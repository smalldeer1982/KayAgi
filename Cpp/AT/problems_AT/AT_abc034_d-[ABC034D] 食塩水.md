# [ABC034D] 食塩水

## 题目描述

有$N$ $(1≤N≤1000)$个装有食盐水的容器。容器从$1$到$N$标号。第$i$号容器有浓度为$p_i\%$ $(0≤p_i≤100)$的食盐水$w_i$ $(1≤w_i≤10^9)$克。高桥君需要选择$K$ $(1≤K≤1000)$个容器，并把选择的容器里的食盐水全部混合在一起。请你编程求出高桥君可以获得的盐水的最大浓度。

## 输入格式

第一行两个整数$N$、$K$；
接下来$N$行，每行两个数$w_i$、$p_i$。

## 输出格式

一行一个实数$x$，表示高桥君能得到的最大浓度的盐水浓度为$x\%$。

## 样例 #1

### 输入

```
3 2
100 15
300 20
200 30

```

### 输出

```
25.000000000

```

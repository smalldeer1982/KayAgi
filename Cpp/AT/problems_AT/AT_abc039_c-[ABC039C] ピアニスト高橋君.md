# [ABC039C] ピアニスト高橋君

## 题目描述

题意描述
高桥君在一个钢琴键上，向右转，查看了20个键的颜色，问他所在的键是哪个？（钢琴键重复着白黑白黑白白黑白黑白黑白，音阶是 Do，#Do，Re，#Re，Mi，Fa，#Fa，So，#So，La，#La,Si）

## 输入格式

一个字符串，W表示白色，B表示黑色

## 输出格式

音阶中的一个，高桥君所在的键盘颜色与输入的第一个相对应

## 样例 #1

### 输入

```
WBWBWWBWBWBWWBWBWWBW
```

### 输出

```
Do
```

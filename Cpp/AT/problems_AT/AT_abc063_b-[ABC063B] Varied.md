# [ABC063B] Varied

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc063/tasks/abc063_b

英小文字からなる文字列 $ S $ が与えられます。$ S $ に含まれる文字がすべて異なるか判定してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ S $

## 输出格式

$ S $ に含まれる文字がすべて異なる場合は `yes`（英小文字）、そうでない場合は `no` と出力せよ。

## 说明/提示

### 制約

- $ 2\ <\ =\ |S|\ <\ =\ 26 $, ここで $ |S| $ は $ S $ の長さを表す。
- $ S $ は英小文字のみからなる。

## 样例 #1

### 输入

```
uncopyrightable
```

### 输出

```
yes
```

## 样例 #2

### 输入

```
different
```

### 输出

```
no
```

## 样例 #3

### 输入

```
no
```

### 输出

```
yes
```

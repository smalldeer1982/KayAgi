# [ABC075A] One out of Three

## 题目描述

读入三个数a,b,c，求只出现一次的数是哪个。

## 输入格式

一行，三个整数a,b,c。

## 输出格式

一行，表示只出现过一次的数
### **输入样例1**
```
5 7 5
```
### **输出样例1**
```
5
```
### **输入样例2**
```
1 1 7
```
### **输出样例2**
```
7
```
### **输入样例3**
```
-100 100 100
```
### **输出样例3**
```
-100
```

## 样例 #1

### 输入

```
5 7 5
```

### 输出

```
7
```

## 样例 #2

### 输入

```
1 1 7
```

### 输出

```
7
```

## 样例 #3

### 输入

```
-100 100 100
```

### 输出

```
-100
```

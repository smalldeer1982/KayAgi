# [ABC079B] Lucas Number

## 题目描述

给你一个数列 $L$，规定：

$L_0=2$

$L_1=1$

而第 $i$ 个数是：$L_i=L_{i-1}+L_{i-2}$。

现在给出一个正整数 $n$，求这个数组的第 $n$ 项。

## 输入格式

一行，一个正整数 $n$。

## 输出格式

一行，即这个数列的第 $n$ 项。

## 说明/提示

$1 \leq n \leq 86$，$L_n$ 保证小于 $10^{18}$。

**【样例解释】**

$L_0=2$

$L_1=1$

$L_2=L_0+L_1=3$

$L_3=L_1+L_2=4$

$L_4=L_2+L_3=7$

$L_5=L_3+L_4=11$

## 样例 #1

### 输入

```
5
```

### 输出

```
11
```

## 样例 #2

### 输入

```
86
```

### 输出

```
939587134549734843
```

# [ABC108A] Pair

## 题目描述

请求出从 $1$ 以上，$K$ 以下（包括$1$和$K$）的正整数中，偶数和奇数各选一个的方法的个数。不考虑选择的顺序，即，$(a,b)$ 和 $(b,a)$ 算作同一种方案。

## 输入格式

一个正整数$K$。

## 输出格式

方案数。

### 数据范围与样例解释

$2 \le K \le 100$ 。

第一组样例：只有 $(2,1)$ 和 $(2,3)$ 两种方案符合条件。

## 样例 #1

### 输入

```
3
```

### 输出

```
2
```

## 样例 #2

### 输入

```
6
```

### 输出

```
9
```

## 样例 #3

### 输入

```
11
```

### 输出

```
30
```

## 样例 #4

### 输入

```
50
```

### 输出

```
625
```

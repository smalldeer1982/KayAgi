# [ABC109A] ABC333

## 题目描述

给出两个正整数 $a, b$ 且 $1 \le a, b \le 3$，判断是否有正整数 $c$ 满足 $1 \le c \le 3$ 且 $a \times b \times c$ 是奇数。若有输出 "Yes"，否则输出 "No"。

## 输入格式

输入两个数 $a, b$。

## 输出格式

如果有数 $c$ 满足条件则输出 "Yes"，否则输出 "No"。

## 样例 #1

### 输入

```
3 1
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
1 2
```

### 输出

```
No
```

## 样例 #3

### 输入

```
2 2
```

### 输出

```
No
```

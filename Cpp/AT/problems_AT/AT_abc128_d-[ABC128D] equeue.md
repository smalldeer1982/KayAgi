# [ABC128D] equeue

## 题目描述

有一个双端队列，初始时队列中共有 $n$ 个元素，元素从头到尾的权值为 $v_{1},v_{2},\cdots,v_{n}$

你可以进行不超过 $k$ 次操作（也可以一次都不操作），每次操作可以选择队头或队尾的一个元素，将它归为己有，或将自己手上的一个元素塞到队头或队尾

问最终你手上所有元素的权值之和的最大值是多少

## 输入格式

第一行两个整数 $n,k$

接下来一行 $n$ 个整数表示 $v_{i}$

## 输出格式

一行一个整数，表示答案

## 说明/提示

$ 1 \le n \le 50,1 \le k \le 100, -10^7 \le v_{i} \le 10^7 $

## 样例 #1

### 输入

```
6 4
-10 8 2 1 2 6
```

### 输出

```
14
```

## 样例 #2

### 输入

```
6 4
-6 -100 50 -2 -5 -3
```

### 输出

```
44
```

## 样例 #3

### 输入

```
6 3
-6 -100 50 -2 -5 -3
```

### 输出

```
0
```

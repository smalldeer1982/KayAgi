# [ABC178A] Not

## 题目描述

输入一个整数$x$。如$x$为$0$，输出$1$； 否则输出$0$。

## 输入格式

输入一个整数，为$1$或$0$。

## 输出格式

输出一个整数$x$，为$1$或$0$。
### 输入输出样例
输入 #1: 1

输出 #1: 0


输入 #2: 0

输出 #2: 1

## 说明/提示

$0≤ x ≤ 1$ 
$x$为整数

## 样例 #1

### 输入

```
1
```

### 输出

```
0
```

## 样例 #2

### 输入

```
0
```

### 输出

```
1
```

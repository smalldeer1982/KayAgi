# [ABC179C] A x B + C

## 题目描述

给定一个正整数 $N$，计算有多少个正整数三元组 $(A,B,C)$ 满足 $A \times B + C = N$。

## 输入格式

输入一行一个正整数 $N$。

## 输出格式

输出一行一个整数，表示满足正整数三元组 $(A,B,C)$ 的个数。

### 【样例解释】

样例 $\#1$ 中满足条件的三元组有：$(1,1,2),(1,2,1),(2,1,1)$。

## 说明/提示

- $2 \le N \le 10^6$。
- 输入全为正整数。

翻译由 @RP_INT_MAX 提供。

## 样例 #1

### 输入

```
3
```

### 输出

```
3
```

## 样例 #2

### 输入

```
100
```

### 输出

```
473
```

## 样例 #3

### 输入

```
1000000
```

### 输出

```
13969985
```

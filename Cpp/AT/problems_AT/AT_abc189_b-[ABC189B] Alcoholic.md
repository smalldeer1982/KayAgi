# [ABC189B] Alcoholic

## 题目描述

高桥君喝酒喝了$N$杯

第$i$个喝的酒是$V_i$ml，酒精度数是$P_i$

高桥君的酒精摄取量超过$X$毫升就会喝醉。

高桥君喝醉是在喝第几杯酒的时候?但是，如果喝了$N$杯酒之后还没有喝醉的话，请代替输出`-1`。

## 输入格式

输入以以下形式从标准输入给出:

$N$ $X$ $V_1$ $P_1$:$V_N$ $P_N$

## 输出格式

高桥君喝醉是在喝了第几杯酒的时候。但是，喝了$N$杯所有的酒之后也没有喝醉的情况，代替输出`-1`。

## 样例 #1

### 输入

```
2 15
200 5
350 3
```

### 输出

```
2
```

## 样例 #2

### 输入

```
2 10
200 5
350 3
```

### 输出

```
2
```

## 样例 #3

### 输入

```
3 1000000
1000 100
1000 100
1000 100
```

### 输出

```
-1
```

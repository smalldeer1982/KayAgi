# [ABC196C] Doubled

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc196/tasks/abc196_c

整数 $ N $ が与えられます。  
 以下の条件を満たす $ 1 $ 以上 $ N $ 以下の整数 $ x $ は何個あるでしょうか？

- $ x $ の十進表記 (先頭に $ 0 $ を付けない) は偶数桁であり、その前半と後半は文字列として等しい。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ N $ は整数
- $ 1\ <\ =\ N\ <\ 10^{12} $

### Sample Explanation 1

$ 11,\ 22,\ 33 $ の $ 3 $ 個が条件を満たします。

### Sample Explanation 2

例えば $ 1313 $ は、十進表記が $ 4 $ 桁で、その前半も後半も $ 13 $ であるため条件を満たします。

## 样例 #1

### 输入

```
33
```

### 输出

```
3
```

## 样例 #2

### 输入

```
1333
```

### 输出

```
13
```

## 样例 #3

### 输入

```
10000000
```

### 输出

```
999
```

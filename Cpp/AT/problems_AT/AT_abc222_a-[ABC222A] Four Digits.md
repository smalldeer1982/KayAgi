# [ABC222A] Four Digits

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc222/tasks/abc222_a

$ 0 $ 以上 $ 9999 $ 以下の整数 $ N $ が与えられます。

$ N $ の先頭に必要なだけ $ 0 $ をつけ、$ 4 $ 桁の文字列にしたものを出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ 0\ \leq\ N\ \leq\ 9999 $
- $ N $ は整数

### Sample Explanation 1

$ 321 $ は $ 3 $ 桁なので、先頭に $ 1 $ つ $ 0 $ をつけると $ 4 $ 桁になります。

## 样例 #1

### 输入

```
321
```

### 输出

```
0321
```

## 样例 #2

### 输入

```
7777
```

### 输出

```
7777
```

## 样例 #3

### 输入

```
1
```

### 输出

```
0001
```

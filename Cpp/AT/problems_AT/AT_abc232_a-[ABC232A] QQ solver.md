# [ABC232A] QQ solver

## 输入格式

一串形如 $a* b$ 长度为 $3$ 的字符串 $S$。

## 输出格式

输出这个字符串中 $a* b$ 的值。

**记得换行**

## 样例 #1

### 输入

```
3x7
```

### 输出

```
21
```

## 样例 #2

### 输入

```
9x9
```

### 输出

```
81
```

## 样例 #3

### 输入

```
1x1
```

### 输出

```
1
```

# [ABC236A] chukodai

## 输入格式

输入一个字符串 $S(2\le|S|\le10)$和两个正整数 $a,b(1\le a,b\le|s|)$。

## 输出格式

将这个字符串的第 $a$ 和第 $b$ 个字符调换后输出。

**记得换行！！**

## 样例 #1

### 输入

```
chokudai
3 5
```

### 输出

```
chukodai
```

## 样例 #2

### 输入

```
aa
1 2
```

### 输出

```
aa
```

## 样例 #3

### 输入

```
aaaabbbb
1 8
```

### 输出

```
baaabbba
```

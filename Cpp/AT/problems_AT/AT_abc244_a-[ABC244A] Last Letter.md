# [ABC244A] Last Letter

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc244/tasks/abc244_a

英小文字からなる長さ $ N $ の文字列 $ S $ が与えられます。$ S $ の末尾の文字を出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ S $

## 输出格式

$ S $ の末尾の文字を出力せよ。

## 说明/提示

### 制約

- $ N $ は整数
- $ 1\ <\ =\ N\ <\ =\ 1000 $
- $ S $ は英小文字からなる長さ $ N $ の文字列

### Sample Explanation 1

$ S\ =\ {} $`abcde` です。$ S $ の末尾の文字は `e` なので、`e` を出力します。

## 样例 #1

### 输入

```
5
abcde
```

### 输出

```
e
```

## 样例 #2

### 输入

```
1
a
```

### 输出

```
a
```

# [ABC277C] Ladder Takahashi

## 题目描述

【题面翻译】

有一座很高的楼，你现在在第一层。

有 $n$ 个传送门，每个传送门连接第 $a_i$ 层与 $b_i$ 层。传送门是双向的。

请你求出你能到达的最高楼层。

translated by @[liangbowen](https://www.luogu.com.cn/user/367488).

## 输入格式

第一行，一个整数 $n$。

接下来 $n$ 行，每行两个数 $a_i$，$b_i$，表示传送门。

## 输出格式

输出你能到达的最高楼层。

## 说明/提示

$1 \le n \le 2 \times 10^5$

$1 \le a_i, b_i \le 10^9$

保证 $a_i \ne b_i$。

## 样例 #1

### 输入

```
4
1 4
4 3
4 10
8 3
```

### 输出

```
10
```

## 样例 #2

### 输入

```
6
1 3
1 5
1 12
3 5
3 12
5 12
```

### 输出

```
12
```

## 样例 #3

### 输入

```
3
500000000 600000000
600000000 700000000
700000000 800000000
```

### 输出

```
1
```

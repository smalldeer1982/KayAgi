# [ABC300A] N-choice question

## 题目描述

给定 $A,B$，在 $C$ 集合中找到 $A+B$。输出其编号。

保证输出唯一。

## 输入格式

第一行三个整数 $N,A,B$，表示 $|C|$（$C$ 集合的大小）、$A$、$B$。

第二行 $N$ 个整数，第 $i$ 个整数表示 $C_i$。

## 输出格式

一行一个整数，表示答案。

## 样例 #1

### 输入

```
3 125 175
200 300 400
```

### 输出

```
2
```

## 样例 #2

### 输入

```
1 1 1
2
```

### 输出

```
1
```

## 样例 #3

### 输入

```
5 123 456
135 246 357 468 579
```

### 输出

```
5
```

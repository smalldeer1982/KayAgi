# [ABC301A] Overall Winner

## 题目描述

两个人比赛，用字符串给出比赛的情况：字符 `A` 表示 A 赢了，字符 `T` 表示 T 赢了，最终的胜者是胜场更多的那一方，如果相同，则先达到总胜场的一方获胜。输出胜方。

## 输入格式

第一行一个正整数 $N$，表示字符串的长度。

第二行一个长度为 $N$ 的字符串。

## 输出格式

共一行一个字符，表示胜方。

## 样例 #1

### 输入

```
5
TTAAT
```

### 输出

```
T
```

## 样例 #2

### 输入

```
6
ATTATA
```

### 输出

```
T
```

## 样例 #3

### 输入

```
1
A
```

### 输出

```
A
```

# [ABC312B] TaK Code

## 题目描述

**题意简述**

对 Tak 二维码的定义为：

- 一个 $9 \times 9$ 的黑白方阵。
- 左上角和右下角 $3 \times 3$ 的方块都为黑块，并且它们周围一圈的 $14$ 个方块都为白块。

给定一个图，求出其中所有 Tak 二维码的左上角字符的坐标。`#` 代表黑块，`.` 代表白块。

如果没看懂，请看下图。其中 `#` 的位置必须为黑块，`.` 的位置必须为白块，`@` 的位置都可以。
```
###.@@@@@
###.@@@@@
###.@@@@@
....@@@@@
@@@@@@@@@
@@@@@....
@@@@@.###
@@@@@.###
@@@@@.###
```

## 输入格式

第一行为两个整数 $N,M$。

接下来 $N$ 行，每行有 $M$ 个字符，表示这个方阵。

## 输出格式

所有 Tak 码的左上角字符的坐标，用换行隔开。如果没有则不输出。

## 说明/提示

- $9 \le N,M \le 100$

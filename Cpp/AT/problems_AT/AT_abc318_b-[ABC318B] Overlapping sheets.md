# [ABC318B] Overlapping sheets

## 题目描述

**题意简述**

在平面直角坐标系中有 $N$ 个矩形，每个矩形的左上角的位置是 $(A_i,C_i)$，右下角的位置是 $(B_i,D_i)$。这些都是指平面直角坐标系上的点，不是指的方格。求所有矩形的覆盖面积总和。

## 输入格式

第一行有一个整数 $N$。

接下来 $N$ 行，每行有 $4$ 个整数 $A_i,B_i,C_i,D_i$，如题意。

## 输出格式

$1$ 个整数，如题意。

## 说明/提示

- $ 2 \le N \le 100 $
- $ 0 \le A_i < B_i \leq 100 $
- $ 0 \le C_i < D_i \leq 100 $

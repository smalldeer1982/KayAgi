# [ABC320A] Leyland Number

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc320/tasks/abc320_a

正整数 $ A,B $ が与えられます。  
 $ A^B+B^A $ の値を出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ A $ $ B $

## 输出格式

答えを整数として出力せよ。

## 说明/提示

### 制約

- $ 2\ \leq\ A\ \leq\ B\ \leq\ 9 $
- 入力される数値はすべて整数
 
### Sample Explanation 1

$ A\ =\ 2,\ B\ =\ 8 $ のとき、$ A^B\ =\ 256,\ B^A\ =\ 64 $ なので $ A^B\ +\ B^A\ =\ 320 $ です。

## 样例 #1

### 输入

```
2 8
```

### 输出

```
320
```

## 样例 #2

### 输入

```
9 9
```

### 输出

```
774840978
```

## 样例 #3

### 输入

```
5 6
```

### 输出

```
23401
```

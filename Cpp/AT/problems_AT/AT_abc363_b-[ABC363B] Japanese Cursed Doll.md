# [ABC363B] Japanese Cursed Doll

## 题目描述

有 $ N $ 个人，第 $ i $ 个人（$ 1\leq\ i\leq\ N $）的当前头发长度为 $ L_i $。  
每个人的头发长度每过一天都会增长 $ 1 $。

请输出第一次出现头发长度达到 $ T $ 或以上的人数超过 $ P $ 的天数，即从现在开始多少天后。  
如果在当前时刻已经有 $ P $ 个或更多头发长度达到 $ T $ 以上的人，则输出 $ 0 $。

## 输入格式

从标准输入中按以下格式给出输入。

> $ N $ $ T $ $ P $ $ L_1 $ $ L_2 $ $ \ldots $ $ L_N $

## 输出格式

输出一个整数，表示第一次出现头发长度达到 $ T $ 或以上的人数超过 $ P $ 的天数。 如果在当前时刻已经满足条件，则输出 $ 0 $。

### 约束条件

- $ 1\leq\ N\leq\ 100 $
- $ 1\leq\ L_i\leq\ 100 $
- $ 1\leq\ T\leq\ 100 $
- $ 1\leq\ P\leq\ N $
- 输入均为整数

### 样例解释 1

有 $ 5 $ 个人，当前时刻的头发长度分别为 $ 3,11,1,6,2 $，因此头发长度达到 $ 10 $ 或以上的人有 $ 1 $ 人。 从现在开始 $ 7 $ 天后，每个人的头发长度分别为 $ 10,18,8,13,9 $，头发长度达到 $ 10 $ 或以上的人有 $ 3 $ 人。 在当前时刻的 $ 6 $ 天后，头发长度达到 $ 10 $ 或以上的人有 $ 2 $ 人，因此不满足条件，所以输出 $ 7 $。

### 样例解释 2

由于当前时刻已经有 $ 2 $ 个头发长度达到 $ 5 $ 或以上的人，满足条件，所以输出 $ 0 $。

## 样例 #1

### 输入

```
5 10 3
3 11 1 6 2
```

### 输出

```
7
```

## 样例 #2

### 输入

```
2 5 2
10 10
```

### 输出

```
0
```

## 样例 #3

### 输入

```
3 10 1
1 2 3
```

### 输出

```
7
```

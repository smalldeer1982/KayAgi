# Integer Preference

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abl/tasks/abl_b

すぬけ君は $ A $ 以上 $ B $ 以下の整数が好きです。 高橋君は $ C $ 以上 $ D $ 以下の整数が好きです。

どちらも好きな整数は存在しますか？

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ A $ $ B $ $ C $ $ D $

## 输出格式

`Yes` または `No` と出力せよ。

## 说明/提示

### 制約

- $ 0\ \leq\ A\ \leq\ B\ \leq\ 10^{18} $
- $ 0\ \leq\ C\ \leq\ D\ \leq\ 10^{18} $
- 入力は全て整数である。

### Sample Explanation 1

たとえば、どちらも $ 25 $ が好きです。

## 样例 #1

### 输入

```
10 30 20 40
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
10 20 30 40
```

### 输出

```
No
```

# [AGC007B] Construct Sequences

## 题目描述

# 题意简述
给你一段长度为n的数列$P_1$,$P_2$,,$P_n$.
让你构造两个新的数列a与b,且满足：   
1.    1<$a_i$,$b_i$<10^9                     
2.    $a_1$<$a_2$<…<$a_n$
3.    $b_1$>$b_2$>…>$b_n$
4.    $a_{p_1}$+$b_{p_1}$<$a_{p_2}$+$b_{p_2}$<…<$a_{p_n}$+$b_{p_n}$
求出满足条件的数列a,b；

## 输入格式

两行，第一行为n,第二行为数列P

## 输出格式

两行，第一行为数列a,第二行为数列b

感谢@该起什么名字 提供的翻译

## 样例 #1

### 输入

```
2
1 2
```

### 输出

```
1 4
5 4
```

## 样例 #2

### 输入

```
3
3 2 1
```

### 输出

```
1 2 3
5 3 1
```

## 样例 #3

### 输入

```
3
2 3 1
```

### 输出

```
5 10 100
100 10 1
```

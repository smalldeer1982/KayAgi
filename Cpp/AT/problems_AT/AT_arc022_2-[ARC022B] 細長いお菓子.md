# [ARC022B] 細長いお菓子

## 题目描述

### 题目简述

有一个长度为$N$的序列。

请从这个序列里面找出**满足全部的值都是不相同的**子序列。

请输出最长子序列的**这个长度**。

## 输入格式

第一行是一个正整数$N(1 \le N \le 10^5)$。

第二行是$N$个正整数用空格分隔，表示这个序列的全部的值。

## 输出格式

输出一个正整数，表示满足要求的最长子序列的长度

## 样例 #1

### 输入

```
7
1 2 1 3 1 4 4
```

### 输出

```
3
```

## 样例 #2

### 输入

```
1
100
```

### 输出

```
1
```

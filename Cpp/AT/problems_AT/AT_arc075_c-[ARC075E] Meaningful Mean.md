# [ARC075E] Meaningful Mean

## 题目描述

**题目大意：**
给一个长度为N的整数序列{a},对于其一共N*(N+1)/2个的非空区间，求有多少个区间的平均数大于等于K。

## 输入格式

第一行两个数字分别为 N K
接下来N行 每行一个数字表示a[i]

## 输出格式

输出一个整数 代表有多少个区间的平均数大于等于K

## 说明/提示

输入的全部为整数

1 ≤ N ≤ 2×10^5

1 ≤ K ≤ 10^9

1 ≤ a[i] ≤ 10^9

## 样例 #1

### 输入

```
3 6
7
5
7
```

### 输出

```
5
```

## 样例 #2

### 输入

```
1 2
1
```

### 输出

```
0
```

## 样例 #3

### 输入

```
7 26
10
20
30
40
30
20
10
```

### 输出

```
13
```

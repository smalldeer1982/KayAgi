# [ARC113B] A^B^C

## 题目描述

这里有 $3$ 个正整数。请求出 $ A^{B^C} $ 的个位。

## 输入格式

按以下格式输入。

> $ A $ $ B $ $ C $

## 输出格式

输出 $ A^{B^C} $ 个位上的数。

### 样例输入 #1

```
4 3 2
```

### 样例输出 #1

```
4
```

## 样例 #2

### 样例输入 #2

```
1 2 3
```

### 样例输出 #2

```
1
```

## 样例 #3

### 样例输入 #3

```
3141592 6535897 9323846
```

### 样例输出 #3

```
2
```

## 说明/提示

- $ 1\leq\ A,B,C\ \leq\ 10^9 $
- $ A,B,C $ 是整数。


$ 4^{3^2}=4^9=262144 $ 的个位是 $ 4 $ 。

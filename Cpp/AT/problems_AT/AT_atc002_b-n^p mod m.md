# n^p mod m

## 题目描述

求 $n^p$ $mod$ $m$ 的值

## 输入格式

一行，为整数 n,m,p(注意顺序)

## 输出格式

一行，为 $n^p$ $mod$ $m$ 的值

## 说明/提示

对于100%的数据 $1≤n,m≤10^9$ $1≤p≤10^{14}$

翻译提供者：Ryan_wxn_

## 样例 #1

### 输入

```
12 15 7
```

### 输出

```
3
```

## 样例 #2

### 输入

```
123456789 234567894 6574837563712
```

### 输出

```
120678297
```

# 円周率

## 题目描述

题目大意：
输入一个数字n，问n出现在圆周率的小数点以下几位（取最前的一位，3在圆周率的第0位上）。

## 输入格式

输入一个数字n

## 输出格式

n出现在圆周率的小数点以下几位

感谢@Andysun06  提供的翻译

## 样例 #1

### 输入

```
0
```

### 输出

```
32
```

## 样例 #2

### 输入

```
1
```

### 输出

```
1
```

## 样例 #3

### 输入

```
2
```

### 输出

```
6
```

## 样例 #4

### 输入

```
3
```

### 输出

```
0
```

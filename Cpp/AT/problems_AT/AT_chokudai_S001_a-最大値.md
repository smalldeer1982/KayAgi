# 最大値

## 题目描述

题目大意：
在数列中找到最大值

## 输入格式

第一行：n
第二行：a【1】~a【n】

## 输出格式

最大值

## 样例 #1

### 输入

```
5
3 1 5 4 2
```

### 输出

```
5
```

## 样例 #2

### 输入

```
6
1 2 3 4 5 6
```

### 输出

```
6
```

## 样例 #3

### 输入

```
7
7 6 5 4 3 2 1
```

### 输出

```
7
```

## 样例 #4

### 输入

```
20
19 11 10 7 8 9 17 18 20 4 3 15 16 1 5 14 6 2 13 12
```

### 输出

```
20
```

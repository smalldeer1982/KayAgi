# 長方形 α

## 题目描述

[problemUrl]: https://atcoder.jp/contests/chokudai_S002/tasks/chokudai_S002_a

$ N $ 個の長方形があり、長方形 $ i $ の幅と高さはそれぞれ $ A_i,\ B_i $ です。

各長方形の面積を求めてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ A_1 $ $ B_1 $ $ A_2 $ $ B_2 $ $ : $ $ A_N $ $ B_N $

## 输出格式

$ i\ (1\ \leq\ i\ \leq\ N) $ 行目に長方形 $ i $ の面積を出力せよ。

## 说明/提示

### 制約

入力は以下の条件を満たす。

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ A_i,B_i\ \leq\ 10^9 $
- 入力される値は全て整数

## 样例 #1

### 输入

```
3
3 4
1000000000 1
111111111 111111111
```

### 输出

```
12
1000000000
12345678987654321
```

# あまり α

## 题目描述

[problemUrl]: https://atcoder.jp/contests/chokudai_S002/tasks/chokudai_S002_b

$ N $ 個の以下の問題に答えてください。

- 整数 $ A_i $ を整数 $ B_i $ で割った余りを求めよ。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ A_1 $ $ B_1 $ $ A_2 $ $ B_2 $ $ : $ $ A_N $ $ B_N $

## 输出格式

$ i\ (1\ \leq\ i\ \leq\ N) $ 行目に $ A_i $ を $ B_i $ で割った余りを出力せよ。

## 说明/提示

### 制約

入力は以下の条件を満たす。

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ A_i,B_i\ \leq\ 10^9 $
- 入力される値は全て整数

## 样例 #1

### 输入

```
3
10 3
1000000000 1
11 92
```

### 输出

```
1
0
11
```

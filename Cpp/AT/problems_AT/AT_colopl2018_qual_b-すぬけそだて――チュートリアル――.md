# すぬけそだて――チュートリアル――

## 题目描述

一个文字游戏（~~日式galgame？？？~~）。  
有n段加载或者剧情，  
n段每段有不同的时间长度- val[i]。  
给出 一个长度为n的01串s

s[i]=-0 表示 为加载 不能跳过
s[i]==1 表示 为剧情 可耗费 x时间跳过  
求最小游玩时间（~~不看剧情玩galgame，玩锤子）~~

## 输入格式

n x  
长度为n 的 01串  
n段 每段的 val[i]

## 输出格式

最小时间总和res

## 样例 #1

### 输入

```
3 5
011
8
10
3
```

### 输出

```
16
```

## 样例 #2

### 输入

```
5 314
10101
159
265
358
979
323
```

### 输出

```
2031
```

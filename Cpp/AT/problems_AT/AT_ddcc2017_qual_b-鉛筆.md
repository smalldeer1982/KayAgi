# 鉛筆

## 题目描述

[problemUrl]: https://atcoder.jp/contests/ddcc2017-qual/tasks/ddcc2017_qual_b

鉛筆 $ 1 $ ダースは $ 12 $ 本、 $ 1 $ グロスは $ 144 $ 本、 $ 1 $ グレートグロスは $ 1728 $ 本です。

$ A $ グレートグロスの鉛筆と、 $ B $ グロスの鉛筆と、 $ C $ ダースの鉛筆と、 $ D $ 本の鉛筆があるとき、合計で鉛筆は何本あるか答えてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ A $ $ B $ $ C $ $ D $

## 输出格式

合計で鉛筆が $ x $ 本あるとき、 $ x $ を出力せよ。

## 说明/提示

### 制約

- $ 0≦A,B,C,D≦11 $
- 入力は整数からなる

### Sample Explanation 1

$ 1 $ グレートグロスは $ 1728 $ 本であり、$ 2 $ グロスは $ 288 $本、 $ 4 $ ダースは $ 48 $ 本なので、合計で $ 2064 $ 本の鉛筆があります。

### Sample Explanation 2

鉛筆が一本もないこともあります。

## 样例 #1

### 输入

```
1 2 4 0
```

### 输出

```
2064
```

## 样例 #2

### 输入

```
0 0 0 0
```

### 输出

```
0
```

## 样例 #3

### 输入

```
8 11 5 4
```

### 输出

```
15472
```

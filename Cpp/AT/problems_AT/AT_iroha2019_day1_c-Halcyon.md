# Halcyon

## 题目描述

Halcyon Days 是 $12$ 月 $N$ 日的冬至和之前的 $1$ 个星期。给出冬至的日期 $N$，输出 $8$ 天 Halcyon Days 的日期。

## 输入格式

$1$ 行，输入 $ N $。

> $ N $

## 输出格式

请将 Halcyon Days 的日期按升序（按从小到大的顺序）输出，每 $1$ 行输入 $1$ 个数，共输 $8$ 行。

## 说明/提示

- $ 8\ \leq\ N\ \leq\ 31 $

样例 $1$ 解释：

冬至是 $ 21$ 日，所以 Halcyon Days 的第一天是 $ 14$ 日。

## 样例 #1

### 输入

```
21
```

### 输出

```
14
15
16
17
18
19
20
21
```

## 样例 #2

### 输入

```
8
```

### 输出

```
1
2
3
4
5
6
7
8
```

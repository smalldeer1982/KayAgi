# ルーク

## 题目描述

有一个 $4000$ 行 $4000$ 列的国际象棋棋盘，其中的 $n$ 个格子内放了车。现在要去掉其中的一部分车，使得剩下的车不互相攻击。要求去掉的车尽可能少。对于每个棋子，输出它是否必须去除或留下。

## 输入格式

第一行为一个整数 $n$。

接下来 $n$ 行，每行两个整数，车的坐标。

## 输出格式

车的去留。

## 样例 #1

### 输入

```
6
1 1
1 3
2 1
2 2
2 3
3 2
```

### 输出

```
3
0
0
0
1
0
-1
```

## 样例 #2

### 输入

```
9
1 2
2 2
2 1
3 3
4 6
4 5
4 4
5 4
6 4
```

### 输出

```
4
-1
1
-1
-1
0
0
1
0
0
```

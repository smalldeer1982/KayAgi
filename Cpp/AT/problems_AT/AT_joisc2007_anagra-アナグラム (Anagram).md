# アナグラム (Anagram)

## 题目描述

### 题目简述

给定一个大写英文字母串 $S$，问将 $S$ 的所有字母重新排列而成的字符串中，$S$ 是第几个？

## 输入格式

一行一个字符串 $S$（$|S|\le 20$）。

## 输出格式

一行一个整数，题目所求。

### 输入输出样例

#### 输入 #1

```
HEART
```

#### 输出 #1

```
55
```

#### 输入 #2

```
IOI
```

#### 输出 #2

```
2
```

## 说明/提示

请注意，$2^{32}\lt 20!\lt 2^{63}$。

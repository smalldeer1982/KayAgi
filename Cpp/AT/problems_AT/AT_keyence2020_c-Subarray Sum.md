# Subarray Sum

## 题目描述

输入三个整数，N，K，S。
请你找到一个N个元素的整数序列，其中每一个元素在1，10的9次方之间。这个序列满足K个子序列的和为S。

## 样例 #1

### 输入

```
4 2 3
```

### 输出

```
1 2 3 4
```

## 样例 #2

### 输入

```
5 3 100
```

### 输出

```
50 50 50 30 70
```

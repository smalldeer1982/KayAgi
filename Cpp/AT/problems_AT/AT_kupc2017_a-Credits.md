# Credits

## 题目描述

## 题目背景

京都大学是残酷的。


在大学里，开设了N门课程，但是积分如果小于K分就会被残酷地留级。

已知上第i门课程会获得 $a_i$ 点积分。

你不想被留级（~~会被家长骂~~），但同时你也不想去学校（~~毕竟学校是个无聊的地方~~），请你编写一个程序，计算你最少上几门课程不会被留级。

## 输入格式

```
N K
a_1 a_2 …… a_n
```

## 输出格式

输出你最少上的课程数。如果你上所有课程都无法不被留级（意思就是一定会被留级），输出-1。

> 感谢 [rui_er](https://www.luogu.org/space/show?uid=122461) 提供的翻译。

## 样例 #1

### 输入

```
5 15
3 8 2 5 6
```

### 输出

```
3
```

## 样例 #2

### 输入

```
5 25
3 8 2 5 6
```

### 输出

```
-1
```

## 样例 #3

### 输入

```
4 20
1 1 20 19
```

### 输出

```
1
```

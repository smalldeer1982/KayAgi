# MUJIN

## 题目描述

来看电脑标准键盘，它上面的`M`,`U`,`J`,`I`,` N`五个键连成了一个块。现在输入一个上文中没有出现过的大写字母，请问在键盘上它在这个块的左边还是右边？

## 输入格式

一个不为上面五个字母之一的大写英文字母。

## 输出格式

若该大写字母在这个块的左边输出`Left`，否则输出`Right`。**输出完毕后，请换行。**

### 输入输出样例

#### 输入 #1
```
A
```
#### 输出 #1
```
Left
```
#### 输入 #2
```
P
```
#### 输出 #2
```
Right
```
#### 输入 #3
```
H
```
#### 输出 #3
```
Left
```

# 出現回数

## 题目描述

[problemUrl]: https://atcoder.jp/contests/past202109-open/tasks/past202109_c

長さ $ N $ の整数列 $ A\ =\ (A_1,A_2,\ldots\ ,A_N) $ が与えられます。この整数列の中に整数 $ X $ は何回現れますか？

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ X $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $

## 输出格式

整数列 $ A $ の中に整数 $ X $ が現れる回数、すなわち $ A_i=X $ をみたす $ i $ の個数を出力せよ。

## 说明/提示

### 注意

### 制約

- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ 1\ \leq\ X\ \leq\ 10^9 $
- $ 1\ \leq\ A_i\ \leq\ 10^9 $
- 入力は全て整数である。

### Sample Explanation 1

整数列 $ A $ の中に $ 3 $ という整数は $ 2 $ 回現れます。

## 样例 #1

### 输入

```
3 3
2 3 3
```

### 输出

```
2
```

## 样例 #2

### 输入

```
3 1
2 3 3
```

### 输出

```
0
```

## 样例 #3

### 输入

```
5 2
2 3 4 2 2
```

### 输出

```
3
```

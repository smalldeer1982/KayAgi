# K番目の絶対値

## 题目描述

给定一个长为 $n$ 的数列 $a$。

众所周知，一个长为 $n$ 的序列中，有 $\frac{n(n+1)}{2}$ 个连续子序列。你需要求出这些连续子序列中，**子序列和的绝对值**的第 $k$ 小值。

## 输入格式

第一行输入两个整数 $n,k$。

第二行输入 $n$ 个整数，表示数列 $a$。

## 输出格式

一行一个整数，题目所求。

## 说明/提示

#### 样例 #1 解释

三个子序列的和的绝对值分别是 $2,3,1$，其中 $2$ 是第 $2$ 小的。

#### 样例 #2 解释

答案有可能爆`int`。

#### 数据规模与约定

对于全部测试点，数据保证：

- $2\le n\le 3\times 10^5$；
- $1\le k\le \frac{n(n+1)}{2}$；
- $-10^9\le a_i\le 10^9$。

## 样例 #1

### 输入

```
2 2
2 -3
```

### 输出

```
2
```

## 样例 #2

### 输入

```
5 10
1000000000 1000000000 1000000000 1000000000 1000000000
```

### 输出

```
3000000000
```

## 样例 #3

### 输入

```
7 22
3 -1 4 -1 5 -9 2
```

### 输出

```
5
```

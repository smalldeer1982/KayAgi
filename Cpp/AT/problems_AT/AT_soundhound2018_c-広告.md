# 広告

## 题目描述

有$r$行$c$列的矩阵，矩阵中的点有`.`，还有`*`。

要求选尽量多的`.`，使其两两不相邻。

## 输入格式

第一行$r$和$c$。

接下来$r$行，每行$c$个字符，是矩阵。

## 输出格式

一行，为最多能选多少个`.`

## 说明/提示

$1\leq r,c\leq 40$

矩阵中只有`.字符`和`*字符`

## 样例 #1

### 输入

```
3 3
...
...
...
```

### 输出

```
5
```

## 样例 #2

### 输入

```
2 2
**
**
```

### 输出

```
0
```

## 样例 #3

### 输入

```
1 1
.
```

### 输出

```
1
```

## 样例 #4

### 输入

```
3 4
*..*
..**
*...
```

### 输出

```
4
```

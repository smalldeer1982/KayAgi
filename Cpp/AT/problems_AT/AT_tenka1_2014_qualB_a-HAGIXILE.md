# HAGIXILE

## 题目描述

天下第一王国的托莫阿基被任命为一篇文章的校对。  
在那篇文章中，天下第一王国的名人 `HAGIXILE` 的名字被误表记为 `HAGIYA`。  
给定的字符串中包含一个字符串 `HAGIYA` ，请替换为 `HAGIXILE` 输出。

## 输入格式

输入共一行一个字符串 $S$。  
其中 $S$ 满足下列条件：  
* $S$ 仅由大写字母组成；  
* $S$ 的长度 $|S|$ 满足 $6 \leq |S| \leq 1000$；  
* $S$ 中 `HAGIYA` 正好只出现一次；
* 输出以换行结束，不包括在 $S$ 中。

## 输出格式

共一行，转换后的字符串，结尾必须有换行符。

## 样例 #1

### 输入

```
MRHAGIYA
```

### 输出

```
MRHAGIXILE
```

# Bit Smaller

## 题目描述

### 题面翻译
求出满足以下条件的所有数字：
- $1\le n\le 20181224$；
- 至少有一种拆分能将 $n$ 拆分成 $2\ \cdot\  k$ 个数 $a_1,b_1,a_2,b_2,\cdots,a_k,b_k$，并使得 $n=a_1^{b_1}\ \cdot\ a_2^{b_2}\ \cdot\ a_3^{b_3}\ \cdots \ a_k^{b_k}$

请按从小到大的顺序输出这些数字。

## 输入格式

无输入。

## 输出格式

共 $x$ 行，每行包含一个满足以上条件的数（$x$ 为满足以上条件的数的个数）。

例如：
```
34425
312325
...
```

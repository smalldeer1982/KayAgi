# [ABC388A] ?UPC

## 题目描述

给定一个字符串 $S$。其中，$S$ 的第 $1$ 个字符为英文大写字母，第 $2$ 个及之后的字符为英文小写字母。

请输出 $S$ 的第 $1$ 个字符与 `UPC` 按此顺序连接而成的字符串。

## 说明/提示

### 限制条件

- $S$ 是长度在 $1$ 到 $100$ 之间的字符串。
- $S$ 的第 $1$ 个字符为英文大写字母。
- $S$ 的第 $2$ 个及之后的字符为英文小写字母。

### 样例解释 1

`Kyoto` 的第 $1$ 个字符是 `K`，因此输出将为 `KUPC`。

由 ChatGPT 4.1 翻译

## 样例 #1

### 输入

```
Kyoto```

### 输出

```
KUPC```

## 样例 #2

### 输入

```
Tohoku```

### 输出

```
TUPC```

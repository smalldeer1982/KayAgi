# [ABC318A] Full Moon

## 题目描述

**题意简述**

满月第一次出现是在第 $M$ 天，之后每个 $P$ 天出现一次。问在 $N$ 天以内有多少次满月。

## 说明/提示

- $1 \le N \le 2 \times 10^5$
- $1 \le M \le P \le 2 \times 10^5$

**样例一解释**

有 $3$ 次，天数分别为 $3,8,13$。

## 样例 #1

### 输入

```
13 3 5```

### 输出

```
3```

## 样例 #2

### 输入

```
5 6 6```

### 输出

```
0```

## 样例 #3

### 输入

```
200000 314 318```

### 输出

```
628```

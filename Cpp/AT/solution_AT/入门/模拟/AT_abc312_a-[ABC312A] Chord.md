# [ABC312A] Chord

## 题目描述

给定一个由大写英文字母组成的长度为 $3$ 的字符串 $S$。如果 $S$ 等于 `ACE`、`BDF`、`CEG`、`DFA`、`EGB`、`FAC`、`GBD` 中的任意一个，请输出 `Yes`，否则输出 `No`。

## 说明/提示

## 限制条件

- $S$ 是由大写英文字母组成的长度为 $3$ 的字符串。

## 样例解释 1

当 $S = $ `ABC` 时，$S$ 不等于 `ACE`、`BDF`、`CEG`、`DFA`、`EGB`、`FAC`、`GBD` 中的任何一个，因此输出 `No`。

由 ChatGPT 4.1 翻译

## 样例 #1

### 输入

```
ABC```

### 输出

```
No```

## 样例 #2

### 输入

```
FAC```

### 输出

```
Yes```

## 样例 #3

### 输入

```
XYX```

### 输出

```

# [ABC358A] Welcome to AtCoder Land

## 题目描述

> 高桥正在去往AtCoder Land。他面前有个告示牌，他想确定上面是不是写着AtCoder Land

给出两个字符串 $ S $ 和 $ T $，判断是否满足 $ S = $ `AtCoder` 且 $ T = $ `Land`。

## 样例 #1

### 输入

```
AtCoder Land```

### 输出

```
Yes```

## 样例 #2

### 输入

```
CodeQUEEN Land```

### 输出

```
No```

## 样例 #3

### 输入

```
aTcodeR lANd```

### 输出

```
No```

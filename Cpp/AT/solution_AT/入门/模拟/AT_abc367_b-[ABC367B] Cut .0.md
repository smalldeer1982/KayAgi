# [ABC367B] Cut .0

## 题目描述

### 问题描述

给出一个实数 \( X \)，精确到小数点后第三位。

以满足下列条件的方式输出实数 \( X \)：

- 对于小数部分，不要在末尾添加 `0`。
- 不要在末尾添加不必要的小数点。

## 样例 #1

### 输入

```
1.012```

### 输出

```
1.012```

## 样例 #2

### 输入

```
12.340```

### 输出

```
12.34```

## 样例 #3

### 输入

```
99.900```

### 输出

```
99.9```

## 样例 #4

### 输入

```
0.000```

### 输出

```
0```

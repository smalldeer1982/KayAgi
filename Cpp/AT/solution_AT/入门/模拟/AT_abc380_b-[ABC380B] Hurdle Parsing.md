# [ABC380B] Hurdle Parsing

## 题目描述

いろは酱有一个长度为 $N$（$N \geq 1$）的正整数序列 $A=(A_1,A_2,\dots,A_N)$。
いろは酱使用 $A$ 生成了一个字符串 $S$，生成方法如下：

- $S$ 以 `|` 开头。
- 按照 $i=1,2,\dots,N$ 的顺序，进行如下操作：
  - 在 $S$ 的末尾添加 $A_i$ 个 `-`。
  - 然后在 $S$ 的末尾添加 $1$ 个 `|`。

给定生成后的字符串 $S$，请还原出正整数序列 $A$。

## 说明/提示

## 限制

- $S$ 是按照题目描述的方法生成的，长度在 $3$ 到 $100$ 之间。
- $A$ 是长度至少为 $1$ 的正整数序列。

## 样例解释 1

$S=$ `|---|-|----|-|-----|` 是由 $A=(3,1,4,1,5)$ 生成的。

由 ChatGPT 4.1 翻译

## 样例 #1

### 输入

```
|---|-|----|-|-----|```

### 输出

```
3 1 4 1 5```

## 样例 #2

### 输入

```
|----------|```

### 输出

```
10```

## 样例 #3

### 输入

```
|-|-|-|------|```

### 输出

```

# Touitsu

## 题目描述

[problemUrl]: https://atcoder.jp/contests/nikkei2019-qual/tasks/nikkei2019_qual_b

三つの文字列 $ A,\ B,\ C $ が与えられます。これらはそれぞれ、英小文字からなる長さ $ N $ の文字列です。

私たちの目標は、これら三つの文字列をすべて等しくすることです。そのために、あなたは次の操作を繰り返し行うことができます。

- 操作: 文字列 $ A,\ B,\ C $ のうち一つを選び、さらに $ 1 $ 以上 $ N $ 以下の整数 $ i $ を指定する。そして、選んだ文字列の先頭から $ i $ 文字目を別の何らかの英小文字に変更する。

目標を達成するためには最小で何回の操作が必要でしょうか？

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 100 $
- $ A,\ B,\ C $ はそれぞれ長さ $ N $ の文字列である。
- $ A,\ B,\ C $ の各文字は英小文字である。

### Sample Explanation 1

この例では、はじめ $ A\ = $ `west`、$ B\ = $ `east`、$ C\ = $ `wait` です。以下のように $ 3 $ 回の操作を行うことで、最小の操作回数で目標を達成できます。 - $ A $ の $ 2 $ 文字目を `a` に変更する。$ A $ は `wast` となる。 - $ B $ の $ 1 $ 文字目を `w` に変更する。$ B $ は `wast` となる。 - $ C $ の $ 3 $ 文字目を `s` に変更する。$ C $ は `wast` となる。

### Sample Explanation 2

はじめから $ A,\ B,\ C $ がすべて等しい場合、必要な操作回数は $ 0 $ となります。

## 问题简述

__这道题我们可以换另一种思路去看待它，就容易理解了：__

给你 $3$ 个长度为 $n$ 字符串 $a,b,c$，问经过最少多少次修改（只能修改一个字符），才能使这 $3$ 个字符串相同。

## 解法综述

直接用 $i$ 同时遍历这 $3$ 个字符串进行操作，会出现这三种情况：

- 若 $a_i=b_i=c_i$，则这些字符都相同，无需进行修改。

- 若 $a_i=b_i$ 或者 $b_i=c_i$ 或者 $a_i=c_i$，则这些字符中有两个是相同的，只需对其中一个不相同的进行修改。
 
- 若 $a_i≠b_i≠c_i$，则这些字符互不相同，只需选取其中一个字符，使其他字符与被选中的字符相同即可。

由于我们只需要计算被修改字符的次数，不需要真正地去对字符进行修改，所所以遍历完之后直接输出答案即可。

## 代码描述
```cpp
#include<cstdio>
using namespace std;
char a[105],b[105],c[105];
int n,ans;
int main()
{
	scanf("%d %s %s %s",&n,a,b,c);
	for(int i=0;i<n;i++)//直接同时遍历字符串
	if(!(a[i]==b[i]&&b[i]==c[i]))//若这些字符不是都相同
	{
		if(a[i]==b[i]||a[i]==c[i]||b[i]==c[i]) ans++;//若这些字符中有两个是相同的
		else ans+=2;//否则，这些字符互不相同
	}
	printf("%d",ans);//直接输出答案即可
	return 0;
}
```

## 方法
我们一位一位地考虑，设第 $i$ 位分别为 $a[i],b[i],c[i]$

分为三种情况讨论：
1. 若三个字符互不相同，那么最少改变 $2$ 次才能让三个均相同
2. 若三个字符中恰有两个相同，那么只需要改变 $1$ 次即可（注：由于判断较复杂，先判断第三种情况，这种直接用 else
3. 若三个字符均相同，那么这一位就不用考虑了，跳过本次循环

所以就简单了，输入 $+$ 循环判断 $+$ 输出 即可
## 代码
[$\mathit{AC}\text{ }\mathit{CODE}$](https://www.luogu.com.cn/record/76870914) ：
```cpp
#include <bits/stdc++.h>
using namespace std;
int n, ans;
string a, b, c;
int main()
{
	cin >> n >> a >> b >> c;
	for (int i = 0; i < n; i++)
	{
		if (a[i] != b[i] && b[i] != c[i] && c[i] != a[i]) ans += 2;
		else if (a[i] == b[i] && b[i] == c[i]) continue;
		else ans ++;
	}
	cout << ans;
	return 0;
}
```

# 题意

有 3 个长度为 $n$ 的字符串 $a,b,c$ ,每次可以修改这 3 个数组中任意一个中的一位，要使最后 $a=b=c$ ，最少要做多少次修改。

# 思路

对于每一位 $i$，我们分情况考虑：

- 如果 $a_i=b_i=c_i$，就说明这一位已经一致了，我们就不需要修改。

- 如果这一位有两个字符一样，我们只需要修改那个不同的字符就行，让答案加 1 。

- 如果 $a_i≠b_i≠c_i$ ，就说明这三个字符互不相同，我们需要把其中两个字符修改成另一个字符，让答案加 2 。

我们把每一位都扫一遍，答案就出来了。

**code:**

```cpp
#include<bits/stdc++.h>
using namespace std;
const int N=110;
int n;
char a[N],b[N],c[N];
int ans;

inline int chck(int i){								//统计每一位有几个字符不同
	if(a[i]==b[i] && b[i]==c[i]) return 0;			
	else if(a[i]==b[i] || a[i]==c[i] || b[i]==c[i]) return 1;
	else return 2;
}

int main(){
	scanf("%d",&n);
	cin>>(a+1)>>(b+1)>>(c+1);
	for(int i=1;i<=n;i++){
		int t=chck(i);
		if(t==1) ans+=1;
		else if(t==2) ans+=2;
	}
	printf("%d\n",ans);
	return 0;
}
```

# 分析
给出字符个数和三个字符串，使三个长度相同的字符串改变字符变成相同的字符串，我们可以一个一个字符看有三个字符串有什么不同，用计数器再加上要改变的字符数量，一共有三种情况，循环字符个数次，一个一个字符判断就行了。三种情况如下：
- 三个字符都相同，不用改变任何字符，也就是不用任何操作。
- 两个字符相同，需要改变一个字符，也就是计数器加一。
- 三个字符都不相同，要改变两个字符，也就是计数器加二。

# 代码
```
#include<bits/stdc++.h>
using namespace std;
int n;
string a,b,c;
int s; //计数器
int main(){
	cin>>n>>a>>b>>c;
	for(int i=0;i<n;i++)
	{
	if(a[i]!=b[i]&&a[i]!=c[i]&&b[i]!=c[i])//字符都不相同	
	s=s+2;
	else if(a[i]!=b[i]||a[i]!=c[i]||b[i]!=c[i])//两个字符相同
	s++;
    //因为全部字符相同不需要任何操作，所以不用判断
	}
	cout<<s;
	return 0; 
}
```

题目大意：给定 $3$ 个长度为 $n$ 的字符串 $a,b,c$，以第 $0$ 位到第 $n-1$ 位的顺序，每次修改一个当前位上的字符，问最少多少次可以使 $a=b=c$。

思路很简单，对于每一个位 $i$，$a_i,b_i,c_i$ 有以下 $3$ 种情况：（首先建立计数器 `cnt = 0;`）

1. $a_i=b_i=c_i$，这种情况不用改，可以直接 `continue;`
2. $a_i=b_i$ 或 $a_i=c_i$ 或 $b_i=c_i$，这种情况只需修改不同的那个字符为其他两个相同的字符即可，`cnt ++;`
3. $a_i \ne b_i \ne c_i$，这种情况需要修改任意两个为第三个即可，`cnt += 2;`

然后就输出 $cnt$ 就可以了，因为每次操作都是独立在这个位上的，和其他位置没关系，（这也是本题简单之处），所以不需要考虑有更优解。

下面是代码：

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
string a, b, c;
int cnt = 0;

int main(){
    cin >> n >> a >> b >> c;
    for (int i = 0; i < n; i ++){ // 每次第 i 位
        if (a[i] == b[i] && b[i] == c[i] && a[i] == c[i]) ; // 其实打不打 continue 都没事，直接潇洒一个分号完事，都是提前进入下一次循环嘛 qwq
        else if (a[i] != b[i] && b[i] != c[i] && a[i] != c[i]) cnt += 2; // 都不同
        else cnt ++; // 只有一个不同
    }
    cout << cnt << endl;
    return 0;
}
```

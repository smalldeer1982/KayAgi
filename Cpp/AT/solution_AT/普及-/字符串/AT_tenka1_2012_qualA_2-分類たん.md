# 分類たん

## 题目描述

[problemUrl]: https://atcoder.jp/contests/tenka1-2012-qualA/tasks/tenka1_2012_qualA_2

 分類たんは、スペースをカンマ区切りにするのが大好き。  
 でも、スペースが続いたときは、カンマ $ 1 $ つで書きたい。

 文字列が与えられるので、与えられた文字列のスペースをカンマ区切りにした文字列を出力してください。

 入力は以下の形式で標準入力から与えられる。 > $ c_1c_2…c_N $

- 入力として文字列が $ 1 $ 行で与えられる。
- 入力の文字列長 $ N $ は、 $ 1\ \leq\ N\ \leq\ 2000 $ を満たす。
- $ i $ 文字目の文字 $ c_i $ は大文字のアルファベット (`A`, $ … $, `Z`) 、カンマ (`,`) 、スペース(` `) のいずれかである。
- 文字列の最初の文字 $ c_1 $ と最後の文字 $ c_N $ はスペースではない。
 
 入力として与えられた文字列のスペースをカンマ区切りにした文字列を標準出力に $ 1 $ 行で出力せよ。  
 ただし、複数の連続したスペースは $ 1 $ つのカンマにする。  
 なお、行の終端には改行が必要である。 ```


## 样例 #1

### 输入

```
X Y Z```

### 输出

```
X,Y,Z```

## 样例 #2

### 输入

```
QWERTY```

### 输出

```
QWERTY```

## 样例 #3

### 输入

```
A  B, C```

### 输出

```
A,B,,C```

# 题解

## 作者：_Qer (赞：5)

题意：给出几个用空格隔开的字符串，改为用逗号隔开并输出（**行末要换行**）

定义一个string类型变量，一直输入，直到键盘不再读入为止。可以用一个循环完成读入部分，每次输入一个字符串后将其输出并添加逗号。

```cpp
#include<bits/stdc++.h>
using namespace std;
string c;//string类型变量
int main(){
    cin>>c;//先输入第一个，因为我 输出时先输出逗号，而第一个字符串前不要逗号
    cout<<c;
    while(cin>>c){//利用while特性，输入不确定个数的字符串
        cout<<","<<c;//先输出逗号再输出字符串
    }
    //如果要先输出字符串再输出逗号就会比较麻烦，要在输出逗号前先输入下一个字符串，判断是否是最后一个
    cout<<endl;//なお、行の終端には改行が必要である。
    return 0;
}
```

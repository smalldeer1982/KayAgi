# map 和 优先队列 解法
题目链接 [AT974](https://www.luogu.com.cn/problem/AT974)。
## 题目大意
将 $1-1000$ 按照字典序升序排列，每行输出一个数。
## 思路
按照字典序排序，可以考虑使用 $map$。因为 $map$ 内的元素是自动按照键从小到大排序的。  
而使用**优先队列**也是可以解决的。
## map解法
将 $1-1000$ 转成 $string$ 类型，并将它作为键赋值。
```cpp
for (int i = 1; i <= 1000; ++i) {
	string s; int t = i;
	while (t != 0) {
		s.insert(s.begin(), t % 10 + '0');
		t /= 10;
	}
	m[s] = true;
}
```
然后遍历 $map$，输出每个元素的键即可。
```cpp
for (map<string, bool >::iterator it = m.begin(); it != m.end(); ++it) {
	cout << it->first << "\n";
}
```
## map解法 AC CODE
```cpp
#include <iostream>
#include <cstdio>
#include <map>
#include <string>
#define IT map<string, bool >::iterator
using namespace std;

map<string, bool > m;

int main() {
	for (int i = 1; i <= 1000; ++i) {
		string s; int t = i;
		while (t != 0) {
			s.insert(s.begin(), t % 10 + '0');
			t /= 10;
		}
		m[s] = true;
	}
	for (IT it = m.begin(); it != m.end(); ++it) {
		cout << it->first << "\n";
	}
	return 0;
}
```

## 优先队列解法
因为按照字典序升序排列，所以先定义一个小根堆。
```cpp
priority_queue<string, vector<string >, greater<string > > q;
```
将 $1-1000$ 转成 $string$ 类型，并压入小根堆中。
```cpp
for (int i = 1; i <= 1000; ++i) {
	string s; int t = i;
	while (t != 0) {
		s.insert(s.begin(), t % 10 + '0');
		t /= 10;
	}
	q.push(s);
}
```
只要小根堆非空，输出堆顶元素并弹出即可。
```cpp
while (!q.empty()) {
	cout << q.top() << "\n";
	q.pop();
}
```
## 优先队列解法 AC CODE
```cpp
#include <iostream>
#include <cstdio>
#include <queue>
#include <string>
#include <vector>
using namespace std;

priority_queue<string, vector<string >, greater<string > > q;

int main() {
	for (int i = 1; i <= 1000; ++i) {
		string s; int t = i;
		while (t != 0) {
			s.insert(s.begin(), t % 10 + '0');
			t /= 10;
		}
		q.push(s);
	}
	while (!q.empty()) {
		cout << q.top() << "\n";
		q.pop();
	}
	return 0;
}
```

 pascal题解
 #### 题目：请将1至1000这1000个数字按字典序升序排列。例如：将1,2,11,12,21按照字典序升序排序后的结果是：1,11,12,2,21。
### 大概思路：
```
1.把1~1000的数化作字符串存在数组里
2.从小到大的顺序将字符串排序1~1000的数（就按普通比较大小即可）。
3.输出
```
#### 那接下来要做的就是把要做的事用代码表示出来就行啦。

以下是代码（pascal）:
```pascal
var
 a:array[0..1001] of string;
 i,n,j:longint;
 t:string;
begin
 for i:=1 to 1000 do str(i,a[i]);//将1~1000的数转化为字符串存到一个数组里
 for i:=1 to 999 do 
  for j:=i+1 to 1000 do 
   if a[i]>a[j] then //比较大小
    begin//交换
     t:=a[i];
     a[i]:=a[j];
     a[j]:=t;
    end;
for i:=1 to 1000 do writeln(a[i]);//输出
end.
```

应用在这道题上就很简单了（建议提交时把语言改为C++11避免CE风险）

```cpp
#include<bits/stdc++.h>
using namespace std;
struct str
{
	string s;
};
bool cmp(str a,str g)
{
	return a.s<g.s;
}
int main()
{
	str st[1001];
	for(int i=1;i<=1000;i++)
		st[i].s=to_string(i);//转化成为string形式
	sort(st+1,st+1001,cmp);
	for(int i=1;i<=1000;i++)
		cout<<st[i].s<<endl;
}
```

发现题解中好像没有用char数组的，于是决定做个题解

```cpp
#include<cstdio>
#include<cstring>
char s[1002][10];
void swap(char *a,char *b){//交换函数
	char s[10];
	strcpy(s,a);
	strcpy(a,b);
	strcpy(b,s);
}
int main(void){
	for(int i=1;i<=1000;i++)
		sprintf(s[i],"%d",i);//放入字符串
	for(int i=1;i<=1000;i++)
		for(int j=i;j<=1000;j++)
			if(strcmp(s[i],s[j])>0)
				swap(s[i],s[j]);//简单的排序
	for(int i=1;i<=1000;i++)
		printf("%s\n",s[i]);//输出
	return 0;
}
```


这里面主要讲三点：

1.char s[1002][10]是指共有1002个长度为10的字符串，不要写反了

2.sprintf函数：sprintf(a,Format,..)：与printf类似不过不是输出，而是放到a所在的字符串中

3.strcpy函数：strcpy(a,b)：将b字符串复制到a里面

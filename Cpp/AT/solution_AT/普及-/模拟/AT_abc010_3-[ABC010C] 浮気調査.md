# [ABC010C] 浮気調査

## 题目描述

已知高桥君从平面直角坐标系的 $(x_a, y_a)$ 处出发，在 $T$ 分钟后到达 $(x_b,y_b)$。已知高桥君每分钟**最大**速度为 $V$。又给出 $n$ 个点，求高桥君有没有可能经过这些点中的**任意一个**。有可能则输出 `YES`， 没有则输出 `NO`。

## 样例 #1

### 输入

```
1 1 8 2 2 4
1
4 5```

### 输出

```
NO```

## 样例 #2

### 输入

```
1 1 8 2 2 6
1
4 5```

### 输出

```
YES```

## 样例 #3

### 输入

```
1 1 8 2 2 5
1
4 5```

### 输出

```
YES```

## 样例 #4

### 输入

```
7 7 1 1 3 4
3
8 1
1 7
9 9```

### 输出

```
YES```

# 積み鉛筆

## 题目描述

有一堆铅笔，被叠成两层，每支铅笔上都有编号，第$ i $层第$ j $支铅笔的编号为$ K_{j} $,$ K_{j} $是第$ i-1 $层中$ K_{j} $与$ K_{j+1} $中的最大值，现在我们得知了第二层的铅笔数量和第一层的铅笔编号，请构造出第二层铅笔的编号。

## 样例 #1

### 输入

```
4
3 5 5```

### 输出

```
1 3 5 4```

## 样例 #2

### 输入

```
6
4 8 8 2 5```

### 输出

```
4 4 8 2 2 5```

## 样例 #3

### 输入

```
5
1 2 3 4```

### 输出

```
1 1 2 3 4```

# [ABC026C] 高橋君の給料

## 题目描述

现在一个公司有 $n$ 名员工。保证 $n\le20$。

除了 $1$ 号员工以外，每个员工都有一个直属上司。保证上司的编号严格小于这个员工的编号。

没有下属的员工工资为 $1$。对于有下属的员工，他的工资是其直接下属中的最大工资和最小工资之和加上 $1$。

求出 $1$ 号员工的工资。

## 样例 #1

### 输入

```
5
1
1
1
1```

### 输出

```
3```

## 样例 #2

### 输入

```
7
1
1
2
2
3
3```

### 输出

```
7```

## 样例 #3

### 输入

```
6
1
2
3
3
2```

### 输出

```
11```

## 样例 #4

### 输入

```
10
1
2
3
4
5
6
7
8
9```

### 输出

```
1023```

# 题目信息

# [ABC257E] Addition and Multiplication 2

## 题目描述

高桥君有一个整数 $x$ 。一开始的时候， $x=0$ 。

高桥君可以无限执行以下操作：

- 选择一个整数 $i$ （ $1 \leq i \leq 9$ ）。支付 $C_i$ 日元，把 $x$ 变为 $10x+i$ 。

高桥君有 $N$ 日元，问 $x$ 最大是多少？

### 约束 ###

$1 \leq N \leq 10^6$

$1 \leq C_i \leq N$

保证 $N,C_i$ 都是整数。

## 样例 #1

### 输入

```
5
5 4 3 3 2 5 3 5 3```

### 输出

```
95```

## 样例 #2

### 输入

```
20
1 1 1 1 1 1 1 1 1```

### 输出

```
99999999999999999999```

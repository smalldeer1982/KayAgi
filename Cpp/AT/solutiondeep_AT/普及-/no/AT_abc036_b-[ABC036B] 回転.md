## 题目描述

给定一个n×n的字符矩阵，要求将其顺时针旋转90度后输出。

输入格式：一个数n，然后输入n×n的字符序列。

输出格式：这个序列顺时针旋转90度后的结果。

样例1：

输入：

3

o x o

x o x

o x o

输出：

o x o

x o x

o x o

样例2：

输入：

3

o o o

x x x

o o o

输出:

o x o

o x o

o x o

【算法分类】
模拟

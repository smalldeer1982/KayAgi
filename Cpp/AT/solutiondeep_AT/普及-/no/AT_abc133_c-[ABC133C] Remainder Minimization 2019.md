# 题目信息

# [ABC133C] Remainder Minimization 2019

## 题目描述

给出非负整数 $L$ 和 $R$，在这个区间里选择两个整数 $i$ 和 $j$ 满足 $L\le i < j\le R$。求 $(i\times j)\mod 2019$ 的最小值。

## 说明/提示

$ 0 \le L < R \le 2 \times 10^9$

## 样例 #1

### 输入

```
2020 2040```

### 输出

```
2```

## 样例 #2

### 输入

```
4 5```

### 输出

```
20```

# 题目信息

# [ABC201C] Secret Number

## 题目描述

高桥忘记了密码。密码是由 $ 0 $ 到 $ 9 $ 之间的数字组成的四位字符串，有时从 $ 0 $ 开始。

对于从 $ 0 $ 到 $ 9 $ 的各数字，高桥有如下要求。密码的要求由长度为 $ 10 $ 的字符串 $S_0 S_1 \ldots S_9 $表示。

要求的表示方法：
- $ S_i $ 为 `o` 时：数字 $ i $ 确实包含在密码中。
- $ S_i $ 为 `x` 时：数字 $ i $ 没有包含在密码中。
- $ S_i $ 为 `?` 时：不知道数字 $ i $ 是否包含在密码中

高桥的密码可能有多少种结果？

## 样例 #1

### 输入

```
ooo???xxxx```

### 输出

```
108```

## 样例 #2

### 输入

```
o?oo?oxoxo```

### 输出

```
0```

## 样例 #3

### 输入

```
xxxxx?xxxo```

### 输出

```
15```

【题解分析与结论】
由于题目要求计算所有可能的四位密码组合，且每个数字的状态（`o`、`x`、`?`）对密码的生成有约束，因此可以通过枚举所有可能的四位密码，并根据约束条件进行筛选。关键在于如何处理 `o` 和 `x` 的约束，以及 `?` 的灵活性。

【通用建议与扩展思路】
1. **枚举所有可能的四位密码**：由于密码是四位数，且每位数字可以是 0 到 9 之间的任意数字，因此总共有 $10^4 = 10000$ 种可能的组合。可以通过遍历所有组合，并检查每个组合是否满足给定的约束条件。
2. **处理约束条件**：对于每个组合，检查是否包含所有 `o` 标记的数字，并且不包含任何 `x` 标记的数字。对于 `?` 标记的数字，可以忽略其存在与否。
3. **优化**：可以通过预处理 `o` 和 `x` 的约束，减少不必要的检查。例如，如果某个数字被标记为 `x`，则可以直接排除所有包含该数字的组合。

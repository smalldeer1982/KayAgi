### 题解分析与结论

由于题目要求两个火源相遇的位置，且导火线的燃烧速度是恒定的，因此可以通过计算两个火源燃烧的总时间来确定相遇的位置。具体来说，可以分别计算从左端和右端燃烧到相遇点所需的时间，这两个时间应该相等。

### 关键思路

1. **总燃烧时间计算**：首先计算所有导火线的总燃烧时间，即从左端燃烧到右端所需的时间。这个时间可以通过累加每根导火线的燃烧时间（$A_i / B_i$）来得到。
2. **相遇点确定**：两个火源相遇的时间是总燃烧时间的一半。然后，从左端开始计算，累加每根导火线的燃烧时间，直到累加的时间达到相遇时间，此时的位置即为相遇点。

### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N;
    cin >> N;
    vector<double> A(N), B(N);
    for (int i = 0; i < N; ++i) {
        cin >> A[i] >> B[i];
    }
    
    double total_time = 0;
    for (int i = 0; i < N; ++i) {
        total_time += A[i] / B[i];
    }
    
    double half_time = total_time / 2;
    double distance = 0;
    for (int i = 0; i < N; ++i) {
        double burn_time = A[i] / B[i];
        if (half_time >= burn_time) {
            distance += A[i];
            half_time -= burn_time;
        } else {
            distance += half_time * B[i];
            break;
        }
    }
    
    cout << fixed << setprecision(15) << distance << endl;
    return 0;
}
```

### 关键代码解释

1. **总燃烧时间计算**：通过循环累加每根导火线的燃烧时间，得到总燃烧时间 `total_time`。
2. **相遇点确定**：计算总燃烧时间的一半 `half_time`，然后从左端开始累加每根导火线的燃烧时间，直到累加的时间达到 `half_time`，此时的位置即为相遇点。

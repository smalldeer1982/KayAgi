### 题解分析与结论

题目要求判断给定的矩阵 $ B $ 是否是从矩阵 $ A $ 中切出的一个矩形区域。由于 $ A $ 的构造规则是固定的，我们可以通过计算 $ B $ 中每个元素的值是否符合 $ A $ 的构造规则来判断。

### 关键思路
1. **矩阵构造规则**：矩阵 $ A $ 的每个元素 $ A_{i,j} = (i-1) \times 7 + j $。
2. **验证方法**：对于 $ B $ 中的每个元素 $ B_{i,j} $，计算其是否符合 $ A $ 的构造规则。具体来说，可以通过计算 $ B_{i,j} $ 对应的 $ i $ 和 $ j $ 是否符合 $ A $ 的构造规则。

### 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    vector<vector<int>> B(N, vector<int>(M));
    for (int i = 0; i < N; ++i) {
        for (int j = 0; j < M; ++j) {
            cin >> B[i][j];
        }
    }
    bool valid = true;
    for (int i = 0; i < N; ++i) {
        for (int j = 0; j < M; ++j) {
            int expected = (B[0][0] - 1) / 7 * 7 + (i * 7) + (j + 1);
            if (B[i][j] != expected) {
                valid = false;
                break;
            }
        }
        if (!valid) break;
    }
    cout << (valid ? "Yes" : "No") << endl;
    return 0;
}
```

### 代码核心思想
1. **输入矩阵 $ B $**：读取矩阵 $ B $ 的大小和内容。
2. **验证规则**：通过计算每个元素 $ B_{i,j} $ 是否符合 $ A $ 的构造规则来验证。
3. **输出结果**：根据验证结果输出 `Yes` 或 `No`。

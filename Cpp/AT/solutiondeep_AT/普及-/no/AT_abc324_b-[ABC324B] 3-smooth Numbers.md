【核心代码实现】
```cpp
#include <iostream>
using namespace std;

bool is3Smooth(long long n) {
    while (n % 2 == 0) n /= 2;
    while (n % 3 == 0) n /= 3;
    return n == 1;
}

int main() {
    long long N;
    cin >> N;
    if (is3Smooth(N)) cout << "Yes" << endl;
    else cout << "No" << endl;
    return 0;
}
```

【代码解释】
1. `is3Smooth` 函数通过不断除以 2 和 3，判断最终剩下的数是否为 1。
2. 主函数读取输入的 $N$，并调用 `is3Smooth` 函数进行判断，输出相应的结果。

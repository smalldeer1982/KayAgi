# 题目信息

# [AGC001A] BBQ Easy

## 题目描述

Snuke 在参加一个户外烧烤派对，他要将 $2n$ 份食材两两配对组成 $n$ 份。  
每个食材有一个价值，为 $L _ i$。一份食物的价值是其两个食材中较小的那个。  
问最大总价值是多少。

## 说明/提示

$1 \le n \le 100, 1 \le L_i \le 100$。保证 $L_i$ 是整数。

## 样例 #1

### 输入

```
2

1 3 1 2```

### 输出

```
3```

## 样例 #2

### 输入

```
5

100 1 2 3 14 15 58 58 58 29```

### 输出

```
135```

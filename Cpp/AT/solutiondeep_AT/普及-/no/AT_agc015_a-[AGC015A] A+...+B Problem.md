# [AGC015A] A+...+B Problem

## 题目描述

[problemUrl]: https://atcoder.jp/contests/agc015/tasks/agc015_a

すぬけ君は、整数を $ N $ 個持っています。このうち最小のものは $ A $、最大のものは $ B $ です。すぬけ君が持っている整数の総和としてありうる値は何通りあるでしょうか。

## 说明/提示

### 制約

- $ 1\ ≦\ N,A,B\ ≦\ 10^9 $
- $ A,B $ は整数である

### Sample Explanation 1

$ 18=4+4+4+6,19=4+4+5+6,20=4+5+5+6,21=4+5+6+6,22=4+6+6+6 $ の $ 5 $ つの値が総和として考えられます。

## 样例 #1

### 输入

```
4 4 6```

### 输出

```
5```

## 样例 #2

### 输入

```
5 4 3```

### 输出

```
0```

## 样例 #3

### 输入

```
1 7 10```

### 输出

```
0```

## 样例 #4

### 输入

```
1 3 3```

### 输出

```
1```

### 题目内容重写
【题目描述】
すぬけ君有 $N$ 个整数，其中最小的整数是 $A$，最大的整数是 $B$。すぬけ君所持有的整数的总和可能有多少种不同的值？

【说明/提示】
- 约束条件：$1 \leq N, A, B \leq 10^9$
- $A$ 和 $B$ 是整数

【样例解释】
样例1：$18=4+4+4+6, 19=4+4+5+6, 20=4+5+5+6, 21=4+5+6+6, 22=4+6+6+6$，总共有5种不同的总和。

### 算法分类
数学

### 题解分析与结论
各题解的核心思路基本一致，即通过计算最小总和和最大总和的差值加1来得到可能的总和数量。难点在于处理特殊情况，如 $N=1$ 或 $A > B$ 时的情况。所有题解都提到了这些特殊情况，并给出了相应的处理方式。

### 所选高星题解
1. **作者：dblark (5星)**
   - **关键亮点**：详细推导了公式 $(b-a) \times (n-2) + 1$，并明确指出了两个坑点（$A > B$ 和 $N=1$ 的情况）。
   - **个人心得**：通过LaTeX公式推导，使得思路更加清晰，代码简洁明了。
   - **核心代码**：
     ```pas
     if a>b then
       writeln(0)
     else
       if n=1 then
         writeln(ord(a=b))
       else
         writeln((b-a)*(n-2)+1);
     ```

2. **作者：MilkyCoffee (4星)**

   - **核心代码**：

     if (n > 1 && b >= a || n == 1 && a == b) cout << (n - 2) * (b - a) + 1 << endl;
     else cout << 0 << endl;
     ```

3. **作者：Cambridge (4星)**

   - **核心代码**：
     ```cpp
     if(n==1&&a==b)cout<<1;
     else if(n>1&&b>=a)cout<<(n-2)*(b-a)+1;
     else cout<<0;
     ```

### 最优关键思路
通过计算最小总和和最大总和的差值加1来得到可能的总和数量，公式为 $(b-a) \times (n-2) + 1$。特别注意处理 $N=1$ 和 $A > B$ 的情况。

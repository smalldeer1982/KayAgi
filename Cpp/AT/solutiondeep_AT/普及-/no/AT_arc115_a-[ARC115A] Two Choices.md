### 关键代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    vector<string> S(N);
    for (int i = 0; i < N; ++i) {
        cin >> S[i];
    }
    
    map<int, int> cnt;
    for (int i = 0; i < N; ++i) {
        int parity = 0;
        for (char c : S[i]) {
            parity ^= (c - '0');
        }
        cnt[parity]++;
    }
    
    long long ans = 0;
    for (auto& p : cnt) {
        ans += 1LL * p.second * (N - p.second);
    }
    ans /= 2;
    
    cout << ans << endl;
    return 0;
}
```

### 代码简述

1. **输入处理**：读取 $ N $ 和 $ M $，以及每个学生的答案。
2. **奇偶性计算**：计算每个学生的答案的 $ 1 $ 的个数的奇偶性，并统计每种奇偶性的学生数量。
3. **组数计算**：根据奇偶性统计结果，计算满足条件的组数。
4. **输出结果**：输出最终的计算结果。

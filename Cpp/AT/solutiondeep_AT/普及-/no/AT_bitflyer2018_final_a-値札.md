# 题目信息

# 値札

## 题目描述

[problemUrl]: https://atcoder.jp/contests/bitflyer2018-final/tasks/bitflyer2018_final_a

すぬけ君は、店を開こうとしています。$ N $ 個の商品を販売する予定です。 $ i $ 番目の商品は、$ p_i $ 円で販売されます。

すぬけ君は、それぞれの価格の末尾についた `0` をたくさん書くのが大変に感じたので、 $ N $ 個全ての商品の値札の末尾に連続する `0` を全ての商品について同じ個数だけ取り除き、 会計の時にその分の `0` を補完することにしました。

商品の値札 $ 1 $ 枚あたり、 `0` を最大何個取り除けるかを求めてください。

## 说明/提示

### 制約

- $ 1\ <\ =\ N\ <\ =\ 100 $
- $ 1\ <\ =\ p_i\ <\ =\ 10^9 $
- $ p_i $ は整数

### Sample Explanation 1

それぞれ、末尾についた `0` を $ 1 $ つだけ取り除くことができ、すると値札には $ 30 $, $ 25 $, $ 600 $, $ 73 $ と書かれることになります。

## 样例 #1

1```

すぬけ君计划开设一家商店，准备销售 $N$ 个商品。第 $i$ 个商品的售价为 $p_i$ 日元。

すぬけ君觉得在每个价格末尾写很多 `0` 非常麻烦，因此他决定在所有商品的价签末尾去掉相同数量的连续 `0`，并在结账时补充这些 `0`。

请计算每个商品的价签最多可以去掉多少个末尾的 `0`。

### 题解综合分析

本题的核心逻辑是计算每个数字末尾的 `0` 的个数，并取所有数字中的最小值。题解中主要采用了两种方法：一种是直接对数字进行除以 `10` 的操作，另一种是将数字转换为字符串后从末尾遍历 `0` 的个数。

### 评分较高的题解

1. **作者：Nuyoah_awa (赞：3)**  
   - **星级：4**  
   - **关键亮点**：代码简洁，直接对数字进行除以 `10` 的操作，时间复杂度为 $O(n)$，思路清晰。  
   - **核心代码**：
     ```cpp
     int sum = 0;
     while(a % 10 == 0)
         sum++, a /= 10;
     ans = min(ans, sum);
     ```

2. **作者：CleverRaccoon (赞：0)**  
   - **星级：4**  
   - **关键亮点**：使用字符串处理，代码直观，适合对字符串操作熟悉的读者。  
   - **核心代码**：
     ```cpp
     int cnt = 0;
     for(int i = num.size() - 1; i >= 0; --i, ++cnt)
         if(num[i] != '0')
             break;
     ans = min(ans, cnt);
     ```

3. **作者：huyixuanpi (赞：0)**  
   - **星级：4**  
   - **关键亮点**：代码简洁，直接对数字进行除以 `10` 的操作，时间复杂度为 $O(n)$，思路清晰。  
   - **核心代码**：
     ```cpp
     while(a % 10 == 0)
         num++, a /= 10;
     ans = min(ans, num);
     ```

### 最优关键思路或技巧

- **数字处理**：直接对数字进行除以 `10` 的操作，直到无法整除为止，记录 `0` 的个数。
- **字符串处理**：将数字转换为字符串后，从末尾遍历 `0` 的个数，直到遇到非 `0` 字符。

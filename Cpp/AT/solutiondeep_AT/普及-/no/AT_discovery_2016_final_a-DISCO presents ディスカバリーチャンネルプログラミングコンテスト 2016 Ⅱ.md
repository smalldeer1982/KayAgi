【题解分析与结论】
该题的核心逻辑是根据参赛者的名次分配奖金，并按照参赛者编号输出奖金。题目本身较为简单，主要考察对数组的使用和基本的输入输出操作。

【题解评分与亮点】
1. **Palind_Rome的题解**（4星）
   - **关键亮点**：使用数组存储奖金，并通过循环输出结果，代码简洁明了。
   - **代码核心思想**：使用一个数组 `d` 存储前五名的奖金，另一个数组 `a` 存储每个参赛者的奖金，最后根据参赛者编号输出奖金。
   - **代码片段**：
     ```cpp
     const int d[101]={0,100000,50000,30000,20000,10000};
     int a[101];
     int main(){
         int n,i,b,_max=0;
         cin>>n;
         for(i=1;i<=n;i++){
             cin>>b;
             a[b]=d[i];
             _max=max(_max,b);
         }
         for(i=1;i<=_max;i++)
             cout<<a[i]<<endl;
         return 0;
     }
     ```

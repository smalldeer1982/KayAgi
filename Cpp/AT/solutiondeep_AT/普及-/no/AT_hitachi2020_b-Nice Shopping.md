# Nice Shopping

## 题目描述

[problemUrl]: https://atcoder.jp/contests/hitachi2020/tasks/hitachi2020_b

あなたは、冷蔵庫と電子レンジを買うために、とある家電量販店に来ました。

この家電量販店では、 $ A $ 種類の冷蔵庫と $ B $ 種類の電子レンジが販売されています。 $ i $ 番目( $ 1\ \le\ i\ \le\ A $ )の冷蔵庫の値段は $ a_i $ 円であり、 $ j $ 番目( $ 1\ \le\ j\ \le\ B $ )の電子レンジの値段は $ b_j $ 円です。

また、あなたは $ M $ 種類の割引券を所持しており、 $ i $ 番目 ( $ 1\ \le\ i\ \le\ M $ )の割引券では、 $ x_i $ 番目の冷蔵庫 と $ y_i $ 番目の電子レンジを同時に買うと、 支払総額が $ c_i $ 円安くなります。ただし、複数の割引券を同時に使うことはできません。

さて、あなたは冷蔵庫と電子レンジをちょうど $ 1 $ 台ずつ買おうと思っています。かかる金額の最小値を求めてください。

## 说明/提示

### 制約

- 入力は全て整数
- $ 1\ \le\ A\ \le\ 10^5 $
- $ 1\ \le\ B\ \le\ 10^5 $
- $ 1\ \le\ M\ \le\ 10^5 $
- $ 1\ \le\ a_i\ ,\ b_i\ ,\ c_i\ \le\ 10^5 $
- $ 1\ \le\ x_i\ \le\ A $
- $ 1\ \le\ y_i\ \le\ B $
- $ c_i\ \le\ a_{x_i}\ +\ b_{y_i} $

### Sample Explanation 1

$ 1 $ 番目の冷蔵庫と $ 2 $ 番目の電子レンジを買うと、割引券の効果により $ 3+3-1=5 $ 円になります。

### Sample Explanation 2

複数の割引券を同時に使うことはできないことに注意してください。

### Sample Explanation 3

この場合は $ 1 $ 番目の冷蔵庫と $ 1 $ 番目の電子レンジを買うと $ 6 $ 円になり、これが最小です。 割引券は使わなくてもよいことに注意してください。

### 题目内容重写

**题目描述**

你为了购买冰箱和微波炉，来到了一家家电卖场。

这家家电卖场出售 $A$ 种冰箱和 $B$ 种微波炉。第 $i$ 台冰箱（$1 \le i \le A$）的价格为 $a_i$ 日元，第 $j$ 台微波炉（$1 \le j \le B$）的价格为 $b_j$ 日元。

此外，你拥有 $M$ 种优惠券，第 $i$ 张优惠券（$1 \le i \le M$）可以在同时购买第 $x_i$ 台冰箱和第 $y_i$ 台微波炉时，使总支付金额减少 $c_i$ 日元。注意，不能同时使用多张优惠券。

现在，你打算购买一台冰箱和一台微波炉。请计算所需支付的最小金额。

**说明/提示**

- 输入均为整数。
- $1 \le A \le 10^5$
- $1 \le B \le 10^5$
- $1 \le M \le 10^5$
- $1 \le a_i, b_i, c_i \le 10^5$
- $1 \le x_i \le A$
- $1 \le y_i \le B$
- $c_i \le a_{x_i} + b_{y_i}$

**样例解释**

- 样例1：购买第1台冰箱和第2台微波炉，使用优惠券后，总价为 $3+3-1=5$ 日元。
- 样例2：不能同时使用多张优惠券，因此最小总价为 $10$ 日元。
- 样例3：购买第1台冰箱和第1台微波炉，总价为 $6$ 日元，不使用优惠券。

### 题解分析与结论

本题的核心逻辑是模拟所有可能的购买组合，并计算最小总价。题解的主要思路是先计算不使用优惠券时的最小总价（即最便宜的冰箱和最便宜的微波炉的总价），然后遍历所有优惠券，计算使用每张优惠券后的总价，最终取最小值。

int a[N], b[N];
int A, B, M;

int main() {
	cin >> A >> B >> M;
	int i;
	rep(i, A) cin >> a[i];
	rep(i, B) cin >> b[i];
	
    int minA = a[1];
    rep(i, A) minA = min(minA, a[i]);
    
    int minB = b[1];
    rep(i, B) minB = min(minB, b[i]);
	
	int res = minA + minB;
	rep(i, M) {
		int x, y, c;
		cin >> x >> y >> c;
		int hoge = a[x] + b[y] - c;
		if (hoge < 0) hoge = 0;
		res = min(res, hoge);
	}
	
	cout << res << endl;
	
	return 0;
}
```

int A, B, M;
int main() {
  cin >> A >> B >> M;
  int a[A], b[B];
  int minA = 1e9;
  int minB = 1e9;
  for (int i = 0; i < A; i++) {
    cin >> a[i];
    minA = min(minA, a[i]);
  }
  for (int i = 0; i < B; i++) {
    cin >> b[i];
    minB = min(minB, b[i]);
  }
  int ans = minA + minB;

  int x, y, c;
  for (int i = 0; i < M; i++) {
    cin >> x >> y >> c;
    x--;
    y--;
    ans = min(ans, a[x] + b[y] - c);
  }
  cout << ans << endl;
  return 0;
}
```

### 最优关键思路
1. **最小总价计算**：首先计算最便宜的冰箱和最便宜的微波炉的总价，作为初始答案。
2. **优惠券遍历**：遍历所有优惠券，计算使用每张优惠券后的总价，并更新最小总价。
3. **边界检查**：确保使用优惠券后的总价不会为负数。

# 题目信息

# Touitsu

## 题目描述

[problemUrl]: https://atcoder.jp/contests/nikkei2019-qual/tasks/nikkei2019_qual_b

三つの文字列 $ A,\ B,\ C $ が与えられます。これらはそれぞれ、英小文字からなる長さ $ N $ の文字列です。

私たちの目標は、これら三つの文字列をすべて等しくすることです。そのために、あなたは次の操作を繰り返し行うことができます。

- 操作: 文字列 $ A,\ B,\ C $ のうち一つを選び、さらに $ 1 $ 以上 $ N $ 以下の整数 $ i $ を指定する。そして、選んだ文字列の先頭から $ i $ 文字目を別の何らかの英小文字に変更する。

目標を達成するためには最小で何回の操作が必要でしょうか？

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 100 $
- $ A,\ B,\ C $ はそれぞれ長さ $ N $ の文字列である。
- $ A,\ B,\ C $ の各文字は英小文字である。

### Sample Explanation 1

この例では、はじめ $ A\ = $ `west`、$ B\ = $ `east`、$ C\ = $ `wait` です。以下のように $ 3 $ 回の操作を行うことで、最小の操作回数で目標を達成できます。 - $ A $ の $ 2 $ 文字目を `a` に変更する。$ A $ は `wast` となる。 - $ B $ の $ 1 $ 文字目を `w` に変更する。$ B $ は `wast` となる。 - $ C $ の $ 3 $ 文字目を `s` に変更する。$ C $ は `wast` となる。

### Sample Explanation 2

はじめから $ A,\ B,\ C $ がすべて等しい場合、必要な操作回数は $ 0 $ となります。

### 题目翻译
#### 题目描述
给定三个字符串 $A, B, C$，它们都是由小写字母组成的长度为 $N$ 的字符串。我们的目标是通过最少的操作使这三个字符串完全相同。每次操作可以选择一个字符串，并指定一个位置 $i$（$1 \leq i \leq N$），将该位置的字符修改为任意一个小写字母。求最少需要多少次操作才能达成目标。

#### 说明/提示
- $1 \leq N \leq 100$
- $A, B, C$ 是长度为 $N$ 的字符串。
- $A, B, C$ 的每个字符都是小写字母。

#### 样例解释
**样例 1**  
输入：  
```
4
west
east
wait
```
输出：  
```
3
```
解释：通过修改 $A$ 的第 2 个字符为 `a`，$B$ 的第 1 个字符为 `w`，$C$ 的第 3 个字符为 `s`，三个字符串都变为 `wast`，共需要 3 次操作。

**样例 2**  
输入：  
```
9
different
different
different
```
输出：  
```
0
```
解释：三个字符串已经相同，不需要任何操作。

### 题解分析与结论
所有题解的核心思路都是逐字符比较三个字符串，根据字符的相同情况决定所需的操作次数。具体来说，对于每个位置 $i$，有以下三种情况：
1. 三个字符都相同：无需操作。
2. 有两个字符相同：只需修改一个字符。
3. 三个字符都不同：需要修改两个字符。

所有题解都采用了这种思路，代码实现也较为相似，主要区别在于代码风格和细节处理。整体来看，这些题解的质量较高，思路清晰，代码简洁，适合初学者理解。

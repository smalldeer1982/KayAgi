# 题目信息

# [ARC001B] リモコン

## 题目描述

高桥君要调整空调的设定温度。现在的设定温度是A度，而他想调到B度。
空调遥控器按一次可以:
- 上调或下调1度
- 上调或下调5度
- 上调或下调10度
高桥君想求出从A调到B度的最小操作数。

## 说明/提示

样例1: 
输入：
```
7 34
```
输出：
```
5
```
依次上调10、10、5、1、1度即可

样例2: 
输入：
```
19 28
```
输出：
```
2
```
上调10度、下调1度即可。


样例3: 
输入：
```
10 10
```
输出：
```
0
```
温度一样时无需调整。

### 题解分析与结论
本题的题解主要分为以下几种思路：
1. **搜索算法**：包括DFS、BFS、双向BFS等，通过遍历所有可能的操作路径，找到最小操作数。
2. **贪心算法**：通过每次选择最大可能的调整幅度（10度、5度、1度）来快速缩小温差，从而减少操作次数。
3. **最短路算法**：将温度变化建模为图的最短路径问题，使用Dijkstra等算法求解。
4. **打表法**：预处理所有可能的温度变化，直接查表输出结果。

#### 题解2：Diamiko (最短路算法) - 4星
**关键亮点**：将温度变化建模为图的最短路径问题，使用Dijkstra算法求解，思路新颖。
**代码核心**：
```cpp
void Dijkstra() {
    priority_queue<pii,vector<pii>,greater<pii>> q;
    q.push({0,A});
    node[A].dis=0;
    while(q.size()) {
        pii tmp=q.top();
        q.pop();
        int d=tmp.first,u=tmp.second;
        if(d!=node[u].dis) continue;
        for(int e=node[u].head;e;e=edge[e].next) {
            int v=edge[e].to;
            if(node[v].dis>d+edge[e].len) {
                node[v].dis=d+edge[e].len;
                q.push({node[v].dis,v});
            }
        }
    }
}
```
**个人心得**：通过将温度变化建模为图的最短路径问题，可以更直观地理解问题，并且Dijkstra算法的效率较高。

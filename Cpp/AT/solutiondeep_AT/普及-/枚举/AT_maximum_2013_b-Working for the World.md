## 题目描述

- **算法要点**

### 所选题解（≥4星）
- **zhanghzqwq**：4星。关键亮点在于思路巧妙，代码简洁，利用不断除尽较小因子的方法，时间复杂度低。
```cpp
#include<iostream>
using namespace std;
int main(){
    int n;
    cin>>n;
    while(n){//判断是否不为零
        for(int i=2;i*i<=n;i++){
            if(n%i==0){
                n/=i;
                i--;//一定要减一
            }
        }
        cout<<n<<endl; //输出
        cin>>n;
    }
    return 0;//华丽的结束
} 
```
**核心实现思想**：从2开始遍历，若n能被i整除，则不断将n除以i，同时i减1以确保再次检查当前i，直到不能整除为止，最后剩下的n即为最大质因数。

### 最优关键思路或技巧
不断除尽较小因子，利用质因数的性质，将时间复杂度优化到$O(\sqrt{n})$。

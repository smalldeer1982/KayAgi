# [ABC050B] Contest with Drinks Easy

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc050/tasks/abc050_b

joisinoお姉ちゃんは、あるプログラミングコンテストの決勝を控えています。 このコンテストでは、$ N $ 問の問題が用意されており、それらには $ 1～N $ の番号がついています。 joisinoお姉ちゃんは、問題 $ i(1≦i≦N) $ を解くのにかかる時間が $ T_i $ 秒であることを知っています。

また、このコンテストでは、$ M $ 種類のドリンクが提供されており、$ 1～M $ の番号がついています。 そして、ドリンク $ i(1≦i≦M) $ を飲むと、脳が刺激され、問題 $ P_i $ を解くのにかかる時間が $ X_i $ 秒になります。 他の問題を解くのにかかる時間に変化はありません。

コンテスタントは、コンテスト開始前にいずれかのドリンクを $ 1 $ 本だけ飲むことができます。 joisinoお姉ちゃんは、それぞれのドリンクについて、それを飲んだ際に、全ての問題を解くのに何秒必要なのかを知りたくなりました。 全ての問題を解くのに必要な時間とは、それぞれの問題を解くのにかかる時間の合計です。 あなたの仕事は、joisinoお姉ちゃんの代わりにこれを求めるプログラムを作成することです。

## 说明/提示

### 制約

- 入力は全て整数である
- $ 1≦N≦100 $
- $ 1≦T_i≦10^5 $
- $ 1≦M≦100 $
- $ 1≦P_i≦N $
- $ 1≦X_i≦10^5 $

### Sample Explanation 1

一つ目のドリンクを飲んだ場合、それぞれの問題を解くのに要する時間は、$ 1 $ 秒、$ 1 $ 秒、$ 4 $ 秒になります。 なので、それらを合計した $ 6 $ 秒が答えになり、$ 6 $ を出力します。 二つ目のドリンクを飲んだ場合、それぞれの問題を解くのに要する時間は、$ 2 $ 秒、$ 3 $ 秒、$ 4 $ 秒になります。 なので、それらを合計した $ 9 $ 秒が答えになり、$ 9 $ を出力します。

# [ABC050B] 轻松的带饮料比赛

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc050/tasks/abc050_b

joisino姐姐即将迎来一场编程竞赛的决赛。这场竞赛准备了 $N$ 道题目，题目编号为 $1$ 到 $N$。joisino姐姐知道解第 $i$ 道题（$1\leq i\leq N$）需要 $T_i$ 秒。

此外，竞赛提供了 $M$ 种饮料，饮料编号为 $1$ 到 $M$。喝了第 $i$ 种饮料（$1\leq i\leq M$）后，大脑会受到刺激，解第 $P_i$ 道题的时间会变为 $X_i$ 秒，解其他题的时间不变。

参赛者可以在比赛开始前仅喝一瓶饮料。joisino姐姐想知道对于每种饮料，喝了之后解完所有题需要多少秒。解完所有题所需的时间是解每道题所需时间的总和。你的任务是编写一个程序来替joisino姐姐解决这个问题。

### 所选题解
- 智子·起源（4星）：关键亮点在于代码简洁，直接利用总时间进行计算，避免了复杂的逻辑。
```cpp
#include<iostream>
#include<cstdio>

using namespace std;
int n,a[100+5],x,y,h;
int main()
{
  scanf("%d",&n);
  for(int i=1;i<=n;i++){scanf("%d",&a[i]);h+=a[i];}
  scanf("%d",&n);
  for(int i=0;i<n;i++)
  {
    scanf("%d%d",&x,&y);
    printf("%d\n",h-a[x]+y);
  }
  return 0;
}
```
核心实现思想：先读入题目时间并累加得到总时间 `h`，再读入饮料信息，对于每种饮料，用总时间 `h` 减去原题目时间 `a[x]` 加上新时间 `y` 得到结果并输出。

- 雪风舞者（4星）：关键亮点在于思路清晰，通过记录总和和计算差值来更新结果。
```cpp
#include<cstdio>
using namespace std;
int n,m,a[105],sum=0;
int main(){
	scanf("%d",&n);
	for(int i=1;i<=n;i++){
		scanf("%d",&a[i]);
		sum+=a[i];
	}
	scanf("%d",&m);
	while(m--){
		int p,x;
		scanf("%d%d",&p,&x);
		printf("%d\n",sum-a[p]+x);
	}
	return 0;
}
```
核心实现思想：先读入题目时间并累加得到总和 `sum`，再读入饮料信息，对于每种饮料，用总和 `sum` 减去原题目时间 `a[p]` 加上新时间 `x` 得到结果并输出。

### 最优关键思路或技巧
先计算出不喝饮料时解完所有题的总时间，然后对于每一种饮料，只需将总时间减去受影响题目的原时间，再加上新时间，即可得到喝该饮料后解完所有题的总时间。

### 可拓展之处

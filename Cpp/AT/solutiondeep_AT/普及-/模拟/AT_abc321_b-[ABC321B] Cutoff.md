# [ABC321B] Cutoff

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc321/tasks/abc321_b

以下の手順で行われる試験があります。

- 試験は $ 1 $ ラウンド目から $ N $ ラウンド目までの $ N $ ラウンドからなる。
- 各ラウンドに対し、 $ 0 $ 以上 $ 100 $ 以下の整数でスコアが与えられる。
- $ N $ ラウンドのスコアのうち、最高スコアと最低スコアを除いた $ N-2 $ ラウンドのスコアの合計が最終結果となる。
  - 厳密には、各ラウンドのスコアを昇順に並べた列を $ S=(S_1,S_2,\dots,S_N) $ としたとき、最終結果は $ S_2+S_3+\dots+S_{N-1} $ となる。
 
現在、試験のうち $ N-1 $ ラウンドが終了し、 $ i $ ラウンド目のスコアは $ A_i $ でした。  
 最終結果を $ X $ 以上とするために $ N $ ラウンド目に取るべきスコアの最小値を出力してください。  
 但し、 $ N $ ラウンド目にどのようなスコアを取っても最終結果が $ X $ 以上にならない場合、代わりに `-1` と出力してください。  
 なお、 $ N $ ラウンド目に取りうるスコアは $ 0 $ 以上 $ 100 $ 以下の整数であることに注意してください。

## 说明/提示

### 制約

- 入力は全て整数
- $ 3\ \le\ N\ \le\ 100 $
- $ 0\ \le\ X\ \le\ 100\ \times\ (N-2) $
- $ 0\ \le\ A_i\ \le\ 100 $
 
### Sample Explanation 1

$ 4 $ ラウンド目までのスコアは $ 40,60,80,50 $ でした。 $ 5 $ ラウンド目にスコア $ 70 $ を取ると、スコアを昇順に並べた列は $ S=(40,50,60,70,80) $ となり、最終結果は $ 50+60+70=180 $ となります。 なお、最終結果を $ 180 $ 以上にするために取るべきスコアの最小値が $ 70 $ であることが示せます。

### Sample Explanation 2

$ 2 $ ラウンド目までのスコアは $ 100,100 $ でした。 $ 3 $ ラウンド目にスコア $ 0 $ を取ると、スコアを昇順に並べた列は $ S=(0,100,100) $ となり、最終結果は $ 100 $ となります。 最大スコアである $ 100 $ が複数ありますが、そのうち $ 1 $ つしか除かれないことに注意してください。(最小スコアについても同様です) なお、最終結果を $ 100 $ 以上にするために取るべきスコアの最小値が $ 0 $ であることが示せます。

### Sample Explanation 3

$ 4 $ ラウンド目までのスコアは $ 0,0,99,99 $ でした。 $ 5 $ ラウンド目にどのようなスコアを取っても、最終結果を $ 200 $ 以上にすることができないことが示せます。

### 题目内容中文重写
有一个按照以下步骤进行的考试：
- 考试由第 1 轮至第 N 轮共 N 轮组成。
- 每一轮都会被赋予一个 0 以上 100 以下的整数分数。
- N 轮的分数中，除去最高分和最低分后的 N - 2 轮分数总和即为最终成绩。
  - 严格来说，将每一轮的分数按升序排列得到序列 $S=(S_1,S_2,\dots,S_N)$ 时，最终成绩为 $S_2+S_3+\dots+S_{N - 1}$。

目前，考试的 N - 1 轮已经结束，第 i 轮的分数为 $A_i$。
请输出为使最终成绩达到 X 以上，在第 N 轮应取得的最低分数。
但是，如果无论第 N 轮取得何种分数，最终成绩都无法达到 X 以上，则改为输出 `-1`。
请注意，第 N 轮可能取得的分数是 0 以上 100 以下的整数。

#### 说明/提示
##### 限制条件
- 所有输入均为整数
- $3 \leq N \leq 100$
- $0 \leq X \leq 100 \times (N - 2)$
- $0 \leq A_i \leq 100$

##### 示例解释 1
第 4 轮的分数分别为 40、60、80、50。若第 5 轮取得 70 分，则分数按升序排列的序列为 $S=(40,50,60,70,80)$，最终成绩为 $50 + 60 + 70 = 180$。另外，可以证明为使最终成绩达到 180 以上应取得的最低分数是 70。

##### 示例解释 2
第 2 轮的分数分别为 100、100。若第 3 轮取得 0 分，则分数按升序排列的序列为 $S=(0,100,100)$，最终成绩为 100。请注意，即使存在多个最高分 100，也只会除去其中一个（最低分同理）。另外，可以证明为使最终成绩达到 100 以上应取得的最低分数是 0。

##### 示例解释 3
第 4 轮的分数分别为 0、0、99、99。可以证明，无论第 5 轮取得何种分数，都无法使最终成绩达到 200 以上。

### 综合分析与结论
由于暂无题解，通用建议是可以先将已有的 N - 1 轮分数进行排序，找出当前的最高分和最低分，然后通过枚举第 N 轮可能的分数（0 到 100），计算加入该分数后除去最高分和最低分的总分，找到满足最终成绩达到 X 以上的最小分数。若所有可能分数都不满足，则输出 -1。

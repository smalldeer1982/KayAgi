# 题目信息

# [ABC112D] Partition

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc112/tasks/abc112_d

整数 $ N,\ M $ が与えられます。

$ a_1\ +\ a_2\ +\ ...\ +\ a_N $ = $ M $ となる正整数からなる長さ $ N $ の数列 $ a $ において、$ a_1,\ a_2,\ ...,\ a_N $ の最大公約数のとり得る最大値を求めてください。

## 说明/提示

### 制約

- 入力はすべて整数である
- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ N\ \leq\ M\ \leq\ 10^9 $

### Sample Explanation 1

$ (a_1,\ a_2,\ a_3)\ =\ (2,\ 4,\ 8) $ としたときこれらの最大公約数が $ 2 $ となり最大です。

## 样例 #1

### 输入

```
3 14```

### 输出

```
2```

## 样例 #2

### 输入

```
10 123```

### 输出

```
3```

## 样例 #3

### 输入

```
100000 1000000000```

### 输出

```
10000```

## 题解对比与结论

### 关键思路总结
1. **字符过滤与转换**：将非辅音字符（元音字母、标点符号）过滤，辅音字母按照映射表转换为数字。
2. **大小写统一处理**：将所有字母转换为小写后处理。
3. **输出格式控制**：使用标记变量控制空格的插入，确保首单词前无空格、末尾无多余空格且必须换行。
4. **空单词处理**：转换后为空的单词不输出。

- 空格控制：通过 `op` 标记实现首个单词前不加空格
- 鲁棒性强：正确处理标点符号和大小写

**关键代码**：
```cpp
int main() {
    int n, op = 0;
    while (n--) {
        string s, t;
        cin >> s;
        for (char c : s) {
            if (is_consonant(c)) t += convert(c); // 过滤并转换字符
        }
        if (!t.empty()) {
            cout << (op++ ? " " : "") << t; // 空格控制
        }
    }
    cout << endl;
}
```

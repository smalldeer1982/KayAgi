# [ABC090D] Remainder Reminder

## 题目描述

高橋君有一个由两个不超过 $N$ 的正整数 $(a,b)$ 组成的数对，但他忘记了具体数值。他只记得 $a$ 除以 $b$ 的余数不小于 $K$。求可能符合条件的数对总数。

### 输入格式
输入两个整数 $N$ 和 $K$。

### 输出格式
输出满足条件的数对个数。

### 说明/提示
- $1 \leq N \leq 10^5$
- $0 \leq K \leq N-1$

**样例解释1**：当输入5 2时，符合条件的7组为：(2,3),(5,3),(2,4),(3,4),(2,5),(3,5),(4,5)

**算法分类**：数学

## 题解综合分析

### 核心思路
枚举除数 $b$，计算每个 $b$ 对应的合法 $a$ 的数量。对于每个 $b$：
1. 完整周期贡献：$\lfloor \frac{n}{b} \rfloor \times (b-K)$
2. 剩余部分贡献：$\max(0, n\%b - K + 1)$
3. 处理 $K=0$ 时的边界情况

### 最优题解

#### 题解作者：RainFestival（4星）
**关键亮点**：
- 数学推导清晰，转化余数问题为区间计算
- 代码简洁高效，时间复杂度 $O(n)$
- 完美处理边界条件（包括 $K=0$ 的特殊情况）

**核心代码**：
```cpp
#include<cstdio>
#include<algorithm> 
int n,k;
long long ans=0;
int main() {
    scanf("%d%d",&n,&k);
    for (int i=k+1;i<=n;i++) {
        int x=n/i, r=n%i;
        ans += 1ll*x*(i-k) + std::max(0, r-k+1);
        if (k==0) --ans;
    }
    printf("%lld\n",ans);
}
```

**实现思想**：
1. 枚举除数 $i$（从 $k+1$ 到 $n$）
2. 计算完整块的贡献 `x*(i-k)`
3. 计算余数部分的贡献 `max(0, r-k+1)`
4. 对 $K=0$ 的情况进行修正

## 关键思路总结
1. **枚举除数思想**：将二维数对问题降维处理，固定除数后可通过数学计算快速求解
2. **分块计算技巧**：利用 $\lfloor \frac{n}{b} \rfloor$ 将数值分解为完整块+余数部分
3. **边界处理艺术**：通过 `max(0, ...)` 简化余数部分判断，避免复杂分支

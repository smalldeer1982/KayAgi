# [ARC138B] 01 Generation

## 题目描述

[problemUrl]: https://atcoder.jp/contests/arc138/tasks/arc138_b

すぬけくんは，$ 0 $ と $ 1 $ からなる長さ $ N $ の整数列を作ろうとしています． 今すぬけ君は空の数列 $ x $ を持っており，これから以下の $ 2 $ 種類の操作を好きな順番で $ N $ 回行います．

- 操作A: $ x $ の要素をすべて flip する．つまり，$ 0 $ ならば $ 1 $ に変え，$ 1 $ ならば $ 0 $ に変える． その後，$ x $ の先頭に $ 0 $ を追加する．
- 操作B: $ x $ の末尾に $ 0 $ を追加する．

$ 0 $ と $ 1 $ からなる長さ $ N $ の整数列 $ A=(A_1,A_2,\cdots,A_N) $ が与えられます． $ x $ を $ A $ に一致させることが可能かどうか判定してください．

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 0\ \leq\ A_i\ \leq\ 1 $
- 入力される値はすべて整数

### Sample Explanation 1

以下のように操作すればよいです． - 始状態：$ x=() $ - 操作Aを行う．$ x=(0) $ となる． - 操作Bを行う．$ x=(0,0) $ となる． - 操作Aを行う．$ x=(0,1,1) $ となる． - 操作Bを行う．$ x=(0,1,1,0) $ となる．

# [ARC138B] 01 Generation

## 题目描述

すぬけ君试图通过两种操作构造一个由0和1组成的长度为N的序列。初始时他有一个空序列x，可以通过以下两种操作各进行N次：

- **操作A**：将x的所有元素翻转（0变1，1变0），然后在序列开头添加0。
- **操作B**：在序列末尾添加0。

给定目标序列A，判断能否通过这些操作得到A。

## 算法分类
构造、贪心

## 题解分析与结论

### 核心思路
采用逆向思维，从目标序列倒推构造过程：
1. **逆向B操作**：不断删除末尾的0（对应正向B的逆操作）
2. **逆向A操作**：当无法删除时，若序列开头为0则删除并翻转整个序列的状态

### 最优题解

#### 作者：4041nofoundGeoge（★★★★☆）
**关键亮点**：
- 使用双指针维护当前处理区间
- 用`res`变量记录当前翻转状态
- 时间复杂度O(n)，代码简洁高效

**核心代码**：
```cpp
int l=1, r=n, res=1;
while(l<=r) {
    while(a[r] != res) r--; // 删除末尾的0
    if(a[l] != res) {       // 执行逆A操作
        res ^= 1;
        l++;
    } else {
        puts("No");
        return 0;
    }
}
puts("Yes");
```

#### 作者：CodingOIer（★★★★☆）
**关键亮点**：
- 使用`rf`标记翻转状态
- 通过异或操作简化状态转换
- 代码逻辑与双指针配合紧密

**核心代码**：
```cpp
int i=1, j=n;
bool rf=0;
while(i<=j) {
    if(get(j) == 0) j--;   // get()包含rf的异或逻辑
    else if(get(i) == 1) {  // 检查开头是否为0（考虑翻转状态）
        i++;
        rf ^= 1;
    } else {
        puts("No");
        return 0;
    }
}
```

### 总结技巧
1. **逆向思维**：将构造问题转化为删除问题，避免正向操作的组合爆炸
2. **状态维护**：用布尔变量记录翻转状态，避免实际翻转数组
3. **双指针优化**：通过左右指针快速定位可删除元素，达到线性复杂度

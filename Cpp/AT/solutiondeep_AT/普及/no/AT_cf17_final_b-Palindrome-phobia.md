# Palindrome-phobia（回文恐惧症）

## 题目描述

すぬけ君有一个仅由字符 `a`、`b`、`c` 组成的字符串 $ S $。他希望重新排列 $ S $ 中的字符，使得新字符串不包含任何长度≥2的回文子串。判断是否存在这样的排列方式。

### 说明/提示

- **输入约束**：$ 1 \leq |S| \leq 10^5 $，且 $ S $ 仅含 `a`、`b`、`c`。
- **样例说明**：例如输入 `abac` 可排列为 `acba` 避免回文。

## 题解分析与结论

### 题解对比
1. **Kvaratskhelia（★★★★★）**
   - **关键思路**：统计字符频次，排序后检查最大频次与最小频次差值是否小于2。
   - **亮点**：逻辑简洁，直接利用频次差判断能否构造无回文字符串，时间复杂度 $O(1)$。
   - **代码核心**：
     ```cpp
     sort(a, a+3);
     if (a[2] - a[0] < 2) cout << "YES";
     ```

2. **wdgm4（★★★★☆）**
   - **思路**：构造 `abcabc...` 循环模式后，剩余字符需均不超过1个。
   - **亮点**：通过构造循环结构直观解释条件，剩余字符处理逻辑清晰。
   - **代码核心**：
     ```cpp
     a[3] -= a[1]; a[2] -= a[1];
     if (a[3] >= 2 || a[2] >= 2) cout << "NO";
     ```

3. **封禁用户（★★★★☆）**
   - **思路**：同时检查最大频次与中间、最小频次的差值。
   - **亮点**：双重条件判断覆盖更多边界情况，但实际等价于频次差≤1。
   - **代码核心**：
     ```cpp
     if (a[2]-a[0]>1 || a[2]-a[1]>1) ok = false;
     ```

### 最优思路总结
通过统计字符频次并排序，若最大频次与最小频次差值 ≤1，则可通过交替排列（如 `abcabc...`）避免回文。反之，若差值 ≥2，必然存在相邻重复字符导致回文。

## 关键代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;
    int cnt[3] = {0};
    for (char c : s) cnt[c - 'a']++;
    sort(cnt, cnt + 3);
    cout << (cnt[2] - cnt[0] <= 1 ? "YES" : "NO");
}
```

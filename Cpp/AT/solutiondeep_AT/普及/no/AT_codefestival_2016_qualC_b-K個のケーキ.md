# K個のケーキ

## 题目描述

[problemUrl]: https://atcoder.jp/contests/code-festival-2016-qualc/tasks/codefestival_2016_qualC_b

$ K $ 個のケーキがあります。高橋君は、1日に一つずつ、$ K $ 日かけてこれらのケーキを食べようと考えています。

ケーキは$ T $ 種類あり、種類$ i\ (1≦i≦T) $ のケーキは$ a_i $ 個あります。

二日連続で同じ種類のケーキを食べると飽きてしまうため、高橋君は、うまくケーキを食べる順番を決めて、前日と同じ種類のケーキを食べる日数を最小にしようと考えました。

高橋君のために前日と同じ種類のケーキを食べる日数の最小値を求めてください。

## 说明/提示

### 制約

- $ 1≦K≦10000,\ 1≦T≦100 $
- $ 1≦a_i≦100 $
- $ a_1+a_2+ $...$ +a_T\ =\ K $

### Sample Explanation 1

ケーキは$ 7 $個あります。例えば種類$ 2,1,2,3,1,3,1 $の順で食べると一度も前日と同じ種類のケーキを食べなくてすみます。

### Sample Explanation 2

ケーキは$ 6 $個あります。種類$ 2,3,2,2,1,2 $の順で食べると4日目だけ前日と同じ種類$ 2 $のケーキを食べることになり、これが最小になるので答えは$ 1 $です。

### Sample Explanation 3

高橋君は一種類のケーキしか持っていないため、2日目以降は毎日前日と同じ種類のケーキを食べるしかありません。

## 样例 #1

### 输入

```
7 3

3 2 2```

### 输出

```
0```

## 样例 #2

### 输入

```
6 3

1 4 1```

### 输出

```
1```

## 样例 #3

### 输入

```
100 1

100```

### 输出

```
99```

# K个蛋糕

## 题目描述

$ K $ 个蛋糕排成一列。高桥君要每天吃一个蛋糕，连续两天不能吃同一种类的蛋糕。给定$ T $种蛋糕的数量（总和为$ K $），求不可避免的连续吃同种蛋糕的最小天数。

### 输入格式
- 第一行两个整数$ K $和$ T $
- 第二行$ T $个整数表示各类型蛋糕数量

### 输出格式
- 输出最小连续天数

### 样例解释
- 样例1：最优排列无连续，输出0
- 样例2：最多蛋糕4个，需插入其他蛋糕后剩余1个导致连续1天
- 样例3：只有一种蛋糕，必须连续99天

### 算法分类
贪心

---

## 题解对比与结论

### 最优思路
核心公式：`max(0, max_count - (K - max_count) - 1)`  
本质是通过贪心策略将最多蛋糕间隔排列，当该类型数量超过其他总和时，剩余部分必然产生连续天数。

### 高分题解精选

#### 1. 快乐一凡（5星）
**亮点**：公式推导清晰，代码简洁高效  
**思路**：  
- 找出最大蛋糕数max_count
- 计算理论最小连续天数`max(0, max_count*2 - K -1)`  
**代码核心**：
```cpp
long long maxn = *max_element(a, a+T);
cout << max(maxn*2 - K -1, 0ll);
```

#### 2. Ca1JH（4星）
**亮点**：代码规范，排序法找最大值  
**思路**：  
- 排序后取最大值更直观
- 公式等价变形为`max(0, max_count - (K - max_count) -1)`  
**代码核心**：
```cpp
sort(a, a+T);
cout << max(a[T-1]*2 -1 -K, 0);
```

#### 3. lky1433223（4星）
**亮点**：数学表达式简写更易理解  
**关键推导**：  
- 当max_count > 其他总和时，答案=max_count - 其他总和 -1  
**公式表达**：  
`max(max_count -1 - (K - max_count), 0)`

## 关键技巧总结
1. **贪心核心**：通过最大值与其他值的关系判断能否完全间隔
2. **数学建模**：将问题转化为最大值与剩余总和的差值计算
3. **边界处理**：使用max(0, ...)保证非负结果

# Stones

## 题目描述

[problemUrl]: https://atcoder.jp/contests/tenka1-2019/tasks/tenka1_2019_c

$ N $ 個の石が一列に並んでおり、すべての石は白か黒で塗られています。 石の状態は長さ $ N $ の文字列 $ S $ で表され、$ S $ の $ i $ 文字目が `.` のとき左から $ i $ 個目の石が白であり、`#` のとき左から $ i $ 個目の石が黒であることを表します。

高橋君は、$ 0 $ 個以上の石の色を黒または白に変更し、黒い石のすぐ右に白い石があるような箇所がないようにしたいです。 色を変更する必要のある石の個数の最小値を求めてください。

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 2\times\ 10^5 $
- $ S $ は `.`, `#` のみからなる長さ $ N $ の文字列である

### Sample Explanation 1

例えば、$ 1 $ 個目の石の色を白に変更すればよいです。

## 题目描述

$ N $ 个石子排成一列，每个石子为白色（`.`）或黑色（`#`）。要求通过改变最少数量的石子颜色，使得所有黑色石子右侧不存在白色石子。求最小修改次数。

## 说明/提示

### 制約
- $ 1 \leq N \leq 2 \times 10^5 $
- $ S $ 为仅包含 `.` 和 `#` 的长度 $ N $ 的字符串

### 题解分析与结论
所有题解均采用**分割点枚举+前缀和优化**的核心思路。通过预处理前缀和数组快速计算任意分割点左侧需要染白的次数（原黑色数）和右侧需要染黑的次数（原白色数），时间复杂度优化至 O(n)。

### 精选题解

1. **作者：zgy_123（★★★★★）**  
   **关键亮点**：  
   - 公式推导最简形式：ans = min(n-i - sum_all_black + 2*pre_black[i])  
   - 代码简洁高效，变量命名清晰  
   **核心代码**：
   ```cpp
   for(int i=0;i<=n;i++) 
       ans = min(ans, n-i - p[n] + 2*p[i]);
   ```

2. **作者：QWQ_SenLin（★★★★☆）**  
   **关键亮点**：  
   - 直观的变量命名和逻辑流程

   for(int i = 1;i <= n + 1;i++)
       ans = min(ans, sum[i-1] + (n - i + 1 - (sum[n] - sum[i-1])));
   ```

3. **作者：fmdd_szx（★★★★☆）**

   for(int i=0;i<=n;i++)
       minn = min(minn, sum[n-i] + i - sum[n] + sum[n-i]);
   ```

### 关键思路总结
**最优解法**：  
1. 预处理前缀和数组统计黑色石子数  
2. 枚举分割点，左侧全白（需改黑数=原黑数），右侧全黑（需改白数=原白数）  
3. 数学变形将总次数转化为前缀和表达式，快速计算最小值  

**代码技巧**：  
- 分割点枚举范围应包含 0（全黑）和 n+1（全白）  
- 利用前缀和差代替重复计算

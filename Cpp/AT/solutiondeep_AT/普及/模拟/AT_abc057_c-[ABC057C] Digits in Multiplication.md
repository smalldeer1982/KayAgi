# [ABC057C] Digits in Multiplication

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc057/tasks/abc057_c

整数 $ N $ が与えられます。   
 ここで、$ 2 $ つの正の整数 $ A,B $ に対して、$ F(A,B) $ を「$ 10 $ 進表記における、$ A $ の桁数と $ B $ の桁数のうち大きい方」と定義します。   
 例えば、$ F(3,11) $ の値は、$ 3 $ は $ 1 $ 桁、$ 11 $ は $ 2 $ 桁であるため、$ F(3,11)=2 $ となります。   
 $ 2 $ つの正の整数の組 $ (A,B) $ が $ N=A×B $ を満たすように動くとき、$ F(A,B) $ の最小値を求めてください。

## 说明/提示

### 制約

- $ 1≦N≦10^{10} $
- $ N $ は整数である。

### Sample Explanation 1

$ (A,B)=(100,100) $ のときに $ F(A,B) $ は最小値をとるため、$ F(100,100)=3 $ を出力します。

### Sample Explanation 2

条件を満たす $ (A,B) $ の組は $ (1,1000003) $ と $ (1000003,1) $ の $ 2 $ 通りで、$ F(1,1000003)=F(1000003,1)=7 $ です。

# [ABC057C] 乘法中的位数

## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc057/tasks/abc057_c

给定一个整数 $N$。
这里，对于两个正整数 $A$ 和 $B$，定义 $F(A,B)$ 为 “在十进制表示中，$A$ 的位数和 $B$ 的位数中较大的那个”。
例如，$F(3,11)$ 的值，因为 $3$ 是 $1$ 位数，$11$ 是 $2$ 位数，所以 $F(3,11) = 2$。
当两个正整数对 $(A,B)$ 满足 $N = A×B$ 时，求 $F(A,B)$ 的最小值。

## 说明/提示
### 限制条件
- $1\leq N\leq 10^{10}$
- $N$ 是整数。

### 样例解释 1

满足条件的 $(A,B)$ 对有 $(1,1000003)$ 和 $(1000003,1)$ 这两种情况，且 $F(1,1000003) = F(1000003,1) = 7$。

### 综合分析与结论
这些题解的核心思路都是通过寻找 $N$ 的因数对 $(A,B)$ 使得 $N = A×B$，并让 $A$ 和 $B$ 的值尽可能接近，这样 $F(A,B)$ 能取到最小值。

#### 算法要点
- **因数搜索**：从 $\sqrt{N}$ 开始向 $1$ 枚举，找到第一个能整除 $N$ 的数 $i$，则 $A = i$，$B = N / i$。
- **位数计算**：计算 $A$ 和 $B$ 的位数，取较大值作为 $F(A,B)$。

#### 解决难点
- **优化搜索范围**：从 $\sqrt{N}$ 开始枚举因数，避免了从 $1$ 到 $N$ 的全量枚举，减少了时间复杂度。
- **位数计算方式**：可以使用循环不断除以 $10$ 计数，也可以转换为字符串求长度。

- **MaoHanKun（5星）**
    - **关键亮点**：思路简洁清晰，代码实现简单直接，直接从 $\sqrt{N}$ 开始找因数，然后计算位数。
```cpp
#include<cstdio>
#include<cmath>
using namespace std;
long long n,ans;
int s;
int main(){
    scanf("%lld",&n);
    for(int i=sqrt(n);i>0;--i)
        if(n%i==0){
            ans=n/i;
            break;
        }
    while(ans>0){
        ++s;
        ans/=10;
    }
    printf("%d\n",s);
    return 0;
}
```

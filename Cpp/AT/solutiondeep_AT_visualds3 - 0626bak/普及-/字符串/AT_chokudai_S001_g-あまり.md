# あまり

## 题目描述

有一个长度为n的数列a，你需要把这个数列里的所有数从左到右拼接起来形成一个数，然后把这个数%$1,000,000,007(1e9+7)$并输出。

## 说明/提示

$1 ≤ N ≤ 100,000$。

别忘了结尾换行！

## 样例 #1

### 输入

```
5

3 1 5 4 2```

### 输出

```
31542```

## 样例 #2

### 输入

```
6

1 2 3 4 5 6```

### 输出

```
123456```

## 样例 #3

### 输入

```
7

7 6 5 4 3 2 1```

### 输出

```
7654321```

## 样例 #4

### 输入

```
20

19 11 10 7 8 9 17 18 20 4 3 15 16 1 5 14 6 2 13 12```

### 输出

```
370453866```

## **综合分析与结论**  
题目要求将数列所有数字拼接后取模，关键难点在于处理超大数无法直接存储的问题。各题解核心思路均围绕**逐位取模**展开，避免存储完整数值。对比发现：

- **字符串直接拼接 + 逐位取模**（da32s1da）是最优解法，时间复杂度 O(L)（L 为总位数），空间高效。

- **Python 整型转换法**（wanghy989/CZQ_King）依赖语言特性，可能在大数据时内存不足。

## **最优思路提炼**  
**核心技巧**：  
- **逐位取模法**：将拼接后的数字视为字符串，从左到右依次计算 `ans = (ans * 10 + digit) % MOD`，避免存储大数。  
- **数学性质应用**：利用取模运算的分配律 `(a*b + c) % mod = ((a*b % mod) + c) % mod`，确保中间结果不溢出。

**代码实现**：  
```cpp
string s = "";
for (int i=0; i<n; i++) {
    string x; cin >> x;
    s += x;
}
long long ans = 0;
for (char c : s) {
    ans = (ans * 10 + (c-'0')) % MOD;
}
```

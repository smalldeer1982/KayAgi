# String Sorting

## 题目描述

设有 $n$ 个正整数（$n \le 100$）长度为 $m$（$m \le 30$），将它们联接成一排，组成一个最大的多位整数。

### **综合分析与结论**
**核心思路**：通过自定义排序规则，比较字符串拼接后的字典序，确保相邻字符串组合后的值最大。  
**解决难点**：正确推导比较规则（比较 `a+b` 与 `b+a` 的字典序），而非简单按数值或字典序排序。  
**关键步骤**：  
1. 将字符串数组按 `a+b > b+a` 的规则排序。  
2. 直接拼接排序后的字符串，得到最大结果。  

**可视化设计**：  
- **动画方案**：  
  - **网格显示**：以像素块展示字符串数组，每一步高亮当前比较的 `a[i]` 和 `a[j]`。  
  - **拼接演示**：生成 `a+b` 和 `b+a` 的字符串，比较结果后决定是否交换位置。  
  - **音效提示**：比较时播放“滴”声，交换时播放“咔嗒”声，完成排序后播放胜利音效。  
- **复古风格**：采用 8-bit 像素字体，背景为深蓝色，字符串块用绿色/红色区分选中状态。

### **最优思路与代码实现**
**核心逻辑**：通过比较 `a+b` 和 `b+a` 的字典序，确定排序顺序。  
```cpp
bool cmp(string a, string b) {
    return a + b > b + a; // 拼接后字典序更大的组合优先
}

// 完整代码（以林家三少为例）
#include <iostream>
#include <algorithm>
using namespace std;
string a[105];
int n, m;
bool cmp(string x, string y) { return x + y > y + x; }
int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> a[i];
    sort(a + 1, a + 1 + n, cmp);
    for (int i = 1; i <= n; i++) cout << a[i];
    cout << endl;
    return 0;
}
```

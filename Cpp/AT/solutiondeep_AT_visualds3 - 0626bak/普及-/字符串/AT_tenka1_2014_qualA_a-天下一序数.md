# 天下一序数

## 题目描述

这是一道提交答案题（虽然在洛谷上，你只能提交程序源代码）。

请将1至1000这1000个数字按字典序升序排列。例如：将1,2,11,12,21按照字典序升序排序后的结果是：1,11,12,2,21。

### 核心思路与难点  
所有题解均围绕「将数字转为字符串后按字典序排序」的核心逻辑，关键难点在于：  
1. 正确生成字符串（前导零处理、边界值1000）  
2. 高效排序（避免O(n²)的冒泡排序）  
3. 数据结构选择（利用STL特性简化代码）

## 题解清单（≥4星）  
### 1. CZQ_King的排序版（⭐️⭐️⭐️⭐️）  
**亮点**：  
- 使用`stringstream`统一处理数字转字符串  
- 清空缓存操作严谨（`ss.str("")`+`ss.clear()`）  
**核心代码**：  
```cpp
for(int i=1;i<=1000;i++){
    ss<<i;
    ss>>a[i-1];
    ss.str(""); // 清空缓存
    ss.clear(); // 重置状态
}
sort(a,a+1000);
```

### 2. Kevin_Zhen的优先队列解法（⭐️⭐️⭐️⭐️）  
**亮点**：  
- 利用`priority_queue`自动维护顺序  
- 逆向构建字符串避免反转（`s.insert(s.begin(), ...)`）  
**核心逻辑**：  
```cpp
priority_queue<string, vector<string>, greater<string>> q;
while(t != 0){
    s.insert(s.begin(), t%10 + '0'); // 逆向插入
    t /= 10;
}
```

### 3. konglk的to_string解法（⭐️⭐️⭐️⭐️⭐️）  
**亮点**：  
- 使用C++11标准函数`to_string`简化代码  
- 结构体封装实现稳定排序  
**关键实现**：  
```cpp
struct str { string s; };
bool cmp(str a, str b) { return a.s < b.s; }
// 转换：
st[i].s = to_string(i);
```

## 最优思路提炼  
1. **字符串转换最佳实践**：  
   - C++11+环境优先使用`to_string`  
   - 旧标准可用`stringstream`或逆向插入法  
2. **排序优化技巧**：  
   - 直接调用`sort`函数（时间复杂度O(n log n)）  
   - 避免手动实现冒泡排序（O(n²)效率低下）  
3. **数据结构活用**：  
   - `map`自动排序适合动态插入场景  
   - 优先队列适合流式数据处理

# 题目信息

# [ABC028C] 数を3つ選ぶマン

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc028/tasks/abc028_c

異なる整数が $ 5 $ 個与えられます。

この中から $ 3 $ つ選んでその和で表すことの出来る整数のうち、$ 3 $ 番目に大きいものを出力してください。

## 说明/提示

### Sample Explanation 1

$ 3 $ つ選んでその和で表すことのできる整数は $ 6,\ 7,\ 8,\ 9,\ 10,\ 11,\ 12 $ です。なので、このなかで $ 3 $ 番目に大きい $ 10 $ を出力します。

## 样例 #1

### 输入

```
1 2 3 4 5```

### 输出

```
10```

## 样例 #2

### 输入

```
1 2 3 5 8```

### 输出

```
14```

## 综合分析与结论

### 核心思路
题目要求从5个不同整数中选出3个求和，找到所有可能和中第三大的值。解题核心分为三步：
1. **枚举所有三元组**：通过三重循环遍历所有组合（C(5,3)=10种），计算和。
2. **去重与排序**：用集合（如`set`）自动去重并排序，或手动存储后排序去重。
3. **定位第三大的值**：从大到小遍历去重后的结果，跳过重复值，取第三项。

### 关键难点
- **去重与高效排序**：需确保和唯一且有序。
- **索引计算**：可能存在多个相同和，需跳过重复项后找到正确位置。

### 题解对比
- **STL `set`法**：利用集合自动去重和排序，代码简洁，但需注意逆序访问。
- **数组手动处理**：需额外处理去重逻辑，代码复杂度较高，易出错。

### 可视化设计
1. **三重循环动画**：高亮当前选中的三个数，显示计算和的过程。
2. **集合插入**：动态展示和插入`set`的过程，颜色标记重复项的自动过滤。
3. **逆序访问**：用红色箭头从`set`末尾回退3步，定位第三大的值。

## 代码实现（关键片段）
```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    int a[5]; set<int> sums;
    for (int i=0; i<5; i++) cin >> a[i];
    for (int i=0; i<3; i++)
        for (int j=i+1; j<4; j++)
            for (int k=j+1; k<5; k++)
                sums.insert(a[i]+a[j]+a[k]);
    auto it = sums.rbegin();
    advance(it, 2); // 移动到第三大的位置
    cout << *it << endl;
}
```

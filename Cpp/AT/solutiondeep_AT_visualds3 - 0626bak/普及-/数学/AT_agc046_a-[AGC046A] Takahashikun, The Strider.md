# [AGC046A] Takahashikun, The Strider

## 题目描述

[problemUrl]: https://atcoder.jp/contests/agc046/tasks/agc046_a

平面上に高橋君がおり、真北を向いて立っています。 高橋君が以下の行動を $ K $ 回繰り返したときに元の位置に戻ってくるような最小の正の整数 $ K $ を求めてください。

- 今向いている方向に $ 1 $ メートル進む。その後、向いている方向を反時計回りに $ X $ 度だけ回転させる。

## 综合分析与结论

### 核心思路与算法要点
题目要求找到最小的正整数 K，使得高桥君在 K 次移动后回到原点。每次移动后逆时针旋转 X 度。通过几何和数论分析：

1. **关键转化**：当且仅当总旋转角度为 360° 的整数倍时，各次移动的向量和为零。此时路径形成闭合正多边形。
2. **数学条件**：求最小 K 使得 K*X ≡ 0 (mod 360)，即 K = 360 / gcd(X, 360)。
3. **实现核心**：计算 X 与 360 的最大公约数（GCD），再求出 360 与该 GCD 的商。

### 解决难点
- **几何直观**：需意识到闭合路径的条件是总旋转角度为 360° 的整数倍。
- **向量和为零**：通过正多边形性质，推导出向量和为零的条件。

### 可视化设计思路
1. **动画演示**：展示每一步移动的方向（箭头）和轨迹，用不同颜色标记当前移动步骤。
2. **高亮关键**：总旋转角度达到 360° 倍数时，路径闭合，突出此时的位置和方向。
3. **交互控制**：允许调整 X 值，实时显示 K 的计算结果和路径变化。

## 最优题解代码

```cpp
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int main() {
    int X;
    cin >> X;
    int d = gcd(X, 360);
    cout << 360 / d << endl;
    return 0;
}
```

### 核心实现思想
- **计算 GCD**：通过辗转相除法求 X 和 360 的最大公约数。
- **数学公式**：K = 360 / gcd(X, 360)，确保 K 是满足条件的最小正整数。

# 题目信息

# 電卓

## 题目描述

在你的计算器上输入非负整数 $ X，\ Y $，然后以 $ \frac{X}{Y} $ 为开头，没有多余的 $ 0 $，小数点后第 $3$ 及以下位的数全部舍弃，显示到小数点后第 $2$ 位。例如，当 $ X=4,Y=2$ 时，计算器上显示  `2.00`；当 $ X=2,Y=3$ 时，计算器上显示 ` 0.66`。但是，如果 $Y=0$ 的时候要显示 `ERROR`。

输入 $ X,Y $，请你编写出求计算器显示的程序。

## 说明/提示

- 输入的所有的数都是整数；
- $ 0\ \leq\ X,\ Y\ \leq\ 100 $

样例解释 $1$：

- $ \frac{X}{Y}= 33.33333 \ \cdots $，计算器上显示的是到小数点后第 $ 2 $ 位 `33.33`。

样例解释 $2$：

- 在 $ Y=0 $ 的情况下输出 `ERROR`。

## 样例 #1

### 输入

```
100 3```

### 输出

```
33.33```

## 样例 #2

### 输入

```
42 0```

### 输出

```
ERROR```

### **综合分析与结论**  
#### **核心思路**  
题目要求将两个整数相除后保留两位小数（直接截断第三位），需处理除数为0的情况。关键点在于避免浮点精度问题，通过整数运算逐位获取小数点后的数字。

#### **解决难点**  
1. **浮点精度问题**：直接使用浮点数运算可能导致精度丢失（如0.666...存储为0.666667），需通过整数逐位计算。  
2. **截断而非四舍五入**：需手动分离小数部分的十分位和百分位。  

#### **算法流程**  
1. **特判除数为0**：若`Y=0`，直接输出`ERROR`。  
2. **计算整数部分**：`X/Y`。  
3. **计算十分位**：`(X * 10 / Y) % 10`（将余数扩大10倍后取商）。  
4. **计算百分位**：`(X * 100 / Y) % 10`（同理）。  

#### **可视化设计**  
- **复古像素动画**：  
  - 用Canvas绘制计算器屏幕，显示当前步骤（整数、十分位、百分位）。  
  - **颜色高亮**：整数部分用红色，十分位绿色，百分位蓝色，当前计算位闪烁。  
  - **音效**：每计算一位播放“哔”声，错误时播放“失败”音效。  
- **交互控制**：支持单步执行，观察每一步的余数变化。

### **最优思路与技巧**  
#### **关键技巧**  
- **逐位分离法**：通过`X * 10^k / Y % 10`获取小数点后第k位，避免浮点误差。  
- **整数运算优先级**：先计算整数部分，再处理余数扩大后的结果。  

#### **代码实现**  
```cpp  
#include<bits/stdc++.h>  
using namespace std;  
int main() {  
    int x, y;  
    cin >> x >> y;  
    if (y == 0) {  
        cout << "ERROR";  
        return 0;  
    }  
    cout << x / y << "." << (x * 10 / y) % 10 << (x * 100 / y) % 10;  
}  
```

# 题目信息

# [ABC317B] MissingNo.

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc317/tasks/abc317_b

ナオヒロ君は $ N+1 $ 個の連続する整数を $ 1 $ 個ずつ持っていましたが、そのうち $ 1 $ 個をなくしてしまいました。

残っている $ N $ 個の整数が順不同で $ A_1,\ldots,A_N $ として与えられるので、なくした整数を求めてください。

なお、なくした整数が一意に定まるような入力のみが与えられます。

## 说明/提示

### 制約

- $ 2\ \leq\ N\ \leq\ 100 $
- $ 1\ \leq\ A_i\ \leq\ 1000 $
- 入力は全て整数である
- なくした整数は一意に定まる
 
### Sample Explanation 1

ナオヒロ君は初め $ 2,3,4,5 $ の $ 4 $ 個の整数を持っており、$ 4 $ をなくし、$ 2,3,5 $ が残っていました。 なくした整数である $ 4 $ を出力します。

2 3 5```

### 输出

4```

## 样例 #2

### 输入

## 综合分析与结论
### 核心思路
题目要求找出丢失的连续整数。通过分析数组的最小值（min）、最大值（max）及差值（diff）与N的关系，可快速定位丢失的数：
- **当diff == N**：丢失的数位于连续序列中间，通过总和公式计算。
- **当diff == N-1**：丢失的数位于序列头或尾，直接通过min-1或max+1确定。

### 解决难点
- **确定连续序列范围**：通过min和max快速定位原序列的可能范围。
- **总和计算优化**：利用连续数列求和公式避免遍历，提高效率。
- **边界条件处理**：当序列头尾丢失时，仅需验证两个候选值。

## 代码实现
```python
n = int(input())
a = list(map(int, input().split()))
a.sort()
min_val = a[0]
max_val = a[-1]
diff = max_val - min_val
total = sum(a)

if diff == n:
    # 中间缺失
    ans = (min_val + max_val) * (n + 1) // 2 - total
else:
    # 头或尾缺失
    x_candidate = min_val - 1
    sum1 = (x_candidate + max_val) * (n + 1) // 2
    if sum1 - total == x_candidate:
        ans = x_candidate
    else:
        ans = max_val + 1
print(ans)
```

## 最优思路提炼
1. **快速定位**：通过min和max缩小问题范围。
2. **数学公式**：利用连续数列总和公式，避免遍历求和。
3. **边界验证**：仅需检查头尾两种候选值，无需复杂遍历。

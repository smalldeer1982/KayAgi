# 题目信息

# [ABC344B] Delimiter

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc344/tasks/abc344_b

$ N $ 個の整数 $ A_1,A_2,\dots,A_N $ が、 $ 1 $ 行に $ 1 $ つずつ、 $ N $ 行にわたって与えられます。但し、 **$ N $ は入力では与えられません。**  
 さらに、以下が保証されます。

- $ A_i\ \neq\ 0 $ ( $ 1\ \le\ i\ \le\ N-1 $ )
- $ A_N\ =\ 0 $
 
$ A_N,\ A_{N-1},\dots,A_1 $ をこの順に出力してください。

## 说明/提示

### 制約

- 入力は全て整数
- $ 1\ \le\ N\ \le\ 100 $
- $ 1\ \le\ A_i\ \le\ 10^9 $ ( $ 1\ \le\ i\ \le\ N-1 $ )
- $ A_N\ =\ 0 $
 
### Sample Explanation 1

繰り返しになりますが、 $ N $ は入力では与えられないことに注意してください。 この入力においては $ N=4 $ で、 $ A=(3,2,1,0) $ です。

### Sample Explanation 2

$ A=(0) $ です。

## 综合分析与结论
题目要求读取不定数量的整数，直到遇到0为止，并逆序输出所有数。核心在于正确处理输入终止条件和逆序输出逻辑。通过动态存储读取的数据并反向遍历即可解决。算法复杂度为O(N)，时间和空间均高效。

**可视化设计思路**：  
- **动画方案**：用动态数组展示每读取一个数就将其加入数组，遇到0时高亮该元素作为终止。逆序输出时逐步从末尾弹出元素，显示到输出区域。  
- **颜色标记**：输入阶段新元素用绿色高亮，0用红色标记；输出阶段当前元素用黄色标记。  
- **步进控制**：允许用户单步执行读取和输出过程，观察数组变化。

**关键亮点**：直接使用vector存储，代码简洁易懂，正确处理逆序输出逻辑。无需复杂结构，时间复杂度最优。

## 代码实现
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> nums;
    int num;
    while (cin >> num) {
        nums.push_back(num);
        if (num == 0) break;
    }
    for (int i = nums.size() - 1; i >= 0; --i) {
        cout << nums[i] << endl;
    }
    return 0;
}
```

**核心逻辑**：  
- 使用`vector`动态存储输入的每个整数。  
- 循环读取直到遇到0，立即终止输入。  
- 逆序遍历`vector`输出所有元素。

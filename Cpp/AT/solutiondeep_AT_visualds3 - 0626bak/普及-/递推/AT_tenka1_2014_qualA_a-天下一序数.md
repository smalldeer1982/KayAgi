# 天下一序数

## 题目描述

这是一道提交答案题（虽然在洛谷上，你只能提交程序源代码）。

请将1至1000这1000个数字按字典序升序排列。例如：将1,2,11,12,21按照字典序升序排序后的结果是：1,11,12,2,21。

### **题解思路、算法要点与解决难点**

1. **核心逻辑**：将数字转换为字符串，按字典序排序。字典序比较规则为逐字符比较（ASCII顺序），例如"10" < "2"（因'1' < '2'）。  
2. **关键步骤**：  
   - **字符串转换**：将1~1000的每个数字转为字符串形式（注意无需补前导零）。  
   - **排序规则**：直接按字符串字典序比较（`strcmp`或`operator<`）。  
3. **解决难点**：  
   - 正确实现字符串比较逻辑，避免数值大小干扰。  
   - 优化排序效率（如使用高效排序算法或数据结构）。

### **代码核心实现**

```cpp
// CZQ_King的排序版核心代码
#include <bits/stdc++.h>
using namespace std;
string a[1000];
int main() {
    for (int i=1; i<=1000; i++) {
        a[i-1] = to_string(i); // 转换为字符串
    }
    sort(a, a+1000, [](const string &x, const string &y) {
        return x < y; // 字典序比较
    });
    for (auto &s : a) cout << s << endl;
    return 0;
}
```

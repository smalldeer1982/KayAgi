# 题目信息

# [ABC010A] ハンドルネーム

## 题目描述

高桥君的秘书小渚从高桥君那里得到了一个昵称。  
然而，很遗憾，这个昵称在她想要注册的服务中已经被使用了。  
没办法，小渚只好模仿她喜欢的 C++ 语言，在昵称的末尾加上 `pp`。  
现在输入原来的昵称，请你输出新的昵称。

## 样例 #1

### 输入

```
chokudai```

### 输出

```
chokudaipp```

## 样例 #2

### 输入

```
sanagi```

### 输出

```
sanagipp```

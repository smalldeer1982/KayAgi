# 题目信息

# [ABC042B] 文字列大好きいろはちゃんイージー

## 题目描述

有n个长为L的字符串
 
 要求把他们按照字典序进行排序并在一行内输出

## 说明/提示

1，1 <= n,L <= 100且n，L 都是正整数
 
 2，对于第i(1 <= i <= n)个字符串，保证长度为L
 
 3，所有字符串都由小写字母构成

感谢@lsy263  提供的翻译

## 样例 #1

### 输入

```
3 3

dxx

axx

cxx```

### 输出

```
axxcxxdxx```

**题解一片段赏析**  
* **亮点**：`scanf`的`%*d`高级用法  
* **核心代码**：
```cpp
scanf("%d%*d",&n);  // 过滤L
for(i=0;i<n;i++) cin>>a[i];
sort(a,a+n);
```
* **代码解读**：  
  > `%*d`如同"跳过卡"，直接忽略第二个输入值。配合0-indexed循环，避免冗余变量声明  
* 💡 **学习笔记**：`scanf`格式控制符是处理非常规输入的利器  

**题解二片段赏析**  
* **亮点**：优先队列实现排序  
* **核心代码**：
```cpp
priority_queue<string, vector<string>, greater<string>> q;
q.push(t);  // 压入队列
cout << q.top(); q.pop(); // 输出最小元素
```
* **代码解读**：  
  > 小根堆（`greater<string>`）保证每次`top()`都是当前最小字符串，如同自动排序的糖果机  
* 💡 **学习笔记**：优先队列适合动态排序场景  

**题解四片段赏析**  
* **亮点**：极简主义实现  
* **核心代码**：
```cpp
sort(b, b+a);  // a即n

* **代码解读**：

* 💡 **学习笔记**：简洁≠简陋，精准把握需求是关键

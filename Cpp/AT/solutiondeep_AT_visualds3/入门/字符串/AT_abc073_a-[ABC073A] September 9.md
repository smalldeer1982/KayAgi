## 限制条件

- $10 \leq N \leq 99$

$29$ 的个位是 $9$。

$72$ 中不包含 $9$。

#### 4. C++核心代码实现赏析
**通用核心实现（综合优质题解）**  
```cpp
#include <iostream>
int main() {
    int n;
    std::cin >> n;
    if (n % 10 == 9 || n / 10 == 9) // 分离个位/十位
        std::cout << "Yes\n";
    else 
        std::cout << "No\n";
}
```
**代码解读概要**：通过取模(%)获个位，整除(/)获十位，用逻辑或(||)合并判断条件，实现最简路径计算。

**题解一（އ_昔年™_）片段**  
```pascal
if(a mod 10=9)or(a div 10=9) // Pascal版核心
```
**亮点**：跨语言展示算法本质一致性  
**学习笔记**：`mod`/`div`对应`%`/`/`，证明核心逻辑与语言无关

**题解二（Minecraft_slime）片段**  
```cpp
if(a%10==9 || a/10==9) // 带优化退出
{ cout<<"Yes"; return 0; } 
```
**亮点**：流程优化减少不必要的else分支  
**学习笔记**：提前return可简化分支嵌套，但需确保无后续操作

# コード川柳

## 题目描述

[problemUrl]: https://atcoder.jp/contests/code-festival-2015-final-open/tasks/codefestival_2015_final_a

コード川柳コーナーでは、プロコンに関する「あるある」を五七五の調べに乗せた川柳を募集しています。

そこで、高橋君はコード川柳コーナーに投稿する川柳を作りました。

高橋君が作った川柳が五七五になっているかどうかをチェックするプログラムを作成してください。

ただし五七五になっている川柳とは、$ 5 $ 文字、$ 7 $ 文字、$ 5 $ 文字の文字列をこの順に並べたものを指すこととします。

## 说明/提示

### Sample Explanation 1

川柳としては素晴らしい作品ですが、この問題においては五七五になっていません。

#### 4. C++核心代码实现赏析
**通用核心实现**  
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s1, s2, s3;
    cin >> s1 >> s2 >> s3;
    if (s1.length()==5 && s2.length()==7 && s3.length()==5) 
        cout << "valid\n";
    else 
        cout << "invalid\n";
    return 0;
}
```
**代码解读概要**：  
1. 读取三个字符串 → 2. 验证长度顺序 → 3. 按规则输出  

**题解一片段赏析**  
* **亮点**：完整包含头文件与换行处理  
* **核心代码**：  
  ```cpp
  if(a.size()==5 && b.size()==7 && c.size()==5) {

  ```
* **代码解读**：  
  > `size()`返回字符串字符数，`endl`确保输出后换行——这是竞赛题的关键细节！  
* 💡 **学习笔记**：输出格式错误可能导致AC失败，务必验证

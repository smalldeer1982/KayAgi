# 题目信息

# DDCC型文字列

## 题目描述

输入一个字符串 $S$ ，其长度为 $4$ 个字符

如果 $S$ 满足以下条件：

- $S$ 的第一个字符等于 $S$ 的第二个字符
- $S$ 的第三个字符等于 $S$ 的第四个字符
- $S$ 的第一、二个字符与第三、四个字符不相等

我们就认为字符串 $S$ 是DDCC型的

## 样例 #1

### 输入

```
DDCC```

### 输出

```
Yes```

## 样例 #2

### 输入

```
AAAA```

### 输出

```
No```

## 样例 #3

### 输入

```
ABCD```

### 输出

```
No```

#### 4. C++核心代码实现赏析
**本题通用核心C++实现参考**  
* **说明**：综合题目要求的安全实现方案
* **完整核心代码**：
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;  // 整体读入字符串
    
    // 验证长度（题目保证4，实际编程建议添加）
    if(s.size() != 4) {
        cout << "No" << endl;
        return 0;
    }
    
    // 核心判断逻辑
    if(s[0]==s[1] && s[2]==s[3] && s[0]!=s[2]) {
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;
    }
    return 0;
}
```
* **代码解读概要**：  
  > 1. 安全读入整个字符串  
  > 2. 显式长度校验（防御性编程）  
  > 3. 三重条件判断符合DDCC定义

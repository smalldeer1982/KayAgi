## 3. 核心难点辨析与解题策略

1.  **难点：识别重复值的逻辑覆盖**
    * **分析**：必须处理三边全等和两两相等两种场景。优质题解通过异或性质或排序后的位置关系保证完整性
    * 💡 **学习笔记**：数学性质可大幅简化代码，但需验证边界情况

2.  **难点：避免冗余条件判断**
    * **分析**：直接比较可能产生6个分支（如题解WSEDSWZD）。优化方案：用异或/排序减少判断，或确保if-else互斥
    * 💡 **学习笔记**：代码分支数≤3即可覆盖所有情况

3.  **难点：特殊场景处理（正方形）**
    * **分析**：正方形是特殊长方形，三条边输入相同。解法需兼容输出任意边长
    * 💡 **学习笔记**：明确几何定义是解题前提

### ✨ 解题技巧总结
- **技巧1：数学性质转化**：将几何特性（对边相等）转化为数学运算（异或/比较）
- **技巧2：排序预处理**：通过排序将随机输入转化为有序序列，简化逻辑
- **技巧3：最小条件覆盖**：用最少的条件分支覆盖所有边界情况

## 4. C++核心代码实现赏析

**通用核心实现参考**  
```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c;
    // 数学异或法：相同值抵消原理
    cout << (a ^ b ^ c) << endl;
    return 0;
}
```
* **说明**：综合三种解法优点，兼顾效率与简洁性
* **解读概要**：  
  1. 读入三条边  
  2. 利用异或性质：相同值两两抵消  
  3. 直接输出剩余值

---

**题解一（异或法）**  
```cpp
printf("%d\n",a^b^c);
```
* **亮点**：数学思维降维打击
* **解读**：  
  > `a^b^c`的魔法：  
  > - 当两数相同（如a=b）：`a^b=0` → `0^c=c`  
  > - 当三数相同：`a^a=0` → `0^a=a`  
  > 💡 **学习笔记**：异或满足交换律，无需考虑输入顺序

**题解二（排序法）**  
```cpp
sort(a+1, a+4);
if(a[1]==a[2]) cout<<a[3];
else cout<<a[1];
```
* **亮点**：空间换可读性
* **解读**：  
  > 排序后相同值必相邻：  
  > - `[2,2,3]` → 排序后`[2,2,3]`，前两数相等输出第三  
  > - `[2,3,2]` → 排序后`[2,2,3]`，同上  
  > 💡 **学习笔记**：排序将输入标准化

**题解三（条件判断）**  
```cpp
if(a==b) cout<<c;
else if(b==c) cout<<a;
else cout<<b;
```
* **亮点**：逻辑直白易扩展
* **解读**：  
  > 分支顺序保障完整性：  
  > 1. 先查a=b？是则输出c  
  > 2. 再查b=c？是则输出a  
  > 3. 剩余情况必是a=c，输出b  
  > 💡 **学习笔记**：else if确保互斥性

#### **4. C++核心代码实现赏析**  
**通用核心实现**  
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    
    // 计算b的位数
    int len = 0, t = b;
    while (t) len++, t /= 10;
    
    // 拼接数字（数学法）
    long long s = a * pow(10, len) + b;
    long long g = sqrt(s);  // 取整数平方根
    
    // 整数回溯验证
    if (g * g == s) cout << "Yes" << endl;
    else cout << "No" << endl;
    return 0;
}
```
**代码解读概要**：  
> 动态计算`b`的位数`len`→用`10^len`移位`a`后拼接→取整平方根`g`→用**整数乘法**验证平方数

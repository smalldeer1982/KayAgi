# [ABC361B] Intersection of Cuboids

## 题目描述

你正在尝试实现 3D 游戏中的碰撞判定。

在三维空间中，设有一个长方体，其对角线为两点 $(a,b,c)$ 和 $(d,e,f)$，且所有面都平行于 $xy$ 平面、$yz$ 平面或 $zx$ 平面。我们用 $C(a,b,c,d,e,f)$ 表示这样一个长方体。  
（根据这个定义，$C(a,b,c,d,e,f)$ 是唯一确定的。）

现在给定两个长方体 $C(a,b,c,d,e,f)$ 和 $C(g,h,i,j,k,l)$，请判断它们的公共部分的体积是否为正。

## 说明/提示

## 限制条件

- $0 \leq a < d \leq 1000$
- $0 \leq b < e \leq 1000$

- $0 \leq g < j \leq 1000$
- $0 \leq h < k \leq 1000$
- $0 \leq i < l \leq 1000$
- 所有输入均为整数。

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：综合投影思想的最简实现
* **完整核心代码**：
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int a,b,c,d,e,f,g,h,i,j,k,l;
    cin >> a >> b >> c >> d >> e >> f >> g >> h >> i >> j >> k >> l;
    
    // 计算三维投影交集
    int left_x = max(a, g), right_x = min(d, j);
    int left_y = max(b, h), right_y = min(e, k);
    int left_z = max(c, i), right_z = min(f, l);
    
    // 验证所有维度存在正交集
    bool intersect = (right_x > left_x) && 
                    (right_y > left_y) && 
                    (right_z > left_z);
    
    cout << (intersect ? "Yes" : "No");
    return 0;
}
```
* **代码解读概要**：通过max/min计算每个维度的有效重叠区间，当且仅当三个维度都存在正长度区间时输出"Yes"

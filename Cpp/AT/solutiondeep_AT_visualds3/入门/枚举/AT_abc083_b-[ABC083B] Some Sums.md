# 题目信息

# [ABC083B] Some Sums

## 题目描述

在 $1$ 以上 $N$ 以下的整数中，求所有 $10$ 进制中各位数的和在 $A$ 和 $B$ 之间的数的总和。

## 样例 #1

### 输入

```
20 2 5```

### 输出

```
84```

## 样例 #2

### 输入

```
10 1 2```

### 输出

```
13```

## 样例 #3

### 输入

```
100 4 16```

### 输出

```
4554```

#### 3. 核心难点辨析与解题策略
1. **难点：高效计算各位数字和**  
   * **分析**：优质解法均采用`num % 10`取个位后`num /= 10`迭代，时间复杂度O(logN)。避免低效的字符串转换（如Anguei的Python解法）。  
   * 💡 学习笔记：**取模+整除是数位拆解的标准操作**。

2. **难点：准确判断区间条件**  
   * **分析**：所有解法统一使用`if(s>=a && s<=b)`判断，注意`&&`确保闭区间。累加的是原数字而非各位和（易错点）。  
   * 💡 学习笔记：**区间判断要同时满足≥A和≤B**。

3. **难点：变量作用域管理**  
   * **分析**：PC_DOS通过函数封装避免污染主作用域；Miller2019用临时变量`d=t`保护循环变量；AlwaysForward在循环内定义`s,d`保证每次重置。  
   * 💡 学习笔记：**临时变量是保护原始数据的护盾🛡️**。

#### 4. C++核心代码实现赏析
**通用核心实现**  
```cpp
#include <iostream>
using namespace std;
int main() {
    int n, a, b;
    cin >> n >> a >> b;
    long long total = 0;              // 累加器（防溢出）
    for (int i = 1; i <= n; ++i) {    // 遍历1~n
        int sum_digits = 0, num = i;  // 保护循环变量i
        while (num) {                 // 拆解数字
            sum_digits += num % 10;   // 取个位加入总和
            num /= 10;                // 移除个位
        }
        if (sum_digits >= a && sum_digits <= b) 
            total += i;               // 条件满足则累加
    }
    cout << total << endl;
    return 0;
}
```
**代码解读概要**：  
> 1. 读入`n,a,b`三参数  
> 2. 遍历每个数并拆解计算各位和  
> 3. 通过区间判断决定是否累加  
> 4. 输出总和  

---

**题解一（PC_DOS）片段**  
```cpp
register long long iSum = 0;          // 寄存器变量优化
for (i = 1; i <= iEnd; ++i) {         // 循环从1到n
    if (IsAddable(i, iLB, iUB))       // 调用判断函数
        iSum += i;                    // 条件满足累加
}
```
* **亮点**：函数封装使主逻辑清晰  
* 💡 学习笔记：**功能拆分提升代码可读性**

**题解二（Miller2019）片段**  
```cpp
while(d) {                    // 当d非0时循环
    s += d % 10;              // 取最后一位
    d /= 10;                  // 移除最后一位
}
```
* **亮点**：最精简的数位计算实现  
* 💡 学习笔记：**while循环是数位拆解的高效工具**

**题解三（AlwaysForward）片段**  
```cpp
for(int j=1;;j++) {           // 无限循环+break退出
    s += d % 10;              
    d /= 10;                  
    if(d==0) break;           // 终止条件
}
```
* **亮点**：展示循环多样性  
* 💡 学习笔记：**break可灵活控制循环退出**

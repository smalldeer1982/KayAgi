# Yahoo

## 题目描述

高桥君打算开办一家新公司，而这个公司决定建立一个搜索网站，因此他决定将 `yahoo` 这个五个字母重新排列后得到的字符串作为公司的名字。

他的朋友青木提议将字符串 $S$ 作为新公司的名字，请你判定这个字符串是否符合高桥君的要求。
### 题意简化
给定一个长度为 $5$ 的字符串，判断其是否由字符串 `yahoo` 重新排列得到。

## 说明/提示

### 数据限制
- 字符串 $S$ 的长度 $|S| = 5$。
- 字符串 $S$ 均为小写字母。

## 样例 #1

### 输入

```
oohay```

### 输出

```
YES```

## 样例 #2

### 输入

```
abcde```

### 输出

```
NO```

## 样例 #3

### 输入

```
yahoo```

### 输出

```
YES```

## 样例 #4

### 输入

```
yohaa```

### 输出

```
NO```

## 3. 核心难点辨析与解题策略

1.  **难点：准确识别字母计数要求**
    * **分析**：易混淆"o"需出现两次（非一次），需明确"yahoo"的字母组成。优质题解通过预设目标值（y=1,a=1,h=1,o=2）化解。
    * 💡 **学习笔记**：先拆解目标字符串的字母组成再编码。

2.  **难点：选择高效验证方法**
    * **分析**：计数法需遍历一次+四次比较；排序法需排序+五次比较。计数法更通用（尤其字母种类多时），排序法在固定组合时更简洁。
    * 💡 **学习笔记**：小规模数据可尝试转换思路（如排序），大规模数据优先计数。

3.  **难点：避免边界错误**
    * **分析**：如循环条件写`i<=5`会导致越界（索引0-4）。优质题解用`i<s.size()`或精确范围`i<5`规避。
    * 💡 **学习笔记**：字符串遍历时，务必验证索引范围。

### ✨ 解题技巧总结
- **技巧1：计数器初始化**：统计前变量归零（`int y=0,a=0...`）
- **技巧2：字符分类技巧**：用`if-else if`链或`switch`避免重复判断
- **技巧3：STL活用**：小规模数据可尝试`sort`简化逻辑

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;
    int y=0, a=0, h=0, o=0;
    for (int i=0; i<s.size(); i++) {
        if (s[i]=='y') y++;
        else if (s[i]=='a') a++;
        else if (s[i]=='h') h++;
        else if (s[i]=='o') o++;
    }
    cout << (y==1 && a==1 && h==1 && o==2 ? "YES" : "NO");
    return 0;
}
```
* **代码解读概要**：读取字符串 → 遍历计数 → 三元运算符输出结果。`else if`确保高效分类。

---

**题解一（来源：liangbob）**
* **亮点**：布尔表达式直接累加计数，代码紧凑
* **核心代码片段**：
```cpp
y += (s[i]=='y');  // 字符'y'时y+1，否则+0
a += (s[i]=='a');
h += (s[i]=='h');
o += (s[i]=='o');
```
* **代码解读**：
> 利用`true=1/false=0`特性，将判断结果直接转为整数累加。相比`if`分支更简洁，但需理解布尔值隐式转换。
* 💡 **学习笔记**：C++中逻辑表达式可参与数值运算。

**题解二（来源：不会dp退役菜鸡）**
* **亮点**：排序转换思维，避免多重判断
* **核心代码片段**：
```cpp
sort(a, a+5);
if(a[0]=='a' && a[1]=='h' && a[2]=='o' && a[3]=='o' && a[4]=='y') 
    cout<<"YES";
```
* **代码解读**：
> 排序后字母按ASCII码升序排列，固定为`a,h,o,o,y`。通过序列比对替代计数，但仅适用于本题特定字母组合。
* 💡 **学习笔记**：排序能将乱序问题转化为有序匹配。

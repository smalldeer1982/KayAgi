#### 4. C++核心代码实现赏析
**通用核心实现参考**  
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cin >> name;
    int len = name.length();
    
    for (int i = 0; i < len/2; i++) {
        if (name[i] != name[len-i-1]) {
            cout << "NO" << endl;
            return 0;
        }
    }
    cout << "YES" << endl;
    return 0;
}
```
**代码解读概要**：  
> 1. 读入字符串后获取长度`len`  
> 2. 循环遍历前半段（`i < len/2`）  
> 3. 发现不对称立即输出"NO"并退出  
> 4. 全部对称则输出"YES"  

**题解一片段赏析**  
```cpp
for(int i=0; i<name.length()/2; ++i) {
    if(name[i] != name[name.length()-i-1]) {
        cout << "NO" << endl;
        return 0; // 亮点：即时终止
    }
}
```
**代码解读**：  
> - `name.length()/2`：自动适配奇偶长度（如长度5则循环2次）  
> - `name.length()-i-1`：精准计算对称位置（如i=0时定位末尾字符）  
> - `return 0`：发现不匹配立即退出，避免无效循环  
> 💡 **学习笔记**：**提前终止**如同消防警报——第一时间响应风险！

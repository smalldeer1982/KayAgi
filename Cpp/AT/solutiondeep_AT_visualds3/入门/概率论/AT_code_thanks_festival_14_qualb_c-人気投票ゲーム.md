# 人気投票ゲーム

## 题目描述

在某个国家，“狐狸派”和“兔子派”正在进行人气投票。你作为“狐狸派”的参谋，需要考虑怎样才能在这次的人气投票中获胜。

这个国家共有$N$个地区，每个地区都会进行投票。这里给出各个地区的总投票数和其中“狐狸派”的得票数，请你编写一个程序，计算“狐狸派”获得过半数选票的地区个数。

但请注意：“过半数”表示比一半大的数量。

例如，总投票数为$100100$时，得票数为$5050$也不能说是过半数。

## 样例 #1

### 输入

```
5

150 130 100 200 150

100 60 50 101 70```

### 输出

```
2```

## 样例 #2

### 输入

```
5

5 4 3 2 1

2 2 2 1 0```

### 输出

```
1```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题看似简单，却隐藏着三个关键陷阱。结合优质题解，我们总结出以下解题心法：
</difficulty_intro>

1.  **整数除法与浮点陷阱**
    * **分析**：当总票数为奇数时（如5票），"过半"需3票而非2.5票。优质解法都采用整数除法（`v[i]/2`）而非浮点数，避免精度误差
    * 💡 **学习笔记**：整数问题优先用整数运算，浮点数是"隐藏的炸弹"

2.  **严格大于的边界条件**
    * **分析**：题目强调"过半"是严格大于一半。如总票100时，50票不达标，51票才达标
    * 💡 **学习笔记**：比较运算符选择决定成败，`>`和`≥`有天壤之别

3.  **零票数的特殊处理**
    * **分析**：当得票数为0时，无论总票数多少都不可能达标。TRZ的解法显式跳过可提升效率
    * 💡 **学习笔记**：边界值是算法的"压力测试点"

### ✨ 解题技巧总结
-   **防御性编程**：显式处理边界情况（如零值），即使逻辑上可能冗余
-   **整数优先原则**：能用整数运算解决的问题，避免引入浮点数
-   **语义强化注释**：对关键判断条件（如`>`和`≥`）添加警示性注释
-   **测试用例设计**：必须包含总票奇偶/零票数/平票等边界情况

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, count = 0;
    cin >> n;
    int total[105], fox[105];  // 总票数、狐狸派票数
    
    // 输入数据
    for(int i=0; i<n; i++) cin >> total[i];
    for(int i=0; i<n; i++) cin >> fox[i];
    
    // 核心逻辑：严格比较得票数 > 总票数/2
    for(int i=0; i<n; i++) {
        if(fox[i] > total[i] / 2) {  // 关键整数比较
            count++;
        }
    }
    cout << count << endl;
    return 0;
}
```
* **代码解读概要**：该实现完整包含输入处理、核心算法和输出模块。通过`fox[i] > total[i]/2`的整数比较，避免浮点误差且满足"严格大于"的要求。数组索引从0开始更符合C++规范。

---

**题解一（TRZ_2007）**
* **亮点**：显式处理零票数边界，整数除法技巧巧妙
* **核心代码片段**：
    ```c
    for(i=1;i<=n;i++) {
        if(sum[i]==0) continue;     // 边界处理：跳过零票数
        if(ans[i]/sum[i]<2) num++;  // 整数除法技巧
    }
    ```
* **代码解读**：
    > 第一行先过滤得票数为0的地区（因0不可能>任何数/2）。第二行用总票数÷得票数<2等价于得票数>总票数/2，巧妙利用整数除法特性。例如总票100得票51：100/51=1.96→取整为1<2 → 达标。
* 💡 **学习笔记**：数学等价转换可简化条件判断

**题解二（杨瀚雄白羊）**
* **亮点**：判断条件直击本质，现代C++风格
* **核心代码片段**：
    ```cpp
    for(int i=1;i<=n;i++) {
        if(f[i] > v[i]/2)  // 直接比较
            ans++;
    }
    ```
* **代码解读**：
    > 直接使用`f[i] > v[i]/2`的核心判断。当`v[i]`为奇数时（如5），整数除法`5/2=2`，要求`f[i]≥3`才成立，完美符合"严格大于一半"的要求。例如总票5时：3>2成立，2>2不成立。
* 💡 **学习笔记**：简洁的实现往往最有力

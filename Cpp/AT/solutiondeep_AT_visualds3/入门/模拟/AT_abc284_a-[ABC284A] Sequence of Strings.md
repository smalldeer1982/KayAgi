# [ABC284A] Sequence of Strings

## 题目描述

给定 $N$ 个字符串 $S_1,S_2,\ldots,S_N$，按照这个顺序给出。

请按照 $S_N,S_{N-1},\ldots,S_1$ 的顺序输出这些字符串。

## 说明/提示

### 限制条件

- $1\leq N\leq 10$
- $N$ 是整数
- $S_i$ 是由英文字母大小写和数字组成的字符串，长度在 $1$ 到 $10$ 之间

### 样例解释 1

$N=3$，$S_1=$ `Takahashi`，$S_2=$ `Aoki`，$S_3=$ `Snuke`。因此，按照 `Snuke`、`Aoki`、`Takahashi` 的顺序输出。

### 样例解释 2

给定的字符串中也可能包含数字。

由 ChatGPT 4.1 翻译

```

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：Kay自建完整实现，突出倒序循环核心逻辑。
* **完整核心代码**：
  ```cpp
  #include <iostream>
  #include <vector>
  using namespace std;
  
  int main() {
      int n;
      cin >> n;
      cin.ignore(); // 跳过换行符
      
      vector<string> v(n);
      for (int i = 0; i < n; i++) {
          getline(cin, v[i]); // 读取含空格的字符串
      }
      
      for (int i = n - 1; i >= 0; i--) {
          cout << v[i] << "\n"; // 倒序输出
      }
      return 0;
  }
  ```
* **代码解读概要**：
  > 1. 读入整数`n`后，用`vector`存储`n`个字符串。  
  > 2. **核心循环**：索引从`n-1`递减至`0`，按逆序输出元素。  
  > 3. 注意`cin.ignore()`和`getline`配合处理输入格式。

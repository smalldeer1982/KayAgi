# 题目信息

# [ABC323A] Weak Beats

## 题目描述

给定一个由 `0` 和 `1` 组成的长度为 $16$ 的字符串 $S$。

请判断 $2$ 到 $16$ 之间所有的偶数 $i$，$S$ 的第 $i$ 个字符是否为 `0`。如果是，则输出 `Yes`，否则输出 `No`。

## 说明/提示

## 限制

- $S$ 是一个由 `0` 和 `1` 组成的长度为 $16$ 的字符串。

## 样例解释 1

由于 $S=1001000000001010$ 的第 $4$ 个字符为 `1`，因此输出 `No`。

## 样例解释 2

由于 $S=1010100000101000$ 的所有偶数位字符均为 `0`，因此输出 `Yes`。

## 样例解释 3

$S$ 的所有偶数位字符均为 `1`。因此不是“全部为 `0`”，所以输出 `No`。

#### 4. C++核心代码实现赏析
**通用核心实现**  
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;
    for (int i = 1; i <= 15; i += 2) { // 遍历偶数下标
        if (s[i] != '0') {             // 发现非0
            cout << "No" << endl;
            return 0;                   // 提前终止
        }
    }
    cout << "Yes" << endl;
    return 0;
}
```
**代码解读概要**：  
> 1. 读入字符串 → 2. 遍历奇数下标（对应偶数位置）→ 3. 遇'1'输出"No"并退出 → 4. 全'0'输出"Yes"

**题解一点评**  
* **亮点**：提前终止机制优化效率  
* **核心代码**：  
  ```cpp
  for (int i = 1; i <= 15; i += 2) {
      if (s[i] != '0') {
          cout << "No"; return 0; // 发现即退出
      }
  }
  ```
* **代码解读**：  
  > `i+=2` 直接跳过奇数位 → `s[i]`访问偶数位 → 一旦不满足条件，`return 0`立即结束程序  
  > **类比**：就像安检发现违禁品，无需检查后续行李
* 💡 **学习笔记**：`return`在循环内可提前退出，避免冗余计算

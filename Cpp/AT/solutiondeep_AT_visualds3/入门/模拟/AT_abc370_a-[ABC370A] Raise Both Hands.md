# 题目信息

# [ABC370A] Raise Both Hands

## 题目描述

高桥决定做章鱼烧给斯努克吃。高桥指示斯努克，如果他想吃章鱼烧，就只举起左手，否则就只举起右手。

我们将告诉你斯努克两只手的状态 $L$ 和 $R$ 。当且仅当 $L=1$ 时，代表他举起左手，当且仅当 $R=1$ 时代表他举起右手。他可能不会按照指示举手，例如举起两只手，或者一只手都不举。

如果斯努克只举起一只手且他举起的是左手（代表他想吃章鱼烧），就打印 ```Yes``` ，如果他举起的是右手，就打印 ```No``` 。如果他同时举起两只手或一只手都不举，则打印 ```Invalid```。

如果斯努克只举起一只手，他就会一直遵守指示。

## 说明/提示

### 约束

- $ L,\ R $ 都是 $ 0 $ 或者 $ 1 $。

### 样例解释1

斯努克想吃章鱼烧，所以他只举起了左手。

### 样例解释2

斯努克举起了两只手。

## 样例 #1

### 输入

```
1 0```

### 输出

```
Yes```

## 样例 #2

### 输入

```
1 1```

### 输出

```
Invalid```

## 4. C++核心代码实现赏析

**通用核心实现参考**
```cpp
#include <iostream>
using namespace std;

int main() {
    int leftHand, rightHand;
    cin >> leftHand >> rightHand;
    
    if (leftHand == 1 && rightHand == 0) 
        cout << "Yes";
    else if (leftHand == 0 && rightHand == 1) 
        cout << "No";
    else 
        cout << "Invalid";
    
    return 0;
}
```
**代码解读概要**：直击问题本质的三段式结构，如同交通信号灯：①允许通行(Yes) ②禁止通行(No) ③故障状态(Invalid)

---

**题解一：基础条件分支**
```cpp
if (L == 1 && R == 0)      // 左手举&右手落
    cout << "Yes";
else if (L == 0 && R == 1) // 右手举&左手落
    cout << "No";
else                       // 其他组合
    cout << "Invalid";
```
**代码解读**：  
> 第1行：检查唯一有效指令`(1,0)` → 想吃章鱼烧  
> 第3行：检查相反有效状态`(0,1)` → 拒绝指令  
> 第5行：用`else`兜住所有无效组合  
> 💡 **学习笔记**：`else`是条件分支的安全网

**题解二：位运算优化**
```cpp
int state = (L << 1) | R;  // 状态编码：0bLR
switch(state) {
    case 0b10: cout << "Yes"; break; // 二进制2
    case 0b01: cout << "No"; break;  // 二进制1
    default:   cout << "Invalid";     // 0或3
}
```
**代码解读**：  
> `L<<1 | R`将状态压缩为单数字：  
> - L=1,R=0 → 0b10(2)  
> - L=0,R=1 → 0b01(1)  
> - 其他值触发default  
> 💡 **学习笔记**：位运算适合处理多二元状态

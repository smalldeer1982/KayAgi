**本题通用核心C++实现参考**

* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;
    int main() {
        int n, x, sum = 0;  // 关键：sum初始化为0
        cin >> n;            // 读取数据量
        for (int i = 0; i < n; i++) {  // 精确循环n次
            cin >> x;        // 读取当前数字
            sum += x;        // 累加到总和
        }
        cout << sum << endl; // 输出结果
        return 0;
    }
    ```
* **代码解读概要**：通过`for`循环控制读取次数，`sum`实时更新总和。代码突出"初始化-读取-累加"三步逻辑链。

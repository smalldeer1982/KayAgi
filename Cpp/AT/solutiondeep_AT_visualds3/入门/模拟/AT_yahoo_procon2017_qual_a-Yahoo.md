# 题目信息

# Yahoo

## 题目描述

高桥君打算开办一家新公司，而这个公司决定建立一个搜索网站，因此他决定将 `yahoo` 这个五个字母重新排列后得到的字符串作为公司的名字。

他的朋友青木提议将字符串 $S$ 作为新公司的名字，请你判定这个字符串是否符合高桥君的要求。
### 题意简化
给定一个长度为 $5$ 的字符串，判断其是否由字符串 `yahoo` 重新排列得到。

## 说明/提示

### 数据限制
- 字符串 $S$ 的长度 $|S| = 5$。
- 字符串 $S$ 均为小写字母。

## 样例 #1

### 输入

```
oohay```

### 输出

```
YES```

## 样例 #2

### 输入

```
abcde```

### 输出

```
NO```

## 样例 #3

### 输入

```
yahoo```

### 输出

```
YES```

## 样例 #4

### 输入

```
yohaa```

### 输出

```
NO```

## 3. 核心难点辨析与解题策略

1.  **难点：准确理解重组本质**  
    * **分析**：重组不要求顺序，本质是频次匹配。优质题解通过统计字母出现次数（而非比较字符串）破解此陷阱  
    * 💡 **学习笔记**：字符串重组问题 = 字母频率匹配问题  

2.  **难点：高效实现字符计数**  
    * **分析**：直接使用if-else链计数（如题解二）比排序法更可靠。排序法（如题解四）可能因相同字母位置误判  
    * 💡 **学习笔记**：固定长度字符串优先选择O(n)遍历而非O(nlogn)排序  

3.  **难点：边界条件处理**  
    * **分析**：必须严格校验长度=5。题解一/二通过for(i<5)循环隐含此约束  
    * 💡 **学习笔记**：题目明示|S|=5时可省略长度检查，但实际开发需验证  

### ✨ 解题技巧总结
- **技巧1 问题降维**：将字符串重组转化为字母频次统计问题  
- **技巧2 布尔转整**：利用`if(s[i]=='y')`返回1的特性简化计数（见题解二）  
- **技巧3 防御性命名**：变量名应明确表达统计目标（如count_y优于rp）

## 4. C++核心代码实现赏析

**本题通用核心实现参考**  
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s; cin >> s;
    int y=0, a=0, h=0, o=0;
    for(int i=0; i<5; ++i) {
        if(s[i]=='y') y++;
        if(s[i]=='a') a++;
        if(s[i]=='h') h++;
        if(s[i]=='o') o++;
    }
    cout << (y==1 && a==1 && h==1 && o==2 ? "YES" : "NO");
    return 0;
}
```
* **说明**：综合优质题解思路，使用标准变量名与三目运算符优化输出  
* **解读概要**：核心分三步——①读入字符串 ②遍历统计字母 ③验证频次输出

## 题目描述

有一天，haruki先生在学校学了素数。所谓素数，是指只有1和本身两个因数的整数。

非常好奇心旺盛的haruki先生有以下疑问。

“n以下的素数中有几个偶数呢？”haruki很在意！

因此，请为haruki先生创建一个程序，求出在n以下的素数中有几个偶数。

## 样例 #1

### 输入

```
3```

### 输出

```
1```

#### 4. C++核心代码实现赏析
**通用核心实现参考**
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    if (n < 2) cout << 0; // 2不在范围内
    else cout << 1;       // 2是唯一偶素数
    return 0;
}
```
**代码解读概要**：通过输入n直接判断2是否在范围内，无需复杂计算。

**题解一核心片段赏析**
```cpp
if(n<2)
    printf("0\n");
else 
    printf("1\n");
```
**代码解读**：
> `if(n<2)` 是关键过滤条件——当n小于2时，素数2不在范围内。`printf`直接输出结果，`\n`保证格式规范。  
> 💡 **学习笔记**：简洁的if-else结构完美匹配二值输出场景。

**题解三核心片段赏析**
```cpp
if(n<2) cout<<0<<endl;
else cout<<1<<endl;
```
**代码解读**：
> 使用C++流式输出，`cout<<0<<endl`中`endl`自动刷新缓冲区并换行。与题解一逻辑相同但更符合C++风格。  
> 💡 **学习笔记**：`cin/cout`在简单题中更易读，但注意避免与`printf`混用。

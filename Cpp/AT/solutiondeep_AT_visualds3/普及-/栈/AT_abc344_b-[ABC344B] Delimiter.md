# 题目信息

# [ABC344B] Delimiter

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc344/tasks/abc344_b

$ N $ 個の整数 $ A_1,A_2,\dots,A_N $ が、 $ 1 $ 行に $ 1 $ つずつ、 $ N $ 行にわたって与えられます。但し、 **$ N $ は入力では与えられません。**  
 さらに、以下が保証されます。

- $ A_i\ \neq\ 0 $ ( $ 1\ \le\ i\ \le\ N-1 $ )
- $ A_N\ =\ 0 $
 
$ A_N,\ A_{N-1},\dots,A_1 $ をこの順に出力してください。

## 说明/提示

### 制約

- 入力は全て整数
- $ 1\ \le\ N\ \le\ 100 $
- $ 1\ \le\ A_i\ \le\ 10^9 $ ( $ 1\ \le\ i\ \le\ N-1 $ )
- $ A_N\ =\ 0 $
 
### Sample Explanation 1

繰り返しになりますが、 $ N $ は入力では与えられないことに注意してください。 この入力においては $ N=4 $ で、 $ A=(3,2,1,0) $ です。

### Sample Explanation 2

$ A=(0) $ です。

## 4. C++核心代码实现赏析

<code_intro_overall>
虽然暂无题解，但我们可以根据通用思路编写一个清晰的核心实现。以下是一个典型的C++解决方案：
</code_intro_overall>

**本题通用核心C++实现参考**  
* **说明**：本代码基于“动态存储+反向遍历”的思路，适用于处理未知数量的输入并逆序输出，逻辑清晰且符合题目要求。  
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <vector>

    using namespace std;

    int main() {
        vector<int> nums; // 动态数组存储输入的数
        int num;

        // 持续读取输入，直到遇到0（包含0）
        while (cin >> num) {
            nums.push_back(num);
            if (num == 0) { // 遇到0时停止读取
                break;
            }
        }

        // 逆序输出数组中的所有元素
        for (int i = nums.size() - 1; i >= 0; --i) {
            cout << nums[i] << endl;
        }

        return 0;
    }
    ```
* **代码解读概要**：  
  代码首先声明一个`vector<int>`类型的`nums`，用于存储输入的整数。通过`while`循环持续读取输入，每次读取后将数存入`nums`，并检查是否为0——若是则退出循环。最后，从`nums`的最后一个元素开始遍历到第一个元素，逐个输出，实现逆序。

# 题目信息

# [ABC022A] Best Body

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc022/tasks/abc022_a

高橋君は太りやすく痩せやすい体質です。そこで彼は $ N $ 日間の体重の変化量を記録してみました。

$ 1 $ 日目の高橋くんの体重は $ W $ キログラムでした。 $ i $ 日目 $ (2≦i≦N) $ の体重の変化量は $ A_i $ キログラムでした($ A_i $ は負になり得ます)。 つまり $ i-1 $ 日目の体重を $ X $ キログラムとすると、 $ i $ 日目の体重は $ X\ +\ A_i $ キログラムだったということです。

ところで、高橋君の個人的な価値観では、体重が $ S $ キログラム以上 $ T $ キログラム以下の体型が「ベストボディー」だと考えています。

体重を記録した $ N $ 日間のうち、高橋君がベストボディーであった日数を求めてください。

## 说明/提示

### Sample Explanation 1

高橋君の体重を $ 1 $ 日目から順に書き出すと $ 50,\ 60,\ 70,\ 80,\ 90 $ になります。 よって$ 2 $ 日目と $ 3 $ 日目にのみ高橋君はベストボディーだったことになります。

## 样例 #1

### 输入

```
5 60 70

50

10

10

10

10```

### 输出

```
2```

## 样例 #2

### 输入

```
5 50 100

120

-10

-20

-30

70```

### 输出

```
2```

> 解决这道题的关键在于“模拟”高橋君每天的体重变化过程。就像玩“体重追踪游戏”——我们需要从第一天的初始体重开始，逐天加上变化量，然后检查每天的体重是否落在[S, T]的“最佳区间”内。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
在解决这道题时，我们需要特别注意以下三个关键点，掌握它们能帮你快速避开常见错误：
</difficulty_intro>

1.  **关键点1**：第一天的体重是否需要判断？
    * **分析**：题目中明确要求统计“N日間”的体重，因此第一天（初始体重W）必须被检查。例如样例1中，初始体重是50，不在[60,70]范围内，所以不计入结果；而第二天开始的体重是60、70等，才被统计。  
    * 💡 **学习笔记**：题目中的“N日間”指的是从第1天到第N天的所有天数，初始体重是第1天的体重，必须检查。

2.  **关键点2**：变化量的数量是N-1个还是N个？
    * **分析**：题目中说明“i日目（2≤i≤N）的体重变化量是A_i”，因此总共有N-1个变化量（对应第2天到第N天）。例如N=5时，需要读取4个A_i。如果错误读取N个，会导致体重计算多一天，结果错误。  
    * 💡 **学习笔记**：变化量的数量是N-1，循环次数应为N-1次。

3.  **关键点3**：体重等于S或T是否算“最佳”？
    * **分析**：题目描述中“体重がSキログラム以上Tキログラム以下”明确包含等于的情况（“以上”“以下”在日语中通常包含边界值）。例如样例1中，体重60（等于S）和70（等于T）都被计入结果。  
    * 💡 **学习笔记**：条件判断用“>=”和“<=”，而非“>”或“<”。

### ✨ 解题技巧总结
<summary_best_practices>
- **变量初始化**：统计类问题中，结果变量（如ans）要初始化为0，避免未定义行为。  
- **输入顺序**：注意输入的第一行是N、S、T，第二行是初始体重W，之后是N-1个变化量A_i。  
- **边界测试**：测试用例时，特别检查初始体重刚好等于S/T，或变化量导致体重跨越边界的情况（如从S-1变为S）。
</summary_best_practices>

## 4. C++核心代码实现赏析

<code_intro_overall>
我们以优质题解为基础，提炼一个通用的核心实现，帮助大家快速掌握代码框架。
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：此代码综合自优质题解，逻辑简洁，正确处理了所有边界条件，适合直接学习。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int n, s, t, w; // n天，最佳区间[S,T]，初始体重W
        cin >> n >> s >> t >> w;
        int ans = 0;
        // 判断第1天的体重是否符合条件
        if (w >= s && w <= t) ans++;
        // 处理第2天到第n天的变化量（共n-1个）
        for (int i = 0; i < n - 1; i++) {
            int a;
            cin >> a; // 读取当天的变化量
            w += a;   // 更新体重
            if (w >= s && w <= t) ans++; // 判断当前体重
        }
        cout << ans << endl; // 输出符合条件的天数
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码首先读取输入的N、S、T和初始体重W。然后检查初始体重是否在[S,T]内（第1天）。接着循环读取N-1个变化量，逐天更新体重并检查是否符合条件。最后输出统计结果。核心逻辑是“初始判断+循环更新”，简洁高效。

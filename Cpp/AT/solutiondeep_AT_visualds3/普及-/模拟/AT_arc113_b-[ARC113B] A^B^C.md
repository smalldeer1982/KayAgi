# 题目信息

# [ARC113B] A^B^C

## 题目描述

这里有 $3$ 个正整数。请求出 $ A^{B^C} $ 的个位。

## 说明/提示

- $ 1\leq\ A,B,C\ \leq\ 10^9 $
- $ A,B,C $ 是整数。


$ 4^{3^2}=4^9=262144 $ 的个位是 $ 4 $ 。

## 样例 #1

### 输入

```
4 3 2```

### 输出

```
4```

## 样例 #2

### 输入

```
1 2 3```

### 输出

```
1```

## 样例 #3

### 输入

```
3141592 6535897 9323846```

### 输出

```
2```

🗣️ **初步分析**：
解决这道题的关键在于发现“个位数字的幂次具有周期性”。比如，2的幂次个位依次是2→4→8→6→2→4…每4次循环一次。我们需要：
1. **找周期**：确定A的个位数字的循环周期T（如2的周期是4）；

3. **求结果**：用 \( A^{\text{指数模周期}} \) 的个位即为答案。

核心难点是如何高效计算 \( B^C \mod T \)（因B、C极大），以及处理特殊情况（如A的个位是0、1、5、6时周期为1）。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决本题时，我们需要重点突破以下三个关键点：
</difficulty_intro>

1.  **关键点1**：如何确定个位数字的周期？
    * **分析**：不同个位数字的周期不同。例如：
      - 0、1、5、6的幂次个位始终不变（周期1）；
      - 4的幂次个位是4→6→4→6（周期2）；
      - 9的幂次个位是9→1→9→1（周期2）；
      - 2、3、7、8的幂次个位每4次循环（周期4）。
    * 💡 **学习笔记**：记住常见个位的周期（1、2、4）能快速解题！

2.  **关键点2**：如何计算 \( B^C \mod \text{周期} \)？
    * **分析**：当周期T=1时，\( B^C \mod 1 = 0 \)，但此时指数需取1（因任何数的1次方是自身）；当T>1时，利用快速幂算法计算 \( B^C \mod T \)（注意处理B≥T的情况）。
    * 💡 **学习笔记**：快速幂是处理大数模运算的利器，能将时间复杂度降到 \( O(\log C) \)。

3.  **关键点3**：如何处理特殊情况（如A=0）？
    * **分析**：若A的个位是0，则 \( A^{B^C} \) 的个位一定是0（只要指数≥1，而B、C≥1，故指数≥1）。
    * 💡 **学习笔记**：特殊值（0、1、5、6）的周期为1，可直接得出结果。

### ✨ 解题技巧总结

- **分情况讨论**：根据a的不同，选择对应的周期；
- **快速幂模运算**：用快速幂计算 \( B^C \mod T \)，避免大数计算。

## 4. C++核心代码实现赏析

<code_intro_overall>
以下是基于上述思路的通用C++实现，代码简洁且覆盖所有情况。
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了数学周期规律与快速幂算法，适用于所有输入范围。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    // 快速幂计算 (base^exponent) % mod
    long long quick_pow(long long base, long long exponent, long long mod) {
        long long result = 1 % mod; // 处理mod=1的情况
        base %= mod;
        while (exponent > 0) {
            if (exponent % 2 == 1) {
                result = (result * base) % mod;
            }
            base = (base * base) % mod;
            exponent /= 2;
        }
        return result;
    }

    int main() {
        long long A, B, C;
        cin >> A >> B >> C;
        int a = A % 10; // 取A的个位
        if (a == 0 || a == 1 || a == 5 || a == 6) {
            cout << a << endl; // 周期1，直接输出
            return 0;
        }

        int T; // 周期
        if (a == 4 || a == 9) T = 2; // 周期2
        else T = 4; // 周期4（2、3、7、8）

        long long exponent;
        if (T == 1) exponent = 1; // 不可能，因前面已处理
        else {
            exponent = quick_pow(B, C, T);
            if (exponent == 0) exponent = T; // 当B^C是T的倍数时，取T
        }

        int ans = 1;
        for (int i = 0; i < exponent; ++i) { // 计算a^exponent的个位
            ans = (ans * a) % 10;
        }
        cout << ans << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    代码首先取A的个位a，根据a判断周期T。若a是0、1、5、6，直接输出a；否则根据a确定周期T（2或4）。接着用快速幂计算 \( B^C \mod T \)，调整指数后计算a的该指数次方的个位，即为答案。

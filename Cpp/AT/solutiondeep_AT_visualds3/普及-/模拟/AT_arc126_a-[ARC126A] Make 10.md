# 题目信息

# [ARC126A] Make 10

## 题目描述

[problemUrl]: https://atcoder.jp/contests/arc126/tasks/arc126_a

長さが $ 2 $ の棒が $ N_2 $ 個、長さが $ 3 $ の棒が $ N_3 $ 個、長さが $ 4 $ の棒が $ N_4 $ 個あります。あなたは、これらの棒に対して次の操作を何度でも行うことができます：

- $ 2 $ つの棒を選ぶ。
- 選んだ棒の長さが $ x,\ y $ であるとき、これらを接着することで、長さ $ x+y $ の棒を作る。

長さがちょうど $ 10 $ に等しい棒を最大でいくつ作れるかを求めてください。

$ T $ 個のテストケースが与えられるので、それぞれについて答えを求めてください。

## 说明/提示

### 制約

- $ 1\leq\ T\leq\ 100 $
- $ 0\leq\ N_2,\ N_3,\ N_4\leq\ 10^{15} $

### Sample Explanation 1

ひとつめのテストケースについて説明します。 長さ $ 2 $ の棒が $ 3 $ 個、長さ $ 3 $ の棒が $ 4 $ 個、長さ $ 4 $ の棒が $ 1 $ 個あります。 例えば以下のようにして、長さがちょうど $ 10 $ の棒を $ 2 $ 個作ることができます。 - 長さが $ 2 $, $ 2 $, $ 3 $, $ 3 $ の棒を適当な順序で接着することで、長さが $ 10 $ の棒をひとつ作ることができます。 - 長さが $ 3 $, $ 3 $, $ 4 $ の棒を適当な順序で接着することで、長さが $ 10 $ の棒をひとつ作ることができます。 - これらの操作の後、長さが $ 2,\ 10,\ 10 $ の棒が手元に残ります。

## 样例 #1

### 输入

```
5

3 4 1

7 0 0

0 0 7

0 0 0

1000000000000000 1000000000000000 1000000000000000```

### 输出

```
2

1

0

0

900000000000000```

## 4. C++核心代码实现赏析

</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了枚举小范围组合次数的思路，适用于处理大范围数值（如1e15），确保高效计算。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <algorithm>
    using namespace std;
    using ll = long long;

    ll solve(ll N2, ll N3, ll N4) {
        ll max_ans = 0;
        // 枚举组合2（2个2+2个3）的使用次数k1（0到4次）
        for (ll k1 = 0; k1 <= 4; ++k1) {
            if (2*k1 > N2 || 2*k1 > N3) break; // 资源不足，提前退出
            ll rem2 = N2 - 2*k1;
            ll rem3 = N3 - 2*k1;
            // 枚举组合5（2个3+1个4）的使用次数k2（0到4次）
            for (ll k2 = 0; k2 <= 4; ++k2) {
                if (2*k2 > rem3 || k2 > N4) break;
                ll rem3_ = rem3 - 2*k2;
                ll rem4 = N4 - k2;
                // 枚举组合4（1个2+2个4）的使用次数k3（0到4次）
                for (ll k3 = 0; k3 <= 4; ++k3) {
                    if (k3 > rem2 || 2*k3 > rem4) break;
                    ll rem2_ = rem2 - k3;
                    ll rem4_ = rem4 - 2*k3;
                    // 剩余资源计算其他组合的最大贡献
                    ll ans = k1 + k2 + k3;
                    // 组合1（5个2）
                    ans += rem2_ / 5;
                    // 组合3（3个2+1个4）
                    ans += min(rem2_ / 3, rem4_);
                    max_ans = max(max_ans, ans);
                }
            }
        }
        return max_ans;
    }

    int main() {
        int T;
        cin >> T;
        while (T--) {
            ll N2, N3, N4;
            cin >> N2 >> N3 >> N4;
            cout << solve(N2, N3, N4) << endl;
        }
        return 0;
    }
    ```
* **代码解读概要**：
    代码通过三层循环枚举组合2（2个2+2个3）、组合5（2个3+1个4）、组合4（1个2+2个4）的使用次数（各0到4次）。对于每种枚举情况，计算剩余资源，并通过除法快速计算剩余资源能组成的其他组合（如5个2、3个2+1个4）的最大次数。最终取所有情况的最大值作为答案。

# 题目信息

# 値札

## 题目描述

[problemUrl]: https://atcoder.jp/contests/bitflyer2018-final/tasks/bitflyer2018_final_a

すぬけ君は、店を開こうとしています。$ N $ 個の商品を販売する予定です。 $ i $ 番目の商品は、$ p_i $ 円で販売されます。

すぬけ君は、それぞれの価格の末尾についた `0` をたくさん書くのが大変に感じたので、 $ N $ 個全ての商品の値札の末尾に連続する `0` を全ての商品について同じ個数だけ取り除き、 会計の時にその分の `0` を補完することにしました。

商品の値札 $ 1 $ 枚あたり、 `0` を最大何個取り除けるかを求めてください。

## 说明/提示

### 制約

- $ 1\ <\ =\ N\ <\ =\ 100 $
- $ 1\ <\ =\ p_i\ <\ =\ 10^9 $
- $ p_i $ は整数

### Sample Explanation 1

それぞれ、末尾についた `0` を $ 1 $ つだけ取り除くことができ、すると値札には $ 30 $, $ 25 $, $ 600 $, $ 73 $ と書かれることになります。

## 样例 #1

### 输入

```
4

300

250

6000

730```

### 输出

```
1```

## 样例 #2

### 输入

```
5

10000000

30000000

150000000

200000000

990000000```

### 输出

```
7```

## 样例 #3

### 输入

```
4

100101100

110010000

100001001

110011000```

### 输出

```
0```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决此题时，需要注意以下核心难点及应对策略：
</difficulty_intro>

1.  **关键点1：正确统计单个数字末尾的0个数**  
    * **分析**：末尾的0是指从最后一位开始连续的0，中间的0不算。例如，数字300的末尾有2个0（300→30→3，两次除以10），而数字3050的末尾只有1个0（3050→305，一次除以10）。无论是用整数除法还是字符串遍历，都需确保只统计末尾连续的0。  
    * 💡 **学习笔记**：统计末尾0时，必须从最后一位开始，遇到非0即停止。

2.  **关键点2：准确取所有数的最小值**  
    * **分析**：最终答案是所有数末尾0个数的最小值。初始化时需将`ans`设为足够大的值（如`INT_MAX`或`1e9`），确保所有数的统计结果都能覆盖到。例如，若第一个数有3个0，`ans`初始化为3，后续遇到更少的数时更新。  
    * 💡 **学习笔记**：最小值问题中，初始值要足够大，避免被错误覆盖。

3.  **关键点3：选择合适的数据处理方式（整数vs字符串）**  
    * **分析**：整数除法适合处理大数（如`p_i=1e9`），无需额外转换，效率更高；字符串遍历更直观，适合对数字分解不熟悉的学习者。本题数据范围小（`p_i≤1e9`），两种方法均可，但整数除法更高效。  
    * 💡 **学习笔记**：整数操作通常比字符串更高效，字符串操作更直观，根据场景选择。

### ✨ 解题技巧总结
- **技巧A：初始化最小值为极大值**：用`INT_MAX`或`1e9`初始化`ans`，确保所有可能的统计值都能被比较。  
- **技巧B：循环终止条件明确**：统计末尾0时，整数除法的终止条件是`a%10!=0`，字符串遍历的终止条件是遇到非'0'字符，避免无限循环。  
- **技巧C：变量命名清晰**：如`sum`表示当前数的0个数，`ans`表示最终答案，提高代码可读性。

## 4. C++核心代码实现赏析

<code_intro_overall>
我们选择整数处理的简洁实现作为通用核心代码，它高效且易于理解。
</code_intro_overall>

**本题通用核心C++实现参考**  
* **说明**：此代码综合了多个优质题解的思路，采用整数除法统计末尾0，代码简洁高效，适合竞赛场景。  
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int n, ans = 1e9; // 初始化为足够大的值
        cin >> n;
        for (int i = 0; i < n; ++i) {
            int p, cnt = 0;
            cin >> p;
            while (p % 10 == 0) { // 统计末尾0的个数
                cnt++;
                p /= 10;
            }
            if (cnt < ans) ans = cnt; // 更新最小值
        }
        cout << ans << endl;
        return 0;
    }
    ```
* **代码解读概要**：  
  代码首先读取商品数量`n`，初始化`ans`为极大值。然后循环读取每个价格`p`，用`cnt`统计其末尾0的个数（通过不断除以10，直到余数不为0）。每次更新`ans`为更小的`cnt`，最终输出`ans`。

* 💡 **学习笔记**：用`while`循环逐个去掉末尾0，是统计末尾0的经典方法。

**题解二：作者fkcufk（字符串处理）**

* **核心代码片段**：
    ```cpp
    string s;
    cin >> s;
    int t = s.size() - 1, x = 0;
    while(s[t] == '0') {
        x++, t--;
    }
    mn = min(x, mn);
    ```
* **代码解读**：  
  将数转为字符串`s`，从最后一位（`s.size()-1`）开始遍历。`t`是当前检查的位置，`x`统计0的个数。只要当前字符是'0'，`x++`并左移`t`，直到遇到非'0'字符。最后更新最小值`mn`。这种方法直观展示了“从右往左找0”的过程。  
* 💡 **学习笔记**：字符串遍历适合直观观察末尾0的位置，适合初学者理解。

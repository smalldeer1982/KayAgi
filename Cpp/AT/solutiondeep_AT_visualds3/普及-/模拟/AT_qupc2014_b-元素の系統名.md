## 3. 核心难点辨析与解题策略

<difficulty_intro>
在解决这道题时，以下三个关键点最容易出错，我们逐一分析：
</difficulty_intro>

1.  **关键点1：首字母大写处理**  
    * **分析**：百位是系统名的第一个字符，必须大写。例如，百位为1时应输出“Un”而非“un”。优质题解通常通过直接拼接大写词根（如“Un”）或后续转换首字母（如将首字符转大写）实现。  
    * 💡 **学习笔记**：首字母大写可通过直接写大写词根（如“Un”）或代码转换（如`a[0] = toupper(a[0])`）实现，前者更简单。  

2.  **关键点2：0的特判（nil vs il）**  
    * **分析**：十位或个位为0时，若前一位是9（如902中的十位0，前一位是百位9），应输出“il”而非“nil”。例如，902的十位0前是9，输出“il”，最终为“Ennilbium”（百位9→Enn，十位0→il，个位2→b，词尾um→bium？需仔细看题目规则）。  
    * 💡 **学习笔记**：0的输出取决于前一位是否为9，需用条件判断`if (前一位是9)`来切换“il”或“nil”。  

3.  **关键点3：词尾的ium vs um**  
    * **分析**：个位为2或3时（对应词根bi/tri），词尾用“um”避免i重复。例如，个位2→bi+um→bium（如902的个位2→b+um→bium）。  
    * 💡 **学习笔记**：词尾判断只需检查个位是否为2或3，用`if (个位 == 2 || 个位 == 3)`即可。  

### ✨ 解题技巧总结  
- **拆分处理**：将三位数拆分为百位、十位、个位，分别处理，避免混淆。  
- **打表优化**：用数组存储词根（如`string a[10] = {"Nil","Un",...}`），减少重复条件判断。  
- **特判优先**：先处理特殊情况（如0的il、词尾um），再处理常规情况，避免遗漏。

* **完整核心代码**：

#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;  // 输入三位数，如"145"
    
    // 处理百位（首字母大写）
    if (s[0] == '0') cout << "Nil";
    else if (s[0] == '1') cout << "Un";
    else if (s[0] == '2') cout << "Bi";
    else if (s[0] == '3') cout << "Tri";
    else if (s[0] == '4') cout << "Quad";

    else if (s[0] == '9') cout << "Enn";
    
    // 处理十位（特判：前一位是9时0输出il）
    if (s[1] == '0') {
        if (s[0] == '9') cout << "il";
        else cout << "nil";
    } else if (s[1] == '1') cout << "un";
    else if (s[1] == '2') cout << "bi";
    else if (s[1] == '3') cout << "tri";
    else if (s[1] == '4') cout << "quad";
    else if (s[1] == '5') cout << "pent";
    else if (s[1] == '6') cout << "hex";
    else if (s[1] == '7') cout << "sept";
    else if (s[1] == '8') cout << "oct";
    else if (s[1] == '9') cout << "enn";
    
    // 处理个位（特判：前一位是9时0输出il）
    if (s[2] == '0') {
        if (s[1] == '9') cout << "il";
        else cout << "nil";
    } else if (s[2] == '1') cout << "un";
    else if (s[2] == '2') cout << "bi";
    else if (s[2] == '3') cout << "tri";
    else if (s[2] == '4') cout << "quad";
    else if (s[2] == '5') cout << "pent";
    else if (s[2] == '6') cout << "hex";
    else if (s[2] == '7') cout << "sept";
    else if (s[2] == '8') cout << "oct";
    else if (s[2] == '9') cout << "enn";
    
    // 处理词尾（个位2/3时用um）
    if (s[2] == '2' || s[2] == '3') cout << "um";
    else cout << "ium";
    
    return 0;
}

  1. **百位**：直接输出大写词根（如1→Un）；  
  2. **十位**：若为0且百位是9，输出“il”，否则输出对应词根（如4→quad）；  
  3. **个位**：若为0且十位是9，输出“il”，否则输出对应词根（如5→pent）；  
  4. **词尾**：个位是2/3时输出“um”，否则输出“ium”。

# オークション

## 题目描述

[problemUrl]: https://atcoder.jp/contests/yahoo-procon2018-qual/tasks/yahoo_procon2018_qual_b

矢風くんはオークションに参加しています。矢風くんの欲しい品物には、現在 $ X $ 円の値が付いています。

矢風くんはこの品物に、以下の条件を満たすような $ Y $ に対して、$ Y $ 円の値段をつけることができます。

- オークションでつける値段として適切である。すなわち、$ X+1\leq\ Y $ である。
- つける金額は細かすぎない。すなわち、$ Y $ の $ 10 $ 進表記の下 $ K $ 桁は全て $ 0 $ である。

矢風くんがつけることのできる最小の金額を求めてください。

## 说明/提示

### 制約

- $ 1\ \leq\ X\ \leq\ 10^9 $
- $ 0\ \leq\ K\ \leq\ 9 $
- 入力はすべて整数である

### Sample Explanation 1

$ 2018 $ 円より大きく、下 $ 2 $ 桁が $ 0 $ であるような最小の金額は、$ 2100 $ 円です。

### Sample Explanation 2

$ X $ 円より大きい値段をつけなければならないことに注意してください。

## 样例 #1

### 输入

```
2018 2```

### 输出

```
2100```

## 样例 #2

### 输入

```
888 0```

### 输出

```
889```

## 样例 #3

### 输入

```
100 7```

### 输出

```
10000000```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
在解决这个问题的过程中，我们通常会遇到以下核心难点。结合优质题解的共性，我为大家提炼了思考方向和策略：
</difficulty_intro>

1.  **关键点1**：如何正确计算10^K的值？
    * **分析**：直接使用pow(10, K)可能因浮点精度丢失导致错误（如K=9时，pow(10,9)在double中可能表示为999999999.999…）。优质题解（如mwhxiaopy）通过循环计算k1=10^K（k1初始为1，循环K次乘以10），确保整数精度。
    * 💡 **学习笔记**：涉及大整数幂次时，优先使用循环计算整数结果，避免浮点运算。

2.  **关键点2**：如何找到大于X的最小10^K倍数？
    * **分析**：Y必须是10^K的倍数且Y>X。数学上，Y等于“X除以10^K的商向上取整后乘以10^K”。例如，X=2018，K=2时，2018/100=20.18，向上取整为21，21×100=2100。
    * 💡 **学习笔记**：模运算可辅助快速计算：Y = (X - (X % k1)) + k1（其中k1=10^K）。

3.  **关键点3**：如何处理K=0的特殊情况？
    * **分析**：K=0时，Y的末尾0位全为0（即无限制），只需Y>X。此时Y=X+1（如样例2：X=888，K=0，输出889）。
    * 💡 **学习笔记**：K=0是边界条件，需单独判断或通过通用公式自动覆盖（如k1=1时，(X - (X%1)) +1 = X+1）。

### ✨ 解题技巧总结
<summary_best_practices>
- **整数优先**：涉及幂次计算时，用循环代替pow函数，避免浮点精度问题。
- **模运算妙用**：通过取余操作快速定位“去尾”部分，简化倍数计算。
- **边界检查**：关注K=0、X=10^K-1等特殊情况，确保代码鲁棒性。
</summary_best_practices>

## 4. C++核心代码实现赏析

<code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：此代码综合了mwhxiaopy题解的思路，采用整数运算避免浮点精度问题，逻辑清晰且鲁棒性强。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int X, K;
        cin >> X >> K;
        long long k1 = 1; // 计算10^K的整数值
        for (int i = 0; i < K; ++i) {
            k1 *= 10;
        }
        // 计算Y = (X - (X % k1)) + k1
        long long Y = (X / k1) * k1 + k1;
        cout << Y << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码首先读取输入X和K，然后通过循环计算k1=10^K（如K=2时，k1=100）。接着，通过(X / k1)得到X除以k1的商（如2018/100=20），乘以k1得到当前倍数（2000），再加k1得到下一个倍数（2100）。此逻辑覆盖了所有情况，包括K=0（k1=1时，Y=X+1）。

        k1=1;
        for(int i=1;i<=k;i++)k1=10*k1; // 计算10^K

        y=x-y; // 去掉末尾K位（得到前部分）

        cout<<y;
    }
    ```
* **代码解读**：
    > 这段代码的核心是通过模运算分解X的结构。`k1=10*k1`循环计算10的K次方（如K=2时，k1=100）。`x%k1`得到X的末尾K位（如2018%100=18），`x-y`去掉末尾K位（2018-18=2000），再加k1得到下一个倍数（2000+100=2100）。此方法直观地展示了“去尾补整”的过程，逻辑清晰。
* 💡 **学习笔记**：模运算能快速分离出“需要调整的部分”，是处理数位问题的常用技巧。

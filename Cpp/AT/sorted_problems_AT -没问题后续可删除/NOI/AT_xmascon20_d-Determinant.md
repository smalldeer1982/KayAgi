# Determinant

## 题目描述

给出正整数 $N$ 和整数 $C$。$N\times N$ 矩阵 $A$ 如下确定：（$i$，$j$）分量（$1\le i\le N$，$1\le j\le N$）为 
$i=j$ 时 $1$ ，
$j$ 不能用 $i$ 除尽时 $C$，
非上述任一种情况下为 $0$。
此时，求 $\text{det A}$ 除以 $998244353$ 的余数（$0$ 以上 $998244353$以下）。

## 输入格式

输入以以下形式从标准输入被给予。

$N\hspace{0.4cm} C\\$

## 输出格式

输出 $\text{det A}$除以 $998244353$ 余数（$0$ 以上 $998244353$ 以下）。

## 样例 #1

### 输入

```
6 3
```

### 输出

```
998244345
```

## 样例 #2

### 输入

```
2020 11
```

### 输出

```
515894850
```

## 样例 #3

### 输入

```
1000000000 2020
```

### 输出

```
4909496
```

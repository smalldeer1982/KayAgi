# [ABC016A] 12月6日

## 输入格式

第一行两个整数 $M, D$。

## 输出格式

输出一行。

如果 $M$ 能被 $D$ 整除，输出 `YES`，否则输出 `NO`。

**记得行末换行！**

## 样例 #1

### 输入

```
1 1
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
2 29
```

### 输出

```
NO
```

## 样例 #3

### 输入

```
12 6
```

### 输出

```
YES
```

# [ABC062B] Picture Frame

## 题目描述

给出一个长H，宽W的图像，用小写字母表示。第i行第j个字母是a(i,j)。

输出该图像最外层用一个'#'包围后的图像

## 输入格式

输入将会按照以下格式：
```
H W
a(1,1) ... a(1,W)
...
a(H,1) ... a(H,W)
```

## 输出格式

输出该图像最外层用一个'#'包围后的图像

## 说明/提示

1 $\le$ H,W $\le$ 100

a(i,j)是一个小写字母

## 样例 #1

### 输入

```
2 3
abc
arc
```

### 输出

```
#####
#abc#
#arc#
#####
```

## 样例 #2

### 输入

```
1 1
z
```

### 输出

```
###
#z#
###
```

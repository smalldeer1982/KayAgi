# [ABC107A] Train

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc107/tasks/abc107_a

$ N $ 両編成の列車があります。 この列車の前から $ i $ 両目は、後ろから何両目でしょうか？

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ i $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 100 $
- $ 1\ \leq\ i\ \leq\ N $

### Sample Explanation 1

$ 4 $ 両編成の列車において、前から $ 2 $ 両目の車両は、後ろから $ 3 $ 両目です。

## 样例 #1

### 输入

```
4 2
```

### 输出

```
3
```

## 样例 #2

### 输入

```
1 1
```

### 输出

```
1
```

## 样例 #3

### 输入

```
15 11
```

### 输出

```
5
```

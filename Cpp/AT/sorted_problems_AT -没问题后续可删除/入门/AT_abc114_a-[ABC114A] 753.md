# [ABC114A] 753

## 题目描述

“七五三”是一个庆祝七岁、五岁、三岁儿童成长的节日
Takahashi 今年$X$岁了,他的成长会在这个节日被庆祝吗？

## 输入格式

一个字母$X$,表示Takahashi 的年龄。

## 输出格式

如果Takahashi的成长会被庆祝，输出YES;
如果不会，输出NO;

## 样例 #1

### 输入

```
5
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
6
```

### 输出

```
NO
```

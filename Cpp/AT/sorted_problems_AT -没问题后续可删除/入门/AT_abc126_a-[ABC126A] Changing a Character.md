# [ABC126A] Changing a Character

## 输入格式

输入共两行。第一行两个整数 $n,k$ 。

第二行一个由 `A`,`B`,`C` 组成字符串  $s$ 。

$n$ 表示字符串的长度。

把字符串 $s$ 的第 $k-1$ 个字符变成小写。（字符串 $s$ 的第一个字符的下标为 $0$ ）

## 输出格式

一行一个字符串 $s$ ，为转换后的字符串。

## 样例 #1

### 输入

```
3 1
ABC
```

### 输出

```
aBC
```

## 样例 #2

### 输入

```
4 3
CABA
```

### 输出

```
CAbA
```

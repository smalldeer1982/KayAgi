# [ABC137A] +-x

## 题目描述

## 题意：

给出两个整数 $A$ 和 $B$。求出 $A + B$，$A\times B$ 和 $ A - B$ 中的最大值。


------------

## 输入格式

一行输入两个整数：$A$ 和 $B$。


------------

## 输出格式

输出 $A + B$，$A\times B$ 和 $ A - B$ 中的最大值。


------------


## 样例输入输出：

无


------------
## 数据规模与约定：

满足所有输入为整数。

$-100 \le A,B \le 100$。

## 样例 #1

### 输入

```
-13 3
```

### 输出

```
-10
```

## 样例 #2

### 输入

```
1 -33
```

### 输出

```
34
```

## 样例 #3

### 输入

```
13 3
```

### 输出

```
39
```

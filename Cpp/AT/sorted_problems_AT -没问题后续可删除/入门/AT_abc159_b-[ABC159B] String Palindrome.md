# [ABC159B] String Palindrome

## 题目描述

一个长度为奇数的字符串 $S$ 被称为强回文串，当且仅当以下几点全部满足。

- $S$ 是一个回文串。
- 设 $N$ 是 $S$ 的长度。$S$的前$(\frac{(N-1)}{2})$个字符所组成的字符串是回文的。
- $S$的后$(\frac{(N-1)}{2})$个字符所组成的字符串是回文的。

判断 $S$ 是否是强回文串。

## 输入格式

标准输入。

一行一个字符串 $S$。

## 输出格式

如果 $S$ 是强回文串，输出 $Yes$；否则输出 $No$。

## 说明/提示

- $S$只包含小写字母。

- $S$的长度是一个属于$[3,99]$的奇数。

## 样例 #1

### 输入

```
akasaka
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
level
```

### 输出

```
No
```

## 样例 #3

### 输入

```
atcoder
```

### 输出

```
No
```

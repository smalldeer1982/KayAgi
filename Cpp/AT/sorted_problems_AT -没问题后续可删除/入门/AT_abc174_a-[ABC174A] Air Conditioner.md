# [ABC174A] Air Conditioner

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc174/tasks/abc174_a

あなたは、室温が $ 30 $ 度以上のとき、またそのときに限り、冷房の電源を入れます。

今の室温は $ X $ 度です。冷房の電源を入れますか？

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ X $

## 输出格式

冷房の電源を入れるならば `Yes`、入れないならば `No` を出力せよ。

## 说明/提示

### 制約

- $ -40\ \leq\ X\ \leq\ 40 $
- $ X $ は整数である。

## 样例 #1

### 输入

```
25
```

### 输出

```
No
```

## 样例 #2

### 输入

```
30
```

### 输出

```
Yes
```

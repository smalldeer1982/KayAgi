# [ABC181B] Trapezoid Sum

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc181/tasks/abc181_b

何も書かれていない黒板があります。 高橋くんは $ N $ 回の操作を行い、黒板に整数を書きます。

$ i $ 回目の操作では、 $ A_i $ 以上 $ B_i $ 以下の整数すべてを $ 1 $ 個ずつ、合計 $ B_i\ -\ A_i\ +\ 1 $ 個の整数を書きます。

$ N $ 回の操作を終えたときの、黒板に書かれた整数の合計を求めてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ A_1 $ $ B_1 $ $ \vdots $ $ A_N $ $ B_N $

## 输出格式

$ N $ 回の操作を終えたときの、黒板に書かれた整数の合計を出力せよ。

## 说明/提示

### 制約

- 入力はすべて整数
- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ 1\ \leq\ A_i\ \leq\ B_i\ \leq\ 10^6 $

### Sample Explanation 1

$ 1 $ 回目の操作では、黒板に $ 1,\ 2,\ 3 $ を書きます。 $ 2 $ 回目の操作では、黒板に $ 3,\ 4,\ 5 $ を書きます。 黒板に書かれた整数の合計は $ 1+2+3+3+4+5=18 $ です。

## 样例 #1

### 输入

```
2
1 3
3 5
```

### 输出

```
18
```

## 样例 #2

### 输入

```
3
11 13
17 47
359 44683
```

### 输出

```
998244353
```

## 样例 #3

### 输入

```
1
1 1000000
```

### 输出

```
500000500000
```

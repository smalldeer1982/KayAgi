# [ABC244D] Swap Hats

## 题目描述

有三顶帽子分别为红色 ```R```，绿色 ```G```，蓝色 ```B```。一开始，这三顶帽子顺序为 $S_1,S_2,S_3$，你可以进行偶数次操作。

#### 操作：

- 每一次操作你可以从三顶帽子中选择两顶交换位置。

询问能否经过偶数次操作使帽子颜色顺序为 $T_1,T_2,T_3$。

## 输入格式

第一行三个字符 $S_1,S_2,S_3$。

第二行三个字符
$T_1,T_2,T_3$。

## 输出格式

如果可以用偶数次操作完成交换输出 ```Yes```，不然输出 ```No```。

## 样例 #1

### 输入

```
R G B
R G B
```

### 输出

```
Yes
```

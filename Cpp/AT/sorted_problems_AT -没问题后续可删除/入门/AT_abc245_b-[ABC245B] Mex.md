# [ABC245B] Mex

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc245/tasks/abc245_b

長さ $ N $ の整数からなる数列 $ A=(A_1,\ldots,A_N) $ が与えられます。

$ A_1,\ldots,A_N $ に含まれない最小の非負整数を求めてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ A_1 $ $ \ldots $ $ A_N $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 2000 $
- $ 0\ \leq\ A_i\ \leq\ 2000 $
- 入力は全て整数である

### Sample Explanation 1

非負整数は $ 0,1,2,3,4,\ldots $ と続きます。 $ 0,1,2,3 $ は $ A $ に含まれ、$ 4 $ は $ A $ に含まれないので、答えは $ 4 $ です。

## 样例 #1

### 输入

```
8
0 3 2 6 2 1 0 0
```

### 输出

```
4
```

## 样例 #2

### 输入

```
3
2000 2000 2000
```

### 输出

```
0
```

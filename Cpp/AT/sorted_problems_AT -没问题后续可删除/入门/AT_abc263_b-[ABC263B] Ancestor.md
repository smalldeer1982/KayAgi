# [ABC263B] Ancestor

## 题目描述

有 $n$ 个人，第 $i$ 个人的上一代是 $p_i(p_i<i)$。

问第 $1$ 个人与第 $n$ 个人之间隔了几代。

## 输入格式

第一行，一个整数 $n$。

第二行，$n-1$ 个整数，表示 $p_2,p_3,\cdots,p_n$。

## 输出格式

输出答案。

## 样例 #1

### 输入

```
3
1 2
```

### 输出

```
2
```

## 样例 #2

### 输入

```
10
1 2 3 4 5 6 7 8 9
```

### 输出

```
9
```

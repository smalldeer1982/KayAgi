# [ABC266A] Middle  Letter

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc266/tasks/abc266_a

英小文字からなる長さが奇数の文字列 $ S $ が与えられます。

$ S $ の中央の文字を出力してください。

中央の文字とはある長さが奇数の文字列 $ T $ について、 $ T $ の長さを $ |T| $ として、$ T $ の前から $ \frac{|T|+1}{2} $ 番目の文字を中央の文字とします。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ S $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ S $ は英小文字からなる長さが奇数の文字列
- $ S $ の長さは $ 1 $ 以上 $ 99 $ 以下

### Sample Explanation 1

`atcoder` の中央の文字は `o` です。

## 样例 #1

### 输入

```
atcoder
```

### 输出

```
o
```

## 样例 #2

### 输入

```
a
```

### 输出

```
a
```

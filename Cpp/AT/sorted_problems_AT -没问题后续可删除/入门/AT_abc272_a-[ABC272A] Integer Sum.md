# [ABC272A] Integer Sum

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc272/tasks/abc272_a

$ N $ 個の整数 $ A_1,A_2,\dots,A_N $ が与えられます。

$ N $ 個の整数を合計した値を求めてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \dots $ $ A_N $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ 1\ \le\ N\ \le\ 100 $
- $ 1\ \le\ A_i\ \le\ 100 $
- 入力はすべて整数

### Sample Explanation 1

$ 3 $ 個の整数 $ 2,7,2 $ が与えられます。 答えは $ 2\ +\ 7\ +\ 2\ =\ 11 $ です。

## 样例 #1

### 输入

```
3
2 7 2
```

### 输出

```
11
```

## 样例 #2

### 输入

```
1
3
```

### 输出

```
3
```

# [ABC302A] Attack

## 题目描述

一个人有 $A$ 点体力值，你可以攻击他，每次攻击使其体力值减 $B$，问要让他体力值降至 $0$ 或以下至少要攻击多少次。

## 输入格式

共一行两个整数 $A,B$。($1\leqslant A,B\leqslant 10^{18}$)

## 输出格式

共一行一个整数，表示最少攻击次数。

## 样例 #1

### 输入

```
7 3
```

### 输出

```
3
```

## 样例 #2

### 输入

```
123456789123456789 987654321
```

### 输出

```
124999999
```

## 样例 #3

### 输入

```
999999999999999998 2
```

### 输出

```
499999999999999999
```

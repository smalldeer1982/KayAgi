# [ABC324A] Same

## 题目描述

给定 $ N $ 个整数 $ A_1, A_2, \ldots, A_N $。

如果这些值都是相等的，则输出 `Yes`，否则输出 `No`。

## 输入格式

输入以以下形式从标准输入读取：

> $ N $ $ A_1 $ $ A_2 $ …… $ A_N $

## 输出格式

在一行中输出 `Yes` 或 `No`。

### 约束条件

- $ 2\leq N\leq100 $
- $ 1\leq A_i\leq100\ (1\leq i\leq N) $
- 输入都是整数

## 样例 #1

### 输入

```
3
3 2 4
```

### 输出

```
No
```

## 样例 #2

### 输入

```
4
3 3 3 3
```

### 输出

```
Yes
```

## 样例 #3

### 输入

```
10
73 8 55 26 97 48 37 47 35 55
```

### 输出

```
No
```

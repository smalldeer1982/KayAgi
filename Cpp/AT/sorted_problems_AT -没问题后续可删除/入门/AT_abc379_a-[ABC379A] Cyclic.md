# [ABC379A] Cyclic

## 题目描述

给你一个由 $1$ 至 $9$ 组成的三位数 $N$。

令 $a$、$b$、$c$ 分别为 $N$ 的百位、十位、个位（即 $N=\overline{abc}$）。请按顺序输出整数 $\overline{bca}$ 和 $\overline{cab}$。

## 输入格式

输入一个正整数 $N$。

## 输出格式

分别输出正整数 $\overline{bca}$ 和 $\overline{cab}$，中间用空格隔开。

## 说明/提示

对于 $100\%$ 的数据，$100\le N\le999$。

## 样例 #1

### 输入

```
379
```

### 输出

```
793 937
```

## 样例 #2

### 输入

```
919
```

### 输出

```
199 991
```

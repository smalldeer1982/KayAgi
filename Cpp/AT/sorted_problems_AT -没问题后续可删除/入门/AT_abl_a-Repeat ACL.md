# Repeat ACL

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abl/tasks/abl_a

整数 $ K $ が与えられます。 文字列 `ACL` を $ K $ 回繰り返してつなげることで得られる文字列を出力してください。

たとえば、$ K\ =\ 3 $ ならば、 `ACLACLACL` を出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ K $

## 输出格式

文字列 `ACL` を $ K $ 回繰り返してつなげることで得られる文字列を出力せよ。

## 说明/提示

### 制約

- $ 1\ \leq\ K\ \leq\ 5 $
- 入力は全て整数である。

## 样例 #1

### 输入

```
3
```

### 输出

```
ACLACLACL
```

# 和

## 题目描述

题目大意：
输入n个数，输出n个数的和。

## 输入格式

第一行：n
第二行：a[1]~a[n]

## 输出格式

n个数的和

## 样例 #1

### 输入

```
5
3 1 5 4 2
```

### 输出

```
15
```

## 样例 #2

### 输入

```
6
1 2 3 4 5 6
```

### 输出

```
21
```

## 样例 #3

### 输入

```
7
7 6 5 4 3 2 1
```

### 输出

```
28
```

## 样例 #4

### 输入

```
20
19 11 10 7 8 9 17 18 20 4 3 15 16 1 5 14 6 2 13 12
```

### 输出

```
210
```

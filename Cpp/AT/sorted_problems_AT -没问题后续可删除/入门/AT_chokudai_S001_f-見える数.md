# 見える数

## 题目描述

[problemUrl]: https://atcoder.jp/contests/chokudai_S001/tasks/chokudai_S001_f

数列 $ a $ に含まれる整数のうち、$ 1\ \leq\ j\ <\ i\ \leq\ N $ を満たす任意の $ j $ において、 $ a_j\ <\ a_i $ を満たすような $ i $ がいくつあるか出力しなさい。

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 100,000 $

## 样例 #1

### 输入

```
5
3 1 5 4 2
```

### 输出

```
2
```

## 样例 #2

### 输入

```
6
1 2 3 4 5 6
```

### 输出

```
6
```

## 样例 #3

### 输入

```
7
7 6 5 4 3 2 1
```

### 输出

```
1
```

## 样例 #4

### 输入

```
20
19 11 10 7 8 9 17 18 20 4 3 15 16 1 5 14 6 2 13 12
```

### 输出

```
2
```

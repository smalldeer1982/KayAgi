# amylasemania IIDX

## 题目描述

给定两个数 $n$ 和 $m$。在这个游戏中， $n$ 为你尝试击打的次数，$m$ 为你的最高连击记录。在游戏中，如果你的一次击打失败了，连击次数就会清零。请你输出最小的可能的失败次数。

## 输入格式

一行，包括两个数 $n$ 和 $m$。

## 输出格式

一行，输出最小的失败次数。

## 说明/提示

$1\le n\le 1000000000$,
$1\le m\le n$。

## 样例 #1

### 输入

```
10 5
```

### 输出

```
1
```

## 样例 #2

### 输入

```
100 9
```

### 输出

```
10
```

# DDCC型文字列

## 题目描述

输入一个字符串 $S$ ，其长度为 $4$ 个字符

如果 $S$ 满足以下条件：

- $S$ 的第一个字符等于 $S$ 的第二个字符
- $S$ 的第三个字符等于 $S$ 的第四个字符
- $S$ 的第一、二个字符与第三、四个字符不相等

我们就认为字符串 $S$ 是DDCC型的

## 输入格式

输入一个字符串 $S$

## 输出格式

如果字符串$S$是$ DDCC $型则输出 `Yes`
,否则输出 `No`
。

## 样例 #1

### 输入

```
DDCC
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
AAAA
```

### 输出

```
No
```

## 样例 #3

### 输入

```
ABCD
```

### 输出

```
No
```

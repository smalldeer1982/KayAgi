# 約数

## 题目描述

[problemUrl]: https://atcoder.jp/contests/past202109-open/tasks/past202109_d

正の整数 $ X,\ Y $ が与えられます。 $ X $ の正の約数の個数と $ Y $ の正の約数の個数を比較し、

- $ X $ の正の約数の個数の方が多いなら `X` と出力し、
- $ Y $ の正の約数の個数の方が多いなら `Y` と出力し、
- 両者が等しいなら `Z` と出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ X $ $ Y $

## 输出格式

答えを出力せよ。

## 说明/提示

### 注意

この問題に対する言及は、2021/10/02 18:00 JST まで禁止されています。言及がなされた場合、賠償が請求される可能性があります。 試験後に総合得点や認定級を公表するのは構いませんが、どの問題が解けたかなどの情報は発信しないようにお願いします。

### 制約

- $ 1\ \leq\ X,\ Y\ \leq\ 10^6 $
- $ X,\ Y $ は整数

### Sample Explanation 1

$ 4 $ の正の約数は $ 1,\ 2,\ 4 $ の $ 3 $ つであり、$ 6 $ の正の約数は $ 1,\ 2,\ 3,\ 6 $ の $ 4 $ つです。 $ 6 $ の正の約数の方が $ 4 $ の正の約数より多いので、`Y` を出力します。

## 样例 #1

### 输入

```
4 6
```

### 输出

```
Y
```

## 样例 #2

### 输入

```
6 4
```

### 输出

```
X
```

## 样例 #3

### 输入

```
3 5
```

### 输出

```
Z
```

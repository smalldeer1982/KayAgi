# Maze

## 题目描述

![](https://cdn.luogu.com.cn/upload/vjudge_pic/AT3743/787af4ce1555338bec23f1fb2e259b2cc200af89.png)

请在上图中，找出所有能从起点（左上）走到终点（右下）的迷宫。

## 输入格式

这个问题没有输入

## 输出格式

输出所有能从起点到终点的迷宫旁边的字母（按字典序输出）。

例如，如果仅 $A$、$T$、$C$、$O$、$D$、$E$、$R$ 能从起点走到终点，则如下输出。

```
ACDEORT
```

## 输入输出样例

无

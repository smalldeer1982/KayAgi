# [ABC204C] Tour

## 题目描述

AtCoder国家包括编号 ${1}$ 到 ${N}$ 的 ${N}$ 个城市和编号为 ${M}$ 的 ${M}$ 条道路。

通过道路 ${i}$ 可以从城市 ${A_i}$ 移动到 ${B_i}$ 。从都市 ${B_i}$ 到都市 ${A_i}$ 不能通行。彪马打算从某个城市开始，使用 ${0}$ 条以上的道路移动，制定以某个城市为终点的旅行计划。

作为起点和终点的城市组合，有几种？

## 输入格式

输入的以下形式由标准输入给出。

$ 
{N M }
$

$
{A_1 B_1⋮ A_M B_M} 
$

## 输出格式

输出一行，包含一个正整数，表示彪马旅行问题的可能性的种数。

## 样例 #1

### 输入

```
3 3
1 2
2 3
3 2
```

### 输出

```
7
```

## 样例 #2

### 输入

```
3 0
```

### 输出

```
3
```

## 样例 #3

### 输入

```
4 4
1 2
2 3
3 4
4 1
```

### 输出

```
16
```

# [ARC125A] Dial Up

## 题目描述

有四个完全由 $0$ 和 $1$ 构成的整数序列 $s,t,a,b$，初始时 $s=a$ 且 $b$ 为空。

每次操作时，你可以选择以下操作之一：

- 将 $a$ 的最后一个数放到开头；
- 将 $a$ 的第一个数放到末尾；
- 在 $b$ 的末尾插入 $a_1$。

请输出能够让 $b=t$ 所需的最少操作次数。若无法达成目的，请输出 $-1$。

## 输入格式

第一行：$s$ 和 $t$ 的长度（$\le 2 \times 10^5$）

第二行：$s$

第三行：$t$

## 输出格式

若目标可实现输出最小操作次数，否则输出 $-1$。

## 样例 #1

### 输入

```
3 4
0 0 1
0 1 1 0
```

### 输出

```
6
```

## 样例 #2

### 输入

```
1 1
0
1
```

### 输出

```
-1
```

# [ABC054C] One-stroke Path

## 题目描述

给定一个没有重边和自环的 $N$ 个点 $M$ 条边的无权无向图，第 $i$ 条边连接顶点 $a _ i$ 和 $b _ i$。

求以顶点 $1$ 为起点，只访问 $1$ 次所有顶点的路径有多少条？特别地，起点和终点也视为被访问。

## 输入格式

第一行两个整数 $N, M$。

接下来 $m$ 行，其中第 $i$ 行两个整数 $a _ i, b _ i$。

$
N M \\
a _ 1 b _ 1 \\
a _ 2 b _ 2 \\
\kern {0.667 em} \vdots \\
a _ M b _ M
$

## 输出格式

输出满足条件的路径有多少。

## 说明/提示

$
2 \le N \le 8 \\
0 \le M \le N(N - 1) \\
1 \le a _ i < b _ i \le N
$

给定的无向图中不包含重边和自环。

## 样例 #1

### 输入

```
3 3
1 2
1 3
2 3
```

### 输出

```
2
```

## 样例 #2

### 输入

```
7 7
1 3
2 7
3 4
4 5
4 6
5 6
6 7
```

### 输出

```
1
```

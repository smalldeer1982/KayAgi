# [ABC230E] Fraction Floor Sum

## 题目描述

求 $\large{\sum\limits_{i=1}^N ⌊\frac{N}{i}⌋}$ 的值

## 输入格式

一行一个整数 $N$

## 输出格式

一行一个整数，你的答案

## 样例 #1

### 输入

```
3
```

### 输出

```
5
```

## 样例 #2

### 输入

```
10000000000
```

### 输出

```
231802823220
```

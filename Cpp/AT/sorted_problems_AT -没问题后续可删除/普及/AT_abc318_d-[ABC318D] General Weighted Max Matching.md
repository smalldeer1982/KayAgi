# [ABC318D] General Weighted Max Matching

## 题目描述

**题意简述**

有一个无向图，$i$ 到 $j$ 的距离为 $D_{i,j}$。你可以选择一些边，使得这些边连接的所有顶点互不相同。求这些边总长度的最大值。

## 输入格式

以以下格式输入：

> $N$\
> $D_{1,2},D_{1,3},\ldots,D_{1,N}$\
> $D_{2,3},\ldots,D_{2,N}$\
> $\vdots$\
> $D_{N-1,N}$

## 输出格式

$1$ 个整数。如题意。

## 说明/提示

- $2 \le N \le 16$
- $1 \le D_{i,j} \le 10^9$

**样例一解释**

选择 $D_{1,3},D_{2,4}$，总和为$5+8=13$。

## 样例 #1

### 输入

```
4
1 5 4
7 8
6
```

### 输出

```
13
```

## 样例 #2

### 输入

```
3
1 2
3
```

### 输出

```
3
```

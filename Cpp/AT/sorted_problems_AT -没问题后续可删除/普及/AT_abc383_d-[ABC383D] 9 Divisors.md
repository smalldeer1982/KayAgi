# [ABC383D] 9 Divisors

## 题目描述

找出不大于 $N$ 且恰好有 $9$ 个因数的正整数的个数。

## 输入格式

只有一行，一个整数 $N$。

## 输出格式

一行一个整数，表示答案。

## 说明/提示

- $1 \leq N \leq 4 \times 10^{12}$

- 所有输入值均为整数。

样例一解释：

三个正整数 $36,100,196$ 满足条件。

## 样例 #1

### 输入

```
200
```

### 输出

```
3
```

## 样例 #2

### 输入

```
4000000000000
```

### 输出

```
407073
```

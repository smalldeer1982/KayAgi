# [ABC393B] A..B..C

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc393/tasks/abc393_b

文字列 $ S $ が与えられます。

$ S $ の中に `A`, `B`, `C` がこの順に等間隔に並んでいる場所が何箇所あるか求めてください。

厳密には、$ 3 $ つの整数の組 $ (i,j,k) $ であって、以下の条件をすべて満たすものの個数を求めてください。 ここで、$ |S| $ で $ S $ の長さを、$ S_x $ で $ S $ の $ x $ 文字目を表すものとします。

- $ 1\leq\ i\ <\ j\ <\ k\leq\ |S| $
- $ j-i\ =\ k-j $
- $ S_i= $ `A`
- $ S_j= $ `B`
- $ S_k= $ `C`

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ S $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ S $ は英大文字からなる長さ $ 3 $ 以上 $ 100 $ 以下の文字列
 
### Sample Explanation 1

$ (i,j,k)=(1,3,5),(2,3,4) $ の $ 2 $ つの組が条件を満たします。

## 样例 #1

### 输入

```
AABCC
```

### 输出

```
2
```

## 样例 #2

### 输入

```
ARC
```

### 输出

```
0
```

## 样例 #3

### 输入

```
AABAAABBAEDCCCD
```

### 输出

```
4
```

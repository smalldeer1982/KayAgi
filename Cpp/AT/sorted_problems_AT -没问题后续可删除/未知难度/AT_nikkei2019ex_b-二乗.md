# 二乗

## 题目描述

[problemUrl]: https://atcoder.jp/contests/nikkei2019-ex/tasks/nikkei2019ex_b

正整数 $ N $ が与えられます。$ A^2 $ が $ N $ を超えないような最大の整数 $ A $ を出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $

## 输出格式

$ A^2 $ が $ N $ を超えないような最大の整数 $ A $ を出力せよ。

## 说明/提示

### 制約

- $ 1\ ≦\ N\ ≦\ 10^9 $

## 样例 #1

### 输入

```
10
```

### 输出

```
3
```

## 样例 #2

### 输入

```
100
```

### 输出

```
10
```

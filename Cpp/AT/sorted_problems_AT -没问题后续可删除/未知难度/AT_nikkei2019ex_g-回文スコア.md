# 回文スコア

## 题目描述

给出一个字符串 $S$。使用 $S$ 中的每个字符，组成若干个回文串，字符不必按照在 $S$ 中的顺序使用。例如 $S\ = $ `aaab` 时，可以组成两个回文串 `aba` 和 `a`。

每组成一个长度为 $L$ 的回文串，就会得到 $L^2$ 的分数。问能得到的最大得分之和是多少？

## 输入格式

一行一个字符串 $S$。

## 输出格式

输出能得到的最大得分之和。

## 说明/提示

- $1 \le |S| \le 10^5$
- $S$ 只含小写字母。

#### 样例解释1

字符串 `aaba` 可以组成 `aba` 和 `a` 两个字符串，此时的得分之和为 $3^2 + 1^2 =10$。能够证明这是可以得到的最大得分之和。

## 样例 #1

### 输入

```
aaba
```

### 输出

```
10
```

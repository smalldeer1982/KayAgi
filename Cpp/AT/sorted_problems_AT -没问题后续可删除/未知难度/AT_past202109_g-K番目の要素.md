# K番目の要素

## 题目描述

有一个由 $n$ 个等差数列依次连接而成的（超长）数列，已知第 $i$ 个数列的首项为 $b_i$ ，末项为 $c_i$ ，长度为 $a_i$ 。请回答：将所有等差数列依次连接后组成的数列中，第 $k$ 项的值是多少？

## 输入格式

输入 $(n+1)$ 行。第一行输入两个正整数 $n,k$ ，中间以单个空格隔开；接下来的 $n$ 行，每行三个正整数 $a_i,b_i,c_i$ ，相邻两个正整数之间以单个空格隔开。

## 输出格式

输出一行一个正整数，即连接而成的数列的第 $k$ 项的值。

## 说明/提示

**数据规模与约定**

对于全部测试点，数据保证：

- $2≤n≤10^5$ ；
- $1≤a_i,b_i,c_i≤10^9$ ；
- $1≤k≤a_1+a_2+...+a_n$ ；
- 所有输入的数值均为整数。

## 样例 #1

### 输入

```
2 4
3 2 2
2 3 4
```

### 输出

```
6
```

## 样例 #2

### 输入

```
2 10
4 1000000000 1000000000
6 1000000000 1000000000
```

### 输出

```
6000000000
```

## 样例 #3

### 输入

```
5 10
3 1 4
1 5 9
2 6 5
3 5 8
9 7 9
```

### 输出

```
16
```

# multiple

## 输入格式

输入通过标准输入给出，格式如下：

## 数据范围

- 输入中的所有数均为整数。
- $1 \leq N \leq 10^5$
- $1 \leq D \leq 10^9$
- $1 \leq A_i \leq 10^9$

# [ABC038B] ディスプレイ

## 题目描述

# 题目大意
高桥君买了两个显示器，第一个**高H1mm**,**宽W1mm**，第二个**高H2mm**,**宽W2mm**,现在将两个显示器**横放**或**竖放**（即每次旋转90度）现在问是否能使两个显示器**一样高**？

## 输入格式

两行，第一行为H1，W1，第二行为H2，W2，中间用空格分开

## 输出格式

可以则输出**YES**，否则输出**NO**，记得**换行**！！

## 样例 #1

### 输入

```
1080 1920
1080 1920
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
1080 1920
1920 1080
```

### 输出

```
YES
```

## 样例 #3

### 输入

```
334 668
668 1002
```

### 输出

```
YES
```

## 样例 #4

### 输入

```
100 200
300 150
```

### 输出

```
NO
```

## 样例 #5

### 输入

```
120 120
240 240
```

### 输出

```
NO
```

# [ABC050A] Addition and Subtraction Easy

## 题目描述

#### 读入a,b,x，如果x='+'就输出a+b，否则输出a-b（注意：a和x之间有一个空格，x和b之间有一个空格！！）。

## 输入格式

```
a，空格，x，空格，b
```

## 输出格式

#### a+b或a-b。
### **输入样例1**
```
1 + 2
```
### **输出样例1**
```
3
```
### **输出样例2**
```
5 - 7
```
### **输出样例2**
```
-2
```

## 样例 #1

### 输入

```
1 + 2
```

### 输出

```
3
```

## 样例 #2

### 输入

```
5 - 7
```

### 输出

```
-2
```

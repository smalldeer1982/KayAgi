# [ABC053B] A to Z String

## 题目描述

题目大意：
输入一个字符串s，问从第一个A到最后一个Z之间有多少个字母（含A和Z）（包括A和Z，如：UYEAERTZUY,就是UYE AERTZ UY,A到Z之间有5个字符。数据保证了开头是A的末尾是Z的部分一定存在的。）

## 输入格式

一个字符串s

## 输出格式

从A到Z之间有多少个字母


感谢@Andysun06 提供的翻译

## 样例 #1

### 输入

```
QWERTYASDFZXCV
```

### 输出

```
5
```

## 样例 #2

### 输入

```
ZABCZ
```

### 输出

```
4
```

## 样例 #3

### 输入

```
HASFJGHOGAKZZFEGA
```

### 输出

```
12
```

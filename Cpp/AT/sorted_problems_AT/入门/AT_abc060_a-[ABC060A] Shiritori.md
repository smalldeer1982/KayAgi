# [ABC060A] Shiritori

## 输入格式

第一行 三个字符串 $A$,$B$,$C$（$1\le |A|,|B|,|C|\le10$）。

## 输出格式

如果这三个字符串构成接龙（即下一个单词的首字母 和当前单词的尾字母相同），输出 `YES`，否则输出 `NO`。

**结尾记得换行！**

## 样例 #1

### 输入

```
rng gorilla apple
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
yakiniku unagi sushi
```

### 输出

```
NO
```

## 样例 #3

### 输入

```
a a a
```

### 输出

```
YES
```

## 样例 #4

### 输入

```
aaaaaaaaab aaaaaaaaaa aaaaaaaaab
```

### 输出

```
NO
```

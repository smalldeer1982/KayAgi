# [ABC077A] Rotation

## 题目描述

给定两个长度为 $3$ 的字符串，如果一个字符串反转后与另一个相等，那么输出 `YES`，否则输出 `NO`。

提示：输出的末尾要输出一个换行符。

## 输入格式

$C_{1,1},C_{1,2},C_{1,3}$  
$C_{2,1},C_{2,2},C_{2,3}$

## 输出格式

按要求输出 `YES` 或 `NO`。

## 说明/提示

输入数据中只会出现两个长度为 $3$ 的，由小写字母组成的字符串。

感谢@福州周圣力  提供的翻译

## 样例 #1

### 输入

```
pot
top
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
tab
bet
```

### 输出

```
NO
```

## 样例 #3

### 输入

```
eye
eel
```

### 输出

```
NO
```

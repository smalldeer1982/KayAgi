# [ABC084A] New Year

## 题目描述

请问从今年的12月30日的M点钟到明年的1月1日要多久,输出时间。(M是一个正整数)

## 输入格式

第一行一个正整数M。

## 输出格式

第一行一个正整数:输出从今年的12月30日的M点钟到明年的1月1日要多久的时间。

第二行:换行

感谢@鼬神 提供的翻译

## 样例 #1

### 输入

```
21
```

### 输出

```
27
```

## 样例 #2

### 输入

```
12
```

### 输出

```
36
```

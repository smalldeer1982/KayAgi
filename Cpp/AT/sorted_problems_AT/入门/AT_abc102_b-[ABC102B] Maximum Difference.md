# [ABC102B] Maximum Difference

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc102/tasks/abc102_b

長さ $ N $ の整数列 $ A $ が与えられます。 $ A $ の（添字の）異なる $ 2 $ 要素の差の絶対値の最大値を求めてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ ... $ $ A_N $

## 输出格式

$ A $ の（添字の）異なる $ 2 $ 要素の差の絶対値の最大値を出力せよ。

## 说明/提示

### 制約

- $ 2\ \leq\ N\ \leq\ 100 $
- $ 1\ \leq\ A_i\ \leq\ 10^9 $
- 入力はすべて整数である。

### Sample Explanation 1

$ A_3-A_1=6-1=5 $ が、異なる $ 2 $ 要素の差の最大値です。

## 样例 #1

### 输入

```
4
1 4 6 3
```

### 输出

```
5
```

## 样例 #2

### 输入

```
2
1000000000 1
```

### 输出

```
999999999
```

## 样例 #3

### 输入

```
5
1 1 1 1 1
```

### 输出

```
0
```

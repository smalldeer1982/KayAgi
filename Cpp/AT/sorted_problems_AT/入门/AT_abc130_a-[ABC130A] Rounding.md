# [ABC130A] Rounding

## 输入格式

输入共一行，两个整数 $a,b$ 。

## 输出格式

输出共一行，如果 $a<b$ ，则输出 $0$ ,否则输出 $10$。

输出完后记得换行！

翻译由 @[Jasoncwx](/user/592684) 整理

## 样例 #1

### 输入

```
3 5
```

### 输出

```
0
```

## 样例 #2

### 输入

```
7 5
```

### 输出

```
10
```

## 样例 #3

### 输入

```
6 6
```

### 输出

```
10
```

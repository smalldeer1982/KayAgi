# [ABC133A] T or T

## 输入格式

输入三个数 $N$,$A$,$B$。

## 输出格式

输出 $N* A$ 和 $B$ 中最小的那个数。如果相同就随便输出一个。

**记得换行。**

## 样例 #1

### 输入

```
4 2 9
```

### 输出

```
8
```

## 样例 #2

### 输入

```
4 2 7
```

### 输出

```
7
```

## 样例 #3

### 输入

```
4 2 8
```

### 输出

```
8
```

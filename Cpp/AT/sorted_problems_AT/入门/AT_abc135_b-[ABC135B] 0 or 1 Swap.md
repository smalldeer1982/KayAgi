# [ABC135B] 0 or 1 Swap

## 题目描述

### 题意：

有一个数列 $p$,你可以任意交换两个数，注意只能交换一次，如果可以使 $p$ 变成升序排列，输出 ```
YES```，否则输出 ```
NO```。

## 输入格式

输入 $n$。

输入 $n$ 个数 $p_i$。

## 输出格式

```YES``` 或 ```
NO```。

## 样例 #1

### 输入

```
5
5 2 3 4 1
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
5
2 4 3 5 1
```

### 输出

```
NO
```

## 样例 #3

### 输入

```
7
1 2 3 4 5 6 7
```

### 输出

```
YES
```

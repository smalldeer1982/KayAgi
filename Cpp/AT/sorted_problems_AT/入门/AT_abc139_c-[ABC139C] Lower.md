# [ABC139C] Lower

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc139/tasks/abc139_c

左右一列に $ N $ 個のマスが並んでいます。

左から $ i $ 番目のマスの高さは $ H_i $ です。

あなたは好きなマスに降り立ち、**右隣の**マスの高さが今居るマスの高さ以下である限り右隣のマスへ移動し続けます。

最大で何回移動できるでしょうか。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ H_1 $ $ H_2 $ $ ... $ $ H_N $

## 输出格式

移動できる回数の最大値を出力せよ。

## 说明/提示

### 制約

- 入力は全て整数である。
- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ 1\ \leq\ H_i\ \leq\ 10^9 $

### Sample Explanation 1

左から $ 3 $ 番目のマスに降り立つと、右に $ 2 $ 回移動できます。

### Sample Explanation 2

左から $ 4 $ 番目のマスに降り立つと、右に $ 3 $ 回移動できます。

## 样例 #1

### 输入

```
5
10 4 8 7 3
```

### 输出

```
2
```

## 样例 #2

### 输入

```
7
4 4 5 6 6 5 5
```

### 输出

```
3
```

## 样例 #3

### 输入

```
4
1 2 3 4
```

### 输出

```
0
```

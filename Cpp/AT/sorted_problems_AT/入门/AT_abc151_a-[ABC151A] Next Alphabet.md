# [ABC151A] Next Alphabet

## 题目描述

# 题目翻译


输入一个小写字母 $z$ ,输出它按字典序排序的下一位。

## 输入格式

按以下的格式输入。

 ```
$ C $
```

## 输出格式

输出它按字典序排序的下一位。

## 说明/提示

$z$是小写字母。


‘ａ’的下一位是‘ｂ’。


‘ｙ’的下一位是‘ｚ’。

## 样例 #1

### 输入

```
a
```

### 输出

```
b
```

## 样例 #2

### 输入

```
y
```

### 输出

```
z
```

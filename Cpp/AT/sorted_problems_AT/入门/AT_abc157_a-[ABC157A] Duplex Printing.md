# [ABC157A] Duplex Printing

## 题目描述

高桥想要打印一有 $N$ 面的文档，因为他用的是双面打印机，所以一张纸可以打印 $2$ 面，求他需要几张纸？

## 输入格式

仅一个整数，表示文档的面数 $N$。

## 输出格式

仅一个整数，表示高桥需要几张纸。

## 说明/提示

$N$ 为整数，

$1\leq N \leq100$。

## 样例 #1

### 输入

```
5
```

### 输出

```
3
```

## 样例 #2

### 输入

```
2
```

### 输出

```
1
```

## 样例 #3

### 输入

```
100
```

### 输出

```
50
```

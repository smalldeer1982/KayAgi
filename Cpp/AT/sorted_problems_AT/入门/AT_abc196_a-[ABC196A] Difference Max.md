# [ABC196A] Difference Max

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc196/tasks/abc196_a

整数 $ a,\ b,\ c,\ d $ が与えられます。  
 $ a\ <\ = x\ <\ = b,\ c\ <\ =\ y\ <\ =\ d $ となるように整数 $ x,\ y $ を選ぶとき、 $ x\ -\ y $ の最大値を求めてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ a $ $ b $ $ c $ $ d $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- 入力は全て整数
- $ -100\ <\ =\ a\ <\ = b\ <\ = 100 $
- $ -100\ <\ =\ c\ <\ = d\ <\ = 100 $

### Sample Explanation 1

$ (x,\ y)\ =\ (10,\ 0) $ のとき最大値 $ x\ -\ y\ =\ 10 $ をとります。

## 样例 #1

### 输入

```
0 10
0 10
```

### 输出

```
10
```

## 样例 #2

### 输入

```
-100 -100
100 100
```

### 输出

```
-200
```

## 样例 #3

### 输入

```
-100 100
-100 100
```

### 输出

```
200
```

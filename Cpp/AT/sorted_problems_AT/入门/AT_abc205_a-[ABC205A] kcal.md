# [ABC205A] kcal

## 题目描述

## 题目翻译

每 $100$ mL 饮料都会有 $A$ 千卡，那么现在有 $B$ mL 饮料，这瓶饮料会有多少千卡呢？

## 输入格式

输入用以下标准输入：

> $A$ $B$

## 输出格式

给出一个数字表示答案。

你给出的答案和假定答案绝对误差或者相对误差小于 $10^{-6}$ 就视为正确的答案。

## 说明/提示

### 限制

* $0 \leq A,B \leq 1000$
* 输入的数字全是整数。


每 $100$ mL 饮料会有 $45$ 千卡，这瓶饮料有 $200$ 毫升，所以有 $90$ 千卡的能量。


答案可能不是整数。

## 样例 #1

### 输入

```
45 200
```

### 输出

```
90
```

## 样例 #2

### 输入

```
37 450
```

### 输出

```
166.5
```

## 样例 #3

### 输入

```
0 1000
```

### 输出

```
0
```

## 样例 #4

### 输入

```
50 0
```

### 输出

```
0
```

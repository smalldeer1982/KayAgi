# [ABC256A] 2^N

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc256/tasks/abc256_a

$ N $ が与えられます。$ 2^N $ を出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ 0\ \leq\ N\ \leq\ 30 $
- $ N $ は整数である

### Sample Explanation 1

$ 2^3=8 $ です。

## 样例 #1

### 输入

```
3
```

### 输出

```
8
```

## 样例 #2

### 输入

```
30
```

### 输出

```
1073741824
```

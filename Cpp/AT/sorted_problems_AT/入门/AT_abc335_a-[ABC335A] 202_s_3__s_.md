# [ABC335A] 202<s>3</s>

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc335/tasks/abc335_a

英小文字と数字からなる文字列 $ S $ が入力されます。  
 $ S $ は `2023` で終わることが保証されます。  
 $ S $ の最後の文字を `4` に変更し、変更後の文字列を出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ S $

## 输出格式

答えを出力せよ。

## 说明/提示

### 制約

- $ S $ は英小文字と数字からなる長さ $ 4 $ 以上 $ 100 $ 以下の文字列
- $ S $ は `2023` で終わる。
 
### Sample Explanation 1

`hello2023` の最後の文字を `4` に変更すると、 `hello2024` となります。

### Sample Explanation 3

$ S $ が `2023` で終わることが保証されていますが、 $ S $ が `2023` そのものである場合もあります。

## 样例 #1

### 输入

```
hello2023
```

### 输出

```
hello2024
```

## 样例 #2

### 输入

```
worldtourfinals2023
```

### 输出

```
worldtourfinals2024
```

## 样例 #3

### 输入

```
2023
```

### 输出

```
2024
```

## 样例 #4

### 输入

```
20232023
```

### 输出

```
20232024
```

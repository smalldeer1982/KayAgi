# [ABC390A] 12435

## 题目描述

给出一个整数序列 $A$，$A$ 里包含 $5$ 个数，肯定是 $1∼5$ 的一种排列。问能不能进行且只进行对相邻两个数的 $1$ 次交换，让这个序列变为升序的。如果能，输出 `Yes`，否则输出 `No`。

## 输入格式

输入五个数以空格分隔。

## 输出格式

输出一行，如果一次交换能变为升序序列，输出 `Yes`，否则输出 `No`。

## 样例 #1

### 输入

```
1 2 4 3 5
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
5 3 2 4 1
```

### 输出

```
No
```

## 样例 #3

### 输入

```
1 2 3 4 5
```

### 输出

```
No
```

## 样例 #4

### 输入

```
2 1 3 4 5
```

### 输出

```
Yes
```

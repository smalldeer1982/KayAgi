# ひふみ　(Hihumi)

## 输入格式

输入一个正整数 $n$。

## 输出格式

输出从 $1$ 到 $n$ 中不为 $123$ 的数的个数。

**记得换行！**

## 样例 #1

### 输入

```
10
```

### 输出

```
10
```

## 样例 #2

### 输入

```
124
```

### 输出

```
123
```

## 样例 #3

### 输入

```
1000
```

### 输出

```
999
```

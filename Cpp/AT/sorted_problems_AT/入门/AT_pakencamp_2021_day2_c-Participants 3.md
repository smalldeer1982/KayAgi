# Participants 3

## 题目描述

帕研君举办了 $2$ 天的编程竞赛。第 $1$ 天的比赛有 $N$ 人参加，第 $2$ 天的比赛有 $M$ 人参加。

每个参赛者都被分配了一个整数 ID。第 $1$ 天参与者的 ID 按顺序为 $A_1,\ A_2,\ \ldots,\ A_N$，第 $2$ 天参与者的 ID 按顺序为 $B_1,\ B_2,\ \ldots,\ B_M$。不同的人没有被分配过相同的 ID。

帕研君想要满足以下条件的人的 ID 的一览表：

- 保证这个人没有参加第 $1$ 天的比赛，但是参加了第 $2$ 天的比赛。

请你替帕研君列举满足条件人的 ID。

## 输入格式

输入按以下格式

> $N$ $M$ 

> $A_1$ $A_2$ $\ldots$ $A_N$ 

> $B_1$ $B_2$ $\ldots $ $B_M$

## 输出格式

满足条件的人数为 $K$。

首先请在第 $ 1 $ 行中输出 $ K $。然后在接下来 $K$ 行，将每个人的 ID 按每行 $1$ 个的格式输出。

ID 按什么顺序输出都可以。

## 说明/提示

- $ 1\ \leq\ N\ \leq\ 3000 $
- $ 1\ \leq\ M\ \leq\ 3000 $
- $ 1\ \leq\ A_i\ \leq\ 3000 $
- $ 1\ \leq\ B_i\ \leq\ 3000 $
- $ i\ \neq\ j\ \implies\ A_i\ \neq\ A_j $
- $ i\ \neq\ j\ \implies\ B_i\ \neq\ B_j $
- 所有输入的值都是整数。

样例 $1$ 解释：

ID 为 $ 6$ 和 $8$ 的人没有参加第 $1$ 天的比赛，但是参加了第 $2$ 天的比赛。ID 按什么顺序输出都可以，所以像下面这样的输出也是正确答案：`2 8 6`。

样例 $2$ 解释：

没有满足条件的人。

样例 $4$ 解释：

出题人：[Forested](https://atcoder.jp/users/Forested)

## 样例 #1

### 输入

```
3 4
1 3 4
6 8 3 1
```

### 输出

```
2
6
8
```

```

```

```

```

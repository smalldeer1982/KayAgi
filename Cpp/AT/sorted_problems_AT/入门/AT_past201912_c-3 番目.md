# 3 番目

## 题目描述

[problemUrl]: https://atcoder.jp/contests/past201912-open/tasks/past201912_c

$ 6 $ つの相異なる整数 $ A $, $ B $, $ C $, $ D $, $ E $, $ F $ が与えられる。

このうち $ 3 $ 番目に大きい数を調べるプログラムを作成せよ。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ A $ $ B $ $ C $ $ D $ $ E $ $ F $

## 输出格式

$ 3 $ 番目に大きい整数を出力せよ。

## 说明/提示

### 注意

この問題に対する言及は、2019年12月29日 05:00 JST まで禁止されています。言及がなされた場合、賠償が請求される可能性があります。

試験後に総合得点や認定級を公表するのは構いませんが、どの問題が解けたかなどの情報は発信しないようにお願いします。

### 制約

- $ 1\ \leqq\ A,\ B,\ C,\ D,\ E,\ F\ \leqq\ 100 $
- $ A,\ B,\ C,\ D,\ E,\ F $ はすべて異なる。
- 入力中の値はすべて整数である。

### Sample Explanation 1

$ 3 $ 番目に大きい数は $ 18 $ である。

## 样例 #1

### 输入

```
4 18 25 20 9 13
```

### 输出

```
18
```

## 样例 #2

### 输入

```
95 96 97 98 99 100
```

### 输出

```
98
```

## 样例 #3

### 输入

```
19 92 3 35 78 1
```

### 输出

```
35
```

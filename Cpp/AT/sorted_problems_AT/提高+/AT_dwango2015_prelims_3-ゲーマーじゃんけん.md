# ゲーマーじゃんけん

## 题目描述

给你一个由 $N$ 个正整数组成的序列 $A = \{A_1, A_2, \dots, A_N\}$。请你计算出满足以下条件的三元组 $(i, j, k)$ 的数量：

- $1 \leq i < j < k \leq N$
- $A_i \times A_j \times A_k$ 是 $6$ 的倍数

## 输入格式

输入由两行组成：

第一行为一个整数 $N$，表示序列的长度。

第二行为 $N$ 个用空格隔开的整数 $A_1, A_2, \dots, A_N$，表示序列中的元素。

## 输出格式

输出一个整数，表示满足条件的三元组的数量。

## 说明/提示

- $1 \leq N \leq 10^3$
- $1 \leq A_i \leq 10^9$

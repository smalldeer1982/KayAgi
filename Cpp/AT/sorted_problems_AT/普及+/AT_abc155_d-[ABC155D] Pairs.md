# [ABC155D] Pairs

## 题目描述

$N$个数两两相乘的结果有 $\frac{N(N-1)}{2}$ 种，问第 $K$ 小的乘积是多少。

## 输入格式

第一行两个整数 $N,K$。  
第二行 $N$ 个整数 $A_i$，为那 $N$ 个要乘起来的数

## 输出格式

一行一个整数，第 $K$ 小的乘积

## 说明/提示

$N \leq 2 \times 10^5,-10^9 \leq A_i \leq 10^9$

## 样例 #1

### 输入

```
4 3
3 3 -4 -2
```

### 输出

```
-6
```

## 样例 #2

### 输入

```
10 40
5 4 3 2 -1 0 0 0 0 0
```

### 输出

```
6
```

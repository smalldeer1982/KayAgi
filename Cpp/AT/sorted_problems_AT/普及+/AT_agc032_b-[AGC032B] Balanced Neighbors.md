# [AGC032B] Balanced Neighbors

## 题目描述

给定整数 $N$，构造一个从 $1$ 到 $N$ 编号的 $N$ 个节点的无向图，使得：

- 该图不含有重边和自环，并且是连通的。
- 每个节点的所有邻接节点的编号之和相同。

可以证明这样的图一定存在。

## 输入格式

一行一个整数 $N$。

## 输出格式

第一行一个整数 $M$，表示构造出的图的边数。

接下来 $M$ 行，每行两个整数 $a_i,b_i$，表示第 $i$ 条边的两个端点。

如果有多种可能的构造，输出其中的任意一种即可。

## 说明/提示

$3 \leq N \leq 100$。

【样例解释】

对于所有节点，其邻接节点的编号之和均为 $3$。

## 样例 #1

### 输入

```
3
```

### 输出

```
2
1 3
2 3
```

# 登山家

## 题目描述

高桥君是登山家，现在正在登山。
这个山脉上N个山中小屋向东西一条直线排列着，
各山小屋从西向东依次排列着，以1到N的号码排序。
第i的山中小屋建于海拔hi。
高桥君很在意从各小屋能看到几间山中小屋?
从第i个山中小屋可以看到第j个（山中小屋）的条件是，
##### 第j（山中小屋）的海拔全部为hi下（hj≤hi）。
求在各山中小屋可以看到的山中小屋的个数。
另外，从第i的山中小屋可以看到的山中小屋不包括第i个山中小屋。

## 输入格式

###### 第一行为山中小屋的个数N（1 ≦ N ≦ 100000）
###### 第二行为各小屋的海拔

## 输出格式

###### 输出N行，其中第i行表示在第i个小屋能看到的小屋数j

## 样例 #1

### 输入

```
3
1
2
3
```

### 输出

```
0
1
2
```

## 样例 #2

### 输入

```

```

```

1

1
0
```

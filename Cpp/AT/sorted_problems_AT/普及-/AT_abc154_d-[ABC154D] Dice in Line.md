# [ABC154D] Dice in Line

## 题目描述

给出 $N$ 个骰子，我们同时投掷 $K$ 个连续骰子。求出可能的最大数学期望值。

## 输入格式

第一行两个数，$N$ 和 $K$，含义在描述中给出  
第二行 $N$ 个数，$p_1 \ldots p_N$，代表每一个骰子的面数

## 输出格式

一行一个数，代表最大的数学期望值

## 说明/提示

本题使用**Special Judge**，你的答案和数据误差小于等于$10^{-6}$就会被评判对

## 样例 #1

### 输入

```
5 3
1 2 2 4 5
```

### 输出

```
7.000000000000
```

## 样例 #2

### 输入

```
4 1
6 6 6 6
```

### 输出

```
3.500000000000
```

## 样例 #3

### 输入

```
10 4
17 13 13 12 15 20 10 13 17 11
```

### 输出

```
32.000000000000
```

# [ABC390B] Geometric Sequence

## 题目描述

给定一个正整数序列 $A$，问这个序列 $A$ 是否为等比数列。如果是，输出 `Yes` ，否则输出 `No`。

## 输入格式

第一行输入一个数 $N$。

第二行输入 $n$ 个数，即序列 $A$。

## 输出格式

如果 $A$ 序列是等比数列，输出 `Yes` ，否则输出 `No`。

## 样例 #1

### 输入

```
5
3 6 12 24 48
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
3
1 2 3
```

### 输出

```
No
```

## 样例 #3

### 输入

```
2
10 8
```

### 输出

```
Yes
```

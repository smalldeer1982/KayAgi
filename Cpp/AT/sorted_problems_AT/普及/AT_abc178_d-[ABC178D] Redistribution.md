# [ABC178D] Redistribution

## 题目描述

给定整数 $ S $。请求出有几个序列，所有项都是 $ 3 $ 以上的整数，其总和是 $ S $。因为答案可能会非常大，所以请输出除以$ 1e9+7 $ 后的余数。

## 输入格式

一个整数$ S $

## 输出格式

答案除以$ 1e9+7 $后的余数

## 样例 #1

### 输入

```
7
```

### 输出

```
3
```

## 样例 #2

### 输入

```
2
```

### 输出

```
0
```

## 样例 #3

### 输入

```
1729
```

### 输出

```
294867501
```

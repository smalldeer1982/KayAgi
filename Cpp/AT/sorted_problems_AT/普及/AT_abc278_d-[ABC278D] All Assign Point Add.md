# [ABC278D] All Assign Point Add

## 题目描述

【题目翻译】

给定长度为 $n$ 的数组 $a$，每次有三种操作：

+ $op_i = 1$，表示将 $a$ 数组全部元素替换成 $k$。
+ $op_i = 2$，表示 $a_i \gets a_i + k$。
+ $op_i = 3$，表示查询 $a_i$ 的值。

对于每个 $op_i = 3$，输出结果。

translated by @[liangbowen](https://www.luogu.com.cn/user/367488).

## 输入格式

第一行一个数 $n$。

接下来 $n$ 个数，表示 $a$ 数组。

第三行一个数 $q$。$q$ 表示操作次数。

接下来 $q$ 行，每行表示一次操作：

+ $op_i = 1$，则再读入一个数 $k$。
+ $op_i = 2$，则再读入两个数 $i$，$k$。
+ $op_i = 3$，则再读入一个数 $i$。

## 输出格式

对于每个 $op_i = 3$，输出结果。

## 说明/提示

$1 \le n, q \le 2 \times 10^5$

保证 $1 \le i \le n$，$1 \le a_i, k \le 10^9$。

## 样例 #1

### 输入

```
5
3 1 4 1 5
6
3 2
2 3 4
3 3
1 1
2 3 4
3 3
```

### 输出

```
1
8
5
```

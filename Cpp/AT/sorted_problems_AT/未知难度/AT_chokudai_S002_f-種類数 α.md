# 種類数 α

## 题目描述

[problemUrl]: https://atcoder.jp/contests/chokudai_S002/tasks/chokudai_S002_f

$ N $ 枚のコインがあります。$ i $ 枚目のコインの片方の面には整数 $ A_i $、もう片方の面には整数 $ B_i $ が書かれています。

書かれている整数の組が同じであるコインを区別しないとき、コインは全部で何種類あるでしょうか？

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $ $ A_1 $ $ B_1 $ $ A_2 $ $ B_2 $ $ : $ $ A_N $ $ B_N $

## 输出格式

コインの種類数を出力せよ。

## 说明/提示

### 制約

入力は以下の条件を満たす。

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ A_i,B_i\ \leq\ 10^9 $
- 入力される値は全て整数

### Sample Explanation 1

書かれた整数の組が $ (1,2),\ (3,4),\ (5,5) $ である $ 3 $ 種類のコインがあります。 $ 1 $ 枚目のコインと $ 2 $ 枚目のコインはひっくり返すと同じ種類のコインであることに注意してください。

## 样例 #1

### 输入

```
5
1 2
2 1
3 4
5 5
3 4
```

### 输出

```
3
```

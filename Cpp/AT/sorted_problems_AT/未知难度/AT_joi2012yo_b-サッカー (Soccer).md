# サッカー (Soccer)

## 题目描述

2077 年，JOI 国大型足球联赛已经圆满落幕，工作人员现在希望让你帮他们统计各个球队的最终排名。

联赛一共有 $N$ 支球队参加，每两支球队之间都要进行一场球赛。不难看出，一共要进行 $\frac{N(N-1)}2$ 场比赛。球队赢一场比赛计 $3$ 分，输一场比赛不计分，平局计 $1$ 分。最终会根据球队总分进行排序，每支球队排序后所在的位置就是其最终排名，如果两支球队的总分一样，则最终排名也一样。

现在，给出所有 $\frac {N(N-1)}2$ 场比赛的信息，包括参赛的双方球队及双方进球个数，求出每支球队的最终排名。

## 输入格式

第一行输入一个正整数 $N$，表示参赛球队数量。  
接下来 $\frac {N(N-1)}2$ 行，第 $i+1$ 行 $4$ 个正整数 $A_i,B_i,C_i,D_i$，描述了第 $i$ 场比赛的信息。其中，$A_i,B_i$ 分别表示参赛的双方球队编号，$C_i$ 表示编号为 $A_i$ 的球队在该比赛中的进球个数，$D_i$ 表示编号为 $B_i$ 的球队在该比赛中的进球个数。

## 输出格式

数出共 $N$ 行，第 $i$ 行表示编号为 $i$ 的球队的最终排名。

**样例 1 输入**

```
4
1 2 0 1
1 3 2 1
1 4 2 2
2 3 1 1
2 4 3 0
3 4 1 3
```

**样例 1 输出**

```
2
1
4
2
```

**样例 1 解释**

编号为 $1$ 的球队赢了 $1$ 场（第 $2$ 场），输了 $1$ 场（第 $1$ 场），平局 $1$ 场（第 $3$ 场），总分为 $3+0+1=4$。  
编号为 $2$ 的球队赢了 $2$ 场（第 $1$、$5$ 场），输了 $0$ 场，平局 $1$ 场（第 $4$ 场），总分为 $3\times 2+1=7$。  
编号为 $3$ 的球队赢了 $0$ 场，输了 $2$ 场（第 $2$、$6$ 场），平局 $1$ 场（第 $4$ 场），总分为 $0\times 2+1=1$。  
编号为 $4$ 的球队赢了 $1$ 场（第 $6$ 场），输了 $1$ 场（第 $5$ 场），平局 $1$ 场（第 $3$ 场），总分为 $3+0+1=4$。

因此，排序之后，编号为 $2$ 的球队是第 $1$ 名，编号为 $1$、$4$ 的球队并列第 $2$，编号为 $3$ 的球队是第 $4$ 名。

**样例 2 输入**

```
5
1 2 1 1
3 4 3 1
5 1 1 2
2 3 0 0
4 5 2 3
1 3 0 2
5 2 2 2
4 1 4 5
3 5 4 0
2 4 0 1
```

**样例 2 输出**

```
2
4
1
4
3
```

## 说明/提示

- $2\leqslant N\leqslant 100$。
- $1\leqslant A_i,B_i\leqslant N$，$0\leqslant C_i,D_i\leqslant 100$。

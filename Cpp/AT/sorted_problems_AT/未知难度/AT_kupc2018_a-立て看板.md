# 立て看板

## 题目描述

有 $N$ 块广告牌，编号从 $1$ 到 $N$，广告牌 $i$ 面积为 $s_i$，美丽程度为$a_i$。

广告牌 $i$ 的破坏程度为 $s_i\times a_i$。

请输出 $N$ 个广告牌中破坏程度的最大值。

## 输入格式

> $N \ \ s_1 \  s_2 \ ... \ s_N \ $
$ \ a_1 \ a_2 \  ... \ a_N  $

## 输出格式

输出 $N$ 个广告牌中破坏程度的最大值。

## 说明/提示

- $1\ \leq\ N\ \leq\ 100$
- $1\ \leq\ s_i\ \leq\ 100 \ (1\ \leq\ i\ \leq\ N)$ 
- $1\ \leq\ a_i\ \leq\ 100  \ (1\ \leq\ i\ \leq\ N)$

## 样例 #1

### 输入

```
3
3 2 1
3 3 4
```

### 输出

```
9
```

## 样例 #2

### 输入

```
1
1
1
```

### 输出

```
1
```

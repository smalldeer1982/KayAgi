# 七目

## 题目描述

有一个 $9 \times 9$ 的格子，第一个格子为白色。

你要用黑色方块，让白色方块不会连续出现 $7$ 个方块，无论是横向、纵向还是斜向！

现在请从这些填充方法中输出一种。

## 输入格式

无输入

## 输出格式

从这些填充方法中输出一种，另外 ```.``` 表示白色方块，```#``` 表示黑色方块。

## 样例 #1

### 输入

```
None
```

### 输出

```
#########
#########
#########
#########
#########
#########
#########
#########
#########
```

## 样例 #2

### 输入

```
None
```

### 输出

```
.#.#.#.#.
#.#.#.#.#
.#.#.#.#.
#.#.#.#.#
.#.#.#.#.
#.#.#.#.#
.#.#.#.#.
#.#.#.#.#
.#.#.#.#.
```

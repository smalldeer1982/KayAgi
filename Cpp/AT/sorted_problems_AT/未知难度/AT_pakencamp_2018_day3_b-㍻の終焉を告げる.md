# ㍻の終焉を告げる

## 题目描述

### 题目简述

给出一个长为 $N$ 的数组 $a$，求出满足 $\displaystyle\sum_{i=1}^{k}a_i\le 2018$ 的正整数 $k$ 的数量（$1\le k\le N$）。

## 输入格式

第一行输入整数 $N$。

接下来 $N$ 行，第 $i$ 行输入整数 $a_i$。

## 输出格式

输出一行一个整数，题目所求。**建议在输出末尾添加换行。**

## 说明/提示

#### 数据规模与约定

$1\le N\le 8000$，$1\le a_i\le 10000$。

## 样例 #1

### 输入

```
3
700
700
700
```

### 输出

```
2
```

## 样例 #2

### 输入

```
2
2018
1
```

### 输出

```
1
```

## 样例 #3

### 输入

```
1
10000
```

### 输出

```
0
```

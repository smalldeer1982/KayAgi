# その計算、合ってますか？

## 题目描述

penguinman 有一个由非负整数构成的非空多重集合。他对这个集合进行了以下计算：

- 每个位上的逻辑或结果为 $A$
- 每个位上的逻辑与结果为 $B$
- 每个位上的异或结果为 $C$

他对这些计算结果有些不确定，希望你帮忙验证。你的任务是判断是否存在一个非负整数的多重集合，使得它满足 penguinman 所描述的性质而不矛盾。

给定 $T$ 组测试数据，请针对每组数据，判断这样的多重集合是否真实存在。

有关逻辑运算的解释，请参考以下维基百科链接：
- 逻辑或：[逻辑或定义](https://ja.wikipedia.org/wiki/%E8%AB%96%E7%90%86%E5%92%8C#:~:text=%E6%95%B0%E7%90%86%E8%AB%96%E7%90%86%E5%AD%A6%E3%81%AB%E3%81%8A%E3%81%84%E3%81%A6%E8%AB%96%E7%90%86,P%20%E3%81%BE%E3%81%9F%E3%81%AF%20Q%E3%80%8D%E3%81%A8%E8%AA%AD%E3%82%80%E3%80%82)
- 逻辑与：[逻辑与定义](https://ja.wikipedia.org/wiki/%E8%AB%96%E7%90%86%E7%A9%8D#:~:text=%E6%95%B0%E7%90%86%E8%AB%96%E7%90%86%E5%AD%A6%E3%81%AB%E3%81%8A%E3%81%84%E3%81%A6%E8%AB%96%E7%90%86,%E7%A4%BA%E3%81%99%E8%AB%96%E7%90%86%E6%BC%94%E7%AE%97%E3%81%A7%E3%81%82%E3%82%8B%E3%80%82)
- 异或：[异或定义](https://ja.wikipedia.org/wiki/%E6%8E%92%E4%BB%96%E7%9A%84%E8%AB%96%E7%90%86%E5%92%8C#:~:text=%E6%8E%92%E4%BB%96%E7%9A%84%E8%AB%96%E7%90%86%E5%92%8C%EF%BC%88%E3%81%AF,%EF%BC%88%E8%AB%96%E7%90%86%E6%BC%94%E7%AE%97%EF%BC%89%E3%81%A7%E3%81%82%E3%82%8B%E3%80%82)

## 输入格式

输入由标准输入给出，格式如下。每组输入的 $A, B, C$ 分别记作 $A_i, B_i, C_i$：

```
\(T\)
\(A_1\) \(B_1\) \(C_1\)
\(A_2\) \(B_2\) \(C_2\)
⋮
\(A_T\) \(B_T\) \(C_T\)
```

## 输出格式

对于每个测试用例，输出一行。若存在符合条件的多重集合，则输出 `Yes`，否则输出 `No`。每行末尾需要换行。

## 说明/提示

- 测试组数 $1 \leq T \leq 100$
- 每个数的范围 $0 \leq A_i, B_i, C_i < 2^{30}$

 **本翻译由 AI 自动生成**

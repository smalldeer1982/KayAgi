# PASTal Code

## 题目描述

[problemUrl]: https://atcoder.jp/contests/past202104-open/tasks/past202104_b

高橋くんは、以下の条件を満たす文字列 $ S $ を入力しました。

- $ S $ は `past` または `post` を合計 $ 1 $ つ以上連結した文字列
- 連結された文字列のうち、 `post` は高々 $ 1 $ つ

あなたへの課題は、連結された文字列のうち、何番目が `post` かを求めることです。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ S $

## 输出格式

もし `post` が $ 1 $ つ連結されていれば、それがはじめから何番目に連結された文字列かを $ 1 $ つの正整数として出力せよ。 `post` が $ 1 $ つも連結されていなければ、 `none` と出力せよ。

## 说明/提示

### 注意

この問題に対する言及は、2021/4/24 18:00 JST まで禁止されています。言及がなされた場合、賠償が請求される可能性があります。 試験後に総合得点や認定級を公表するのは構いませんが、どの問題が解けたかなどの情報は発信しないようにお願いします。

### 制約

- $ |S|\ \le\ 1000 $
- $ S $ は `past` または `post` を合計 $ 1 $ つ以上連結した文字列
- 連結された文字列のうち、 `post` は高々 $ 1 $ つ

### Sample Explanation 1

$ S= $ `pastpastpostpast` です。 $ 3 $ 番目に連結された文字列が `post` です。

### Sample Explanation 2

$ S= $ `pastpastpast` です。 `post` は $ 1 $ つも連結されていません。

## 样例 #1

### 输入

```
pastpastpostpast
```

### 输出

```
3
```

## 样例 #2

### 输入

```
pastpastpast
```

### 输出

```
none
```

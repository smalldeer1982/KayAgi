# 3-順列

## 题目描述

给出 $M$ 个大于等于1、小于等于 $N$ 的不同整数对 $(a_{i},b_{i})$。
从 $1$ 到 $N$ 的排列 $P=(P_{1},P_{2},\cdots,P_{N})$，其中任何 $i$ 都满足 $1 \le i \le M$ ,请判断是否存在 $P_{A_{i}}+P_{B_{i}}$ 为 $3$ 的倍数。

## 输入格式

输入为以下形式:

> $N \ M \ A_{1} \ B_{1} \ : \ A_{M} \ B_{M}$

## 输出格式

如果存在满足条件的排列，则输出`Yes`，否则输出`No`。

## 样例 #1

### 输入

```
5 2
1 2
2 3
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
3 2
1 2
2 3
```

### 输出

```
No
```

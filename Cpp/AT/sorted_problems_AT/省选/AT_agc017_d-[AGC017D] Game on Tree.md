# [AGC017D] Game on Tree

## 题目描述

有一棵 $N$ 个节点的树，节点标号为 $1,2,⋯,N$，边用 $(x_i,y_i)$表示。 Alice 和 Bob 在这棵树上玩一个游戏，Alice先手，两人轮流操作：


选择一条树上存在的边，把它断开使树变成两个连通块。然后把不包含 $1$ 号点的联通块删除


当一个玩家不能操作时输，你需要算出：假如两人都按最优策略操作，谁将获胜。

## 输入格式

$N$

$x_1$ $y_1$

$x_2$ $y_2$

$...$

$x_{n-1}$ $y_{n-1}$

## 输出格式

若Alice获胜，输出```Alice```

否则输出```Bob```

## 说明/提示

$1 \leq N \leq 100000$

$1\leq x_i,y_i \leq N$

保证给出的是一棵树

## 样例 #1

### 输入

```
5
1 2
2 3
2 4
4 5
```

### 输出

```
Alice
```

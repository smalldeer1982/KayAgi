# ukuku

## 题目描述

给定一个长为 $n$ 的整数序列 $a$，表示一个字符串中从前往后数以第 $i$ 位为中心的回文串的最长长度。请输出任意一个满足条件的字符串。数据保证有解。

## 输入格式

第一行为整数 $n$；第二行为 $n$ 个整数，表示序列 $a$。

## 输出格式

输出任意一个长为 $n$ 且符合要求的小写英文字母串即可。

## 说明/提示

#### 数据规模与约定

$1 \le n \le 2 \times 10^5$，$1 \le a_i \le n$。

## 样例 #1

### 输入

```
5
1 3 5 3 1
```

### 输出

```
ukuku
```

## 样例 #2

### 输入

```
7
1 1 3 5 5 3 1
```

### 输出

```
ukekeke
```

# 集合写真

## 题目描述

有N个人（1<=N<=50）按照从矮到高左右排列，这时候又来了一个人，请你确定这个人的位置

## 输入格式

第一行：一个正整数N，表示多少人（1<=N<=50）

第二行：N个正整数H1,H2,H3...HN（1<=H<=100）,表示N个人高度

第三行：

一个正整数X，表示后来那个人的身高，（X与前面的任何一个H都不相同）

## 输出格式

一个正整数I，表示后来那个人的位置，注意（岛国的题要换行）

Translated by @a2954898606

## 样例 #1

### 输入

```
3

11 18 22

17
```

### 输出

```
2
```

## 样例 #2

### 输入

```
5

30 40 50 60 70

10
```

### 输出

```
1
```

## 样例 #3

### 输入

```
6

11 24 44 56 78 99

100
```

### 输出

```
7
```

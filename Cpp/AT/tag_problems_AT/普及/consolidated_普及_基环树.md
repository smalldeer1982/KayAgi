# [ABC311C] Find it!

## 题目描述

**题意简述**

给定一个有向图，其中有 $N$ 个顶点和 $N$ 条边。保证其中存在一个环。请你找到任意一个环。

## 输入格式

第一行为整数 $N$。
第二行有 $N$ 个整数，第 $i$ 个整数为 $A_i$ ，表示从顶点 $i$ 到顶点 $A_i$ 有边。

## 输出格式

第一行为环的长度。

第二行输出这个环，可以以任意顺序。

## 说明/提示

- $2 \le N \le 2 \times 10^5$
- $1 \le A_i \le N$
- $A_i \ne i$

## 样例 #1

### 输入

```
7

6 7 2 1 3 4 5
```

### 输出

```
4

7 5 3 2
```

## 样例 #2

### 输入

```
2

2 1
```

### 输出

```
2

1 2
```

## 样例 #3

### 输入

```
8

3 7 4 7 3 3 8 2
```

### 输出

```
3

2 7 8
```

# [ABC029D] 1

## 题目描述

高桥先生用十进制标记一次性地在纸上写了  $1$ ~ $N$  以下的所有整数。 在这项工作中，高桥君写了几个  $1$  这样的数字呢？即求从 
 $1$ ~ $N$  中数字  $1$  出现的次数。（本题类似 P1980）

## 输出格式

说明： 对于  $20\%$  的数据，有  $1 \leq N \leq 999$ 。

对于  $100\%$  的数据，有  $1 \leq N \leq 10^9$ 。

感谢@初音Miku 提供的翻译

## 样例 #1

### 输入

```
12
```

### 输出

```
5
```

## 样例 #2

### 输入

```
345
```

### 输出

```
175
```

## 样例 #3

### 输入

```
999999999
```

### 输出

```
900000000
```

# Polycarp's Pockets

## 题目描述

题意   
输入$n$个数，输出其中最多有多少个数是相同的。

## 输入格式

第一行为一个整数$n$ ( 1 $\le$ $n$ $\le$ 100 ) , 第二行为$n$个整数。

## 输出格式

一个整数，表示其中最多相同的数的个数。

## 样例 #1

### 输入

```
6
1 2 4 3 3 2

```

### 输出

```
2

```

## 样例 #2

### 输入

```
1
100

```

### 输出

```
1

```

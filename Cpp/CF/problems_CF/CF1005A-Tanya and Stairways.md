# Tanya and Stairways

## 题目描述

给你N个数（1≤N≤1000），这N个数是由M个从1开始的连续自然序列组成的

## 输入格式

第一行，一个数：N

第二行N个数，分别是组成M个序列的元素

## 输出格式

第一行，一个数：M

第二行，M个数，分别是这M个序列的长度

## 样例 #1

### 输入

```
7
1 2 3 1 2 3 4

```

### 输出

```
2
3 4 
```

## 样例 #2

### 输入

```
4
1 1 1 1

```

### 输出

```
4
1 1 1 1 
```

## 样例 #3

### 输入

```
5
1 2 3 4 5

```

### 输出

```
1
5 
```

## 样例 #4

### 输入

```
5
1 2 1 2 1

```

### 输出

```
3
2 2 1 
```

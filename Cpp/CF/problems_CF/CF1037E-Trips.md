# Trips

## 题目描述

**题目大意：**

一共有$n$个人，他们开始互不认识，而每天早上不认识的两个人会变成朋友。一共有$m$天，每天晚上有的人要去旅行，去旅行的人必须满足ta有至少$k$个朋友也去旅行

求每天去旅行的最大人数

## 输入格式

第一行3个整数，表示$n,m,k$

往下m行，每行两个整数$x,y$，表示这天早上$x$和$y$会变成朋友

## 输出格式

共$m$行，每行一个整数，表示每天晚上最多能去旅游的人数

## 样例 #1

### 输入

```
4 4 2
2 3
1 2
1 3
1 4

```

### 输出

```
0
0
3
3

```

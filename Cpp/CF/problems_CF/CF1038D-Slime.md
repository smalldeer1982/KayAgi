# Slime

## 题目描述

### 题目大意：
有$n$只史莱姆qwq，每只史莱姆有一个分数，每次一只史莱姆可以吞掉左边的或者右边的相邻史莱姆（要是有的话），然后ta的分数会减去被吞的史莱姆的分数，问最后剩下的史莱姆分数最大为多少

## 输入格式

第一行一个整数$n$

第二行$n$个整数，表示史莱姆的分数

## 输出格式

一个整数，即最大分数

## 样例 #1

### 输入

```
4
2 1 2 1

```

### 输出

```
4
```

## 样例 #2

### 输入

```
5
0 -1 -1 -1 -1

```

### 输出

```
4
```

# Coffee Break

## 题目描述

### 题目大意：
给定$n$个数和一个$k$，这$n$个数都不超过$m$

每次从没被去掉的数里面选一个数$a$，去掉$a$，然后可以任意一个$b(b>a+k)$，然后去掉任意一个$c(c>b+k)$，以此类推

问最少能选多少个$a$，然后输出每个数都是选第几个$a$的时候被去掉的

## 输入格式

一行三个整数$n,m,k$

再一行$n$个整数，表示给定的数

## 输出格式

第一行一个整数，表示最少选$a$的个数

第二行$n$个整数，表示每个数都是选第几个$a$时被去掉的

## 样例 #1

### 输入

```
4 5 3
3 5 1 2

```

### 输出

```
3
3 1 1 2 

```

## 样例 #2

### 输入

```
10 10 1
10 5 7 4 6 3 2 1 9 8

```

### 输出

```
2
2 1 1 2 2 1 2 1 1 2 

```

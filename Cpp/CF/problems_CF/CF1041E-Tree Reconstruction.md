# Tree Reconstruction

## 题目描述

### 题目大意：
有一棵树，现在给你每条树边被去掉时，形成的两个联通块中点的最大的编号分别是多少，问满足条件的树存不存在

## 输入格式

第一行一个整数$n$

往下$n-1$行，每行两个整数，表示去掉这条边后两个联通块中点的最大的编号是多少

## 输出格式

如果不存在满足条件的树，输出"NO"

否则输出"YES"，下面$n-1$行每行描述一条树边（两端点的编号）

## 样例 #1

### 输入

```
4
3 4
1 4
3 4

```

### 输出

```
YES
1 3
3 2
2 4

```

## 样例 #2

### 输入

```
3
1 3
1 3

```

### 输出

```
NO

```

## 样例 #3

### 输入

```
3
1 2
2 3

```

### 输出

```
NO

```

# Equations of Mathematical Magic

## 题目描述

给出 $ a $，判断方程 $ a-(a \oplus x)-x=0 $ 的解的个数。（其中 $ \oplus $ 为异或）

## 输入格式

第一行是数据组数 $ t $，接下来 $ t $ 行每行输入一个整数 $ a $。

## 输出格式

对于每组数据，输出对应的答案。

## 样例 #1

### 输入

```
3
0
2
1073741823

```

### 输出

```
1
2
1073741824

```

# Binary Numbers AND Sum

## 题目描述

## 题目大意

现在，给你两个位数为 $n$ 和 $m$ 的两个二进制数$a$,$b$，现在，我们要进行如下操作：

* 计算$a$&$b$
* 答案累加上一个操作的值
* $b$右移一位，最后一位直接舍弃

现在,请你算出最终的答案，并输出，答案对998244353取模

## 输入格式

第一行，两个整数$n$,$m$,$(1≤n,m≤2 \times 10^5)$

第一行，一个长度为$n$的二进制数$a$

第一行，一个长度为$m$的二进制数$b$

## 输出格式

一行，一个数，表示答案

## 样例 #1

### 输入

```
4 4
1010
1101

```

### 输出

```
12

```

## 样例 #2

### 输入

```
4 5
1001
10101

```

### 输出

```
11

```

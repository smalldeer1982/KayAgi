# Card Game

## 题目描述

一种纸牌游戏。
有许多张卡牌，每张卡牌有一个套装（“S”，“H”，“D”和“C”）和一个等级（按升序排列“6”，“7”，“8”，“9”，“T”， “J”，“Q”，“K”和“A”）
**规则：**
以下情况卡牌A才能赢卡牌B:
1、A有王牌套装，B没有
2、A和B的套装相同，但A的等级比B高
否则B赢

## 输入格式

第一行 一个字母（代表王牌套装）
第二行 卡牌A 卡牌B

## 输出格式

卡牌A能赢卡牌B输出"YES"，否则输出"NO".

## 样例 #1

### 输入

```
H
QH 9S

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
S
8D 6D

```

### 输出

```
YES
```

## 样例 #3

### 输入

```
C
7H AS

```

### 输出

```
NO
```

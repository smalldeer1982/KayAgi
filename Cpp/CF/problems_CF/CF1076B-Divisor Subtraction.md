# Divisor Subtraction

## 题目描述

给定一个整数$n$，按照如下算法进行操作  
1. 如果$n=0$，结束算法；  
1. 找到$n$的最小质因子$d$；  
1. $n-=d$并回到操作$1$

## 输入格式

一行一个整数$n(2\leq n\leq 10^{10})$

## 输出格式

输出一个整数——该算法进行循环操作的次数

## 样例 #1

### 输入

```
5

```

### 输出

```
1

```

## 样例 #2

### 输入

```
4

```

### 输出

```
2

```

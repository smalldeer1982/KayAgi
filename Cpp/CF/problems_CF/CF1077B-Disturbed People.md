# Disturbed People

## 题目描述

对于一个给定的长度为$n$的$01$序列$a_1,a_2,\cdots,a_n$  
如果存在$1<i<n$满足$a_{i-1}=a_{i+1}=1,a_i=0$，那么这个序列是不优美的  
求最少需要将多少个$1$变为$0$使得原序列变为优美的序列

## 输入格式

第一行一个整数$n(3\leq n\leq100)$   
第二行$n$个整数$a_1,a_2,\cdots,a_n(a_i\in{0,1})$表示给定序列

## 输出格式

输出一个整数表示答案

## 样例 #1

### 输入

```
10
1 1 0 1 1 0 1 0 1 0

```

### 输出

```
2

```

## 样例 #2

### 输入

```
5
1 1 0 0 0

```

### 输出

```
0

```

## 样例 #3

### 输入

```
4
1 1 1 1

```

### 输出

```
0

```

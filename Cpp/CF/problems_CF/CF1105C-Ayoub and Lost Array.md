# Ayoub and Lost Array

## 题目描述

已知有一个长度为$n$($1\leq n\leq 2 \times 10^5$)的数列，每一个数的大小在$[l,r]$($1\leq l \leq r \leq 10^9$)之间。求出有多少种方案使得这个数列的和为$3$的倍数。答案对$10^9+7$取模。当然，如果无法组成任何一个合法的数列，答案即为$0$。

## 输入格式

一行三个正整数$n$,$l$,$r$

## 输出格式

一行，表示方案数模$10^9+7$的结果。

## 样例 #1

### 输入

```
2 1 3

```

### 输出

```
3

```

## 样例 #2

### 输入

```
3 2 2

```

### 输出

```
1

```

## 样例 #3

### 输入

```
9 9 99

```

### 输出

```
711426616

```

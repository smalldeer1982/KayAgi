# Double Happiness

## 题目描述

给定闭区间 $ [l,r] $，找出区间内满足 $  t=a^{2}+b^{2} $ 的所有素数 $ t $ 的个数（$ a,b $ 为任意正整数）。

## 输入格式

输入只有一行
给定两个正整数 $ l,r $ 表示区间范围 。

## 输出格式

输出一个整数，表示共有多少个素数 $t$。

## 样例 #1

### 输入

```
3 5

```

### 输出

```
1

```

## 样例 #2

### 输入

```
6 66

```

### 输出

```
7

```

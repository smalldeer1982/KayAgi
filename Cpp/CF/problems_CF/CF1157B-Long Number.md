# Long Number

## 题目描述

给定一个一个n位数，以及1~9数字的替换数。现请你在这个n位数上选一个连续的数段，将段内的每个数字替换成相应数字的替换数。要求：输出按照要求替换后的最大n位整数。

## 输入格式

第一行一个整数n$ ( 1 \le n \le 2 \cdot 10^5 ) $

第二行一个n位整数

第三行为9个替换数(一位正整数)，第i个为数字i的替换数

## 输出格式

一个n位整数，即按照要求替换后你的最大结果

## 样例 #1

### 输入

```
4
1337
1 2 5 4 6 6 3 1 9

```

### 输出

```
1557

```

## 样例 #2

### 输入

```
5
11111
9 8 7 6 5 4 3 2 1

```

### 输出

```
99999

```

## 样例 #3

### 输入

```
2
33
1 1 1 1 1 1 1 1 1

```

### 输出

```
33

```

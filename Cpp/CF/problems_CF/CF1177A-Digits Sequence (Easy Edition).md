# Digits Sequence (Easy Edition)

## 题目描述

序列123456789101112131415161718192021222324252627282930313233343536...是无穷无尽的,现在你要输出它的第k项。

## 输入格式

一个整数k。

## 样例 #1

### 输入

```
7

```

### 输出

```
7

```

## 样例 #2

### 输入

```
21

```

### 输出

```
5

```

# Vus the Cossack and a Contest

## 题目描述

## 题意简述

Vus the Cossack举办了一场比赛。这场比赛有$n$个人参加。Vus the Cossack为他们准备了奖品——$m$只笔和$k$个笔记本。问是否每一个人都可以获得至少一支笔和一个笔记本。

## 输入格式

$n,m,k(1≤n,m,k≤100)$。

## 输出格式

如果每个人都可以获得至少一支笔和一个笔记本，请输出Yes，反之输出No。

## 样例 #1

### 输入

```
5 8 6

```

### 输出

```
Yes

```

## 样例 #2

### 输入

```
3 9 3

```

### 输出

```
Yes

```

## 样例 #3

### 输入

```
8 5 20

```

### 输出

```
No

```

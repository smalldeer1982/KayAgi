# Increasing Sequence

## 题目描述

### 题面描述

把数列中一个数加上 $d$ 算作一次操作，求最少几次操作能把这个数列变为递增数列。

## 输入格式

输入共 $2$ 行。

第一行 $2$ 个整数 $n$ 和 $d$，$n$ 表示数列中数的数量，$d$ 含义如题目所示。

第二行 $n$ 个整数，表示原始数列。

## 输出格式

一行一个整数 $m$，表示最少 $m$ 次操作能把这个数列变为递增数列。

## 样例 #1

### 输入

```
4 2
1 3 3 2

```

### 输出

```
3

```

# Uniqueness

## 题目描述

给你一个n(1≤n≤2000)个数的序列A1,A2...An(1≤Ai≤1e9)，我们要删除一个子串，使得数列中余下的数两两不重复，问删除字串的最短长度。

## 输入格式

第一行一个整数n;

第二行n个整数，表示A数列。

## 输出格式

一行一个整数，为最小删除长度。

## 样例 #1

### 输入

```
3
1 2 3

```

### 输出

```
0

```

## 样例 #2

### 输入

```
4
1 1 2 2

```

### 输出

```
2

```

## 样例 #3

### 输入

```
5
1 4 1 4 9

```

### 输出

```
2

```

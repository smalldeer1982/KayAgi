# Ice Cream

## 题目描述

Tanya 想在 $n$ 天内吃掉 $k$ 份冰淇凌，在第 $i$ 天她至少吃 $a_i$ 份，至多吃 $b_i$ 份，冰淇凌的价格为 $c_i$，当天买必须当天吃完，求最少的花费。

## 输入格式

第一行两个整数 $n$ 和 $k$.

接下来 $n$ 行每行三个整数 $a_i$, $b_i$, $c_i$.

## 输出格式

输出最小花费。如果无解输出 -1.

（请使用 Kotlin 语言）

## 样例 #1

### 输入

```
3 7
3 5 6
0 3 4
3 3 3

```

### 输出

```
31

```

## 样例 #2

### 输入

```
1 45000
40000 50000 100000

```

### 输出

```
4500000000

```

## 样例 #3

### 输入

```
3 100
2 10 50
50 60 16
20 21 25

```

### 输出

```
-1

```

## 样例 #4

### 输入

```
4 12
2 5 1
1 2 2
2 3 7
3 10 4

```

### 输出

```
35

```

# Marbles

## 题目描述

## 题目大意

有 n $ (n \le 4 * 10^5) $ 个珠子 ， 第$i$个珠子颜色是$ c_i (c_i \le 20)$ , 每次操作把**相邻**的两个珠子交换。现在要把相同颜色的珠子排列在相连的一段，问至少要多少次操作 。

## 输入格式

第一行：一个数n

第二行：n个数，表示珠子的颜色

## 输出格式

一行：至少交换的次数

## 样例 #1

### 输入

```
7
3 4 2 3 4 2 2

```

### 输出

```
3

```

## 样例 #2

### 输入

```
5
20 1 14 10 2

```

### 输出

```
0

```

## 样例 #3

### 输入

```
13
5 5 4 4 3 5 7 6 5 4 4 6 5

```

### 输出

```
21

```

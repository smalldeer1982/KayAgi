# Equalize Prices Again

## 题目描述

## 题目大意

有n个商品，其中第i个商品的成本为a_i。

现在要求你求出一个平均价格，使得每件商品按照此平均价格卖出后不亏本，而且赚到的钱最少（挣的钱可以为0）

注意题目有多组数据。

## 输入格式

第一行输入数据组数q（1≤q≤100，q为整数）。

接下来q组数据。对于每组数据，先输入商品个数n（1≤n≤100，n为整数），接下来一行包含n个整数：a_1，a_2，... ... ，a_n，其中a_i表示第i个商品的成本。

## 输出格式

对于每组数据，每行输出一个平均价格。

## 样例 #1

### 输入

```
3
5
1 2 3 4 5
3

4
1 1 1 1

```

### 输出

```
3

1
```

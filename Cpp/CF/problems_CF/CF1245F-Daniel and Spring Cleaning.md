# Daniel and Spring Cleaning

## 题目描述

给定$l,r$，求
$$\sum_{a=l}^r\sum_{b=l}^r[a+b=a\oplus b]$$

## 输入格式

第一行$t(1\le t\le100)$，表示数据组数

接下来$t$行，每行$l,r$

## 输出格式

共$t$行，答案为题目所述

## 样例 #1

### 输入

```
3
1 4
323 323
1 1000000

```

### 输出

```
8
0
3439863766

```

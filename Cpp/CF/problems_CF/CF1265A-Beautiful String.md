# Beautiful String

## 题目描述

### 题意简述

给定字符串 $s$，由 `a`，`b`，`c`，`?` 四种字符构成。

您需要将 $s$ 中的所有 `?` 修改为 `a`，`b`，`c` 中的一种，使得 $s$ 相邻两个字符均不相同。

## 输入格式

第一行是一个整数 $t(t≤1000)$，表示数据组数。

对于每组数据，一行一个字符串 $s$，**保证满足上述限制**。

保证 $s$ 的长度之和不超过 $10^5$。

## 输出格式

对于每组数据，输出一行一个字符串，表示修改之后的 $s$。如果有多组解，输出**任意一组**。如果无解，输出 `-1`。

翻译贡献者 U108949

## 样例 #1

### 输入

```
3
a???cb
a??bbc
a?b?c

```

### 输出

```
ababcb
-1
acbac

```

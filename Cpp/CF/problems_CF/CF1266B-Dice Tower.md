# Dice Tower

## 题目描述

鲍勃正在玩 $6$ 面的骰子。他有不限量的这样的骰子。骰子可以搭成骰子塔，骰子塔上可以看见的点子的数量称为骰子塔的点数。

## 输入格式

有两行。第一行包含一个整数 $T$，表示 $T$ 次询问。第二行包含 $T$ 个整数 $x$，表示骰子塔的点数。

## 输出格式

有 $T$ 行，表示每一次询问的结果。如果可以搭成有 $x$ 个点数的骰子塔输出`YES`，否则输出`NO`。

## 样例 #1

### 输入

```
4
29 34 19 38

```

### 输出

```
YES
YES
YES
NO

```

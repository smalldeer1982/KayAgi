# Modulo Equality

## 题目描述

你有一个正整数$m$和两个整数序列:

$a=[a_1, a_2, a_n]$和$b=[b_1,b_2b_n]$。

这两个序列的长度都是$n$。

然后将$a$序列中的数加上一个数$x$，$mod \ m$之后得到的序列改变或者不改变顺序与$b$序列相等.

## 输入格式

第一行 $n, m(1≤n≤2000,1≤m≤10^9)$

第二行 n个整数表示$a_1...a_i...a_n (0\leq a_i<m)$

第三行 n个整数表示$b_1...b_i...b_n (0\leq b_i <m)$

## 输出格式

一个数，最小的x

## 样例 #1

### 输入

```
4 3
0 0 2 1
2 0 1 1

```

### 输出

```
1

```

## 样例 #2

### 输入

```
3 2
0 0 0
1 1 1

```

### 输出

```
1

```

## 样例 #3

### 输入

```
5 10
0 0 0 1 2
2 1 0 0 0

```

### 输出

```
0

```

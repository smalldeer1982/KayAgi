# Скрытый друг

## 题目描述

## 题面
```
对于给定的朋友图 找到隐藏的友谊 即用户 a 是与用户 b 的朋友，但用户 b 不是与用户 a 的朋友的情况。
```

------------

## 输入格式

第一行包含一个整数 $n$   ( $1 < n \le 1000 $) 
指朋友圈人数 $ 1 至 n $ 。


接下来的 $n$ 行显示每个人的朋友图： 

第 $ i $ 行中，首先给出第 $i$ 人的朋友数 及 其朋友列表 (以空格分隔）。

朋友编号都不会重复。

## 输出格式

在第一行中，输出数字 $k$ 表示 隐藏的朋友关系的数量

在接下来的$ k $ 行中，输出2个成对的数字$ u $ , $v$     表示用户 $ u $ 对朋友隐藏了用户 $v$ 。

## 样例 #1

### 输入

```
5
3 2 3 4
4 1 3 4 5
0
2 1 2
3 4 3 1

```

### 输出

```
6
3 5
4 5
5 2
3 1
1 5
3 2

```

## 样例 #2

### 输入

```
2
0
1 1

```

### 输出

```
1
1 2

```

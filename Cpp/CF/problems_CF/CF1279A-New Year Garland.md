# New Year Garland

## 题目描述

### 题意简述

给定 $r$ 个字符 R，$g$ 个字符 G，$b$ 个字符 B，问是否存在一种方案，使得所有字符排列组成的字符串不存在两个相邻且相同的字符。

## 输入格式

**本题有多组数据**。

第一行一个正整数 $t(1\leq t \leq 100)$，表示数据组数。

对于每组数据，一行三个整数 $r,g,b(1\leq r,g,b \leq 10^9)$。

## 输出格式

对于每组数据，如果方案存在，输出字符串 `Yes`。否则输出 `No`。

翻译贡献者 U108949

## 样例 #1

### 输入

```
3
3 3 3
1 10 2
2 1 1

```

### 输出

```
Yes
No
Yes

```

# Elementary!

## 输入格式

输入一个仅包含大写字母的字符串，长度小于10个字符。

## 输出格式

输出"YES"或"NO"。

## 样例 #1

### 输入

```
GENIUS
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
DOCTOR
```

### 输出

```
NO
```

# Most Unstable Array

## 题目描述

给定两个整数 $n$ 和 $m$ ，构造一个长度为 $n$ 且每个元素均为非负整数的数组 $a$ ，使得数组内每个元素的和为 $m$ ，且 $\sum \limits_{i=1}^{n-1}|a_i-a_{i+1}|$ 的值尽可能大，输出这个最大值。

## 输入格式

第一行一个整数 $t$ ，接下来 $t$ 行，每行两个整数 $n,m$ 。

## 输出格式

对于每组数据，输出题目要求的最大值。

## 样例 #1

### 输入

```
5
1 100
2 2
5 5
2 1000000000
1000000000 1000000000
```

### 输出

```
0
2
10
1000000000
2000000000
```

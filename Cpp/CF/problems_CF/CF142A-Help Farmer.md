# Help Farmer

## 题目描述

原本有A×B×C块干草，被偷走若干块后，剩下N块，刚好等于(A−1)×(B−2)×(C−2)。现在已知N的值，不知道ABC的值，求被偷走的干草的最小值和最大值。

## 输入格式

第一行，一个整数n。

## 输出格式

第一行，两个整数，分别表示最小值和最大值。

感谢@xukuan 提供的翻译

## 样例 #1

### 输入

```
4

```

### 输出

```
28 41

```

## 样例 #2

### 输入

```
7

```

### 输出

```
47 65

```

## 样例 #3

### 输入

```
12

```

### 输出

```
48 105

```

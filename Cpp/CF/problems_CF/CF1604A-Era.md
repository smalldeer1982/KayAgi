# Era

## 题目描述

给一个长度为 $n$ 的序列 $a_1,a_2,\dots,a_n$，每次可以往序列中插入任意个整数，求最少插入多少个整数时 $\forall i,a_i\le i$。

## 输入格式

第一行输入一个正整数 $t$ 表示数据组数。

每组数据输入两行，第一行为一个正整数 $n$ 表示一个序列，第二行 $n$ 个正整数表示序列 $a$。

## 输出格式

对于每组数据输出一行一个整数表示答案。

## 说明/提示

$1\le t\le200,1\le n\le100,1\le a_i\le10^9$。

## 样例 #1

### 输入

```
4
3
1 3 4
5
1 2 5 7 4
1
1
3
69 6969 696969
```

### 输出

```
1
3
0
696966
```

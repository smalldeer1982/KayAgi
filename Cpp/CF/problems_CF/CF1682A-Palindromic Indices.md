# Palindromic Indices

## 题目描述

给定一个回文字符串 $s$ ，它的长度为 $ n $ $ (2 \leq n \leq 10^5) $ ，问一共有多少种方式使它去掉一个字符后仍是回文字符串。

## 输入格式

共有 $ t $  $ (1 \leq t \leq 10^3) $ 组数据，对于每组数据，先输入字符串的长度 $ n $ ，再输入回文字符串 $ s $ 。

## 输出格式

对于每组数据，输出一共有多少种方式使输入的字符串去掉一个字符后仍是回文字符串。

## 说明/提示

数据保证所有的 $ n $ 之和不超过 $ 2 \cdot 10^5 $ 。

## 样例 #1

### 输入

```
3
3
aba
8
acaaaaca
2
dd
```

### 输出

```
1
4
2
```

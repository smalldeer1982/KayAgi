# Creep

## 题目描述

请你构造出一个有 $a$ 个 $0$ 和 $b$ 个 $1$ 的字符串，使它的 $0$ 和 $1$ 的个数相差最多的前缀，$0$ 和 $1$ 的个数相差最少。若有多种答案，输出任意一种。多组数据。

## 输入格式

先输入数据组数 $t$，然后 $t$ 行，每行两个整数 $a,b$，表示一组数据。

## 输出格式

输出 $t$ 行，每行一个字符串，表示一组数据的答案。

## 说明/提示

$1\leq t\leq 1000$

$1\leq a,b\leq 100$

Translate by [庄nnnn额](https://www.luogu.com.cn/user/521592)

## 样例 #1

### 输入

```
5
1 1
1 2
5 2
4 5
3 7
```

### 输出

```
10
011
0011000
101010101
0001111111
```

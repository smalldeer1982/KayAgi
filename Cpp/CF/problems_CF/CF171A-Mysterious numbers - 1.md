# Mysterious numbers - 1

## 输入格式

输入包括两个整数 $a_1,a_2(0\le a_i\le 10^9)$，两个整数被空格隔开。

## 输出格式

输出一个整数。

## 样例 #1

### 输入

```
3 14

```

### 输出

```
44

```

## 样例 #2

### 输入

```
27 12

```

### 输出

```
48

```

## 样例 #3

### 输入

```
100 200

```

### 输出

```
102

```

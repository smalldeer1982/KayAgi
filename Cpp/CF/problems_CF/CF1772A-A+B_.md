# A+B?

## 题目描述

有一个算式 $a{+}b$ ，$a$ 、$b$ 都是一位数。你需要输出这个算式的答案。

## 输入格式

第一行，一个整数 $t$ ( $1\le t\le100$ ) 表示输入的算式个数。

下面 $t$ 行，每行一个 $a{+}b$ 的算式，注意加号旁边没有空格。( $0\le a,b\le9$，$a$ 和 $b$ 都是整数 )

## 输出格式

对于每个算式，输出这个算式的结果。

## 样例 #1

### 输入

```
4
4+2
0+0
3+7
8+9
```

### 输出

```
6
0
10
17
```

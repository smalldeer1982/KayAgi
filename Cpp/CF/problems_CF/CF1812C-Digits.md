# Digits

## 题目描述

无

## 输入格式

输入的第一行为一个整数 $ t $ ( $ 1 \le t \le 32 $ )，表示数据组数。

接下来每组数据 $n$ 行，第 $i$ 行为一个整数 $ a_i $ ( $ 1 \le a_i \le 9 $ ).

所有组的 n 之和不超过 $155$。

## 输出格式

对每组数据，输出一个整数表示答案。

## 样例 #1

### 样例输入 #1

```
3
2
1
4
7
1
2
3
5
```

### 样例输出 #1

```
8
7
30
```

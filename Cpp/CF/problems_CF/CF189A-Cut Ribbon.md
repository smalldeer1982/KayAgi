# Cut Ribbon

## 题目描述

给一长度为n的缎带，要求将其剪成若干长度为a,b,c的缎带，且缎带数量尽可能多。

## 输入格式

输入仅一行，四个正整数n,a,b,c(n,a,b,c≤4000)。

## 输出格式

输出仅一行，即缎带数量的最大值。

## 样例 #1

### 输入

```
5 5 3 2

```

### 输出

```
2

```

## 样例 #2

### 输入

```
7 5 5 2

```

### 输出

```
2

```

# X Axis

## 题目描述

你得到了三个点 $x_1$，$x_2$ 和 $x_3$。 你需要求出一个点使得 $x_1$，$x_2$，$x_3$ 与这个点的距离和最小。

## 输入格式

第一行为一个整数 $t$，表示测试数据的组数。

接下来 $t$ 行，每行三个整数 $x_1$，$x_2$ 和 $x_3$。

## 输出格式

输出共 $t$ 行，每行一个整数表示与给出的三个点最小的距离和。

## 说明/提示

$1 \le t \le 10^3$，$1 \le x_i \le 10$。

### 输入

```
8

```

### 输出

```
0
8
6
7
1
3
4
8
```

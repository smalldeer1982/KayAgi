# Simple Palindrome

## 题目描述

构造一个 **长度为** $n$ ，且只由 $a$，$e$，$i$，$o$，$u$ **这 $5$ 种字母** 构成字符串，使得这个字符串的 **所有子序列** 是 **回文串** 的个数最少。请输出这个 **最小值** 。

## 输入格式

第一行一个正整数 $T$ ，表示有 $T$ 组数据。
 接下来 共 $T$ 行，每行一个 **正整数** $n$ ，需要构造的 **字符串的长度** 。

## 输出格式

共 $T$ 行，每行一个 **正整数** ，表示答案的最小值。

## 说明/提示

对于 $100\%$ 的测试点，$1\le T\le 100$， $1\le n\le 100$ 。

## 样例 #1

### 输入

```
3
2
3
6
```

### 输出

```
uo
iae
oeiiua
```

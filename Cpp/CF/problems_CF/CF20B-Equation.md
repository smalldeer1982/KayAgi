# Equation

## 题目描述

### 题目大意

给定 $A, B, C$，求方程 $Ax^2+Bx+C=0$ 的解的个数，如果有无穷个解，则输出 `-1`。

## 输入格式

第一行：三个整数表示上式的 $A, B, C$。

## 输出格式

第一行：一个整数，表示解的个数，如果有无数个解，请输出 `-1`。

接下来若干行：每行一个解，请按大小顺序输出。

## 样例 #1

### 输入

```
1 -5 6

```

### 输出

```
2
2.0000000000
3.0000000000
```

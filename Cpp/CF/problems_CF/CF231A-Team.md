# Team

## 题目描述

## 题目要求
三位选手要去参加编程竞赛，比赛共有N道题目，他们商定: 对于每一道题目，只有两位及以上的选手确认找到解法时才做。
试问，他们一共能做多少题?

## 输入格式

第一行一个整数N，表示题目的数量。
第二行到第N+1行，每行三个整数(0或者1)，其中0表示这位选手无法解出此题，1表示这位选手想到了解法。

## 输出格式

一行，一个整数，表示他们所能做的题目的数量。

感谢@PC_DOS 提供翻译

## 样例 #1

### 输入

```
3
1 1 0
1 1 1
1 0 0

```

### 输出

```
2

```

## 样例 #2

### 输入

```
2
1 0 0
0 1 1

```

### 输出

```
1

```

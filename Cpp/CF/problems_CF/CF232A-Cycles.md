# Cycles

## 题目描述

构造一个无向图（没有自环），使这个无向图恰好有 $m$ 个三元环，输出这个无向图的 $01$ 矩阵。

$($无向图的顶点数不超过 $100,1 \leq m \leq 10^5)$

## 输入格式

一行一个整数 $m$ 。

## 输出格式

第一行为无向图的顶点数 $n$，接下来 $n$ 行为这个无向图的 $01$ 矩阵，不需要输出空格。

## 样例 #1

### 输入

```
1

```

### 输出

```
3
011
101
110

```

## 样例 #2

### 输入

```
10

```

### 输出

```
5
01111
10111
11011
11101
11110

```

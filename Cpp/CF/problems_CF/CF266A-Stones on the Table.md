# Stones on the Table

## 题目描述

桌子上有N个石头，每一块都可以是红的、绿的或蓝的。计算从桌子上取下的石头的最小数量，以便任何两块相邻的石头都有不同的颜色。如果没有其他石头，一排石头被认为是相邻的。

## 输入格式

第一行是一个整数n（1<=n<=50）--桌上石头的数量

第二行是一个字符串s，代表着石头的颜色，石头的排法为从1到n，从左到右。s中的第i个字母如果是"R",那么第i个石头就是红色，"G"是绿色，"B"是蓝色

## 输出格式

输出一个整数——这个问题的答案

感谢U91783提供的翻译

## 样例 #1

### 输入

```
3
RRG

```

### 输出

```
1

```

## 样例 #2

### 输入

```
5
RRRRR

```

### 输出

```
4

```

## 样例 #3

### 输入

```
4
BRBG

```

### 输出

```
0

```

# Lunch Rush

## 题目描述

$n$ 个餐馆，每个餐馆都有两个值 $f_i$，$t_i$，分别代表在该餐馆的快乐值和所要停留的时间，给出一个正整数 $k$，如果 $t_i>k$，则在该餐馆的快乐值为 $f_i-(t_i-k)$，否则，仍为 $f_i$，选择一个餐馆，使获得的快乐值最大。

## 输入格式

第一行两个正数 $n$，$k$。

接下来 $n$ 行每行两个数 $f_i$，$t_i$。

## 输出格式

一个数，表示最大快乐值。

## 说明/提示

$1\leq n \leq 10^4$，$1\leq k,f_i,t_i \leq 10^9$。

## 样例 #1

### 输入

```
2 5
3 3
4 5

```

### 输出

```
4

```

## 样例 #2

### 输入

```
4 6
5 8
3 6
2 3
2 2

```

### 输出

```
3

```

## 样例 #3

### 输入

```
1 5
1 7

```

### 输出

```
-1

```

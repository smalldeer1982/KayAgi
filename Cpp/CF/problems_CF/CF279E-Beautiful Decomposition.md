# Beautiful Decomposition

## 题目描述

给定一个长度不超过 $10^6$
  的二进制数 $s$，求至少要用几个 $2^k$ 和 $-2^k(k\ge0)$ 加起来值等于 $n$。

## 输入格式

仅一行包含字符串 $s(1\le|s|\le10^6)$。

## 输出格式

输出一个整数表示最少的次数

by [@_I°Fear](https://www.luogu.com.cn/user/326452)

## 样例 #1

### 输入

```
10

```

### 输出

```
1

```

## 样例 #2

### 输入

```
111

```

### 输出

```
2

```

## 样例 #3

### 输入

```
1101101

```

### 输出

```
4

```

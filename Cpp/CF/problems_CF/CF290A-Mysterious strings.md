# Mysterious strings

## 输入格式

The input contains a single integer $ a $ ( $ 1<=a<=40 $ ).

## 输出格式

Output a single string.

## 样例 #1

### 输入

```
2

```

### 输出

```
Adams

```

## 样例 #2

### 输入

```
8

```

### 输出

```
Van Buren

```

## 样例 #3

### 输入

```
29

```

### 输出

```
Harding

```

# The least round way

## 题目描述

给定由非负整数组成的 $n\times n$ 的正方形矩阵，你需要寻找一条路径：

+ 以左上角为起点。
+ 每次只能向右或向下走。
+ 以右下角为终点。
+ 如果我们把沿路遇到的数进行相乘，积应当以尽可能少的 $0$ 结尾。

## 输入格式

第一行包含一个整数 $n (2 \leq n \leq 1000)$，$n$ 为矩阵的规模，接下来的 $n$ 行包含矩阵的元素（不超过 $10^9$ 的非负整数）。

## 输出格式

第一行应包含结尾最少的 $0$ 的个数，第二行打印出相应的路径（译注：`D` 为下，`R`  为右）。

## 样例 #1

### 输入

```
3
1 2 3
4 5 6
7 8 9

```

### 输出

```
0
DDRR

```

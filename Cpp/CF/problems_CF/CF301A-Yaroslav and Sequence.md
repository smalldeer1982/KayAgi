# Yaroslav and Sequence

## 题目描述

现有 $2n-1$ 个整数，每次操作可以使其中的 $n$ 个数 $\times (-1)$。

现在请问进行任意次操作后，这些整数的最大和是多少？

---

## 输入格式

输入共两行。第一行包含一个整数 $n$。

第二行包含 $2n-1$ 个数，表示所给的整数。

---

## 输出格式

输出一个整数，表示最大和。

---

## 说明/提示

对于 $100\%$ 的数据，$1\le n\le100,1\le|x|\le1000$

## 样例 #1

### 输入

```
2
50 50 50

```

### 输出

```
150

```

## 样例 #2

### 输入

```
2
-1 -100 -1

```

### 输出

```
100

```

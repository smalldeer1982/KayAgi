# Borze

## 题目描述

### 题面描述

三进制数字符号在Berland很受欢迎。如果用borze编码表示电报的三进制数。数字 $0,1,2$ 分别被作为`.`，`-.`和`--`。你需要为borze编码解码。（把borze编码转换为三进制数）。

## 输入格式

第一行包含在Borze编码。字符串的长度介于 $1$ 到 $200$ 个字符之间。这是保证给定的字符串是一个有效的一些三元数borze编码（这个数可能有前导零）。

## 输出格式

一个三进制数（如果有前导零要输出）。

## 样例 #1

### 输入

```
.-.--

```

### 输出

```
012
```

## 样例 #2

### 输入

```
--.

```

### 输出

```
20
```

## 样例 #3

### 输入

```
-..-.--

```

### 输出

```
1012
```

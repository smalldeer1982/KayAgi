# Group of Students

## 题目描述

有一些学生考了试，他们在考试中获得了$1$~$m$的分数，获得 i 分的有 $C_i$人。\
你需要求出一个分数线，满足低于分数线的人数和高于分数线(含)的人数均在$[x,y]$范围之内

## 输入格式

------------
第一行一个整数，$m$\
第二行$m$个整数，为$C_1$~$C_m$
第三行两个整数$x,y$

## 输出格式

------------
输出分数线，特别的，若没有满足要求的分数线，输出0

## 样例 #1

### 输入

```
5
3 4 3 2 1
6 8

```

### 输出

```
3

```

## 样例 #2

### 输入

```
5
0 3 3 4 2
3 10

```

### 输出

```
4

```

## 样例 #3

### 输入

```
2
2 5
3 6

```

### 输出

```
0

```

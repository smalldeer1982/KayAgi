# Counting Rectangles is Fun

## 题目描述

给定一个${n * m}$的01矩阵, q次询问, 每次询问指定一个子矩形, 求该子矩形种有多少个只包含0的子矩阵.

矩阵从上到下编号1~n, 从左到右编号1~m.

## 输入格式

第一行三个整数n,m,q.
接下来n行, 每行m个整数描述该矩阵.
接下来q行, 每行四个整数a,b,c,d表示询问的子矩阵左上角为(a,b), 右下角为(c,d).

## 输出格式

对于每个询问, 输出一个整数表示答案.

## 样例 #1

### 输入

```
5 5 5
00101
00000
00001
01000
00001
1 2 2 4
4 5 4 5
1 2 5 2
2 2 4 5
4 2 5 3

```

### 输出

```
10
1
7
34
5

```

## 样例 #2

### 输入

```
4 7 5
0000100
0000010
0011000
0000000
1 7 2 7
3 1 3 1
2 3 4 5
1 2 2 7
2 2 4 7

```

### 输出

```
3
1
16
27
52

```

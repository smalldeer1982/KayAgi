# Bear and Raspberry

## 题目描述

给定一个序列，请求出相邻两个数之间（前数$-$后数）差的最大值与$c$的差。

如果结果小于$0$的话就输出$0$。

## 输入格式

第一行输入$n$和$c$。

接下来一行$n$个数字，描述整个序列。

## 输出格式

输出一行表示结果。

## 样例 #1

### 输入

```
5 1
5 10 7 3 20

```

### 输出

```
3

```

## 样例 #2

### 输入

```
6 2
100 1 10 40 10 40

```

### 输出

```
97

```

## 样例 #3

### 输入

```
3 0
1 2 3

```

### 输出

```
0

```

# On Sum of Fractions

## 题目描述

我们设：

- $v(n)$是不超过$n$的最大素数；

- $u(n)$是大于$n$的最小素数。

计算![](https://cdn.luogu.org/upload/vjudge_pic/CF396B/35810ad29d9f6c4fb0ebb0082bbbd3e3b2a406a1.png)。

## 输入格式

第一行一个整数$t(1<=t<=500)$表示数据组数。

接下来$t$行每行一个整数$n(1<=n<=10^9)$。

## 输出格式

输出$t$行，第$i$行一个最简分数“$p/q$”表示第$i$组数据的答案，$p,q$是整数。

## 样例 #1

### 输入

```
2
2
3

```

### 输出

```
1/6
7/30

```

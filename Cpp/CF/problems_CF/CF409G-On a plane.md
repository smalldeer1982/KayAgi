# On a plane

## 题目描述

### 题目背景
CodeForces 2013年愚人节题目。

### 题目要求
给出一系列坐标(X,Y)，求Y的平均值与5的和。

## 输入格式

第一行一个整数n(1<=<=1000)，第2到第n+1行，每行两个浮点数X和Y(-1000<=X,Y<=1000)。

## 输出格式

1行，1个单精度实数θ，θ=5+Average(Y)，θ和标准答案的差不应超过0.01。

## 样例 #1

### 输入

```
8
-2.14 2.06
-1.14 2.04
-2.16 1.46
-2.14 0.70
-1.42 0.40
-0.94 -0.48
-1.42 -1.28
-2.16 -1.62

```

### 输出

```
5.410

```

# Sereja and Mugs

## 题目描述

# 题意

给定 $n$ 个数和 $s$，求其中最小的 $n-1$ 个数加起来是否超过 $s$

## 输入格式

第一行两个正整数 $n$ 和 $s$。

接下来一行 $n$ 个整数表示这 $n$ 个数字。

## 输出格式

如果不超过 $s$，则输出 ```YES```，反之输出 ```NO```。

## 样例 #1

### 输入

```
3 4
1 1 1

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
3 4
3 1 3

```

### 输出

```
YES

```

## 样例 #3

### 输入

```
3 4
4 4 4

```

### 输出

```
NO

```

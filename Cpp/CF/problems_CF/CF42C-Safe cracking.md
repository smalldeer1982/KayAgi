# Safe cracking

## 输入格式

一行，四个用空格分开的正整数，分别代表四个在圆上的数字，这四个数都不大于10^9。

## 输出格式

如果有多于一个解法，输出其中任意一个即可。

## 样例 #1

### 输入

```
1 2 4 2

```

### 输出

```
/2
/3

```

## 样例 #2

### 输入

```
3 3 1 1

```

### 输出

```
+1
/1
/1

```

## 样例 #3

### 输入

```
2 1 2 4

```

### 输出

```
/3
/4

```

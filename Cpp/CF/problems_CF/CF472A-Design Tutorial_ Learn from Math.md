# Design Tutorial: Learn from Math

## 题目描述

输入一个数n(12<=n<=10^6)，找出两个合数x,y，使得x+y=n,如果有多组x,y,输出任意一组即可

## 样例 #1

### 输入

```
12

```

### 输出

```
4 8

```

## 样例 #2

### 输入

```
15

```

### 输出

```
6 9

```

## 样例 #3

### 输入

```
23

```

### 输出

```
8 15

```

## 样例 #4

### 输入

```
1000000

```

### 输出

```
500000 500000

```

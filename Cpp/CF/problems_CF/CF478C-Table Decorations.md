# Table Decorations

## 题目描述

您正在为宴会装饰桌子，有 $r$ 个红色，$g$ 个绿色和 $b$ 蓝色的气球，一张桌子需要三个颜色不完全相同的气球来装饰。

您需要求出可以用所需方式进行装饰的桌子的最大数量 $t$。

## 输入格式

单行包含三个整数 $r,g,b (0<=r,g,b<=2*10^{9})$，表示红色，绿色和蓝色气球的数量。

## 输出格式

输出一个整数 $t$ 表示可以按要求的方式装饰的桌子的最大数量。

## 样例 #1

### 输入

```
5 4 3

```

### 输出

```
4

```

## 样例 #2

### 输入

```
1 1 1

```

### 输出

```
1

```

## 样例 #3

### 输入

```
2 3 3

```

### 输出

```
2

```

# Exams

## 题目描述

学生进行期末考试，每个考试时间在$a_i$，Valera和老师约定提前于时间$b_i$进行考试，要求所有考试按照$a_i$的顺序，求Valera最早哪个时间可以结束考试。

## 输入格式

第一行一个整数，考试数目nnn。接下来n+1行每行两个整数代表$a_i$和$b_i$。

## 输出格式

一个整数，代表最小结束时间。
##### 数据规模
n<=5000,$a_i$、$b_i$在int范围

## 样例 #1

### 输入

```
3
5 2
3 1
4 2

```

### 输出

```
2

```

## 样例 #2

### 输入

```
3
6 1
5 2
4 3

```

### 输出

```
6

```

# Diverse Permutation

## 题目描述

## 题面翻译
构造一个 $n$ 的排列，使得 $|a_i-a_{i+1}|\ (1\leq i<n)$ 恰有 $k$ 个不同的值。

本题有 Special Judge。

## 输入格式

仅一行两个整数 $n,k$。

## 输出格式

符合题意的一个 $n$ 的排列。如有多组答案，输出任意一组。

翻译贡献者：尘染梦

题面修改：[Hszzzx](/user/676412)

## 说明/提示

$1\leq k < n \leq 10^5$.

## 样例 #1

### 输入

```
3 2

```

### 输出

```
1 3 2

```

## 样例 #2

### 输入

```
3 1

```

### 输出

```
1 2 3

```

## 样例 #3

### 输入

```
5 2

```

### 输出

```
1 3 2 4 5

```

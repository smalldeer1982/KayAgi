# Flea travel

## 题目描述

一个虫子位于绕城一圈的$n$片树叶中的一片上，在第$k$分钟，虫子会跳过$k-1$片树叶并在脚下的树叶上做一个记号。在这里，我们假设虫子的$life=+∞$。你的任务是判断该虫子是否将$n$片树叶都做了记号。

## 输入格式

只有一行，一个数字$n$，为树叶总数量。

## 输出格式

若将$n$片树叶都做了记号，输出$YES$。否则，输出$NO$。

### 输入

```
1

```

### 输出

```
YES

3

```

### 输出

NO

# LCS Again

## 题目描述

给定一个字符串 $S$，求有多少个与 $S$ 等长字符串 $T$，满足 $\mathrm{LCS}(S,T)=|S|-1$，其中 $S,T$ 只包含前 $m$ 个小写字母。

## 输入格式

第一行包括两个数字 $n$ 和 $m$，第二行是给定的字符串 $S$。

## 输出格式

输出一行一个整数表示答案。

## 样例 #1

### 输入

```
3 3
aaa

```

### 输出

```
6

```

## 样例 #2

### 输入

```
3 3
aab

```

### 输出

```
11

```

## 样例 #3

### 输入

```
1 2
a

```

### 输出

```
1

```

## 样例 #4

### 输入

```
10 9
abacadefgh

```

### 输出

```
789

```

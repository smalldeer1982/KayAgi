# Kefa and Company

## 题目描述

Kefa 想要去餐馆庆祝一下，他有 $n$ 个朋友，每个朋友有一定的友谊值和工资。没人想觉得自己穷，所以 Kefa 邀请的朋友中两两工资差小于 $d$。现在给出朋友的信息，请求出最大友谊值是多少。

## 输入格式

第一行是 $n$ 和 $d$，有 $n$ 个朋友，以及工资差值小于 $d$。接下来 $n$ 行给定 $n$ 个朋友的信息，第 $i$ 行表示他的工资为$m_i$，友谊值为 $s_i$。

## 输出格式

一个整数，满足要求的前提下可以获得的最大友谊值。

## 样例 #1

### 输入

```
4 5
75 5
0 100
150 20
75 1

```

### 输出

```
100

```

## 样例 #2

### 输入

```
5 100
0 7
11 32
99 10
46 8
87 54

```

### 输出

```
111

```

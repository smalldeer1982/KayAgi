# Duff in Love

## 题目描述

## 问题描述

如果不存在一个数 $ a (a>1)$ ，使得 $ a^2 $ 是 $ n $ 的约数，我们就称 $ n $ 是一个可爱的数。

给出一个正整数 $ n $ ，在 $ n $ 的所有约数中，寻找最大的可爱的数。

## 输入格式

输入一个整数 $ n $ 。

## 输出格式

输出本题的答案。

## 样例 #1

### 输入

```
10

```

### 输出

```
10

```

## 样例 #2

### 输入

```
12

```

### 输出

```
6

```

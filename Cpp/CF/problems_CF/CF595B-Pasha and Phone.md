# Pasha and Phone

## 题目描述

构造长度为 $n$ 的电话号码，满足将其均分成长度为 $k$ 的段后，第 $i$ 段能被 $a_i$ 整除且开头不为 $b_i$ 的个数总和，答案对 $10^9+7$ 取模。

## 输入格式

第一行两个整数 $n,k$。

第二行 $\dfrac{n}{k}$ 个数，表示 $a_i$。

第三行 $\dfrac{n}{k}$ 个数，表示 $b_i$。

## 输出格式

一个整数，电话号码情况总数。

$1<=n<=100000,1<=k<=\min(n,9)$

## 样例 #1

### 输入

```
6 2
38 56 49
7 3 4

```

### 输出

```
8

```

## 样例 #2

### 输入

```
8 2
1 22 3 44
5 4 3 2

```

### 输出

```
32400

```

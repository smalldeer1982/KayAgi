# Chocolate

## 题目描述

## 分割序列

给一个01序列，将之分割成若干段，要求每段恰好只含有1个1，计算分解方案的总数。

## 输入格式

第一行为正整数n(≤100)，第二行为用空格分隔的n个0或1。

## 输出格式

输出分解方案总数。

#### 输入 #1

3

0 1 0

#### 输出 #1

1

#### 输入 #2

5

1 0 1 0 1

#### 输出 #2

4

## 样例 #1

### 输入

```
3
0 1 0

```

### 输出

```
1

```

## 样例 #2

### 输入

```
5
1 0 1 0 1

```

### 输出

```
4

```

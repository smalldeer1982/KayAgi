# Bear and Reverse Radewoosh

## 题目描述

Limak 和 Radewoosh 将要竞争参加一场算法比赛。他们实力相当，但不以同样的顺序答题。

有 $n$ 个问题。第 $i$ 个问题有 $p_i$ 分，需要 $t_i$ 分钟解出。题目按难度排序，也就是说 $p_i<p_{i+1},t_i<t_{i+1}$。

另给出常量 $c$ ，代表分数扣除的速度。在 $x$ 分钟时上传的题目 $i$ （$x$ 指比赛开始的时间），可以得到 $max(0,p_i-c\times x)$ 分。

Limak 按照难度升序解题（从易到难）。Radewoosh 按照难度降序解题（从难到易）。

你的任务是输出赢家的名字，如果是平局则输出 `Tie`。

你可以假设时间大于或等于 $t_i$ 的总和，也就是说他们有时间做完所有的题目。

## 输入格式

第一行包含两个整数 $n$ 和 $c$ ($1\le n\le 50,1\le c\le 1000$)，代表题目数量和扣分速度。

第二行包含 $n$ 个整数$p_1,p_2,p_3,...,p_n$，代表题目分值

第三行包含 $n$ 个整数$t_1,t_2,t_3,...,t_n$，代表题目解答时间

## 输出格式

如果 Limak 赢了，输出`Limak`；

如果 Radewoosh 赢了，输出`Radewoosh`；

如果平局，输出`Tie`。

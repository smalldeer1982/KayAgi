# Mahmoud and a Triangle

## 题目描述

# 题目大意：
给出n个数，若其中3个数能形成一个非退化的三角形则输出‘YES’，否则输出‘NO’（没有引号）

## 输入格式

第一行包含单个整数n（3<=n<=10^5）表示有n条边可供选择
第二行有n个a1,a2,a3...an（1<=ai<=10^9）表示每条边的长度

## 输出格式

一行，如果符合，则输出‘YES’，否则输出‘NO’（没有引号）

## 样例 #1

### 输入

```
5
1 5 3 2 4

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
3
4 1 2

```

### 输出

```
NO

```

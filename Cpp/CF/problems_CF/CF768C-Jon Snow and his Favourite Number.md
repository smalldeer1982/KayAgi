# Jon Snow and his Favourite Number

## 题目描述

题意:

你有一串长度为n的序列a,重复k次操作,问操作后的序列的极值值.
操作:
将序列从小到大排序,从1标号,对序号为奇数的数^(xor)x.

## 输入格式

第一行三个数n,k,x.
接线来一行n个整数,为序列元素.

## 输出格式

一行两个数,第一个为最大值,第二个为最小值.

## 说明/提示

n,k:[1,1e5]
x,a[i]:[1,1e3]

注意:

None.

感谢@尘染梦 提供的翻译

## 样例 #1

### 输入

```
5 1 2
9 7 11 15 5

```

### 输出

```
13 7
```

## 样例 #2

### 输入

```
2 100000 569
605 986

```

### 输出

```
986 605
```

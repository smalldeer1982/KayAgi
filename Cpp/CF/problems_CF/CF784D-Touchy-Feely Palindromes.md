# Touchy-Feely Palindromes

## 输入格式

The only line of the input contains a string of digits. The length of the string is between $ 1 $ and $ 10 $ , inclusive.

## 输出格式

Output "Yes" or "No".

## 样例 #1

### 输入

```
373

```

### 输出

```
Yes

```

## 样例 #2

### 输入

```
121

```

### 输出

```
No

```

## 样例 #3

### 输入

```
436

```

### 输出

```
Yes

```

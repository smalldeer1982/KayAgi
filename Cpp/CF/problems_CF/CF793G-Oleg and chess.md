# Oleg and chess

## 题目描述

**题目翻译:**

有一个 $n×n$ 的矩阵，每行每列至多能放一个棋子，另外有 $q$ 个矩形的区域不能放棋子（这些矩形区域互不相交），问最多能放多少个棋子。$n,q≤10^4$

## 输入格式

第一行一个正整数 $n≤ 10^4$

第二行一个正整数 $q≤10^4$

接下来 $q$ 行，每行$4$个正整数$(x_1,y_1),(x_2,y_2)$，表示矩形的左上角和右下角

## 输出格式

输出一行表示答案

## 样例 #1

### 输入

```
5
5
1 1 2 1
1 3 1 5
4 1 5 5
2 5 2 5
3 2 3 5

```

### 输出

```
3

```

## 样例 #2

### 输入

```
8
4
2 2 4 6
1 8 1 8
7 1 8 2
5 4 6 8

```

### 输出

```
8

```

# k-Factorization

## 题目描述

给一个正整数$n$ ，找到$k$ 个整数（不必不同），这些整数都严格大于$1$ ，并且它们的乘积等于$n$ 。

## 输入格式

第一行包含两个整数$n$ 和$k$ （$2\leq n\leq 100000,1\leq k\leq 20$ ）

## 输出格式

如果不可能将$n$ 表示为$k$ 个数的乘积，输出-1。

否则，按任意顺序输出这$k$ 个整数。它们的乘积必须等于$n$ 。如果有多种解，任意输出一种。

感谢@Khassar 提供的翻译

## 样例 #1

### 输入

```
100000 2

```

### 输出

```
2 50000 

```

## 样例 #2

### 输入

```
100000 20

```

### 输出

```
-1

```

## 样例 #3

### 输入

```
1024 5

```

### 输出

```
2 64 2 2 2 

```

# Beaver

## 题目描述

给定一个字符串s和n个字符串b，求s中最长的连续子串使得其中不包含任意b。

## 输入格式

第一行为字符串s

第二行为n

接下来n行为b

## 输出格式

两个整数：最长子串的长度和它的起始位置（中间用空格隔开）

## 样例 #1

### 输入

```
Go_straight_along_this_street
5
str
long
tree
biginteger
ellipse

```

### 输出

```
12 4

```

```
IhaveNoIdea

```

### 输出

```

```

## 样例 #3

### 输入

```
unagioisii
```

# Minimum number of steps

## 题目描述

题面：

你有一串字符串,仅由a,b组成,一次操作为"ab"->"bba",求使原串中没有a在b前面的操作次数.

## 输入格式

仅一行：
一串长为l的字符串(只由a,b组成).

## 输出格式

一个整数,即操作次数(%1e9+7).

## 说明/提示

l:[1,1e6]
注意：

None.
翻译贡献者：尘染梦

## 样例 #1

### 输入

```
ab

```

### 输出

```
1

```

## 样例 #2

### 输入

```
aab

```

### 输出

```
3

```

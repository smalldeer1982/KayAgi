# 3-palindrome

## 题目描述

题意
给定n，求长度为n的以'a','b'或'c'构成的字符串，使得其中不存在长为3的回文子串（如"abc","abca"均符合，但"aba"不符），并使'c'的个数尽可能少

## 输入格式

第一行输入一个整数n（1<=n<=2*10^5）为字符串长度

## 输出格式

输出一个满足要求的字符串，若有多个任选一个输出
翻译提供者：mxy1

## 样例 #1

### 输入

```
2

```

### 输出

```
aa

```

## 样例 #2

### 输入

```
3

```

### 输出

```
bba

```

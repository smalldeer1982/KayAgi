# Treasure Hunt

## 题目描述

Bill船长收到一个有趣的挑战，有一些陌生人给了他一张地图和一些能够远程传送的药水…………

简单而言，给你移动方式，让你判断能否从(x1,y1)的位置移动到(x2,y2)的位置

而移动方式如下：输入x,y，你一共有四种移动方式可选择

## 输入格式

第一行输入x1,y1,x2,y2第二行输入x,y

## 输出格式

如果可以，输出YES，如果不行，输出NO

## 样例 #1

### 输入

```
0 0 0 6
2 3

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
1 1 3 6
1 5

```

### 输出

```
NO

```

# Generous Kefa

## 题目描述

读入一个字符串（都是小写字母），让你求出里边最多的一种字母的个数是否小于等于 $k$。，若大于输出"NO",否则输出"YES"。   
第二行为一个字符串。

## 样例 #1

### 输入

```
4 2
aabb

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
6 3
aacaab

```

### 输出

```
NO

```

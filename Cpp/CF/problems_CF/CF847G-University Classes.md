# University Classes

## 题目描述

在一个大学里，有$n$个学生小组。在学习交流日里，每个小组可以参与不超过$7$个课程。总共有$7$个时间段分别称为$1$至$7$将分配给这些课程。

现在已经知道了每个小组星期一的计划，即这些小组参与课程的时间段。

你的任务是决定星期一至少需要多少个教室来供这些小组参与课程。一个教室在一个时间段只能容纳一个小组。

## 样例 #1

### 输入

```
2
0101010
1010101

```

### 输出

```
1

```

## 样例 #2

### 输入

```
3
0101011
0011001
0110111

```

### 输出

```
3

```

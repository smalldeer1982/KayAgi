# Olya and Energy Drinks

## 题目描述

有一NxM的迷宫，'#'是墙，‘.’是路，一秒钟可以向四个方向中的一个移动1~k步，求从起点到终点的最短时间。

## 输入格式

第一行n、m、k，下面是NxM的迷宫，最后是起点到终点的坐标。

## 输出格式

输出最短时间。如果不能离开迷宫，输出-1。

## 样例 #1

### 输入

```
3 4 4
....
###.
....
1 1 3 1

```

### 输出

```
3
```

## 样例 #2

### 输入

```
3 4 1
....
###.
....
1 1 3 1

```

### 输出

```
8
```

## 样例 #3

### 输入

```
2 2 1
.#
#.
1 1 2 2

```

### 输出

```
-1
```

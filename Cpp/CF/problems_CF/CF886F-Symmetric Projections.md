# Symmetric Projections

## 题目描述

>给出$n$个点$(x,y)$,对于一条过原点的直线,如果$n$个点在它上面的投影是中心对称的`即存在一个点使得投影旋转180度后能和旋转前重合,这个点不一定是原点`,那么称这是一条好的直线.

>给出所有的点,问有多少条好的直线,如果有无限条好的直线,输出`-1`.

## 输入格式

>第一行,一个正整数$n$
>接下来$n$行,每行包含两个整数$x_i,y_i$

## 输出格式

>如果有无限条符合条件的,输出`-1`,否则输出一个整数表示数量

## 样例 #1

### 输入

```
3
1 2
2 1
3 3

```

### 输出

```
3

```

## 样例 #2

### 输入

```
2
4 3
1 2

```

### 输出

```
-1

```

# Magic Forest

## 题目描述

### 题目大意

给定一个正整数$n$ ，求满足如下条件的三元组$(a,b,c)$ 的个数：

- $1 \le a \le b \le c \le n$ 
- $a \space xor \space b \space xor \space c=0$ 
- 存在一个边长分别为$a,b,c$ 的三角形。

## 输入格式

一行一个正整数$n(1 \le n \le 2500)$

## 输出格式

输出满足题意的三元组个数。

感谢U3144 浮尘ii 提供的翻译

## 样例 #1

### 输入

```
6

```

### 输出

```
1

```

## 样例 #2

### 输入

```
10

```

### 输出

```
2

```

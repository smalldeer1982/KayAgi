# Love Triangle

## 题目描述

飞机跟人一样，也会有三角恋。现在有 $n$ 架飞机，编号从 $1 \sim n$ 且第 $i$ 号飞机喜欢 $f_i$ 号飞机，且 $1 \le f_i \le n$，$f_i \ne i$。

当飞机 $A$ 喜欢飞机 $B$，且飞机 $B$ 喜欢飞机 $C$，且飞机 $C$ 喜欢飞机 $A$，我们称这种情况为三角恋。现在请你找出是否存在三角恋。

## 输入格式

第一行一个数，飞机的个数 $n$（$2 \le n \le 5000$）。

第二行 $n$ 个数，$f_1, f_2, \ldots , f_n$，表示 $i$ 号飞机喜欢 $f_i$ 号飞机。

## 输出格式

如果存在三角恋则输出 `YES`，否则输出 `NO`，输出大小写都没问题。

## 样例 #1

### 输入

```
5
2 4 5 1 3

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
5
5 5 5 5 1

```

### 输出

```
NO

```

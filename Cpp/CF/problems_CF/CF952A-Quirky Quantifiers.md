# Quirky Quantifiers

## 输入格式

The input contains a single integer $ a $ ( $ 10<=a<=999 $ ).

## 输出格式

Output 0 or 1.

## 样例 #1

### 输入

```
13

```

### 输出

```
1

```

## 样例 #2

### 输入

```
927

```

### 输出

```
1

```

## 样例 #3

### 输入

```
48

```

### 输出

```
0

```

# Equator

## 题目描述

已知 Polycarp 共有 $n$ 天准备编程竞赛，他每天预计做的题数为 $a_i$，请问直到第几天能做完所有题目的一半或更多。

## 输入格式

第一行，一个整数 $n$。

第二行，$n$ 个整数，表示 $a_i$。

## 输出格式

一个整数，表示答案。

## 样例 #1

### 输入

```
4
1 3 2 1

```

### 输出

```
2

```

## 样例 #2

### 输入

```
6
2 2 2 2 2 2

```

### 输出

```
3

```

# Row

## 题目描述

### 读入一个字符串，只含有$01$两种字符，合法则输出"$Yes$",否则输出"$No$"。
### 不合法情况  
$1$.一个为$0$的字符两旁没有$1$。   
$2$.一个为$1$的字符两旁还有$1$。

## 输入格式

共两行。第一行为一个正整数$n$，表示字符串的长度。第二行为一个字符串。

## 样例 #1

### 输入

```
3
101

```

### 输出

```
Yes

```

## 样例 #2

### 输入

```
4
1011

```

### 输出

```
No

```

## 样例 #3

### 输入

```
5
10001

```

### 输出

```
No

```

# High School: Become Human

## 题目描述

### 题目大意
输入一个 $x$ ,一个 $y$ ，求是  $x^y$  大还是  $y^x$  大。 $(1≤x,y≤10^9)$

## 输入格式

一行，两个整数 $x,y$ 。

## 输出格式

若 $x^y$ > $y^x$ ,输出">",若 $x^y$ < $y^x$ ,输出"<",相等则输出"="。

感谢@二元长天笑 提供的翻译

## 样例 #1

### 输入

```
5 8

```

### 输出

```
>

```

## 样例 #2

### 输入

```
10 3

```

### 输出

```
<

```

## 样例 #3

### 输入

```
6 6

```

### 输出

```
=

```

# Mishka and Contest

## 题目描述

给出长度为 $n$ 的数列，每次只能删除右端或左端小于等于 $k$ 的数，求最多能删除几个数。

## 输入格式

- 第一行 $n, k$；
- 第二行输入 $n$ 个数，表示要删除的数列。

## 输出格式

- 一个数，表示最多能删除几个数。

## 样例 #1

### 输入

```
8 4
4 2 3 1 5 1 6 4

```

### 输出

```
5

```

## 样例 #2

### 输入

```
5 2
3 1 2 1 3

```

### 输出

```
0

```

## 样例 #3

### 输入

```
5 100
12 34 55 43 21

```

### 输出

```
5

```

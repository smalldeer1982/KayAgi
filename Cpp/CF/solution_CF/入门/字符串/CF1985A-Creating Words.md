# Creating Words

## 题目描述

Matthew 得到了两个字符串 $a$ 和 $b$，它们的长度均为 $3$。他觉得把 $a$ 的第一个字符和 $b$ 的第一个字符交换来生成两个新单词非常有趣。请你输出交换后的 $a$ 和 $b$。

注意，交换后得到的新单词不一定不同。

## 说明/提示



由 ChatGPT 4.1 翻译

## 样例 #1

### 输入

```
6
bit set
cat dog
hot dog
uwu owo
cat cat
zzz zzz```

### 输出

```
sit bet
dat cog
dot hog
owu uwo
cat cat
zzz zzz```

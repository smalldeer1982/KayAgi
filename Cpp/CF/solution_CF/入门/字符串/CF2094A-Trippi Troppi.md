# Trippi Troppi

## 题目描述

Trippi Troppi 居住在一个奇异的世界里。每个国家的古代名称由三个字符串组成，将每个字符串的首字母连接起来就形成了该国家的现代名称。

给定一个国家的古代名称，请输出其现代名称。

## 样例 #1

### 输入

```
7
united states america
oh my god
i cant lie
binary indexed tree
believe in yourself
skibidi slay sigma
god bless america```

### 输出

```
usa
omg
icl
bit
biy
sss
gba```

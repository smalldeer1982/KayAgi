## 题目描述

给出 3 个数 $a_1,a_2,a_3$，现有一种操作可以在 $a_1,a_2,a_3$ 中任选两个数，使其中一个 +1，另一个 -1。操作次数不限。

要求最小化 $|a_1+a_3-2\times a_2|$。

## 说明/提示

$1\le t\le 5000,1\le a_1,a_2,a_3\le 10^8$。

## 样例 #1

### 输入

```
3
3 4 5
2 2 6
1 6 5```

### 输出

```
0
1
0```

# 题目大意

给定三个数 $a,b,c$，每次操作可以选择两个数分别进行 `+1` 和 `-1` 操作。

问至少多少次操作可以使得 $|a+c-2b|$ 最小。

# 题目分析

令 $f(a,b,c)=|a+c-2b|$。

显然，如果让 $a,c$ `+1` 或 `-1`，答案不会改变，一定不是最优解，所以分类讨论：

1. $f(a+1,b-1,c)=|a+1+c-2\times(b-1)|=|a+c-2b+3|$。

2. $f(a-1,b+1,c)=|a-1+c-2\times(b+1)|=|a+c-2b-3|$。

3. $f(a,b+1,c-1)=|a+c-1-2\times(b+1)|=|a+c-2b-3|$。

4. $f(a,b-1,c+1)=|a+c+1-2\times(b-1)|=|a+c-2b+3|$。

$\therefore$ 综上，经过一次操作后会与上次操作结果相差 $3$。

---
$\boxed{\texttt{结论：经过多次操作后，答案一定为 1 或 0。}}$

证明：

如果经过多次操作后，当前为最优解，且 $a,b,c$ 中有两个数之差大于等于 $2$。

那么将其进行几次 `+3` 或 `-3` 操作后，答案一定会更小。

这与当前为最优解矛盾，所以结论成立。

---

所以我们只需要判断 $a+c-2b$ 的结果是否是 $3$ 的倍数即可。

# 代码

```cpp
while T--:
	in(a),in(b),in(c);
    if (a+c-2*b)%3==0:
        print(0)
    else:
        print(1)
```

# 题目大意:

给定三个数，并可以进行以下操作若干次，使得三个数中， $\left|a_1+a_3-2a_2\right|$ 最小。

+ 将一个数减少一，然后将另一个数增加一。

# 分析：
操作不改变和，且可以将这些数的值随意分配。

如果 $a_1+a_2+a_3$ 是 $3$ 的倍数，答案是 $0$ 。

如果 $a_1+a_2+a_3$ 除以 $3$ 余 $1$ ，我们将剩下的 $1$ 分配给 $a_3$ ，答案是 $1$ 。如果余 $2$ ，就分配个 $a_1,a_2$ 各一。

		if ((a+b+c)%3==0)
			cout << "0" << endl;
		
		else
			cout << "1" << endl;
	}

## Solution

由于操作不限次数，那么在总和一定的情况下，$a_1,a_2,a_3$ 都可以是任意一个自然数。

此时要让 $|a_1+a_3-2\times a_2|$ 最小，我们可以令 $a_1=0$，那么就是让 $a_3=2\times a_2$，因此 $a_1+a_2+a_3$ 如果是 3 的倍数，答案为 0。否则只能让 $a_3=2\times a_2+1$ 或 $a_3=2\times a_2-1$，此时答案为 1。前者为余数为 1，后者为余数为 2。

## Code

```cpp
#include<cstdio>

int main()
{
	scanf("%d",&t);
	while (t--)
	{
		scanf("%d%d%d",&x,&y,&z);
		s=x+y+z;
		if (s%3==0) printf("0\n");
		else printf("1\n");	
	}

### Description

给定 $a_1, a_2, a_3$，可以做无限次以下操作：

选择不同的两个 $i,j$，给 $a_i+1,a_j-1$.

目标是最小化 $|a_1+a_3-2\times a_2|$.

多测。

**数据范围**

$1\leq a_1,a_2,a_3 \leq 10^8$

$1\le T\le 5\times 10^3$

### Solution

容易发现，选 $a_1,a_3$ 是没有区别的，所以我们考虑将问题转化成：

有两个整数 $A=a_1+a_3,B=a_2\times 2$，每次可以将 $A-1,B+2$，或是将 $A+1,B-2$，最小化 $|A-B|$.

这个时候我们发现每次会将差值减小 $3$，所以最后的答案一定不会大于 $1$，接着就会发现，如果 $A-B$ 能被 $3$ 整除，则答案就是 $0$，反之为 $1$.

### Code

```cpp
inline void solve() {
	in(a, b, c);
	if ((a + c - b * 2) % 3 == 0) {
		out('\n', 0);
	} else {
		out('\n', 1);
	}
}
```

## 作者：wzy2021 (赞：0)

我们来看一下这道题：

首先我们令 $k = \left\vert a_1 + a_3 - 2 \times a_2\right\vert$，

因为 $a_1$ 与 $a_3$ 是等价的，

所以操作 $a_1$ 和 $a_3$ 的结果不会改变，以及操作 $a_1$ 和 $a_2$ 与操作 $a_3$ 和 $a_2$ 是等价的，

那么我们只需考虑操作 $a_1$ 和 $a_2$:

1. $a_1 + 1,a_2 - 1$，则 $k$ 会增加 $3$，
2. $a_1 - 1,a_2 + 1$，则 $k$ 会减少 $3$，

我们推出了：若 $k\equiv{x}\pmod{3}$，则 $k$ 就能变成 $x$，

可是这样结束了吗？

我们来看一下 $x = 2$ 的情况：

1. $k$ 不取绝对值为正，那么我们在原结果上再减一个 $3$，则 $\left\vert 2 - 3 \right\vert = 1$，就更小了！
2. $k$ 不取绝对值为负，那么我们在原结果上再加一个 $3$，则
$\left\vert -2 + 3 \right\vert = 1$，也跟小了！

我们在推出以下结论：令 $t = \left\vert a_1 + a_3 - 2 \times a_2\right\vert$

1. $3 \mid t$，则答案为 $0$，
2. $3 \nmid t$，则答案为 $1$，

接下来是代码时间！

```cpp
#include <cstdio>
#include <iostream>
#include <algorithm>
using namespace std;
 
int main() {
    int T; scanf ("%d", &T);
    while (T --) {
        int a1, a2, a3; scanf ("%d%d%d", &a1, &a2, &a3);
        printf ("%d\n", !(abs (a1 + a3 - 2 * a2) % 3 == 0));
    }
    return 0;
}
```

## CF1605A题解

**题目翻译：**

给你三个数 $a_1,a_2,a_3(1\leq a_1,a_2,a_3 \leq 10^8)$，可以任意选择两个数改变大小（一个变大，一个变小，增加或减少相同的数），求最小的 $\left| a_1+a_3-2\cdot a_2\right|$ 的值。

**本题思路：**

1. 我们可以将这个式子变化一下：若 $\left| a_1+a_3-2\cdot a_2\right|$ 为正： $a_1+a_3-2 \cdot a_2$ 最小，若 $\left| a_1+a_3-2\cdot a_2\right|$ 负： $2 \cdot a_2-a_1-a_3$ 最小。

2. 又因为和不变，所以当 $a_1+a_2+a_3 \equiv 0 \pmod{3}$ 时，输出 ``0`` ，否则输出 ``1`` 。

接下来——上代码。
```cpp
#include<bits/stdc++.h>
using namespace std;
int read()//快读模板
{
   	int x=0,f=1;
	char ch=getchar();
   	while(ch<'0'||ch>'9')
	{
		if(ch=='-')
			f=-1;
		ch=getchar();
	}
   	while(ch>='0'&&ch<='9')
	{
		x=(x<<1)+(x<<3)+(ch^48);
		ch=getchar();
	}
    return x*f;
}
void write(int x)//快输模板
{
	if(x<0)
		putchar('-'), x=-x;
	if(x>9)
		write(x/10);
	putchar(x%10+'0');
	return ;
}
int main()
{
	cin.tie(0);
	cout.tie(0);
	int t=read();
	while(t--)
	{
		int a=read(), b=read(), c=read();
		if((a+b+c)%3==0)//判断
			putchar('0'), putchar('\n');
		else
			putchar('1'), putchar('\n');
	}
	return 0;
}
```

## 前言
一道数论题。
## 题意概述
给定三个整数 $a_1,a_2,a_3$，允许你无限次的将其中一个数增加一并将另一个数减一，问 $|a_1+a_3-2\times a_2|$ 最小是多少。
## 解题思路
1. 首先，很明显，我们无论怎么对 $a_1$ 与 $a_3$ 操作，答案总不会改变。因为 $a_1 +a_3$ 的值总不会改变。
2. 接下来，若我们增加 $a_1$ 或 $a_3$，减少 $a_2$，会对答案产生什么贡献呢？显然，答案会增加 $3$，因为 $a_2\times 2$ 减少了 $2$，$a_1+a_3$ 增加了 $1$。
3. 最后，若减少 $a_1$ 或 $a_3$，增加 $a_2$ 呢？显然，答案会减少 $3$。

由上述可得，我们只需要考虑 $a_1+a_3-2\times a_2 \bmod 3$ 的值就好了。若上式 $\bmod 3$ 为 $1$ 或 $2$，则答案为 $1$，否则就为 $0$。
## 代码
```cpp
#include<cstdio>
int main(void){
    int t,a1,a2,a3;
    scanf("%d",&t);
    while(t--){
        scanf("%d%d%d",&a1,&a2,&a3);
        printf("%d\n",((a1+a2+a3)%3!=0));
    }
    return 0;
}
```

# Yogurt Sale

## 题目描述

在“Vosmiorochka”商店中，一杯酸奶的价格为 $a$ 布尔，但现在有一个促销活动，可以用 $b$ 布尔购买两杯酸奶。

Maxim 需要恰好购买 $n$ 杯酸奶。在购买两杯酸奶时，他可以选择以原价购买，也可以选择以促销价购买。

Maxim 购买 $n$ 杯酸奶所需花费的最少布尔是多少？

## 说明/提示

在样例的第三个测试用例中，直接购买三杯酸奶共需 $15$ 布尔，比用 $11$ 布尔买两杯再用 $5$ 布尔买一杯更划算。

在样例的第四个测试用例中，需要购买四杯酸奶，每杯 $5$ 布尔。

# 题解

## 作者：yitian_ (赞：10)

## 题目分析
题目描述了一个购买酸奶的场景，商店提供两种购买方式：单独购买一瓶酸奶和促销组合购买两瓶酸奶。我们需要编写一个程序来计算以最小的花费购买给定数量的酸奶。

对于给定的情况，我们可以通过比较单独购买和促销组合购买的价格来决定购买方式。如果促销组合购买的价格更便宜，我们就尽可能多地选择促销组合购买酸奶；否则，我们就采用单独购买的方式。最后将花费累加起来即可得到最小花费。

## 思路
这个问题的思路相对简单，主要是通过比较正常价和促销价来决定购买的方式。

1. 首先，我们需要判断在当前的价格情况下，是购买单独一瓶酸奶还是两瓶酸奶的促销组合更划算。
2. 如果促销组合更划算，我们尽可能多地购买促销组合，然后再考虑是否需要单独购买一瓶酸奶。
3. 如果单独购买一瓶酸奶更划算，我们直接购买单独的酸奶即可。

这个思路比较直观，基于贪心的原则，每次都选择当前最划算的购买方式。在实现时，我们只需要比较一下促销价和两倍的单独购买价的大小关系，然后做出相应的购买决策即可。

## C++ 代码实现

```c
#include<bits/stdc++.h>
using namespace std;

int main() 
{
    int t;
    cin >> t; // 输入测试用例数量
    while(t--) 
	{
        int n,a,b;
        cin >> n >> a >> b; // 输入酸奶数量，单独购买价格，促销组合购买价格
        // 如果促销价比正常价划算，则尽可能多地使用促销价
        if(b<a*2) 
		{
			int c=n/2,d=n%2; // 计算促销组合购买的数量和单独购买的数量
            cout << c*b+d*a << endl; // 输出最小花费
        } 
		else 
		{
            cout << n*a << endl; // 否则，直接单独购买
        }
    }
    return 0;
}
```

### CF1955A Yogurt Sale 题解

题目输入 $ t $ 组数据，我们只需要在每组数据中最优。
分两种情况：
- 一起 $ 2 $ 倍 $ a $ 价钱大于 $ b $ 价钱就只用单价买。

- 一起 $ 2 $ 倍 $ a $ 价钱小于等于 $ b $ 价钱就用 $ a $ 价钱和 $ b $ 价钱一起买。

上代码：

~~~cpp
#include <bits/stdc++.h>
using namespace std;
int t,n,a,b;
int main(){
    scanf("%d",&t);
    for (int i = 1;i <= t;i++){
        scanf("%d%d%d",&n,&a,&b);
        if (a * 2 > b){
              if (n % 2 == 1) printf("%d\n",n / 2 * b + a);
              else printf("%d\n",n / 2 * b);
        }
        else printf("%d\n",n * a);
    }
    return 0;
}
~~~

## 思路

分为两种情况，一种是用从 $ a $ 买两个划算，一种是用 $ b $ 买两个划算。

## Code:

```cpp
#include<bits/stdc++.h>
using namespace std;
int t;
int a,b,c;
int main(){
	cin>>t;
	while(t--){
		cin>>c>>a>>b;
		if(2*a>b){
			if(c%2==0){
				cout<<b*(c/2)<<endl;
			}else{
				cout<<b*((c)/2)+a<<endl;
			}
		}else{
			cout<<a*c<<endl;
		}
	}
	return 0;
}
```

题意解释：其实就是你可以直接用单价买，或者用优惠价直接买两瓶，再用单价补上缺的。  
解题方式：根据上面的分析，暴力枚举即可。  
代码如下：
```
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int main()
{
	ll n,x,a,b,t1,t2,t3,i;
	cin>>n;
	for(i=1;i<=n;i++)
	{
		cin>>x>>a>>b;
		t1=a*x;
		t2=(x/2)*b;
		t3=(x%2)*a;
		cout<<min(t1,t2+t3)<<endl;
	}
	return 0;
}
```

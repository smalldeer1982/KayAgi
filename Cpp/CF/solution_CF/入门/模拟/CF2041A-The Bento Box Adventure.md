# The Bento Box Adventure

## 题目描述

![](https://cdn.luogu.com.cn/upload/vjudge_pic/CF2041A/d196e410706f95eae92e5b015241db6ecb085028.png) 图片由 ChatGPT 4o 生成。台湾的便当非常普遍，为学生和上班族提供了方便且价格实惠、营养均衡的选择。这些便当通常包含多种蔬菜、蛋白质和米饭，能够满足均衡饮食的需求。在本地自助餐厅有许多选择，因此成为了追求快捷健康午餐的人的热门选择。

有五家台湾自助餐厅，编号为 1 到 5。你计划在周一到周五每天去不同的餐厅打包便当作为午餐。你已经在周一到周四分别去过了四家不同的餐厅。现在是周五，你想去本周还没有去过的那家餐厅。

请编写一个程序，输入你在周一到周四已经去过的四家餐厅的编号（顺序为周一到周四），输出你周五应该去的那家餐厅的编号。

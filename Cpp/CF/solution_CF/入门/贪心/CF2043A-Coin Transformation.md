# Coin Transformation

## 题目描述

最开始，你有一枚价值为 $n$ 的硬币。你可以任意多次地执行以下操作：

- 将一枚价值为 $x$ 的硬币（其中 $x > 3$）转换成两枚价值为 $\lfloor \frac{x}{4} \rfloor$ 的硬币。

经过一系列操作后，你最多能得到多少枚硬币？

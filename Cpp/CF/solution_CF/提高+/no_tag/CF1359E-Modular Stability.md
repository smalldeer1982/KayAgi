# Modular Stability

## 题目描述

求有多少个长度为 $k$ 的序列 $a$，满足以下条件：

- $\forall 1 \le i < k,a_i < a_{i+1}$
- $\forall 1 \le i \le k,1 \le a_i \le n$
- 对于任意一个 $1$ 至 $k$ 的排列 $p$，满足 $( (((x \bmod a_1)\bmod a_2)\bmod a_3)\bmod \cdots \bmod a_k) = ((((x \bmod a_{p_1})\bmod a_{p_2})\bmod a_{p_3} \bmod \cdots \bmod a_{p_k}$。其中 $x$ 为任意非负整数。

结果对 $998,244,353$ 取模。

## 说明/提示

$1 \le n,k \le 5 \times 10^5$。

# 题解

## 作者：fighter (赞：11)

我们先拉出序列中最小的那个数$a$，假设现在有一个大于$a$的模数$b$，那么思考一下它们的顺序对答案的影响。

对于任意正整数$x$，我们要满足$x\%a\%b= x\%b\%a$，又因为$b>a$，所以$x\%a\%b=x\%a$，所以上式即$x \equiv x\% b\pmod a$

然后设$x=tb+y$，那么$tb+y\equiv y \pmod a$，由于$t$是任意的，所以只有当$a|b$时，等式恒成立。

那么意会一下，这个结论也可以推广到整个数列，于是我们得出结论：序列中的每一个数都是最小的那个数的倍数。

那么我们只要从1开始枚举最小的数，求出值域范围内有多少它的倍数，然后组合数选出k-1个即可（最小数自己是钦定的）。

```cpp
#include <bits/stdc++.h>
#define ll long long
#define MAX 1000005
#define P 998244353
using namespace std;

ll n, k;
ll fac[MAX], inv[MAX];

void init(int n){
    fac[0] = 1, inv[0] = inv[1] = 1;
    for(int i = 1; i <= n; i++) fac[i] = fac[i-1]*i%P;
    for(int i = 2; i <= n; i++) inv[i] = (P-P/i)*inv[P%i]%P;
    for(int i = 2; i <= n; i++) inv[i] = inv[i]*inv[i-1]%P;
}

ll C(ll n, ll m){
    if(!m) return 1;
    return fac[n]*inv[m]%P*inv[n-m]%P;
}


int main()
{
    init(MAX-1);
    cin >> n >> k;
    ll ans = 0;
    for(int i = 1; i <= n; i++){
        if(n/i < k) break;
        ans = (ans+C(n/i-1, k-1))%P;
    }
    cout << ans << endl;

    return 0;
}
```

## 作者：duyi (赞：5)

## 题解 CF1359E Modular Stability

> 引理：
>
> 对于两个正整数$a,b$，不妨设$a<b$。$\forall x:(x\bmod a)\bmod b=(x\bmod b)\bmod a$，当且仅当$b\bmod a=0$，也就是$b$是$a$的倍数。

> 证明：
>
> 充分性显然。
>
> 必要性可以考虑$x=b$时，左边$=x\bmod a=b\bmod a$，右边$=0$。所以必要条件是$b\bmod a=0$。

发现当$a_1,\dots a_k$从小到大排列时，结果就等于$x\bmod a_1$。

相当于我们要让任意排列的结果都等于$x\bmod a_1$。

任意排列之后，$a_1$后面的数可以不用管，因为不会影响结果。

如果$a_1$前面的数，都是$a_1$的倍数，那么不管他们模出来什么结果，最后$\bmod a_1$都是一样的。

所以发现，一个数列合法，当且仅当所有数都是$a_1$的倍数。

枚举$a_1=i$，则$a_1$的倍数有$\lfloor\frac{n}{i}\rfloor-1$个，我们从中选出$k-1$个，方案数是${\lfloor\frac{n}{i}\rfloor-1\choose k-1}$。

预处理阶乘，即可$O(1)$求组合数。

时间复杂度$O(n+k)$。

参考代码（片段）：

```cpp
const int MAXN=5e5,MAXK=20;
int n,K;
const int MOD=998244353;
inline int mod1(int x){return x<MOD?x:x-MOD;}
inline int mod2(int x){return x<0?x+MOD:x;}
inline void add(int& x,int y){x=mod1(x+y);}
inline void sub(int& x,int y){x=mod2(x-y);}
inline int pow_mod(int x,int i){int y=1;while(i){if(i&1)y=(ll)y*x%MOD;x=(ll)x*x%MOD;i>>=1;}return y;}
int fac[MAXN+5],ifac[MAXN+5];
inline int comb(int n,int k){
	if(n<k)return 0;
	return (ll)fac[n]*ifac[k]%MOD*ifac[n-k]%MOD;
}
void facinit(int lim=MAXN){
	fac[0]=1;
	for(int i=1;i<=lim;++i)fac[i]=(ll)fac[i-1]*i%MOD;
	ifac[lim]=pow_mod(fac[lim],MOD-2);
	for(int i=lim-1;i>=0;--i)ifac[i]=(ll)ifac[i+1]*(i+1)%MOD;
}

int main() {
	facinit();
	cin>>n>>K;
	int ans=0;
	for(int i=1;i<=n;++i){
		add(ans,comb(n/i-1,K-1));
	}
	cout<<ans<<endl;
	return 0;
}
```

## 作者：nothingtoeatit (赞：2)

[zxx 的恩情还不完系列](https://www.luogu.com.cn/training/168096#problems)。

### [题意](https://www.luogu.com.cn/problem/CF1359E)

求有多少个长度为 $n$ 递增的严格序列 $a_i$ 满足每个元素在 $1$ 到 $n$ 之间，且对于任意正整数 $x$，让 $x$ 对 $a_i$ 的任何一个排列顺次取模，结果相同，$n,k\le 5\times 10^5$。

### 分析

首先考虑不打乱，结果一定是 $n\bmod a_1$，然后考虑 $a_1$ 排在第二位的情况，不妨假设 $a_i$ 在第一位，如果 $a_i$ 不是 $a_1$ 的倍数，那么令 $x=a_i$，结果就是 $0$，与 $x\bmod a_1$ 不同。

所以这个结论就轻松被转化成了所有数的最大公约数就是最小值，或者说所有的数都是 $a_1$ 的倍数。

那么从 $1$ 到 $n$ 枚举 $a_1$，答案就是：$$\sum\limits_{i=1}^nC_{\lfloor n/i\rfloor-1}^{k-1}$$

这题如果多测的话，可以做到 $O(n+T\sqrt n)$，这里由于只有一组测试数据，实际的复杂度即为 $O(n)$。

```c++
#include<bits/stdc++.h>
#define up(a,b,c)for(int a=b;a<=c;++a)
const int N=55e4,p=998244353;
using namespace std;
int n,k,fc[N],nf[N],as;
int C(int n,int m)
{
	return m<=n?1ll*fc[n]*nf[m]%p*nf[n-m]%p:0;
}
int main()
{
	cin>>n>>k;
	nf[0]=nf[1]=fc[0]=1;
	up(i,2,n)nf[i]=1ll*(p-p/i)*nf[p%i]%p;
	up(i,1,n)fc[i]=1ll*fc[i-1]*i%p,nf[i]=1ll*nf[i-1]*nf[i]%p;
	up(i,1,n)as=(as+C(n/i-1,k-1))%p;
	cout<<as;
	return 0;
} 
```

## 作者：Ginger_he (赞：2)

## 题解

显然，对于正整数 $y$，都有 $x\bmod y<y$。故对于左式，有

$$(((x\bmod a_1)\bmod a_2)\ldots)\bmod a_k=x\bmod a_1$$

由于 $p$ 是任意一个 $1$ 到 $k$ 的排列，所以对于任意 $j=2,3,\ldots,k$ 都有

$$((((((((x\bmod a_j)\bmod a_1)\bmod a_2)\ldots)\bmod a_{j-1})\bmod a_{j+1})\bmod a_{j+2})\ldots)\bmod a_k=x\bmod a_1$$

从而有

$$(x\bmod a_j)\bmod a_1=x\bmod a_1$$

则

$$x=a_jr+a_1s+q=a_1p+q$$

其中 $p,q,r,s$ 均为非负整数，且 $a_1s+q<a_j,q<a_1$。化简可得

$$a_1(p-s)=a_jr$$

由于 $x$ 为任意非负整数，所以 $p-s$ 和 $r$ 均为变量。若要上述条件恒成立，只能 $a_j$ 为 $a_1$ 的倍数。

所以对于任意 $j=2,3,\ldots,k$，$a_j$ 均为 $a_1$ 倍数，结合序列 $a$ 严格递增的性质，不难得出答案为

$$\sum_{i=1}^n\binom{\lfloor\dfrac{n}{i}\rfloor-1}{k-1}$$

### Code

```cpp
#include<bits/stdc++.h>
using namespace std;
#define ll long long
#define N 500005
const ll p=998244353;
int n,k; 
ll a[N],f[N],g[N],ans;
ll quickpow(ll a,ll b)
{
	ll res=1;
	while(b)
	{
		if(b&1)
			res=res*a%p;
		b>>=1;
		a=a*a%p;
	}
	return res;
}
ll c(ll n,ll m)
{
	if(m>n) return 0;
	if(!m) return 1;
	return f[n]*g[m]%p*g[n-m]%p;
}
int main()
{
	scanf("%d%d",&n,&k);
	f[0]=1;
	for(int i=1;i<=n;i++)
		f[i]=f[i-1]*i%p;
	g[n]=quickpow(f[n],p-2);
	for(int i=n-1;i>=0;i--)
		g[i]=g[i+1]*(i+1)%p;
	for(int i=1;i<=n;i++)
		ans=(ans+c(n/i-1,k-1))%p;
	printf("%lld\n",ans);
	return 0;
}
```

## 作者：快斗游鹿 (赞：1)

## 思路

先来看题目中给的限制，$((((x \bmod a_1)\bmod a_2)\bmod a_3)\bmod \cdots \bmod a_k) = ((((x \bmod a_{p_1})\bmod a_{p_2})\bmod a_{p_3} )\bmod \cdots \bmod a_{p_k})$。因为 $a_1$ 是最小的，所以其实这个式子等价于 $(x \bmod a_1) = ((((x \bmod a_{p_1})\bmod a_{p_2})\bmod a_{p_3} )\bmod \cdots \bmod a_{p_k})$。

随便找一个大于 $a_1$ 的模数，假设是 $a_2$，则应该满足 $x\bmod a_1=(x\bmod a_2)\bmod a_1$。即 $x=pa_1+s=qa_2+ra_1+s$，移项得到 $a_1(p-r)=qa_2,a_1=\dfrac{q}{p-r}a_2$，其中 $p,q,r,s$ 都是非负整数。 又因为 $a_i$ 均为整数，所以只有当 $a_2$ 是 $a_1$ 倍数时，条件成立。

得到这个结论后，就好做了。我们枚举 $a_1$，每次从剩下的 $a_1$ 的倍数中取出 $k-1$ 个数即可，这就是个组合数。预处理一下就可以做到 $O(1)$ 求组合数。

## 代码

```
#include<bits/stdc++.h>
#define int long long 
using namespace std;
const int p=998244353;
const int N=5e5+5;
int n,k,ans,inv[N],fac[N];
int read(){
	int x=0,f=1;
	char ch=getchar();
	while(ch<'0'||ch>'9'){if(ch=='-')f=-1;ch=getchar();}
	while(ch>='0'&&ch<='9'){x=(x<<3)+(x<<1)+ch-'0';ch=getchar();}
	return x*f;
}
int C(int n,int m){
	if(m==0)return 1;
	return fac[n]*inv[m]%p*inv[n-m]%p;
}
signed main(){
	n=read();k=read();
	inv[0]=1;inv[1]=1;fac[1]=1;
	for(int i=2;i<=n;i++)inv[i]=(p-p/i)*inv[p%i]%p;
	for(int i=2;i<=n;i++)inv[i]=inv[i-1]*inv[i]%p;
	for(int i=2;i<=n;i++)fac[i]=fac[i-1]*i%p;
	if(k>n){
		puts("0");return 0;
	}
	for(int i=1;i<=n;i++){
		if(n/i<k)break;
		ans+=C(n/i-1,k-1);ans%=p;
	}
	cout<<ans;
	return 0;
}

```

## 作者：elbissoPtImaerD (赞：1)

首先，我们有 $\forall a<b$，$x\bmod a \bmod b = x \bmod a$。

左边的式子化简完就是 $x \bmod a_1$。  
右边可以变成，$x \bmod a_{b_1} \cdots \bmod a_{b_m} \bmod a_1$，$b$ 满足 $b_i > b_{i+1}$。

考虑把 $x$ 写成 $ya_1+z$ 的形式。

这样左边就是 $z$，右边 $x \bmod a_{b_1} \cdots \bmod a_{b_m} \bmod a_1$ 要恒等 $z$。

我们先考虑 $m=1$ 会怎样。  
$m=1$ 时，有 $x\bmod a_{b_1}\bmod a_1=x \bmod a_1$。

由于 $a_{b_1} > a_1$，所以 $a_1|a_{b_1}$。  
由反证法知道这个是充要的。

再对 $m$ 归纳一下，那就有 $\forall a_i$，$a_1 | a_i$。

枚举 $a_1$ 的取值就能算了。

是 $\sum\limits_{i=1}^n \binom{\lfloor\frac n i \rfloor-1}{k-1}$。

求有多少数列 $a_1,a_2,\dots,a_k(1\le a_{i-1}<a_i<a_{i+1}\le n)$ 满足 $\forall x \in N$，对于数列 $a$ 的任意一个排列  $p$，均有 $x\% a_{p_1}\% a_{p_2}\%\dots\%a_{p_n}=\text{定值}$ ，即 $(((x \bmod a_1) \bmod a_2) \dots \bmod a_{k - 1}) \bmod a_k$ 的值，无论 $a$ 数列如何排列，都是同一个值。

**Solution**

题目中是 $k$ 个模数，我们先考虑一般情况，即两个模数的情况。

设 $a<b$

$$((x\bmod a)\bmod b=((x\bmod b)\bmod a)$$

由于 $b>a$

显然有 $(x\bmod a)< b$ ，即 $((x\bmod a)\bmod b=(x\bmod a)$

即

$$x\equiv (x\bmod b)\pmod a$$

设 $x=tb+y$（由于 $x\in N$，故 $t\in N$，即 $t$ 为任意整数）

则 
$$\begin{aligned}&x\equiv (x\bmod b)\pmod a&\\&tb+y\equiv((tb+y)\bmod b)\pmod a&\\&tb+y\equiv y\pmod a&\\&tb\equiv0\pmod a\end{aligned}$$

由于 $t$ 为任意整数，所以 $b\pmod a=0$，即对于任意一个 $a$ 的倍数的序列，均满足题意。

所以直接 $O(n)$ 枚举 $a$，显然 $1\sim n$ 中 $a$ 的倍数一共有 $\lfloor\dfrac{n}{a}\rfloor$ 个，去掉 $a$ 之后还有 $\lfloor\dfrac{n}{a}\rfloor$ 个，我们从中任意选取 $k-1$ 个数，与 $a$ 组成长度为 $k$ 的序列，方案数为 $C_{\lfloor\frac{n}{a}\rfloor-1}^{\ k - 1}$，总和即为答案。


**Code**

```cpp
// Problem: CF1359E Modular Stability
// Contest: Luogu
// URL: https://www.luogu.com.cn/problem/CF1359E
// Memory Limit: 500 MB
// Time Limit: 2000 ms
// 
// Powered by CP Editor (https://cpeditor.org)

#include <bits/stdc++.h>

using namespace std;
using ll = long long;
const int N = 6e5 + 7, mod = 998244353;
int n, m, k, t;
int fact[N], infact[N];
int inv[N];

void init(int n)
{
	fact[0] = infact[0] = inv[0] = inv[1] = 1;
	for (int i = 2; i <= n; ++ i) 
		inv[i] = 1ll * (mod - mod / i) * inv[mod % i] % mod;
	for (int i = 1; i <= n; ++ i) {
		fact[i] = 1ll * fact[i - 1] * i % mod;
		infact[i] = 1ll * infact[i - 1] * inv[i] % mod;
	} 
}

int C(int n, int m)
{
	if(m == 0 || n == m) return 1; 	
	if(m > n) return 0; 
	return 1ll * fact[n] * infact[m] % mod * infact[n - m] % mod;
}

int main ()
{
	init (N - 7);
	scanf ("%d%d", &n, &k);
	ll ans = 0;
	for (int i = 1; i <= n; ++ i) { 
		ans = (ans + C((n / i) - 1, k - 1)) % mod;
	}
	cout << ans << endl;
	return 0;
}
```

对于一个 $x$，所有排列中必然存在一个排列，满足 $x\% a_{p_1} \% \cdots \% a_{p_m}$ 为 $x\%a_1$（$p_1=1$ 时）。所以我们的需要满足对于所有排列都有 $\forall p_j<p_1, x\% a_j=x\% a_1$，即所有数都满足 $x\% a_i=x\% a_1$，即 $a_i=ka_1$。众所周知，$\le n$ 的数中有 $\lfloor\frac{n}{x}\rfloor$ 个 $x$ 的倍数，我们相当于要从中选取 $k$ 个数。我们考虑枚举 $a_1$ 的取值。所以答案为：

$$
\sum_{x=1}^{n} \binom{\lfloor\frac{n}{x}\rfloor-1}{k-1}
$$

$\texttt{Code} \downarrow$

```cpp
#define per(i,a,b) for(register int i=(a);i>=(b);i--)

const int N=5e5+9,mod=998244353;

inline int read() {
    register int x=0, f=1; register char c=getchar();

}

int n,k,fac[N],inv[N],ifac[N],ans;

    rep(i,1,n) fac[i]=fac[i-1]*i%mod;

}

    if(x<0||y<0||x<y) return 0;

}
```

# Lucky Number Representation

## 题目描述

定义由 $4,7,0$ 三个数字构成的数称为幸运数，例如 $0,47,7074$ 就是幸运数。

现在给定 $t$ 个 $n$，问 $n$ 能否被拆成 $6$ 个幸运数的和，能则输出这六个数，否则输出 $-1$

## 说明/提示

$1 \leq t \leq 5000,1 \leq n \leq 10^{18}$


如果有多种方案，输出任意一种。

另：请不要用 ``%lld`` 读 ``long long``，请用 ``cin`` 和 ``cout``（题目特别声明）。

## 样例 #1

### 输入

```
5
42
17
444
7
51
```

### 输出

```
7 7 7 7 7 7
-1
400 0 40 0 4 0
7 0 0 0 0 0
47 4 0 0 0 0
```

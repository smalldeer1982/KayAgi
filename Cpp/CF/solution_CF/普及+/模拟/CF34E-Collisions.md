# Collisions

## 题目描述

在数轴上有n个球，可视为质点。现在我们知道它们的坐标位置，质量和速度（可能是负数）。现在问t秒后，这n个小球的位置。在这t秒间，小球可能会发生弹性碰撞，现在给你弹性碰撞的公式![1](https://cdn.luogu.org/upload/vjudge_pic/CF34E/939b5bce3bf45a602cb8d3e9f25a3ee89c788ac0.png)

## 样例 #1

### 输入

```
2 9
3 4 5
0 7 8
```

### 输出

```
68.538461538
44.538461538
```

## 样例 #2

### 输入

```
3 10
1 2 3
4 -5 6
7 -8 9
```

### 输出

```
-93.666666667
-74.666666667
-15.666666667
```

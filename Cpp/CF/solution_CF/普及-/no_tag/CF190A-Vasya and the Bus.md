# Vasya and the Bus

## 题目描述

--

一天，Vasya 听到一个故事，它说：“在 High Bertown 城，一辆 $62$ 路公交车带着 $n$ 位大人和 $m$ 位小孩，从公交车站开了出来……”

至于那些后续，都是无关紧要的。Vasya 是一位会计，而且他很享受数钱数到手抽经的快感。因此，他想知道这趟车上的这些乘客所付车票钱的最小值和最大值是多少

在 High Bertown 城，公交车的车票为 $1$ 卢布 。但是，那么简单的题目想都别想遇到—— $1$ 位大人可以免票带 $1$ 位小孩。这意味着 $1$ 位大人和 $k$ 个小孩共同乘车，总的需要 $k$ 卢布的车票；他自己需要 $1$ 张车票，孩子们需要 $(k-1)$  张。除此之外，大人们可以不带孩子乘车，这样他们就只要支付自己的 $1$ 卢布车票钱。

同时，我们还知道在 High Bertown 城，孩子们不能在没有大人的陪同下单独乘车。

请你帮帮 Vasya 计算一下这辆车上乘客需付金额总和的最小值和最大值。

---

## 说明/提示

---

在第一个样例中， $1$ 位大人和 $2$ 位孩子乘车，不管怎么样，都需要支付 $2$ 卢布。

在第二个样例中，只有孩子乘车是不合法的。

在第三个样例中，有两种情况：

1.  $2$ 位大人各带 $1$ 位小孩，各付 $1$ 卢布车票钱。此时，乘客们一共支付了 $2$ 卢布。

2. 另一种情况下，一位大人带 $2$ 位孩子，支付 $2$ 卢布车票钱；另一位独自乘车，花费 $1$ 卢布。因此，他们一共支付了 $3$ 卢布。

# 题解

## 作者：Chtholly_L (赞：6)

### 题意：

公交车上有 $n$ 个大人 和 $m$ 个小孩，一个大人只可以免费带一个小孩，小孩不能单独乘车，一张票 $1$ 块钱，求赚钱的最大值和最小值。

---

### 分析：

一共分为 $3$ 种情况。

1. 车上只有大人。那么不用考虑小孩的免费问题，直接输出两次大人们所花的钱，即大人的人数。

2. 车上只有小孩。这样是不合法的，直接输出 "Impossible" 。

3. 车上既有大人也有小孩。考虑两种最极端的情况。      
   （1）一位大人带了所有的小孩，那么其中只有一个小孩不能付钱，其他所有人都要付钱，那么结果是 $n+m-1$ 。

   （2）每位大人都带了一个小孩，那么所付的钱数即为大人人数或小孩人数的最大值。因为如果大人较多，那么所有的小孩都不用付钱；如果小孩较多，那先付多出来的小孩的钱，再付接下来大人的钱，其余的小孩不用付钱，即小孩的人数。
   
### 代码：

```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,m;
	cin>>n>>m;
	int maxn;
	if(m>=n)maxn=m; //求出第三种中的最大值
	else maxn=n;
	if(m==0)cout<<n<<" "<<n;  //没有小孩
	else if(n==0)cout<<"Impossible";  //没有大人
	else cout<<maxn<<" "<<m+n-1;  //大人小孩都有
	return 0;
}
```

# 题意：
一辆~~宝宝巴士~~上有n位大人和m位小孩，每人的车票为一卢布，而一个大人可以免费带一名小孩，但是小孩不能自己坐车（雾），也就是说车上不能没有大人，然后求赚钱的最大值和最小值。
# 本题思路：
由题意得，我们可以分为三种情况：

1.车上没有大人，只有小孩，输出“Impossible”；
       
2.车上有大人有小孩，并且一个大人带一个小孩；

3.车上有大人有小孩，但是是一个大人带所有小孩，剩下的人支付自己的车票（大雾）；

分析完毕，那么我们可以写出代码去实现它。
## 代码如下：
```c
#include<iostream>
#include<cstdio>
#include<algorithm>//max所对应的头文件 
using namespace std;
int main() {
	int n,m,ans;
	cin>>n>>m;//简单的输出
	ans=max(m,n);//求出m和n的最大值赋给ans
	if(n==0&&m!=0) {//车上没有大人时必须要有小孩，车上不能没有人！！！ 
		cout<<"Impossible"; 
		return 0;//直接结束 
	} //当没有大人，只有小孩的时候 乘车是不合法的，所以输出Impossible
	else if(m==0) {
		cout<<n<<" "<<n;
		return 0;
	} //当没有小孩，只有大人的时候 就输出大人每人一张票需要的钱数就可以了 
	else {
		cout<<ans<<" "<<m+n-1;
	}//当大人和小孩都有的时候，最小值为一个大人带一个小孩 最大值为一个大人带了所有的孩子
	return 0;//完美结束 
}
```

## 作者：Waaifu_D (赞：0)

### 题面解释

$n$ , $m$ 分别为大人和小孩的人数，每个大人可以带无数个小孩（雾）， $1$ 位大人可以免费带 $1$个 小孩，剩下的就另外收费（一个大人一块钱），所以 $1$个 大人带 $ n $ 个小孩需要支付 $n$ 元( $1+n-1=n$ ）,同时小孩不可以自己乘车，要求我们求出唯一的一种钱数或者输出最大值和最小值

### 思路讲解

先从简单的想起，小孩不可以自己乘车，大人可以带无数个小孩，所以如果没有大人且小孩数不为 $0$ 我们就要输出 $Impossible$ 
```cpp
if(n==0&&m!=0) 
{
	cout<<"Impossible"; return 0;
}
```
然后是只有大人没有小孩的情况，这个也简单，直接输出每个大人所需支付的钱数就可以

```cpp
if(m==0)
{
	cout<<n<<" "<<n; return 0;
}
```

最后就是需要一点思维的情况，也就是样例3的情况

```
输入：         输出：

2  2           2  3
```

这种情况有两种支付方式

1. 一位大人把2个小孩都带上，另外一个自己支付，共支付3元

2. 每位大人分别带一个小孩，这样只需要支付2元

分析一下我们可以得出，这种有两种情况的答案其实就是 $n$ ，$m$ 之中的最大值和     $m+n-1$ ,因为花最少钱数的方法就是每个大人带一个小孩，这样需支付 $m$ , $n$ 之中的最大值元，而花最多钱数就是一个大人带上所有的小孩，这样可以让最多的小孩付钱，只有一个小孩不需要付钱，即 $m+n-1$ 元

完整代码如下

```cpp
#include<cstdio>
#include<iostream>
#include<cmath>
using namespace std;
int n,m,maxx;
int main()
{
	cin>>n>>m;
	maxx=max(n,m);
	if(n==0&&m!=0) {
	cout<<"Impossible"; return 0;}//记得结束程序
	if(m==0) {
	cout<<n<<" "<<n; return 0;}
	cout<<maxx<<' '<<m+n-1;
	return 0;
}
```

$1$道简单的判断题（~~只要你知道情况和答案~~）

这一题的答案主要分为$3$种情况：

- 没有小孩，也就是$m=0$的情况，这种情况我们只要输出两次大人的数量即可，因为每个人付$1$卢布，$n$个大人就付$n$卢布

- 没有大人，这种最简单，直接输出$Impossible$

- 有大人也有小孩，最小的：$max(n,m)$：大人多直接大小孩的票省了，小孩多就是先付了$n$卢布，再加上$m-n$，算出来还是$m$；最大的：$n+m-1$：一个大人带所有孩子，只省了$1$卢布，最后是大人数量$+$小孩数量$-$省的钱（$1$卢布）

然后记得要加$else$，中间加" "什么的

## 作者：糖葫芦 (赞：0)

可以直接将题目分为三个情况

1.没有孩子，即m=0,这时最大，最小值都为n

2.没有大人,即n=0，此时应输出"Impossible"

3.大人和孩子都存在,最小值在n>=m时是每个孩子都分别被一个大人带着,即为n张票,因为孩子票都被省了；在m>=n时是每个大人都至少带了一个孩子,此时最小值为m，因为大人票都被省了。整理一下可知最小值为m和n中的大的那个。最大值是一个大人带了所有孩子，此时只省了一张票,即m+n-1


```cpp

using namespace std;
int n,m;
int main(){
    scanf("%d%d",&n,&m);
    if(m==0){
    	printf("%d %d\n",n,n);
    }else if(n==0){
        printf("Impossible");
    }else{
        printf("%d %d\n",max(n,m),n+m-1);
	}
    return 0;
}
```

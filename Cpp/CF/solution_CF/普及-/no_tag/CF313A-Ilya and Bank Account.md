# Ilya and Bank Account

## 题目描述

Ilya是一只灰常聪明的狮子，他住在一座不同寻常的城市中——动物园！在这座城市中，每一只动物都有它的权利与义务。它们甚至都有自己的银行账户。银行账户包含一个整数，其中有可能是负数（说明它欠钱了）。

Ilya最近要过生日了，所以他收到了一系列礼物。其中一个礼物（动物园总行）可以让它消除它银行账户上的最后一位或倒数第二位数字（**就一次**）。比如说，如果Ilya的账户上余额为-123，那么他可以消除3得到-12，或者消除2得到-13（当然它不傻，知道欠钱越少越好）。Ilya也可以不消除数字。

现在，Ilya要你帮他算出它最多有多少钱。

## 说明/提示

第一个样例中，狮子没有用那个礼物消除数。
第二个样例中，狮子消除了1得到0。

## 样例 #1

```
2230
```

### 输出

```
2230
```

## 样例 #2

### 输入

```
-10
```

### 输出

```
0
```

## 样例 #3

### 输入

```
-100003
```

### 输出

```
-10000
```

## 作者：封禁用户 (赞：5)

题目传送>>[CF313A](https://www.luogu.com.cn/problem/CF313A)  
### 题意简述：  
- 给定一个银行账户余额 $n$，现有一次机会可消除其最后一位或倒数第二位数字,求最终其最大值。  
- $10\le|n|\le10^9$。  

### 题目分析:  
据题意可知账户余额可以为正，也可以为负（此时表示欠钱）。为正的时候不管消除其最后一位还是倒数第二位账户余额都会变小，所以我们此时就放弃消除，初始余额即最大。而为负时，当然欠的钱越少越好，我们就可以使用消除的机会对其进行改变，我们只需要分别计算出消除其最后一位和倒数第二位得到的最终余额，再进行比较，越大的即表示欠钱越少，账户余额也就最大。  
### 问题解决:  
#### 判断:  
我们可以使用 if...else... 进行判断，当然我们也可以使用一个小东西 —— 三目运算符 “?:”。  
例：a?b:c 的含义是：如果 a 为真，则表达式的值是 b，否则是 c。  
### Code:  
```cpp
#include<iostream>
using namespace std;
int main()
{
	ios::sync_with_stdio(0);   //关同步流，加速cin输入和cout输出
	int n;   //定义账户余额
	int xl,xs,r;   //定义消除其最后一位的结果，消除其倒数第二位的结果，最终结果
	cin>>n;
	xl=n/10;    //计算消除其最后一位的结果
	xs=(n/100)*10+n%10;   //计算消除其倒数第二位的结果
	r=(n>0)? n:max(xl,xs);   //利用三目运算符判断其最终结果，如果为正即为初始值，为负则比较两个经消除后的数值取其最大
	cout<<r;
	return 0;   //结束整个程序
}
```

## 作者：Chtholly_L (赞：2)

### 题意：

给你一个数字，你可以选择它的最后一位或倒数第二位，将这一位删除，你也可以选择不删除。请你求出消除后最大的数。

---

### 分析：

由于只能消除最后一位或倒数第二位，我们便直接计算出消除后的两种结果，再将这三个数进行排序，找出最大的数。

对于消除最后一位，我们可以直接将原数整除十，这样就相当于直接去掉最后一位。

对于消除倒数第二位，我们可以将原数整除一百，这样就得到了没有后两位的数，再将这个数乘十，就相当于去掉了最后一位，再将倒数第二位清零，这时，我们再加上最后一位就得到了去掉倒数第二位的数。

拿样例一来举个例子：

$2230/100=22$ 

$22 \times 10=220$

$220+(2230 \bmod 10)=220$

---

### 代码：

```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>  //用 sort 排序

using namespace std;

int a[5];

int main(){
	int n;
	cin>>n;
	int yi=n/10;  //去最后一位
	int er=n/100*10+n%10;  //去倒数第二位
	a[1]=n,a[2]=yi,a[3]=er;  //录入数组
	sort(a+1,a+4);  //排序
	cout<<a[3];
	return 0;
}
```

### 思路：  
如果输入的数是正数，无论消除哪一个数字，都只会让钱变少，所以选择不消数字。  
如果输入的数是负数，就消除倒数两位之中数字更大的一个，注意如果消除后数为  $0$需要特判（如 $-10$ 消除后为 $-0$ 即 $0$）。

```python
import sys #头文件
a=input()  #输入的是字符串

if int(a)>=0:
    print(a)     #正数直接输出
    sys.exit(0)  #跳出程序
if int(a[len(a)-1]) >= int(a[len(a)-2]):  #如果倒数第一位大于等于第二位，则舍去倒数第一位

        b+=a[i]
else:                                     #倒数第二位大于倒数第一位
```  

## 作者：chenkaiwen (赞：1)

这个算是一道比较基础的数论题，这一题的题意是：

给你一个数$n$，你可以对他进行以下几种操作
1. 不变
2. 删除它的倒数第一位	
3. 删除它的倒数第二位，并把最后一位向前与倒数第三位连接

那么，我们怎么用代码实现上面的三种情况呢。首先第一种十分的简单，就是$n$本身；第二种要删除最后一位，就是$n/10$；第三种是删除倒数第二位，也就是倒数第二位以前的位数（n/100）+倒数第一位（n%10），也就是$ ((n/100)* 10+n%10) $

所以代码十分的简单，我们只要在三种情况中取最大值就好了，对于取最大值，我们可以用$max$，但是$max$只能比较两个数的掉线，这里告诉大家一种方法可以用：max(a,max(b,c))来获取三个数的最大值。

其他细节上的问题我会在代码的注释中写出，这里就不一一解释，代码如下：

[![](https://cdn.luogu.com.cn/upload/image_hosting/bhfc7ota.png)证明](https://www.luogu.com.cn/record/32443326)

```cpp
#include<bits/stdc++.h>//多余的头文件最好要删掉 
using namespace std;
int main() {
	int n;
	cin>>n;//输入n 
	cout<<max(n,max(((n/100)*10+n%10),n/10))<<endl;//三个中取最大值 
	return 0; 
}
```

## Content
有一个人的银行账户里有 $n$ 元钱，他可以删去倒数第二位获最后一位数字来使得账户里面的钱最大。请求出他的银行账户余额的最大值。

**数据范围：$10\leqslant |n|\leqslant 10^9$。**
## Solution
这道题目很简单，枚举倒数第二位和最后一位数字删去之后的账户余额，然后比较哪个更大就好。当然，如果是正数的话不需要删，肯定是越删余额越少的，毕竟谁都不希望把自己原来就有的钱变少。

具体我们用一个字符串实现，读入，如果第一个字符不是负号，那么就直接输出结束。否则，将负号以外的数一一枚举删除，然后比较答案更新。最后取最小值带负号输出，相当于最大值。不过这里要注意的一点就是，$-0$ 是不合法的，所以，当答案是 $0$ 时，我们不需要输出前面的负号。
## Code
说明一下：

- $\texttt{isdigit}$ 函数用来判断这个字符是否是数字。
- $\texttt{(3f3f3f3f)}_{16}$ 表示的是十进制下的 $1061109567$，可以通过这道题目的数据。注意取最小值初始化时一定要尽可能大！
- 中间那个注释的部分使我中间调试时用的。这对今后的竞赛中很有用，但注意正式提交代码时要把中间调试的部分注释掉！

```cpp
#include <cstdio>
#include <cstring>
#include <string>
#include <algorithm>
using namespace std;

char a[17];

int main() {
	scanf("%s", a);
	if(isdigit(a[0]))	return printf("%s", a), 0;
	else {
		int len = strlen(a), ans = 0x3f3f3f3f;
		for(int i = len - 2; i < len; ++i) {
			int s = 0;
			for(int j = 1; j < len; ++j)
				if(j != i)	s = s * 10 + a[j] - '0';
//			printf("delete %d: %d\n", i, s);
			ans = min(ans, s); 
		}
		if(ans)	printf("-");
		return printf("%d", ans), 0;
	}
}
```

### 氵

这一道题真的很简单，大概意思就是：

- 如果$n$是正数，就直接输出

- 如果$n$是负数，就看$n$倒数第一位(个位)大，还是倒数第二位(十位)大，然后输出删掉大的那一位后的$n$(因为欠的钱越少越好)

求出个位的代码是：

	n/10;
    
求出个位的代码是：

	((n/100)*10+n%10);
    
然后我们就可以看代码了：

```cpp
#include<iostream>
#include<cstdio>
#include<cmath>
#include<string>
#include<cstring>
#include<algorithm>
using namespace std;
int n;
int ge,shi;
int main(){
	cin>>n; //输入
	if(n>=0) cout<<n;  //正数输出
	else{
		ge=n/10; //个位
		shi=((n/100)*10+n%10); //十位
		if(ge>shi) cout<<ge;
		else cout<<shi;  //比较大小输出
	}
	return 0; //完结撒花
}
```

# Splits

## 题目描述

我们定义正整数$n$的分裂为一个由正整数组成的不上升序列，且序列数字和为$n$

举个栗子：下列这些序列都是$8$的分裂：$[4,4],[3,3,2],[2,2,1,1,1],[5,2,1]$

下列这些序列不是$8$的分裂：$[1,7],[5,4],[11,-3],[1,1,4,1,1]$

一个分裂的权是序列第一个数出现的次数，举个例子：$[1,1,1,1,1]$的权是$5$，$[5,5,3,3,3]$的权是$2$，$[9]$的权是$1$

现在给出$n$，求$n$的分裂有多少个不同的权

## 样例 #1

### 输入

```
7
```

### 输出

```
4
```

## 样例 #2

### 输入

```
8
```

### 输出

```
5
```

## 样例 #3

### 输入

```
9
```

### 输出

```
5
```

# 题解

## 作者：Tweetuzki (赞：4)

考虑一个贪心，我们只用两种数 $1$ 和 $2$ 组成 Split。因为选用更大的数对答案是没有贡献的。

于是 Split 不同权重的数量就是能割出最多的 $2$ 的个数（因为可以在这些 $2$ 当中随便拆，拆成这么多种不同的权重）。容易求得，$n$ 中有 $n/2$ （下取整）个 $2$。再加上 $n$ 个 $1$ 组成的 Split。于是就有 $n / 2 + 1$ 种不同的权重。

所以答案就是 $n/2+1$。

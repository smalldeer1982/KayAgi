# Restaurant Tables

## 题目描述

在一个小餐馆里，有$a$ 张单人桌和$b$ 张双人桌。
（单人桌就是只能坐一个人，双人桌是能坐两个，原文是只能容纳一人的坐的桌子和能容纳两人的坐的桌子，太长了）

可以知道的是，今天将会有$n$ 组人来，每组都是一人或两人。

如果一组只有一人，他将被安排坐在一个空的单人桌。如果不存在（空的单人桌），他将被安排坐在一个空的双人桌。如果不存在（空的双人桌），他将被安排坐在一个有一个人坐的双人桌。如果仍然不存在（一个人坐的双人桌），餐馆将拒绝为这组人服务。

如果一组有两人，他们将被安排坐在一个空的双人桌。如果不存在（空的双人桌），餐馆将拒绝为这组人服务。

你被给与了这些组按时间到来的情况。你要确定餐馆将拒绝为多少人提供服务。

## 说明/提示

在第一个样例中，第一组有一个人，它坐在一个空的单人桌上。下一组坐了一整个双人桌。第三组有一个人，坐在剩下的双人桌上的一个位置。第四组有一个人，他坐在双人桌的剩余的座位上。因此，所有顾客能被服务。

在第二个样例中，，第一组有一个人，它坐在一个空的单人桌上。下一组有一个人，坐在双人桌上的一个位置上。已经不可能坐下两个人，所以餐馆拒绝为他们（第三组的两个人）服务。第四组有一个人，他坐在双人桌的剩余的座位上。因此，该餐馆拒绝为$2$ 名顾客提供服务。

## 作者：LLGZ (赞：2)

**题意**

其实这题并不难，只是题意难以读懂，这里我解释一下：

输入$m$,表示有m组顾客，输入这个饭店有$Own$个单人桌，$Twin$个双人桌。  
然后输入$m$组数据，每组数据有一个数$s$,这个数只能是1或2，表示这组顾客有多少人。  
Then，在每一组数据中判断是否拒绝        
- 如果s为1  

优先选择单人桌，$Own--$；

没有单人桌，选择双人桌,$Twin--$,$Twin1++$($Twin1$指只做了一位的双人桌);

如果再没有，选择只坐了单人的双人桌，$Twin1--$;

还是没有的话则拒绝$sum++$;
- 如果s为2

只能选择空的双人桌$Twin--$;

没有则拒绝$sum++$;

贴上代码
```
#include<iostream>
using namespace std;

int m,Own,Twin,s,sum;
int Twin1;

int main()
{
	cin >> m>> Own >> Twin;
	
	for(int i=1;i<=m;i++)
	{
		
		cin >> s;
		
		if(s==1)
		{
			if(Own > 0)
			Own --;
			
			else
			if(Twin > 0)
			{
				Twin --;
				Twin1 ++;
			}
			
			else
			if(Twin1 > 0)
			Twin1 --;
			
			else sum ++;
		}
		else 
		{
			if(Twin > 0)
			Twin --;
			
			else sum += 2;
		}
		
	}
	
	cout<<sum;
	
	return 0;
}
```

## 作者：_Qer (赞：1)

本题不需要什么高端东西，老老实实模拟就能AC。

开3个变量，分别存单人桌、双人桌和坐了一个人的双人桌，分别存对应桌子的剩余个数，按顺序输入来的顾客，按照题目要求安排进座位就可以了。

```cpp
#include<bits/stdc++.h>
using namespace std;
int n, a, b, hb, t, ans;
//依次为顾客组数、单人桌数、双人桌数、
//坐了一个人的双人桌数、每组顾客的数量、拒绝顾客的数量
int main(){
    cin >> n >> a >> b;
    //输入
    //这里省略了一句话
    //hb = 0;全局变量默认初始值为0
    for (int i = 0; i < n; ++i) {
        cin >> t;//输入人数
        if (t == 1) {//如果是一个人
            if (a > 0) {//优先单人桌
                --a;
            } else if (b > 0) {//然后空双人桌
                --b;
                ++hb;//双人桌坐了一个人，变成了hb
            } else if (hb > 0){//最后看有没有坐了一个人的双人桌
                --hb;
            } else {
                ++ans;//已经没座位了，拒绝顾客数+1
            }
        } else {
            if (b > 0) {//只能做双人桌，没有就只能拒绝
                --b;
            } else {
                ans += 2;//注意这里是拒绝两个顾客（一组来两个人）
            }
        }
    }
    cout << ans << endl;
    return 0;
}
```

## 思路讲解：

首先，先把单人桌剩余、双人桌剩余和容纳两人却坐了一个人的桌目前剩余的数量用 $p,q,r$ 存下来，然后如果来了 $1$ 个人，那么优先给单人桌，没有的话给双人，再没有的话给一个已经坐了一个人的双人桌，再不行就不能接待了。来了 $2$ 个人也是同理，先给双人，双人不行直接不接待。（好像比单人简单）

## AC CODE:

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,a,b;
	cin>>n>>a>>b;
	int p,q,r;
	p=a,q=b,r=0;//单人、双人、双人做了一人
	int ans=0;
	int k;
	for(int i=1;i<=n;i++)
	{
		cin>>k;
		if(k==1)//来了1人
		{
			if(p>0)p--;
			else if(q>0)
			{
			    q--;
			    r++;
			}
			else if(r>0)r--;
			else ans++; 
		}
		if(k==2)//来了2人
		{
			if(q>0)q--;
			else ans+=2;
		}
	}
	cout<<ans;
	return 0;
}

```

## 作者：ZepX_D (赞：0)

## 题意
有 $ a $ 张单人桌和 $ b $ 张双人桌，有 $ n $ 组人，让你安排桌子，看有多少人坐不下。

## 分析
- 单人：有单人桌就先坐单人桌，否则坐双人桌（要先坐空的）。
- 双人：有空的双人桌就服务，否则计数器加二。

注意的是要输出饭店拒绝服务了**多少人**，而**不是多少组**。

## 代码
```cpp
#include<cstdio>

using namespace std;

int main()
{
	int n,a,b,ans=0,sum=0;
	scanf("%d%d%d",&n,&a,&b);
	
	for (int i=1;i<=n;i++)
	{
		int op;
		scanf("%d",&op);
		switch(op)
		{
			case 1:
			{
				if (a) a--;//如果有空的单人桌就把他安排到单人桌
				else if (b)//否则如果有双人桌，就把他安排到双人桌
				{
					b--;//空的双人桌要少一个
					sum++;//坐了一个人的双人桌多了一个
				}
				else if (sum)//如果有坐了一个人的双人桌，就把他安排到那里
					sum--;
				else ans++;//否则饭店拒绝为他服务
				break;
			}
			case 2:
			{
				if (b) b--;//如果有空的双人桌就服务
				else ans+=2;//否则拒绝服务
				break;
			}
		}
	}
	printf("%d",ans);
	return 0;
}
```

## 作者：lightningZ (赞：0)

# 思路
因为单人肯定先坐单人桌，如果没有单人桌就去坐双人桌。由于这个双人桌已经被人坐了，所以其他的双人不会选那个双人桌，就需要一个数组记录有多少个被坐了但是没坐满的双人桌，而剩下的双人桌才是能坐的，否则就只能拒绝接客。

但是要注意，单人会第一时间选择空着的双人桌，所以只有双人桌都有人坐的时候才会去拼桌，而这时就不可能接待双人的客人。
# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,x,y,a[200005],ans,z;
//x是单人桌
//y是双人桌
//z是做了一个人的双人桌
int main()
{
	scanf("%d%d%d",&n,&x,&y);
	for(int i=1;i<=n;i++)
	{
		scanf("%d",&a[i]);
		if(a[i]==1)
		{
			if(x!=0) x--;//有单人桌就直接坐
			else if(z<2*y)//只要没坐满就可以一直坐双人桌，因为是一个人坐双人桌，所以可以坐2*y个人
			{
				z++;
			}
			else ans++;//没桌子就拒绝
		}
		else
		{
			if(z<y) y--;//如果有空闲的双人座就直接坐，z>y就证明所有双人座都有一人（讨厌的人）
			else ans+=2;//否则拒绝
		}
	}
	printf("%d",ans);
	return 0;
}
```

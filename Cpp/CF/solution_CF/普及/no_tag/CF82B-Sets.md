# Sets

## 题目描述

小瓦西亚非常喜欢玩由正整数组成的集合。为了使游戏更有趣，瓦西亚选择了n个非空集，使它们中没有两个具有共同的元素。

有一天，他想向他的朋友们展示玩数字是多么有趣。为此，他在n*(n-1)/2张纸片上写下了每两个不同集合之间所有可能的并集。然后他把纸片洗了一遍。之后按照任意的顺序写下了不同并集中的数字。

例如，如果 n=4 ，并且实际集合的形式如下{1，3},{5},{2，4},{7}则集合对的数量等于 6。这六张纸可以包含以下数字：

* 2,7,4 ;

* 1,7,3 ;

* 5,4,2 ;

* 1 ,3,5 ;

* 3,1,2,4 ;

* 5,7.

然后瓦西亚把这些纸片给他的朋友看，但对他们保密了原来的n个集合。他的朋友设法成功计算出瓦西亚最初想到了哪些集合。那你呢，你能用给定的纸片恢复原来的n个集合吗？

## 样例 #1

### 输入

```
4
3 2 7 4
3 1 7 3
3 5 4 2
3 1 3 5
4 3 1 2 4
2 5 7
```

### 输出

```
1 7 
2 2 4 
2 1 3 
1 5 
```

# 题解

## 作者：_edge_ (赞：0)

CF 评分 1700 ，实际上就是一道 pj T2 的题目。

我们发现每一个集合里面的数都是不同的，所以如果两个并集里面有相同的元素，他们两个的交集肯定是一个完整的集合。

上面的想法很容易想到，具体怎么实现 …… 以及复杂度的问题，这些细节还是要考虑周到。

我们发现集合里面的数都是不同的，因此只要维护一个桶，去枚举一遍这里，把这里的数全部都扔进去，或者之前他们就以及被扔进去了，这时候直接枚举两个集合的相同元素。

代码难度不大，但是有几个坑，如果你是 RE 请检查数组，有些数组要开到 $n \times (n-1)/2$ ！！！！

如果你是 WA 了 test $33$ 或者 $34$，请特判 $n=2$，因为这个情况下只有一个并集。

代码附上，如有问题还请指出。

```cpp
#include <iostream>
#include <cstdio>
#include <map>
#include <vector>
#define pb push_back
using namespace std;
const int INF=205;
const int INFN=INF*INF;
int n,a[INFN][INF],k[INFN],Map[INFN],Map1[INFN],tot;
vector <int> ans[INFN];
signed main()
{
        ios::sync_with_stdio(false);
        cin>>n; int kk=n*(n-1)/2;
        for (int i=1; i<=kk; i++) {
                cin>>k[i];
                for (int j=1; j<=200; j++) Map1[j]=0;
                for (int j=1; j<=k[i]; j++) {
                        cin>>a[i][j]; Map1[a[i][j]]=1;
                        // cout<<a[i][j]<<" "<<Map1[4]<<"\n";
                }
                for (int j=1; j<=k[i]; j++) {
                        if (Map[a[i][j]]==-1) continue;
                        if (Map[a[i][j]]!=0) {
                                int l=Map[a[i][j]]; tot++;
                                for (int p=1; p<=k[l]; p++) {
                                        if (Map1[a[l][p]]) {
                                                // cout<<a[l][p]<<" over\n";
                                                ans[tot].pb(a[l][p]);
                                                Map[a[l][p]]=-1;
                                        }
                                }
                                // cout<<"ok\n";
                                Map[a[i][j]]=-1;
                        }
                        else Map[a[i][j]]=i;
                }
        }
        for (int i=1; i<=tot; i++) {
                cout<<ans[i].size()<<" ";
                int len=ans[i].size();
                for (int j=0; j<len; j++)
                        cout<<ans[i][j]<<" ";
                cout<<"\n";
        }
        if (n==2) {
                cout<<"1 "<<a[1][1]<<"\n";
                cout<<k[1]-1<<" ";
                for (int i=2; i<=k[1]; i++)
                        cout<<a[1][i]<<" ";
                cout<<"\n";
        }
        return 0;
}

```

# Pasha and Tea

## 题目描述

Pasha 决定去邀请他的朋友参加茶会。在那样的场合中，他有一个容量为 $w$ 毫升的茶壶和 $2n$ 个茶杯。每个茶杯只供 Pasha 的一个朋友使用（也就是说不能多个人共用一个茶杯）。第 $i$ 个茶杯可以装下 $a_i$ 毫升的水。

结果，在 Pasha 的朋友之中，有 $n$ 个男孩和 $n$ 个女孩，并且他们都会来到茶话会。为了让每个人都满意，Pasha 决定按照如下的方法来为茶倒水：
- Pasha 可以通过向茶壶中倒入 $w$ 毫升的水来把水一次烧开；
- Pasha 给每个男孩倒入相同体积的茶水；
- Pasha 给每个女孩倒入相同体积的茶水；
- 如果每个女孩能得到 $x$ 毫升的水，那么每个男孩可以得到 $2x$ 毫升的水。

换句话说，每个男孩应该得到的茶水是每个女孩的两倍。

Pasha 非常和蔼礼貌，所以他想使他倒给朋友的茶水的总量最大。你的任务是帮助他决定给他的每个朋友倒水的最佳方案。

## 说明/提示

### 进入正题，题目主要意思：
不得不说一句，题目的题面真的太绕了。

题目意思大概就是：Pasha 要给 $n$ 个男生和 $n$ 个女生倒茶，整个茶壶有 $w$ 毫升，所有男生喝的茶水量**一样多**，所有女生喝的茶水量也**一样多**，而女生喝的茶水量是男生的**一半**。

问题：求 Pasha 要倒的最大水量是多少。

---------------------------------------

### 思路分析：
很简单的贪心。

为了让答案**更大**，我们就应该**保证所有茶杯都填满了。**

我们做此题的步骤是：  
1. 先把茶杯的装水量从小到大进行排序。  
`sort(a+1,a+n*2+1);`
2. 接着计算每个人最少分配多少水。前 $n$ 人是女生，后面的都为男生。因此，$s$ 应该是第 $1$ 个数和第 $\displaystyle \frac{n+1}{2}$ 的最小值，也就是最小的茶杯容量和男生茶杯容量的一半的最小值。     
`double s=min(a[1],a[1+n]/2);`
4. 最后计算总的茶水量，用 $s \times 3 \times n$ 就是最终的答案 $ans$。但很坑的一点是 $ans$ 不能超过一次倒水量，所以答案要和茶壶总容积 $w$ 取最小值。   
`double ans=min(s*3*n,w*1.0);`

---------------------------------------

### AC Code（带注释）：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,w;
double a[200005];
int main(){
	cin>>n>>w;
	for(int i=1;i<=n*2;i++){
		cin>>a[i];
		a[i]=a[i]*1.0;
	}	
	sort(a+1,a+n*2+1);//因为男生的水量比女生的大，所以要排序。

	double ans=min(s*3*n,w*1.0);//得出的答案就是茶水量总和和茶壶总容积中的最小值。 
	printf("%.7lf",ans);//答案要保留7位小数。 
	return 0;
}
```

## 题目大意

## 分析

一道贪心题。

显然男生比女生的水量大，那么就要让男生的茶杯比女生的大一些，所以排序。

```cpp
sort(a + 1, a + n * 2 + 1);
```

但是结果 $cnt$ 不能超过一次倒水量，所以将它和 $x \times 3 \times n$ 进行比较，取较小值。

```cpp
cnt = min (x * 3 * n, w * 1.0000000);
```

最后记得保留精度即可。

```cpp
printf("%.7lf", cnt);
```

## Code

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, w;
double x, cnt;
double a[200007];
signed main()
{
	scanf ("%d%d", &n, &w);
	// 有2*n个人，输入2*n个茶杯 
	for( int i = 1; i <= 2 * n; i++)
	{
		scanf ("%lf", &a[i]);
		// 7位精度 
		a[i] *= 1.0000000; 
	}
	// 排序 
	sort(a + 1, a + n * 2 + 1);
	// 一次倒的x要在女生<=的a[1]和2*男生<=的a[n+1]里选较小值 
	x = min (a[1], a[n + 1] / 2);
	// 在总量和一次倒的水量取较小值，作为答案 
	cnt = min (x * 3 * n, w * 1.0000000);
	// 输出保留7位小数 
	printf("%.7lf", cnt);
	return 0;
}
```

### 解题思路
为了解决这个问题，我们需要理解问题的约束和目标。目标是**最大化**给所有朋友倒的水的总量，同时满足以下条件：

1. 每个男孩得到的茶水是每个女孩的两倍。

2. 每个女孩得到 $x$ 毫升的水，每个男孩得到 $2x$ 毫升的水。

3. 男孩和女孩的人数都是 $n$，所以总共有 $2n$ 个茶杯。

4. 茶壶的容量是 $w$ 毫升。

所以，我们需要确定每个女孩和每个男孩能得到的**最大水量**。由于茶壶的容量**是固定的**，我们需要**确保所有茶杯都能被填满**。因此，我们需要找到一个 $x$ 值，使得所有茶杯的容量之和不超过 $w$。

### 带注释代码

```cpp
#include<bits/stdc++.h>//万能头
using namespace std;
int boy,girl;//声明女,男
double sum,cmp;
double a[200001];
double qp=1.0000000;
int main(){
	cin>>boy>>girl;
	for(int i=1;i<=2*boy;i++){
		cin>>a[i];
		a[i]*=qp;
	}
	sort(a+1,a+boy*2+1);//sort排序
	sum=min(a[1],a[boy+1]/2);//计算一下
	cmp=min(sum*3*boy,girl*qp);

}

## 题目大意：
#### 关于题面：
虽说题目大意很简单，但由于题面问题，很多人依旧看不懂，看不懂的可以参考一下上面的链接，接下来我将会正式开始讲解题目意思。
#### 意思：
有 $n$ 个男生和相同数量的女生，一共有 $w$ 毫升水，我们将要把这些水分给他们，一共有和男女生数量一样多的杯子，每个男生的水的数量是一样的，女生也同理，男生喝的水是女生的两倍，我们要尽可能让分给大家的水更多。
## 思路：
超级简单，显然男生喝的水肯定比女生多，因此，大的杯子可以给男生用，小的可以给女生用，给杯子排序，一个 `sort` 函数就可以解决。  
男生和男生之间，女生和女生之间，每个人喝的水都得一样多，显然，要想让大家喝的尽可能多，我们要按照一个标准来，我们以最小杯子为标准，每个人喝的水都是这个杯子的容积，这样才能让每个人喝的水最多，如果比标准少了（除不够之外），就不能最优：比标准多了，就人没法喝水了，同时我们也要注意男女生喝的水量不同，这样的话该怎么立标准呢？很简单，我们可以用男生标准的一半和女生标准做对比，取其中更小的作为标准，这个标准是女生的标准，因为男生的标准是女生的两倍，因此在不考虑水量的情况下一个男生和一个女生加起来的水是标准的三倍，如果 $ans$ 表示标准，那么最多能分的水就是 $ans \times 3 \times n$。  
注意，本题要保留七位小数，因为误差不能超过 $10^{-6}$，也就是小于 $0.000001$。

当你写出这个代码后开开心心地去提交时，你惊奇地发现居然错了，想必你肯定是忘了 $w$ 这个数，我们给男女生们准备的水可不能超过 $w$，于是我们还得加个判断。
#### 正确代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,w;
    cin>>n>>w;
    double a[2*n+1];
    for(int i=1;i<=2*n;i++){
    	cin>>a[i]; 
	}
	sort(a+1,a+2*n+1);     //按杯子的大小来排序。
	double ans=min(a[1],a[n+1]/2);   //找出标准。
	ans=ans*3*n;    //最多能分的水。

### 思路
简单贪心 $+$ 小数精度问题。

题目非常的绕，其实意思如下：

- 有 $n$ 个男生，$n$ 个女生。（$2n$ 个人）
- 水壶有 $w$ 毫升。（容量）

- 求出最大的水体积数。（问题）

每个女生和男生得到的水都由得到水最少的女生决定。

为什么呢，因为所有男生女生水的体积一样多，设女生水的体积为 $x$，那么男生为 $2x$。

每个人得到水的多少由水壶的容积决定，所以先对水壶容积从小到大排序，然后在第 $1$ 个数（$x$）和第 $\frac{n + 1}{2}$ 个数取一个最小值再乘上 $3n$，因为不能超过一次倒水量，所以结果再与 $w$ 取一个最小值即可，最后输出结果保留 $7$ 位小数。

## 前言。
简单的贪心。

题意简述：有 $2\times n$ 个杯子和 $W$ 升的水，其中每个杯子有自己的体积，男生女生各有 $n$ 个人，一人一个杯子，要求男生的水的体积一样多，女生的体积一样多，且女生喝的水是男生的一半，求喝掉的最大的水的体积数。
## 分析。
明显我们要使男生的杯子的体积尽可能大，然后分配杯子，计算男生喝的和女生喝的总水量。具体来说，将杯子平均分成两类，一类我们称之为较大的杯子，另一类我们称之为较小的杯子。当较小的杯子中最小的那个杯子的体积的两倍，比较大的杯子中体积最小的那个杯子的体积要小的时候，说明女生喝的水的水量最大就是该小杯子的体积。反之，当小杯子中最小的那个杯子的体积的两倍，比大杯子中体积最小的那个杯子的体积要大的时候，说明男生喝的水的水量最大就是该大杯子的体积。

这么选取的原因是：因为要平均分配，不能说单纯地考虑说直接取较大的杯子中的体积最大的杯子，因为它的一半可能会比较小的杯子中的那个最小的要多，不成立，或者水不够分。

```

### 题目大意
现在有 $n$ 个男生，$n$ 个女生，每个人都有一个茶杯，容量为 $a_i$。每个男生必须喝相同量的茶，每个女生也必须喝相同量的茶，男生喝的茶量是女生喝的茶量的 $2$ 倍。现在有一个容量为 $w$ 的茶壶，问倒给朋友的茶水的总量最大是多少。

### 思路
由于男生分到的茶杯容量肯定大于女生，所以我们先把 $a$ 数组从小到大进行排序，再将 $a_1 \thicksim a_n$ 分给女生，$a_{n+1} \thicksim a_{2n}$ 分给男生。

但考虑到男生喝的茶量是女生喝的茶量的 $2$ 倍这个条件，如果 $2 \times a_1 \gt a_{n+1}$，女生喝的茶量就要减小。所以女生喝的茶量就是 $\min(a_1,\frac{1}{2}a_{n+1})$。

这里还有一个隐藏条件。如果发现男生和女生喝的茶量比 $w$ 大，那么他们最多也只能喝 $w$ 的茶。

### 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
const double eps=1.0000000;
int n,w;
double a[200001],tmp,ans;//注意有男生和女生范围，数组要乘 2 
int main(){
	cin>>n>>w;
	for(int i=1;i<=n*2;i++){//注意一共有 2n 个人 
		cin>>a[i];
		a[i]*=eps;//精度 
	}
	sort(a+1,a+n*2+1);//对茶杯容量进行排序
	tmp=min(a[1],a[n+1]/2);//女生喝的茶量
	ans=min(n*tmp*3,w*eps);//答案
	printf("%.7lf",ans); 
}
```

## 题目分析

共有 $w$ 升水，$n$ 个男生，$n$ 个女生。

因为每个女生和男生各自都得到相同体积的水，所以每个女生和男生得到的水都由得到水最少的女生决定。

也就是说，若女生中得到最少水的人有 $x$ 升，那么其他每个女生都有 $x$ 升，男生都有 $2x$ 升。


## 思路

得到水的多少由茶壶的容积决定。对茶壶容积从小到大进行排序，第 $1$ 个数就是上文所说的 $x$。

但这并不能保证答案的正确性，因为现在求出的 $x$ 仅仅考虑了女生而没有考虑男生。

错误原因可以通过例子理解：

假如有 $1$ 个女生，$1$ 个男生，茶壶容积为 $2$ 和 $3$。

这时 $x$ 为 $2$，但 $2x$ 为 $4$，超过了男生茶壶的容积 $3$，出现错误。

在这种情况下，$x$ 就不是 $a$ 数组的第一个数而是 $a$ 数组的第 $n+1$ 个数除以 $2$。

还有一个需要注意的点是给出的总容积 $w$ 的大小。

继续给出例子：

假如有 $1$ 个女生，$1$ 个男生，总容积为 $3$，茶壶容积为 $2$ 和 $4$。

现在男生茶壶的容积满足 $2x$ 的要求，但总共只有 $3$ 升水，不能把所有的茶壶倒满。

在这种情况下，水一定可以倒完，所以答案就是总容积 $w$。

需要注意的是，由于茶壶的容积并不一定是整数，所以要使用浮点数类型存储 $a$ 数组的数据。

本题要求误差小于 $10^{-6}$，结果保留 $7$ 位小数输出即可通过本题。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
#define maxn 200005
int n,w;
double ans,a[maxn];
int main(){
	cin>>n>>w;
	for(int i=1;i<=2*n;i++){
		cin>>a[i];
		a[i]*=1.0000000;
	}
	sort(a+1,a+2*n+1);
	ans=min(a[1],a[n+1]/2);//得出x的大小 
	ans=min(ans*n*3,w*1.0000000);//验证总容积是否满足x 
	cout<<fixed<<setprecision(7)<<ans;
	return 0;
} 
```

# CF557B 题解

[题目传送门](https://www.luogu.com.cn/problem/CF557B)

### 题目大意

Pasha 有一个大小为 $w$ 的大茶壶，他请来了 $n$ 个男生和 $n$ 个女生。他一共有 $2 \times n$ 个体积分别为 $a_i$ 茶杯且男生喝水量为女生的一倍。求他能倒给他的朋友的水的最大体积。

### 题目分析

- **核心算法：贪心**
- **关键部分**
  
  + 排序：为了方便划分男女茶杯。
    ```cpp
    	 sort(cup + 1, cup + n * 2 + 1);
    // 对每个茶壶的体积排序
    ```
  + 求解最小值：从倒茶实际数量和茶壶容积中选。
    ```cpp
    v = cup[1] > cup[n + 1] / 2 ? cup[n + 1] / 2 : cup[1];
    // 在男生与女生之间选最小值
    ans = min(v * 3 * n, w * 1.0000000);
    // 在一次倒的量与茶壶体积取最小值
    ```
- **小贴士：运算与输出保留7位以上小数。**

### AC Code 

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, w;
double v, ans;
double cup[200010];
signed main()
{
    cin >> n >> w; // 读入茶杯数和茶壶大小
    for (int i = 1; i <= 2 * n; i++)
    {
        cin >> cup[i];
        // 读入每个茶杯的大小
        cup[i] *= 1.0000000000;
        // 为了应对题目的精度，最好把答案×10的-10次方
    }
    sort(cup + 1, cup + n * 2 + 1);
    // 对每个茶壶的体积排序
    v = cup[1] > cup[n + 1] / 2 ? cup[n + 1] / 2 : cup[1];
    // 在男生与女生之间选最小值
    ans = min(v * 3 * n, w * 1.0000000);
    // 在一次倒的量与茶壶体积取最小值
    printf("%.7lf", ans);
    // 对于题目的数据范围，保留7位小数
    return 0;
}
```

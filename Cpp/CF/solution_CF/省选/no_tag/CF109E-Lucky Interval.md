# Lucky Interval

## 题目描述

Petya 喜欢幸运数字。

幸运数字的定义是：在这个数的十进制表示中，它只含有数字 $4$ 或 $7$。

现在给定一个区间 $\left[a,a+l-1\right]$。

让你求一个最小的整数 $b$ 使得 $f(a)=f(b)$， $f(a+1)=f(b+1)$，$f(a+2)=f(b+2)$......
$f(a+l-1)=f(b+l-1)$。

定义： $f(x)$ 表示数字 $x$ 中幸运数字出现的个数。

## 说明/提示

$1 \le a,l \le 10^9$。

## 样例 #1

### 输入

```
7 4
```

### 输出

```
17
```

## 样例 #2

### 输入

```
4 7
```

### 输出

```
14
```

# Chess Championship

## 题目描述

一场国际象棋比赛，有 $M$ 个玩家参加，每对玩家恰好玩一场比赛。胜利加 $2$ 分，平局 $1$ 分，输不加分。

你不知道具体的情况，只知道排名前 $n$ 的人的分数，构造一种胜负情况，使得这个排名是正确的。


第二行 $n$ 个整数，表示排名前 $n$ 的人的分数。

## 样例 #1

### 输入

```
5 5
8 6 4 2 0
```

### 输出

```
yes
XWWWW
LXWWW
LLXWW
LLLXW
LLLLX
```

## 样例 #2

### 输入

```
5 1
9
```

### 输出

```
no
```

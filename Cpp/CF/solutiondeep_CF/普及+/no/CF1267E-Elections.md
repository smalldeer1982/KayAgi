# Elections

## 题目描述

有n(n<=100)个人,m(m<=100)个投票站,每个投票站对每个人都有一定的票数(<=1000).  
第n个是坏蛋,请问去除掉哪几个投票站才能使他的票数不大于其他每个人中的一个，也就是不能让第n个人的总票数最多。

## 样例 #1

### 输入

```
5 3
6 3 4 2 8
3 7 5 6 7
5 2 4 7 9
```

### 输出

```
2
3 1 
```

## 样例 #2

### 输入

```
2 1
1 1
```

### 输出

```
0

```

## 样例 #3

### 输入

```
3 3
2 3 8
4 2 9
3 1 7
```

### 输出

```
3
1 2 3 
```

### 综合分析与结论
这些题解的核心思路都是先枚举前 $n - 1$ 个人中哪一个最终票数超过第 $n$ 个人，然后针对每个人计算需要删除哪些投票站才能使该人的总票数超过第 $n$ 个人，最后取删除投票站数量的最小值。

- **思路**：枚举前 $n - 1$ 个人，对于每个人，计算每个投票站中第 $n$ 个人与该人的票数之差，将这些差值排序，然后贪心选择差值大的投票站删除，直到该人的总票数超过第 $n$ 个人。
- **算法要点**：枚举、贪心、排序。
- **解决难点**：如何确定删除哪些投票站能使某个人的总票数超过第 $n$ 个人。通过计算票数差并排序，优先删除差值大的投票站，保证每次删除都能最大程度缩小差距。

### 重点代码
```cpp
#include<bits/stdc++.h>
using namespace std;
#define ll long long
const ll N=110;
ll n,m,a[N][N],ans=LONG_LONG_MAX,t,p2[N],ct,p[N];
struct Vote{
    ll v,id;
}a2[N];
bool cmp(Vote l,Vote r){
    return l.v>r.v; // 按差从大到小排序 
}
ll solve(ll x){
    t=ct=0;
    for(ll i=1;i<=m;i++) a2[i].v=a[i][n]-a[i][x],t+=a2[i].v,a2[i].id=i;
    // 数组a2存的是反叛者与第x个人每轮中的票数之差
    // id用来存放投票站编号 
    sort(a2+1,a2+m+1,cmp); // 排序 
    while(t>0) t-=a2[++ct].v,p2[ct]=a2[ct].id; // 贪心取差的最大值 
    return ct;
}
int main(){
    ios::sync_with_stdio(false);cin.tie(0);cout.tie(0);
    cin>>n>>m;
    for(int i=1;i<=m;i++)
        for(int j=1;j<=n;j++) cin>>a[i][j];
    for(int i=1;i<n;i++){ // 枚举哪个人票数超过反叛者 
        t=solve(i);
        if(t<ans){
            ans=t;
            for(int j=1;j<=ans;j++) p[j]=p2[j];
            // 更新答案 
        }
    }        
    cout<<ans<<"\n";
    for(int i=1;i<=ans;i++) cout<<p[i]<<" ";
    return 0;
}
```
**核心实现思想**：
1. `solve` 函数：计算为了使第 $x$ 个人的总票数超过第 $n$ 个人，需要删除的投票站数量。先计算每个投票站中第 $n$ 个人与第 $x$ 个人的票数之差，排序后贪心选择差值大的投票站删除，直到第 $x$ 个人的总票数超过第 $n$ 个人。
2. `main` 函数：枚举前 $n - 1$ 个人，调用 `solve` 函数计算需要删除的投票站数量，取最小值作为最终答案。

### 关键思路或技巧

# Brackets

## 题目描述

一个由括号组成的序列被称作**平衡的**，当且仅当这个序列可以通过在字符串中加入“+”号或数字 $1$。例如，序列"(())()", "()" 和 "(()(()))"是**平衡的**，但是")(", "(()" 和 "(()))(" 不是平衡的。  



给定一个由左右圆括号组成的字符串，请你判断它是否是**平衡的**。

## 样例 #1

### 输入

```
(()(()))()
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
())()
```

### 输出

```
NO
```

### 题解综合分析与结论
这些题解主要围绕判断括号序列是否平衡展开，核心思路是检查左右括号的匹配情况。主要有以下两类做法：
1. **栈做法**：遇到左括号入栈，遇到右括号时，若栈非空则弹出栈顶元素（左括号），否则说明右括号无匹配，序列不平衡。遍历结束后，若栈为空则序列平衡，否则不平衡。此方法直观，符合括号匹配的逻辑顺序。
2. **非栈做法**：使用一个或两个变量记录左右括号的数量关系。一个变量时，遇到左括号变量加1，遇到右括号变量减1，过程中若变量小于0或遍历结束后变量不为0则不平衡；两个变量时，分别记录左右括号数量，过程中若右括号数量大于左括号数量则不平衡，遍历结束后若两者数量不相等则不平衡。这种做法空间复杂度低，代码简洁。

```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
stack<int> st;
int main()
{
  cin>>s;
  int n=s.size();
  for(int i=0;i<n;i++)
  {
    if(s[i]=='(')st.push(1);
    if(s[i]==')')
    {
      if(st.empty())
      {
        cout<<"NO";
        return 0;
      }
      else st.pop();
    }
  }
  if(st.empty())cout<<"YES";
  else cout<<"NO";
  return 0;
}
```

```cpp
#include<bits/stdc++.h>
using namespace std;
string s; 
int ans,cnt;
int main()
{
	cin>>s;
	int l=s.size();
	for(int i=0;i<l;i++){
		if(s[i]=='(') ans++;
		if(s[i]==')') cnt++;
		if(cnt>ans){
			puts("NO");
			return 0;
		}
	}
	puts(ans!=cnt?"NO":"YES");
}
```

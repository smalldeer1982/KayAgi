# Hot Days

## 题目描述

### 题意简述
$A$ 地和 $B$ 地由一条贯穿 $n$ 个区域的公路连接起来。每个区域都有独特的气候，所以第 $i$  $\left(1\le i \le n\right)$ 个区域的夏季气温会稳定在 $t_i$ 度。

有 $m$ 个学生要从 $A$ 地到 $B$ 地去旅游，他们乘坐大巴车旅游，但巴士内的温度会随地区与车内人数的变化而变化，行驶在第 $i$ 个地区的温度为 $t_i+k$ ( $k$ 为车内人数)，但当车内温度超过 $T_i$ 时，学生会要求组织旅行的人赔偿，需赔偿每人 $x_i$ 元。

组织者需要在每个地区**都**购买任意数量公交车以完成旅行，且他们可以分配孩子到公交车上，分配不需要代价，在第 $i$ 个地区买一辆公交车需 $cost_i$ 。

```

120
```

## 样例 #2

### 输入

### 题解综合分析与结论
所有题解思路基本一致，均认为每个地区的决策相互独立，采用贪心策略解决问题。对于每个地区，考虑两种策略：一是不赔偿，计算需要租的大巴车数量并算出花费；二是只租一辆大巴车并支付赔偿费用，算出该花费。同时注意到当Ti≤ti时为特殊情况，只能采用租一辆车并赔偿的方式。最后取两种策略花费的最小值，累加每个地区的最小花费得到总花费。各题解主要差异在于代码实现的细节和表述清晰度。

```cpp
#include <bits/stdc++.h>

using ll = long long;
using namespace std;

int main()
{
    ll n, m, sum = 0;
    cin >> n >> m;

    for(int i = 1; i <= n; i++)
    {
        ll t, T, x, cost;
        cin >> t >> T >> x >> cost;

        if(T <= t)
            sum += cost + m * x;

        else
            sum += min((ll)(ceil(1.0 * m / (T - t)) * cost), (cost + m * x));
    }

    cout << sum << endl;
    return 0;
}
```

```cpp
#include <bits/stdc++.h>
using namespace std;
#define int long long
int t[100005], T[100005], x[100005], a[100005];

signed main() {
    int n, m;
    cin >> n >> m;
    int ans = 0;
    for (int i = 1; i <= n; i++) {
        cin >> t[i] >> T[i] >> x[i] >> a[i];
        int k = a[i] + m * x[i];
        if (T[i] <= t[i])
            ans += k;
        else {
            int p = ceil(m * 1.0 / (T[i] - t[i])) * a[i];
            ans += min(p, k);
        }
    }
    cout << ans;
}
```

```cpp
#include<bits/stdc++.h>
using namespace std;
const long long N = 100000 + 10;
long long t[N],T[N],x[N],cost[N];
long long n,m;
long long sum = 0;
int main()
{
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);
    cin >> n >> m;
    for (long long i = 1;i <= n;i++){
        cin >> t[i] >> T[i] >> x[i] >> cost[i];
    }
    for (long long i = 1;i <= n;i++){
        long long k = cost[i] + m * x[i];
        long long kk = ceil(1.0 * m / (T[i] - t[i])) * cost[i];
        if (T[i] <= t[i]) sum += k;
        else sum += min(k,kk);
    }
    cout << sum << "\n"; 
    return 0;
}
```

### 最优关键思路或技巧
1. **贪心策略**：将每个地区看作独立子问题，分别考虑不赔偿和只租一辆车赔偿这两种策略，取花费最小的方案，这是解决本题的核心思路。
2. **特殊情况处理**：当Ti≤ti时，意识到无论如何都要赔偿，只能选择租一辆车并赔偿的方案，避免出现错误结果。

# Triangle

## 题目描述

There is a right triangle with legs of length $ a $ and $ b $ . Your task is to determine whether it is possible to locate the triangle on the plane in such a way that none of its sides is parallel to the coordinate axes. All the vertices must have integer coordinates. If there exists such a location, you have to output the appropriate coordinates of vertices.

## 样例 #1

### 输入

```
1 1
```

### 输出

```
NO
```

## 样例 #2

### 输入

```
5 5
```

### 输出

```
YES
2 1
5 5
-2 4
```

## 样例 #3

### 输入

```
5 10
```

### 输出

```
YES
-10 4
-2 -2
1 2
```

### 综合分析与结论
这些题解的核心思路大多基于数学原理，通过固定一个顶点在原点，利用勾股定理、向量垂直关系（或相似三角形性质）来寻找满足条件的其他顶点坐标。主要难点在于如何确保找到的坐标满足三边不平行于坐标轴且均为整数的条件。部分题解通过暴力枚举相关变量来寻找解，不同题解在枚举方式、判断条件及代码实现细节上有所差异。

    - **重点代码核心思想**：枚举 $x$，根据勾股定理计算 $y$，通过求最大公约数得到最简比例，再结合另一直角边长度 $b$ 计算出另一组坐标，判断是否满足条件。
    ```cpp
    for (int x = 1; x < a; x++) {
        int y = int(sqrt(a*a - x*x) + 0.001);
        if (y * y + x * x == a * a) { 
            int g = __gcd(x, y);
            x /= g, y /= g; 
            int k = int(sqrt(b*b / (x*x + y*y)) + 0.001);
            if (k*k * (x*x + y*y) == b*b) { 
                if (g*x!= -k*y && g*y!= k*x) { 
                    cout << "YES\n"; 
                    cout << "0 0\n";
                    cout << g*x << " " << g*y << "\n";
                    cout << -k*y << " " << k*x << "\n";
                    return 0;
                }
            }
            x *= g; 
        }
    }
    ```

### 最优关键思路或技巧
固定一个顶点在原点，利用向量垂直关系（或相似三角形性质）结合勾股定理，通过合理的枚举方式来寻找满足条件的顶点坐标。在代码实现中，注意处理浮点数运算的精度问题（如使用 `sqrt` 函数后判断结果是否为整数）以及对不满足三边不平行坐标轴条件的解进行排除。

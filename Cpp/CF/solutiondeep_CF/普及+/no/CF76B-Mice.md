# 题目信息

# Mice

## 题目描述

在平面上给出两条直线 $y = Y_0$ 和 $y = Y_1$。在 $y = Y_0$ 上有 $n$ 只老鼠，第 $i$ 只老鼠横坐标为 $x_{1, i}$，在 $y = Y_1$ 上有 $m$ 个奶酪，第 $i$ 个奶酪横坐标为 $x_{2, i}$。已知一只老鼠的捕食策略如下：

$1.$ 如果离这只老鼠最近的奶酪有且只有一个，那么这只老鼠会往这个奶酪移动。

$2.$ 如果有多个奶酪离老鼠距离最近，那么这只老鼠会选择向使所有老鼠中挨饿个数最少的奶酪移动。

每只老鼠的移动速度都是一样的，当某些老鼠到达某个奶酪并且当前奶酪还没有被吃掉时，他们会吃掉奶酪并且不再挨饿。如果某个老鼠在到达奶酪时奶酪已经被吃掉了，那么它不会再进行移动，此时我们认为它处于挨饿状态。

请输出最终处于挨饿状态的老鼠个数。

## 样例 #1

### 输入

```
3 2 0 2
0 1 3
2 5
```

### 输出

```
1
```

### 综合分析与结论
这三道题解均围绕如何确定老鼠选择奶酪的策略从而得出挨饿老鼠数量展开。Lily_White 提出贪心策略，从左到右考虑老鼠选择奶酪，证明了贪心的正确性，时间复杂度 \(O(n + m)\)，代码实现清晰；chenxia25 提出用 DP 解决，通过处理红点（奶酪）对应黑点（老鼠）集合，对红点序列进行 DP 转移，但未给出详细代码，思路理解有一定难度；WaterSun 同样采用贪心，先确定唯一选择奶酪的老鼠，再处理可选择两个奶酪的老鼠，给出证明过程和代码，代码实现稍显复杂。综合来看，Lily_White 的题解思路清晰，代码可读性强，质量较高。

### 所选的题解
- **Lily_White（5星）**
  - **关键亮点**：提出清晰的贪心策略，并详细证明其正确性，代码简洁明了。
  - **个人心得**：无
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long LL;
#define rep(i, n) for (int i = 0; i < (n); i++)
#define repn(i, n) for (int i = 1; i <= (n); i++)
int n, m, ans;
const int N = 100010, INF = 0x3f3f3f3f;
int x[N], y[N], best[N];
int main() {
  scanf("%d%d%*d%*d", &n, &m);
  rep(i, n) scanf("%d", x + i);
  repn(i, m) scanf("%d", y + i);
  y[0] = -INF;
  y[m + 1] = INF;
  memset(best, INF, sizeof(best));
  rep(i, n) {
    int j = lower_bound(y, y + m + 2, x[i]) - y;
    int dis = min(x[i] - y[j - 1], y[j] - x[i]);
    if (y[j] - x[i] > dis ||
        (x[i] - y[j - 1] == dis && (best[j - 1] == dis || best[j - 1] == INF)))
      --j;
    if (best[j] == INF || best[j] == dis) ++ans;
    best[j] = min(best[j], dis);
  }
  printf("%d\n", n - ans);
  return 0;
}
```
核心代码实现思想：先对奶酪位置数组进行边界扩展，遍历每只老鼠，通过二分查找找到离它最近的两个奶酪，根据贪心策略确定选择的奶酪，并更新相关状态，最后统计挨饿老鼠数量。

# 题目信息

# The least round way

## 题目描述

给定由非负整数组成的 $n\times n$ 的正方形矩阵，你需要寻找一条路径：

+ 以左上角为起点。
+ 每次只能向右或向下走。
+ 以右下角为终点。
+ 如果我们把沿路遇到的数进行相乘，积应当以尽可能少的 $0$ 结尾。

## 样例 #1

### 输入

```
3
1 2 3
4 5 6
7 8 9
```

### 输出

```
0
DDRR
```

### 题解分析与总结

#### 1. 算法思路
所有题解都采用了动态规划（DP）的思路，核心思想是通过统计路径上每个位置的2和5的因子数量，找到一条使得路径上2和5因子数量最少的路径。最终的结果是取2和5因子数量的最小值，因为一个0的产生需要一对2和5。

#### 2. 解决难点
- **因子统计**：每个位置的数需要分解质因数，统计2和5的因子数量。
- **路径选择**：在DP过程中，需要记录路径的选择（向右或向下），以便最终输出路径。
- **0的处理**：如果矩阵中存在0，那么经过0的路径的尾0个数恒为1，需要与不经过0的路径进行比较。

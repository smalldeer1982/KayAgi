# Kids' Riddle

## 题目描述

程序员的孩子们能在5-10分钟内解决这个谜题，你能多快完成？

给定一个十进制整数，将其转换为十六进制后，统计各位数字中"圈圈"的数量。规则如下：
- 数字0、4、6、9、A、D各含1个圈
- 数字8、B各含2个圈
- 其他字符无圈

## 样例 #1

### 输入
```
11
```

### 输出
```
2
```

## 样例 #2

### 输入
```
14
```

### 输出
```
0
```

## 题解分析与结论

### 关键思路
1. **查表法**：预先建立0-15的圈数映射表，直接通过取余运算获取十六进制各位的值，避免字符串转换。
2. **位运算优化**：用`n&15`替代`n%16`，`n>>=4`替代`n/=16`提升效率。
3. **边界处理**：通过`ans=!n`的初始化技巧，简洁处理n=0的特殊情况。

### 最优题解

#### 题解作者：CZQ_King（⭐⭐⭐⭐⭐）
**核心亮点**：
- 位运算与查表法结合，代码极简（仅10行）
- 预处理答案变量巧妙处理边界
- 时间复杂度O(logn)，空间复杂度O(1)

```cpp
#include<cstdio>
int n,cir[]={1,0,0,0,1,0,1,0,2,1,1,2,0,1,0,0};
int main(){
    scanf("%d",&n);
    int ans=!n;  // 预处理n=0的情况
    while(n){
        ans+=cir[n&15]; // 取最后4位（十六进制位）
        n>>=4;          // 右移4位等价于除以16
    }
    printf("%d",ans);
}
```

```cpp
const int dic[]={1,0,0,0,1,0,1,0,2,1,1,2,0,1,0,0};
int main() {
    int n,sum=0;
    cin>>n;
    if(n==0) sum=1;     // 特判n=0
    while(n) {
        sum += dic[n%16]; // 查表累加
        n /= 16;
    }
    cout<<sum;
}
```

## 扩展与总结

### 技巧总结
- **进制转换优化**：直接取余比字符串转换效率更高，适用于无需输出进制结果的情况
- **预处理映射表**：将离散判断转换为数组查表，提升代码可维护性
- **位运算加速**：在进制问题中，位运算比算术运算更高效

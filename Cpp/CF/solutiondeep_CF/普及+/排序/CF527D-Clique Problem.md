# Clique Problem

## 题目描述

数轴上有 $n$ 个点，第 $i$ 个点的坐标为 $x_i$，权值为  $w_i$。两个点 $i,j$ 之间存在一条边当且仅当 $abs(x_i-x_j)\geq w_i+w_j$ 。 你需要求出这张图的最大团的点数。

团的定义：两两之间有边的顶点集合。

## 样例 #1

### 输入

```
4
2 3
3 1
6 1
0 2
```

### 输出

```
3
```

# 团问题

## 题目描述

数轴上有 $n$ 个点，第 $i$ 个点的坐标为 $x_i$，权值为 $w_i$。两个点 $i,j$ 之间存在一条边当且仅当 $|x_i-x_j|\geq w_i+w_j$。你需要求出这张图的最大团的点数。

团的定义：两两之间有边的顶点集合。

## 样例 #1

### 输入
```
4
2 3
3 1
6 1
0 2
```

### 输出
```
3
```

---

### 题解分析与结论
**关键思路**：将每个点转化为区间 $[x_i-w_i, x_i+w_i]$，问题转化为寻找最多互不重叠的区间数（最大不相交区间问题）。贪心策略按右端点排序后依次选取，时间复杂度 $O(n\log n)$。

**最优解法亮点**：
1. 问题转化巧妙：将几何条件转化为区间覆盖问题
2. 贪心策略高效：经典右端点排序+线性扫描
3. 边界处理严谨：初始右端点设为极小值避免遗漏

### 高星题解推荐
#### 1. NaCly_Fish（⭐⭐⭐⭐⭐）
**核心思路**：
- 将点转化为线段，推导出 $l_i \geq r_j$ 的充要条件
- 结构体排序实现优雅，代码注释详细
- 包含快读优化，适合大数据量

**关键代码**：
```cpp
sort(a+1,a+1+n);
int r = -inf, ans = 0;
for(int i=1;i<=n;++i){
    if(a[i].l >= r) ++ans, r = a[i].r;
}
```

#### 2. Siyuan（⭐⭐⭐⭐）
**实现亮点**：
- 使用索引数组排序节省内存
- 变量命名清晰（las代替r增强可读性）
- 代码简洁高效，无冗余逻辑

**代码片段**：
```cpp
std::sort(id+1,id+n+1,cmp);
int las = -2e9;
for(int i=1;i<=n;++i) if(las <= l[id[i]]) ++ans, las = r[id[i]];
```

#### 3. Randyhoads（⭐⭐⭐⭐）
**特色分析**：
- 运算符重载实现直观比较
- 变量初始化采用位运算极值
- 代码风格紧凑，适合竞赛快速实现

**核心实现**：
```cpp
sort(t+1,t+n+1);
int r = -(1<<30), ans=0;
for(int i=1;i<=n;i++)
    if(t[i].l >= r) ans++, r = t[i].r;
```

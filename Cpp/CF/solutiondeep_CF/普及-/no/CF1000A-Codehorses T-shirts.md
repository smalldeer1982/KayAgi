# Codehorses T-shirts

## 题目描述

**题目大意：**

$n$个字符串和$n$个模板串，在$1s$的时间内你可以选择一个字符串（不是模板串）修改任意多的字符，但是不能删除或增加字符，问至少需要多少s使得字符串变为模板串，注意不考虑串的顺序，即最后每个字符串出现次数=这个串在模板串中的出现次数

# AI分析结果



### 题目翻译
**题目名称**：Codehorses T-shirts

**题目描述**：  
给定两组各 $n$ 个字符串，第一组为原字符串，第二组为目标模板串。每次操作可以修改任意一个原字符串的任意多个字符（不能增删字符），要求将原字符串组修改为与模板串组完全匹配（出现次数相同，顺序无关）。求最少需要多少次操作。

**输入样例**：  
```
3
XS
XS
M
XL
S
XS
```
**输出样例**：  
```
2
```

**分类标签**：贪心、字符串、模拟

---

### 题解分析与结论
**核心思路**：  
1. **字符串频次统计**：使用哈希表（如 `map`）统计原字符串和目标模板串的出现次数。
2. **直接抵消匹配项**：遍历模板串，若原字符串中存在相同项，则抵消（频次减1）。
3. **剩余未匹配项即答案**：未被抵消的原字符串和模板串的数量即为最少操作次数（因为每次操作可以完全修改一个字符串）。

**关键技巧**：  
- 利用哈希表高效统计和操作字符串频次。
- 贪心策略优先抵消完全相同的字符串，减少需处理的问题规模。

### 精选题解（评分≥4星）
#### 1. 冰糖鸽子（五星）
**亮点**：  
- 思路清晰，代码简洁高效。
- 直接通过频次抵消计算未匹配项，时间复杂度 $O(n \log n)$。
**核心代码**：  
```cpp
map<string, int> ans;
int rans = n;
// 统计原字符串频次
for (int i=1; i<=n; i++) {
    cin >> fs;
    ans[fs]++;
}
// 抵消匹配项并计算剩余操作次数
for (int i=1; i<=n; i++) {
    cin >> fs;
    if (ans[fs]) {
        ans[fs]--;
        rans--;
    }
}
cout << rans << endl;
```

#### 2. 风浔凌（四星）
**亮点**：  
- 通过差值统计未匹配项，思路与抵消法等价。
- 代码清晰，使用迭代器遍历哈希表。
**核心代码**：  
```cpp
map<string, int> m;
// 统计原字符串频次
for (int i=1; i<=n; i++) {
    cin >> s;
    m[s]++;
}
// 统计模板串频次并计算差值
for (int i=1; i<=n; i++) {
    cin >> s;
    m[s]--;
}
// 累加正差值（未匹配项）
int ans = 0;
for (auto it = m.begin(); it != m.end(); it++) {
    ans += max(0, it->second);
}
cout << ans << endl;
```

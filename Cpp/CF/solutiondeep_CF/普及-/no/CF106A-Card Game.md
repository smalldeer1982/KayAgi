# 题目信息

# Card Game

## 题目描述

一种纸牌游戏。
有许多张卡牌，每张卡牌有一个套装（“S”，“H”，“D”和“C”）和一个等级（按升序排列“6”，“7”，“8”，“9”，“T”， “J”，“Q”，“K”和“A”）
**规则：**
以下情况卡牌A才能赢卡牌B:
1、A有王牌套装，B没有
2、A和B的套装相同，但A的等级比B高
否则B赢

## 样例 #1

### 输入

```
H
QH 9S
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
S
8D 6D
```

### 输出

```
YES```

## 样例 #3

### 输入

```
C
7H AS
```

### 输出

```
NO```

1. **条件拆分**：明确两种获胜条件：①A有王牌套装且B无；②同套装时比较等级。
2. **等级映射**：用数组或map预处理等级顺序，便于快速比较。
3. **输入处理**：直接读取卡牌字符串的第二个字符判断套装，第一个字符判断等级。

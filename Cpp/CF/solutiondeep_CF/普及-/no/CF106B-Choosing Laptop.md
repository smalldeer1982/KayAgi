# Choosing Laptop

## 题目描述

Vasya想购买一台笔记本电脑。商场提供了n台电脑给顾客体验。
Vasya只对以下三个电脑指标比较感兴趣：CPU速度，内存和硬盘。因为Vasya是一个程序员而不是游戏玩家，因此他对其他指标并不感兴趣。
如果一台电脑的三个指标都小于另一台电脑，那么Vasya认为这台电脑是过时的。Vasya不考虑那些过时的电脑，他会在其他（不过时）电脑中选择一台最便宜的。

## 说明/提示

第一和第五台电脑是过时的，因为它们的三个指标都小于第三台。剩下的电脑中第四台最便宜，因此，输出是4。

## 样例 #1

### 输入

```
5
2100 512 150 200
2000 2048 240 350
2300 1024 200 320
2500 2048 80 300
2000 512 180 150
```

### 输出

```
4```

# 题目重写

## 题目描述

Vasya想购买一台笔记本电脑。商场提供了n台电脑供顾客体验。  
Vasya只关注以下三个指标：CPU速度、内存和硬盘。若某台电脑的三个指标均严格小于另一台，则视为过时。Vasya会排除所有过时的电脑，并在剩余的电脑中选择最便宜的一台。

## 输入格式

第一行输入整数n，表示电脑数量。  
接下来n行，每行输入四个整数，分别表示CPU速度、内存、硬盘和价格。

## 输出格式

输出所选电脑的编号（从1开始）。

## 样例输入

```
5
2100 512 150 200
2000 2048 240 350
2300 1024 200 320
2500 2048 80 300
2000 512 180 150
```

## 样例输出

```
4
```

## 题解分析与结论

### 核心思路
1. **过时判定**：对每台电脑i，遍历所有其他电脑j，若存在j的三个指标均大于i，则i过时。
2. **最优选择**：在未过时的电脑中，选择价格最低且编号最小的。

### 关键技巧
- **双重循环枚举**：通过两重循环实现所有电脑对的比较，时间复杂度O(n²)在n≤100时可行。
- **状态标记优化**：使用布尔数组或变量实时标记过时状态，减少后续无效判断。
- **结构体封装**：将电脑属性封装为结构体提升代码可读性。

**亮点**：变量命名清晰，逻辑简洁，利用布尔标记快速判定。

int main() {

            if(s[i]<s[j] && r[i]<r[j] && h[i]<h[j]) pd = true;
        }
        if(!pd && minn>c[i]) { // 未过时且价格更低

            ans = i;
        }
    }
    cout << ans;
}
```

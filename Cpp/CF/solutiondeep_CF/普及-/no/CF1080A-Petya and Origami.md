# Petya and Origami

## 题目描述

Petya 将要举行一场派对，她要发送大量的邀请函给朋友们。

每一张邀请函需要耗费 2 张红色纸，5 张绿色纸和 8 张蓝色纸。

然而，商店里卖纸是一堆一堆卖的，每一堆有 $k$ 张且只含有红，绿，蓝三种颜色之一的纸，Petya 可以选择买哪种颜色的一堆纸。

现在 Petya 要给 $n$ 名朋友发送邀请函，请问她至少要到商店买几堆纸。

## 说明/提示

$1 \leq n,k \leq 10^8$

## 样例 #1

### 输入

```
3 5
```

### 输出

```
10
```

## 样例 #2

### 输入

```
15 6
```

### 输出

```
38
```

## 题解综合分析

所有题解均采用向上取整计算各颜色所需堆数的思路。核心公式为：  
总堆数 = ⌈2n/k⌉ + ⌈5n/k⌉ + ⌈8n/k⌉  
关键差异在于向上取整的实现方式：

1. **浮点运算派**：使用`ceil`函数，需注意转为浮点数
2. **整数运算派**：通过`(a + k - 1)/k`实现，避免浮点误差
3. **模运算派**：用`if(a % k != 0)`手动加1

### 高星题解推荐

#### 题解作者：Ptilopsis_w（5星）
**关键亮点**：使用整数运算技巧`(a + k -1)/k`，避免了浮点数的精度风险，代码简洁高效，适合大数场景。
```cpp
int s = (n*2+k-1)/k + (n*5+k-1)/k + (n*8+k-1)/k;
```

#### 题解作者：Zechariah（4星）

int a[4] = {0 ,2 ,5 ,8};
for(int i = 1; i <= 3; i++)

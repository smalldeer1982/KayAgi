### 题目中文重写
**统一字符串**

#### 题目描述
给定两个整数 $n$ 和 $k$，构造一个长度为 $n$ 的字符串 $S$，要求满足以下条件：
1. 使用英文字母表前 $k$ 个小写字母，每个字母至少出现一次；
2. 最大化所有字母中出现次数最少的字母的出现次数。

你需要回答 $t$ 组独立询问。

#### 样例输入
```
3
7 3
4 4
6 2
```

#### 样例输出
```
cbcacab
abcd
baabab
```

### 题解综合分析

#### 核心思路对比
所有题解的核心思路均基于**循环轮转分配字符**的构造策略：
1. **鸽巢原理**：通过 `i % k` 的余数循环分配字符，保证每个字母的出现次数为 `⌊n/k⌋` 或 `⌈n/k⌉`。
2. **贪心分配**：依次输出前 $k$ 个字母并循环，确保剩余字符均匀分配。

#### 解决难点
- 确保每个字符至少出现一次。
- 剩余字符的分配要尽可能均匀，以最大化最小出现次数。

---

### 精选题解（评分≥4星）

#### 题解1：Yi_Fan（5星）
**关键亮点**：
- 代码简洁高效，直接通过 `i % k` 实现循环分配。
- 时间复杂度 $O(n)$，无额外空间消耗。
```cpp
for (int i = 1; i <= n; i++)
    cout << char(i % k + 'a');
```

#### 题解2：Ouaoan（5星）
**关键亮点**：
- 与 Yi_Fan 方法等效，但使用显式循环维护字符偏移。
- 直接输出 `j%y + 'a'` 避免中间变量。
```cpp
for (int j = 1; j <= x; j++)
    printf("%c", j % y + 'a');
```

#### 题解3：Zechariah（4星）
**关键亮点**：
- 使用动态索引 `now` 轮转字符，逻辑清晰。
- 适用于需要动态调整循环起点的场景。
```cpp
int now = 0;
for (int i = 1; i <= n; ++i)
    putchar('a' + now), (++now) %= k;
```

---

### 最优关键思路
**循环轮转分配**：通过取模运算或动态索引，将字符依次分配到前 $k$ 个字母，使得每个字母的出现次数尽可能均匀。数学上保证最小出现次数为 `⌊(n-1)/k⌋ + 1`，满足最大化要求。

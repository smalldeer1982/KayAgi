# Two distinct points

## 题目描述

给定你数轴上的两条线段 $[l_1, r_1],[l_2, r_2]$（可能相交，重叠或重合）。你需要找到两个整数 $a,b$ 满足 $l_1 \leq a \leq r_1, l_2 \leq b \leq r_2$，且 $a\neq b$。你需要回答 $q$ 个询问，可能存在多种答案，输出任意一个即可。

## 样例 #1

### 输入

```
5
1 2 1 2
2 6 3 4
2 4 1 3
1 2 1 3
1 4 5 8
```

### 输出

```
2 1
3 4
3 2
1 2
3 7
```

### 题解对比与结论

#### 核心思路总结
所有解法均围绕「选择端点+冲突处理」展开，通过固定选择特定端点（如左端点或右端点）并处理相等情况，确保解的存在性。关键点在于利用区间端点特性快速构造答案，避免复杂计算。

#### 高星题解推荐

1. **Reywmp（5星）**  
   **关键亮点**：  
   - 优先选择第一个区间的左端点 `a=l1` 和第二个区间的右端点 `b=r2`。  
   - 若 `l1 == r2`（冲突），则将 `b` 调整为第二个区间的左端点 `l2`。  
   **代码简洁性**：直接处理冲突，逻辑清晰。  
   **代码片段**：  
   ```cpp
   if (l1 == r2) printf("%d %d\n", l1, l2);
   else printf("%d %d\n", l1, r2);
   ```

### 最优思路总结
**关键技巧**：  
1. **端点优先**：选择端点（如左端点或右端点）作为候选值，减少计算量。  
2. **冲突调整**：若端点组合出现 `a == b`，调整其中一个端点至另一合法端点（如另一区间的端点）。  
3. **利用区间性质**：题目保证区间长度至少为1，调整后的端点必合法。

# Petya and Inequiations

## 题目描述

对于给出的整数 $n, x, y,$ 找出一个正整数数组 $a$ ，满足：

$\begin{cases}a_1^2+a_2^2+...+a_n^2\geq x\\a_1+a_2+...+a_n\leq y\end{cases}$

## 说明/提示

对于 $100 \%$ 的数据：

$1\leq n\leq 10^5$

$1\leq x\leq 10^{12}$

$1\leq y\leq 10^6$

## 样例 #1

### 输入

```
5 15 15
```

### 输出

```
4
4
1
1
2
```

## 样例 #2

### 输入

```
2 3 2
```

### 输出

```
-1
```

## 样例 #3

### 输入

```
1 99 11
```

### 输出

```
11
```

## 题解综合分析

所有题解的核心思路均为**贪心构造**：将前 $n-1$ 个数设为 $1$，最后一个数设为 $y-n+1$，此时平方和达到最大可能值 $(y-n+1)^2 + (n-1)$。若此值仍不足 $x$ 或 $y < n$，则无解。该思路的正确性基于数学证明：将两个数的平方和最大化时，应使其中一个尽可能大，其余尽可能小。

## 关键思路与技巧
1. **极端构造法**：在总和约束下，将尽可能多的元素设为最小值（通常为1），剩余元素取最大值
2. **平方和性质**：$(a+b)^2 > a^2 + b^2$（当$a,b>1$时），证明将数值集中更优
3. **边界特判**：当 $y < n$ 时无法构造正整数数组，直接返回-1

# Nastya Is Reading a Book

## 题目描述

Nastya决定读一本有 $n$ 个章节的书。这本书的每一页都属于一章中，同样的，每一章至少有 $1$ 页。

昨晚 Nastya 没有看完书, 所以她在第 $k$  页做下了记号，第 $k$ 页就是还没有读的第 $1$ 页（换句话说，她读了从第 $1$ 页到第 $\left(k-1\right)$ 页。

第二天， Nastya 的朋友 Igor 来问她,还有多少章没读。 但是 Nastya 太忙了, 所以让你来求出她还有多少章节**没完全读完**。

### 题目中文重写
# Nastya 在读书

## 题目描述

Nastya 决定阅读一本包含 $n$ 个章节的书。每个章节至少包含 1 页，且所有页码连续分配（每页严格属于一个章节）。

昨晚 Nastya 在第 $k$ 页做了标记，表示从该页开始尚未阅读（即已阅读完第 $1$ 到 $(k-1)$ 页）。

请计算她还有多少章节**未完全读完**（即章节中存在至少一页未被阅读）。

---

### 算法分类
**模拟**

### 题解分析与结论

#### 核心思路
所有题解均基于以下观察：
1. 当第 $k$ 页位于第 $i$ 章区间内（即 $l_i \leq k \leq r_i$），则：
   - 第 $i$ 章未完成（包含未读页）
   - 所有后续章节（$i+1$ 到 $n$）也包含未读页
   - 总未完成章节数为 $n-i+1$

2. 解法只需遍历章节，找到第一个满足 $k \leq r_i$ 的章节即可。

#### 最优思路提炼
直接**线性遍历章节**，时间复杂度 $O(n)$，空间复杂度 $O(1)$。此方法无需预处理，适用于任意数据规模，是效率最高的解法。

### 精选题解

#### 1. 作者：HoshizoraZ（5星）
**关键亮点**  
- 代码简洁高效，逻辑清晰  
- 使用立即返回优化执行  

```cpp
#include <cstdio>
int n,k,l[110],r[110];
int main(){
    scanf("%d",&n);
    for(int i=1;i<=n;i++) scanf("%d%d",&l[i],&r[i]);
    scanf("%d",&k);
    for(int i=1;i<=n;i++){
        if(k <= r[i]){
            printf("%d\n",n-i+1);
            return 0; // 立即终止，无需后续判断
        }
    }
    return 0;
}
```

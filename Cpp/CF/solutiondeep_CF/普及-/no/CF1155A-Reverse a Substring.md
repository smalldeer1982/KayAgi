# Reverse a Substring

## 题目描述

给定一个**仅含小写字母**的字符串$s$，其长度为$n$

我们定义子串为一个字符串中连续的一段，比如```acab```是```abacaba```的子串（位置是```3~6```），而```aa```和```d```不是。所以对于一个字符串$s$，它的位置为$[l,r]$的子串可以表示成$s[l;r]$，即$s_ls_{l+1}...s_r$

您需要指定$s$的**一个**子串并翻转这个子串，使得新字符串的字典序比原来的字符串$s$小。注意不是最小。

如果可以满足题意，输出```YES```，再输出反转的区间。否则输出```NO```

我们认为字符串$x<y$当且仅当存在一个 $i$ $(1 \leq i\leq min(|x| ,|y|))$，使得 $x_i < y_i$ 并且$x_j =y_j (1 \leq j < i)$ 此处的绝对值符号```|x|``` 指的是字符串长度。在某些语言中您可以用 $<$ 运算符比较字符串字典序

## 说明/提示

样例$1$中，翻转后的字符串是```aacabba```

## 题目描述

给定一个**仅含小写字母**的字符串$s$，其长度为$n$。

子串定义为字符串中连续的一段。例如，`acab`是`abacaba`的子串（位置为3~6），而`aa`和`d`不是。你需要选择一个子串进行翻转，使得新字符串的字典序比原字符串小。输出任意一个可行解。

## 题解综合分析

### 核心思路
所有题解均基于以下关键观察：**若存在相邻字符满足$s[i] > s[i+1]$，反转这两个字符即可使字典序减小**。该思路本质是贪心策略，只需找到第一个逆序对即可。

### 解决难点
- **字典序比较特性**：逆序对的第一个不同字符决定了字典序大小，只需保证存在至少一个位置使得翻转后的字符更小。
- **线性扫描优化**：无需考虑复杂情况，遍历字符串一次即可找到解，时间复杂度$O(n)$。

## 关键思路总结
**贪心策略**：找到第一个相邻逆序对$(i, i+1)$，翻转这两个字符即可。由于字典序比较的特性，第一个逆序对的反转必然导致新字符串字典序更小。

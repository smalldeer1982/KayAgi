# Long Number

## 题目描述

给定一个一个n位数，以及1~9数字的替换数。现请你在这个n位数上选一个连续的数段，将段内的每个数字替换成相应数字的替换数。要求：输出按照要求替换后的最大n位整数。

## 样例 #1

### 输入

```

1 2 5 4 6 6 3 1 9
```

```
1557
```


### 综合分析与结论
这些题解的核心思路都是基于贪心策略。因为要使替换后的n位整数最大，应优先考虑高位数字的增大，所以从高位向低位遍历原数字。当遇到可替换数字大于当前数字时，开始替换，并持续替换后续数字，直到遇到可替换数字小于当前数字时停止。各题解的不同之处主要在于细节处理和代码实现方式，如判断开始替换和停止替换的条件写法、数据的存储方式（数组或字符串）等。

核心实现思想：遍历数组存储的数字，比较当前数字与其可替换数字，若可替换数字大则替换并标记开始替换；若可替换数字小且已开始替换则停止。

### 最优关键思路或技巧
1. **贪心策略**：利用高位数字对整体数值影响更大的特点，优先从高位开始寻找可增大的替换机会，一旦遇到替换后会使数字变小的情况则立即停止替换，以保证得到最大的n位整数。
2. **边界条件判断**：在判断是否开始替换和停止替换时，准确设置条件。开始替换条件为可替换数字大于当前数字；停止替换条件为可替换数字小于当前数字且已经开始了替换操作。

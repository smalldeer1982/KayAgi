# 题目信息

# Little Pigs and Wolves

## 题目描述

在一个大小为$n*m$的二维网格里有几只猪和几只狼，这个网格里每一个单元只包含一只猪或一只狼。

如果猪和狼在同一条公共边上，则我们就说他们相邻。因为小猪怕狼，所以每只猪旁边只可能会有一只狼，但是每只狼旁边可能会有好几头猪。

狼将选择其中与其相邻的一头并吃掉，此过程将不再重复。也就是说，一头狼只能吃一头猪，而小猪在被吃掉后会消失且不能被别的狼吃。

求最多有多少只猪会被狼吃掉。

## 样例 #1

### 输入

```
2 3
PPW
W.P
```

### 输出

```
2
```

## 样例 #2

### 输入

```
3 3
P.W
.P.
W.P
```

### 输出

```
0
```

# 中文题目重写

## 题目描述

在一个大小为 $n \times m$ 的二维网格中有若干只小猪和狼，每个单元格只包含一只小猪（用 `P` 表示）或一只狼（用 `W` 表示），或为空（用 `.` 表示）。

当小猪和狼相邻（共享一条公共边）时，狼可以选择吃掉这只小猪。每只狼只能吃一头小猪，且每头小猪被吃掉后无法再被其他狼食用。求最多有多少只小猪会被吃掉。

## 样例 #1

### 输入

```
2 3
PPW
W.P
```

### 输出

```
2
```

## 样例 #2

### 输入

```
3 3
P.W
.P.
W.P
```

### 输出

```
0
```

# 题解分析与结论

## 关键思路总结
1. **模拟法核心**：利用题目中「每只小猪周围最多只有一只狼」的隐含条件，直接统计每只狼周围是否存在可吃的小猪。
2. **边界处理**：在检查狼的四个相邻方向时，必须判断坐标是否合法（不越界）。
3. **标记已吃小猪**：部分题解通过修改地图标记被吃小猪，避免重复统计（但根据题意，此操作非必需）。

## 高分题解推荐

### 题解作者：dang_dang（5星）
**关键亮点**：
- 完全正确处理边界条件，通过显式判断坐标合法性避免越界。
- 代码逻辑清晰，条件判断顺序合理。
- 使用 `continue` 跳过无效方向，减少冗余计算。

**核心代码**：
```cpp
#include<iostream>
using namespace std;
#define MAXN 11
char arr[MAXN][MAXN];
int main() {
    int n,m, eaten=0;
    cin >> n >> m;
    for(int i=0; i<n; i++)
        for(int j=0; j<m; j++)
            cin >> arr[i][j];
    
    for(int i=0; i<n; i++) {
        for(int j=0; j<m; j++) {
            if(arr[i][j] != 'W') continue;
            if(i+1 < n && arr[i+1][j] == 'P') {
                eaten++;
                arr[i+1][j] = '.';
                continue;
            }
            if(i-1 >= 0 && arr[i-1][j] == 'P') {
                eaten++;
                arr[i-1][j] = '.';
                continue;
            }
            if(j+1 < m && arr[i][j+1] == 'P') {
                eaten++;
                arr[i][j+1] = '.';
                continue;
            }
            if(j-1 >= 0 && arr[i][j-1] == 'P') {
                eaten++;
                arr[i][j-1] = '.';
            }
        }
    }
    cout << eaten;
    return 0;
}
```

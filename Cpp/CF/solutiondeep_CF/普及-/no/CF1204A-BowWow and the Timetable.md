# BowWow and the Timetable

## 题目描述

已知圣彼得堡的车站会在第 $4^k(k\geq 0)$ 分钟开出新的一趟火车，已知BowWow团队将在第 $s(0\leq s\leq 2^{100})$ 分钟时到达火车站，他们想知道自己错过了多少趟火车；换句话说，在第 $s$ 分钟之前开出了多少趟火车。举个例子，当 $s=20$ 时，他们错过了第 1、4 和 16 分钟的火车。现在只有你知道这个时间 $s$，请你求出他们错过了多少趟火车。

### 综合分析与结论
这些题解主要围绕如何利用二进制数的特性来解决问题。由于输入的\(s\)为二进制形式，且火车出发时间为\(4^k\)，而\(4 = 2^2\)，所以重点在于分析二进制数的位数及各位数字情况。各题解思路相似，主要差别在于实现细节和对特殊情况的处理方式。

```cpp
#include <cstdio>
#include <cstring>
using namespace std;
char in[110];
int l,ans=0;
int main() {
	scanf("%s",in);
	l=strlen(in);
	if(l%2)
		for(int i=l-1; i>=1; i--) 
        if(in[i]=='1') {
				printf("%d",1+l/2);
				return 0;
			}
	printf("%d",l/2);
	return 0;
}
```
核心实现思想：读入二进制字符串，获取长度。若长度为奇数，检查除最高位外是否有\(1\)，若有则结果为长度除以\(2\)再加\(1\)，否则直接为长度除以\(2\)。

- **作者：So_noSlack (赞：1)  星级：4星  关键亮点：思路清晰，对二进制长度奇偶两种情况分类讨论明确，代码简洁**
```cpp
#include<iostream>
using namespace std;

string str;

int main(){
    cin >> str; 
    int n = str.length(); 
    if(!(n & 1)) {
        cout << n / 2;
        return 0; 
    }
    for(int i = 1; i < n; i ++)
        if(str[i] == '1'){
            cout << n / 2 + 1; 
            return 0;
        }
    cout << n / 2; 
    return 0;
}
```
核心实现思想：读入二进制字符串并获取长度。若长度为偶数，直接输出长度除以\(2\)。若长度为奇数，检查除最高位外是否有\(1\)，若有则输出长度除以\(2\)再加\(1\)，否则输出长度除以\(2\)。

### 最优关键思路或技巧
利用二进制数的位数和各位数字情况来确定错过火车的数量。通过分析\(4^k\)在二进制下的形式（\(1\)后面跟\(2k\)个\(0\)），将问题转化为对输入二进制数的位数及各位数字的判断。对于二进制数长度为偶数时，错过火车数量直接为长度除以\(2\)；长度为奇数时，需判断除最高位外是否有\(1\)，若有则错过火车数量为长度除以\(2\)再加\(1\)，否则为长度除以\(2\)。

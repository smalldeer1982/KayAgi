# Good ol' Numbers Coloring

## 题目描述

对于一个非负整数的数列$0,1,2...$，给定两个整数$a,b(1\le a,b\le10^4)$，我们把每个数从小到大染色



每个数字只能被染成黑白两种颜色，而且对$i$染色的规则如下



- 如果$i=0$，则染为白色

- 如果$i\ge a$且$i-a$是白色的，则染为白色

- 如果$i\ge b$且$i-b$是白色的，则染为白色

- 否则染为黑色



举个例子，如果$a=3,b=5$，那么这个序列的染色情况就是（0代表白色，1代表黑色）

$$0,1,1,0,1,0,0,1,0,0,...$$

注意：



- 有可能无限多个数被染为黑色。举个例子，当$a=10,b=10$的时候，只有$10$的倍数是白色的，其余数都是黑色

- 也有可能有限多个数被染为黑色。再举个例子，当$a=1,b=10$的时候，没有一个数是黑色的。



你的任务是判断对于每个$a,b$，数列的黑色数是不是无限的。



如果有无限个数是黑色的，则输出$Infinite$（不含引号）。否则输出$Finite$（不含引号）。

# 经典数字染色问题

## 题目描述
对于非负整数数列$0,1,2,\cdots$，给定两个整数$a,b$（$1\leq a,b\leq10^4$），我们对每个数从小到大进行染色。
每个数字只能染成黑白两种颜色，对$i$染色的规则如下：
- 如果$i = 0$，则染为白色。
- 如果$i\geq a$且$i - a$是白色的，则染为白色。
- 如果$i\geq b$且$i - b$是白色的，则染为白色。
- 否则染为黑色。

举个例子，如果$a = 3$，$b = 5$，那么这个序列的染色情况就是（0代表白色，1代表黑色）：
$0,1,1,0,1,0,0,1,0,0,\cdots$

注意：
- 有可能有无限多个数被染为黑色。例如，当$a = 10$，$b = 10$的时候，只有10的倍数是白色的，其余数都是黑色。
- 也有可能有限多个数被染为黑色。例如，当$a = 1$，$b = 10$的时候，没有一个数是黑色的。

你的任务是判断对于每个$a,b$，数列中黑色数的个数是否为无限个。
如果有无限个数是黑色的，则输出$Infinite$（不含引号）。否则输出$Finite$（不含引号）。

### 题解综合分析与结论
这些题解的核心思路都是通过判断$a$与$b$的最大公约数是否为1，来确定黑色数字的个数是有限还是无限。所有题解均指出，当$\gcd(a,b)=1$时，黑色数字个数有限；反之则无限。其依据在于，白色数字可表示为$ax + by$（$x\geq0$，$y\geq0$）的形式，若两个白色数字相邻，即$ax + by = ax' + by' + 1$，变形可得$a(x - x') + b(y - y') = 1$，此方程当且仅当$\gcd(a,b)=1$时有解且有无限个解，意味着存在相邻白色数字，进而黑色数字数量有限。各题解主要区别在于证明过程的详细程度以及代码实现方式，如求最大公约数的方法（库函数、辗转相除法等）、输入输出方式、代码风格等。

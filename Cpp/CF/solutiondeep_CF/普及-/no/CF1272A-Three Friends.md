# Three Friends

## 题目描述

有三个人站在一个平面直角坐标系的$x$轴上，他们的纵坐标分别为$a,b,c$  
定义这三个的距离和为$|a-b|+|b-c|+|a-c|$

接下来的一分钟，他们每一个人可以选择：向左走一个单位长度 或 向右走一个单位长度，求一分钟后的三个人的最小距离和

## 说明/提示

$1\leq q \le 1000$  
$1\leq a,b,c \le 10^9$  
感谢 @_Wolverine  提供的翻译

# 三个朋友

## 题目描述

### 综合分析与结论
这几道题解主要围绕如何求三个人移动后距离和的最小值。题解思路分为两类，一类是暴力枚举，另一类是通过数学分析得出\(O(1)\)解法。暴力枚举是对每个人的三种移动情况（向左、向右、不动）进行全排列，计算所有情况的距离和并取最小值，时间复杂度为\(O(q\times3^3)\)。而通过数学分析的方法，先对三个数排序，根据最大数与最小数的差值判断能否走到同一位置，进而得出最小距离和的计算公式，时间复杂度为\(O(1)\)。对比可知，数学分析的方法效率更高，代码也更简洁。

### 所选的题解
- **作者：Rainbow_qwq（5星）**
    - **关键亮点**：通过数学分析得出简洁的\(O(1)\)解法，思路清晰，代码简洁高效。
    - **核心代码**：
```cpp
inline int read()
{
    char c=getchar();int x=0;bool f=0;
    for(;!isdigit(c);c=getchar())f^=!(c^45);
    for(;isdigit(c);c=getchar())x=(x<<1)+(x<<3)+(c^48);
    if(f)x=-x;return x;
}
void work()
{
	int a=read(),b=read(),c=read();
	if(a>b)swap(a,b);if(a>c)swap(a,c);if(b>c)swap(b,c);//排序
	if(c-a<=2)puts("0");
	else printf("%d\n",((c-1)-(a+1))*2);
}
```
    - **核心实现思想**：先对输入的三个数排序，若最大数与最小数差值小于等于\(2\)，则最小距离和为\(0\)；否则，通过计算\(((c - 1)-(a + 1))*2\)得到最小距离和。

### 最优关键思路或技巧
通过对问题进行数学分析，根据排序后最大数与最小数的差值，得出最小距离和的计算公式，避免了暴力枚举带来的高时间复杂度，优化了算法。

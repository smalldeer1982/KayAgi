# Nastya and Rice

## 题目描述

有$n$个范围为$[a-b,a+b]$的数，判断他们的和是否可能在$[c-d,c+d]$范围中。

输入有多组数据（$1 \le T \le 1000$），保证：

- $1 \le n \le 1000$
- $0 \le b < a \le 1000 $
- $0 \le d < c \le 1000$

## 样例 #1

### 输入

5
7 20 3 101 18
11 11 10 234 2
8 9 7 250 122
19 41 21 321 10
3 10 8 6 1```

### 输出

```
Yes
No
Yes
No
Yes```

### 综合分析与结论
所有题解思路基本一致，均是先求出\(n\)个数和的最小值\(n\times(a - b)\)与最大值\(n\times(a + b)\)，然后判断这两个最值与区间\([c - d, c + d]\)的关系。若和的最大值小于\(c - d\)或者和的最小值大于\(c + d\)，则输出`No`；否则输出`Yes`。这些题解的核心难点在于理解“可能在某区间”的含义，即两个区间有交集即可，而非一个区间完全包含在另一个区间内。

### 通用建议与扩展思路
此类题目关键在于分析数据范围边界，通过确定最值来判断条件是否满足。类似题目可能会改变区间形式或条件判断方式，但核心思路不变。在解题时，需仔细理解题目中对区间关系的描述。对于此类区间判断问题，还可以拓展到更高维度的区间，如二维平面上矩形区域的相交判断等，其思路都是通过确定边界值来判断是否存在交集。

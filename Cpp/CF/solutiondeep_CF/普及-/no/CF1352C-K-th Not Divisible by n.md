# K-th Not Divisible by n

## 题目描述

给定两个正整数 $n$ 和 $k$。求出第 $k$ 个不能被 $n$ 整除的正整数。

例如，如果 $n=3$，$k=7$，那么所有不能被 $3$ 整除的数为：$1,2,4,5,7,8,10,11,13 \ldots$ 在这其中的第 $7$ 个数字为 $10$。

## 样例 #1

### 输入

```
6
3 7
4 12
2 1000000000
7 97
1000000000 1000000000
2 1```

### 输出

```
10
15
1999999999
113
1000000001
1```

### 题解综合分析与结论
这些题解的核心思路都是基于数学规律来求解。关键在于发现每 $n$ 个数中有 $n - 1$ 个不能被 $n$ 整除这一规律，并据此通过不同方式计算第 $k$ 个不能被整除的数。主要难点在于处理 $k$ 与 $n - 1$ 的倍数关系，不同题解在处理方式和表达方式上有所不同。

### 所选的题解
- **作者：Allen_lml (5星)**
    - **关键亮点**：思路清晰，通过对 $k$ 是否为 $n - 1$ 的倍数进行分类讨论，直接得出简洁的计算公式，代码可读性高。
    - **重点代码**：
```cpp
if(k%(n - 1)!= 0) {
    ans = k / (n - 1)+k;
    printf("%d\n", ans);
    continue;
}
else {
    ans = k / (n - 1)*n - 1;
    printf("%d\n", ans);
    continue;
}
```
    - **核心实现思想**：当 $k$ 不是 $n - 1$ 的倍数时，$k / (n - 1)$ 表示轮数，第 $k$ 个数为 $k$ 加上轮数；当 $k$ 是 $n - 1$ 的倍数时，第 $k$ 个数为 $k / (n - 1)$ 乘以 $n$ 再减 1。
- **作者：ADay (4星)**
    - **关键亮点**：通过构建数表，利用行列关系计算结果，方法新颖且逻辑清晰，同时实现了快读和向上取整函数。
    - **重点代码**：
```cpp
int x = myceil(k, n - 1), y = (k - 1) % (n - 1)+1;
printf("%d\n", n * (x - 1)+y);
```
    - **核心实现思想**：先计算出第 $k$ 个数所在的行 $x$ 和列 $y$，然后根据行列关系得出结果为 $n(x - 1) + y$。
- **作者：b6e0_ (4星)**
    - **关键亮点**：从答案 $x$ 包含的能被 $n$ 整除和不能被 $n$ 整除的数的数量关系出发，设计出公式并处理边界情况，代码简洁明了。
    - **重点代码**：
```cpp
if(k % (n - 1))
    cout << k + k / (n - 1) << endl;
else
    cout << k + k / (n - 1)-1 << endl;
```
    - **核心实现思想**：通过 $k + k / (n - 1)$ 计算答案，当 $k$ 能被 $n - 1$ 整除时，结果减 1。

### 最优关键思路或技巧
利用每 $n$ 个数中有 $n - 1$ 个不能被 $n$ 整除这一规律，根据 $k$ 与 $n - 1$ 的倍数关系进行分类讨论得出计算公式，避免暴力枚举，提高效率。

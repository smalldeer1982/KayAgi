# Honest Coach

## 题目描述

你面前有 n 名运动员 运动员的编号从左到右分别是1--n你知道每个运动员的力量 编号为i的运动员的力量为si

   	你需要把所有运动员分到两队里，每队至少有一人，每名运动员必须在其中一队里。

    你需要在第一队里最强壮的运动员和第二队里最虚弱的运动员的差距尽可能的小。

    正式的，你想把运动员分成A，B两队导致价值∣max(A)−min(B)∣尽可能的小。

    其中max（A）是A队中力量最大的，min（B）是队伍中力量最小的。

    例如，如果n=5，力量分别为s={3,1,2,4,6}；则最理想的分配是：

    第一队 A={1,2,4}；第二队B={3，6}；

    这样一来最大的价值的绝对值是|4-3|=1，这个例子说明了一种最佳的分成两个团队的方法。输出最小的价值|max(A)-min(B)|。

# 诚实的教练

## 题目描述
你面前有 n 名运动员，运动员的编号从左到右分别是1 - n，你知道每个运动员的力量，编号为i的运动员的力量为si。

你需要把所有运动员分到两队里，每队至少有一人，每名运动员必须在其中一队里。

你需要让第一队里最强壮的运动员和第二队里最虚弱的运动员的差距尽可能的小。

正式地，你想把运动员分成A，B两队，使得价值∣max(A)−min(B)∣尽可能的小。

其中max（A）是A队中力量最大的，min（B）是队伍中力量最小的。

例如，如果n = 5，力量分别为s = {3, 1, 2, 4, 6}；则最理想的分配是：

第一队 A = {1, 2, 4}；第二队B = {3，6}；

这样一来最大的价值的绝对值是|4 - 3| = 1，这个例子说明了一种最佳的分成两个团队的方法。输出最小的价值|max(A)-min(B)|。

### 综合分析与结论
三道题解思路基本一致，均是先对运动员力量数组进行排序，然后通过遍历相邻元素差值来寻找最小值，此最小值即为所求的最小差距。其核心思想在于：排序后相邻元素差值最小，将小于等于较小值的运动员放一队，大于等于较大值的放另一队，能保证一队最大值与另一队最小值的差最小。

### 通用建议与扩展思路
 - **通用建议**：在解决此类问题时，要善于观察数据特征，通过排序简化问题求解过程。同时，在代码实现上，注意边界条件处理以及变量初始化，确保程序的正确性和健壮性。
 - **扩展思路**：类似的贪心问题通常需要找到一种合理的策略，使得局部最优能导致全局最优。对于本题，可拓展到将运动员分为多组的情况，或者改变分组条件，如每组人数有特定要求等，解题时需重新分析寻找新的贪心策略。

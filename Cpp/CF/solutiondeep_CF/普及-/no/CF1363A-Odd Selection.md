# Odd Selection

## 题目描述

给出一个 $n$ 项的数列，要求选出 $x$ 个数（不要求连续），使这 $x$ 个数的和为奇数。判断这是否可能。

## 说明/提示

对于第一组数据，我们必须选出 $999$ ，其为奇数。

对于第二组数据，我们必须选出 $1000$ ，其不是奇数。

对于第三组数据，我们可以选择 $51$ ，使得总和为奇数。

对于第四组数据，我们必须选择 $51$ 与 $50$ ，和为 $51+50=101$ 为奇数。

对于第五组数据，我们必须选择全部的三个数，但是总和并不是奇数。

## 样例 #1

### 输入

```
5
1 1
999
1 1
1000
2 1
51 50
2 2
51 50
3 3
101 102 103```

### 输出

```
Yes
No
Yes
Yes
No```

### 题解综合分析与结论
这些题解的核心思路都是利用奇数和偶数相加的奇偶性规律来解决问题。关键在于统计数列中奇数和偶数的个数，然后根据不同情况判断能否选出和为奇数的x个数。各题解的主要差异在于对不同情况的分类讨论方式和代码实现细节。

### 所选的题解
- **作者：__shadow__ (5星)**
    - **关键亮点**：思路清晰，对各种情况进行了详细的分类讨论，代码逻辑严谨，易于理解。
    - **重点代码核心实现思想**：先统计奇数个数`sum`，若没有奇数则直接输出`No`。若奇数个数为偶数，则减1并相应调整`n`。然后通过比较`x`与`sum`以及偶数数量的关系，判断是否满足条件。
```cpp
int n, x, t, sum;//sum是奇数个数 
int main()
{
    int Q;//Q是数据组数 
    scanf ("%d", &Q);
    while (Q--)
    {
        sum = 0;//奇数个数归零 
        scanf ("%d%d", &n, &x);//n是有几个数  x是要取几个数 
        for (int i = 1;i <= n; i++)
        {
            scanf ("%d", &t);//输入这个数 
            if (t % 2!= 0)//如果是奇数 
                sum++;//sum 加一 
        }
        if (!sum)//如果没有奇数，肯定不行 
        {
            printf ("No\n");
            continue;
        } 
        if (sum % 2 == 0)//如果有偶数个奇数 
        {
            sum--;//奇数个数减一 
            n--;//因为要用（n-sum）算偶数个数，所以n也要减一 
        }
        if (x >= sum && (x - sum) <= (n - sum))
        //如果x比sum大，并且偶数数量足够补空 
        {
            printf ("Yes\n");//输出可行 
            continue;
        }
        if (x < sum)//如果x比sum小 
        {
            if((n - sum) > 0 || x % 2!= 0)
            //当有一个偶数或要奇数个数时 
            {
                printf ("Yes\n");//输出可行 
                continue;
            }
        }
        printf ("No\n");//剩下的情况输出不行 
    }
    return 0;
}
```

    - **关键亮点**：通过维护奇数和偶数的数量，遍历奇数个数来判断是否满足条件，算法简洁明了。
    - **重点代码核心实现思想**：遍历0到奇数数量`odd`，当扫过奇数个奇数`i`时，判断`x - i`是否不超过偶数数量`even`，若满足则说明可以凑出和为奇数的x个数。
```cpp
int main() {
    int t;
    cin >> t;
    
    while (t--) {
        int n, x;
        cin >> n >> x;
        
        int odd = 0, even = 0;
        rep(i, n) {
            int a;
            cin >> a;
            (a & 1? odd : even)++;
        }
        bool ok = false;
        rep(i, odd + 1) {
            if (i & 1 && i <= x && x - i <= even) ok = true;
        }
        cout << (ok? "Yes" : "No") << '\n';
    }
    
    return 0;
}
```
- **作者：KSToki (4星)**
    - **关键亮点**：先取出一个奇数，根据剩余需求量`x`的奇偶性进一步判断，逻辑较为清晰。
    - **重点代码核心实现思想**：统计奇数和偶数数量，若没有奇数则输出`No`。取出一个奇数后，若`x`为奇数且没有偶数则输出`No`，否则再取出一个偶数。最后判断剩余奇数和偶数数量之和是否大于等于当前`x`。
```cpp
int t,n,x,k,a,b;
int main()
{
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d%d",&n,&x);
        a=0;
        b=0;
        for(register int i=1;i<=n;++i)
        {
            scanf("%d",&k);
            if(k&1)
                ++a;
            else
                ++b;
        }
        if(a==0)
        {
            puts("No");
            continue;
        }
        --a;
        --x;
        if(x&1)
        {
            if(b==0)
            {
                puts("No");
                continue;
            }
            --b;
            --x;
        }
        a=a/2*2;
        if(a+b>=x)
            puts("Yes");
        else
            puts("No");
    }
    return 0;
}
```

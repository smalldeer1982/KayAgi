# 01 Game

## 题目描述

**题目大意**

`Alice`和`Bob`在玩游戏。

初始有一个仅由`01`构成的字符串。`Alice`和`Bob`轮流进行游戏，`Alice`先行。轮到某个人的时候，他需要从原串中找到并删除两个相邻且不同的字符（`01`或`10`），无法操作者输。

两人都用最优的策略进行，你需要确定谁能够赢得游戏。

### 综合分析与结论
这些题解的核心思路基本一致，都基于以下原理：每次操作会删除一个`0`和一个`1`，所以游戏的操作次数取决于字符串中`0`和`1`个数的最小值。判断这个最小值的奇偶性，若为奇数则`Alice`赢（输出`DA`），若为偶数则`Bob`赢（输出`NET`）。
不同点主要在于实现方式，部分题解直接遍历字符串统计`0`和`1`的个数，部分使用了`STL`中的`count`函数来统计。还有赛时用栈模拟删除过程的做法，但赛后也发现该做法相对复杂。整体来看，各题解思路清晰，主要差异在于代码简洁性和对`STL`的运用上。

### 通用建议与扩展思路
对于此类题目，关键在于理解游戏规则下操作对数据的影响规律，从而找到决定结果的核心因素（如本题中`0`和`1`个数的最小值）。可拓展思路为改变游戏规则，如改变可删除的字符组合，或者改变获胜条件等，类似的基于规则找规律的题目都可以用类似思路分析。

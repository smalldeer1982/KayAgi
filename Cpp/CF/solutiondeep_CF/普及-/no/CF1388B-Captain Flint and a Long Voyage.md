# Captain Flint and a Long Voyage

## 题目描述

若有一个 $n$ 位正整数 $x$，定义 $k$ 的值为将 $x$ 的每一位上的数字转换为二进制（不含前导 $0$）。比如 $x=729$ 时，$k=111101001$。将 $k$ 的后 $n$ 位截去得到 $r$。比如 $x=729$ 时，$r=111101$。

现在给定 $n$，你需要找到 $r$ 最大时 $x$ 的最小值。

## 说明/提示

$1\le t\le1000$，$1\le n\le 10^5$。

数据保证所有数据中的 $n$ 之和不超过 $2\times10^5$。

### 综合分析与结论
这些题解的核心思路均围绕贪心策略展开。由于要使 $r$ 最大，而 $1 - 9$ 中 $8$ 和 $9$ 的二进制表示位数最多（均为 $4$ 位），所以 $x$ 应由 $8$ 和 $9$ 组成。为使 $x$ 最小，在会被截去的二进制位对应的位置用 $8$，其他位置用 $9$。各题解主要区别在于计算 $8$ 和 $9$ 个数的方式略有不同，但本质思路一致。

#include<bits/stdc++.h>
using namespace std;
int n,t;

    - **核心代码**：
```cpp
#include<iostream>
#include<cstdio>
#include<cstring>
#define ll long long
#define rgt register int
using namespace std;
const int mxn = 2e5+5;
int n,g;
char ans[mxn];
int main(){
    int test;
    scanf("%d",&test);
    while(test--){
        scanf("%d",&n);
        g=n/4;
        memset(ans,'8',sizeof(ans));
        for(rgt i=1;i<=n-g;i++)
            ans[i]='9';
        if(n%4)
            ans[n-g]='8';
        for(rgt i=1;i<=n;i++)
            printf("%c",ans[i]);
        printf("\n");
    }
    return 0;
}
```
    - **核心实现思想**：先计算要被删掉的数位 `g` 即 $8$ 的个数，先将数组 `ans` 初始化为全 $8$，再将不用被删的数位填 $9$，最后根据余数处理细节输出结果。
 - **作者：白鲟 (4星)**
    - **关键亮点**：分析过程自然流畅，对为什么选择 $8$ 和 $9$ 以及二者使用场景解释到位，代码简洁。
    - **核心代码**：
```cpp
#include<cstdio>
using namespace std;
int T,n;
int main()
{
    scanf("%d",&T);
    while(T--)
    {
        scanf("%d",&n);
        int num_8=n/4+(n%4>0);
        for(int i=1;i<=n-num_8;++i)
            putchar('9');
        for(int i=1;i<=num_8;++i)
            putchar('8');
        putchar('\n');
    }
    return 0;
}
```
    - **核心实现思想**：通过 `n / 4 + (n % 4 > 0)` 计算出 $8$ 的个数 `num_8`，然后分别输出相应个数的 $9$ 和 $8$。

### 最优关键思路或技巧
利用贪心思想，优先选择二进制表示位数多的数字（$8$ 和 $9$）来构造原数 $x$，同时根据截去二进制位的规则，在满足 $r$ 最大的前提下，尽量使用较小的数字 $8$ 来使 $x$ 最小。

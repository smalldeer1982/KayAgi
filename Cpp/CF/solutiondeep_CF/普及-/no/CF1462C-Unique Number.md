# Unique Number

## 题目描述

You are given a positive number $ x $ . Find the smallest positive integer number that has the sum of digits equal to $ x $ and all digits are distinct (unique).

## 样例 #1

### 输入

```
4
1
5
15
50```

### 输出

```
1
5
69
-1```

# 唯一数字

## 题目描述
给定一个正数 $x$ 。找到最小的正整数，其各位数字之和等于 $x$ 且所有数字都不同（唯一）。

## 样例 #1
### 输入
```
4
1
5
15
50
```
### 输出
```
1
5
69
-1
```

### 算法分类
贪心

### 题解综合分析与结论
这些题解主要围绕贪心策略展开，部分题解还提到了打表和暴力搜索的方法。所有题解都意识到当 $x > 45$ 时无解，因为 $1$ 到 $9$ 的数字之和最大为 $45$ 且要保证数字各不相同。贪心思路核心是为使所求数最小，应让位数尽可能少，所以从大数字 $9$ 开始往低位填充，直到满足数字和为 $x$ 。

各题解的不同点主要在于实现细节和优化方式。如有的使用数字桶标记已使用数字，有的直接通过计算和模拟填充过程；部分题解还考虑了输入输出优化、结果还原等。打表法通过预先计算好结果存储在数组中，直接查询输出，适合多次查询且数据范围不大的情况。暴力搜索则通过深度优先搜索并结合剪枝策略来求解，但效率相对贪心策略较低。

### 所选的题解
- **作者：asasas (5星)**
    - **关键亮点**：思路清晰，代码简洁明了，详细注释。利用数字桶标记已使用数字，最后通过循环还原结果顺序。
    - **重点代码**：
```cpp
bool uss[15];//数字桶
int main(){
    int t;
    cin >> t;
    while(t--){
    	int n;
    	cin >> n;
    	memset(uss,0,sizeof(uss));//清空桶
    	if (n>45){
    		cout << -1 << endl;//n>45时直接输出无解
    		continue ;
    	}
    	int ans=0;
    		for (register int i=9;i>=1;i--){
    			if (uss[i]==0&&n>=i){
    				uss[i]=1;
    				ans=ans*10+i;
    				n-=i;//能填就填
    			}
    			if (n==0) break;//填完了就退出
    	}
    	int num=0;
    	while(ans){
    		num=num*10+ans%10;//由于之前求的是答案数的倒序，所以最后要将它还原一下，还原成正序
		ans/=10; 
    	}
    	cout << num << endl;//输出
    }
    return 0;
}
```
    - **核心实现思想**：从9到1枚举数字，若该数字未使用且当前和n不小于它，则将其加入答案并更新n，最后将得到的倒序答案还原为正序输出。

```cpp
int core(int input){
    int ans=0,tmp=10;
    if(input>45){
        return -1;
    }
    for(int i=9;input-i>0;i-=1){
        input-=i;
        tmp=i;
    }
    if(input>0){
        ans+=input;
    }
    for(int i=tmp;i<=9;i++){
        ans*=10;
        ans+=i;
    }
    return ans;
} 
```
    - **核心实现思想**：先从9开始减去能减的数字，记录最后一个减去的数字tmp，若还有剩余则加到ans，再将tmp到9的数字依次加入ans。

- **作者：EuphoricStar (4星)**

    - **重点代码**：
```cpp
int main()
{
    ios::sync_with_stdio(0);
    cin >> t;
    while (t--)
    {
        cin >> x;
        if (x > 45) cout << -1 << endl;
        else
        {
            string res = "";
            for (int i = 9;; --i)
            {
                if (x < i) continue;
                x -= i;
                res = char(i + '0') + res;
                if (!x) break;
            }
            cout << res << endl;
        }
    }
    return 0;
}
```
    - **核心实现思想**：从9到1枚举数字，若x不小于当前数字，则将其加入字符串res开头，更新x，x为0时结束循环并输出res。

### 最优关键思路或技巧
贪心策略下从大数字往小数字填充，能有效减少数字位数从而得到最小数。在实现上，使用合适的数据结构（如数字桶标记数字使用情况）或简单数据类型（如字符串）来简化代码逻辑。对于多次查询且数据范围有限的情况，打表法可极大提高效率。

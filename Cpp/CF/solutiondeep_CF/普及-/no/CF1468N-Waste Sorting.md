# Waste Sorting

## 题目描述

有三个桶，五种垃圾，每个桶有固定的容量。

$1$垃圾放入$1$号桶中，$2$垃圾放入$2$号桶中，$3$垃圾放入$3$号桶中，$4$垃圾可以放入$1$、$3$号桶中，$5$垃圾可以放入$2$、$3$号桶中。

对于给定的桶容量$c$和垃圾量$a$，请问垃圾是否可以全部放入桶中？

### 综合分析与结论
这些题解思路基本一致，均采用贪心策略。先处理只能放入特定桶的1、2、3号垃圾，若对应桶容量不足则直接判定无法全部放入。接着处理4、5号垃圾，优先将它们放入1、2号桶，剩余部分再尝试放入3号桶，根据3号桶剩余容量判断能否装下所有垃圾。各题解在细节处理和代码实现方式上略有不同，如处理负数情况、代码风格等。

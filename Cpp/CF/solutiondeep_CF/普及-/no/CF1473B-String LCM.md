# String LCM

## 题目描述

如果字符串 $s$ 可以变成 $n$ 个字符串 $s_1$ 首尾相连，则说 $s$ 能被 $s_1$ 整除或 $s_1$ 能整除 $s$。

定义两个字符串 $s_1,s_2$ 的最短公倍串为：**可以被 $s_1$ 和 $s_2$ 整除的最短的非空串**。

例如： $baba$ 和 $ba$ 的最短公倍串为 $baba$；$aa$ 和 $aaa$ 的最短公倍串为 $aaaaaa$；$aba$ 和 $ab$ 没有最短公倍串。

### 输入

```
3
baba
ba
aa
aaa
aba
ab```

### 输出

```
baba
aaaaaa
-1```

    - **核心代码**：
```cpp
#include<bits/stdc++.h>
using namespace std;
string s,t;
int main()
{
    int q;
    cin>>q;
    while(q--)
    {
    	cin>>s>>t;
    	int ls=s.size(),lt=t.size();
    	int x=__gcd(ls,lt); 
    	int lp=x*(ls/x)*(lt/x);
    	int t1=lp/ls,t2=lp/lt;
    	string ss="",tt=""; 
    	for(int i=1;i<=t1;i++)
    	{
    		ss+=s; 
		}
		for(int i=1;i<=t2;i++)
    	{
    		tt+=t; 
		}
		if(ss==tt)
		{
			cout<<ss<<endl;
		}
		else
		{
			cout<<-1<<endl;
		}
	}
    return 0;
}
```
    - **核心实现思想**：先计算两个字符串长度的最大公约数 `x`，进而得到最小公倍数 `lp`。根据最小公倍数计算出两个字符串分别需要重复的次数 `t1` 和 `t2`，将字符串 `s` 和 `t` 分别重复相应次数得到 `ss` 和 `tt`，比较 `ss` 和 `tt` 是否相等，相等则输出 `ss`，否则输出 `-1`。

    - **核心代码（Python）**：
```python
from math import gcd
def solve():
    	a=input()
    	b=input()
    	glen=gcd(len(a),len(b))
    	glen=len(a)*len(b)//glen
    	aa=a*(glen//len(a))
    	bb=b*(glen//len(b))
    	if aa==bb:print(aa)
    	else:print(-1)
t=int(input())
for i in range(0,t):solve()
```
    - **核心实现思想**：借助 `math` 库中的 `gcd` 函数求出两个字符串长度的最大公约数 `glen`，进而得到最小公倍数 `glen`。根据最小公倍数计算出字符串 `a` 和 `b` 分别需要重复的次数，将 `a` 和 `b` 分别重复相应次数得到 `aa` 和 `bb`，比较 `aa` 和 `bb` 是否相等，相等则输出 `aa`，否则输出 `-1`。

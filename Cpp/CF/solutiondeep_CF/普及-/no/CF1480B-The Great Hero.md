# The Great Hero

## 题目描述

### 题意

  我们定义一个人物为一个二元组$(x, y)$, 称其中 $x$ 为攻击力, $y$ 为血量. 一个英雄是一个人物. 现在有 $n$ 个怪物, 每个怪物是一个人物. 我们这样定义两个人物 $A$ 与 $B$ 交战:

- $A$ 的血量减少等同于 $B$ 的攻击力的数值, $B$ 的血量也减少等同于 $A$ 的攻击力的数值.
- 然后, $A$ 和 $B$ 中所有血量小于等于 $0$ 的人物死亡.

  现在英雄需要消灭所有怪物, 消灭怪物的方式是与之交战. 请求出英雄能不能消灭所有的怪物, 即使英雄本人在消灭所有怪物后死亡.

# 伟大的英雄

## 题目描述

### 题意
我们定义一个人物为一个二元组$(x, y)$，称其中$x$为攻击力，$y$为血量。一个英雄是一个人物。现在有$n$个怪物，每个怪物也是一个人物。我们这样定义两个人物$A$与$B$交战：
- $A$的血量减少等同于$B$的攻击力的数值，$B$的血量也减少等同于$A$的攻击力的数值。

### 综合分析与结论
这几道题解均围绕英雄能否消灭所有怪物展开，核心在于考虑英雄与怪物交战时的血量损耗及攻击次数。难点在于如何处理英雄与最后一只怪物同归于尽的情况，使方案最优。
 - **云浅知处题解**：先计算消灭所有怪物英雄需消耗的总血量，再通过判断加上某只怪物攻击力后英雄剩余血量是否大于0，来确定能否杀光所有怪物，时间复杂度$O(n)$。
 - **fls233666题解**：最初思路是模拟杀怪，但因同归于尽情况导致按顺序模拟非最优，改进为按攻击力对怪物排序后模拟杀怪。
 - **fanfansann题解**：先模拟计算打完所有怪英雄剩余血量，再加上攻击力最大怪物的攻击力，判断此时血量是否大于0。

整体来看，云浅知处题解思路清晰，直接从整体消耗和剩余血量判断，代码简洁明了；fls233666题解通过分析模拟思路的问题，提出贪心改进策略；fanfansann题解抓住将攻击力最大怪物放最后的关键，但相比之下，云浅知处题解更具通用性和简洁性。

```cpp
int myceil(int x,int y){
    if(x%y==0)return x/y;
    else return (int)(x/y)+1;
}

void solve(){
    
    cin>>A>>B>>n;

    for(int i=1;i<=n;i++){
        cin>>a[i];
    }
    for(int i=1;i<=n;i++){
        cin>>b[i];
    }

    int sum=0;
    for(int i=1;i<=n;i++){
        sum+=myceil(b[i],A)*a[i];
    }

    bool f=0;
    for(int i=1;i<=n;i++){
        if(B-sum+a[i]>0){
            cout<<"YES\n";
            f=1;
            break;
        }
    }
    if(!f)cout<<"NO\n";
}
```
核心实现思想：先通过自定义函数`myceil`计算杀死每只怪物所需攻击次数，累加计算消灭所有怪物英雄消耗的总血量`sum`，再遍历每只怪物，判断加上该怪物攻击力后英雄剩余血量是否大于0，若有大于0的情况则输出`YES`，否则输出`NO`。

### 最优关键思路或技巧
通过计算消灭所有怪物英雄的总血量消耗，再结合与单只怪物攻击力的关系判断能否杀光怪物，这种从整体到局部的分析方式，避免了复杂的模拟过程，有效降低时间复杂度，是解决此类问题的关键技巧。

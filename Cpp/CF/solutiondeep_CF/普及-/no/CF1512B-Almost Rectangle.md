# Almost Rectangle

## 题目描述

There is a square field of size $ n \times n $ in which two cells are marked. These cells can be in the same row or column.

You are to mark two more cells so that they are the corners of a rectangle with sides parallel to the coordinate axes.

For example, if $ n=4 $ and a rectangular field looks like this (there are asterisks in the marked cells):

  $$\begin{matrix}
.&.&*&.\\
.&.&.&.\\
*&.&.&.\\
.&.&.&.\\
\end{matrix}$$

Then you can mark two more cells as follows

 $$\begin{matrix}
*&.&*&.\\
.&.&.&.\\
*&.&*&.\\
.&.&.&.\\
\end{matrix} $$

If there are several possible solutions, then print any of them.

# 近乎矩形

## 题目描述
有一个大小为 $n \times n$ 的正方形区域，其中有两个单元格被标记。这些单元格可以在同一行或同一列。

你需要再标记另外两个单元格，使它们成为一个边与坐标轴平行的矩形的角。

例如，如果 $n = 4$，一个矩形区域看起来像这样（标记的单元格中有星号）：

  $$\begin{matrix}
.&.&*&.\\
.&.&.&.\\
 *&.&.&.\\
.&.&.&.\\
 \end{matrix}$$

那么你可以按如下方式标记另外两个单元格

  $$\begin{matrix}
 *&.&*&.\\
.&.&.&.\\
 *&.&*&.\\
.&.&.&.\\
 \end{matrix} $$

如果有多种可能的解决方案，则打印其中任何一种。

### 综合分析与结论
这些题解的核心思路都是对给定的两个标记点的位置关系进行分类讨论，从而确定另外两个要标记的点。主要分为以下三种情况：
1. 两个点行列均不同，此时另外两个点坐标为 $(x_1,y_2)$ 和 $(x_2,y_1)$。
2. 两个点在同一行，需在该行上方或下方添加两个点，要注意边界情况。
3. 两个点在同一列，需在该列左侧或右侧添加两个点，同样要注意边界情况。

各题解在实现细节上略有不同，如输入输出方式、边界处理方式等，但整体思路一致。

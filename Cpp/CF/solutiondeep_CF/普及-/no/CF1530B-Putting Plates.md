# Putting Plates

## 题目描述

有一张桌子，分为 $h$ 行 $w$ 列。

桌子**边缘**上的某一行某一列可以放一个盘子（也可以不放）。但是，一个盘子的上面、下面、左面、右面、左上、右上、左下、右下都不能有盘子。

问题来了：这个桌子，放盘子最多的方案是什么？请输出这个方案。当然，答案不止一种，你也不用全部输出，输出任意一个，即可。

• **算法分类**：贪心
• **综合分析与结论**：这些题解主要分为两种思路。一种是找规律贪心放置盘子，如zct_sky、Orin等作者，利用奇数行/列放置盘子能使数量最多的特性，先在上下两行奇数位全填，再在左右两列特定奇数行填充；另一种是暴力模拟，如SUPERXWX、__int127等作者，通过遍历边缘位置，判断每个位置能否放置盘子，能放则放并标记周围位置。找规律贪心的方法思路更巧妙，代码简洁高效，时间复杂度低；暴力模拟虽易理解但代码复杂，时间复杂度相对较高，在数据规模增大时效率较低。
• **所选的题解**
  - **作者zct_sky，星级：5星，关键亮点**：思路清晰，简洁地阐述了先在第1行和第h行奇数列放盘子，再在第1列和第w列的第2行到第h - 2行的偶数行放盘子的贪心策略，代码实现简洁明了。
```cpp
// 核心代码片段
for(int i=0;i<w;i++)//第1行和第h行奇数列全放盘子
    if(!(i%2))f[0][i]=f[h-1][i]=1;

for(int i=2;i<h-2;i++)//第1列和第w列的第2行到第h-2行的偶数行全放盘子
    if(!(i%2))f[i][0]=f[i][w-1]=1;
```

# 2-Letter Strings

## 题目描述

换句话来说，找到所有的数字对$(i,j)(i<j)$使得第$i$个字符串与第$j$个字符串有唯一一个数字$p(1 \leq p \leq 2)$使得$s_{i_p}\neq s_{j_p}$

答案可能超出32位数的范围，所以你在使用c++时应该使用long long等64位变量类型来存储数据避免溢出

注：本题有多数据

## 题目描述

### 综合分析与结论
题解通过枚举每个字符串与其他字符串在某一位相等，而另一位不相等的情况来统计符合条件的字符串对数。算法要点在于利用数组 `c` 记录每个字符组合出现的次数，然后对每个字符串，分别枚举第一位不同且第二位相同以及第一位相同且第二位不同的情况，累加结果。解决难点在于要注意不关心顺序，最后需将结果除以2。

### 所选的题解
- **星级**：4星
  - **关键亮点**：思路清晰，通过数组统计字符组合出现次数，简化了枚举过程。利用双重循环分别处理不同位不同的情况，逻辑简洁明了。
  - **重点代码核心实现思想**：首先读取数据并记录每个字符组合的出现次数，然后对每个字符串，通过枚举不同位的字符，累加符合条件的字符串对数，最后除以2得到结果。
  - **核心代码片段**：
```cpp
for (int i = 1; i <= n; ++i) {
  char x, y;
  cin >> x >> y;
  a[i][0] = x - 'a', a[i][1] = y - 'a';
  ++c[a[i][0]][a[i][1]];
}
for (int i = 1; i <= n; ++i) {
  for (int j = 0; j <= 10; ++j) {
    if (j!= a[i][0]) {
      s += c[j][a[i][1]];
    }
    if (j!= a[i][1]) {
      s += c[a[i][0]][j];
    }
  }
}
cout << s / 2 << endl;
```
### 最优关键思路或技巧
利用二维数组记录字符组合出现次数，通过枚举不同位不同字符的方式来统计符合条件的字符串对数，简化了暴力枚举的过程，同时注意结果需除以2以避免重复计数。

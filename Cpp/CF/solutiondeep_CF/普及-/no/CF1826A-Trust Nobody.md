# 题目信息

# Trust Nobody

## 题目描述

有由 $n$ 个人组成的群体，其中的一部分人被称为“说谎者”，总是说谎话，另一部分人总是说真话。对于 $1\leq i \leq n$ ，第 $i$ 个人说：“在我们中间至少有 $l_i$ 个人说谎话。”写一个程序判断人们所说的是矛盾的，还是可能存在的。如果是可能存在的，输出群体中说谎者的数量，如果有多种可能，输出其中任意一种即可。

## 样例 #1

### 输入

```
7
2
1 2
2
2 2
2
0 0
1
1
1
0
5
5 5 3 3 5
6
5 3 6 6 3 5```

### 输出

```
1
-1
0
-1
0
3
4```

# Easy Number Challenge

## 题目描述

Let's denote $ d(n) $ as the number of divisors of a positive integer $ n $ . You are given three integers $ a $ , $ b $ and $ c $ . Your task is to calculate the following sum:

![](https://cdn.luogu.com.cn/upload/vjudge_pic/CF236B/6b4d9893ce96bd0459ff1289a8bf3491052ac12a.png)Find the sum modulo $ 1073741824 $ $ (2^{30}) $ .

## 说明/提示

For the first example.

- $ d(1·1·1)=d(1)=1 $ ;
- $ d(1·1·2)=d(2)=2 $ ;
- $ d(1·2·1)=d(2)=2 $ ;
- $ d(1·2·2)=d(4)=3 $ ;
- $ d(2·1·1)=d(2)=2 $ ;
- $ d(2·1·2)=d(4)=3 $ ;
- $ d(2·2·1)=d(4)=3 $ ;
- $ d(2·2·2)=d(8)=4 $ .

So the result is $ 1+2+2+3+2+3+3+4=20 $ .

## 样例 #1

### 输入

```
2 2 2
```

### 输出

```
20
```

## 样例 #2

### 输入

```
5 6 7
```

### 输出

```
1520
```

# 简单数字挑战

## 题目描述
设 \(d(n)\) 为正整数 \(n\) 的约数个数。给定三个整数 \(a\)、\(b\) 和 \(c\) ，你的任务是计算以下和式：

\(\sum_{i = 1}^{a}\sum_{j = 1}^{b}\sum_{k = 1}^{c}d(i \times j \times k)\)

求该和式对 \(1073741824\)（\(2^{30}\)）取模的结果。

## 说明/提示
对于第一个样例：
 - \(d(1×1×1)=d(1)=1\)；
 - \(d(1×1×2)=d(2)=2\)；
 - \(d(1×2×1)=d(2)=2\)；
 - \(d(1×2×2)=d(4)=3\)；
 - \(d(2×1×1)=d(2)=2\)；
 - \(d(2×1×2)=d(4)=3\)；
 - \(d(2×2×1)=d(4)=3\)；
 - \(d(2×2×2)=d(8)=4\)。

所以结果为 \(1 + 2 + 2 + 3 + 2 + 3 + 3 + 4 = 20\)。

2 2 2

所有题解均基于暴力枚举的思路来解决问题，通过三重循环遍历 \(a\)、\(b\)、\(c\) 的所有组合，计算每个组合乘积的约数个数并求和，最后对结果取模。解决难点在于优化计算约数个数的过程以及处理重复计算的情况。大部分题解采用记忆化的方式来避免重复计算乘积的约数个数，从而提高效率。但整体时间复杂度仍较高，为 \(O(n^3\sqrt{n})\)。

# 题目信息

# Bear and Raspberry

## 题目描述

给定一个序列，请求出相邻两个数之间（前数$-$后数）差的最大值与$c$的差。

如果结果小于$0$的话就输出$0$。

## 样例 #1

### 输入

```
5 1
5 10 7 3 20
```

### 输出

```
3
```

## 样例 #2

### 输入

```
6 2
100 1 10 40 10 40
```

### 输出

```
97
```

## 样例 #3

### 输入

```
3 0
1 2 3
```

### 输出

```
0
```

### 综合分析与结论
所有题解的核心思路都是遍历序列，计算相邻两数（前数 - 后数）差值，并找出最大差值，最后将最大差值与 c 相减，若结果小于 0 则输出 0。不同之处在于数据存储方式和实现细节：
 - 部分题解使用数组存储整个序列，如作者“PrincessYR✨～”、“18lxxrz”（普通思路）、“CobaltChloride”；
 - 部分题解仅用两个变量滚动记录相邻数字，避免使用数组，减少空间开销，如作者“LRL65”、“18lxxrz”（蒟蒻思路）、“damage”。在代码实现上，有的使用 cin/cout，有的使用 scanf/printf，还有的实现了快读。整体来看，这些题解思路较为直接，实现难度不高。

### 通用建议与扩展思路
对于此类简单模拟题，关键在于清晰理解题意并准确实现相应逻辑。优化方向可考虑减少空间占用（如避免不必要的数组使用）和提高输入输出效率（如使用快读、scanf/printf 替代 cin/cout）。类似题目套路通常围绕对给定数据的特定运算和处理，重点在于准确模拟题目描述的操作过程。

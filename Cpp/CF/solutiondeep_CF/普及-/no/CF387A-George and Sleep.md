# 题目信息

# George and Sleep

## 题目描述

给出一个时间点 $s$ 和一个时间段 $t$，求出从时间点 $s$ 向前回推时间段 $t$ 得到的时间点 $p$。（所有时间用 `HH:MM` 的格式表示）

## 说明/提示

对于 $100\%$ 的数据，$0\le \text{HH}\le 23$，$0\le \text{MM}\le 59$。

时间点 $s$ 和时间点 $p$ 可能不在同一天。

## 样例 #1

### 输入

```
05:50
05:44
```

### 输出

```
00:06
```

## 样例 #2

### 输入

```
00:00
01:00
```

### 输出

```
23:00
```

## 样例 #3

### 输入

```
00:01
00:00
```

### 输出

```
00:01
```

### 综合分析与结论
这些题解主要围绕时间的计算展开，核心思路都是将时间转化为分钟进行运算，再处理可能跨天的情况，最后将结果转化回 `HH:MM` 的格式输出。不同之处在于具体实现方式，如输入处理、运算逻辑的细节以及输出格式化。有的题解逻辑较为直接，通过多次判断处理不同情况；有的则通过巧妙的公式简化判断过程。

### 所选的题解
- **作者：songxiao（5星）**
    - **关键亮点**：思路清晰，代码注释详细，详细讲解了输入输出格式控制，先将时间统一转换为分钟计算，再根据时间先后关系判断是否跨天并进行相应计算。
    - **个人心得**：无
    ```cpp
    #include <bits/stdc++.h>
    int main()
    {
        int h,m,h0,m0;
        int t,t0,s;
        scanf("%d:%d\n%d:%d",&h,&m,&h0,&m0);
        t=h*60+m;
        t0=h0*60+m0;
        if(t>t0)
        {
            s=t-t0;
            printf("%02d:%02d",s/60,s%60);
        }
        else if(t<t0)
        {
            s=24*60-t0+t;
            printf("%02d:%02d",s/60,s%60);
        }
        else
            printf("00:00");
        return 0;
    }
    ```
- **作者：CZQ_King（5星）**

    - **个人心得**：无

### 最优关键思路或技巧

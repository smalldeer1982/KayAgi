【重点代码】
```cpp
#include <bits/stdc++.h>
using namespace std;
long long n,x,y,m=-1,s=1000000005,a[500000];
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        cin>>a[i];
        if(a[i]==m) x++;
        if(a[i]>m) m=a[i],x=1;
        if(a[i]==s) y++;
        if(a[i]<s) s=a[i],y=1;
    }
    if(m==s) cout<<0<<" "<<n*(n-1)/2;
    else cout<<m-s<<" "<<x*y;
    return 0;
}
```
**核心实现思想**：通过一次遍历输入数据，边读边更新最大值 `m`、最小值 `s` 及其个数 `x` 和 `y`，最后根据 `m` 和 `s` 是否相等进行分类讨论，相等时方案数为 `n*(n - 1)/2`，不相等时方案数为 `x*y`。

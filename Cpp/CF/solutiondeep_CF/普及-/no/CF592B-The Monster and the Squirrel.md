# 题目信息

# The Monster and the Squirrel

## 题目描述

### 题目概要

有一个正n多边形，每个顶点从1到n标号，从顶点1开始，向每个顶点的方向上绘制一条线。线到达顶点或与之前绘制的另一条线相交时停止。你需要经过所有线划分成的区域，当你在一个区域时，当且仅当另一个区域与你所在的区域共边时，你才能到这个区域，求你最少需要穿过几次所画的线。

## 样例 #1

### 输入

```
5
```

### 输出

```
9
```

## 样例 #2

### 输入

```
3
```

### 输出

```
1
```

### 综合分析与结论
这些题解的核心思路基本一致，都是先通过找规律猜测答案为 $(n - 2)^2$，再通过分析每个顶点能连出的边数来证明该结论。具体来说，顶点 $1$ 能连出 $n - 3$ 条边，$2$ 和 $n$ 号点连出 $n - 3$ 条边（但部分会断掉），$3$ 到 $n - 1$ 号点（共 $n - 3$ 个点）连出 $n - 4$ 条边，最后将这些边数相加并加 $1$ 得到结果，再化简式子得到 $(n - 2)^2$。

### 重点代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin>>n;
    cout<<(n-2)*(n-2)<<endl;
    return 0;
}
```
**核心实现思想**：读取输入的 $n$，直接计算并输出 $(n - 2)^2$ 的结果。

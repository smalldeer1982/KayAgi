# 题目信息

# Uncowed Forces

## 题目描述

凯文刚刚完成了 CF 的比赛。测试时间为 120 分钟，有 5 个题目。题目 1~5 的满分分别是: 500、1000、1500、2000  和 2500。他作答了所有的题。

他知道自己每个问题的提交时间、每个问题上他提交错误的次数，以及它提交正确和错误的总数。

但 CF 评分的机制很复杂， 凯文想知道他的最终得分。

评分机制：设一个问题的最高分是 x ,提交者在第 m 分钟时提交正确，提交者错误提交了 w 此。那么，他的每道题的得分是： 

![](https://cdn.luogu.com.cn/upload/vjudge_pic/CF604A/3fb27769d723e1978a19f6c6254058e35687ad16.png)

他的总成绩是他每道题成绩的和。此外，凯文每通过一道题，他的总分数将增加 100 ；反之，他每错误一次，他的总分数将减少 50 分。

所有运算都是觉得准确的，没有四舍五入。凯文的得分绝对是整数。

## 样例 #1

### 输入

```
20 40 60 80 100
0 1 2 3 4
1 0
```

### 输出

```
4900
```

## 样例 #2

### 输入

```
119 119 119 119 119
0 0 0 0 0
10 0
```

### 输出

```
4930
```

### 综合分析与结论
这些题解的核心思路都是按照题目给定的评分机制进行模拟计算。具体步骤为：先输入每道题的提交时间、错误次数以及正确和错误的总数，然后根据公式计算每道题的得分并累加，最后根据正确和错误的题目数量进行加分和减分得到最终总分。

各题解的差异主要体现在数据存储方式（如数组、结构体）、每题满分的计算方式（打表、循环累加）以及对精度问题的处理上。大部分题解都使用 `double` 类型来避免精度损失，部分题解还给出了处理精度的具体技巧。

### 所选题解
- **作者：封禁用户 (赞：4)，4星**
    - **关键亮点**：使用结构体存储每道题的相关数据，思路清晰，代码可读性高，同时注意到了精度问题并使用 `double` 类型处理。
    - **核心代码**：
```cpp
struct problem{   //利用结构体存数据
    double x;   //每题满分
    double m;   //每题的提交时间
    double w;   //提交错误次数
    double f;   //每题得分
}t[6];
double score=0,ac,uac;   //定义最终得分、提交通过的次数、提交未通过的次数
int main()
{
    for(int i=1;i<=5;i++)
    {
        cin>>t[i].m;
    }
    for(int i=1;i<=5;i++)
    {
        cin>>t[i].w;
    }
    cin>>ac>>uac;
    for(int i=1;i<=5;i++)
    {
        t[i].x=500.0*i;   //计算保存每题满分
    }
    for(int i=1;i<=5;i++)
    {
        t[i].f=max(0.3*t[i].x,(1-t[i].m/250)*t[i].x-50*t[i].w);   //依据公式计算每题得分
        score+=t[i].f;   //累加得分
    }
    score=score+100*ac-50*uac;   //根据奖罚机制调整最终得分
    cout<<score;
    return 0;
}
```
核心实现思想：通过结构体存储每道题的信息，方便后续计算。先输入数据，再计算每题满分，接着根据公式计算每题得分并累加，最后根据正确和错误的题目数量调整最终得分。

- **作者：Chtholly_L (赞：4)，4星**

```cpp
double m[6];
double w[6];
double s[6];
double ac,wa,x=500,sum;
int main()
{
    for(int i=1;i<=5;i++)
    {
        cin>>m[i];
    }
    for(int i=1;i<=5;i++)
    {
        cin>>w[i];
    }
    cin>>ac>>wa;
    for(int i=1;i<=5;i++)
    {
        s[i]=max(0.3*x,(1-m[i]/250)*x-50*w[i]);
        x+=500;
    }
    for(int i=1;i<=5;i++)
    {
        sum+=s[i];
    }
    cout<<sum+100*ac-50*wa;
    return 0;
}
```
核心实现思想：使用数组存储每道题的提交时间和错误次数，通过一个变量 `x` 循环累加得到每题满分，再根据公式计算每题得分并累加，最后根据正确和错误的题目数量得到最终得分。

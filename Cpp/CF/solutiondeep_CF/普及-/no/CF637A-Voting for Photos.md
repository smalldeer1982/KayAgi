# 题目信息

# Voting for Photos

## 题目描述

给定一个数组，找出出现最多的数，若有多个，输出最后一次出现时最靠前的数。

## 样例 #1

### 输入

```
5
1 3 2 2 1
```

### 输出

```
2
```

## 样例 #2

### 输入

```
9
100 200 300 200 100 300 300 100 200
```

### 输出

```
300
```

### 通用建议与扩展思路
- 对于此类统计元素出现次数的问题，若数据范围较小，可优先考虑使用数组（桶），时间复杂度低；若数据范围较大，可使用 `map` 容器节省空间。
- 当需要处理多个条件（如出现次数相同但最后出现位置不同）时，要仔细分析判断条件，确保逻辑正确。

### 重点代码
以下是使用数组（桶）实现的核心代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int a,b[1000002],c=0,d,e,f,g,h,i,j;
int main()
{
    cin>>a;
    for(i=1;i<=a;i++)
    {
        cin>>d;
        b[d]++; //入桶
        if(b[d]>c) //更新最大值
        {
            c=b[d];
            e=d; //存哪位选手
        }
    }
    cout<<e; //输出
    return 0;
}
```
核心实现思想：通过数组 `b` 作为桶来统计每个数的出现次数，在输入过程中不断更新出现次数的最大值和对应的数字。

# 题目信息

# Game of Robots

## 题目描述

有$n(n\leq100000)$个机器人，每个机器人都有一个唯一的整数序号，该序号在$1$到$10^9$之间。他们在做一个叫“滚雪球”的游戏，即第一个机器人说出第一个机器人的序号，第二个机器人说出第一到第二个机器人的序号，第三个机器人说出第一到第三个机器人的序号……以此类推。求第$k(k\leq min(2\cdot10^9,n\cdot(n+1)/2)$个被说出的序号。

## 样例 #1

### 输入

```
2 2
1 2
```

### 输出

```
1
```

## 样例 #2

### 输入

```
4 5
10 4 18 3
```

### 输出

```
4
```

### 综合分析与结论
这些题解的核心思路都是模拟机器人“滚雪球”叫序号的过程，来找出第 $k$ 个被说出的序号。
- **思路对比**：大部分题解是通过循环依次累加每个机器人说出的序号个数，直到找到包含第 $k$ 个序号的机器人；而 henrywyh 的题解通过数学推导，先确定包含第 $k$ 个序号的机器人，减少了循环次数。
- **算法要点**：各题解都围绕着计算每个机器人说出的序号个数和确定第 $k$ 个序号所在的位置展开。
- **解决难点**：关键在于准确计算出第 $k$ 个序号是由哪个机器人说出的，以及在该机器人说出的序号中是第几个。

### 所选题解
- **LJN1117（4星）**：
    - **关键亮点**：思路清晰，代码简洁，使用前缀和的思想，通过简单的循环和判断就能找到答案。
    - **核心代码**：
```cpp
#include<bits/stdc++.h>
#define int long long
#define maxn 100000+10

using namespace std;

int n,k,a[maxn],sum;

signed main(){
    scanf("%lld %lld",&n,&k);
    for(int i=1;i<=n;i++) scanf("%lld",&a[i]);
    for(int i=1;i<=n;i++){
        sum+=i;
        if(sum>=k){
            sum=i-(sum-k);
            printf("%lld\n",a[sum]);
            break;
        }
    }
    return 0;
}
```
核心实现思想：用 `sum` 记录前 $i$ 个机器人说出的序号总数，当 `sum` 大于等于 $k$ 时，计算出第 $k$ 个序号在第 $i$ 个机器人说出的序号中的位置，然后输出对应序号。

- **henrywyh（4星）**：
    - **关键亮点**：通过数学推导，从 $\sqrt{2k}$ 开始往下试，减少了循环次数，优化了时间复杂度。
    - **核心代码**：
```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long
const int maxn=1e5+5;
int num[maxn];
signed main()
{
    int n,k,i;
    cin>>n>>k;
    for(i=1;i<=n;i++)
    {
        cin>>num[i];
    }
    for(i=sqrt(2*k);;i--)
    {
        if(i*(i+1)<2*k)
            break;
    }
    int s=k-i*(i+1)/2;
    cout<<num[s];
    return 0;
}
```
核心实现思想：先通过数学公式找到最大的 $m$，使得前 $m$ 个机器人说出的序号总数小于 $k$，然后计算出第 $k$ 个序号是第 $m + 1$ 个机器人说出的第几个序号，最后输出对应序号。

### 最优关键思路或技巧
henrywyh 的题解通过数学推导，从 $\sqrt{2k}$ 开始往下试，减少了循环次数，优化了时间复杂度。这种利用数学公式进行优化的思路值得学习。

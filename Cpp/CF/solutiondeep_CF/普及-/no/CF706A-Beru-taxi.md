### 题目翻译
Vasiliy位于点 (a, b)，他周围有 n 辆出租车，这些出租车的坐标为点 (x, y)，速度为 v。假设 Vasiliy 已经知道他最早能坐上哪辆出租车，请你求出他最早能坐上出租车的时间（出租车将直线朝 Vasiliy 行驶）。

### 综合分析与结论
这些题解的核心思路都是利用数学中的勾股定理计算出租车到 Vasiliy 位置的距离，再结合出租车的速度计算出每辆出租车到达的时间，最后找出最短时间。
- **思路对比**：DreamsChaser 仅给出思路，未提供代码；happybob 直接在循环中更新最小时间；Eason_AC 通过结构体存储信息并排序找出最小时间；sysu_yzc 在循环中比较更新最小时间。
- **算法要点**：都运用勾股定理计算距离，再用距离除以速度得到时间。
- **解决难点**：准确计算距离和时间，并找出最小时间。

### 重点代码
```cpp
#include <iostream>
#include <iomanip>
#include <algorithm>
#include <cmath>
using namespace std;

int main()
{
	double a, b, n, ans = 1e14;
	cin >> a >> b >> n;
	for (int i = 1; i <= n; i++)
	{
		double x, y, v;
		cin >> x >> y >> v;
		ans = min(ans, sqrt(pow(x - a, 2) + pow(y - b, 2)) / v);
	}
	cout << fixed << setprecision(20) << ans << endl;
	return 0;
}
```
**核心实现思想**：先初始化最小时间为一个较大值，然后在循环中读取每辆出租车的坐标和速度，计算其到达 Vasiliy 位置的时间，并更新最小时间，最后输出最小时间。

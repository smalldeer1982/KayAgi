# Alyona and copybooks

## 题目描述

Alyona 在学校学习 4 门课程，她要去书店买笔记本。她想让 4 门课程的笔记本数量相同。

Alyona 一开始时有 $n$ 个笔记本。书店有 $3$ 种笔记本购买方式：

 1. 买一个笔记本，花费 $a$ 卢布；
 2. 买两个笔记本，花费 $b$ 卢布；
 3. 买三个笔记本，花费 $c$ 卢布。
 
Alyona 可以购买任意次笔记本。问最少要花多少卢布？

## 样例 #1

3

### 算法分类
数学

### 综合分析与结论
这些题解的核心思路都是先计算初始笔记本数量 $n$ 除以 4 的余数，再根据余数的不同情况，枚举所有可能的购买组合，计算每种组合的花费，最后取最小值作为最少花费。

- **思路**：通过计算 $n$ 对 4 取余，根据余数为 0、1、2、3 分别讨论，找出每种余数情况下花费最少的购买方案。
- **算法要点**：取余运算和枚举不同购买组合并比较花费。
- **解决难点**：全面考虑所有可能的购买组合，避免遗漏，如余数为 2 时，要考虑用两次买 3 本的方式。

#include<iostream>

#include<bits/stdc++.h>

int main()
{
    ll n,a,b,c,s;
    cin>>n>>a>>b>>c;
    ll d=4-n%4;
    if(d==1) s=min(a,min(b+c,c*3));
    if(d==2) s=min(b,min(a*2,c*2));
    if(d==3) s=min(a*3,min(a+b,c));
    if(d==4) s=0;
    cout<<s;
    return 0;
}

### 核心实现思想
先读取输入的 $n$、$a$、$b$、$c$，计算 $n$ 对 4 的余数或 $4$ 减去该余数得到 $d$，根据不同的余数或 $d$ 值，枚举所有可能的购买组合，计算每种组合的花费，使用 `min` 函数找出最小花费并输出。

### 拓展思路
同类型题目可能会改变课程数量（即除数）或购买方式，解题思路仍然是先取余，再枚举所有可能的组合并比较花费。

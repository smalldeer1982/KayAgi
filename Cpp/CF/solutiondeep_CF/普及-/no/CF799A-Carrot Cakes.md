# 题目信息

# Carrot Cakes

## 题目描述

一共需要烤 $n$ 个蛋糕，用一个烤箱烤 $k$ 个蛋糕需要 $t$ 分钟。现有一个烤箱，可以再用 $d$ 分钟建一个。建烤箱时正在建的烤箱不能工作，但另一个烤箱可以。问：新建烤箱烤完 $n$ 个蛋糕的时间是否比不新建要严格快（即不能等于）？

## 样例 #1

### 输入

```
8 6 4 5
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
8 6 4 6
```

### 输出

```
NO
```

## 样例 #3

### 输入

```
10 3 11 4
```

### 输出

```
NO
```

## 样例 #4

### 输入

```
4 2 1 4
```

### 输出

```
YES
```

### 综合分析与结论
这些题解主要围绕比较新建烤箱和不新建烤箱烤完蛋糕所需时间来判断是否新建烤箱更快。大部分题解采用比较加烤箱后的平均速度 $\frac{(n - 1)t}{k}$ 与建造时间 $d$ 的方法；而作者为⊱⋛赫宇⋚⊰的题解则分别计算了不新建烤箱和新建烤箱所需的时间，再进行比较。

各题解思路总结：
- 多数题解思路：通过计算加烤箱后的平均速度 $\frac{(n - 1)t}{k}$ 与建造时间 $d$ 比较，若平均速度大于 $d$ 则输出 `YES`，否则输出 `NO`。
- ⊱⋛赫宇⋚⊰的题解思路：分别计算不新建烤箱的时间 $T1$ 和新建烤箱的时间 $T2$，若 $T2 < T1$ 则输出 `YES`，否则输出 `NO`。

难点对比：多数题解的难点在于理解平均速度的计算和比较；⊱⋛赫宇⋚⊰的题解难点在于准确计算新建烤箱时的时间，要考虑建造过程中已有烤箱的工作情况。

### 所选题解
- 作者：⊱⋛赫宇⋚⊰ (赞：3)，4星。关键亮点：思路清晰，通过分别计算两种情况下的时间进行比较，逻辑明确，且代码中有详细注释。
```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long
signed main(){
    int n,t,k,d;
    cin >> n >> t >> k >> d;
    //计算不用建烤箱所要的时间
    int T1 =  t*( (n/k) + (n%k!=0));
    //计算需要烤箱所要的时间，在此过程中要注意，在建造的过程中也可以用烤箱
    int T2 =  d + t*(( n-(d/t)*k )/(2*k) +(( n-(d/t)*k )%(2*k)!=0));
    if(T2 < T1)puts("YES");
    else puts("NO");
    return 0;
}
```
核心实现思想：先计算不新建烤箱时，根据蛋糕总数 $n$、每次烤的数量 $k$ 和每次烤制时间 $t$ 算出总时间 $T1$。再计算新建烤箱的时间 $T2$，先减去建造烤箱时间 $d$ 内已有烤箱烤制的蛋糕数，然后考虑两个烤箱同时工作的情况算出剩余时间，最后加上建造时间 $d$ 得到 $T2$，比较 $T1$ 和 $T2$ 输出结果。

### 最优关键思路或技巧
对于此类问题，可以先明确不同情况的计算方式，如本题分别计算不新建和新建烤箱的时间，再进行比较。同时，在计算过程中要仔细考虑各种条件，如本题中建造烤箱时已有烤箱的工作情况。

# 题目信息

# Diplomas and Certificates

## 题目描述

有n名学生参加了比赛，现在奖项已经公布了下来。   
n名学生中，中有人获得了文凭，也有人获得了证书，但还有人什么都没有。  

其中官方颁奖还有个规则：    
1.证书的数量必须恰好是文凭数量的k倍。   
2.获奖者人数不得超过所有学生人数的一半(即不超过n的一半)。

当然也可能没有赢家。

而题目让你求出最大的赢家数量，以及这种情况下，有文凭的学生人数，有证书的学生人数和没有获奖的学生人数。

【综合分析与结论】
这些题解思路基本一致，都是通过设未知数建立数学方程来求解。先设获得文凭的学生人数为\(x\)，根据证书数量是文凭数量的\(k\)倍，得出获得证书的学生人数为\(k\times x\)，再结合获奖者人数不超过总人数一半的条件，列出方程\(x + k\times x = \frac{n}{2}\)，进而求出\(x=\frac{n}{2\times(k + 1)}\)，最后算出获得证书的人数和未获奖的人数。各题解的差异主要体现在代码风格和公式推导的详细程度上。

【核心实现思想】
先读取输入的总人数\(n\)和倍数\(k\)，然后根据数学推导得出获得文凭的人数\(x=\frac{n}{2\times(k + 1)}\)，再计算出获得证书的人数为\(k\times x\)，最后用总人数减去获得文凭和证书的人数，得到未获奖的人数，将这三个结果输出。

【关键思路或技巧】
- 运用数学建模的思想，通过设未知数建立方程来求解问题。
- 注意数据范围，由于\(n\)和\(k\)的取值范围较大，需要使用`long long`类型来避免数据溢出。

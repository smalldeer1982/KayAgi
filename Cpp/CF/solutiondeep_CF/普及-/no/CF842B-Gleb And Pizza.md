# Gleb And Pizza

## 题目描述

### 题目大意

Gleb 点了披萨，他发现有几片香肠在外皮上，但他讨厌披萨的外皮。

披萨一个半径为  $r$ 的圆，圆心位于原点，披萨中间部分是半径为  $r-d$ 的圆，圆心也在原点。外皮部分就是外面一圈宽为  $d$ 的环。

香肠片也是圆的，对于第  $i$ 片香肠，他的半径为  $r_i$ ，圆心位于  $(x_i,y_i)$

现在求有多少个香肠片整个都在外皮上。

### 综合分析与结论
这些题解的核心思路基本一致，都是将问题抽象为几何问题，通过计算香肠圆心到披萨圆心的距离，结合香肠半径和披萨的内外半径来判断香肠是否完全在披萨外皮上。具体做法是，连接香肠圆心与披萨圆心，计算这条直线与香肠边缘两个交点到披萨圆心的距离，若这两个距离分别满足大于等于小圆半径且小于等于大圆半径，则该香肠完全在披萨外皮上。

#include<iostream>
#include<cmath>
using namespace std;
int r,d,n,ans,x,y,z;
int main(){
    cin>>r>>d>>n;

        dis=sqrt(x*x+y*y);    //计算距离

**核心实现思想**：先读取披萨的大圆半径 `r`、大圆半径与小圆半径的差 `d` 以及香肠的数量 `n`。然后循环读取每个香肠的圆心坐标 `(x, y)` 和半径 `r1`，计算香肠圆心到披萨圆心的距离 `dis`，判断 `dis - r1` 是否大于等于 `r - d` 且 `dis + r1` 是否小于等于 `r`，如果满足条件则计数器加1，最后输出计数器的值。

# 题目信息

# Maxim Buys an Apartment

## 题目描述

Maxim想在大都会的莱恩大道上买一座新的公寓。这幢楼房共有 n 座公寓，从 1 到 n 编号排列在一个数列中。两座相邻公寓的编号相差为 1。有一些公寓已经被售出了，另外的一些是待售的。

Maxim时常拜访他的邻居，因此如果有一座待售公寓，其相邻公寓中至少有一座是已售出的，那么这座公寓很适合Maxim。Maxim已经知道了有 k 座公寓已被售出，但是他还不知道这些已售出公寓的编号是多少。

计算出适合Maxim的公寓数可能的最小值及最大值。

## 说明/提示

在样例中，如果编号为 1，2，3 的公寓被售出了，那么只有编号为 4 的公寓适合Maxim。如果编号为 1，3，5 的公寓被售出了，那么编号为 2，4，6 的公寓都是适合Maxim的。

### 综合分析与结论
这些题解的核心思路都是分别计算适合Maxim的公寓数的最小值和最大值。
- **最小值**：当`k == 0`或`k == n`时，适合的公寓数为0，否则将已售公寓集中排列，适合的公寓数为1。
- **最大值**：将已售公寓每隔两个位置放置，当`3 * k < n`时，每个已售公寓左右两边的待售公寓都适合，数量为`2 * k`；当`3 * k >= n`时，剩下的所有待售公寓都适合，数量为`n - k`。

各题解的区别主要在于代码实现风格和细节处理，思路基本一致。

**核心实现思想**：先判断`k`和`n`的关系，如果`k == 0`或`k == n`，则直接输出`0 0`；否则先输出最小值1，再根据`3 * k`和`n`的大小关系输出最大值（`2 * k`或`n - k`）。

### 关键思路或技巧
- **贪心思想**：在计算最小值时，将已售公寓集中排列，使得适合的公寓数最少；在计算最大值时，将已售公寓间隔排列，充分利用每个已售公寓的相邻位置。
- **数学分析**：通过分析已售公寓和待售公寓的数量关系，得出最大值的计算公式。

# 题目信息

# Two Cakes

## 题目描述

除夕将至，因此Ivan觉得应该开始摆盘了。Ivan买了两个蛋糕并且把他们切成了块：第一个蛋糕被切成a块，第二个被切成b块。 Ivan知道包括他在内将会有n个人参加庆祝活动，所以他已经为蛋糕准备了n个盘子。现在他在思考怎样分配盘子和蛋糕。Ivan想要满足一下所有条件： 1.每块蛋糕都被放在盘子上； 2.每个盘子至少放了一块蛋糕； 3.没有盘子放了两个不同的蛋糕（即切开前分别属于两个不同的大蛋糕）； 为了给客人带来更多快♂感，Ivan想让（数量）最小份的蛋糕数量尽可能大。他希望知道最大的x，使得他使得他可以根据上述条件分发蛋糕，并且每个盘子包含至少x块蛋糕。 帮助伊万来计算这个数字 x！

## 说明/提示

在第一个样例中，只有一种方法可以将蛋糕分配进盘子，即每个盘子1个蛋糕。 在第二个样例中，你可以在两个盘子分别放3块第一个蛋糕和4块第二个蛋糕，另外两个盘子都放5块第二个蛋糕。最少的块数是3。

## 样例 #1

### 输入

```
5 2 3
```

### 输出

```
1
```

## 样例 #2

### 输入

```
4 7 10
```

### 输出

```
3
```

### 综合分析与结论
- **思路**：题目的核心是要找到一个最大的数 `x`，使得将两个蛋糕分别按每盘 `x` 块来分配，所用盘子数之和大于等于总人数 `n`。题解采用从两个蛋糕块数的最小值开始递减枚举的方式，依次检查每个数是否满足条件。
- **算法要点**：通过 `for` 循环从 `min(m, k)` 开始递减，对于每个数 `i`，判断 `(m / i + k / i)` 是否大于等于 `n`，若满足则输出该数并结束程序。
- **解决难点**：难点在于理解题目要求，将其转化为数学模型，即找到满足特定条件的最大数。题解通过枚举的方式，从大到小依次尝试，确保找到的数是最大的。

### 题解评分与亮点
- **星级**：3星
- **关键亮点**：思路简单直接，代码实现简洁，对于数据范围较小的情况可以直接使用 `for` 循环解决问题。

### 重点代码
```c
#include<stdio.h>
#include<algorithm>
using namespace std;
int main(){
    int n,m,k,i;
    scanf("%d%d%d",&n,&m,&k);
    for(i=min(m,k);i>0;i--){
        if((m/i+k/i)>=n){
            printf("%d",i);
            return 0;
        }
    }
    return 0;
}
```
**核心实现思想**：从两个蛋糕块数的最小值开始递减枚举，对于每个数 `i`，计算两个蛋糕按每盘 `i` 块分配所需的盘子数之和，若该和大于等于总人数 `n`，则输出 `i` 并结束程序。

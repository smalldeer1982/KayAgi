# Weird Subtraction Process

## 题目描述

两个数 $a$ 和 $b$ 要对它们做以下操作:

1. 如果 $a=0$ 或 $b=0$ ，结束程序。否则执行第 $2$ 步。

2. 如果 $a\geq2b$ ，把 $a$ 变为 $a-2b$ ，并返回第 $1$ 步。否则执行第 $3$ 步。

3. 如果 $b\geq 2 a$ ，把 $b$ 变为 $b-2a$，并返回第 $1$ 步。否则结束程序。

你需要将 $a,b$ 持续进行操作直到结束程序，输出结束程序时的 $a,b$。

## 样例 #1

### 输入

```
12 5
```

### 输出

```
0 1
```

## 样例 #2

### 输入

```
31 12
```

### 输出

```
7 12
```

### 重点代码
```cpp
// zhang_kevin 优化后的代码
#include<iostream>
using namespace std;
typedef long long ll;
int main(){
    ll a, b;
    cin >> a >> b;
    while(true){                     //循环 
        if((a==0)||(b==0)){          //步骤 1 
            break;                   //结束循环 
        }else if(a>=2*b){            //步骤 2
            a %= 2 * b;              //改变 a 的值
            continue;                //回到步骤 1
        }else if(b>=2*a){            //步骤 3 
            b %= 2 * a;              //改变 b 的值 
        }else{                       //否则 
            break;                   //结束循环 
        }
    }
    cout << a << " " << b << endl;
    return 0;
}
```
**核心实现思想**：使用 `while` 循环不断判断 `a` 和 `b` 的关系，根据不同条件更新 `a` 或 `b` 的值，当 `a` 或 `b` 为 0 或者不满足 `a >= 2 * b` 和 `b >= 2 * a` 时，结束循环并输出结果。

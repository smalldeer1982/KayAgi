# 题目信息

# Mike and palindrome

## 题目描述

麦克有一个只有小写英文字母的字符串 _s_ 。他想通过改变一个字符 的方式来让整个字符串成为回文字符串。

回文字符串是一个正着读和反着读都一样的字符串。比如说"z"， "aaa"，"aba"， "abccba"是回文字符串，而"codeforces"， "reality"和"ab"不是回文字符串。

## 样例 #1

### 输入

```
abccaa
```

### 输出

```
YES
```

## 样例 #2

### 输入

```
abbcca
```

### 输出

```
NO
```

## 样例 #3

### 输入

```
abcda
```

### 输出

```
YES
```

### 题解分析与结论

#### 综合分析
本题的核心是通过改变一个字符使字符串成为回文。大多数题解采用了双指针法，通过比较字符串的首尾字符，统计不匹配的次数，并根据不匹配的次数和字符串长度的奇偶性来判断是否可以通过改变一个字符使字符串成为回文。

#### 最优关键思路
1. **双指针法**：使用两个指针分别从字符串的首尾向中间移动，统计不匹配的次数。
2. **特判处理**：当字符串已经是回文且长度为奇数时，可以通过改变中间字符使其仍然是回文。
3. **提前跳出循环**：当不匹配次数超过1时，可以提前判断为“NO”，减少不必要的计算。

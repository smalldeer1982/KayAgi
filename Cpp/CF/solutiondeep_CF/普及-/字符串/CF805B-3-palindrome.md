# 题目信息

# 3-palindrome

## 题目描述

题意
给定n，求长度为n的以'a','b'或'c'构成的字符串，使得其中不存在长为3的回文子串（如"abc","abca"均符合，但"aba"不符），并使'c'的个数尽可能少

## 样例 #1

### 输入

```
2
```

### 输出

```
aa
```

## 样例 #2

### 输入

```
3
```

### 输出

```
bba
```

- **共同点**：所有题解都认识到可以通过构造一个循环节来避免长度为 3 的回文子串，并且都尽量不使用 `'c'`。
- **不同点**：不同题解选择的循环节不同，如 `"bbaa"`、`"abba"`、`"aabb"` 等，但核心思想都是通过交替输出 `'a'` 和 `'b'` 来避免回文。

#### 2. 最优思路与技巧
- **关键思路**：通过构造一个长度为 4 的循环节（如 `"bbaa"` 或 `"abba"`），可以确保任意长度为 3 的子串都不会形成回文，同时避免使用 `'c'`。
- **实现技巧**：使用模运算（`i % 4`）来控制字符的输出顺序，代码简洁且高效。

#### 3. 可拓展之处
- 类似问题：构造字符串时避免特定模式的子串（如回文、重复子串等）。

# Nearest Minimums

## 题目描述

You are given an array of $ n $ integer numbers $ a_{0},a_{1},...,a_{n-1} $ . Find the distance between two closest (nearest) minimums in it. It is guaranteed that in the array a minimum occurs at least two times.
找到一个长度为n的序列中，距离最短的最小值的距离。

## 样例 #1

### 输入

```
2
3 3
```

### 输出

```
1
```

## 样例 #2

### 输入

```
3
5 6 5
```

### 输出

```
2
```

## 样例 #3

### 输入

```
9
2 1 3 5 4 1 2 3 1
```

### 输出

```
3
```

### 所选题解
- da32s1da（4星）
    - 关键亮点：不用数组，在输入过程中动态更新最小值、答案和位置，空间复杂度低。
    - 核心代码：
```c
#include<cstdio>
const int inf=1e9+7;
int minn,wz,n,x,ans;
int main(){
    scanf("%d",&n);
    minn=inf;ans=n;
    for(int i=1;i<=n;i++){
        scanf("%d",&x);
        if(minn==x&&ans>i-wz)ans=i-wz;
        if(minn==x)wz=i;
        if(minn>x)minn=x,wz=i,ans=n;
    }
    printf("%d\n",ans);
}
```
    - 核心实现思想：在输入每个数时，判断该数与当前最小值的关系，若等于最小值则更新答案和位置，若小于最小值则更新最小值、答案和位置。

### 最优关键思路或技巧

# Simple Palindrome

## 题目描述

构造一个 **长度为** $n$ ，且只由 $a$，$e$，$i$，$o$，$u$ **这 $5$ 种字母** 构成字符串，使得这个字符串的 **所有子序列** 是 **回文串** 的个数最少。请输出这个 **最小值** 。

## 说明/提示

对于 $100\%$ 的测试点，$1\le T\le 100$， $1\le n\le 100$ 。

## 样例 #1

### 输入

```
3
2
3
6```

### 输出

```
uo
iae
oeiiua```

```cpp
#include <iostream>
#include <string>

std::string constructString(int n) {
    std::string result;
    std::string chars = "aeiou";
    for (int i = 0; i < n; ++i) {
        result += chars[i % 5];
    }
    return result;
}

int main() {
    int T, n;
    std::cin >> T;
    while (T--) {
        std::cin >> n;
        std::cout << constructString(n) << std::endl;
    }
    return 0;
}
```

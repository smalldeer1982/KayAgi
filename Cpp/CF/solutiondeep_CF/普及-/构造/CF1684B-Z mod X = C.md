# 题目信息

# Z mod X = C

## 题目描述

给定三个整数 $a,b,c$，求三个整数 $x,y,z$，满足

$$x \bmod{y} =a$$
$$y \bmod{z} =b$$
$$z \bmod{x} =c$$

对于给定数据，保证答案存在。

## 说明/提示

$1 \le t \le 10^4$

$1 \le a < b < c \le 10^8$

## 样例 #1

### 输入

```
4
1 3 4
127 234 421
2 7 8
59 94 388```

### 输出

```
12 11 4
1063 234 1484
25 23 8
2221 94 2609```

### 综合分析与结论
这些题解主要围绕构造满足条件的 $x,y,z$ 展开，多数题解采用了两种思路：
- **思路一**：令 $x = a + b + c$，$y = b + c$，$z = c$。此思路通过取模运算的性质，简单直观地构造出满足条件的解，证明过程也较为清晰。
- **思路二**：令 $y = b$，$z = c$，再找到一个大于 $c$ 且满足 $x \bmod b = a$ 的 $x$，即 $x = k * b + a$（$k > \frac{c - a}{b}$）。

难点在于如何快速找到 $x,y,z$ 与 $a,b,c$ 之间的关系，通过对取模运算性质的观察和利用，以及对样例的分析，可以得到构造思路。

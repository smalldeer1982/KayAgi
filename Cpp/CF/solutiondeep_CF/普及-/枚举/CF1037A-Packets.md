# 题目信息

# Packets

## 题目描述

你有$n$个硬币，每个硬币价值都为$1$。

你要把它们分成若干个小包裹，使得在$1$与$n$之间的所有面额都能用这其中某几个小包裹凑出。

每个小包裹只能作为一个整体使用。

请求出最少要分几个包裹。

## 样例 #1

### 输入

```
6
```

### 输出

```
3```

## 样例 #2

### 输入

```
2
```

### 输出

```
2```

### 综合分析与结论
这些题解主要围绕二进制原理解决将硬币分成最少包裹以凑出$1$到$n$所有面额的问题。各题解思路本质相同，但实现方式有差异。
- **思路对比**：大部分题解都利用了二进制数的特性，即$2^0, 2^1, 2^2, \cdots, 2^n$可以组合出$1$到$2^{n + 1} - 1$之间的所有数。部分题解通过累加$2$的幂次判断，部分使用对数函数计算，还有通过找规律和枚举等方式。
- **算法要点**：核心要点是找到满足条件的最小$k$，使得$2^k - 1 \geq n$。可以通过循环累加$2$的幂次、使用对数函数$\log_2(n) + 1$、不断除以$2$统计次数等方法实现。
- **解决难点**：关键在于理解二进制拆分与组合的原理，以及如何根据这个原理设计算法来确定最少包裹数。

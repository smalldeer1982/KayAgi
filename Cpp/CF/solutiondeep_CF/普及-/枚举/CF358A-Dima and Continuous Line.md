# 题目信息

# Dima and Continuous Line

## 题目描述

在 $x$ 轴上存在 $n$ 个点 $(x_1,0),(x_2,0),(x_3,0),\dots ,(x_n,0)$，以相邻两点 $(x_i,0)$ 与 $(x_{i+1},0)$ 的连线作为直径向上作半圆，问是否有除 $x$ 轴上的交点。

## 说明/提示

$1 \le n \le 10^3, -10^6 \le x_i \le 10^6$。

## 样例 #1

### 输入

```
4
0 10 5 15
```

### 输出

```
yes
```

## 样例 #2

### 输入

```
4
0 15 5 10
```

### 输出

```
no
```

# AI分析结果

### 题解综合分析与结论
这些题解的核心目标都是判断以相邻两点为直径作的半圆是否存在除 $x$ 轴上的交点。思路主要分为两类：
- **几何关系判断**：通过计算圆心、半径和圆心距，利用两圆相交的几何条件 $\vert r_1 - r_2 \vert < d < r_1 + r_2$ 来判断半圆是否相交。如 muyang_233、lizhengdong、conti123、yutong_Seafloor、GuoJZ2010 的题解。
- **区间相交判断**：将半圆转化为区间，通过判断区间是否相交来确定半圆是否相交。如 WhiteSunFlower、czyczyczyczyczy、Last_kiss_Snow_Dog6、我梦见一片焦土、Moon_Traveller、Floating_Trees 的题解。还有 Muruski 的题解则是直接列举出所有可能的相交情况进行判断。

各题解的难点主要在于如何准确计算圆心、半径和圆心距，以及如何正确判断区间是否相交。部分题解还需要注意处理点的顺序和特判情况。

```cpp
inline bool check(int _1,int _2){
    double O1=(a[_1]+a[_1+1])/2.0;
    double O2=(a[_2]+a[_2+1])/2.0;
    double R1=abs(a[_1]-O1);
    double R2=abs(a[_2]-O2);
    return abs(O1-O2)<R1+R2&&abs(O1-O2)>abs(R1-R2);
}
```
核心实现思想：通过中点公式计算两圆的圆心，再计算半径，最后根据两圆相交的几何条件判断两圆是否相交。

- **WhiteSunFlower（4星）**

```cpp
for(int i=1;i<=n-1&&check;i++){
    int l=u[i],r=v[i];

        if(i==j)continue;
        int L=u[j],R=v[j];
        if(L<l&&l<R&&R<r||l<L&&L<r&&r<R){
            check=0;
        }
    }
}
```
核心实现思想：枚举所有半圆对，将半圆端点排序后，根据推导的区间相交条件判断半圆是否相交。

# 题目信息

# Pineapple Incident

## 题目描述

Ted有一个菠萝。这个菠萝可以像狗一样叫！在第 $t$ 秒它第一次叫。然后每隔 $s$ 秒之后它会叫一次，在叫这一次之后还会在后一秒再叫一次。意思就是说它会在 $t$ 秒，$t+s$ 秒，$t+s+1$ 秒，$t+2s$ 秒，$t+2s+1$ 秒分别叫一次，以此类推。 

Barney在早上醒来，他想吃个菠萝，但这只菠萝叫的时候Barney不能吃它。Barney打算在 $x$ 秒吃它，所以请你告诉Barney在x秒的时候，这个菠萝会不会叫。

## 样例 #1

### 输入

```
3 10 4
```

### 输出

```
NO
```

## 样例 #2

### 输入

```
3 10 3
```

### 输出

```
YES
```

## 样例 #3

### 输入

```
3 8 51
```

### 输出

```
YES
```

## 样例 #4

### 输入

```
3 8 52
```

### 输出

```
YES
```

### 题目翻译

### 综合分析与结论
这些题解的核心目标都是判断在第 $x$ 秒时菠萝是否会叫。主要思路可分为两类：
1. **枚举法**：通过循环模拟菠萝叫的时间，判断 $x$ 是否在这些时间点上。如Michigan_King、NCC79601、Zofia、雪鸮、我是一个蒟蒻、_Error的题解。
2. **取模法**：根据菠萝叫的时间规律，将问题转化为判断 $(x - t)$ 对 $s$ 取模的结果是否为 0 或 1。如lnkkerst、NOILinux_Ubuntu、棒棒糖是猪、留守新手村、shenmadongdong、wpj20241021005、赤霞QvQ的题解。

难点在于处理题目翻译错误，即 $t + 1$ 秒时菠萝不会叫，多数题解都通过特判来解决这个问题。

### 评分较高的题解
- **lnkkerst（5星）**
    - **关键亮点**：思路清晰，代码简洁，直接利用取模判断，对翻译错误的处理明确。
    - **个人心得**：指出翻译错误导致 $t + 1$ 秒的判断问题。
    - **核心代码**：
```cpp
#include <cstdio>
int main() {
    int t, s, x;
    scanf("%d%d%d", &t, &s, &x);
    int tmp = x - t;
    if ((tmp % s == 0 || tmp % s == 1) && tmp >= 0 && tmp!= 1)
        printf("YES");
    else
        printf("NO");
    return 0;
}
```
- **NOILinux_Ubuntu（4星）**
    - **关键亮点**：详细分析了枚举法的优化过程，最终得出取模法，逻辑严谨。
    - **个人心得**：指出题目结合了枚举、优化、数论、翻译四种算法，强调要略过翻译看原题。
    - **核心代码**：

```cpp
using namespace std;
int t,s,x;
int main(){

```
- **留守新手村（4星）**
    - **关键亮点**：通过假设菠萝从时间 0 开始叫，简化了问题，代码逻辑清晰。
    - **核心代码**：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int t,s,x;
    scanf("%d%d%d",&t,&s,&x);
    x-=t;
    if(x < s && x!= 0) printf("NO");
    else if(x % s == 0 || x % s == 1) printf("YES");
    else printf("NO");
}
```

### 最优关键思路或技巧
取模法是最优思路，通过将问题转化为取模运算，避免了枚举的高时间复杂度。同时，要注意题目翻译可能存在的错误，对特殊情况进行特判。

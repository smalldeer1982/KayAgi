# Heist

## 题目描述

昨晚有一家电子商店被抢劫了。

昨天在商店里的所有键盘都是从$x$开始按升序编号的。例如，如果$x=4$，并且商店中有$3$个键盘，那么编号就为$4,5,6$；如果$x=10$，并且有$7$个键盘，那么编号就为$10,11,12,13,14,15,16$。

在抢劫之后，只有$n$个键盘仍然存在，他们的编号分别为$a_1,a_2,\dots ,a_n$。计算被盗的键盘的最小可能数量。工作人员不记得$x$也不记得商店里原来有几个键盘。

## 样例 #1

### 输入

```
4
10 13 12 8
```

### 输出

```
2
```

## 样例 #2

### 输入

```
5
7 5 6 4 8
```

### 输出

```
0
```

### 题目内容（已为中文，无需翻译）
昨晚有一家电子商店被抢劫了。

昨天在商店里的所有键盘都是从$x$开始按升序编号的。例如，如果$x=4$，并且商店中有$3$个键盘，那么编号就为$4,5,6$；如果$x=10$，并且有$7$个键盘，那么编号就为$10,11,12,13,14,15,16$。

在抢劫之后，只有$n$个键盘仍然存在，他们的编号分别为$a_1,a_2,\dots,a_n$。计算被盗的键盘的最小可能数量。工作人员不记得$x$也不记得商店里原来有几个键盘。

### 综合分析与结论
- **思路对比**：大部分题解思路一致，即找出剩余键盘编号中的最大值和最小值，计算从最小值到最大值的连续编号数量（最大值 - 最小值 + 1），再减去剩余键盘的数量$n$，得到被盗键盘的最小数量。部分题解采用排序后遍历数组，计算相邻元素差值减 1 并累加的方式。
- **算法要点**：
    - 主流方法：找出最大值和最小值，公式计算结果。
    - 遍历方法：排序后遍历数组，累加相邻元素差值减 1。
- **解决难点**：本题难点在于理解题意，明确要计算被盗键盘的最小数量，关键是确定连续编号的范围。

### 评分较高的题解
- **作者：piggymichael（5星）**
    - **关键亮点**：思路清晰，代码简洁，无需排序和开数组，直接在输入时记录最大值和最小值，时间复杂度$O(n)$，空间复杂度$O(1)$。
    - **核心代码**：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a,minn=0x3f3f3f3f,maxn=0;
int main()
{
    cin>>n;
    for(int i=1;i<=n;++i)
    {
        cin>>a;
        minn=min(a,minn);
        maxn=max(a,maxn);
    }
    cout<<maxn-minn+1-n<<endl;
    return 0;
}
```

```cpp
#include<iostream>

int main()

    for(int i=1;i<=n;i++)

    return 0;

```

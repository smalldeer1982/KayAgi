# DZY Loves Chessboard

## 题目描述

一个棋盘上有一些格子是坏的，另一些是正常的。对于每一个正常的格子，都要在上面放上棋子。
请找到一组解使没有两个相同颜色的棋子相邻(两个格子相邻为它们存在共同的边)

## 样例 #1

### 输入

```
1 1
.
```

### 输出

```
B
```

## 样例 #2

### 输入

```
2 2
..
..
```

### 输出

```
BW
WB
```

## 样例 #3

### 输入

```
3 3
.-.
---
--.```

### 输出

```
B-B
---
--B```

# AI分析结果

### 题目内容（已为中文，无需翻译）
一个棋盘上有一些格子是坏的，另一些是正常的。对于每一个正常的格子，都要在上面放上棋子。
请找到一组解使没有两个相同颜色的棋子相邻(两个格子相邻为它们存在共同的边)

### 综合分析与结论
这些题解的思路主要分为两类：一是利用棋盘黑白相间的规律直接填充；二是使用搜索算法（DFS或BFS）来找出符合条件的填充方案。
- **算法要点对比**：
    - **规律填充法**：通过判断坐标 `(i + j)` 的奇偶性来确定该位置放置黑棋（'B'）还是白棋（'W'），若该位置是坏格子则直接填充 '-'。这种方法代码简洁，时间复杂度为 $O(nm)$。
    - **搜索算法**：从某个正常格子开始，递归或迭代地尝试填充棋子，保证相邻格子颜色不同。DFS 可能会有超时风险，BFS 相对稳定，但代码实现相对复杂。
- **解决难点**：
    - 规律填充法的难点在于发现棋盘黑白相间的规律，以及处理坏格子的情况。
    - 搜索算法的难点在于控制搜索范围，避免越界，以及处理已填充的格子。

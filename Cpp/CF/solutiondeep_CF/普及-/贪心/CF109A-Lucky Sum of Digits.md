# Lucky Sum of Digits（幸运数字的数位和）

## 题目描述

Petya喜欢幸运数字。幸运数字是仅由4和7组成的正整数。例如47、744、4是幸运数字，而5、17、467不是。

给定数位和n，求满足该和的最小幸运数字。若不存在则输出-1。

---

## 题解对比与结论

### 最优思路总结
1. **贪心构造**：最小数的两个条件——**位数最少**且**字典序最小**。
2. **逆序枚举7的个数**：从最多可能的7开始枚举，剩余部分判断是否能被4整除。
3. **4在前7在后**：构造时先输出所有4再输出7，保证字典序最小。

### 关键技巧
- 当总位数相同时，4排列在前更优（如47 < 74）
- 7的权重（7）比4大，优先用7可以减少总位数

## 高分题解推荐

### 题解1：PC_DOS（5星）
**亮点**：逆序枚举7的个数，代码高效清晰  
**核心思路**：
```cpp
for (nSeven = iSum / 7; nSeven>=0; --nSeven) {
    nFour = iSum - 7 * nSeven;
    if (nFour % 4 == 0) {
        // 输出nFour/4个4和nSeven个7
        return 0;
    }
}
```

### 题解2：hensier（5星）
**亮点**：代码极简，循环逻辑与输出合并  
**核心实现**：
```cpp
for(i=n/7; i>=0; i--) {
    if((n-i*7)%4) continue;
    // 输出对应数量的4和7
    return 0;
}
```

### 题解3：灵光一闪（4星）
**亮点**：正向枚举4的个数，思路直观  
**代码要点**：
```cpp
for(int i=0; i<=n/4; i++) {
    if ((n-4*i)%7 == 0) {
        // 输出i个4和剩余7的个数
        return 0;
    }
}
```

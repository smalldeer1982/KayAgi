## 综合题解分析

所有题解均发现**颜色种类数直接决定操作次数**。通过统计不同颜色数量 $k$，最少操作次数为 $k-1$。这一规律来源于每次操作可消除至少一种颜色类型，最终将所有颜色统一。

### 关键思路总结
- **颜色种类统计**：使用集合或数组统计不同的颜色数量
- **数学规律**：操作次数 = 颜色种类数 - 1
- **无需考虑具体分布**：无论颜色是 2+2、3+1 或 1+1+1+1 分布，答案仅与颜色种类数相关

sort(c, c+4);

## 关键代码实现
以 Dry_ice 的代码为例：
```cpp
#include <stdio.h>
int main() {
    int t; scanf("%d", &t);
    while(t--) {
        int cnt[26] = {0}, res = 0;
        for(int i=0; i<2; i++) {
            char s[3]; scanf("%s", s);
            for(int j=0; j<2; j++) 
                if(++cnt[s[j]-'a'] == 1) res++;
        }
        printf("%d\n", res-1);
    }
}
```
**核心思想**：使用桶数组统计首次出现的颜色，统计总数后输出 res-1。

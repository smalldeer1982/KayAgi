# Nene's Game

## 题目描述

有 $n$ 个人玩一个游戏，把这 $n$ 个人分别编号为 $1$ 到 $n$。

每一轮，编号为 $ a_1, a_2, \ldots, a_k $ 的人会被踢出这个游戏，剩下的人会补齐空位并重新从 $1$ 开始编号。

当某一轮没有人被踢出时，游戏结束，剩下没有被踢出的人成为赢家。

现在请你求出最后会有多少人成为赢家。

由于 Nene 还没有确定参加游戏的具体人数，所以她会问你 $q$ 个问题，请你对 $q$ 个 $n$ 分别给出答案。

## 题解综合分析与结论

### 核心算法思想
所有题解的核心思路都是基于以下观察：
1. 每轮操作会持续淘汰编号不小于最小淘汰编号的人
2. 最终存活人数只与最小的淘汰编号 $m = \min(a_i)$ 有关
3. 结果可直接通过公式 $ans = \min(n, m-1)$ 计算

### 最优解法亮点
- **时间复杂度**：$O(k+q)$，预处理最小值后单次查询O(1)
- **空间复杂度**：O(1)，仅需存储最小值
- **思维突破点**：发现淘汰过程的最终状态仅由最小编号决定，无需模拟过程

## 关键思路与技巧

### 核心思维模式
1. **逆向思维**：从最终稳定状态倒推，发现当人数 < 最小淘汰编号时无法操作
2. **最值特性**：淘汰过程的终止条件仅由最小编号决定
3. **数学归纳**：通过观察多轮操作后的收敛特性，避免模拟过程

### 实现技巧
- 使用 `INT_MAX` 或 `0x7f7f7f7f` 初始化最小值
- 边读入边更新最小值，节省存储空间
- 利用 `min()` 函数简化条件判断

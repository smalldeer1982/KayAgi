# Fun Game

## 题目描述

### 题目大意

Vanya 有一个 01 串 $s=s_1s_2\dots s_n$。他可以对 $s$ 进行一系列变换。每次变换中，Vanya 会取两个整数 $l,r\left(1\le l\le r\le n\right)$，然后对所有 $i\in\left[l,r\right]$，将 $s_i$ **同时**替换为 $s_i\oplus s_{i-L+1}$，其中 $\oplus$ 表示按位异或。

Vanya 想知道 $s$ 能否经过一系列变换，得到另一个长度与 $s$ 相同的 01 串 $t$。

### 题目重写

**题目名称**：Fun Game

**题目描述**：

Vanya 有一个 01 串 $s = s_1s_2\dots s_n$。他可以进行若干次变换操作。每次操作中，Vanya 选择两个整数 $l$ 和 $r$（$1 \le l \le r \le n$），并对所有 $i \in [l, r]$，将 $s_i$ **同时**替换为 $s_i \oplus s_{i-l+1}$（其中 $\oplus$ 表示按位异或）。

Vanya 想知道能否通过一系列操作将 $s$ 转换为另一个长度相同的 01 串 $t$。

### 题解综合分析

#### 关键思路总结
1. **无法修改的区域**：在 $s$ 的第一个 `1` 出现之前的所有位置无法被操作改变，因此这些位置在 $t$ 中必须与 $s$ 完全一致。
2. **可修改的区域**：在 $s$ 的第一个 `1` 出现之后的位置，可以通过多次异或操作调整值。
3. **特殊情况处理**：
   - 若 $s$ 全为 `0`，则 $t$ 必须全为 `0`。
   - 若 $s$ 的第一个 `1` 的位置比 $t$ 的第一个 `1` 的位置靠后，则无法转换。

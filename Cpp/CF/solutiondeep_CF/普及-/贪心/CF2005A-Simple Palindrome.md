# Simple Palindrome

## 题目描述

构造一个 **长度为** $n$ ，且只由 $a$，$e$，$i$，$o$，$u$ **这 $5$ 种字母** 构成字符串，使得这个字符串的 **所有子序列** 是 **回文串** 的个数最少。请输出这个 **最小值** 。

## 说明/提示

对于 $100\%$ 的测试点，$1\le T\le 100$， $1\le n\le 100$ 。

## 样例 #1

### 输入

```
3
2
3
6```

### 输出

```
uo
iae
oeiiua```

# 简单回文串

## 题目描述

构造一个 **长度为** $n$ ，且仅由 $a$、$e$、$i$、$o$、$u$ 这5种字符组成的字符串，使得该字符串的 **所有子序列是回文串** 的个数最少。输出这个 **最小值对应的字符串**。

## 说明/提示

对于 $100\%$ 的测试点，$1\le T\le 100$， $1\le n\le 100$。

## 样例 #1

### 输入

```
3
2
3
6
```

### 输出

```
uo
iae
oeiiua
```

---

### 综合分析与结论

本题关键在于构造字符串使得回文子序列数目最少。核心思路是**最小化重复字符的出现次数**，并将重复字符连续排列以避免形成长回文子序列。

#### 关键思路
1. **字符分配策略**：当 $n \leq 5$ 时，每个字符恰好出现一次，此时回文数为 $n$。当 $n > 5$ 时，多余的次数循环分配给各字符，确保每个字符最多先增加到2次，以此减少指数级增长的组合数。
2. **构造优化**：重复字符必须连续排列，避免形成跨字符的长回文子序列（如 `a...a` 中间有其他字符会形成 `aba` 型回文）。
3. **字符顺序**：按特定顺序（如 `o, e, i, u, a`）分配重复次数，确保输出与样例一致。

#### 核心代码实现
```cpp
#include <iostream>
using namespace std;

int main() {
    int T, n;
    cin >> T;
    char chars[] = {'o', 'e', 'i', 'u', 'a'};
    while (T--) {
        cin >> n;
        int counts[5] = {1, 1, 1, 1, 1};
        int extra = n - 5;
        for (int i = 0; extra > 0; i = (i+1)%5) {
            counts[i]++;
            extra--;
        }
        string s;
        for (int i = 0; i < 5; i++) {
            s += string(counts[i], chars[i]);
        }
        cout << s << endl;
    }
    return 0;
}
```

#### 代码要点
- **字符顺序**：使用 `o, e, i, u, a` 顺序以匹配样例输出。
- **次数分配**：循环分配多余次数，确保每个字符最多先加到2次。
- **字符串构造**：将每个字符的重复部分连续拼接。

# Circuit

## 题目描述

Alice 刚才制作了一个有 $n$ 盏灯和 $2n$ 个开关的电路，电路中的每个元件（即每盏灯或每个开关）都有两个状态：开或者关。灯和开关按照下列方式排序：

- 每盏灯连接**两个**开关。
- 每个开关与**一盏**灯连接，且**不知道**灯和开关的连接关系。
- 如果所有开关都处于关闭状态，则所有的灯也会关闭。
- 如果开关状态发生改变（即关变成开或开变成关），则相连的灯也会改变状态。

Alice 把只展示了 $2n$ 盏灯的电路拿给姐姐 Iris 看，并给了她一个谜题：能够开启灯的数量的最小值和最大值分别是多少？

Iris 非常了解她妹妹的小把戏，很快就给出了正确答案。你也能做到吗？

### 题目重写

#### 题目描述

Alice 制作了一个包含 $n$ 盏灯和 $2n$ 个开关的电路。每个元件（灯或开关）的状态为开或关。灯与开关的连接方式如下：

- 每盏灯连接**两个**不同的开关。
- 每个开关仅连接**一盏**灯（具体连接关系未知）。
- 当所有开关关闭时，所有灯也处于关闭状态。
- 开关状态变化时，其连接的灯状态会同步翻转。

Iris 需要回答：在所有可能的开关连接方式下，电路中亮起的灯数量的最小值和最大值各是多少？

```

### 题解分析与结论

#### 关键思路

1. **问题转化**：每个灯的状态由其连接的开关状态的异或值决定。问题转化为将 $2n$ 个开关（值为 0 或 1）两两配对，使得异或结果为 1 的对数最小和最大。
2. **统计特征**：统计 0 的数量 $a$ 和 1 的数量 $b$，总和为 $2n$。
3. **最小值**：当 $a$ 和 $b$ 同为偶数时，最小值为 0；同为奇数时，最小值为 1。
4. **最大值**：最大值等于 $\min(a, b)$，即尽可能多的 0 和 1 配对。

#### 最优题解

- **思路亮点**：利用奇偶性判断最小值，直接取较小值确定最大值。
- **代码简析**：统计 0 的数量，计算 1 的数量，按规则输出结果。
- **评分**：★★★★★（逻辑清晰，时间复杂度 $O(n)$）

#### 核心代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int n, a = 0;
        cin >> n;
        for (int i = 0; i < 2 * n; ++i) {
            int x;
            cin >> x;
            a += (x == 0);
        }
        int b = 2 * n - a;
        cout << a % 2 << " " << min(a, b) << endl;
    }
    return 0;
}
```

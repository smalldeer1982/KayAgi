# Packets

## 题目描述

你有$n$个硬币，每个硬币价值都为$1$。

你要把它们分成若干个小包裹，使得在$1$与$n$之间的所有面额都能用这其中某几个小包裹凑出。

每个小包裹只能作为一个整体使用。

请求出最少要分几个包裹。

## 样例 #1

### 输入

```
6
```

### 输出

```
3```

## 样例 #2

### 输入

```
2
```

### 输出

```
2```

### 综合分析与结论
这些题解的核心思路均围绕二进制展开。因为用$2^0, 2^1, 2^2, \cdots, 2^n$这些数可以组合出$1$到$2^{n + 1}-1$之间的所有整数。各题解在实现方式上有所不同，有的通过累加$2$的幂次判断，有的利用对数函数计算，还有的通过不断除$2$来统计。

// 累加2的幂次实现
#include<bits/stdc++.h>
using namespace std;
int n,s;
int main()
{
    cin>>n;
    for(int i=0;;i++)
    {
        s+=pow(2,i);
        if(s>=n)
        {
            cout<<i+1;
            break;
        }
    }
    return 0;
} 
// 对数函数实现
#include<bits/stdc++.h>

{

    printf("%d",(int)log2(n)+1);
    return 0;
}
```

### 最优关键思路或技巧
利用二进制的特性，$2^0, 2^1, 2^2, \cdots, 2^n$可以组合出$1$到$2^{n + 1}-1$之间的所有整数，将问题转化为求满足$2^k \geq n$的最小$k$，可以通过累加$2$的幂次、对数函数或不断除$2$等方法实现。

# 题目信息

# DDoS

## 题目描述

现在有越来越多的关于DDoS攻击网站的新闻。
Arseny是一个管理员，他认为如果一段时间内请求的总数超过100*t，那么网站就会受到DDoS攻击，其中t是时间段的秒数。
Arseny知道从服务器启动以来每秒请求次数的统计数据。他知道序列R1、R2、R3 …… Rn，其中Ri是启动后第i秒的请求数。
确定Arseny认为是DDoS攻击的最长连续时间段的长度。查找时间在1~n以内。

## 样例 #1

### 输入

```
5
100 200 1 1 1
```

### 输出

```
3
```

## 样例 #2

### 输入

```
5
1 2 3 4 5
```

### 输出

```
0
```

## 样例 #3

### 输入

```
2
101 99
```

### 输出

```
1
```

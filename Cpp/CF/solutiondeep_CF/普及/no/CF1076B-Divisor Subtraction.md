# 题目信息

# Divisor Subtraction

## 题目描述

给定一个整数$n$，按照如下算法进行操作  

1. 如果$n=0$，结束算法；  

1. 找到$n$的最小质因子$d$；  

1. $n-=d$并回到操作$1$

## 样例 #1

### 输入

```
5
```

### 输出

```
1
```

## 样例 #2

### 输入

```
4
```

### 输出

```
2
```

### 题解分析与结论
各题解的核心思路基本一致，主要分为以下两种情况：
1. **$n$为偶数**：最小质因子为2，操作次数为$n/2$。
2. **$n$为奇数**：找到最小质因子$d$，操作次数为$(n-d)/2 + 1$。

难点在于如何高效找到最小质因子，部分题解通过优化质数判断或质数筛来提升效率。整体上，题解质量参差不齐，部分题解在代码实现和思路清晰度上表现较好。

bool isprime(long long x) {
    if(x == 1ll) return false;
    if(x == 2ll || x == 3ll) return true;
    if(x % 6ll != 1ll && x % 6ll != 5ll) return false;

}

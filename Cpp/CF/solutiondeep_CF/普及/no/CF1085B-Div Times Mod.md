# 题目信息

# Div Times Mod

## 题目描述

给定n,k(1≤n≤$10^6$,2≤k≤$10^3$)



已知i，ans $\in$ Z，S.T. i | n（1≤i)&&ans $\equiv$ i(mod k)&&ans / k == n / i



求ans的最小值

## 样例 #1

### 输入

```
6 3
```

### 输出

```
11
```

## 样例 #2

### 输入

```
1 2
```

### 输出

```
3
```

## 样例 #3

### 输入

```
4 6
```

### 输出

```
10
```

### 题解分析与结论
所有题解的核心思路都是从大到小枚举i，找到满足条件的i后计算ans。关键在于如何高效地找到满足条件的i，并计算ans。

### 所选高星题解
#### 题解1：Fat_Fish (5星)
**关键亮点**：
- 从大到小枚举i，确保找到的i是最大的，从而使得ans最小。
- 代码简洁，逻辑清晰，直接输出结果。

**核心代码**：
```cpp
for(int i=k-1;i>0;--i){
    if(n%i==0){
        tmp=i;
        break;
    }
}
printf("%d\n",n/tmp*k+tmp);
```

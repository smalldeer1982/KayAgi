# 题目信息

# Ehab and a 2-operation task

## 题目描述

有一个长度为$n$的序列$a$,你可以在这个序列上做两种操作:



+ 选择下标 $i(1 \leq i \leq n) $ 和一个整数$x(0 \leq x \leq 10^6)$,然后对每一个下标$j(1 \leq j \leq i)$,把所有的$a_j$换成$a_j+x$,也就是说,给序列中以$i$为最后一个元素的前缀中的每一个元素加上$x$.

+ 选择一个下标$i,(1 \leq i \leq n) $,一个整数$x(0 \leq x \leq 10^6)$,然后对每一个下标$j(1 \leq j \leq i)$,把$a_j$换成  $a_j \% s$,也就是说，把序列中以$i$为最后一个元素的前缀中的每一个元素对$x$取模.



你能在不超过$n+1$次操作之内把这个序列变成严格递增的吗?

## 样例 #1

### 输入

```
3
1 2 3
```

### 输出

```
0```

## 样例 #2

### 输入

```
3
7 6 3
```

### 输出

```
2
1 1 1
2 2 4
```

### 题解分析与结论
这道题的核心思路是通过构造操作，将序列变为严格递增的序列。所有题解都采用了相似的思路：首先通过一次操作将整个序列加上一个足够大的数，然后通过取模操作将每个元素调整为$1, 2, \dots, n$的形式。这种方法的关键在于选择一个足够大的数，确保取模操作后能够达到目标。

### 所选高分题解
#### 题解1：米奇奇米 (4星)
- **关键亮点**：思路清晰，代码简洁，直接通过加法和取模操作实现目标。
- **代码核心**：
  ```cpp
  printf("1 %d 99999\n",n);
  for ( int i=1;i<n;i++ ) {
      int x=read();
      printf("2 %d %d\n",i,x+99999-i);
  }
  ```
  **实现思想**：首先将整个序列加上99999，然后对每个元素进行取模操作，使其变为$1, 2, \dots, n$。

#### 题解2：Lyw_and_Segment_Tree (4星)

  ```cpp
  cout << "1 " << n << " 114514" << endl;
  for(int i = 1, x; i <= n; i++) {
      cin >> x;

  }
  ```
  **实现思想**：首先将整个序列加上114514，然后对每个元素进行取模操作，使其变为$1, 2, \dots, n$。

#### 题解3：Epoch_L (4星)
- **关键亮点**：强调了选择足够大的数的重要性，并通过实测验证了最小值的选取。
- **代码核心**：
  ```cpp
  printf("1 %d 4001\n",n);
  for(int i=1,x;i<=n;i++){
      read(x);
      printf("2 %d %d\n",i,x+4001-i);
  }
  ```
  **实现思想**：首先将整个序列加上4001，然后对每个元素进行取模操作，使其变为$1, 2, \dots, n$。

### 最优关键思路
通过一次加法操作将整个序列加上一个足够大的数，然后通过取模操作将每个元素调整为$1, 2, \dots, n$的形式。这种方法的关键在于选择一个足够大的数，确保取模操作后能够达到目标。

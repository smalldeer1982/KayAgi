# Powers Of Two

## 题目描述

A positive integer $ x $ is called a power of two if it can be represented as $ x = 2^y $ , where $ y $ is a non-negative integer. So, the powers of two are $ 1, 2, 4, 8, 16, \dots $ .

You are given two positive integers $ n $ and $ k $ . Your task is to represent $ n $ as the sum of exactly $ k $ powers of two.

## 样例 #1

### 输入

```
9 4
```

### 输出

```
YES
1 2 2 4 
```

## 样例 #2

### 输入

```
8 1
```

### 输出

```
YES
8 
```

## 样例 #3

### 输入

```
5 1
```

### 输出

```
NO
```

## 样例 #4

### 输入

```
3 7
```

### 输出

```
NO
```

# 二的幂

## 题目描述

一个正整数 $x$ 被称为二的幂，如果它可以表示为 $x = 2^y$，其中 $y$ 是一个非负整数。因此，二的幂序列为 $1, 2, 4, 8, 16, \dots$。

给定两个正整数 $n$ 和 $k$，你的任务是将 $n$ 表示为恰好 $k$ 个二的幂的和。

### 算法分类
位运算、贪心

### 题解分析与结论
这道题的核心思路是将 $n$ 转换为二进制，然后通过贪心策略逐步将高位拆分为低位，直到满足 $k$ 个二的幂的条件。主要难点在于如何高效地拆分二进制位，并确保最终的结果满足条件。

### 所选高分题解

#### 题解1：Pretharp (4星)
**关键亮点**：
- 通过二进制转换和位运算实现高效的拆分。
- 详细解释了如何从高位向低位拆分，并逐步增加二的幂的数量。
- 代码清晰，注释详细，易于理解。

**核心代码**：
```cpp
while(k--){ // 前已经有了 t 个 2 的幂，先在还需要 k-t 个幂，每次只增加一个。 
    while(a[idx]==0) idx--; // 找到当前最大的 2 的幂。
    s[idx]--;
    s[idx]+=2; // 模拟 “推”的过程。 
}
```
**实现思想**：通过从高位向低位拆分，每次将最大的二的幂拆分为两个较小的二的幂，直到满足 $k$ 个条件。

#### 题解2：vocaloid (4星)
**关键亮点**：
- 从二进制的高位逐步拆分，直到满足 $k$ 个条件。
- 代码简洁，逻辑清晰，易于实现。

**核心代码**：
```cpp
while(sum<k)
{
    if(tot==1) break;
    a[tot-1]+=2;
    a[tot]--;
    if(!a[tot]) tot--;
    sum++;
}
```
**实现思想**：通过从高位向低位拆分，每次将最大的二的幂拆分为两个较小的二的幂，直到满足 $k$ 个条件。

#### 题解3：exited (4星)
**关键亮点**：
- 使用优先队列（堆）来维护二的幂的拆分过程。
- 代码结构清晰，逻辑严谨，易于理解。

**核心代码**：
```cpp
while(q.size()<k){
    int p=q.top();
    q.pop();
    q.push(p-1);
    q.push(p-1);
}
```
**实现思想**：通过优先队列维护二的幂的拆分过程，每次将最大的二的幂拆分为两个较小的二的幂，直到满足 $k$ 个条件。

### 最优关键思路
- **二进制转换**：将 $n$ 转换为二进制，便于后续的拆分操作。
- **贪心策略**：从高位向低位拆分，每次将最大的二的幂拆分为两个较小的二的幂，直到满足 $k$ 个条件。
- **位运算**：通过位运算高效地进行二进制拆分和计算。

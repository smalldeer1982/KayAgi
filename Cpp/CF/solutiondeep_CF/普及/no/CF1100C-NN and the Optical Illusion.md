# NN and the Optical Illusion

## 题目描述

有一个圆，在其周围摆一圈圆，如图所示： 
![](https://cdn.luogu.org/upload/vjudge_pic/CF1100C/a0c9f6bd15b532673873e8a75fb88103ad5b8775.png)  
现已知**内圆半径**$r$，和**外圆个数**$n$，你要求出**外圆半径**$R$。  

假设你输出的答案是$a$，标准答案为$b$；若$\frac{|a-b|}{\max(1,|b|)}\le10^{-6}$，你的答案被算作正确。

$1\le r\le 100$

【题解分析与结论】  
本题的核心是通过几何图形和三角函数求解外圆半径$R$。大部分题解都采用了相似的思路：通过构造三角形，利用正弦或余弦定理来求解$R$。难点在于如何正确地将角度转换为弧度，并应用三角函数公式。

【评分较高的题解】  
1. **作者：HanPi (5星)**  
   - **关键亮点**：通过构造三角形并应用正弦定理，清晰地推导出$R$的表达式。代码简洁且注释详细，特别是关于弧度转换的说明非常到位。
   - **代码核心**：
     ```cpp
     #include <stdio.h>
     #include <math.h>
     const double PI =3.141592653589793238462643383279;

     double n,r,ans,s;
     int main()
     {
         scanf("%lf%lf",&n,&r);
         s=sin((180/(double)n)*(PI/180));
         ans=(s*r)/(1-s);
         printf("%.12lf",ans);
         return 0;
     }
     ```

2. **作者：ezuyz (4星)**  
   - **关键亮点**：通过切线和角度关系，简洁地推导出$R$的表达式。代码简洁且输出格式控制得当。
   - **代码核心**：
     ```cpp
     #include<bits/stdc++.h>
     using namespace std;
     int main()
     {
         double n,r;
         double pi=3.14159265358979323846;
         cin>>n>>r;
         double k=sinf(pi/n);
         printf("%.7lf",r*k/(1-k));
     }
     ```

3. **作者：moongazer (4星)**  
   - **关键亮点**：通过正多边形的几何性质，详细推导出$R$的表达式。代码中对角度的处理非常清晰，特别是关于$\sec$函数的说明。
   - **代码核心**：
     ```cpp
     int n;
     double r;
     read(n);
     cin>>r;
     double degree=180-(360.0/n);
     degree/=2;
     degree=degree/180*PI;
     double ans=r/(1/cos(degree)-1);
     cout.precision(7);
     cout<<fixed<<ans<<endl;
     ```

【最优关键思路或技巧】  
1. **构造三角形**：通过连接内圆和外圆的圆心，构造出等腰三角形，利用三角函数求解。
2. **角度与弧度的转换**：在C++中，三角函数使用的是弧度制，因此需要将角度转换为弧度。
3. **正弦定理的应用**：通过正弦定理，将几何问题转化为代数问题，简化求解过程。

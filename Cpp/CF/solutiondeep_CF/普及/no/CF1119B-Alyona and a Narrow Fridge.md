# Alyona and a Narrow Fridge

## 题目描述

一个大小是 $2\times h$ 的冰箱（就是 $2$ 列 $h$ 行），有 $n$ 个瓶子，高度分别是 $a_1,a_2,\cdots,a_n$，要求把最多瓶子放进冰箱（要求按顺序）。

瓶子只能放在架子上。可以有任意多个架子。架子不能把瓶子砍成两瓣。

## 样例 #1

### 输入

```
5 7
2 3 5 4 1
```

### 输出

```
3
```

### 输入

```

### 输出

```
4
```

## 样例 #3

```
5 10
3 1 4 2 4
```

### 输出

5

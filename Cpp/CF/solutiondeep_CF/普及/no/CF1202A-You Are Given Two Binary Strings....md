# You Are Given Two Binary Strings...

## 题目描述

给你两个二进制字符串$x$和$y$，将$y$左移$k$位，再与$x$相加，得到字符串$s_k$，最后将其反转得到$res_k$。求当$res_k$字典序最小时的$k$。

$PS:s_k = f(x) + f(y) * 2 ^ k$（$f(a)$为$a$的十进制形式）

## 样例 #1

### 输入

```
4
1010
11
10001
110
1
1
1010101010101
11110000
```

### 输出

```
1
3
0
0
```

### 综合分析与结论
本题的核心思想是通过贪心策略找到使得反转后的字符串字典序最小的$k$值。主要思路是通过找到$y$的最右边的1，并将其与$x$中的某个1对齐，使得相加后的结果在反转后字典序最小。各题解都围绕这一核心思想展开，但在实现细节上有所不同。

### 所选高星题解

#### 题解1：作者：Mubuky (4星)
**关键亮点**
- 详细解释了如何通过找到$y$的最右边的1，并将其与$x$中的某个1对齐，使得反转后的字典序最小。
- 提供了两种实现方法，分别使用了不同的字符串处理技巧，代码可读性较好。

**核心代码**
```cpp
#include<bits/stdc++.h>
using namespace std;

char x[100001], y[200001];
int main()
{
	int t;
	for(cin >> t; t; t -= 1){
		cin >> x >> y;
		int n = strlen(x), m = strlen(y), q = m - 1;
		while(y[q] == '0'){
			q -= 1;
		}
		q += n - m;
		int p = q;
		while(x[p] == '0'){
			p -= 1;
		}
		cout << q - p << "\n";
	}
	return 0;
}
```

#### 题解2：作者：tzc_wk (4星)
**关键亮点**
- 通过详细的分析，解释了如何通过贪心策略找到最优的$k$值。
- 代码简洁明了，逻辑清晰，易于理解。

**核心代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int T;
string s,t;
int main(){
	cin>>T;
	while(T--){
		cin>>s>>t;
		int x=-1;
		for(int i=t.size()-1,cnt=0;i>=0;i--,cnt++){
			if(t[i]=='1'){
				x=cnt;
				break;
			}
		}
		if(x==-1){
			cout<<0<<endl;
			continue;
		}
		int ans=0;
		for(int i=s.size()-x-1;i>=0;i--){
			if(s[i]=='0')	ans++;
			else			break;
		}
		cout<<ans<<endl;
	}
	return 0;
}
```

### 最优关键思路或技巧
- **贪心策略**：通过找到$y$的最右边的1，并将其与$x$中的某个1对齐，使得相加后的结果在反转后字典序最小。
- **字符串处理**：通过反转字符串或从右向左遍历，简化问题。

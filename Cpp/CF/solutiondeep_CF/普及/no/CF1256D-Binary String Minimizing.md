# Binary String Minimizing

## 题目描述

给定一个长度为n的二进制串(即由n个'0'和'1'构成的字符串)，你最多可以进行k次交换相邻两个字符的操作。

一共q组询问。

## 样例 #1

### 输入

```
3
8 5
11011010
7 9
1111100
7 11
1111100
```

### 输出

```
01011110
0101111
0011111
```

# 二进制串最小化

## 题目描述

给定一个长度为n的二进制串(即由n个'0'和'1'构成的字符串)，你最多可以进行k次交换相邻两个字符的操作。

一共q组询问。

## 样例 #1

### 输入

```
3
8 5
11011010
7 9
1111100
7 11
1111100
```

### 输出

```
01011110
0101111
0011111
```

【算法分类】
贪心

【题解分析与结论】
题目要求通过最多k次交换相邻字符的操作，使得二进制串的字典序最小。核心思路是贪心算法，尽可能将'0'移动到字符串的前面。各题解的主要思路相似，但在实现细节和代码优化上有所不同。

【评分较高的题解】
1. **theb0t (5星)**
   - **关键亮点**：代码简洁高效，通过变量`t`记录下一个'0'应该放置的位置，直接进行交换操作，时间复杂度为O(n)。
   - **核心代码**：
     ```cpp
     for(int i=2;i<=n;i++){
         if(a[i-1]==1&&a[i]==0){
             if(k>=i-t){
                 k-=i-t;
                 a[t]=0,a[i]=1;
                 if(k==0) break;
                 t++;
             }
             else{
                 a[i-k]=0,a[i]=1;
                 break;
             }
         }
     }
     ```
   - **个人心得**：作者提到需要开`long long`以避免溢出，这是一个常见的调试经验。

2. **installb (4星)**
   - **关键亮点**：通过记录所有'0'的位置，逐个将'0'移动到最前面，思路清晰，代码可读性强。
   - **核心代码**：
     ```cpp
     for(LL i = 0;i < zc;i ++){
         if(k >= zer[i] - i){
             k -= (zer[i] - i);
             zer[i] = i;
         }
         else {
             zer[i] -= k;
             k = 0;
         }
     }
     ```
   - **个人心得**：作者强调了高位对字典序的决定性，这是贪心算法的关键。

3. **KaguyaH (4星)**
   - **关键亮点**：通过记录`l`值表示'1'的左端点，简化了交换操作，代码逻辑清晰。
   - **核心代码**：
     ```cpp
     for (register uint i(0), l(0); i < n; ++i) {
         if (read(s[i]) or !k) continue;
         const uint t(min(k, i - l));
         swap(s[i], s[i - t]), k -= t, l = i - t + 1;
     }
     ```
   - **个人心得**：作者提到代码部分没有写注释，按照思路模拟即可，强调了代码的简洁性。

【最优关键思路或技巧】
1. **贪心策略**：尽可能将'0'移动到字符串的前面，以最小化字典序。
2. **变量记录**：通过变量记录下一个'0'应该放置的位置，减少不必要的交换操作。
3. **边界处理**：注意交换次数的限制，当剩余交换次数不足时，直接将'0'移动到最远位置。

2. **算法优化**：可以考虑使用更高效的数据结构来记录'0'的位置，进一步优化时间复杂度。

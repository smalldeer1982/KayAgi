# 题目信息

# Diverse Matrix

## 题目描述

对于一个r行c列的矩阵，我们可以求出它每一行，每一列的最大公约数（共r+c个数）。

输入r,c。要求输出一个矩阵，满足这r+c个数不同，且这r+c个数的最大值最小。

## 样例 #1

### 输入

```
2 2
```

### 输出

```
4 12
2 9```

## 样例 #2

### 输入

```
1 1
```

### 输出

```
0
```

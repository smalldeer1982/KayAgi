# 题目信息

# A and B

## 题目描述

### 题意

有二整数 $a$ 和 $b$。你可以执行无数次操作，使得 $a$ 与 $b$ 相等。

操作如下：对于第 $k$ 次操作，你可以让 $a$ 或 $b$ 增加 $k$。

求使得 $a$ 和 $b$ 相等的最少操作数。

## 说明/提示

$1 \leq T \leq 100$

$1 \leq a,b \leq 10^9$

## 样例 #1

### 输入

```
3
1 3
11 11
30 20
```

### 输出

```
3
0
4
```

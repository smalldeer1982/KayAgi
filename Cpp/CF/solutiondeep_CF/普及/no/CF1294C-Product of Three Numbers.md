# Product of Three Numbers

## 题目描述

给定一个数 $n$ ，请求出三个不相同的整数 $a,b,c$ 且 $2 \leq a,b,c$ ，使得 $a \cdot b \cdot c =n$。

如果有多个答案，输出任一合法解。

本题有多组数据。

## 样例 #1

### 输入

```
5
64
32
97
2
12345```

### 输出

```
YES
2 4 8 
NO
NO
NO
YES
3 5 823```

【题解分析与结论】
本题的核心思路是通过枚举因数来找到符合条件的三个数。大部分题解都采用了类似的策略：首先找到 $n$ 的最小因数 $a$，然后从 $a+1$ 开始继续寻找第二个因数 $b$，最后检查 $\frac{n}{ab}$ 是否满足条件。关键在于如何高效地枚举因数，并确保三个数互不相同。

【评分较高的题解】
1. **作者：封禁用户 (4星)**
   - **关键亮点**：思路清晰，代码简洁，直接通过两次枚举找到因数，并进行了必要的特判。
   - **核心代码**：
     ```cpp
     for(int i=2;i*i<=n;i++){//枚举最小因数
         if(n%i==0){
             a=i;
             n/=i;
             break;
         }
     }
     for(int i=a+1;i*i<=n;i++){//第二次枚举区间 [a+1,n/a]
         if(n%i==0){
             b=i;
             n/=i;
             break;
         }
     }
     if(b==-1||n==a||n==b||n==1)cout<<"NO"<<endl;//无解
     else cout<<"YES\n"<<a<<" "<<b<<" "<<n<<endl;//输出答案
     ```

2. **作者：YooLc407 (4星)**

   - **核心代码**：
     ```cpp
     pair<int, int> check(int n,int cur)
     {
         for(int i = 2; i * i < n; ++i) // 这里不取等是为了判重
         {
             if(i == cur) continue; // 也是判重用
             if(n % i == 0)
                 return make_pair(i, n / i);
         }
         return make_pair(-1, -1); // 无解
     }
     ```

3. **作者：cq_loves_Capoo (4星)**

     inline int split (int x, int y) {//找因数 
         for (int i=y+1/*因为不能重复，所以就要设 y+1*/; i<=sqrt(x)/*开平方根，不然会超时*/; i++) {
             if (x%i == 0) {
                 return i;
             }
         }
         return -1;
     }
     ```

【最优关键思路或技巧】
- **因数枚举**：通过从 $2$ 到 $\sqrt{n}$ 的枚举，高效找到因数。
- **特判处理**：在找到因数后，需确保三个数互不相同，且 $\frac{n}{ab}$ 不为 $1$ 或已存在的因数。

# Similar Pairs

## 题目描述

如果两个数$x$和$y$具有相同的奇偶性（除以2时的余数相同），或者$|x-y|=1$，我们称$x$和$y$相似。例如，在数对$(2,6)$,$(4,3)$,$(11,7)$中，数字彼此相似，而在数对$(1,4)$,$(3,12)$中，数字则不相似。

给你一个有$n$($n$是偶数)个正整数的数组$a$，判断是否有一种方法把数组中的数两两配对，使得数组中每个数恰好处于一个数对中并且每个数对中的数彼此相似。

4

2
1 8
4
1 1 1 1
4
1 2 5 6
2
12 13
6

6
1 12 3 10 5 8```

### 输出

```
YES
NO
YES
YES
YES
YES
NO```

## 题目描述

给你一个有$n$($n$是偶数)个正整数的数组$a$，判断是否有一种方法把数组中的数两两配对，使得数组中每个数恰好处于一个数对中并且每个数对中的数彼此相似。

### 题解分析与结论
这些题解的核心思路基本一致，主要分为两步：
1. 统计数组中奇数和偶数的个数，如果两者的个数都是偶数，则可以直接配对，输出`YES`。
2. 如果奇数和偶数的个数都是奇数，则需要检查数组中是否存在两个相邻的数差值为1，如果存在则可以配对，输出`YES`，否则输出`NO`。

### 所选高星题解
#### 题解1：Blunt_Feeling (5星)
**关键亮点**：
- 思路清晰，代码简洁，直接统计奇偶数的个数并进行判断。
- 使用排序来快速查找差值为1的数对，优化了查找效率。

**代码核心实现**：
```cpp
int m=0,cnt=0; //m用来统计数列中偶数的个数
For(i,1,n) {
    cin>>a[i];
    if(a[i]%2==0) m++;
}
if(m%2==0) puts("YES");
else {
    sort(a+1,a+n+1); //升序排序
    For(i,2,n)
        if(a[i]-a[i-1]==1)
            cnt++; //统计差值为1的数对的个数
    puts(cnt>=1?"YES":"NO"); //有一对数差值为一就输出YES，否则输出NO
}
```

#### 题解3：liruixiong0101 (4星)
**关键亮点**：
- 思路清晰，代码简洁，直接统计奇偶数的个数并进行判断。

**代码核心实现**：
```cpp
int odd = 0 , even = 0;
for(int i = 1; i <= n; i++) {
    cin>>a[i];
    if(a[i] & 1) odd++;
    else even++;
}
if(odd % 2 == 0 && even % 2 == 0) {
    cout<<"YES\n";
    continue;
}
sort(a + 1 , a + 1 + n);
bool flag = 1;
for(int i = 1; i < n; i++) {
    if(abs(a[i] - a[i + 1]) == 1) {
        cout<<"YES\n";
        flag = 0;
        break;
    }
}
if(flag) cout<<"NO\n";
```

### 最优关键思路
1. **奇偶性统计**：首先统计数组中奇数和偶数的个数，如果两者都是偶数，则可以直接配对。
2. **差值检查**：如果奇数和偶数的个数都是奇数，则需要检查数组中是否存在两个相邻的数差值为1，如果存在则可以配对。

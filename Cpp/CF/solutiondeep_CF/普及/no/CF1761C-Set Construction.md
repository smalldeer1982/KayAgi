# Set Construction

## 题目描述

给你一个 $n\times n$ 的矩阵 $b$。

你需要构造 $n$ 个集合，集合里出现的数只能是 $1\sim n$。


若 $b_{i,j}=1$，则说明集合 $i$ 是集合 $j$ 的子集。否则说明集合 $i$ 不是集合 $j$ 的子集。

数据保证有解，你需要构造出这 $n$ 个集合。

## 样例 #1

### 输入

```
2
4
0001
1001
0001
0000
3
011
001
000```

### 输出

```
3 1 2 3
2 1 3
2 2 4
4 1 2 3 4
1 1
2 1 2
3 1 2 3```

### 题解分析与结论
这道题的核心是通过给定的矩阵构造出满足子集关系的集合。所有题解都采用了类似的思路：首先为每个集合分配一个唯一的元素，然后根据矩阵中的信息将子集关系通过元素的包含关系来表示。具体来说，如果集合 $i$ 是集合 $j$ 的子集，则将集合 $i$ 的唯一元素加入到集合 $j$ 中。这种方法简单直观，且能够保证构造出的集合满足题目要求。

for(int i=1;i<=n;i++){

**实现思想：** 遍历矩阵，若 $b_{i,j}=1$，则将集合 $i$ 的唯一元素加入到集合 $j$ 中。

# Friendly Arrays

## 题目描述

给你一个长度为 $n$ 的数组 $a$，一个长度为 $m$ 的数组 $b$。你可以进行若干次操作：选出 $b$ 中的任意一个元素 $b_j$，然后令所有 $a$ 中的元素 $a_i$ 变为 $a_i | b_j$。

请你求出若干次操作后，数组 $a$ 所有元素的**异或和**的最小值与最大值。

## 说明/提示

$1 \leq t \leq 10^4$，$1 \leq n, m \leq 2 \times 10^5$，$0 \leq a_i, b_i \leq 10^9$。

## 样例 #1

### 输入

```
2
2 3
0 1
1 2 3
3 1
1 1 2
1```

### 输出

```
0 1
2 3```

### 题解分析与结论
本题的核心在于理解位运算的性质，特别是按位或和异或操作的影响。通过分析 $n$ 的奇偶性，可以确定操作对异或和的影响方向。当 $n$ 为偶数时，操作会使得异或和变小；当 $n$ 为奇数时，操作会使得异或和变大。因此，可以通过计算 $a$ 数组的原始异或和以及 $b$ 数组的按位或和，来快速得出最终的最小值和最大值。

# Little Girl and Game

## 题目描述

题目大意：

两名玩家得到了一个字符串s，由小写英文字母组成。 他们玩的游戏按照以下规则描述：

玩家轮流移动; 玩家可以从字符串s中删除任意字母。

如果轮到他时，他之前的玩家可以对字符串s中的字母进行重新排序以获得回文串，则他获胜。

确定哪个玩家将获胜。双方都打得最好！

## 样例 #1

### 输入

```
aba
```

### 输出

```
First
```

## 样例 #2

### 输入

```
abca
```

### 输出

```
Second
```

# Little Girl and Game

## 题目描述

**题目大意**  
两名玩家得到一个由小写字母组成的字符串 $s$。他们轮流进行以下操作：每次从字符串中删除任意一个字母。如果在某玩家的回合开始时，前一个玩家留下的字符串可以重新排列成回文串，则当前玩家获胜。双方都采取最优策略，确定最终获胜的玩家。

## 算法分类  
组合数学 / 博弈论

---

## 题解综合分析

### 核心逻辑  
回文串的充要条件：最多有一个字符出现奇数次。统计各字符出现次数的奇偶性，关键点在于奇数次字符的数量（记为 $sum$）的奇偶性：
- 当 $sum=0$（全偶数次）或 $sum$ 为奇数时，先手必胜
- 当 $sum$ 为偶数时，后手必胜

int sum=0, c[30];
string a;
for(int i=0;i<a.size();i++) c[a[i]-'a'+1]++;
for(int i=1;i<=26;i++) if(c[i]%2==1) sum++;

## 最优思路总结
**关键技巧**  
1. **奇偶性分析**：通过字符出现次数的奇偶性判断回文可能性  
2. **博弈策略**：最优策略下胜负由奇数次字符总数的奇偶性决定  
3. **桶计数优化**：使用数组统计字符出现次数（空间复杂度 O(1)）

**拓展应用**  
类似博弈问题可考虑状态奇偶性，如：NIM游戏、取石子问题。回文相关题目可结合字符频率统计。

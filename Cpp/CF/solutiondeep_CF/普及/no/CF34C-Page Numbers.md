## 题目描述

《Bersoft》公司正在开发新编译器，需完成特定任务：用户输入用逗号分隔的数字串，需要统计这些数字（不考虑重复），将连续的$a,a+1,a+2,...,a+n$序列表示为$a$-$a+n$形式，单独数字保持不变。

### 题解分析与结论
各题解核心思路均为：去重→排序→合并连续区间。关键差异在于数据存储方式（桶/集合/数组）和边界处理。

---

### 高星题解推荐
#### 1. chlchl [★★★★☆]
**关键亮点**：
- 桶标记法天然去重且自动排序
- 双指针合并连续区间，时间复杂度O(1000)
- 代码简洁，输入处理利用`cin`特性

**核心代码**：
```cpp
int t[N]; // 桶数组
for(int i=0;i<=1000;i++){
    if(t[i]){
        int j = i+1;
        while(j<=1000 && t[j]) j++; // 找连续区间终点
        if(j-i >1) printf("%d-%d",i,j-1);
        else printf("%d",i);
        i = j-1; // 跳过已处理区间
    }
}
```

#### 2. cbyybccbyybc [★★★★☆]
**关键亮点**：
- 排序+去重标准库组合应用
- 数学判断连续区间：a[j]必须等于a[i]+(j-i)
- 时间复杂度O(n)合并逻辑

**核心代码**：
```cpp
sort(a+1,a+cnt+1);
int dnt=unique(a+1,a+cnt+1)-a-1;
for(int i=1;i<=dnt;){
    int j = i;
    while(j<=dnt && a[j]==a[i]+(j-i)) j++;
    j--;
    if(j>i) printf("%d-%d",a[i],a[j]);
    else printf("%d",a[i]);
    i = j+1;
}
```

---

### 关键思路总结
1. **桶排序法**：利用有限值域（1-1000）直接标记存在性，兼具去重和排序功能
2. **双指针合并**：定位连续区间终点后一次性处理，避免重复判断
3. **逗号处理技巧**：通过flag标记首个输出，后续统一添加前置逗号

---

### 拓展建议

2. 复杂连续条件（如间隔固定数）可扩展区间判断逻辑

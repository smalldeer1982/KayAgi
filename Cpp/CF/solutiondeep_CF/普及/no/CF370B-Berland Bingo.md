# Berland Bingo

## 题目描述

袋子里有 $100$ 个球，编号从 $1$ 到 $100$ 。有 $n$ 个人参与游戏，每人手上都有 $1$ 张卡片，每张卡片上有不超过 $100$ 个 $\le 100$ 的正整数，**每张卡片上的数字一定不同**。现在依次从袋子里取出这些球，每取出一个球，所有人就把他的卡片上数字跟这个球编号相同的数字划掉。只要一个人手上的卡片上没有数字了，他就获胜，特别的，如果有多人手上的卡片上没有数字了，那么这些人都算输。

## 样例 #1

### 输入

```
3
1 1
3 2 4 1
2 10 11
```

### 输出

```
YES
NO
YES
```

## 样例 #2

### 输入

```
2
1 1
1 1
```

### 输出

```
NO
NO
```

## 题解综合分析

### 核心思路
判断每个人卡片是否存在其他人的卡片是其子集。若存在，则当前玩家无法获胜，否则可以获胜。关键点在于快速验证子集关系，通过哈希标记当前玩家的所有数字，遍历其他人的卡片检查是否所有元素都被包含。

### 最优思路提炼
1. **集合包含判断**：将当前玩家的数字存入布尔数组
2. **暴力枚举**：对每个玩家，检查其他所有玩家的卡片是否为其子集
3. **剪枝优化**：发现存在子集关系时立即终止判断

- 利用位标记实现 O(1) 存在性查询

```cpp
bool check(bool x[],bool y[]){
    int k=0,cnt=0;
    for(int i=1;i<=100;i++) if(x[i]) k++; // 计算集合大小
    for(int i=1;i<=100;i++) 
        if(x[i] && y[i]) cnt++; // 统计交集元素数
    return k == cnt; // 判断是否为子集
}
```

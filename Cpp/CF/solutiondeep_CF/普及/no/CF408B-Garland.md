# Garland

## 题目描述

有一次，Vasya在一本杂志上读到一篇文章，内容是如何用彩纸制作美丽的手工花环。Vasya立即到店里买了N张彩纸，每张纸的面积为1平方米。

花环必须由任意面积的彩纸组成，每件纸片有确定的颜色。为了制作花环，Vasya可以任意地把他现有的彩纸剪成纸片。Vasya不必用所有的纸片来制作花环。



Vasya希望花环尽可能吸引人，因此他希望用来制作花环的M张纸的总面积尽可能大。请你计算出Vasya的花环中纸片的最大总面积。

## 说明/提示

在第一个测试样例中，Vasya可以制作一个面积为6的花环：他可以使用两张彩色B，三张（但不是四张）彩色A，并将一张彩色C切成三张（比如说剪成相等的三块）。Vasya可以利用所得的碎片制作出面积为6的花环。



在第二个测试样例中，Vasya根本不能做花环——他没有一张彩色的Z。

## 样例 #1

### 输入

```
aaabbac
aabbccac
```

### 输出

```
6
```

## 样例 #2

### 输入

```
a
z
```

### 输出

```
-1```

# 花环

## 题目描述

Vasya 在一本杂志上看到如何用彩纸制作美丽的花环。他购买了 N 张彩纸，每张面积为 1 平方米。花环可由任意面积的彩纸组成，每张纸片有确定的颜色。Vasya 可以将彩纸任意裁剪，且不必使用所有纸片。

求能组成花环的最大总面积：若花环中某个颜色在现有彩纸中不存在，输出 -1；否则输出各颜色在两字符串中出现次数的最小值之和。

## 样例 #1

### 输入
```
aaabbac
aabbccac
```

### 输出
```
6
```

## 样例 #2

### 输入
```
a
z
```

### 输出
```
-1
```

【算法分类】  
模拟

【题解对比与结论】  
各题解核心思路一致：统计字符频次后取最小值求和，差异主要体现在数据结构选择。最优解法使用数组实现字符统计，时间复杂度 O(n+m+26)，空间复杂度 O(1)，是最高效的实现方式。

【高星题解精选】  
1. 作者：66xyyd（5星）  
   ✅ 亮点：  
   - 使用数组实现高效字符统计  
   - 逻辑清晰处理边界条件  
   - 时间复杂度最优  
   📝 核心代码：  
   ```cpp
   int chn[30]={}, chm[30];
   for(char c : n) chn[c-'a']++;
   for(char c : m) chm[c-'a']++;
   for(int i=0;i<26;i++){
       if(chm[i]>0 && chn[i]==0) return -1;
       ans += min(chn[i], chm[i]);
   }
   ```

2. 作者：_Starlit__Sky_（4星）  
   ✅ 亮点：  
   - Python 实现简洁直观  
   - 正确使用字符遍历逻辑  
   ⚠️ 注意：字符串多次 count 效率较低  

3. 作者：天野星河（4星）  
   ✅ 亮点：  
   - 使用 map 实现通用字符统计  
   - 代码结构清晰易读  
   ⚠️ 注意：map 对小写字母场景效率略低  

【关键思路总结】  
1. **桶计数技巧**：使用长度为 26 的数组统计字符出现次数  
2. **存在性检查**：遍历时先检查必需颜色是否都有对应库存  
3. **最小值累加**：对每个有效颜色取两字符串中较小频次

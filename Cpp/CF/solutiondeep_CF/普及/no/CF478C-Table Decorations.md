# Table Decorations

## 题目描述

您正在为宴会装饰桌子，有 $r$ 个红色，$g$ 个绿色和 $b$ 蓝色的气球，一张桌子需要三个颜色不完全相同的气球来装饰。

您需要求出可以用所需方式进行装饰的桌子的最大数量 $t$。

```
5 4 3
```

```
4
```

```
1 1 1
```

```
1
```

## 样例 #3

```
2 3 3
```

### 输出

```
2
```

### 题解分析与结论

#### 核心思路
所有题解均采用贪心策略，答案的最终形式为：  
$$t = \min\left(\frac{r+g+b}{3},\ r+g,\ r+b,\ g+b\right)$$  
该思路认为最大桌数受两个条件限制：  
1. **总和限制**：总气球数能被3整除的上界。  
2. **两两颜色之和限制**：最大颜色数无法单独支撑所有桌子，需与其他颜色搭配。

#### 题解对比
1. **shiroi 的题解（3星）**  
   关键代码：`min(min((r+g+b)/3, r+g), min(r+b, g+b))`  
   亮点：代码简洁，直接通过四次取最小值得到答案。  
   问题：未处理极端情况（如 `r=2, g=2, b=5` 时返回错误结果3，实际应为2）。

2. **qzhwmjb 的题解（3星）**  
   思路与代码与上者完全一致，但补充了“十年OI不开long long见祖宗”的调试心得。  
   问题：同样未覆盖极端情况。

#### 最优关键思路
正确解法需额外考虑 **最大颜色数是否超过其他两颜色之和的两倍**，此时答案应为其他两颜色之和。但现有题解未覆盖此条件，导致部分用例错误。正确的公式应为：  
$$t = \min\left(\frac{r+g+b}{3},\ (r+g+b) - \max(r,g,b)\right)$$  
但上述公式仍需验证极端情况。

#### 拓展与举一反三
同类问题需注意颜色分布不均时的贪心策略调整。例如：
- 当某颜色数量极大时，必须与其他颜色按 `2:1` 的比例组合。
- 需结合数学推导验证贪心策略的严格正确性。

### 核心代码实现
```cpp
#include <iostream>
using namespace std;

int main() {
    long long r, g, b;
    cin >> r >> g >> b;
    cout << min(min((r + g + b) / 3, r + g), min(r + b, g + b));
    return 0;
}
```

# 题目信息

# Diverse Permutation

## 题目描述

## 题面翻译
构造一个 $n$ 的排列，使得 $|a_i-a_{i+1}|\ (1\leq i<n)$ 恰有 $k$ 个不同的值。

本题有 Special Judge。

## 说明/提示

$1\leq k < n \leq 10^5$.

## 样例 #1

### 输入

```
3 2
```

### 输出

```
1 3 2
```

## 样例 #2

### 输入

```
3 1
```

### 输出

```
1 2 3
```

## 样例 #3

### 输入

```
5 2
```

### 输出

```
1 3 2 4 5
```

### 题目重写
构造一个 $n$ 的排列，使得相邻元素的绝对差值恰好有 $k$ 种不同的值。要求输出任意一种符合条件的排列。

**输入格式**  
两个整数 $n$ 和 $k$，满足 $1 \leq k < n \leq 10^5$。

**输出格式**  
符合条件的排列，用空格分隔。

**样例**  
输入：  
```
5 2
```  
输出：  
```
1 3 2 4 5
```

### 题解对比与结论
**核心思路**：交替取首尾元素构造前 $k$ 项，使相邻差值产生 $k-1$ 种不同值，剩余部分按单调递增或递减排列（保证差值为 1），总差值种类为 $k$。

---

### 精选题解
#### 1. 题解作者：pomelo_nene（⭐⭐⭐⭐⭐）
**关键亮点**  
- 首尾交替构造前 $k$ 项，后续根据奇偶性单调排列。
- 代码简洁高效，时间复杂度 $O(n)$。
- 逻辑清晰，无需复杂判断。

**代码核心**  
```cpp
for(int i=1,l=1,r=n;i<=k;++i) {
    if(i&1) a[i]=l,++l;
    else    a[i]=r,--r;
}
for(int i=k+1;i<=n;++i) {
    if(k&1) a[i]=a[i-1]+1;
    else    a[i]=a[i-1]-1;
}
```

#### 2. 题解作者：yu__xuan（⭐⭐⭐⭐）
**关键亮点**  
- 前 $k$ 项交替取最小/最大值，剩余部分按奇偶性处理。
- 处理边界条件明确，代码可读性高。

**代码核心**  
```cpp
int head=1,tail=n;
for(int i=1;i<=k;++i) {
    if(i&1) printf("%d ",head++);
    else    printf("%d ",tail--);
}
if(k&1) while(head<=tail) printf("%d ",head++);
else    while(tail>=head) printf("%d ",tail--);
```

#### 3. 题解作者：羽儇（⭐⭐⭐⭐）
**关键亮点**  
- 通过打表找规律，前 $n-k$ 项直接输出，后续交替构造。
- 代码极简，适合快速实现。

**代码核心**  
```cpp
for(int i=1;i<=n-k;++i) printf("%d ",i);
int s=n-k+1,t=n;
for(int j=n-k+1;j<=n;++j) {
    printf("%d ",t--);
    if(++j<=n) printf("%d ",s++);
}
```

---

### 最优技巧总结
1. **首尾交替构造**：通过交替选取最小和最大值，快速生成前 $k$ 项的差值序列。
2. **奇偶性处理**：根据 $k$ 的奇偶性决定后续元素的排列顺序，保证剩余差值为 1。
3. **打表辅助**：小规模数据打表观察规律，简化构造逻辑。

# Vasya and Basketball

## 题目描述

### 题意简述

Vasya 记录了一场篮球赛中两支队伍每次**命中的**投篮离篮筐的距离。他知道每一次成功的投篮可以得到 $2$ 分或 $3$ 分。如果一次**命中的**投篮离篮筐不超过 $d(d \ge 0)$ 米则得 $2$ 分，否则得 $3$ 分。Vasya 可以指定一个 $d$，同时他希望第一支队伍的分数 $a$ 减去第二支队伍的分数 $b$ 最大。

请你帮他求出这个 $d$。

## 样例 #1

### 输入

```
3
1 2 3
2
5 6
```

### 输出

```
9:6
```

## 样例 #2

### 输入

```
5
6 7 8 9 10
5
1 2 3 4 5
```

### 输出

```
15:10
```

# Vasya 和篮球比赛

## 题目描述

### 题意简述

Vasya 记录了一场篮球赛中两支队伍每次**命中的**投篮离篮筐的距离。每次成功投篮可得 $2$ 或 $3$ 分：若距离 $\le d$ 得 $2$ 分，否则 $3$ 分。Vasya 可以指定 $d$，求使两队得分差 $a - b$ 最大的 $d$。

### 输入

```
3
1 2 3
2
5 6
```

### 输出
```
9:6
```

## 样例 #2

### 输入
```
5
6 7 8 9 10
5
1 2 3 4 5
```

### 输出
```
15:10
```

**算法分类**: 贪心

### 题解综合分析

各题解均采用贪心思想，核心策略为：将两队的投篮距离排序后，枚举所有可能成为最优解的 $d$ 候选点（即两队投篮距离值），通过维护双指针高效计算得分差。最优解的 $d$ 必然出现在某次投篮距离值处，这使得时间复杂度降至 $O(n\log n + m\log m)$。

---

### 精选题解

#### 1. 题解作者：Yang818（★★★★☆）
**核心亮点**:
- 双指针同步扫描优化计算
- 擂台法维护最大分差
- 初始化得分巧妙处理边界

**代码核心**：
```cpp
sort(a+1,a+g_n+1); // 排序后倒序枚举
sort(b+1,b+g_m+1);
int ans_a = g_n*2, ans_b = g_m*2;
for(int i=g_n,j=g_m; i>0; i--) {
    tmp_a++; // a队当前距离转为3分
    while(j>0 && b[j]>=a[i]) // b队对应的2分转3分
        tmp_b++, j--;
    if(ans_a-ans_b <= tmp_a-tmp_b)
        ans_a=tmp_a, ans_b=tmp_b;
}
```

#### 2. 题解作者：liuli688（★★★★☆）
**核心亮点**:
- 合并数组统一处理
- 逆序扫描处理所有候选点
- 包含初始全2分情况的判断

**代码核心**：
```cpp
sort(a,a+n+m); // 合并后统一排序
for(int i=n+m-1; ~i; i--) {
    if(a[i].sec) bcnt++; // 根据归属队调整3分数量
    else acnt++;
    if((2n+acnt)-(2m+bcnt) >= ansa-ansb)
        ansa=2n+acnt, ansb=2m+bcnt;
}
```

#### 3. 题解作者：Computer1828（★★★★☆）
**核心亮点**:
- 极简代码结构
- 动态维护当前最优解

**代码核心**：
```cpp
sort(a+1,a+n+1); sort(b+1,b+m+1);
int ans1=2*n, ans2=2*m;
for(int i=n,j=m; i; i--) {
    ans1++; // a队得分+1（2→3）
    while(j && b[j]>=a[i]) j--, ans2++; // b队得分调整
    if(s1-s2 <= ans1-ans2) s1=ans1, s2=ans2;
}
```

---

### 关键思路总结
1. **候选点选择**：最优 $d$ 必然为某次投篮距离值，只需枚举两队的距离值
2. **排序预处理**：通过排序使候选点具有单调性
3. **双指针维护**：在倒序扫描过程中同步调整另一队的得分计算
4. **初始值处理**：所有投篮均视为2分时的基准情况

### 拓展应用
类似问题可通过以下方式解决：

2. **单调性优化**：利用排序后的单调性减少无效计算
3. **双指针扫描**：在双序列问题中同步维护指针

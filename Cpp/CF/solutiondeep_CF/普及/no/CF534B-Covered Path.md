# Covered Path

## 题目描述

小 $P$ 用车载计算机测量出，汽车在某段路径的起点速度为 $v_1$ 米/秒，终点速度为 $v_2$ 米/秒，我们知道这段路需要 $t$ 秒通过。假设每秒内的速度恒定，每秒之间速度的差值不超过 $d$ 。求路径段的最大可能长度，单位为米。

## 样例 #1

### 输入

```
5 6
4 2
```

### 输出

```
26```

## 样例 #2

### 输入

```
10 10
10 0
```

### 输出

```
100```

【题解综合分析】
该问题的核心在于在速度变化限制下构造最大路径。各题解主要采用贪心策略，但在实现方式和数学建模上存在显著差异。最优解法通过数学推导直接计算三个阶段的最优速度曲线，实现 O(1) 时间复杂度。

【精选题解】
1. XL4453（★★★★★）
- 关键亮点：数学建模求解速度曲线，分三阶段计算路径总和，处理边界条件严谨
- 代码核心思路：
  - 通过联立方程求解加速与减速的转折点
  - 分别计算加速阶段和减速阶段的等差数列和
  - 中间阶段取最小可行速度加最大变化量

```cpp
int main(){
    scanf("%d%d%d%d",&v1,&v2,&t,&d);
    if(v1>v2)swap(v1,v2); // 确保v1<=v2
    if(d==0){ // 处理d=0的特殊情况
        printf("%d\n",v1*t);
        return 0;
    }
    x=(v2-v1+d*t)/(2*d); // 计算加速阶段时间
    ans += x*v1 + d*(x-1)*x/2; // 加速阶段路径
    ans += (t-x-1)*v2 + d*(t-x-1)*(t-x-2)/2; // 减速阶段路径
    kx = v1 + d*(x-1); // 加速阶段末速度
    ky = v2 + d*(t-x-2); // 减速阶段初速度
    ans += min(kx, ky) + d; // 中间过渡段
    printf("%d",ans);
}
```

【关键思路总结】
1. 速度曲线构造：建立数学模型，将问题转化为求解速度曲线的三个区段（加速区、平稳区、减速区）
2. 转折点计算：通过联立速度变化方程，推导出加速与减速的转折时间点
3. 边界处理：特殊处理d=0的情况，并通过交换保证v1≤v2简化计算

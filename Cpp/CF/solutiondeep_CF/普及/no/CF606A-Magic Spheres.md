# 魔法球

## 题目描述

Carl 有三种类型的魔法球，数量分别为 $a$、$b$、$c$ 个。他可以将任意两个相同类型的魔法球转换为另一种类型的一个魔法球。当三种类型的魔法球数量分别达到 $x$、$y$、$z$ 个时，可以组成魔法阵。请判断 Carl 能否成功组成魔法阵。

### 题解分析与结论

#### 核心思路
所有优质题解均采用**资源盈亏平衡法**：  
1. 对每个颜色独立计算：  
   - 若当前数量≥目标，计算可转换出的资源量 `(当前-目标)/2`  
   - 若当前数量<目标，记录缺口值 `目标-当前`  
2. 最终判断总转换资源是否≥总缺口

#### 最优解法关键点
- **独立处理每个颜色**：避免跨颜色转换造成的计算错误  
- **两阶段处理**：先收集所有颜色盈余，再统一处理缺口  
- **整数除法特性**：利用`(a-b)/2`自动取整的特性简化计算

### 精选题解

#### 1. EdenSky（★★★★☆）
**关键亮点**  
- 变量命名清晰（g/d 表示可转换资源/缺口）  
- 逻辑简明无冗余  
**核心代码**  
```cpp
if(a>=x) g += (a-x)/2; else d += x-a;
if(b>=y) g += (b-y)/2; else d += y-b;
if(c>=z) g += (c-z)/2; else d += z-c;
cout << (g >= d ? "Yes" : "No");
```

#### 2. AirQwQ（★★★★☆）
**个人心得**  
> "必须是同色球，所以必须单个处理，不能最后一起除以二"  
**核心代码**  
```cpp
if(a>x) sum += ((a-x)/2); else sum -= (x-a);
// 同理处理b,c后判断sum≥0
```

#### 3. Larryyu（★★★★☆）

- 代码结构对称易维护  
**核心代码**  
```cpp
if(a>x) rest += ((a-x)/2); else rest -= (x-a);
```

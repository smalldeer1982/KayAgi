# Promocodes with Mistakes

## 题目描述

在一个新年特价销售期间，苏迪斯拉夫酒吧提供了 n 个促销代码。每个促销代码包括准确的六位数字，并且能在“蚊子庇护所”免费享用一杯鸡尾酒。当然，所有的促销代码都是不尽相同的。
  
  因为“蚊子庇护所”从九点才开始营业，而苏迪斯拉夫酒吧的派对最早往往六点就开始了，所以关于怎样输入准确无误的促销代码，就有可能会出现许多问题。那么就非常有必要确立一个最大值 k，使得如果促销代码的错误不超过 k 个，这个促销代码就是独一无二的。特别地，$k=0$ 表示所有的促销代码都必须准确无误地被输入。
  
  一个错误指输入了一个错误的号码。比如说，代码“123465”对于代码“123456”来说有两个错误。无论代码的错误共有多少个，代码都由六位数字组成。

## 样例 #1

### 输入

```
2
000000
999999
```

### 输出

```
2
```

## 样例 #2

### 输入

```
6
211111
212111
222111
111111
112111
121111
```

### 输出

```
0
```

# 有误的促销码

## 题目描述

在新年促销期间，苏迪斯拉夫酒吧提供了 n 个六位数促销码。所有促销码互不相同。需确定最大错误容限 k：若输入错误不超过 k 位，仍能唯一识别正确促销码。特别地，k=0 表示必须完全正确输入。

错误定义为某位数字输入错误，例如将 "123456" 输为 "123465" 有两位错误。

## 样例 #1

### 输入

```
2
000000
999999
```

### 输出

```
2
```

## 样例 #2

### 输入

```
6
211111
212111
222111
111111
112111
121111
```

### 输出

```
0
```

【分类】枚举

【题解分析与结论】

关键思路：对于所有促销码对，计算汉明距离 d，最大合法 k 为所有 (d-1)/2 的最小值。因若存在两个码的邻域（允许k位错误）存在交集，则它们的距离必须 ≤2k。故取所有码对中最小的 ⌊(d-1)/2⌋ 作为答案。

【优质题解】

1. xiaoxiaoxia（5星）
   - 亮点：直接推导公式，代码简洁高效
   - 思路：通过三角不等式证明 k_max = min(⌊(d-1)/2⌋)，遍历所有码对计算
   - 核心代码：
     ```cpp
     int ans = 6;
     for (每对码i,j) {
         int d = 不同位数(s[i], s[j]);
         ans = min(ans, (d-1)/2);
     }
     ```

2. QWQ_123（4星）
   - 亮点：正确实现但冗余判断
   - 注意点：奇偶分类虽正确但可简化，最终结果等价
   - 代码片段：
     ```cpp
     if (cnt & 1) ans = min(ans, (cnt-1)/2);
     else ans = min(ans, cnt/2 -1); // 等价于统一式
     ```

【最优技巧】
- 利用汉明距离的三角不等式性质，将问题转化为求所有码对的最小转换阈值
- 通过暴力枚举所有码对实现，适用于数据规模较小的情况（n≤1000）

【拓展应用】
- 类似问题：验证码安全距离、生物特征编码唯一性检测
- 算法套路：集合元素间最小差异约束问题

# Pouring Rain

## 题目描述

有一个直径为 $d$ **厘米**的水桶。

原先桶里水深度为 $h$ **厘米**。

你在喝桶里的水，喝水的速度是 $v$ **毫升每秒**。

不喝水时桶内水面上升速度为 $e$ **厘米每秒**。

现在一边下雨，你一边喝桶里的水。

如果你喝的速度赶不上降水速度，输出"NO"，否则输出"YES"和你需要几秒喝光水。

## 样例 #1

### 输入

```
1 2 3 100
```

### 输出

```
NO
```

## 样例 #2

### 输入

```
1 1 1 1
```

### 输出

```
YES
3.659792366325
```

### 题解综合分析

各题解核心思路一致：将喝水的体积速度转换为高度变化速度，与雨水上升速度比较。关键步骤：
1. 计算底面积 $S = π(d/2)^2$
2. 喝水的高度速度 $v_{h} = v / S$
3. 比较 $v_{h}$ 与 $e$ 判断是否可喝完
4. 时间计算 $t = h / (v_{h} - e)$

**共同难点**：单位转换（毫升→立方厘米→高度变化）、浮点精度处理。所有题解均正确实现核心逻辑，差异主要在代码简洁度和精度处理方式。

### 精选题解（评分≥4⭐）

#### 1. 作者：CLCK（5⭐）
**核心亮点**：
- 直接转换喝水速度为高度单位，简化计算
- 代码简洁（仅10行核心逻辑）
- 使用精确的π值并保留12位小数

```cpp
#include<bits/stdc++.h>
using namespace std;
double d,h,v,e;
const double pi=3.1415926535898;
int main() {
    cin>>d>>h>>v>>e;
    v=v/(pi*(d/2)*(d/2));
    if(v<e)cout<<"NO"<<endl;
    else  {
        cout<<"YES"<<endl;
        printf("%.12lf",h/(v-e));
    }
    return 0;
}
```

### 最优思路总结
**关键技巧**：
1. **单位统一转换**：将喝水量 $v$（毫升/s）转换为高度变化速度 $v/S$（cm/s）
2. **速率差计算**：只需比较高度变化速度差，避免直接处理体积运算
3. **精确输出控制**：使用 `printf("%.12lf")` 或 `setprecision(12)` 保证小数精度

**易错点**：
- 底面积计算时半径应为 $d/2$ 而非 $d$
- 输入变量需用 double 类型存储（避免整数除法问题）

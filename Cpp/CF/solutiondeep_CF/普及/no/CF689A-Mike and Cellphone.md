# Mike and Cellphone

## 题目描述

迈克在海滩游泳时不小心将手机放入水中。他买了一个带有老式键盘的手机。键盘只有十个数字大小的键，位于以下方式：

| 1 | 2 | 3 |
| -----------: | -----------: | -----------: |
| 4 | 5 | 6 |
| 7 | 8 | 9 |
|   | 0 |   |   |
联系人与他的旧手机一起消失了，他现在只能记住他的手指在他输入一些数字时的移动方式。人们可以将手指动作视为连接按下按键的一系列动作。例如，数字“586”的手指移动动作与数字“253”的手指移动动作相同。
Mike通过他的“手指记忆”输入了一个数字并开始调用它，所以他现在担心，有没有其他数字，有相同的手指动作？

NO

## 样例 #3

```

### 输出

YES
```

## 样例 #4

### 输入

YES

## 题解分析与结论

通过分析键盘边界按键的移动限制，判断输入序列是否完全无法在某个方向移动。若存在至少一个方向可移动，则存在其他可能序列。核心思路是维护四个方向的移动可能性，遍历输入字符更新限制条件。

bool up, down, lft, rgt;

## 最优技巧总结
1. **边界分析法**：将键盘布局转化为方向限制条件表  
2. **状态压缩**：用四个布尔变量表示移动可能性，空间复杂度O(1)  
3. **复合条件优化**：利用逻辑或运算合并同类限制条件

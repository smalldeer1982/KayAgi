## 题目描述

当瓦西里在疗养院度假回来，发现他完全忘了他的度假的细节！



每天疗养院的餐厅里都有早餐、午餐和晚餐（当然，按这个顺序）。瓦西里现在唯一拥有的就是一张餐厅的卡片，上面写着他吃过多少次早餐、午餐和晚餐（因此，卡片包含三个整数）。瓦西里有时可能会错过一些食物，例如，他可以吃早饭和晚餐，但晚餐不吃，或者也许在某些天，他根本没有在餐厅。



瓦西里不记得一天中他到达疗养院的时间（早餐前、午餐前、晚餐前或晚餐后），以及他离开疗养院的时间（早餐前、午餐前、晚餐前或晚餐后）。所以他考虑了这些选择中的任何一个。瓦西里来到疗养院后，一直呆在那里直到离开。请注意，瓦西里可能是在抵达疗养院的同一天离开的。



根据卡片上的说明，帮助瓦西里确定他可能错过的餐厅的最少用餐次数。我们不应算作瓦西里抵达前一天的误餐，也不应算作瓦西里离开后一天的误餐。



根据卡片上的说明，帮助瓦西里确定他可能错过的餐厅的最少用餐次数。我们不应算作瓦西里抵达前一天的没吃的餐，也不应算作瓦西里离开后一天的没吃的餐。

## 题目描述

瓦西里从疗养院度假回来后，完全忘记了度假期间的细节！

每天疗养院的餐厅都会按顺序提供早餐、午餐和晚餐。瓦西里现在只有一张记录他实际用餐次数的卡片（包含三个整数）。瓦西里可能会错过某些餐点，例如某天可能只吃早餐和晚餐而错过午餐，甚至某些天根本不在餐厅用餐。

瓦西里不记得自己到达和离开疗养院的具体时间（可能在早餐前、午餐前、晚餐前或晚餐后）。请求根据卡片上的数据，计算他可能错过的最少用餐次数（不计入到达前和离开后的日期）。

## 最优题解分析

### 关键思路
1. **数学建模**：设三餐次数为 `a, b, c`，最大值为 `max_val = max(a, b, c)`。
2. **补餐规则**：若某餐次数少于 `max_val-1`，则必须补到 `max_val-1`，此时错过的次数为 `max_val-1 - 当前次数`。
3. **统一计算**：对每个餐次计算其与 `max_val-1` 的差值（负数取0），总和即为最小错过次数。

### 代码实现
```cpp
#include<bits/stdc++.h>
using namespace std;
int main() {
    long long a, b, c;
    cin >> a >> b >> c;
    long long max_val = max({a, b, c}) - 1;
    a = max(max_val - a, 0LL);
    b = max(max_val - b, 0LL);
    c = max(max_val - c, 0LL);
    cout << a + b + c;
    return 0;
}
```

### 综合分析与结论

题目核心是贪心策略：当 $x < y$ 时，将最大的 $x$ 个数放入 $X$；当 $x > y$ 时，将最大的 $y$ 个数放入 $Y$。字典序最小通过调整相同数值元素的选取顺序实现。唯一正确的题解为 CarroT5656 的解法。

---

### 精选题解（评分：⭐⭐⭐⭐⭐）

**关键亮点**：  
1. **贪心策略正确性**：根据 $x$ 与 $y$ 的大小关系，动态调整数值和编号的排序优先级。  
2. **字典序处理**：相同数值时，按编号方向排序以保证字典序最小。  
3. **代码简洁性**：通过自定义排序函数统一处理所有情况。

**核心代码**：  
```cpp
struct nod{ll x,id;}a[N];
bool cmp(nod a,nod b){
    if(a.x!=b.x) return a.x>b.x;
    return x<=y ? a.id <b.id : a.id >b.id;
}

sort(a+1,a+n+1,cmp);
if(x<y) for(int i=1;i<=x;i++) ans[a[i].id]=1;
else if(x>y) for(int i=n;i>=n-x+1;i--) ans[a[i].id]=1;
```

### 关键思路与技巧

1. **贡献最大化**：元素 $a$ 的贡献为 $\frac{a}{\min(x,y)}$，需优先将大值放入更小分母的集合。
2. **字典序优化**：相同数值时，根据 $x/y$ 关系决定编号顺序（小编号优先选入更优集合）。
3. **排序技巧**：降序主排序，配合动态调整的次排序规则。

# 题目信息

# Mister B and Angle in Polygon

## 题目描述

在一个边为 $n$ 的正凸多边形上，有一个角 $a$。

  现在要找到三个不同的顶点 $v1$ , $v2$ , $v3$ 使得 $∠ v1 v2 v3$ （其中 $v2$ 是角度的顶点，并且 $v1$ 和 $v3$ 位于其两侧）尽可能接近 $a$（换句话说，该值应该尽可能小）。

 如果有多个最优解，可以输出**任意一个**。

## 样例 #1

### 输入

```
3 15
```

### 输出

```
1 2 3
```

## 样例 #2

### 输入

```
4 67
```

### 输出

```
2 1 3
```

## 样例 #3

### 输入

```
4 68
```

### 输出

```
4 1 2
```

### 题解对比与结论
#### 关键思路
所有题解均利用正多边形的角度均匀分割特性：  
1. **角度分割**：正 $n$ 边形内角为 $\frac{(n-2) \cdot 180^\circ}{n}$，每个小段角度为 $\frac{180^\circ}{n}$。  
2. **固定顶点优化**：固定 $v1=1$ 和 $v2=2$，只需寻找 $v3$，通过数学计算或遍历快速确定最优解。  

#### 最优题解亮点
1. **BLuemoon_（4星）**  
   - **边界处理**：分三种情况处理极端值（极小/极大角度）。  
   - **倒序遍历优化**：从 $n$ 到 $3$ 倒序查找，发现差值增大时立即终止，时间复杂度接近 $O(1)$。  
   - **代码清晰**：逻辑简洁，覆盖所有情况。  

2. **nydzsf_qwq（4星）**  
   - **直接数学计算**：通过取整确定最接近的段数，无需遍历。  
   - **高效性**：时间复杂度 $O(1)$，代码极简。  
   - **心得**：强调通过分类讨论快速锁定结果。  

#### 核心代码实现
**BLuemoon_ 的核心逻辑**  
```cpp
for (int i = n, j = 1; i >= 3; i--, j++) {
    double d = k * j; // k=180/n
    if (abs(d - a) < maxa) {
        maxa = abs(d - a);
    } else {
        cout << "1 2 " << i + 1 << '\n';
        return 0;
    }
}
```
**实现思想**：倒序遍历可能的 $v3$，计算角度差值，发现差值不再减小时立即输出最优解。  

**nydzsf_qwq 的核心逻辑**  
```cpp
int tmp3 = alpha / tmp1; // tmp1=180/n
if (alpha - tmp3*tmp1 < (tmp3+1)*tmp1 - alpha) {
    cout << "1 2 " << n - tmp3 + 1;
} else {
    cout << "1 2 " << n - tmp3;
}
```
**实现思想**：通过取整确定最接近的段数，直接计算 $v3$ 的编号。

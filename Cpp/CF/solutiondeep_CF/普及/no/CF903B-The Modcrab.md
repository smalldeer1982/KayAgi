# The Modcrab

## 题目描述

沃瓦又在玩一个RPG游戏。在游戏里沃瓦扮演的角色收到了一个任务：去干死一只叫做魔之蟹的大 Boss。

在打了两个小时游戏后沃瓦知道了魔之蟹的各种数值。魔之蟹有 $h_2$ 点生命值和 $a_2$ 点攻击力。沃瓦是个手残，他不想死那么多回，于是就去商店氪了一大堆强力生命回复药剂来准备战斗。

沃瓦的角色有 $h_1$ 点初始生命和 $a_1$ 点攻击力。他还有无限多的强力生命回复药剂，每一瓶都可以给他回复 $c_1$ 点血。沃瓦的每一瓶药剂都是完全一样的。题目数据可以保证 $c_1 > a_2$ 。

战斗包含多个回合。在沃瓦的回合中，它既可以攻击魔之蟹（可以将他的生命值减少 $a_1$ ），也可以喝下一瓶药剂（可将沃瓦的生命值增加 $c_1$ ，沃瓦的生命值值可以超过 $h_1$ ）。现在如果沃瓦和魔之蟹都没有死的话，魔之蟹就会攻击沃瓦，将沃瓦的生命值减少 $a_2$ 。战斗会在沃瓦或魔之蟹中任意一方死亡后结束。

当然，沃瓦想赢，而且他还想赢的越快越好。所以他想制作一个让他用最短时间获胜的策略。

帮帮沃瓦把策略写出来吧！你可以假设沃瓦有无数瓶药水，而且测试数据能保证沃瓦总能赢。

# 魔之蟹

## 题目描述

沃瓦在玩一个RPG游戏时接到击败魔之蟹的任务。魔之蟹拥有 $h_2$ 点生命值和 $a_2$ 点攻击力。沃瓦的角色初始有 $h_1$ 点生命和 $a_1$ 点攻击力，并携带无限个能回复 $c_1$ 点生命的药剂（保证 $c_1 > a_2$）。

战斗回合制进行：
1. 沃瓦回合可选择：攻击（魔之蟹生命减 $a_1$）或喝药（自身生命加 $c_1$）
2. 双方存活时，魔之蟹攻击沃瓦（生命减 $a_2$）

求沃瓦获胜的最短回合策略。

### 题解分析与结论

各题解均采用贪心策略，核心思路为：
1. **必杀判断**：若当前能击杀Boss则攻击
2. **濒死判断**：若下回合会被Boss击杀则必须回血
3. **安全攻击**：其他情况优先攻击

**关键难点**在于正确处理攻击与回血的优先级关系，尤其注意：
- 回血后的Boss攻击判定
- 攻击后Boss可能立即死亡的终止条件

---

### 精选题解

#### 1. 文·和（★★★★☆）
**亮点**：通过队列记录操作，修复了未判断必杀情况的BUG  
**心得**：通过第三个测试点WA发现必须添加`h2>a1`的判断条件  
```cpp
do {
    if (h1 <= a2 && h2 > a1) { // 必须回血条件
        h1 += c1;
        q.push(1);
    } else { // 攻击条件
        h2 -= a1;   
        q.push(2);
    }
    h1 -= a2;
} while (h2 > 0);
```

#### 2. zhanghengrui（★★★★☆）
**亮点**：条件判断最简化的Python实现  
**核心逻辑**：合并判断安全攻击条件  
```python
if h1 > a2 or a1 >= h2:  # 安全攻击条件
    h2 -= a1
    lis.append('STRIKE')
else:
    h1 += c1
    lis.append('HEAL')
```

#### 3. Histone（★★★☆☆）
**技巧**：使用字符串拼接操作记录结果  
**注意点**：需处理Boss攻击前死亡的情况  
```cpp
if (h2 <= a1) {
    s += "STRIKE\n";
    h2 -= a1;
} else if (h1 <= a2) {
    s += "HEAL\n";
    h1 += (c - a2); // 提前抵消后续的Boss攻击
}
```

### 最优策略总结
1. **回合优先级**：必杀 > 保命 > 常规攻击  
2. **数值处理**：回血后的生命值计算需包含后续的Boss攻击扣除  
3. **实现技巧**：使用容器记录操作步骤，避免二次模拟

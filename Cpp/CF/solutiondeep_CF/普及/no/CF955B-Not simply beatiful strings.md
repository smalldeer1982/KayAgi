# Not simply beatiful strings

## 题目描述

如果一个字符串的字母可以重新排列成两组相等的符号（注意不同的组必须包含不同的符号），那么我们就称它为可爱的字符串。例如，字符串ababa是可爱的字符串（你可以把它转换成aaabb，其中前三个字母组成一组a-s，其他字母组成一组b-b-s），但c并不是因为在每一个可能的后续分区字母中这两组是重合的。





你得到了一个字符串s。检查是否可以将其拆分为两个非空子序列，以使这些子序列形成的字符串非常可爱。这里的子序列是字符串的任意索引集。

### 题解分析与结论
本题的核心逻辑是通过枚举字符串中字母的种类和数量，判断是否可以将字符串拆分为两个子序列，使得每个子序列都包含且仅包含两种不同的字母。题解主要集中在以下几个要点：
1. **字母种类和数量的统计**：使用数组或`map`来统计字符串中每种字母的出现次数。
2. **分类讨论**：根据字母种类的数量（2、3、4）进行不同的判断：
   - 如果字母种类为2，则每种字母的数量必须大于等于2。
   - 如果字母种类为3，则至少有一种字母的数量大于等于2。
   - 如果字母种类为4，则直接满足条件。
3. **特判**：如果字符串长度小于4或字母种类小于2或大于4，则直接输出`No`。

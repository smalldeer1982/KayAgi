# Lightsabers (easy)

## 题目描述

海蒂大师面前有 $n$ 名绝地武士，每个人都拿着一把光剑，光剑有 $m$ 种可能的颜色。她需要选择一些连续的武士，使得在选中的武士中，有 $k_1$ 名手持 $1$ 号颜色的光剑，$k_2$ 名手持 $2$ 号颜色的光剑，以此类推，$k_m$ 名手持 $m$ 号颜色的光剑。海蒂想知道这个方案是否可行，如果可行，输出 $\texttt{YES}$，否则输出 $\texttt{NO}$。

## 样例 #1

### 输入

```
5 2
1 1 2 2 1
1 2
```

### 输出

```
YES
```

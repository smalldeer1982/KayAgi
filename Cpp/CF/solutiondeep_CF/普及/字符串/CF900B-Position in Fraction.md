# 题目信息

# Position in Fraction

## 题目描述

给你三个整数 $a,b,c$，问 $\dfrac{a}{b}$ 的小数部分第几位是 $c$，如果无法找到，请输出 `-1`。

## 说明/提示

对于 $100\%$ 的数据：$1\le a<b\le 10^5,0\le c\le 9$。

## 样例 #1

### 输入

```
1 2 0
```

### 输出

```
2```

## 样例 #2

### 输入

```
2 3 7
```

### 输出

```
-1```

# AI分析结果

### 综合分析与结论
这些题解主要围绕模拟除法过程来确定 $\frac{a}{b}$ 的小数部分中数字 $c$ 的位置，若出现循环节且未找到 $c$ 则输出 `-1`。

- **思路对比**：大部分题解都采用模拟竖式除法的思路，部分题解先将分数化为最简形式，但对于本题数据范围，此步骤并非必要。还有题解使用 Java 的 `BigDecimal` 处理，但存在精度和性能问题。
- **算法要点**：模拟除法时，每次将余数乘以 10 再除以除数得到小数位，同时记录余数是否重复，重复则说明出现循环节。
- **解决难点**：主要难点在于判断循环节的出现，通过记录余数是否重复来解决。

#include<iostream>
#include<cstdio>
using namespace std;
const int mxn = 1e5+5;
int a,b,c,d,cnt;
bool ap[mxn];
int main(){
    scanf("%d%d%d",&a,&b,&c);
    a%=b;  //去除整数部分
    cnt=1;  //当前小数位数为1
    while(1){  //循环模拟除法
        a*=10;
        d=a/b;   //算小数部分
        a%=b;   //取余数
        if(d==c){  //找到第一个出现的位置，直接输出
            printf("%d",cnt);
            return 0;
        }
        if(ap[a])  //余数出现两次，有循环节，跳出
            break;
        ap[a]=true;
        cnt++;   //增加小数位数
    }
    printf("-1");  //未找到解
    return 0;
}
```

### 最优关键思路或技巧
- **模拟竖式除法**：通过不断将余数乘以 10 再除以除数，模拟手动计算除法的过程，获取小数部分。
- **记录余数**：使用数组记录余数是否重复，以此判断是否出现循环节。

# Binary Period

## 题目描述

假设有一个字符串$s$，如果它的第$i$ 个字符和第$i+k$ 个字符相等$(1\le i\le|s|-k)$ ，那么这个字符串的周期为$k$ 。其中，$|s|$ 表示字符串$s$ 的长度。



现在给你一个字符串$t$ ，$t$ 内只包括```0```和```1```，请你找出这个$s$ ，使其满足如下条件：

1. 字符串$s$ 也只包括```0```和```1```。

2. $|s|\le 2\times|t|$（$s$ 的长度不能超过$t$ 的长度的两倍）。

3. $t$ 是$s$ 的子串。

4. 在满足上面3个条件的情况下，$s$ 需要有最小的周期$k$ 。



$t$ 是$s$ 的子串就是说当$s$ 删除$0$个或更多个字符后（不能改变顺序），$s$ 就变成了$t$ 。例如```011```就是```10101```的子串。

## 样例 #1

### 输入

```
4
00
01
111
110```

### 输出

```
00
01
11111
1010```

### 题目内容
【题目描述】
假设有一个字符串 $s$，如果它的第 $i$ 个字符和第 $i + k$ 个字符相等 $(1 \leq i \leq |s| - k)$，那么这个字符串的周期为 $k$。其中，$|s|$ 表示字符串 $s$ 的长度。

1. 字符串 $s$ 也只包括 `0` 和 `1`。
2. $|s| \leq 2\times|t|$（$s$ 的长度不能超过 $t$ 的长度的两倍）。
3. $t$ 是 $s$ 的子串。

$t$ 是 $s$ 的子串就是说当 $s$ 删除 0 个或更多个字符后（不能改变顺序），$s$ 就变成了 $t$。例如 `011` 就是 `10101` 的子串。

### 综合分析与结论
这些题解整体思路一致，都是先判断字符串 $t$ 是否全为 `0` 或全为 `1`，若是则直接输出 $t$，因为此时周期为 1 最小；若不是，则构造周期为 2 的字符串 $s$。
- **思路**：均是分情况讨论，根据 $t$ 的字符组成决定 $s$ 的构造方式。
- **算法要点**：判断 $t$ 中字符是否单一，若单一则直接输出，否则构造周期为 2 的字符串。
- **解决难点**：核心难点在于构造满足条件的最小周期字符串。不同题解在构造周期为 2 的字符串时方法略有不同，有的直接构造长度为 $2|t|$ 的 `01` 或 `10` 串，有的在 $t$ 中插入不同字符使其周期为 2。

    - **核心代码**：
```cpp
#include <iostream>
using namespace std;
int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);
    int T;
    cin >> T;
    while (T--) {
        string t;
        cin >> t;
        int n = t.size();
        if (t == string(n, '0') || t == string(n, '1')) cout << t << '\n';
        else {
            string s(2 * n, '0');
            for (int i = 0; i < n; i++) s[2 * i] = '1';
            cout << s << '\n';
        }
    }
    return 0;
}
```

### 最优关键思路或技巧
- **分类讨论**：将问题分为 $t$ 为单一字符和非单一字符两种情况，简化问题。
- **周期分析**：明确周期为 1 和 2 的字符串形式，优先考虑最小周期。

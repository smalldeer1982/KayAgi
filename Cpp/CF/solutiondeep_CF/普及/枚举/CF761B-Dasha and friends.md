# Dasha and friends

## 题目描述

小A和小B参加障碍跑，跑道可能有多个，所有跑道是环形的，按逆时针方向跑，长度都是是L，都有n个障碍。 小A从起点A开始跑，沿途记录下经过的障碍物离开起点A的距离，如2,4,6；小B从起点B开始跑，沿途记录下经过的障碍物离开起点B的距离，如1,5,7。距离均为整数。 现在请你判断，小A和小B的使用的是否是同一个跑道。

### 综合分析与结论
这些题解的核心思路都是判断两个环形跑道是否可以通过旋转后重合。具体思路和方法有所不同：
- **最小表示法**：___new2zy___ 的题解将障碍之间的距离作为判断量，利用最小表示法在 $O(n)$ 时间内判断两个环是否循环同构。
- **暴力旋转法**：BushHuang、kaaaai、_cpp 的题解通过不断旋转其中一个环，每次旋转后与另一个环比较，最多旋转 $L$ 次，时间复杂度为 $O(L^2)$。
- **枚举旋转法**：A_small_WA 的题解将小A记录的所有障碍物每次 $+1$，模拟顺时针转动，旋转 $L$ 次，每次旋转后排序再与小B的记录比较。

### 所选题解
- ___new2zy___（5星）
  - 关键亮点：使用最小表示法，时间复杂度为 $O(n)$，效率高。
  - 核心代码：
```cpp
inline int Min_show(int s[])
{
    int i = 0, j = 1, k = 0;
    while (i < n && j < n && k < n)
    {
        if (s[(i + k) % n] == s[(j + k) % n])
            k++;
        else
        {
            if (s[(i + k) % n] > s[(j + k) % n])
                i += k + 1;
            else
                j += k + 1;
            if (i == j)
                i++;
            k = 0;
        }
    }
    return min(i, j);
}
```
核心实现思想：通过两个指针 $i$ 和 $j$ 以及匹配长度 $k$，不断比较两个位置的元素，若相同则 $k$ 加1，若不同则移动指针，最终找到串的最小表示。

### 最优关键思路或技巧
最小表示法可以在 $O(n)$ 时间内判断两个长度相等的字符串是否是循环同构的，适用于环形结构的判断问题。

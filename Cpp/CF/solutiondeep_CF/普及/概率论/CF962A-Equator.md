# Equator

## 题目描述

已知 Polycarp 共有 $n$ 天准备编程竞赛，他每天预计做的题数为 $a_i$，请问直到第几天能做完所有题目的一半或更多。

## 样例 #1

### 输入

```
4
1 3 2 1
```

### 输出

```
2
```

## 样例 #2

### 输入

```
6
2 2 2 2 2 2
```

### 输出

```
3
```

### 综合分析与结论
这些题解主要围绕模拟和二分查找两种思路来解决问题。模拟思路是通过累加每天做的题数，直到达到或超过总题数的一半，输出对应的天数；二分查找思路则是利用前缀和的单调性，通过二分法找到满足条件的最小天数。

#### 思路对比
- **模拟思路**：简单直接，遍历数组累加题数，判断是否达到一半。
- **二分查找思路**：利用前缀和单调性，减少不必要的遍历，时间复杂度更低。

#### 算法要点
- **模拟**：累加题数，判断是否达到总题数一半。
- **二分查找**：计算前缀和，二分查找满足条件的最小天数。

#### 解决难点
- **总题数为奇数**：直接除以2会向下取整，需要加1处理。
- **前缀和应用**：二分查找需要前缀和数组来快速计算区间和。

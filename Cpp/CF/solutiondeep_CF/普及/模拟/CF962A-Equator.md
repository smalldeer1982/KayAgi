# Equator

## 题目描述

已知 Polycarp 共有 $n$ 天准备编程竞赛，他每天预计做的题数为 $a_i$，请问直到第几天能做完所有题目的一半或更多。

## 样例 #1

### 输入

```
4
1 3 2 1
```

### 输出

```
2
```

## 样例 #2

### 输入

```
6
2 2 2 2 2 2
```

### 输出

```
3
```

### 综合分析与结论
这些题解的核心目标都是找出直到第几天能做完所有题目的一半或更多。主要思路可分为模拟和二分查找两类。

#### 模拟思路
- **算法要点**：先求出所有题目的总数，再计算出一半的题数（考虑奇数情况），然后从第一天开始累加每天做的题数，当累加值达到或超过一半题数时，输出当前天数。
- **解决难点**：处理总题数为奇数时，整数除法向下取整的问题，通过`(sum + 1) / 2`或`sum * 1.0 / 2`等方式解决。

#### 二分查找思路
- **算法要点**：使用前缀和维护前`i`天做的题数之和，由于前缀和单调递增，利用二分查找来找到满足条件的最小`k`。
- **解决难点**：除了处理奇数题数的精度问题，还需注意二分查找的边界条件和最终结果的调整。

# 题目信息

# T-primes

## 题目描述

我们知道质数是只有两个不同的正数因数的正整数。相似的，我们把一个正整数 _t_ 叫做 T质数，如果 _t_ 恰好有三个不同的正整数因数。 

你被给了一个含有 _n_ 个正整数的数组。你要给其中所有的数判断它是否是 T质数。

## 说明/提示

给定样例有三个数。第一个数4恰好有三个因数--1、2和4，因此对这个数字输出YES。第二个数5有两个因数（1和5），第三个数6有四个因数（1、2、3和6），因此这两个数的答案是NO。

## 样例 #1

### 输入

```
3
4 5 6
```

### 输出

```
YES
NO
NO
```

##### 题目描述

### 分析与结论
本题要求判断给定数组中的每个正整数是否为 T 质数（即恰好有三个不同正因数的正整数）。各题解的核心思路都是先判断该数是否为完全平方数，若是则进一步判断其平方根是否为质数，若是质数则该数为 T 质数，否则不是。

### 关键思路或技巧
- **数学性质运用**：利用完全平方数因数个数为奇数的性质，以及 T 质数是质数的平方这一结论，将问题转化为判断数是否为完全平方数且其平方根是否为质数。
- **质数判断优化**：如 Laser_Crystal 提到的除 2、3 外质数除以 6 余数为 1 或 5 的技巧，减少质数判断的时间复杂度。
- **筛法运用**：如 Siyuan 的线性筛、Minecraft万岁的埃筛、pipiispig 的欧拉筛，用于预处理质数。

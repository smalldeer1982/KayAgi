# System Administrator

## 题目描述

给定 $n$ 个顶点，可以在这些顶点之间连接 $m$ 条双向边，要求连接后整个图联通，并且去掉 $v$ 点后整个图不连通。

若有解，输出所连得的 $m$ 条边，随意输出一种方案即可；若无解，输出 ```-1```。

## 说明/提示

【题解分析与结论】

各题解的思路基本一致，主要分为以下几个步骤：
1. **判断无解情况**：当 $m < n-1$ 或 $m > \frac{(n-1)(n-2)}{2} + 1$ 时无解。
2. **构造连通图**：首先构造一个以 $v$ 为中心的菊花图，确保图连通。
3. **构造割点**：在菊花图的基础上，尽可能将其他点连成完全图，确保去掉 $v$ 后图不连通。

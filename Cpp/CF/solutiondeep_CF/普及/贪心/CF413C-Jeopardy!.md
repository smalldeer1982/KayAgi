# Jeopardy!

## 题目描述

题意简述

“Jeopardy！”的决赛将有n问题，每个问题都有对应的得分ai，其中有m个问题可以选择不得分，而将现有总得分翻倍。你可以安排关卡的通过顺序和策略，求最大得分。

## 样例 #1

### 输入

```
4 1
1 3 7 5
3
```

### 输出

```
18
```

## 样例 #2

### 输入

```
3 2
10 3 8
2 3
```

### 输出

```
40
```

## 样例 #3

### 输入

```
2 2
100 200
1 2
```

### 输出

```
400
```

### 题解分析与结论

#### 综合分析与结论
本题的核心在于贪心算法的应用。所有题解都采用了贪心策略，主要思路是：
1. **先处理不能翻倍的问题**，直接将其得分累加到总得分中。
2. **对可以翻倍的问题进行排序**，通常按得分从大到小排序，以确保在翻倍时能获得最大收益。
3. **逐个处理可以翻倍的问题**，比较翻倍和直接加分的收益，选择收益更大的操作。

### 最优关键思路与技巧
1. **贪心策略**：先处理不能翻倍的问题，再处理可以翻倍的问题，确保在翻倍时能获得最大收益。
2. **排序优化**：对可以翻倍的问题按得分从大到小排序，确保在翻倍时能获得最大收益。
3. **数据结构**：使用结构体或数组存储问题的得分和是否可以翻倍，便于排序和处理。

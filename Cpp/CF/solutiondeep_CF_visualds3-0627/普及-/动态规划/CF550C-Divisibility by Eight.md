# 题目信息

# Divisibility by Eight

## 题目描述

You are given a non-negative integer $ n $ , its decimal representation consists of at most $ 100 $ digits and doesn't contain leading zeroes.

Your task is to determine if it is possible in this case to remove some of the digits (possibly not remove any digit at all) so that the result contains at least one digit, forms a non-negative integer, doesn't have leading zeroes and is divisible by 8. After the removing, it is forbidden to rearrange the digits.

If a solution exists, you should print it.

## 样例 #1

### 输入

```
3454
```

### 输出

```
YES
344
```

## 样例 #2

### 输入

```
10
```

### 输出

```
YES
0
```

## 样例 #3

### 输入

```
111111
```

### 输出

```
NO
```

# 【综合分析与结论】

## 核心思路
利用**数学特性**：一个数能被8整除当且仅当其后三位数能被8整除。所有题解基于以下策略：
1. 特判存在0的情况（直接输出0）
2. 暴力枚举所有可能的一位、两位、三位数组合
3. 按顺序检查组合是否能被8整除

## 解决难点
- **前导零处理**：优先判断0的存在，单独处理
- **高效枚举**：按数字顺序进行三层嵌套循环，优先检查较短组合（1位→2位→3位）
- **顺序约束**：严格保持原数字顺序，不重新排列

## 可视化设计思路
使用**像素风格动画**展示数字选择过程：
1. 在Canvas中绘制数字序列，初始为灰色
2. 枚举三位数时：
   - 用黄色高亮候选的第一位数字
   - 绿色高亮第二位，蓝色高亮第三位
   - 计算数值时显示半透明覆盖层展示"XXX%8=?"
3. 找到合法组合时：
   - 触发上扬音效
   - 被选数字闪烁三次
4. 失败时播放低沉音效，所有数字变为红色

## 灵光一闪题解的关键逻辑
```cpp
// 特判0的存在
if(s.find('0') != string::npos) {
    cout << "YES\n0";
    return 0;
}

// 三层嵌套检查
for(int i=0; i<s.size(); i++){
    if((s[i]-'0') %8 ==0){ // 一位数检查
        cout << "YES\n" << s[i];
        return 0;
    }
    for(int j=i+1; j<s.size(); j++){
        int twoDig = (s[i]-'0')*10 + (s[j]-'0');
        if(twoDig %8 ==0){ // 两位数检查
            cout << "YES\n" << s[i] << s[j];
            return 0;
        }
        for(int k=j+1; k<s.size(); k++){
            int threeDig = (s[i]-'0')*100 + (s[j]-'0')*10 + (s[k]-'0');
            if(threeDig %8 ==0){ // 三位数检查
                cout << "YES\n" << s[i] << s[j] << s[k];
                return 0;
            }
        }
    }
}
```

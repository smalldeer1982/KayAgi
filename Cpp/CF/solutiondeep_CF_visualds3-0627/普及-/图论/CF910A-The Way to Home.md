# The Way to Home

## 题目描述

一只青蛙现在在一个数轴上，它现在要从点 $1$ 跳到点 $n$ ，它每次可以向右跳不超过 $d$ 个单位。比如，它可以从点 $x$ 跳到点 $x+a$  $( 1<=a<=d )$ 。特别的，青蛙只能在有百合花的点上停留。保证点 $1$ 和点 $n$ 之间有一些点有百合花。请输出青蛙到达点 $n$ 的最小跳跃次数。

## 说明/提示

在样例1中，青蛙可以从点 $1$ 跳3个单位到点 $4$ ，再从点 $4$ 跳4个单位到点 $8$ .
在样例2中，青蛙不能到达点 $n$ ，因为它至少需要跳3个单位，但它最多只能跳2个单位。

## 综合分析与结论
### 核心思路与算法要点
题目本质是典型的最优跳跃问题，所有题解可归纳为三类思路：
1. **贪心策略**：从当前点尽可能跳最远，保证步数最少（wky32768、Lips、Vector_）
2. **动态规划**：维护到达每个点的最小步数（空气树、jiangXxin）
3. **路径搜索**：DFS/BFS或最短路算法遍历所有可能路径（学而思李老师、Siyuan）

**最优解法**为贪心策略，其正确性基于以下两点：
1. **局部最优性**：每次选择当前可跳范围内最远的点，必然属于全局最优解
2. **无后效性**：后续跳跃只与当前所处位置有关，与到达当前点的路径无关

### 解决难点与优化技巧
- **关键难点**：如何证明贪心策略的正确性
- **优化突破**：通过反证法可证明，若存在更优路径，必然可以通过调整跳跃步长得到矛盾
- **代码实现**：倒序遍历检查跳跃范围（`for i from min(pos+d, n) downto pos+1`）

### 可视化设计思路
1. **动画流程**：
   - 用像素方块表示数轴，绿色方块表示荷叶
   - 青蛙从起点逐帧向右跳跃，当前检查区间高亮黄色边框
   - 发现可跳点时播放跳跃音效，目标点变蓝
2. **交互设计**：
   - 步进控制：空格键单步执行，方向键调节速度
   - 失败时屏幕抖动+红色闪烁，成功时烟花特效+胜利音效
3. **复古风格**：
   - 8-bit 音效：跳跃（短促"哔"声）、成功（上扬旋律）、失败（低沉嗡鸣）
   - 16色像素调色板，仿 FC 游戏界面

## 核心代码实现
### 贪心算法（最优解）
```cpp
int ans = 0, pos = 0;
while(pos < n-1) { // 数轴从0开始
    int next = pos;
    for(int i = pos+1; i <= min(pos+d, n-1); i++) {
        if(s[i] == '1') next = i; // 记录最远可跳点
    }
    if(next == pos) return -1; // 无进展则失败
    pos = next;
    ans++;
}
return ans;
```

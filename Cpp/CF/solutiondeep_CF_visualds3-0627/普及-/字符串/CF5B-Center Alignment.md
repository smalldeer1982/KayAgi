# 题目信息

# Center Alignment

## 题目描述

几乎每一个文本编辑器都有文字居中对齐的功能，因此Berland的著名文本编辑器Textpad的开发者们希望在产品的第四次更新中加入此功能。

您的任务是尽快编写一个文本居中对齐的原型程序，祝您好运!

## 样例 #1

### 输入

```
This  is

Codeforces
Beta
Round
5
```

### 输出

```
************
* This  is *
*          *
*Codeforces*
*   Beta   *
*  Round   *
*     5    *
************
```

### **题解思路与核心难点**
1. **核心思路**  
   - 读取所有输入行，计算最大长度 `max_len`。
   - 每行首尾添加星号，中间根据 `max_len` 居中对齐。
   - 若无法完美对齐（`max_len - 当前行长度` 为奇数），交替左/右调整。

2. **解决难点**  
   - **空格分配**：当长度差为奇数时，需交替调整左右空格数。例如，差为3时，第一次左补1、右补2，第二次左补2、右补1。
   - **空行处理**：空行需填充 `max_len` 个空格。
   - **交替逻辑**：用变量（如 `cnt`、`bt`）记录调整方向，每次翻转。

3. **关键变量**  
   - `max_len`：最长行长度。
   - `cnt/bt/o`：交替调整的标记变量。
   - `left_space`、`right_space`：左右空格数。

### **代码片段（核心逻辑）**
```cpp
// 以 SIXIANG32 的代码为例
int cnt = 0; // 交替计数器
for (每行字符串 s) {
    int diff = max_len - s.size();
    if (diff % 2 == 0) { // 完美对齐
        int space = diff / 2;
        cout << "*" << string(space, ' ') << s << string(space, ' ') << "*";
    } else { // 交替调整
        cnt++;
        int left = diff / 2;
        int right = diff - left;
        if (cnt % 2 == 1) { // 左少右多
            cout << "*" << string(left, ' ') << s << string(right, ' ') << "*";
        } else { // 右少左多
            cout << "*" << string(left + 1, ' ') << s << string(right - 1, ' ') << "*";
        }
    }
}
```

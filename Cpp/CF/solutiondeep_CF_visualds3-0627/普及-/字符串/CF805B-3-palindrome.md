# 题目信息

# 3-palindrome

## 题目描述

题意
给定n，求长度为n的以'a','b'或'c'构成的字符串，使得其中不存在长为3的回文子串（如"abc","abca"均符合，但"aba"不符），并使'c'的个数尽可能少

## 样例 #1

### 输入

```
2
```

### 输出

```
aa
```

## 样例 #2

### 输入

```
3
```

### 输出

```
bba
```

## **题解思路与核心分析**

### **核心思路**
所有题解均采用**构造循环模式**的策略，通过周期性排列字符 `a` 和 `b` 避免出现长度为3的回文子串，并完全舍弃 `c` 以满足最小化其数量的要求。

#### **关键步骤**
1. **循环节选择**：大多数题解使用长度为4的循环节（如 `bbaa`、`abba`、`aabb` 等），确保任意连续三字符不构成回文。
2. **边界处理**：对奇数长度的末尾字符进行调整，确保不破坏回文约束。
3. **代码实现**：直接通过模运算循环输出预定义的字符序列。

#### **解决难点**
- **回文避免**：通过循环节设计，使得每个字符与前两字符不同，自动排除所有可能的回文子串。
- **无 `c` 构造**：所有题解均证明 `a` 和 `b` 的排列足以满足条件，无需引入 `c`。

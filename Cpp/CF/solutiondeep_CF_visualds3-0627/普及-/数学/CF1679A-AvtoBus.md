# AvtoBus

## 题目描述

春天来了，车队的巴士需要更换轮胎。

轮胎有 $n$ 个，巴士有两种，第一种有 $4$ 个轮子，第二种有 $6$ 个。

求车队最少和最多有几辆巴士。

## 样例 #1

### 输入

```
4
4
7
24
998244353998244352```

### 输出

```
1 1
-1
4 6
166374058999707392 249561088499561088```

### **题解思路、算法要点与解决难点**
**核心问题**：求解方程 \(4a + 6b = n\) 的非负整数解中 \(a+b\) 的最小值和最大值。

**关键数学推导**：
1. **无解条件**：
   - \(n\) 为奇数或 \(n < 4\) 时无解。
   - 方程可化简为 \(2a + 3b = \frac{n}{2}\)，故 \(n\) 必须为偶数。

2. **最小值计算**：
   - 尽量多用 6 轮车（减少总数量）。
   - 若 \(n \equiv 0 \mod 6\)，则最小值 \(= \frac{n}{6}\)。
   - 若 \(n \equiv 2 \mod 6\)，需将 1 辆 6 轮车拆为 2 辆 4 轮车：最小值 \(= \frac{n-8}{6} + 2\)。
   - 若 \(n \equiv 4 \mod 6\)，直接加 1 辆 4 轮车：最小值 \(= \frac{n-4}{6} + 1\)。
   - 统一公式：最小值 \(= \lceil \frac{n}{6} \rceil\)。

3. **最大值计算**：
   - 尽量多用 4 轮车（增加总数量）。
   - 若 \(n \equiv 0 \mod 4\)，则最大值 \(= \frac{n}{4}\)。
   - 若 \(n \equiv 2 \mod 4\)，拆 1 辆 4 轮车为 1 辆 6 轮车：最大值 \(= \frac{n-6}{4} + 1 = \frac{n}{4} - 0.5 + 1\)（整数运算下等价于 \(\frac{n}{4}\)）。
   - 统一公式：最大值 \(= \frac{n}{4}\)（向下取整）。

**解决难点**：余数的处理需要构造合法解，确保拆分后轮胎总数不变。

### **核心代码实现**
```cpp
#include <iostream>
using namespace std;

int main() {
    int t;
    cin >> t;
    while (t--) {
        long long n;
        cin >> n;
        if (n < 4 || n % 2 != 0) {
            cout << "-1\n";
            continue;
        }
        // 最小值计算
        long long min_val = (n + 5) / 6; // 等价于 ceil(n/6)
        // 最大值计算
        long long max_val = n / 4;
        cout << min_val << " " << max_val << "\n";
    }
    return 0;
}
```

**代码解释**：
- **最小值**：`(n + 5) / 6` 实现向上取整，避免浮点运算。
- **最大值**：直接整除 4，利用整数除法自动向下取整。

# 题目信息

# Simple Palindrome

## 题目描述

构造一个 **长度为** $n$ ，且只由 $a$，$e$，$i$，$o$，$u$ **这 $5$ 种字母** 构成字符串，使得这个字符串的 **所有子序列** 是 **回文串** 的个数最少。请输出这个 **最小值** 。

## 说明/提示

对于 $100\%$ 的测试点，$1\le T\le 100$， $1\le n\le 100$ 。

## 样例 #1

### 输入

```
3
2
3
6```

### 输出

```
uo
iae
oeiiua```

## 综合分析与结论
本题要求构造一个由特定字符组成的字符串，使得其所有子序列中回文的数量最少。核心在于如何合理分配字符的出现次数以最小化回文子序列的总数。

### 核心公式推导
回文子序列的总数由每个字符的出现次数决定。对于出现次数为 \( c_i \) 的字符，其贡献为 \( 2^{c_i} - 1 \)。总和为所有字符的贡献之和。为使总和最小，需将字符的出现次数尽可能均匀分配。

### 分配策略
将 \( n \) 分解为 \( k = \lfloor n/5 \rfloor \) 和 \( r = n \mod 5 \)。前 \( 5-r \) 个字符出现 \( k \) 次，后 \( r \) 个字符出现 \( k+1 \) 次。这一分配确保了总和的最小化。

### 构造字符串
将字符按字典序排列（a, e, i, o, u），后 \( r \) 个字符分配更多的次数。构造时按顺序拼接各字符的重复次数，保证字典序最小。

## 代码实现
```python
t = int(input())
for _ in range(t):
    n = int(input())
    k, r = divmod(n, 5)
    ans = []
    chars = ['a', 'e', 'i', 'o', 'u']
    for i in range(5):
        count = k + 1 if i >= 5 - r else k
        ans.append(chars[i] * count)
    print(''.join(ans))
```

### 核心逻辑
- **divmod计算**：分解 \( n \) 为基数和余数。
- **后r分配**：后 \( r \) 个字符次数加1，确保均匀分布。
- **顺序拼接**：按字符顺序构造字符串，保证字典序最小。

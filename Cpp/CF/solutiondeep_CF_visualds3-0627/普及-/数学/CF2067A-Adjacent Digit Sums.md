# Adjacent Digit Sums

## 题目描述

给定两个数字 $x, y$。你需要判断是否存在一个整数 $n$ 使得 $S(n) = x$ 且 $S(n + 1) = y$。

其中，$S(a)$ 表示数字 $a$ 在十进制系统中的各位数字之和。


## 说明/提示

在第一个测试用例中，例如 $n = 100$ 满足条件。此时 $S(100) = 1$，$S(101) = 2$。

在第二个测试用例中，可以证明对于所有 $n$ 均有 $S(n) \neq S(n+1)$，因此答案为 No。

在第四个测试用例中，$n = 10^{111}-1$ 满足条件，这是一个由 $111$ 个 $9$ 组成的数字。

## 综合分析与结论

### 核心思路与难点
题目要求判断是否存在整数 `n` 使得 `S(n) = x` 且 `S(n+1) = y`，其中 `S(a)` 表示 `a` 的各位数字之和。核心在于通过数学推导，找到 `x` 和 `y` 必须满足的条件。

1. **进位分析**：当 `n` 加 1 时，若末尾有 `k` 个 9，则 `S(n+1) = S(n) - 9k + 1`。  
2. **条件推导**：存在非负整数 `k`，使得 `x - y + 1 = 9k`，且需满足：  
   - 若 `k=0`，则 `y = x + 1`（无进位）。  
   - 若 `k≥1`，则 `x ≥ 9k`（确保构造的 `n` 有效）。

### 可视化设计思路
1. **颜色标记**：高亮 `n` 的末尾连续 9 的个数（`k`），并动态展示进位后的变化。  
2. **步进控制**：展示 `n` 的构造过程，如从 `n=999` 到 `n+1=1000` 的各位变化。  
3. **像素风格**：用 8 位网格表示数字，红色标记 9 的连续段，绿色标记进位后的高位。

## 题解清单 (≥4星)

### 题解：数学条件判断（5星）
**关键亮点**：
- 直接通过数学公式推导条件，无需复杂结构。  
- 时间复杂度 O(1)，空间复杂度 O(1)。  
- 覆盖所有边界情况，如 `k=0` 和 `k≥1` 的精确判断。

**代码片段**：
```python
t = int(input())
for _ in range(t):
    x, y = map(int, input().split())
    d = x - y + 1
    if d < 0 or d % 9 != 0:
        print("No")
        continue
    k = d // 9
    if k == 0:
        print("Yes" if y == x + 1 else "No")
    else:
        print("Yes" if x >= 9 * k else "No")
```

## 最优思路提炼
1. **数学条件**：  
   - `x - y + 1` 必须是非负且能被 9 整除。  
   - 当 `k=0` 时，要求 `y = x + 1`；当 `k≥1` 时，要求 `x ≥ 9k`。  
2. **构造方法**：  
   - 若条件成立，末尾 `k` 个 9 的构造总能满足要求，如 `n = 10^k - 1`。

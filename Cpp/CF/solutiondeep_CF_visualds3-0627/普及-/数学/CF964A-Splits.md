# Splits

## 题目描述

我们定义正整数$n$的分裂为一个由正整数组成的不上升序列，且序列数字和为$n$

举个栗子：下列这些序列都是$8$的分裂：$[4,4],[3,3,2],[2,2,1,1,1],[5,2,1]$

下列这些序列不是$8$的分裂：$[1,7],[5,4],[11,-3],[1,1,4,1,1]$

一个分裂的权是序列第一个数出现的次数，举个例子：$[1,1,1,1,1]$的权是$5$，$[5,5,3,3,3]$的权是$2$，$[9]$的权是$1$

现在给出$n$，求$n$的分裂有多少个不同的权

## 样例 #1

### 输入

```
7
```

### 输出

```
4
```

## 样例 #2

### 输入

```
8
```

### 输出

```
5
```

## 样例 #3


### 输出

```
5
```

## 综合分析与结论

### 核心思路与数学推导
题目要求计算正整数 $n$ 的所有分裂中不同权值的数量。权值定义为序列第一个数出现的次数。通过观察样例和数学推导，发现答案与 $n$ 的奇偶性无关，直接由公式 $\lfloor n/2 \rfloor + 1$ 给出。

**关键推导步骤**：
1. **贪心构造**：最优策略是仅用数值 $1$ 和 $2$ 构造分裂序列。这样能最大化权值的多样性。
2. **最大 $2$ 的数量**：若最多能拆出 $m = \lfloor n/2 \rfloor$ 个 $2$，则权值可以是 $1, 2, \dots, m$，加上全为 $1$ 的情况（权值为 $n$），但通过数学归纳发现实际只需计算 $m + 1$ 种。
3. **公式证明**：对于任意 $n$，权值种类数为 $\lfloor n/2 \rfloor + 1$。例如，当 $n=7$，$m=3$，答案 $3+1=4$。

### 解决难点
- **构造正确性验证**：通过贪心策略确保每个可能的权值 $t$ 都能被构造，例如拆分成 $t$ 个 $2$ 和剩余 $1$。
- **公式适用范围**：无论 $n$ 的奇偶性如何，公式均成立。

### 可视化设计思路
- **动画演示**：在 Canvas 上动态拆解 $n$ 为 $2$ 和 $1$ 的组合。每步高亮当前拆分的 $2$，剩余部分以不同颜色标记。
- **步进控制**：用户可单步观察拆分过程，例如 $n=8$ 拆为 $4$ 个 $2$，再逐步替换 $2$ 为两个 $1$，权值从 $4$ 递减到 $1$。
- **复古像素风格**：用 8-bit 风格显示拆分步骤，音效提示每步操作。

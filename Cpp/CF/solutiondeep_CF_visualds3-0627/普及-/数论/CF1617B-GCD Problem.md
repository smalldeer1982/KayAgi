# GCD Problem

## 题目描述

Given a positive integer $ n $ . Find three distinct positive integers $ a $ , $ b $ , $ c $ such that $ a + b + c = n $ and $ \operatorname{gcd}(a, b) = c $ , where $ \operatorname{gcd}(x, y) $ denotes the [greatest common divisor (GCD)](https://en.wikipedia.org/wiki/Greatest_common_divisor) of integers $ x $ and $ y $ .

## 说明/提示

In the first test case, $ 6 + 9 + 3 = 18 $ and $ \operatorname{gcd}(6, 9) = 3 $ .

In the second test case, $ 21 + 39 + 3 = 63 $ and $ \operatorname{gcd}(21, 39) = 3 $ .

In the third test case, $ 29 + 43 + 1 = 73 $ and $ \operatorname{gcd}(29, 43) = 1 $ .

### 综合分析与结论

#### 核心算法思路
题目核心是通过数学构造法快速生成符合条件的解。所有题解的核心逻辑可归纳为：
1. **固定 c=1**：将问题转化为寻找互质数对 (a, b) 使得 a + b = n-1。
2. **分奇偶构造**：
   - **偶数 n**：直接取相邻两数 (n/2-1, n/2)，其和为 n-1 且互质。
   - **奇数 n**：
     - 若 (n-1) 是 4 的倍数：取中心对称的两个奇数 (d-1, d+1)，差为 2 且互质。
     - 否则：取差为 4 的两个数 (d-2, d+2)，利用更大间隔保证互质。

#### 解决难点
- **互质验证**：相邻奇数/特定间隔数的互质性是核心数学基础。
- **覆盖所有情况**：通过奇偶与模 4 分类确保构造覆盖所有可能的 n。

#### 可视化设计
- **动画流程**：
  1. 输入 n 后，高亮显示「固定 c=1」的逻辑。
  2. 根据奇偶性分叉动画路径：
     - **偶数路径**：显示拆分相邻数对，验证其和为 n-1，并弹出 GCD=1 的提示。
     - **奇数路径**：根据模 4 结果切换分支，显示对称数的构造过程。
  3. 最终输出结果时，用像素风格爆炸特效强调答案。
- **复古风格**：
  - **颜色方案**：使用 FC 红白机经典的 8 色调色板（如 #FF0000 表示错误，00FF00 表示正确）。
  - **音效**：选择操作时播放 FC 风格的“嘀”声，正确解触发经典过关音效。

### 最优思路代码实现
**核心逻辑实现（C++）**：
```cpp
void solve(int n) {
  if (n % 2 == 0) {
    cout << n/2-1 << " " << n/2 << " 1\n";
  } else {
    int d = (n-1)/2;
    if ((n-1) % 4 == 0) {
      cout << d-1 << " " << d+1 << " 1\n";
    } else {
      cout << d-2 << " " << d+2 << " 1\n";
    }
  }
}
```

# 题目信息

# Dora's Set

## 题目描述

Dora 有一个整数集合 $s$。在最开始，她会将所有满足 $l\le x\le r$ 的整数 $x$ 放入 $s$ 中。然后她允许你进行如下操作：
- 首先，在 $s$ 中选择三个不同的整数 $a,b,c$，并且需要确保它们满足 $\gcd(a,b)=\gcd(b,c)=\gcd(a,c)=1$。
- 然后，将这三个整数从 $s$ 中删除。

其中 $\gcd(x,y)$ 是整数 $x$ 与 $y$ 的[最大公因数](https://en.wikipedia.org/wiki/Greatest_common_divisor)。

你最多能进行多少次操作呢？

## 样例 #1

### 输入

```
8
1 3
3 7
10 21
2 8
51 60
2 15
10 26
1 1000```

### 输出

```
1
1
3
1
2
3
4
250```

## 【综合分析与结论】
本题考察数学构造与奇偶性质分析。核心思路基于以下观察：每次操作需选取两奇一偶且两两互质的三元组。最大操作次数由奇偶数量共同决定，取「奇数数//2」与「偶数数」的较小值。

**算法流程：**
1. 计算区间内奇数数量 `odd = ((r+1)//2) - (l//2)`
2. 计算偶数数量 `even = (r-l+1) - odd`
3. 结果为 `min(odd//2, even)`

## 【代码实现】
```python
t = int(input())
for _ in range(t):
    l, r = map(int, input().split())
    odd = ((r + 1) // 2) - (l // 2)
    even = (r - l + 1) - odd
    print(min(odd // 2, even))
```

## 【最优思路提炼】
**关键技巧**：利用奇偶性质构造三元组。每个操作选两个互质奇数和一个偶数，三者必两两互质。此时最大操作次数受限于奇数对数与偶数数量。

**数学推导**：
- 奇数间互质概率较高，偶数为2的幂时与任意奇数互质。
- 奇偶数量决定操作上限：`min(奇数//2, 偶数)`。

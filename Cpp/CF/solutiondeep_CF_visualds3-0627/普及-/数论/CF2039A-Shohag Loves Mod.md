# 题目信息

# Shohag Loves Mod

## 题目描述

Shohag 有一个整数 $n$，请帮助他找到一个升序序列 $1\le a_1< a_2<\cdots<a_n\le100$，使得对于所有满足 $1\le i < j \le n$ 的数对 $(i,j)$，都满足 $a_i \bmod i\neq a_j \bmod j$。

可以证明，在给定限制下，一定存在合法的序列。

$a\bmod b$ 表示 $a$ 除以 $b$ 的余数。比如，$7\bmod 3=1, 8\bmod 4=0, 69\bmod 10=9$。

## 样例 #1

### 输入

```
2
3
6```

### 输出

```
2 7 8
2 3 32 35 69 95```

## 综合分析与结论  

### 核心算法流程  
1. **构造核心**：每个元素 $a_k$ 满足 $a_k \equiv k-1 \mod k$，确保余数唯一。  
2. **递增保证**：选择 $a_k = k \cdot m + (k-1)$，其中 $m$ 为满足 $a_k > a_{k-1}$ 的最小非负整数。  
3. **边界控制**：所有元素需严格递增且不超过 100。

### 可视化设计思路  
- **动画演示**：  
  - **高亮当前构造的 $a_k$**，显示其 $k$ 和选择的 $m$。  
  - **颜色标记余数**：不同余数用不同颜色区分，突出唯一性。  
  - **步进控制**：展示每个元素如何逐步生成，并检查递增性。  
- **复古像素风格**：  
  - 用 8-bit 网格展示序列构造过程，每个格子代表元素值，底部标注 $k$ 和余数。  
  - **音效触发**：每次成功生成元素播放“确认音”，余数冲突时播放“错误音”。

## 最优思路或技巧提炼  
**关键构造法**：  
- **余数唯一性**：强制每个元素的余数为 $k-1$，利用余数自然递增特性避免重复。  
- **动态调整系数**：对每个 $k$ 选择最小的 $m$ 使 $a_k > a_{k-1}$，确保序列紧凑且不越界。

## 代码实现与核心逻辑  
```cpp
#include <iostream>
using namespace std;

int main() {
    int t, n;
    cin >> t;
    while (t--) {
        cin >> n;
        int last = 0;
        for (int k=1; k<=n; k++) {
            int m = (last - (k-1)) / k + 1; // 最小m满足 a_k > last
            if (m < 0) m = 0;
            int a = k * m + (k-1);
            cout << a << " ";
            last = a;
        }
        cout << endl;
    }
    return 0;
}
```

**核心逻辑解释**：  
- **系数计算**：$m = \left\lfloor \frac{last - (k-1)}{k} \right\rfloor + 1$，确保 $a_k > last$。  
- **余数保证**：$a_k = k \cdot m + (k-1)$ 天然满足 $a_k \equiv k-1 \mod k$。

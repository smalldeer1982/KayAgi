# 题目信息

# GCD vs LCM

## 题目描述

本题有多测。  
给定一个正整数 $n$，求一组正整数 $a$, $b$, $c$, $d$，使得 $a+b+c+d=n$，并且 $\gcd(a,b) = \operatorname{lcm}(c,d)$。本题有 SPJ，求出任意一组即可。

## 说明/提示

$4\le n\le 10^9$  
$1\le t\le 10^4$

## 样例 #1

### 输入

```
5
4
7
8
9
10```

### 输出

```
1 1 1 1
2 2 2 1
2 2 2 2
2 4 2 1
3 5 1 1```

### 综合分析与结论  
#### 核心思路与难点  
**关键思路**：通过构造让 $\gcd(a,b)$ 和 $\operatorname{lcm}(c,d)$ 都为 1，简化问题。  
1. **构造目标**：$\gcd(a,b)=1$ 且 $\operatorname{lcm}(c,d)=1$。  
   - 任何数与 1 的 $\gcd$ 为 1，只需让 $c=1, d=1$，则 $\operatorname{lcm}(c,d)=1$。  
   - 让 $a=1, b=n-3$，此时 $\gcd(1,n-3)=1$，满足条件。  
2. **和为 $n$**：$a+b+c+d = 1 + (n-3) + 1 + 1 = n$。  

**解决难点**：  
- 避免复杂分类讨论，直接固定三个变量为 1，剩余变量为 $n-3$，保证时间复杂度为 $O(1)$。  

#### 可视化设计思路  
1. **动画流程**：  
   - 初始状态：四个空位（a, b, c, d）和当前总和 $0$。  
   - 逐步填入 $c=1$（绿色高亮），$d=1$（绿色高亮），$\operatorname{lcm}(c,d)=1$（显示公式）。  
   - 填入 $a=1$（蓝色高亮），计算剩余 $b=n-3$（红色高亮），验证 $\gcd(a,b)=1$。  
2. **像素风格**：  
   - 使用 8-bit 风格网格，变量用不同颜色方块表示。  
   - 成功时播放经典 FC 过关音效，背景音乐为循环的 8-bit 旋律。

### 最优思路与代码实现  
#### 核心代码片段  
```cpp
printf("1 %d 1 1\n", n-3);
```  
**关键思想**：固定三个变量为 1，剩余变量为 $n-3$，确保 $\gcd$ 和 $\operatorname{lcm}$ 均为 1。  

#### 完整代码  
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int t; cin >> t;
    while (t--) {
        int n; cin >> n;
        cout << 1 << " " << n-3 << " 1 1\n";
    }
    return 0;
}
```

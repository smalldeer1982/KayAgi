# 问题描述

小白有 n 个对手，他每天都要和这些对手PK。对于每一天，如果 n 个对手全部到齐，那么小白就输了一场，否则小白就赢了一场。特别的，如果某天一个对手都没有到，也算小白赢。现在已知对手 d 天的出场情况，请计算小白最多能连胜多少场。

## 综合分析与结论

### 核心算法流程
1. **胜负判断**：逐日检查对手出场情况字符串中是否存在 '0'，存在则视为胜利。
2. **连胜统计**：维护两个变量：当前连胜场次 `current_streak` 和最大连胜场次 `max_streak`。
3. **动态更新**：若当前日胜利，则 `current_streak += 1` 并更新 `max_streak`；否则重置 `current_streak` 为 0。
4. **边界处理**：循环结束后需再比较一次，避免漏判最后连续胜利的情况。

### 可视化设计思路
- **动画方案**：以逐日推进的网格显示每日对手状态（0/1），胜利日高亮为绿色，失败日红色。
- **状态标记**：实时显示当前连胜值和最大连胜值，用动态进度条表示连胜增长。
- **音效交互**：胜利时播放上升音调，失败时播放下降音调，破纪录时触发特殊音效。
- **复古风格**：采用 8-bit 像素风格，用 16 色调色板，胜负状态用不同色块表示。

## 最优思路提炼

### 关键技巧
- **胜负判断优化**：使用字符串内置方法（如 `find`）替代逐字符遍历，提升代码可读性。
- **在线处理**：无需存储所有日期的胜负状态，直接逐日处理，空间复杂度 O(1)。
- **动态更新极值**：类似 Kadane 算法维护当前最优解，时间复杂度 O(dn)。

## 核心代码实现
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n, d, max_streak = 0, current_streak = 0;
    cin >> n >> d;
    while (d--) {
        string s;
        cin >> s;
        if (s.find('0') != string::npos) {
            current_streak++;
            max_streak = max(max_streak, current_streak);
        } else {
            current_streak = 0;
        }
    }
    cout << max_streak << endl;
    return 0;
}
```

# Binary String Constructing

## 题目描述

给你 $3$ 个整数 $a$ ,$b$ ,$x$ ，让你求出一个包含了 $a$ 个 $0$，$b$ 个 $1$ 的二进制串 $s$。要求其中 $s_i\not =s_{i+1}$ 的对数正好等于 $x$。

## 样例 #1

### 输入

```
2 2 1
```

### 输出

```
1100
```

## 样例 #2

### 输入

```
3 3 3
```

### 输出

```
101100
```

## 样例 #3

### 输入

```
5 3 6
```

### 输出

```
01010100
```

## 题解思路与核心难点

### 核心思路
各题解均采用**分阶段构造**策略：
1. 确定较多字符（0或1），生成交替块（如01或10）
2. 处理奇偶性：x为奇数时最后补长段，偶数时先补短段
3. 剩余字符连续输出

### 解决难点
- **交替次数控制**：通过生成x/2个交替块确保基础次数
- **奇偶调整**：通过剩余字符顺序补足x的奇偶差异
- **边界处理**：正确处理字符用完后的拼接顺序

## 最优思路提炼

### 关键策略
1. **统一化处理**：通过swap将问题统一为a≥b的情况
2. **交替块构造**：生成x/2个01块保证基础相邻差异数
3. **奇偶补偿**：
   - x奇：先输出剩余长段（产生1次差异）
   - x偶：先输出短段（无额外差异）

### 代码实现
```cpp
if(a < b) swap(a,b), swap(c1,c2); // 统一处理
for(int i=0; i<x/2; i++) cout<<c1<<c2; // 构造基础块
if(x%2) { // 奇数次补偿
    cout<<string(a, c1) << string(b, c2);
} else {
    cout<<string(b, c2) << string(a, c1);
}
```

## 核心代码示例
```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
    int a, b, x; 
    cin >> a >> b >> x;
    char c1 = '0', c2 = '1';
    
    if(a < b) swap(a,b), swap(c1,c2); // 统一处理
    
    // 构造交替块
    for(int i=0; i<x/2; ++i) {
        cout << c1 << c2;
        a--; b--;
    }
    
    // 处理剩余字符
    if(x%2) {
        cout << string(a, c1) << string(b, c2);
    } else {
        cout << string(b, c2) << string(a, c1);
    }
    return 0;
}
```

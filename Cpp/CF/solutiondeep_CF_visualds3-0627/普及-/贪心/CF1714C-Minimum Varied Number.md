# 题目信息

# Minimum Varied Number

## 题目描述

找出数码和为 $s$ 的最小数字，使得其中的所有数字都是不同的（即所有数字都是唯一的）。

例如，如果 $s=20$ ，那么答案是 $389$。这是最小的数字，其中所有数字都不同，数字的总和为 $20$ （$3+8+9=20$）。

对于给定的 $s$ ，输出这个最小数字。

## 样例 #1

### 输入

```
4
20
8
45
10```

### 输出

```
389
8
123456789
19```

# AI分析结果



### 综合分析与结论

**核心思路**：通过贪心策略从大到小选取数字，确保位数最少，并将结果升序排列形成最小数。  
**难点**：正确性证明（为何从大数开始选最优）和顺序调整（升序排列确保高位最小）。  
**解决方案**：贪心选最大数减少位数，逆序存储后正序输出，形成高位小数。

### 代码实现（核心贪心逻辑）

```cpp
// FQR_ 的贪心实现（关键部分）
string ans;
for(int i=9; i>=1; i--) {
    if(s > i) {
        ans += (i + '0');
        s -= i;
    } else {
        ans += (s + '0');
        break;
    }
}
reverse(ans.begin(), ans.end()); // 逆序转为升序
```

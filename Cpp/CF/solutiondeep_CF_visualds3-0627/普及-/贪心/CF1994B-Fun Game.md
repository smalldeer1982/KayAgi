# 题目信息

# Fun Game

## 题目描述

### 题目大意

Vanya 有一个 01 串 $s=s_1s_2\dots s_n$。他可以对 $s$ 进行一系列变换。每次变换中，Vanya 会取两个整数 $l,r\left(1\le l\le r\le n\right)$，然后对所有 $i\in\left[l,r\right]$，将 $s_i$ **同时**替换为 $s_i\oplus s_{i-L+1}$，其中 $\oplus$ 表示按位异或。

Vanya 想知道 $s$ 能否经过一系列变换，得到另一个长度与 $s$ 相同的 01 串 $t$。

---

2. **题解思路与核心难点**  
   **核心思路**：  
   - 通过异或操作的性质发现，只有在前缀存在 `1` 时才能改变后续位的值。  
   - 关键约束：若 `t` 中某个位置需要修改（与 `s` 不同），则在此位置之前 `s` 必须出现过 `1`。  
   - 特例处理：若 `s` 全为 `0`，则 `t` 必须也为全 `0`。  

   **解决难点**：  
   - 确定哪些位置无法被修改：`s` 的第一个 `1` 之前的前缀无法被改变，必须与 `t` 完全一致。  
   - 通过一次遍历维护“是否已遇到 `1`”的标记，判断是否满足约束条件。

**核心代码片段**（基于 a_girl 的题解）：
```cpp
bool f = 0; // 是否已遇到 s 的 1
for (int i = 0; i < n; i++) {
    if (s[i] == '1') f = 1;
    if (t[i] == '1' && !f) {
        cout << "No\n";
        return;
    }
}
cout << "Yes\n";
```

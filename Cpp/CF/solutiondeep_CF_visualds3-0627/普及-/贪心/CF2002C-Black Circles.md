# 题目信息

# Black Circles

## 题目描述

在一个二维平面上有 $n$ 个圆。第 $i$ 个圆的圆心位于 $(x_i, y_i)$。最初，所有圆的半径都是 $0$。

这些圆的半径以每秒 $1$ 单位的速度增长。

你现在的位置是 $(x_s, y_s)$，你的目标是到达 $(x_t, y_t)$，并且在这个运动过程中不能碰到任何一个圆的边缘（包括你到达 $(x_t, y_t)$ 的那一刻）。你可以向任意方向移动。然而，你的速度被限制在每秒 $1$ 单位。

请判断是否有可能实现这一目标。

## 综合分析与结论  

### 核心思路与难点
1. **核心逻辑**：走直线是唯一可能成功的路径。若存在任意圆心到终点的距离 ≤ 起点到终点的距离，则无法安全到达终点。  
2. **数学推导**：  
   - 到达终点的时间为两点间欧氏距离 $d$，此时所有圆的半径都增长到 $d$。  
   - 若圆心到终点的距离 ≤ $d$，则此时终点已被该圆覆盖。  
3. **优化技巧**：避免浮点运算，直接比较距离平方。  

### 可视化设计思路  
- **动画流程**：  
  1. 绘制起点（绿色）、终点（红色）、圆心（灰色点）和路径（蓝色直线）。  
  2. 动态显示随时间 $t$ 的推移，圆的半径逐步扩大（灰色圆圈渐显）。  
  3. 当人物（移动绿点）抵达终点时，检查是否有圆覆盖终点（红色变黄表示碰撞）。  
- **高亮逻辑**：  
  - 关键比较：圆心到终点的距离平方（红色连线）与起点到终点的距离平方（蓝色连线）。  
  - 失败条件触发时，对应圆变为红色闪烁。  
- **复古像素风格**：  
  - 使用 8-bit 像素画风，路径显示为蓝色方块路径，圆用同心像素环扩散。  
  - 音效：移动时播放 "beep" 音效，碰撞时播放 "boom" 音效，成功时播放胜利旋律。

function drawFrame(t) {  
    // 清空画布  
    ctx.fillStyle = "#000";  
    ctx.fillRect(0, 0, W, H);  

    // 绘制所有圆  
    circles.forEach(circle => {  
        let radius = t * SCALE;  
        drawCircle(circle.x, circle.y, radius);  
    });  

    // 绘制人物位置  
    let progress = Math.min(t / totalTime, 1);  
    let px = start.x + (end.x - start.x) * progress;  
    let py = start.y + (end.y - start.y) * progress;  
    drawPixel(px, py, "#0F0");  

    // 碰撞检测  
    if (t >= totalTime) {  
        checkCollision();  
    }  
}

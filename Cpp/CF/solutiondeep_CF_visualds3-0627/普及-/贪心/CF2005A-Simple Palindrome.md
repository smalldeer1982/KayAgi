# 题目信息

# Simple Palindrome

## 题目描述

构造一个 **长度为** $n$ ，且只由 $a$，$e$，$i$，$o$，$u$ **这 $5$ 种字母** 构成字符串，使得这个字符串的 **所有子序列** 是 **回文串** 的个数最少。请输出这个 **最小值** 。

## 说明/提示

对于 $100\%$ 的测试点，$1\le T\le 100$， $1\le n\le 100$ 。

## 样例 #1

### 输入

```
3
2
3
6```

### 输出

```
uo
iae
oeiiua```

### **综合分析与结论**
题目要求构造由a、e、i、o、u组成的字符串，使得所有子序列中回文的数量最少。通过数学推导和贪心策略得出以下结论：

1. **核心思路**  
   - **回文子序列数目公式**：若字符出现次数为 `k_i`，总回文数为 `Σ(k_i + k_i*(k_i-1)/2)`。
   - **最优策略**：尽可能使字符均匀分布，避免高重复次数。当必须重复时，相邻放置以减少更长的回文。

2. **构造方法**  
   - **n ≤5**：用5个不同字符，回文数为 `n`。
   - **5 < n ≤10**：重复 `m = n-5` 个字符各两次，其余各一次，回文数为 `5 + 2*(n-5)`。
   - **n >10**：重复某个字符多次，回文数为 `15 + 3*(n-10)`。

3. **可视化设计**  
   - **动画演示**：以像素风格展示字符串构造过程，高亮重复字符的相邻位置，动态计算回文数。
   - **音效**：每次添加字符时播放音效，重复字符时触发“警告”音，成功构造时播放胜利音效。

### **代码实现**
```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        string base = "aeiou";
        if (n <= 5) {
            cout << base.substr(0, n) << endl;
        } else {
            string res;
            int m = n - 5;
            // 构造前5个字符各一次
            res = base;
            // 按需重复字符，相邻放置
            for (int i = 0; i < m; i++) {
                res += base[i % 5];
            }
            cout << res << endl;
        }
    }
    return 0;
}
```

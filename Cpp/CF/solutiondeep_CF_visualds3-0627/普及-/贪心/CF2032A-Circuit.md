# Circuit

## 题目描述

Alice 刚才制作了一个有 $n$ 盏灯和 $2n$ 个开关的电路，电路中的每个元件（即每盏灯或每个开关）都有两个状态：开或者关。灯和开关按照下列方式排序：

- 每盏灯连接**两个**开关。
- 每个开关与**一盏**灯连接，且**不知道**灯和开关的连接关系。
- 如果所有开关都处于关闭状态，则所有的灯也会关闭。
- 如果开关状态发生改变（即关变成开或开变成关），则相连的灯也会改变状态。

Alice 把只展示了 $2n$ 盏灯的电路拿给姐姐 Iris 看，并给了她一个谜题：能够开启灯的数量的最小值和最大值分别是多少？

Iris 非常了解她妹妹的小把戏，很快就给出了正确答案。你也能做到吗？

## 综合分析与结论

### 题解思路
题目核心在于将 2n 个开关状态分组成 n 对，每对对应一盏灯。灯亮当且仅当该对的和为奇数（即一个 0 和一个 1）。需找到所有可能分组方式中亮灯数量的最小值和最大值。

**关键推导**：
1. **最大值**：最多有 `min(0的数量, 1的数量)` 对异号（0和1）。
2. **最小值**：若 0 和 1 的数量均为偶数，可全同色配对，最小为 0；否则必存在至少 1 对异号。

### 可视化设计
1. **动画展示**：将开关按数值分组（0和1），用不同颜色区分。动态模拟配对过程：
   - **最大配对**：异色块逐个配对，直到用尽较少的颜色。
   - **最小配对**：若颜色奇偶相同，展示一对异色，其余同色。
2. **像素风格**：用红蓝方块表示 0 和 1，配对时高亮当前操作块，伴随 8-bit 音效。

---

## 最优思路提炼

1. **统计奇偶性**：计算 0 和 1 的总数。
2. **直接公式计算**：
   - 最大值 = `min(0的数量, 1的数量)`
   - 最小值 = `0`（若两者均为偶数）或 `1`（否则）。

## 代码实现

```python
t = int(input())
for _ in range(t):
    n = int(input())
    switches = list(map(int, input().split()))
    c0 = switches.count(0)
    c1 = len(switches) - c0
    max_val = min(c0, c1)
    min_val = 0 if (c0 % 2 == 0 and c1 % 2 == 0) else 1
    print(min_val, max_val)
```

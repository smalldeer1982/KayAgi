# 问题描述

小白有 n 个对手，他每天都要和这些对手PK。对于每一天，如果 n 个对手全部到齐，那么小白就输了一场，否则小白就赢了一场。特别的，如果某天一个对手都没有到，也算小白赢。现在已知对手 d 天的出场情况，请计算小白最多能连胜多少场。

## 样例 #1

### 输入

```
2 2
10
00
```

### 输出

```
2
```


## 综合分析与结论

### 核心思路
题目本质是求最长连续满足条件的子数组长度。通过遍历每一天的对手状态，判断当天是否满足胜利条件（至少一个对手缺席），维护当前连胜次数和最大连胜次数。

### 关键算法流程
1. **每日状态判断**：检查当日字符串是否存在'0'，存在则算胜利。
2. **贪心维护连胜**：若当日胜利则当前连胜+1，否则重置为0。每一步更新最大连胜值。
3. **边界处理**：循环结束后可能残留未比较的连胜值（部分题解需要补一次比较）。

### 算法可视化设计
- **像素动画**：用绿色方块表示胜利日，红色方块表示失败日。每步动态展示当前连胜进度条和最高记录。
- **高亮变化点**：当连胜中断时，将当前连胜进度条用闪烁红色标记。
- **音效反馈**：连胜增长时播放上升音调，打破记录时播放8-bit欢呼音效。
- **自动演示**：以0.5秒/步的速度自动遍历所有天数，可暂停观察关键节点。

## 最优代码实现

### Register版核心代码
```cpp
int maxn=0, sum=0;
while(d--) {
    bool flag = false;
    for(int i=0; i<n; i++){
        cin >> t;
        if(t=='0') flag=true;
    }
    if(flag) sum++, maxn=max(maxn,sum);
    else sum=0;
}
```

### 核心思想
- **即时更新**：每次胜利立即更新最大值，确保实时性
- **空间O(1)**：仅用两个变量维护状态，无需存储历史数据

```html
<canvas id="game" width="600" height="200"></canvas>
<script>
const colors = { win: '#00FF00', lose: '#FF0000', bg: '#000000' };
let streak = 0, maxStreak = 0;

function drawDay(ctx, day, x, isWin) {
    ctx.fillStyle = isWin ? colors.win : colors.lose;
```

# Splits

## 题目描述

我们定义正整数$n$的分裂为一个由正整数组成的不上升序列，且序列数字和为$n$

举个栗子：下列这些序列都是$8$的分裂：$[4,4],[3,3,2],[2,2,1,1,1],[5,2,1]$

下列这些序列不是$8$的分裂：$[1,7],[5,4],[11,-3],[1,1,4,1,1]$

一个分裂的权是序列第一个数出现的次数，举个例子：$[1,1,1,1,1]$的权是$5$，$[5,5,3,3,3]$的权是$2$，$[9]$的权是$1$

现在给出$n$，求$n$的分裂有多少个不同的权

## 样例 #1

### 输入

```
7
```

### 输出

```
4
```

## 样例 #2

### 输入

```
8
```

### 输出

```
5
```

## 样例 #3

### 输入

```
9
```

### 输出

```
5
```

### **综合分析与结论**

**题解思路与核心难点**  
题目要求计算所有可能的分裂权值的数目。权值定义为序列中第一个数的出现次数。关键点在于发现权值的可能取值仅与构造方式有关，通过贪心策略拆分为最多可能的2和1的组合，权值数目为 `floor(n/2) + 1`。

**核心算法流程**  
1. **贪心构造**：用尽可能多的2构造分裂，剩下的用1填充。例如，n=7拆分为`[2,2,2,1]`，权值为3。
2. **权值范围**：构造的权值为`1`到`floor(n/2)`，再加上全1序列的权值n。
3. **公式推导**：总权值数目为`floor(n/2) + 1`。

# Creating Words

## 题目描述

Matthew 得到了两个字符串 $a$ 和 $b$，它们的长度均为 $3$。他觉得把 $a$ 的第一个字符和 $b$ 的第一个字符交换来生成两个新单词非常有趣。请你输出交换后的 $a$ 和 $b$。

注意，交换后得到的新单词不一定不同。

## 4. C++核心代码实现赏析

<code_intro_overall>
下面是一个**完整的、能解决本题的C++核心实现**——逻辑清晰，符合题目要求，适合新手参考！

**本题通用核心C++实现参考**
* **说明**：本代码是“模拟交换过程”的典型实现，覆盖了多组输入、字符交换、结果输出的全流程。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string> // 处理字符串需要的头文件
    using namespace std;

    int main() {
        int n; // 测试用例数量
        cin >> n; // 读入n
        while (n--) { // 循环n次，每次处理一组数据
            string a, b; // 两个长度为3的字符串
            cin >> a >> b; // 读入a和b
            // 交换第一个字符
            char temp = a[0];
            a[0] = b[0];
            b[0] = temp;
            // 输出结果
            cout << a << " " << b << endl;
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码的核心逻辑很直白：① 先读入要处理的“组数n”；② 用`while(n--)`循环n次（每次处理一组）；③ 读入当前组的a和b；④ 交换a和b的第一个字符（用临时变量temp）；⑤ 输出交换后的a和b。

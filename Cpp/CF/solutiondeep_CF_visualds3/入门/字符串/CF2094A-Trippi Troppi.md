# Trippi Troppi

## 题目描述

Trippi Troppi 居住在一个奇异的世界里。每个国家的古代名称由三个字符串组成，将每个字符串的首字母连接起来就形成了该国家的现代名称。

给定一个国家的古代名称，请输出其现代名称。

## 样例 #1

### 输入

```
7
united states america
oh my god
i cant lie
binary indexed tree
believe in yourself
skibidi slay sigma
god bless america```

### 输出

```
usa
omg
icl
bit
biy
sss
gba```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
虽然这道题看起来简单，但新手容易在三个关键点上“卡壳”。结合字符串处理的通用技巧，Kay帮大家梳理了解决策略：
</difficulty_intro>

1.  **关键点1**：如何正确读取多组输入？
    * **分析**：题目第一行是输入的组数`n`，之后有`n`行，每行3个字符串。用`cin`读取最方便——因为`cin`会自动跳过空格和换行符。比如先读`n`，然后循环`n`次，每次读三个`string`变量（比如`s1, s2, s3`）。
    * 💡 **学习笔记**：`cin`读取字符串时，默认以 whitespace（空格、换行）为分隔符，刚好适合本题的输入格式！

2.  **关键点2**：如何提取字符串的首字母？
    * **分析**：C++中`string`是字符数组的封装，用`s[0]`就能拿到第一个字符（注意：字符串不能为空哦，但题目保证输入是三个有效字符串）。比如`s1[0]`就是第一个单词的首字母。
    * 💡 **学习笔记**：`string`的索引从0开始，`s[i]`对应第`i+1`个字符~

3.  **关键点3**：如何拼接首字母并输出？
    * **分析**：直接把三个首字母用`cout`连起来就行！比如`cout << s1[0] << s2[0] << s3[0] << endl;`。注意不需要额外的变量，直接输出更简洁。
    * 💡 **学习笔记**：`cout`可以连续输出多个字符，就像“串珠子”一样把首字母串起来~

## 4. C++核心代码实现赏析

<code_intro_overall>
这道题的核心代码非常简洁，我们直接看一个**通用且完整的实现**——它覆盖了所有输入输出逻辑，而且极易理解！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码是字符串处理的典型入门实现，逻辑直接，适合新手模仿学习。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    using namespace std;

    int main() {
        int n;
        cin >> n; // 读取组数
        while (n--) { // 循环n次处理每组输入
            string s1, s2, s3;
            cin >> s1 >> s2 >> s3; // 读三个字符串
            cout << s1[0] << s2[0] << s3[0] << endl; // 输出首字母拼接
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分为三步：①读入组数`n`；②循环`n`次，每次读三个字符串；③直接输出三个字符串的首字母。核心逻辑就是“读→取→输”，没有多余的步骤，非常高效！

# Word Capitalization

## 题目描述

Capitalization is writing a word with its first letter as a capital letter. Your task is to capitalize the given word.

Note, that during capitalization all the letters except the first one remains unchanged.

## 样例 #1

### 输入

```
ApPLe
```

### 输出

```
ApPLe
```

## 样例 #2

### 输入

```
konjac
```

### 输出

```
Konjac
```

- **题解思路**：所有题解的核心逻辑高度一致：① 读入字符串 → ② 处理首字符（小写→大写） → ③ 输出字符串。

## 3. 核心难点辨析与解题策略

在解决字符串问题时，这3个“坑”你一定遇到过！结合优质题解，我帮你总结了“避坑指南”：

1. **关键点1：字符串的索引从0开始！**  
   * **分析**：C++里的字符串（不管是`string`还是字符数组），第一个字符的位置是`[0]`，不是`[1]`！比如输入“konjac”，`s[0]`是`'k'`，`s[1]`是`'o'`。优质题解都明确用了`[0]`，比如Luban的`s[0]`、Trans_Portal的`a[0]`。  
   * 💡 **学习笔记**：记不住索引？想成“排队时第一个人站在‘第0位’”！

2. **关键点2：转大写的两种正确方式！**  
   * **分析**：转大写有两种方法：① 小写字母减32（`s[0]-=32`）——因为`'a'`的ASCII码是97，`'A'`是65，差32；② 用`toupper()`函数（`s[0]=toupper(s[0])`）——这个函数会自动处理所有情况（比如输入已经是大写，就不变）。优质题解里，Trans_Portal用了`toupper()`，Luban用了`+'A'-'a'`（和减32一样），都是正确的。  
   * 💡 **学习笔记**：怕算错ASCII？直接用`toupper()`！

3. **关键点3：输入方式要选对！**  
   * **分析**：读入字符串可以用`cin>>s`（适合`string`或字符数组）、`getchar()`（逐字符读入）、`scanf("%s", b)`（字符数组）。优质题解里，Luban用`cin>>s`（最简单），紫金依旧用`getchar()`（处理换行符更灵活），都是可行的。但要注意：`cin>>s`会自动跳过空格和换行，适合读“单个单词”（本题正好是单个单词）。  
   * 💡 **学习笔记**：读单个单词用`cin>>s`，读整行用`getline(cin, s)`！


### ✨ 解题技巧总结  
- **技巧A：优先用`string`类**：不用手动算长度，操作更简单。  
- **技巧B：用库函数偷懒**：`toupper()`、`tolower()`这些函数能帮你少写代码，减少错误。  
- **技巧C：测试边界情况**：比如输入已经是大写（像样例1的“ApPLe”），或者输入只有一个字符（比如“a”→“A”），要确保代码能正确处理！


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考  
* **说明**：综合了Luban和Trans_Portal的思路，用`string`和`toupper()`函数，是最简洁、最易读的实现。  
* **完整核心代码**：  
```cpp
#include <iostream>
#include <cctype> // 包含toupper()函数的头文件
using namespace std;

int main() {
    string s;
    cin >> s;               // 读入字符串
    s[0] = toupper(s[0]);   // 首字符转大写
    cout << s << endl;      // 输出结果
    return 0;
}
```
* **代码解读概要**：  
  1. 用`string s`定义字符串（不用管长度，自动扩容）；  
  2. `cin >> s`读入用户输入的单词；  
  3. `toupper(s[0])`把首字符转大写（不管原来是不是大写）；  
  4. `cout << s`输出处理后的字符串。


### 针对各优质题解的片段赏析

#### 题解一：Luban的string写法  
* **亮点**：用`string`类简化代码，转大写的方式直观。  
* **核心代码片段**：  
```cpp
string s;
cin >> s;
if (s[0] >= 'a' && s[0] <= 'z') 
    s[0] += 'A' - 'a'; // 等价于s[0] -= 32
cout << s << endl;
```
* **代码解读**：  
  - `'A' - 'a'`是多少？`'A'`的ASCII是65，`'a'`是97，所以差是`-32`？不对！等一下，`s[0] += 'A' - 'a'`其实是`s[0] = s[0] + ('A' - 'a')`——比如`s[0]`是`'k'`（97+10=107），`'A'-'a'`是`-32`，所以107-32=75，正好是`'K'`！哦，原来如此，这样写比直接减32更直观（告诉你是“从a转到A”）。  
* 💡 **学习笔记**：`'A'-'a'`是转大写的“魔法数”，记住它！

#### 题解二：Trans_Portal的toupper函数  
* **亮点**：用库函数一步解决转大写，代码极简。  
* **核心代码片段**：  
```cpp
char a[1001];
cin >> a;
a[0] = toupper(a[0]);
cout << a;
```
* **代码解读**：  
  - `toupper()`函数在头文件`<cctype>`里（有些编译器可能自动包含，但最好加上）；  
  - 不管`a[0]`是大写还是小写，`toupper(a[0])`都会返回正确的大写形式——比如`a[0]`是`'A'`，返回`'A'`；是`'k'`，返回`'K'`。  
* 💡 **学习笔记**：库函数是“偷懒神器”，但要记得包含对应的头文件！

#### 题解三：Higashikata_Jousuke的字符数组提醒  
* **亮点**：特意提醒“输入默认在`a[0]`”，帮新手避坑。  
* **核心代码片段**：  
```cpp
char a[10001];
cin >> a;
if (a[0] >= 'a' && a[0] <= 'z')
    a[0] -= 32; // 直接减32转大写
cout << a;
```
* **代码解读**：  
  - 字符数组`a[10001]`足够存下题目中的输入（题目没说长度，但10001足够）；  
  - `cin >> a`会把输入的字符串存到`a`里，首字符在`a[0]`；  
  - 条件判断`a[0] >= 'a' && a[0] <= 'z'`确保只有小写才转大写。  
* 💡 **学习笔记**：字符数组的首字符永远是`[0]`，别记成`[1]`！

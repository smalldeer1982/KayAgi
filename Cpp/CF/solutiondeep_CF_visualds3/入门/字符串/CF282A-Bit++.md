# Bit++

## 题目描述

The classic programming language of Bitland is Bit++. This language is so peculiar and complicated.

The language is that peculiar as it has exactly one variable, called $ x $ . Also, there are two operations:

- Operation ++ increases the value of variable $ x $ by 1.
- Operation -- decreases the value of variable $ x $ by 1.

A statement in language Bit++ is a sequence, consisting of exactly one operation and one variable $ x $ . The statement is written without spaces, that is, it can only contain characters "+", "-", "X". Executing a statement means applying the operation it contains.

A programme in Bit++ is a sequence of statements, each of them needs to be executed. Executing a programme means executing all the statements it contains.

You're given a programme in language Bit++. The initial value of $ x $ is $ 0 $ . Execute the programme and find its final value (the value of the variable when this programme is executed).

## 样例 #1

### 输入

```
1
++X
```

### 输出

```
1
```

## 样例 #2

### 输入

```
2
X++
--X
```

### 输出

```
0
```

## 1. 题目解读与核心算法识别  

✨ **本题主要考察**：模拟（或「编程技巧应用」）  

🗣️ **初步分析**：  
Bit++语言只有一个变量`x`（初始值0），每条语句是`++X`、`X++`、`--X`、`X--`中的一种。我们需要**模拟每条语句的执行**，最后输出`x`的最终值。  

模拟的核心就像「按指令做事」——比如妈妈说“拿苹果”，你就去拿苹果；题目说“++X”，你就把`x`加1。题解的思路分两类：  
- **直接判断**：检查字符串是否是`++X`或`X++`（加1），否则减1；  
- **找规律简化**：不管`++`在`X`前还是后，**第二个字符都是`+`**（比如`++X`的第2位是`+`，`X++`的第2位也是`+`）；同理，`--X`和`X--`的第2位都是`-`。只需判断第2位就能快速确定操作！  

**核心算法流程**：读入语句数量`n`→循环`n`次→每次读字符串→判断第2位是`+`则`x++`，否则`x--`→输出`x`。  

**可视化设计思路**：我们会做一个「像素小助手的加减任务」动画（FC红白机风格）——用像素数字显示`x`的值，加1时数字闪烁绿色+“叮”音效，减1时闪烁红色+“咚”音效。控制面板支持「单步执行」「自动播放」，帮你直观看到`x`的变化！

## 3. 核心难点辨析与解题策略  

### 核心难点与解决方法  
1. **难点1：如何快速判断操作类型？**  
   - 分析：直接判断四个字符串太麻烦！找规律——`++X`和`X++`的第2位都是`+`，`--X`和`X--`的第2位都是`-`，只需判断`S[1]`即可。  
   - 💡 学习笔记：找规律能**简化问题**，避免冗余判断！  

2. **难点2：字符串的下标处理？**  
   - 分析：C++中`string`的下标从**0**开始！比如`"++X"`的第0位是`+`，第1位是`+`，第2位是`X`。不要把第2位写成`S[2]`哦！  
   - 💡 学习笔记：字符串下标从0开始，记牢！  

3. **难点3：输入的循环处理？**  
   - 分析：需要读入`n`条语句，用`for`循环`n`次，每次用`cin >> S`读字符串（自动跳过空格/换行）。  
   - 💡 学习笔记：循环处理输入要正确，避免多余的空格干扰！  

### ✨ 解题技巧总结  
- 技巧1：**找规律简化判断**（比如本题的“第2位字符”规律）；  
- 技巧2：**明确变量命名**（比如用`ans`或`x`代表结果，避免混淆）；  
- 技巧3：**简洁的循环结构**（避免嵌套过深，保持代码可读性）。  


## 4. C++核心代码实现赏析  

### 本题通用核心C++实现参考  
* **说明**：综合了Bronya18C和紫金依旧的思路，是**最简洁高效的实现**。  
* **完整核心代码**：  
  ```cpp
  #include <iostream>
  #include <string>
  using namespace std;

  int main() {
      int n;
      long long ans = 0; // 初始值0
      cin >> n;
      for (int i = 0; i < n; ++i) {
          string s;
          cin >> s;
          if (s[1] == '+') ans++; // 第2位是+，加1
          else ans--;             // 否则减1
      }
      cout << ans << endl;
      return 0;
  }
  ```  
* **代码解读概要**：  
  1. 读入语句数量`n`，初始化`ans`为0（`x`的初始值）；  
  2. 循环`n`次，每次读一个字符串`s`；  
  3. 判断`s`的第2位（`s[1]`）是`+`则`ans++`，否则`ans--`；  
  4. 输出最终的`ans`。

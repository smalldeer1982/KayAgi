# 题目信息

# Way Too Long Words

## 题目描述

有时候，像“localization”或“internationalization”这样的词汇如此之长，以至于在一篇文章中多次写下这些词语令人厌烦。

让我们考虑一个词太长，如果它的长度严格超过 10个字符。太长的单词应该用一个特殊的缩写代替。

这个缩写是这样写的：我们写下一个单词的第一个和最后一个字母，并在它们之间写出第一个和最后一个字母之间的字母数。该数字是十进制系统，不包含任何前导零。

因此，“localization”将拼写为“l10n”，而“internationalization”将拼写为”i18n”。

建议您使用缩写来自动更改单词的过程。因为所有太长的单词应该用缩写代替，不太长的单词不应该经历任何改变。

## 样例 #1

### 输入

```
4
word
localization
internationalization
pneumonoultramicroscopicsilicovolcanoconiosis
```

### 输出

```
word
l10n
i18n
p43s
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题看似简单，但新手容易在「细节」上栽跟头。结合题解的共性，我帮你提炼了3个核心难点和解决方法：
</difficulty_intro>

1.  **关键点1：如何正确获取字符串长度？**
    * **分析**：C++中字符串有两种存储方式——`string`类和`char`数组。`string`类用`size()`或`length()`（两者完全一样）；`char`数组要用`strlen()`函数（需要包含`<cstring>`头文件）。比如题解中的`a.length()`（`string`）、`strlen(b)`（`char`数组）都是正确的做法。
    * 💡 **学习笔记**：选对字符串类型，长度获取就成功了一半！

2.  **关键点2：如何访问首尾字符？**
    * **分析**：字符串的字符是「按顺序排列」的，第一个字符的索引是0（比如`string s = "word"`，`s[0]`是'w'），最后一个字符的索引是「长度-1」（`s[s.length()-1]`是'd'）。如果用`char`数组，逻辑一样（`b[0]`是首字符，`b[strlen(b)-1]`是尾字符）。
    * 💡 **学习笔记**：索引从0开始，别把最后一个字符写成`s[length()]`哦！

3.  **关键点3：如何处理多组输入？**
    * **分析**：题目要求输入n个单词，所以需要用循环——比如`for(int i=0; i<n; i++)`，每次循环读入一个单词并处理。注意循环的起始和结束条件（比如从0到n-1，或从1到n，只要次数对就行）。
    * 💡 **学习笔记**：多组输入的核心是「循环n次」，别漏读或多读！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份「通用核心代码」——它综合了所有优质题解的优点，逻辑清晰、代码简洁，适合直接套用！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自「封禁用户」的题解，调整了变量名使其更规范，保留了最核心的逻辑。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>  // 处理string需要的头文件
    using namespace std;

    int main() {
        int word_count;  // 单词个数
        cin >> word_count;
        
        for (int i = 0; i < word_count; ++i) {
            string current_word;  // 当前要处理的单词
            cin >> current_word;
            
            if (current_word.length() > 10) {
                // 输出首字符 + 中间字符数 + 尾字符
                cout << current_word[0] 
                     << current_word.length() - 2 
                     << current_word[current_word.length() - 1] 
                     << endl;
            } else {
                cout << current_word << endl;
            }
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分三步：① 读入要处理的单词个数`word_count`；② 循环`word_count`次，每次读入一个单词`current_word`；③ 判断单词长度——超过10就缩写，否则原样输出。核心逻辑全在`if-else`里，像“开关”一样简单！

* 💡 **学习笔记**：模拟题的核心就是「把题目规则写成代码」，别加多余的逻辑！

**题解二：(来源：SKTelecomT1_Faker)**
* **亮点**：优化循环变量，加`register`加速。
* **核心代码片段**：
    ```cpp
    for(register int i=1;i<=n;i++)//做循环方便输入
    {
        string s;//定义字符串
        cin>>s;//输入
        if(s.size()>10) cout<<s[0]<<s.size()-2<<s[s.size()-1]<<endl;
        else cout<<s<<endl;//无脑输出
    }
    ```
* **代码解读**：
    > 作者用了`register int i`——`register`关键字会让编译器把变量存到CPU寄存器里，循环次数多的时候能变快！虽然这道题数据小看不出差别，但养成这个习惯，以后处理大数据会很有用～
* 💡 **学习笔记**：小优化能让代码更高效，哪怕现在用不上，也要知道有这个技巧！

**题解三：(来源：PC_DOS)**
* **亮点**：输入输出加速，适合大数据。
* **核心代码片段**：
    ```cpp
    ios::sync_with_stdio(false); //输入输出加速
    int nCount, iLength;
    string sInput;
    cin >> nCount;
    for (i = 1; i <= nCount; ++i){
        cin >> sInput;
        iLength = sInput.length();
        if (iLength > 10)
            cout << sInput[0] << iLength - 2 << sInput[iLength - 1] << endl;
        else
            cout << sInput << endl;
    }
    ```
* **代码解读**：
    > 第一行`ios::sync_with_stdio(false)`是C++的「输入输出加速神器」！默认情况下，C++的`cin/cout`会和C的`scanf/printf`同步，导致速度慢。加了这行，`cin/cout`的速度会提升好几倍——处理10万级数据时超有用！
* 💡 **学习笔记**：处理大数据时，记得加输入输出加速！

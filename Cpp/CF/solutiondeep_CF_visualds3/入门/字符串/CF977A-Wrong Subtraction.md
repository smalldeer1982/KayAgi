# 题目信息

# Wrong Subtraction

## 题目描述

Little girl Tanya is learning how to decrease a number by one, but she does it wrong with a number consisting of two or more digits. Tanya subtracts one from a number by the following algorithm:

- if the last digit of the number is non-zero, she decreases the number by one;
- if the last digit of the number is zero, she divides the number by 10 (i.e. removes the last digit).

You are given an integer number $ n $ . Tanya will subtract one from it $ k $ times. Your task is to print the result after all $ k $ subtractions.

It is guaranteed that the result will be positive integer number.

## 说明/提示

The first example corresponds to the following sequence: $ 512 \rightarrow 511 \rightarrow 510 \rightarrow 51 \rightarrow 50 $ .

## 样例 #1

### 输入

```
512 4
```

### 输出

```
50
```

## 样例 #2

### 输入

```
1000000000 9
```

### 输出

```
1
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
虽然这题简单，但第一次做时很容易踩「小坑」～ 我总结了3个最常见的难点，帮你提前避开！
</difficulty_intro>

1.  **关键点1：如何处理「大数溢出」？**
    * **分析**：题目里`n`可以大到`1e9`，如果用`int`（最多装`2e9`左右）看似够，但如果`n`更大（比如`1e18`），`int`就会「爆掉」！优质题解给了两个方案：① 用`long long`（能装到`9e18`）；② 用字符串（能装任意长度的数字）。  
    * 💡 **学习笔记**：遇到「可能很大的数」，先想`long long`，再想字符串！

2.  **关键点2：如何正确判断「最后一位」？**
    * **分析**：判断最后一位的秘诀是`n%10`——比如`512%10=2`（非0），`510%10=0`。但要注意：如果用字符串，就是取最后一个字符（`st.back()`或`st[st.size()-1]`）。  
    * 💡 **学习笔记**：数字的最后一位用`%10`，字符串的最后一位用「索引到末尾」！

3.  **关键点3：如何避免「循环中的错误操作」？**
    * **分析**：比如当最后一位是0时，我们要「除以10」，而不是「先减1再除以10」！题解里用`if-else`或`continue`避免了这个错误——比如thename的题解中，`continue`跳过了后面的`n--`，保证「除以10」后不会多减1。  
    * 💡 **学习笔记**：模拟题的核心是「按顺序执行规则」，千万不要「多做或少做」！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份「通用核心代码」——它综合了优质题解的优点，简洁又正确，适合直接套用！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：这份代码来自「引领天下」的题解，修正了输入输出的格式错误（原代码用`%d`读`long long`会出错），是最简洁的「模拟版」。
* **完整核心代码**：
    ```cpp
    #include <cstdio>
    long long n, k; // 用long long装大数
    int main() {
        scanf("%lld%lld", &n, &k); // 注意用%lld读long long
        while (k--) { // 循环k次，每次k减1
            if (n % 10) { // 最后一位非0（n%10≠0）
                n--; // 减1
            } else {
                n /= 10; // 除以10（删最后一位）
            }
        }
        printf("%lld\n", n); // 输出结果
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分三步：① 读入`n`和`k`；② 循环`k`次，每次按规则修改`n`；③ 输出结果。核心是`while(k--)`（循环`k`次）和`n%10`（判断最后一位）——这两行就把题目中的规则「翻译」成了代码！

**题解一：(来源：引领天下)**
* **亮点**：用`long long`处理大数，`while(k--)`简洁循环。
* **核心代码片段**：
    ```cpp
    while (k--) {
        if (n % 10) n--;
        else n /= 10;
    }
    ```
* **代码解读**：
    > ① `while(k--)`：先判断`k`是否大于0，然后`k`减1——相当于「做k次操作」。  
    > ② `n%10`：如果结果非0（比如`512%10=2`），就执行`n--`（512→511）；如果结果是0（比如`510%10=0`），就执行`n/=10`（510→51）。  
    > 这几行代码把题目中的规则「浓缩」成了最简洁的形式，超高效！
* 💡 **学习笔记**：`while(k--)`是「循环k次」的常用写法，比`for`更简洁！

**题解二：(来源：666yuchen)**
* **亮点**：用字符串处理超大数，避免溢出。
* **核心代码片段**：
    ```cpp
    string st;
    int n;
    cin >> st >> n;
    for (int i = 1; i <= n; i++) {
        if (st[st.size()-1] == '0') {
            st.erase(st.size()-1, 1); // 删最后一位
        } else {
            st[st.size()-1]--; // 最后一位减1（字符减1，比如'2'→'1'）
        }
    }
    ```
* **代码解读**：
    > ① `st.size()-1`：取字符串的最后一个字符的索引（比如`"512"`的最后一位是`st[2]`）。  
    > ② `st.erase(st.size()-1, 1)`：从最后一位开始，删1个字符（比如`"510"`→`"51"`）。  
    > ③ `st[st.size()-1]--`：字符减1——比如`'2'`的ASCII码是50，减1变成49（即`'1'`），正好对应数字减1！  
    > 这个方法超适合处理「比`long long`还大的数」，比如`1e20`，字符串能轻松装下～
* 💡 **学习笔记**：字符串是处理「超大数」的神器！

**题解三：(来源：thename)**
* **亮点**：用`for`循环+`continue`，逻辑清晰适合萌新。
* **核心代码片段**：
    ```cpp
    for (; k > 0; k--) {
        if (n % 10 == 0) {
            n /= 10;
            continue; // 跳过后面的n--
        }
        n--;
    }
    ```
* **代码解读**：
    > ① `for (; k > 0; k--)`：循环条件是`k>0`，每次循环`k`减1——和`while(k--)`效果一样，但更直观。  
    > ② `continue`：当最后一位是0时，执行`n/=10`后，直接跳过后面的`n--`——避免「先除以10再减1」的错误！  
    > 这个写法把「规则的顺序」写得明明白白，萌新看了也不会乱～
* 💡 **学习笔记**：`continue`能帮你「跳过不需要的操作」，避免逻辑错误！

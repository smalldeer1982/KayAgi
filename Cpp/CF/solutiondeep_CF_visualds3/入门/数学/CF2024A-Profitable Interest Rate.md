# 题目信息

# Profitable Interest Rate

## 题目描述

Alice 有 $a$ 个硬币。她可以开设一个名为“有利”的银行存款账户，但开设该账户的最低金额要求是 $b$ 个硬币。

还有一种名为“无利”的存款账户，可以用任意数量的硬币开设。Alice 注意到，如果她使用 $x$ 个硬币开设“无利”账户，开设“有利”账户所需的最低金额会减少 $2x$ 个硬币。然而，这些硬币之后不能再存入“有利”账户。

请帮助 Alice 确定她在先向“无利”账户存入一些硬币（可能为 $0$ ）后，最多能向“有利”账户存入多少硬币。如果 Alice 永远无法开设“有利”账户，则输出 $0$。

## 说明/提示

在第一个测试用例中，$a \geq b$，所以 Alice 可以直接用全部的 10 个硬币开设“有利”账户。

在第二个测试用例中，Alice 可以用 2 个硬币开设“无利”账户。然后她剩下 5 个硬币，而开设“有利”账户的最低要求将减少 4 个硬币，变为 5 个硬币。因此，Alice 将能够用 5 个硬币开设“有利”账户。

在第三个测试用例中，Alice 无法开设“有利”账户。

## 样例 #1

### 输入

```
5
10 5
7 9
5 100
1 1
1 2```

### 输出

```
10
5
0
1
0```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`数学推导`  

🗣️ **初步分析**：  
解决这道题的核心是**找到“存入无利账户的最小金额”**——因为存入有利账户的金额等于总硬币数减去无利账户的存款（`s = a - x`），所以要让`s`最大，就得让`x`尽可能小。  

那`x`的最小值怎么算？我们需要满足两个条件：  
1. 存入`x`后，有利账户的最低要求`b-2x`不能超过剩余硬币数`a-x`（否则开不了户）；  
2. `x`不能是负数（总不能从无利账户“取钱”吧～）。  

把这两个条件转化为数学式子，就能得到`x`的最小值：`x_min = max(0, b - a)`（如果`b ≤ a`，`x`可以是0；如果`b > a`，`x`至少要等于`b - a`才能让`b-2x ≤ a-x`）。  

接下来只要检查`x_min`是否≤总硬币数`a`（毕竟不能存比总硬币还多的钱到无利账户）：如果满足，最大有利存款就是`a - x_min`；否则输出0（永远开不了有利账户）。

  关键是**不要怕写数学式子**——把文字条件“翻译”成代数符号，问题往往会变得很简单！


## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的难点在于“把文字条件转化为数学逻辑”，我总结了3个核心思考点：
</difficulty_intro>

1.  **关键点1：如何将“开户条件”转化为数学式子？**  
    * **分析**：题目说“存x到无利后，有利的最低要求减少2x，且剩余硬币≥新的最低要求”。翻译过来就是`b - 2x ≤ a - x`（新的最低要求≤剩余硬币）。再加上`x≥0`（不能存负数），就能得到`x_min = max(0, b - a)`。  
    * 💡 **学习笔记**：文字条件→数学式子是解决这类题的第一步！

2.  **关键点2：为什么“最小化x”能得到最大有利存款？**  
    * **分析**：因为有利存款`s = a - x`——`x`越小，`s`越大（比如总硬币是7，x=2时s=5，x=3时s=4，显然x越小越好）。  
    * 💡 **学习笔记**：目标函数的单调性决定了极值的方向！

3.  **关键点3：如何验证推导的正确性？**  
    * **分析**：用样例数据测试！比如样例2中`a=7`、`b=9`，`x_min = max(0,9-7)=2`，`s=7-2=5`，和样例输出一致；样例3中`a=5`、`b=100`，`x_min=95`>5，输出0，正确。  
    * 💡 **学习笔记**：样例是验证推导的“试金石”！


### ✨ 解题技巧总结
- **技巧1：目标函数转化**：把“最大化有利存款”转化为“最小化无利存款”，简化问题。  
- **技巧2：条件代数化**：将题目中的“开户要求”“金额限制”等文字转化为不等式，解出变量范围。  
- **技巧3：样例验证**：用题目给的样例测试推导的公式，确保正确性。  


## 4. C++核心代码实现赏析

<code_intro_overall>
这道题的代码非常简洁——核心逻辑就是计算`x_min`并判断条件。我们先看通用实现：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了数学推导的核心逻辑，直接计算最优解，适用于所有测试用例。  
* **完整核心代码**：
  ```cpp
  #include <iostream>
  #include <algorithm> // 用于std::max函数

  using namespace std;

  int main() {
      int t; // 测试用例数
      cin >> t;
      while (t--) {
          int a, b; // a:总硬币数，b:有利账户初始最低要求
          cin >> a >> b;
          int x_min = max(0, b - a); // 无利账户的最小存款
          if (x_min > a) { // 无法满足条件（存的钱超过总硬币）
              cout << 0 << endl;
          } else {
              cout << a - x_min << endl; // 最大有利存款
          }
      }
      return 0;
  }
  ```
* **代码解读概要**：  
  1. 读取测试用例数`t`；  
  2. 循环处理每个测试用例，读取`a`和`b`；  
  3. 计算`x_min`（无利账户的最小存款）；  
  4. 判断`x_min`是否超过总硬币数`a`：如果是，输出0；否则输出`a - x_min`（最大有利存款）。

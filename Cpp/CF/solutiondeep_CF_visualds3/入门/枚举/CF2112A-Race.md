# 题目信息

# Race

## 题目描述

Alice 和 Bob 正在参加一场电视节目的游戏。游戏开始的时候，奖品会被放在某个点上，第一个到达奖品所在位置的人将获得奖品。

Alice 决定她将从点 $a$ 起跑。Bob 还没有决定他的出发点。

Bob 知道奖品会落在点 $x$ 或点 $y$ 处，他可以在 Alice 之前到达奖品的位置当且仅当他的出发点和奖品的距离严格小于 Alice 的出发点和奖品的距离。点 $c$ 和点 $d$ 之间的距离是 $\vert c-d\vert$。

你的任务是回答 Bob 能不能选出一个整点，使得无论奖品是在点 $x$ 还是点 $y$ 处出现，都能保证他比 Alice 先到达奖品处。他可以选择除了 $a$ 以外的所有整点，包括 $x$ 和 $y$。

## 说明/提示

**样例解释**

对于第一组数据，Bob 可以选择点 $4$，如果奖品出现在点 $x$，Bob 和它之间的距离为 $\vert 4-3\vert=1$，Alice 和它之间的距离为 $\vert 1-3\vert=2$。如果奖品出现在点 $y$，Bob 和它之间的距离为 $\vert 4-4\vert=0$，Alice 和它之间的距离为 $\vert 1-4\vert=3$。

对于第二组数据，Bob 可以选择点 $2$，如果奖品出现在点 $x$，Bob 和它之间的距离为 $\vert 2-3\vert=1$，Alice 和它之间的距离为 $\vert 5-3\vert=2$。如果奖品出现在点 $y$，Bob 和它之间的距离为 $\vert 2-1\vert=1$，Alice 和它之间的距离为 $\vert 5-1\vert=4$。

对于第三组数据，Bob 无法选择一个点保证自己胜利。


## 样例 #1

### 输入

```
3
1 3 4
5 3 1
3 1 5```

### 输出

```
YES
YES
NO```

#### 1. 题目解读与核心算法识别
✨ **本题主要考察**：`区间位置判断`（编程技巧应用）  
🗣️ **初步分析**：  
> 解决"Race"题目的核心在于**几何位置关系的快速判断**。想象数轴上有三个关键点：Alice的起点`a`，以及奖品可能出现的两个位置`x`和`y`。Bob需要找到一个"魔法位置"（除`a`外的任意整数点），使得无论奖品出现在`x`或`y`，他都比Alice更接近奖品。  
> - **关键洞察**：当Alice位于`x`和`y`构成的区间内部时（含端点），Bob永远无法同时满足两个距离条件；若Alice在区间外，则存在无数符合条件的点。  
> - **可视化设计**：我们将用**8位像素风格**的数轴动画展示位置关系。数轴上用不同颜色像素块标记`a`、`x`、`y`，动态绘制区间范围。当`a`进入区间时触发"禁止通行"音效；当`a`在区间外时，显示Bob可选区域（绿色闪烁区域）并播放"可行"音效。

#### 2. 精选优质题解参考
**题解：区间位置判断法（综合推导）**  
* **点评**：此解法直接抓住问题本质——**Alice位置与奖品区间的几何关系**。思路极为清晰：通过比较`a`与`min(x,y)`和`max(x,y)`的关系直接得出结论。代码实现简洁高效（时间复杂度`O(1)`），变量命名直观（`L`/`R`表示区间边界），边界处理严谨（包含端点值）。亮点在于**用基础数学替代复杂计算**，避免无意义的遍历，是竞赛中空间时间双优的典范。

#### 3. 核心难点辨析与解题策略
1. **难点1：理解距离条件的几何含义**  
   * **分析**：两个不等式`|b-x|<|a-x|`和`|b-y|<|a-y|`要求Bob的位置必须同时比Alice更靠近`x`和`y`。这等价于要求**Bob的位置必须比Alice更接近整个区间`[x,y]`**。  
   * 💡 **学习笔记**：在数轴问题中，绝对值条件往往对应几何位置关系。

2. **难点2：发现Alice位置的临界特性**  
   * **分析**：当Alice位于`[min(x,y), max(x,y)]`区间内时，任何Bob的位置都无法同时满足两个严格不等式（可用三角不等式证明）；反之则存在解。  
   * 💡 **学习笔记**：区间端点值需要包含，这是易错点。

3. **难点3：避免无效遍历的思维转换**  
   * **分析**：暴力枚举所有整数点`b`会超时。优质解法通过**问题抽象**将算法复杂度从`O(n)`降为`O(1)`。  
   * 💡 **学习笔记**：先数学分析再编码，常能大幅优化效率。

##### ✨ 解题技巧总结
- **几何转化技巧**：将距离条件转化为数轴区间关系  
- **边界处理技巧**：闭区间端点需用`>=`和`<=`严谨判断  
- **问题抽象技巧**：用`min`/`max`压缩分支逻辑

#### 4. C++核心代码实现赏析
**本题通用核心C++实现参考**  
* **说明**：综合几何思想的最简实现  
* **完整核心代码**：
  ```cpp
  #include <iostream>
  #include <algorithm>
  using namespace std;

  int main() {
      int t;
      cin >> t;
      while (t--) {
          int a, x, y;
          cin >> a >> x >> y;
          int L = min(x, y);
          int R = max(x, y);
          cout << ((a >= L && a <= R) ? "NO" : "YES") << endl;
      }
      return 0;
  }
  ```
* **代码解读概要**：  
  > 1. 读取测试数据组数`t`  
  > 2. 对每组数据：  
  >    - 计算区间边界`L=min(x,y)`, `R=max(x,y)`  
  >    - 判断`a`是否在`[L,R]`内（含端点）  
  >    - 输出`NO`（在区间内）或`YES`（在区间外）

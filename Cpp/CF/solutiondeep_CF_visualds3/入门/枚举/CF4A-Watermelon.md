## 题目描述

一个炎热的夏天，Pete 和他的朋友 Billy 感到十分口渴，于是在西瓜摊上买了一个重 $w$ 公斤的西瓜。

他们回到家准备分西瓜。因为两个人都十分喜爱偶数，所以他们俩希望分出来的两块西瓜的重量都是偶数公斤。帮助他们判断是否可能做到这一点。

8

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：数学（奇偶性应用 + 边界条件处理）

🗣️ **初步分析**：
解决「Watermelon」的关键，其实是一道“小学奥数题”——**偶数+偶数=偶数**。但别着急下结论！题目要求两块西瓜都是**正偶数**（即重量>0的偶数），所以有个“隐藏陷阱”：最小的可分解数是4（2+2），而2无法分成两个正偶数（2=0+2，但0不是正数；2=1+1，但1是奇数）。

所有题解的核心思路高度一致：**先判断输入是否为偶数，再排除“2”这个特殊情况**。比如：
- 输入8（偶数且≠2）→ 可以分成2+6 → 输出YES；
- 输入2（偶数但=2）→ 无法分解 → 输出NO；
- 输入3（奇数）→ 直接输出NO。

**可视化设计思路**：我们会用8位像素风格模拟“西瓜分解”过程——
- 屏幕中央是一个像素化西瓜，上方显示输入数字；
- 点击“分解”后，西瓜会分裂成两个小方块，用**绿色**表示偶数、**红色**表示奇数；
- 若分解成功（两个绿色方块），播放“叮”的像素音效；若失败（出现红色或0），播放“嗒”的提示音；
- 还能通过“单步”按钮慢动作看分解逻辑，比如输入2时，分裂出两个红色1，直观理解为什么不行。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“陷阱”藏在细节里。结合优质题解的共性，我总结了3个核心难点和解决策略：
</difficulty_intro>

1.  **关键点1：如何利用奇偶性？**
    * **分析**：偶数才能分成两个偶数之和，所以第一步要判断`n%2==0`（或`n&1==0`）。如果是奇数，直接输出NO。
    * 💡 **学习笔记**：奇偶性是解决很多数学题的“敲门砖”，先想数学规律再写代码！

2.  **关键点2：为什么要特判2？**
    * **分析**：题目要求“正偶数”，两个正偶数最小是2+2=4，所以2无法分解。必须单独处理`n==2`的情况。
    * 💡 **学习笔记**：遇到“看起来符合规律但实际不行”的情况，一定要加**边界条件判断**！

3.  **关键点3：如何写简洁的判断逻辑？**
    * **分析**：可以把两个条件合并（`n%2==0&&n!=2`），或者用位运算优化（`!(n&1)&&n!=2`），减少代码行数。
    * 💡 **学习笔记**：简洁的代码不是“少打几个字”，而是**逻辑更集中**，让人一眼看懂核心条件。

### ✨ 解题技巧总结
- **技巧A：先想数学规律**：遇到“分块”“求和”问题，先回忆数学公式（比如偶数+偶数=偶数），再转化为代码。
- **技巧B：重视边界条件**：像“2”“0”“负数”这些特殊情况，一定要单独测试，避免WA。
- **技巧C：代码优化小技巧**：用位运算（`&1`）代替取模（`%2`），用三目运算符简化if-else，让代码更高效。

<code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自优质题解的整合，用最直观的方式实现核心逻辑，适合新手学习。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int n;
        cin >> n;
        if (n % 2 == 0 && n != 2) {
            cout << "YES" << endl;
        } else {
            cout << "NO" << endl;
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分为三步：1. 读入西瓜重量`n`；2. 判断`n`是否是偶数且不等于2；3. 输出结果。核心逻辑就藏在`if`条件里——**符合条件输出YES，否则NO**。

接下来看几个优质题解的“精华片段”，学习它们的亮点：

**题解一：来源：Loner_Knowledge**
* **亮点**：用位运算和三目运算符，把判断逻辑压缩成一行，非常简洁！
* **核心代码片段**：
    ```cpp
    printf("%s", (w&1 || w==2) ? "NO" : "YES");
    ```
* **代码解读**：
    > `w&1`等价于`w%2==1`（判断奇数），`||`表示“或者”。整句话的意思是：如果`w`是奇数**或者**`w`等于2，输出NO；否则输出YES。三目运算符`? :`把if-else写成了一行，特别高效！
* 💡 **学习笔记**：位运算比取模运算更快，适合优化简单判断！

**题解二：来源：Kujo_Jotaro**
* **亮点**：把“特判2”放在最前面，直接结束程序，避免无效计算。
* **核心代码片段**：
    ```cpp
    if (n == 2) { cout << "NO"; return 0; }
    if (n % 2 == 0) cout << "YES";
    else cout << "NO";
    ```
* **代码解读**：
    > 先检查`n`是不是2，如果是，直接输出NO并结束程序（`return 0`）。这样后面的代码不用再处理2的情况，逻辑更清晰！
* 💡 **学习笔记**：提前处理特殊情况，能让代码更高效！

**题解三：来源：土川崇贺**
* **亮点**：用暴力枚举验证规律，适合理解“为什么2不行”。
* **核心代码片段**：
    ```cpp
    for (int i=2; i<=n/2; i+=2) {
        if ((n-i)>=2 && (n-i)%2==0) {
            cout << "YES"; return 0;
        }
    }
    cout << "NO";
    ```
* **代码解读**：
    > 从2开始，每次加2枚举第一个数`i`（保证是正偶数），然后计算第二个数`n-i`。如果`n-i`也是≥2的偶数，说明可以分解，输出YES并结束。如果循环结束都没找到，输出NO。这种“暴力枚举”能直观验证我们的数学规律！
* 💡 **学习笔记**：当不确定规律时，用暴力枚举验证是个好方法！

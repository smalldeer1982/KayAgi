# Rounding

## 题目描述

Vasya has a non-negative integer $ n $ . He wants to round it to nearest integer, which ends up with $ 0 $ . If $ n $ already ends up with $ 0 $ , Vasya considers it already rounded.

For example, if $ n=4722 $ answer is $ 4720 $ . If $ n=5 $ Vasya can round it to $ 0 $ or to $ 10 $ . Both ways are correct.

For given $ n $ find out to which integer will Vasya round it.

## 说明/提示

In the first example $ n=5 $ . Nearest integers, that ends up with zero are $ 0 $ and $ 10 $ . Any of these answers is correct, so you can print $ 0 $ or $ 10 $ .

## 样例 #1

### 输入

```
5
```

### 输出

```
0
```

## 样例 #2

### 输入

```
113
```

### 输出

```
110
```

## 样例 #3

### 输入

```
1000000000
```

### 输出

```
1000000000
```

## 样例 #4

### 输入

```
5432359
```

### 输出

```
5432360
```

    * 💡 **学习笔记**：处理大数字时，先想“会不会溢出”，`long long`是你的好朋友~

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**通用且安全**的核心实现——它综合了“条件判断”和“大数处理”，适合所有情况！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自“向日葵小班”的题解，修改了变量类型为`long long`，处理大数更安全。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        long long n; // 用long long存大数
        cin >> n;
        long long ge = n % 10; // 取个位
        if (ge <= 5) {
            n -= ge; // 舍：减去个位，比如5→0，113→110
        } else {
            n += (10 - ge); // 入：加上(10-个位)，比如5432359→5432360
        }
        cout << n << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分3步：1. 读入数字`n`（用`long long`防溢出）；2. 用`n%10`取个位`ge`；3. 根据`ge`的值选择“舍”或“入”，最后输出结果。逻辑超直白，新手也能看懂！

**题解一：来源：Reywmp**
* **亮点**：用一行公式实现四舍五入，简洁到极致！
* **核心代码片段**：
    ```cpp
    int a;
    scanf("%d", &a);
    a = (a + 5) / 10 * 10; // 关键公式
    printf("%d", a);
    ```
* **代码解读**：
    > 比如`a=5`：5+5=10→10/10=1→1*10=10；`a=113`：113+5=118→118/10=11（C++除法自动向下取整）→11*10=110；`a=5432359`：5432359+5=5432364→5432364/10=543236→543236*10=5432360。完美！
* 💡 **学习笔记**：这个公式是“四舍五入到10的倍数”的“万能公式”，记下来能省好多代码！

**题解二：来源：向日葵小班**
* **亮点**：用`long long`处理大数，条件判断清晰！
* **核心代码片段**：
    ```cpp
    long long n;
    cin >> n;
    if (n % 10 <= 5) n -= n % 10;
    if (n % 10 > 5) n += (10 - n % 10);
    ```
* **代码解读**：
    > 比如`n=1e9`（1000000000）：`n%10=0`，所以不处理，直接输出1e9；`n=5`：`5%10=5≤5`，所以`5-5=0`；`n=5432359`：`5432359%10=9>5`，所以`5432359+(10-9)=5432360`。逻辑超稳！
* 💡 **学习笔记**：处理大数时，`long long`比`int`更安全，一定要记得用！

**题解三：来源：不到前10不改名**
* **亮点**：利用C++除法的“向下取整”特性，逻辑直接！
* **核心代码片段**：
    ```cpp
    int n;
    scanf("%d", &n);
    if (n % 10 >= 5)
        n = (n / 10 + 1) * 10; // 入：比如5→(0+1)*10=10
    else
        n = n / 10 * 10; // 舍：比如113→11*10=110
    ```
* **代码解读**：
    > `n/10`会去掉最后一位（比如5/10=0，113/10=11）。如果个位≥5，就加1再乘10（比如5→0+1=1→10）；否则直接乘10（比如113→11→110）。这个思路超符合“人类直觉”！
* 💡 **学习笔记**：C++的整数除法会“自动去掉小数部分”，比如118/10=11，而不是11.8，这个特性超有用！

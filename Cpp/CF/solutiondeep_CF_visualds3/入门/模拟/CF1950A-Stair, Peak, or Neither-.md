## 题目描述

给定三个数字 $a$、$b$ 和 $c$，判断它们是否构成“阶梯”、“峰值”或都不是。

- “阶梯”满足条件 $a < b < c$。
- “峰值”满足条件 $a < b > c$。

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：模拟应用  

🗣️ **初步分析**：  
解决这道题的关键是「模拟」——就像按照妈妈给的蛋糕配方一步步操作：题目说“阶梯要满足a<b<c”，那我们就**严格对照规则检查数字关系**；说“峰值要满足a<b>c”，我们就逐一验证条件。模拟的核心思想就是“用代码还原题目描述的规则”，没有复杂的算法，却超考验逻辑的严谨性～  

在本题中，模拟的流程超清晰：  
1. 读入三个数字a、b、c；  
2. 先检查是不是“阶梯”（a < b 且 b < c）；  
3. 再检查是不是“峰值”（a < b 且 b > c）；  
4. 都不满足就输出“NONE”。  

**核心难点**：别把条件搞反（比如把峰值写成a>b>c）、别漏掉“严格小于/大于”（比如a等于b时，两个条件都不满足）。  

**可视化设计思路**：我打算用8位像素风做三个“数字方块”——a、b、c排成一行。如果是阶梯，三个方块会**依次变高（像楼梯）+ 变成绿色**；如果是峰值，中间的b方块会**猛地“长高”，两边a、c变矮 + 变成黄色**；都不是的话，三个方块会**灰掉**。关键步骤会用“闪烁”和“像素音效”提醒你：比如检查a<b时，a和b会闪一下，满足就“叮”一声～

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题看似简单，但新手常犯“想当然”的错误。结合常见问题，我帮大家提炼了3个核心思考点：
</difficulty_intro>

1.  **关键点1：准确理解“阶梯”和“峰值”的定义**  
    * **分析**：题目里的“阶梯”是“a < b < c”（严格递增），不是“a ≤ b ≤ c”；“峰值”是“a < b > c”（中间高、两边低），不是“a > b > c”（递减）。比如输入“1 2 2”，因为b=c，所以不是阶梯；输入“3 2 1”，因为a>b，所以不是峰值。  
    * 💡 **学习笔记**：题目规则是“圣旨”，哪怕你觉得“应该这样”，也要严格按题目写的来！

2.  **关键点2：用“逻辑与”（&&）组合条件**  
    * **分析**：阶梯需要同时满足“a < b”和“b < c”，这两个条件缺一不可。比如输入“1 3 2”，虽然a < b，但b > c，所以不是阶梯；输入“2 1 3”，a > b，所以也不是阶梯。代码里一定要用`&&`把两个条件绑在一起，不能分开写！  
    * 💡 **学习笔记**：“同时满足”用&&，“满足其一”用||——别搞混啦！

3.  **关键点3：处理“相等”的情况**  
    * **分析**：只要a等于b，或者b等于c，两个条件都不满足。比如输入“0 0 0”，三个数都相等，输出NONE；输入“4 5 5”，b等于c，不是阶梯；输入“5 5 3”，a等于b，不是峰值。  
    * 💡 **学习笔记**：“严格小于/大于”是本题的“隐形规则”，别漏掉！

### ✨ 解题技巧总结
<summary_best_practices>
通过这道题，我们可以总结出“模拟类题目”的通用技巧：
</summary_best_practices>
- **技巧1：把题目规则“拆成代码句子”**：比如“阶梯是a<b<c”→ 写成`(a < b) && (b < c)`；“峰值是a<b>c”→ 写成`(a < b) && (b > c)`。  
- **技巧2：用“多组测试用例”练手感**：比如样例输入有7组，用循环读入每组数据，避免“只处理一组”的错误。  
- **技巧3：写代码前先“手动算样例”**：比如样例输入“1 5 3”，手动算一遍：a=1 < b=5，b=5 > c=3 → 是峰值，输出PEAK——这样写代码时更有底气！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**能解决所有测试用例的通用C++代码**，它的逻辑超清晰，适合新手模仿～
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了模拟题的经典写法，处理多组输入，严格对照题目条件判断，代码简洁易懂。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int n; // 测试用例数量
        cin >> n;
        while (n--) { // 循环n次，处理每组数据
            int a, b, c;
            cin >> a >> b >> c;
            if (a < b && b < c) { // 检查阶梯
                cout << "STAIR" << endl;
            } else if (a < b && b > c) { // 检查峰值
                cout << "PEAK" << endl;
            } else { // 都不是
                cout << "NONE" << endl;
            }
        }
        return 0;
    }
    ```
* **代码解读概要**：  
  1. 先读入测试用例数量`n`；  
  2. 用`while(n--)`循环n次，每次读入三个数a、b、c；  
  3. 先判断是不是阶梯（`a < b && b < c`），是就输出STAIR；  
  4. 不是的话，再判断是不是峰值（`a < b && b > c`），是就输出PEAK；  
  5. 都不是就输出NONE。

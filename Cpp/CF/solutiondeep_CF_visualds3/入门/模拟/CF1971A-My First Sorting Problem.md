## 3. 核心难点辨析与解题策略

<difficulty_intro>
  这道题虽然简单，但新手常踩3个“小坑”。结合基础编程逻辑，Kay帮你把“坑”变成“知识点”：
</difficulty_intro>

1.  **关键点1：如何正确比较两个数的大小？**
    * **分析**：用`if`语句或者`min/max`函数！比如`if (x > y)`说明x更大，这时候要输出`y x`；或者直接用C++自带的`min(x,y)`（取小数）和`max(x,y)`（取大数），代码更简洁～
    * 💡 **学习笔记**：比较是排序的“第一步”，先明确“谁大谁小”才能排对顺序！

2.  **关键点2：如何处理“多组输入”？**
    * **分析**：题目第一行是`n`（表示有`n`组数据），所以要先读`n`，再用`while(n--)`循环`n`次——每次读一组`x`和`y`，处理后输出～
    * 💡 **学习笔记**：多组输入别忘“先读组数”，循环是处理多组数据的“神器”！

3.  **关键点3：输出格式怎么保证？**
    * **分析**：样例输出中，每组结果占一行，两个数之间有空格。所以输出时要写`cout << 小数 << " " << 大数 << endl;`——`" "`是空格，`endl`是换行～
    * 💡 **学习笔记**：输出格式错了会“扣分”，一定要和样例对齐！

## 4. C++核心代码实现赏析

<code_intro_overall>
  下面是一个**清晰完整的核心实现**，综合了基础思路，帮你快速理解整体框架：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码用`min/max`函数简化比较，循环处理多组输入，是最基础也最常用的实现方式。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <algorithm> // 包含min和max函数
    using namespace std;

    int main() {
        int n; // 组数
        cin >> n; // 读入组数
        while (n--) { // 循环n次，处理每组数据
            int x, y;
            cin >> x >> y; // 读入当前组的两个数
            cout << min(x, y) << " " << max(x, y) << endl; // 输出小数和大数
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 1. 先读入组数`n`；  
    > 2. 用`while(n--)`循环`n`次：每次读`x`和`y`，用`min(x,y)`取小数，`max(x,y)`取大数；  
    > 3. 按“小数 大数”的顺序输出，每个结果占一行。

# 题目信息

# Minimize!

## 题目描述

给你两个整数 $a$ 和 $b$ ($a≤b$) 。
在 $c$ ($a≤c≤b$) 的所有可能整数值中，求 $(c−a)+(b−c)$ 的最小值。

## 样例 #1

### 输入

```
3
1 2
3 10
5 5```

### 输出

```
1
7
0```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：数学（表达式化简）

🗣️ **初步分析**：  
解决“Minimize!”的关键，是**拆穿表达式的“伪装”**！题目要求在`a≤c≤b`的范围内，找`(c−a)+(b−c)`的最小值。但我们把括号拆开算一算：  
`(c−a)+(b−c) = c - a + b - c = b - a`  
哦！原来不管`c`选什么值，结果都是`b-a`——**变量`c`完全抵消了**！所以最小值就是`b-a`本身，根本不用遍历所有`c`！  

这就是数学化简的力量：把“需要循环遍历”的复杂问题，直接变成“一步计算”的简单题。  

对于可视化设计，我们会用**像素动画展示“c抵消”的过程**：用彩色像素块代表`a`（红）、`b`（蓝）、`c`（黄），动画演示`c`和`-c`碰撞消失，结果区的紫色块（代表`b-a`）始终不变，再加上“叮”的音效强调化简的关键——让抽象的数学变直观！

**本题通用核心C++实现参考**
* **说明**：本代码处理多组输入，直接计算`b-a`，是本题最简洁高效的实现。
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int t;
      cin >> t; // 读取测试用例数量
      while (t--) { // 循环处理每组数据
          int a, b;
          cin >> a >> b;
          cout << b - a << endl; // 直接输出化简结果
      }
      return 0;
  }
  ```
* **代码解读概要**：  
  代码分三步：①读测试用例数`t`；②循环`t`次，每次读`a`和`b`；③输出`b-a`。核心逻辑就是**利用数学化简的结论**，没有任何多余的计算——这就是“聪明的代码”！

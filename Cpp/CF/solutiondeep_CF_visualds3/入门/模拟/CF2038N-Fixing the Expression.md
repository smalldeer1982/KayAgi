# 题目信息

# Fixing the Expression

## 题目描述

一个表达式是由三个字符组成的字符串，其中第一个和最后一个字符是数字（从 $0$ 到 $9$ ），中间的字符是一个比较符号（$<$ 、$=$ 或 $>$ ）。

如果比较符号与数字相匹配，那么这个表达式就是成立的（例如，如果第一个数字严格小于最后一个数字，那么比较符号就应该是 $<$ ）。

例如，表达式 $1<3$ 、$4>2$ 、$0=0$ 是成立的，而 $5>5$ 、$7<3$ 则不成立。

给定一个字符串，它是一个表达式。尽可能少地改变字符，使变成一个成立的表达式。注意，如果已经是成立的，就应该保持原样。

## 样例 #1

### 输入

```
5
3&lt;7
3&gt;7
8=9
0=0
5&lt;3```

### 输出

```
3&lt;7
8&gt;7
8&lt;9
0=0
0&lt;3```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
刚开始做这道题时，你可能会问：“为什么改符号是最优的？”“怎么正确读入表达式？”“字符和数字怎么比较？”别着急，我们一个个拆解～
</difficulty_intro>

1.  **关键点1：为什么改符号是最少修改？**
    * **分析**：修改一个字符的代价是1。如果改符号，最多需要1次（符号错了）；如果改数字，比如把「3>7」改成「8>7」，也需要1次，但**改符号的逻辑更简单**——不用考虑数字的变化，直接根据原数字找正确符号就行。所以改符号是“又快又好”的选择～
    * 💡 **学习笔记**：解决“最少修改”问题时，先想“修改哪个部分能覆盖所有情况”，优先选逻辑简单的方案！

2.  **关键点2：怎么正确读入表达式？**
    * **分析**：表达式是三个字符（比如“3<7”），有两种读入方式：①读成字符串（用`string`），然后取`s[0]`（第一个数字）、`s[1]`（符号）、`s[2]`（第二个数字）；②分开读入三个变量（比如`char x, op, y`）。两种方式都对，选你习惯的就行～
    * 💡 **学习笔记**：处理固定长度的字符串时，用索引取字符超方便！

3.  **关键点3：字符和数字怎么比较？**
    * **分析**：字符'0'到'9'的ASCII值是连续递增的（'0'=48，'1'=49，…，'9'=57）。所以直接比较字符的大小，就等于比较它们代表的数字大小！比如`'3' < '7'`是对的（51<55），不用转成int～
    * 💡 **学习笔记**：字符数字的ASCII值是连续的，直接比较字符就能代替数字比较！


### ✨ 解题技巧总结
<summary_best_practices>
通过这道题，我们能学到3个通用技巧：
</summary_best_practices>
-   **技巧1：优先选“修改逻辑简单”的方案**：比如改符号比改数字更直接，减少出错可能。
-   **技巧2：固定长度字符串用索引取字符**：比如`s[0]`取第一个字符，比拆分成多个变量更省代码。
-   **技巧3：利用字符ASCII值的连续性**：字符数字直接比较，不用额外转int，提高效率～


## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：本代码用字符串处理输入，直接比较字符大小，是最简洁的实现方式。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    using namespace std;

    int main() {
        int n;
        cin >> n;
        while (n--) { // 处理n组输入
            string s;
            cin >> s; // 读入表达式（比如"3>7"）
            char a = s[0], b = s[2]; // 取第一个和第三个字符（数字）
            cout << a; // 输出第一个数字
            if (a < b) cout << '<'; // 比较后输出正确符号
            else if (a == b) cout << '=';
            else cout << '>';
            cout << b << endl; // 输出第二个数字
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分三步：①读入测试用例数量`n`；②循环处理每组输入，读入表达式字符串`s`；③提取两个数字字符`a`和`b`，比较大小后输出正确的符号。整个流程像“流水线”，一步接一步，特别好懂～

**题解一：来源：Vct14**
* **亮点**：直接修改字符串的符号位，代码超短！
* **核心代码片段**：
    ```cpp
    string c; cin >> c;
    if(c[0]<c[2]) c[1]='<';
    if(c[0]>c[2]) c[1]='>';
    if(c[0]==c[2]) c[1]='=';
    cout<<c<<"\n";
    ```
* **代码解读**：
    > 这段代码的巧妙之处是“直接修改原字符串的符号位”（`c[1]`）。比如输入`"3>7"`，`c[0]='3'`、`c[2]='7'`，因为`'3'<'7'`，所以把`c[1]`改成'<'，最后输出修改后的字符串。这种写法省掉了“重新拼接字符串”的步骤，超高效～
* 💡 **学习笔记**：修改字符串的特定位置，比重新构造字符串更简洁！

**题解二：来源：adsd45666**
* **亮点**：拆分输入变量，逻辑更直观！
* **核心代码片段**：
    ```cpp
    char x, op, y;
    cin >> x >> op >> y;
    if(x < y) cout << x << "<" << y;
    else if(x == y) cout << x << "=" << y;
    else cout << x << ">" << y;
    ```
* **代码解读**：
    > 这段代码把输入拆成三个变量：`x`（第一个数字）、`op`（符号）、`y`（第二个数字）。虽然`op`没被用到，但这种拆分让逻辑更清晰——即使你忘了字符串索引，也能通过变量名看懂每个部分的作用。适合刚学C++的同学～
* 💡 **学习笔记**：拆分变量能让代码更易读，尤其是新手阶段！

**题解三：来源：JoeZYQ**
* **亮点**：把数字读成int，避免字符比较的困惑！
* **核心代码片段**：
    ```cpp
    int a, b;
    char c;
    cin >> a >> c >> b;
    if(a > b) cout << a << ">" << b << "\n";
    if(a == b) cout << a << "=" << b << "\n";
    if(a < b) cout << a << "<" << b << "\n";
    ```
* **代码解读**：
    > 这段代码把`a`和`b`读成int类型，直接比较数值大小。比如输入`"5<3"`，`a=5`、`b=3`，因为`5>3`，所以输出`5>3`。这种写法完全不用考虑字符ASCII值，逻辑最直观，适合对字符操作不熟悉的同学～
* 💡 **学习笔记**：如果字符操作容易错，就转成int再处理！

有五家台湾自助餐厅，编号为 1 到 5。你计划在周一到周五每天去不同的餐厅打包便当作为午餐。你已经在周一到周四分别去过了四家不同的餐厅。现在是周五，你想去本周还没有去过的那家餐厅。

请编写一个程序，输入你在周一到周四已经去过的四家餐厅的编号（顺序为周一到周四），输出你周五应该去的那家餐厅的编号。

> 问题的核心超级简单——五家餐厅编号1-5，周一到周四已经去了4家不同的，周五要补剩下的那一家。就像你有5颗彩虹糖，吃了4颗，剩下的那颗就是"目标糖"！

> 1. **总和相减法**：1+2+3+4+5=15（所有餐厅的总和），减去周一到周四去过的4家的总和，结果就是缺失的餐厅编号（比如样例1输入1+3+2+5=11，15-11=4，正好是答案）。

## 4. C++核心代码实现赏析

<code_intro_overall>
我们先看**两种核心方案的完整代码**，再拆解关键片段——学完就能直接写程序啦！
</code_intro_overall>

### 方案一：总和相减法（最简实现）
* **说明**：这是本题最简洁的实现，利用1-5的总和固定为15，直接计算缺失值。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int a, b, c, d;
        cin >> a >> b >> c >> d;  // 读入周一到周四的餐厅编号
        cout << 15 - (a + b + c + d) << endl;  // 总和相减得结果
        return 0;
    }
    ```
* **代码解读概要**：  
  代码只有3步——读入4个数字→计算总和差→输出结果。就像"吃了4颗糖，算剩下的糖数"一样直接！

### 方案二：标记数组法（最直观实现）
* **说明**：用数组记录每家餐厅的访问状态，最后找没被访问的，逻辑更贴近人类思考。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <vector>  // 用vector当"小本本"
    using namespace std;

    int main() {
        vector<bool> visited(6, false);  // 索引0不用，1-5对应餐厅，初始都没访问
        int x;
        for (int i = 0; i < 4; ++i) {  // 读4次输入
            cin >> x;
            visited[x] = true;  // 标记为已访问
        }
        for (int i = 1; i <= 5; ++i) {  // 找没被访问的餐厅
            if (!visited[i]) {
                cout << i << endl;
                break;  // 找到就退出，不用继续找
            }
        }
        return 0;
    }
    ```
* **代码解读概要**：  
  用`vector<bool> visited(6, false)`创建一个"小本本"（索引1-5对应餐厅），读入每个餐厅编号时标记为`true`（已访问），最后遍历"小本本"找`false`（没访问）的那个！

**标记数组法核心片段**

        visited[x] = true;  // 标记已访问

* **代码解读**：

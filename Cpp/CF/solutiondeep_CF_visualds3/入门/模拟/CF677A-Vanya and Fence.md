# 题目信息

# Vanya and Fence

## 题目描述

有n个人，如果这个人的高度大于k，那么马路宽度+2，否则+1，输出最终的宽度值。

## 样例 #1

### 输入

```
3 7
4 5 14
```

### 输出

```
4
```

## 样例 #2

### 输入

```
6 1
1 1 1 1 1 1
```

### 输出

```
6
```

## 样例 #3

### 输入

```
6 5
7 6 8 9 10 5
```

### 输出

```
11
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
刚开始学模拟题时，容易踩3个「小坑」。结合题解的经验，我帮你把坑填上~
</difficulty_intro>

1.  **关键点1：别搞错「大于k」的条件**
    * **分析**：题目说「高度大于k加2」，不是「大于等于」！比如样例2中所有人都是1，k=1，所以每人加1，总6——如果写成`>=`就会错。解决方法很简单：**用样例验证条件**（比如样例2的输入，如果你写成`a>=k`，输出会是12，明显不对）。
    * 💡 **学习笔记**：条件判断前，一定要用样例「测试」你的逻辑！

2.  **关键点2：不用数组存所有身高**
    * **分析**：很多同学刚学的时候，会习惯性用数组存所有输入——比如题解TRZ_2007用了`a[N]`。但其实**每个身高只用一次**，读完就可以直接判断累加，根本不用存！比如题解FP·荷兰猪的代码，直接读`x`然后处理，节省了数组空间。
    * 💡 **学习笔记**：如果数据只用一次，别存数组——省空间又省时间！

3.  **关键点3：初始化计数器要正确**
    * **分析**：计数器（比如`gs`、`ans`、`q`）的初始值要对。比如题解wyy134的`q`初始化为n（基础费用），而其他题解的`ans`初始化为0。如果初始化错了（比如把`q`初始化为0），结果肯定不对。解决方法：**想清楚计数器的「起点」**——是从0开始累加，还是从n开始补？
    * 💡 **学习笔记**：计数器的初始值，要和你的逻辑匹配！

### ✨ 解题技巧总结
<summary_best_practices>
模拟题的「通关秘诀」就3条，记好啦：
</summary_best_practices>
- **按题说话**：题目说什么就做什么，别自己加规则（比如别把「大于」改成「大于等于」）。
- **用多少读多少**：不用的数别存，直接处理——像吃零食一样，拆一包吃一包，别全拆了堆着。
- **用样例查错**：写完代码先跑样例，对不上就回头看条件、初始化、累加逻辑。

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份「通用核心代码」——它综合了所有优质题解的优点，逻辑清晰、代码简洁，适合直接用~
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自「FP·荷兰猪」的题解，调整了变量名使其更易懂，保留了极简风格。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int n, k;
        cin >> n >> k;
        int total = 0; // total表示总宽度
        for (int i = 1; i <= n; ++i) {
            int height;
            cin >> height;
            if (height > k) {
                total += 2;
            } else {
                total += 1;
            }
        }
        cout << total << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分3步：①读入n（人数）和k（栅栏高度）；②用循环读n次身高，每次根据身高加1或2；③输出总宽度。核心逻辑就是「循环+判断」，没有任何多余的东西~

**题解一：作者666yuchen**
* **亮点**：完全贴合题意的「直接累加」，新手友好。
* **核心代码片段**：
    ```cpp
    for(int i=1;i<=n;i++) {
        cin>>a;
        if(a>k) gs+=2;
        else gs++;
    }
    ```
* **代码解读**：
    > 循环从1到n，每次读入一个身高`a`。如果`a`大于k，`gs`（计数）加2；否则加1。这就是「题目说什么，代码做什么」的完美例子——你读题目时的思路，直接变成了代码！
* 💡 **学习笔记**：新手写模拟题，先学「直接对应题意」，再学优化。

**题解二：作者wyy134**
* **亮点**：先加基础值，再补差值，减少计算步骤。
* **核心代码片段**：
    ```cpp
    q = n; // 先加所有人的基础1
    for (i=1;i<=n;i++) {
        cin>>a;
        if (a>k) q++; // 高个子补1
    }
    ```
* **代码解读**：
    > 比如n=3，先加3（每人1）；如果有1个高个子，再加1，总4——和样例1一致！这种思路把「加1或2」变成了「加1 + 补1」，减少了一次加法操作（不用每次都判断加1还是加2，而是先加1，再补）。
* 💡 **学习笔记**：遇到「两种情况差1」的问题，可以试试「基础值+补差值」的技巧。

**题解三：作者FP·荷兰猪**
* **亮点**：极简代码，没有多余变量。
* **核心代码片段**：
    ```cpp
    int ans=0;
    for (i=1;i<=n;i++) {
        cin>>x;
        if (x>k) ans+=2;
        else ans+=1;
    }
    ```
* **代码解读**：
    > 变量`ans`从头用到尾，没有`a`、`gs`这类别名——直接用`x`读入身高，判断后加1或2。代码里每一行都有用，没有废话，特别适合学「写干净的代码」。
* 💡 **学习笔记**：变量名越简洁（但要易懂），代码越干净！

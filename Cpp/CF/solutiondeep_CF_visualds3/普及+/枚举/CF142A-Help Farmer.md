## 题目描述

## 样例 #1

### 输入

```
4
```

### 输出

```
28 41
```

## 样例 #2

### 输入

```
7
```

### 输出

```
47 65
```

## 样例 #3

### 输入

```
12
```

### 输出

```
48 105
```

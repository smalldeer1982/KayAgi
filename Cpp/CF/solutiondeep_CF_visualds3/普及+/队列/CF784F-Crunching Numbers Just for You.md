## 3. 核心难点辨析与解题策略

<difficulty_intro>
排序题看似简单，但新手常遇到这三个问题。结合优质题解的经验，我帮你总结了应对策略：
</difficulty_intro>

1.  **关键点1：如何选择合适的排序算法？**
    * **分析**：排序算法有很多（sort、桶排、快排、归并等），选择的关键是**数据范围**。比如本题中，若`a_i`的范围很小（比如≤100），用桶排（`O(n + k)`）比sort（`O(n log n)`）更快；若数据范围很大（比如≤1e9），则用sort更合适。优质题解中，作者同时给出了两种算法，就是因为考虑了不同数据范围的情况。
    * 💡 **学习笔记**：数据范围是选择算法的重要依据！

2.  **关键点2：如何处理重复元素？**
    * **分析**：重复元素是排序中常见的情况，比如输入中有多个`3`。桶排的处理方式非常巧妙——用`a[x]++`记录`x`出现的次数，最后循环输出`x`的次数次。这种方法不仅能处理重复，还能保持排序的稳定性（相同元素的相对位置不变）。
    * 💡 **学习笔记**：桶排是处理小范围重复数据的“神器”！

3.  **关键点3：如何写出简洁的排序代码？**
    * **分析**：STL的`sort`函数是C++中最简洁的排序工具，只需`sort(a, a+n)`就能完成升序排序。优质题解中的sort代码只有5行（读入、排序、输出），充分体现了STL的便利性。
    * 💡 **学习笔记**：善用STL可以大大减少代码量！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**通用核心实现**（用STL sort），这是排序题最常用的解法：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自“做梦想Peach”的题解，是C++中排序的“标准模板”，简洁高效。
* **完整核心代码**：
    ```cpp
    #include <cstdio>
    #include <algorithm>
    using namespace std;
    int main() {
        int a[100], n; // a数组存输入的数字，n是数字个数
        scanf("%d", &n); // 读入n
        for (int i = 0; i < n; i++) scanf("%d", &a[i]); // 读入n个数字
        sort(a, a + n); // 对a数组的前n个元素升序排序
        for (int i = 0; i < n; i++) printf("%d ", a[i]); // 输出排序后的结果
        return 0;
    }
    ```
* **代码解读概要**：代码分为三步：①读入`n`和`n`个数字；②用`sort`函数排序；③输出排序结果。其中`sort(a, a + n)`是核心，它会自动将`a[0]`到`a[n-1]`的元素按升序排列。


<code_intro_selected>
接下来，我们剖析“做梦想Peach”题解中的**桶排序片段**，看看它是如何处理小范围数据的：
</code_intro_selected>

**题解一：来源：做梦想Peach**
* **亮点**：针对小范围数据（`a_i ≤ 100`），用桶排实现`O(n + k)`的线性时间排序，效率极高。
* **核心代码片段**：
    ```cpp
    int a[110], n, x; // a是“桶”，存每个数字出现的次数
    int main() {
        scanf("%d", &n);
        for (int i = 0; i < n; i++) {
            scanf("%d", &x);
            a[x]++; // 记录x出现的次数（比如x=3，a[3]加1）
        }
        for (int i = 0; i < 110; i++) { // 从0到109循环（覆盖所有可能的x值）
            if (a[i] >= 1) { // 如果i出现过
                for (int j = 0; j < a[i]; j++) { // 输出a[i]次i
                    printf("%d ", i);
                }
            }
        }
        return 0;
    }
    ```
* **代码解读**：
    > 这段代码的核心是“桶”（`a`数组）。比如输入`3 3 1 2`，`a[1]`会变成1（1出现1次），`a[2]`变成1（2出现1次），`a[3]`变成2（3出现2次）。然后循环`i`从0到109，只要`a[i]`≥1，就输出`i`的`a[i]`次。这样就能得到`1 2 3 3`（注意样例输入是`3 3 1 2`，输出是`1 2 3`？不对，等一下，样例输入应该是`4`个数字？哦，样例输入中的`3 3 1 2`可能是`n=4`，输出是`1 2 3 3`？可能题目中的样例输入描述有误，但桶排的逻辑是对的。）
* 💡 **学习笔记**：桶排的关键是“用空间换时间”——用一个大数组记录每个数字的出现次数，然后按顺序输出。

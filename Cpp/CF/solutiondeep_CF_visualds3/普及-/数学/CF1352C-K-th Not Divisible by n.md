# 题目信息

# K-th Not Divisible by n

## 题目描述

给定两个正整数 $n$ 和 $k$。求出第 $k$ 个不能被 $n$ 整除的正整数。

例如，如果 $n=3$，$k=7$，那么所有不能被 $3$ 整除的数为：$1,2,4,5,7,8,10,11,13 \ldots$ 在这其中的第 $7$ 个数字为 $10$。

## 样例 #1

### 输入

```
6
3 7
4 12
2 1000000000
7 97
1000000000 1000000000
2 1```

### 输出

```
10
15
1999999999
113
1000000001
1```

#### 3. 核心难点辨析与解题策略
1. **难点1：发现周期性规律**  
   * **分析**：序列每 n 个数出现一个"坑"（n的倍数），有效数字周期为 n-1。优质题解通过举例（如 n=3, k=7→10）归纳规律。  
   * 💡 **学习笔记**：遇到序列跳过问题，先观察重复模式。

2. **难点2：处理周期边界**  
   * **分析**：当 k 是 n-1 的倍数时，位于周期末尾，需特殊处理（如 `-1`）。Allen_lml 和 b6e0_ 用分支逻辑解决。  
   * 💡 **学习笔记**：边界测试用例：k = n-1, 2(n-1), ... 

3. **难点3：避免整数溢出**  
   * **分析**：k≤1e9 时，`k*n` 可能超 int 范围。ADay 用 `long long`，Scrutiny 未处理（潜在风险）。  
   * 💡 **学习笔记**：10^9 级运算优先用 `long long`。

✨ **解题技巧总结**：  
- **规律转化**：将抽象序列 → 周期数学模型  
- **边界四步法**：① 列出小样例 ② 归纳特例 ③ 公式调整 ④ 验证  
- **防御性编码**：用 `long long` 处理大数，添加整除判断分支

#### 4. C++核心代码实现赏析
**通用核心实现**  
```cpp
#include <iostream>
using namespace std;
int main() {
    int t; cin >> t;
    while (t--) {
        long long n, k;  // 防溢出
        cin >> n >> k;
        if (k % (n-1) == 0) 
            cout << k/(n-1)*n - 1 << endl;
        else 
            cout << k/(n-1)*n + k % (n-1) << endl;
    }
    return 0;
}
```
* **说明**：综合自 Allen_lml 和 b6e0_ 的最优公式  
* **解读概要**：  
  - 读取 t 组数据  
  - 分支处理整除/非整除情况  
  - 公式核心：`周期数 * n + 余数`（余数=0时需-1）  

---

**题解一：Allen_lml 片段**  
```cpp
if(k%(n-1)!=0) {
    ans = k/(n-1) + k;  // 非整除公式
} else {
    ans = k/(n-1)*n - 1;  // 整除调整
}
```
* **亮点**：双分支显式逻辑，易读性强  
* **代码解读**：  
  > ① `k/(n-1)` 计算完整周期数 → 每个周期跳过 1 个数  
  > ② 非整除时：总数字 = 原序号 + 跳过次数  
  > ③ 整除时：末尾数字 = 周期数×n - 1  
* 💡 **学习笔记**：跳过次数 = k/(n-1)，是公式核心推导  

**题解二：b6e0_ 片段**  
```cpp
if(k%(n-1)) 
    cout << k + k/(n-1);  // 非整除
else 
    cout << k + k/(n-1) - 1;  // 整除调整
```
* **亮点**：用加法等价替代乘法，更简洁  
* **代码解读**：  
  > `k + k/(n-1)` 等价于 `k/(n-1)*n + k%(n-1)`  
  > - 数学证明：`k = q*(n-1) + r` → 结果 = q*n + r  
* 💡 **学习笔记**：公式可数学变换，但保持可读性更重要  

**题解三：ADay 片段**  
```cpp
int x = myceil(k, n-1);        // 行号=周期数（向上取整）
int y = (k-1) % (n-1) + 1;    // 列号=周期内位置
cout << n*(x-1) + y;           // 数=前x-1行总数+y
```
* **亮点**：表格思维直观映射序列位置  
* **代码解读**：  
  > ① 每行 n-1 个有效数 → 总行数 x = ceil(k/(n-1))  
  > ② 列号 y ∈ [1, n-1]，用取模定位  
  > ③ 前 x-1 行有 (x-1)*n 个数 → 当前行第 y 个  
* 💡 **学习笔记**：二维建模是复杂序列问题的通用技巧

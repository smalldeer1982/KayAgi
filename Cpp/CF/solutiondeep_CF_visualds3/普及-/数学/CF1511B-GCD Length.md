# 题目信息

# GCD Length

## 题目描述

You are given three integers $ a $ , $ b $ and $ c $ .

Find two positive integers $ x $ and $ y $ ( $ x > 0 $ , $ y > 0 $ ) such that:

- the decimal representation of $ x $ without leading zeroes consists of $ a $ digits;
- the decimal representation of $ y $ without leading zeroes consists of $ b $ digits;
- the decimal representation of $ gcd(x, y) $ without leading zeroes consists of $ c $ digits.

 $ gcd(x, y) $ denotes the [greatest common divisor (GCD)](https://en.wikipedia.org/wiki/Greatest_common_divisor) of integers $ x $ and $ y $ .

Output $ x $ and $ y $ . If there are multiple answers, output any of them.

## 说明/提示

In the example:

1. $ gcd(11, 492) = 1 $
2. $ gcd(13, 26) = 13 $
3. $ gcd(140133, 160776) = 21 $
4. $ gcd(1, 1) = 1 $

## 样例 #1

### 输入

```
4
2 3 1
2 2 2
6 6 2
1 1 1```

### 输出

```
11 492
13 26
140133 160776
1 1```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决本题需突破三个关键点：

1.  **GCD的位数控制**
    * **分析**：必须构造恰好c位的整数。优质解法统一采用10^{c-1}，既保证首位非零（避免前导0），又通过尾随零精确控制位数。
    * 💡 **学习笔记**：10^{k}是构造k+1位数的黄金工具

2.  **互质关系的构造**
    * **分析**：当GCD固定为10^{c-1}时，x/GCD与y/GCD必须互质。两种主流方案：
      - 质数对法：用两个不同质数（如2和3）确保无公因子
      - 连续9法：9的倍数不含2和5因子，自然与10的幂次互质
    * 💡 **学习笔记**：因子互补性是构造互质的利器

3.  **乘积的位数保障**
    * **分析**：x = GCD * P 需满足位数a。通过：
      - 质数对法：预存质数使其位数恰好为a-c+1
      - 连续9法：输出a-1位数字符串（首位1）
    * 💡 **学习笔记**：位数=对数值取整+1，构造时注意边界

## 4. C++核心代码实现赏析

<code_intro_overall>
下面展示综合优化的通用实现，采用最简洁的连续9法：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：综合自xiaoyuchenp和Mophie题解，兼具简洁性与可读性
* **完整核心代码**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int t, a, b, c;
    cin >> t;
    while (t--) {
        cin >> a >> b >> c;
        // 构造x：1后跟a-1个0
        cout << '1';
        for (int i = 1; i < a; i++) cout << '0';
        
        cout << ' ';
        
        // 构造y：b-c+1个9 + c-1个0
        for (int i = 0; i < b - c + 1; i++) cout << '9';
        for (int i = 0; i < c - 1; i++) cout << '0';
        cout << '\n';
    }
    return 0;
}
```
* **代码解读概要**：
> 1. 读取测试用例数t
> 2. 构造x：输出'1'后接a-1个'0'（确保a位）
> 3. 构造y：先输出b-c+1个'9'（与GCD商互质），再接c-1个'0'（与GCD拼接）
> 4. 关键点：9的倍数不含2/5因子，保证与GCD的商(10^{a-c})互质

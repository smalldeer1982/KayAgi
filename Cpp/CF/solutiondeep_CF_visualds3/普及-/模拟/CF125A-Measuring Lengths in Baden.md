# Measuring Lengths in Baden

## 题目描述

Lengths are measures in Baden in inches and feet. To a length from centimeters it is enough to know that an inch equals three centimeters in Baden and one foot contains 12 inches.

You are given a length equal to $ n $ centimeters. Your task is to convert it to feet and inches so that the number of feet was maximum. The result should be an integer rounded to the closest value containing an integral number of inches.

Note that when you round up, 1 cm rounds up to 0 inches and 2 cm round up to 1 inch.

## 样例 #1

### 输入

```
42
```

### 输出

```
1 2
```

## 样例 #2

### 输入

```
5
```

### 输出

```
0 2
```

1.  **关键点1**：如何正确实现四舍五入规则？
    * **分析**：题目要求1cm→0英寸（0.333英寸→0），2cm→1英寸（0.666英寸→1）。这等价于将n/3的结果四舍五入到最近的整数。优质题解中，Alex_Wei用`(n+1)/3`（整数除法自动向下取整），Eason_AC用`floor(n/3.0 + 0.5)`，都能正确实现。例如，n=5时，5/3≈1.666，+0.5后是2.166，floor后为2（正确）。
    * 💡 **学习笔记**：四舍五入可通过“加0.5后取整”或利用整数除法特性（如(n+1)/3）实现。

2.  **关键点2**：如何将总英寸数分解为英尺和剩余英寸？
    * **分析**：1英尺=12英寸，因此总英寸数除以12的商是英尺数，余数是剩余英寸数。例如，总英寸数14→14/12=1（英尺），14%12=2（英寸）。这一步是简单的整数运算，关键是确保总英寸数正确。
    * 💡 **学习笔记**：分解单位时，除法得高位单位，取模得低位单位。

3.  **关键点3**：如何选择简洁的代码实现方式？

**题解一：作者：Alex_Wei**
* **亮点**：用`(n+1)/3`极简实现四舍五入，利用整数除法特性，无需浮点运算，高效且避免精度问题。
* **核心代码片段**：
    ```cpp
    cin >> n, c = (n + 1) / 3;
    cout << c / 12 << " " << c % 12;
    ```
* **代码解读**：
    > 这两行代码是核心！`(n+1)/3`为什么能实现四舍五入？例如，n=5时，5+1=6，6/3=2（正确）；n=1时，1+1=2，2/3=0（正确）。整数除法会自动向下取整，所以`(n+1)/3`等价于将n/3的结果四舍五入到最近的整数。
* 💡 **学习笔记**：整数运算比浮点运算更高效，且无精度丢失，简单问题中优先考虑。

**题解二：作者：封禁用户**
* **亮点**：自定义`round_double`函数，清晰展示四舍五入的通用逻辑，适合理解原理。
* **核心代码片段**：
    ```cpp
    int round_double(double number) {
        return (number > 0.0) ? floor(number + 0.5) : ceil(number - 0.5);
    }
    cout << round_double(n / 3.0) / 12 << " " << round_double(n / 3.0) % 12;
    ```
* **代码解读**：
    > `round_double`函数对正数和负数分别处理：正数时，`number+0.5`后向下取整（如1.666+0.5=2.166，floor后为2）；负数时类似。本题中n为非负，所以直接`floor(n/3.0 + 0.5)`即可。
* 💡 **学习笔记**：自定义函数可提高代码复用性，适合需要多次四舍五入的场景。

**题解三：作者：Eason_AC**
* **亮点**：用`floor(n/3.0 + 0.5)`直接实现四舍五入，逻辑清晰，代码简洁。
* **核心代码片段**：
    ```cpp
    int inch = floor(n / 3.0 + 0.5);
    printf("%d %d", inch / 12, inch % 12);
    ```
* **代码解读**：
    > `n/3.0`得到浮点型的英寸数（如5/3.0=1.666），加0.5后（2.166）用`floor`取整得到2。这一步完美实现四舍五入，然后分解为英尺和英寸。
* 💡 **学习笔记**：浮点运算+取整函数是通用的四舍五入方法，适合理解原理。

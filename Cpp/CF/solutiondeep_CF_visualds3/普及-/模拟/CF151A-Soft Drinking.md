# 题目信息

# Soft Drinking

## 题目描述

This winter is so cold in Nvodsk! A group of $ n $ friends decided to buy $ k $ bottles of a soft drink called "Take-It-Light" to warm up a bit. Each bottle has $ l $ milliliters of the drink. Also they bought $ c $ limes and cut each of them into $ d $ slices. After that they found $ p $ grams of salt.

To make a toast, each friend needs $ nl $ milliliters of the drink, a slice of lime and $ np $ grams of salt. The friends want to make as many toasts as they can, provided they all drink the same amount. How many toasts can each friend make?

## 说明/提示

A comment to the first sample:

Overall the friends have $ 4*5=20 $ milliliters of the drink, it is enough to make $ 20/3=6 $ toasts. The limes are enough for $ 10*8=80 $ toasts and the salt is enough for $ 100/1=100 $ toasts. However, there are 3 friends in the group, so the answer is $ min(6,80,100)/3=2 $ .

## 样例 #1

### 输入

```
3 4 5 10 8 100 3 1
```

### 输出

```
2
```

## 样例 #2

### 输入

```
5 100 10 1 19 90 4 3
```

### 输出

```
3
```

## 样例 #3

### 输入

```
10 1000 1000 25 23 1 50 1
```

### 输出

```
0
```

🗣️ **初步分析**：
解决“Soft Drinking”的关键在于理解“材料限制”的数学关系。简单来说，就像分糖果时要考虑每个人分到的数量不能超过总糖果数，这里每个朋友做吐司需要的饮料、柠檬片、盐的量，需要被总材料量限制。  
题目要求所有朋友做的吐司数量相同，因此我们需要找到三种材料能支持的最大吐司总数的最小值，再均分给n个朋友。  
- **题解思路**：所有题解均采用“计算三种材料的最大可用吐司数→取最小值→除以朋友数n”的逻辑。核心难点是正确计算每种材料的可用次数（如饮料总量是k*l，每次需要nl毫升，所以可用次数是k*l/nl）。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决此题的关键在于正确计算每种材料的可用次数，并理解“取最小值后均分”的逻辑。以下是三个核心难点及应对策略：
</difficulty_intro>

1.  **关键点1：正确计算每种材料的可用次数**  
    * **分析**：饮料的总量是k瓶×每瓶l毫升，每次需要nl毫升，所以可用次数是`k*l/nl`（整数除法向下取整）。柠檬的总量是c个×每个d片，每次需要1片，所以可用次数是`c*d`。盐的总量是p克，每次需要np克，所以可用次数是`p/np`。  
    * 💡 **学习笔记**：每种材料的可用次数=总材料量÷单次需求量（注意整数除法自动向下取整）。

2.  **关键点2：理解“取最小值”的意义**  
    * **分析**：三种材料必须同时满足才能做吐司，因此最终能做的吐司总数由“最紧缺”的材料决定。例如，如果饮料只能支持5次，柠檬支持10次，盐支持8次，那么最多只能做5次。  
    * 💡 **学习笔记**：短板效应——最终结果由最小的可用次数决定。

3.  **关键点3：理解“除以n”的原因**  
    * **分析**：题目要求所有朋友做的吐司数量相同，因此需要将总吐司数均分给n个朋友。例如，总吐司数是6，n=3，每个朋友能做2个。  
    * 💡 **学习笔记**：均分是题目要求“所有朋友喝相同量”的直接体现。

### ✨ 解题技巧总结
- **问题拆解**：将复杂问题拆分为“计算各材料可用次数→取最小→均分”三个步骤，逐步解决。  
- **变量命名清晰**：用x、y、z分别表示饮料、柠檬、盐的可用次数，代码更易读。  
- **边界处理**：整数除法自动向下取整，无需额外处理（如样例1中k*l=20，nl=3，20/3=6）。

<code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：此代码综合了Reywmp和ShineEternal的思路，保留核心逻辑并简化输入方式（使用`scanf`），适合新手直接学习。  
* **完整核心代码**：
    ```cpp
    #include <cstdio>
    #include <algorithm> // 用于fmin函数

    int main() {
        int n, k, l, c, d, p, nl, np;
        scanf("%d%d%d%d%d%d%d%d", &n, &k, &l, &c, &d, &p, &nl, &np);
        
        // 计算三种材料的可用吐司数
        int drink = k * l / nl;       // 饮料可用次数
        int lemon = c * d;            // 柠檬可用次数
        int salt = p / np;            // 盐可用次数
        
        // 取最小值后均分给n个朋友
        int total = std::min({drink, lemon, salt});
        printf("%d\n", total / n);
        
        return 0;
    }
    ```
* **代码解读概要**：  
  代码首先读取输入的8个整数，然后分别计算饮料（`k*l/nl`）、柠檬（`c*d`）、盐（`p/np`）的可用吐司数。通过`std::min({...})`取三者的最小值，最后除以朋友数n并输出结果。

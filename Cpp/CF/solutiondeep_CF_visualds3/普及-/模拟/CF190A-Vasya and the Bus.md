# 题目信息

# Vasya and the Bus

## 题目描述

--

一天，Vasya 听到一个故事，它说：“在 High Bertown 城，一辆 $62$ 路公交车带着 $n$ 位大人和 $m$ 位小孩，从公交车站开了出来……”

至于那些后续，都是无关紧要的。Vasya 是一位会计，而且他很享受数钱数到手抽经的快感。因此，他想知道这趟车上的这些乘客所付车票钱的最小值和最大值是多少

在 High Bertown 城，公交车的车票为 $1$ 卢布 。但是，那么简单的题目想都别想遇到—— $1$ 位大人可以免票带 $1$ 位小孩。这意味着 $1$ 位大人和 $k$ 个小孩共同乘车，总的需要 $k$ 卢布的车票；他自己需要 $1$ 张车票，孩子们需要 $(k-1)$  张。除此之外，大人们可以不带孩子乘车，这样他们就只要支付自己的 $1$ 卢布车票钱。

同时，我们还知道在 High Bertown 城，孩子们不能在没有大人的陪同下单独乘车。

请你帮帮 Vasya 计算一下这辆车上乘客需付金额总和的最小值和最大值。

---

## 说明/提示

---

在第一个样例中， $1$ 位大人和 $2$ 位孩子乘车，不管怎么样，都需要支付 $2$ 卢布。

在第二个样例中，只有孩子乘车是不合法的。

在第三个样例中，有两种情况：

1.  $2$ 位大人各带 $1$ 位小孩，各付 $1$ 卢布车票钱。此时，乘客们一共支付了 $2$ 卢布。

2. 另一种情况下，一位大人带 $2$ 位孩子，支付 $2$ 卢布车票钱；另一位独自乘车，花费 $1$ 卢布。因此，他们一共支付了 $3$ 卢布。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
在解决这个问题时，核心难点在于理解“最小和最大花费”的计算逻辑。以下是关键步骤和解决策略：
</difficulty_intro>

1.  **关键点1**：如何确定“最小花费”？
    * **分析**：最小花费的本质是“让尽可能多的小孩被大人免费带”。如果大人数量（n）≥小孩数量（m），每个小孩都能被一个大人带，此时花费等于大人数量（n）；如果小孩更多（m>n），每个大人带一个小孩后，剩下的小孩需要自己买票，总花费等于小孩数量（m）。因此，最小花费是`max(n, m)`。
    * 💡 **学习笔记**：最小花费的关键是“最大化免费带的小孩数”，取n和m中的较大值。

2.  **关键点2**：如何确定“最大花费”？
    * **分析**：最大花费的本质是“让尽可能少的小孩被免费带”。让一个大人带所有小孩（只省1张票），其他大人单独买票。总花费为“大人数量+小孩数量-1”（因为只省了1张票）。例如，2个大人和2个小孩，总花费是2+2-1=3（一个大人带2个小孩省1张，另一个大人单独买1张）。
    * 💡 **学习笔记**：最大花费的关键是“只省1张票”，公式为`n + m - 1`。

3.  **关键点3**：如何处理边界条件？
    * **分析**：边界条件包括“只有大人”（m=0，花费n）和“只有小孩”（n=0，不合法）。需优先判断这些特殊情况，避免后续逻辑错误。例如，若先处理“有大人和小孩”的情况，可能漏掉“n=0且m>0”的非法情况。
    * 💡 **学习笔记**：编程时，优先处理特殊/非法情况，再处理一般情况，能避免逻辑漏洞。

### ✨ 解题技巧总结
<summary_best_practices>
- **问题分解**：将复杂问题拆解为“只有大人”“只有小孩”“两者都有”三种情况，分别处理。
- **数学抽象**：通过观察极端情况（如“每个大人带一个小孩”“一个大人带所有小孩”），抽象出数学公式（`max(n,m)`和`n+m-1`）。
- **边界优先**：代码中先判断非法/特殊情况（如`n==0且m>0`），再处理一般情况，提高鲁棒性。
</summary_best_practices>

* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <algorithm> // 用于max函数

        int n, m;
        cin >> n >> m;

        if (m == 0) { // 只有大人
            cout << n << " " << n << endl;
        } else if (n == 0) { // 只有小孩（非法）
            cout << "Impossible" << endl;

    }
    ```

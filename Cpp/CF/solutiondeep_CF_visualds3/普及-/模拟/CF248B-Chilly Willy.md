# 题目信息

# Chilly Willy

## 题目描述

Chilly Willy loves playing with numbers. He only knows prime numbers that are digits yet. These numbers are $ 2 $ , $ 3 $ , $ 5 $ and $ 7 $ . But Willy grew rather bored of such numbers, so he came up with a few games that were connected with them.

Chilly Willy wants to find the minimum number of length $ n $ , such that it is simultaneously divisible by all numbers Willy already knows ( $ 2 $ , $ 3 $ , $ 5 $ and $ 7 $ ). Help him with that.

A number's length is the number of digits in its decimal representation without leading zeros.

## 样例 #1

### 输入

```
1
```

### 输出

```
-1
```

## 样例 #2

### 输入

```
5
```

### 输出

```
10080```

**本题通用核心C++实现参考**
* **说明**：本代码综合了多个优质题解的思路，通过特判和循环节数组实现，逻辑清晰，适合初学者。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    using namespace std;

    int main() {
        int n;
        cin >> n;
        string cycle[] = {"050", "080", "170", "020", "200", "110"}; // 循环节数组
        if (n == 1 || n == 2) {
            cout << -1 << endl;
        } else if (n == 3) {
            cout << 210 << endl;
        } else {
            cout << 1; // 首位1
            for (int i = 1; i <= n - 4; ++i) { // 中间n-4个0
                cout << 0;
            }
            int idx = (n - 3) % 6; // 计算循环节索引
            cout << cycle[idx] << endl;
        }
        return 0;
    }
    ```
* **代码解读概要**：
    代码首先处理n=1、n=2的无解情况，n=3时直接输出210。对于n≥4的情况，先输出首位1，接着输出n-4个0（填充中间部分），最后根据(n-3)%6计算循环节索引，输出对应的后三位。

* **核心代码片段**：
    ```cpp
    string s[6]={"110","050","080","170","020","200"};
    int main(){
        int n; cin>>n;
        switch(n){
            case 1:case 2:cout<<-1;break;
            case 3:cout<<210;break;
            default:{
                cout<<1;
                for(int i=1;i<=n-4;i++) cout<<0;

            }
        } 
    }
    ```
* **代码解读**：
    这段代码通过switch语句简洁处理n=1、2、3的情况。default分支中，先输出首位1，然后循环输出n-4个0（中间部分），最后用(n-3)%6计算索引p，输出s[p]作为后三位。数组s的顺序与n的对应关系通过观察规律确定（如n=4时p=1，对应s[1]="050"）。
* 💡 **学习笔记**：switch语句适合处理多分支特判，代码更易读。

    ```cpp
    const string msw[10] = {"110", "050", "080", "170", "020", "200"};
    int main() {
        cin >> n;

            cout << 1;

* **代码解读**：

* 💡 **学习笔记**：数组存储循环节时，需验证索引与实际值的对应关系。

**题解三：bj12z_donglinxi**
* **亮点**：动态计算后三位，避免硬编码循环节。
* **核心代码片段**：
    ```cpp
    int t = 50;
    for(int i = 1 ; i <= n - 4 ; i++) {
        cout << 0;
        t = t * 10 % 210;
    }
    printf("%03d\n", t);
    ```
* **代码解读**：
    初始t=50（对应n=4的后三位050）。每次循环输出0时，t更新为t*10%210（模拟后三位左移一位并取模210）。例如n=5时，循环执行1次（n-4=1），t=50*10%210=500%210=80，输出080，与答案10080的后三位一致。
* 💡 **学习笔记**：动态计算可避免记忆循环节，适合理解数学原理。

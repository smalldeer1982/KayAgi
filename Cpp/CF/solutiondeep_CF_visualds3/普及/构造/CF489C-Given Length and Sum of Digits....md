# Given Length and Sum of Digits...

## 题目描述

You have a positive integer $ m $ and a non-negative integer $ s $ . Your task is to find the smallest and the largest of the numbers that have length $ m $ and sum of digits $ s $ . The required numbers should be non-negative integers written in the decimal base without leading zeroes.

## 样例 #1

### 输入

```
2 15
```

### 输出

```
69 96
```

## 样例 #2

### 输入

```
3 0
```

### 输出

```
-1 -1
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：贪心算法（Greedy）

🗣️ **初步分析**：
解决这道题的关键是**贪心算法**——就像分蛋糕时，要“最小化”蛋糕的“前面大小”（比如想让数字最小，就把大数字放在后面），要“最大化”就把大数字放在前面。简单来说：  
- **最小数**：把大数字尽量往低位放（比如15分成2位，要最小就是69，而不是96）；  
- **最大数**：把大数字尽量往高位放（比如15分成2位，最大就是96）。  

但有两个核心难点：  
1. **特判无解情况**：比如数字和s=0但长度m>1（不能有前导零），或者s>9*m（所有位都是9也达不到和）；  
2. **最小数的构造**：最高位必须是1（不能有前导零），剩下的和要尽量给低位。  

### 可视化设计思路
我会用**8位像素风格**做动画：左边是“最小数构造机”，右边是“最大数构造机”。动画里用不同颜色的像素块代表数字（比如蓝色是已填数字，黄色是当前处理位），填数字时会有“叮”的音效，完成时播放胜利音乐。比如构造最小数时，最高位先亮“1”，然后从最后一位开始逐个填“9”，剩下的余数填在对应位；构造最大数时从第一位开始填“9”，直到用完所有和。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”都藏在细节里！结合优质题解，我提炼了3个核心难点和解决策略：
</difficulty_intro>

1.  **关键点1：如何判断无解？**  
    * **分析**：有两种情况一定无解：  
      - s=0但m>1（比如要3位数，和为0，只能是000，但前导零不允许）；  
      - s>9*m（比如3位数，和为30，所有位都是9也才27，不够）。  
      优质题解都会先处理这两种情况，直接输出-1 -1。  
    * 💡 **学习笔记**：先判无解，再构造数字，能避免后续逻辑出错！

2.  **关键点2：如何构造最小数？**  
    * **分析**：最小数的核心是“高位尽量小，低位尽量大”，但最高位必须≥1。所以步骤是：  
      1. 先给最高位填1（占住位置，避免前导零）；  
      2. 剩余和s-1，从低位到高位填9（能填9就填，不够填余数）。  
      比如样例1：m=2，s=15，先填最高位1，剩余14，从低位开始填9（用掉9，剩5），然后填到第二位（1+5=6），结果就是69。  
    * 💡 **学习笔记**：最小数要“反着填”——从最后一位开始！

3.  **关键点3：如何构造最大数？**  
    * **分析**：最大数的核心是“高位尽量大”，所以从第一位开始填9，直到用完所有和。比如样例1：m=2，s=15，第一位填9（用掉9，剩6），第二位填6，结果就是96。  
    * 💡 **学习笔记**：最大数要“顺着填”——从第一位开始！

### ✨ 解题技巧总结
- **技巧A：先判无解**：永远把边界情况（比如s=0、s>9*m）放在最前面处理，避免后续逻辑绕晕；  
- **技巧B：数组模拟数字**：用数组存每一位数字，方便控制填数顺序（比如最小数从低位到高位）；  
- **技巧C：剩余和跟踪**：用一个变量（比如`tmp`）跟踪剩余的数字和，填数时不断减去已用的数字，避免算错。


## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个综合优质题解的**通用核心实现**，帮你把握整体框架：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了“晨曦墨凝”和“Zaku”的思路，逻辑清晰、特判全面，适合作为模板使用。
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;

  const int MAXN = 105;
  int min_num[MAXN], max_num[MAXN];

  int main() {
      int m, s;
      cin >> m >> s;

      // 特判无解
      if ((s == 0 && m != 1) || s > 9 * m) {
          cout << "-1 -1" << endl;
          return 0;
      }
      if (s == 0 && m == 1) {
          cout << "0 0" << endl;
          return 0;
      }

      // 构造最小数：最高位1，剩余和s-1，从低位到高位填
      int tmp = s - 1;
      min_num[1] = 1; // 最高位（数组第1位）
      for (int i = m; i >= 1 && tmp > 0; --i) {
          if (tmp >= 9) {
              min_num[i] = 9;
              tmp -= 9;
          } else {
              min_num[i] = tmp;
              tmp = 0;
          }
      }

      // 构造最大数：从高位到低位填9
      tmp = s;
      for (int i = 1; i <= m && tmp > 0; ++i) {
          if (tmp >= 9) {
              max_num[i] = 9;
              tmp -= 9;
          } else {
              max_num[i] = tmp;
              tmp = 0;
          }
      }

      // 输出结果
      for (int i = 1; i <= m; ++i) cout << min_num[i];
      cout << " ";
      for (int i = 1; i <= m; ++i) cout << max_num[i];
      cout << endl;

      return 0;
  }
  ```
* **代码解读概要**：  
  1. **特判**：先处理无解情况；  
  2. **最小数构造**：用`min_num`数组，最高位先填1，剩余和从低位到高位填9；  
  3. **最大数构造**：用`max_num`数组，从高位到低位填9；  
  4. **输出**：依次输出最小数和最大数。

<code_intro_selected>
接下来看优质题解的**核心片段**，挖掘它们的“小技巧”：
</code_intro_selected>

**题解一：来源：晨曦墨凝**  
* **亮点**：用`tmp`变量跟踪剩余和，逻辑直接，遍历顺序明确。
* **核心代码片段**：
  ```cpp
  // 最小数构造
  tmp = m - 1; // 这里m是原题的s，注意变量名！
  mi[1] = 1;
  for (i = n; i >= 1 && tmp; --i) { // n是原题的m（长度）
      if (tmp >= 9) {
          mi[i] += 9;
          tmp -= 9;
      } else {
          mi[i] += tmp;
          break;
      }
  }
  ```
* **代码解读**：  
  作者用`mi`数组存最小数，`i`从长度`n`（低位）往1（高位）遍历。`tmp`是剩余和（s-1），能填9就填，否则填剩余的数。比如样例1中，`tmp=14`，i从2开始：填9（tmp剩5），然后i=1时填5，结果就是69。  
* 💡 **学习笔记**：遍历顺序是最小数的关键——从低位到高位！

**题解二：来源：Zaku**  
* **亮点**：用`sum`变量直接处理剩余和，代码简洁。
* **核心代码片段**：
  ```cpp
  // 最大数构造
  sum = s;
  for(int i=0;i<m;i++){
      if(sum-9>=0) Max[i]=9;
      else Max[i]=sum;
      sum-=Max[i];
  }
  ```
* **代码解读**：  
  作者用`Max`数组存最大数，从i=0（高位）到m-1（低位）遍历。`sum`是剩余和，能填9就填，否则填剩余的数。比如样例1中，sum=15，i=0时填9（sum剩6），i=1时填6，结果就是96。  
* 💡 **学习笔记**：最大数要“顺着填”——从高位到低位！

**题解三：来源：TheSky233**  
* **亮点**：用`num`数组复用，减少变量。
* **核心代码片段**：
  ```cpp
  // 最小数构造
  len=m,tot=s;
  while(tot>0 && len>=1){
      if(tot>9) num[len--]=9,tot=tot-9;
      else {num[len]=tot-1,num[1]++;break;}
  }
  ```
* **代码解读**：  
  作者用`num`数组先存最小数，`len`是当前处理的位（从m开始，即低位），`tot`是剩余和。如果`tot>9`，填9并减9；否则填`tot-1`，并给最高位（num[1]）加1（因为最高位要≥1）。比如样例1中，tot=15，len=2时填9（tot剩6），然后len=1时填6-1=5，num[1]加1变成6，结果就是69。  
* 💡 **学习笔记**：复用数组能减少代码量，但要注意变量的含义！

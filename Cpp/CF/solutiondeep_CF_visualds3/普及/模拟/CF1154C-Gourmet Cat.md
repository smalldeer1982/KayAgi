# Gourmet Cat

## 题目描述

Polycarp has a cat and his cat is a real gourmet! Dependent on a day of the week he eats certain type of food:

- on Mondays, Thursdays and Sundays he eats fish food;
- on Tuesdays and Saturdays he eats rabbit stew;
- on other days of week he eats chicken stake.

Polycarp plans to go on a trip and already packed his backpack. His backpack contains:

- $ a $ daily rations of fish food;
- $ b $ daily rations of rabbit stew;
- $ c $ daily rations of chicken stakes.

Polycarp has to choose such day of the week to start his trip that his cat can eat without additional food purchases as long as possible. Print the maximum number of days the cat can eat in a trip without additional food purchases, if Polycarp chooses the day of the week to start his trip optimally.

## 说明/提示

In the first example the best day for start of the trip is Sunday. In this case, during Sunday and Monday the cat will eat fish food, during Tuesday — rabbit stew and during Wednesday — chicken stake. So, after four days of the trip all food will be eaten.

In the second example Polycarp can start his trip in any day of the week. In any case there are food supplies only for one week in Polycarps backpack.

In the third example Polycarp can start his trip in any day, excluding Wednesday, Saturday and Sunday. In this case, the cat will eat three different dishes in three days. Nevertheless that after three days of a trip there will be $ 99 $ portions of rabbit stew in a backpack, can cannot eat anything in fourth day of a trip.

## 样例 #1

### 输入

```
2 1 1
```

### 输出

```
4
```

## 样例 #2

### 输入

```
3 2 2
```

### 输出

```
7
```

## 样例 #3

### 输入

```
1 100 1
```

### 输出

```
3
```

## 样例 #4

### 输入

```
30 20 10
```

### 输出

```
39
```

> 解决“Gourmet Cat”这道题，关键在于结合“模拟”和“枚举”两种方法。简单来说，“枚举”就是逐一尝试所有可能的出发日（共7种），而“模拟”则是按照猫咪的饮食规律，逐日消耗食物，直到某类食物不足为止。在本题中，由于食物量可能很大（如题目中a、b、c可达7×10⁸），直接暴力模拟所有天数会超时，因此需要先用数学方法计算“整周数”，将问题规模缩小，再对剩余食物进行枚举和模拟。
   - **题解思路**：所有优质题解的核心思路一致：先计算能完整吃多少周（整周数=min(a/3, b/2, c/2)），减去整周消耗的食物后，枚举7种出发日，模拟剩余食物能吃的天数，最终总天数=整周数×7+剩余天数的最大值。
   - **核心难点**：如何正确计算整周数？如何高效模拟剩余食物的消耗？如何确保枚举所有出发日时不遗漏最优解？

1.  **关键点1**：如何计算整周数？
    * **分析**：猫咪每周固定消耗3鱼、2兔、2鸡，因此整周数由三种食物能支撑的周数的最小值决定（即`min(a/3, b/2, c/2)`）。例如，若a=2、b=1、c=1（样例1），则`a/3=0`，整周数为0，剩余食物即为原始量。
    * 💡 **学习笔记**：整周数是“木桶效应”的体现，由最紧缺的食物决定。

2.  **关键点2**：如何正确模拟剩余食物的消耗？
    * **分析**：剩余食物量（a', b', c'）需满足`a'=a-整周数×3`，同理b'、c'。模拟时，从出发日开始逐日消耗，直到某类食物不足。例如，出发日为周日（对应样例1），消耗顺序为周日（鱼）、周一（鱼）、周二（兔）、周三（鸡），4天后食物耗尽。
    * 💡 **学习笔记**：模拟时需严格按照每周的饮食规律，用取模运算（如`j%7`）处理日期循环。

3.  **关键点3**：如何枚举所有出发日并找到最大值？
    * **分析**：共7种出发日（周一到周日），需逐一模拟每种情况，记录能吃的天数，最终取最大值。例如，样例3中a=1、b=100、c=1，整周数为0，枚举发现无论选哪三天（如鱼、兔、鸡各1次），最多吃3天。
    * 💡 **学习笔记**：枚举是解决“有限可能性”问题的有效方法，本题因只有7种可能，枚举成本极低。

## 4. C++核心代码实现赏析

<code_intro_overall>
在深入分析具体题解的精妙片段之前，让我们先来看一个能够完整解决本题的通用核心C++实现参考。

**本题通用核心C++实现参考**
* **说明**：本代码综合了yeshubo_qwq和一扶苏一的题解思路，结构清晰，适合学习。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <algorithm>
    using namespace std;

    int main() {
        int a, b, c;
        cin >> a >> b >> c;
        // 计算整周数
        int weeks = min({a / 3, b / 2, c / 2});
        a -= weeks * 3;
        b -= weeks * 2;
        c -= weeks * 2;
        int max_days = 0;
        // 枚举7种出发日（0-6对应周日到周六）
        for (int start = 0; start < 7; ++start) {
            int ra = a, rb = b, rc = c;
            int days = 0;
            int current_day = start;
            while (true) {
                switch (current_day % 7) {
                    case 0: case 1: case 4: // 周日（0）、周一（1）、周四（4）
                        if (ra-- <= 0) break;
                        break;
                    case 2: case 6: // 周二（2）、周六（6）
                        if (rb-- <= 0) break;
                        break;
                    default: // 周三（3）、周五（5）
                        if (rc-- <= 0) break;
                }
                days++;
                current_day++;
            }
            max_days = max(max_days, days);
        }
        cout << weeks * 7 + max_days << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码首先计算整周数（`weeks`），减去整周消耗的食物后，枚举7种出发日（`start`从0到6，对应周日到周六）。对每个出发日，模拟逐日消耗食物（`switch`语句根据当前日期选择食物类型），直到某类食物不足。最终总天数为整周数×7加上剩余天数的最大值。

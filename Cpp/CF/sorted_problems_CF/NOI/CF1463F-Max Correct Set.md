# Max Correct Set

## 题目描述

规定一组正整数 $S$。当且仅当满足以下条件时该组正整数成立：

- $S \subseteq \{1,2,...,n\}$

- 如果 $a \in s$ 并且 $b \in s$，那么 $|a - b| \not ={x}$ 并且 $|a - b| \not ={y}$
  
对于给定的数值 $n,x,y$，你需要找到成立数组的最大长度

## 输入格式

一行，包括三个整数 $n,x,y$ $(1 \leq n \leq 10 ^ 9; 1 \leq x, y \leq 22)$

## 输出格式

输出一个整数——成立数组的最大长度

## 样例 #1

### 输入

```
10 2 5
```

### 输出

```
5
```

## 样例 #2

### 输入

```
21 4 6
```

### 输出

```
9
```

## 样例 #3

### 输入

```
1337 7 7
```

### 输出

```
672
```

## 样例 #4

### 输入

```
455678451 22 17
```

### 输出

```
221997195
```

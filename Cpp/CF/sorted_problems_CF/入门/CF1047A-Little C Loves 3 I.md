# Little C Loves 3 I

## 题目描述

**题目大意：**
找三个数$a,b,c$，使得$a+b+c==n$且$a,b,c$都不能被$3$整除

## 输入格式

一行一个正整数$n$

## 输出格式

三个正整数$a,b,c$，输出任意一组即可

## 样例 #1

### 输入

```
3

```

### 输出

```
1 1 1
```

## 样例 #2

### 输入

```
233
```

### 输出

```
77 77 79

```

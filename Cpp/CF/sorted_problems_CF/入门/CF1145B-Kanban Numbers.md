# Kanban Numbers

## 输入格式

The input contains a single integer $ a $ ( $ 1 \le a \le 99 $ ).

## 输出格式

Output "YES" or "NO".

## 样例 #1

### 输入

```
5

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
13

```

### 输出

```
NO

```

## 样例 #3

### 输入

```
24

```

### 输出

```
NO

```

## 样例 #4

### 输入

```
46

```

### 输出

```
YES

```

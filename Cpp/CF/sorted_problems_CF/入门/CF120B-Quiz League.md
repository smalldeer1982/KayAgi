# Quiz League

## 题目描述

#### 题意简述

给定 $n$ 个 $0$ 或 $1$ 的数围成一个环，第一个数的位置是 $1$。现在给定一个 $k$，从第 $k$ 个数开始顺时针遍历整个环，求第一个 $1$ 的位置。

## 输入格式

第一行 $2$ 个数，$n$ 和 $k$。

第二行 $n$ 个数，以顺时针顺序依次读入整个环上的数。

## 输出格式

一个数，代表从第 $k$ 个数开始顺时针遍历整个环时第一个 $1$ 的位置。

#### 数据范围与约定
$1 \leqslant k \leqslant n \leqslant 1000$。

保证环上的每一个数都是 $0$ 或 $1$。

## 样例 #1

### 输入

```
5 5
0 1 0 1 0

```

### 输出

```
2

```

## 样例 #2

### 输入

```
2 1
1 1

```

### 输出

```
1

```

# Badges

## 题目描述

简化版题意:

有两个取值为 $[0, b]$ 和 $[0, g]$ 的整数 $B$ 和 $G$ , 求有多少对 $(B, G)$ 满足 $B + G = n$ ?

## 输入格式

三行, 每行一个整数, 依次为 $b$ , $g$ , $n$ .

## 输出格式

一行一个整数, 表示满足条件的不同 $(B, G)$ 对的个数.

## 说明/提示

$1 \leq b, g \leq 300$, $1 \leq n \leq b + g$

## 样例 #1

### 输入

```
5
6
3

```

### 输出

```
4

```

## 样例 #2

### 输入

```
5
3
5

```

### 输出

```
4

```

# Canvas Frames

## 题目描述

有$n$根木棍，每根木棍有一个长度，问你用这些木棍最多能够拼成几个矩形（木棍不能被折断）。

例如拼成一个$h \times w$的矩形就需要两根长度为$h$的木棍和两根长度为$w$的木棍。

## 输入格式

第一行一个整数$n$。

下一行共有$n$个整数，表示每根木棍的长度。

## 输出格式

一行一个整数，表示最多能够拼成的矩形数。

## 样例 #1

### 输入

```
5
2 4 3 2 3

```

### 输出

```
1
```

## 样例 #2

### 输入

```
13
2 2 4 4 4 4 6 6 6 7 7 9 9

```

### 输出

```
3
```

## 样例 #3

### 输入

```
4
3 3 3 5

```

### 输出

```
0
```

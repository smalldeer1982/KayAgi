# Minutes Before the New Year

## 题目描述

新年将至，我们想知道离新年还有多久。（新年在后一天的0时0分，我们用24时计时法）

## 输入格式

第一行一个整数t，表示数据组数。

接下来t行，每行有两个不含前导零的整数hh和mm，表示当前是hh时mm分。

（$0≤hh<24,0≤mm<60$ ）

## 输出格式

对于每组数据，输出当前的时间离新年还有多少分钟。

## 样例 #1

### 输入

```
5
23 55
23 0
0 1
4 20
23 59

```

### 输出

```
5
60
1439
1180
1

```

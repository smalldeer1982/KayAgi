# ...And after happily lived ever they

## 输入格式

The input contains a single integer $ a $ ( $ 0 \le a \le 63 $ ).

## 输出格式

Output a single number.

## 样例 #1

### 输入

```
2
```

### 输出

```
2
```

## 样例 #2

### 输入

```
5
```

### 输出

```
24
```

## 样例 #3

### 输入

```
35
```

### 输出

```
50
```

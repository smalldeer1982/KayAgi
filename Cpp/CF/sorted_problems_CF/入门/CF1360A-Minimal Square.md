# Minimal Square

## 题目描述

## 题意

有两个完全一致且**不重合**的矩形，长为 $a$，宽为 $b$。二矩形都在一个大正方形型内。矩形可以任意旋转，但是要求**所有矩形的边和正方形的边满足其中任意二条边都互相平行或垂直**。

当然有很多情况满足上述条件。您需要给出正方形**最小**可能的**面积**。

## 输入格式

**本题含有多组测试数据**。

第一行一个整数 $t$，为数据组数。

对于每一组数据一行二整数 $a$ 和 $b$，表示矩形的边长。

## 输出格式

对于每一组数据一行一个数，表示可能的最小正方形面积。

## 说明/提示

$1 \leq t \leq 10^4$

$1 \leq a,b \leq 100$

## 样例 #1

### 输入

```
8
3 2
4 2
1 1
3 1
4 7
1 3
7 4
100 100
```

### 输出

```
16
16
4
9
64
9
64
40000
```

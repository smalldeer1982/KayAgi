# Required Remainder

## 题目描述

输入$x,y,n(2\le x\le10^9,0\le y<x,y\le n\le10^9)$  

求满足$k\bmod x=y\ (0\le k\le n)$的$k$的最大值  

多组数据, 数据组数$1\le t\le5\cdot10^4$

## 输入格式

第一行,一个数$t$  

第$2$到第$t+1$行, 每行一组$x,y,n$

## 输出格式

一行一个满足条件的非负整数$k$  

可以证明在给定的条件下, $k$一定存在

## 样例 #1

### 输入

```
7
7 5 12345
5 0 4
10 5 15
17 8 54321
499999993 9 1000000000
10 5 187
2 0 999999999
```

### 输出

```
12339
0
15
54306
999999995
185
999999998
```

# LCM Problem

## 题目描述

规定 $\text{LCM(x,y)}$ 表示 $x,y$ 的最小公倍数。

现在给出两个整数 $l,r$ 。请找出一组满足条件的 $x,y$ ，使得 $l \le x \lt y \le r$ 且 $l \le \text{LCM(x,y)} \le r$。

## 输入格式

输入的第一行只有一个整数 $t$ ( $1 \le t \le 10000$ )。

接下来的$t$行，每行输入两个整数 $l,r$ ( $1 \le l \lt r \le 10^9$ ) ，表示一组数据。

## 输出格式

对于每组数据，输出两个整数：

- 如果无法找到符合条件的 $x,y$ ，则输出两个 $-1$ 。

- 如果能够找到符合条件的 $x,y$ ，输出 $x,y$ 。

## 样例 #1

### 输入

```
4
1 1337
13 69
2 4
88 89
```

### 输出

```
6 7
14 21
2 4
-1 -1
```

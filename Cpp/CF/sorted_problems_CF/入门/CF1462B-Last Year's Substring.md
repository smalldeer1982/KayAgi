# Last Year's Substring

## 题目描述

本题给你一个字符串，问删去其中一个可以为空的字串后，能否使其变为"$2020$"。

## 输入格式

第一行一个整数，为测试数据数量。

对于每个数据，第一行一个整数，为字符串长度。第二行输入字符串。

## 输出格式

对于每个数据点，如果可以，输出$YES$，否则输出$NO$。

## 样例 #1

### 输入

```
6
8
20192020
8
22019020
4
2020
5
20002
6
729040
6
200200
```

### 输出

```
YES
YES
YES
NO
NO
NO
```

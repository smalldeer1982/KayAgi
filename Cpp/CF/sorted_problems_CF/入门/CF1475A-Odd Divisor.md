# Odd Divisor

## 题目描述

给定一个正整数 $n$，如果 $n$ 有大于 $1$ 的奇因数，输出`YES`，否则输出`NO`。

## 输入格式

第一行一个正整数 $t$ 表示数据组数。

接下来 $t$ 行，每行一个正整数 $n$。

## 输出格式

$t$ 行，每行一个字符串表示答案。
### 注意事项
$1\le t\le10^4,1\le n\le10^{14}$。

输出的字符串不区分大小写。（所以你输出`yEs`也是阔以的）

### 输入

```
6

```

```
YES

YES

NO
```

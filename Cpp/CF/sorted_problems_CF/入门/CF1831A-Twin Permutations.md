# Twin Permutations

## 题目描述

### 题目大意
给出一个长为 $n$ 的排列 $a$ 请构造一个长为 $n$ 的排列 $b$ 使 $c_i = a_i + b_i$ 并且 $c$ 是一个不下降序列

## 输入格式

第一行，一个数 $T$ ,表示 $T$ 组数据

接下来 $T$ 组数据

第一行一个整数 $n$
, 第二行 $n$ 个数表示 $a_i$

## 输出格式

共 $T$ 行 ,每行 $n$ 个数表示 排列 $b$

## 样例 #1

### 输入

```
5
5
1 2 4 5 3
2
1 2
1
1
3
3 2 1
4
1 4 3 2
```

### 输出

```
1 2 4 3 5
2 1
1
1 2 3
1 2 3 4
```

# Wallet Exchange

## 题目描述

Alice 和 Bob 感到无聊，所以他们决定用钱包玩个游戏。Alice 的钱包里有 $a$ 枚硬币，而 Bob 的钱包里有 $b$ 枚硬币。

两位玩家轮流进行，Alice 先走。在每一轮中，玩家将**按顺序**依次执行以下步骤：

1. 可以选择与对方交换钱包，也可以不交换。
2. 从玩家当前钱包中取出 $1$ 枚硬币。在执行之前钱包里的硬币数量不能为 $0$。

轮到自己时无法进行有效进行的玩家输。如果 Alice 和 Bob 都以最佳方式玩，确定谁将赢得比赛。

## 输入格式

每个测试包含多个测试用例。第一行包含一个整数 $t$（$1 \le t \le 1000$）——测试用例的数量。测试用例的描述如下。


每个测试用例的第一行也是唯一一行包含两个整数 $a$ 和 $b$（$ 1 \le a, b \le 10^9 $）——分别是 Alice 和 Bob 钱包中的硬币数量。

## 输出格式

对于每个测试用例，如果 Alice 将赢得游戏，则输出 `Alice`，如果 Bob 将赢得游戏，则输出 `Bob`。

## 说明/提示

对于第一个测试用例，下面展示了该游戏的一个示例：

- Alice 在她的这一步行动选择不与 Bob 交换钱包。此时，$a=0$ 且 $b=1$。
- 由于 Alice 的钱包是空的，Bob 在他的这一步行动必须选择不交换他们的钱包。此时，$a=0$ 且 $b=0$。
- 由于 Alice 和 Bob 的钱包都空了，Alice 无法行动。因此，Bob 获胜。

对于第二个测试用例，下面展示了该游戏的一个示例：

- Alice 在她的这一步行动选择与 Bob 交换钱包。此时，$a=3$ 且 $b=1$。
- Bob 在他的这一步行动选择与 Alice 交换钱包。此时，$a=1$ 且 $b=2$。
- Alice 在她的这一步行动选择不与 Bob 交换钱包。此时，$a=0$ 且 $b=2$。
- 由于 Alice 的钱包是空的，Bob 在他的这一步行动只能选择不与 Alice 交换钱包。此时，$a=0$ 且 $b=1$。
- 由于 Alice 的钱包是空的，Alice 在她的这一步行动只能选择和 Bob 交换钱包。此时，$a=0$ 且 $b=0$。
- 由于 Alice 和 Bob 的钱包都空了，Bob 无法行动。因此，Alice 获胜。

# Catch the Coin

## 题目描述

# Catch the Coin


Monocarp 参观了一个复古街机俱乐部，在那里他对一个名为“捉硬币”游戏机产生了兴趣。

游戏非常简单。屏幕表示一个坐标网格，其中：

- X 轴从左到右；
- Y 轴从下到上；
- 屏幕的中心坐标是 $ (0, 0) $ 。

游戏开始时，  Monocarp 位于屏幕中心，屏幕上有 $ n $ 个硬币，且第 $ i $ 个硬币位于坐标 $ (x_i, y_i) $ 。所有硬币的坐标都不同，并且不等于 $ (0, 0) $ 。

在一秒钟内，Monocarp 可以移动到八个方向之一。如果  Monocarp 在坐标 $ (x, y) $ ，那么它可以到达以下任意一个坐标：$ (x, y + 1) $，$ (x + 1, y + 1) $，$ (x + 1, y) $，$ (x + 1, y - 1) $，$ (x, y - 1) $，$ (x - 1, y - 1) $，$ (x - 1, y) $，$ (x - 1, y + 1) $。

如果  Monocarp 到达有硬币的坐标，则他可以收集该硬币。

在 Monocarp 移动后，所有硬币向下掉落 $ 1 $ ，即它们从 $ (x, y) $ 移动到 $ (x, y - 1) $。假设游戏场地在所有方向上都是无限的。

Monocarp 想要收集至少一个硬币，但他无法决定去捡哪个硬币。请帮助他确定，对于每个硬币，他是否可以收集到。

## 输入格式

第一行包含一个整数 $ n $ ( $ 1 \le n \le 500 $ )，表示硬币的数量。

接下来的 $ n $ 行中，第 $ i $ 行包含两个整数 $ x_i $ 和 $ y_i $ ( $ -50 \le x_i, y_i \le 50 $ )，表示第 $ i $ 个硬币的横坐标和纵坐标。所有硬币的坐标都不同，且没有硬币位于 $ (0, 0) $。

## 输出格式

对于每个硬币，如果 Monocarp 可以收集到它，则输出 "YES"。否则，输出 "NO"。

## 样例 #1

### 样例输入 #1

```
5
24 42
-2 -1
-1 -2
0 -50
15 0
```

### 样例输出 #1

```
YES
YES
NO
NO
YES
```

## 说明/提示

注意样例中的第二个硬币。Monocarp 可以先从 $ (0, 0) $ 移动到 $ (-1, -1) $。然后硬币向下掉落 $ 1 $ 到 $ (-2, -2) $。最后 Monocarp 移动到 $ (-2, -2) $ 并收集硬币。

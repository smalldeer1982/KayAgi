# Minimize!

## 题目描述

给你两个整数 $a$ 和 $b$ ($a≤b$) 。
在 $c$ ($a≤c≤b$) 的所有可能整数值中，求 $(c−a)+(b−c)$ 的最小值。

## 输入格式

第一行包含 $t$ ($1≤t≤55$) -测试用例数。
每个测试用例包含两个整数 $a$ 和 $b$ ($1≤a≤b≤10$) 。

## 输出格式

对于每个测试用例，另起一行输出 $(c−a)+(b−c)$ 的最小可能值。

## 样例 #1

### 输入

```
3
1 2
3 10
5 5
```

### 输出

```
1
7
0
```

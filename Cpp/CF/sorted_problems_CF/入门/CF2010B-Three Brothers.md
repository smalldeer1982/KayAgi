# Three Brothers

## 题目描述

### 题目翻译：

三兄弟相约见面。我们按以下顺序给他们编号：老大是 $1$ 号，老二是 $2$ 号，老三是 $3$ 号。

到了约定的时间，有一个兄弟迟到了。现在给出准时到达的两个兄弟的编号，你需要确定那个迟到的兄弟的编号。

## 输入格式

输入一行包含两个不同的整数 $a$ 和 $b$ （$1 \leq a, b \leq 3$，且 $a \neq b$），代表准时到达的两个兄弟的编号。编号顺序是任意的。

## 输出格式

输出一个整数，代表迟到的兄弟的编号。
```

## 样例 #1

### 输入

```
3 1
```

### 输出

```
2
```

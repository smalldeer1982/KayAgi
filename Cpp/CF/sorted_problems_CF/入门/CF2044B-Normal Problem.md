# Normal Problem

## 题目描述

玻璃上写着一串仅包含小写字母 $p$，$q$，$w$ 的字符串 $a$。求该玻璃水平翻转之后，从左到右看到的新字符串 $b$。

## 输入格式

第一行输入一个整数 $t$ $(1≤t≤100)$ ——测试数据的组数。

对于每一组测试数据，输入一行字符串 $a$ ——玻璃正面所看到的字符串。保证 $a$ 只包含小写字母 $p$，$q$，$w$。

## 输出格式

对于每一组测试数据，输出一行字符串$b$ ——该玻璃水平翻转之后所看到的字。

## 样例 #1

### 输入

```
5
qwq
ppppp
pppwwwqqq
wqpqwpqwwqp
pqpqpqpq
```

### 输出

```
pwp
qqqqq
pppwwwqqq
qpwwpqwpqpw
pqpqpqpq
```

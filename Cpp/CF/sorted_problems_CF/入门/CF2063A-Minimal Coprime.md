# Minimal Coprime

## 题目描述

如果 $l$ 和 $r$ 互质 $\text{*}$，则一段正整数 $[l，r]$ 称为互质段。

如果一个互质段 $[l，r]$ 不包含$\text{†}$任何不等于其自身的互质段，则称为最小互质段。

给定 $[l，r]$，一个正整数段，求出 $[l，r]$ 中包含的最小互质段的数量。 

$\text{*}$ 如果两个整数 $a$ 和 $b$ 只有一个正公约数，则它们互质。例如，数字 $2$ 和 $4$ 不是互质，因为它们都被 $2$ 和 $1$ 整除，但数字 $7$ 和 $9$ 是互质的，因为它们唯一的正公约数是 $1$ 。 

$\text{†}$ 当且仅当$l\le l'\le r'\le r$时，段$[l'，r']$包含在段$[l，r]$中

## 输入格式

每个测试包含多个测试数据。第一行包含测试用例的数量$t$（$1\le t\le 100$）。测试用例的描述如下。

每个测试数据输入两个整数 $l$ 和 $r$（$1\le l\le r\le 10^9$）。

## 输出格式

对于每个测试数据，在单独的一行中输出 $[l，r]$ 中包含的最小互质段的数量。

## 说明/提示

在第一个测试数据中，给定的段是 $[1,2]$。$[1,2]$ 中包含的段如下。

- $[1,1]$：此段是互质，因为数字 $1$ 和 $1$ 是互质的，并且此段中不包含任何其他段。因此，$[1,1]$ 是最小的互质段。 

- $[1,2]$：这段是互质段。然而，由于它包含 $[1,1]$这个互质段，因此 $[1,2]$ 不是最小互质段。 

- $[2,2]$：此段不是互质段，因为 $2$ 和 $2$ 共享 $2$ 正公约数：$1$ 和 $2$。 
因此，段 $[1,2]$ 包含 $1$ 个最小互质段。

### 输入

```
6
1 2
1 10
49 49
69 420
1 1
9982 44353
```

### 输出

```

9
0
```

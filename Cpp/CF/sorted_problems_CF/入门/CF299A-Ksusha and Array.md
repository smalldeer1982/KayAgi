# Ksusha and Array

## 题目描述

输入 $N$ 个数字，你的任务是**在这些数字中**找到一个能够整除所有这些数字的数。

## 输入格式

第一行输入 $N$（$1 \le N \le {10}^5$），第二行输入 $N$ 个数字 $a_i$（$1 \le a_i \le {10}^9$）。

## 输出格式

输出那个数字，若没有则输出 $-1$，若有多个，可任意输出其中一个。

## 样例 #1

### 输入

```
3
2 2 4

```

### 输出

```
2

```

## 样例 #2

### 输入

```
5
2 1 3 1 6

```

### 输出

```
1

```

## 样例 #3

### 输入

```
3
2 3 5

```

### 输出

```
-1

```

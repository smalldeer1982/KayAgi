# Shell Game

## 题目描述

经典的转纸杯游戏。给定小球初始所在的纸杯位置和 $3$ 次交换纸杯的操作，问你最后小球的位置。

## 输入格式

第一行一个整数。标识小球的初始位置，之后 $3$ 行每行两个整数 $a,b$ 表示交换 $a,b$ 纸杯。（要注意的是：最左边的杯子编号永远为 $1$，最右边的编号永远为 $3$，不会随着交换而改变）

## 输出格式

一个整数，表示最后小球所在的纸杯编号。

本题目中所有数字均不超过 $3$。

## 说明/提示

本题需要从 input.txt 读入输入，输出到 output.txt

## 样例 #1

### 输入

```
1
1 2
2 1
2 1

```

### 输出

```
2

```

## 样例 #2

### 输入

```
1
2 1
3 1
1 3

```

### 输出

```
2

```

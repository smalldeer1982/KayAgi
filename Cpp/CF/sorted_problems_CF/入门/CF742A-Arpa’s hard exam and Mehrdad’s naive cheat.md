# Arpa’s hard exam and Mehrdad’s naive cheat

## 题目描述

给你一个数N，问你1378的N次幂个位是多少

## 说明/提示

在第一个样例，1378^1是1378，所以个位是8

在第二个样例，1378^2是1898884，所以个位是4

## 样例 #1

### 输入

```
1

```

### 输出

```
8
```

## 样例 #2

### 输入

```
2

```

### 输出

```
4
```

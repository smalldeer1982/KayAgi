# Book Reading

## 题目描述

【题目大意】 最近Luba买了一本书，他希望在接下来的$n$  天里抽出$t$  秒看完这本书。（总计$t$  秒） 但是Luba每天会有一些工作，对于第$i$天，他需要花费$a_i$ 秒时间来工作。 问Luba最早在什么时候能看完这本书。（注意一天有$86400$ 秒）

## 输出格式

Translated by @Durant_Lee

## 样例 #1

### 输入

```
2 2
86400 86398

```

### 输出

```
2

```

## 样例 #2

### 输入

```
2 86400
0 86400

```

### 输出

```
1

```

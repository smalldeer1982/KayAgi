# Hamster Farm

## 题目描述

Dima有$n$ 只仓鼠，有$k$ 种盒子，每种盒子编号为1~k，可以装$a_i$ 只仓鼠，所有盒子都要装满仓鼠，剩下的仓鼠不装，Dima想让剩下的仓鼠最少，求应选盒子和盒子总数，如果有多种情况，输出任意一种。

## 输出格式

感谢 @WDLGZH2017  提供的翻译。

## 样例 #1

### 输入

```
19 3
5 4 10

```

### 输出

```
2 4

```

## 样例 #2

### 输入

```
28 3
5 6 30

```

### 输出

```
1 5

```

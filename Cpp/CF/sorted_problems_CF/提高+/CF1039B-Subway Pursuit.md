# Subway Pursuit

## 题目描述

### 题目大意：
交互题

在$1$到$n$里有一个运动的点，要求找到这个点，每次可以查询一个区间内有没有这个点，每次这个点往左或者往右移动$1$到$k$个位置，要求要在$4500$次查询内找到这个点的位置

## 输入格式

一行两个整数$n,k$

## 输出格式

每行输出两个整数$l,r$，表示查询区间

## 样例 #1

### 输入

```
10 2

Yes

No

Yes

Yes

```

### 输出

```
3 5

3 3

3 4

5 5

```

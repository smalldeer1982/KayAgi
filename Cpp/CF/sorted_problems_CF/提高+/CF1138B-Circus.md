# Circus

## 题目描述

**题意：** 有$n$组数（保证$n$是偶数），每组数中有两个数，第$i$组数中的数为$c_i$和$a_i$，其中$c_i,a_i$都为$0$或$1$

您需要将$n$组数分成两部分，满足：

- 每部分$\frac{n}{2}$组数。

- 第一部分中$c_i$的和=第二部分中$a_i$的和。

**数据范围：** $2\le n\le 5000$

## 输出格式

**样例1说明：** $c_1+c_4=1=a_2+a_3$，于是输出1 4.

## 样例 #1

### 输入

```
4
0011
0101

```

### 输出

```
1 4

```

## 样例 #2

### 输入

```
6
000000
111111

```

### 输出

```
-1

```

## 样例 #3

### 输入

```
4
0011
1100

```

### 输出

```
4 3

```

## 样例 #4

### 输入

```
8
00100101
01111100

```

### 输出

```
1 2 3 6

```

# XOR Inverse

## 题目描述

给定长度为 $n$ $(1\le n\le3\times 10^5)$ 的数列 $\{a_n\}$ $(0\le a_n\le 10^9)$，请求出最小的整数 $x$ 使 $\{a_n\oplus x\}$ 的逆序对数最少，其中 $\oplus$ 是异或

## 输入格式

第一行 $n$，第二行 $\{a_n\}$

## 输出格式

两个数，逆序对数和 $x$

## 样例 #1

### 输入

```
4
0 1 3 2
```

### 输出

```
1 0
```

## 样例 #2

### 输入

```
9
10 7 9 10 7 5 5 3 5
```

### 输出

```
4 14
```

## 样例 #3

### 输入

```
3
8 10 3
```

### 输出

```
0 8
```

# XOR Triangle

## 题目描述

给你一个数 $n$，问：有多少对数 $0\leq a,b,c \leq n$ 满足 $a \oplus b,b \oplus c,a \oplus c$  。三个数字构成了一个非退化三角形，也就是两条短边之和大于第三边的长度。$\oplus$ 表示二进制下的异或操作。

## 输入格式

一个数字 $n$，表示给定的 n 在二进制下的表示。

## 输出格式

输出答案 mod 998244353。

## 样例 #1

### 输入

```
101
```

### 输出

```
12
```

## 样例 #2

### 输入

```
1110
```

### 输出

```
780
```

## 样例 #3

### 输入

```
11011111101010010
```

### 输出

```
141427753
```

# Oranges and Apples

## 题目描述

已知有 $2N-1$ 个箱子，每个箱子里有一些苹果和橘子。你的任务是从中选择 $N$ 个箱子使得这 $N$ 个箱子中的苹果数目不小于所有箱子里苹果总数的一半，这 $N$ 个箱子中的橘子数目也不小于所有箱子里橘子总数的一半。

## 输出格式

Translated by 稀神探女

## 样例 #1

### 输入

```
2
2
10 15
5 7
20 18
1
0 0

```

### 输出

```
YES
1 3
YES
1

```

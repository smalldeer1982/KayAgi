# Jeff and Rounding

## 题目描述

题意:

你有一个长度为2n的序列,将其中的n个数向下取整,n个数向上取整,求|原序列的和 - 取整后序列的和|的最小值.

## 输入格式

第一行一个n.
第二行2n个三位小数x.

## 输出格式

|ans|的最小值(保留三位小数).

## 说明/提示

x:[0,10000]
n:[1,2000]

注意:

None.

感谢@尘染梦 提供的翻译

## 样例 #1

### 输入

```
3
0.000 0.500 0.750 1.000 2.000 3.000

```

### 输出

```
0.250

```

## 样例 #2

### 输入

```
3
4469.000 6526.000 4864.000 9356.383 7490.000 995.896

```

### 输出

```
0.279

```

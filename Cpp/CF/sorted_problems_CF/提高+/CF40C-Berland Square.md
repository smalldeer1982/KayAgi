# Berland Square

## 题目描述

集合(K,z)是以点(z,0)为圆心，分别以1、2、………K 为半径画出的同心圆。在平面上画了一个(N,x)集和一个(M,y)集，你必须找出这些集合把平面分成了多少部分。

## 输入格式

第一行包含整数N x M y
（1<=N,M<=100000,−100000<=x,y<=100000,x！=y）

## 输出格式

一个整数

## 样例 #1

### 输入

```
1 0 1 1

```

### 输出

```
4

```

## 样例 #2

### 输入

```
1 0 1 2

```

### 输出

```
3

```

## 样例 #3

### 输入

```
3 3 4 7

```

### 输出

```
17

```

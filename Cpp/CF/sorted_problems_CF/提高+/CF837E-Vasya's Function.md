# Vasya's Function

## 题目描述

Vasya正在学习数论。他定义了一个函数f(a, b)：
### 
f(a, 0) = 0;
f(a, b) = 1 + f(a, b – gcd(a, b)),gcd(a, b)就是a和b的最大公因数。

Vasya有两个数字x和y，并且他想要算出f(x,y)的值。他想要自己去算，但发现可能会需要很长的时间。所以他向你求助，请你给出一个能够快速得出答案的程序。

## 输出格式

------------
第一次提翻译，求采纳。

## 样例 #1

### 输入

```
3 5

```

### 输出

```
3

```

## 样例 #2

### 输入

```
6 3

```

### 输出

```
1

```

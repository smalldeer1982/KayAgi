# Pair Of Lines

## 题目描述

在平面直角坐标系上给出 $n$ 个点，求是否存在两条直线穿过所有点。

## 输入格式

第一行一个正整数 $n(1 \leq n \leq 10^5)$ 表示点数，接下来 $n$ 行每行两个整数 $x_i , y_i(|x_i| , |y_i| \leq 10^9)$ 表示一个点。保证不存在两个点的 $x_i,y_i$ 均相同。

## 输出格式

如果存在方案输出 `YES`，否则输出 `NO`。

## 样例 #1

### 输入

```
5
0 0
0 1
1 1
1 -1
2 2

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
5
0 0
1 0
2 1
1 1
2 3

```

### 输出

```
NO

```

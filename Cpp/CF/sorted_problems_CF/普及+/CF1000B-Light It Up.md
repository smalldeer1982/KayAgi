# Light It Up

## 题目描述

**题目大意：**

有一台灯，这个灯在时间为$0$时打开，$m$时关闭，在$0$到$m$这段时间内有$n$个时间点灯的状态会改变（即开变关，关变开），现在可以在**剩余**的时间点选一个让灯的状态改变一次，求这个灯最大亮着的时间

## 输入格式

第一行两个整数，$n,m$，$n<=10^5,m<=10^9$

以下一行有$n$个整数，即改变时间点

## 输出格式

一个整数，即最大亮着的时间

感谢@守望 提供翻译

## 样例 #1

### 输入

```
3 10
4 6 7

```

### 输出

```
8

```

## 样例 #2

### 输入

```
2 12
1 10

```

### 输出

```
9

```

## 样例 #3

### 输入

```
2 7
3 4

```

### 输出

```
6

```

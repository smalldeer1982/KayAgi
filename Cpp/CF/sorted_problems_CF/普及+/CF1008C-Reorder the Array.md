# Reorder the Array

## 题目描述

**题目大意：**

给定一个序列，序列里的元素可以以任意顺序重新排好

序列里某位置排序后的数可能会大于原数，问这样的位置最多能有多少个

## 输入格式

第一行一个整数$n$，表示序列长度

第二行$n$个整数，表示初始序列

## 输出格式

一个整数，表示满足条件的位置最多能有多少个

## 样例 #1

### 输入

```
7
10 1 1 1 5 5 3

```

### 输出

```
4

```

## 样例 #2

### 输入

```
5
1 1 1 1 1

```

### 输出

```
0

```

# Zero Quantity Maximization

## 题目描述

# 零数最大化
给出两个长度为 $n$ 的数组 $a,b$。

你想要创建一个长度为 $n$ 的数组 $c$，且$c_i=d*a_i+b_i,i\in[1, n]$，要求你给 $d$ 赋值，使得数组 $c$ 中的 $0$ 的个数最多并输出最多的 $0$ 的个数。

## 输入格式

第一行一个整数 $n$，
第二行 $n$ 个整数为数组 $a$，
第二行 $n$ 个整数为数组 $b$，

## 输出格式

一个数为数组 $c$ 中最多的 $0$ 的个数

## 样例 #1

### 输入

```
5
1 2 3 4 5
2 4 7 11 3

```

### 输出

```
2

```

## 样例 #2

### 输入

```
3
13 37 39
1 2 3

```

### 输出

```
2

```

## 样例 #3

### 输入

```
4
0 0 0 0
1 2 3 4

```

### 输出

```
0

```

## 样例 #4

### 输入

```
3
1 2 -1
-6 -12 6

```

### 输出

```
3

```

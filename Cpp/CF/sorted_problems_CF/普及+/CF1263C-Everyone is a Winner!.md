# Everyone is a Winner!

## 题目描述

给定$n$ 对于所有正整数$k$求出$\lfloor \frac{n}{k} \rfloor$可以有多少种不同的值 并求出所有可能的值

## 输入格式

第一行一个数$T$代表数据组数  
接下来$T$行每行一个数 代表这组数据的$n$

## 输出格式

第一行一个数$m$ 代表总共可能的不同取值数  
第二行$m$个数 用空格分隔 将所有可能的取值按递增顺序输出

## 样例 #1

### 输入

```
4
5
11
1
3

```

### 输出

```
4
0 1 2 5 
6
0 1 2 3 5 11 
2
0 1 
3
0 1 3 

```

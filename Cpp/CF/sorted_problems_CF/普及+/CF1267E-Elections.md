# Elections

## 题目描述

有n(n<=100)个人,m(m<=100)个投票站,每个投票站对每个人都有一定的票数(<=1000).  
第n个是坏蛋,请问去除掉哪几个投票站才能使他的票数不大于其他每个人中的一个，也就是不能让第n个人的总票数最多。

## 输入格式

第一行是两个整数，分别是n和m.  
接下来m行n列，代表每个投票站对这n个人的票数。

## 输出格式

第一行一个整数k，代表需要去掉几个投票站。  
第二行有k个数，代表投票站的号数，不要求字典序.

## 样例 #1

### 输入

```
5 3
6 3 4 2 8
3 7 5 6 7
5 2 4 7 9

```

### 输出

```
2
3 1 

```

## 样例 #2

### 输入

```
2 1
1 1

```

### 输出

```
0


```

## 样例 #3

### 输入

```
3 3
2 3 8
4 2 9
3 1 7

```

### 输出

```
3
1 2 3 

```

# Present to Mom

## 题目描述

求一个 $n\times m$ 的字符矩阵内，有多少个不同的矩形包含不少于 $k$ 个下述图形：

```
 1
111
 1
```

注意，这些图形可以相互重叠，但是不能完全重叠。

## 输入格式

第一行三个正整数，$n,m,k$。

接下来 $n$ 行，每行 $m$ 个字符描述矩阵。这些字符只可能是```0```或```1```。

## 输出格式

一行一个正整数，代表询问的答案。

（提示：本题请用```%I64d```或```cout```输出```long long```）

## 样例 #1

### 输入

```
4 6 2
111000
111100
011011
000111

```

### 输出

```
6

```

## 样例 #2

### 输入

```
5 5 4
11111
11111
11111
11111
11111

```

### 输出

```
9

```

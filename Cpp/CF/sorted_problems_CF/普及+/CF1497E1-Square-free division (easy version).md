# Square-free division (easy version)

## 题目描述

### 题意描述

- 这是简单版，此题中 $k=0$。

给出一串长为 $n$ 的序列 $a_1,a_2,a_3...a_n$。

把它分成尽量少的块使每一块中任意两数的乘积不是一个完全平方数。

输出最少的块数。

## 输入格式

- 多组数据 $T$ $(1<=T<=1000)$ 。
- 对每组数据，第一行为两个整数 $n,k$ $(1<=n<=2*10^5,k=0)$ 。
- 第二行给出 $n$ 个整数 $a_i$ $(1<=a_i<=10^7)$。
- 保证所有数据中 $n$ 的总和不超过 $2*10^5$。

## 输出格式

- 每组数据输出最小块数。

## 样例 #1

### 输入

```
3
5 0
18 6 2 4 1
5 0
6 8 1 24 8
1 0
1
```

### 输出

```
3
2
1
```

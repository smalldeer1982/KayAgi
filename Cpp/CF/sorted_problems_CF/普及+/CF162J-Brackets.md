# Brackets

## 题目描述

一个由括号组成的序列被称作**平衡的**，当且仅当这个序列可以通过在字符串中加入“+”号或数字 $1$。例如，序列"(())()", "()" 和 "(()(()))"是**平衡的**，但是")(", "(()" 和 "(()))(" 不是平衡的。  

给定一个由左右圆括号组成的字符串，请你判断它是否是**平衡的**。

## 输入格式

输入的第一行包含一个长度在$1$到$100$之间的字符串（包含1和100）。字符串中的每个字符只有可能是"("或者")"。

## 输出格式

如果这个括号序列是平衡的，输出"YES"，否则输出"NO"。（不含引号）  

翻译By @若如初见

## 样例 #1

### 输入

```
(()(()))()

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
())()

```

### 输出

```
NO

```

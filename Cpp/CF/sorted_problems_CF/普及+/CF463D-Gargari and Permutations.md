# Gargari and Permutations

## 题目描述

给你k个长度为n的排列，求这些排列的最长公共子序列的长度

## 输入格式

第一行包含n(1<=n<=1000)和k(2<=k<=5)。
后面的k行分别表示k个排列。

## 输出格式

输出最长公共子序列的长度

## 说明/提示

第一个测试样本的答案是子序列[1,2,3]。

## 样例 #1

### 输入

```
4 3
1 4 2 3
4 1 2 3
1 2 4 3

```

### 输出

```
3

```

# Pretty Song

## 题目描述

给你一个字符串，计算一下它的优美程度。
   	优美程度是指它的所有每一段连续子串优美程度的和。
    优美程度的计算方法：这一段连续子串中I，E，A，O，U，Y的总数/字符串长度。

## 输入格式

一串字符。

## 输出格式

一个小数，表示该字符串的优美程度（误差不超过1e-6）。

## 说明/提示

在第一个样例中，所有的字母都是I，E，A，O，U，Y其中的一个。每个子串的优美程度为1。长度为7的字符串有28个连续子串。所以这个字符串的优美程度为28.

## 样例 #1

### 输入

```
IEAIAIO

```

### 输出

```
28.0000000

```

## 样例 #2

### 输入

```
BYOB

```

### 输出

```
5.8333333

```

## 样例 #3

### 输入

```
YISVOWEL

```

### 输出

```
17.0500000

```

# Number of Parallelograms

## 题目描述

给定平面上$n$个点，问用这些点能组成多少平行四边形（多个平行四边形可以共用点，保证任意三点不共线）

## 输入格式

第一行输入点数$n$

随后$n$行输入点的坐标$(x_i,y_i)$，$x$和$y$用空格隔开（见样例）

## 输出格式

输出能组成的四边形数量$c$

## 说明/提示

$1 \leq n \leq 2000$ , $0\leq x_i,y_i \leq 10^{9}$

## 样例 #1

### 输入

```
4
0 1
1 0
1 1
2 0

```

### 输出

```
1

```

# Binary String Constructing

## 题目描述

给你 $3$ 个整数 $a$ ,$b$ ,$x$ ，让你求出一个包含了 $a$ 个 $0$，$b$ 个 $1$ 的二进制串 $s$。要求其中 $s_i\not =s_{i+1}$ 的对数正好等于 $x$。

## 输入格式

一行 $3$ 个整数 $a$ ,$b$ ,$x$。

## 输出格式

一行一个字符串 $s$ ,表示**任意**一符合要求的答案。

### 输入输出样例
无。

感谢 @win10 提供的翻译。

## 样例 #1

### 输入

```
2 2 1

```

### 输出

```
1100

```

## 样例 #2

### 输入

```
3 3 3

```

### 输出

```
101100

```

## 样例 #3

### 输入

```
5 3 6

```

### 输出

```
01010100

```

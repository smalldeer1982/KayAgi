# Uniform String

## 题目描述

```plain

给你两个数, $n$,$k$

你的任务是构造一个长度为$n$的字符串 $S$, 使得英文字母表中前 $k$ 个字符在字符串中出现至少一次,并且不出现其他字母,你需要最大化出现次数最小的字母的出现次数.

你要回答 $t$ 组询问,询问之间独立.

你要回答 $t$ 组询问,询问之间独立.

## 输入格式

第一行一个数 $T$

接下来 $T$ 行, 每行两个数 $n$,$k$.
```

# Ilya and a Colorful Walk

## 题目描述

ilya 有很多房子，每个房子都有一个颜色。因为 ilya 喜欢散步，所以想要选最远的两个不同颜色的房子作为散步的起点和终点。输出这个最远距离。

## 输入格式

第一行一个整数 $n$，表示有多少房子。

第二行 $n$ 个整数，表示颜色，相同的数字表示相同的颜色，不同的数字表示不同的颜色。

## 输出格式

输出最大距离。

## 样例 #1

### 输入

```
5
1 2 3 2 3

```

### 输出

```
4

```

## 样例 #2

### 输入

```
3
1 2 1

```

### 输出

```
1

```

## 样例 #3

### 输入

```
7
1 1 3 1 1 1 1

```

### 输出

```
4

```

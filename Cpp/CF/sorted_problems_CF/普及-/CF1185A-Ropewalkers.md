# Ropewalkers

## 题目描述

数轴上有3个整点A,B,C，分别分布在数a,b,c的地方。现需要让三个点之间两两距离不小于d。每秒钟有且仅有一个点可以移动一步，每步可以向左或向右移动恰好1个单位长度。我们想知道：为了达成目标，至少需要多少秒？

## 输入格式

一行，四个正整数a,b,c,d，含义如题所示。

## 输出格式

一行，一个整数，表示答案。

## 说明/提示

对于100%的数据，保证1<=a,b,c,d<=10^9。

## 样例 #1

### 输入

```
5 2 6 3

```

### 输出

```
2

```

## 样例 #2

### 输入

```
3 1 5 6

```

### 输出

```
8

```

## 样例 #3

### 输入

```
8 3 3 2

```

### 输出

```
2

```

## 样例 #4

### 输入

```
2 3 10 4

```

### 输出

```
3

```

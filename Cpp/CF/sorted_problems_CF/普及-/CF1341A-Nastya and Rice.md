# Nastya and Rice

## 题目描述

有$n$个范围为$[a-b,a+b]$的数，判断他们的和是否可能在$[c-d,c+d]$范围中。

输入有多组数据（$1 \le T \le 1000$），保证：

- $1 \le n \le 1000$
- $0 \le b < a \le 1000 $
- $0 \le d < c \le 1000$

## 输入格式

- 第一行为数据组数$T$。
- 接下来每行五个整数，$n,a,b,c,d$。

## 输出格式

- 如果所有数的和可能在$[c-d,c+d]$范围中，输出`Yes`，否则输出`No`。

## 样例 #1

### 输入

```
5
7 20 3 101 18
11 11 10 234 2
8 9 7 250 122
19 41 21 321 10
3 10 8 6 1
```

### 输出

```
Yes
No
Yes
No
Yes
```

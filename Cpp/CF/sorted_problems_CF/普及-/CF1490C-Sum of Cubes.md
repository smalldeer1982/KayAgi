# Sum of Cubes

## 题目描述

给您一个正整数  $x$ ，问这个正整数能否拆分成两个立方数之和。

也就是说，是否存在  $1\le a, b$ 满足  $a^3+b^3=x$ 。

## 输入格式

第一行是数据个数 $t(1\le t\le100)$。

后 $T$行每行一个 $x (1\le x \le 10^{12})$，为要判断的数。

请注意，32位整型变量可能存放不下所有的 $x$，所以你应该选择你所用的编程语言里至少64位的整型变量。（要开`long long`）

## 输出格式

对于每一个 $x$，输出单独一行。

- 如果输入的 $x$满足条件就输出"YES"

- 否则输出"NO"

## 样例 #1

### 输入

```
7
1
2
4
34
35
16
703657519796
```

### 输出

```
NO
YES
NO
NO
YES
YES
YES
```

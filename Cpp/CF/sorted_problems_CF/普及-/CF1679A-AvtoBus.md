# AvtoBus

## 题目描述

春天来了，车队的巴士需要更换轮胎。

轮胎有 $n$ 个，巴士有两种，第一种有 $4$ 个轮子，第二种有 $6$ 个。

求车队最少和最多有几辆巴士。

## 输入格式

第一行一个整数 $t$，表示测试数据的数量。

接下来 $t$ 行，每行一个整数 $n$，表示轮胎的总数。

## 输出格式

共 $t$ 行，每行两个整数，分别表示巴士数量的最小值和最大值。

若无解，输出 $-1$。

## 样例 #1

### 输入

```
4
4
7
24
998244353998244352
```

### 输出

```
1 1
-1
4 6
166374058999707392 249561088499561088
```

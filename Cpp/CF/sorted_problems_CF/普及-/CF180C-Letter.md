# Letter

## 题目描述

给你一个字符串，我们每一次操作都可以将一个大写字母变成任意小写字母，当然同理也可以将小写字母变成任意大写字母，问最少操作多少次，能够使得字符串变成前边都是大写字母，后边都是小写字母。

## 输入格式

输入的唯一行包含一个由大写字母和小写字母组成的非空字符串。字符串的长度不超过10^5个

## 输出格式

输出一个数字最少删除次数

## 样例 #1

### 输入

```
PRuvetSTAaYA

```

### 输出

```
5

```

## 样例 #2

### 输入

```
OYPROSTIYAOPECHATALSYAPRIVETSTASYA

```

### 输出

```
0

```

## 样例 #3

### 输入

```
helloworld

```

### 输出

```
0

```

# Special Characters

## 题目描述

### 题意
给定一个整数 $n$，请你构造一个恰好含有 $n$ 个特殊字符的字符串，或报告无解。

称一个字符是特殊的，当且仅当与它相邻的字符中恰好有一个与它相同。

## 输入格式

第一行一个整数 $t$，代表测试组数。

接下来 $t$ 行，每行一个整数 $n$，意义如题意。

## 输出格式

若有解，输出 `YES`，并在下一行输出构造的字符串；若无解，输出 `NO`。

## 样例 #1

### 输入

```
3
6
1
2
```

### 输出

```
YES
AAABAACC
NO
YES
MM
```

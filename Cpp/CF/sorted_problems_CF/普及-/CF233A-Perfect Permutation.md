# Perfect Permutation

## 题目描述

现有一个$1$~$n$的排列$p_1,p_2,...,p_n$。

$Nickolas$喜欢排列，他认为有一种排列是完美的。用$p_i$表示排列中的第$i$个元素，则完美的排列符合如下性质：

1. $p_{p_i}=i$

2. $p_i≠i$

现在给出$n$的值，请求出这个完美的排列。

## 输入格式

仅一行，一个整数$n(1≤n≤ 100)$

# Pashmak and Flowers

## 题目描述

Pashmak 想要给 Parmida一对花。有n朵花在花园里，每朵花都有一个魅力值bi，Pashmak想要两朵魅力值相差最大的花，问有多少种方案？

## 输入格式

首先输入n (2<= n <= 2*10^5)。
接下来一行有n个空格隔开的整数b1, b2, …, bn (1 <= bi <= 10 ^ 9).

## 输出格式

输出最大差异和方案数，中间用空格隔开

## 样例 #1

### 输入

```
2
1 2

```

### 输出

```
1 1
```

## 样例 #2

### 输入

```
3
1 4 5

```

### 输出

```
4 1
```

## 样例 #3

### 输入

```
5
3 1 2 3 1

```

### 输出

```
2 4
```

# Factory

## 题目描述

有一个工厂，在一天的开始时，共有x条细节（？），在这一天结束时，会生产x mod m条细节。
 
 现在已知第一天所拥有的细节数a，以及m的值
 
请问，是否存在一个时刻，整个工厂会停止生产（x mod m = 0）？

## 输入格式

两个整数，a 和 m

## 输出格式

当生产会停止时，输出`Yes`，否则输出`No`

## 样例 #1

### 输入

```
1 5

```

### 输出

```
No

```

## 样例 #2

### 输入

```
3 6

```

### 输出

```
Yes

```

# Parking Lot

## 题目描述

停车场共有 $2n-2$ 个停车位。共有 $4$ 种品牌的汽车，每种汽车的数量都远大于停车位的数量。

该公司首席执行官认为，如果停车场有 **恰好** $n$ 个连续汽车的品牌相同，则停车场会更漂亮。

给定n的值，问有多少的方案使停车场满足条件。

## 输入格式

一行，包含一个整数 $n$（$3\leq n\leq 30$）。

## 输出格式

输出一个整数，即总方案数。

## 样例 #1

### 输入

```
3

```

### 输出

```
24
```

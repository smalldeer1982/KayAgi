# Substrings Sort

## 题目描述

给定一个数n和n个字符串，求一个重新排列，对于新排列中的每个字符串
都满足在它前面的串都是它的子串。 


感谢@南方不败 提供翻译

## 输入格式

第一行n，第二行开始n个字符串

## 样例 #1

### 输入

```
5
a
aba
abacaba
ba
aba

```

### 输出

```
YES
a
ba
aba
aba
abacaba

```

## 样例 #2

### 输入

```
5
a
abacaba
ba
aba
abab

```

### 输出

```
NO

```

## 样例 #3

### 输入

```
3
qwerty
qwerty
qwerty

```

### 输出

```
YES
qwerty
qwerty
qwerty

```

# A Tide of Riverscape

## 题目描述

**题目大意：**

给定一个由'0','1','.'构成的字符串，问可不可能无论'.'是什么，这个字符串都存在长度为$p$的周期

## 输入格式

一行两个数，$n,p$，$1<=p<=n<=2000$

以下一行，一个长度为$n$的字符串

## 输出格式

如果不满足条件，输出不满足条件的字符串（把'.'用'0'或'1'替换），否则输出"No"

感谢@守望 提供翻译

## 样例 #1

### 输入

```
10 7
1.0.1.0.1.

```

### 输出

```
1000100010

```

## 样例 #2

### 输入

```
10 6
1.0.1.1000

```

### 输出

```
1001101000

```

## 样例 #3

### 输入

```
10 9
1........1

```

### 输出

```
No

```

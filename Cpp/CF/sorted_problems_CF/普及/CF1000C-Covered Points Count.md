# Covered Points Count

## 题目描述

**题目大意：**

给你n个区间，求被这些区间覆盖层数为$k(k<=n)$的点的个数

## 输入格式

第一行一个整数，$n$，$n<=2*10^5$

以下$n$行，每行有两个整数，即这个区间的左右端点$l,r(0<=l<=r<=10^{18})$

## 输出格式

$n$个整数，即每个被覆盖层数对应的点的个数

感谢@守望 提供翻译

## 样例 #1

### 输入

```
3
0 3
1 3
3 8

```

### 输出

```
6 2 1 

```

## 样例 #2

### 输入

```
3
1 3
2 4
5 7

```

### 输出

```
5 2 0 

```

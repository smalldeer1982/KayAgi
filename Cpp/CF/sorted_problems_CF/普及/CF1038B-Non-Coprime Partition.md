# Non-Coprime Partition

## 题目描述

### 题目大意：
给定一个整数$n$，要求把$1$到$n$分别放入两个序列$s1,s2$，使得$\gcd(sum(s1),sum(s2))>1$

## 输入格式

一个整数$n$

一个整数$n$

## 输出格式

如果不能构成符合条件的序列，输出"No"，否则输出"Yes"，并在下两行输出$s1,s2$


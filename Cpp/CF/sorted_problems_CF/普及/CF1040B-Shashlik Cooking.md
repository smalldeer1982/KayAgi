# Shashlik Cooking

## 题目描述

### 题目大意：
有$n$个烤串，翻动第$i$根烤串会连带着$max(1,i-k)$到$min(n,i+k)$根烤串翻动，问把所有烤串都翻动一遍最少需要翻动几根烤串

## 输入格式

两个整数$n,k$

## 输出格式

第一行一个整数 $ans$，表示最少翻动烤串根数。

接下来一行 $ans$ 个整数，表示方案。

## 样例 #1

### 输入

```
7 2

```

### 输出

```
2
1 6 

```

## 样例 #2

### 输入

```
5 1

```

### 输出

```
2
1 4 

```

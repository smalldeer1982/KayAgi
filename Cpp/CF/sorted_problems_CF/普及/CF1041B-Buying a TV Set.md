# Buying a TV Set

## 题目描述

### 题目大意：
给你$a,b,c,d$，求满足$\frac{x}{y}=\frac{c}{d},x\leq a,y\leq b$的$x,y$的对数

## 输入格式

一行$4$个整数$a,b,c,d$

## 输出格式

一个整数，表示满足条件的对数

## 样例 #1

### 输入

```
17 15 5 3

```

### 输出

```
3

```

## 样例 #2

### 输入

```
14 16 7 22

```

### 输出

```
0

```

## 样例 #3

### 输入

```
4 2 6 4

```

### 输出

```
1

```

## 样例 #4

### 输入

```
1000000000000000000 1000000000000000000 999999866000004473 999999822000007597

```

### 输出

```
1000000063

```

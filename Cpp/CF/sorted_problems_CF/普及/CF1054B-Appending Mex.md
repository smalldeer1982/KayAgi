# Appending Mex

## 题目描述

一开始有一个空的序列，每一次可以选取这个序列的一个子序列，并将这个子序列的 $\text{mex}$ 值加入到序列的尾部。

给定长度为 $n$ 的序列 $a_i$，求最小的 $t$ 使得无法通过若干次操作得到序列 $a_1,\ldots,a_t$。

## 输入格式

第一行一个正整数 $n$。

第二行 $n$ 个正整数，第 $i$ 个表示 $a_i$。

## 输出格式

如果可以通过若干次操作得到整个序列输出 $-1$，否则输出最小的 $t$ 使得若干次操作后无法得到 $a_1,\ldots,a_t$。

## 样例 #1

### 输入

```

```

```
1
```

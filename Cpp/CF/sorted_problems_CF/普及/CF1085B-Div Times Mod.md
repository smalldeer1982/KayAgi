# Div Times Mod

## 题目描述

给定n,k(1≤n≤$10^6$,2≤k≤$10^3$)

已知i，ans $\in$ Z，S.T. i | n（1≤i)&&ans $\equiv$ i(mod k)&&ans / k == n / i

求ans的最小值

## 输入格式

一行，分别为n,k

## 输出格式

一行，ans

## 输入输出样例

样例搬运过来就行了

## 样例 #1

### 输入

```
6 3

```

### 输出

```
11

```

## 样例 #2

### 输入

```
1 2

```

### 输出

```
3

```

## 样例 #3

### 输入

```
4 6

```

### 输出

```
10

```

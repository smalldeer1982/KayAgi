# Build a Contest

## 题目描述

### 题目大意
有$n$个题，每个题有一个难度$a_i(1\le a_i\le m)$，从左往右加入题，当加入的题中$m$个难度都出现时，输出$1$并把每个难度都删除一道题，否则输出$0$，求输出序列

## 输入格式

第一行两个数$m,n$，表示难度个数和题目个数

第二行$n$个整数，表示题目难度

## 输出格式

一行，表示输出序列

## 样例 #1

### 输入

```
3 11
2 3 1 2 2 2 3 2 2 3 1

```

### 输出

```
00100000001

```

## 样例 #2

### 输入

```
4 8
4 1 3 3 2 3 3 3

```

### 输出

```
00001000

```

# Diverse Matrix

## 题目描述

对于一个r行c列的矩阵，我们可以求出它每一行，每一列的最大公约数（共r+c个数）。
输入r,c。要求输出一个矩阵，满足这r+c个数不同，且这r+c个数的最大值最小。

## 输入格式

输入一行两个整数，分别为r,c。(1≤r,c≤500)

## 输出格式

如果无解输出0，否则输出你构造的r行c列的矩阵，要求矩阵中的每个元素满足1≤a≤1e9 。

## 样例 #1

### 输入

```
2 2

```

### 输出

```
4 12
2 9
```

## 样例 #2

### 输入

```
1 1

```

### 输出

```
0

```

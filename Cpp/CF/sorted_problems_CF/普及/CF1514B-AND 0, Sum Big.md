# AND 0, Sum Big

## 题目描述

#### 题目大意：

给定两个正整数 $n$ 和 $k$ ，求有多少个序列满足下列条件：

1. 序列中的所有元素都在 $[0,2^k-1]$ 之间。
2. 它的所有元素的与运算之和为 $0$ 。
3. 它的元素之和是尽可能大的。

答案对 $10^9+7$ 取模。

## 输入格式

第一行一个整数 $t$ ($1 \le t \le 10$) 表示数据组数。

每组测试数据包含两个整数 $n$ 和 $k$ ，意义如题面所述。

## 输出格式

对于每组数据，输出符合条件的序列对 $10^9+7$ 取模的结果。

## 样例 #1

### 输入

```
2
2 2
100000 20
```

### 输出

```
4
226732710
```

# Trees in a Row

## 题目描述

## 题意
有$n$个正整数，可以对一个数进行修改(修改后也是正整数)，要求修改之后满足$a_i+k=a_{i+1}$,求最少的修改次数以及具体的修改方案

## 输入格式

第一行:$n$和$k$

第二行:$n$个数,表示初始的$a_i$

## 输出格式

若不用修改,只要输出0

 若需要修改;
 
 - 第一行:最小的修改次数
 - 接下来的若干行,对于**被修改的**$a_i$,其修改后值为$b_i$,若$b_i>a_i$,输出:"+ i ($b_i-a_i$)";若$a_i>b_i$,输出:"- i $(a_i-b_i)$"

## 样例 #1

### 输入

```
4 1
1 2 1 5

```

### 输出

```
2
+ 3 2
- 4 1

```

## 样例 #2

### 输入

```
4 1
1 2 3 4

```

### 输出

```
0

```

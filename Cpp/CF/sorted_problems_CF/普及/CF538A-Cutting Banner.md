# Cutting Banner

## 题目描述

### 题意简述

给定只由大写英文字母组成的字符串 $s$，问删掉几个连续的字符（即删去一个子串）之后是否能得到 `CODEFORCES`。

## 输入格式

一个长度不超过 $100$ 的字符串 $s$。

## 输出格式

如果可以得到 `CODEFORCES`，输出 `YES`。反之输出 `NO`。

翻译贡献者 U108949

## 样例 #1

### 输入

```
CODEWAITFORITFORCES

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
BOTTOMCODER

```

### 输出

```
NO

```

## 样例 #3

### 输入

```
DECODEFORCES

```

### 输出

```
YES

```

## 样例 #4

### 输入

```
DOGEFORCES

```

### 输出

```
NO

```

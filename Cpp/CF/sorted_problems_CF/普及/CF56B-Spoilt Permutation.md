# Spoilt Permutation

## 题目描述

给出一个长度为 $ n $ 的序列（保证序列中 $ 1-n $ 每个数有且仅出现1次），问能否恰好反转序列中的一段，使得序列变成 $ 1,2...n $ 的顺序。

## 样例 #1

### 输入

```
8
1 6 5 4 3 2 7 8

```

### 输出

```
2 6

```

## 样例 #2

### 输入

```
4
2 3 4 1

```

### 输出

```
0 0

```

## 样例 #3

### 输入

```
4
1 2 3 4

```

### 输出

```
0 0

```

# Pouring Rain

## 题目描述

有一个直径为 $d$ **厘米**的水桶。

原先桶里水深度为 $h$ **厘米**。

你在喝桶里的水，喝水的速度是 $v$ **毫升每秒**。

不喝水时桶内水面上升速度为 $e$ **厘米每秒**。

现在一边下雨，你一边喝桶里的水。

如果你喝的速度赶不上降水速度，输出"NO"，否则输出"YES"和你需要几秒喝光水。

## 输入格式

四个整数 $d,h,v,e$。

## 输出格式

"NO"(不带引号)或者"YES"和喝光水需要的时间（一个实数）。

## 样例 #1

### 输入

```
1 2 3 100

```

### 输出

```
NO

```

## 样例 #2

### 输入

```
1 1 1 1

```

### 输出

```
YES
3.659792366325

```

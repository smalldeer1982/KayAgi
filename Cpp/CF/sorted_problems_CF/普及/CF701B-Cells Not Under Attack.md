# Cells Not Under Attack

## 题目描述

给定一个 $n \times n$ 的棋盘和 $m$ 个车以及这 $m$ 个车的坐标，问在放下第 $i$ 个车的时候，有多少个格子不会被已放下的车攻击到。

## 输入格式

第 $1$ 行输入两个数 $n$ 和 $m$；  
第 $2$ 行到第 $m+1$ 行，输入第 $i$ 个车的坐标 $x_{i}$ 和 $y_{i}$ 。

## 输出格式

输出 $m$ 个数字，第 $i$ 个数字表示放置 $i$ 个车后不会被攻击到的格子数量。

## 样例 #1

### 输入

```
3 3
1 1
3 1
2 2

```

### 输出

```
4 2 0 

```

## 样例 #2

### 输入

```
5 2
1 5
5 1

```

### 输出

```
16 9 

```

## 样例 #3

### 输入

```
100000 1
300 400

```

### 输出

```
9999800001 

```

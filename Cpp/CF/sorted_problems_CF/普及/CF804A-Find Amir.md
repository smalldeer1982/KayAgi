# Find Amir

## 题目描述

题面：

有n座城市,标号为1~n,定义标号为i,j的两座城市的距离为(i+j)%(n+1).求走完所有城市所需的最小花费(起点任意).

## 输入格式

仅一个n.

## 输出格式

走完所有城市的最小距离.
翻译贡献者：尘染梦

## 样例 #1

### 输入

```
2

```

### 输出

```
0

```

## 样例 #2

### 输入

```
10

```

### 输出

```
4

```

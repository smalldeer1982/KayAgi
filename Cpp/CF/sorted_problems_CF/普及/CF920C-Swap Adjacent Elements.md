# Swap Adjacent Elements

## 输入格式

第一行一个整数 n (2≤n≤200000)

第二行 n个整数 a1,a2,⋯,an(1≤ai​≤200000) 。每个 1 到 n 间的整数正好出现一次。

第三行一个字符串，只包含 0 或者 1 。如果第 i个字符是1 ，你就可以把 ai与 ai+1交换。

## 输出格式

如果可以排序，输出 YES，否则输出 NO。

感谢@小粉兔 提供的翻译

## 样例 #1

### 输入

```
6
1 2 5 3 4 6
01110

```

### 输出

```
YES

```

## 样例 #2

### 输入

```
6
1 2 5 3 4 6
01010

```

### 输出

```
NO

```

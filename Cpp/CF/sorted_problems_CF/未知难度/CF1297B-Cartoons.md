# Cartoons

## 题目描述

Tanya 是一个动画片迷。她得知在她钟爱的影院中，将会有 $n$ 部新动画片即将上映。第 $i$ 部动画片的上映日期从第 $a_i$ 天开始，持续到第 $b_i$ 天（$1 \le a_i \le b_i \le 10^9$）。

影院推出了一个特别优惠：每天如果只有一部动画片在放映，就会有巨大的折扣。

Tanya 并不介意具体看哪一部动画片，但她很想省钱。所以，她希望你帮忙找出一个日期 $x$，这一天恰好只有一部动画片上映。形式上，即需要找出一个 $x$，使得仅存在一个 $i$ 满足 $a_i \le x \le b_i$。如果这样的一天不止一个，任选其一输出即可；如果没有这样的日期，输出 -1。

## 输入格式

第一行为一个整数 $t$（$1 \le t \le 1000$），表示测试用例的数量。接下来是 $t$ 组测试用例。

每个测试用例的第一行是一个整数 $n$（$1 \le n \le 2000$），表示动画片的数量。

随后 $n$ 行描述动画片的上映日期，每行由两个整数 $a_i$ 和 $b_i$ 构成，分别表示第 $i$ 部动画片的首映日和终映日。

保证所有测试用例中 $n$ 的总和不大于 $2000$。

## 输出格式

依次输出每个测试用例的答案：对于第 $i$ 个测试用例，找出一个 $x$，使得这一天只有一部动画片在放映，或若无这样的日期，则输出 -1。

## 说明/提示

- 在第三个测试用例中：第 1 天和第 2 天，第一部和第三部动画片同时上映；第 3 天和第 4 天，第二部和第三部动画片同时上映。因此没有某一天只有一部动画片上映。
- 在第四个测试用例中，$11$ 也是一个可能的答案。

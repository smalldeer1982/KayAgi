# Mountain Climber

## 输入格式

第一行输入$t （1 \le t \le 25） -$ 测试数据组数。

接下来 t 行，各为一个字符串 $s（1 \le |s| \le 100）$,只包含小写英文字母。

## 输出格式

对于每一行数据，输出 YES 或 NO。

# Help, what does it mean to be "Based"

## 输入格式

给出一个单独的整型变量 $ x $ ( $ 1 \le x \le 4 $ )。

## 输出格式

如果 $ x = 1 $,输出一份代码，要求如下：输入变量 $ a $,$ b $,输出 $ a + b $ 。

如果 $ x = 2 $，输出一份代码，要求如下：输入 $ x $,输出 $ x $ 的绝对值。

如果 $ x = 3 $，输出一份代码，要求如下：输入 $ n $ 和 长度为 $n$ 的数组 $a$,输出数组 $ a $ 的最大值。

如果 $ x = 4 $，输出一份代码，要求如下：输入 $ n $ 和 长度为 $n$ 的数组 $a$ 与 $ k $,输出数组 $ a $ 中的第 $ k $ 大值。

*slaps a on top of b*
yeet b
go touch some grass

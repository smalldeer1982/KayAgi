# 比赛会变成 Unrated 吗？

## 题目描述

去年愚人节比赛的时候 CF 崩了，导致比赛变成了 Unrated。没人知道这些可怜的出题人们之后经历了什么。后来经过仔细研究找到了问题的元凶：出题人忘了预测试点和正式系统测试点是等价的，都加进去了导致题目测试点太多。今年出题人不想再让历史重演，所以来找你，让你帮他们检查一下这题有多少个测试点。

## 输入格式

一行字符串，问你这题有多少个测试点。

## 输出格式

一行一个整数，本题测试点的数量。

## 说明/提示

作者的温馨提示：“我原谅你这回撒谎，但是我再问你的时候你最好把答案说对！”

## 样例 #1

### 输入

```
Hello! Can you please tell me how many tests does this problem have?
```

### 输出

```
25
```

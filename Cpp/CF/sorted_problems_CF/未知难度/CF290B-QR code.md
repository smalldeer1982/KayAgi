# QR code

## 题目描述

 ![](https://cdn.luogu.com.cn/upload/vjudge_pic/CF290B/d0a8acb40db4893f5cd05b650f4206e9b618f001.png)

## 输入格式

The input contains two integers $ a_{1},a_{2} $ ( $ 0<=a_{i}<=32 $ ), separated by a single space.

## 输出格式

Output a single integer.

## 样例 #1

### 输入

```
1 1

```

### 输出

```
0

```

## 样例 #2

### 输入

```
3 7

```

### 输出

```
0

```

## 样例 #3

### 输入

```
13 10

```

### 输出

```
1

```

# Feed the Golorp

## 题目描述

Golorps 是以变量为食的神秘生物。Golorp的名字是某种编程语言的程序。一些科学家认为，这种语言是Befunge;golorp们沉默得令人着迷。golorps 使用的变量的值可以从 0 到 9（包括 0 和 9）。对于每只 golorp，它的日常饮食都由它的名字定义。有些golorp非常挑剔，根本喂不饱。此外，所有的golorp都非常注重健康，并尽量少吃。给定几个有效的变量值序列的选择，每个 golorp 将选择字典上最小的一个。为了解决这个问题，你可以假设一只golorp由下巴和胃组成。喂养golorp所需的变量数量由其下巴的形状定义。变量只能通过下颌到达胃部。一只饥饿的golorp正在拜访你。你知道它的名字;喂它或弄清楚这是不可能的。

## 输入格式

输入是一个字符串（长度在 13 到 1024 个字符之间）——来访的 golorp 的名称。所有名称都相似，并且与示例中给出的名称相似。该名称保证有效。

## 输出格式

输出按词典顺序排列的最小变量值序列，适合喂养这只 golorp。值应按照它们进入下颌的顺序列出。如果 golorp 无法喂食，则输出“false”。

## 样例 #1

### 输入

```
?(_-_/___*__):-___&gt;__.

```

### 输出

```
0010

```

## 样例 #2

### 输入

```
?(__-_+_/_____):-__&gt;__,_____&lt;__.

```

### 输出

```
false

```

## 样例 #3

### 输入

```
?(______________________/____+_______*__-_____*______-___):-__&lt;___,___&lt;____,____&lt;_____,_____&lt;______,______&lt;_______.

```

### 输出

```
0250341

```

## 样例 #4

### 输入


### 输出

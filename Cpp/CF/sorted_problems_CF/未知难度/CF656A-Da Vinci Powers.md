# Da Vinci Powers

## 输入格式

The input contains a single integer $ a $ ( $ 0<=a<=35 $ ).

## 输出格式

Output a single integer.

## 样例 #1

### 输入

```
3

```

### 输出

```
8

```

## 样例 #2

### 输入

```
10

```

### 输出

```
1024

```

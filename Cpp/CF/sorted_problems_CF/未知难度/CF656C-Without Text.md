# Without Text

## 题目描述

## 输入格式

The only line of the input is a string (between 1 and 50 characters long, inclusive). Each character will be an alphanumeric character or a full stop ".".

## 输出格式

Output the required answer.

## 样例 #1

### 输入

```
Codeforces

```

### 输出

```
-87

```

## 样例 #2

### 输入

```
APRIL.1st

```

### 输出

```
17

```

# Ace It!

## 输入格式

The only line of the input is a string of 7 characters. The first character is letter A, followed by 6 digits. The input is guaranteed to be valid (for certain definition of "valid").

## 输出格式

Output a single integer.

## 样例 #1

### 输入

```
A221033

```

### 输出

```
21

```

## 样例 #2

### 输入

```
A223635

```

### 输出

```
22

```

## 样例 #3

### 输入

```
A232726

```

### 输出

```
23

```

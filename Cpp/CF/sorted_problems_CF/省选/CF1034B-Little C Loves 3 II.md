# Little C Loves 3 II

## 题目描述

**题目大意：**

给定一个$n\times m$的棋盘，每次可以在上面放一对棋子$(x,y)(x',y')$，要求$\mid x-x'\mid+\mid y-y'\mid==3$，问最多可以放多少**个**棋子

## 输入格式

一行两个整数$n,m$

## 输出格式

一个整数，表示最多能放多少个棋子

## 样例 #1

### 输入

```
2 2

```

### 输出

```
0
```

## 样例 #2

### 输入

```
3 3

```

### 输出

```
8
```

# Ray in the tube

## 题目描述

### 题目大意：
下边界有$n$个给定点，上边界有$m$个给定点，可以从任意一个点发出一条激光，激光碰到边界会反射

激光到达边界必须打到整数点，问最多可以打到几个给定点

## 输入格式

第一行两个整数$n,y_1$，表示下边界给定点的个数和下边界的纵坐标

第二行$n$个整数，表示给定点的横坐标

第三行两个整数$m,y_2$，表示上边界给定点的个数和上边界的纵坐标

第四行$m$个整数，表示给定点的横坐标

## 输出格式

一个整数，表示最多能打到多少点

## 样例 #1

### 输入

```
3 1
1 5 6
1 3
3

```

### 输出

```
3

```

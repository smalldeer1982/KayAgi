# What Is It?

## 题目描述

给定一个 $n$，你需要构造一个排列 $\{p_i\}$ 和一个操作序列。  
每次操作你可以交换 $i$ 位置和 $j$ 位置的值，要求 $i\ne j$ 且 $i = p _ j$，并获得 $(j-i)^2$ 的贡献。  
要求最后满足 $p_i = i$。  
你需要最大化总贡献。  
可以证明，操作次数 $<n$。

## 输入格式

多组数据，每组数据给定一个 $n$。

## 输出格式

首先，第一行一个数，表示你求出的最大贡献。  
然后第二行给出你构造的 $\{p_i\}$。  
第三行给出一个数 $m$，表示你需要的操作次数。  
接下来 $m$ 行，每行两个数，代表当前操作的 $i$ 和 $j$（注意，**有顺序要求**）

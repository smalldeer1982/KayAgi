# Triangles

## 题目描述

给定一个有规律的金字塔形的图形，其中一部分三角形是黑色的（见图）。 要求统计所有从最顶端出发，沿着黑线走了一条闭合简单路径又回到最顶端，且没有任何黑色三角形在闭合简单路径内部的路径条数

## 输入格式

n(2 ≤ n ≤$10^6$)

## 输出格式

条数%1000000009.
Translated by @liyifeng

## 样例 #1

### 输入

```
2

```

### 输出

```
10

```

## 样例 #2

### 输入

```
4

```

### 输出

```
74

```

# Yet Another Minimization Problem

## 输出格式

$2 \leq n \leq 10^5$，$2 \leq k \leq \min(n,20)$，$1 \leq a_i \leq n$ 。

## 样例 #1

### 输入

```
7 3
1 1 3 3 3 2 1

```

### 输出

```
1

```

```
10 2

```

```
8

```

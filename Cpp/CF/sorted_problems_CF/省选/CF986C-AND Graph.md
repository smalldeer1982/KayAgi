# AND Graph

## 题目描述

给定一个 $m$ 个整数的集合，每个整数在 $0$ 到 $2^n-1$ 之间，以每一个整数作为顶点建无向图，当两个点 $x$ 和 $y$ 做与运算值为 $0$ 时，则认为 $x$ 和 $y$ 是连通的，即 $x$ 和 $y$ 之间有一条无向边。请求出图中连通块的个数。

## 输入格式

第一行输入两个整数 $n$ 和 $m$（$0 \leq  n\leq22$，$1 \leq  m\leq2^n$）。  
第二行输入 $m$ 个整数，即集合里的元素。

## 输出格式

一个整数，表示图中连通块的个数。

## 样例 #1

### 输入

```
2 3
1 2 3

```

### 输出

```
2

```

## 样例 #2

### 输入

```
5 5
5 19 10 20 12

```

### 输出

```
2

```

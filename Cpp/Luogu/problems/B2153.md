# 求阶乘的和
## 题目描述

给定正整数 $n$，求不大于 $n$ 的正整数的阶乘的和（即求 $1!+2!+3!+\dots+n!$），输出阶乘的和。

阶乘定义为 $n!=n\times (n-1)\times (n-2)\times \cdots \times 1$。例如，$5! = 5 \times 4 \times 3 \times 2 \times 1=120$。
## 输入格式

输入一行，包含一个正整数 $n(1 < n < 12)$。
## 输出格式

输出一行，表示阶乘的和。
## 样例

### 样例输入 #1
```
5
```
### 样例输出 #1
```
153
```

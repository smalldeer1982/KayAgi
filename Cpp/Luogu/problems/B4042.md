# [语言月赛 202410] 顺序结构
## 题目描述

顺序结构中，为什么一定要讲究几句话的顺序呢？在一次经历中，小 Y 意识到了顺序的重要性。

---

小 Y 写了一个程序，按顺序执行下面四步：

1. 输入变量 $a$。
2. 将 $a$ 增加 $ 5$。
3. 将 $a$ 变成原来的 $3$ 倍。
4. 输出现在的 $a$。

小 L 写了一个程序，按顺序执行下面四步：

1. 输入变量 $a$。
2. 将 $a$ 变成原来的 $3$ 倍。
3. 将 $a$ 增加 $5$。
4. 输出现在的 $a$。

给出输入的 $a$，请判断两人的程序分别会输出什么。
## 输入格式

输入一行一个正整数 $a$，表示程序的输入。
## 输出格式

输出一行两个自然数 $a_y,a_l$，分别表示小 Y 的输出和小 L 的输出。
## 样例

### 样例输入 #1
```
4

```
### 样例输出 #1
```
27 17

```
### 样例输入 #2
```
10

```
### 样例输出 #2
```
45 35

```
### 样例输入 #3
```
27

```
### 样例输出 #3
```
96 86

```
## 提示

【样例 1 解释】

在小 Y 的程序输入 $a$ 之后：

- 先把 $a$ 加上 $5$，变成 $4+5=9$。
- 然后把 $a$ 乘上 $3$，得到 $9\times 3 = 27$。

在小 L 的程序输入 $a$ 之后：

- 先把 $a$ 乘上 $3$，得到 $4\times 3 = 12$。
- 然后把 $a$ 加上 $5$，得到 $12+5=17$。

【数据范围】

$1\le a \le 28$，并且 $a$ 是正整数。

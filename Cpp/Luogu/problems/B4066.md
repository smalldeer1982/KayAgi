# [GESP202412 三级] 数字替换
## 题目描述

小杨有一个包含 $n$ 个数字的序列 $A$，即 $A=[a_1,a_2,\ldots,a_n]$，他想将其中大于 $k$ 的数字都替换为序列的最大值，将其中小于 $k$ 的数字都替换为序列的最小值，请你帮他计算出替换后的序列。


## 输入格式

第一行包含两个正整数 $n,k$，含义如题面所示。

第二行包含 $n$ 个数字，代表序列 $A$。
## 输出格式

输出 $n$ 个整数，代表替换后的结果。
## 样例

### 样例输入 #1
```
5 0
-2 -1 0 1 2
```
### 样例输出 #1
```
-2 -2 0 2 2
```
## 提示

对于全部数据，保证有 $1\le n\le 10^5$，$|k|,|a_i|\le 10^5$。

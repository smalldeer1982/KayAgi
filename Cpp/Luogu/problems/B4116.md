# [JOI2025 预选赛 R1H3] 现金
## 题目描述

比太郎有 $A$ 张 $1000$ 日元的纸币，$B$ 张 $10000$ 日元的纸币，求比太郎手里还有多少日元。
## 输入格式

第一行输入一个正整数 $A$。

第二场输入一个正整数 $B$。
## 输出格式

输出一行一个整数表示答案。
## 样例

### 样例输入 #1
```
7
2
```
### 样例输出 #1
```
27000
```
### 样例输入 #2
```
11
1
```
### 样例输出 #2
```
21000
```
## 提示

### 数据范围

$1\leq A,B\leq 100$。

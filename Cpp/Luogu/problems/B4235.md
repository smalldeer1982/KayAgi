# [四川青少年 C++ 算法设计大赛 2024] 4-5 年级组第 6 题
## 题目背景

第二届四川省青少年 C++ 算法设计大赛小学 4-5 年级组全省总决赛试题。数据为洛谷自造。
## 题目描述

小 X 想用星号画一个正方形，但是全用星号画他觉得太枯燥了，所以他打算将整个正方形的正中间挖掉一个正方形，使得整体看上去是一个环形。现在给定外面的大正方形和中间挖掉的正方形的边长，请你画出整个图形。
## 输入格式

一行两个整数 $a,b$，分别表示外面大矩形的边长以及中间挖掉的正方形的边长。
## 输出格式

共 $a$ 行，每行 $a$ 个字符，表示小 X 最终画出来的图形。
## 样例

### 样例输入 #1
```
6 2
```
### 样例输出 #1
```
******
******
**  **
**  **
******
******
```
### 样例输入 #2
```
5 3
```
### 样例输出 #2
```
*****
*   *
*   *
*   *
*****
```
## 提示

对于 $100\%$ 的数据，$1\leq b<a\leq 50$，且 $b$ 和 $a$ 的奇偶性相同。

# [GESP202503 二级] 等差矩阵
## 题目描述

小 A 想构造一个 $n$ 行 $m$ 列的矩阵，使得矩阵的每一行与每一列均是等差数列。小 A 发现，在矩阵的第 $i$ 行第 $j$ 列填入整数 $i \times j$，得到的矩阵能满足要求。你能帮小 A 输出这个矩阵吗？
## 输入格式

一行，两个正整数 $n, m$。
## 输出格式

共 $n$ 行，每行 $m$ 个由空格分割的整数，表示小 A 需要构造的矩阵。
## 样例

### 样例输入 #1
```
3 4
```
### 样例输出 #1
```
1 2 3 4
2 4 6 8
3 6 9 12
```
## 提示

对于所有测试点，保证 $1\leq n,m \leq 50$。

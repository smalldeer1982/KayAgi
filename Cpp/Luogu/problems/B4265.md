# [朝阳区小学组 2019] rectangle
## 题目背景

数据为洛谷自造。
## 题目描述

给出一个长方形的长和宽，求长方形的周长和面积。
## 输入格式

两个整数 $a,b$（$1\leq a,b\leq 10\,000$），分别表示长方形的长和宽，以空格隔开。
## 输出格式

两个整数，分别表示长方形的周长和面积，以 $1$ 个空格隔开。
## 样例

### 样例输入 #1
```
5 3
```
### 样例输出 #1
```
16 15
```

# [蓝桥杯青少年组省赛 2022] 统计数字
## 题目描述

给定一个正整数 $N$，找出 $3$ 到 $N$ 之间的正整数中，个位数为 $3$ 的有多少个。

例如：$N=25$，$3$ 至 $25$ 之间个位数为 $3$ 的有 $3$、$13$、$23$，一共有 $3$ 个。

## 输入格式

输入一个正整数 $N$（$3 \leq N \leq 99993$）。
## 输出格式

输出一个整数，表示 $3$ 到 $N$ 之间的正整数中，个位数为 $3$ 的个数（包含 $3$ 和 $N$）。
## 样例

### 样例输入 #1
```
25
```
### 样例输出 #1
```
3
```

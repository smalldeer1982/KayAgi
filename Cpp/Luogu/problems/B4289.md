# [蓝桥杯青少年组省赛 2022] 字母组合
## 题目描述

给定 $N$ 个小写字母，然后将这 $N$ 个小写字母按照字典序排序后组合成一个字符串输出。

例如 $N=4$，$4$ 个小写字母分别为 $\tt c$、$\tt d$、$\tt a$、$\tt c$，按照字典序排序后组合成的字符串为：$\tt{accd}$。
## 输入格式

第一行输入一个正整数 $N$（$2 < N < 100$），表示小写字母的个数。

第二行输入 $N$ 个小写字母，且小写字母之间以一个空格隔开。
## 输出格式

将 $N$ 个小写字母按照字典序排序后组成一个字符串并输出。
## 样例

### 样例输入 #1
```
4
c d a c
```
### 样例输出 #1
```
accd
```

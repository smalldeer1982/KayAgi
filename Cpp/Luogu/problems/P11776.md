# [COTS 2013] 图形计数 / POLIGONI
## 题目描述

枚举平面直角坐标系具有下列性质的多边形：

- 多边形顶点为坐标系整数点。
- 多边形每条边平行于坐标轴。
- 对于顶点 $(x,y)$，满足 $0 \le x \le N,0\le y\le K$。

形状相同，位置不同的两个多边形我们认为是不同的，且根据定义，多边形的边是一条不接触自身的闭合多边形线。

## 输入格式

一行两个整数 $K,N$。
## 输出格式

一行一个整数，为多边形的数量，答案对 $10007$ 取模。
## 样例

### 样例输入 #1
```
2 2 
```
### 样例输出 #1
```
13
```
### 样例输入 #2
```
3 2
```
### 样例输出 #2
```
40
```
### 样例输入 #3
```
3 3 
```
### 样例输出 #3
```
213 
```
## 提示

【数据范围与约定】

- 对于所有数据，均满足 $1\le K \le 6,1 \le N \le 1000$。

| 测试点编号 | $K=$ | $N\le$ | 特殊性质 | 分值 |
| -----------: | -----------: | -----------: | -----------: | -----------: |
| $1$ | $1$ | $1000$ | 是 | $5$  |
| $2$ | $1$ | $1000$ | 否 | $5$ |
| $3$ | $2$ | $1000$ | 是 | $5$ |
| $4$ | $2$ |$1000$  | 否 | $5$  |
| $5$ | $3$ | $1000$ | 是 | $10$ |
| $6$ | $3$ |$1000$  | 否 |$10$  |
| $7$ | $4$ |$1000$  | 是 | $10$ |
| $8$ | $4$ | $1000$ | 否 | $10$ |
| $9,10$ | $5$ | $1000$ | 否 |$10$  |
| $11,12$ | $6$ |$1000$  | 否  |$10$  |

- 特殊性质： $N\times K \le 20$。

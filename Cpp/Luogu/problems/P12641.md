# [KOI 2024 Round 1] 上学
## 题目背景

试题来源：<https://koi.or.kr/archives/>。中文翻译做了少量本土化修改。

## 题目描述

小明为了不迟到，必须在 $X$ 分钟内到达学校。为了前往学校，他需要选择搭乘一辆在车站停靠的 $N$ 辆公交车中的一辆。

懒惰的小明为了尽可能晚些出发，查阅了 $N$ 辆公交车的所有信息。他已经知道每辆公交车距离现在还有多少分钟会从车站出发（记为 $S$），也知道每辆公交车从车站到学校需要多少分钟（记为 $T$），但他尚未决定搭乘哪一辆。

请帮助小明，在所有能够使他不迟到的公交车中，找出最晚出发的那一辆，并输出该公交车距离现在还有多少分钟会从车站出发。如果没有任何一辆公交车能让他准时到达学校，请输出 $-1$。
## 输入格式

第一行给出 $N$ 和 $X$，用一个空格隔开。

从第二行起，给出 $N$ 行，每行包含一个公交车的信息，其中每行包含两个数字：$S$（公交车从车站出发的时间）和 $T$（公交车从车站到学校的时间），两者用空格隔开。
## 输出格式

如果没有公交车能够在 $X$ 分钟内到达学校，则输出 $-1$。

如果有公交车能够在 $X$ 分钟内到达学校，则输出最晚出发的公交车从现在起出发所需的时间。
## 样例

### 样例输入 #1
```
3 8
2 1
6 3
4 4
```
### 样例输出 #1
```
4
```
### 样例输入 #2
```
1 30
15 20
```
### 样例输出 #2
```
-1
```
## 提示

**限制条件**
- 所有给定的数字均为整数。
- $1 \leq N \leq 100$
- $2 \leq X \leq 200$
- 对于每辆公交车，其出发时间 $S$ 和行驶时间 $T$ 满足：
  - $1 \leq S \leq 100$
  - $1 \leq T \leq 100$
  
**子问题**
1. （10 分）$N = 1$
2. （15 分）所有公交车都能在 $X$ 分钟内到达学校。
3. （30 分）所有公交车的 $T = 20$
4. （45 分）无额外限制条件

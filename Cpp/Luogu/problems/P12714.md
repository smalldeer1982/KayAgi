## 题目描述

你有 $n$ 个字符串，请求出满足字符串里**所有字符互不相同**的字符串数量。

**字符串中只含有小写字母。**
## 输入格式

第一行输入一个整数 $n$。

接下来 $n$ 行每行输入一个字符串。
## 输出格式

输出一行，表示满足条件的字符串数量。
## 样例

### 样例输入 #1
```
3
takahashi
aoki
snuke
```
### 样例输出 #1
```
2
```
## 提示

**【数据范围】**

- $1 \le n \le 100$。
- 每个字符串的长度不超过 $100$。

# [PA 2017] 换钱
## 题目背景

译自 [PA 2017](https://sio2.mimuw.edu.pl/c/pa-2017-1/) R1T1。


## 题目描述

有 $n$ 枚硬币，第 $i$ 枚硬币面额为 $2^{a_i}$。

可以将两枚面额均为 $2^j$ 的硬币合成一枚面额 $2^{j+1}$ 的硬币。求出**可能得到**的最大的金币面额。

## 输入格式

第一行，正整数 $n$。

第二行，$n$ 个非负整数 $a_1,\cdots,a_n$。
## 输出格式

令答案为 $2^k$，输出一行一个非负整数 $k$。
## 样例

### 样例输入 #1
```
5
3 4 1 3 3
```
### 样例输出 #1
```
5
```
## 提示

- $1\le n\le 10^6$；
- $0\le a_i\le 201\, 718$。

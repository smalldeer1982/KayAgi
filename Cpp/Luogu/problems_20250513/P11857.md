# [CCC 2025 Senior] 挂画 / Positioning Peter’s Paintings
## 题目背景

译自 CCC 2025 Senior T1。本题满分为 $15$。
## 题目描述

有两幅矩形画，一幅长为 $A$，宽为 $B$；另一幅长为 $X$，宽为 $Y$。

要把这两幅画用一个矩形画框裱起来挂在墙上，但是要满足：

- 这两幅画必须全部在画框中；
- 这两幅画不能重叠（但是允许边缘触碰）；
- 这两幅画不能旋转；
- 这两幅画边缘必须与地面平行，或者垂直于地面。

求出符合条件的画框周长的最小值。
## 输入格式

一行四个正整数 $A,B,X,Y$。
## 输出格式

一行一个正整数，表示答案。
## 样例

### 样例输入 #1
```
3 3 3 3
```
### 样例输出 #1
```
18
```
### 样例输入 #2
```
2 2 4 4
```
### 样例输出 #2
```
20
```
### 样例输入 #3
```
1 2 3 1
```
### 样例输出 #3
```
12
```

# [ICPC 2016 WF] Polygonal Puzzle

## 题目描述

有两个多边形，可以平移旋转（但不能对称、缩放等），求这两个多边形贴在一起但不重合的情况下贴贴部分的最大总长度。

## 说明/提示

$3 \leq n_1, n_2 \leq 50$，横纵坐标均为整数且绝对值不超过 $100$。

## 样例 #1

### 输入

```
8
0 0
0 10
10 10
15 15
24 6
24 10
30 10
30 0
7
-5 0
-5 10
10 10
15 5
20 10
35 10
35 0
```

### 输出

```
30.142135624
```

## 样例 #2

### 输入

```
3
1 0
0 30
40 0
3
1 0
0 30
40 0
```

### 输出

```
50
```

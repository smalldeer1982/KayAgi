# [CSP-J2019 江西] 面积

## 题目描述

Alice 有一个边长为 $a$ 的正方形，Bob 有一个长宽分别为 $b,c$ 的矩形，请你告诉他们俩谁的图形面积更大。

## 说明/提示

【数据范围】   
对于 $30\%$ 的数据，$1 \le a,b,c \le 100$，$b=c$；  
对于 $80\%$ 的数据，$1\le a,b,c \le 10^4$；  
对于 $100\%$ 的数据，$1\le a,b,c \le 10^9$。

【样例 $1$ 解释】  
正方形面积为  $25$，矩形面积为 $24$。  

【样例 $2$ 解释】  
正方形面积为  $49$，矩形面积为 $50$。  

## 样例 #1

### 输入

```
5 4 6```

### 输出

```
Alice```

## 样例 #2

### 输入

```
7 5 10```

### 输出

```
Bob```

## 作者：ctq1999 (赞：68)

## 思路

前置知识：正方形、长方形的面积公式

$Alice$的正方形的面积公式：

$$S = a^2$$

$Bob$的长方形的面积公式：

$$S = b \times c$$

现在给你$a, b, c$的值，那么代入计算即可

注意数据范围：$1\le a,b,c \le 10^9$

$int$最大只可以存$2^{31} - 1$，差不多是$2 \times 10^9$，算面积时会爆$int$，但是$long long$可以存下$2^{63} - 1$的数据，差不多是$10^{19}$，而算出来的面积最大也就是$10^9 \times 10^9 = 10^{18}$了

所以使用$longlong$就可以存下面积的值了

但是算的过程中会爆$int$，所以要乘上$1ll$，相当于强制转换成$longlong$

```cpp
#include <bits/stdc++.h>

#define ll long long

using namespace std;

int a, b, c;//题目给出的a,b,c
ll S1, S2;//Alice的面积和Bob的面积

int main() {
	scanf("%d%d%d", &a, &b, &c);//输入
	
	S1 = 1ll * a * a;	
    S2 = 1ll * b * c;//算出两者的面积，注意强制转换
	if (S1 > S2) cout << "Alice" << endl;//比较，并输出
			else cout << "Bob" << endl;
	
	return 0;//结束
}
```

### P5681 题解

思路：特判+算面积

本题很容易想到的是直接边长X边长，长X宽，一比就完事儿了。

然而还有某些特殊情况，**不用算**就能知道哪个几何图形的面积大一些。

例如：有一个边长为6的正方形（RED）和一个长为5，宽为4的长方形（YELLOW）。

![](https://cdn.luogu.com.cn/upload/image_hosting/elndkp41.png)

不难发现，正方形的边长**既大于长方形的长，同时也大于它的宽**。

所以遇到bc>c&&bc>k的情况，正方形面积肯定大一些。

当然，如果bc<=c&&bc<=k，就说明了正方形面积是不大于长方形面积的。

剩下的就只能算面积辽~

### code：

```cpp
#include <iostream>
#include <cstdio>
#define re register int
using namespace std;

long long bc,c,k,s1,s2;  //不开long long要爆。bc，c，k分别存边长，长，宽 

int main()
{
	std::scanf("%lld%lld%lld",&bc,&c,&k);
	
	if(bc<=c&&bc<=k)
	{
		std::printf("Bob");
		return 0;
	}
	if(bc>c&&bc>k)
	{
		std::printf("Alice");
		return 0;
	}//如上所述的特判过程 
	
	s1=bc*bc;
	s2=c*k;  //带公式算面积 
	
	if(s1>s2)
		std::printf("Alice");
	else
		std::printf("Bob");
	//比较&输出 
	return 0;
}
```

Tips：因为 $10^9 \times 10^9 = 10^{18}$ 会爆 `int`，所以要开 `long long`。

## CODE：

```cpp
#include <iostream>
using namespace std;
long long a, b, c; 
int main()
{
    cin >> a >> b >> c;
    if (a * a > b * c) cout << "Alice";
    else cout << "Bob";
    return 0;
}
```

长为 $a$，宽为 $b$ 的长方形面积为 $a \times b$。

# 题目解法

把 Alice 和 Bob 的图形面积算出来，再比较大小就行了。

正方形面积公式：$S=a^2$。

长方形面积公式：$S=ab$。

注意事项：

- 不要把人名打错！
- 注意到 $1\le a,b,c\le 10^9$，面积可以达到 $10^{18}$，所以十年 OI 一场空，不开 long long 见祖宗。

代码：
```cpp
//有点C语言风格？
#include<cstdio>
int main(){
	long long a,b,c;
	scanf("%lld%lld%lld",&a,&b,&c);
	printf("%s",(a*a>b*c)?"Alice":"Bob");
	return 0;
}
```

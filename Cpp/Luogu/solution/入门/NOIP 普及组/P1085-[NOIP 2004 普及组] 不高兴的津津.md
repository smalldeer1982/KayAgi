## 题目大意
输入七组数，分别表示津津在学校上课的时间和妈妈安排她上课的时间。求在学校上课的时间和妈妈安排她上课的时间的和是否会让津津不高兴。
## 题目思路
可以通过遍历数组来求解最大值出现的位置。

输入数据，边输入边统计最不高兴的是周几，也就是上课时间最长的是第几天。

这里要注意，如果上课时间相同，则统计时间靠前的那一天。
## AC code

```cpp
#include <bits/stdc++.h>

            maxn=a[i];
            maxm=i;
        }
    }
    if(maxn==-1)printf("0");
    else printf("%d",maxm);
    return 0;
}
```

## 作者：cff_0102 (赞：19)

可以通过遍历数组来求解最大值出现的位置。
- 可以利用我们在 P1001 学到的知识计算出一天的不高兴度。用一个变量 $mx$ 存储最大的不高兴度，$pos$ 用于保存最大值出现的对应天数。
- 首先将 $mx$ 设为 $8$，$pos$ 设为 $0$。如果所有天数的信息都不能更新 $mx$ 和 $pos$，表示津津不会不高兴，那么就会输出最初的 $pos=0$。
- 接下来，依次输入每天的信息。如果这一天的不高兴度大于 $mx$，说明找到了新的最大值，此时更新 $mx$ 的值，并将 $pos$ 更新为当前的天数。
- 最终，$mx$ 就是最大的不高兴度，而 $pos$ 则是对应的天数。

代码如下：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	ios::sync_with_stdio(0);cin.tie(0);
	int mx=8,pos=0;
	for(int i=1;i<=7;i++){
		int a,b;cin>>a>>b;
		if(mx<a+b)mx=a+b,pos=i;
	}
	cout<<pos;
	return 0;
}
```

# 题目解析
输入 $7$ 组数，每组 $2$ 个数，让你求哪组 $2$ 个数的和超过 $8$ 且最大，如有多个，输出第一个，如果每天都没有超过 $8$，输出```0```。
# 解法
循环输入 $7$ 组数，每组输入 $2$ 个数 $t1$，$t2$，求出他们的和，如果第 $i$ 组时，$max<$ 他们的和，那么 $max\gets t1+t2$，$maxi\gets i$。  
最后输出 $maxi$。
## 易错点解析
一开始，$max\gets 8$，$maxi\gets 0$。  
因为津津一天内学习的时间**超过** $8$ 才会不开心，
所以 $max\gets 7$ 是错的。  
如果没有，则输出```0```,所以 $maxi\gets 0$。

还有，如果有两天或两天以上不高兴的程度相当，则输出时间最靠前的一天。那么判断时不应用 $max\le t1+t2$,因为这样会输出最靠后的一天（想一想，为什么），而应该用 $max< t1+t2$。
# code
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	int max=8,maxi=0;
	for(int i=1;i<=7;i++){
		int t1,t2;//津津在学校上课的时间和妈妈安排她上课的时间
		cin>>t1>>t2;
		if(max<t1+t2){
			max=t1+t2;
			maxi=i;
		}
	}
	cout<<maxi;
	return 0;
}
```

# 题解：P1085 [NOIP2004 普及组] 不高兴的津津
[题目传送门](https://www.luogu.com.cn/problem/P1085)
- 首先，我们要通过循环输入两个数 $a$ 和 $b$，定义一个数 $sum$ 来计算每天的上课时间。
- 接下来，我们需要定义一个 $temp$ 来存储当前大于 $8$ 的值。当 $sum > 8$ 且 $sum > temp$ 时，更新 $temp$。这样写可以保证如果有两天或两天以上不高兴的程度相当，输出的是时间最靠前的一天。
- 然后定义一个 $ans$ 来记录 $sum$ 的下标，即记录令津津不高兴的是哪一天。
- 最后输出这个 $ans$ 即可。
- 注意 $temp$ 要初始化为 $0$，这样才可以更新它的值。


```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
ll a,b,sum,ans=0,temp=0;
int main(){
    for(int i=1;i<=7;i++){
		cin>>a>>b;
		sum=a+b;
	    if(sum>8&&sum>temp) temp=sum,ans=i;
	}
	cout<<ans;
	return 0;
}
```

### 题意

有 $7$ 组数，$(a_1,b_1)$ 到 $(a_7,b_7)$。

如果 $a_i+b_i>8$，那么津津会不高兴。如果 $a_i+b_i>8$，则 $a_i+b_i$ 的值越大他就越不高兴。

问津津最不高兴的是哪一天，如果有相同的，输出最靠前的。

### 思路

这个题的题意是求 $a_i+b_i>8$ 的数中的最大值。

每次输入 $a_i$ 和 $b_i$，由于只要考虑 $a_i+b_i$ 的值，直接将两数相加，$a_i\larr a_i+b_i$。记 $maxx$ 表示当前的最大值。如果相加后的 $a_i>8$ 并且 $a_i>maxx$，就把 $a_i$ 记为最大值，$i$ 记为答案。

因为如果有相同的，输出最靠前的，所以 $a_i$ 必须要严格大于 $maxx$ 才能记。

如果没有让津津不高兴的，要输出 $0$。

### AC Code

这是一份来自 $4$ 年前的代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main() {
	int a,b,max=0,x=0;
	for(int i=1;i<=7;i++){
		cin>>a>>b;
		a+=b;
		if(a>8&&a>max){
			x=i;
			max=a;
		}
	}
```

# 题意

津津上课的时间 $=$ 在学校上课的时间 $+$ 妈妈安排他上课的时间。如果津津上课的时间 $>8$ 小时他就会不高兴。要求输出他最不高兴的时间是周几。（上课时间越长他越不高兴）如果有两天或两天以上不高兴的程度相当，则输出时间最靠前的一天。

# 思路

我们可以先依次输入津津从周一到周日每天在学校上课的时间和妈妈安排他上课的时间。（具体见输入格式）然后再计算出他每天的总上课时间。我们设他在学校上课的时间为 $a_i$，妈妈安排他上课的时间为 $b_i$，他每天总上课的时间为 $c_i$。则有：

$c_i=a_i+b_i$。（根据题意可得）（此处的 $i$ 用 
$1,2,3,4,5,6,7$ 分别表示周一，周二，周三，周四，周五，周六，周日）

我们可以利用此公式算出他每天的总上课时间。然后创建一个变量 `maxn` 用来存储津津从周一至周日的最长的总上课时间和一个变量 `ans` 用来存储答案。并先将变量 `maxn` 赋值为 $0$。

我们可以从周一至周日依次比较 $c_i$ 与 `maxn` 的大小，如果 $c_i$ 大于（注意：没有等于！）`maxn`，则将 `maxn` 赋值为 $c_i$（维护最大值），并且将 `ans` 赋值为 $i$。（存储最大值所在的编号）

最后输出 `ans` 即可。

# 题目大意

津津当一天上课超过八个小时会不高兴，计算下周是否会不高兴，如果会的话，哪天最不高兴。

# 解法

定义津津在学校上课的时间是 $a_i$，妈妈安排她上课的时间是 $b_i$，随后令 $max=a_1+b_1,sum=1$，随后遍历后 $6$ 天，如果有一天 $i$ 的 $a_i+b_i\ge 8$ 并且 $a_i+b_i>max$，说明津津不高兴而且更不高兴，让 $max$ 更新值，$sum$ 记录这一天。最后判断 $max=0$，是说明不存在一个 $i$ 使得 $a_i+b_i\ge 8$；否则输出 $sum$。

代码（远古代码，建议大家定义变量不定义为 $max$）：

复杂度：$O(day)$，$day$ 是一周中的天数。

```cpp
#include<iostream>
using namespace std;
int main(){
	int a[8],b[8],sum=0,max=0;
	for(int i=1;i<=7;i++){
		cin>>a[i]>>b[i];
	}
	max=a[1]+b[1];
	for(int i=1;i<8;i++){
		if(a[i]+b[i]>=8&&max<a[i]+b[i]){
			sum=i;
			max=a[i]+b[i];
		}
	}
	if(max){
		cout<<sum<<endl;
	}
	else{
		cout<<"0"<<endl;
	}
	return 0;
}
```

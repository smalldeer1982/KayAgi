## 题目描述

输入两个整数 $a$ 和 $b$（其绝对值不超过 $10000$），想知道：

1. $a$ 是否大于 $b$？
2. $a$ 是否小于或等于 $b$？
3. $a$ 是否不等于 $b$？

## 样例 #1

### 输入

```
2 3
```

### 输出

```
0 1 1```

# [语言月赛 202408] 游戏与共同语言

## 题目背景

> 风与清都喜欢玩某款游戏。

## 题目描述

在该游戏的职业联赛中，战队 A 和战队 B 的排名先后关系取决于胜局数、净胜数、平局记录数。具体排名规则如下：

- 胜局数**高**者排名靠前
- 若胜局数相同，净胜数**高**者排名靠前
- 若净胜数仍相同，平局记录**低**者排名靠前
- 不存在平局记录仍相同的情况

现在，给出两支战队相关数据，请给出排名更靠前的战队。

## 说明/提示

对于 $100\%$ 的测试数据，$1 \le w_a,c_a,t_a,w_b,c_b,t_b \le 10^9$。

## 样例 #1

### 输入

```
10 5 4
10 6 1
```

### 输出

```
B
```

# 题解

## 作者：Igallta (赞：3)

## 题意

给定胜局数、净胜局、平局数量。

再给定比较规则：

- 胜局数高者排名靠前。
- 若胜局数相同，净胜数高者排名靠前。
- 若净胜数仍相同，平局记录低者排名靠前。

求哪个队更厉害。

## 思路

首先，我们要输入。

- 注：这里用数组记录。

```cpp
cin>>w[1]>>c[1]>>t[1]>>w[2]>>c[2]>>t[2];
```

然后我们要对各个数组的各个位依次比较。

首先登场的是胜局数，直接比较然后输出即可。

这部分的代码:

```cpp
if(w[1]!=w[2]){
  cout<<(w[1]>w[2]?"A":"B");
}
```

其次，如果它们一样，那就比较净胜局。

```cpp
else if(c[1]!=c[2]){
  cout<<(c[1]>c[2]?"A":"B");
}
```

最后，如果他们也一样，那就比较平局记录。

```cpp
else if(t[1]!=t[2]){
  cout<<(t[1]>t[2]?"B":"A");
}
```

由于不可能平局记录也一样，所以到此为止。

## 文字题解

本题考察三关键字比较。

给出两只战队的 $w,c,t$ 三个指标，

1. 首先，判断 $w_a$ 与 $w_b$ 是否相等，如果不相等，根据 $w_a$ 和 $w_b$ 的大小关系输出答案，并结束程序。
2. 如果 $w_a=w_b$，则判断 $c_a$ 与 $c_b$ 是否相等，如果不相等，根据 $c_a$ 和 $c_b$ 的大小关系输出答案，并结束程序。
3. 根据 $t_a$ 和 $t_b$ 的大小关系输出答案。

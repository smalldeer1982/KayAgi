# [CSP-X 2018] 快递费用

## 题目描述

某快递公司按邮寄物品的重量收费，收费标准如下：

重量在 $500$ 克以内的，一律 $20$ 元；

超过 $500$ 克的，超重的部分按每 $500$ 克加收费用。超出的重量不足 $500$ 克的，按 $500$ 克计算。例如：$1020$ 克，超重 $520$ 克，需加收两份费用。

根据目的地的不同，加收的费用是不一样的。快递公司划分了五个目的地区域：

- 区域 $1$：每超重 $500$ 克加收 $4$ 元；

- 区域 $2$：每超重 $500$ 克加收 $6$ 元；

- 区域 $3$：每超重 $500$ 克加收 $9$ 元；

- 区域 $4$：每超重 $500$ 克加收 $10$ 元；

- 区域 $5$：每超重 $500$ 克加收 $17$ 元。

给出物品的重量 $w$ 和目的地区域编号 $n$，请你计算快递费用。

## 说明/提示

$1020$ 克的物品寄到区域 $3$，前 $500$ 克收费 $20$ 元；

超重 $1020 - 500 = 520$ 克，还需缴纳 $2$ 份加收的费用；

区域 $3$ 每超重 $500$ 克加收 $9$ 元，共计：$20 + 9 \times 2 = 38$ 元。

数据保证 $1\leq w\leq 10^8$，$1\leq n\leq 5$。

## 样例 #1

### 输入

```
1020 3```

### 输出

```
38```

本题考查分支结构。

对于读入的 $w,n$，我们首先判断 $w$ 是否小于等于 $500$。如果成立的话，则输出 $20$。

接着再处理超重的部分。首先，要计算的是：要加收几次费用。例如：$1020$ 克要加收 $2$ 次超重费用，因为 $1020-500=520$，$520-500=20$（第一次计算超重），留下的 $20$ 克用于第二次计算超重。

可以得出式子，扣除 $500$ 克计算超重的次数是：$(w-500) \div 500$。如果这个除法运算有余数（也就是会有剩余的不足 $500$ 克的部分），那么次数 $+1$。

接下来，根据目的地区域编号计算加收费用。可以使用 `switch` 语句或者 `if else` 结构。参考代码中采用的是 `if else` 结构。需要注意两点：
1. 不要抄错数字，确保每个区域的加收价格在程序中正确填写。
2. $20$ 元的费用是基础费用，一定要加进答案中。

参考代码（部分）：

```cpp
if (w <= 500)
    cout << 20 << endl;
else {
    int ex = (w - 500) / 500; //计算加收次数
    if ((w - 500) % 500 != 0) //如果有剩余的不足 500 克的重量，则再额外加上 1 次
        ex++;
    if (n == 1)
        cout << 20 + ex * 4 << endl;
    else if (n == 2)
        cout << 20 + ex * 6 << endl;
    else if (n == 3)
        cout << 20 + ex * 9 << endl;
    else if (n == 4)
        cout << 20 + ex * 10 << endl;
    else
        cout << 20 + ex * 17 << endl;
}
```

**（易错问题）** 思考：代码中的 `if else` 结构改成下面的代码，为什么是错误的呢？

```cpp
if (n == 1)
    cout << 20 + ex * 4 << endl;
if (n == 2)
    cout << 20 + ex * 6 << endl;
if (n == 3)
    cout << 20 + ex * 9 << endl;
if (n == 4)
    cout << 20 + ex * 10 << endl;
else
    cout << 20 + ex * 17 << endl;
```

答案：`else` 结构承接在 `if (n == 4)` 里，因此只要 $n$ 不等于 $4$，就会进入 `else` 语句，输出多余的内容。

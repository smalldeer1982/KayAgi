# [JOI2025 预选赛 R1H1] 铅笔 2

## 题目描述

JOI 君有 $A$ 日元，一根铅笔 $5$ 日元，问 JOI 君最多可以买多少根铅笔。

## 说明/提示

### 样例解释
$1$ 根铅笔 $5$ 日元，$2$ 根铅笔 $10$ 日元，$3$ 根铅笔 $15$ 日元。

### 数据范围

$1\leq A\leq 100$。

## 样例 #1

### 输入

```
9```

### 输出

```
1```

## 样例 #2

### 输入

```
10```

### 输出

```
2```

## 样例 #3

### 输入

```
3```

### 输出

```
0```

## 样例 #4

### 输入

```
100```

### 输出

```
20```

# [JOI2025 预选赛 R1H1] 铅笔 2

## 知识点

- 【1】算术运算：加、减、乘、除、整除、求余

## 分析

考虑 $5$ 日元可以买一个铅笔，说明我们需要找到最大的 $r$ 满足 $5\times r \leq A$。所以 $r=\lfloor A\div 5 \rfloor$，其中 $\lfloor x\rfloor$ 表示 $\leq x$ 的最大整数。

## 代码

```cpp
int n;
cin >> n
cout << n / 5 << endl;
```

# [JOI2025 预选赛 R1H1] 散步

## 题目描述

JOI 君在一条直线型道路上散步。

JOI 君会交替执行以下两种操作：
- 前进 $3\textrm{ m}$
- 后退 $2\textrm{ m}$

JOI 君的初始位置为 $0$，问 $X$ 次操作后 JOI 君在哪个位置。

## 说明/提示

### 样例解释

JOI 君的位置变化如下：

$$0\rightarrow 3\rightarrow 1 \rightarrow 4 \rightarrow 2\rightarrow 5 \rightarrow 3\rightarrow \ldots$$

### 数据范围

$1\leq X\leq 100$。

## 样例 #1

### 输入

```
3```

### 输出

```
4```

## 样例 #2

### 输入

```
6```

### 输出

```
3```

## 样例 #3

### 输入

```
1```

### 输出

```
3```

## 样例 #4

### 输入

```
37```

### 输出

```
21```

## 样例 #5

### 输入

```
100```

### 输出

```
50```

# [JOI2025 预选赛 R1H1] 散步

## 知识点

- 【2】for 语句、while 语句、do while 语句
- 【2】if 语句、switch 语句、多层条件语句
- 【1】算术运算：加、减、乘、除、整除、求余

## 分析

我们使用循环语句，从 $1$ 开始枚举，设当前枚举的数为 $i$，如果 $i$ 是奇数，那么我们执行 $+3$ 操作；否则执行 $-2$ 操作。

## 代码

```cpp
int ans=0;
for(int i=1;i<=x;i++){
  if(i%2==1)ans+=3;
  else ans-=2;
}
cout<<ans<<endl;
```

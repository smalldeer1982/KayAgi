# [JOI2025 预选赛 R1H3] 黑桃 J

## 题目描述

给定三个正整数 $A,B,C$，如果三个数的和 $\leq 21$，输出 $1$，否则输出 $0$。

## 说明/提示

### 数据范围

$1\leq A,B,C\leq 13$。

## 样例 #1

### 输入

```
5
10
6```

### 输出

```
1```

## 样例 #2

### 输入

```
7
8
13```

### 输出

```
0```

# 题解

## 作者：luogu_gza (赞：0)

直接按照题意模拟即可。

```cpp
int A,B,C;
cin>>A>>B>>C;
putchar(A+B+C<=21?'1':'0');
```

# [语言月赛 202501] 数字谜

## 题目描述

给出一个两位数 $a$，问至少要把 $a$ 加上几（可以 $+0$），才能让 $a$ 的个位变成 $b$。

## 说明/提示

【样例 1 解释】

$47+2=49$，个位为 $9$。

【样例 2 解释】

$89+4=93$，个位为 $3$。

【样例 3 解释】

$a$ 的个位已经是 $0$，所以最少只要增加 $0$。

【数据范围】

$10\le a\le 90, 0\le b\le 9$。

## 样例 #1

### 输入

```
47 9
```

### 输出

```
2
```

## 样例 #2

### 输入

```
89 3
```

### 输出

```
4
```

## 样例 #3

### 输入

```
30 0
```

### 输出

```
0
```

# 题解

## 作者：yummy (赞：8)

本题考查顺序结构。

可以考虑先让 $a$ 的个位变成 $b$，然后再让增加的数尽量小。

如果仅是让 $a$ 的个位变成 $b$，由于 $a\le 90$，我们可以直接把 $a$ 加到 $90+b$。

然后我们发现如果增加的数超过 $10$，那么少增加 $10$，个位不变，因此增加的数可以直接对 $10$ 求余数。

因此，最终答案就是 $(90+b-a)\bmod 10$。

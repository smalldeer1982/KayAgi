# [语言月赛 202501] 会场座位

## 题目描述

一个会场有 $100$ 个座位，座位的编号从左到右依次是 $99,97,\ldots,7,5,3,1,0,2,4,6,\ldots,96,98$。

给出两个**不同的**座位 $a,b$，请判断座位 $a,b$ 之间间隔了多少个座位。

## 说明/提示

【样例 1 解释】

座位 $2,5$ 之间有 $0,1,3$ 这三个座位，所以输出 $3$。

【样例 2 解释】

座位 $2,6$ 之间有 $4$ 这个座位，所以输出 $1$。

【样例 3 解释】

座位 $85,83$ 之间没有座位，所以输出 $0$。

【数据范围】

保证 $0\le a,b \le 99$ 并且 $a\ne b$。

## 样例 #1

### 输入

```
2 5
```

### 输出

```
3
```

## 样例 #2

### 输入

```
2 6
```

### 输出

```
1
```

## 样例 #3

### 输入

```
85 83
```

### 输出

```
0
```

# 题解

## 作者：yummy (赞：9)

本题考查分支结构。

把座位 $0,1$ 之间的位置称为“中缝”（也就是奇数和偶数座位号的切换点），不难注意到，对于任意一个座位 $x$，$x$ 和中缝之间的座位个数恰好是 $x\div 2$ 的整数部分，记为 $\lfloor x\div 2 \rfloor$。

那么对于原问题，分两种情况讨论：

- 如果 $a,b$ 奇偶性不同（在中缝两侧），那么答案就是”$a$ 和中缝之间的座位个数 + $b$ 和中缝之间的座位个数“，也就是 $\lfloor a\div 2\rfloor + \lfloor b\div 2\rfloor$。
- 如果 $a,b$ 奇偶性相同（在中缝同一侧），那么答案是 $\lfloor a\div 2\rfloor, \lfloor b\div 2\rfloor$ 的差（大数减小数）。

# [XJTUPC 2024] 交小西的礼物

## 题目背景

小西给参加这场比赛的大家都准备了精美的礼物！

## 题目描述

小西准备给 $a$ 名志愿者，$b$ 支单挑队，$c$ 支双人队，$d$ 支三人队的**每个人**恰好发一份奖品，她想知道一共要发几份奖品？

## 作者：zsh_haha (赞：3)

### 思路讲解

题目很简单，主要考察表达式的应用。

已知有 $a$ 名志愿者，$b$ 支单挑队，$c$ 支双人队，$d$ 支三人队，给每个人发一个奖品。

因为每支单挑队有一个人，每支双人队有两个人，每支三人队有三个人，所以我们要准备 $a+b+2c+3d$ 个奖品。

由于数据范围很小，用 `short` 或 `int` 就能过。

### AC 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b,c,d;
	cin>>a>>b>>c>>d;
	cout<<a+b+c*2+d*3; 
	return 0;
}
```

## 作者：Double_Light (赞：2)

容易发现 $a$ 个志愿者要发 $a$ 份礼物，$b$ 支单挑队发 $b$ 份礼物，$c$ 支双人队有 $2c$ 个人，送 $2c$ 份礼物，$d$ 支三人队有 $3d$ 个人，送 $3d$ 份礼物。

所以答案就是 $a+b+2c+3d$，输出即可。

## 作者：封禁用户 (赞：1)

题意很重要。

志愿者以“**名**”为单位，所以每个志愿者只需要 $1$ 个奖品，共需要 $a$ 个。

玩过游戏的人都知道，“单挑”是指一个人对战一个人的比赛，所以单挑队的每队也只有 $1$ 个人，也就是每队需要 $1$ 个奖品，共需要 $b$ 个。

同理，“双人队”是两个人组成的队伍，所以双人队的每队有 $2$ 个人，也就是每队需要 $2$ 个奖品，共需要 $2c$ 个。

“三人队”是三个人组成的队伍，所以三人队的每队有 $3$ 个人，也就是每队需要 $3$ 个奖品，共需要 $3d$ 个。

所以需要准备的奖品总个数就是：$(a+b+2c+3d)$ 个。

## 作者：Vitamin_B (赞：1)

# 思路
$a$ 名志愿者，每名 $1$ 份，共 $a$ 份；$b$ 支单挑队，每支 $1$ 份，共 $b$ 份；$c$ 支双打队，每支 $2$ 份，共 $2c$ 份；$d$ 支三人队，每支 $3$ 份，共 $3d$ 份。总共 $a+b+2c+3d$ 份。
# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int a,b,c,d;
    cin>>a>>b>>c>>d;
    cout<<a+b+c*2+d*3;
}
```

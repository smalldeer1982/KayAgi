# [APC001] A - CT

## 题目背景

Compressed Towel.

~~什么，这不是公开赛，这是压缩毛巾，遇水变大变高~~。

## 题目描述

这是一个有魔力的压缩毛巾，可以无限变大。

压缩毛巾的初始长、宽、高分别为 $a, b, h$ 厘米。它遇到水 $k$ 次，每次遇到水，长宽高增加 $1$ 厘米。请输出最后压缩毛巾的体积。单位为立方厘米。

但是某人觉得这题太简单了，懒惰的他把这题丢给了你，所以你需要做出这道题。

## 说明/提示

### 样例解释 $1$

计算可得 $(1+5)\times(1+5)\times(4+5)=6\times6\times9=324$。

### 样例解释 $2$

请注意压缩毛巾可以不遇到水。

### 数据范围

对于 $100\%$ 的数据，$1\leq a,b,h\leq 500$，$0\leq k\leq 500$。

## 样例 #1

### 输入

```
1 1 4 5```

### 输出

```
324```

## 样例 #2

### 输入

```
100 100 100 0```

### 输出

```
1000000```

## 样例 #3

### 输入

```
114 51 419 198```

### 输出

```
47933496```

## 作者：Breath_of_the_Wild (赞：4)

签到题。按题意模拟即可。

输入 $a,b,h$，遇水 $k$ 次后，长宽高即变为 $(a+k),(b+k),(h+k)$。体积即为它们的乘积。注意，答案对 $988444333$ 取模。

```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
const ll M=988444333;
ll a,b,c,k;
int main(){
    cin>>a>>b>>c>>k;
    cout<<(a+k)*(b+k)%M*(c+k)%M;
    return 0;
}
```

### 思考

遇水 $k$ 次，每次长、宽、高增加 $1$ 厘米，则最后等于长、宽、高增加了 $k$ 厘米。

### 做题

我们输入后可以直接把 $a,b,c$ 加上 $k$，然后每次取模即可。

### 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
const int mod = 988444333;
int main(){
    int a,b,c,k;
    cin>>a>>b>>c>>k;
    a+=k,b+=k,c+=k;
    cout<<a*b%mod*c%mod;
    return 0;
}
```

## 作者：jiangyunuo (赞：0)

### 题目大意及思路：
有一个压缩毛巾，它的初始长、宽、高分别为 $a$、$b$、$h$ 厘米，此时压缩毛巾遇到了水，长、宽、高各增长了 $k$ 厘米，求现在压缩毛巾的体积。长方体的体积就是长、宽、高的积，我们可以先求出遇水后的长、宽、高，再求出压缩毛巾的体积。
### 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b,h,k;
	cin>>a>>b>>h>>k;
	a+=k;
	b+=k;
	h+=k;   //计算遇水后的长、宽、高。
	long long ans;
	ans=a*b*h;     //计算体积。
	ans%=988444333;    //取模。
	cout<<ans<<endl;
}
```

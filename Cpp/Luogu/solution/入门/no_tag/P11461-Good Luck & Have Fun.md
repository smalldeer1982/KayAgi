# Good Luck & Have Fun

## 题目背景

> 欢迎报考东北大学**秦皇岛**分校

## 题目描述

众所不周知，NEUQ 第一届图灵杯在 $2013$ 至 $2014$ 学年开始线下举办，并且之后每学年举办一届。

但众所周知，从第七届图灵杯开始，NEUQ 面向全国高校举行图灵杯线上同步赛，然而由于一些因素，第十届图灵杯并没有举行线上同步赛。

现在给定一个正整数 $x$，现在会长 @BrotherCall 想知道 $x$ 至 $x + 1$ 学年举办的图灵杯有没有线上同步赛。

如果有则输出 `Yes`，没有则输出 `No`。




## 说明/提示

$2013 \le x \le 2024$。

## 样例 #1

### 输入

```
2013```

### 输出

```
No```

## 样例 #2

### 输入

```
2024```

### 输出

```
Yes```

# 题解

## 作者：_Jocularly_ (赞：2)

如果没有线上赛，说明要不然是一九年以前，要不然是二二年，只有这两种情况是没有的，其他的都是有的，直接输出即可。


```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
int main(){
    cin >> n;
    if(n < 2019 || n == 2022) cout << "No";
    else cout << "Yes";
    return 0;
}
```

# 题目实现
其实这道题目很简单，我们只要搞清楚第七届比赛是在 $2019$ 至 $2020$ 学年举办，第十届比赛是在 $2022$ 至 $2023$ 学年举办即可。

为什么呢，让我们为一些脑子没转过弯的 OIer 解释一下：

  $2013$ 至 $2014$ 学年 ——— 第一届比赛（线下）
 
  $2014$ 至 $2015$ 学年 ——— 第二届比赛（线下）

  $2015$ 至 $2016$ 学年 ——— 第三届比赛（线下）

  $2016$ 至 $2017$ 学年 ——— 第四届比赛（线下）
  
  $2017$ 至 $2018$ 学年 ——— 第五届比赛（线下）
  
  $2018$ 至 $2019$ 学年 ——— 第六届比赛（线下）
  
  $2019$ 至 $2020$ 学年 ——— 第七届比赛（线上）
  
  $2020$ 至 $2021$ 学年 ——— 第八届比赛（线上）
  
  $2021$ 至 $2022$ 学年 ——— 第九届比赛（线上）
  
  $2022$ 至 $2023$ 学年 ——— 第十届比赛（线下）

  $2023$ 至 $2024$ 学年 ——— 第十一届比赛（线上）

  $2024$ 至 $2025$ 学年 ——— 第十二届比赛（线上）

现在应该大家都懂了吧。

第 $i$ 届举办时间为 $2012+i$ 到 $2013+i$ 年。

第七届开始有线上赛，即 $2019$ 到 $2020$ 年开始有线上赛。

特别的，第十届没有线上赛，即 $2022$ 到 $2023$ 年没有线上赛。

那么只需要简单判断下第 $n$ 年的时候是不是已经开始举办线上赛了以及 $n$ 是否为 $2022$ 即可。

## 作者：collegiate (赞：0)

### 简化题意
从 $2013$ 年开始举办比赛，之后每年举办一次，第七届以后有线上同步赛，第十届没有，求第 $x$ 年到 $x+1$ 年有没有线上同步赛。

### 思路
因为从 $2013$ 年开始比赛，所以第七届举办时间就是 $2013+7-1 = 2019$ 年，而第十届没有举办，也就是 $2013+10-1 = 2022$ 年没有举办。因此只要判断一下是否是 $2022$ 年还有是否年份小于了 $2019$ 年即可。


```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;
ll x;
int main(){
	cin>>x;
	if(x<2019 || x==2022) cout<<"No";//判断
	else cout<<"Yes";
	return 0;
}


```

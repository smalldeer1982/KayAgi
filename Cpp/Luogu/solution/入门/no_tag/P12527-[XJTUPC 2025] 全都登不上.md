## 思路
对于每个字符串，逐字符判断，若不属于需删除的集合 $\{\tt{1}, 
\tt{i}, \tt{I}, \tt{l}\}$ 中，输出即可。

## 代码  
```cpp
#include <bits/stdc++.h>
using namespace std;
#define is(c) (c == 'i' || c == '1' || c == 'I' || c == 'l')
string s;
signed main() {
    cin.tie(0), cout.tie(0) -> sync_with_stdio(0);
    int T; cin >> T;
    while(T--) {
        cin >> s;
        for(char c : s)
            if(!is(c)) cout << c;  // 保留非删除字符
        cout << '\n';
    }
    return 0;
}
```

## 题目分析：
本题考查对字符串的理解与应用。

简单来说：给定一个字符串 $s$，判断字符串里是否含有 `1`，`i`，`I`，`l`。如果有的话，那这个字符。就不输出，否则就输出这个字符。

注意：输出最后记得要换行！空串也是要输出的！

## 正解：

```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long
int n;
inline int read() {
	int x = 0, f = 1;
	char ch = getchar();
	while (ch < '0' || ch > '9') {
		if (ch == '-')
			f = -1;
		ch = getchar();
	}
	while (ch >= '0' && ch <= '9') {
		x = (x << 3) + (x << 1) + (ch ^ 48);
		ch = getchar();
	}
	return x * f;
}
inline void write(int x) {
	if (x < 0) {
		putchar('-');
		x = -x;
	}
	if (x > 9)
		write(x / 10);
	putchar(x % 10 + '0');
}
signed main() {
	ios::sync_with_stdio(0);
	cin.tie(0), cout.tie(0);
	cin >> n;
	while (n--) {
		string s;
		cin >> s;
		for (int i = 0; i < s.size(); i++) {
			if (s[i] == '1' || s[i] == 'i' || s[i] == 'I' || s[i] == 'l') continue;
```

对于每个字符串遍历一遍，如果当前字符不是 `1`、`i`、`I`、`l` 就原样输出。

### 代码实现

```cpp
#include<iostream>
using namespace std;
int main(){
    int n;
	cin>>n;
    while(n--){
        string s;
        cin>>s;
        for(int i=0;i<s.size();i++){
            if(s[i]!='1'&&s[i]!='i'&&s[i]!='I'&&s[i]!='l')cout<<s[i];
        }
        cout<<'\n';
    }
	return 0;
}
```

## [P12527 [XJTUPC 2025] 全都登不上](https://www.luogu.com.cn/problem/P12527)

### 思路
首先题目说将每个密码中的容易混淆的字母都删掉，所以可以新建一个字符串，一旦不是容易混淆的字母，就加上，之后输出这个新创建的字符串就行了。

### Code


```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin>>n;
    while (n--){
        string s;
        cin>>s;
        string ans;
        for (int i=0;i<s.size();i++){
            if (s[i]!='1' && s[i]!='i' && s[i]!='l' && s[i]!='I') ans+=s[i];
        }
        cout<<ans<<endl;
    }
    return 0;
}

```

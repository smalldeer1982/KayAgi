# 小玉家的电费

## 题目描述

夏天到了，各家各户的用电量都增加了许多，相应的电费也交的更多了。小玉家今天收到了一份电费通知单。小玉看到上面写：据闽价电 [2006]27 号规定，月用电量在 $150$ 千瓦时及以下部分按每千瓦时 $0.4463$ 元执行，月用电量在 $151\sim 400$ 千瓦时的部分按每千瓦时 $0.4663$ 元执行，月用电量在 $401$ 千瓦时及以上部分按每千瓦时 $0.5663$ 元执行。小玉想自己验证一下，电费通知单上应交电费的数目到底是否正确呢。请编写一个程序，已知用电总计，根据电价规定，计算出应交的电费应该是多少。


## 样例 #1

### 输入

```
267```

### 输出

```
121.5```

# 题解

## 作者：MyukiyoMekya (赞：18)

最近在学Python，然而并没有AFO，所以想到了一个顽固Python语法的好方法：在luogu刷题

这题可以先判是否超过401，超过就把那部分计算掉，然后再判剩下的是否超过151，超过也把那部分计算掉，然后剩下的直接计算即可

```cpp
a=int(input()) #读入用电量
b=0.0		#储存答案的
if a>=401:		#判断是否超过400
	b+=(a-400)*0.5663	#计算超过部分钱数
	a=400				#减掉超过部分
if a>=151:
	b+=(a-150)*0.4663
	a=150
b+=a*0.4463
print('%.1f' %b)		#保留一位输出答案

```

## 作者：GoldenSTEVE7 (赞：7)

## 思路
这是一道典型的分段计费问题。我们跟着题目条件来，设输入的数为 $n$，所交电费为 $S$，分以下三种情况讨论：

- 若 $n \le 150$，则 $S = n \times 0.4463$。
- 若 $150 < n \le 400$，则 $S = 150 \times 0.4463 + (n - 150) \times 0.4663$。
- 若 $400 < n$，则 $S = 150 \times 0.4463 + 250 \times 0.4663 + (n - 400) \times 0.5663$。

总结一下，分段计费问题的核心思路就是分段讨论。（~~就像你七年级学的零点分段~~）

还有个问题就是四舍五入到小数点后一位，我们把这个过程形式化地写下来。设待进行四舍五入的小为 $a$，答案为 $a'$。

- 若 $100a \bmod 10 < 5$，$a' = [10a]\div 10$。
- 若 $100a \bmod 10 \ge 5$，$a' = [10a+1]\div 10$。

其中，$[a]$ 表示 $a$ 向下取整。

其实，上述过程可以简化为
$$
a' = [10a+0.5] \div 10
$$

请大家牢记这种东西，因为如果你在这种地方出现错误，那就是非常可惜了。

## 代码 
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int n; cin >> n;
    double b = 0;
    if(n <= 150) { // 讨论，求解答案
        b = n * 0.4463;
    } else if(n > 150 && n <= 400) {
        b = 150 * 0.4463 + (n - 150) * 0.4663;
    } else {
        b = 150 * 0.4463 + 250 * 0.4663 + (n - 400) * 0.5663;
    } cout << floor(b * 10 + 0.5) / 10.0; // 四舍五入
    return 0;
}
```

> 月用电量在 $150$ 千瓦时及以下部分按每千瓦时 $0.4463$ 元执行，月用电量在 $151\sim 400$ 千瓦时的部分按每千瓦时 $0.4663$ 元执行，月用电量在 $401$ 千瓦时及以上部分按每千瓦时 $0.5663$ 元执行。

那么我们就可以分三种情况，设月用电量为 $x$：
1. $0 < x \le 150$\
那么输出 $x \times 0.4463$ 即可。
3. $150 < x \le 400$\
那么输出 $150 \times 0.4463 + (x-150) \times 0.4663$ 即可。
4. $400 < x$\
那么输出 $150 \times 0.4463 + 250 \times 0.4663 + (x-400) \times 0.5663$ 即可。

按照以上所述模拟即可，剩下来的看代码。

### 思路
分支结构，注意认真读题，然后模拟即可。

- 月用电量在 $150$ 千瓦时及以下部分按每千瓦时 $0.4463$ 元算。

- 月用电量在 $151\sim 400$ 千瓦时的部分按每千瓦时 $0.4663$ 元算，再加上 $150 \times 0.4463$ 即可。

- 月用电量在 $401$ 千瓦时及以上部分按每千瓦时 $0.5663$ 元算，再加上 $150 \times 0.4463$ 以及 $250 \times 0.4663$ 即可。

### 代码
```c++
#include <bits/stdc++.h>
using namespace std;

double a;
int main() {
	cin >> a;
	if (a <= 150) cout << setprecision(1) << fixed << a * 0.4463;
	else if (a >= 151 && a <= 400) cout << setprecision(1) << fixed << 150 * 0.4463 + (a - 150) * 0.4663;
	else cout << setprecision(1) << fixed << 150 * 0.4463 + 250 * 0.4663 + (a - 400) * 0.5663;
	return 0;
}

# [入门赛 #19] 分饼干 II

## 题目描述

老师有 $N$ 块饼干，要分给 $k$ 名小朋友。

每名小朋友至少拿到一块饼干，老师想让每名小朋友拿到的饼干数量都不一样多，请问老师能否实现这个目标。

## 说明/提示

### 数据规模与约定

- 对于 $50\%$ 的测试数据 $1 \le k \le 1000$，$1 \le N \le 10^6$。
- 对于 $100\%$ 的测试数据，$1 \le k,N \le 10^9$。
- $1 \le T \le 10^5$

## 样例 #1

### 输入

```
1
1 1
```

### 输出

```
Yes
```

## 样例 #2

### 输入

```
1
5 3
```

### 输出

```
No
```

# 题解

## 作者：LogicNotFound (赞：43)

   想要满足 $k$ 个小朋友，每个小朋友的饼干数都不同，那么最低满足要求的分配方式就是：从 小朋友 $1$ 到小朋友 $k$，分配的饼干数量依次为：$1$，$2$，...，$k-1$，$k$。即最低满足要求的饼干总数为 $(k+1) * k/2 $， 公差为 $1$ 等差数列求和嘛。

   而只要饼干总数 $n$ 大于最低满足要求的饼干数，那么说明能够分配。多出来的饼干，都给最后一个小朋友就行。
   
   最后上代码
   ```
#include<bits/stdc++.h>
using namespace std;
int main(){
	long long n,k;
	int t;
	cin>>t;
	while(t--){
		cin>>n>>k;
		if(n>=((k+1)*k)/2) cout<<"Yes\n";
		else cout<<"No\n";	
	}
 
	return 0;
}

```

## 作者：Genius_Star (赞：8)

### 思路：

~~水一道简单题……~~

考虑求出分饼干数量的最小值。

因为每名小朋友的饼干数量都 $>0$，且都互不相同。

明显只要是 $[1,2,3,\cdots,k]$ 需要的糖果数量是最少的，考虑等差数列求和公式，其和为 $\frac{k \times (k+1)}{2}$。

只需要判断 $\frac{k \times (k+1)}{2}$ 是否大于 $n$ 即可，如果大于，则分不了。

**注意开 long long！**

### 完整代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
typedef double db;
inline ll read(){
    ll x=0,f=1;
    char c=getchar();
    while(c<'0'||c>'9'){
        if(c=='-')
          f=-1;
        c=getchar();
    }
    while(c>='0'&&c<='9'){
        x=(x<<1)+(x<<3)+(c^48);
        c=getchar();
    }
    return x*f;
}
inline void write(ll x){
	if(x<0){
		putchar('-');
		x=-x;
	}
	if(x>9)
	  write(x/10);
	putchar(x%10+'0');
}
ll T,n,k; 
int main(){
	T=read();
	while(T--){
		n=read(),k=read();
		if(k*(k+1)/2<=n)
		  puts("Yes");
		else
		  puts("No");
	}
	return 0;
}
```

### 思路
这个题目很简单，只要从 $1$ 加到 $k$ 的和大于 $n$ 的话，就是不可以的。

那么我们就可以用到[高斯求和](https://baike.baidu.com/item/%E7%BA%A6%E7%BF%B0%C2%B7%E5%8D%A1%E5%B0%94%C2%B7%E5%BC%97%E9%87%8C%E5%BE%B7%E9%87%8C%E5%B8%8C%C2%B7%E9%AB%98%E6%96%AF/9963604?fromtitle=%E9%AB%98%E6%96%AF%E6%B1%82%E5%92%8C&fromid=7487795&fr=aladdin)了，代价应该都知道怎么求吧。（~~可是我们没有高斯聪明~~）

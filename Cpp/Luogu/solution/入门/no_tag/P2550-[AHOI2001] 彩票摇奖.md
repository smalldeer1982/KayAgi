# [AHOI2001] 彩票摇奖

## 题目描述

为了丰富人民群众的生活、支持某些社会公益事业，北塔市设置了一项彩票。该彩票的规则是：

1. 每张彩票上印有 $7$ 个各不相同的号码，且这些号码的取值范围为 $1\sim33$。
2. 每次在兑奖前都会公布一个由七个各不相同的号码构成的中奖号码。
3. 共设置 $7$ 个奖项，特等奖和一等奖至六等奖。

兑奖规则如下：
- 特等奖：要求彩票上 $7$ 个号码都出现在中奖号码中。
- 一等奖：要求彩票上有 $6$ 个号码出现在中奖号码中。
- 二等奖：要求彩票上有 $5$ 个号码出现在中奖号码中。
- 三等奖：要求彩票上有 $4$ 个号码出现在中奖号码中。
- 四等奖：要求彩票上有 $3$ 个号码出现在中奖号码中。
- 五等奖：要求彩票上有 $2$ 个号码出现在中奖号码中。
- 六等奖：要求彩票上有 $1$ 个号码出现在中奖号码中。

注：兑奖时并不考虑彩票上的号码和中奖号码中的各个号码出现的位置。例如，中奖号码为 $23\ 31\ 1\ 14\ 19\ 17\ 18$，则彩票 $12\ 8\ 9\ 23\ 1\ 16\ 7$ 由于其中有两个号码（$23$ 和 $1$）出现在中奖号码中，所以该彩票中了五等奖。

现已知中奖号码和小明买的若干张彩票的号码，请你写一个程序帮助小明判断他买的彩票的中奖情况。

## 说明/提示

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1 \leq n\lt1000$。

## 样例 #1

### 输入

```
2
23 31 1 14 19 17 18
12 8 9 23 1 16 7
11 7 10 21 2 9 31```

### 输出

```
0 0 0 0 0 1 1```

## 作者：_____Elysia_____ (赞：13)

数组入门题。给出彩票的中奖号码和 $n$ 张彩票的号码，求各个奖项的获奖次数。特等奖是要求彩票上 $7$ 个号码出现在中奖号码中，一等奖是 $6$ 个，以此类推，六等奖是 $1$ 个。

可以开一个数组存放中奖号码，然后依次读入 $n$ 张彩票的号码，每读入一个号码就判断是否与中奖号码之一相等，读入完 $7$ 个号码后就把相等的数量累加到答案数组里，最后输出答案即可。

```cpp
#include<bits/stdc++.h>
using namespace std;
int zhong[11],ans[11];
int main(){
    int n;
    cin>>n;
    for(int i=1;i<=7;i++)cin>>zhong[i];
    for(int i=1;i<=n;i++){
        int d=0;
        for(int j=1;j<=7;j++){
            int x;
            cin>>x;
            for(int k=1;k<=7;k++){
                if(x==zhong[k]){
                    d++;
                    break;
                }
            }
        }
        ans[d]++;
    }
    for(int i=7;i>=1;i--)cout<<ans[i]<<' ';
}
```

# 题目大意
### 题意
给出中奖号码和小明买的 $n$ 张彩票的号码，求他买的彩票的中奖情况。
### 思路
暴力枚举小明买的第 $i$ 张彩票上有几个号码出现在中奖号码中，最后输出答案。
## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, ans[10], a[10];//定义

int main() {
	cin >> n;
	for (int i = 1; i <= 7; i++) {
		cin >> a[i];//输入中奖号码
	}
	for (int j = 1; j <= n; j++) {
		int sum = 0;
		for (int i = 1; i <= 7; i++) {
			int b;
			cin >> b;
			for (int k = 1; k <= 7; k++) {
				if (a[k] == b) {//判断有几个号码中奖
					sum++;
				}
			}
		}

		ans[sum]++;//记录答案
	}
	for (int i = 7; i >= 1; i--) {
		cout << ans[i] << " ";//输出答案
	}

	return 0;
}
```

## 作者：Ashankamiko (赞：2)

# 题目简述
### 题意
给定 $n$ 个人的彩票数字和中间序列，求中奖情况。
- 特等奖：要求彩票上 $7$ 个号码都出现在中奖号码中。
- 一等奖：要求彩票上有 $6$ 个号码出现在中奖号码中。
- 二等奖：要求彩票上有 $5$ 个号码出现在中奖号码中。
- 三等奖：要求彩票上有 $4$ 个号码出现在中奖号码中。
- 四等奖：要求彩票上有 $3$ 个号码出现在中奖号码中。
- 五等奖：要求彩票上有 $2$ 个号码出现在中奖号码中。
- 六等奖：要求彩票上有 $1$ 个号码出现在中奖号码中。
### 思路
只需按照题意模拟，每次输入的时候记录有多少个数字中奖了，最后用输出中奖情况。

由于数字不是一一对应，所以我们输入中奖序列时，可以用 $vis$ 数组表示 $i$ 是中奖号码，这样处理的时候只需要令 $cnt \gets cnt + vis_x$ 即可，其中，$x$ 表示输入的这个数字。
## [AC 代码](https://www.luogu.com.cn/record/209152977)
```cpp
#include <bits/stdc++.h>
using namespace std;
#define in cin
#define out cout
int vis[34], _end[10];
int main() {
	int n;
	in >> n;
	for (int i = 0, x; i < 7; i++) {
		in >> x;
		vis[x] = 1; //标记这个数，说明若有这个数，则算中奖
	}
	while (n--) {
		int cnt = 0;
		for (int i = 0, x; i < 7; i++) {
			in >> x;
			cnt += vis[x]; //中奖了就可以+1，否则不+1
		}
		_end[cnt]++; //记录中奖情况
	}
	for (int i = 7; i > 0; i--) //输出中奖情况
		out << _end[i] << ' ';
	return 0;
}
```

# 思路

首先，我们仔细阅读题目可知，中奖号码范围非常小，仅仅在 1 至 **33** 之间，所以我们就可以使用 vis 数组来记录中奖号码，再枚举所有号码与中奖号码相同的个数即可。

# 代码


```cpp
#include <bits/stdc++.h>

#define int long long

using namespace std;

const int maxn = 10005;

int n;

int vis[maxn]; // 记录中奖号码

int ans1, ans2, ans3, ans4, ans5, ans6, ans7; // 答案

inline int read() // 快读
{
	int ret = 0, f = 1;
	char ch = getchar();
	while (ch < '0' || ch > '9')
	{
		if (ch == '-')
			f = -f;
		ch = getchar();
	}
	while (ch >= '0' && ch <= '9')
		ret = ret * 10 + ch - '0',
		ch = getchar();
	return ret * f;
}

signed main()
{
	n = read();
	for (int i = 1; i <= 7; ++i)
		vis[read()] = 1;
	for (int i = 1; i <= n; ++i)
	{
		int a, b, c, d, e, f, g;
		int sum = 0;
		// 输入
		a = read();
		b = read();
		c = read();
		d = read();
		e = read();
		f = read();
		g = read();
		// 判断是否有相同号码
		if (vis[a])
			++sum;
		if (vis[b])
			++sum;
		if (vis[c])
			++sum;
		if (vis[d])
			++sum;
		if (vis[e])
			++sum;
		if (vis[f])
			++sum;
		if (vis[g])
			++sum;
		// 记录答案
		if (sum == 7)
			++ans1;
		if (sum == 6)
			++ans2;
		if (sum == 5)
			++ans3;
		if (sum == 4)
			++ans4;
		if (sum == 3)
			++ans5;
		if (sum == 2)
			++ans6;
		if (sum == 1)
			++ans7;
	}
	printf("%lld %lld %lld %lld %lld %lld %lld\n", ans1, ans2, ans3, ans4, ans5, ans6, ans7); // 输出
	return 0;
}
```

## 作者：GuoMingXuants1 (赞：1)

给定 $ 7 $ 个中奖号码，以及 $ n $ 个彩票，兑奖规则题目已阐述，现在，求每张彩票中有几个号码出现在中奖号码中。

本题较易，使用桶思想即可解决。

开个桶记录一个号码是否为中奖号码，然后判断，记录一张彩票中中奖号码的数量，最后统计不同奖项的数量。


```cpp
#include <iostream>
using namespace std;
int p[7],a[35];
int main()
{
    int n;
    cin>>n;
    for(int i=1;i<=7;i++)
    {
        int k;
        cin>>k;
        a[k]++;
    }
    for(int i=1;i<=n;i++)
    {
        int sum = 0;
        for(int j=1;j<=7;j++)
        {
            int x;
            cin>>x;
            if(a[x] > 0)
            {
                sum++;
            }
        }
        p[7-sum]++; //通过观察可以发现，7和中奖号码数量的差是几就是几等奖，0表示特等奖。
    }
    for(int i=0;i<=6;i++)
    {
        cout<<p[i]<<" ";
    }
}
```

## 作者：chenhanzheapple (赞：0)

# 思路

首先使用布尔数组记录每个数是否出现。

对于小明购买的每张彩票，遍历每一个数字，看它是否在布尔数组中出现，如果出现，就将计数变量加一。依照题意判断中了几等奖，在答案数组中加一即可。

# 代码


```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
int ans[10];
bool vis[205];
int main(){
    cin >> n;
    for(int i=1;i<=7;i++){
        int x;
        cin >> x;
        vis[x] = 1;//记录每个数是否在中奖号码中出现过
    }
    for(int i=1;i<=n;i++){
        int cnt = 0;//记录有多少个号码和中奖号码一样
        for(int j=1;j<=7;j++){
            int x;
            cin >> x;
            if(vis[x]){//如果在中奖号码中出现了
                cnt++;//将计数变量加一
            }
        }
        ans[7-cnt]++;//判断是几等奖并将答案加一
    }
    for(int i=0;i<=6;i++){
        cout << ans[i] << " ";//输出
    }
    return 0;
}
```

# 【深基15.例1】询问学号

## 题目描述

有 $n(n \le 2 \times 10^6)$ 名同学陆陆续续进入教室。我们知道每名同学的学号（在 $1$ 到 $10^9$ 之间），按进教室的顺序给出。上课了，老师想知道第 $i$ 个进入教室的同学的学号是什么（最先进入教室的同学 $i=1$），询问次数不超过 $10^5$ 次。

## 样例 #1

### 输入

```
10 3
1 9 2 60 8 17 11 4 5 14
1 5 9
```

### 输出

```
1
8
5```

## STL里的vector
__vector就是变长数组，我们使用push_back(val)的方式就能向vector里
添加新元素了__


```cpp
#include <iostream>
#include <vector>
using namespace std;
vector<long > vec;
int main() {
	ios::sync_with_stdio(false);
	cin.tie(0);
	int n, m;
	cin >> n >> m;
	while (n--) {
		long x;
		cin >> x;
		vec.push_back(x); 
	}
	
	while (m--) {
		int idx;
		cin >> idx;
        // 注意i从1开始，但下标从0开始，所以要-1
		cout << vec[idx - 1] << endl;
	}
	return 0;
}

## 作者：hensier (赞：12)

这道题考察了数组的用法。我们设置一个数组保存学生即可。当然，我们首先输入$n,m$，然后在$i∈[1,n]$区间输入$a[i]$。输入完毕后，再$j∈[0,m)$输入参数$x$即可（可以不用数组）。

我们可以得到代码：

```cpp
#include<cstdio>
int n,m,x,a[2000001];//定义变量
int main()
{
    scanf("%d%d",&n,&m);
    for(int i=1;i<=n;i++)scanf("%d",&a[i]);//不停输入
    while(m--)
    {
        scanf("%d",&x);
        printf("%d\n",a[x]);//实现一边输入一边输出，不需要申请数组来浪费空间
    }
}
```

## 作者：chufuzhe (赞：10)

### 题意
有n个同学进了教室，进行m次询问，问第几个进入教室的同学。
### 实现
我们可以开一个数组，输入进入教室的学号，然后再输出每次询问为下标的那个数。注意n和m都比较大，建议用快读和快出。
### 代码
```cpp
#include <bits/stdc++.h> //头文件
using namespace std;
inline int read() //快读
{
   int s=0,w=1;
   char ch=getchar();
   while(ch<'0'||ch>'9'){if(ch=='-')w=-1;ch=getchar();}
   while(ch>='0'&&ch<='9') s=s*10+ch-'0',ch=getchar();
   return s*w;
}
inline void write(int x) //快出
{
    if(x<0)
	{
    	putchar('-');
		x=-x;
	}
    if(x>9)
		write(x/10);
    putchar(x%10+'0');
}
int a[2000005]; //定义一个数组
int main()
{
	int n,m; //定义，和题目中表示的一样
	n=read(); //输入
	m=read();
	for(register int i=1;i<=n;i++)
		a[i]=read();
	for(register int i=1;i<=m;i++) //输出
		write(a[read()]),puts(""); //以询问的数为下标的那个数
	return 0;
}

```

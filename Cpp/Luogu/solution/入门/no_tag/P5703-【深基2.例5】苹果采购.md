# 【深基2.例5】苹果采购

## 题目描述

现在需要采购一些苹果，每名同学都可以分到固定数量的苹果，并且已经知道了同学的数量，请问需要采购多少个苹果？

## 样例 #1

### 输入

```
5 3
```

### 输出

```
15
```

# 题解

## 作者：Make_China_Great (赞：15)

## 思路

根据题意描述，我们设每人分到的数量为 $a$，人数为 $b$，那么需要的苹果数就是 $a\times b$。

然后输入 $a$ 和 $b$，输出 $a\times b$ 即可，题目本身没什么难度，考验的是你基础语法以及变量学没学会。

## 代码


```cpp
#include<bits/stdc++.h>
using namespace std;
int a,b;
int main(){
    cin>>a>>b;
    cout<<a*b;
}
```

## 后记

什么叫头文件？


```cpp
#include<bits/stdc++.h>
```

头文件有很多，但就用这个万能头文件就可以了。

什么叫名字空间？


```cpp
using namespace std;
```

写在头文件的后面，一般为第二行。

第三行定义 `int` 类型的变量，我们声称全局变量，当然，在这道题里，定义在主函数里也可以。

以后你们会知道，定义成全局变量的自动赋值 $0$，但是跑程序慢，定义在主函数内或子函数内的变量需要虽然快，但需要手动赋值。

第四行表示主函数，写法如下


```cpp
int mian(){

}
```

后面的大括号打在哪里取决于主函数内内容长度。

之后输入用 `cin>>a;  //a为变量名` 输入，输出用 `cout<<a;  //a为变量名` 输出。

需要注意每一行后面都应该有分号。

然后在最后一行打上主函数的右括号就完美结束了，记住，一定不要把 `cin>>a;` 给打成 `cin<<a;` 哦。

## 作者：ChenHaoQi (赞：5)

简化题意，给出每个同学得到苹果的数量 $a$ 和同学的数量 $b$，求苹果的总数。

根据小学二年级的等量关系相关知识可得，答案应该是 $a \times b$ 的值。现在让我们看看怎么用 C++ 语言实现这一点。

在 C++ 中，我们首先要判断数据类型。题目说“输入两个不超过 $10^9$ 的正整数”，容易发现 $10^9<2^{31}-1$，所以使用 `int` 类型即可。

乘法运算，在 C++ 语言中不是用 `x` 或 `X` 表示，而是采用 `*` 表示（在键盘上的位置如下图），代码中的 `a*b` 就是数学中的 $a \times b$ 。

![](https://cdn.luogu.com.cn/upload/image_hosting/m2ond2yh.png)

有了这些，我们就可以通过这道题了。代码如下：
```cpp
#include<iostream>
using namespace std;

int main(){
    int a,b;
    cin>>a>>b;
    cout<<a*b; //计算a乘b的值并输出
    return 0;
}

### 题目大意

现在需要采购一些苹果，每名同学都可以分到固定数量的苹果，并且已经知道了同学的数量，请问需要采购多少个苹果？

### 题目分析

假设有 $a$ 个同学，每人 $b$ 个苹果，那么所需要采购的苹果数就是 $a$ 个 $b$ 之和，也就是 $a \times b$ 个苹果。

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int a,b;
    cin>>a>>b;
    cout<<a*b;
    return 0;
}
```

## 题意
给定人数和每个人需要的苹果数，求需要多少个苹果。

题目保证输入和答案都是 int 范围内的整数。

## 思路
根据题意，假设有 $a$ 个人，每个人需要 $b$ 个苹果，那么一共就需要 $a \times b$ 个苹果，于是直接照此模拟即可。

## 代码
```cpp
#include <iostream>
using namespace std;

int a, b;

int main() {
	scanf("%d%d", &a, &b); // 输入

    printf("%d\n", a * b); // 输出答案
	
	return 0;
}
```

## 作者：Tachibana27 (赞：0)

考察小学数学，数据读取，数据输出。

思考算法：有 $a$ 个人，每个人要 $b$ 个苹果，显然需要 $a\times b$ 个苹果才能够满足。

首先需要引入头文件，一般直接使用 `bits/stdc++.h` 即可。

对于数据读取，可以使用 C++ 风格的 `std::cin` 来读取。具体的，```std::cin>>a>>b``` 可以将读到的第一个整数赋值给 $a$，第二个赋值给 $b$。

对于输出，同样可以使用 C++ 风格的 `std::cout` 实现。`std::cout<<a*b` 即可输出 $a\times b$。

在程序的最后，一般会加上 `return 0` 以避免其他奇怪的错误。

```cpp
#include<bits/stdc++.h>
int a,b;
int main(){
    std::cin>>a>>b;
    std::cout<<a*b;
    return 0;
}
```

# 【深基2.例6】字母转换

## 题目描述

输入一个小写字母，输出其对应的大写字母。例如输入 q[回车] 时，会输出 Q。

## 样例 #1

### 输入

```
q```

### 输出

```
Q```

# 题解

## 作者：meyi (赞：32)

提供一种完全依赖库函数的解法
```cpp
#include<cctype>	//toupper(char)的库
#include<cstdio>	//getchar()和putchar(char)的库
int main(){
	/*	
    	getchar():输入
    	putchar(char):输出
        toupper(char):把小写字母转为大写
    */
	putchar(toupper(getchar()));
    return 0;
}
```

## 题目大意
给定一个小写英文字母，输出他的大写版本。
## 前置知识
要想解决这道题，我们要知道美国信息交换标准代码，简称 ASCII。它到目前为止定义个 $128$ 个字符。简单来说，任何一个字符都可以用一个数来代替。以下是该表：
![](https://cdn.luogu.com.cn/upload/image_hosting/a6x63sqi.png)
## 题目解法
见表，我们会发现所有的小写字母的 ASCII 值都比他的大写版本大 $32$，所以我们只需要让输入的字符减去 $32$ 就可以得出正确结果。

这里我们需要知道一下 `char` 类型与 `int` 类型的转换规则。

如果你用一个 `char` 类型的字符减去一个 `int` 类型的数，那么得到的结果是 `int` 类型的。所以结果我们还需要让他强制转换为 `char` 类型，详见代码。

## AC code
```.cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    char c;
    cin>>c;
    cout<<char(c-32);
}
```

### 题目大意

将一个小写字符转换为其大写形式并输出。

### 题目分析

本题实际上考察的是字符的 ASCII 码值。

C++ 在存储字符的时候实际上存的是一个 $0 \sim 127$ 的数字，表示着这个字符的 ASCII 码值。

ASCII 码值的 $65 \sim 90$ 对应着 $\text{A} \sim \text{Z}$，$97 \sim 122$ 对应着 $\text{a} \sim \text{z}$，因此我们只要将输入的字符减去 $32$ 即可将小写字符转换为其大写形式。

### 代码

注意在输出的时候要使用强制类型转换将其变成 `char` 类型，否则会输出一个整数（其实是我们要输出的字符的 ASCII 码值）

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	char ch;
	cin>>ch;
	cout<<(char)(ch-32);
}
```

## 作者：ko114 (赞：1)

### 题目大意
将输入的一个小写字母转换为大写字母。
### 前置知识
对于 ASCII 码值位于 $97\sim 122$ 的字符，存在大写形式。

在 Python 中，`str.upper` 可将一个字符串中的存在大写形式的字符转换为这个字符的大写形式，无大写形式的保持不变。

### 实现方式
1. 读入一个字符 `x`；
2. 将 `ans` 设为 `x.upper()`；
3. 输出变量 `ans`。

### 具体代码
```python
x = input()[0]  # 只存入一个字符
ans = x.upper()  # 转为大写
print(ans)  # 输出答案
```

注意到这些中间变量是无用的，省略后的代码如下：
```python
print(input()[0].upper())
```

# 【深基2.例8】再分肥宅水

## 题目描述

现在有 $t$ 毫升肥宅快乐水，要均分给 $n$ 名同学。每名同学需要 $2$ 个杯子。现在想知道每名同学可以获得多少毫升饮料（严格精确到小数点后 $3$ 位），以及一共需要多少个杯子。



## 说明/提示

对于所有数据，$0\leq t\leq 10000$ 且小数点后不超过 $3$ 位，$1\leq n\leq 1000$。

## 样例 #1

### 输入

```
500.0 3```

### 输出

```
166.667
6```

第二问：一个人需要2个杯子，所以一共需要$2 \times n$个杯子。

注意$t$要是`double`型（`float`也没问题就是咯），结果保留3位小数。

代码：
```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
	double a;
	int b;
	cin>>a>>b;
	cout<<setprecision(3)<<fixed<<a/b<<endl<<b*2;
	return 0;
}
```

## 作者：Clare613 (赞：5)

## 思路：
这道题我们需要认识 `prinf();`，这个语句可以有利于我们输出小数，如以下语句，是输出小数 $a$ 并保留 $6$ 位小数：
```
#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
    double a;
    cin>>a;
    printf("%.6lf",a);
    return 0;
}
```
只要学会这条语句怎么用，我们就可以秒切这道题了，剩下来的已知有 $k$ 毫升肥宅水和 $n$ 个同学，那么每个同学就可以分到 $\dfrac{k}{n}$ 毫升肥宅水，注意保留三位小数，而杯子数量自然就是 $n \times 2$ 了。
## code：
```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
    double k;
    int n;
    cin>>k>>n;
    printf("%.3lf\n%d",k/n*1.000,n*2);
    return 0;
}
```

# P5706 题解

欢迎来到 AstralNahida 的题解频道！

这题所需的第一个输出是需要精确到小数点后 $3$ 位的，我们在 C++ 中有两种做法（保留不同位数只用把代码里面的 $3$ 改成其他数字就行）：

第一种需要用到两个头文件，即 `<iostream>`（用于提供流输出操作）和 `<iomanip>`（用于提供保留小数位数的操作）。一般形式为：
```cpp
std::cout << std::fixed << std::setprecision(3) << k;
```
其中 $k$ 是需要输出的数。

第二种做法用到 `<cstdio>`（是从 C 语言的 `<stdio.h>` 继承而来的，提供了 `printf` 的输出操作），一般形式为：
```cpp
printf("%.3lf", k);
```
其中 $k$ 也是需要输出的数。\
注意，使用 `%.3lf` 的格式会认为 $k$ 是 `double` 类型，如果是 `float` 的话，应当使用 `%.3f`。

接下来就是大家喜闻乐见的 AC Code 环节了：
```cpp
// 这里使用第一种做法。

#include <iostream>
#include <iomanip>

int main(void) {
    double t;
    int n;
    std::cin >> t >> n;
    std::cout << std::fixed << std::setprecision(3) << t / n << '\n';
    std::cout << 2 * n << '\n';
}
```

## 作者：AFO_Lzx (赞：1)

简单的小学乘除法。

$t$ 毫升的肥宅水平均分给 $n$ 个人，那么每个人可以分到 $\dfrac{t}{n}$ 毫升的肥宅水。一共 $n$ 个人，每个人要两个杯子，那么一共要 $2\times n$ 个杯子。

保留三位小数使用 ```fixed``` 和 ```setprecision(3)``` 实现。

```cpp
#include<bits/stdc++.h>
using ll = long long;
using namespace std;

signed main() {
	ios::sync_with_stdio(0);
	cin.tie(0), cout.tie(0);
	
	double t;
	int n;
	
	cin >> t >> n;
	cout << fixed << setprecision(3) << t / n << '\n';
	cout << n * 2 << '\n';
	return 0;
}
```

## 作者：luogu_gza (赞：1)

### 题目大意

现在有 $t$ 毫升肥宅快乐水，要均分给 $n$ 名同学。每名同学需要 $2$ 个杯子。现在想知道每名同学可以获得多少毫升饮料（严格精确到小数点后 $3$ 位），以及一共需要多少个杯子。

### 题目分析

如果有 $t$ 毫升肥宅快乐水要均分给 $n$ 名同学，那每名同学可以获得 $t \div n$ 毫升的饮料（保留三位小数），每名同学需要 $2$ 个杯子，那么一共需要 $2 \times n$ 个杯子。

在 C++ 中，输出一个数 $x$，并保留小数点后三位数（其他位数只需要把代码里面的 $3$ 改成其他数字即可）有两种方法：

1. `cout<<fixed<<setprecision(3)<<x;`

2. `print("%.3lf",x);`，注意此处是 `lf` 是假定 $x$ 是 `double` 类型的，如果是别的类型需要进行替换。

### 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	double a;
	int b;
	scanf("%lf%d",&a,&b);
	printf("%.3lf\n%d",a/b,b*2);
    return 0;
}
```

- 将 $t$ 毫升肥宅快乐水分给 $n$ 个人，每个人可以得到 $\frac{t}{n}$ 毫升肥宅快乐水。
- 每个同学需要两个杯子，因此一共需要 $2\times n$ 个杯子。
直接输出即可。c++ 中可以使用 `printf("%.3lf",x)` 来保留三位小数输出。

```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
	double t;
	int n;
	cin>>t>>n;
	double a=n;
	double ml=t/a;
	printf("%.3lf\n%d",ml,n*2);
    return 0;
}
```

# 【深基2.习6】Apples Prologue / 苹果和虫子

## 题目描述

小 B 喜欢吃苹果。她现在有 $m$（$1 \le m \le 100$）个苹果，吃完一个苹果需要花费 $t$（$0 \le t \le 100$）分钟，吃完一个后立刻开始吃下一个。现在时间过去了 $s$（$1 \le s \le 10000$）分钟，请问她还有几个完整的苹果？

## 说明/提示

如果你出现了 RE，不如检查一下被零除？

## 样例 #1

### 输入

```
50 10 200```

### 输出

```
30```

# 题解

## 作者：piggymichael (赞：443)

题目还是挺简单的，但是要注意细节！

首先，八尾勇吃了$s/t$个苹果，那么问题就来了，第一个坑：如果$t$是0，那么八尾勇肯定能吃完所有的苹果，输出0，但是0又不能做除数，这种情况要特判。

接着往下想，如果$s$是$t$的整数倍，那么八尾勇就吃了$s/t$个完整的苹果，剩下$m-s/t$个完整的苹果；如果$s$并不是是$t$的整数倍，八尾勇就吃了$s/t$个完整的苹果和一个残缺的苹果，那么就有$s/t+1$个苹果不完整了，剩下$m-s/t-1$个苹果。可是，第二个坑：说不定他在$s$秒内就吃完了这堆苹果了呢，那么就没有剩下的苹果，输出0。

上代码：
```cpp
#include<bits/stdc++.h>//万能头文件
using namespace std;
int m,t,s;
int main()
{
	cin>>m>>t>>s;
	if(t==0)//特判t=0的情况
	{
		cout<<0<<endl;
		return 0;
	}
	if(s%t==0)//如果s是t的整数倍
		cout<<max(m-s/t,0)<<endl;
	else//如果不是
		cout<<max(m-s/t-1,0)<<endl;
	return 0;//程序结束
}
```
程序中max(a,b)就是a,b中的最大值，如果$m-s/t$比0小，大的就是0了，输出的就是0。

### 题目
[P5709 【深基2.习6】Apples Prologue](https://www.luogu.com.cn/problem/P5709)

### 思路
数学题。吃一个苹果要 $t$ 分钟，吃了 $s$ 分钟，那么一共吃完了 $\lfloor\frac{s}{t}\rfloor (\lfloor x \rfloor \text{表示对 x 下取整})$ 个苹果，可能有正在吃没有吃完的苹果答案要减去 $1$。

### $Code$

```cpp
#include<iostream>//使用cin,cout要有的头文件

using namespace std;

int main() {//主函数
	int m,t,s;//三个整形变量
	cin>>m>>t>>s;//输入
	if(t==0) std::cout<<m<<'\n';//特判t=0
	else {
		m-=s/t;//苹果总数减去已经吃完了的数量
		if(s%t!=0) --m;//有正在吃没有吃完的苹果
		std::cout<<(m>=0?m:0)<<'\n';
		//条件?a:b 这个语句的作用是判断条件是否为真若为真返回 a 反之返回 b
	}
	return 0;
}
```

## 作者：Dark_Knight_AK_ALL (赞：9)

# 思路：
有 $m$ 个苹果，每 $t$ 分钟能吃一个苹果，吃了 $s$ 分钟，问还有几个完整的苹果。我们可以知道，在 $s$ 分钟内，可以吃 $s \div t$ 个苹果。那么剩下的就是 $m - s \div t$ 个苹果。注意 $t$ 是否为 $0$。$s \div t$ 要向下取整。
# 代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int m,t,s;
int main()
{
	cin>>m>>t>>s;
	if(t==0)
	{
		cout<<0<<endl;
		return 0;
	}
	if(s%t==0)
		{
			if(m-s/t>=0)
			cout<<m-s/t;
			else cout<<0;
		}
	else
		{
			if(m-s/t-1>=0)
			cout<<m-s/t-1;
			else cout<<0;
		}
	return 0;
}
```

## 作者：AFO_Lzx (赞：6)

简单题，题目好心地提醒了我们要特判 $t=0$ 的情况。

当 $t\ne0$ 时，我们实际上只需要判断 $s$ 是否是 $t$ 的倍数，如果不是的话就是正在吃一个苹果，已经啃了一些，不是完整的，就像这样：

![](https://cdn.luogu.com.cn/upload/image_hosting/cak7th45.png)

所以这就不能算一个完整的苹果，要将答案减去 $1$。

代码实现：

```cpp
#include<bits/stdc++.h>
using ll = long long;
using namespace std;

signed main() {
	ios::sync_with_stdio(0);
	cin.tie(0), cout.tie(0);
	
	int m, t, s;
	cin >> m >> t >> s;
	
	if (t == 0) cout << 0 << '\n';
	else {
		int ans = m - s / t;
		if (s % t != 0) ans--;
		cout << max(0, ans) << '\n';
	}
	
	return 0;
}
```

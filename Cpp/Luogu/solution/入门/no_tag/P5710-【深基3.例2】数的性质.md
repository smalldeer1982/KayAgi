# 【深基3.例2】数的性质

## 题目描述

一些整数可能拥有以下的性质：

- 性质 1：是偶数；
- 性质 2：大于 $4$ 且不大于 $12$。

小 A 喜欢这两个性质同时成立的整数；Uim 喜欢这至少符合其中一种性质的整数；小 B 喜欢刚好有符合其中一个性质的整数；正妹喜欢不符合这两个性质的整数。现在给出一个整数 $x$，请问他们是否喜欢这个整数？

## 样例 #1

### 输入

```
12```

### 输出

```
1 1 0 0```

# 题解

## 作者：TLE_qwq (赞：44)

## 题目分析
给定一个整数 $x$，判断它是否满足以下两个性质：
- 是偶数。
-  大于 $4$ 且不大于 $12$。
 
然后根据这两个性质的满足情况，输出四个人是否喜欢这个数字：
- 小 A：同时满足两个性质。
- Uim：至少满足一个性质。
- 小 B：恰好满足一个性质。
- 正妹：两个性质都不满足。

## 解题思路
1. 使用布尔变量 $prop1$ 和 $prop2$ 分别记录两个性质是否满足
2. 通过逻辑运算判断每个人的喜好：

## 代码实现
```cpp
#include <iostream>
using namespace std;
int main() {
    int x; cin >> x;
    bool p1 = x%2 == 0, p2 = x>4 && x<=12;
    cout << (p1&&p2) << " " << (p1||p2) << " " << (p1!=p2) << " " << (!p1&&!p2);
    return 0;
}
```

## 作者：A_Đark_Horcrux (赞：23)

看到没有用位运算的我就来发篇题解。

我们可以写出判断是否满足条件的两个函数：

```
int a(int x) {return !(x%2);}
int b(int x) {return x>4&&x<=12;} 
```

小A想要两个都满足 $\Rightarrow$ ```a(x)&b(x)=1;```

Uim想要至少一个满足 $\Rightarrow$ ```a(x)|b(x)=1;```

八尾勇想要刚好一个满足 $\Rightarrow$ ```a(x)^b(x)=1;```

正妹想要两个都不满足 $\Rightarrow$ ```!a(x)&!b(x)=1.```

由于符合想要的就输出1，不符合想要的就输出 0，那我们直接输出表达式的值就好了，后面可以不判断是否等于1.

```cpp
#include<cstdio>
using namespace std;
int x;
int a(int x) {return !(x%2);}//判断第一个条件
int b(int x) {return x>4&&x<=12;} //判断第二个条件
int main()
{
	scanf("%d",&x);//输入
	return !printf("%d %d %d %d",a(x)&b(x),a(x)|b(x),a(x)^b(x),!a(x)&!b(x));//分别判断
 } //结束awa
```

## 题目大意：
有一个整数，可能具有如下性质：
- 性质 $1$：是偶数。
- 性质 $2$：大于 $4$ 且不大于 $12$。

有四个人，他们喜欢的整数如下：
- 第一个喜欢两个性质都成立的整数。
- 第二个喜欢**至少**一个性质成立的整数。
- 第三个喜欢**仅有**一个性质成立的整数。
- 第四个喜欢两个性质都不成立的整数。

若喜欢则输出 $1$，否则输出 $0$。
## 算法分析：
我们可以先判断这个整数满足几个性质，再去和四个人喜欢的整数比较并输出，详细可以见代码。
## 代码展示：
```cpp
# include <bits/stdc++.h>

# define ll long long
# define int long long
# define rint register int

long long read(){long long s = 0 , w = 0; char c = getchar(); while(!isdigit(c)) w |= (c == '-') , c = getchar(); while(isdigit(c)) s = (s << 1) + (s << 3) + (c ^ 48) , c = getchar(); return w ? -s : s;}
void write(long long x){if(x < 0) putchar('-') , x = ~ (x - 1); if(x > 9) write(x / 10); putchar(x % 10 | 48);}
void writesp(long long x){write(x) , putchar(' '); }
void writeln(long long x){write(x) , putchar('\n');}

using namespace std;

long long n;
bool f1,f2;

signed main() {
  n = read();//输入 
  if(n % 2 == 0) f1 = true;
  if(n >= 4 && n <= 12) f2 = true;//判断满足几个性质 
  if(f1 + f2 == 2) writesp(1);
  else writesp(0);//第一个人
  if(f1 + f2 >= 1) writesp(1);
  else writesp(0); //第二个人
  if(f1 + f2 == 1) writesp(1);
  else writesp(0);//第三个人
  if(f1 + f2 == 0) write(1);
  else write(0);//第四个人
  return 0;
}
```

# 题解：P5710 【深基3.例2】数的性质

这道题非常简单，我给一种最简洁的代码。

首先我们可以使用 $cnt$ 存储这两个条件符合多少个，这两个条件转化成代码分别是 `x%2==0` 和 `x>4&&x<=12`。

我们再看看这四个人喜欢这个数的条件：

1.  两个条件必须成立，条件为 `cnt==2`。

2.  两个条件至少一个成立，条件为 `cnt>=1`。

3.  两个条件只能成立一个，条件为 `cnt==1`。

4.  两个条件都不能成立，条件为 `cnt==0`。

我们可以直接判断，就如：

```cpp
printf("%d %d %d %d",cnt==2,cnt>=1,cnt==1,cnt==0);
```

因为在输出时，我使用的是 `%d`，也就是整型，所以条件满足就会将布尔类型的 `true` 强制转换成 `1`，因此输出 `1`，反之亦然。

所以代码如下咩：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int x;
    int cnt=0;
    scanf("%d",&x);
    if(x%2==0){
        cnt++;
    }
    if(x>4&&x<=12){
        cnt++;
    }
    printf("%d %d %d %d",cnt==2,cnt>=1,cnt==1,cnt==0);
    return 0;
}
```

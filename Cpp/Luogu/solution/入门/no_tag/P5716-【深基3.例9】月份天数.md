# 【深基3.例9】月份天数

## 题目描述

输入年份和月份，输出这一年的这一月有多少天。需要考虑闰年。

## 说明/提示

数据保证 $1583 \leq y \leq 2020$，$1 \leq m \leq 12$。

## 样例 #1

### 输入

```
1926 8```

### 输出

```
31```

## 样例 #2

### 输入

```
2000 2```

### 输出

```
29```

# 题解

## 作者：sycqwq (赞：18)

这一题，只要是学过c++语法的并且有一点常识的都能过，，

大月31天，小月30天，二月28天（闰年29天）。

大月：1,3,5,7,8,10,12.

小月：2,4,6,9,11.

难点就在判断闰年。

闰年判断方法：

年份整除4或者年份整除400不整除100.

代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int rn(int y)//闰年函数
{
	return y%400==0||y%4==0&&y%100!=0;
 } 
int m[13]={0,31,28,31,30,31,30,31,31,30,31,30,31};//每个月的天数
int main(){
	int n,y;
	cin>>n>>y;
	if(rn(n))//如果是闰年，则二月有29天
		m[2]=29;
	cout<<m[y];//输出第y月，这里不需要特判，因为如果是闰年前面已经修改过天数了。
	
	return 0;
}

```

## 前言

~特判出奇迹~。

## 思路

这道题是求某一年的某一月有多少天，众所周知，一年有十二个月。

有一首童谣这样唱：

> 一三五七八十腊，
>
> 三十一天永不差。
>
> 四六九冬三十整，
>
> 二月二八或二九。

因此，我们除了二月，其余月份均可得出普遍天数，而闰年（即二月有 $29$ 天）的判定条件是被 $400$ 整除或被 $4$ 整除且不被 $100$ 整除，根据此模拟即可。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
	long long n,m;
	scanf("%lld %lld",&n,&m);
	if (m==1 || m==3 || m==5 || m==7 || m==8 || m==10 || m==12) {
        printf("31");
        return 0;
    }//对应第一句。
	if (m==4 || m==6 || m==9 || m==11) {
        printf("30");
        return 0;
    }//对应第三句。
	if ((n%400==0 || (n%4==0 && n%100!=0)) && m==2)
	{//闰年的二月。
		printf("29");
		return 0;
	}else printf("28");//平年的二月。
	return 0;
}
```

## 解题分析

首先判断该月份是否为 $2$ 月。

如果不是，则按 $1$、$3$、$5$、$7$、$8$、$10$、$12$ 月作大月 $31$ 天处理。

$2$、$4$、$6$、$9$、$11$ 月作小月 $30$ 天处理。

如果是，则判断该年是否为闰年。

年数被 $400$ 整除或不被 $100$ 整除但被 $4$ 整除的年叫做闰年，$2$ 月有 $29$ 天。

否则不是闰年，$2$ 月只有 $28$ 天。

代码如下：


```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;
int n,m,mo[13]={0,31,28,31,30,31,30,31,31,30,31,30,31};
int main(){
    ios::sync_with_stdio(0);
    cin.tie(0);cout.tie(0);
    cin>>n>>m;
    if(m!=2)cout<<mo[m];
    else if(n%4==0 and (n%100!=0 or n%400==0))cout<<29;
    else cout<<28;
    return 0;
}
```

### 思路
用一个数组记录 $1 \sim 12$ 月的天数，判断是否是闰年，如果是，$2$ 月有 $29$ 天，否则有 $28$ 天，输出对应月份即可。

```c++
#include <bits/stdc++.h>
using namespace std;

int y, m;
int a[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
bool check(int year) {
	return year % 400 == 0 || year % 4 == 0 && year % 100 != 0;
}
int main() {
	cin >> y >> m;
	if (check(y)) a[2] = 29;
	cout << a[m];
	return 0;
}

## 作者：Vct14 (赞：1)

如果是 $1,3,5,7,8,10,12$ 月，则答案一定为 $31$；如果是 $4,6,9,11$ 月，则答案一定为 $30$；如果是 $2$ 月，则需要进行闰年判断：若年份为 $4$ 的倍数且不为 $100$ 的倍数，或是年份为 $400$ 的倍数，则该年为闰年，否则不是。

```cpp
#include<bits/stdc++.h>
using namespace std;

int isrun(int a){
	if(a%4==0){
		if(a%100==0){
			if(a%400==0){
				return 1;
			}
			else{
				return 0;
			}
		}
		else{
			return 1;
		}
	}
	else{
		return 0;
	}
}

int main() {
	int n,m;
	cin>>n>>m;
	if(m==1 || m==3 || m==5 || m==7 || m==8 || m==10 || m==12){
		cout<<"31";
		return 0;
	}
	else if(m==2){
		int run=isrun(n);
		if(run==1) cout<<"29";
		else cout<<"28";
		return 0;
	}
	else cout<<"30";
    return 0;

## 作者：GoldenSTEVE7 (赞：1)

## 题意简述
给定年份和月份，求这个月有多少天。

## 思路
我们用小学学过的 N 种方法都可以知道，$1,3,5,7,8,10,12$ 月有 $31$ 天，除 $2$ 月之外，其余是 $30$ 天，但 $2$ 月要关注一下年份问题，如果是闰年， $2$ 月是 $29$ 天，否则 $28$ 天。

那我们可以使用一个数组，其下标对应月份，数值对应天数。在输出之前处理闰年的问题即可。

更详细的解释看下面的注释。

## 代码
```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;
int a[13];
signed main() {
    ios::sync_with_stdio(0);
    cin.tie(0); cout.tie(0);
    int y, m; cin >> y >> m;
    a[1] = a[3] = a[5] = a[7] = a[8] = a[10] = a[12] = 31; // 处理大月
    a[4] = a[6] = a[9] = a[11] = 30; a[2] = 28; // 处理小月
    if(y % 4 == 0 && y % 100 != 0) a[2] = 29; 
    if(y % 400 == 0) a[2] = 29;
    // 处理闰年 2 月（四年一闰，百年不闰，四百年再闰）
    cout << a[m]; // 输出
    return 0;
}
```

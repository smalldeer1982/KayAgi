## [P5721 【深基4.例6】数字直角三角形](https://www.luogu.com.cn/problem/P5721)

### 思路

首先可以知道要输出的是一个第一行为 $n$ 个数，第二行输出 $n-1$ 个数，第三行输出 $n-2$ 个数，最后一行输出 $1$ 个数。一共有 $n$ 行，所以我们可以用两层循环，第一层是遍历第几行的，因为第一行有 $n$ 个数，所以从 $n$ 开始遍历。第二层则是每行有几个数。还要判断前导 $0$ 哦。

### Code


```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin>>n;
    int c=1;
    for (int i=n;i>=1;i--){
        for (int j=1;j<=i;j++){
            if (c<10){
                cout<<"0"<<c;
            }
            else{
                cout<<c;
            }
            c++;
        }
        cout<<endl;
    }
    return 0;
}

```

不难发现，这个数字三角形的第 $1$ 行有 $n$ 个数，第 $2$ 行有 $n-1$ 个数，以此类推，第 $n$ 行就只有 $1$ 个数。

用我们的等差数列求和公式就可以算出总共有 $\dfrac{n(n+1)}{2}$ 个数。先用一个 ```for``` 循环从 $1$ 遍历到 $\dfrac{n(n+1)}{2}$。但是题目要求要输出两位数，所以我们要特判一下，如果当前数字只有一位数，先输出一个 $0$ 在输出当前数字，否则直接输出当前数字。再开两个变量，一个叫 $b$，表示当前这一行的数字个数，另一个叫 $j$，表示当前行已输出的数字个数。一开始把 $b$ 的初始值设为 $n$，把 $j$ 的初始值设为 $0$，每输出一个数，$j$ 就加一，如果 $j$ 等于 $b$,那么 $b$ 就减一，$j$ 归零。

废话不多，上代码。

```
#include <bits/stdc++.h>
using namespace std;
int n,b;
int j;
int main(){
    cin>>n;
    b=n;
    n=n*(n+1)/2;
    j=0;
    for(int i=1;i<=n;i++){
        if(i<10){//如果当前数字只有一位数，就加上前导0。
            cout<<0<<i;
        }
        else{
            cout<<i;
        }
        j++;
        if(j==b){
            b--;
            j=0;
            cout<<'\n';
        }
    }
    return 0;
}
```

# P5718 【深基4.例6】数字直角三角形

## 分析

第一行要输出 $n$ 个数字，第二行输出 $n - 1$ 个数字……最后一行输出 $1$ 个数字。可以用一个从 $n$ 到 $1$ 的倒序循环来枚举每一行输出的数字个数。在输出 $i$ 个数的一行，可以用一个 $0$ 到 $i - 1$ 的循环输出这 $i$ 个数。

## 代码

### C++

在 C++ 里，可以使用 for 循环嵌套来执行这个过程。外层的 for 循环令 $i$ 从 $n$ 到 $1$ 倒序执行，表示每一行的输出数量；内层的 for 循环令 $j$ 从 $0$ 到 $i - 1$，用来输出 $i$ 个数。可以维护一个变量 $\mathrm{tot}$，表示当前输出的数。

关于输出，使用 `printf("%02d", x)` 可以按两格宽度输出整数，不足两格补 $0$。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int n, tot = 0;
  cin >> n;
  for (int i = n; i; --i) {
    for (int j = 0; j < i; ++j) {
      printf("%02d", ++tot);
    }
    printf("\n");
  }
}
```

### Rust

在 Rust 里，进行从 $n$ 到 $1$ 的倒序 for 循环的方法是 `for i in (1..=n).reverse() { Block }`。内层的 for 循环可以同前顺序枚举的过程。

可以写出代码如下：

```rust
/*

fn main() {
  let mut sc = Scanner::new();
  let n = sc.next_int();
  let mut tot = 0;
  for i in (1..=n).rev() {
    for _ in 0..i {
      tot -=- 1;
      print!("{:02}", tot);
    }
    print!("\n");
  }
```

### TypeScript

在 ts 里，类似的 for 循环方法与 cpp 一致。可以用 number 的 `toString()` 方法把数字转换成字符串，再用 `padStart()` 方法给字符串开头补 $0$。把一行内要输出的内容放在 `curLine` 里，然后统一用 `console.log` 输出。

```typescript
import { Reader } from './Reader';
// https://www.luogu.com.cn/paste/r7vtnbaz

const data = fs.readFileSync('/dev/stdin')
const lines = data.toString('ascii').trim().split('\n');
const cin = new Reader(lines);

const n = cin.nextInt();
let tot = 0;
for (let i = n; i; --i) {
  let curLine : string = '';
  for (let j = 0; j < i; ++j) {
    curLine += (++tot).toString().padStart(2, '0')
  }
  console.log(curLine);
}
process.exit(0);
```

```cpp
		for(int j=1;j<=n-i+1;j++){
			if(cnt>=1&&cnt<=9) cout<<"0"<<cnt;
			else cout<<cnt;
			cnt++;
		}
		cout<<"\n";
	}
	return 0;
}
```

## 1. 题目大意
- 输出直角边长度是 $n$ 的数字直角三角形；

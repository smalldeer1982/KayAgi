```cpp
#include<iostream>
```

### 所以：接着改！
这时，一个蒟蒻想到：为什么不用一个**快速幂+记忆化**呢？

**于是法三就出来了：**
```cpp
#include<iostream>
using namespace std;
int n,a[101],s;
void sum(int x)
{
    if(x>n)
    {
        return;
    }
    if(x%2==0)
    {
        a[x]+=2*a[x/2];
    }
    else
    {
        a[x]+=2*a[x/2]+1;
    }
    s+=a[x];
    sum(x+1);
}
int main()
{
    cin>>n;
    sum(1);
    cout<<s;
    return 0;
}
```
15ms,788kb,感觉还不错

**比较一下：**
- 法一：15ms,820kb
- 法二：14ms,1.10mb
- 法三：15ms,788kb

 **根据比较，法三在综合方面还不错，暂时如果时间紧迫，使用法一也是推荐的**

```cpp
#include<iostream>
```

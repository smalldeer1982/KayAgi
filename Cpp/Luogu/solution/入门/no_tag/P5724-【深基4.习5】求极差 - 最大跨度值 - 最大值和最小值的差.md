# 【深基4.习5】求极差 / 最大跨度值 / 最大值和最小值的差

## 题目描述

给出 $n$ 和 $n$ 个整数 $a_i$，求这 $n$ 个整数中的极差是什么。极差的意思是一组数中的最大值减去最小值的差。

## 说明/提示

数据保证，$1 \leq n\leq 100$，$0\le a_i \le 1000$。

## 样例 #1

### 输入

```
6
4 1 5 1 4 1```

### 输出

```
4```

# 题解

## 作者：Skies (赞：104)

我的思路是将存在a数组里，然后sort一边，用a[n] 即最大- a[1] 即最小即可食用
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[100001];
int main()
{
	int n;
	cin>>n;
    for(int i=1;i<=n;i++)
    {
    	scanf("%d",&a[i]);
	}
	sort(a+1,a+n+1);
	cout<<a[n]-a[1];
    return 0;
}
```

## 作者：警策看取 (赞：62)

仍然不需要数组。

定义两个变量，一个更新最大值，一个更新最小值，最后算出最大值与最小值的差即可。

记得把最大值设置成小的数，最小值设置成大的数。

代码：
```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
	int n,maxn=-1,minn=1001;
	cin>>n;
	for(int i=1;i<=n;++i){
		int tmp;
		cin>>tmp;
		maxn=max(tmp,maxn);//算最大值
		minn=min(tmp,minn);//算最小值
	}
	cout<<maxn-minn;//求差
	return 0;
}
```

## 作者：Sea_Level (赞：52)

这题十足是一道大水题，题意简化为：

> 给出一串数列，求最大值与最小值的差。

上代码：

```cpp
#include<iostream>
using namespace std;
int n,a,zx=10000,zd=0;//不用开数组，只需要一个变量就可以了。a就是上述的那个变量。zx是是最小值因为a[i]最大只能是1000，所以10000一定比1000大，zd是最大值。
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a;//
		if(a<zx){
			zx=a;
		}
		if(a>zd){
			zd=a;
		}
	}
	cout<<zd-zx<<endl;//输出差
	return 0;//拜拜！
}
```

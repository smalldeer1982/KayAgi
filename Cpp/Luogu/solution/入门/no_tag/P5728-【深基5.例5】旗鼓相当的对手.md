# 【深基5.例5】旗鼓相当的对手

## 题目描述

现有 $N$ 名同学参加了期末考试，并且获得了每名同学的信息：语文、数学、英语成绩（均为不超过 $150$ 的自然数）。如果某对学生 $\lang i,j\rang$ 的每一科成绩的分差都不大于 $5$，且总分分差不大于 $10$，那么这对学生就是“旗鼓相当的对手”。现在想知道这些同学中，有几对“旗鼓相当的对手”？同样一个人可能会和其他好几名同学结对。

## 说明/提示

数据保证，$2 \le N\le 1000$ 且每科成绩为不超过 $150$ 的自然数。

## 样例 #1

### 输入

```
3
90 90 90
85 95 90
80 100 91```

### 输出

```
2```

## 题意

现在给出每组有 $3$ 个数的长度为 $n$ 的序列，要求出其中有几对每个数的差不超过 $5$ 且总差不超过 $10$ 的组。

## 思路

这就是一道纯模拟题，我们可以设立 $4$ 个数组，分别表示语文、数学、英语和总分。

对这 $4$ 个依次进行判断，直接加到答案里面。

时间复杂度为 $O(n^2)$。

在本题的数据范围内（$2\le n\le1000$），这种时间复杂度是完全可行的。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int yw[1005],sx[1005],yy[1005],zf[1005];//拼音万岁！
int ans=0;
int main(){
	int n;
	cin>>n;
	for(int i=1;i<=n;++i){
		cin>>yw[i]>>sx[i]>>yy[i];
		zf[i]=yw[i]+sx[i]+yy[i]; 
	}
	for(int i=1;i<=n;i++){
		for(int j=i+1;j<=n;j++){
			if((abs(yw[i]-yw[j])<=5)&&(abs(sx[i]-sx[j])<=5)&&(abs(yy[i]-yy[j])<=5)&&(abs(zf[i]-zf[j])<=10)){//就是题目里面的意思。
                ans++;
            }
		}
	}
	cout<<ans;
	return 0;
}
```

# 题目简述
### 题意
给定 $N$ 个学生及其三科的成绩，求有多少对学生满足：
- 对学生 $⟨i,j⟩$ 的每一科成绩的分差都不大于 5，且总分分差不大于 10。
### 思路
要求每一对学生是否符合要求，就需要暴力枚举每一种可能（用结构体储存），并做出判断，如果符合要求，令 $cnt \gets cnt+1$，最后输出 $cnt$ 即可。

我们是暴力枚举每一种可能，所以时间复杂度为 $O(N^2)$，$N \le 1000$，所以不可能超时。
## [AC 代码](https://www.luogu.com.cn/record/203459974)
```cpp
#include <bits/stdc++.h>
using namespace std;
int cnt = 0, k = 0;

struct node {
	int a, b, c, ans; //结构体储存三科成绩和总分
};

int main() {
	int t;
	cin >> t;
	node v[t];
	for (int i = 0; i < t; i++) { //读入学生的数据
		cin >> v[i].a >> v[i].b >> v[i].c;
		v[i].ans = v[i].a + v[i].b + v[i].c;
	}
	for (int i = k++; i < t; i++) {
		int x = v[i].a, y = v[i].b, z = v[i].c, sum = v[i].ans;
		for (int j = i + 1; j < t; j++)
			if (abs(v[j].a - x) < 6 && abs(v[j].b - y) < 6 && abs(v[j].c - z) < 6 && abs(v[j].ans - sum) <= 10) //如果符合要求
				cnt++; //视为是一对旗鼓相当的对手，令cnt++
	}
	cout << cnt; //输出有多少对学生满足要求
	return 0;
}
```

## 题意

输入一些同学的语数英成绩，判断其中有几对单科分差不大于 $5$，总分差不大于 $10$ 的对手。

## 思路

数据量很小，因此只需要枚举即可。

1. 输入每个同学的各科成绩，同时计算总分。
2. 循环枚举判断即可。外层循环枚举一位同学，内层循环再枚举一位。注意不要重复枚举。

## 代码

```cpp
#include<iostream>
using namespace std;
int grade[1010][5],sum,n;
//特别古老的代码了（x
int main(){
    cin>>n;
    for(int i=1;i<=n;i++){
        for(int j=1;j<=3;j++){
            cin>>grade[i][j];
            grade[i][4]+=grade[i][j];
        }
    }
    for(int i=1;i<=n;i++){
        for(int j=i+1;j<=n;j++){
            if(abs(grade[i][1]-grade[j][1])<=5)
            if(abs(grade[i][2]-grade[j][2])<=5)
            if(abs(grade[i][3]-grade[j][3])<=5)
            if(abs(grade[i][4]-grade[j][4])<=10)
                sum++;
        }
    }
    cout<<sum;
    return 0;
}
```

注意到 $n\le1000$，可以直接枚举 $i,j$ 判断他们是否满足题目要求。

输入一名同学的信息时，我们用结构体存下该同学的三科成绩以及总分，然后枚举已经输入了的同学判断其能否与之配对为“旗鼓相当的对手”。

## 题目介绍

给定所有人的语数英三科成绩，求旗鼓相当的人的对数，即语数英这对人分差 $\le5$ 且总分差 $\le10$，且数据范围不大，只需要枚举并且判断即可。\
注意，不要与自己进行比较，会错，也不要重复比较。

```cpp
#include<bits/stdc++.h>
using namespace std;
const int N=1e3+5;
int n,ans=0;
struct node {
	int a,b,c,num;
} aa[N];//结构体 
int main() {
	cin>>n;
	for(int i=1; i<=n; i++) {
		cin>>aa[i].a>>aa[i].b>>aa[i].c;//输入 
		aa[i].num=aa[i].a+aa[i].b+aa[i].c;//提前求总分 
	}
	for(int i=1; i<=n; i++) {
		for(int j=i+1; j<=n; j++) {//保证不重复 
			if(abs(aa[i].a-aa[j].a)<=5&&abs(aa[i].b-aa[j].b)<=5&&abs(aa[i].c-aa[j].c)<=5&&abs(aa[i].num-aa[j].num)<=10) {//判断条件 
				ans++;//对数加一 
			}
		}
	}
	cout<<ans<<endl;//输出答案 
	return 0;
}
```

# 题目大意
### 题意
给出每个学生各科成绩，求有多少对学生 $\lang i,j\rang$ 满足每一科成绩的分差都不大于 $5$，且总分分差不大于 $10$。
### 思路
暴力枚举每一对学生 $\lang i,j\rang$ 每科成绩是否满足条件，如果满足，让 $ans \gets ans+1$。
## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, ans = 0, a[1010][4];//定义

int main() {
	cin >> n;//输入
	for (int i = 0; i < n; i++) {
		cin >> a[i][0];
		cin >> a[i][1];
		cin >> a[i][2];
		//输入三科成绩
		a[i][3] = a[i][0] + a[i][1] + a[i][2];//总分
	}

	for (int i = 0; i < n; i++) {//枚举
		for (int j = i + 1; j < n; j++) {
			if (abs (a[i][0] - a[j][0]) <= 5 &&
			        abs (a[i][1] - a[j][1]) <= 5 &&
			        abs (a[i][2] - a[j][2]) <= 5 &&
			        abs (a[i][3] - a[j][3] ) <= 10)
				ans = ans + 1;
		}
	}
	cout << ans;//输出
	return 0;
}

```

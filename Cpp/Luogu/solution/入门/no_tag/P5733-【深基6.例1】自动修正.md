在这里我介绍一种**不用循环**(即$for()$和$while()$)
的方法。

首先，我给大家科普一个知识，在$C++$中，$main()$函数是可以递归的。

那我们就可以判断当前是否输入了一个字符，并检测输入字符是否为小写字母，
如果是，就将其转换成大写字母即可。转换方法就是利用$ASCII$码，将小写字母$+'A'-'a'$即可。

### Code ：

```cpp
#include<bits/stdc++.h>
using namespace std;
char ch;
int main(){
	if(cin>>ch){//判断是否输入 
		if(ch>='a'&&ch<='z')ch+='A'-'a';//转换大小写 
		cout<<ch;//输出 
		main();//递归调用 
	} 
	return 0;//如果没有输入就退出 
}

```

## 作者：2011hym (赞：15)

作为一个基础题，思路非常简单。

## 题目分析
我们需要将一个字符串中的所有小写字母转换为大写字母，其他字符保持不变。

[更好的阅读体验。](https://blog.csdn.net/Hymmeishili/article/details/148385510?sharetype=blogdetail&sharerId=148385510&sharerefer=PC&sharesource=Hymmeishili&spm=1011.2480.3001.8118)

## 解决思路

我们可以利用 ASCLL 码的性质，将每个小写字母加上 `'A'-'a'`，就可以转换成大写字母。

对于每个字符，如果是小写字母（$a-z$），则转换为对应的大写字母（$A-Z$）。

而且，常数级别的复杂度保证不会超时。

## 代码实现
```cpp
#include <iostream>
using namespace std;
char c;
int main() {
    while (cin >> c) {//依次读入。
        if (c >= 'a' && c <= 'z') {
            c += 'A' - 'a';
        }
        cout << c;
    }
    return 0;
}
```

# 思路

输入一行字符串，枚举每一位。设当前的字符为 $S_i$，由于要求将小写字母转换为大写字母，只需要判断 $S_i$ 是否在 `a` 到 `z` 这个区间即可。


```cpp
if (a[i] >= 'a' && a[i] <= 'z') cout << char(a[i] - 32);
```

输出可以看作 `cout<<char(a[i]-'a'+'A')`，学过 ASCII 码表的都知道，`A` 在 `a` 的前面，用当前的小写字母减去 `a`，就能知道这个字母是第几个，就可以转换为大写字母。

其余直接输出即可。

# 代码


```cpp
#include<iostream>
#include<string>
using namespace std;
int main()
{
	string a;
	cin >> a;
	for (int i = 0; i < a.size(); i++) {
		if (a[i] >= 'a' && a[i] <= 'z') cout << char(a[i] - 32);
		else cout << a[i];
	}
	return 0;
}
```

# 题目简述
### 题意
给定一个字符串，把所有的小写字母全部转化为大写字母，输出转化后的字符串。
### 思路
遍历字符串，如果是小写字母，输出转为大写后的字母，否则输出这个字符。
### STL 函数
函数 `islower(x)`。
```cpp
bool islower(char ch) {
    if (ch >= 'a' && ch <= 'z')
      return true;
    return false;
}
```
作用：如果 $x$ 是小写字母返回 `true` 值，否则返回 `false`。

---
函数 `toupper(x)`。
```cpp
int toupper(x) {
    return x - ('a' - 'A');
}
```
作用：返回 $x$ 转化为大写字母后的字符。

## [AC 代码](https://www.luogu.com.cn/record/202122887)
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	string s;
    cin >> s;
	for (int i = 0; i < s.size(); i++) //遍历字符串
		if (islower(s[i])) //如果是小写字母
            cout << toupper(s[i]); //转化为大写字母并输出
      else
            cout << s[i]; //否则正常输出
	return 0;
}
```

双倍经验：[大写](https://www.luogu.com.cn/problem/P8761)。可以看看[这篇题解](https://www.luogu.com.cn/article/fo2rvqhd)。

这里介绍 c++ 中的 `toupper(int c)` 函数。该函数的功能为：
- 若 $c$ 在 ASCII 码中对应字符为小写字母，则返回对应大写字母的 ASCII 码。
- 否则直接返回 $c$。

于是这题中可以直接在不判断当前字符是否为小写字母的情况下用 `toupper` 函数直接将字小写字母转为大写。

```cpp
#include<bits/stdc++.h>
using namespace std;

char a[102];

int main(){
	cin>>a;
	for(int i=0; i<strlen(a); i++) a[i]=toupper(a[i]);//转换
	cout<<a;
	return 0;
}
```

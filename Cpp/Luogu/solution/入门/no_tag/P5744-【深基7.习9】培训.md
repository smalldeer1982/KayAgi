# 【深基7.习9】培训

## 题目描述

某培训机构的学员有如下信息：

- 姓名（字符串）
- 年龄（周岁，整数）
- 去年 NOIP 成绩（整数，且保证是 $5$ 的倍数）

经过为期一年的培训，所有同学的成绩都有所提高，提升了 $20\%$（当然 NOIP 满分是 $600$ 分，不能超过这个得分）。

输入学员信息，请设计一个结构体储存这些学生信息，并设计一个函数模拟培训过程，其参数是这样的结构体类型，返回同样的结构体类型，并输出学员信息。

## 说明/提示

数据保证，$1 \leq n \leq 5$。年龄为 $0 \sim 100$（含 $0$ 与 $100$）的整数。成绩为 $0 \sim 600$（含 $0$ 与 $600$）的 $5$ 的整倍数。

## 样例 #1

### 输入

```
3
kkksc03 24 0
chen_zhe 14 400
nzhtl1477 18 590```

### 输出

```
kkksc03 25 0
chen_zhe 15 480
nzhtl1477 19 600```

# 题解
### 题目大意
给定 $N$ 个人的名字、年龄、去年 NOIP 分数，设他们每个人今年的分数都会比去年提高 $20\%$，求他们今年的 NOIP 成绩。
### 思路
直接模拟，名字不变，年龄需要 $+1$，分数最高上限为 $600$，所以我们直接输出 $\min(600, \lfloor score \times 1.2 \rfloor)$ 就行了。

注意：`score` 必须为 `double` 类型，输出的时候直接输出 `min(600, int(score * 1.2))` 就是最终答案。

## [AC 代码](https://www.luogu.com.cn/record/221902615)
```cpp
#include <bits/stdc++.h>
using namespace std;
#define in cin
#define out cout

signed main() {
	int t;
	in >> t;
	while (t--) {
		string s; //名字
		int year; //年龄
		double score; //分数
		in >> s >> year >> score;
		out << s << ' ' << ++year << ' ' << min(600, int(score * 1.2)) << '\n'; //如上所述
	}
	return 0;
}
```

## 题目分析

本题要求根据提供的学员信息，计算出一年后的 NOIP 成绩，按题意模拟即可。

### 实现细节

创建一个结构体存放每一个学生的信息，包括姓名，年龄和 NOIP 得分。一年后，学生姓名不变，年龄增长 $1$ 岁，NOIP 得分提升 $20 \%$。

**注意**：得分不能超过 $600$ 分。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;
struct student
{
    string name;
    int age;
    int score;
};
student s[100005];
int n;
int main()
{
    cin >> n;
    for (int i = 1; i <= n; i++)
    {
        cin >> s[i].name >> s[i].age >> s[i].score;
        s[i].age++;
        s[i].score *= 1.2;
        if (s[i].score > 600)   // 得分不能超过 600 分
            s[i].score = 600;
    }
    for (int i = 1; i <= n; i++)
        cout << s[i].name << ' ' << s[i].age << ' ' << s[i].score << endl;
    return 0;
}
```

## 知识铺垫

用户可以优先阅读 [结构体](https://oi-wiki.org/lang/struct/) 相关知识点。

结构体可以帮助我们更好的对同种目的的数据进行合并表示。

例如：如果我们需要对成员的成绩进行分析，可以定义一个名为 `student` 的结构体。

使用结构体，应当先定义结构体类型，它的基本语法格式如下：

```cpp
struct 结构体名称
{
    成员类型1 成员名称1;
    成员类型2 成员名称2;
    ...
    成员类型n 成员名称n;
};
```

`struct` 是**定义结构体的关键字**，结构体名称用于**标识这个新的数据类型**，花括号内部包含了**若干成员变量的声明**。**每个成员变量都有自己的类型和名称**，它们之间用分号分隔。

下面展示本题对于成员的结构体命名：

```cpp
struct students
{
    string name; // 表示 string 类型的 name，题目中存储学生姓名
    int age; // 表示 int 类型的 age，题目中存储学生年龄
    int score; // 表示 int 类型的 score，题目中存储学生成绩
}s[MAXN];
```

结构体是一种类型，而不是变量，不能赋值；定义了结构体类型后，还需**要声明结构体变量才能使用**。

声明结构体变量有两种方式：先定义类型再声明变量，或者在定义类型的同时声明变量。

```cpp
// 方式 1：先定义类型，再声明变量
struct students s[MAXN];  //使用了已经定义的结构体类型 student
// 方式 2：定义类型的同时声明变量
struct students
{
    string name; // 表示 string 类型的 name，题目中存储学生姓名
    int age; // 表示 int 类型的 age，题目中存储学生年龄
    int score; // 表示 int 类型的 score，题目中存储学生成绩
}s[MAXN];
```

## 题目分析

**题目中要求我们使用结构体进行存储**。所以可以设计一个名为 `students` 的结构体，同时分别命名了姓名，年龄和分数。

根据题目要求，将年龄的数据加 $1$，去年 NOIP 成绩乘以 $120\%$ 即可。

需要注意的是，我们应当对乘以 $120\%$ 后的成绩进行特殊判断。若成绩大于 $600$，应当将数据强行赋值为 $600$。

#include <bits/stdc++.h>

using namespace std;

const int MAXN = 10;

struct students
{
    string name;
    int age;
    int score;
}s[MAXN];

int n;

int main()
{
    // freopen("code.in", "r", stdin);
    scanf("%d", &n);
    for (int i = 1; i <= n; i++)
    {

    }
    for (int i = 1; i <= n; i++)
    {
        s[i].age += 1;
        s[i].score *= 1.2;
        if (s[i].score >= 600)
        {
            s[i].score = 600;
        }
    }
    for (int i = 1; i <= n; i++)
    {
        cout << s[i].name << " " << s[i].age << " " << s[i].score << endl;
    }
    return 0;
}

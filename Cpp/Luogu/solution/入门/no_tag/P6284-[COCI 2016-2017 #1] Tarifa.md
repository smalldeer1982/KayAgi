# [COCI 2016/2017 #1] Tarifa

## 题目描述

为了能够在网上冲浪，Pero 购买了一个数据套餐：

- 每月他将获得 $x \rm{MB}$ 流量。
- 每月未用完的流量，可以在次月继续使用。

现在，我们知道了 Pero 在使用这个套餐的前 $n$ 个月中，每月使用了多少 $\rm{MB}$ 流量。请你帮忙计算，第 $n+1$ 月他能使用多少 $\rm{MB}$ 流量。

## 说明/提示

#### 样例 1 解释

第一月，Pero 能使用 $10\rm{MB}$ 流量，实际使用 $4\rm{MB}$ 流量，剩余 $6\rm{MB}$ 流量。 

第二月，Pero 能使用 $6+10=16(\rm{MB})$ 流量，实际使用 $6\rm{MB}$ 流量，剩余 $10\rm{MB}$ 流量。 

第三月，Pero 能使用 $10+10=20(\rm{MB})$ 流量，实际使用 $2\rm{MB}$ 流量，剩余 $18\rm{MB}$ 流量。
 
第四月，Pero 能使用 $18+10=28(\rm{MB)}$ 流量。

---

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1\le x\le 100$，$1\le n\le 100$，$0\le p_i\le 10^4$。

------------

### 本题自我感觉难度入门，适合刚学习编程的同学们写。

我们先来分析一下题面：

Pero买了一个流量套餐。

这个套餐每个月会给他 x MB的流量，如果没有用完下个月还可以接着用。

给你前 n 个月他每月使用了多少 MB 流量。

请你帮忙计算，第 n+1 月他还能使用多少 MB 流量。

----

【我的思路】

这一题我的思路很简单。

就是把前 n 个月可以使用的最多流量计算出来，然后减去前 n 个月用过的，再加上 x 就可以了。其实这很暴力，但这也应该是最简单的。

----

有了思路和题目的分析，我们就可以写代码了。

下面是我的AC代码：

```cpp
#include<iostream>//输入输出的头文件。
using namespace std;
int main()
{
	int x, n, a[10001], s = 0;
	cin >> x >> n;
	s = x * n;//计算前 n 个月的流量最多用的总量。
	for (int i = 1; i <= n; i++)
	{
		cin >> a[i];
		s -= a[i];//边输入边减。
	}
	cout << s + x;//最后加上 x ，并且输出。
	return 0;
}
```

这题没人用 PY？

其实这题就是为 PY 而生的（至少我是这么认为的）。

为什么呢？这道题满足以下两个特点：

- 不卡常（PY 比较慢）；
- 输入都是一个数一行（不用打那个长长的`map`函数）。

而 PY 的代码量小，可以秒 A 这道题。

我们来看思路。

要求第$n+1$月可用的流量，我们可以用

$n+1$月总共得到的流量$-$前$n$个月总共用去的流量

这个公式进行计算。

第二部分非常简单，直接边输入边减即可；

第一部分相当于每月的流量$x \times$总共$n+1$个月。

那么代码就很好得到了。先来看代码。

```python
x=int(input()) #input函数自动输入一行一个数
n=int(input())
ans=(n+1)*x #先算总流量
for i in range(n): #n次执行
    m=int(input()) #输入每一个月的流量
    ans-=m #从总流量里减去
print(ans) #输出答案
```

***
\[思路\]：问Pero第n+1个月能用多少流量，可以把Pero的n+1个月可用流量的总和算出来，再用循环把前面月份用的流量减掉，就是第n+1个月剩余的流量了  
***  
我的代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int x,n;//x是每月可使用的流量，n是n个月中每月用的流量 
	int a[105];//a数组用来存储n个月中每月用的流量，例如a[0]就是第一个月中用的流量 
	cin>>x>>n;
	int sum=x*(n+1);//这里的sum是到n+1个月的可用流量总和 
	for (int i=0;i<n;i++) cin>>a[i];//输入n行数据，注意这个程序中是从a[0]开始存储，存到a[n-1] 
	for (int i=0;i<n;i++) sum-=a[i];//按照刚才的顺序，循环n次，每次都用sum减去这个月用掉的流量(sum-=a[i]与sum=sum-a[i]是同一个意思），最后剩下的就是这个月可以用的流量了 
	cout<<sum;//最后输出这个月可用的流量 
	return 0;
} 

```

## 作者：毕瑞成 (赞：2)

[题目链接](https://www.luogu.com.cn/problem/P6284)

### 前言
题目比较水，按照题目模拟就行，刚入门的可以练练手。

### 做法分析：
通过求出前n月剩余的流量数，然后加上第$n+1$月获得的流量数后输出。


#### 定义：

数组$p$:每月的花费

数组$res$:当前月后剩余的流量数

变量$x$:每月可获得的MB数量

变量$ans$:最后的答案


**备注：**

$res_{0}=0$,$ans=res_{n}+x$

代码中不设变量$ans$,直接输出。

#### 做法一：递归
通过题目，可以发现：

第$ans$=$res_{n}+x$

而$res_{n}$=$x+res_{n-1}-p_{n}$

同理$res_{n-1}$=$x+res_{n-2}-p_{n-1}$

$\cdots$

$res_{2}$=$x+res_{1}-p_{2}$

$res_{1}$=$x+res_{0}-p_{1}$

$res_{0}$=$0$


[附新手参考代码](https://www.luogu.com.cn/paste/so1mavko)

[简洁代码](https://www.luogu.com.cn/paste/1alapgq1)


#### 做法二：递推
通过题目，可以发现：

对于第$1$月有:$res_{1}$=$x-p_{1}$

对于第$2$月有:$res_{2}$=$x+res_{1}-p_{2}$

对于第$2$月有:$res_{3}$=$x+res_{2}-p_{3}$

$\cdots$

对于第$i$月有:$res_{i}$=$x+res_{i-1}-p_{i}$(i>=2)

同理,对于第$n$月有:$res_{n}$=$x+res_{n-1}-p_{n}$

则$ans$=$res_{n}+x$


[附新手参考代码](https://www.luogu.com.cn/paste/vgm2i1dh)

[附"滚动"后代码](https://www.luogu.com.cn/paste/8hrem9fi)

#### 做法三：结论法（我也不知道叫什么方法）
对于前两个方法，进行进一步的化简，得到结论

#### 1.化简递归：
$res_{n}$=$x+res_{n-1}-p_{n}$

$res_{n-1}$=$x+res_{n-2}-p_{n-1}$

$\cdots$

得到:

$res_{n}$=$x+x+res_{n-2}-p_{n-1}-p_{n}$

$res_{n}$=$x+x+x+res_{n-3}-p_{n-2}-p_{n-1}-p_{n}$

$res_{n}$=$x+x+x+x+res_{n-4}-p_{n-3}-p_{n-2}-p_{n-1}-p_{n}$

$\cdots$

$res_{n}$=$x+x+…+x-p_{1}-p_{2}-…p_{n-2}-p_{n-1}-p_{n}$

$res_{n}$=$x·n\sum_{i=1}^{n}{p_{i}}$

#### 2.化简递推

对于题意，可得（与上递推处相同）：

对于第$1$月有:$res_{1}$=$x-p_{1}$

对于第$2$月有:$res_{2}$=$x+res_{1}-p_{2}$

对于第$2$月有:$res_{3}$=$x+res_{2}-p_{3}$

$\cdots$

对于第$i$月有:$res_{i}$=$x+res_{i-1}-p_{i}$(i>=2)

同理,对于第$n$月有:$res_{n}$=$x+res_{n-1}-p_{n}$


**将等式两边相加可得：**

$res_{1}+res_{2}+…+res_{n}$=$x+x+…+x+res_{0}+res_{1}+\cdots+res_{n-1}-p_{1}-p_{2}-…-p_{n}$

$res_{n}$=$x·n+res_{0}-\sum_{i=1}^{n}{p_{i}}$

$res_{n}$=$x·n-\sum_{i=1}^{n}{p_{i}}$

**结论：综合以上两种方法，都可以得到如下式子**

$res_{n}$=$x·n-\sum_{i=1}^{n}{p_{i}}$

可得$ans$

=$res_{n}+x$

=$x·n-\sum_{i=1}^{n}{p_{i}}+x$

=$x·(n+1)-\sum_{i=1}^{n}{p_{i}}$

[代码~](https://www.luogu.com.cn/paste/gcn85x16)

## 作者：_tommysun_ (赞：0)

个人感觉题目中流量的概念比较抽象，可以将其理解为工资：Pero每月的工资是$x$元，每月都要花费$p_i$元。如果一个月的前花不完，会放进保险柜中。问前n个月他有多少钱？

**思路：**

读入$x$和$n$,再输入$p_i$。定义一个变量ans,来模拟上文中所说的“保险柜”。存入保险柜的钱便是这个月的工资减去花费的钱（也有可能透支，见下面的代码）。最后再加上第$n+1$月拿到的工资，输出即可。

**代码：**
```cpp
#include<bits/stdc++.h>
using  namespace std;
int main(){
	int x,n;
	cin>>x>>n;
	int a[n+10];
	int ans=0; //上文中的保险箱 
	for(int i=0;i<n;i++){
		cin>>a[i];
		ans+=x-a[i]; //将钱放入保险箱内。如果透支了就往里面拿钱补贴。 
	}
	cout<<ans+x; //还有第n+1个月的工资 
	return 0;	
}
```

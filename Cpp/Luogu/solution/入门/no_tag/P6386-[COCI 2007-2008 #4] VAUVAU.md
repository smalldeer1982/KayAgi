# [COCI 2007/2008 #4] VAUVAU

## 题目描述

在一个小村子里，邮递员、送奶工、垃圾清理工每天早晨都面临着同样的难题：18 号房子的门前有两条看门狗。他们所不知道的是，这两条狗的表现是有迹可循的。

当一天开始时，其中一条狗会先暴躁 $a$ 分钟，然后安静 $b$ 分钟，而另一条狗则会先暴躁 $c$ 分钟，然后安静 $d$ 分钟。这两条狗在一天中会无限地重复上述行为。

给定这三个人到达 18 号房子的时刻，请求出该时刻有几条狗是处于暴躁状态的。

## 说明/提示

#### 数据规模与约定

对于全部的测试点，保证 $1 \leq a, b, c, d, p, m, g \lt 10^3$。

## 作者：Scrutiny (赞：36)

这是一道%你题。

对于一个时间点$t$，以及给定的$a,b,c,d$,

当$t$模$a+b$的余数$>0$且$\leq a$时（此时已经过了整数个周期），第一只狗是暴躁的，否则它不暴躁。

第二只狗同理，因此最后算一下一共几只狗暴躁。

最后将上面的思路转换为代码后打包在函数里，然后调用三次即可。


### CODE:



```cpp

#include<bits/stdc++.h>
using namespace std;
int a,b,c,d,p[10];
string check(int x){
	if(0<x%(a+b)&&x%(a+b)<=a&&0<x%(c+d)&&x%(c+d)<=c){
		return "both";
	}
	if((x%(a+b)>a||x%(a+b)==0)&&(x%(c+d)>c||x%(c+d)==0)){
		return "none";
	}
	else{
		return "one";
	}
}
int main(){
	cin>>a>>b>>c>>d>>p[1]>>p[2]>>p[3];
	for(int i=1;i<=3;++i){
		cout<<check(p[i])<<endl;
	}
	return 0;
} 

```

# 1 Analysis

说实在的，并不需要标记每个时刻的暴躁和安静时间，可以算出来。

由于**在一天中会无限地重复**，就可以把一次暴躁和安静当做一个循环，两只狗完成这一个循环需要的时间分别是 $a + b$ 和 $c + d$。

那么，怎么计算在任意时刻这两只狗是在暴躁状态还是安静状态呢？

**先计算这个时刻是循环的第几分钟，再判断**即可。可以通过取模计算。

举例：如果某人到达的时间为 $10$，规定 $a, b, c, d$ 分别为 $1, 2, 2, 3$。可得，这个人到达的时间正好在第一只狗的第四个循环的第一分钟（正好在暴躁时间），在第二只狗的第二个循环正好结束（安静时间）。

# 2 Code

```cpp
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using namespace std;

int cnt;								// 计数器。

int a, b, c, d;
int p, m, g;

void calc(int);
void check();

int main() {
    scanf("%d%d%d%d", &a, &b, &c, &d);
    scanf("%d%d%d", &p, &m, &g);
    calc(p);							// 计算和输出。
    check();
    cnt = 0;
    calc(m);
    check();
    cnt = 0;
    calc(g);
    check();
    return 0;
}

void calc(int x) {
	// 判断和计算有几只狗在暴躁状态。
    if (x % (a + b) <= a && x % (a + b) != 0) {
        cnt++;
    }
    if (x % (c + d) <= c && x % (c + d) != 0) {
        cnt++;
    }
}

void check() {
    switch (cnt) {
    case 0:
        puts("none");
        break;
    case 1:
        puts("one");
        break;
    case 2:
        puts("both");
        break;
    }
}
```

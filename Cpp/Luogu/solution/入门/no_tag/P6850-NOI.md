## 题目背景

## 题目描述

由于小 L 不会算数，现在小 L 告诉了你他每题的分数和队线，请你告诉他是否进队。

NOI 分数的计算方式是：

1. 笔试基础有 $50$ 分，每通过一题加 $1$ 分。
2. 实际得分是每题的分数加上笔试的分数。
3. 如果是 A 类，总分在此基础上再加 $5$ 分。

# 题意及注意点：

- 给出一堆考试分数，要求把这些分数相加求出小 L 的总分，并把它与集训队分数线进行比较，若**大于等于**集训队分数线，则输出 AKIOI，否则输出 AFO。

# 思路：


窝的思路这样的：

1. 先把小 L 分数（下文均用 $s$ 代替）读入，加上基准值 $50$。

2. 依次加上每题的分数 $x$。

3. 接下来，判断小 L 是否为 A 类选手，若他是，则 $s$ 加上 $5$。

4. 将 $s$ 与集训队分数线作比较，由于不需要考虑第 $50$ 名同分的情况，所以如果 $s$ 大于等于集训队分数线，就输出 AKIOI。否则输出 AFO。


------------

# 放代码啦qwq：
```cpp
#include<bits/stdc++.h>
using namespace std;
long long s,x;
int main()
{
	cin>>s;s+=50;//输入笔试分数
    for(int i=1;i<=6;i++)
     {
       cin>>x;//输入 D1D2 每题的分数
       s+=x;//加进总分中
     }
	cin>>x;//输入是否是 A 队的标志
    if(x) s+=5;
	cin>>x;//输入集训队分数线
    if(s>=x) cout<<"AKIOI";//判断小 L 是否达到了集训队分数线
    else cout<<"AFO";
    return 0;
}
```

## 作者：Eason_AC (赞：7)

## Content
小 L 参加了 $\texttt{NOI}$，现在他告诉你九个数 $a,b,c,d,e,f,g,h,i$，分别表示——笔试作对的题数、D1T1、D1T2、D1T3、D2T1、D2T2、D2T3 分别得到的分数、是否是 A 类选手（$1$ 表示是，$0$ 表示不是）和进队线。已知 $\texttt{NOI}$ 的计分方式是：

1. 笔试基础分为 $50$，每做对一题加 $1$ 分。
2. 总分为笔试分数加上上机题目每题的分数。
3. 如果是 A 类选手还有 $5$ 分的加成。

请问小 L 是否能够进队。

**数据范围：$0\leqslant a\leqslant 50,0\leqslant b,c,d,e,f,g\leqslant 100,0\leqslant h\leqslant 1,205\leqslant i\leqslant 705$。**
## Solution
直接按照上面的规则计算分数，很明显可以推出来分数是  $a+b+c+d+e+f+g+50+h\times5$。

这里我来解释一下：

- 为什么要加 $a,b,c,d,e,f,g$ 不需要讲吧。
- 笔试基础分有 $50$ 分，需要加上去。
- 因为有 A 类名额的话，$h=1,h\times5=5$；没有的话，$h=0,h\times 5=0$。所以 $h\times 5$ 就相当于看有没有 A 类名额的加成，没有？那这个 $h\times 5$ 也是白加。

然后再判断这个分数是否 $\geqslant i$ 即可。
## Code
```cpp
#include <cstdio>
using namespace std;

int a[17], sum;

int main() {
	for(int i = 1; i <= 9; ++i)	scanf("%d", &a[i]);
	sum = 50 + a[1] + a[2] + a[3] + a[4] + a[5] + a[6] + a[7] + a[8] * 5;
	if(sum >= a[9])	printf("AKIOI");
	else	printf("AFO");
} 
```

### P6850 NOI题解

[题目传送](https://www.luogu.com.cn/problem/P6850)

#### 思路：

基础分 $50$ 分，笔试通过了 $a$ 题，$6$ 道实际得分分别为 $b,c,d,e,f,g$。$h$ 代表是不是 $A$ 队，若是则加 $5$ 分，$i$ 为进集训队分数线。则总分为 $50+a+b+c+d+e+f+g+h*5$，判断其值与 $i$ 的大小即可。

注意: 所谓基础分，是本来就拥有的 $50$ 分，所以答案应该加上 $50$ 分。

$Code:$

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a,b,c,d,e,f,flag,sum,ans;
inline int read(){
    int x=0,f=1;
    char ch=getchar();
    while(ch<'0'||ch>'9'){
        if(ch=='-')
            f=-1;
        ch=getchar();
    }
    while(ch>='0'&&ch<='9'){
        x=(x<<1)+(x<<3)+(ch^48);
        ch=getchar();
    }
    return x*f;
}
int main(){
    scanf("%d",&n);
    scanf("%d%d%d%d%d%d",&a,&b,&c,&d,&e,&f);
    scanf("%d%d",&flag,&sum);
    ans=ans+n+50+a+b+c+d+e+f+flag*5;
    if(ans>=sum)cout<<"AKIOI\n";
    else cout<<"AFO\n";
    return 0;
}
```

# 正文
首先，第一天的笔试成绩是$a+50$分,因为是50分的基础分加上$a$分做题分。

其次，对于$D1T1,D1T2,D1T3,D2T1,D2T2,D2T3$这六道题来说，只需要把它们的分数总和都相加，得出$b+c+d+e+f+g$。

最后上代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int a, b, c, d, e, f, g, h, i;
int main(){
    cin>>a>>b>>c>>d>>e>>f>>g>>h>>i;
    if(a + b + c + d + e + f + g + 5 * h + 50 >= i) cout<<"AKIOI";
    else cout<<"AFO";
    return 0;
}
```

## 作者：REAL_曼巴 (赞：0)

此题的关键就是在题目的理解，我们要搞清楚如何计算得分，才能判断进没进队。总得分有好几部分，我们一个个来看。

首先，笔试分。因笔试每答对一题给一分，况且又五十分底分，我们仅需把第一个输入的数，也就是答对的题目数与底分加起来，就完成了第一部分的得分，笔试分的计算。注意！一定要加底分！

我们接下来看第二部分，题目得分。题目得分没什么好说的，直接把第一天一二三题的得分与第二天一二三题的得分加起来即可，题目也说得很清楚。

第三部分是 $A$ 类的分，仅需一个判断决定是否加五分。

最后，将几部分分加起来如大于进队的分就输出`AKIOI`，反之亦然。

```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int d1,d2,d3,d4,d5,d6,d7,f,i,ans=0;
int main(){
    cin>>d1>>d2>>d3>>d4>>d5>>d6>>d7>>f>>i;//输入
    ans=d1+d2+d3+d4+d5+d6+d7+50;//计算每部分答案
    if(f==1)//A类
        ans=ans+5;
    if(ans>=i)//判断是否进队
        printf("AKIOI");
    else
        printf("AFO");
    return 0;
}

```

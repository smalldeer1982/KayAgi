# [RC-04] 信息学竞赛

## 题目描述

小 R 今天学习了余角有关的数学知识，请你帮帮他计算一个角的余角吧！

一个角的余角的计算公式如下：

$$\angle B=90\degree -\angle A$$

其中 A 是给出的角，B 是你要计算的角。

## 说明/提示

对于 $100\%$ 的数据，$x$ 在 C++ 语言的 `int` 范围内。即，$x\in[-2^{31},2^{31}-1]$。

## 样例 #1

### 输入

```
100```

### 输出

```
-10```

## 作者：Eason_AC (赞：4)

## Content
给定一个角 $\alpha$，求 $\beta=90^\circ-\alpha$。

**数据范围：$\alpha\in[-2^{31},2^{31}-1]$。**
## Solution
根据公式计算即可，注意，$\alpha$ 在 `int` 范围内并不意味着 $\beta$ 就会在 `int` 范围内（比如 $\alpha=-(2^{31})^\circ,\beta=(2^{31}+90)^\circ$，超出了 `int` 范围）。
## Code
```cpp
int main() {
    long long a;
    scanf("%lld", &a);
    printf("%lld", 90 - a);
    return 0;
}
```

# [COCI 2016/2017 #4] Bridž

## 题目描述

Mirko 发明了一种卡牌游戏。卡牌只包含 $\text{A, K, Q, J}$ 和 $\text{X}$。它们的分值分别为 $4,3,2,1,0$。

给定 $N$ 个包含 $13$ 张卡牌的卡组，求这 $N$ 个卡组中所有卡牌分值的总和。

## 说明/提示

**【样例 1 解释】**

|卡牌类型|卡牌数量|每张卡牌分值|累计分值|
| :----------: | :----------: | :----------: | :----------: |
|$\text{A}$|$4$|$4$|$4 \times 4=16$|
|$\text{K}$|$2$|$3$|$2 \times 3=6$|
|$\text{Q}$|$1$|$2$|$1 \times 2=2$|
|$\text{J}$|$1$|$1$|$1 \times 1=1$|
|$\text{X}$|$5$|$0$|$5 \times 0=0$|

因此分值总和为 $16+6+2+1+0=25$。

**【数据规模与约定】**

对于 $100\%$ 的数据，$1 \le N \le 10^4$。

**【提示与说明】**

**题目译自 [COCI 2016-2017](https://hsin.hr/coci/archive/2016_2017/) [CONTEST #4](https://hsin.hr/coci/archive/2016_2017/contest4_tasks.pdf) _T1 Bridž_。**

**本题分值按 COCI 原题设置，满分 $50$。**

## 作者：ZBAA_MKC (赞：6)

这是一道非常简单的字符串统计题。

#### 大致思路

对于每次输入的字符串 $s$, 挨个判断字符串中的每一个字符是否是 A,K,Q,J 中的一个，如果是就按照题目要求累加分数，反之如果是 X 那就不进行任何操作。

代码如下：

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int ans = 0;
    while (n--)
    {
    	string s;
    	cin >> s;
    	for (int i = 0; i < 13; i++)
    	{
    		if (s[i] == 'A')
			{
				ans += 4;
			} 
			else if (s[i] == 'K')
			{
				ans += 3;
			}
			else if (s[i] == 'Q')
			{
				ans += 2;
			} 
			else if (s[i] == 'J')
			{
				ans += 1;
			}
		}
	}
	cout << ans;
	return 0;
}

```

思路：

这题应该是个入门题。

输入 $n$ 个字符串（长度 $13$ ），每次从头到尾扫描一遍，遇到字母 `A` 答案 $sum$ 就加 $4$ ，遇到字母 `K` 答案 $sum$ 就加 $3$ ，遇到字母 `Q` 答案 $sum$ 就加 $2$ ，遇到字母 `J` 答案 $sum$ 就加 $1$ ，遇到字母 `X` 答案 $sum$ 就要加 $0$ ，但其实这相当于没加，就不必考虑了。

代码：

```
#include<bits/stdc++.h>
using namespace std;
int n,sum=0;
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		string a;
		cin>>a;
		for(int i=0;i<=12;i++){
			if(a[i]=='A') sum+=4;
			if(a[i]=='K') sum+=3;
			if(a[i]=='Q') sum+=2;
			if(a[i]=='J') sum+=1;
		}
	}
	cout<<sum;
	return 0;
}
```

## 分析
模拟统计过程。读入字符串后，从 $0$ 开始循环十三次，判断字符串每一位上字母，对应的计数器累加，统计每种字母的个数。

输出时按照题意进行计算，比如统计得到的 $A$ 的数量最后就要乘 $4$。将每个字母数量计算后得到的结果输出即可。注释见代码部分。

## Code
```cpp
#include<iostream>
using namespace std;
string s;
int n,a,k,q,j;
int main(){
	cin>>n;
	while(n--){
		cin>>s;
		for(int i=0;i<13;i++){ //统计字母数量 
			if(s[i]=='A')a++;
			if(s[i]=='K')k++;
			if(s[i]=='Q')q++;
			if(s[i]=='J')j++;
		}
	}
	cout<<a*4+k*3+q*2+j; //按照题意计算 
	return 0;
} 
```

## 作者：_cmh (赞：0)

本题根据题意模拟即可通过。

这是一道考察字符串知识的题目，依次遍历字符串的每一位，判断是否为 $\text{A}$, $\text{K}$, $\text{Q}$, $\text{J}$ 和 $\text{X}$ ，是则分别加上 $4,3,2,1,0$ 。

但是写那么多 `if` ，太麻烦了。于是我们可以进行优化，再开一个数组，存放 $\text{A}$, $\text{K}$, $\text{Q}$, $\text{J}$ 和 $\text{X}$ 这几个字母，但顺序是倒序。这样我们就可以在遍历过程中直接加上它们对应字母在数组中的下标，即为它们对应的值。

详见代码：

```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>
#include<string>
#include<cstdlib>
using namespace std;
int n,ans;
const int tot[5]={'X','J','Q','K','A'};
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		string s;
		cin>>s;
		for(int j=0;j<s.length();j++)
			for(int k=0;k<5;k++)
				ans+=k*(tot[k]==s[j]);
	}
	cout<<ans;
    return 0;
}
```

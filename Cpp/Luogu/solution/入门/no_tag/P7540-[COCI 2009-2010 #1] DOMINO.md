# [COCI 2009/2010 #1] DOMINO

## 题目描述

求所有满足 $0 \le x \le y \le N$ 的非负整数对 $(x,y)$ 中，$\sum{x} + \sum{y}$ 是多少。

## 说明/提示

#### 【样例 2 解释】

满足要求的所有数对为：$(0,0)$、$(0,1)$、$(0,2)$、$(0,3)$、$(1,1)$、$(1,2)$、$(1,3)$、$(2,2)$、$(2,3)$、$(3,3)。$

#### 【数据范围】

对于 $100\%$ 的数据，$1 \le n \le 10^3$。

#### 【说明】

本题分值按 COCI 原题设置，满分 $50$。

题目译自 [**COCI2009-2010 CONTEST #1**](https://hsin.hr/coci/archive/2009_2010/contest1_tasks.pdf) _**T2 DOMINO**_。

# 题解

## 作者：Fat_Fish (赞：12)

一道简单的结论题。手玩几组数据得到所有的数对组数为$\tfrac{(n+1)*(n+2)}{2}$,

再一求和，发现答案就是$\tfrac{n*(n+1)*(n+2)}{2}$

所以，对于这道题，我们只要$O(1)$输出答案即可

$Code$
```cpp
#include<iostream>
#include<bits/stdc++.h>
using namespace std;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);
	int n;
	cin>>n;
	cout<<(n*(n+1)*(n+2)>>1)<<'\n';//右移一位相当于/2，但运行速度更快
	return 0;
}
```

# 一、求和符号$\sum$
$\sum_{i=0}^n$表示所有$0≤i≤n(i\in N)$的$i$的和。
例如：$\sum_{i=1}^5$意为$1+2+3+4+5$。
> 求所有满足 $0 \le x\le y $ 的非负整数对 $(x,y)$ 中，$\sum{x} + \sum{y}$ 是多少

意为所有满足 $0 \le x\le y $ 的非负整数对 $(x,y)$ 中所有$x$和$y$的和。
# 二、思路
看到这个题第一个想法就是：数学。因为肯定是不能真的暴力把它们加起来的。

然而看到数据范围：
对于 $100\%$ 的数据，$1 \le n \le 10^3$ ......

**暴力大法好！**

# $Code$
```cpp
#include<iostream>
#include<cstdio>
#define LL unsigned long long
using namespace std;
LL n,ans; 
int main()
{
	cin>>n;
	for(LL i=0;i<=n;i++)
		for(LL j=i;j<=n;j++)
			ans+=i+j;
	cout<<ans;
	return 0;
}
```

## 算法 $\bf 1$

暴力。注意 $i$ 的初始值为 $0$。

```cpp
#include <bits/stdc++.h>
using namespace std;
#define _for(i,a,b) for(int i=a;i<=b;i++)
int main() {
  int n,s=0;cin>>n;
  _for(i,0,n)_for(j,i,n)s=s+i+j;
  cout<<s<<endl;
}
```

## 算法 $\bf 2$

$\sum x + \sum y = \sum(x+y) = \sum_{i=0}^n\sum_{j=i}^n(x+y)$。

提出第 $2$ 个求和，得到 $\sum_{i=0}^n\sum_{j=i}^n(x+y) = \sum_{i=0}^ni\times (n+2)$。

前面是 $1 \sim n$ 的和，值为 $\dfrac{n(n+1)}{2}$。

可以求出这个式子的值为 $\dfrac{n(n+1)(n+2)}{2}$。

```cpp
#include <bits/stdc++.h>
using namespace std;
#define _for(i,a,b) for(int i=a;i<=b;i++)
int main() {
  int n;cin>>n;
  cout<<n*(n+1)/2*(n+2);
}
```

## 作者：_lfxxx_ (赞：1)

**题目传送门：[P7540 [COCI2009-2010#1] DOMINO](https://www.luogu.com.cn/problem/P7540)**

### 题意：
给定 $n$ ，求
$$\sum_{x=0}^n\sum_{y=x}^n(x+y)$$
### 思路：
虽然数据范围是 $1\le n\le 10^3$ ，可以用暴力 $O(n^2)$ 做。但是，我们要把眼光往前看，追求完美。

$0$ 会在 $(0,0),(0,1)\cdots $ 出现 ，$1$ 会在 $(0,1)$ （注意，这里有 $1$） $(1,1),(1,2)\cdots$ ， $2$ 会在 $(0,2),(1,2)$ （跟上面一样，这里有 $2$ ） $(2,2),(2,3)\cdots$ 。

不难看出，每个数字所在的数对出现了 $n+1$ 次，但数字本身出现了 $n+2$ 次（会有两个数相同的情况） 。

于是现在的问题变为求
$$(n+2)\cdot\sum_{x=0}^nx$$

这个样子相信大家都会求，于是最终结果是
$$\dfrac{1}{2}\cdot n(n+1)(n+2)$$

直接 $O(1)$ 求出。

## 作者：幼儿园第一名 (赞：1)

### 思路：
我们可以看到数据范围中： $n≤1000$。

所以我们可以直接暴力，双重循环累计答案，当 $i>j$ 时 $continue$ 。

### 核心代码：
```
	int n=read();
	ll ans=0;
	for(int i=0;i<=n;i++) {
		for(int j=0;j<=n;j++) {
			if(i>j) {
				continue;
			}
			ans+=i,ans+=j;
		}
	}
	printf("%lld",ans);
	return 0;
```

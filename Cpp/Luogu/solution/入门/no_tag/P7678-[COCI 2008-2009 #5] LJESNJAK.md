# [COCI 2008/2009 #5] LJESNJAK

## 题目描述

不久以前，在操作系统还未像今天如此发达的时候，计算机无法处理克罗地亚字母表中的某些字母，于是程序员被允许用二到三个计算机能够显示的字符替换这个不能显示的字母，如下表：

| 不能显示的字母 | 用来替换的字符 |
| :----------: | :----------: |
| `č` | `c=` |
| `ć` | `c-` |
| `dž` | `dz=` |
| `ñ` | `d-` |
| `lj` | `lj` |
| `nj` | `nj` |
| `š` | `s=` |
| `ž` | `z=` |

例如说，`ljes=njak` 这一个字符串可以用来表示克罗地亚字母表中的六个字母：`lj`、`e`、`š`、`nj`、`a`、`k`。

现在给出一个字符串，请你求出解码后这一个字符串里面包含多少个克罗地亚字母表中的字母。字母的匹配是贪心的，尽可能长的匹配当前字母，例如 `lj` 不应该被认为是两个英文字母 `l` 和 `j`。

克罗地亚字母表中的字母除了有上表中的几个字母外，还有英语中的 `a`~`z` 这 26 个字母。

## 说明/提示

**【数据范围】**

对于所有数据，满足字符串的长度不超过 $100$，且仅包含小写字母和 `-`、`=` 字符。

**【题目来源】**

本题来源自 **_[COCI 2008-2009](https://hsin.hr/coci/archive/2008_2009/) [CONTEST 5](https://hsin.hr/coci/archive/2008_2009/contest5_tasks.pdf) T1 LJESNJAK_**，按照原题数据配置，满分 $30$ 分。

由 [Eason_AC](https://www.luogu.com.cn/user/112917) 翻译整理提供。

## 作者：yzy1 (赞：17)

严格一行 Python 3。

```python
print(len(input().strip().replace('c=','W').replace('c-','Y').replace('dz=','F').replace('d-','A').replace('lj','K').replace('nj','I').replace('s=','O').replace('z=','I')))
```

说下原理：Python 的 `s.replace(a,b)` 函数能将源字符串 `s` 中的所有子串 `a` 替换为字符串 `b`。题目让我们求的是「一共有多少个字母」，我们只要将「多字符字母」替换为任意的单个字符，再求出字符长度就可以得到答案。

### 题目分析：  
我们将字符串读入后对其一一检查题目所示特别替换的字符，计数，如果不符则为无需替换的字符，也计数，最后输出结果即可。  
### Code：  
```cpp
#include <iostream>
#include <cstdio>
#include <string>
#include <cstring>
using namespace std;
string str;			 //定义字符串
int le = 0, ans = 0; //定义字符串长度与结果
int main()
{
	ios::sync_with_stdio(0); //关同步流加速cin输入和cout输出
	cin >> str;
	le = str.length();			 //求字符串长度
	for (int i = 0; i < le; i++) //注意起始为0
	{
		if (str[i] == 'c' and str[i + 1] == '=')
			ans++, i++; //查找特别替换字符，计数，并推i
		else if (str[i] == 's' and str[i + 1] == '=')
			ans++, i++;
		else if (str[i] == 'z' and str[i + 1] == '=')
			ans++, i++;
		else if (str[i] == 'c' and str[i + 1] == '-')
			ans++, i++;
		else if (str[i] == 'd' and str[i + 1] == '-')
			ans++, i++;
		else if (str[i] == 'l' and str[i + 1] == 'j')
			ans++, i++;
		else if (str[i] == 'n' and str[i + 1] == 'j')
			ans++, i++;
		else if (str[i] == 'd' and str[i + 1] == 'z' and str[i + 2] == '=')
			ans++, i += 2;
		else
			ans++; //无需替换，直接计数
	}
	cout << ans;
	return 0;
}
```

## 作者：LRY314 (赞：0)

看其他题解都是拿来加，这里来一篇用来减且情况合并的代码

一开始把 $ans$ 设为 字符数，但字符串中是有替换字母的，它把一个字母当成了二或三个，再扫一遍字符串遇到替换字母时减去即可。

顺带把一些首或末字母相同的替换字母放在一个判断里了。

```
#include<bits/stdc++.h>
using namespace std; 
int ans;
string s;
int main()
{
	cin>>s;
	ans=s.size();
	for(int i=0;i<s.size();i++)
	{
		if(s[i]=='c'&&(s[i+1]=='='||s[i+1]=='-'))ans--;
		if(s[i]=='d'&&(s[i+1]=='-'||(s[i+1]=='z'&&s[i+2]=='=')))ans--;//你可能会问为什么'dz='只减1，下面'z='会又减一次。 
		if(s[i+1]=='j'&&(s[i]=='l'||s[i]=='n'))ans--;
		if(s[i+1]=='='&&(s[i]=='s'||s[i]=='z'))ans--;//看似把情况合并，实际上码量没少多少。 
	}
	cout<<ans;
}
```

# [COCI 2016/2017 #5] TUNA

## 题目描述

渔夫在昨晚捕到 $N$ 条金枪鱼。他准备将这些鱼卖给一个平台。

对于每一条鱼，平台会给出两个估测值 $P_1,P_2$。如果这两个值之差不超过 $X$，则取较大的值作为该条鱼的价值；否则如果差超过 $X$，则取另一个值 $P_3$。

现在给定 $N$ 条鱼的值（可能会有 $2$ 个或 $3$ 个），求所有鱼的总价值。

## 说明/提示

**【样例 2 解释】**

渔夫捕到了 $4$ 条金枪鱼，而 $X$ 的值为 $2$：

|金枪鱼编号|$p_1$|$p_2$|$p_3$|差值|最终价值|
| :----------: | :----------: | :----------: | :----------: | :----------: | :----------: |
|$1$|$3$|$5$|/|$2 \le 2$|$5$|
|$2$|$2$|$8$|$4$|$6 \gt 2$|$4$|
|$3$|$6$|$5$|/|$1 \le 2$|$6$|
|$4$|$6$|$3$|$7$|$3 \le 7$|$7$|

故价值总和为 $5+4+6+7=22$。

**【数据规模与约定】**

对于 $100\%$ 的数据，$1 \le N \le 20$，$1 \le X \le 10$，$1 \le P_1,P_2,P_3 \le 100$。

## 问题简述

__这道题我们可以换另一种思路去看待它，就容易理解了：__

有 $N$ 个数和一个数 $X$，每个数附有三种值分别为 $P_1,P_2,P_3$，若 $|P_1-P_2|$ 大于 $X$，则取 $P_3$ 为值，否则取 $P_1$ 与 $P_2$ 之间的最大值，求这 $N$ 个数的值的和。

## 解法综述

根据问题简述模拟即可。对 $N$ 个数的取值进行操作，读入两个数 $P_1$ 和 $P_2$，若发现 $P_1$ 与 $P_2$ 之间的差不大于 $X$，则用 $ans$ 累加它们之间的最大值，否则读入一个数 $P_3$ 并用 $ans$ 累加这个数，最后输出 $ans$ 即可。

## 代码描述
```cpp
#include<cstdio>
#define max(a,b) (a>b?a:b)//取两个数之间最大值
#define abs(a,b) (a>b?a-b:b-a)//取两个数之间的绝对值
int N,X,P1,P2,P3,ans;
int main()
{
	scanf("%d%d",&N,&X);
	while(N--)//对N个数的取值进行操作
	{
		scanf("%d%d",&P1,&P2);//读入两个数P1和P2
		if(abs(P1,P2)<=X) ans+=max(P1,P2);//若发现P1与P2之间的差不大于X，则用ans累加它们之间的最大值
		else//否则
		{
			scanf("%d",&P3);//读入一个数P3
			ans+=P3;//用ans累加这个数
		}
	}
	printf("%d",ans);//最后输出ans即可
	return 0;
}
```

### 思路：
读入 $P_1,P_2$ 判断一下 $P_1$ 和 $P_2$ 之差是否小于 $X$ 就行了，如果小于 $X$ 则在 $ans$ 里加上 $P_1$ 和 $P_2$ 里的最大值，否则，在 $ans$ 里加上 $P_3$ 。

最后输出 $ans$ 即可。

### Code:
```c++
#include<bits/stdc++.h>
using namespace std;
inline int read(){//快读 
	int x=0,y=1;
	char ch=getchar();
	while(ch<'0'||ch>'9'){
		if(ch=='-')y=-1;
		ch=getchar();
	}
	while(ch>='0'&&ch<='9')x=(x<<3)+(x<<1)+ch-'0',ch=getchar();
	return x*y;
}
inline void write(int x){//快输 
	if(x<0) putchar('-'),x=-x;
	if(x>9) write(x/10);
	putchar(x%10+48);
	return;
}
int n,x,ans,p1,p2,p3;
int main(){
	n=read();x=read();
	for(int i=0;i<n;i++){
		p1=read();
		p2=read();
		if(p2<p1) swap(p1,p2);//如果p1>p2,则交换p1和p2，使p2为最大值 
		if(p2-p1>x) p3=read(),ans+=p3;//判断，如果p2-p1>x，则加上p3 
		else ans+=p2;//否则，加上p2(因为前面已经使p2为最大值了) 
	}
	write(ans);//输出 
	return 0; 
}
```

## 作者：AmaZingFantasy (赞：3)

描述：农夫有 $N$ 条鱼，要把它们卖掉。每条鱼会给出两个价钱 $P_1$ 和 $P_2$。如果它们的差小于等于 $X$。那么这条鱼的价钱就是 $P_1$ 和 $P_2$ 的最大值，否则价格就是 $P_3$。

思路：给定 $N$ 组数据和一个正整数 $X$，定义一个变量 $cnt$，每组数据首先给出 $P_1$ 和 $P_2$。如果 $p_1$ 和 $P_2$ 的差 $\leq X$，则让 $cnt$ 加 $P_1$ 和 $P_2$ 中的较大值；否则给出 $P_3$，让 $cnt$ 加上 $P_3$。

代码如下：

```cpp
#include <iostream>
typedef long long l;
using namespace std;
int main(){
    l n,x;
    cin>>n>>x;//输入n，x。
    l cnt=0;
    for(l i=0;i<n;i++){
        l p1,p2;
        cin>>p1>>p2;//输入p1,p2。
        if(abs(p2-p1) <= x){//
            cnt+=max(p2,p1);
        }else{
            l p3;
            cin>>p3;
            cnt+=p3;
        }
    }
    cout<<cnt;
    return 0;
}

```

每部分的格式为下列二者之一。

原来当前两个数差值大于 $X$ 后才输入第三个数字！

## 正确解法

在每一次输入时，先判断两个数的绝对值之差是否大于 $X$ ，如果大于则输入第三个数字并将第三个数字累加至答案中，否则将前两个数的最大值累加进答案中，最后输出答案即可。

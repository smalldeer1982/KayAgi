# 题意
给定一个长度为 $8$ 的字符串 $s$，统计 $s$ 中大小写字母和数字字符的个数。
## 本题可以使用字符数组或 string 类型，这里给出两种方法代码，代码如下。
### 1. 使用数组
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[10];
int num, big, small;//num记录数字出现个数，big记录大写字母，small记录小写字母。

int main()
{
	scanf("%s", s);//读入。
	for (int i = 0; i < strlen(s); i++)//循环查找每一个字符
	{
		if ('0' <= s[i] && s[i] <= '9') num++;//数字
		else if ('a' <= s[i] && s[i] <= 'z') small++;//小写字母
		else if ('A' <= s[i] && s[i] <= 'Z') big++;//大写字母
	}
	printf("%d %d %d\n", num, small, big);//输出
	return 0;
}
```

```cpp
#include <bits/stdc++.h>
using namespace std;

{
	cin >> s;//cin读到空格结束，如果要读取整行，使用getline(cin, s);。
	for (int i = 0; i < s.size(); i++)
	{
		if ('0' <= s[i] && s[i] <= '9') num++;
		else if ('a' <= s[i] && s[i] <= 'z') small++;
```

## 作者：_O_v_O_ (赞：0)

此题主要考验字符串处理。

我们可以用三个变量来存储每种字符出现次数，将字符串遍历一遍然后再判断每个字符的类型。

关键在于，如何判断字符类型。

众所周知，字符有个编码，名叫 ASCII 码，编程中的字符也是按此编码，而在此码中，$0 \sim 9$，$a\sim z$，$A\sim Z$ 的编码都是连续的，并且字符之间的比较正是以该码为依据进行比较的，因此，我们只用判断每个字符是否在每个范围内即可。

代码：

```cpp
#include<bits/stdc++.h>
using namespace std;

string s;

signed main(){
	cin>>s;
	int l=s.size(),sum1=0,sum2=0,sum3=0;  //.size函数能获取一个字符串的长度。
	for(int i=0;i<l;i++){
		if(s[i]>='0'&&s[i]<='9') sum1++;  //字符判断。
		else if(s[i]>='a'&&s[i]<='z') sum2++;
		else sum3++;
	}
	cout<<sum1<<' '<<sum2<<' '<<sum3;
	return 0;
}
```

# 题目大意

给定一个长度为 $8$ 字符串 $s$，依次输出 $s$ 内数字、小写字母、大写字母的个数。

---

# 思路

比较水的一道题目，~~那我就不讲了~~。

## 大致分析

~~怎么水的题目还要什么分析啊~~。

首先我们可以把这个字符串拆成 $8$ 个部分，每一个字符为 $1$ 个部分。

对于每一次输入，我们读入 $1$ 个部分。对于这个字符，我们需要去判断属于哪一种情况，把那个情况的计数器加一即可。

## 具体实现

对于输入，十分简单，定义一个字符类型的变量，只需要循环八次每一次重复读入变量即可。

那么我们判断的时候就有一点难了，因为我们要判断是否属于一个范围内的字符，不能直接像数字一样用 `>=` 和 `<=`（记住，后面要考），但是在计算机中，有一个东西很方便，那就是 ascii 码，他是 c++ 中把字符转换成数字的一种最快的方法。

由于 ascii 码中，这几个范围都是连续的，所以我们只需要用 ascii 就可以直接过。

但是记 ascii 码的位置比较麻烦，实际上 c++ 直接可以字符比较，也就是（打个比方）`a<='Z'` 或者 `a>='A'` 即可。

# CODE

`Talk is cheap.Show me the code. —— AprendiZ`

```cpp
#include<bits/stdc++.h>
using namespace std;
char a; //字符
int b,c,d; //计数器
signed main()
{
	for(int i=1;i<=8;i++) //循环
	{
		cin>>a; //输入字符
		if(a>='0' && a<='9') //判断
			b++;
		if(a>='a' && a<='z') //同上
			c++;
		if(a>='A' && a<='Z') //同上
			d++;
	}
	cout<<b<<" "<<c<<" "<<d<<"\n"; //输出计数器
	return 0;
}
```

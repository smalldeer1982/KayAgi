# [CCC 2008 J1] Body Mass Index

## 题目描述

身体质量指数（BMI）是医生用来评估成年人健康状况的计算方法之一。医生测量患者的身高（以米为单位）和体重（以千克为单位），然后使用公式计算 BMI：

$$\text{BMI} = \dfrac{\text{weight}}{\text{height} \times \text{height}}$$

编写一个程序，提示输入患者的身高和体重，计算 BMI，并显示下表中的相应信息。

| BMI 分类 | 信息 |
| :----------: | :----------: |
| 超过 $25$ | 超重 |
| 在 $18.5$ 和 $25.0$ 之间（含） | 正常体重 |
| 小于 $18.5$ | 体重过轻 |

## 说明/提示

**样例输入 1 的输出解释**：

BMI 为 $69 \div (1.73 \times 1.73)$，约为 $23.0545$。根据表格，这是“正常体重”。

**样例输入 2 的输出解释**：

BMI 为 $84.5 \div (1.8 \times 1.8)$，约为 $26.0802$。根据表格，这是“超重”。

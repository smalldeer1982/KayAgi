# 甲流病人初筛

## 题目描述

目前正是甲流盛行时期，为了更好地进行分流治疗，医院在挂号时要求对病人的体温和咳嗽情况进行检查，对于体温超过 $37.5$ 度（含等于 $37.5$ 度）并且咳嗽的病人初步判定为甲流病人（初筛）。现需要统计某天前来挂号就诊的病人中有多少人被初筛为甲流病人。

## 样例 #1

### 输入

```
5
Zhang 38.3 0
Li 37.5 1
Wang 37.1 1
Zhao 39.0 1
Liu 38.2 1```

### 输出

```
Li
Zhao
Liu
3```

# 题解

## 作者：我和鱼过不去 (赞：19)

### 思路
+ 使用计数器 $ans$ 来记录被初筛为甲流的病人数量；  
+ 依次读入病人的信息，判断此人体温是否**大于等于** $37.5$ 度且咳嗽，如果是的话，直接输出病人名字，$ans$ ++；
+ 最后输出 $ans$。  

### Code
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,cough,ans;
string name;   //循环n次读入，可以重复使用变量
double temp;
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        cin>>name>>temp>>cough;   //读入病人的信息
        if(temp>=37.5&&cough)     //如果符合筛查条件
        {
            cout<<name<<endl;   //输出名字
            ans++;              //被初筛为甲流的病人数量+1
        }
    }
    cout<<ans<<endl;     //输出被初筛为甲流的病人数量
    return 0;
}
```

## 题意
给定 $n$ 组数据，每组数据由一个字符串，一个浮点型变量，一个整形变量组成，若浮点型变量大于 37.5 且整形变量为 1 则输出字符串，最后输出满足条件的数据组数。
## 解法
### I 在线做法
~~初学者可以忽略什么是在线/离线做法~~，输入数据组数 $n$ ,并循环读入数据，用条件语句判断是否满足条件并输出即可。

代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,k,ans;
//n组数据，ans存储满足条件的数据组数，k表示是否咳嗽 
double tem;
//存储体温 
string name;
//存储姓名 
int main()
{
    cin>>n;
    for(int i=0;i<n;i++)
    {
        cin>>name>>tem>>k; 
        //字符串类型变量可以直接用cin读入 
        if(tem>=37.5 && k) 
        //判断 
        {
            cout<<name<<endl;
            ans++;
        } 
    }
    cout<<ans<<endl;
    return 0;
}
```

### II 离线做法
用结构体存储每一组数据，最后统一判断并输出

代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,ans;//n组数据，ans存储满足条件的数据组数，
struct person
{
    string name;//存储姓名 
    double tem;//存储体温 
    int k;//k表示是否咳嗽 
}a[300]; 
int main()
{
    cin>>n;
    for(int i=0;i<n;i++) cin>>a[i].name>>a[i].tem>>a[i].k; 
    //字符串类型变量可以直接用cin读入
    for(int i=0;i<n;i++)
        if(a[i].tem>=37.5 && a[i].k) //判断 
        {
            cout<<a[i].name<<endl;
            ans++;
        }
    cout<<ans<<endl;
    return 0;
}
```

## 作者：int127 (赞：3)

对于每个读入，判断体温是否大于 $37.5$ 度，是否咳嗽即可。

详情见代码：

```cpp
#include<bits/stdc++.h>
#define fr(i,a,b,step) for(rg int i=a;i<=b;i+=step)
#define rp(i,a,b,step) for(rg int i=a;i>=b;i-=step)
#define rs(a,b) memset(a,b,sizeof(a))
#define ie inline
#define rg register
using namespace std;
int n;
struct patient{//这个结构体可以用3个数组 (p[],t[],k[]) 替代
	string p;//名字 (patient)
	double t;//体温 (temperature)
	bool k;//是否咳嗽 (kesou)
}a[210];
string ill[210];//初筛病人 (illness)，可以用vector<int> ill;来替代
int cnt;//病人数
signed main(){
	cin>>n;
	fr(i,1,n,1){
		cin>>a[i].p>>a[i].t>>a[i].k;
		if(a[i].t>=37.5&&a[i].k) ill[++cnt]=a[i].p;//病人！存下来！
	}
	fr(i,1,cnt,1) cout<<ill[i]<<'\n';
	cout<<cnt;
	return 0;
}
```

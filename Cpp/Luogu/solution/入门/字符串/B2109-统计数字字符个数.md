# 统计数字字符个数

## 题目描述

输入一行字符，统计出其中数字字符的个数。

## 样例 #1

### 输入

```
Today is 2021-03-27```

### 输出

```
8```

# 题解

## 作者：__Cartesian__Tree__ (赞：54)

题目要求求出字符串里的数字字符个数，可以根据字符的 ASCII 码来判定。
```cpp
if (s[i] <= '9' && s[i] >= '0')
    cnt++;
```
这里我使用了 C++ 独有的 ```string``` 类型的字符串。

在读入时需要注意，如果你写成这样：

```cin >> s;```

那么你就只能获得 50 分，因为字符串中有空格，需要整行读入。

AC Code:
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    getline(cin, s);//要用 getline
    int cnt = 0;
    for (int i = 0; i <= s.length() - 1; i++){//注意范围
        if (s[i] <= '9' && s[i] >= '0')//判断
            cnt++;
    }
    cout << cnt << endl;
    return 0;
}
```

这道题目是一个很容易的字符串处理，读入一行，每个遍历即可，代码：

```cpp
#include <iostream>
#include <cstdio>
#include <string>
using namespace std;

int main()
{
    int cnt = 0;
    string s;
    getline(cin, s);
    for (int i = 0; i < s.length(); i++) cnt += isdigit(s[i]);
    cout << cnt << endl;
    return 0;
}
```

## 题目大意
给定一个字符串，求其中数字的个数。
## 思路
### I 小知识
在c++中有一个 ```isdigit( )``` 函数，会判断一个字符是否为数字，并返回一个整形的值。

## II 具体做法
用字符型数组存字符串，循环内依次使用 ```isdigit( )``` 函数，来判断每一个字符是否为数字。
## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
char a[300]; //char型数组存字符串
int ans,lena;
int main()
{
    gets(a);
    lena=strlen(a); //strlen函数返回char型数组的长度
    for(int i=0;i<lena;i++)
        if(isdigit(a[i])) ans++;
    //若该字符是数字，则累加器加一
    cout<<ans<<endl;
    return 0;
}
```

# 题目分析
遍历字符串，遇见一个数字字符则数字字符个数增加1即可统计数字字符个数。
# C#代码
```c#
using System;
class MainClass{
    public static int num(string s){//统计数字字符个数函数
        int n = 0;//数字字符个数
        for (int i = 0; i < s.Length; i++) if (s[i] <= '9' && s[i] >= '0') n++;//遍历字符串，遇见一个数字字符则数字字符个数增加1
        return n;
    }
    public static int Main(){
        string s;
        s = Console.ReadLine();
        Console.WriteLine("{0}",num(s));//输出数字字符个数
        return 0;
    }
};
```

# 基因相关性

## 题目描述

为了获知基因序列在功能和结构上的相似性，经常需要将几条不同序列的 DNA 进行比对，以判断该比对的 DNA 是否具有相关性。

现比对两条长度相同的 DNA 序列。首先定义两条 DNA 序列相同位置的碱基为一个碱基对，如果一个碱基对中的两个碱基相同的话，则称为相同碱基对。接着计算相同碱基对占总碱基对数量的比例，如果该比例大于等于给定阈值时则判定该两条 DNA 序列是相关的，否则不相关。

## 样例 #1

### 输入

```
0.85
ATCGCCGTAAGTAACGGTTTTAAATAGGCC
ATCGCCGGAAGTAACGGTCTTAAATAGGCC```

### 输出

```
yes```

# 题解

## 作者：墨忧 (赞：27)

## 题目大意
给定两个字符串以及一个阈值 $c$ ,若两个字符串中字符相同的比例大于 $c$ ,则输出 “yes” ，否则输出 “no” 。
## 思路
使用一个浮点型变量存储阈值，分别用两个字符型数组存储两个字符串，循环内依次比对字符串的每一位，最后判断并输出即可
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
double c,ans;//c存储阈值，ans存储相似度 
char a[1000],b[1000];//a，b两个数组存储字符串 
int len;//字符串长度 
int main()
{
    cin>>c>>a>>b;//char型数组可以直接用cin读入 
    len=strlen(a);//strlen函数返回数组长度 
    for(int i=0;i<len;i++) if(a[i]==b[i]) ans++;//依次比对 
    ans/=len;
    if(ans>=c) cout<<"yes"<<endl;//输出答案 
    else cout<<"no"<<endl;
    return 0;
} 
```

## 作者：MCH_Satrimiten (赞：12)

本蒟蒻的一篇题解，不喜勿喷

我们先要求比例。比例就是两个字符串中**对应一样的字母个数**除以**字符串总长度**。

样例中 `ATCGCCGTAAGTAACGGTTTTAAATAGGCC` 和`ATCGCCGGAAGTAACGGTCTTAAATAGGCC` 两个长度相同的字符串，串长为 $30$ ,相似的字母个数为 $28$ ， $28$ / $30$是大于 $0.85$ 的，所以输出 “ `yes` ”。

所以，我们只需要求出比例，问题就迎刃而解了。

### 上代码

```cpp
#include<bits/stdc++.h>
using namespace std;
char a[1000001],b[1000001];
int main(){
	double f;
	cin>>f;//阈值
	cin>>a>>b;//两个字符串
	int len=strlen(a),x=strlen(a);//len为串长，x记录相同的字母
	for(int i=0;i<len;i++)if(a[i]!=b[i])x--;//不一样就减一
	double s=x,l=len;//转成double类型
	s/=l;//比例
	if(s<f){cout<<"no";return 0;}//小于阈值就输出no
	cout<<"yes";//否则输出yes
	return 0;
}
```

# 题意
输入浮点数，再输入两个字符串（保证长度相等），若两个字符串同一位置相同记作一对，求出总相同对数，再求两个字符串相似性（即  对数 $\div$ 字符串长度 ），若相似性大于等于给出的浮点数，则输出 yes ，否则输出 no 。
# 注意
输入浮点数换行需要按回车键，系统会将此回车当做第一个字符串，所以要用一个新的字符串“吃”下这个回车，然后就可以正常输入了。

# 代码
```cpp
#include<iostream>
using namespace std;

double p,ans;
string a,b,t;


int main()
{
	cin>>p;
	getline(cin,t);//用来吸收回车键的字符串
	getline(cin,a);
	getline(cin,b);
	for(int i=0;i<a.size();i++)
		if(a[i]==b[i])
			ans+=1;//逐一配对
	if(ans/a.size()>=p)//若大于等于，则输出yes
		cout<<"yes";
	else
		cout<<"no";//反之
	return 0;//好习惯
}
```

# 题目分析
字符串裸题，遍历字符串寻找相同碱基对即可。
# C#代码
```c#
using System;
class MainClass{
    public static double check(string s1, string s2){//判断相关性函数
        double a = 0, b = s1.Length;//a 是相同碱基对的对数，b 是总数
        for (int i = 0; i < b; i++) if (s1[i] == s2[i]) a = a + 1.0;//如果相关，a 增加1
        return a / b;//返回相同碱基对占总碱基对数量的比例
    }
    public static int Main(){
        string xx;
        xx = Console.ReadLine();
        double x = Convert.ToDouble(xx);//判定出两条 DNA 序列是否相关的阈值，这里将读入的字符串数据转换成了 double 数据
        string s1, s2;
        s1 = Console.ReadLine();
        s2 = Console.ReadLine();//输入两条 DNA 序列
        if (check(s1, s2) >= x) Console.WriteLine("yes");//如果相同碱基对占总碱基对数量的比例大于等于给定阈值，输出 yes
        else Console.WriteLine("no");//否则输出 no
        return 0;
    }
};
```

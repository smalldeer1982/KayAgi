# 输出亲朋字符串

## 题目描述

亲朋字符串定义如下：给定字符串 $s$ 的第一个字符的 ASCII 值加第二个字符的 ASCII 值，得到第一个亲朋字符；$s$ 的第二个字符加第三个字符得到第二个亲朋字符；依此类推。注意：亲朋字符串的最后一个字符由 $s$ 的最后一个字符 ASCII 值加 $s$ 的第一个字符的 ASCII 值。

## 样例 #1

### 输入

```
1234```

### 输出

```
cege```

# 题解

## 作者：alvis (赞：48)

## 题意
给定一个字符串，输出两相邻的字符 $\text{ACSII}$ 码之和，并规定首字符与末字符相邻。
## 小知识
对于字符串类的问题，我们可以使用 $\text{string}$ 来解决问题。

求一个 $\text{string}$ 类型 的字符串长度：
```cpp
string s = "12313212";
int l = s.length();
cout << l;
```
上面程序的输出结果是：$8$

另外，$\text{c++}$ 对于 $\text{ACSII}$ 码的处理，可以将两个字符直接相加，所得结果即为一个 $\text{ACSII}$ 码为**原两字符 $\text{ACSII}$ 码的和**的字符。

其次，$\text{string}$ 类型的字符串可以直接加一个字符：
```cpp
string s = "123132";
s += s[0];
for(int i = 0;i < s.length();i ++){
	cout << s[i];
}
```
上述代码的结果为：$1231321$

即将所加的字符添加到**原字符串的末尾**。

## 代码
```cpp
#include <iostream>
#include <string>
using namespace std;

string a, s;

int main(){
    cin >> a;
    //求字符串长度
    int l = a.length();
    //将第一个字符添加到末尾，便于处理
    a += a[0];
    //求得目标字符串
    for(int i = 0;i < l;i ++ ){
        s[i] = a[i] + a[i+1];
    }
    //输出
    for(int i = 0;i < l;i ++) {
        cout << s[i];
    }
    return 0;
}
```

## 题目分析
此题只用将新字符串按题意操作即可，注意在循环中需要判断是否是最后一个字符，如果是，就要进行特殊操作。
## 代码实现
```cpp
#include<bits/stdc++.h>
using namespace std;
string s,k;
int main(){
	cin>>s;
	for(int i=0;i<s.size();i++){
		if(i+1==s.size())
			k[i]=s[i]+s[0];//最后一个字符
		else
			k[i]=s[i]+s[i+1];
	}
	for(int i=0;i<s.size();i++)
		cout<<k[i];
	return 0;
}
```

# 思路
先把字符串每相邻两位 ASCII 值放入一个数组里，最后单独输出首字符与末字符相加得到的 ASCII 值。
# 代码：
```c
#include <bits/stdc++.h>
using namespace std;
int x[105];
char s[105];
int main(){
	cin>>s;
	for(int i=0;i<strlen(s)-1;i++){
		x[i]=s[i]+s[i+1];//把每相邻两位相加的ASCII值存入x数组
	}
	for(int i=0;i<strlen(s)-1;i++){
		cout<<char(x[i]);
	}
	cout<<char(s[strlen(s)-1]+s[0])<<endl;//输出首字符与末字符相加得到的ASCII值
    return 0;
}


```

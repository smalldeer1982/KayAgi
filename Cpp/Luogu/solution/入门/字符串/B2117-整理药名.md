# 整理药名

## 题目描述

医生在书写药品名的时候经常不注意大小写，格式比较混乱。现要求你写一个程序将医生书写混乱的药品名整理成统一规范的格式，即药品名的第一个字符如果是字母要大写，其他字母小写。

如将 ASPIRIN 、 aspirin 整理成 Aspirin。

## 样例 #1

### 输入

```
4
AspiRin
cisapride
2-PENICILLIN
Cefradine-6```

### 输出

```
Aspirin
Cisapride
2-penicillin
Cefradine-6```

# 题解

## 作者：￥隐姓埋名￥ (赞：36)

题目大意
------------
输入数字 $n$，并输入 $n$ 个字符串，将每个字符串改写成除开头字母大写外，其余字母小写的形式并输出，其它字符不变。

**难点：大小写转化。**

这道题我们可以定义字符串并用```for```循环解决，再用上大写转小写和小写转大写的方法，如下所示：

```cpp
//小写转大写
if(s[0]>='a'&&s[0]<='z')
	s[0]-=32;
```

```cpp
//大写转小写
if(s[0]>='A'&&s[0]<='Z')
	s[0]+=32;
```

$A$ 的序号是 $65$，$a$ 的序号是 $97$，两者作差 $32$。

实在不理解的可以看表格:

![](http://www.elecfans.com/article/UploadPic/2009-6/200963010427214.jpg)

代码
------------
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int n;
    cin>>n;
    string s;
    for(int i=1;i<=n;i++)
    {
        cin>>s;
        if(s[0]>='a'&&s[0]<='z')
            s[0]-=32;
        for(int j=1;j<=s.size()-1;j++)
            if(s[j]>='A'&&s[j]<='Z')
                s[j]+=32;
        cout<<s<<endl;
    }
    return 0;
}
```

## 题目大意
给定 $n$ 个字符串，对于每个字符串，若其开头字符为小写字母，则转变为大写字母，对于其他字母，若为大写，则转变为小写，字母以外不做变动。
## 思路
### I 前置小知识
#### 1.
在 c++ 中有一个 ```islower( )``` 函数，会判断一个字符是否为小写字母，并返回一个整型值，还有一个函数 ```isupper( )``` ，会判断一个字符是否为大写字母，同样返回一个整型的值。
#### 2.
在 c++ 中，小写的字母的 ASCII 码与大写字母的 ASCII 码相差 32 ，因此**大写字母加 32 可转化为小写字母**，同理，**小写字母减 32 可转化为大写字母**。
### II 解法
用字符型数组存储字符串，循环判断每一位字符是否为大/小写字母，转变大小写即可。
## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,lena;
char a[30];//char型数组存字符串
int main()
{
    cin>>n;
    for(int i=0;i<n;i++)
    {
        cin>>a;
        lena=strlen(a);;//strlen返回char型数组的长度
        if(islower(a[0])) a[0]-=32;
        //判断开头字符是否是小写字母，若是，则转化为大写
        for(int j=1;j<lena;j++) if(isupper(a[j])) a[j]+=32;
        //判断其他字符是否为大写字母，若是，则转化为小写
        cout<<a<<endl;
    }
    return 0;
}
```

# 题目分析
这是一道字符串的裸题，我们可以循环 $n$ 次，每次输入药名后先将药名的首位转换为大写（如果不是字母则不变），再将药名的其余字母转换为小写。

# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;cin>>n;
    for(int i = 0;i < n;i ++){
        string drugname;cin>>drugname;//输入药名
        transform(drugname.begin(),drugname.begin()+1,drugname.begin(),::toupper);//将药名的首位转换为大写（如果不是字母则不变）
        transform(drugname.begin()+1,drugname.end(),drugname.begin()+1,::tolower);//将药名的其余字母转换为小写
        /*
        关于将字符串转换为大/小写，STL为我们提供了函数 transform(A.begin(), A.end(), B.begin(), ::toupper); 。
        其中 A.begin() 是需要转换的字符串的起始位置，A.end() 是需要转换的字符串的终止位置，B.begin() 是输出字符串的起始位置，而 ::toupper 是转换为大写的函数，小写是 ::tolower 。
        */
        cout<<drugname<<endl;//输出新的药品名
    }
    return 0;
}
```

# B2117 整理药名

### 题目大意：

我们需要把字符串改写成这种格式：$1\times$大写$+(n-1)\times$小写

一个字符是否大写判断方式：

```cpp
('A' <= c && c <= 'Z')
```

一个字符是否小写判断方式：

```cpp
('a' <= c && c <= 'z')
```

一开始对于第一个字符进行判断，将其改为大写。

然后对其他字符进行判断，全部改为小写。

```cpp
#include<iostream>
using namespace std;
int main()
{
	int n;
	cin >> n;
  	while(n--)
  	{
		string str;
		cin >> str;
 		if('a' <= str[0] && str[0] <= 'z')//如果是小写
		{
			str[0] = str[0] - 'a' + 'A';//改大写
		}
		for(int i = 1; i < str.size(); i++)
		{
			if('A' <= str[i] && str[i] <= 'Z')//如果是大写
			{
				str[i] = str[i] - 'A' + 'a';//改小写
			}
		}
  	}
	return 0;
}
```

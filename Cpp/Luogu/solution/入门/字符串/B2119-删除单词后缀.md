# 删除单词后缀

## 题目描述

给定一个单词，如果该单词以 `er` 、 `ly` 或者 `ing` 后缀结尾，则删除该后缀（题目保证删除后缀后的单词长度不为 $0$），否则不进行任何操作。

## 样例 #1

### 输入

```
referer
```

### 输出

```
refer
```

# 题解

## 作者：Jorisy (赞：36)

提供一种较为~~玄学~~做法。  
我们先不看这种做法，先来看一个 $75pts$ 的做法：
```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	string s;
	cin>>s;
	cout<<(s.find("er",0)==s.size()-2||s.find("ly",0)==s.size()-2?s.substr(0,s.size()-2):(s.find("ing",0)==s.size()-3?s.substr(0,s.size()-3):s));
	return 0;
}
```
这种方法是想按序找到后缀，然后删除，然而像输入 `singing` 时，其不是我们所希望的 `sing`，而是 `singing`。因为先找到的是 `s` 后的子串 `ing`。  
既然正着不行，那就——**倒着**找！  
这里涉及到函数 `reverse()` 及 `size()`，详见[此](https://www.luogu.com.cn/blog/JY-Cjy/solution-b2145)。但除此还涉及到 `find()` 和 `erase()`。  

---

对于 `find()` 函数的用法：这个函数共有两个参数，一个是查找子串，一个是开始查找位置，对于查找，其会返回首次查到到位置；若找不到，则会返回 `string::npos`。如在字符串 `I Love Luogu.` 中查找子串 `Luogu`，会返回 $7$。  
对于 `erase()` 函数的用法：这个函数也有两个参数，一个是删除子串的起始位置，一个是删除子串的长度。如在字符串 `I Don't Love Luogu.` 中删除子串 `Don't`，则字符串变成 `I Love Luogu.`。  
完整代码：
```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	string s;
	cin>>s;
	reverse(s.begin(),s.end());//第一次翻转
    //由于翻转了，所以子串也要翻转
	if(s.find("re",0)==0||s.find("yl",0)==0) s.erase(0,2);//删除长度是二的
	else if(s.find("gni",0)==0) s.erase(0,3);//删除长度是三的
	reverse(s.begin(),s.end());//重新翻转回来
	cout<<s;
	return 0;
}
```

## 题意
给定一个字符串并输出，若字符串以 "er" , "ly" , "ing" 后缀结尾，则不输出后缀。
## 思路
用一个字符型数组存储字符串，判断是否以 "er" , "ly" , "ing" 后缀结尾，若是，则将其删除最后输出字符串即可。
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
char a[50];//存储字符串 
int lena;
int main()
{
    cin>>a;
    lena=strlen(a);//strlen函数返回char型数组的长度 
    if(a[lena-1]=='r' && a[lena-2]=='e') a[lena-1]=a[lena-2]=' ';
    if(a[lena-1]=='y' && a[lena-2]=='l') a[lena-1]=a[lena-2]=' ';
    if(a[lena-1]=='g' && a[lena-2]=='n' && a[lena-3]=='i') a[lena-1]=a[lena-2]=a[lena-3]=' ';
    //删除后缀 
    cout<<a;//输出删除后后缀后的字符串 
    return 0;
 } 
```

## 作者：Griseo_nya (赞：21)

按照题目要求照做即可。输入字符串后，从后往前逐个字符匹配后缀，如为 er , ly , ing 则删除后缀。

下面我们附使用 STL string 类和 char 数组的两种方式。
## STL string 类做法
### 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
int main(){
	cin>>s;																	//输入 
	int siz=s.size();														//取得大小 
	if(s[siz-1]=='y'&&s[siz-2]=='l')s.erase(siz-2,2);						//如果结尾为 ly ,删除 ly 
	else if(s[siz-1]=='r'&&s[siz-2]=='e')s.erase(siz-2,2);					//如果结尾为 er ,删除 er 
	else if(s[siz-1]=='g'&&s[siz-2]=='n'&&s[siz-3]=='i')s.erase(siz-3,3);	//如果结尾为 ing ,删除 ing
	cout<<s<<endl;															//输出 
	return 0;
} 
```
### 关于 STL string 类的一些类方法
在这篇题解中用到了 STL 中的 string 类，简单介绍一下在这里用到的类方法：

* ```string::erase( pos , len )``` : 从第 $pos$ 个位置开始，擦除之后的 $len$ 个字符。

* ```string::size()``` : 返回该字符串的长度。

## char 数组做法
### 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
char s[50];
int main(){
	cin>>s;																//输入 
	int siz=strlen(s);													//取得大小 
	if(s[siz-1]=='y'&&s[siz-2]=='l')s[siz-2]='\0';						//如果结尾为 ly ,删除 ly 
	else if(s[siz-1]=='r'&&s[siz-2]=='e')s[siz-2]='\0';					//如果结尾为 er ,删除 er 
	else if(s[siz-1]=='g'&&s[siz-2]=='n'&&s[siz-3]=='i')s[siz-3]='\0';	//如果结尾为 ing ,删除 ing
	cout<<s<<endl;														//输出 
	return 0;
} 
```
### 讲解
我们之所以可以这么做，是因为在 C++ 内部表示中，认为 char 数组所表示的字符串以 ```'\0'``` 结尾，所以我们相当于将结尾位放在了被赋 ```'\0'``` 的位置，系统自然就不会继续读下去了。

# 题目分析
依照题目一步一步照做即可，这是一道字符串的题，同时考察了分支结构。

过程：先取出单词的后2位与单词的后3位，再判断单词的后2位是不是 `er` 或 `ly`，如果是删去它们并输出新单词；如果不是，但单词的后3位是 `ing`,删去它们并输出新单词；否则直接输出单词。
# 代码
```cpp
#include<bits/stdc++.h>//万能头
using namespace std;//使用std命名空间
int main(){//主函数
    string words;cin>>words;//定义题目中的单词并输入
    string l2="",l3="";//l2 为单词的后2位，l3 为单词的后3位
    l2+=words[words.size()-2],l3+=words[words.size()-3],l2+=words[words.size()-1],l3+=l2;//取出单词的后2位与单词的后3位（其中 words.size() 是单词的长度）
    if(l2=="er"||l2=="ly"){words[words.size()-2]='\0',words[words.size()-1]='\0';cout<<words;}//如果单词的后2位是 er 或 ly,删去它们并输出新单词
    else if(l3=="ing"){words[words.size()-3]='\0',words[words.size()-2]='\0',words[words.size()-1]='\0';cout<<words;}//如果单词的后3位是 ing,删去它们并输出新单词
    else cout<<words;//否则直接输出单词
    return 0;//返回0
}
```

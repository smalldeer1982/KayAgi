# 单词翻转

## 题目描述

小明同学写单词的时候喜欢反着写，比如 `hello` 他会写成 `olleh`。给出小明同学写的一个句子，请你将所有的单词复原。

## 说明/提示

句子长度不超过 $100$，仅包含小写字母和空格，不存在多余的空格。


## 样例 #1

### 输入

```
olleh dlrow```

### 输出

```
hello
world```

## 作者：StarryWander (赞：45)

## 字符串基础。

### 解题步骤：
1.  定义一个字符串。 
1.  输入**多个**字符串。
1.  字符串**倒序**输出。

- 对于**输入**字符串，可以采用以下的形式，更加方便地读入**多个字符串**：
```cpp
while(cin>>s)
```

- 对于输出有几个点需要注意：

1. 通过以下函数获取**字符串长度**：
```cpp
s.size()
```
2. 字符串下标从：**长度 $ - 1$ 开始**，**到 $0$** 遍历。

3. **记得换行**。

---
### 代码。
```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;
int main(){
	string s;//定义字符串。
	while(cin>>s){//输入字符串。
		for(int i=s.size()-1;i>=0;i--){//循环输出。
			cout<<s[i];
		}
		cout<<endl;//换行。
	}
	return 0;
}

```

## 作者：baiABC (赞：13)

这道题目是考字符串的操作。
- - -
#### 声明一个字符串：
- ```char s_c[k]```：可以声明一个最大长度为 $k-1$ 的 C 字符串。其中 ```k``` 是常量表达式。只有结尾是 ```'\0'```（空字符）时才可以用作字符串，否则就是普通的字符数组。
- ```string s_cpp```：这是 C++ 字符串，在 ```<string>```头文件中（或者直接用万能头文件 ```<bits/stdc++.h>```）。

#### 读入：
本题读单词最好用  ```scanf("%s", s_c)``` 或 ```cin >> s_cpp```。
#### 输出：
用 ```puts(s_c)```（会输换行） 或 ```cout << s_cpp << endl```。
### 示例代码：
#### C++：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
   string s;
   while(cin >> s) // 一直读入单词直到文件末尾
   {
      reverse(s.begin(), s.end()); // STL 大法好！
      cout << s << endl;
   }
   return 0;
}
```
#### C：
这里用到了 strlen 函数，用来求字符串长度。它在 ```<string.h>``` 头文件中。
```c
#include <stdio.h>
#include <string.h> // 用了 strlen
int main()
{
   char s[101]; // 注意数据范围！要给空字符留下位置（说不定整个句子就一个单词呢）
   while(~scanf("%s", s)) // 不加 &!
   { // 多组数据读入固定格式
      for(int i = strlen(s)-1; i >= 0; --i)
         putchar(s[i]); // 输出字符
    /* 其实不用真正翻转，倒着输出就可以了 */
      putchar('\n');
   }
   return 0;
}
```

### 题意
给定一系列字符串，分别将每个字符串翻转后输出。


### 介绍一个有用的函数：

```reverse(s.begin(), s.end())```
的方法调用。

### 代码如下
```
#include<bits/stdc++.h>
using namespace std;
int main(){
    string s;
    while(cin >> s){
        reverse(s.begin(), s.end());
        cout << s << endl;
    }
    
    return 0;
}
```

## 作者：Jorisy (赞：7)

对于题目，我们可以知道输入的字符串是不定的，那么我们可以用 `while(cin>>s)` 来实现。这句话的意思是：在获取 `Ctrl+Z` 时一直输入字符串 $s$（评测机会自动输入 `Ctrl+Z`）。  
接下来，是本题的关键。  
对于翻转字符串，我们可以使用一个 STL 字符串函数——`reverse()`。这个函数我在其他题解中说过，这里再详细说一遍。  
这个函数有两个参数：`begin` 和 `end`，表示开始和结束。如字符串 $s$ 为 `Luogu`，那对于一下几个代码就有不同的结果：

- `reverse(s.begin(),s.end());`，结果为 `ugouL`；
- `reverse(s.begin()+1,s.end());`，结果为 `Lugou`；
- `reverse(s.begin(),s.end()-2);`，结果为 `ouLgu`；
- ……

AC 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	string s;
   	while(cin>>s)
   	{
     	reverse(s.begin(),s.end());
    	cout<<s<<endl;
   	}
	return 0;
}
```

## 作者：zmza (赞：4)

这道题目要我们将所有输入的串全部翻转。

怎样翻转呢？

我们可以将从前往后数的第 $i$ 个和从后往前数的第 $i$ 个交换。

比如说 $olleh$ 翻转时， $h$ 和 $o$ 交换， $e$ 和 $l$ 交换。于是原串就变成了 $hello$ 。

代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
char a[105];
void reverse()
{
	for (int i = 0; i < strlen(a) / 2; i++)
		swap(a[i], a[strlen(a) - i - 1]);
	for (int i = 0; i < strlen(a); i++)
		printf("%c", a[i]);
	puts("");
}
signed main()
{
	while (cin >> a)
		reverse();
	return 0;
}
```

## 作者：thomas_zjl (赞：3)

### 题目描述。

输入几个单词，把它们翻转后输出。

### 代码。

题目是简单地，但对于代码来说，有些小点。

- 输入时。

```cpp
while(cin>>a)
```

这样可以以空格为一个单位，来对每个字符串进行划分。

- 在翻转时。

```cpp
for(int i=a.length()-1;i>=0;i--)
	cout<<a[i];
```
注意：i 一开始要为 ```a.lenghth()-1```，在 $i=0$ 时结束,因为输入是从下标为零开始输入。

如果要改变，可以用 ```cin>>a+1```或```scanf("%s",a+1)```来改变初始下标的位置。

- 别忘了换行。

接下来就是代码了。

```
#include<bits/stdc++.h>
using namespace std;
int main(){
	string a;
	while(cin>>a){
		for(int i=a.length()-1;i>=0;i--)
			cout<<a[i];
		cout<<endl;
	}	
	return 0;
}

```

在 C++ STL 里面有一个 `reverse` 的函数。

使用方法如下：

`reverse(_begin,_end);//_begin 是数组或字符串开头，_end 同理`

```cpp
#include<bits/stdc++.h>
using namespace std;
string word;
signed main(){
	while(cin>>word){//读入
		reverse(word.begin(),word.end());//反转
		cout<<word<<'\n';//输出
	}
	return 0;
}
```

# 最高分数的学生姓名

## 题目描述

输入学生的人数，然后再输入每位学生的分数和姓名，求获得最高分数的学生的姓名。

## 说明/提示

hanmeimei 的分数最高 $99$ 分。

## 样例 #1

### 输入

```
5
87 lilei
99 hanmeimei
97 lily
96 lucy
77 jim```

### 输出

```
hanmeimei
```

# 题解

## 作者：TensorFlow_js (赞：25)

# 题目分析
依题意一步一步模拟即可。

模拟过程：用一个 `for`循环，边输入边判定分数是否大于目前的最高分数，如果是则更新目前的最高分数。
# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;cin>>n;//输入题目中的N
    int max=-1;string ans;//max为目前的最高分数,ans为目前最高分数的学生名字
    for(int i = 0;i < n;i ++){
        int x;string s;cin>>x>>s;//输入
        if(x>max)max=x,ans=s;//如果这位学生的分数大于目前的最高分数,则更新目前的最高分数与学生名字
    }
    cout<<ans;//输出最高分数的学生名字
    return 0;
}
```

## 作者：Ryan_Yu (赞：10)

此题是大号的结构题练习题。

读入后用`sort`排序，分数多的靠前（需编写`cmp`程序）

最后输出排名最靠前的即可。

```cpp
#include<bits/stdc++.h>
using namespace std;
struct node
{
    string name;int score;
}a[105];
int n,i;
bool cmp(node c,node b)
{
    return c.score>b.score;
}//分数大的靠前
int main()
{
	cin>>n;
    for(i=1;i<=n;i++)
    {
        cin>>a[i].score>>a[i].name;
    }
    sort(a+1,a+1+n,cmp);
    cout<<a[1].name;//输出分数最靠前的
    return 0;
}
```

## 作者：lighthouse (赞：8)

从题面分析：  
本题可以用一个结构体数组来储存学生的信息。  
定义一个 $maxn$ 来储存最大值，用循环读入时，用`max`函数判断，如果读入的这个分数比 $maxn$ 大，就更新当前值。  
最后用循环扫一遍数组，如果学生的分数与 $maxn$ 相等就输出名字，然后直接结束程序。  
看数据范围：  
$n$ 与学生成绩都在`int`范围之间，名字可以用`string`定义。
代码：  
```cpp
#include <bits/stdc++.h>

using namespace std;

struct student{//定义结构体
	string name;
	int score;
}a[110];

int main(){
	int n;
	cin >> n;//输入n
	int maxn = 0;//定义最大值储存器maxn
	for(int i = 1;i <= n;i++){
		cin >> a[i].score >> a[i].name;//输入学生信息
		maxn = max(maxn, a[i].score);//存最大值
	}
	for(int i = 1;i <= n;i++){//循环扫描数组
		if(maxn == a[i].score){//找到了
			cout << a[i].name;//输出名字
			return 0;//直接结束
		}
	}
	return 0;
}
```

## 题目分析
此题只用多加一个二维数组，先通过循环求到最大值的下标，然后再将对应这个下标的那一行的学生的名字，注意最大值最好初始化成负一。
## 代码实现
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[110];
string s[110];
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a[i];
		cin>>s[i];
	}
	int cmp,maxi=-1;
	for(int i=1;i<=n;i++)
		if(a[i]>maxi){
			cmp=i;//下标
			maxi=a[i];//最大值
		}
	for(int i=0;i<s[cmp].size();i++)
		cout<<s[cmp][i];
	return 0;
}
```

## 作者：happybob (赞：7)

本题可以写 `cmp` 函数，但是这里教大家写运算符重载。

运算符重载可以使两个结构体 $a, b$ 做到像 `int` 一样可以判断 `a < b`，而不用 `cmp`。通常重载运算符是在结构体内定义（也可以用友元类，两个不同结构体的关系），定义形式如：

```cpp
struct Node // class也可以
{
	int a;
	……; // 成员
	bool operator<(const Node& a) const // bool 也可以是其他的类型
    {
		return ……; // 两个结构体之间的大小关系判断式
	}
}

```

代码：

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Node
{
    string s;
    int score;
    bool operator<(const Node& x) const
    {
        return score < x.score;
    }
    Node();
};

Node::Node() // 构造函数
{
    score = 0;
    s = "";
}

Node a[105];

int main()
{
    ios::sync_with_stdio(false);
    int n;
    cin >> n;
    for (auto& i : a) // 用c++11或以上可以提交，本地测试一定输入完要crtl+z（vs和dev都是这样）
    {
        cin >> i.score >> i.s;
    }
    sort(a + 1, a + n + 1);
    cout << a[n].s << endl;
    return 0;
}
```

# [蓝桥杯青少年组省赛 2024] 数字交换

## 题目描述

前导 $0$：整数前对数值无影响的 $0$。

例如：
- $0201$ 去除前导 $0$ 后为 $201$；
- $00321$ 去除前导 $0$ 后为 $321$。

给定一个正整数 $n$，请将 $n$ 的最高位与最低位的数字进行交换，并输出交换后的结果。如果交换后的结果有前导 $0$，去除前导 $0$ 后再输出结果。

例 1：$n = 173$，将 $173$ 的最高位 $1$ 与最低位 $3$ 交换，交换后的结果为 $371$；

例 2：$n = 10200$，将 $10200$ 的最高位 $1$ 与最低位 $0$ 交换，交换后的结果为 $00201$，结果有前导 $0$，去除前导 $0$ 后的结果为 $201$。

## 样例 #1

### 输入

```
173```

### 输出

```
371```

## 题意描述

给定数字 $n$，交换 $n$ 的最高位和最低位，并去除前导 $0$。

## 思路

字符串函数！

分步解决：

- 交换 $n$ 的最高位和最低位：利用 `swap` 函数交换最高位和最低位。即 `swap(n[0],n.back())`。

- 去除前导零：利用 `stoi` 函数将交换后的字符串转为整型数字，而整型会自动去除前导零，就不用我们自己手写啦！

## AC 代码

```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	string a;
	cin>>a;
	swap(a[0],a.back());
	cout<<stoi(a);
	return 0;
}
```

### 解题思路

按照题目要求进行模拟即可。

由于整数进行交换两个位置上的数字较为麻烦，所以我们可以选用字符串进行处理。

交换完毕之后，我们可以去找第一个非零位，那么从这个位置上输出即表示去除前导零的结果。

### AC_Code

```cpp
#include <iostream>

using namespace std;

int main()
{
    string str;
    cin >> str;

    swap(str[0], str.back());

    int k = 0;
    while (k < str.size() - 1 && str[k] == '0')
        k ++;
    
    cout << str.substr(k) << endl;
    
    return 0;
}
```

## 作者：LINYUHENG2 (赞：1)

## 思路
用数字来处理比较麻烦，可以采用 `string` 来读入，这样用 `swap` 函数就可以进行交换。

对于删去前导 $0$，可以将 $n$ 翻转一下，不断删去末尾的 $0$ 直到末尾不是 $0$，再翻转回来。
## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    string n;
    cin>>n;
    swap(n[0],n.back());
    reverse(n.begin(),n.end());
    while(n.back()=='0') n.pop_back();
    reverse(n.begin(),n.end());
    cout<<n;
    return 0;
}
```

## 作者：yyycj (赞：1)

## 题目简述
给定数字 $n$，要求交换 $n$ 的最高位和最低位，并去除前导 $0$。

## 主要思路
用数字交换 $n$ 的最高位和最低位会比较麻烦，所以可以采用 `string` 来读入，这样用一个 `swap` 就可以完成交换。

对于去除前导 $0$，可以先将 $n$ 翻转，不断判断 $n$ 的末尾是否为 $0$，然后删去，直到末尾不为 $0$，再翻转回来即可。

## AC Code
```cpp
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;

int main() {
	string n; cin >> n;
	// ----------------------------
	swap(n[0], n.back());
	reverse(n.begin(), n.end());
	while (n.back() == '0') n.pop_back();
	reverse(n.begin(), n.end());
	// ----------------------------
	cout << n;
	return 0;
}
```

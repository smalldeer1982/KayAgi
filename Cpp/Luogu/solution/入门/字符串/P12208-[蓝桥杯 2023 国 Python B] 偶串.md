# [蓝桥杯 2023 国 Python B] 偶串

## 题目描述

小蓝特别喜欢偶数，当他看到字符串时，他总数要检查一下是不是每种字符都是出现偶数次。给定一个字符串，请帮助小蓝检查一下该字符串是否满足要求。

## 说明/提示

### 评测用例规模与约定

- 对于 $50\%$ 的评测用例， $1 \leq$ 字符串长度 $\leq 1000$；
- 对于所有评测用例，$1 \leq$ 字符串长度 $\leq 10^{6}$。

## 样例 #1

### 输入

```
banana```

### 输出

```
NO```

## 样例 #2

### 输入

```
bbnana```

### 输出

```
YES```

# 题解

## 作者：Suwenluo (赞：3)

首先明确题目内容，我们需要判断字符串中的每种字符是否都是出现偶数次，那我们只需用一个桶记录字符串中每种字母出现的次数，然后在逐一判断每个在字符串中出现的字母是否都出现了偶数次就好了。
```python
l=[0]*1000 #Python中的数组初始化方式，为避免数组太小，我索性开大了一点。
a=input() #简洁的输入。
for i in a: #遍历每个元素（这里i就是元素）。
    l[ord(i)]+=1 #ord就是强行把单字符转为int类型。在桶中记录。
for i in range(1,150): #直接把范围开大搜。
    if (l[i]%2==1): #如果出现了奇数次。
        print("NO") #那当然是NO。
        quit() #独特的退出方式。
print("YES") #没退出就补输出。
```

## 作者：Lysea (赞：2)

看到奇偶次数，很容易想到异或哈希。

具体来讲，我们可以给 $a$ 到 $z$ 每个字母定一个随机的哈希值，然后利用 $x\oplus x=0$ 的性质，来判断每个字母出现了奇数还是偶数次。

举个例子，假设 $a$ 的哈希值为 $9283$，$b$ 的哈希值为 $2395$，$c$ 的哈希值为 $8374$。

那么对于字符串 `abab`，最终得到的值就为 $9283\oplus2395\oplus 9283\oplus 2395=0$。

而对于字符串 `abac`，最终得到的值就为 $9283\oplus 2395\oplus 9283\oplus 8374=2395\oplus 8374 \ne 0$。

当然这种做法无法保证绝对的正确性，例如若 $b$ 和 $c$ 的哈希值不小心相同了，$abac$ 也会被判定为符合要求。

不过在采用随机赋值的情况下错误概率可以忽略不计，这就是哈希算法的魅力。


```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;
mt19937_64 rnd(time(NULL));
int v[205],a;
signed main(){
    for(int i='a';i<='z';i++) v[i]=rnd();
    string s; cin>>s;
    for(int i=0;i<s.size();i++) a^=v[s[i]];
    cout<<(a?"NO":"YES"); return 0;
}
```

## 作者：easy42 (赞：2)

首先，异或有一个性质，异或两次就会变为原数。也就是说，我们开一个数组，对应每个字母，遇到一个字母，就把它对应的数组异或 $1$。

最后看它是否等于零就好了。

下面是代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int a[26];
int main(){
    char c;
    while(cin>>c){
        a[c-'a']^=1;
    }
    bool flag=true;
    for(int i=0;i<26;i++){
       if(a[i]) flag=false;
    }
    if(flag) puts("YES");
    else puts("NO");
    return 0;
}
```

## 作者：Show_me (赞：1)

### 理解题意
题目让我们检查一下一个字符串是不是每种字符都是出现偶数次，那我们就可以统计每种字符出现的次数，再来判断它的次数是不是都是偶数。（详细请见代码，已注释）
### 代码

```python
s=input()#输入字符串
d=['a','b','c','d','e','f','g','h','i','j','k','l','m','n','o','p','q','r','s','t','u','v','w','x','y','z']#26字母表，用于核对字母的编号
c=[0]*26#统计它们字母的个数，创建26个
for i in s:#循环
    for j in range(26):#枚举
        if i==d[j]:#如果字母和这个字母表里面的一样
            c[j]+=1#它对应的编号，也就是他对应的字母，的个数+1
f=0#标记，看是否达到要求
for i in c:#枚举
    if i%2!=0:#如果有一个不是偶数
        f=1#标记
        break#退出
if f==0:#如果他都是偶数
    print('YES')#输出
else:#反之
    print('NO')#输出
```

## 作者：niuniudundun (赞：1)

[题目传送门](https://www.luogu.com.cn/problem/P12208)

定义输入的字符串叫做 $s$，因为我们检查一下是不是每种字符都是出现偶数次，这种最好是使用桶或 `map`，选择码量较小的桶。将每个 $s$ 中的字符减 `a` 后放入桶中，接着判断每个桶内字符是否为偶数。因为桶记录的是字符出现次数，是偶数说明字符出现偶数次。

对 $s$ 遍历是循环 $s$ 的长度次，设它为 $\operatorname{len}(s)$，以及对桶的遍历是 $26$ 次，记为 $A$，那么复杂度是 $O(\operatorname{len}(s)+A)$。如果使用 `map` 会有 $O(\log \operatorname{len}(s))$ 的常数。

```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
int a[26];
int main(){
	cin>>s;
	int l=s.length();
	for(int i=0;i<l;i++){
		a[s[i]-'a']++;
	}
	for(int i=0;i<26;i++){
		if(a[i]%2==1){
			cout<<"NO\n";
			return 0;
		}
	}
	cout<<"YES\n";
	return 0;
}
/*
*/
```

## [题目传送门](https://www.luogu.com.cn/problem/P12208)

用桶数组记录每一个字符的出现次数，最后遍历字符串，如果有一个字符出现的次数不是偶数就直接输出 $\texttt{NO}$，如果遍历完了，都是偶数次就输出 $\texttt{YES}$。

用 C++14 的 `auto` 遍历会方便一些。

### $\texttt{Code}$

```cpp
#include<bits/stdc++.h>
using ll = long long;
using namespace std;

const int N = 1e5 + 5;
string s;
int a[N];

signed main() {
	ios::sync_with_stdio(0);
	cin.tie(0), cout.tie(0);
	
	cin >> s;
	
	for (auto c : s) a[c]++;
	for (auto c : s) {
		if (a[c] % 2 != 0) {
			cout << "NO" << '\n';
			return 0;
		}
	}
	
	cout << "YES" << '\n';
	return 0;
}
```

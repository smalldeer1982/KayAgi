## 题目描述

作为一名守法好谷民，我们当然要支持正版的产品。

现在有 $T$ 个不一定标准的洛谷比赛名（仅存在大小写字母和数字），你每次操作可以交换任意两个字符，使得不存在 `LGR` 子串与 `CSP` 子串共存的情况。

例如 `LGRCSP` 可以经过一次操作可变为 `LGRSCP`。

请输出最小操作次数。

## 说明/提示

**本题采用捆绑测试。**

令 $t$ 为字符串 $s$ 的长度。

| 子任务编号 | $t$ | 分值 |
| :-----: | :-: | :-----: |
|$1$|$\le 10^3$|$40$|
|$2$|$\le 10^5$|$60$|

对于 $100\%$ 的数据，保证 $1\le T\le 10$，$1\le t\le10^5$。

LGR201Div3CSP2024dierlunmoni

LGRLGR114514Div2CSPCSP2025dierlunmoni```

### 输出

0
1```

# 题解

## 作者：Anemones (赞：13)

设 $s$ 中有 $x$ 个 `CSP` 子串和 $y$ 个 `LGR` 子串，那么答案即为 $\lceil\min(\frac{x}{2},\frac{y}{2})\rceil$，即只更改一种类型的子串。

对于方案，以更改 `LGR` 为例，若为偶数，则两两配对交换，比如 `LGRLGR` 变为 `GGRLLR`；若为奇数，则将其中一个 `LGR` 变为 `LRG`。

```cpp
//luogu uid:Anemones 736184
#include<bits/stdc++.h>
using namespace std;
signed main(){
    int T;
    cin>>T;
    while(T--){
        string s;
        cin>>s;
        int n=s.size();
        int t1=0,t2=0;
        for(int i=0;i<n-2;i++){
            if(s.substr(i,3)=="LGR") t1++;
    		if(s.substr(i,3)=="CSP") t2++;
        }
        cout<<min(ceil(t1/2.),ceil(t2/2.))<<endl;
    }
    return 0;
}
```

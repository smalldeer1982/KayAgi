## 题目描述

Mirko 的奶奶至今仍在使用一部古老的脉冲拨号电话机，其拨号盘如下图所示：

![](https://cdn.luogu.com.cn/upload/image_hosting/44fn1zm8.png)

每当我们要拨一个数字时，需要顺时针旋转拨号盘，直到选中的数字到达金属挡片（指停）。然后松开拨号盘，等待其返回原位后，才能拨下一个数字。在我们这个追求即时满足的现代社会里，这个拨号盘的回弹时间往往比我们的耐心还要长。更具体地说，拨数字 $1$ 需要 $2$ 秒，拨更大的数字则每增加一个数字（从 $1$ 到所拨数字），总时间再增加 $1$ 秒（如上图所示）。

Mirko 的奶奶通过记忆单词来记住电话号码，每个字母对应拨号盘上写有该字母的数字。例如，字母 S 对应数字 $7$。比如，单词 **UNUCIC**$^{1}$ 对应的号码是 $868242$。你的任务是，对于给定的单词，计算拨完该单词所需的总时间。

$^{1}$：在克罗地亚语中意为“小孙子”。

## 说明/提示

第一个样例说明：对应的电话号码为 $92$，拨号总时间为 $10 + 3 = 13$ 秒。

# [题目传送门](https://www.luogu.com.cn/problem/P13419)

***1、题目大意***

这题要求我们用古老的脉冲拨号电话机来拨字母，每个字母对应拨号盘上写有该字母的数字。每当我们要拨一个数字时，需要顺时针旋转拨号盘，直到选中的数字到达金属挡片（指停）。然后松开拨号盘，等待其返回原位后，才能拨下一个数字。

***2、思路***

我们可以先用 map 把拨每个字母所需的时间先记录下来，枚举字符串的每个字母，并用 sum 累加起来。

***3、代码***


```cpp
#include<bits/stdc++.h>
//#pragma GCC optimize(2,"Ofast","inline")
using namespace std;
map<char,int>a;
string s;
int sum;
signed main(){
	//处理拨A~Z的次数 
	for(char i='A';i<='C';i++) a[i]=3;
	for(char i='D';i<='F';i++) a[i]=4;
	for(char i='G';i<='I';i++) a[i]=5;
	for(char i='J';i<='L';i++) a[i]=6;
	for(char i='M';i<='O';i++) a[i]=7;
	for(char i='P';i<='S';i++) a[i]=8;
	for(char i='T';i<='V';i++) a[i]=9;
	for(char i='W';i<='Z';i++) a[i]=10; 
	//读入，用sum累加 
	getline(cin,s);
	for(int i=0;i<s.size();i++)
	sum+=a[s[i]];
	cout<<sum;
}


```

题目：P13419
## 题面大意：
输入一个字符串，求字符串的“字符和”：\
遍历字符串的每一个字符，
- 如果字符在 $\texttt A\sim\texttt C$，将字符和增加 $3$；
- 如果字符在 $\texttt D\sim\texttt F$，将字符和增加 $4$；
- 如果字符在 $\texttt G\sim\texttt I$，将字符和增加 $5$；
- 如果字符在 $\texttt J\sim\texttt L$，将字符和增加 $6$；
- 如果字符在 $\texttt M\sim\texttt O$，将字符和增加 $7$；
- 如果字符在 $\texttt P\sim\texttt S$，将字符和增加 $8$；
- 如果字符在 $\texttt T\sim\texttt V$，将字符和增加 $9$；
- 如果字符在 $\texttt W\sim\texttt Z$，将字符和增加 $10$。

## 解题流程：
遍历字符串的每一个字符，然后判断字符所在的区间并递加字符和，最后输出字符和就可以了。

## 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
char c,s[20];
int a=0;
int main()
{
    cin>>s;//输入：输入字符串
    for(int i=0;i<strlen(s);i++)//遍历：遍历字符串
    {
        /*判断：分别判断每一个区间*/
        c=s[i];
        if('A'<=c && c<='C') a+=3;
        if('D'<=c && c<='F') a+=4;
        if('G'<=c && c<='I') a+=5;
        if('J'<=c && c<='L') a+=6;
        if('M'<=c && c<='O') a+=7;
        if('P'<=c && c<='S') a+=8;
        if('T'<=c && c<='V') a+=9;
        if('W'<=c && c<='Z') a+=10;
    }
    cout<<a;//输出：输出“字符和”
    return 0;
}
```

# 主要思路

这道题，我们不难发现：需要的时间就是每个字母对应的数字 $+1$，然后求和即可。我们可以打表求出每个字母对应的数字，然后拿一个 $ans$ 累加这些字母对应的数字 $+1$，输出 $ans$ 即可。

# [AC](https://www.luogu.com.cn/record/226486894) code
```cpp
#include<iostream>
using namespace std;
int a[26]={2,2,2,3,3,3,4,4,4,5,5,5,6,6,6,7,7,7,7,8,8,8,9,9,9,9};
int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr),cout.tie(nullptr);
    string s;
    cin>>s;
    int ans=0;
    for(int i=0;i<s.size();i++){
        ans+=a[s[i]-'A']+1;
    }
    cout<<ans;
    return 0;
}
```

```cpp
a['Z'+5] = {};
for(int i='A'; i<'S'; i+=3)
    a[i] = a[i+1] = a[i+2] = (i-'A')/3+2;
```

## 作者：b__b (赞：2)

水。

我们先考虑如何将字母转成数字，设当前字符在英语字母表中的编号减 $1$ 为 $c$，数字为 $x$：
- 当 $0 \leq c \leq 17$（即字母在 `A` 到 `R` 范围内）时，$x=\lfloor \frac{c}{3} \rfloor+1$；
- 当 $17 < c \leq 24$（即字母在 `S` 到 `Y` 范围内）时，$x=\lceil \frac{c}{3} \rceil+1$；
- 当 $c=25$ 时，$x=9$。
> 拨数字 $1$ 需要 $2$ 秒，拨更大的数字则每增加一个数字（从 $1$ 到所拨数字），总时间再增加 $1$ 秒。

因此拨一个数字所需要的时间就是 $x+1$ 秒。

实现时遍历字符串的每一个字符即可。
```cpp
#include <cstdio>
#include <cctype>
char s[114];
int main() {
    fgets(s, 114, stdin);
    int ans = 0;
    for (int i = 0; isalpha(s[i]); ++i) {
        int si = s[i] - 'A';
        if (si <= 'R' - 'A') ans += si / 3 + 3;
        else if (si <= 'Y' - 'A') ans += (si + 2) / 3 + 2;
        else ans += 10;
    }
    printf("%d", ans);
}
```

# 思路
因为拨动数字 $1$ 需要 $2$ 秒，从数字 $1$ 到别的数字只需要 $1$ 秒，所以拨动每个数字键需要此数字加 $1$ 秒。

所以进行枚举，用一些判断，求出字符串中每个字母对应的数字，再将此字母对应的数字加 $1$ 就是拨动此字母需要的时间。算出字符串中每一个字母需要拨动的时间，加起来就是总共要拨动的时间。
# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin>>s;
    int sum=0;
    for(int i=0;i<=s.length();i++){
        if(s[i]=='A'||s[i]=='B'||s[i]=='C')sum+=3;
         if(s[i]=='D'||s[i]=='E'||s[i]=='F')sum+=4;
         if(s[i]=='G'||s[i]=='H'||s[i]=='I')sum+=5;
         if(s[i]=='J'||s[i]=='K'||s[i]=='L')sum+=6;
         if(s[i]=='M'||s[i]=='N'||s[i]=='O')sum+=7;
         if(s[i]=='P'||s[i]=='Q'||s[i]=='R'||s[i]=='S')sum+=8;
         if(s[i]=='V'||s[i]=='T'||s[i]=='U')sum+=9;
         if(s[i]=='X'||s[i]=='Y'||s[i]=='Z'||s[i]=='W')sum+=10;
        
    }
    cout<<sum<<endl;
    return 0;
}
```

```cpp
}
```

# [信息与未来 2023] 精密计时

## 题目描述

小小有一个非常精密的计时器，每秒可以计数 100 次 (两个连续的时钟计数之间恰好间隔百分之一秒)。例如，以下是从 $13:01:02.37$ 到 $13:01:03.01$ 的计时过程：
```plain
13:01:02.37 (13 时 1 分 2 秒.37)
13:01:02.38
13:01:02.39
...
13:01:02.98
13:01:02.99 (13 时 1 分 2 秒.99)
13:01:03.00 (13 时 1 分 3 秒.00)
13:01:03.01
```
小小记录了计时器上的两个时刻，你能帮助他计算这两个时刻之间经过了多少个 “百分之一秒”吗？

## 说明/提示

对于 $100\%$ 的数据，两个时刻都来自 $24$ 小时制的同一天 （$00:00:00.00$ 到 $23:59:59.99$），且保证后一个时刻晚于前一个时刻。

>本题原始满分为 $15\text{pts}$。

## 样例 #1

### 输入

```
01:02:34.56 01:03:00.10```

### 输出

```
2554```

## 样例 #2

### 输入

```
00:00:00.00 23:59:59.99```

### 输出

```
8639999```

# 题解

## 作者：hahaloo (赞：11)

### 思路
我们首先要把两个时间的转化成**百分之一秒**，直接**相减**就可以得出答案。那怎么将时间化成百分之一秒呢？显然我们只需要将时分秒乘以**相应的进率**即可。
## 参考代码
```cpp
#include <iostream>
#include <iomanip> // 用于控制输出流格式
using namespace std;

int main()
{
    // 读入两个时间
    int h1, m1, s1, ms1, h2, m2, s2, ms2;//分别代表时分秒和百分之一秒 
    char c; // 用于读取冒号和小数点
    cin >> h1 >> c >> m1 >> c >> s1 >> c >> ms1;
    cin >> h2 >> c >> m2 >> c >> s2 >> c >> ms2;

    // 时间转换为百分之一秒
    long long t1 = h1 * 360000 + m1 * 6000 + s1 * 100 + ms1;
    long long t2 = h2 * 360000 + m2 * 6000 + s2 * 100 + ms2;

    // 计算两个时间的差值
    cout << t2 - t1 << endl;

    return 0;
}
```

## 本题思路：
这道题十分简单，其实只要知道时分秒之间是六十的转化，然后我们直接算出每一位的差，把这一位的差去乘以六十放倒后一位的计算，注意一下秒的转化是 100，这样最后输出秒的差就可以了。
## 本题代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	long long x=0,y=0,z=0,o=0,x1=0,y1=0,z1=0,o1=0;
	for(int i=1;i<=11;i++)
	{
		char e;
		cin>>e;
		if(i==1 || i==2) x=x*10+e-'0';
		if(i==4 || i==5) y=y*10+e-'0';
		if(i==7 || i==8) z=z*10+e-'0';
		if(i==10 || i==11) o=o*10+e-'0';
	}
	for(int i=1;i<=11;i++)
	{
		char e;
		cin>>e;
		if(i==1 || i==2) x1=x1*10+e-'0';
		if(i==4 || i==5) y1=y1*10+e-'0';
		if(i==7 || i==8) z1=z1*10+e-'0';
		if(i==10 || i==11) o1=o1*10+e-'0';
	}	
	y1+=(x1-x)*60;
	z1+=(y1-y)*60;
	o1+=(z1-z)*100;
	cout<<o1-o;
	return 0;
}

```

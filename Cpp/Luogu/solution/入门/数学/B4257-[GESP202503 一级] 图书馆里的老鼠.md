# [GESP202503 一级] 图书馆里的老鼠

## 题目描述

图书馆里有 $n$ 本书，不幸的是，还混入了一只老鼠，老鼠每 $x$ 小时能啃光一本书，假设老鼠在啃光一本书之前，不会啃另一本。请问 $y$ 小时后图书馆里还剩下多少本完整的书。

## 说明/提示

对于所有测试点，保证 $1\leq n,x,y\leq 1000$，保证 $y$ 小时后至少会剩下⼀本完整的书。

## 样例 #1

### 输入

```
10
2
3```

### 输出

```
8```

## 样例 #2

### 输入

```
5
2
4```

### 输出

```
3```

# 题解

本题考察分支结构。

我们要解决的问题是：$y$ 小时后，还剩多少本完整的书。为了知道剩下多少，我们得先知道老鼠**啃了或者开始啃了多少本书**。

以样例 1 为例，图书馆里有 $n=10$ 本书，老鼠啃完一本需要 $x=2$ 小时，时间过去了 $y=3$ 小时。这意味着老鼠完全啃了一本书，又再开始啃第二本书，所以只有 $8$ 本是完整的；

以样例 2 为例，图书馆里有 $n=5$ 本书，老鼠啃完一本需要 $x=2$ 小时，时间过去了 $y=4$ 小时。这意味着老鼠恰好啃完了两本书，因此还有 $3$ 本是完整的。

我们可以用总时间 $y$ 除以啃一本书需要的时间 $x$ 得知老鼠能**完整啃几本书**。比如 $y=3, x=2$，那么 $3 \div 2 = 1$ 余 $1$。老鼠完整啃掉了 $1$ 本书。如果说除不尽有余数，那么说明老鼠在啃下一本书。我们假设变量 `d` 是啃过的书的数目，可以写出：

- 如果 `y % x == 0` （$y$ 能被 $x$ 整除），那么 `d = y / x`。
- 如果 `y % x != 0` （$y$ 不能被 $x$ 整除），那么 `d = y / x + 1`。

最后，根据 `n - d` 即可计算出答案了。参考代码：

```cpp
if (y % x == 0) {
    // 如果 y 小时刚好够啃完整数本书
    d = y / x;
} else {
    // 如果 y 小时啃完一些书后还有剩余时间，老鼠会开始啃下一本
    d = y / x + 1;
}
cout << n - d << endl;
```

拓展思考：如果不想使用分支结构，可以使用 `d = (y + x - 1) / x;` 计算得到一样的值。想一想，这是为什么呢？

# [科大国创杯小学组 2025] 饼干

## 题目背景

Subtask 0 为民间数据，Subtask 1 为官方数据。

## 题目描述

小可可有一个 $1 \times n$ 的长方形盘子，现在他想在里面放一些 $1 \times m$ 的饼干，饼干不能重叠，他想知道他最多能放多少块饼干？

## 说明/提示

### 约定和数据范围

- 数据点 $1 \sim 3$，$1 \leq n \leq 10^9$，$m = 1$。
- 数据点 $4 \sim 5$，$1 \leq n, m \leq 10^9$，$n = m$。
- 数据点 $6 \sim 10$，$1 \leq n, m \leq 10^9$。

## 样例 #1

### 输入

```
19 6```

### 输出

```
3```

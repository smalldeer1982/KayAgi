# [P12868 [蓝桥杯 2025 国 Python A] 最大周长](https://www.luogu.com.cn/problem/P12868)
## 题目介绍：
一个长方形面积是 $2025$，长和宽均是正整数，求周长最大是多少。

## 分析做法：
**红题**难度，**枚举**即可过。

假设一条边是 $i$，另外一条边是 $j$，范围是从 $1$ 至 $2025$，便可以枚举所有情况，如果发现 $i$ 乘 $j$ 的值为 $2025$，便和之前的最大周长比较，更新最大值。

因为只要答案，不考虑时间复杂度。

最终答案 $4052$。

奉上代码：
```cpp
#include <bits/stdc++.h>
using namespace std;
#define int long long 
signed main() {
	int maxn = INT_MIN;//初始化最大值
    for (int i = 1; i <= 2025; i++) {//枚举
		for (int j = 1; j <= 2025; j++) {
			if (i * j == 2025) {//判断条件
				maxn = max(maxn, (i + j) * 2);//和最大值比较。
			}
		}
	}
	cout << maxn << endl;
    return 0;
}
```

## 作者：Tsz1024_AK (赞：1)

### 解析思路

求一个面积为 $2025$ 的长方形周长最大是多少。

周长等于长方形的长宽之和乘以二。

因为一个长方形的长与宽差得越大，它的周长也越大。那么很显然当长方形的长为 $2025$，宽为 $1$ 时，周长之和最大。

举个例子，$2025$ 是 $45$ 的平方，而正方形是长方形中特殊的一种，所以长（正）方形的长和宽（边长）都是 $45$ 是可行的，但其周长仅有 $45 \times 4 = 180$。反观长为 $2025$，宽为 $1$ 的长方形，其周长便是 $(1+2025) \times 2 = 4052$.

### 结论答案

根据上面的解析得出答案为 $4052$，下面是 Python 和 C++ 的输出答案的代码：

```python
print(4052)
```

```c++
#include<bits/stdc++.h>
using namespace std;
int main(){
	cout<<<4052;
	return 0;
}
```

```cpp
int ans = 0, n = 2025;
for (int i = 1; i * i <= n; i++)
    if (!(n % i))
    {
        int j = n / i, C = 2 * (i + j);
        ans = max(ans, C);
    }
cout << ans;
```

答案是 $\bf{4052}$。

# 思路
简单数学。我们都知道**和同近积大**，即当两个数的和确定时，两个数的差越小，两个数的积越大。  
证明：  
设正整数 $n$ 和 $m$（$n \ge m$），使 $n+m=x$，且 $n-m$ 的值最小。则 $mn\ge (n+k)(m-k)$。  


知道了这个结论，发现积是确定的，那么就可以用倒推法。所以我们就有**积同远和大**。  

回到题目，我们知道了面积，就可以推出长、宽在最优情况下分别为 $2025$ 和 $1$。之后套长方形面积公式就行了。

最终答案为：$4052$

我们可以枚举长方形的长和宽，检查是否乘积是否是 $2025$ 这个数字，用公式 $C=2(a+b)$ 计算周长，取最大值即为答案。

上代码。

```cpp
#include <bits/stdc++.h>
using namespace std;
int mx=-1;
signed main(){
	for(int i=1;i<=2025;i++){
		for(int j=1;j<=2025;j++){
			if(i*j==2025){
				mx=max(mx,(i+j)*2);
			}
		}
	}cout<<mx;
	return 0;
} 
```

# 主要思路

首先，我们需要知道：几个面积相等的长方形，长和宽差得越大，周长越长。我们都知道几个周长相等的长方形长和宽越接近面积越大，[证明](https://mp.weixin.qq.com/s?__biz=MzAxNTY1NTM4MQ==&mid=2651626602&idx=1&sn=4080610d721284fbacc2177b7e65726e&chksm=8078f4d0b70f7dc633b095ffef35774a3564d9769f5278fcef82024d9e107b6ef49d0ba6c269&scene=27)，那么我们反推一下，要保持面积相等，那么长和宽差得越远是不是就需要用更长的周长来保持面积的相等？知道了这一点，题目就简单了。我们知道了面积是 $2025$，既然长宽都是正整数，那么最小的正整数就是 $1$，这就是长方形的宽，那么长方形的长就是 $2025\div1=2025$ 了，周长就出来了：$\left(1+2025\right)\times2$，输出即可。

## 题目分析
希望审核大大给过。

这是一道数学题。在上小学的时候，我们学过一句话，叫：**和同近积大，积同近和小**，意思就是如果两个数和相同，那么越接近，积越大；两个数积相同，那么越接近，和越小。我们要让两数和最大，就要让两数尽可能的远，也就是当两数为 $1$ 和 $2025$ 时，答案就为 $(1 + 2025) \times 2 = 4052$ 为最大值，输出即可。

下面的代码我是用循环暴力做的，大家也可以直接输出 $4052$。
## 题目[代码](https://www.luogu.com.cn/record/220693678)

```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long
#define uint unsigned long long
#define speed ios::sync_with_stdio(0),cin.tie(0), cout.tie(0);
signed main() {
	speed
	int maxx=0;
	for(int i=1;i<=2025;i++){
		if(2025%i==0){
			int k=((2025/i)+i)*2;
			maxx=max(maxx,k);
		}
	}

}
```

总面积较小，可以直接暴力枚举长，反求宽，记录最大值即可。    
直接上代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	cin.tie(0)->sync_with_stdio(0);
	cout.tie(0);
	int maxx = -1;
	for(int i = 1;i*i<=2025;i++){//i*i为剪枝，因为i>sqrt(2025)就会重复枚举
		if(2025%i)continue;//注意长宽是整数
		maxx = max(maxx,2*(i+2025/i));//周长公式
	}
	cout<<maxx;
	return 0;
}
```

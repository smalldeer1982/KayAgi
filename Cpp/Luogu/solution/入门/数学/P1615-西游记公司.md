# 西游记公司

## 题目背景

一道极其无厘头的题目。

## 题目描述

事情是这样的：  

西游记中的孙沙猪三徒弟在西天取经之后开始进入厦门大学经贸系学习经济，在 $1$ 个小时的学习后，他们用特殊手段毕业了。

然后，他们创办了三个公司：“花果山生态旅游山庄集团公司” “高老庄猪肉美食城有限公司” 和 “流沙河轮渡有限公司”。虽然这三家公司出自 “经卷出版社” 社长唐三藏的徒弟们，但是每家公司都仍然保持着 $0$ 元以下的财政收入。  

于是，他们想到了一个无聊的方法：向别人偷！其中，孙悟空偷猪八戒的平底锅之前，猪八戒可以一直抢沙和尚的笔记本。

现在，作为猪八戒手下首席智士的你，要帮助既聪明又愚蠢的猪八戒用那种不正当的手段挽救这个公司！  

你可以这样做：你已经知道孙悟空什么时候要来偷猪八戒的东西，而猪八戒又要在什么时间去抢沙和尚的东西，又知道猪八戒每秒钟可以偷沙和尚 $n$ 台笔记本，帮猪八戒算一算，自己在有限的时间内能偷到沙和尚多少台笔记本，使得他有足够的时间来防御大师兄。

## 说明/提示

对于 $100\%$ 的数据，保证两个时间合法，$n<32768$。

## 样例 #1

### 输入

```
00:0:00
0:00:10
10```

### 输出

```
100```

3. 计算结果并输出：

#### AC代码如下：

```cpp
#include <cstdio>
using namespace std;

//因为题目说答案有可能会超int，所以这里所有都用long long比较保险
typedef long long ll;

//用一个时间类来保存一个时间
//PS：实际上写个类就是用来装X的
class Time {
private:
	//分别存储小时，分钟，秒
	ll hour, minute, second;
	//存储时间戳（类似于句柄之类的）
	ll point;
public:
	//构造函数，用输入的内容进行初始化
	Time() {
		this->input();
	}

	void input() {
		//输入时间
		scanf("%lld:%lld:%lld", &hour, &minute, &second);
		//计算时间戳，时间戳以1秒为1单位
		point = hour * 3600 + minute * 60 + second;
	}

	//对减法进行重载，方便计算
	ll operator-(Time& cmp) {
		//返回时间戳的差值
		return this->point - cmp.point;
	}
};

int main()
{
	ll n;
	Time t1, t2;	//在初始化的时候已经输入了数据
	scanf("%lld", &n);
	//最后输出相差秒数乘上每秒偷的东西的数量即可
	printf("%lld\n", (t2 - t1) * n);
	return 0;
}

## 作者：NishikinoMaki (赞：2)

这道题可以拿来练快读

快读的基本思想就是把数字当作字符一位一位的读入，然后把字符转化成数字返回。

因为getchar的时间复杂度很低，所以比cin和scanf不知道高到哪里去了

题目本身不难，单纯的模拟，所以拿来练习快读再合适不过了

而且快读能自动“过滤”掉一些奇奇怪怪的字符，在本题中的优势就更明显了

快读的具体实现看代码吧啦啦啦~~~

```cpp
#include <cstdio> //调用getchar必须用cstdio
#include <iostream>
using namespace std;

inline long long read()//快读
{
    long long n = 0,k = 1;//我们用n来记录所读到的数字（默认为正），k来记录正负（毕竟long long也有负数）
    char ch = getchar();//现在我们读入了第一个字符，但并不知道它是什么
    while ((ch > '9' || ch < '0') && ch != '-')  ch = getchar();
     //所以如果ch既不是0~9的数字，也不是决定n正负的负号，我们就多读一个字符，看下一个是不是要读的数字
    if(ch == '-') k = -1, ch = getchar();
    //如果读到了一个负号，用k记一下，然后再读下一位
    //这里一定要有ch = getchar()这一句！不然ch还是负号后面的while循环就崩了！
    while (ch <= '9' && ch >= '0')//当ch是数字时，我们更新n的值
      {
          n = n \* 10 + ch - '0';
        //因为是从高位向低位读入，所以n先乘十再加上新读入的数
        //ch是字符型，参与运算的是ASCII码值，所以减去0的ACSII才是其读入的数字
          ch = getchar();
        //再读一位，不是数字就直接跳出循环
        //所以这就很好的解决了输入中那个恶心的冒号
      }
    return n \* k;//n就是绝对值，k就是正负，一乘就是读入的数
}

  then begin write(2908824891);halt;end;
  a:=pos(':',s);{运用字符串进行处理，因为题目说可能有2个0，也可能有1个0，如果写过奖学金的应该都明白}
  x:=copy(s,1,a-1);
  val(x,c);{将字符串转为数字}
  delete(s,1,a);
  a:=pos(':',s);
  x:=copy(s,1,a-1);
  val(x,d);

  val(s,e);

  y:=copy(s1,1,b-1);
  val(y,c1);

  a:=pos(':',s1);
  y:=copy(s1,1,b-1);
  val(y,d1);
  delete(s1,1,a);
  val(s1,e1);

  h:=c1-c;{将所有时间进行对比，以及加减}
  m:=d1-d;
  second:=e1-e;
  if m<0 then begin h:=h-1;m:=m+60;end;{如果不够减，则退位-1，自己+60}
  if second<0 then begin m:=m-1;second:=second+60;end;
  if m<0 then begin h:=h-1;m:=m+60;end;
  write((h*3600+m*60+second)*speed);

end.

思路：

1. 读取开始与结束的时间，分别存储时分秒。

2. 将时间统一换算成秒。
- 1小时=3600秒
- 1分钟=60秒

3. 用结束时间-开始时间求得相差的秒数，乘以每秒偷的个数。
- 注意：结果的大小有可能超过int型，需要用long long型。

```c
#include <stdio.h>

int main(void)
{
	int hour1, min1, sec1,	//猪八戒的开始时间
	    hour2, min2, sec2;	//孙悟空的开始时间，即猪八戒的结束时间
	int time1,		//猪八戒的开始时间换算成秒
	    time2;		//猪八戒的结束时间换算成秒
	int num;		//每秒能偷的笔记本数
	long long n;		//猪八戒能偷沙和尚笔记本的个数

	scanf("%d:%d:%d", &hour1, &min1, &sec1);
	scanf("%d:%d:%d", &hour2, &min2, &sec2);
	scanf("%d", &num);

	time1 = hour1 * 3600 + min1 * 60 + sec1;
	time2 = hour2 * 3600 + min2 * 60 + sec2;

	n = (long long)num * (time2 - time1);

	printf("%lld\n", n);

	return 0;
}

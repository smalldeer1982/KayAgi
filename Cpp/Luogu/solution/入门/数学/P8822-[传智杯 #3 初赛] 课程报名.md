# [传智杯 #3 初赛] 课程报名

## 题目描述

传智播客推出了一款课程，并进行了一次促销活动。具体来说就是，课程的初始定价为 $v$ 元；每报名 $m$ 个学员，课程的定价就要提升 $a$ 元。由于课程能够容纳的学生有限，因此报名到 $n$ 人的时候就停止报名。

现在老师想知道，当课程停止报名时，一共可以获得多少学费呢？

## 说明/提示

### 样例解释
每卖出 $1$ 个课程，价格就会提高 $1$ 元，所以总共获得 $1+2+3+4+5=15$ 元。

### 数据规模与约定
对于 $50\%$ 的数据，满足 $1 \leq n,m,v,a\leq 10$；

对于额外 $20\%$ 的数据，满足 $a=0$；

对于 $100\%$ 的数据，满足 $0 \leq n,m,v,a \leq 1000$。

然而，由于本次比赛是 ACM 赛制，因此您必须通过 $100\%$ 的数据才能够获得本题的得分，后题同。

## 样例 #1

### 输入

```
5 1 1 1```

### 输出

```
15```

# 题解

## 作者：Loser_Syx (赞：8)

## 思路

题目还是很水的，可以纯模拟。

由于每报名完 $m$ 个人，课程会涨价 $a$ 元，那么我们只要模拟这涨价的过程和卖出的课程的钱，直到卖给 $n$ 个人后，最后输出答案就可以了。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
main(){
    int n, v, m, a, ans = 0;
    cin >> n >> v >> m >> a;
    while(n){//还有课的情况
        for(int i = 1; i <= m && n != 0; i++, n--){//注意这里一定要判断 n!=0，否则肯定 WA
            ans += v;//卖课
        }
        v += a;//涨价
    }
    cout << ans;
    return 0;
}
```

# 题意简述

有 $n$ 个人报名课程，初始的课程定价为 $v$ 元/节，每报名 $m$ 个人，定价就会提升 $a$ 元，问所有人的费用总和是多少？

# 思路

我们可以将 $n$ 个人分割成若干个长度为 $m$ 的小段（最后一段的长度可能不是 $m$），这样的一段之中，所有人所花的费用是相同的。

那么对于第 $i$ 段（$i$ 从 $1$ 开始），定价应为 $(v + (i - 1) \times a)$ 元（自己模拟一下就能得出）。

所以对于前面的整段，第 $i$ 段所花的费用应该是：$(v + (i - 1) \times a) \times m$ 元。对于最后面可能会有的非整段，所需费用就是：$(v + (i - 1) \times a) \times (n \bmod m)$ 元。

# 代码

```cpp
#include <iostream>
using namespace std;

int n, v, m, a;
int sum;

int main()
{
    cin >> n >> v >> m >> a;
    for(int i = 0; i < n / m; i++) // 这里为了方便，可以直接从 i=0 开始遍历，这样就不用算 i-1 了，直接用 i 即可。
    {
        sum += (v + i * a) * m;
    }
    sum += (v + n / m * a) * (n % m);
    cout << sum << endl;
    return 0;
}
```

## 作者：kkxacj (赞：1)

#### [题目传送门](https://www.luogu.com.cn/problem/P8822)

#### 题目大意：

课程的初始定价为 $v$ 元，报名 $m$ 个学员，定价提升 $a$ 元。报到 $n$ 人停止报名，问一共可以获得多少学费。
#### 思路：

模拟报名，每次看是否还可以报 $m$ 个人，如果可以则学费加上 $v \times m$ 元, $v$ 加上 $a$ 元，否则学费加上 $v \times n$ 元，结束循环。

code
```c
#include<bits/stdc++.h>
using namespace std;
int n,ans,m,k,v;
int main()
{
//	freopen(".in","r",stdin);
//	freopen(".out","w",stdout);
	scanf("%d%d%d%d",&n,&v,&m,&k);
	while(n > 0)
	{
		if(n - m >= 0)
		{
			n -= m;
			ans += v * m;
			v += k;
		}
		else
		{
			ans += n * v;
			n = 0;
		}
	}
	printf("%d",ans);
	return 0;
}
```

# 数组逆序重存放

## 题目描述

将一个数组中的值按逆序重新存放。例如，原来的顺序为 $8,6,5,4,1$。要求改为 $1,4,5,6,8$。

## 样例 #1

### 输入

```
5
8 6 5 4 1```

### 输出

```
1 4 5 6 8```

# 题解

## 作者：hmya (赞：29)

这题，输入进来再倒序输出即可。

摆上代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;
    int a[150];
    scanf("%d",&n);
    for(int i=1;i<=n;i++){
         scanf("%d",&a[i]);
    }
    for(int i=n;i>=1;i--){
         printf("%d ",a[i]);
    }
    return 0;
}
```

## 作者：cyhyyds (赞：21)

题目要求把数组逆序存放，那该怎么办呢？

既然我们存储的时候是从第一个存储到第 $n$ 个，那么要将数组逆序存放，我们就可以从第 $n$ 个开始输出到第 $1$ 个，这样就实现了倒序的操作。

代码如下：
```
#include <iostream>

using namespace std;

int a[105];

int main() {
	int n;
	cin >> n;
	for (int i = 1; i <= n; i ++) {
		cin >> a[i];
	}
	for (int i = n; i >= 1; i --) {
		cout << a[i] << " ";
	}
	//输入从第一个到第n个
	//输出从第n个到第一个
	//就实现了倒序 
	return 0;
} 
```

## 题解

### 解法 1

直接使用 $\verb!reverse!$ 函数翻转整个序列。例如， `reverse(a+1,a+1+n)` 可以翻转序列 $a$ 的第 $1$ 到 $n$ 项。

### 解法 2

考虑使用循环进行枚举。从 $1$ 枚举到 $\left\lfloor\dfrac{n}{2}\right\rfloor$ ，交换对应的数字。其实某种程度上相当于手写了一遍 $\verb!reverse!$ 了。

### 解法 3

使用栈的特性。一个栈具有**先进先出**的特性，我们先将所有数字入栈，再将所有数字出栈即可。在 C++ 中可以直接用 $\verb!stack<int>!$ 定义一个 $\verb!int!$ 类型的栈。
  
### 解法 4

我们为什么不能直接倒序输出呢？这是最简单也很容易想到的解法了。

## 参考代码

```cpp
#include<bits/stdc++.h>
#define up(l,r,i) for(int i=l,END##i=r;i<=END##i;++i)
#define dn(r,l,i) for(int i=r,END##i=l;i>=END##i;--i)
using namespace std;
typedef long long i64;
const int MAXN =1e3+3;
int n,A[MAXN];
int main(){
    scanf("%d",&n); up(1,n,i) scanf("%d",A+i);
    dn(n,1,i) printf("%d ",A[i]);
    return 0;
}
```

`reverse(_begin,_end);//_begin 是数组或字符串开头，_end 同理`

```cpp

```

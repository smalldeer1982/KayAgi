# 年龄与疾病

## 题目描述

某医院进行一项研究，想知道某项疾病是否与年龄有关。因此对以往的诊断记录进行整理，统计 0-18 、 19-35 、 36-60、 61 及以上这四个年龄段的患者人数占总患者人数的比例。

## 样例 #1

### 输入

```
10
1 11 21 31 41 51 61 71 81 91```

### 输出

```
20.00%
20.00%
20.00%
40.00%```

## 样例 #2

### 输入

```
12
10 19 86 94 58 57 34 19 88 69 5 13```

### 输出

```
25.00%
25.00%
16.67%
33.33%```

# 题解

## 作者：hanyuchen2019 (赞：22)

这道题只关注各个患者所处的年龄段，并不在意患者的具体年龄。

所以我们可以定义 $4$ 个变量 `a[1],a[2],a[3],a[4]`，用来存储各个年龄段的人数。（类似桶排序）

在读入数据时直接判断年龄段，并将所对应的变量 $+1$，就完成了对各个年龄段的人数的统计。

因为本题要保留两位小数，我选择直接使用 `printf` 进行输出。

```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int a[5],n,c;
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>c;
		if(c<=18)        a[1]++;
		if(19<=c&&c<=35) a[2]++;
		if(36<=c&&c<=60) a[3]++;
		if(c>60)         a[4]++;
	}
	printf("%.2lf%%\n",double(a[1])/n*100);//百分数要x100!
	printf("%.2lf%%\n",double(a[2])/n*100);
	printf("%.2lf%%\n",double(a[3])/n*100);
	printf("%.2lf%%\n",double(a[4])/n*100);
	//由于a[1~4]除以n时，两边都是整数，会导致直接整除，所以要进行类型转换，使其变成浮点数除法。 
 	return 0;
}
```

## 作者：CSP_Sept (赞：14)

## Description

>对于一个非负整数数列 $\{a_n\}$，分别输出这个数列中在 $[0,18],[19,35],[36,60],[61,+\infty]$ 区间内的占比情况。\
>百分数前保留 $2$ 位小数。

## Solution

### 分析

非常直接的方法就是开四个变量 $a,b,c,d$ 分别记录四个区间内数的个数，最后除以 $n$ 算百分比。

这里就要用到 if...else... 语句。

### 算法介绍

```cpp
if (语句 1) 操作 1;
else if (语句 2) 操作 2;
else if (语句 3) 操作 3;
...
else 操作 n;
```

代表：

1. 如果语句 $1$ 成立，则执行操作 $1$；
2. **否则**，如果语句 $2$ 成立，则执行操作 $2$；
3. ……
4. 如果上述 $n-1$ 个语句都不成立，则执行操作 $n$。

### 实现

对于每个 $a_i(i\in[1,n])$)，挨个判断即可。

## Code

```cpp
#include <cstdio>

using namespace std;
int n, a, b, c, d;
int main(){
    scanf("%d", &n);
    for(int i = 0 ; i < n ; i++){
        int t;
        scanf("%d", &t);
        if(t <= 18) a++;
        else if(t <= 35) b++;
        else if(t <= 60) c++;
        else d++;
    }
    printf("%.2lf\%\n%.2lf\%\n%.2lf\%\n%.2lf\%", a * 100.0 / n, b * 100.0 / n, c * 100.0 / n, d * 100.0 / n);
    return 0;
}
```

## 作者：5k_sync_closer (赞：12)

# 思路
输入每个病人的年龄后，判断属于哪个年龄段，累加每个年龄段的人数。

最后用每个年龄段的人数 ÷ 总人数 × 100 输出即可。

注意 printf 输出**一个**百分号是`printf("%%");`
# 代码
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
double n, a, b, c, d, t; //全定义成double，除法方便
int main()
{
	cin >> n;
	for(int i = 0;i < n;++i)
	{
		cin >> t; //下面4行是统计四个年龄段的人数
		if(t >= 0 && t <= 18) ++a;
		if(t >= 19 && t <= 35) ++b;
		if(t >= 36 && t <= 60) ++c;
		if(t >= 61) ++d;
	}
	printf("%.2lf%%\n%.2lf%%\n%.2lf%%\n%.2lf%%", a / n * 100, b / n * 100, c / n * 100, d / n * 100);
	return 0;
}
```

## 作者：·糯· (赞：11)

## 题目分析
此题只用按每个患者的年龄将每个患者分到四个计数器内，表示四个不同的年龄段的人数，最后再求一下总和，然后保留**两位小数**按**双精度浮点型变量**输出每个计数器内的数占总和的百分比即可。
## 代码展示
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[110],sum1,sum2,sum3,sum4,all; 
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a[i];
		if(a[i]>=0&&a[i]<=18)
			sum1++;
		else if(a[i]>=19&&a[i]<=35)
			sum2++;
		else if(a[i]>=36&&a[i]<=60)
			sum3++;
		else if(a[i]>=61)
			sum4++;
	}
	all=sum1+sum2+sum3+sum4;
	cout<<fixed<<setprecision(2)<<sum1*100.0/all<<"%"<<endl;
	cout<<fixed<<setprecision(2)<<sum2*100.0/all<<"%"<<endl;
	cout<<fixed<<setprecision(2)<<sum3*100.0/all<<"%"<<endl;
	cout<<fixed<<setprecision(2)<<sum4*100.0/all<<"%"<<endl;
	return 0;
}

```

## 作者：MCH_Satrimiten (赞：6)

这道题目只要分年龄阶段计数即可。
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[10001];
int w,x,y,z;
int main(){
	int n;
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a[i];
		if(a[i]>=0&&a[i]<=18)w++;//代表 0-18 岁这个年龄段的患者人数
		if(a[i]>18&&a[i]<=35)x++;//代表 19-35 岁这个年龄段的患者人数
		if(a[i]>35&&a[i]<=60)y++;//代表 36-60岁这个年龄段的患者人数
		if(a[i]>60)z++;//代表 61岁及以上这个年龄段的患者人数
	}
	double ww=w,xx=x,yy=y,zz=z,nn=n;//转成浮点数
	printf("%.2lf%\n",ww/nn*100);//保留两位小数输出，记得乘100！！
	printf("%.2lf%\n",xx/nn*100);//百分号记得加上
	printf("%.2lf%\n",yy/nn*100);
	printf("%.2lf%\n",zz/nn*100);
	return 0;
}
```

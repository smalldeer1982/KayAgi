# 查找特定的值

## 题目描述

在一个序列（下标从 $0$ 开始）中查找一个给定的值，输出第一次出现的位置。

## 样例 #1

### 输入

```
5
2 3 6 7 3
3```

### 输出

```
1```

# 题解

## 作者：一只大龙猫 (赞：23)

[原题传送门](https://www.luogu.com.cn/problem/B2093)

对于这道题，我们固然可以一个一个去查找（即“顺序查找法”），但我们还可以用更优的方法——**哨兵查找法**。

先看一下代码。

```cpp
#include<iostream>
using namespace std;
int n,a[20000],x,p;
int main(){
	cin>>n;
	for(int i=0;i<n;i++){
		cin>>a[i];
	}
	cin>>x;
	a[n]=x;
	while(a[p]!=x)p++;
	if(p==n)cout<<"-1";
	else cout<<p;
	return 0;
}

```

“哨兵”指的就是 $a_n \gets x$ 一句中放在 $a_n$ 里的数。只要把 $x$ 放在 $a_n$ 里，当 $p=n$ 时就一定能找到 $x$，不会出现 $p > n$ 的情况。因此，我们就不用在`while`循环中确认 $p$ 的值是否达到了 $n$。

换句话说，我们把原来的

```cpp
while(p<n){
	if(a[p]==x){
		cout<<p;
		return 0;
	}
	p++;
}
cout<<"-1";
return 0;
```

写成了

```cpp
while(a[p]!=x)p++;
if(p==n)cout<<"-1";
else cout<<p;
return 0;
```

不但使代码看上去更加简洁，还省去判断 $p$ 是否等于 $n$ 的语句，节约了大约 $1 \over 4$ 的时间。在 $n ≤ 10000$ 时，节约的时间还不明显。但当 $n$ 达到 $1 \times 10^5$ 甚至是 $1 \times 10^6$ 时，函数的执行时间就会有显著差距了。

[参考文献](https://www.luogu.com.cn/paste/zsscxeo7)

```cpp
#include<bits/stdc++.h>

int a[10007];

    cin>>n;
    for(int i=0;i<n;i++){
        cin>>a[i];

## 作者：sid_shi1 (赞：11)

题目：[B2093 查找特定的值](https://www.luogu.com.cn/problem/B2093)

思路：

原本本题我想要用一个数组来记录的，记录某个值第一次出现的位置（毕竟数也不大），但是由于用有负数的存在，数组下标也不能为负数，这种思路写起来就稍微有一些麻烦（虽然我知道某些大佬觉得这很水），那么就直接暴力查询就好了。从头到尾遍历一遍，遇到要找的值就直接输出当前下标（注意下表从 $0$ 开始），然后退出循环，并拿一个变量标记找到了。循环结束后看一下有没有找到，若没有输出 $-1$ 就行了。

代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[10001]={0},x;
int main(){
	bool flag=false;//一开始标记没找到
	cin>>n;
	for(int i=0;i<=n-1;i++) cin>>a[i];//注意下标从0开始
	cin>>x;
	for(int i=0;i<=n-1;i++){
		if(a[i]==x){//找到了
			cout<<i;//输出当前下标
			flag=true;//标记为找到
			break;//退出循环
		}
	}
	if(flag==false) cout<<-1;//若循环里没有找到，输出-1
	return 0;
}
```

## 作者：rain_dew (赞：8)

本题解为 Python 题解，原题链接：[B2093 查找特定的值](https://www.luogu.com.cn/problem/B2093)

在 Python 中，`count()` 如果查找成功返回查找到的个数，否则返回 $0$；而 `index()` 返回它查找到的第一个值，因此本题代码如下：

```python
n=int(input());
a=input().split();
m=input();
if a.count(m):
    print(a.index(m));
else:
    print("-1");
```

## 作者：cyhyyds (赞：6)

我们只需要按照题意模拟，输入的数存储在数组中。

但是我们看到题目，要求下标从 $0$ 开始，也就是说我们输入时要从 $0$ 这个下标开始。

最后我们只需要遍历这个数组，同样，遍历时下标也要从 $0$ 开始。在寻找到这个元素时，将其下标输出，结束程序。在最后如果没有找到该元素，我们在程序最后输出 $-1$ 即可。

代码如下：
```
#include <iostream>
#include <cstdio>
#include <cstring>
#include <algorithm> //sort需要的库 
using namespace std;
int a[10005];
int main() {
	int n;
	cin >> n;
	for (int i = 0; i < n; i ++) { //下标从0开始 
		cin >> a[i];
	}
	int x;
	cin >> x;
	for (int i = 0; i < n ; i ++) { //下标从0开始 
		if (a[i] == x) {//如果出现相等的情况 
			cout << i;//输出下标 
			return 0;//同时结束程序 
		} 
	} 
	cout << "-1";//如果程序未结束，那么就没有x这个元素，则输出-1 
	return 0;
}
```

## 作者：·糯· (赞：5)

## 题目分析
这题我们直接用循环判断，一旦找到与特定值相同的元素，就输出下标，并结束进程，若在循环外仍未结束进程，说明没有找到，输出“-1”。注意题目中要求**下标从零开始**。
## 代码实现
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[10010]; 
int main(){
	cin>>n;
	for(int i=0;i<n;i++)//下标从零开始
		cin>>a[i];
	int x;//特定值
	cin>>x;
	for(int i=0;i<n;i++)
		if(a[i]==x){//判断
			cout<<i<<endl;
			return 0;//找到了，结束进程
		}
	cout<<-1<<endl;//未找到
	return 0;
}

```

# 整数去重

## 题目描述

给定含有 $n$ 个整数的序列，要求对这个序列进行去重操作。所谓去重，是指对这个序列中每个重复出现的数，只保留该数第一次出现的位置，删除其余位置。

## 样例 #1

### 输入

```
5
10 12 93 12 75```

### 输出

```
10 12 93 75```

# 题解

## 作者：·糯· (赞：79)

## 题目分析
因为此题的每一个数的值都不大，所以我们可以建立一些桶来记录每个数的出现次数，每输入一次就将对应的桶内的数加一，然后判断是否只出现一次，是，就输出。
## 代码实现
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[20010],num[110]; 
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a[i];
		num[a[i]]++;//出现次数
		if(num[a[i]]<2)//如果是第一次出现就输出
			cout<<a[i]<<" ";
	}
	return 0;
}

```

## 作者：DreamFox (赞：31)

本题要求我们将序列去重，所以我们可以用一个数组记录此时输入的那个数之前有没有出现过。

如果没有出现过，那么将其输出，并将数组中以这个数为下标的值计为 $1$（假设刚开始所有值为 $0$），表示这个数已经出现过，否则就不进行操作。

由于 $10 \leq a_i \leq 100$，所以并不用开 `map` 数组，直接开一个大小为 $105$ 的布尔数组即可。

AC 代码（含注释）：
```cpp
#include <iostream>
using namespace std;
int a, n;
bool vis[105];  //记录输入的数是否在之前出现过
int main() {
	cin >> n;
	for(int i = 1; i <= n; i ++) {
		cin >> a;
		if(!vis[a]) cout << a << ' ', vis[a] = 1;  //如果没出现过，就将其输出，并标记这个数已经出现过。
	}
	return 0;
}
```

## 作者：spark_minous (赞：21)

这道题可以将每个整数都存到一个初始化的数组里，如果为 $0$ ，就说明这个数第一次出现，输出这个数，并将这个数组变为 $1$ ，表示已经出现过这个数。 

下面放代码：
```C
#include<bits/stdc++.h>
int i,n,cnt[1000000],t;
int main(){
scanf("%d", &n);
for(i=0;i<n;i++){
  scanf("%d", &t);
  if(cnt[t]==0){//如果这个数第一次出现
    printf("%d ", t);
    cnt[t]=1;
  }
}
  return 0;
}
```

## 作者：墨忧 (赞：12)

## 题意
给定 $n$ 和 $n$ 个数，依次输出这 $n$ 个数，对于已经输出过的数不再输出。
## 思路
用一个数组来存储每一个数，输出时用另外一个数组来记录这个数是否输出过，若输出过则不再输出，类似于桶排序。
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,t[100000],a[100000]; //a数组记录每个数，t数组记录数字是否出现过
int main()
{
    cin>>n;
    for(int i=0;i<n;i++) cin>>a[i];
    for(int i=0;i<n;i++)
    {
        if(t[a[i]]==0) cout<<a[i]<<" "; //若出现过，则不再输出
        t[a[i]]++; //记录下这个数已经输出过
    }
    return 0;
}

## 作者：int127 (赞：7)

## 前置芝士

`map` 是一个关联容器，将一个类型变量映射到另一个类型的变量。

如：

```cpp
map<string,int> q;
...
q["EXAMPLE"]=20210703;
```

并不用深究 `map` 的其他知识，就可以完成此题。

## 思路

边读入边统计每个数字出现过的次数，如果超过 $1$ 就重复了，不输出。

结合代码思考更佳。

```cpp
#include<bits/stdc++.h>
using namespace std;
int a;
map<int,int> p;//可以替换成数组，毕竟 a[i] 的值域不大
int main(){
    int n;
    cin>>n;
    for(int i=1;i<=n;i++){
    	cin>>a;
    	if(p[a]>0) continue;
    	cout<<a<<' ';
    	p[a]++;
    }
    return 0;
}
```

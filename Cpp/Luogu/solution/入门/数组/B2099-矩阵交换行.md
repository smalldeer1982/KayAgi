# 矩阵交换行

## 题目描述

给定一个 $5 \times 5$ 的矩阵(数学上，一个 $r \times c$ 的矩阵是一个由 $r$ 行 $c$ 列元素排列成的矩形阵列)，将第 $n$ 行和第 $m$ 行交换，输出交换后的结果。

## 样例 #1

### 输入

```
1 2 2 1 2
5 6 7 8 3
9 3 0 5 3
7 2 1 4 6
3 0 8 2 4
1 5```

### 输出

```
3 0 8 2 4
5 6 7 8 3
9 3 0 5 3
7 2 1 4 6
1 2 2 1 2```

# 题解

## 作者：TensorFlow_js (赞：28)

# 题目分析

只需使用 `for` 输入矩阵后交换行并输出即可。

# 字符串解法

```cpp
#include<bits/stdc++.h>
using namespace std;
string jz[5]={"","","","",""};//定义字符串数组 jz （题目中的矩阵的行）
int main(){
    for(int i = 0;i < 5;i ++)getline(cin,jz[i]);
    int a,b;cin>>a>>b;
    swap(jz[a-1],jz[b-1]);//使用STL函数 swap 交换
    for(int i = 0;i < 5;i ++)cout<<jz[i]<<endl;
    return 0;
}
```

# 二维数组解法 1

```cpp
#include<bits/stdc++.h>
using namespace std;
int jz[5][5];//定义整型数组 jz （题目中的矩阵）
int main(){
    for(int i = 0;i < 5;i ++)for(int j = 0;j < 5;j ++)cin>>jz[i][j];
    int a,b;cin>>a>>b;
    swap(jz[a-1],jz[b-1]);//使用STL函数 swap 交换
    //这里 swap 是将一整个一维数组 jz[a-1] 与 jz[b-1] 进行了交换
    for(int i = 0;i < 5;i ++){for(int j = 0;j < 5;j ++)cout<<jz[i][j]<<' ';cout<<endl;}
    return 0;
}
```

# 二维数组解法 2（手打`swap`）

```cpp
#include<bits/stdc++.h>
using namespace std;
int jz[5][5];//定义整型数组 jz （题目中的矩阵）
int main(){
    for(int i = 0;i < 5;i ++)for(int j = 0;j < 5;j ++)cin>>jz[i][j];
    int a,b;cin>>a>>b;
    for(int i = 0;i < 5;i ++){int temp = jz[a-1][i];jz[a-1][i]=jz[b-1][i];jz[b-1][i]=temp;}//使用临时变量 temp 进行交换
    for(int i = 0;i < 5;i ++){for(int j = 0;j < 5;j ++)cout<<jz[i][j]<<' ';cout<<endl;}
    return 0;
}
```

## 作者：sylqwq (赞：24)

题目不难，注意事项写到代码注释里啦~       

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int a[5][5];
    int m,n;
    //将数组中的数字输入，用两个for循环即可 
    for(int i=0;i<5;i++)
    {
        for(int j=0;j<5;j++)
            cin>>a[i][j];
    }
    cin>>m>>n;
    //循环一次二维数组
    for(int i=0;i<5;i++)
    {
        for(int j=0;j<5;j++)
        {
            if(i==m-1)//当i等于m-1时，将取n-1行的数组 
                cout<<a[n-1][j];
            else if(i==n-1)//同样的，当等于n-1时 ，取m-1的数组 
                cout<<a[m-1][j];
            else
                cout<<a[i][j];//其他的数组不变 
            cout<<" "; //空格隔开每一个数 
    } 
    cout<<endl;//j循环结束再空行，这样那就是每行空格 
    } 
    return 0;
}
```

## 作者：墨忧 (赞：20)

## 题目大意
给定一个矩阵，交换矩阵的第 $m$ 和第 $n$ 行后输出。
## 思路
使用二维数组存储矩阵，循环交换两行的每个值即可。
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[10][10],m,n;//二维数组存矩阵 
int main()
{
    for(int i=1;i<=5;i++)
        for(int j=1;j<=5;j++) cin>>a[i][j];
    cin>>m>>n;
    for(int i=1;i<=5;i++) swap(a[m][i],a[n][i]);//交换两行 
    for(int i=1;i<=5;i++)
    {
        for(int j=1;j<=5;j++) cout<<a[i][j]<<" ";
        cout<<endl;
    }
         
    return 0;
} 
```

## 作者：rain_dew (赞：7)

本题解为 Python 题解，原题链接：[B2099 矩阵交换行](https://www.luogu.com.cn/problem/B2099)

Python 中没有具体**交换值**的函数（但有**交换空间**的函数 `swap`），而交换值的办法一般为 `x,y=y,x`，指 $x$ 与 $y$ 进行交换值。本题读入后与其交换（注意数组初始下标为 $0$）输出即可，代码如下：

```python
a=[];
for i in range(5):
    a.append(input());
x,y=input().split();
x=int(x);y=int(y);
a[x-1],a[y-1]=a[y-1],a[x-1];
for i in range(5):
    print(a[i]);
```

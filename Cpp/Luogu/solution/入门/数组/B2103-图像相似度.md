# 图像相似度

## 题目描述

给出两幅相同大小的黑白图像（用 $0-1$ 矩阵）表示，求它们的相似度。

说明：若两幅图像在相同位置上的像素点颜色相同，则称它们在该位置具有相同的像素点。两幅图像的相似度定义为相同像素点数占总像素点数的百分比。

## 样例 #1

### 输入

```
3 3
1 0 1
0 0 1
1 1 0
1 1 0
0 0 1
0 0 1```

### 输出

```
44.44```

# 题解

## 作者：·糯· (赞：25)

## 题目分析
此题可以直接对比，相同的话就将计数器加一，最后再除成百分比的形式，注意题目中**没有要求输出百分号**，并且要保留两位小数。
## 代码实现
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[110][110],b[110][110],m,n;
double s;
int main()
{
	cin>>n>>m;
	for(int i=1;i<=n;i++)
		for(int j=1;j<=m;j++)
			cin>>a[i][j];
	for(int i=1;i<=n;i++)
		for(int j=1;j<=m;j++)
			cin>>b[i][j];//输入两个矩阵
	for(int i=1;i<=n;i++)
		for(int j=1;j<=m;j++)
			if(a[i][j]==b[i][j])
				s++;//相同的点数
	cout<<fixed<<setprecision(2)<<s/(n*m)*100<<endl;//图像相似度
	return 0;
}
```

## 作者：Eason_AC (赞：8)

## Content
给定两个 $m\times n$ 的矩阵 $A,B$，求 $A,B$ 两个矩阵的相似度，精确到小数点后 $2$ 位。

定义两个矩阵的相似度为两个矩阵对应相同元素个数占矩阵大小的比例。

**数据范围：$1\leqslant n,m\leqslant 100$。**
## Solution
强烈建议先做 [B2019](https://www.luogu.com.cn/problem/B2019)。

对于两个矩阵对应相同元素个数，我们可以开个计数器，然后枚举对应元素，将对应元素相等当做一个 $0/1$ 值加入计数器中。设计数器最终的结果为 $x$，最终的答案就是 $\dfrac{x}{nm}$。注意为了避免整除运算，需要将分子分母中的一个在输出时强制转换为浮点型变量，具体见代码。
## Code
代码中再次运用到了省空间的技巧，即将只在输入中有用的数组直接用变量代替，然后再一边判断。
```cpp
#include <cstdio>
using namespace std;

int n, m, ans, a[107][107];

int main() {
    scanf("%d%d", &m, &n); //注意看题！！！
    for(int i = 1; i <= m; ++i) for(int j = 1; j <= n; ++j) a[i][j] = Rint;
    for(int i = 1; i <= m; ++i) for(int j = 1; j <= n; ++j) {int x; scanf("%d", &x); ans += (a[i][j] == x);}
    return printf("%.2lf", ans * 100.0 / (n * m)), 0;
}
```

## 作者：Miracle_ZX (赞：7)

### 思路
此题其实就是先输入两个矩阵，对比后求出相同个数，最后再算百分比即可
### 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
  
int a[1010][1010], b[1010][1010], m, n;
double cnt ,ans;
  
int main()
{
    cin >> m >> n;
    for(int i = 0; i < m; i++)
    {
        for(int j = 0; j < n; j++)
  		{
  			cin >> a[i][j];
  		}
  	}
    for(int i = 0; i < m; i++)
    {
        for(int j = 0; j < n; j++)
  		{
  			cin >> b[i][j];
  		}
  	}//输入部分 
  	
  	for(int i = 0; i < m; i++)
    {
        for(int j = 0; j < n; j++)
  		{
  			if(a[i][j] == b[i][j]) cnt++;
  		}
  	}//逐个对比 
  	
    ans = cnt / (m * n) * 100; //百分数的转换，一定要熟记 
    cout << fixed << setprecision(2) << ans;//保留两位小数 + 四舍五入 输出
    return 0;
}
```

## 作者：ForeverCC (赞：6)

输入，记录 $a_{i,j}$ 和 $b_{i,j}$ 相同的数量，再计算相似度输出即可。

代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int a[105][105],m,n;
double ans;
int main() {
	scanf("%d%d",&m,&n);
	for(int i=1; i<=m; i++)
		for(int j=1; j<=n; j++) 
			scanf("%d",&a[i][j]); 
	for(int i=1; i<=m; i++)
		for(int j=1; j<=n; j++) {
			int b;
			scanf("%d",&b);
			if(a[i][j]==b) 
				ans++;//相同时记录 
		}
	printf("%.2f\n",ans/(m*n)*100);
	return 0;
}
```

## 作者：墨忧 (赞：5)

## 题意
给定两个矩阵，求两个矩阵中所有对应元素相同个数占矩阵大小的百分数
## 思路
用两个二维数组存储两个矩阵，再每一位对比求出对应元素相同个数，除以矩阵大小输出即可
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[105][105],b[105][105],m,n;
double xsd,ans;
int main()
{
    cin>>m>>n;
    for(int i=0;i<m;i++)
        for(int j=0;j<n;j++) cin>>a[i][j];//读入第一个矩阵
    for(int i=0;i<m;i++)
        for(int j=0;j<n;j++)
        {
            cin>>b[i][j];//读入第二个人矩阵
            if(a[i][j]==b[i][j]) xsd++;//对比，求相同个数
        }
    ans=xsd/(m*n)*100;//注意，题目里要求以百分数形式输出答案，因此答案应再乘100
    printf("%.2f",ans);//保留两位小数输出
    return 0;
}
```

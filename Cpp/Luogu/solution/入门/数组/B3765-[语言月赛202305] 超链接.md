# [语言月赛202305] 超链接

## 题目描述

在某局域网中，一共有 $N$ 个网页，依次从 $1$ 编号到 $N$。

每个网页上都有一些超链接，第 $i$ 个网页上一共有 $T_i$ 个超链接，依次指向 $A_{i,1},\cdots,A_{i,T_i}$ 号网页。

某 E 现在从 $1$ 号网页开始，点击不超过两次超链接，一共能到达多少网页？

## 说明/提示

### 样例解释

 - 点击 $0$ 次：$1$ 号页面；
 - 点击 $1$ 次：$2,3$ 号页面；
 - 点击 $2$ 次：$1, 2, 3, 4,5$ 号页面。

共 $5$ 个页面。

### 数据规模与约定

 - 对于 $30\%$ 的测试数据，$T_i = 1$;
 - 对于 $100\%$ 的测试数据，$1 \le N \le 1000$，$0 \le T_i \le 100$，$1 \le A_{i,j} \le N$，同一个网页中不同超链接指向的网页编号不同，不保证不存在指向自己的超链接。

## 样例 #1

### 输入

```
6
2 2 3
3 3 4 1
2 4 5
1 6
1 6
1 5
```

### 输出

```
5```

## 文字题解

题目要求，点击不超过两次的页面，那么可以分为点击零次、一次、两次三类。

### 第一类 点击零次

对于第一类，点击零次，也就是我们的出发页面 $1$。

### 第二类 点击一次

对于第二类，点击一次，也就是页面 $1$ 上超链接指向的所有页面。

### 第三类 点击两次

对于第三类，点击两次。点击两次的页面，都是可以从点击一次的页面，再经过一次点击到达。我们循环枚举所有第二类页面，其超链接指向的页面均为第三类页面。

对于三类页面，我们还需要进行去重，因为一个页面可以同时属于以上三类。我们使用一个 bool 数组 $a$，若 $a[i]$ 为 true，则代表该页面属于以上三类中的某一类，为 false 就不属于。在完成三类页面的计算后，统计数组 $a$ 中有多少位置为 true 即可。

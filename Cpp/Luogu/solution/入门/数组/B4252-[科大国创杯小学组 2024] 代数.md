# B4252 [科大国创杯小学组 2024] 代数 题解
~~安徽是我老家~~
## 题目大意
给出两个序列 $a,b$ 且长度均为 $n$，求出 $a_1b_1 + a_2b_2 + \cdots + a_nb_n$。

## 题目解法
本题目只需采用循环即可。

注意到 $a$ 与 $b$ 的长度相等，所以在输入后再次循环 $1\sim n$，接着只要在每次循环内求出 $a_ib_i$，最后将所有的 $a_ib_i$ 相加就可以解决该问题。

## 注意事项
注意到数据范围 $1 \le n, a_i, b_i \le 1000000$，所以仅仅用 `int` 是肯定会爆的（[友情提醒](https://www.luogu.com.cn/record/209129174)）。所以本题依旧是不开 `long long` 见祖宗。

本题时间复杂度为 $O(n)$，可以放心写。

## AC code
```
#include <bits/stdc++.h>
#define int long long
using namespace std;
int a[1000005],b[1000005];
signed main(){
    int n;
    cin>>n;
    for(int i=1;i<=n;i++){
        cin>>a[i];
    }
    for(int i=1;i<=n;i++){
        cin>>b[i];
    }
    int sum=0;
    for(int i=1;i<=n;i++){
        sum+=a[i]*b[i];
    }
    cout<<sum;
}
```

原来还担心数据太大爆 long long 的（指数据大小超过 long long 导致答案错误）不过最大好像就是 $10^{18}$，可以被 long long 装下，也不需要担心数据太多，用 cin 输入会超时，（用 cin 输入比较慢）所以可以直接将  $a_1$ 到 $a_n$ 存在数组里，再读入每一个 $b_i$ 让答案加上 $a_i+b_i$ 就行了。

代码如下：

```
#include <bits/stdc++.h>
using namespace std;
long long a[1000005];//不开 long long 泪两行。
int main()
{
    long long n, b, ans = 0;
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 1; i <= n; i++)
    {
        cin >> b;
        ans += a[i] * b;
    }
    cout << ans;
    return 0;
}
```

## 题意
和向量没有关系，其实就是两个数组：

$a_1, a_2, \cdots, a_n$ 和 $b_1, b_2, \cdots, b_n$。

求：$a_1b_1 + a_2b_2 + \cdots + a_nb_n$。
## 思路
使用两个数组来存储。

一个 for 循环遍历两个位置的数，求积，再使用一个变量累加。

## 代码
因为两个数的乘积最大可以达到 $10 \times 10^{12}$，然后又有 $n$ 个乘积，所以乘积的和可以达到 $10 \times 10^{18}$，故需要 long long，不用高精度。

```cpp
#include<bits/stdc++.h>
using namespace std;
const int N = 1000000 + 10;
int a[N],b[N];
long long sum;
int main()
{
    int n;cin >> n;
    for (int i = 1;i <= n;i++) cin >> a[i];
    for (int i = 1;i <= n;i++) cin >> b[i];
    for (int i = 1;i <= n;i++){
        sum += (long long)a[i] * b[i];
    }
    cout << sum << "\n";
    return 0;
}
```

# B4252 [科大国创杯小学组 2024] 代数 题解

[题目传送门](https://www.luogu.com.cn/problem/B4252)

## 分析

一道模拟题，考察对数组和循环结构的运用。

题意：给定序列 $a,b$ 和序列长度 $n$，求 $a_1\times b_1+a_2\times b_2+...+a_n\times b_n$。

用数组和循环结构实现即可。

很容易写出代码：

```cpp
#include <stdio.h>
const int N = 1000005;
int n, a[N], b[N], sum;
signed main(void) {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i) scanf("%d", &a[i]);
    for (int i = 1; i <= n; ++i) scanf("%d", &b[i]);
    for (int i = 1; i <= n; ++i) sum += a[i] * b[i];
    printf("%d", sum);
    return 0;
}
```
但是，交上去之后只得了 $60$ 分，我们发现 $1000000\times 1000000=1000000000000=10^{12}$，爆 `int`，要使用 `long long` 类型（最大点积为 $(10^6)^3= 10^{18}$，不会炸）。

## $AC$ $code$

```cpp
#include <stdio.h>
const int N = 1000005; int n, a[N], b[N]; long long sum;
signed main(void) {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i) scanf("%d", &a[i]);
    for (int i = 1; i <= n; ++i) scanf("%d", &b[i]);
    for (int i = 1; i <= n; ++i) sum += 1ll * a[i] * b[i];
    printf("%lld", sum);
    return 0;
}
```


---


事实上我们可以把 $b$ 数组压掉，一读入就计算，加入 $sum$。

```cpp
#include <stdio.h>
int n, a[1000005]; long long sum;
signed main(void) {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i) scanf("%d", &a[i]);
    for (int i = 1, b; i <= n; ++i) scanf("%d", &b), sum += 1ll * a[i] * b;
    printf("%lld", sum);
    return 0;
}
```

# Problem

给定 $n$ 和向量 $a,b$，计算 $a_1b_1 + a_2b_2 + \cdots + a_nb_n$。

# Solution

依题意模拟。输入 $n$，随后输入 $a,b$。接着使用循环不断让结果 $sum$ 加上 $a_ib_i$。最后输出。

因为 $n,a_i,b_i\le 10^6$，有 $a_ib_i\ 10^{12}$，$n$ 个 $a_i+b_i$ 就是最高有 $10^{18}$，需要使用 `long long`。

# Code

```cpp
#include<bits/stdc++.h>
using namespace std;

const int maxn=1e6+1;
int n,a[maxn],b[maxn],sum;

	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a[i];
	}
	for(int i=1;i<=n;i++){
		cin>>b[i];
	}
	for(int i=1;i<=n;i++){
		sum+=a[i]*b[i];
	}
	cout<<sum<<endl;
	return 0;
}

# B4252 [科大国创杯小学组 2024] 代数
根据题意，我们知道点积的运算方式其实是很简单的，利用数组乘上每一项并累加即可。  
要紧的是**数据范围**（这一类题拿到都得先看数据范围），最大 $1 \leq n, a_i, b_i \leq 1000000$。因此每次 $a_ib_i$ 最大为 $10^{12}$，$n$ 个 $a_ib_i$ 相加最大是 $10^{18}$，很明显开`long long`就足够了，连`unsigned long long`都不需要。然后我们按照过程相加即可。  
当然不看数据范围，凭对数据大小的预判直接写高精也可以。但**显然没必要**。

```cpp
#include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    int a[n], b[n];//根据数据大小定数组，提前开好也可以
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }//读入a
    for (int i = 0; i < n; i++) {
        cin >> b[i];
    }//读入b
    long long s = 0;//记得开long long
    for (int i = 0; i < n; i++) {
        s += (long long)a[i] * b[i];//强制转换为long long型防止计算过程中溢出
    }
    cout << s << endl;
    return 0;
}
```

## 思路讲解
这道题就是一道模拟题（其实连模拟也算不上），我们先把数据输入进来，用 $sum$ 记录结果，然后写一个执行 $n$ 次的循环，每次 $sum$ 加上 $a_i\times b_i$，最后输出 $sum$ 即可。

小提示：数据较大，建议使用 long long。

## [AC](https://www.luogu.com.cn/record/209234103) 代码

```cpp
#include <bits/stdc++.h>

int main()
{
    //数据的输入
    cin >> n;
    for (int i = 1;i <= n;i++)
    {
        cin >> a[i];
    }
    for (int i = 1;i <= n;i++)
    {
        cin >> b[i];
    }
    //记录结果
    for (int i = 1;i <= n;i++)

# [蓝桥杯 2025 省 Java B] 逃离高塔

## 题目背景

本站蓝桥杯 2025 省赛测试数据均为洛谷自造，与官方数据可能存在差异，仅供学习参考。

## 题目描述

小蓝一觉醒来，发现自己被困在一座高耸的塔中。这座塔共有 $2025$ 层，每一层都刻有一个数字的立方值，从底层的 $1^3,2^3,3^3,\dots$ 一直到顶层的 $2025^3$，层层叠叠，直入云霄。塔顶有一扇门，门旁刻着一行字：“若想离开此塔，需找出这些立方数中个位数字为 $3$ 的数的个数。”

小蓝非常着急，因为他需要尽快离开这座塔，去参加即将到来的蓝桥杯比赛。时间紧迫，请你帮助他解答这个问题。

## 作者：chenyuan3 (赞：5)

模拟题。枚举 $i$ 从 $1$ 到 $2025$，判断 $i^3$ 除以 $10$ 的余数是否为 $3$，如果是，那么答案加一。

``` cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
ll ans = 0;
void solve()
{
    for (ll i = 1; i <= 2025; i++)
    {
        if (i * i * i % 10 == 3)
        {
            ans++;
        }
    }
    cout << ans;
}
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	cout.tie(nullptr);
 	// ll t; cin >> t; while (t--)
	solve();
	return 0;
}
```

把所有可能的数列出来，枚举即可。

```cpp
#include<bits/stdc++.h>
using namespace std;
```

可得答案为 $202$。

本题还有另一种做法，计算可得正整数的个位数与其立方的个位数有以下关系：

|个位数|立方的个位数|
|:--|:--|
|$0$|$0$|
|$1$|$1$|
|$2$|$8$|
|$3$|$7$|
|$4$|$4$|
|$5$|$5$|
|$6$|$6$|
|$7$|$3$|
|$8$|$2$|
|$9$|$9$|

也就是说，当且仅当一个正整数的个位数为 $7$ 的时候，其立方的个位数为 $3$。

在 $1$ 和 $2025$ 之间一共有 $202$ 个个位数为 $7$ 的正整数，输出 $202$。

## 题意

求 $1$ 到 $2025$ 中立方末尾为 $3$ 的数的个数。

## 解法

立方的末尾取决于这个数的末尾，我们可以求这个数的末尾的立方的末尾。

当然，也可以直接求这个数立方的末尾，不过记得开 `long long`。

## CODE
```cpp
#include<iostream>
using namespace std;
#define mt i%10//计算末尾
int main(){
    int ans=0;
    for(int i=1;i<=2025;i++){
        if(mt*mt*mt%10==3)ans++;//记得还要再次%10
    }
    cout<<ans;
    return 0;
}
```

## 作者：xzy_awa (赞：2)

# 题解：P12154 [蓝桥杯 2024 省 Java B] 逃离高塔

## 思路

一个数的立方数个位仅由**该数的个位数**决定。因此，我们只需确定哪些个位数的立方结果以 $3$ 结尾并统计以这些个位数结尾的数的出现次数。

因为在 $0$ 至 $9$ 这个范围内，只有 $7$ 的立方数个位是 $3$（$7^3=343$）。

然后从 $1$ 到 $2025$ 中每 $10$ 个就会有一个数的个位为 $7$，最小为 $7$，最大为 $2017$。

所以，我们用这些数构成等差数列，首项为 $7$，末项为 $2017$，公差为 $10$。总项数公式为：  

$$\text{项数} = \frac{\text{末项} - \text{首项}}{\text{公差}} + 1 = \frac{2017 - 7}{10} + 1 = 202$$

因此答案为 $202$。

## 代码

C++ 代码：

```cpp
#include <bits/stdc++.h>
#define endl '\n'
using namespace std;

int main()
{
    // ios::sync_with_stdio(false);
    // cin.tie(nullptr);
    // cout.tie(nullptr);
    cout << 202 << endl;
    return 0;
}
```

Java 代码：

```java
public class Main {
    public static void main(String[] args) {
        System.out.println(202);
    }
}
```

## 作者：ysgosv9 (赞：2)

## 题意分析：
  求 $1^3, 2^3, 3^3,\dots 2024^3, 2025^3$ 中个位为 $3$ 的数的个数。

## 求解过程：
  通过枚举可以发现 $1^3$ 的个位数字为 $1$，$2^3$ 的个位数字为 $8 \dots 7^3$ 的个位数字为 $3$。而一个数的立方的**个位**数字只会受这个数的**个位**的影响。如 $11^3$ 的个位数字和 $111^3$ 的个位数字实际上也是 $1$。这使得本题可以转化为 “求 $1, 2, 3 \dots 2024, 2025$ 中，有多少个数字的个位数字为 $7$ ”这个较为简单的问题。

  接下来，我们需要知道在 $1, 2, 3 \dots 2024, 2025$ 中，有多少个个位数字为 $7$ 的数字就好了，这将是一个较为简单的方法。

## 模拟代码：

```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;

signed main()
{
	int ans =  0;
	for (int i = 1; i<= 2025; i++)
	{
		if (i % 10 == 7)
		{
			ans++;
		}
	}
	cout << ans;
	return 0;
}
```
 ## 最终思路
  当然，我们也可以直接计算：

  因为每 $10$ 个数中才存在一个个位数字为 $7$ 的数，又因为 $2021, 2022, 2023, 2024, 2025$ 中显然不存在个位数字为 $7$ 的数，所以答案便是 $1, 2, 3 \dots 2019, 2020$ 这个序列的总长度除以 $10$，即 $2020 \div 10$，也就是 $202$。

## 正解代码


```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;

signed main()
{
	cout << 202;
	return 0;
}
```

## 作者：Vct14 (赞：1)

对于一个数 $x$，设它等于 $10s+t$，其中 $s\ge0$ 且 $0\le t<10$。那么 $x^3\bmod 10=(10s+t)^3\bmod 10=t^3\bmod 10$。我们枚举 $t=0,1,2,\cdots ,9$，发现当且仅当 $t=7$ 时 $t^3\bmod 10=3$。则答案为 $1,2,\cdots ,2025$ 中个位数字为 $7$ 的数的个数。答案为 $202$。

## 作者：LINYUHENG2 (赞：0)

## 思路
这道题主要使用枚举算法，从 $1$ 枚举到 $2025$，如果其中有一个数的立方个位为 $3$，累加计数器。

因为 $2025^3=8303765625$，超出了 `int` 所表示的范围，所以要使用 `long long`。
## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int cnt=0;
    for(long long i=1;i<=2025;i++){
        if(i*i*i%10==3) cnt++;
    }
    printf("%d",cnt);
    return 0;
}
```
最终的答案是 $202$。

简单的枚举 $1$ 到 $2025$ 中的每个数的三次方，判断个位数是否为 $3$ 即可。

```cpp
	}
```

# 小鱼的游泳时间

## 题目描述

伦敦奥运会要到了，小鱼在拼命练习游泳准备参加游泳比赛，可怜的小鱼并不知道鱼类是不能参加人类的奥运会的。

这一天，小鱼给自己的游泳时间做了精确的计时（本题中的计时都按 $24$ 小时制计算），它发现自己从 $a$ 时 $b$ 分一直游泳到当天的 $c$ 时 $d$ 分，请你帮小鱼计算一下，它这天一共游了多少时间呢？

小鱼游的好辛苦呀，你可不要算错了哦。


## 说明/提示

对于全部测试数据，$0\le a,c \le 24$，$0\le b,d \le 60$，且结束时间一定晚于开始时间。

## 样例 #1

### 输入

```
12 50 19 10```

### 输出

```
6 20```

	cout<<x/60<<' '<<x%60;
	return 0;
}
```

### 分析
题意为：给你两个时间点，且不会超过一天。  

我们可以先把起始时间和截止时间都化成分钟，然后求出茶后，再转回去。  
例如样例：
```
12 50 19 10
```
把 `12:50` 转成分钟后是 $12 \times 60 + 50 = 770$ 分钟。  
把 `19:10` 转成分钟后是 $19 \times 60 + 10 = 1150$ 分钟。  
把两数相减后是 $1150 - 770 = 380$ 分钟。
再把 $380$ 分钟转成小时 $ 380 \div 60 = 6$ 余 $10$，这就是答案。

### AC代码
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int sh, sm, eh, em;
    cin >> sh >> sm >> eh >> em;//输入起始时间和截至时间
    int tim = (eh * 60 + em) - (sh * 60 + sm);//算出差值
    cout << tim / 60 << " " << tim % 60;//进行转换
    return 0;
}
```

先用 `cin` 输入开始时间 $a$ 时 $b$ 分到结束时间 $c$ 时 $d$ 分。

#include <iostream>
using namespace std;
int main()
{
    int a,b,c,d;
    cin>>a>>b>>c>>d;
    int x=c-a,y=d-b;
    if(y<0){x--;y+=60;}
    cout<<x<<" "<<y;
    return 0;
}
```

## Solution
我们可以先算出小鱼游了多少分钟，再将其转化为输出所要求的形式。  

我们可以先假设小鱼从 $a$ 时 $0$ 分游到 $c$ 时 $d$ 分，那么这时候小鱼游的时间就是 $60(c-a)+d$，但是小鱼其实是从 $a$ 时 $b$ 分开始的。因此要在原基础上减去 $b$，即 $60(c-a)+d-b$ 分钟。  

接着将其转化为输出的形式。设上文的 $60(c-a)+d-b$ 为 $s$，那么小时数就是 $\lfloor\frac{s}{60}\rfloor$，分钟数就是 $s\bmod60$。

## AC code


```cpp
#include <iostream>
using namespace std;
int main(){
    int a,b,c,d;cin>>a>>b>>c>>d;
    int s=(c-a)*60+d-b;
    cout<<s/60<<' '<<s%60;
    return 0; 
}
```

### 题意
求从 $a$ 时 $b$ 分开始，直到 $c$ 时 $d$ 分，共有多少分钟。
### 解法
分类讨论 $d<b$ 和 $d\ge b$ 的情况。
+ 如果 $d\ge b$，则非常简单，小时数是 $c-a$，分钟数是 $d-b$。
+ 如果 $d<b$，则采取退位：将 $c$ 变为 $c-1$，$d$ 变为 $d+60$，然后进行如上计算。不需要取模，因为 $d<b$，$d+60-b$ 显然小于 $60$。

代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int a,b,c,d,e,f;
int main()
{
	cin>>a>>b>>c>>d;
	if(d<b){
		e=c-a-1;
		f=d+60-b;
	}
	else{
		e=c-a;
		f=d-b;
	}
	cout<<e<<" "<<f;
	return 0;
}
```

### 思路
注意到，文中的两个时间是以**时**加**分**来计算的，所以可以将其转化为多少小时或多少分钟。已经有题解写的是转化为分钟的了，那我来写一个转化为小时的吧！

其实最重要的就是时与分之间的进制转化了，某些人~~比如我~~输入的是一小时六十分钟，输出就是一小时一百分钟了，所以只需要注意比例就好了！

### 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	int a,b,c,d;cin>>a>>b>>c>>d;
	double t1=a+(b*1.0/60.0),t2=c+(d*1.0/60.0);
	double ans=t2-t1;int as=ans;
	cout<<as<<" "<<(ans-as)*60;
	return 0;
}
```

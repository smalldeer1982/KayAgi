# 【MX-X13-T1】「KDOI-12」在现实和志向中选择现实，否则会一个都得不到的。

## 题目背景

现实中的你加入了一个群聊，这个群聊总是有人说批话。

## 题目描述

定义一句话为 $A\underline{\texttt{ : }}B\underline{\texttt{ is }}C\underline{\texttt{.}}$，其中 $A,B,C$ 均为字符串，分别表示这个人的名称，句子的主语以及表语。

一句话是批话当且仅当这个人在说自己菜，也就是，$A$ 与 $B$ 相同且 $C=\underline{\texttt{weak}}$。

现在，你先判断给出的这句话是否是批话。如果是批话输出 `Plang.`，否则输出 `Not plang.`。

## 说明/提示

**【样例解释 \#1】**

在第一个样例中，$A=\text{zxx}$，$B=\text{zxx}$，$C=\text{weak}$。这句话中，$\text{zxx}$ 在说自己菜，这是一句批话。因此，应当输出 `Plang.`。

**【数据范围】**

| 测试点编号 | 特殊性质 |
|:--:|:--:|
| $1\sim7$ | A |
| $8\sim20$ | 无特殊性质 |

* 特殊性质 A：$C=\underline{\texttt{weak}}$。

对于所有数据：保证 $A,B,C$ 为长度不少于 $1$，不超过 $10$ 的由小写英文字母组成的字符串。

## 样例 #1

### 输入

```
zxx : zxx is weak.```

### 输出

```
Plang.```

## 样例 #2

### 输入

```
zxx : ou is strong.```

### 输出

```
Not plang.```

## 样例 #3

### 输入

```
zxx : zxx is cute.```

### 输出

```
Not plang.```

### 意思与思路：
联系一下犇犇的发言，如果我说我是蒟蒻（~~话说今天不宜装弱~~），那我们就可以看到如下内容（请看下图中第一条犇，请忽略时间等其他因素）：

我们可以先看到发送者，然后我们又会看见发送的内容，表示一下就变成 `jiangyunuo : jiangyunuo 是 蒟蒻`，接着，按文中的表示就变成了 `jiangyunuo : jiangyunuo is weak.`。  
如果要自己说自己菜，首先那个发送者（即文中的 $A$）与其发送的话的主语（即文中的 $B$）要相同，这才是自己说自己；其次表语（即文中的 $C$）的内容必须是 `weak`,因为你是说自己菜，不是说别的。  
所以，$A = B$ 与 $C=\underline{\texttt{weak}}$ 同时满足时，这句话就是批话。  
### 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    string a,b,c,d;  //a、b、c即原文的A、B、C，d 起辅助作用。  
    cin>>a>>d>>b>>d>>c;
    if(a==b&&c[0]=='w'&&c[1]=='e'&&c[2]=='a'&&c[3]=='k')cout<<"Plang."<<endl;  //当满足全部条件时，这句话是批话。
    else cout<<"Not plang."<<endl;  //有任何一个条件不满足时，这句话不是批话。
    return 0;
}
```

水题。

一句话是批话当且仅当这个人在说自己菜，也就是，$A$ 与 $B$ 相同且 $C=\underline{\texttt{weak}}$。

那么我们可以考虑模拟，把字符串中的 $A,B,C$ 分割出来，再按题意判断即可。

### Code
```cpp
#include <bits/stdc++.h>
#define qwq(i,a,b) for(int i=(a);i<=(b);++i)
#define qaq(i,a,b) for(int i=(a);i>=(b);--i)

using namespace std;

int main() {
	string s;
	getline(cin, s);
	
	// 分割A、B、C
	size_t col = s.find(" : ");
	size_t is = s.find(" is ");
	size_t dot = s.find(".");
	
	string A = s.substr(0, col);
	string B = s.substr(col + 3, is - (col + 3));
	string C = s.substr(is + 4, dot - (is + 4));
	
	// 判断是否是批话
	if (A == B && C == "weak") {
		cout << "Plang.";
	} else {
		cout << "Not plang.";
	}
}
```

## 思路

要判断句子 `A : B is C` 是否为批话，需检查是否满足 `A==B` 且 `C=="weak"`。通过字符串解析，提取 $A$，$B$，$C$，然后比较 `A==B` 和 `C=="weak"`。若两者都满足，输出 `Plang.`，否则输出 `Not plang.`。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	string s,a,b,c;
	getline(cin,s);
	int p1=s.find(":"),p2=s.find("is"),p3=s.find(".");
	a=s.substr(0,p1-1);
	b=s.substr(p1+2,p2-p1-3);
	c=s.substr(p2+3,p3-p2-3);
	if(a==b&&c=="weak")cout<<"Plang.";
	else cout<<"Not plang.";
	return 0;
}

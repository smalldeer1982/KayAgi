# 『STA - R6』b20

## 题目背景

我的 B 站 20 粉丝啦！

## 题目描述

小 A 在各种平台都很活跃，大家经常通过祂在各种平台的粉丝数来称呼祂。

具体地，取出网站名字的第一个字母并拼接上粉丝数量，即得到对应称呼。例如，小 A 在 Bilibili 有 $20$ 粉丝，就称其为 `B20`，小 A 在 wikidot 有 $0$ 粉丝，就称其为 `w0`。

现在，给定小 A 在活跃的网站名字和粉丝数量，输出其对应称呼。

## 说明/提示

对于 $30\%$ 的数据，保证网站名为 `bilibili`。

对于全部数据，网站名称长度不超过 $20$ 且仅含英文字母，粉丝数是不超过 $10^7$ 的非负整数。

## 样例 #1

### 输入

```
bilibili
20```

### 输出

```
b20```

## 样例 #2

### 输入

```
Luogu
463```

### 输出

```
L463```

## 作者：Licis_Subway (赞：2)

本题思路极其简单，就是第一行提取第一个字符，第二行输入后全部输出即可。

我们可以尝试使用 `getchar` 进行读取第一行字符。

就像这样：
```cpp
bool flag=false;
char first='\0';
while(char ch=getchar())
{
	if(ch=='\n') break;
	if(!flag) first=ch;
	flag=true;
}
```
其中，`flag` 用来记录是否为第一个字符。

最后我们再输出 `first` 和第二行的数。

整体代码：
```cpp
#include<cstdio>
int main()
{
	bool flag=false;
	char first='\0';
	while(char ch=getchar())
	{
		if(ch=='\n') break;
		if(!flag) first=ch;
		flag=true;
	}
	int k;
	scanf("%d",&k);
	printf("%c%d",first,k);
	return 0;
}
```

### 分析
通过题目中所举的例子可以发现：
- 在 Bilibili 上的名称 `B20` 由 `B` 和 `20` 组成，`B` 是 Bilibili 的首字母，`20` 是小 A 的粉丝数量。
- 在 witidot 上的名称 `w0` 由 `w` 和 `0` 组成，`w` 是 witidot 的首字母，`0` 是小 A 的粉丝数量。

由此可以得到：
- 这个称呼就是网站名称首字母与小 A 粉丝数量组合在一起的字符串。

直接输出即可。
### Code
时间复杂度 $\mathcal O(1)$。
```cpp
#include<iostream>
using namespace std;
signed main() {
	ios::sync_with_stdio(false),cin.tie(0),cout.tie(0);//加快读写速度
	string name;//网站名称
	int n;//粉丝数量
	cin>>name>>n;
	cout<<name[0]/*下标从0开始*/<<n;//先输出网站名称第一个字符，在输出粉丝数量
	return 0;
}

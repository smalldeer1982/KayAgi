## 题目描述

求 $1 \sim 20250412$ 中，有多少个数可以通过改变其数字顺序后含有 $2025$。

例如，$5220$、$21520$ 可以，而 $205$、$225$、$2200$、$222555111$ 则不行。

提示：要求的数就是含有至少 $1$ 个 $0$、$2$ 个 $2$、$1$ 个 $5$ 的数。

## 分析

求在 $1 \sim 20250412$ 中，有多少个数字至少包含 $1$ 个 $0$、$2$ 个 $2$ 和 $1$ 个 $5$。

## 思路

将 $1 \sim 20250412$ 中每个数进行拆位，用三个变量分别表示 $0$ 的数量、$2$ 的数量和 $5$ 的数量，如果 $0$ 的数量大于等于 $1$，$2$ 的数量大于等于 $2$，$5$ 的数量大于等于 $1$，就将答案 $ans$ 加 $1$。最后直接输出 $ans$ 即可。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int ans=0;//答案
    for(int i=1;i<=20250412;i++){//开始枚举
        int x=i,c0=0,c2=0,c5=0;//c0是0的个数，c2是2的个数，c5是5的个数
        while(x){//拆位
            if(x%10==0)c0++;
            if(x%10==2)c2++;
            if(x%10==5)c5++;
            x/=10;
        }
        if(c0>=1&&c2>=2&&c5>=1)ans++;//如果满足条件就将ans+1
    }
    cout<<ans;//输出答案
    return 0;
}
```

可以发现最后答案是 $506754$。

## 题目分析

题目要求我们求 $1 \sim 20250412$ 中，有多少个数可以通过改变其数字顺序后含有 2025。其实也就是求有多少个含有至少 $1$ 个 $0$、$2$ 个 $2$、$1$ 个 $5$ 的数。~~（这题目里面不是有吗）~~

## 解决方案

这里我用的是桶排序的方法，统计从 $1 \sim 20250412$ 中每一个数所含的各个数字数量。当这个数中 $0$ 的数量大于 $1$ 、$2$ 的数量大于 $2$ 、$5$ 的数量大于 $1$  时，我们便认定这个数是我们要求的数之一，所以使计数器加一，最后输出。

## 代码

话不多说，上代码。
```cpp
#include<iostream>//标准开头 
using namespace std;
int a[10];//数组a用于统计i中各数字的个数 
int main()
{
	int t,sum=0;//t用于分离i的每一位数字;sum为计数器 
	for(int i=1;i<=20250412;i++)//从1~20250412枚举i 
	{
		if(i<1000)continue;//当i为一、二、三位数时，无法凑成四位数2025

		while(t!=0)
		{
			a[(t%10)]++;
			t/=10;

		if(a[0]>=1&&a[2]>=2&&a[5]>=1)
		{
			sum++;
		}//当符合条件时计数器加一 
		for(int j=0;j<10;j++)a[j]=0;//重置a 
	}
	cout<<sum;//输出
```

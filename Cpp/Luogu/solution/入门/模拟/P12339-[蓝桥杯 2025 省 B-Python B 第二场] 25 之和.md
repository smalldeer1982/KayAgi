# [蓝桥杯 2025 省 B/Python B 第二场] 25 之和

## 题目描述

小蓝最近对求和很着迷，给定一个正整数 $n$，他想求从 $n$ 开始的连续 $25$ 个整数的和，即 $n + (n + 1) + (n + 2) + \cdots + (n + 24)$，请帮帮他吧。

## 说明/提示

### 评测用例规模与约定

- 对于 $40\%$ 的评测用例，$1 \leq n \leq 100$；
- 对于所有评测用例，$1 \leq n \leq 10000$。

## 样例 #1

### 输入

```
1```

### 输出

```
325```

## 样例 #2

### 输入

```
100```

### 输出

```
2800```

## 作者：Clare613 (赞：2)

## 思路
本题考查循环结构，模拟 $n$ 到 $n+24$，把和加起来输出即可。
```cpp
#include<bits/stdc++.h>
using namespace std;

signed main(){
	cin.tie(0)->sync_with_stdio(0);
	int n,ans;
	cin>>n;
	for(int i=1;i<=25;i++){
		ans+=n;
		n++;
	}
	cout<<ans;
	return 0;
}
```

## 题目思路：
这道题让我们计算 $n+(n+1)+(n+2)+ \dots +(n+24)$ 这一长串数字。这时，我们想到了一个好思路。我们先把括号去掉，整理一下不难发现，此时有 $25$ 个 $n$ 和 $1$ 到 $24$ 的数字，相加即可通过此题。

## 正解：

```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long
int n,ans;
signed main(){
	ios::sync_with_stdio(0);
	cin.tie(0),cout.tie(0);
	cin>>n;
	for(int i=1;i<=25;i++) ans+=n;
	for(int i=1;i<=24;i++) ans+=i;
	cout<<ans<<"\n";
	return 0;
}
```

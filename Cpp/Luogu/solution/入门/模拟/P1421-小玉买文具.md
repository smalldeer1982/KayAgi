# 小玉买文具

## 题目描述

班主任给小玉一个任务，到文具店里买尽量多的签字笔。已知一只签字笔的价格是 $1$ 元 $9$ 角，而班主任给小玉的钱是 $a$ 元 $b$ 角，小玉想知道，她最多能买多少只签字笔呢。


## 说明/提示

#### 数据规模与约定

对于全部的测试点，保证 $0 \leq a \leq 10^4$，$0 \leq b \leq 9$。

## 样例 #1

### 输入

```
10 3
```

### 输出

```
5
```

首先，签字笔的价钱是1.9元，而班主任给了小玉a.b元。

看到这样的题目，初学时的我是崩溃的，现在看来，其实1.9元就是19角嘛，那a.b元不\*就是a\*10+b角嘛（1元=10角），这样想想这道题压根没难度。


接下来解决下一个问题：除不尽怎么办呢？比如说现在我有38角，我可以买两支笔没错，但如果我只有37角，怎么办呢？我们在小学时也碰到过这样的题，买了一支笔以后剩下18角，不够买一支笔了，那么我们要懂得舍弃，只买一支笔。


可是这样的思想过程要如何在代码实现呢？我们来举个例子，37/19，在现实生活中这道题的得数应该是1点几，接近2，四舍五入来说应该是2，但计算机在没有特殊指令的情况下只会保留整数（用了比较浅显的语言讲，神犇们不要在意），就算是1.9999999999的现实答案，到了计算机里还是1。


```cpp
#include<cstdio>
int main()
{
    int a,b;//a元b角
    int c;//转换后的班主任给的价钱
    scanf("%d %d",&a,&b);
    c=a*10+b;//把a元b角转换为a*10+b角
    printf("%d",c/19);//19在这里表示19角，原来是1.9元，不过已经被转换掉了
    return 0;
}
```

## P1421 小玉买文具 题解
这道题是一道比较简单的题目，适合初学者拿来练习数学中的转化，还是很典型了。
### 思路

题目中是给的**元和角**，可以把它转化为多少**元**，~~另一位大佬的题解是转换为角~~，将样例拿来模拟，小玉的老师给了她 $10$ 元 $3$ 角，转化后就成为了 $10.3$ 元，然后稳稳地一步一步减 $1.9$ 元。~~虽然但是，数据小，随便跑。~~

### 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	int a,b,ans=0;cin>>a>>b;
	double p=1.9,sum=a+double(b*1.0/10.0);
	while(true){
		if(sum<p) break;
		sum-=p;ans++;
	}
	cout<<ans;
	return 0;
}
```

想要优化的可以将 $p$ 和 $sum$ 乘十后直接除，这就类似于转化为角了。

### 优化后的代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	int a,b,ans=0;cin>>a>>b;
	double p=1.9,sum=a+double(b*1.0/10.0);
	cout<<int(sum*10)/int(p*10);
	return 0;
}
```

### 题目大意

已知每支笔价格 $1$ 元 $9$ 角，问 $a$ 元 $b$ 角能购买多少只笔。

### 解题思路

我们已知 $1$ 元等于 $10$ 角，为了统一单位，我们可以将 $a$ 元 $b$ 角转化为 $a \times 10 + b$ 角。同理可得一支笔为 $19$ 角。

## 题目简述

这道题让我们根据班主任给小玉的钱数，计算能买多少只 $1$ 元 $9$ 角的签字笔。

## 思路

在现实的数学大题中，我们可以很轻易地利用小数除法解开这道题，即 $\lfloor (a + b \div 10.0) \div 1.9 \rfloor$。使用 C++ 语言编写就是：

```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
    int a, b;
    cin >> a >> b;
    double tot = a + b / 10.0;//转换为浮点数，以元为单位
    int ans = floor(tot / 1.9); //浮点数除法后向下取整
    cout << ans;
    return 0;
}
```


同样的，我们可以使用一种方法，不需要使用 `double` 类型，对入门者更加友好：

- 把小玉的钱都转化为**以角为单位**。即 $a \times 10 + b$。
  
- 用**总钱数**除以每只笔的价格（即**单价**），得到能买的笔的数量。

## 重点代码解释

- **计算总钱数（以角为单位）**：
  
``` cpp
int m = a * 10 + b;
```
首先，我们要想一下，元和角的进率是什么？是的， $1$ 元 = $10$ 角。

因此，我们把以元为单位的变量 $a$ 乘以 $10$ 转化为以角为单位。再加上以角为单位的 $b$，得到以角为单位的总钱数 $m$。

- **计算能买的笔的数量**：
  
```cpp
int ans=m/19;
```

在解释本行之前，我们需要先了解，整型变量的除法特征：**它的结果会直接舍弃小数部分，只保留整数部分**。通俗来讲就是常说的去尾。

而我们知道，签字笔的单价是 $1$ 元 $9$ 角，也就是 $19$ 角。因此，我们只需要用总钱数 $m \div 19$，就能得到能买的签字笔数量啦！

或许有人会问：那小数部分怎么办？

那么就回到刚才说的**整型变量的除法特征**了。因为是整数除法，会自动舍去小数部分。因此，得到的商就是能买的笔的数量，余数部分因为钱不够买一只笔，所以直接舍去啦！

说了这么多，该上代码了！

## 实现代码

不要复制，这只是参考。

```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	int a,b;
	cin>>a>>b;
	int m=a*10+b;
	int ans=m/19;
	cout<<ans;
	return 0;
}

```

# 题解：P1421 小玉买文具
## 题目大意
一只笔价格 $1$ 元 $9$ 角，有 $a$ 元 $b$ 角钱，求最多能买多少只签字笔。

## 题目分析
这道题，我们只需要把钱的单位转化为角，并用钱数除以笔的单价，就可以解决问题了。

因为 $1$ 元等于 $10$ 角，所以 $1$ 元 $9$ 角等于 $1 \times 10+9=19$ 角，$a$ 元 $b$ 角等于 $a \times 10+b=10a+b$ 角。能购买的笔的数量就等于 $\lfloor{\frac{10a + b}{19}}\rfloor$ 支。

## 代码
拒绝抄袭，从我做起！

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a, b;
    cin >> a >> b; // 输入
    int num = (a * 10 + b) / 19; // 计算答案
    cout << num; // 输出
    
    return 0;
}
```

[题目](https://www.luogu.com.cn/problem/P1421)  
**思路**：  
首先会看到元和角的单位不统一，那我们可以先转化为角。那小玉得到的钱就是 $10\times a+b$ 了。   
根据小学数学知识，我们知道：总价除以单价等于个数。这里的个数可能会有小数的情况，但是笔肯定没有小数支，我们需要向下取整来舍弃多余的小数部分。（向下取整的意思是取不大于这个数的最大整数，比如 $3.5$ 向下取整就是 $ 3$。）不过我们也并不需要写特别的判断来向下取整，因为 ```int``` 类型就是**默认向下取整**的。  
那么整理一下，我们首先将小玉得到的钱和签字笔的单价的单位统一成角，然后用小玉钱除以单价就是答案啦。    
时间复杂度显然是 $O(1)$ 的。  
**code:**
```
#include<iostream>
using namespace std;
int a,b,c,sum;
int main()
{
	cin>>a>>b;
	sum=a*10+b;//统一单位
	c=sum/19;
	cout<<c;
	return 0;
}
```

### 思路分析

首先，读入两个整数 $a,b$。

那么**班主任给小玉的钱**就是 $a$ 元 $b$ 角，但是单位**不统一**，怎么计算呢？

我们可以将**班主任给小玉的钱**和**一只签字笔的价格**转化成以**角**作为单位的数，这样直接**整除**即可。

那么，**班主任给小玉的钱**就是 $a \times 10 + b$ **角**， **一只签字笔的价格**就是 $1 \times 10 + 9 = 19$ **角**。

所以，直接用 $a \times 10 + b$ 整除 $19$ **取下整**即可。

### 代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
int a, b;
int main() {
	scanf("%d%d", &a, &b);
	printf("%d\n", (a * 10 + b) / 19);
	return 0;
}
```

# [COCI 2012/2013 #1] DOM

## 题目描述

给定一个仅包含大写字母的单词，删除该单词中的以下九个字母：$\texttt{C,A,M,B,R,I,D,G,E}$。

## 说明/提示

#### 【数据范围】

对于 $100\%$ 的数据，保证 $3 \le$ 单词长度 $\le 100$。

#### 【说明】

本题分值按 COCI 原题设置，满分 $50$。

题目译自 **[COCI2012-2013 CONTEST #1](https://hsin.hr/coci/archive/2012_2013/contest1_tasks.pdf)  _T1 DOM_**。

# 思路

这道题还是很水的，我来发一下我的题解吧。

这道题就是用 Char 类型字符串判断是否符合  $\texttt{C,A,M,B,R,I,D,G,E}$
的其中一个，如果是这其中几个，那么就将这个单个字符跳过，否则跳过。

这样就可以 AC 了

AC 代码：
```cpp
#include<cstdio>
#include<cstring>
#include<iostream>

using namespace std;

int main()
{
	char s[1000];
	cin>>s;
	for(int i=0;i<=strlen(s);i++){
		if(s[i]=='C' || s[i]=='A' || s[i]=='M' || s[i]=='B' || s[i]=='R' || s[i]=='I' || s[i]=='D' || s[i]=='G'|| s[i]=='E')
			continue;
		else
			putchar(s[i]);
	}
	return 0;
}
```

### [P7583 [COCI2012-2013#1] DOM](https://www.luogu.com.cn/problem/P7583)

#### 题意

- 给定一个字符串，删除该字符串中的以下九个字母：$\texttt{C,A,M,B,R,I,D,G,E}$。

- $3\le$ 字符串长度 $\le 100$。

#### 分析

十分基础的一道字符串题目。

我的做法是先将整个字符串读入，然后一个字符一个字符比较，如果当前字符为 $\texttt{C,A,M,B,R,I,D,G,E}$ 中任意一个就不输出，否则输出。这样就达到了删除的目的。

#### 代码

code below ↓

```cpp
#include<bits/stdc++.h>
#define FOR(i,j,k)  for(int i=(j);i<=(k);i++)
using namespace std;
char s[110];
int main()
{
	cin>>s;
	int len=strlen(s);
	for(int i=0;i<len;i++)
		if(!(s[i]=='C'||s[i]=='A'||s[i]=='M'||s[i]=='B'||s[i]=='R'||s[i]=='I'||s[i]=='D'||s[i]=='G'||s[i]=='E'))
			cout<<s[i];
	puts("");
	return 0;
}

```

# $\text{Problems}$
给定一个仅包含大写字母的单词，删除该单词中的以下九个字母：$\texttt{C,A,M,B,R,I,D,G,E}$。
# $\text{Answer}$
用 `for` 循环跑一遍，当遇到 $s_{i}=\texttt{C,A,M,B,R,I,D,G,E}$ 中的任意一个时，跳过这次循环，即可达成删除的效果。

排除九个字符后就可以直接输出了。
# $\text{Code}$
```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
int main()
{
    getline(cin,s);
    for(int i=0;i<s.length();++i)
    {
        if(s[i]=='C') continue;
        if(s[i]=='A') continue;
        if(s[i]=='M') continue;
        if(s[i]=='B') continue;
        if(s[i]=='R') continue;
        if(s[i]=='I') continue;
        if(s[i]=='D') continue;
        if(s[i]=='G') continue;
        if(s[i]=='E') continue;
        cout<<s[i];
    }
    return 0;
}
```

## 算法 $\bf 1$

比较给出字符串与字符串 `CAMBRIDGE` 是否有相同的字符，有则不输出此字符。

```cpp
string cp="CAMBRIDGE";bool f;
int main() {
  string s;cin>>s;
  for(auto i:s){
    f=0;
    for(auto j:cp){
      if(i==j){f=1; break;}
    }
    if(f)continue;
    else cout<<i;
  }
}
```

## 算法 $\bf 2$

同样比较字符串，我们可以发现判断是否存在可以使用 `find()` 函数代替。

```cpp
string cp="CAMBRIDGE";
int main() {
  string s;cin>>s;
  for(auto i:s){
    if(cp.find(i)==cp.npos)cout<<i;
  }
}
```

# [蓝桥杯 2020 省 AB1] 解码

## 题目描述

小明有一串很长的英文字母，可能包含大写和小写。

在这串字母中，有很多连续的是重复的。小明想了一个办法将这串字母表达得更短：将连续的几个相同字母写成字母 + 出现次数的形式。 例如，连续的 $5$ 个 `a`，即 `aaaaa`，小明可以简写成 `a5`（也可能简写成 `a4a`、`aa3a` 等）。

对于这个例子：`HHHellllloo`，小明可以简写成 `H3el5o2`。为了方便表达，小明不会将连续的超过9个相同的字符写成简写的形式。

现在给出简写后的字符串，请帮助小明还原成原来的串。

## 说明/提示

对于所有评测用例，字符串由大小写英文字母和数字组成，长度不超过 $100$。请注意原来的串长度可能超过 $100$。

蓝桥杯 2020 第一轮省赛 A 组 F 题（B 组 G 题）。

## 样例 #1

### 输入

```
H3el5o2```

### 输出

```
HHHellllloo```

## 作者：minVan (赞：28)

**解题思路**

对于每个大小写字母，如果下一个字符也为大小写字母，则代表这个字母只有一个，直接输出即可；如果下一个字符为数字 $k$，则输出 $k$ 个这个字母。

**AC 代码，请勿抄袭**

```cpp
#include <bits/stdc++.h>
using namespace std;
string s;
int main() {
    cin >> s;
    for(int i = 0; i < s.size(); i++) {
        if(s[i] < '0' || s[i] > '9') {
            if(s[i + 1] < '0' || s[i + 1] > '9') {
                cout << s[i];
                continue;
            }
            for(int j = 1; j <= s[i + 1] - '0'; j++) {
                cout << s[i];
            }
        }
    }
    return 0;
}

```

## P8706 [蓝桥杯 2020 省 AB1] 解码 题解

考虑用 `string` 类型来实现。

输入字符串后，从头到尾进行判断，如果第 $i$ 个字符是大小写字母，则直接输出，如果是一个数字，证明这是代表第 $i-1$ 个字符 $x$ 的个数。由于我们前面已经输出了一个 $x$ 了，所以我们还差第 $i$ 个字符对应的数字减一个 $x$ 没有输出，我们新做一个循环来输出就可以了。

本题的 AC 代码如下：

```cpp
#include<iostream>
#include<string>
using namespace std;
string a;
int main(){
	cin>>a;
	int len=a.size();//len表示字符串a的长度
	for(int i=0;i<len;i++){
		if(a[i]<='9'&&a[i]>='0'){//判断数字的情况
			int n=a[i]-'0';
			for(int j=2;j<=n;j++){//输出a[i]-1个a[i-1]
				cout<<a[i-1];
			}
		}
		else cout<<a[i];//判断字母的情况
	}
	return 0;
}
```

## 作者：Mortidesperatslav (赞：12)

这是一道模拟题，由于保证数字不大于 $9$，那就好办了。

我们先读入一个字符串，只要遇到一个字符，就判断下一个字符是什么，如果是数字（姑且称作 $n$）就重复输出 $n$ 次当前字符，否则输出一次。

代码如下：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    string i;
    int tmp,p=0;
    cin>>i;
    while(p<i.length()){
        char ch=i[p];
        if(i[p+1]>='0'&&i[p+1]<='9'){
            tmp=i[p+1]-48;
            p++;
        }else{
            tmp=1;
        }
        while(tmp--)cout<<ch;
        p++;
    }
}
```

      } else {

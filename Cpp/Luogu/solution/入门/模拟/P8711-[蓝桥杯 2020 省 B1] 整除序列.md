# [蓝桥杯 2020 省 B1] 整除序列

## 题目描述

有一个序列，序列的第一个数是 $n$，后面的每个数是前一个数整除 $2$，请输出这个序列中值为正数的项。

## 说明/提示

对于 $80\%$ 的评测用例，$1\le n\le10^9$。

对于所有评测用例，$1\le n\le10^{18}$。

蓝桥杯 2020 第一轮省赛 B 组 F 题。

## 样例 #1

### 输入

```
20```

### 输出

```
20 10 5 2 1```

#### 题目大意

有一个序列，序列的第一个数是 $n$，后面的每个数是前一个数整除 $2$，请输出这个序列中值为正数的项。

#### 解题思路

序列的第一个数为 $n$，所以可以先直接输出 $n$。

因为序列到 $1$ 就不能再除了，所以要在 $n$ 不等于 $1$ 的情况下不断除以 $2$，并输出 $n$。 

#### 代码

[AC 记录](https://www.luogu.com.cn/record/102820420)

```c++
#include<bits/stdc++.h>
#define ll long long
using namespace std;
ll n;
int main() {
	cin>>n;
	cout<<n<<" ";
	while(n!=1){
	 	n/=2;
        cout<<n<<" ";
    }
	return 0;
}
```

### 描述

有一个序列，序列的第一个数是 $n$，后面的每个数是前一个数**整除** $2$，请输出这个序列中值为正数的项。

### 思路

这是一道面向样例题，我们分析样例发现，这其实就是短除法，一直循环操作输出该数，再除以 $2$ 就行。

然后我们发现，请输出 `序列中值为正数的项`，也就是说，不输出 $0$，那我们短除法的终止条件为 $0$。

### 解法

按照思想说的解就行。

```cpp
#include <iostream>
using namespace std;

int main() {
    
    int n;
    cin >> n;
    
    while(n){
        cout << n << " ";
        n /= 2;
    }
    cout << endl;
    
    return 0;
}
```

然后你就会惊喜地发现：$80$ 分！

请看看最底下的两行字：

>对于 $80\%$ 的评测用例，$1\le n\le10^9$。
>
>对于所有评测用例，$1\le n\le10^{18}$。

所以我们要开 long long。

```cpp
#include <iostream>
using namespace std;

int main() {
    
    long long n;
    cin >> n;
    
    while(n){
        cout << n << " ";
        n /= 2;
    }
    cout << endl;
    
    return 0;
}
```

#include <iostream>

		N /= 2; //整除2
	}

这是一道简单的**模拟**题。

***

### 题意简述

给出一个整数 $n$，不断用 $2$ 整除 $n$，直到结果为非正，按顺序输出整除过程中得到的所有正数。

***

### 解法

数据范围为 $10^{18}$，显然要开 `long long`，而且不能使用数组存储运算过程中的数，于是考虑算出来一个就输出一个。

显然，计算过程中不可能出现负数，只会有 $0$。而当算出一个 $0$ 之后，接下来所有数就都是 $0$，于是出现 $0$ 就可以结束程序。

***

### 代码

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;
int n;
signed main(){
    scanf("%lld",&n);//输入
    while(n!=0){//一直循环，直到 n 非正（变为 0）
        printf("%lld ",n);//输出
        n/=2;//用 2 整除
    }
	return 0;
}
```

### P8711 [蓝桥杯 2020 省 B1] 整除序列 题解

### 思路：

循坏模拟 $n$ 除以 $2$，如果当前的 $n$ 为 $1$，则暂停循环。

**$1\le n\le 10^{18}$，一定要开 `long long`。**

### Code:

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
  long long n;
  cin >> n;
  for (; ; ) {
    cout << n << ' ';
    n /= 2;
    if (n == 1) {
        cout << "1";   // 最后一次输出
        break;
    }
  }
  return 0;
}

```

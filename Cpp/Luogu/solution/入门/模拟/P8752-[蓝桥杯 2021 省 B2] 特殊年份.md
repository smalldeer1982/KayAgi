# [蓝桥杯 2021 省 B2] 特殊年份

## 题目描述

今年是 2021 年，2021 这个数字非常特殊, 它的千位和十位相等, 个位比百位大 $1$，我们称满足这样条件的年份为特殊年份。

输入 $5$ 个年份，请计算这里面有多少个特殊年份。


## 说明/提示

**【样例说明】** 

2021 和 9899 是特殊年份，其它不是特殊年份。 

蓝桥杯 2021 第二轮省赛 B 组 F 题。

## 样例 #1

### 输入

```
2019
2021
1920
2120
9899```

### 输出

```
2```

## 题意
今年是 2021 年，2021 这个数字非常特殊，它的千位和十位相等，个位比百位大 $1$，我们称满足这样条件的年份为特殊年份。

输入 $5$ 个年份，请计算这里面有多少个特殊年份。

## 分析
这道题我们把每个年份当作 $4$ 个字符来处理。

输入 $a,b,c,d$ 这 $4$ 个字符，分别为年份的千位、百位、十位、个位。

判断 $a$ 是否与 $c$ 相等，$d$ 是否比 $b$ 大 $1$ 即可。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int ans=0;
	for(int i=1;i<=5;i++)
	{
		char a,b,c,d;
		cin>>a>>b>>c>>d;
		if(a==c&&d-b==1) ans++;
	}
	cout<<ans;
   	return 0;
}
```

#### 题目大意

输入 $5$ 个年份，请计算这里面有多少个千位和十位相等, 个位比百位大 $1$ 的年份。

#### 解题思路

将每一个年份按[分离数位规则](https://www.luogu.com.cn/blog/677671/fen-li-shuo-wei-gui-ze)把每一位都分离，赋给 $a,b,c,d$ 四个数，再判断是否满足千位和十位相等, 个位比百位大 $1$；如果满足，计数器加一；最后输出计数器即可。



#### 代码

[AC记录](https://www.luogu.com.cn/record/97692632)

```c++
#include<bits/stdc++.h>
#define ri register int
using namespace std;
int s[6],ans=0;
int main() {
	for(ri i=1;i<=5;i++){
		cin>>s[i];
		int a=s[i]%10,b=s[i]%100/10,c=s[i]/100%10,d=s[i]/1000;
		if(b==d&&a-c==1)
			ans++;
	}
	cout<<ans;
	return 0;
}
```

## 作者：guozhetao (赞：2)

## 思路
先提取出年份的每一个数，再按照题意去模拟。

首先我用 $a$ 表示四位的十进制数，$b1$ 表示 $a$ 的个位，$b2$ 表示 $a$ 的十位，$b3$ 表示 $a$ 的百位，$b4$ 表示 $a$ 的千位。

很明显：

- $b1 = a \bmod 10$.
- $b2 = \dfrac{a - b1}{10} \bmod 10 $.但是因为在 c++ 中整数除法会去掉小数部分，所以 $b2 = \lfloor \dfrac{a}{10}\rfloor \bmod 10$.

同理可得：

- $b3 = \lfloor \dfrac{a}{100}\rfloor \bmod 10 $.
- $b4 = \lfloor \dfrac{a}{1000}\rfloor$.

得出这几个数后，我们再用题目的方式模拟，最后得出答案。

## 代码
```cpp
int ans = 0;
int main() {
	for(int i = 1;i <= 5;i++) {//循环
		int a = read();
		int b1 = a % 10,b2 = (a / 10) % 10,b3 = (a / 100) % 10,b4 = a / 1000;//提取出每一位
		if(b2 == b4 and b1 == b3 + 1) ans++;//按照题意模拟
	} 
	printf("%d\n",ans);
    return 0;
}

```

## 作者：KK_lang (赞：1)

## 大意

定义千位与十位相同、各位比百位多 $1$ 的四位数为“特殊数字”，给定 $5$ 个四位数，求其中又多少个“特殊数字”。

## 思路

 只有 $5$ 个数、而且是 $4$ 位数、时限 $1$ 秒，所以我们不用考虑效率（时间复杂度）。
 
我们观察，对于四位数的操作只有逐位特判，所以其实四位数本身是多少不太重要，最重要的是我们要取到四个位上的数。

四位数之间没有空格，我们考虑用 ```char``` 来存四位数。

```cpp
char a2, b2, c2, d2;
cin >> a2 >> b2 >> c2 >> d2;
```

我们就把四位数存在了 $a2$、$b2$、$c2$、$d2$ 中。

接下来，字符转整数，直接用 ASCII 码相减就可以实现

```cpp
int a = a2 - '0', b = b2 - '0', c = c2 - '0', d = d2 - '0'; 

```

然后就是特判啦！特殊数字条件：```a == c && b + 1 == d```。

外层套一个循环，做 $5$ 次，就可以了。

## AC Code

整体代码献上。

都讲的很明白啦，就不写代码注释了。

```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
    int ans = 0;
    for (int i = 1; i <= 5; i++)
    {
        char a2, b2, c2, d2;
        cin >> a2 >> b2 >> c2 >> d2;
        int a = a2 - '0', b = b2 - '0', c = c2 - '0', d = d2 - '0'; 
        if (a == c && b + 1 == d) ans++;
    }
    cout << ans << endl;
    return 0;
}
```

## Solution

这道题用 `int` 还是比较麻烦的，我们考虑使用字符串 `string`，它在 `cstring` 库里。

```cpp
#include<bits/stdc++.h>
using namespace std;
string s;//使用字符串
int ans;

	for(int i = 1;i<=5;i++){//只需要输入5个数
		cin>>s;
		if(s[0]==s[2]/*千位是否和十位相等*/&&s[3]==s[1]+1/*个位是否比百位大 1*/;) ans++;
	}
	cout<<ans<<endl;	
	return 0;
}

## 作者：Algo_Sl_al (赞：0)

### 题意简述

[题目传送门](https://www.luogu.com.cn/problem/P8752)

输入五个年份，判断它们是不是“特殊年份”。

## 思路

输入五个数可以使用 `for` 循环来读入，之后对每个年份进行判断。

判断方式如下：一个四位数的各位分别为 `n%10`（个位），`n/10%10`（十位），`n/100%10`（百位），`n/1000`（千位）。根据题意，“特殊年份”应该满足以下两个条件：

**1.千位等于十位**，可以用 `n/1000==n/10%10` 判断；

**2.个位比百位大1**，可以用 `n%10-1==n/100%10` 判断。

于是，我们就能写出代码：

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,cnt=0;//记录个数
    for(int i=0;i<5;i++){//循环输入
        cin>>n;
        if((n/1000==n/10%10)&&(n%10-1==n/100%10))cnt++;//判断+计数
    }
    cout<<cnt;
    return 0;
}
```

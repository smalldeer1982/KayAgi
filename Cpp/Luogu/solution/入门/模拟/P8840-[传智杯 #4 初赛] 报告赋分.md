# [传智杯 #4 初赛] 报告赋分

## 题目描述

花栗鼠科技大学的计算机组成原理实验最终的结课考核方式是提交一份报告。

然而作为任课老师，萝老师不希望大家过于内卷，所以指定了如下规定：

每份报告有一个卷面基础分 $a$。

在此基础上：

- 若是报告字数低于 $16$ 页，则扣 $10$ 分，如果分数低于 $0$ 分了，则记作 $0$ 分。

- 若是报告字数超过 $20$ 页，每超过 $1$ 页扣 $1$ 分 ，直到分数扣到 $0$ 分为止。

现在你知道了一份报告的卷面基础分 $a$ 和它的页数 $p$ ，请你计算这份报告的最终得分。我们保证 $1 \leq a \leq 100,1 \leq p \leq 50$.

## 样例 #1

### 输入

```
2
70 17
80 10```

### 输出

```
70
70```

## 作者：Zvelig1205 (赞：13)

分支结构的练习题。

首先需要解决的是多组询问。

显然可以用一个 `for` 循环解决。

但一个更为简便的方法是这样：

```
while(T--)
{
	...
	...
	...
}
```

这里只做简介，不深究，有兴趣的可以自行百度。

然后，由于只有两个扣分选项，可以直接用两个 `if` 判断。

当然，最低分为 $0$ 分，所以每次扣分都需要和 $0$ 取个 $\max$。

AC Code:

```cpp
#include<iostream>
#include<algorithm>
using namespace std;
int main()
{
	int qwq;cin>>qwq;
	while(qwq--)
	{
		int a,p;cin>>a>>p;
		if(p<16)a=max(0,a-10);
		if(p>20)a=max(0,a-p+20);
		cout<<a<<'\n';
	}
	return 0;
}
```

#include<bits/stdc++.h>
using namespace std;
int main() {
	int t;
	cin >> t;
	for (int i = 1; i <= t; i++) {
		int a, p;
		cin >> a >> p;
		if (p < 16) a -= 10; // 若是报告字数低于 16 页，则扣 10 分，如果分数低于 0 分了，则记作 0 分。
		if (p > 20) a -= p - 20; // 若是报告字数超过 20 页，每超过 1 页扣 1 分 ，直到分数扣到 0 分为止。
		if (a <= 0) cout << 0 << endl; // 加个特判，如果分数被扣成了负数，还是输出 0。
		else cout << a << endl;
	}
	return 0; // 最后收尾
}

## 作者：InversionShadow (赞：4)

### P8840 [传智杯 #4 初赛] 报告赋分 题解

[题目传送门](https://www.luogu.com.cn/problem/P8840)

### 题意：

$T\ (1 \leq T \leq 1000)$ 组数据，每次输入 $a\ (1 \leq a \leq 100)$ 和 $p\ (1 \leq p \leq 50)$。

+ 若是 $p$ 小于 $16$，则 $a$ 减 $10$，如果 $a$ 小于 $0$，则记作 $0$ 。

+ 若是 $p$ 大于 $20$，每超过 $1$，$a$ 减 $1$，直到 $a$ 减到 $0$ 为止。

### 思路：

按题意模拟，根据题目给出的两种情况分类讨论即可。

根据第一条线索，我们可以推出以下代码：
```cpp
if (p < 16) {
  cout << max(0, a - 10) << endl;  
}   

```

这里之所以要用 $a\ -\ 10$ 与 $0$ 取最大值，是因为题目说如果分数低于 0 分了，则记作 0 分。

举个例子：

+ $a$ 等于 $20$，$a\ -\ 10$ 等于 $10$，$10$ 与 $0$ 取最大值是 $10$，所以 $a$ 为 $10$。

+ $a$ 等于 $5$，$a\ -\ 10$ 等于 $-5$，$-5$ 与 $0$ 取最大值是 $0$，所以 $a$ 为 $0$。

根据第二条线索，我们可以推出以下代码：

```cpp
if (p > 20) {
  cout << max(0, a - (p - 20)) << endl;    
}
```
这里的 $p\ -\ 20$ 指的是超过 $20$ 的部分，每超过 $1$，$a$ 减 $1$，所以这里其实是 ```max(0, a - 1 * (p - 20))```
，不过为了简便，$1$ 和 ```*``` 可以不写。

**注意：**

大家想一想，如果 $p$ 既大于 $16$，又小于 $20$，那该怎么办？

对了，当然是不做操作。

所以有以下代码：

```cpp
else {  
  cout << a << endl;
}
```


### 完整代码:

```cpp
#include <bits/stdc++.h>

using namespace std;

int t, a, p;

int main() {
  cin >> t;
  while (t--) {
    cin >> a >> p;
    if (p < 16) {
      cout << max(0, a - 10) << endl;  
    } else if (p > 20) {
      cout << max(0, a - (p - 20)) << endl;    
    } else {  
      cout << a << endl;
    }
  }
  return 0;
}

```

## 作者：REAL_曼巴 (赞：3)

这道题是一个简单的模拟题，考察对判断语句的应用。

先输入 $a$ 和 $b$ 两个变量，完全按照题来写程序。

第一个要求：

- 若是报告字数低于 $16$ 页，则扣 $10$ 分 。

代码可以这么写：

```
if(b<16) a-=10;

```

- 若是报告字数超过 $20$ 页，每超过 $1$ 页扣 $1$ 分 。

代码可以这么写：

```
if(b>20) a-=(b-20);

```

最后他还说，如果分数低于 $0$ 分了，则记作 $0$ 分。直接特判就好了

代码是这样：

```
#include<iostream>
#include<cstdio>
using namespace std;
int main(){
    
    int n;
    cin>>n;
    while(n--){
        int a,b;
        cin>>a>>b;
        if(b<16)
            a-=10;
        if(b>20)
            a-=(b-20);
        if(a<0)cout<<0<<endl;
        else cout<<a<<endl;
    }
}

# [传智杯 #4 初赛] 竞争得分

## 题目描述

为了鼓励大家写出更好的作业，花栗鼠科技大学（Hualishu University of Science and Technology, HUST）的组原实验采用了竞争得分的方式。

具体来说，假设有 $n$ 个人提交了作业，并且其中原始得分最低的人记作 $a_{\min}$ ，原始得分最高的人记作 $a_{\max}$，第 $i$ 个人的原始得分为 $a_i$，那么第 $i$ 个人的得分就是：
$$100 \times \frac{a_i-a_{\min}}{a_{\max}-a_{\min}}$$

由于成绩系统的问题，最终录入的成绩只能是整数，**采用直接去掉小数部分的方法**。

## 样例 #1

### 输入

```
3
1 2 3```

### 输出

```
0 50 100```

# 题解

## 作者：Dws_t7760 (赞：14)

我们首先定义一个数组 $a$，记录每个人的原始得分，再定义两个变量 $minn$ 和 $maxx$，分别记录原始得分的最大值和最小值。我们可以用公式 
$$100 \times \frac{a_i-a_{min}}{a_{max}-a_{min}}$$

来求出每个人的得分。注意最终录入的成绩只能是整数，需要直接去掉小数部分，也就是向下取整。

AC代码：

```cpp
#include<iostream>
#include<cmath>
using namespace std;
int n,a[1000],maxx,minn=10000;
int main()
{
	cin>>n;
	for(int i=0;i<n;i++) {
		cin>>a[i];
		if(a[i]>maxx) maxx=a[i];
		if(a[i]<minn) minn=a[i];
	}
	for(int i=0;i<n;i++) cout<<floor(100.0*(a[i]-minn)/(maxx-minn))<<' ';
	return 0;
}
```

忽然发现很久之前水的比赛加到主题库里了。

题意十分明了，直接模拟即可。

感觉像是语言篇的检测。

首先，边输入边找到最大值和最小值。

```cpp
cin>>n>>a[1];
maxn=minn=a[1];
for(int i=2;i<=n;i++)
{
	cin>>a[i];
	if(a[i]>maxn)maxn=a[i];
	if(a[i]<minn)minn=a[i];
}
```

这段代码的含义就是：

先假设第一个数为整个数列的最大值、最小值。然后在 `for` 的过程中，判断我们的假设是否成立。如果不成立，就推翻，将现在的值设置为最大（或最小）值。

然后，由于分母为 $a_{max}-a_{min}$，是定值，可以先求出来。

所以，再一次 `for` 循环，利用题目中的公式：

$$100\times \dfrac{a_i-a_{min}}{a_{max}-a_{min}}$$

就可以求出来每一个人的得分了。求出来之后便可以直接输出了。

AC Code:

```cpp
#include<iostream>
using namespace std;
int n,maxn,minn;
int a[1001];
int main()
{
	cin>>n>>a[1];
	maxn=minn=a[1];
	for(int i=2;i<=n;i++)
	{
		cin>>a[i];
		if(a[i]>maxn)maxn=a[i];
		if(a[i]<minn)minn=a[i];
	}
	int fenmu=maxn-minn;
	for(int i=1;i<=n;i++)
		cout<<100*(a[i]-minn)/fenmu<<' ';
	return 0;
}
```

## 作者：xvl_ (赞：1)

[题目传送门](https://www.luogu.com.cn/problem/P8841)

这道题主要考察的是 "打擂台" 算法，

也就是求最大或求最小值。

就像这样：

```cpp
if (x > maxn) maxn = x;
```
也可以写成这样：

```cpp
maxn = max(maxn, x);
```

最小值同理。

然而光求出最大最小值还不行，还必须求出每个人更新后的得分，计算方法题目已经给我们了，如下：
$$100 \times \frac{a_i-a_{min}}{a_{max}-a_{min}}$$

最后注意一个点，题目原文说：

>由于成绩系统的问题，最终录入的成绩只能是整数，**采用直接去掉小数部分的方法**。

所以最后直接强转成 ```int``` 即可。


### 最后贴上代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[1005];
int main() {
	int n;
	double a_max = -1e9, a_min = 1e9; // 分别代表最大值与最小值，也要注意是 double 类型，否则计算结果是整数。
	cin >> n;
	for (int i = 1; i <= n; i++) {
		cin >> a[i];
		if (a[i] > a_max) a_max = a[i]; // 求最大值。
		if (a[i] < a_min) a_min = a[i]; // 求最小值。
	}
	for (int i = 1; i <= n; i++) cout << (int)(100 * ((a[i] - a_min) / (a_max - a_min))) << " "; // 直接套公式，注意强转 int 。
	return 0; // 最后收尾。
}
```

总结：

易错点只有最后要强转成 ```int``` 一个，其他的并不需要特别注意。

## 作者：Vibration886 (赞：0)

### [P8841 [传智杯 #4 初赛] 竞争得分](https://www.luogu.com.cn/problem/P8841)

## 题意

有 $n$ 个人提交了作业，$a_{min}$ 为原始得分最低的人的分数，$a_{max}$ 为原始得分最高的人的分数，设第 $i$ 个人的原始得分为 $a_i$，那么第 $i$ 个人的得分就是：

$$\lfloor 100 \times \frac{a_i-a_{min}}{a_{max}-a_{min}} \rfloor$$

求经过更新后每个人的得分。

## 思路

按照题意计算即可。

注意，通过乘以 $1.0$ 或除以 $1.0$ 的方式可以将一个整数强制转换成小数。

## AC代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int a[1001],n;
int maxn,minn=1001;
int ans[1001];
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>a[i];
		maxn=max(maxn,a[i]);
		minn=min(minn,a[i]);
	}
	int dis=maxn-minn;
	for(int i=1;i<=n;i++)
	{
		int t=a[i]-minn;
		ans[i]=floor(100.0*((t*1.0)/(dis*1.0)));
     	//floor()为向下取整函数
	}
	for(int i=1;i<=n;i++)
	{
		cout<<ans[i]<<" ";
	}
	return 0;
}
```

## 一些话
本题适合入门的新生做，考察数组的运用，找一组数据中的最大值与最小值。  
本题建议评红。  

## 做法
先读入数据，可以读入数据同时找 $a_{max}$ 和 $a_{min}$ 来优化代码长度。  
然后根据题目描述中的公式计算即可。

## 细节问题
题目要求“采用直接去掉小数部分的方法”，在 C++ 语言中，`int` 类型和 `int` 类型的 `/` 除法运算是**向零取整**，在本题中数据都为正数，可以起到直接去掉小数部分的作用。  

## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;

int n,a[1005];
int amax,amin=1001;

int main()
{
    cin >> n;
    for (int i=1;i<=n;i++){
        cin >> a[i];
        amax=max(amax,a[i]),amin=min(amin,a[i]);
    }
    for (int i=1;i<=n;i++){
        cout << 100*(a[i]-amin)/(amax-amin) << ' ';
    }
	return 0;
}

```

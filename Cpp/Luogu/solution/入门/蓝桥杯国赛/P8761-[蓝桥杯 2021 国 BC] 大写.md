# [蓝桥杯 2021 国 BC] 大写

## 题目描述

给定一个只包含大写字母和小写字母的字符串，请将其中所有的小写字母转换成大写字母后将字符串输出。

## 说明/提示

对于所有评测用例, 字符串的长度不超过 $100$。 

蓝桥杯 2021 国赛 B 组 E 题（C 组 E 题）。

## 样例 #1

### 输入

```
LanQiao```

### 输出

```
LANQIAO```

### 题意

给定一个字符串，将所有字母转化为大写输出。

### 解法

#### 方法 $1$

直接使用 `ASCII` 码值判断转换。

```
#include<bits/stdc++.h>
using namespace std;

char a[102];//也可以使用 string

int main(){
	cin>>a;
	for(int i=0; i<strlen(a); i++) if(a[i]<=122 && a[i]>=97) a[i]-=32;//判断并转换
	cout<<a;
	return 0;
}
```

#### 方法 $2$

在 `c++` 中，可以使用用单引号求出字符对应的 `ASCII` 码值。

```
#include<bits/stdc++.h>
using namespace std;

char a[102];//也可以使用 string

int main(){
	cin>>a;
	for(int i=0; i<strlen(a); i++) if(a[i]<='z' && a[i]>='a') a[i]+='A'-'a';//判断并转换
	cout<<a;
	return 0;
}
```

#### 方法 $3$

可以使用 `islower` 函数判断小写字符。

```
#include<bits/stdc++.h>
using namespace std;

char a[102];//也可以使用 string

int main(){
	cin>>a;
	for(int i=0; i<strlen(a); i++) if(islower(a[i])) a[i]+='A'-'a';//判断并转换
	cout<<a;
	return 0;
}
```

#### 方法 $4$

可以使用 `isupper` 函数判断大写字符。

```
#include<bits/stdc++.h>
using namespace std;

char a[102];//也可以使用 string

int main(){
	cin>>a;
	for(int i=0; i<strlen(a); i++) if(!isupper(a[i])) a[i]+='A'-'a';//判断并转换
	cout<<a;
	return 0;
}
```

#### 方法 $5$

直接使用 `toupper` 函数将字母转大写。

```
#include<bits/stdc++.h>
using namespace std;

char a[102];//也可以使用 string

int main(){
	cin>>a;
	for(int i=0; i<strlen(a); i++) a[i]=toupper(a[i]);//转换
	cout<<a;
	return 0;
}
```

## 题意简述
输入一个字符串，把字符串中的小写字母全部转换为大写字母，输出转换后的字符串。
## 题目分析
### 方法一：
输入字符串，用循环枚举字符串的每一个字符。

通过当前字符的 ASCII 码判断这个字符是否为小写字符。如果为小写字符，让这个字符的 ASCII 码减 `a` 的 ASCII 码，再加上 `A` 的 ASCII 码，就把它转换为了大写字母。

输出字符串。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	string a;//定义字符串a
	cin>>a;
	for(int i=0;i<a.size();i++){//循环枚举a的每一个字符
		if(a[i]>='a'&&a[i]<='z'){//如果当前的字符为小写字母
			a[i]=a[i]-'a'+'A';//更改为大写字母
		}
	}
	cout<<a;
}
```
### 方法二：
输入字符串后循环枚举字符串的每一个字符，使用 `toupper` 函数把当前字符更改为大写字母。最后输出字符串。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	string a;//定义字符串a
	cin>>a;
	for(int i=0;i<a.size();i++){//循环枚举a的每一个字符
        a[i]=toupper(a[i]);//转换为大写字符
	}
	cout<<a;
}
```
### 方法三：
输入字符串，使用 `transform` 函数和 `towupper` 函数转换为大写，最后输出字符串。

`transform` 函数的用法是 `transform(a,b,c,d)`，`a` 表示区间的开始，`b` 表示区间的结尾，`c` 表示储存结果的容器，`d` 表示要进行操作的函数。因为我们要把字符串转为大写，所以要进行操作的函数为 `towupper`。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	string a;//定义字符串a
	cin>>a;
	transform(a.begin(),a.end(),a.begin(),towupper);//转换为大写
	cout<<a;
}
```

#include<bits/stdc++.h>

## P8761 [蓝桥杯 2021 国 BC] 大写 题解

题意很容易理解，将字符串中的小写字母转化为大写即可。

由于前一篇题解已经介绍了 ```toupper``` 函数，那么这里就来介绍剩下三个关于大小写的函数。



------------
- ```isupper```：判断字符是否为大写字符。
- ```islower```：判断字符是否为小写字符。
- ```tolower```：返回该字符对应的小写字符，若没有对应字符，则不改变原字符。


------------
附上 [AC](https://www.luogu.com.cn/record/108518374) 代码：
```cpp
#include<iostream>
#include<cstring>
#include<cstdio>
using namespace std;
int main(){
	string s;
	//定义字符串变量。
	cin>>s;
	//输入。
	for(int i=0;i<s.length();i++){
		if(islower(s[i])){  //判断是否为小写字符，如果是，则更改。
			s[i]-=32; //一个小写字符的ASCII码值与它对应的大写字符相差32。
		}
	}
	cout<<s<<endl;
	return 0;
}
```

# 分析
这道题让我们将小写字母改成大写，那么可以直接运用 ASCII 代码进行编写，代码如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
char arr[105];
int main(){
	scanf("%s",arr);
	for(int i = 0;i<strlen(arr);i++){
		if(arr[i] >= 97 && arr[i] <= 122){
			arr[i] -= 32;
		}
	}
	printf("%s",arr);
	return 0;
}
```
我们还可以用 ```toupper``` 函数将当前的字母转换成大写字母，这样就可以省去循环内的特判，改过的代码如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
char arr[105];
int main(){
	scanf("%s",arr);
	for(int i = 0;i<strlen(arr);i++){
		arr[i] = toupper(arr[i]);
	}
	printf("%s",arr);
	return 0;
}
```

## 作者：fish_love_cat (赞：0)

**题意：**

给你一个字符串，请你把其中的小写字母转成大写字母并输出。

**思路：**

众所周知，小写字母的 ```ASCII``` 码减去 32 会变成对应大写字母的 ```ASCII``` 码。我们对每一位字符进行判断，只要是小写字母，就将它的 ```ASCII``` 码减成大写后在输出，不然原样输出即可。

**代码：**

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    char ch;
    while(ch=getchar()){
        if(ch=='\n') break;
        if(ch>='a'&&ch<='z'){
            ch-=32;
        }
        cout<<ch;
    }
    cout<<endl;
    return 0;
}
```

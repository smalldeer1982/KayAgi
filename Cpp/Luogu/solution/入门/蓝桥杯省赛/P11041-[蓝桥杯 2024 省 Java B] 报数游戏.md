# [蓝桥杯 2024 省 Java B] 报数游戏

## 题目描述

小蓝和朋友们在玩一个报数游戏。由于今年是 $2024$ 年，他们决定要从小到大轮流报出是 $20$ 或 $24$ 倍数的正整数。前 $10$ 个被报出的数是：$20,24,40,48,60,72,80,96,100,120$。请问第 $202420242024$ 个被报出的数是多少？

这是一道结果填空的题，你只需要算出结果后提交即可。本题的结果为一个整数，在提交答案时只输出这个整数，填写多余的内容将无法得分。

# 题解

## 作者：_Punainen_ (赞：75)

一道简单的找规律题。

观察前 $10$ 个数，很难不注意到，每个偶数项报的数都为 $i \times 12$，所以便简简单单地直接进行计算 $202420242024 \times 12$，得到答案 $2429042904288$。

完美结束。

## 作者：zhuxiangrui_2010 (赞：8)

一眼找规律。

通过自己的推理我们可以写出这个数列的前 $20$ 个数：

$20,24,40,48,60,72,80,96,100,120,140,144,160,168,180,192,200,216,220,240$。

可以发现，后十个数就是前十个数加上 $120$。同理可得，第三组十个数是第二组十个数加上 $120$，第四组十个数是第三组十个数加上 $120$…

所以，第 $202420242020$ 个数就是 $\frac{202420242020}{10} \times 120$，即 $2429042904240$。

第 $202420242024$ 个数在该数的基础上加 $48$（第 $4$ 个数）即可。

## 作者：lcfollower (赞：7)

首先 $\operatorname{lcm}\left(20,24\right) = 120$，为序列第 $10$ 个数，而前 $10$ 个数都是 $20$ 的倍数和 $24$ 的倍数交替。

因此考虑把 $202420242024$ 拆分成 $20242024202\times 10 + 4$，这样答案就是 $20242024202\times 120 + a_4$，$a = \{20,24,40,48,60,72,80,96,100,120\}$（下标从 $1$ 开始）。

然后这就是直接输出板子。


```java
public class Main {
    public static void main(String []args) {
       System.out.println("2429042904288");
    }
}
```

## 作者：_xdd_ (赞：4)

You can't view the English version of this solution.

三年级数学题。

定义 $10$ 个数为一组，容易发现每组加 $120$。接下来就是小学数学了，所以我是用小学数学的格式写的：

$202420242024 \div 10=20242024202$ 余 $4$，这一步是求出有几组、最后一个数在一组中的第几位，一组中的第 $4$ 位是 $48$。

$20242024202 \times 120=2429042904240$，把这几组加起来。

$2429042904240+48=2429042904288$，加上最后一个数 $48$。

故答案为 $2429042904288$。

### 思路分析
我们先十个十个为一组，把前 $30$ 个数枚举出来找规律，具体如下。

前 $1$ 到 $10$ 个分别为：$20$，$24$，$40$，$48$，$60$，$72$，$80$，$96$，$100$，$120$。

前 $11$ 到 $20$ 个分别为：$140$，$144$，$160$，$168$，$180$，$192$，$200$，$216$，$220$，$240$。

前 $21$ 到 $30$ 个分别为：$260$，$264$，$280$，$288$，$300$，$312$，$320$，$336$，$340$，$360$。

显而易见，第 $i$ 组，也就是前 $i \times 10 + 1$ 到 $(i+1) \times 10$ 的数分别为：

$120 \times i+20$，$120 \times i+24$，$120 \times i+40$，$120 \times i+48$，$120 \times i+60$，$120 \times i+72$，$120 \times i+80$，$120 \times i+96$，$120 \times i+100$，$120 \times i+120$。

所以，第 $20242024202$ 组的第 $4$ 个数便为最后的答案，代入公式得 $120 \times 20242024202+48$，答案即为以上这个数，即 $2429042904288$。

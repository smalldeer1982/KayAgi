# 题目大意
在 $1$ 至 $202504$ 中有几个数数位之和是 $5$ 的倍数。
# 个人思路
这道题目数据不大，直接模拟即可。循环枚举 $1$ 至 $202504$ 的每一个数，将其一个个拆分数位，再把每个数位依次累加，最后判断累加的总和是否满足 $5$ 的倍数，若满足，则累加答案。

```cpp
#include<bits/stdc++.h>
using namespace std;
int ans;
int main(){
	ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
	for(int i=1;i<=202504;i++){//循环枚举
		int sum=0,tmp=i;
		while(tmp){//拆分数位
			sum+=tmp%10;
			tmp/=10;
		}
		if(sum%5==0)ans++;//统计答案
	}
	cout<<ans;
	return 0;
}

```
最后得到答案为 `40500`。

这是一道非常简单的模拟题，直接模拟即可。

题目让你找从 $1$ 到 $202504$ 中间的一共 $202504$ 个整数中，有多少个数的各个数位之和是 $5$ 的倍数。

既然这是一个填空题，数据已经完全确定了，所以我们可以直接写个代码来寻找。我们从 $1$ 一直循环到 $202504$，然后计算每个数的各数位之和，然后分别判断这个和是不是 $5$ 的倍数，如果是的话，就对答案贡献 $1$，否则就继续往后查找。

这样看来，代码并不难写，我写出了这样的代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int ans = 0;
    for(int i = 1; i <= 202504; i ++)
    {
        int x = i, sum = 0;
        while(x)
        {
            sum += x % 10; //取出现在这个数的最后一位
            x /= 10; //把最后一位去掉
        }
        if(sum % 5 == 0) ans ++;
    }
    cout << ans << endl;
    return 0;
}
```

这样，输出结果是 $40500$，提交一下，确实是这个数。所以，这道题我们就做完了。

我这个代码的时间复杂度是 $O(n)$。如果这不是一个填空题而是一个编程题，那我觉得，如果它让你找从 $1$ 到 $n$ 之间所有的整数中，有多少个数的各个数位之和是 $5$ 的倍数，如果 $n$ 的范围是 $10^7$ 以下，也能完美通过。

## 思路
本题主要运用枚举思想。

我们可以从 $1$ 枚举到 $202504$，如果其中有一个数字的各位数之和是 $5$ 的倍数，累加计数器。

那如何求各位数之和呢？我们可以定义一个 `check` 函数，用于返回一个数的各位数之和，判断返回值是否是 $5$ 的倍数即可。

其中，`check` 函数的实现方式为：设需求各位数之和的数为 $n$，让累加值不断加上 $n$ 的个位数，同时将 $n$ 除以 $10$，直到 $n$ 的值为 $0$。
## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int check(int n){
    int sum=0;
    while(n>0){
        sum+=n%10;
        n/=10;
    }
    return sum;
}

int main(){
    int cnt=0;
    for(int i=1;i<=202504;i++){
        if(check(i)%5==0) cnt++;
    }
    printf("%d",cnt);
    return 0;
}
```

# P12162 题解
### Step 0 前言 Preface
~~[博客食用更佳](https://www.luogu.com.cn/article/okdh0x9t)~~ 打算把蓝桥杯的题（P12162~P12168）全写一篇题解，这是 T1 的题解。
### Step 1 思路 Idea
很明显的一道大模拟，就是问 $x\in[1, 202504]$ 中有几个 $x$ 的数位之和能整除 $5$。要解决这一道题，我们需要用到一个知识：数位拆分。

问：如果想要求出 $x$ 的个位是多少该怎么求？  
解：我们可以把 $x$ 和 $10$ 做取余运算，求出它除以 $10$ 的余数，就是把除 $x$ 的个位的所有数字全部删掉，就能求出 $x$ 的个位。

问：如果想要把 $x$ 的个位去除该怎么做？  
解：我们可以把 $x$ 整除 $10$，这样就能把 $x$ 的个位去除了。

所以求数位之和的思路如下：
1. 判断 $x$ 是否为 $0$，如果是，跳转到第 $4$ 步，否则往下执行。
2. 将数位之和等于数位之和加上 $x\bmod10$。
3. 让 $x$ 等于 $x\div10$，返回第 $1$ 步。
4. 返回数位之和。
### Step 2 代码 Code
思路就在上面，完整代码如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
int get(int x) {  // 得到数位之和
	int ans=0;
	while(x) {  // 数位拆分
		ans+=x%10;
		x/=10;
	}
	return ans;
}
int main() {
	int ans=0;
	for(int i = 1;i<=202504;i++) {  // 枚举
		if(!(get(i)%5)) {  // 判断是否是 5 的倍数
			ans++;
		}
	}
	printf("%d", ans);
	return 0;
}
```

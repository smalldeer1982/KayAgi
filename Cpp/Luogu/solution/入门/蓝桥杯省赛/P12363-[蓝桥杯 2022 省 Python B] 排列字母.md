# [蓝桥杯 2022 省 Python B] 排列字母

## 题目描述

小蓝要把一个字符串中的字母按其在字母表中的顺序排列。

例如，$\tt{LANQIAO}$ 排列后为 $\tt{AAILNOQ}$。

又如，$\tt{GOODGOODSTUDYDAYDAYUP}$ 排列后为 $\tt{AADDDDDGGOOOOPSTUUYYY}$。

请问对于以下字符串，排列之后字符串是什么？

$$\tt{WHERETHEREISAWILLTHEREISAWAY}$$

# [P12363 [蓝桥杯 2023 省 Python B] 排列字母 ](https://www.luogu.com.cn/problem/P12363)

### 思路

根据题目，`LANQIAO` 排列后为 `AAILNOQ`。

又如，`GOODGOODSTUDYDAYDAYUP` 排列后为 `AADDDDDGGOOOOPSTUUYYY`。

我们可以发现是将给定的字符串进行从小到大的排序。

### Code

```cpp
#include<bits/stdc++.h>
using namespace std;
string s="WHERETHEREISAWILLTHEREISAWAY";
signed main() 
{
	ios::sync_with_stdio(false);
	cin.tie(0);cout.tie(0);
	
	sort(s.begin(),s.end());//sort 函数默认按升序排序
	
	cout<<s<<endl;
	
	return 0;
}
```

# [蓝桥杯 2022 省 Python B] 纸张尺寸

## 题目描述

在 ISO 国际标准中定义了 A0 纸张的大小为 $1189 \ \text{mm} \times 841\ \text{mm}$, 将 A0 纸沿长边对折后为 A1 纸, 大小为 $841\ \text{mm} \times 594 \ \text{mm}$, 在对折的过程中长度直接取下整 (实际裁剪时可能有损耗)。将 A1 纸沿长边对折后为 A2 纸, 依此类推。

输入纸张的名称, 请输出纸张的大小。

## 样例 #1

### 输入

```
A0```

### 输出

```
1189
841```

## 样例 #2

### 输入

```
A1```

### 输出

```
841
594```

# 题解

## 作者：_Null_Ptr (赞：4)

由题意得，我们可以依据纸张编号进行相应次数的对折操作。对折即为就将长边除以 $2$ 并取整，注意始终保持 $l$ 为长边。

下面贴上代码。
```python
l = 1189
w = 841
inp = input()
n = int(inp[1])
for i in range(n):
    if l > w:
        l = l//2
    else:
        w = w//2
    if l < w:
        l, w = w, l
print(l)
print(w)
    
```

## 做法：
我们读一遍题可以发现，$A$ 字母后面的数字越大，那么纸张的大小越小。

而且，原本的宽会变成长，原本的长的长度 $\div 2$ 就是宽。

所以代码就有了，接下来看代码。

## Code
```
#include <bits/stdc++.h>
using namespace std;
int main()
{
    string s;
    cin >> s;
    int n = s[1] - '0',x = 1189,y = 841;
    for(int i = 1; i <= n; i++)
    {
        int xx = x;//原本的x
        x = y;//把宽变成长
        y = xx / 2;//把长除以2变成宽
    }
    cout << x << "\n" << y;
    return 0;
}
```

## P12365 [蓝桥杯 2023 省 Python B] 纸张尺寸

### Problem

A0 的纸大小为 $1189 \ \text{mm} \times 841\ \text{mm}$，后面依次为对折长边，求 A0 到 A9 对应的纸张大小。

### Solution

可以直接拿计算机算 $9$ 次就做完了，但是我懒得打开计算机，这题没啥可说的直接依据题意模拟，用 map 存字符串和对应二元组即可，然后可以字符串拼接实现，具体看代码。

### Code


```cpp
#include <bits/stdc++.h>
#define int long long
int a, b;
std::string s;
std::map<std::string, std::pair<int, int> > mp;
int32_t main(){
	std::cin.tie(nullptr) -> std::ios::sync_with_stdio(false);
	int a = 1189, b = 841;
	mp["A0"] = {a, b};
	for(int i = 1; i <= 9; i++){
		if(a < b){
			std::swap(a, b);
		}
		a >>= 1;
		std::string t = "A";
		t += (char)(i + '0');
		mp[t] = {b, a};
		
	}
	std::cin >> s;
	std::cout << mp[s].first << '\n';
	std::cout << mp[s].second << '\n';
	return 0;
}
```

# 题解：P12365 [蓝桥杯 2023 省 Python B] 纸张尺寸

## 思路：

首先，这道题目一看就十分的水，已知 A0 的长边为 1189mm，短边为 841mm。让我们求从 A0 到 A9 中，所有对应的纸张的长边和短边的长度。

那么，我们观察样例就能发现，沿着长边对折以后，长边的长度就是短边的长度，而短边的长度就是长边的一半，那么我们就能将这道题目给解出来了。

## AC Code:

```cpp
#include <bits/stdc++.h>
using namespace std;
int cb=1189,db=841;
int main(){
    string s;
    cin>>s;
    int i=s[1]-'0';
    if(i==0){cout<<cb<<"\n"<<db;return 0;}
    while(i>=1){
        i--;
        int zc;
        zc=cb;
        cb=db;
        db=zc/2;
    }
    cout<<cb<<"\n"<<db;
}
```

### 思路

A0 纸张的大小为 $1189 \ \text{mm} \times 841\ \text{mm}$，在对折的过程中长度直接取下整 (实际裁剪时可能有损耗）为 A1 的长度，依此类推求出所有纸张的边长。
 
题意很好理解，由于只有十个数，废话不多说，直接暴力求解。

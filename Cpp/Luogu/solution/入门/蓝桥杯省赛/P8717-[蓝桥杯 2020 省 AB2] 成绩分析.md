# [蓝桥杯 2020 省 AB2] 成绩分析

## 题目描述

小蓝给学生们组织了一场考试，卷面总分为 $100$ 分，每个学生的得分都是一个 $0$ 到 $100$ 的整数。

请计算这次考试的最高分、最低分和平均分。



## 说明/提示

对于 $50 \%$ 的评测用例, $1 \leq n \leq 100$。

对于所有评测用例, $1 \leq n \leq 10000$。

蓝桥杯 2020 第二轮省赛 A 组 F 题（B 组 F 题）。

### 输入

```
7
80

```

### 输出

## 作者：YuTianQwQ (赞：14)

其题为题一水，可为三步骤：

1. 求和，以方便求出平均值；
2. 求最大值打擂台；
3. 求最小值同上。

#### 这里先放出求和的伪代码：

```cpp
int sum = 0;
for(int i = 0;i < 10;i++)
{
   int a;
   scanf("%d",&a);
   sum+=a;
}
```

#### 求最大值的：

```cpp
int max = 0;
for(int i = 0;i < 0;i++)
{
   int a;
   scanf("%d",&a);
   if(max < a) max = a;
}
```

#### 求最小值的：

```cpp
int min = INT_MAX;
for(int i = 0;i < 0;i++)
{
   int a;
   scanf("%d",&a);
   if(min > a) min = a;
}
```

#### 我们再综合一下就是这样：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
   int n,sum = 0,maxn = 0,minn = INT_MAX;
   scanf("%d",&n);
   for(int i = 0;i < n;i++)
   {
      int a;
      scanf("%d",&a);
      sum+=a;
      if(a > maxn) maxn = a;
      if(a < minn) minn = a;
   }
   printf("%d\n%d\n%d",maxn,minn,sum);
   return 0;
}
```

但这只是输出了总和、最大值、最小值，求平均数只需**除以一下 $n$** 就行了。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
   int n,sum = 0,maxn = 0,minn = INT_MAX;
   scanf("%d",&n);
   for(int i = 0;i < n;i++)
   {
      int a;
      scanf("%d",&a);
      sum+=a;
      if(a > maxn) maxn = a;
      if(a < minn) minn = a;
   }
   printf("%d\n%d\n%.2f",maxn,minn,float(sum)/n);
   return 0;
}
```

**注意！$sum$ 需要用 $\texttt{float}$ 强制类型转换，否则可能出现奇怪的错误**。

题意不再赘述。我们依次解决三个问题。

### Problem 1 求最大值

因为要与每个分数作比较求出最大值，所以初值一定是要定义极小值。

题目上说每个分数都大于等于 $0$，所以我们定义最大值 $maxx$ 的初值为 $-1$ 即可。

```cpp
int maxx=-1;
```

接下来对于每次输入的成绩，我们要进行比大小，用到 $\max$ 函数，$\max(a,b)$ 表示 $a,b$ 中较大的值，所以代码为：

```cpp
maxx=max(maxx,a);
```

### Problem 2 求最小值

因为要与每个分数作比较求出最小值，所以初值一定是要定义极大值。

题目上说每个分数都小于等于 $100$，所以我们定义最小值 $minn$ 的初值为 $999$ 即可。

```cpp
int minn=999;
```

接下来对于每次输入的成绩，我们要进行比大小，用到 $\min$ 函数，$\min(a,b)$ 表示 $a,b$ 中较小的值，所以代码为：

```cpp
minn=min(minn,a);
```

### Problem 3 求平均值

求平均值我们需要知道总值和个数。

总值 $sum$ 初值定义为 $0$，之后加上每次输入的分数 $a$。

输入完后要再除以个数 $n$，记得要先强制转换为 $double$ 类型的，再保留两位小数。

输出代码：
```cpp
printf("%.2lf",sum*1.0/n);
```

```cpp

    for(int i=1;i<=n;i++)

    	int a;
    	cin>>a;
    	maxx=max(maxx,a);
    	minn=min(minn,a);
    	sum+=a;
	}
	printf("%d\n%d\n%.2lf",maxx,minn,sum*1.0/n);
```

## 分析

首先输入 $n$ 和数列 $a$，然后遍历一遍 $a$，把最大值记为 $mx$，最小值记为 $mn$，和记录为 $sum$。如果 $a_i>mx$，那么就把 $mx$ 赋值为 $a_i$。如果 $a_i<mn$，那么就把 $mn$ 赋值为 $a_i$。每次 $sum$ 都加上 $a_i$。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
const int N = 1e5+5;
int n,a[N],mx = INT_MIN,mn = INT_MAX,sum;
int main()
{
	scanf("%d",&n);//输入n
	for(int i = 1;i<=n;i++)//输入a
		scanf("%d",&a[i]);
	for(int i = 1;i<=n;i++)//遍历数组a
	{
		if(mn>a[i])//记录最小值
			mn = a[i];
		if(mx<a[i])//记录最大值
			mx = a[i];
		sum+=a[i];//加和
	}
	printf("%d\n%d\n%.2lf",mx,mn,(double)sum/n);//printf("%.2lf",...) 保留两位小数 
	return 0;
}

```

### 题意简述

- 给定一个数列 $a$。
- 求这个数列中的**最大值、最小值、平均值**。

### 算法一

若选手不会使用数组存储数据，可以使用**一个变量**来进行数列的输入和求值。

选手需要熟练掌握**打擂台求最大、最小值**这一算法。本题解**使用了 `max` 和 `min` 函数**来替代用于判断大小的 `if` 语句。

请注意将 `int` 类型的变量赋值给 `double` 类型的变量需要先 `* 1.0`。

该算法时间复杂度 $O(n)$，空间复杂度 $O(1)$，可以通过本题。参考代码如下：

```cpp
#include<bits/stdc++.h>
using namespace std;
int tmp,maxi = -1,mini = 101,n,sum;
double a;
signed main()
{
	cin >> n;
	for(int i = 1;i <= n;i++)
	{
		cin >> tmp;
		maxi = max(tmp,maxi);
		mini = min(tmp,mini);
		sum += tmp;
	}
	cout << maxi << "\n" << mini;
	a = sum * 1.0 / n;
	printf("\n%.2lf",a);
	return 0;
}

```

### 算法二

选手可以使用数组来存储 $a$ 数列，读入后做法同上。该算法时间复杂度为 $O(n)$，空间复杂度为 $O(n)$，可以通过本题。

### 算法三

选手可以将 $a$ 数列进行**排序**，以求得最大值和最小值。若选手使用的是快速排序、`sort` 排序等 $O(n\log n)$ 的算法，或冒泡排序等 $O(n^2)$ 的算法均可通过本题。

又注意到本题中 $1\le a_i\le100$，所以选手也可以使用**计数排序**。

### 后记

本题数据范围极小，即使乱搞也能通过。在算法竞赛中，请尽量使用算法一中的做法。

## 作者：runner_362 (赞：2)

一道非常**经典**的题，[传送门](https://www.luogu.com.cn/problem/P8717)。

## Solution

### 题目简意

- 输入 $n$ 个数。
- 输出这些数的最大值、最小值、平均数。
- 注意，最大值与最小值都是整数，而平均数是**实数**，需要保留 $2$ 位小数。

### 核心分析过程

首先，是求最大、最小数。我们首先来看看最大数的定义是“一个数列中最大的数”（最小数亦然），那么想要求出最大数，我们只需将当前数列中的每个数与当前的最大数进行比较：

- 若当前数更大，就说明目前的“最大数”不是数列中最大的，我们就需要将最大数更新为目前遍历的数。
- 若最大数更大，就说明目前的“最大数”没有问题（暂时），不需要进行任何操作。

最小数亦然。

所以，我们可以写出下面代码（`maxn` 表示最大数）：
```cpp
if (maxn < a[i]) {
    maxn = a[i];
}
```

但是，我们还可以简化这行代码，不需要 `if`，怎么做呢？

我们知道，STL库中有 $2$ 个函数：`max()` 和 `min()`。他们的作用分别是：返回 $2$ 个参数的最大数、最小数。那么，我们如何利用这 $2$ 个函求最大最小值？

答案很简单，只要一行代码：
```cpp
maxn = max(maxn, a[i]);
```

看懂了吗？这段话翻译过来就是：“更新最大值 = 求最大(当前最大值, 遍历到的数);”。

好了，现在解决了最大数和最小数，只剩一个平均数了。小学知识：$平均数=总和\div 个数$，所以，我们要求出平均数，必须知道 $2$ 个值：数列总和与数列中数的个数，个数就是 $n$，总和呢？

很简单！总和就是把数列中的每个数加起来，所以，我们只需遍历每个数，并且加到一个变量中，最后再除以 $n$。

最后，我们需要注意一些细节：
- 首先，是这些变量的初值，`sum`（也就是上文提到的“总和”）的初值不用说，肯定是0。但是 `maxn` 与 `minn` 就不一样了，他们的初值是 $2$ 个极端—— $-1$ 与 $2\times 10^9$，因为我们不知道在做第一次比较时的数是多少，只好让第一次变更的成功率为 $100\%$。
- 第二，就是题目中的“第三行包含一个**实数**，四舍五入保留正好**两位小数**”。怎么保证保留 $2$ 位小数呢？我们要了解这段代码：`cout << fixed << setprecision(2) << n << endl;`，这段代码的意思大致就是：将一个 `double` 类型的变量 `n` 四舍五入致 $2$ 为小数，再输出。
- 第三，我们没必要再开一个循环遍历每一个数，只需在输入时操作就可以了。

## Code

**AC代码，仅供参考，切勿抄袭！**

```cpp
# include <iostream>
# include <cmath>
# include <cstring>
# include <algorithm>
# include <bits/stdc++.h>
# define ll long long
using namespace std;

int a[10005], n, minn = 2e9, maxn = -1, sum = 0; // 注意初值

int main() {
    cin >> n;
    for(int i = 1; i <= n; ++i) {
        cin >> a[i];
        minn = min(minn, a[i]); // 最小值
        maxn = max(maxn, a[i]); // 最大值
        sum += a[i]; // 总和
    }
    cout << maxn << endl;
    cout << minn << endl;
    cout << fixed << setprecision(2) << 1.0 * sum / n << endl; // 注意输出格式
    
    return 0;
}
```

### 题目描述

给定一个 $n$ 和 $n$ 个数，输出最大值，最小值，算术平均值。

### 题目解法

平均值的解法：先将所有的数相加，最后除以 $n$。

最大值和最小值写法：

1. 使用 sort 排序，最后输出最大值和最小值。算法复杂度：$O(n \cdot \log n)$，可以通过此题。

1. 使用箱排序，最后输出最大值和最小值。算法复杂度：$O(n + c)$，可以通过此题。

1. 使用打擂台求最值。定义变量 $maxx$ 表示最高分，$minx$ 表示最低分，每输入一个数进行一次更新。算法复杂度：$O(n)$，可以通过此题。

最后一种时间与空间均踩爆前两种，故采用最后一种。

细节：在保留两位小数时可以用 `cout << fixed << setprecision(2) << endl;` 语句来解决，头文件：`iomanip`。

代码：

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main()
{
    int n, sum, maxx = -1, minx = 101;
    cin >> n;
    for (int i = 1; i <= n; i++)
    {
        int x;
        cin >> x;
        maxx = max(maxx, x);
        minx = min(minx, x);
        sum += x;
    }
    cout << maxx << endl;
    cout << minx << endl;
    cout << fixed << setprecision(2) << 1.0 * sum / n << endl;
    return 0;   
}
```

## P8717 [蓝桥杯 2020 省 AB2] 成绩分析 题解

这里一共提供两种做法。

**一、$O(n \log n)$ 做法**

在输入的时候累计输入成绩的加和，再除以 $n$ 就是平均分。除此之外，使用 STL 的 `sort` 排序，记录成绩的数组第一个数就是最低分，最后一个数就是最高分。

代码如下：

```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>
using namespace std;
int n,a[10005],sum;
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a[i];
		sum+=a[i];//sum记录加和
	}
	sort(a+1,a+n+1);//这部分时间复杂度最高，达到n log n
	cout<<a[n]<<endl<<a[1]<<endl;
	printf("%.2lf",sum*1.0/n);//保留两位小数这样写
	return 0;
} 
```
**二、$O(n)$ 做法**

整个程序中，`sort` 排序的时间复杂度最高，如果不用排序，时间复杂度就会降到 $O(n)$。

我们直接在输入的时候开两个变量，一个记录最高分，一个记录最低分，不断更新它们的值，最后不用排序也能得到最高分和最低分。

这样除了时间复杂度降低以外，由于数组只在输入时用，所以可以直接变成一个变量，使空间复杂度降低。

代码如下：

```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>
using namespace std;
int n,m,sum,x=-1,y=101;//记录分数的变量要初始化
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>m;//直接变成变量输入
		sum+=m;
		if(m>x)x=m;//更新最高分
		if(m<y)y=m;//更新最低分
	}
	cout<<x<<endl<<y<<endl;
	printf("%.2lf",sum*1.0/n);
	return 0;
} 
```

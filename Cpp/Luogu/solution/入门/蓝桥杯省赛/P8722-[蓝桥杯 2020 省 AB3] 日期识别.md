# [蓝桥杯 2020 省 AB3] 日期识别

## 题目描述

小蓝要处理非常多的数据, 其中有一些数据是日期。

在小蓝处理的日期中有两种常用的形式：英文形式和数字形式。

英文形式采用每个月的英文的前三个字母作为月份标识，后面跟两位数字表示日期，月份标识第一个字母大写，后两个字母小写, 日期小于 $10$ 时要补前导 $0$。$1$ 月到 $12$ 月英文的前三个字母分别是 `Jan`、`Feb`、`Mar`、`Apr`、`May`、`Jun`、`Jul`、`Aug`、`Sep`、`Oct`、`Nov`、`Dec`。

数字形式直接用两个整数表达，中间用一个空格分隔，两个整数都不写前 导 `0`。其中月份用 $1$ 至 $12$ 分别表示 $1$ 月到 $12$ 月。

输入一个日期的英文形式, 请输出它的数字形式。

## 说明/提示

蓝桥杯 2020 第三轮省赛 AB 组 F 题。

## 样例 #1

### 输入

```
Feb08```

### 输出

```
2 8```

## 样例 #2

### 输入

```
Oct18```

### 输出

```
10 18```

## 作者：LegendaryGrandmaster (赞：36)

由题目可知，月份缩写都是三个字符。

把对应的月数找到，剩余后两位转成数字就是哪一天。

找到对应的月数的方法：用数组把每一个月的英文缩写打表下来，直到前三位和这个字符串匹配，那么就是这个月。

代码如下：

```cpp
#include<bits/stdc++.h>
using namespace std;
string a[12]={"Jan","Feb","Mar","Apr","May","Jun","Jul","Aug","Sep","Oct","Nov","Dec"};
int main()
{
    int m=0,d=0;
    string st,month="";
    cin>>st;
    month=st.substr(0,3);
    for(int i=0;i<12;i++){
		if(month==a[i]){
			m=i+1;
			break;
		}
    }
    d=int(st[3]-48)*10+int(st[4]-48);
    cout<<m<<' '<<d;
}
```

## 题目大意
输入英文字母月份的前 $3$ 位以及有前导 $0$ 数字日期，让你输出月和日。

## 题目解法
一道很简单的题目，使用`if`就加上一点字符串的处理就可以解决的。

主要问题就是处理前导 $0$，处理方法也是十分简单，就是把字符串的第 $4$ 位转换成数字乘 $10$，再加上第 $5$ 位转换成数字的和就可以。就能把前导 $0$ 去掉了。

### 参考代码
```cpp
#include<iostream>
#include<string>
using namespace std;
string s;
int main()
{
    cin>>s;
    int month,day;
    if(s[0]=='J') 
	{
        if (s[1]=='a') month=1;
        else if(s[2]=='n') month=6;
        else month=7;
    } 
	else if(s[0]=='F') month=2;
    else if(s[0]=='M')
	{
        if(s[2]=='r') month=3;
        else month=5;
    } 
	else if(s[0]=='A')
	{
        if(s[1]=='p') month=4;
        else month=8;
    } 
	else if(s[0]=='S') month=9;
    else if(s[0]=='O') month=10;
    else if (s[0]=='N') month=11;
    else month=12;
    day=(s[3]-'0')*10+(s[4]-'0');
    cout<<month<<" "<<day<<endl;
    return 0;
}
```

## 作者：Loser_Syx (赞：0)

## 思路

由于此题每个月份的英文都是 $3$ 位的，而每个日期都是 $2$ 位的，那么总共就是 $5$ 位，我们只需要分成 $2$ 部分来判断，一部分是 `s[0]~s[2]`，另一部分是 `s[3]~s[4]`。其中 `s[0]~s[2]` 是月份，`s[3]~s[4]` 是日期

其中分辨月份的代码是这么写的：

```cpp
    if(s[0] == 'J'){
		if(s[1] == 'a') cout << 1 << " ";//Jan 1月
		else if(s[2] == 'n') cout << 6 << " ";//Jun 6月
		else cout << 7 << " "; // Jul 7月
	}else if(s[0] == 'F') cout << 2 << " ";// Feb 2月
	else if(s[0] == 'M'){
		if(s[2] == 'r') cout << 3 << " ";// Mar 3月
		else cout << 5 << " ";// May 5月
	}else if(s[0] == 'A'){
		if(s[1] == 'p') cout << 4 << " "; //Apr 4月
		else cout << 8 << " ";// Aug 8月
	}else if(s[0] == 'S') cout << 9 << " ";//Sep 9月
	else if(s[0] == 'O') cout << 10 << " ";//Oct 10月
	else if(s[0] == 'N') cout << 11 << " ";//Nov 11月
	else cout << 12 << " ";//Dec 12月
```

而分辨日期我们就可以从高位开始，每次进行一个 `ans = ans * 10 + (s[i] - '0')` 的操作，其实就是快读模板，然后输出就可以了。

## 完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;
//Jan、Feb、Mar、Apr、May、Jun、Jul、Aug、Sep、Oct、Nov、Dec
int main(){
	string s;
	cin >> s;
	if(s[0] == 'J'){
		if(s[1] == 'a') cout << 1 << " ";
		else if(s[2] == 'n') cout << 6 << " ";
		else cout << 7 << " ";
	}else if(s[0] == 'F') cout << 2 << " ";
	else if(s[0] == 'M'){
		if(s[2] == 'r') cout << 3 << " ";
		else cout << 5 << " ";
	}else if(s[0] == 'A'){
		if(s[1] == 'p') cout << 4 << " ";
		else cout << 8 << " ";
	}else if(s[0] == 'S') cout << 9 << " ";
	else if(s[0] == 'O') cout << 10 << " ";
	else if(s[0] == 'N') cout << 11 << " ";
	else cout << 12 << " ";
	int ans = 0;
	for(int i = 3; i <= 4; i++){
		ans = ans * 10 + (s[i] - '0');
	}
	cout << ans;
	return 0;
}
```

#### 思路简述

先输入一个字符串 $ s $，之后判断 $ s $ 的前三位确定月份，然后用 ` stringstream ` 提取出日期输出即可。

```cpp
#include<bits/stdc++.h>
using namespace std;
int mon,day;
int main(){
    string s;
    cin>>s;
    if(s.find("Jan")!=string::npos)mon=1;
    if(s.find("Feb")!=string::npos)mon=2;
    if(s.find("Mar")!=string::npos)mon=3;
    if(s.find("Apr")!=string::npos)mon=4;
    if(s.find("May")!=string::npos)mon=5;
    if(s.find("Jun")!=string::npos)mon=6;
    if(s.find("Jul")!=string::npos)mon=7;
    if(s.find("Aug")!=string::npos)mon=8;
    if(s.find("Sep")!=string::npos)mon=9;
    if(s.find("Oct")!=string::npos)mon=10;
    if(s.find("Nov")!=string::npos)mon=11;
    if(s.find("Dec")!=string::npos)mon=12;//判断月份
    for(int i=0;i<s.size();i++){
        if(isalpha(s[i]))s[i]=' ';//将字母改成空格方便使用stringstream
    }
    stringstream sin;//转换
    sin<<s;
    sin>>day;
    cout<<mon<<' '<<day;//输出
}

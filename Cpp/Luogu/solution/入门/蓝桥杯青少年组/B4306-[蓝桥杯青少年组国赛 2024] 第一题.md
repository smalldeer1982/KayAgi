# [蓝桥杯青少年组国赛 2024] 第一题

## 题目描述

给定一个奇数 $n$ 和一个大写字母 $Center$，生成一个 $n \times n$ 的字符矩阵。字符矩阵由大写字母和 `'.'`（注意：`'.'` 为英文句号）组成，并且所有大写字母组成一个 $n$ 行的菱形，菱形的中心位置为大写字母 $Center$，菱形中的大写字母规律参照下列三组样例。

例 1，$n = 3$, $Center = \tt C$，字符矩阵为:

```
.B.
BCB
.B.
```

例 2，$n = 5$, $Center = \tt Z$，字符矩阵为:

```
..X..
.XYX.
XYZYX
.XYX.
..X..
```

## 样例 #1

### 输入

```
9
E```

### 输出

```
....A....
...ABA...
..ABCBA..
.ABCDCBA.
ABCDEDCBA
.ABCDCBA.
..ABCBA..
...ABA...
....A....```

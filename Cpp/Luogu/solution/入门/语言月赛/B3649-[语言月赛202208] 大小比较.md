# [语言月赛202208] 大小比较

## 题目描述

给出整数 $a,b$，若 $a \le b$，输出 `YE5`，否则输出 `N0`（第二个字符为阿拉伯数字）。


## 说明/提示

对于 $100\%$ 的数据，$0 \le a,b \le 10^5$。

## 样例 #1

### 输入

```
3 5```

### 输出

```
YE5```

## 样例 #2

### 输入

```
5 3```

### 输出

```
N0```

## 文字题解

### 简要题意

若 $a \le b$，输出 `YE5`（审题），若 $a>b$，输出 `N0`（审题）

### 关键代码

考察 if...else 结构。

```cpp
if(a <= b) puts("YE5");
else puts("N0");
```

当然，也可以用两个 if 代替。

```cpp
if(a <= b) puts("YE5");
if(a > b) puts("N0");
```

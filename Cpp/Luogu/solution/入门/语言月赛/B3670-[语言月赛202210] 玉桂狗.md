# [语言月赛202210] 玉桂狗

## 题目背景

某 E 非常喜欢玉桂狗。

## 题目描述

在商店里共有 $n$ 只玉桂狗玩偶，每只玩偶有可爱度 $k$ 和价格 $p$。$k$ 越大的玉桂狗玩偶越可爱。

某 E 现在有 $R$ 元钱，她想知道自己可以买到的最可爱的玉桂狗玩偶的可爱度为多少。

保证某 E 一定能买到至少一只玉桂狗。

## 说明/提示

对于 $30\%$ 的数据，$n=1$；  
对于另外 $30\%$ 的数据，$R \ge \max p$；  
对于 $100\%$ 的数据，$1 \le n \le 10^5, 1 \le k,p,R \le 10^6$。

## 样例 #1

### 输入

```
4 10
100 20
80 10
90 15
10 1
```

### 输出

```
80```

## 文字题解

### 题目大意

有 $n$ 个商品，你有 $R$ 元钱，每个商品有价值 $k$ 和价格 $p$，求能买到的**最大价值**的物品。

### 解析

输入时，利用循环读入 $k,p$。然后判断是否买得起，即 $R$ 是否大于 $p$。若大于，则用 $ans$ 和 $k$ 作比较，当当前的 $k$ 比 $ans$ 还大的时候，$ans$ 自动更新。


**注意事项：**

- 注意读入有 $n$ 次，需调用 ```while(n--)``` 或 ```for(int i=1;i<=n;i++)``` 进行读入。
- 注意 $ans$ 的更新，当且仅当有比它大的 $k$。

# B3670 [语言月赛202210] 玉桂狗 題解

## Source & Knowledge

本题考察对**循环语句**的应用。

## 文字题解

### 题目大意

给定 $n$ 组数据及自身的金钱数 $R$，每组数据都有 2 个数据 $k,p$，求出数据中最大的 $k$ 值并满足该数据的 $p \le R$。

### 解析

当看到求某数据的最大值时，我们很容易想到“打擂台”，使用方法如下：

- 我们定义一个存最大值的变量 $\text{maxx}$；
- 从头遍历所有元素，把每组元素的 $k$ 和 $\text{maxx}$ 比较，如果这个 $k$ 比当前最大值（即 $\text{maxx}$）,那么就更新 $\text{maxx}$ 的值。

如果选手不会使用**数组及结构体**，可以边读入边比较。

**打擂台代码如下**：

```cpp
int maxx=-10000;
for (int i = 1; i <= n; i++){
	std::cin >> k >> p;
    if (p <= R && k > maxx) maxk = k; 
}
```

**注意事项**：

- 需要将 $\text{maxx}$ 定义一个很小的值（比元素中最小的值要小）；

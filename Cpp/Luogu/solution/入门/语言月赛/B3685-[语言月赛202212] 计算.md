# [语言月赛202212] 计算

## 题目描述

给定一个 $3$ 位整数 $n$，计算：
1. $n$ 的各位数字之和；
2. $n$ 的各位数字之和的平方；
3. $n$ 的各位数字之和的立方；

例如，当这个三位数是 $123$ 时：

1. 各位数字之和为：$1+2+3=6$；
2. 各位数字之和的平方为：$(1+2+3)^2=(1+2+3)\times (1+2+3)=36$；
3. 各位数字之和的立方为：$(1+2+3)^3=(1+2+3)\times (1+2+3)\times (1+2+3)=216$。

## 说明/提示

### 样例 1 解释

各位数字之和为：$8+1+7=16$；  
各位数字之和的平方为：$16^2=16\times 16 = 256$；   
各位数字之和的立方为：$16^3 =16\times 16\times 16 = 4096$。

### 数据规模与约定

对于全部的测试点，保证 $100 \leq n \leq 999$。

## 样例 #1

### 输入

```
817```

### 输出

```
16
256
4096
```

## 文字题解

### 题目大意

给定一个 $3$ 位整数 $n$，计算：
1. $n$ 的各位数字之和；
2. $n$ 的各位数字之和的平方；
3. $n$ 的各位数字之和的立方；

### 解析

首先，将这个 $3$ 位整数 $n$ 的个位、十位、百位分离。

- 个位：`n % 10`
- 十位：`n / 10 % 10` 或 `n / 100 % 10`
- 百位：`n / 100`

然后将个位、十位、百位的数字求和，记录到一个变量 `s` 中。最后再按照题意输出即可。

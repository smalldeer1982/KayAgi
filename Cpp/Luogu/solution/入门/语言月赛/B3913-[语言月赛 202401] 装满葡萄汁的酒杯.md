# [语言月赛 202401] 装满葡萄汁的酒杯

## 题目描述

小 H 有五个酒杯可供选择，它们的容量分别为 $100\text{mL}$、$150\text{mL}$、$300\text{mL}$、$400\text{mL}$ 和 $1000\text{mL}$。

现在，小 H 想要在这五个酒杯中选择一个来喝葡萄汁。小 H 打算喝 $a\text{ mL}(a \leq 1000)$ 的葡萄汁。请问，小 H 至少需要选择多大容量的酒杯，才能装下 $a\text{ mL}$ 的葡萄汁（葡萄汁不能溢出）。

## 说明/提示

### 样例解释 #1

如果要装 $50\text{ mL}$ 的葡萄汁，那么用 $100\text{ mL}$ 的酒杯就可以了。 

### 数据规模与约定

对于 $100\%$ 的数据，满足 $1 \leq a \leq 1000$。



## 样例 #1

### 输入

```
50```

### 输出

```
100```

## 样例 #2

### 输入

```
112```

### 输出

```
150```

## 样例 #3

### 输入

```
300```

### 输出

```
300```

## 题目大意

给定容量为 $100\text{mL}$、$150\text{mL}$、$300\text{mL}$、$400\text{mL}$ 和 $1000\text{mL}$ 的五个酒杯。求问如果想要装下 $a\text{ mL}$ 的葡萄汁，至少需要选择多大容量的酒杯。
	
## 题目分析

本题考察对分支结构的运用。

由于 $a \leq 1000$，因此酒杯一定可以装下。将 $a$ 按次序依次和 $100, 150, 300, 400, 1000$ 比较，当找到大于等于 $a$ 的值时，立刻输出即可。

具体来讲，

- 如果 $a \leq 100$，使用 $100 \text{ mL}$ 酒杯，输出 `100`；
- 如果 $100 < a \leq 150$，输出 `150`；
- 如果 $150 < a \leq 300$，输出 `300`；
- 如果 $300 < a \leq 400$，输出 `400`；
- 如果 $400 < a \leq 1000$，输出 `1000`。

# [语言月赛 202408] 相识于 2016

## 题目背景

> 谨以此作别。
>
> 风，2024 年 7 月 27 日夜

## 题目描述

风和清曾经是很好的朋友。清曾说，“……可能走不到最后，但友谊可是能长久一辈子的”。风也曾这么相信。但是，语言的承诺总是那样无力。

风与清相识于 2016 年 8 月。2016 年 8 月是他们认识的第 1 个月，2016 年 9 月是他们认识的第 2 个月……

时光荏苒，现在已经是 $x$ 年 $y$ 月，请问这是他们认识的第几个月？

## 说明/提示

对于 $20\%$ 的测试数据，$x=2016$；  
对于另外 $20\%$ 的测试数据，$y=8$；  
对于 $100\%$ 的测试数据，$2016 \le x \le 9999$，$1 \le y \le 12$。特别地，若 $x=2016$，则保证 $y\ge 8$。

## 样例 #1

### 输入

```
2016 8
```

### 输出

```
1
```

## 样例 #2

### 输入

```
2024 7
```

### 输出

```
96
```

## 文字题解

$x$ 年 $y$ 月至 $2016$ 年 $8$ 月，共过去了 $x-2016$ 年 $y-8+1$ 个月。

因此，答案即为 $12\times (x-2016) + y-7$。

## 题意

给你一个日期，求和 $2016$ 年 $8$ 月的月份距离。

## 思路

首先，我们要输入。

```cpp
cin>>y>>d;
```

根据题意：$2016$ 年 $8$ 月是第一个月。

每过一年加 $12$ 个月。

这部分代码：

```cpp
ans += (y-2016)*12;
```

再算月份即可，注意算是 $8$ 月到这个月，这个月也要算！

```cpp
ans+=(d-8+1);
```

最后记得输出 $ans$！

```cpp
cout<<ans;
```

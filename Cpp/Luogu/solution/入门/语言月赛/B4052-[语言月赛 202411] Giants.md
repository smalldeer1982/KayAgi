# [语言月赛 202411] Giants

## 题目描述

在《迪迪卫》中，一次魔法的**能量**和五个参数有关：能量类型 $type$；发起者的攻击力 $A$，魔法强度 $B$；承受者的物理抗性 $C$，魔法抗性 $D$。

**能量**的数值计算方法如下：

- 如果 $type=0$，则**能量**的数值为 $A+B$。
- 如果 $type=1$，则**能量**的数值为 $\max(A-C,0)+\max(B-D,0)$。

其中 $\max(x,y)$ 表示 $x,y$ 中的最大值。

现在给你这五个参数，请你计算本次魔法的**能量**的数值。

## 说明/提示

### 样例 #1 解释
能量类型为 $0$，数值为 $A+B=3+4=7$。

### 样例 #2 解释
能量类型为 $1$，数值为 $\max(A-C,0)+\max(B-D,0)=1+2=3$。

### 样例 #3 解释
能量类型为 $1$，数值为 $\max(A-C,0)+\max(B-D,0)=0+2=2$。

### 数据范围

对于 $100\%$ 的数据，$0\le A,B,C,D\le 1000,0\le type\le 1$。

对于 $40\%$ 的数据，$type=0$。

另有 $20\%$ 的数据，$B=D=0$。

## 样例 #1

### 输入

```
0 3 4 2 2
```

### 输出

```
7
```

## 样例 #2

### 输入

```
1 3 4 2 2
```

### 输出

```
3
```

## 样例 #3

### 输入

```
1 2 4 3 2
```

### 输出

```
2
```

# 题解

## 作者：未来姚班zyl (赞：5)

考察变量，输入输出，基础分支结构。

按照题目描述定义变量 $A,B,C,D,type$，用 cin 输入，根据 $type$ 输出答案即可，取最大值可以用 `std::max`。核心代码如下：


```cpp
int type,A,B,C,D;
cin>>type>>A>>B>>C>>D;
if(type==0)cout<<A+B;
else cout<<max(A-C,0)+max(B-D,0);
```

# [语言月赛 202412] 聪明猪猪大赛

## 题目描述

ICPC（Interactive Clever Pig Contest，交互聪明猪猪大赛）和 CCPC（Classic Clever Pig Contest，经典聪明猪猪大赛）是两个国际知名猪猪竞赛。其中，ICPC 自猪历 $a$ 年起举办（包括猪历 $a$ 年），每年举办 $b$ 场比赛；CCPC 自猪历 $c$ 年起举办（包括猪历 $c$ 年），每年举办 $d$ 场比赛。

你想知道猪历 $x$ 年这一年 ICPC 和 CCPC 共举办了多少场比赛。

## 说明/提示

### 数据规模与约定

| 测试点编号 | 约定 |
| :-: | :-: |
| $1$ | $a = b=c=d = 1$ |
| $2$ | $c = a = 1$ |
| $3$ | $a = 1$ |
| $4$ | $c = 1$ |
| $5$ | $x = 1$ |
| $6$ | $x \geq a, c$ |
| $7,8,9,10$ | 无特殊约定 |

对全部的测试数据，保证 $1 \le a,b,c,d,x \leq 10^9$。

## 样例 #1

### 输入

```
1
2
3
4
5
```

### 输出

```
6
```

## 样例 #2

### 输入

```
5
1
2
1
4
```

### 输出

```
1
```

## 文字题解
### 题目大意
本题要求计算在猪历 $x$ 年，ICPC 和 CCPC 两个竞赛共举办了多少场比赛。ICPC 从猪历 $a$ 年开始每年举办 $b$ 场比赛，而 CCPC 从猪历 $c$ 年开始每年举办 $d$ 场比赛。
### 解析
#### 读入
首先需要读取五个整数 $a, b, c, d, x$，分别代表两个竞赛的开始年份和每年的比赛场次以及需要查询的年份。
```cpp
int a, b, c, d, x;
cin >> a >> b >> c >> d >> x;
```
#### 计算
用一个变量 $ans$ 表示当前这一年的比赛数量。

如果 $x \geq a$，则今年举办了 $b$ 场 ICPC，于是给答案加上 $b$。
```cpp
if (x >= a) ans += b;
```

同理，如果 $x \geq c$，则今年举办了 $d$ 场 CCPC，于是给答案加上 $d$。
```cpp
if (x >= c) ans += d;
```
#### 输出
输出答案即可。
```cpp
cout << ans << endl;
```

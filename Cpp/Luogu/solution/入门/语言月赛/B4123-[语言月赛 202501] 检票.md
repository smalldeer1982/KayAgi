# [语言月赛 202501] 检票

## 题目描述

在机场的安检入口，有 $n$ 个人正在排队。从前往后第 $i$ 个人的航班在 $t_i$ 分钟后就要起飞了。

为了让旅客不耽误行程，机场决定让航班起飞时间 $\le 15$ 分钟的旅客排到队伍的最前面（这些旅客之间的相对顺序保持不变）。

请输出调整后的队伍信息，但是为了方便，对于每个人你只需要输出他的航班还有几分钟起飞即可。

## 说明/提示

【样例 1 解释】

有 $6$ 个人排队，他们的航班分别在 $39,{\color{red}12},{\color{red}15},40,{\color{red}8},33$ 分钟后起飞。我们让标红的几个人排到最前面即可。

【样例 2 解释】因为所有人都不需要排到最前面，所以队伍不发生改变。

【样例 3 解释】因为所有人都需要排到最前面，所以队伍仍然不发生改变。

【数据范围】

本题共有 $5$ 个测试点，每个测试点 $20$ 分。

对于所有测试点，$1\le n,t_i\le 10000$。前三个测试点还有一些额外的特殊性质：

- 测试点 $1$ 满足 $n=1$。
- 测试点 $2$ 满足 $n=2$。
- 测试点 $3$ 满足 $t_i\le 15$。

## 样例 #1

### 输入

```
6
39 12 15 40 8 33
```

### 输出

```
12 15 8 39 40 33
```

## 样例 #2

### 输入

```
3
55 66 77
```

### 输出

```
55 66 77
```

## 样例 #3

### 输入

```
3
1 1 1
```

### 输出

```
1 1 1
```

# 题解

## 作者：yummy (赞：5)

本题考查一维数组的使用。

我们将所有人的 $t_i$ 存进一个一维数组后，可以分两次循环输出所有人。

第一次循环让 $i$ 从 $1\sim n$ 枚举，如果 $t_i\le 15$ 那么就输出 $t_i$。这样，所有要优先排队的旅客就一定先输出了。

第二次循环仍然让 $i$ 从 $1\sim n$ 枚举，如果 $t_i>15$ 就输出 $t_i$，换言之输出剩下的所有旅客。

## 作者：Igallta (赞：2)

## （较为）形式化题意

给定一个长度为 $n$ 的序列 $t$，需要你对序列 $t$ 进行分类。$i$ 越小且 $t_i \leqslant 15$ 的放最前面，然后 $i$ 越小的放前面。

## 举例说明

例如：序列 `200 15 300 7`，它应该输出 `15 7 200 300`，因为是 $15$ 这个人先到的，然后才是 $7$。（它讲究先来后到而不是越小越前）

## 思路

首先，输入 $n$。

整理：题目大概就是对这个序列进行了分类，那我们也边输入边分类不就行了。在输入的同时，我们判断如果 $t_i \leqslant 15$，我们就把它放进一个数组 $frt$ 里面，否则放进 $normal$。（这一步里的 $frt$、$normal$ 就是队列，你也可以使用 queue 或者 vector 等 STL 容器来完成）

这部分的代码：

```cpp
cin>>n;
for(int i=1;i<=n;i++){
	cin>>x;
	if(x<=15){
		frt[++cntf]=x;
	}else normal[++cntn]=x;
}
```

在这里说一下：有一些同学可能会死板地认为输入的时候是存放在同一个数组里，输出的时候也应该是直接输出同一个数组里的内容。实则不然，当我们遇到分类的时候我们就需要用多个数组。

当然，这里有些同学可能无法理解，我画个图：

![](https://pic1.imgdb.cn/item/67c80906d0e0a243d40c8ee8.png)

好了，接下来直接分类输出即可。先输出 $frt$ 后输出 $normal$。

这部分的代码：

```cpp
for(int i=1;i<=cntf;i++)cout<<frt[i]<<' ';
for(int i=1;i<=cntn;i++)cout<<normal[i]<<' ';
```

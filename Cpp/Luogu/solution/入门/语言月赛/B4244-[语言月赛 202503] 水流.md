# [语言月赛 202503] 水流

## 题目描述

有一片池塘，池塘中间的一道非常高的闸门将池塘分隔分为左右两侧。左侧水面高为 $a$ 米，右侧水面高为 $b$ 米。

现在将闸门的高度降低到 $c$ 米，请问此时水是从左向右流动，还是从右向左流动，还是不流动？


## 说明/提示

### 样例 1 解释

初始时左右两侧水面依次为 $7, 3$ 米高，左侧水面比右侧水面高。

在闸门降低至 $4$ 米后，尽管右侧水面低于闸门，但是左侧水面高于闸门，闸门不妨碍左侧的水流动。因此水会从左向右流动。

### 样例 2 解释

初始时左右两侧水面依次为 $7, 7$ 米高，两侧水面一样高。在闸门降低至 $6$ 米后，尽管两侧水面都高于闸门，但是由于两侧水面一样高，因此水不会流动。

### 样例 3 解释

初始时左右两侧水面依次为 $7, 6$ 米高，左侧水面比右侧水面高。

在闸门降低至 $7$ 米后，左侧水面没有高过闸门，无法向右侧流动。因此水不流动。

### 数据规模与约定

对于 $100\%$ 的数据，$1 \leq a, b, c \leq 10000$。

## 样例 #1

### 输入

```
7 3 4
```

### 输出

```
LeftToRight
```

## 样例 #2

### 输入

```
7 7 6
```

### 输出

```
None
```

## 样例 #3

### 输入

```
7 6 7
```

### 输出

```
None
```

# [语言月赛 202503] 水流 题解
## Source & Knowledge

本题来源于 2025 年 3 月的语言月赛，主要考察简单分支结构的运用。

## 文字题解

题目假设一个池塘被一个闸门分为两半。左侧一半水面高度为 $a$ 米，右侧水面高度为 $b$ 米。假设闸门降到 $c$ 米高，询问水流的流向。

我们不妨首先考虑在哪些情况下水流的流向会是**从左向右**、**从右向左**、**不流动**。

- 如果水流从左向右流动，那么一定会满足以下条件：

    1. 左侧水面高于右侧水面 $a > b$；
    2. 左侧水面高于闸门 $a > c$，这是水能够流动的必须要求。

    满足上面两条条件后，水流一定会从左向右流动。如果任何一条条件不满足，水则一定不会流动。

- 水流从右向左流动同理，一定会满足以下条件：

    1. 右侧水面高于左侧水面 $b > a$；
    2. 右侧水面高于闸门 $b > c$。
    
除上述两种情况外，其他任何情况下水都不会流动。

因此我们可以编写如下代码：

```cpp
// 变量定义、读入部分省略

if (a > b && a > c) {
    cout << "LeftToRight" << endl;
} else if (b > a && b > c) {
    cout << "RightToLeft" << endl;
} else {
    cout << "None" << endl;
}
```

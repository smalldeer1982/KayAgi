# [语言月赛 202505] 翻书

## 题目描述

双面打印是常见的节约纸张的方式。小 A 双面打印了某份文件，第 $1,2$ 页在第一张纸上，第 $3,4$ 页在第二张纸上……

页数更小的一面称为正面，另一面称为反面。翻开 $k$ 张纸，可以看到第 $k$ 张纸的反面和第 $k+1$ 张纸的正面。

请问，小 A 想要阅读第 $p$ 页的内容，应该翻开多少张纸。

## 说明/提示

对于 $100\%$ 的测试数据：

- $1 \le p \le 10^{18}$

## 样例 #1

### 输入

```
1```

### 输出

```
0```

## 样例 #2

### 输入

```
4```

### 输出

```
2```

## 文字题解

翻开 $k$ 张纸，可以看到第 $k$ 张纸的正面和反面。第 $k$ 张纸的正面是第 $2k-1$ 页，反面是第 $2k$ 页。

因此，想要看到第 $p$ 页，需要翻开 $\lfloor \dfrac{p}{2} \rfloor$ 张纸。

> C/C++ 中的除法运算为向零取整。例如 `5/2=2`，`-5/2=-2`。

直接输入 $p$，输出 `p/2` 的值即可。注意数据范围需要使用 long long 类型。

# [语言月赛 202507] 购票

## 题目描述

H 市 W 公园的门票分为单次票和年票，单次票的售价为 $a$ 元，年票的售价为 $b$ 元。若购买年票，则可以在今年无限次进入公园。

今年你想去 $n$ 次 W 公园，你最少要花多少钱购买门票？

## 说明/提示

### 样例 1 解释

如果购买单次门票，则总花费为 $100\times 1=100$ 元。

如果购买年票，则总花费为 $50$ 元。

因此购买年票可以使总花费最少，最少花费为 $50$ 元。

### 样例 2 解释

如果购买单次门票，则总花费为 $2\times 10=20$ 元。

如果购买年票，则总花费为 $50$ 元。

因此购买单次门票可以使总花费最少，最少花费为 $20$ 元。

### 样例 3 解释

请注意答案可能超过 `int` 类型所能表示的范围。

### 样例 4 解释

输入的数据中可能含有 $0$。

### 数据范围与约定

对于全部数据，满足 $0\le a,b,n\le 10^{18}$。各测试点的详细数据范围见下表。

| 测试点 | 特殊性质 |
| :---: | :---: |
| $1\sim 12$ | $0\le a,b,n\le 10^4$ |
| $13\sim 16$ | $0\le a,b,n\le 10^9$ |
| $17\sim 20$ | 无 |

# 题解

## 作者：Dr_Gilbert (赞：0)

## 入门赛 #37 B 题解

题目大意：输入三个整数 $a,b,c$，输出 $\min(a\times c,b)$。其中 $0\le a,b,c\le 10^{18}$。

要通过本题的全部数据，就需要考虑当 $a,c$ 均为 `long long` 范围的数时如何处理 $a\times c$。可以使用 `__int128` 或者取 $a,b,c$ 的对数，判断 $\log a+\log c$ 和 $\log b$ 的大小关系。这两种做法在此不作展开。

本题的标准做法是将乘积的比较转化为除法的比较。

考虑实际意义，$\left\lfloor\dfrac{b}{a}\right\rfloor$ 表示，用年卡的价格买普通票，可以买几张。如果用年卡的价格能买到的单次门票数量**少于**入场的次数，说明用年卡更划算，否则说明买单次门票更划算或者两种购票方案花费相同。

也可以计算 $\left\lfloor\dfrac{b}{n}\right\rfloor$，即计算买年卡时的单次入场价格。如果买年卡时的单次入场价格**小于**单次门票价格，说明用年卡更划算，否则说明买单次门票更划算或者两种购票方案花费相同。

以上两种方案可以任选一种，这样我们就可以规避大整数乘法完成本题。

需要注意判断 $a,n$ 是否为 $0$。本题包含 $a=0$ 和 $n=0$ 的测试点各一个。

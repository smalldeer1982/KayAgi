# [语言月赛 202508] 机型判断

## 题目背景


扶苏有一页飞行日志，但是这页飞行日志上记录的飞机型号看不清了。但是扶苏知道这架飞机的一些其它信息，所以希望能够复原她乘坐的航班的型号。

## 题目描述


部分常见客机机型的特点如下：

| 型号 | 制造公司 | 宽/窄体 | 发动机数量 |
| :-: | :-: | :-: | :-: |
| A320 | 空客 | 窄体 | $2$ |
| A350 | 空客 | 宽体 | $2$ |
| A380 | 空客 | 宽体 | $4$ |
| B737 | 波音 | 窄体 | $2$ |
| B747 | 波音 | 宽体 | $4$ |
| B787 | 波音 | 宽体 | $2$ |
| C919 | 商飞 | 窄体 | $2$ |

已知该飞行日志记录的飞机型号只可能是上表列出的几个型号之一。可以发现，只要给定了制造公司、宽/窄体、发动机数量三个信息，就能唯一确定飞机的型号。

现在，给定制造公司、宽/窄体、发动机数量三个信息，请你求出该飞机的型号。


## 说明/提示

对全部的测试数据，保证 $x \in \{\texttt{A}, \texttt{B}, \texttt{C}\}$，$y \in \{\texttt{N}, \texttt{W}\}$，$z \in \{2, 4\}$。

## 样例 #1

### 输入

```
A W 4```

### 输出

```
A380```

## 样例 #2

### 输入

```
B N 2```

### 输出

```
B737```

## 样例 #3

### 输入

```
C N 2```

### 输出

```
C919```

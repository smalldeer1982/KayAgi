# [GXPC-S 2024] 数字谜题

## 题目背景

小林最近迷上了数字谜题。

## 题目描述


给定十进制正整数 $x$，你需要输出它在二进制表示下的最长的连续的 $1$ 的个数。

你需要解决 $T$ 组数据。

## 说明/提示

对于第三组数据：注意到 $(10)_{10}=(1010)_2$。最长的连续的 $1$ 的个数为 $1$。

**本题采用捆绑测试。**

- Subtask 1（30pts）：保证 $1\le x\le 100$，$1\le T\le 10$；
- Subtask 2（70pts）：无额外约束。

对于 $100\%$ 的数据，保证：

- $1\le x\le 10^{18}$；
- $1\le T\le 10^5$。

## 样例 #1

### 输入

```
3
3
7
10```

### 输出

```
2
3
1```

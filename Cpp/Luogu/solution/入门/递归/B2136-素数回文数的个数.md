# 素数回文数的个数

## 题目描述

求 $11$ 到 $n$ 之间（包括 $n$），既是素数又是回文数的整数有多少个。

## 说明/提示

回文数指左右对称的数，如：$11$，$12121$。

## 样例 #1

### 输入

```
23```

### 输出

```
1```

# 题解

## 作者：lujunyu (赞：50)

## 思路
由题可知，素数回文数既是一个素数，也是一个回文数。所以可以定义两个函数来判断是否符合素数回文数的两个条件。
## 方法
### 判断素数
使用循环让这个数对 $2$ 至这个数减去一的每一个数进行取模，如果都没有取模尽，说明这个数是一个素数。
### 判断回文数
先定义两个数组。用循环将数的每一位分解出来放入第一个数组，再将第一个数组倒过来装进第二个数组。最后判断第一个数组与第二个数组是否完全相等即可。
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
bool ss(int x)//判断素数
{
	for(int i=2;i<=x-1;i++)
		if(x%i==0)
			return false;
	return true;			
}
bool hw(int x)//判断回文数
{
	int t=0,m=0,a[15],b[15];
	while(x!=0)
	{
		t++;
		a[t]=x%10;
		x/=10;
	}
	for(int i=t;i>=1;i--)
	{
		m++;
		b[m]=a[i];
	}
	for(int i=1;i<=t;i++)
		if(a[i]!=b[i])
			return false;
	return true;		
}
int main()
{
	int n,ans=0;
	cin>>n;
	for(int i=11;i<=n;i++)
		if(hw(i)&&ss(i))//如果两个条件均符合，计数器就加一
			ans++;
	cout<<ans;
	return 0;		
}

第一步：（函数）判断是否是素数；

第二步：（函数）判断是否是回文数；

第三步：主函数，循环从 $11$ 到 $n$，每一个数用两个函数验证，是的话计数器加 $1$，最后输出计数器的值。

代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
bool IsPrime(int n){    // 判断素数 
	if (n < 2) return false; // 小于2都不是素数
	for (int i = 2; i <= sqrt(n); i ++) // 从2枚举到根号n
	if (n % i == 0) return false; // 有因数！不是素数
	return true; // 是素数
}
bool IsRever(int n){    // 判断回文数
	int tmp = n, NEW = 0; // tmp存储n，new为翻转后的n
	while (tmp){ // tmp到零循环结束
		NEW = NEW * 10 + tmp % 10; // 反向相加
		tmp /= 10; // tmp去掉最后一位
	}
	return NEW == n; // 如果new == n，则是回文数，反之不是
}
int main(){
    int n, cnt = 0; // 计数器cnt
	cin >> n; // 输入
	for (int i = 11; i <= n; i ++) // 枚举
	if (IsPrime(i) && IsRever(i)) cnt ++;	// 满足素回文数
	cout << cnt << endl; // 输出总数
	return 0; // done
}
```

# 【深基4.习9】打分

## 题目描述

现在有 $n(n \le 1000)$ 位评委给选手打分，分值从 $0$ 到 $10$。需要去掉一个最高分，去掉一个最低分（如果有多个最高或者最低分，也只需要去掉一个），剩下的评分的平均数就是这位选手的得分。现在输入评委人数和他们的打分，请输出选手的最后得分，精确到 $2$ 位小数。 

## 说明/提示

数据保证，$3 \leq n \leq 1000$，每个评委打出的分值为为 $0$ 到 $10$（含 $0$ 与 $10$）之间的整数。

## 样例 #1

### 输入

```
5
9 5 6 8 9```

### 输出

```
7.67```

# 题解

## 作者：Lithium_Chestnut (赞：117)

本题大概思路：

注意此题答案类型为`double`，读入后快排一下，第 $1$ 位和第 $n$ 位就分别是最小值和最大值，那求和时就从 $2$ 开始一直到 $n-1$ 位。最后输出分数和除以总人数 $-2$（因为去掉了最大值和最小值），注意用`printf`的`.2lf`输出，保留 $2$ 位小数。

$Code:$

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[10001];
double ans;
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++) cin>>a[i];
	sort(a+1,a+n+1);
	for(int i=2;i<=n-1;i++) ans+=a[i];
	printf("%.2lf",ans/(n-2));
	return 0;
}
```

## 作者：C171501 (赞：43)

我们可以定义一个表示最大值 $maxn$ 和最小值 $minn$ 的变量，分别给 $maxn$ 赋值 $0$，给 $minn$ 赋予一个较大的值。

输入 $n$ 组数据，每输入一个数据，就判断是否最大和最小，同时将数据加到 $sum$ 中，最后将 $sum$ 减去最大值和最小值除以 $n-2$ 并输出。

如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
double n,m,maxn=0,minn=999999999,sum;
int main()
{
	cin>>n;
	for(int i=1;i<=n;++i)
	{
		cin>>m;
		maxn=max(maxn,m);
		minn=min(minn,m);
		sum+=m;
	}
	sum-=maxn;
	sum-=minn;
	printf("%.2f",sum/(n-2));
	return 0;
}
```

## 作者：AveMarina (赞：31)

__记录最大最小值和总和，最后减去最大值最小值即可__
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int main() {
	int n;
	cin >> n;
	float ans = 0, maxNum = -1, minNum = 100;
	int p = n;
	while (n--) {
		float x;
		cin >> x;
		ans += x;
        // 记录最大值和最小值，最后再减去这两个值
		maxNum = max(x, maxNum), minNum = min(x, minNum);
	}
	printf("%.2f", (ans - maxNum - minNum) / (p - 2.0));
	return 0;
}
```

## 作者：Lylighte (赞：21)

[题目传送门](https://www.luogu.com.cn/problem/P5726) & [双倍经验](https://www.luogu.com.cn/problem/P5738)

~~想起当年被找最大数/最小数支配的恐惧。~~

真的入门题，找出数组中最大和最小的元素，计算剩余元素之和的平均数，没了。

解释见代码：

```cpp
#include<bits/stdc++.h>
using namespace std;

int score[10010], n;		//score 数组存放评分，n 存元素总个数
int maxn, minn, tot;		//maxn 存最大值，minn 存最小值，tot 存元素总和

int main(){
	cin >> n;
	for(int i=1; i<=n; i++)
		cin >> score[i];	//读入数据
	tot = maxn = minn = score[1];	//初始化，假设最大值（最小值）为第一个元素
	for(int i=2; i<=n; i++){		//循环第 2~n 个元素
		if(score[i]>maxn)			//如果比当前最大值更大，更新
			maxn = score[i];
		if(score[i]<minn)			//如果比当前最小值更小，更新
			minn = score[i];
		tot += score[i];			//计算元素总和
//maxn 和 minn 可以这样理解：在循环第 i 个元素时，maxn（或 minn）存的是前 i-1 个元素的最大值（或最小值），然后把第 i 个元素与之对比，循环完 n 个元素后，就得出整个数组中的最大值（或最小值）
	}
	cout << fixed << setprecision(2) << double(tot-maxn-minn)/(n-2) << endl;
//输出答案，注意数据类型转换和保留位数
	return 0;
}
```

数据范围是 $n\le 1000$，对数组排序也不会超时，所以：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n, score[1010];
int ans;

int main(){
	cin >> n ;
	for(int i=1; i<=n; i++)
		cin >> score[i];
	sort(score+1, score+n+1);		//STL 自带的排序算法
	for(int i=2; i<=n-1; i++)		//此时第一个元素为最小值，第 n 个元素为最大值，直接跳过
		ans += score[i];
	cout << fixed << setprecision(2) << (double)ans/(n-2) << endl;
	return 0;
}
```

## 作者：FR_qwq (赞：16)

这是一道比较简单的模拟题。

我们珂以这样做：在读入时，将最小和最大分别求出，再同时加上那个分数，用到下列代码：
```pascal
read(x);//读入
if x<min then min:=x;//求最小
if x>max then max:=x;//求最大
sum:=sum+x;//加上分数
```
然后再减去最高分和最低分，求出总数减2的平均值，完美输出。用下列代码：
```pascal
writeln((sum-min-max){减去最高分和最低分}/(n-2){n-2是应为有两个数减去了}:0:2{记得保留两位小数哦});
```
现在整个代码就浮现在我们眼前了：
```pascal
var i,n:longint;

  begin
  read(x);

  if x>max then max:=x;

  end;

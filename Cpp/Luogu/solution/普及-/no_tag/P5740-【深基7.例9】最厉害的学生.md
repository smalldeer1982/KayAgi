# 【深基7.例9】最厉害的学生

## 题目描述

现有 $N$ 名同学参加了期末考试，并且获得了每名同学的信息：姓名（不超过 $8$ 个字符的仅有英文小写字母的字符串）、语文、数学、英语成绩（均为不超过 $150$ 的自然数）。总分最高的学生就是最厉害的，请输出最厉害的学生各项信息（姓名、各科成绩）。如果有多个总分相同的学生，输出靠前的那位。

## 说明/提示

数据保证，$1 \leq N \leq 1000$，姓名为长度不超过 $8$ 的字符串，语文、数学、英语成绩均为不超过 $150$ 的自然数。

## 样例 #1

### 输入

```
3
senpai 114 51 4
lxl 114 10 23
fafa 51 42 60```

### 输出

```
senpai 114 51 4```

**[题目传送门](https://www.luogu.com.cn/problem/P5740)**
## 题意
给定 $n$ 个同学的姓名与成绩，找出总分最高，输入顺序最前的人的名字与每门科目的成绩。
## 思路
用结构体存下每个同学的信息，然后用 sort 排序，这里 cmp 函数要自己写。这题的难点在于输出输入时最靠前的人，我们可以用 $id$ 存下每个人是第几个输入的，如果总分相同就比较谁 $id$ 更小的。细节看代码。
## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int n;
struct node
{
	char name[18];
	int id, x, y, z, sum;
} a[1010];
bool cmp(node a, node b) 
{ 
    if (a.x + a.y + a.z == b.x + b.y + b.z)
        return a.id < b.id;
    return a.x + a.y + a.z > b.x + b.y + b.z; 
}
int main()
{
	scanf("%d", &n);
	for (int i = 1; i <= n; i++)
    	scanf("%s%d%d%d", a[i].name, &a[i].x, &a[i].y, &a[i].z), a[i].id = i;
	sort(a + 1, a + n + 1, cmp);
	printf("%s %d %d %d", a[1].name, a[1].x, a[1].y, a[1].z);
	return 0;
}
```

# 题解：P5740 【深基7.例9】最厉害的学生
## 思路：
先建立一个结构体来记录数据和总分，然后把总分排序，最后输出第一个人的信息。这样就完美地解决了有多个总分相同的学生输出靠前的那位的问题。
## 代码：
```cpp
#include<bits/stdc++.h>//万能头
using namespace std;
const int N=1005;
int t;
struct u//建立结构体
{
	string n;
	int a,b,c,m;//n 记录名字；a、b、c 分别记录 语、数、英三门功课；m 记录总分
} s[N];
int cmp(u x,u y)
{
	return x.m>y.m;
}
int main()
{
	cin>>t;
	for(int i=1;i<=t;i++)
	{
		cin>>s[i].n>>s[i].a>>s[i].b>>s[i].c;//输入
		s[i].m=s[i].a+s[i].b+s[i].c;//记录每一个人的总分
	}
	stable_sort(s+1,s+t+1,cmp);//将总分进行排序
	cout<<s[1].n<<' '<<s[1].a<<' '<<s[1].b<<' '<<s[1].c;//输出第一名的信息
	return 0;//好习惯
}
```

# 思路

定义了一个结构体 `node`，里面包含五个成员变量 $name$、$x$、$y$、$z$ 和 $sum$，分别表示姓名、语文分数、数学分数、英语分数和总分，然后通过选择排序排序，最后输出排序后第一项即可。

# 代码实现

给出代码：

```cpp
#include<iostream>
using namespace std;
const int N=1005;
int n;
struct node
{
	string name;
	int x,y,z,sum;
}a[N];
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(0);cout.tie(0);
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>a[i].name>>a[i].x>>a[i].y>>a[i].z;
		a[i].sum=a[i].x+a[i].y+a[i].z;
	}
	for(int i=1;i<=n;i++)
		for(int j=i+1;j<=n;j++)
			if(a[i].sum<a[j].sum)
				swap(a[i],a[j]);
	cout<<a[1].name<<' '<<a[1].x<<' '<<a[1].y<<' '<<a[1].z<<endl;
	return 0;
}
```

# [题目传送门](https://www.luogu.com.cn/problem/P5740)

## 题目大意

由题意可知，题目会输入 $n$ 个学生的名字与成绩，总分最高的学生就是最厉害的，让我们输出最厉害的学生各项信息（姓名、各科成绩）。

## 方法概述

我们可以用一个结构体来存储学生的名字与各科成绩，再通过排序来找出最高的**总成绩**就可以了。

但是有一个非常**重要**的一点一定要注意！题目告诉了我们如果有多个总分相同的学生，输出**靠前**的那位。这时候就需要在结构体中加入一个变量来记录输入的顺序，并且在比较函数中判断总成绩相等的情况，否则就是[90pts](https://www.luogu.com.cn/record/222039865)。

## AC code

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 1010;

int n;
struct stu {
    string name;
    int chinese, maths, english, sum, location;
} x[maxn];
bool cmp(stu a, stu b) {
    if (a.sum == b.sum) {
        return a.location < b.location;
    }
    return a.sum > b.sum; 
}
int main() {
	cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> x[i].name >> x[i].chinese >> x[i].maths >> x[i].english;
        x[i].location = i, x[i].sum = x[i].chinese + x[i].maths + x[i].english;
    }
    sort(x, x + n, cmp);
    cout << x[0].name << " " << x[0].chinese << " " << x[0].maths << " " << x[0].english << endl;
	return 0;
}
```

# 题目简述
### 题意
给定 $N$ 个学生的名字及三科成绩，求总分最高的学生的姓名，如果多个学生总分相同，输出靠前的那一位。
### 思路
依次比较，设 $maxx$ 用于存储最高分学生的信息，那么我们只需求出第 $i$ 个学生的总成绩比较，如果高于 $maxx$ 中的成绩，更新 $maxx$，否则不更新。

但由于这个代码是我半年前写的，所以用的排序，和上述思路一致，都是优先排成绩，后排出现顺序。
## [AC 代码](https://www.luogu.com.cn/record/2219087810)
```cpp
#include <bits/stdc++.h>
using namespace std;

struct node {
	string name; //名字
	int x, y, z, score, id; //三科成绩及总分、序号
};

bool cmp(node k, node t) {
	if (k.score != t.score) //分数不同
		return k.score > t.score; //排分数
	return k.id < t.id; //否则排序号（出现顺序）
}

int main() {
	int n;
	cin >> n;
	node a[n];
	for (int i = 0; i < n; i++) {
		cin >> a[i].name >> a[i].x >> a[i].y >> a[i].z;
		a[i].score = a[i].x + a[i].y + a[i].z, a[i].id = i + 1; //建立每个学生的信息
	}
	sort(a, a + n, cmp); //排序
	cout << a[0].name << " " << a[0].x << ' ' << a[0].y << " " << a[0].z; //输出最高分学生的信息
	return 0;
}
```

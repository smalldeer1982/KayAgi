# [COCI 2008/2009 #1] PTICE

## 题目描述

Adrian 、Bruno 和 Goran 想加入鸟类爱好者俱乐部。然而，他们并不知道所有的申请者都必须通过加入俱乐部的考试。考试由 $n$ 道题组成，每道题有三个可能的答案：`A`、`B` 和 `C`。

不幸的是，他们不能区分各种鸟类，所以他们试图猜测正确的答案。每一个男孩都有一套关于哪一种答案序列最有效的理论：

Adrian 声称最好的顺序是：`A`，`B`，`C`，`A`，`B`，`C`，`A`，`B`，`C`，`A`，`B`，`C`$\ldots$

Bruno 确信这样更好：`B`，`A`，`B`，`C`，`B`，`A`，`B`，`C`，`B`，`A`，`B`，`C`$\ldots$

Goran 嘲笑他们，并使用这个序列：`C`，`C`，`A`，`A`，`B`，`B`，`C`，`C`，`A`，`A`，`B`，`B`$\ldots$

写一个程序，根据给出的正确答案，确定三个人中谁对的题目最多。

## 说明/提示

#### 数据规模与约定
对于 $100\%$ 的数据，有 $1≤n≤100$。

## 作者：SUNCHAOYI (赞：6)

首先先要得出这三个人的循环节，分别为：`ABC`，`BABC`与`CCAABB`。然后就用这三个不同的循环节去比较正确答案。**其中需要注意的是，在得出最高的正确数后，还要判断有几个人都获得了最高的分数，也就是说，输出的姓名不只$1$个，这一点需要额外注意**。
```
#include <bits/stdc++.h>
using namespace std;
int main()
{
	int n,ans[3],num = 0;string str,d[3] = {"ABC","BABC","CCAABB"},name[3] = {"Adrian","Bruno","Goran"};
	cin>>n>>str;
	for(int i = 0;i < 3;i++)
	{
		ans[i] = 0;//每个人的答案初始化 
		for(int j = 0;j <= str.size() - 1;j++)
			if(str[j] == d[i][j % d[i].size()]) ans[i]++;//依次比较循环节 
		num = max(num,ans[i]);//记录最高的正确数量 
	}
	cout<<num<<endl;
	for(int i = 0;i < 3;i++)//可能不只一个 
		if(ans[i] == num) cout<<name[i]<<endl;
	return 0;
}

```

## 作者：wuyonghuming (赞：2)

## 原理：
如果一个位置对字符串的长度取余数，如果那个位置的字符与答案中对位置取余数的那个字符相同，就说明回答是对的。
## 思路：
先记录答题顺序，输入答案，按照上面的原理做就行了，如果答案相同计数器加一。

最后输出答对题目最多的人答对的题目数，如果一个人答对的数量与答对题目最多的人答对的题目数相同，就把他的名字输出。**注意如果有多个人答对了一样多的题目且他们答对的数量与最大值相同，就要全部输出。**
## 代码：
```cpp
#include <bits/stdc++.h>//头文件
using namespace std;
int main()
{
	int n,a=0,b=0,c=0,m;//计数器清零
	string Adrian="ABC",Bruno="BABC",Goran="CCAABB";//记录答题顺序
	char s;
	cin>>n;//输入题目数量
	for(int i=0;i<n;i++)//循环
	{
		cin>>s;//输入位置是i时的答案
		a+=Adrian[i%3]==s;//验证Adrian是否答对这题
		b+=Bruno[i%4]==s;//验证Bruno是否答对这题
		c+=Goran[i%6]==s;//验证Goran是否答对这题
        //可以加上一个判断式的值，如果成立，它会返回1
	}
	m=max(a,max(b,c));//m是答对题目最多的人答对的题目数
	cout<<m<<endl;//输出答对题目最多的人答对的题目数
	if(a==m)//判断Adrian是否答对这么多题目
		cout<<"Adrian"<<endl;//是就输出
	if(b==m)//判断Bruno是否答对这么多题目
		cout<<"Bruno"<<endl;//是就输出
	if(c==m)//判断Goran是否答对这么多题目
		cout<<"Goran"<<endl;//是就输出
  	return 0;//别忘了
}

```

## 分析题目
我们先生成三个人的答卷，然后再一题一题地核对，统计答对的题数。然后，我们找出三个人中答对题目数最多的人，输出。

## 代码
```cpp
#include<iostream>
using namespace std;
int n,len;
string ans,a,b,g;

void init(){//输入并生成三个人的答卷
	cin>>n;
	cin>>ans;
	while(len<n){
		a+="ABC";
		len+=3;
	}
	len=0;
	while(len<n){
		b+="BABC";
		len+=4;
	}
	len=0;
	while(len<n){
		g+="CCAABB";
		len+=6;
	}
}

//算出某人的分数
inline int cmp(string a,string b,int r){
	int ret=0;
	for(int i=0;i<r;i++)
		if(a[i]==b[i])
			++ret;
	return ret;
}

int main(){
	init();
	int A=cmp(ans,a,n);
	int B=cmp(ans,b,n);
	int G=cmp(ans,g,n);
	int res=max(max(A,B),G);//找出最大分数
	printf("%d\n",res);
   	//输出分数最大的人
	if(A>=B&&A>=G)
		puts("Adrian");
	if(B>=A&&B>=G)
		puts("Bruno");
	if(G>=A&&G>=B)
		puts("Goran");
	return 0;
}
```

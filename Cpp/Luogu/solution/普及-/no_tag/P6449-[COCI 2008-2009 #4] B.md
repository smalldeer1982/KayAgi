# [COCI 2008/2009 #4] B

## 题目描述

给定一个 $2009$ 年的日期，请确定这一天是周几。

## 说明/提示

#### 数据规模与约定

对于全部的测试点，保证给出的日期是一个真实存在于 $2009$ 年的日期。

#### 提示

- $2009$ 年 $1$ 月 $1$ 日是周四。
- 请注意输入时先输入日再输入月。

#### 说明
**题目译自 [COCI2008-2009](https://hsin.hr/coci/archive/2008_2009/) [CONTEST #4](https://hsin.hr/coci/archive/2008_2009/contest4_tasks.pdf) *T2 DATUM***。

## 样例 #1

```
1 1
```

### 输出

```
Thursday```

## 样例 #2

### 输入

```
17 1
```

### 输出

```
Saturday
```

## 样例 #3

### 输入

```
25 9
```

### 输出

```
Friday```

## 作者：Utilokasteinn (赞：7)

其实不用打表也不用循环，可以套用求星期几的公式来做。

原本求y年m月d天的公式是这样的：
```cpp
	if(m==1||m==2)m+=12,y--;
	int c=y/100,y%=100;
	int w=c/4-2*c+y+y/4+26*(m+1)/10+d-1;
	if(w<0)w=(w+(-w/7+1)*7)%7;
	else w%=7;
```
但是因为固定是2009年的某一天，所以可以简化很多：c由y/100变为20，因为2009/100=20,还可以把里面的c全部变为20，c/4变成20/4即5,2*c变成2 *20即
40，因为年分一定，还可以盛掉最后的判断，简化后如下：
```cpp
	int y=9;
	if(m==1||m==2)m+=12,y--;
	int w=(c/4-2*c+y+y/4+26*(m+1)/10+d-1)%7;
```
于是有了下面这份代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	string a[7]={"Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday"};
	//0为星期天，1为星期一，2为星期二，3为星期四，4为星期四，5为星期五，6为星期六。 
	int y=9,m,d;//y代表2009年，简化后为9，m代表月，d代表日 
	cin>>d>>m;//输入日和月 
	if(m==1||m==2)m+=12,y--;//这里是套公式 
	cout<<a[(y+y/4+26*(m+1)/10+d-1)%7];
	return 0;//结束 
}
```
最后附上无注释代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	string a[7]={"Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday"};
	int y=9,m,d;
	cin>>d>>m;
	if(m==1||m==2)m+=12,y--;
	cout<<a[(y+y/4+26*(m+1)/10+d-1)%7];
	return 0;
}
```

## 作者：wuyonghuming (赞：5)

## 思路：
按照时间顺序，一天一天地循环，直到循环到那一天，最后判断那一天是星期几。因为$2009$年是平年，所以那年二月有$28$天。
## 代码：
```cpp
#include <bits/stdc++.h>//头文件
using namespace std;
string day[7]={"Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday"};
int main()
{
	int m,d,s=3;//2008年12月31日是星期三
	cin>>d>>m;//输入日和月
	for(int i=1;i<=12;i++)//一年十二个月
	{
		int l;//l记录这个月的天数
		if(i==1||i==3||i==5||i==7||i==8||i==10||i==12)//这些月份31天
			l=31;
		else if(i==4||i==6||i==9||i==11)//这些月份30天
			l=30;
		else if(i==2)//2月28天
			l=28;
		for(int j=1;j<=l;j++)//循环
		{	
			s++;//s加一，用来计算星期几
			if(j==d&&i==m)//如到了这一天
			{
				cout<<day[s%7];//输出答案
				return 0;//结束
			}
		}
	}
	return 0;//别忘了
}
```

## 作者：sukimo (赞：1)

这道题写起来比较简单，但是有一些边边角角的地方需要留心。我的思路是离线处理出从第$1$个月到第$i$个月的天数前缀和，然后对于第$i$个月第$j$天，算出从$1$月$1$日到这一天一共过去了多少天，中间进行取模化简，再计算包括$1$月$1$日的星期四，现在是星期几，最后细心一点结果就出来了。注意如果是星期天，$7\%7==0 $，但是实际上不是“第$0$天”而是第$7$天，特判一下。

上代码：

```
#include<bits/stdc++.h>
using namespace std;
int pre[13]={0,31,59,90,120,151,181,212,243,273,304,334,365};
void print(int x){
	switch(x){
		case 1:printf("Monday");break;
		case 2:printf("Tuesday");break;
		case 3:printf("Wednesday");break;
		case 4:printf("Thursday");break;
		case 5:printf("Friday");break;
		case 6:printf("Saturday");break;
		case 0:printf("Sunday");break;
	}
}
int main(){
	int day,month;scanf("%d%d",&day,&month);
	print(((pre[month-1]+day-1)%7+4)%7);
	return 0;
}
```

	for (int i=1;i<m;++i) days+=day[i]; //注意一定是m，因为第m个月还没有过完
	days+=d;cout<<st[days%7]; //days%7表示星期几
	return 0; //完美的结束
}

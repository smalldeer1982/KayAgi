# [信息与未来 2019] 鸡兔同笼

## 题目描述

有三种动物被关在笼子里，它们分别是：
- 鸡，每只鸡有一个头、两只脚；
- 三脚猫，每只三脚猫有一个头，三只脚；
- 兔子，每只兔子有一个头，四只脚。

现在知道笼子中一共有 $x$ 个头、$y$ 只脚，但这个信息并不足以推导出鸡、兔和三脚猫的准确数量。这时候就需要编程来帮忙了——请你求出笼子里分别至少、至多有几只兔子。

## 说明/提示

对于 $50\%$ 的测试数据，有 $1\le x\le 10^6$。

对于 $100\%$ 的测试数据，有 $1\le x\le10^9,1\le T\le20$。
> 本题原始满分为 $20\text{pts}$。

## 样例 #1

### 输入

```
4
6 12
6 21
6 24
12345 40000```

### 输出

```
0 0
3 4
6 6
2965 7655```

# 题解

本题考察数学、方程组。

我们可以令鸡的数量为 $a$、三脚猫的数量为 $b$、兔子的数量为 $c$。题目给出两个方程：

$$\begin{cases}
a+b+c=x\\
2a+3b+4c=y
\end{cases}$$


我们可以消去 $a$。令 $a=x-b-c$，代入第二个方程得到

$$2(x-b-c)+3b+4c=y$$

化简后，

$$2x-2b-2c+3b+4c=y \quad\Longrightarrow\quad 2x+b+2c=y$$

即 $b=y-2x-2c$。

同时，由于 $a= x-b-c = x-(y-2x-2c)-c = 3x-y+c$，为了保证 $a\ge0$ 和 $b\ge0$（动物数量不能为负），我们必须有：
- $3x-y+c\ge0$  即 $c\ge y-3x$；
- $y-2x-2c\ge0$  即 $c\le \frac{y-2x}{2}$。

另外，$c$ 本身也必须非负，所以综合起来：

$$c\ge \max(0,\, y-3x) \quad \text{且} \quad c\le \frac{y-2x}{2}$$

因此，所有满足条件的兔子数量的取值范围为：

$$\text{下界} = \max(0,\, y-3x) \qquad \text{上界} = \left\lfloor \frac{y-2x}{2} \right\rfloor$$

参考代码：

```cpp
// 下界：保证 a >= 0
long long low = max(0LL, y - 3 * x);
// 上界：保证 b >= 0
long long high = (y - 2 * x) / 2;
cout << low << " " << high << "\n";
```

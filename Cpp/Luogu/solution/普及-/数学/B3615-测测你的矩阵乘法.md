# 测测你的矩阵乘法

## 题目描述

给定两个大小为均为 $512 \times 512$，每个元素均为整数，值域为 $ [0, 1024) $ 矩阵 $A, B$，定义为

$$ \begin{aligned}
	A\left[i, j\right] &= \left(\left(i \mathbin{\mathrm{or}} j\right) + j\right) \mathbin{\mathrm{xor}} \mathrm{seed}_A \\
    B\left[i, j \right] &= \left( \left(i \mathbin{\mathrm{and}} j \right) + i \right) \mathbin{\mathrm{xor}} \mathrm{seed}_B
\end{aligned} $$

其中 $i, j \in [0, 512)$。

请计算 $C = A \times B$。

```
0 0```

### 输出

14942208

29097984

# 题解

## 作者：览遍千秋 (赞：8)

对于 $n \times p$ 的矩阵 A 和 $p \times m$ 的矩阵 B，$A \times B$ 可以得到一个 $n \times m$ 的矩阵 C。

$C_{i,j}$ 由 A 矩阵第 $i$ 行与 B 矩阵第 $j$ 列对应元素乘积和得到。A  矩阵的第 $i$ 行共有 $p$ 个元素，B 矩阵的第 $j$ 列共有 $p$ 个元素。对应元素即 $A_{i,k},B_{k,j}$，其中 $k=1,2,3, \cdots,p$。

即有，

$$C_{i,j}=\sum\limits_{k=1}^pA_{i,k}\times B_{k,j}$$

具体到本题中，$n=m=p=N$。

先枚举 $i,j$，再枚举 $k$ 即可。

```cpp
void multiply (int c[N][N], int a[N][N], int b[N][N]) {
	for(int i = 0; i < N; i++) {
		for(int j = 0; j < N; j++) {
			for(int k = 0; k < N; k++) {
				c[i][j] += a[i][k] * b[k][j];
			}
		}
	} 
}
```

# [NOIP 2007 普及组] 奖学金

## 题目背景

NOIP2007 普及组 T1

## 题目描述

某小学最近得到了一笔赞助，打算拿出其中一部分为学习成绩优秀的前 $5$ 名学生发奖学金。期末，每个学生都有 $3$ 门课的成绩：语文、数学、英语。先按总分从高到低排序，如果两个同学总分相同，再按语文成绩从高到低排序，如果两个同学总分和语文成绩都相同，那么规定学号小的同学排在前面，这样，每个学生的排序是唯一确定的。

任务：先根据输入的 $3$ 门课的成绩计算总分，然后按上述规则排序，最后按排名顺序输出前五名名学生的学号和总分。

注意，在前 $5$ 名同学中，每个人的奖学金都不相同，因此，你必须严格按上述规则排序。例如，在某个正确答案中，如果前两行的输出数据（每行输出两个数：学号、总分) 是：

```plain
7 279  
5 279
```

这两行数据的含义是：总分最高的两个同学的学号依次是 $7$ 号、$5$ 号。这两名同学的总分都是 $279$ (总分等于输入的语文、数学、英语三科成绩之和) ，但学号为 $7$ 的学生语文成绩更高一些。

如果你的前两名的输出数据是：

```plain
5 279  
7 279
```

则按输出错误处理，不能得分。

**结构体排序严重好评**

众所周知，`sort`是一个特别好的东西，但当排序面对的对象是*结构体*时，`sort`就不能自动排了，还需要手写填一个`comp`函数。

**comp函数**框架：
```
bool comp(结构体名称 x,结构体名称 y)
{
	按照题目的要求进行比较，并返回true或false;
}
```
在这道题中，comp的框架完全可以这样写：
```
bool comp(xx x,xx y)
{
	if(x.zf!=y.zf)//如果两个变量的总分不相同
		return x.zf>y.zf;//直接比较，总分大的在前面
	else
		if(x.yw!=y.yw)//如果两个学生总分相同、语文分数不相同
			return x.yw>y.yw;//语文分数大的排在前面
		else
			return x.bh<y.bh;//如果总分相同、语文分数也相同，编号小的在前面
}
```

# 题目思路
直接根据题意模拟。我们用一个结构体存储每个学生的各科成绩，计算总分，同时记录学号。

那么，如何对数据排序呢？

我们先比较两者总分是否相同，相同则比较语文，语文相同则比较学号，语文不同则成绩高的在前，总分不同则总分高的在前。

最后输出前 $5$ 名的学号以及总分即可。

# 代码
注：本代码仅供参考。

```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>
using namespace std;
int n; 
struct node{
	int chines,math,eng,sum,num;
} stu[305];
bool cmp(node x,node y){
	if(x.sum==y.sum){ //如果两者总分相同，进一步比较语文成绩 
		if(x.chines==y.chines){ //如果语文成绩相同，进一步比较学号 
			return x.num<y.num;
		}
		else{
			return x.chines>y.chines; //如果语文成绩不同，成绩高的排在前面 
		}
	}
	else{
		return x.sum>y.sum; //如果两者总分不同，总分高的排在前面 
	}
}
int main(){
	//输入 
	scanf("%d",&n);
	for(int i=1;i<=n;i++){
		scanf("%d %d %d",&stu[i].chines,&stu[i].math,&stu[i].eng);
		stu[i].sum=stu[i].chines+stu[i].math+stu[i].eng; //计算总分 
		stu[i].num=i; //记录学号 
	}
	sort(stu+1,stu+n+1,cmp); //根据 cmp 函数排序 
	for(int i=1;i<=5;i++){ //输出前 5 名的学号和总分 
		printf("%d %d\n",stu[i].num,stu[i].sum);
	}
	return 0;
}
```

# 题解：P1093 [NOIP2007 普及组] 奖学金

## 题意

输入 $n$ 个学生的语、数、英成绩，按照总分排序，并输出前五名。
要求：
+ 如果两个学生总分相同，则按照语文成绩排序
+ 如果语文成绩相同，则学号小的排在前面。

## 思路

用结构体记录每名学生的语数英总分和学号，使用 `sort` 排序，`cmp` 函数特判，最后输出。

## 前置知识

### 结构体

结构体（$\texttt{struct}$）可以看成将多个变量封装在一个变量内。
例如定义结构体数组 $s$ 的代码：

```cpp
struct S{
    string name;
    int age;
}s[100];
```

大写 $S$ 表示结构体**类型名**，在这个结构体中，包含 $\texttt{string}$ 类型的变量 `name` 和 $\texttt{int}$ 类型的变量 `age`。这个结构体数组的名字叫作 $s$，长度为 $100$。

## 代码及具体讲解

```cpp
#include<bits/stdc++.h>//万能头 
#define int long long
using namespace std;
struct S{//定义结构体 
    int a,b,c;
    int sum;
    int m;
}s[305];
bool cmp(S a,S b){//构建比较函数 （排序用） 
    if(a.sum==b.sum){//当总分相同时 
        if(a.a==b.a) return a.m<b.m;//如果语文成绩相同就按学号从小到大排序 
        else return a.a>b.a;//如果语文成绩不相同就按语文成绩从大到小排序 
    }
    else
        return a.sum>b.sum;//如果总分不相同就按总分从大到小排序 
}
signed main(){
    int n;
    cin>>n;
    for(int i=1;i<=n;i++){
        cin>>s[i].a>>s[i].b>>s[i].c;//输入语数英成绩 
        s[i].sum=s[i].a+s[i].b+s[i].c;//计算总分 
        s[i].m=i;//标记学号 
    }
    sort(s+1,s+1+n,cmp);//用快排函数以及自定义比较函数将结构体排序 
    for(int i=1;i<=5;i++){//输出前五名 
        cout<<s[i].m<<' '<<s[i].sum<<'\n';
    }
    return 0;//完美结束 
}
```

bool cmp(node x,node y){
	if(x.sum!=y.sum)
		return x.sum>y.sum;
	if(x.c!=y.c)
		return x.c>y.c;
	return x.id<y.id;
}
```

# 题解：P1093 [NOIP2007 普及组] 奖学金

[![](https://img.shields.io/badge/题目-传送门-f39c11)](https://www.luogu.com.cn/problem/P1093)

## 思路

这道题可以用结构体排序的方法做。

### 什么是结构体？

> **结构体**（struct），可以看做是一系列称为成员元素的组合体。
>
> 可以看做是自定义的数据类型。

（以上摘自 [OI-Wiki](https://oi-wiki.org/lang/struct/)）

如果有些数据可以放在一起使用（如本题中一个学生的语数英三科放在一起排序），此时可以用结构体。

### 怎么定义结构体？

``` cpp
struct abc
{
    int a, b, c;
};
```

定义了一个结构体 `abc`，它的每个变量都拥有 `a`、`b`、`c` 三个成员。

### 怎么定义一个结构体类型的变量？

和普通变量一样。

``` cpp
abc x, y = {3, 4}, *p, *q = &x, a[100] = {{0}};
```

这段代码定义了 `abc` 类型的两个变量（`x` 和 `y`）、两个指向 `abc` 类型变量的指针（`p` 和 `q`）、一个 `abc` 类型的数组。

`abc y = {3, 4}` 表明初始化 `y` 的 `a` 成员为 `3`、`b` 成员为 `4`。

### 怎么使用这个结构体变量？

可以用 `y.c` 来表示 `y` 的 `c` 成员。下面是一些例子：

``` cpp
y.c = 10; // 将 y 的 c 成员赋值为 10
cout << y.c; // 输出 y 的 c 成员
cin >> y.c; // 输入 y 的 c 成员
```

可以用 `q->a` 或 `(*q).a` 表示 `*q`（即 `q` 指向的那个变量）的 `a` 成员。

可以用 `a[i].b` 表示 `a` 数组的第 `i` 个元素的 `b` 成员。

### 如何排序？

``` cpp
int b[100];
sort(b + l, b + r); // 将 b 数组的 l 到 r 从小到大排序。
sort(b + l, b + r, greater<int>()); // 将 b 数组的 l 到 r 从大到小排序。
bool cmp(abc/*类型*/ x, abc y)
{
    // return x.a < y.a; // 按照 a 成员排序。（注意 sort 是不稳定排序，所以 a 成员相等时顺序可能会换）
    return x.a < y.a || x.a == y.a && x.b < y.b || x.a == y.a && x.b == y.b && x.c < y.c;
    // a 成员为第一关键字、b 成员为第二关键字、c 成员为第三关键字排序。
}
sort(a + l, a + r, cmp); // 将 a 数组“按照 cmp 排序”
```

### 思路

介绍一种神秘方法。

假如你有两个数 $a$ 和 $b$，$b<100$，那么将这个数存为 $a\times 100+b$，变成一个新数 $c$，那么 $c\bmod 100 = b$，$c\div 100=a$。在此基础上观察本题，学号 $n\le 300$，分数总和 $\le 300$，语文 $\le 100$，乘起来等于 $9\times 10^6$，远小于 `int` 上限的 $2^{31}-1$，所以我们可以把每个人按照这个方法来存成一个数。

但是这么存有什么用呢？还是新数 $c$，让它和另一个由 $a'$ 和 $b'$ 组成的 $c'$ 比较，那么这两个数的比较可以看作是先比较 $a$ 和 $a'$，再比较 $b$ 和 $b'$。这样就可以很自然地形成排序的优先级，只需要让更优先的值成为更大的数就好了。

设一个数的学号为 $i$，语文分数为 $x_i$，总分为 $s_i$，那这个数可以被保存为 $s_i\times 10^6+a_i\times10^3+n-i$。注意学号的排序方式是小的优先，因此存为 $n-i$。

然后把这些保存好的数从小到大排序，最后面的五个就是最优秀的了。之后用上面所说的方法输出即可。

### 代码


```cpp
#include<bits/stdc++.h>
using namespace std;
int calc[405], x, y, z, n;
int main() {
	cin >> n;
	for (int i = 1; i <= n; i++) {
		cin >> x >> y >> z;
		calc[i] = (x + y + z) * 1000000 + x * 1000 + n - i;
	}
	sort(calc + 1, calc + n + 1);
	for (int i = n; i > n - 5; i--) cout << n - calc[i] % 1000 << " " << calc[i] / 1000000 << "\n";
	return 0;
```

## Solution

考察结构体的基础题。

# 题目分析

一道结构体排序简单题，按照题目要求输入后用结构体存储，考虑到题目中是按照总成绩、语文成绩、学号作为关键字，因此结构体中用三个变量来表示。

接着自定义排序，先按总成绩排序，相同的语文高的优先，语文成绩相同的学号前的优先，在主函数部分使用 `stable_sort()` 来排序。

最后用循环输出排序后前五位。

## Code

```cpp
#include<bits/stdc++.h>

using namespace std ;
typedef long long ll;
const int N=1e8;
struct stu{
    ll sum;
    ll y;
    ll ad;
}a[N];
bool cmp(stu x,stu y){//自定义排序
    if(x.sum>y.sum){
        return true;//无需交换，下面同理
    }
    else if(x.sum==y.sum){
        if(x.y>y.y){
            return true;
        }
        else if(x.y==y.y){
            if(x.ad<y.ad){
                return true;
            }
            else{
                return false;
            }
        }
        else{
            return false;
        }
    }
    else{
        return false;//需要交换
    }
}
int main ()
{
  ll n;
  cin>>n;
  for(int i=1,x,y,z;i<=n;i++){
	cin>>x>>y>>z;
    a[i].y=x;//语文成绩
    a[i].sum=x+y+z;//总成绩
    a[i].ad=i;//记录地址
  }
  stable_sort(a+1,a+n+1,cmp);//排序

  }

# [常州市赛 2021] 鸡羊同栏

## 题目背景

搬运自 <http://czoj.com.cn/p/439>。数据为民间数据。

## 题目描述

小 $\text{X}$ 是 $\text{CZ}$ 市著名的牧场主。他把 $A$ 只鸡和 $B$ 只羊圈养在一个围栏中。   
一只正常的鸡有 $0$ 只角、 $2$ 条腿；一只正常的羊有 $2$ 只角、 $4$ 条腿。   
某一天，一道神秘的宇宙射线击中了牧场，导致其中的**至多一个动物**发生了变异。   
变异的动物角的只数与腿的条数中**至少有一个**数量变化了。例如，一只正常的鸡变异成了 $3$ 只角 $2$ 条腿。注意：**变异以后动物也不可能有负数个数的角或腿**。   
小 $\text{X}$ 只知道鸡和羊的数量、以及笼子中的动物一共有 $C$ 只角 $D$ 条腿。他想请你帮忙搞清楚变异的情况是什么。 

## 说明/提示

### 样例 $\textbf 1$ 解释
只可能是一只羊变异成了 $1$ 只角 $7$ 条腿 $(0+0+1+2=3,2+2+7+4=15)$。  
### 样例 $\textbf 2$ 解释
$0+0+2+2=4,2+2+4+4=12$，不可能有动物变异。

## 作者：封禁用户 (赞：3)

## 思路

我们先假设一只鸡变异，变异后鸡的角数 $x$ 为总角数 $c$ 减去正常羊的角数 $2\times b$。

此时，变异后鸡的腿数 $y$ 为总腿数 $d$ 减去正常鸡和羊的腿数 $2\times a+4\times b$ 再加上一只正常鸡的腿数 $2$。

我们计算羊变异，变异后羊的角数 $x$ 为 $x+2$，变异后羊的腿数 $y$ 为 $y+2$。

如果没有变异，输出 $-1$。

## 代码


```cpp
#include<bits/stdc++.h>
using namespace std;
long long a,b,c,d,x,y;
int main(){
	cin>>a>>b>>c>>d;
	x=c-(2*b),y=d-(2*a)-(4*b)+2;
	if(x<0||y<0)cout<<-1<<endl;
	else cout<<x<<" "<<y<<endl;
	if(x+2<0||y+2<0)cout<<-1;
	else cout<<x+2<<" "<<y+2;
}
```

## 作者：Sliarae (赞：2)

考虑算出变异后角和腿的数量增加了多少。原来角的数量是 $2b$，腿的数量是 $2a + 4b$，现在分别变成了 $c, d$。于是我们令 $c \gets c - 2b, d \gets d - 2a - 4b$。那么角的数量增加了 $c$，腿的数量增加了 $d$（$c, d$ 可能为负数）。

此时如果一只动物发生了变异，那么它的角的数量应该增加 $c$，腿的数量应该增加 $d$。首先特判掉 $c = d = 0$ 的情况，此时不可能发生变异，输出两个 `-1`。

否则假设变异的是鸡，那么它有 $c$ 个角，$d + 2$ 条腿，如果这两个数都非负，那么就输出，否则输出 `-1`。变异的是羊的情况同理。

```cpp
#include <iostream>

using namespace std;
using LL = long long;

int main () {
	cin.tie(0)->sync_with_stdio(0);
	LL a, b, c, d;
	cin >> a >> b >> c >> d;
	c -= 2 * b, d -= 2 * a + 4 * b;
	if (!c && !d) return cout << -1 << '\n' << -1 << '\n', 0;
	LL x = c, y = 2 + d;
	if (x >= 0 && y >= 0) cout << x << ' ' << y << '\n';
	else cout << -1 << '\n';
	x = 2 + c, y = 4 + d;
	if (x >= 0 && y >= 0) cout << x << ' ' << y << '\n';
	else cout << -1 << '\n';
	return 0; 
}
```

# Solution
这道题用到较为基础的数学，我们可以将题目分情况讨论。令变异后的角数为 $x$，腿数为 $y$。

当鸡发生变异时，可列出 $x=c-2b$，$y=d-(2a+4b)+2$，即 $y=d-2a-4b+2$。

当羊发生变异时，我们可以看到羊的腿数和角数都比鸡多 $2$，所以可以列出 $x=c-2b+2$，$y=d-2a-4b+4$。

有了以上式子，我们直接代入计算即可。题目中要求变异之后的角和腿数量均为非负整数，故我们只需要判断一下 $x$ 和 $y$ 是否非负即可。
# Code

```cpp
#include<bits/stdc++.h>
using namespace std;
long long a,b,c,d,x,y,x_,y_;
int main()
{
	ios::sync_with_stdio(0);
	cin.tie(0),cout.tie(0);
	cin>>a>>b>>c>>d;
	x=c-(2*b),y=d-(2*a)-(4*b)+2;
	x_=x+2,y_=y+2;//羊的腿数和角数都比鸡多 2
	if(x<0||y<0) cout<<-1<<endl;//判断是否非负 
	else cout<<x<<" "<<y<<endl;
	if(x_<0||y_<0) cout<<-1<<endl;
	else cout<<x_<<" "<<y_<<endl;
	return 0;//下次再见 
}
```

## 思路

~~第一眼还以为是鸡兔同笼就进来了~~。

首先我们知道正常的鸡有 $0$ 个角和 $2$ 条腿，正常的羊有 $2$ 个角和 $4$ 条腿。因此我们可以先求出**变异后的角、腿数量**。即变异后的角总数是 $c = c - 2b$，变异后的腿总数是 $d = 2a + 4b$。而为 $0$ 时，不满足变异后的条件，所以输出两个  `-1` 即可，**注意换行**。

然后就是和鸡兔同笼里很像的假设法了：

- 假设变异的是一只鸡，那鸡变异后的角数为 $c$，腿数为 $d + 2$。因为正常的鸡有 $2$ 只脚。如果 $c \ge 0$ 且 $d + 2 \ge 0$，说明可能是一只鸡变异了，输出变异后的角和腿数。否则输出 `-1`。

- 假设变异的是一只羊，同理，只不过注意正常羊的角数为 $2$，腿数为 $4$。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	int a,b,c,d;
	cin>>a>>b>>c>>d;
	c-=2*b,d-=2*a+4*b;
	//变异的情况 
	if(c==0&&d==0){
		cout<<"-1\n-1";	
		return 0; 
	}
	//如果一个动物都没有变异 
	int jj=c,jt=d+2;
	//这个是鸡角、鸡腿的变异后数量。
	//因为正常的鸡有2只角，所以加2 
	if(jj<0||jt<0) cout<<"-1\n";
	else cout<<jj<<" "<<jt<<endl;
	int yj=c+2,yt=4+d;
	//这个是羊角、羊腿的变异后数量。
	//因为正常的羊有2只角，所以加2；有4只角，所以加4 
	if(yj<0||yt<0) cout<<-1;
	else cout<<yj<<" "<<yt;
	return 0;
}
```

## 作者：hezhiran (赞：1)

## 题意

有 $A$ 只鸡和 $B$ 只羊，一只正常的鸡有 $0$ 只角、 $2$ 条腿；一只正常的羊有 $2$ 只角、 $4$ 条腿。有**至多一种**动物在角数和腿数之中有**至少一种**的数量会变，但不为负。

## 思路

我们假设变异的为鸡，那么我们可以发现：

1、鸡现在的角数为总角数减去兔子数量乘上兔子角数，即 $c - 2 \times b$。

2、鸡现在的腿数为总腿数减去兔子数量乘上兔子腿数再减去鸡数量乘上鸡腿数再加上鸡原来的腿数，即  $d - 2 \times a - 4 \times b + 2$。

我们再假设变异的为兔子，那么兔子的角数就为变异的为鸡时鸡的角数加二，即 $c - 2 \times b + 2$，兔子的腿数同理，为 $d - 2 \times a - 4 \times b + 2 + 2$。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
long long a,b,c,d;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);
	cin>>a>>b>>c>>d;
	int j=c-2*b;
	int t=d-2*a-4*b+2;
	if(j<0||t<0){
		cout<<-1<<'\n';
	}
	else{
		cout<<j<<' '<<t<<'\n';
	}
	if(j+2<0||t+2<0){
		cout<<-1<<'\n';
	}
	else{
		cout<<j+2<<' '<<t+2<<'\n';
	}
	return 0;
}
```

## 解题思路
根据题意，有两种变异情况，分别为鸡变异和羊变异。 
- 鸡变异情况  
   - 变异角数的情况：
   ```x1 = c - (2 * b)```
   - 变异腿数的情况：
   ```y1 = d - (2 * a) - (4 * b) + 2```
   - 若 $x1 < 0$ 或 $y1 < 0$，则输出 `-1`，否则输出 $x1$ 和 $y1$。
- 羊变异情况
   - 题目中正常的每只羊都比鸡多两只脚和两只腿
   - 变异角数的情况：
   ```x2 = x1 + 2```
   - 变异腿数的情况：
   ```y2 = y1 + 2```
   - 若 $x2 < 0$ 或 $y2 < 0$，则输出 `-1`，否则输出 $x2$ 和 $y2$。
## 输出细节
每行若输出两个数则用空格隔开，每行之间用换行隔开。  
输出有两行，不要多打一些没用的。
## AC CODE
```cpp
#include <bits/stdc++.h>

int main() {
	int a, b, c, d;

	int x2 = x1 + 2, y2 = y1 + 2; //羊的变异情况
	if (x1 < 0 || y1 < 0) { //第一轮判断
		cout << -1 << endl;
	} else {
		cout << x1 << " " << y1 << endl;

		cout << -1;
	} else {
		cout << x2 << " " << y2;
	}
	//输出注意空格和换行
	return 0;
}
```

### B4203 [常州市程序设计小能手 2021] 鸡羊同栏 Solution

[AC 记录](https://www.luogu.com.cn/record/206986615)  
[题目传送门](https://www.luogu.com.cn/problem/B4203)

#### 我的思路

1. 判断鸡变异：
   - 变异鸡的角数：$x = C - 2B$
   - 变异鸡的腿数：$y = D - 4B - 2A + 2$
   - 若 $x \geq 0$ 且 $y \geq 0$，且 $x \neq 0$ 或 $y \neq 2$，则输出 $x$ 和 $y$，否则输出 `-1`。

2. 判断羊变异：
   - 变异羊的角数：$x = C - 2(B - 1)$
   - 变异羊的腿数：$y = D - 4(B - 1) - 2A$
   - 若 $x \geq 0$ 且 $y \geq 0$，且 $x \neq 2$ 或 $y \neq 4$，则输出 $x$ 和 $y$，否则输出 `-1`。

3. 特殊情况：
   - 若总角数和总腿数与正常情况一致，则直接输出两个 `-1`。

#### 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	long long a, b, c, d;//不开LL见祖宗（bushi
	cin >> a >> b >> c >> d;

	// 判断鸡变异
	long long x_c = c - 2 * b;
	long long y_c = d - 4 * b - 2 * a + 2;
	if (x_c >= 0 && y_c >= 0 &&
		(x_c != 0 || y_c != 2)) {
		cout << x_c << " " << y_c << endl;
	} else {
		cout << -1 << endl;
	}

	// 判断羊变异
	long long x_s = c - 2 * (b - 1);
	long long y_s = d - 4 * (b - 1) - 2 * a;
	if (x_s >= 0 && y_s >= 0 && (x_s != 2 || y_s != 4)) {
		cout << x_s << " " << y_s << endl;
	} else {
		cout << -1 << endl;
	}

	return 0;
}

```

# 解析
假设鸡变异，变异后鸡的角数 $h_1$ 为总角数 $c$ 减去正常羊的角数 $2 \times b$。假设鸡变异，变异后鸡的腿数 $l_1$ 为总腿数 $d$ 减去正常鸡和羊的腿数 $2 \times a + 4 \times b$ 再加上一只正常鸡的腿数 $2$。计算假设羊变异时，变异后羊的角数 $h_2$ 为 $h_1 + 2$，变异后羊的腿数 $l_2$ 为 $l_1 + 2$。

最后判断是否羊或鸡是否变异，不是就输出 -1（特判原角数和腿数是否已经达到要求，是则输出 -1）。

# CODE

```cpp
#include <iostream>
using namespace std;
#define P(x,y) (x>=0&&y>=0?cout<<x<<" "<<y:cout<<-1)<<endl
int main() {
    long long a,b,c,d;
    cin>>a>>b>>c>>d;
    long long h1=c-2*b,l1=d-2*a-4*b+2,h2=h1+2,l2=l1+2;
    if(c==(2*b)&&d==(2*a+4*b)){
    	cout<<-1<<"\n"<<-1;
    	return 0;
	}
    P(h1,l1);P(h2,l2);
    return 0;
}
```

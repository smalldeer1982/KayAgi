# [蓝桥杯 2023 国 Python A] 跑步计划

## 题目描述

小蓝计划在某天的日期中出现 $1$ 时跑 $5$ 千米，否则只跑 $1$ 千米。注意日期中出现 $1$ 不仅指年月日也指星期。

请问按照小蓝的计划，$2023$ 年小蓝总共会跑步锻炼多少千米？

例如，$5$ 月 $1$ 日、$1$ 月 $13$ 日、$11$ 月 $5$ 日、$4$ 月 $3$ 日（星期一）小蓝会跑 $5$ 千米，而 $5$ 月 $23$ 日小蓝会跑 $1$ 千米。

（示例日期均为 $2023$ 年）

### 思路：

---

我们可以把每个月枚举一遍：

$1$ 月：每个日子都有 $1$，所以每天跑 $5$ 千米。一共 $5 \times 31 = 155$ 千米，星期一全重复。

$2$ 月：有 $2$ 月 $1$ 日，$2$ 月 $10$ 日 $\sim 2$ 月 $19$ 日，$2$ 月 $21$ 日有 $1$, 除上述日期外，还有 $3$ 天是**星期一**（别忘了）。有 $14$ 天，加上剩下的 $14$ 天，每天 $1$ 千米。一共 $14 \times 5 + 14 \times 1 = 84$ 千米。

$3$ 月 $\sim$ $9$ 月与 $2$ 月同理，除天数不同以外，分别有 $16$、$14$、$16$、$14$、$16$、$16$、$13$ 天要跑 $5$ 千米（**星期一**！！！）。一共 $16 + 14 + 16 + 14 + 16 + 16 + 13 = 105$ 天。有 $15 + 16 + 15 + 16 + 15 + 15 + 17 = 109$ 天只要跑 $1$ 千米。跑了 $105 \times 5 + 109 \times 1 = 634$ 公里。

$10$ 月 $\sim$ $12$ 月每天都有 $1$。一共 $31 + 30 + 31 = 92$ 天。跑了 $92 \times 5 = 460$ 公里，星期一全重复。

### 答案：

---

一共 $155 + 84 + 634 + 460 = 1333$ 公里。

# P10983 题解

Python 题解。

## 思路

检查某一个数是否含有数位 $1$，其实可以直接 `"1" in str(k)`。

枚举 $month$，当 $month \in [1,3,5,7,8,10,12]$，则 $days=31$，当 $month \in [4,6,9,11]$，则 $days=30$，当 $month=2$，则 $days=28$，然后枚举 $day$，每次枚举之后把 $xq$ 星期数 $+1$，$xq=8$ 的时候将 $xq$ 重置为 $1$。

由于 $2023$ 年 $1$ 月 $1$ 日是周日，所以 $xq$ 初始为 $7$。

## 代码

```py
ans=0
xq=7

def check(mon,day,xq):
    return ("1" in str(mon)) or ("1" in str(day)) or (xq==1)

for month in range(1,13):
    days=0
    if month in [1,3,5,7,8,10,12]:
        days=31
    elif month!=2:
        days=30
    else:
        days=28
    for day in range(1,days+1):
        if check(month,day,xq):
            ans+=5
        else:
            ans+=1
        xq+=1
        if xq==8:
            xq=1

print(ans)
```

### 思路
直接模拟即可。  
$x \bmod 10$ 获取个位数，$\lfloor\dfrac{x}{10}\rfloor$ 获取十位数（`x // 10`），`wdotw` 函数获取星期几（详见代码）。

### 代码


```python
months = [0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
# 求 m 月 d 日是星期几
def wdotw(m, d):
    sum = 0
    # 求 1 月 1 日至 m - 1 月最后一天总的天数
    for i in range(1, m):
        sum += months[i]
    # 求 1 月 1 日至 m 月 d 日总的天数
    sum += d
    # 1 月 2 日为星期一所以要减一再模七（星期日为 0）
    return (sum - 1) % 7
ans = 0
for i in range(1, 13):
	for j in range(1, months[i] + 1):
		if wdotw(i, j) == 1 or i % 10 == 1 or i // 10 == 1 or j % 10 == 1 or j // 10 == 1:
			ans += 5
		else:
			ans += 1
print(ans)
```

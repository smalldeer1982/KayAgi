# [传智杯 #3 练习赛] 评委打分

## 题目描述

小 A 参加一个综艺节目。一共有 $n(3 \le n \le 10^6)$ 名评委参与打分（分数范围是 0 到 100 的整数），每个评委依次亮出自己的得分。

为了节目效果，要求从第三个评委开始，每当第 $i$ 个评委给出打分后，立刻计算出出这个选手在前 $i$ 名评委的打分中，去掉一个最高分和一个最低分，剩下 $i-2$ 个评委的平均分，保留 $2$ 位小数。

## 样例 #1

### 输入

```
6
11 45 14 19 19 81```

### 输出

```
14.00
16.50
17.33
24.25```

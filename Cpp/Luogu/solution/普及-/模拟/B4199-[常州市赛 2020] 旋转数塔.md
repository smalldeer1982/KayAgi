## 题目描述

小 $\text X$ 是 $\text C$ 城著名的考古学家。一日，他被重金聘请调查一座荒漠中的宫殿。  
宫殿大门紧闭，但这难不倒聪明的小 $\text X$ 。他在隐蔽处发现了两个数字正方形： 
![](https://cdn.luogu.com.cn/upload/image_hosting/sdypvsfa.png)
小 $\text X$ 略加思索便发现了其中的奥妙：把数字从小到大依次填入正方形中，每次填最外面的一圈；每一圈从左上角开始，按照顺时针、逆时针、顺时针……的顺序填。  
作为小 $\text X$ 的助手，他希望你帮助他以相同的规律填上旁边 $n\times n$ 的空白方阵。这里方阵是数字正方形的简称，通常用二维数组来存放其中的数字。

## 作者：Sliarae (赞：1)

容易发现最终的方阵可以拆成若干个环，其中从外到内，第一个环的方向为顺时针，第二个环的方向为逆时针，第三个环的方向为顺时针，以此类推。

从外往内每次填一个环的数字，假设当前环为 $x \in \{l, r\} \vee y \in \{l, r\}$ 的所有格子 $(x, y)$。从 $(x, x)$ 开始，沿顺/逆时针方向转，如果碰到边界或者已经访问过的格子就切换方向，如果这个环上所有格子都被填完就退出。

```cpp
#include <iostream>

using namespace std;

const int kN = 52;

int n, now, a[kN][kN];

int main () {
	cin.tie(0)->sync_with_stdio(0);
	cin >> n;
	for (int l = 1, r = n; l <= r; ++l, --r) {
		const int dx[4] = {0, 1, 0, -1};
		const int dy[4] = {1, 0, -1, 0};
		int d = (l & 1 ? 0 : 1), c = (l & 1 ? 1 : 3);  
		for (int x = l, y = l; !a[x][y] && (x == l || y == l || x == r || y == r); ) {
			a[x][y] = ++now;
			if (x + dx[d] < 1 || x + dx[d] > n || y + dy[d] < 1 || y + dy[d] > n || a[x + dx[d]][y + dy[d]])
				d = (d + c) % 4;
			x += dx[d], y += dy[d];
		}
	}
	for (int i = 1; i <= n; ++i) {
		for (int j = 1; j <= n; ++j)
			cout << a[i][j] << ' ';
		cout << '\n';
	}
	return 0; 
}
```

## 作者：dg114514 (赞：1)

一道比较毒瘤的上位橙模拟题。\
首先定义 $a,i,j,cnt$，表示答案矩阵和当前要填的下标以及要填的数。\
然后由于方便定义一个 `work(int dir)` 函数，当 $dir=1$：往右移动；当 $dir=2$：往下移动；当 $dir=3$：往左移动；当 $dir=4$：往上移动。详见：

```cpp
void work(int dir){
	switch(dir){
		case 1:while(j<n&&!a[i][j+1]) a[i][++j]=++cnt;break;//只要 j<n 并且 a[i][j+1] 没有填，就向右填。
		case 2:while(i<n&&!a[i+1][j]) a[++i][j]=++cnt;break;//只要 i<n 并且 a[i+1][j] 没有填，就向下填。
		case 3:while(j>1&&!a[i][j-1]) a[i][--j]=++cnt;break;//只要 j>1 并且 a[i][j-1] 没有填，就向左填。
		case 4:while(i>1&&!a[i-1][j]) a[--i][j]=++cnt;break;//只要 i>n 并且 a[i-1][j] 没有填，就向上填。
	}
}
```

**坑点**：这里的代码由于种种原因需要初始化为 `i=1,j=0;`

然后发现：题目的过程就相当于正着填一圈然后往右一格后又反着填一圈往下一格又正着填一圈这样一直循环。而正着填一圈就是 `work(1),work(2),work(3),work(4);`，反着就是 `work(2),work(1),work(4),work(3);`。\
可以写出以下代码：

```cpp
for(int time=1;time<=n;time++){
		work(1),work(2),work(3),work(4);
		a[i][++j]=++cnt;
		work(2),work(1),work(4),work(3);
		a[++i][j]=++cnt;
	}
	for(int i=1;i<=n;i++)
		for(int j=1;j<=n;j++)
			cout<<a[i][j]<<" \n"[j==n];
```

然而交上去 [WA $10\ pts$](https://www.luogu.com.cn/record/205905105)。随便测了一个 `2` 发现直接输出

```
1 2
4 5
```

很容易发现，是因为填的圈数过多，`a[i][++j]=++cnt;` 和 `a[++i][j]=++cnt;` 会把原有数据覆盖。可以增加一个限定条件当 $cnt=n^2$ 时终止循环。在 `a[++i][j]=++cnt;` 和 `a[i][++j]=++cnt;` 前面加上即可。\
然而。交上去 [WA $90\ pts$](https://www.luogu.com.cn/record/205902370)。发现测试点 $1$ 中 $n=1$。测试 $n=1$ 时发现输出 `0`。因为~~我也不知道~~ $n$ 过小，导致出现问题，加上特判即可。

## AC Code：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[110][110],x,y,cnt;
bool flag=0; 
int main(){
	cin>>n;
	flag=1;
	x=1;y=n;
	while(x<=y){
		if(x==y)a[x][x]=++cnt;
		else if(flag){
			for(int j=x;j<=y;j++)a[x][j]=++cnt;
			for(int i=x+1;i<=y;i++)a[i][y]=++cnt;
			for(int j=y-1;j>=x;j--)a[y][j]=++cnt;
			for(int i=y-1;i>x;i--)a[i][x]=++cnt;
		}
		else{
			for(int i=x;i<=y;i++)a[i][x]=++cnt;
			for(int j=x+1;j<=y;j++)a[y][j]=++cnt;
			for(int i=y-1;i>=x;i--)a[i][y]=++cnt;
			for(int j=y-1;j>x;j--)a[x][j]=++cnt;
		}
		x++;y--;flag=!flag;
	}
	for(int i=1;i<=n;i++){
		for(int j=1;j<=n;j++)
			cout<<a[i][j]<<" ";
		cout<<endl;
	}
``````

```cpp
#include<bits/stdc++.h>
using namespace std;
int c[100],a[100][100],n;//c数组表示每一圈所需要填充的数字数量 
void dfs(int step,int s,int k,int x,int y){
	//step:总共进行的步数
	//s:该圈进行的步数
	//k:正在进行的圈数，奇数为顺时针填充，偶数为逆时针填充 
	//x,y:当前填充的坐标 
	if(step==n*n+1)return;//全部填充完了，退出 
	a[x][y]=step;//填充 
	if(s==c[k]){//这一圈已经填完了 
		if(k%2==0)dfs(step+1,1,k+1,x+1,y);//进行顺下一圈(顺时针) 
		else dfs(step+1,1,k+1,x,y+1);//进行下一圈(逆时针) 
	}
	else{//多写几个样例，找填充的规律 
		if(k%2==0){//逆时针 
			if(a[x][y-1]!=0&&a[x+1][y]==0)dfs(step+1,s+1,k,x+1,y);
			//如果左边被填充过了且下面没填充过，往下搜 
			else if(a[x+1][y]!=0&&a[x][y+1]==0)dfs(step+1,s+1,k,x,y+1);
			//如果下面被填充过了且右边没填充过，往右搜 
			else if(a[x][y+1]!=0&&a[x-1][y]==0)dfs(step+1,s+1,k,x-1,y);
			//如果右边被填充过了且上面没填充过，往上搜 
			else if(a[x-1][y]!=0&&a[x][y-1]==0)dfs(step+1,s+1,k,x,y-1);
			//如果上面被填充过了且左边没填充过，往左搜 
		}
		else{//顺时针 
			//和上面差不多，不说了 
			if(a[x-1][y]!=0&&a[x][y+1]==0)dfs(step+1,s+1,k,x,y+1);
			else if(a[x][y+1]!=0&&a[x+1][y]==0)dfs(step+1,s+1,k,x+1,y);

			else if(a[x][y-1]!=0&&a[x-1][y]==0)dfs(step+1,s+1,k,x-1,y);
		}
	}
}
```

## 作者：tkm2013 (赞：0)

不难发现走一轮只有 $10$ 次操作，其中有两次只会走一步，所以这两次我们特殊处理，那么我们先写出方向数组:

```cpp
int dx[8]={0,1,0,-1,1,0,-1,0};
int dy[8]={1,0,-1,0,0,1,0,-1};
```

然后在经历完第四次操作后需要往右走一格，经历完第八次操作后会往下走一格。剩下的代码就和蛇形方阵是一样的。

## Code

```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
int dx[8]={0,1,0,-1,1,0,-1,0};//方向数组
int dy[8]={1,0,-1,0,0,1,0,-1};
int a[55][55];
int cnt=1;
int x=1,y=0;
bool check(){
	if(cnt<=n*n && !a[x][y] && x>=1 && y>=1 && x<=n && y<=n){//是否满足条件
		return 1;
	}
	return 0;
}
int main(){
	ios::sync_with_stdio(false);
	cin.tie(0);cout.tie(0);
	cin>>n;
	while(1){
		for(int i=0;i<4;i++){
			x+=dx[i];y+=dy[i];
			while(check()){
				a[x][y]=cnt;
				cnt++;
				x+=dx[i];y+=dy[i];
			}
			x-=dx[i];y-=dy[i];
		}
		if(cnt>n*n){//特殊处理
			break;
		}
		y++;
		a[x][y]=cnt;
		cnt++;
		for(int i=4;i<8;i++){
			x+=dx[i];y+=dy[i];
			while(check()){
				a[x][y]=cnt;
				cnt++;
				x+=dx[i];y+=dy[i];
			}
			x-=dx[i];y-=dy[i];
		}
		if(cnt>n*n){//特殊处理
			break;
		}
		x++;
		a[x][y]=cnt;
		cnt++;
	}
	for(int i=1;i<=n;i++){
		for(int j=1;j<=n;j++){
			cout<<a[i][j]<<' ';
		}
		cout<<"\n";
	}
	return 0;
}
```

## 作者：Manchester_City_FC (赞：0)

[题目传送门](https://www.luogu.com.cn/problem/B4199)

题目大意就不讲了，蛇形方阵加强版。

考虑按照从外向内的顺序每一圈分别去填，即第一圈为顺时针，第二圈为逆时针，第三圈为顺时针，以此类推。

我们从该圈的起点填数，先按原先方向填数，若发现下一个应该填的位置已经有数了，则转弯继续填数，直到结束。还是不太理解的可以看一下下面的示意图：

![](https://cdn.luogu.com.cn/upload/image_hosting/gz01i26b.png)

总时间复杂度是 $\mathcal O(n^2)$，可以通过。

### AC 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
const int N=52;
int n,a[N][N],cur=1;
int main(){
    cin>>n;
    for(int i=1;i<=n;i++){
        // 上、下、左、右边界
        int x=i,y=n-i+1,l=i,r=n-i+1;
        // 当边界不可相交时停止
        if(x>y||l>r) break;
        // 判断当前层的方向：奇数层顺时针，偶数层逆时针
        if(i&1){
            // 顺时针填充
            // 1. 填充上边行，从左到右
            for(int j=l;j<=r;j++) a[x][j]=cur++;
            // 2. 填充右边列，从上+1到下
            for(int j=x+1;j<=y;j++) a[j][r]=cur++;
            // 3. 填充下边行，从右-1到左（注意条件：避免和上边重复）
            if(x< y) for(int j=r-1;j>=l;j--) a[y][j]=cur++;
            // 4. 填充左边列，从下-1到上+1（避免重复）
            if(l<r) for(int j=y-1;j>x;j--) a[j][l]=cur++;
        }
		else{
            // 逆时针填充
            // 1. 填充左边列，从上到下
            for(int j=x;j<=y;j++) a[j][l]=cur++;
            // 2. 填充下边行，从左+1到右
            for(int j=l+1;j<=r;j++) a[y][j]=cur++;
            // 3. 填充右边列，从下-1到上（注意条件：避免和左边重复）
            if(l<r) for(int j=y-1;j>=x;j--) a[j][r]=cur++;
            // 4. 填充上边行，从右-1到左+1（避免重复）
            if(x<y) for(int j=r-1;j>l;j--) a[x][j]=cur++;
        }
    }
    // 输出结果，注意每行之间用一个空格隔开
    for(int i=1;i<=n;i++){
        for(int j=1;j<=n;j++) cout<<a[i][j]<<' ';
        cout<<endl;
    }
}
```

## 题目思路
本题是 [P5731](/problem/P5731) 的升级版。我们把两道题进行对比。

可以发现，这道题在顺时针走完一圈之后还要逆时针走上一圈。在切换方向**前**还有一个数字的过渡期（即下图蓝圈）。

![](https://cdn.luogu.com.cn/upload/image_hosting/cyt1y7t7.png)

所以我将一个周期设置为 $4\times 2+2\times 1=10$ 个数字，即在原来的八个方向加上两个切换时的过渡期。

因为过渡期只有一个数字，所以我们需要及时判断并切换状态。

其他细节可以查看 [P5731 的题解区](/problem/solution/P5731)，这里不再阐述。

```cpp
		for(int j=1;j<=n;j++)cout<<a[i][j]<<" ";
```

# 解析
一道简单的模拟题。

定义一个变量作为判断顺逆时针的依据，每次操作完就改变，方便判断当前应该顺或逆时针填充。若顺时针，通过四个循环分别完成从左到右填充顶行、从上到下填充右列、从右到左填充底行、从下到上填充左列的操作。若逆时针，通过四个循环分别完成从上到下填充左列、从左到右填充底行、从下到上填充右列、从右到左填充顶行的操作。
# CODE
```cpp
#include <iostream>
using namespace std;
int a[55][55];
int main() {
    int n,num=1,s=0,e,cw=1;
    cin>>n; e=n-1;
    while (s<=e){
        if (s==e)a[s][s]=num++;
        else if(cw){
            for (int j=s;j<=e;a[s][j++]=num++);
            for (int i=s+1;i<=e;a[i++][e]=num++);
            for (int j=e-1;j>=s;a[e][j--]=num++);
            for (int i=e-1;i>s;a[i--][s]=num++);
        }else{
            for (int i=s;i<=e;a[i++][s]=num++);
            for (int j=s+1;j<=e;a[e][j++]=num++);
            for (int i=e-1;i>=s;a[i--][e]=num++);
            for (int j=e-1;j>s;a[s][j--]=num++);
        }
        s++;e--;cw=!cw;
    }
    for (int i=0;i<n;i++) {
        for (int j=0;j<n;cout<<a[i][j++]) if (j) cout<<" ";
        cout<<'\n';
    }
    return 0;
}
```

## 题目描述

小蓝特别喜欢蓝色。最近，小蓝学习了颜色在计算机中的一种表示方法：用三个 $0$ 至 $255$ 之间的整数（包含 $0$ 和 $255$）分别表示颜色的红、绿、蓝三个分量。

在这种颜色的表示方法下，小蓝定义了一种颜色是偏蓝的，是指蓝色分量大于红色分量，且蓝色分量大于绿色分量。例如，红、绿、蓝分别为 $10$、$10$、$11$ 时是偏蓝的；红、绿、蓝分别为 $100$、$200$、$200$ 时不是偏蓝的。

小蓝想知道，有多少种不同的颜色是偏蓝的。两种颜色如果在红、绿、蓝中至少有一个分量值不同，就认为是不同的。

# 题解

## 作者：LostKeyToReach (赞：4)

简单题。

钦定 $b$，则我们要统计满足 $r < b, g < b$ 的数量，这里贡献了 $b^2$，故我们需要求：

$$
\sum_{i = 0} ^ {255} b^2.
$$

使用平方和公式计算 $\dfrac{255 \times 256 \times 511}{6} = 5559680$，输出即可。

代码实现如下：

```python
print(5559680)
```

## 作者：_Epsilon_ (赞：1)

# 题意
有三个数字 $0\leq R,G,B \leq 255$，求出 $B>R$ 且 $B>G$ 的情况数量。
# 分析
我们可以看到这道题是一个提交答案题，所以提供两种思路：
## 思路一 数学
设 $B=n$ 时有 $k$ 个排列满足需求。通过排列的计算公式不难发现，当 $n$ 增加了 $1$，相应的 $\sqrt{k}$ 相比前一个 $\sqrt{k}$ 增加了 $1$。

即可推出公式：$k_n=n^2$。

此时求出 $\sum{k}$ 即可。

复杂度 $O(B)$。

## 程序一

```python
ans = 0
for m in range(256):
    ans = ans + m * m
print(ans)

```
## 思路二 枚举
顾名思义，我们只需要枚举每一个 $R,G,B$ 的情况，然后判断是否符合 $B>R$ 且 $B>G$ 即可。

复杂度 $O(B^3)$。
## 程序二

```python
ans = 0
for R in range(256):
    for G in range(256):
        for B in range(256):
            if (B > R and B > G):
                ans = ans + 1
print(ans)
```

枚举即可。

三重循环先枚举所有可能的红绿值，然后从红和绿的较大值加 $1$ 开始循环，这样所有遍历到的红绿蓝值都是偏蓝的。

### C++

```cpp
#include<iostream>
using namespace std;
int ans,r,g,b;
int main(){
    for(r=0;r<=255;r++)
        for(g=0;g<=255;g++)
            for(b=max(r,g)+1;b<=255;b++)
                ans++;
    cout<<ans;
}
```

ans = 0
for r in range(256): #这里是判断小于

## 作者：htl2025 (赞：0)

## 方法思路
要计算满足蓝色分量 $B$ 大于红色分量 $R$ 和绿色分量  $G$ 的所有颜色组合数目，每个颜色分量的取值范围是   $0$ 到 $255$。我们需要统计所有满足 $B > R$ 且 $B > G$ 的组合数。

**分析条件**：对于每个 $B$，$R$ 和 $G$ 的取值范围是 $0$ 到 $B-1$，因此每个 $B$  对应的组合数是   $B^2$。

**求和公式**：所有可能的 $B$ 值从 $1$ 到 $255$，对应的组合数之和为从 $1$ 到 $255$ 的平方和。使用平方和公式计算：$sum = 1^2 + 2^2 + \cdots + 255^2$。

**平方和公式**：利用数学公式 
$$\frac{n(n+1)(2n+1)}6$$ 
其中 $n=255$，代入公式计算结果。


```cpp
#include <iostream>
using namespace std;
int main() {
    int n = 255;
    int result = n * (n + 1) * (2 * n + 1) / 6;
    cout << result << endl;
    return 0;
}
```

## 作者：封禁用户 (赞：0)

我们可以发现，当蓝色取 $0$ 时，红色和绿色都没有取值可能（因为蓝色取 $0$ 时红色和绿色都不可能小于蓝色），蓝色取 $1$ 时，红色和绿色各有 $1$ 种取值可能（取 $0$），蓝色取 $2$ 时红色和绿色各有 $2$ 种取值可能（取 $0$ 或 $1$），以此类推，蓝色取 $0 \sim 255$ 中的任何一个数，都有这个数的平方种可能。于是答案就是 $\sum_{i=0}^{255} i^2$。做完了。

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int ans=0;
    for(int i=1;i<=255;i++)ans+=i*i;
    cout<<ans;
}
```

运行一遍可得答案为 $5559680$。

## 作者：dashabi35 (赞：0)

[题目传送门](https://www.luogu.com.cn/problem/P12184)

一道典型的循环题。

可以用一个三重循环。

分别表示蓝值，红值，绿值。

当蓝值大于红值且蓝值大于绿值时，就使得 $ans$ 加一。

最后输出 $ans$ 即可。

**CODE:**


```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int ans=0;
	for(int i=0;i<=255;++i)
	{
		for(int j=0;j<=255;++j)
		{
			for(int k=0;k<=255;++k)
			{
				if(k>i&&k>j)  ++ans;
			}
		}
	}
  //i和j分别表示红值和绿值（无序），k表示蓝值
	printf("%d",ans);
 } 
```

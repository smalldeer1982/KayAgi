# [蓝桥杯 2025 省 B/Python B 第二场] 脉冲强度之和

## 题目描述

在蓝桥电子工坊，工程师小蓝正在设计一款智能脉冲生成器，用于驱动一种新型设备。该设备的运行依赖于特定的脉冲强度，用正整数 $p$ 表示，其必须满足以下三个条件：

1. 可由连续 $10$ 个正整数之和组成：即存在一个正整数 $k$，使得脉冲强度 $p = k + (k + 1) + (k + 2) + \cdots + (k + 9)$。
2. 各个数位上的数字都相同：例如 $1111$、$22222$、$333333$ 等。
3. 数值不超过 $20255202$：即 $1 \leq p \leq 20255202$。

通过计算所有符合条件的脉冲强度之和，小蓝能够优化设备运行模式。对此，请帮助他计算这一总和。

# 题解

## 作者：Clare613 (赞：3)

## 思路
由 $p = k + (k + 1) + (k + 2) + \cdots + (k + 9)$ 可以化简，得到 $p = 10k + 45$，所以个位为 $5$。又因为 $1 \leq p \leq 20255202$，所以可以得到符合条件的有 $55$、$555$、$5555$、$55555$、$555555$、$5555555$，加起来的和等于 $6172830$，输出即可。

```cpp
    cin.tie(0)->sync_with_stdio(0);
    cout<<"6172830";
	return 0;
}
```

## 作者：Suwenluo (赞：0)

考察基础数学。

因为 $p=k+(k+1)+(k+2)+⋯+(k+9)$，所以很显然我们得到 $p=10k+1+2+…+9=10k+45$。  
不管 $k$ 是多少，乘 $10$ 以后末尾肯定有个 $0$，$10k$ 的末尾就是 $0$，再加上 $45$，$p$ 的末尾数字必定为 $5$。

又因为题目说 $p$ 的各位数字都相同，所以 $p$ 的每一位都是 $5$。

又因为 $k$ 是正整数，所以我们得到 $p\ge45$。

枚举所有可能的 $p$：
$55$、$555$、$5555$、$55555$、$555555$、$5555555$，加起来就是本题答案，$6172830$，直接输出即可。

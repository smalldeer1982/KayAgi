# [蓝桥杯 2021 省 B] 时间显示

## 题目描述

小蓝要和朋友合作开发一个时间显示的网站。在服务器上，朋友已经获取了当前的时间，用一个整数表示，值为从 1970 年 1 月 1 日 00:00:00 到当前时刻经过的毫秒数。

现在，小蓝要在客户端显示出这个时间。小蓝不用显示出年月日，只需要 显示出时分秒即可，毫秒也不用显示，直接舍去即可。

给定一个用整数表示的时间，请将这个时间对应的时分秒输出。

## 说明/提示

对于所有评测用例, 给定的时间为不超过 $10^{18}$ 的正整数。 

蓝桥杯 2021 第一轮省赛 B 组 F 题。

## 样例 #1

### 输入

```
46800999```

### 输出

```
13:00:00```

## 样例 #2

### 输入

```
1618708103123```

### 输出

```
01:08:23```

# 题解

## 作者：lichenzhen (赞：19)

### 思路
先看一下题目，发现就是一道模拟题，但是又一些细节的。

首先是数据的规模：时间不超过 $10^{18}$,所以需要开 long long。

然后首先要忽略掉毫秒。

这里普及一个知识: $1s$ = $1000ms$

所以需要忽略后面的三位。直接整除 $1000$ 就可以了。

之后就是处理时分秒了：

#### 1.小时
由于只是算一天，$24$ 小时之内，所以要把完整的一天去掉，也就是把对每天的秒数取余，余数就是除了整天剩余的秒数，每天的秒数用`24*60*60`来表示（当然你也可以直接算出来结果是 $86400$）。然后用最后一天剩余的秒数整除每小时的秒数`60*60`（也就是 $3600$），就是小时值了。
#### 2.分钟
分钟要先对每小时的秒数`60*60`取余，余数就是除了整小时以外剩余的秒数。之后整除 $60$ 就是剩余的秒数中整分钟的数，就是分钟值了。
#### 3.秒
直接对60取余，剩下的就是秒了。

时间的处理就搞好了。下面就是输出了，题目要求时、分、秒不足两位时补前导 $0$。所以这是一个细节。

当然你可以用printf进行域宽的输出，但是我这里介绍一个用cin的方法。

要用到两个函数：

一个是`setw()`函数，这个函数可以限定域宽，宽度就是括号里的数字。

另外一个`setfill()`函数，这个函数可以设置当实际的宽度不足`setw()`函数的值时，用某个字符来补足。（`setfill()`函数最好放在`setw()`函数前面）

这样就可以了。

## 参考代码

```cpp
#include<bits/stdc++.h>
using namespace std; 
int main()
{
	long long t,hh,mm,ss;
	cin>>t;
	t/=1000;
	hh=t%(24*60*60)/(60*60);//时
	mm=t%(60*60)/60;//分
	ss=t%60;//秒
	cout<<setfill('0')<<setw(2)<<hh<<setfill('0')<<setw(2)<<mm<<setfill('0')<<setw(2)<<ss;
   //注意setfill()函数内的值是字符，要加单引号。
   return 0;
}
```

## 题目分析

根据题意模拟即可，注意开 long long。

设输入的毫秒数为 $n$，解题步骤如下：

首先，忽略不足 1 秒的毫秒数，即 `n/=1000`。

其次，计算出时、分、秒。其中，时可以用 `n%86400/3600` 来表示，分可以用 `n%3600/60` 来表示，秒可以用 `n%60` 来表示。这里的 86400 秒是一天 24 小时内所经过的秒数，3600 秒是 1 小时内所经过的秒数，60 秒是 1 分钟内所经过的秒数。

最后，按照题目要求输出时间，注意时、分、秒不足两位时要补前导 0。

## 参考代码

```cpp
#include <cstdio>
int main() {
	long long n;
	scanf("%lld", &n);
	n /= 1000;	//忽略不足1秒的毫秒数 
	printf("%02lld:%02lld:%02lld\n", n % 86400 / 3600, n % 3600 / 60, n % 60);
	return 0;
}
```

## 作者：Double_Light (赞：5)

一道模拟题，挺适合初学者的。

对于一个毫秒数 $n$，我们先忽略大于前面的年月日，让 $n$ 只表示从今天 00:00:00 到现在的毫秒数。

然后将 $n$ 除以 $1000$，让 $n$ 从毫秒数变成秒数。

最后输出，小时数是 ```n/3600```，分钟数是 ```n%3600/60```，秒数是 ```n%60```。不足两位的补足两位。

参考代码：

```cpp
#include<iostream>
using namespace std;
long long n;//开long long
int main(){
	cin>>n;
	n%=60*60*24*1000;//变成今天的毫秒数
	n/=1000;//变成今天的秒数
	if(n/3600<=9)cout<<"0";//判断补零，下面同理
   cout<<n/3600<<":";//输出，下面同理
   if(n%3600/60<=9)cout<<"0";
   cout<<n%3600/60<<":";
   if(n%60<=9)cout<<"0";
   cout<<n%60;
	return 0;
}
```

*[Problem](https://www.luogu.com.cn/problem/P8748)*

-----------
### 题意：

给出一个数，表示从 1970 年 1 月 1 日 00:00:00 到当前时刻经过的毫秒数，请你求出现在的时分秒，毫秒舍去。

------------
### 解题思路：

首先将毫秒舍去。 ```n /= 1000``` 

然后由于题目不要求显示日期，所以先求出当天的秒数。（用总秒数取模一天的秒数） ```n % (24 * 60 * 60)```

最后根据时分秒的转换公式进行计算即可。 ```1时 = 60分 = 3600秒```

**注意：**

> 时、分、秒不足两位时补前导 0 

输出还要补足位数，可以用 ```printf``` 输出，具体细节见代码。

------------
### code：

```cpp
#include <bits/stdc++.h>
using namespace std;

long long a;

int main()
{
	scanf("%lld", &a);
	int day_s = 24 * 60 * 60, hour_s = 60 * 60, min_s = 60;
	a /= 1000;
	int th = a % day_s / hour_s;
	int tm = a % day_s % hour_s / min_s;
	int ts = a % day_s % min_s;
	printf("%02d:%02d:%02d", th, tm, ts);
	return 0;
}

```

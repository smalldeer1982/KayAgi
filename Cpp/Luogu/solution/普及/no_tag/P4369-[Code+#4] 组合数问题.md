# [Code+#4] 组合数问题

## 题目描述

众所周知，小葱同学擅长计算，尤其擅长计算组合数，所以小葱给了你两个数 $x$ 和 $k$，希望你把 $x$ 分成恰好 $k$ 个不同的组合数的和。所谓不同，即对于两个组合数 $C(n_1,m_1)$ 和 $C(n_2,m_2)$，如果 $n_1\neq n_2$​​ 或者 $m_1\neq m_2$​，我们就称这两个组合数是不同的。为了使得计算不过于复杂，你需要保证你给出的任何一个组合数 $C(n,m)$ 满足 $0\leq m\leq n\leq x$。数据保证一定有解。

## 说明/提示

对于 $20\%$ 的数据，$k=1$。

对于另外 $20\%$ 的数据，$x\leq 100$。

对于另外 $20\%$ 的数据，$k=2$。

对于 $100\%$ 的数据，$1\leq x\leq 10^9,1\leq k\leq 10^3.$

Credit: https://www.luogu.org/discuss/show/38908

### 输入

```
6 2```

### 输出

```
3 1
3 2```

# 题目大意
将x写成k个组合数的和，要求：在组合数 $C _n ^m$中，$n\le x$。   
# 题目分析

我们先列出杨辉三角的前几行：   
```cpp
            1
          1   1
        1   2   1
      1   3   3   1
    1   4   6   4   1
  1   5   10 10   5   1
1   6  15  20  15   6   1
```

写到这里我们发现：其实任何一个数都是某个组合数：  
    $n=C_n^1=C_n^{n-1}$  
    
而且1还有多种表示形式：

$1=C_n^0=C_n^n$

那我们就可以把x写成下面的形式了：   
$x=\sum _{i=1} ^{k-1} 1+(x-k+1)$   
$\ \ \ =\sum _{i=1} ^{k-1} C_i^0+C_{x-k+1}^1$  
容易验证这样构造的$k$个组合数都满足下标限制且两两不同。证毕。

## 作者：EdisonBa (赞：6)

#### 题目分析：

题目大意为把 $x$ 分成 $k$ 个组合数 $C(n_i,m_i)$。

 $C(n_1,m_1)$ 和 $C(n_2,m_2)$

首先，我们要明确什么是组合数。

从 $n$ 个不同元素中每次取出 $m$ 个不同元素 $(0\leq m\leq n)$，不管其顺序合成一组，所有这样的组合的种数称为组合数。在线性写法中被写作 $C(n,m)$。

看到这个题，我们可以想到组合数的一些互补性质：$C(n,0)=1$ , $ C(n,n)=1 $ , $ C(n,1)=n$ 。

那么，我们可以先假设要把 $x$ 分成 $x$ 个组合数，这样每个组合数就等于 1 。想要让每个组合数不相等，就可以使 $x$ 被分成：$C(1,0)+C(2,0)+C(3,0)+~...~+C(x,0)$。

但是，题目中说让分成 $k$ 个组合数。我们可以先分配 $k-1$ 个组合数给 $C(i,0)$ ，这样，$x$ 这个数里没有被分配的值为 $x-k+1$ 。由$ C( n,1)=n$，我们可以把最后的一个组合数用 $C( x-k+1,1)$ 来表示。

综上，把 $x$ 分成 $k$ 个组合数可以分为：
$$
\sum ^{k-1}_{i=1}\tbinom{i}{0} + \tbinom{x-k+1}{1}
$$

#### 代码

```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
	int x,k;
	cin>>x>>k;
	for(int i=1;i<=k-1;++i)
	printf("%d 0\n",i,i);
	printf("%d 1",x-k+1);
	return 0;
}
```

### Problem.
给定一个$n$和$k$，求$k$个不同的组合数和为$n$。  
组合数不同的定义：$C_n^m$不同于$C_k^p$当且仅当$n\not=k$或$m\not=p$。

### PS.
笔者是个大彩笔，刚开始甚至没有看出答案。。。  
然后笔者在~~观摩~~看杨辉三角时突然发现边上全都是$1$，~~笔者大彻大悟~~。

### Solution.
首先我们发现，不同的定义不是答案不同而使$n\not=k$或$m\not=p$！  
所以$C_5^0\not=C_6^0$！  
所以我们可以构造出$k-1$个不同的$1$以及一个$n-k+1$，然后这些数共$k$个，和为$n$，证毕！QwQ  
顺便再做个简单的证明：$C_n^m>=1$，所以$k$个组合数的和$>=k$，所以只有$n>=k$时才会满足`数据保证一定有解。`

### Coding.
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int n,k;scanf("%d%d",&n,&k);
    for(int i=1;i<=k-1;i++) printf("%d 0\n",i);
    return printf("%d 1\n",n-k+1),0;
}
```

## 作者：我杀文化课 (赞：1)

题目大意：给定$x$，输出$k$对组合数，使得任意所有组合数$C(n_i,m_i)$之和为$x$且互不相同。不同的定义为：对于两组组合数$C(n_1,m_1)$和$C(n_2,m_2)$,$n_1\ne n_2$**或**$m_1\ne m_2$

首先看见这个题有多种答案且只是任意输出一种，我脑子里马上冒出了组合数的几个性质。

- $C(n,0)=1(n\in Z)$
- $C(n,n)=1(n\in Z)$
- $C(n,1)=n(n\in Z)$

那么对于任何一个$x,k$，我们只用输出$(1,0/1)(2,0/2)...(k-1,0/k-1)$以及$(x-k+1,1)$就完美的把这个题氵过去了。

## Code:
```cpp
#include<cstdio>
using namespace std;
int n,k;
int main()
{
	scanf("%d%d",&n,&k);
	for(int i=1;i<k;i++) printf("%d 0\n",i);
	printf("%d 1",n-k+1);
}
```

## 作者：songhongyi (赞：1)

题意：给定$n$和$k$，给定一种方案，把$n$拆成$k$个组合数（$C_a^b$）的和。  
我们注意到，$\forall t \in \mathbb{N}$，有$C_t^0=1.C_t^1=t$，那我们很容易就能想到一个构造思路，先给$k-1$个$C_t^0$，此时和为$k-1$，然后补一个$C_{n-k+1}^1=n-k+1$，和变为$n$，目标达成。  
代码如下：
```cpp
#include <iostream>
using namespace std;
int main()
{
    int n, k;
    cin >> n >> k;
    for ( int i = 1; i < k; i++ )
        cout << i << " " << 0 << endl; //k-1个1
    cout << n - k + 1 << " " << n-k << endl;// 最后补齐
    return 0;
}
```
题目评价：一道比较有趣的构造题。

# [SEERC 2020] Simple Hull

## 题目描述

Gary 创造了一个含有 $n$ 个点的多边形，每个点的坐标形如 $(x,y)$。

你需要进行一些连边，保证包含所有的点，同时连出来的图是一条“多边形链”（即任何一个点只能连向或被连向 $2$ 个点，且连出来的边可以在图中与其他边相交）。

你还需要保证每两个存在连边的点 $i$ 和 $j$，这两个点形成的边在图中是垂直或水平的。

输出连出来的图的最小面积。

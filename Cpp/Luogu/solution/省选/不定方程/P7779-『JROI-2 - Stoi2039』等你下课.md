## 题目描述

给定正整数 $d$，定义

$$S_v=\{(x,y):x,y \in \mathbb{Z_{\ge 0}},x^2-dy^2=v\}$$

求：

$$\sum_{v=1}^{\lfloor\sqrt{d}\rfloor}v[S_v \neq \varnothing]$$

多组询问。

## 说明/提示

#### 样例解释

- 对于 $d=10$，只有 $v=1$ 满足 $S_v \neq \varnothing$。
- 对于 $d=13$，有 $v=1,3$ 满足 $S_v \neq \varnothing$。
- 对于 $d=16$，有 $v=1,4$ 满足 $S_v \neq \varnothing$。
- 对于 $d=19$，有 $v=1,4$ 满足 $S_v \neq \varnothing$。

#### 数据范围

- 对于 $30\%$ 的数据，$t=1$，$1 \le d \le 70$。
- 对于 $60\%$ 的数据，$t=1$。
- 对于 $100\%$ 的数据，$1 \le t \le 4 \times 10^3$，$1 \le d \le 2 \times 10^6$。

# 题解

## 作者：灰鹤在此 (赞：8)

本题解是基于 [VinstaG173](https://www.luogu.com.cn/user/59388) 的[题解](https://59388.blog.luogu.org/solution-p7779)所做的一些补充和改进。

题目要求的是所有 $x^2-d\cdot y^2(x,y\in N)$ 的值 $v$ 的和。且 $v\in[1,\sqrt{d}]$。

我们设 $k=\sqrt{d}$，于是我们可以得到 $k\ge (x+k\cdot y)(x-k\cdot y)\ge1$ 这样一条恒不等式。但是我们要求的是正整数解，那么这个问题就变成了佩尔方程的变形。

- 因为当 $k\in Z_{\ge1}$ 的时候，即 $d$ 为平方数的时候，这个不等式非常好解。因为当 $y\ge 1$ 的时候，$x+k\cdot y\ge1$，所以 $x-k\cdot y\ge1$。

于是我们得到 $x\ge k\cdot y+1$，代回得 $x+k\cdot y\ge 2k\cdot y+1$，而因为 $k\ge (x+k\cdot y)(x-k\cdot y)$ 且 $x-k\cdot y\ge1$，这个不等式无解，所以 $y=0$。

当 $y=0$ 的时候，只要 $v$ 是在限制范围内的平方数，那么一定有对应的 $x$ 可以使原方程成立，所以当 $d$ 为平方数的时候，我们所求的答案为

$$\sum_{i=1}^{\sqrt{k}}i\times i$$

- 当 $k$ 是无理数的时候，我们可以用连分数来解决这个问题。

先来介绍一下 $Pell$ 方程：

形如 $x^2-d\cdot y^2=1$ 的方程。

假设 $(x_0,y_0)$ 是它的最小正整数解，那么这个方程的所有正整数解可写为：

$x_n=\frac{(x_1+\sqrt d\cdot y_1)^n+(x_1-\sqrt d\cdot y_1)^n}{2}$

$y_n=\frac{(x_1+\sqrt d\cdot y_1)^n+(x_1-\sqrt d\cdot y_1)^n}{(2\sqrt d)}$

所以 $x_n+\sqrt d\cdot y_n=(x_0+\sqrt d\cdot y_0)^{n+1}$。

且 $x_n$ 和 $y_n$ 之间存在线性关系。

$x_n=2\cdot x_1\cdot x_{n-1}-x_{n-2}$

$y_n=2\cdot y_1\cdot y_{n-1}-y_{n-2}$

况且，我们还可以用连分数来求解 $Pell$ 方程。

我们现在要处理的是 $Pell$ 方程的变形。

因为 $(x+k\cdot y)(x-k\cdot y)=v$，如果当 $\frac{x}{y}$ 等于 $k$ 的一个渐进分数，那么 $v$ 就是我们再求 $k$ 的渐进分数中的过渡数 $c$。

接下来就是如何求二次无理数的连分数和过渡数：

请阅读[连分数](https://www.luogu.com.cn/blog/huihe/lian-fen-shuo)这篇博客里面不失精度的求无限连分数的方法。

我们要求的是所有过渡数的和，而且每一种数我们只能计算一次。因为我们求得的过渡数都是正的，但是我们不能保证 $x$ 和 $y$ 都是正的，因为 $(-1)^n\cdot c_n=p_{n-1}^2-d\cdot q_{n-1}^2$，所以我们要先把所有偶数次求的过渡数先加上（因为肯定是正数），然后再根据我们求得的连分数的循环节的奇偶性来判断我们后面是否需要把奇数次求得的过渡数再加上。

因为如果循环节是奇数，那么它下一次循环的时候，奇数次位置上的数会跑到偶数次位置上来，所以如果循环节是奇数，那么所有奇数次上的数都要再加一遍。

代码

```cpp
#include<bits/stdc++.h>
using namespace std;
const int MaxN=2e6+5;
const int MaxD=1e4+5;
const int MaxK=1414;
int T,d,vis[MaxN],state[MaxK+5],a[MaxD],c[MaxD],k[MaxD],cnt,ans,sk,ssk;
int main(){
	for(register int i=1;i<=MaxK;i++){
		vis[i*i]=1;
	}
	scanf("%d",&T);
	while(T--){
		scanf("%d",&d);
		sk=sqrt(d);
		ssk=sqrt(sk);
		cnt=ans=0;
		for(register int i=1;i<=ssk;i++){
			ans+=i*i;
			state[i*i]=1; 
		}
		if(!vis[d]){
			a[0]=sk;
			c[0]=1;
			k[0]=0;
			while(a[cnt]!=sk*2){
				k[cnt+1]=a[cnt]*c[cnt]-k[cnt];
				c[cnt+1]=(d-k[cnt+1]*k[cnt+1])/c[cnt];
				a[cnt+1]=(sk+k[cnt+1])/c[cnt+1];
				cnt++;
				if(!(cnt&1)){
					if(c[cnt]<=sk){
						if(!state[c[cnt]]){
							ans+=c[cnt];
							state[c[cnt]]=1;
						}
					}
				}
			}
			if(cnt&1){
				for(register int i=1;i<=cnt;i+=2){
					if(c[i]<=sk){
						if(!state[c[i]]){
							ans+=c[i];
							state[c[i]]=1;
						}	
					}
				}
			}
			for(register int i=1;i<=sk;i++){
				state[i]=0;
			}
		}
		printf("%d\n",ans);
	}
	return 0;
}
```

## 作者：VinstaG173 (赞：6)

首先 $d$ 为平方数时，设 $d=k^2$。则问题即

$$\sum_{t=1}^{k}[S_t \neq \varnothing].$$

由于 $x^2-dy^2=(x+ky)(x-ky)>0$，当 $y>0$ 时有 $x-ky \ge 1$，$x+ky>k$，故 $x^2-dy^2>k$。因此 $S_t$ 中的元素只可能满足 $y=0$。故答案为所有 $[1,k]$ 中的完全平方数之和，容易求出。

下面考虑 $d$ 不为平方数的情况。

根据连分数理论，有如下一些定义和定理（证明可参见潘承洞、潘承彪《初等数论》，我也在写证明但没写完）。为方便起见，以下 $a_n$ 均为正整数。

下面称无限连分数为形如

$$x=a_0+\dfrac{1}{a_1+\dfrac{1}{a_2+\dots}}$$

的分数，可表示为

$$x=\left<a_0,a_1,a_2,\dots\right>.$$

称一个无限连分数的第 $n$ 个渐近分数为

$$x_n=\left<a_0,a_1,a_2,\dots,a_n\right>,$$

另记

$$y_n=\left<a_n,a_{n+1},a_{n+2},\dots\right>.$$

称 $x$ 是循环的，当存在 $m \in \mathbb{Z_{\ge 0}},l \in \mathbb{Z_+}$，有 $\forall i \ge m$，$a_i=a_{i+l}$。称 $l$ 为其循环节，最小的满足条件的 $l$ 称为其最小循环节，以下为方便简称最小循环节为循环节。

循环连分数可记为

$$x=\left<a_0,a_1,\dots,a_{m-1},\overline{a_m,\dots,a_{m+l-1}}\right>,$$

含义如上所述。

**定理 $1.$** 将 $\sqrt{d}$ 表示成无限连分数之后必为循环连分数，且 $m=1$ 为最小循环开始点，即形如

$$\sqrt{d}=\left<a_0,\overline{a_1,\dots,a_l}\right>,$$

同时有 $a_l=2a_0$。

**定理 $2.$** 当 $t \le \lfloor\sqrt{d}\rfloor$ 时，有 $x^2-dy^2=t$ 的正整数解 $(x,y)$ 满足 $\dfrac{x}{y}$ 是 $\sqrt{d}$ 表示成无限连分数后的一个渐近分数 $x_n$。

**定理 $3.$** 每个 $y_n$ 均可以表示成形如

$$y_n=\dfrac{\sqrt{d}+c_n}{t_n}$$

的形式，其中 $c_n \in \mathbb{Z_{\ge 0}},t_n \in \mathbb{Z_+}$，且满足 $t_n|d-c_n^2$。

**定理 $4.$** 设 $x_n=\dfrac{p_n}{q_n}$ 为其最简分数表示，则有

$$(-1)^nt_n=p_{n-1}^2-dq_{n-1}^2.$$

另外有一些显然的事实：$c_n,t_n$ 随 $a_n$ 循环而循环；$x_n$ 必是有理数。

同时，在定理 $3$ 中还可以增加一点：

$$c_{n+1}=a_nt_n-c_n,t_{n+1}=\dfrac{d-c_{n+1}^2}{t_n}.$$

这样我们就得到了 $O(l)$ 递推出所有 $t_n$ 的方法，再视 $l$ 的奇偶性把出现在负位置上的那些数排掉就行了。

而 $l$ 是 $O(\sqrt{d})$ 的。故总时间复杂度 $O(\sqrt{d})$。

同时注意，在这种情况下 $\lfloor\sqrt{d}\rfloor$ 以内的所有平方数也是满足条件的，也需要计入答案中。

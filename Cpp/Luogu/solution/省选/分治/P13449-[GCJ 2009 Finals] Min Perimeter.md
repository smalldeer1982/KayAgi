# [GCJ 2009 Finals] Min Perimeter

## 题目描述

你将得到一组整数坐标的点集。你的任务是计算，从这些点中选取三个互不相同的点作为顶点，能够构成的三角形的最小周长。


## 说明/提示

**限制条件**

- $1 \leq T \leq 15$
- $0 \leq x_i, y_i \leq 10^9$

**小数据集（5 分）**

- 时间限制：15 秒
- $3 \leq n \leq 10000$

**大数据集（15 分）**

- 时间限制：30 秒
- $3 \leq n \leq 1000000$

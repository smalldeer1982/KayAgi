# 题目信息

# [NOIP 2007 普及组] 奖学金

## 题目背景

NOIP2007 普及组 T1

## 题目描述

某小学最近得到了一笔赞助，打算拿出其中一部分为学习成绩优秀的前 $5$ 名学生发奖学金。期末，每个学生都有 $3$ 门课的成绩：语文、数学、英语。先按总分从高到低排序，如果两个同学总分相同，再按语文成绩从高到低排序，如果两个同学总分和语文成绩都相同，那么规定学号小的同学排在前面，这样，每个学生的排序是唯一确定的。

任务：先根据输入的 $3$ 门课的成绩计算总分，然后按上述规则排序，最后按排名顺序输出前五名名学生的学号和总分。

注意，在前 $5$ 名同学中，每个人的奖学金都不相同，因此，你必须严格按上述规则排序。例如，在某个正确答案中，如果前两行的输出数据（每行输出两个数：学号、总分) 是：

```plain
7 279  
5 279
```

这两行数据的含义是：总分最高的两个同学的学号依次是 $7$ 号、$5$ 号。这两名同学的总分都是 $279$ (总分等于输入的语文、数学、英语三科成绩之和) ，但学号为 $7$ 的学生语文成绩更高一些。

如果你的前两名的输出数据是：

```plain
5 279  
7 279
```

则按输出错误处理，不能得分。

# AI分析结果



### 算法分类
排序

### 题解分析与结论
所有题解均围绕结构体排序展开，核心差异集中在比较函数实现方式与代码简洁度。关键要点：
1. **多级排序条件**：总分降序 → 语文降序 → 学号升序
2. **结构体设计**：只需存储总分、语文、学号（数学英语无需参与排序）
3. **比较函数优化**：利用短路逻辑简化多条件判断

### 高星题解推荐
1. **codesonic（5星）**  
   - 亮点：步骤拆解清晰，注释详尽，适合新手理解多级排序逻辑  
   - 代码片段：
     ```cpp
     bool cmp(stu a,stu b) {
         if(a.sum != b.sum) return a.sum > b.sum;
         if(a.c != b.c) return a.c > b.c;
         return a.num < b.num;
     }
     ```

2. **wpy233（4星）**  
   - 亮点：利用条件表达式简化比较逻辑，代码紧凑高效  
   - 代码片段：
     ```cpp
     bool comp(xx x,xx y) {
         if(x.zf != y.zf) return x.zf > y.zf;
         if(x.yw != y.yw) return x.yw > y.yw;
         return x.bh < y.bh;
     }
     ```

3. **张鑫杰（4星）**  
   - 亮点：面向对象实现，重载运算符展示结构体排序的另一种范式  
   - 关键代码：
     ```cpp
     bool operator<(const student& right) const {
         if(sum > right.sum) return true;
         if(sum == right.sum && chinese > right.chinese) return true;
         return (sum == right.sum && chinese == right.chinese && id < right.id);
     }
     ```

### 关键技巧总结
- **结构体精简**：仅存储必要字段（总分、语文、学号）
- **比较函数短路**：`if(a.sum != b.sum) return ...` 提前终止判断
- **稳定排序**：使用 `stable_sort` 可优化相同元素的原始顺序保持（但本题无需）

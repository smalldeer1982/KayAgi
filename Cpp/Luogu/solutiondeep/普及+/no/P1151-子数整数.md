# 题目信息

# 子数整数

## 题目描述

对于一个五位数 $\overline{a_1a_2a_3a_4a_5}$，可将其拆分为三个子数：

$sub_1=\overline{a_1a_2a_3}$

$sub_2=\overline{a_2a_3a_4}$

$sub_3=\overline{a_3a_4a_5}$

例如，五位数 $20207$ 可以拆分成

$sub_1=202$

$sub_2=020\ (=20)$

$sub_3=207$

现在给定一个正整数 $K$，要求你编程求出 $10000$ 到 $30000$ 之间所有满足下述条件的五位数，条件是这些五位数的三个子数 $sub_1,sub_2,sub_3$ 都可被 $K$ 整除。


## 说明/提示

$0<K<1000$


## 样例 #1

### 输入

```
15```

### 输出

```
22555
25555
28555
30000```

### 题解分析与结论
题目核心是暴力枚举所有五位数并验证三个子数的整除性。题解均采用枚举法，差异主要体现在子数提取方式和循环优化策略。最优思路通过分步生成子数减少无效枚举，如先确定第一个子数后再推导后续子数。需注意处理前导零（如020视为20）和特判边界值（如30000）。

### 精选题解
1. **Dilute (4星)**  
   **关键亮点**：分步生成子数+循环优化  
   **代码思想**：先找到第一个能被K整除的三位数，逐步生成后续子数，避免全范围遍历。特判30000处理边界情况。  
   ```cpp
   int i = 100; // 前三位起始值
   while(i % x != 0) i++; // 找到首个可被K整除的sub1
   for( ; i < 300; i += x){ // 按步长K枚举sub1
       qe = (i % 100) * 10; // 生成sub2前两位
       for(int j = 0; j < 10; j++){ // 枚举第四位
           if((qe + j) % x != 0) continue;
           es = (qe % 100 + j) * 10; // 生成sub3前两位
           for(int u = 0; u < 10; u++){ // 枚举第五位
               if((es + u) % x != 0) continue;
               printf("%d%d%d\n", i, j, u); // 构造完整五位数
   }}}
   ```

2. **疯的、风 (4星)**  
   **关键亮点**：极简数学运算提取子数  
   **代码思想**：通过除法和取模快速计算三个子数，代码简洁高效。  
   ```cpp
   if(i/100%k==0)             // sub1 = 前三位
   if((i/10 - i/10000*1000)%k==0) // sub2 = 中间三位
   if((i - i/1000*1000)%k==0)    // sub3 = 后三位
   ```

3. **5ab_juruo (4星)**  
   **关键亮点**：函数封装+逻辑清晰  
   **代码思想**：将子数计算封装为函数，提升可读性，且使用逻辑表达式简化判断。  
   ```cpp
   int sub(int m, int n) {
       switch(n) { // 按索引返回子数
           case 1: return m/100; case 2: return m/10%1000; case 3: return m%1000;
       }
   }
   if(!(sub(i,1)%k) && !(sub(i,2)%k) && !(sub(i,3)%k)) // 判断整除
   ```

### 优化技巧总结
1. **分步剪枝**：先确定sub1的有效范围，再生成sub2和sub3，减少无效枚举（Dilute）。
2. **数学运算优化**：用除法和取模快速提取子数，避免字符串转换开销。
3. **边界特判**：单独处理30000等边界值，避免循环遗漏。

昴出了一身冷汗，战战兢兢地问道：“那么问题是？”

魔女拾起了身边的一个棋子，缓缓说道：“如你所见，这是一张 $n\times n$ 的棋盘，由 $n^2$ 个方格组成，每个方格上最多只能存在一个棋子。**不同于普通的棋盘，这个棋盘是循环的，也就是说，第 $i$ 行 $n$ 列的格子右边是第 $i$ 行 $1$ 列的格子，第 $n$ 行 $i$ 列的格子下面是第 $1$ 行 $i$ 列的格子，以此类推。** 我手中的棋子名叫「大罪司教」，它一步可以向任意斜线方向移动任意多格。一个棋子能一步走到的格子称为其掌控的格子。如果一个棋子掌控了另一个棋子所在的方格，则这种棋盘布局是不稳定的，反之称其为稳定的。现在，请问你能否向棋盘中放入恰好 $m$ 个「大罪司教」，使得整个棋盘布局是稳定的呢？”

## 说明/提示

样例一：将一个棋子放入 $1\times1$ 的棋盘中显然只有一种方案。

样例二：一共有以下四种稳定的布局（用 `.` 代表空位，`o` 代表「大罪司教」）：

```plain
oo .o .. o.
.. .o oo o.
```

### 综合分析与结论
本题的核心在于如何计算在循环棋盘上放置“大罪司教”棋子的稳定布局方案数。题解主要围绕组合数学展开，尤其是对于奇数棋盘和偶数棋盘的不同处理方式。奇数棋盘可以转化为经典的车布局问题，而偶数棋盘则需要将棋盘分为黑白两部分分别处理。各题解在思路清晰度和代码实现上有所不同，但整体上都基于组合数学的基本原理。

// 核心代码片段
int main() {
    int n, m;
    cin >> n >> m;
    if (n % 2 == 1) {
        // 奇数棋盘的处理
        cout << comb(n, m) * comb(n, m) * factorial(m) % MOD;
    } else {
        // 偶数棋盘的处理
        int half = n / 2;
        long long ans = 0;
        for (int i = 0; i <= m; ++i) {
            ans += comb(half, i) * comb(half, i) * factorial(i) % MOD * comb(half, m - i) * comb(half, m - i) * factorial(m - i) % MOD;
        }
        cout << ans % MOD;
    }
    return 0;
}
```

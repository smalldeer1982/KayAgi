# 题目信息

# 删数问题

## 题目描述

键盘输入一个高精度的正整数 $n$（不超过 $250$ 位），去掉其中任意 $k$ 个数字后剩下的数字按原左右次序将组成一个新的非负整数。编程对给定的 $n$ 和 $k$，寻找一种方案使得剩下的数字组成的新数最小。

## 说明/提示

用 $\operatorname{len}(n)$ 表示 $n$ 的**位数**，保证 $1 \leq k < \operatorname{len}(n) \leq 250$。

## 样例 #1

### 输入

```
175438 
4
```

### 输出

```
13```

# AI分析结果



### 综合分析与结论
本题核心为贪心策略：通过局部最优（高位尽可能小）实现全局最优解。关键点在于每次选择可删除范围内的最小数，并处理前导零与全零等边界情况。高质量题解需清晰体现贪心思路与边界处理。

### 高星题解推荐（评分≥4星）

#### 1. 作者：expin (⭐⭐⭐⭐⭐)
**关键亮点**  
- **贪心策略**：每次在允许范围内（前 `m+1` 位）选最左侧最小数作为当前位，保证高位最小化  
- **高效处理**：直接输出中间结果，避免存储完整字符串  
- **边界处理**：通过 `flag` 标记处理前导零，结尾特判全零情况  
**核心代码**  
```cpp
while (cnt < rest) {
    minp = t;
    for (int i = t; i <= k + t; ++i) // 在可删范围内找最小数
        if (a[minp] > a[i]) minp = i;
    if (a[minp]) flag = 1; // 标记非零开始
    if (flag) cout << a[minp];
    k -= minp - t; // 更新剩余删除次数
    t = minp + 1;  // 移动起点
    cnt++;
}
if (!flag) cout << 0; // 全零特判
```

#### 2. 作者：xh39 (⭐⭐⭐⭐)
**关键亮点**  
- **峰删法**：循环删除第一个递减区间的首字符（即"山峰"）  
- **简洁实现**：利用 `erase` 函数直接操作字符串，逻辑简明  
- **前导零处理**：循环删除开头的零直到有效位  
**核心代码**  
```cpp
while (s) {
    for (i = 0; n[i] <= n[i + 1];) // 找第一个递减点
        i++;
    n.erase(i, 1); // 删除该位
    s--;
}
while (n[0] == '0' && n.size() > 1) // 去前导零
    n.erase(0, 1);
```

#### 3. 作者：KesdiaelKen (⭐⭐⭐⭐)
**关键亮点**  
- **逆向思维**：将删数转化为选数，每次在允许范围内选最小高位  
- **精准定位**：通过动态计算可选范围起始点，避免重复遍历  
- **零处理优化**：在选数阶段跳过前导零，减少后期处理  
**核心代码**  
```cpp
while (bl - js) { // 需要选bl位数
    minn = 2e9;
    for (int i = sgw + 1; i <= len - bl + js; i++)
        if (shu[i] < minn) minn = shu[sgw = i]; // 找最小数
    bln[js++] = shu[sgw]; // 记录结果
}
while (bln[start] == 0 && bl - start - 1) start++; // 去前导零
```

### 关键思路总结
1. **贪心选择**：高位优先选择最小数，允许范围随删除次数动态调整  
2. **数据结构优化**：使用字符串或数组直接操作，结合双指针减少遍历次数  
3. **边界处理**：特判全零、递增序列删除末尾、前导零的多阶段处理

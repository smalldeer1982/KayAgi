# 题目信息

# 「DTOI-5」校门外的枯树

## 题目背景

某天放学，你走出了校门，发现校门外又双叒叕出现了一排树。只不过因为正值寒冬时节，树的叶子都掉光了，树们在寒风中瑟瑟发抖，让人担心它们会不会在某一时刻失去平衡，然后倒下来。

## 题目描述

给你校门外的一排 $T$ 棵无向有根树（每棵树的根节点编号均为 $1$），每棵树的每条边有其重量 $m_i$，现在请你算出每棵树的不平衡度 **或** 该树的**每个节点的子树**的不平衡度，好让校方帮忙加固。$\color{white}\sout{\text{不要问我为什么重量的字母是 }m\text。}$

注意这里的**树的边是有顺序的**，~~你总不可能把树枝掰断然后嫁接到另一个地方吧，这可是枯树啊喂~~。

---

对于一颗有根树，定义其不平衡度为该树被在根节点与某一叶子节点的一条最短路径分割为左右两部分（两个边集）后（两个边集均不含该最短路径中的边）两部分的边的总重之差的最小值。特别地，**单个点作为树的不平衡度为 $0$**；空边集内边的总重为 $0$。

### 综合分析与结论

本题的核心在于计算树的不平衡度，即通过选择一条从根到叶子的路径，将树分割为左右两部分，并计算这两部分边权总和的差值的最小值。题解中主要采用了DFS遍历、前缀和、二分查找、启发式合并等技巧来优化计算过程。

- **y_kx_b** 的题解通过DFS预处理每个叶子节点的左右边权和，并利用二分查找来快速找到最小差值，思路清晰且代码简洁。
- **_EEA_** 的题解通过维护每个节点的左右边权和，并利用二分查找来优化计算，思路清晰且复杂度较低。
- **_edge_** 的题解通过启发式合并和懒标记来维护每个节点的左右边权和，虽然复杂度较高，但思路新颖且代码实现较为复杂。

### 所选高星题解

#### 1. y_kx_b (★★★★★)
- **关键亮点**：通过DFS预处理每个叶子节点的左右边权和，并利用二分查找来快速找到最小差值，思路清晰且代码简洁。
- **核心代码**：
```cpp
int wsum = 0, w1 = 0;
int ans = 0x7f7f7f7f;
void dfs(int u, int w2) {
    if(head[u] == -1) {
        ans = min(ans, abs((wsum - w2 - w1) - w1));
        return;
    }
    for(int i = head[u]; ~i; i = ne[i]) {
        int &v = to[i];
        dfs(v, w2 + w[i]);
        w1 += w[i];
    }
}
```
- **个人心得**：作者提到链式前向星的遍历顺序可能是一个坑点，但在这题中并不影响结果。

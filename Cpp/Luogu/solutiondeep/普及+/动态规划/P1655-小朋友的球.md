# 小朋友的球

## 题目描述

@发源于 小朋友最近特别喜欢球。有一天他脑子抽了，从口袋里拿出了 $N$ 个不同的球，想把它们放到 $M$ 个相同的盒子里，并且要求每个盒子中至少要有一个球，他好奇有几种放法，于是尝试编程实现，但由于他天天不好好学习，只会上 B 站看游泳教练，于是他向你求助。

## 说明/提示

- 对于 $20\%$ 的数据，满足 $N,M \leq 10$；
- 对于 $100\%$ 的数据，满足 $1 \leq N,M \leq 100$，一个测试文件最多有 $10$ 组测试数据。


## 样例 #1

### 输入

```
4 2
1 1```

### 输出

```
7
1```

# AI分析结果



### 综合分析与结论
题目本质是求第二类斯特林数 $S(n,m)$，核心解法为动态规划结合高精度计算。所有题解均基于递推公式 $f[i][j] = f[i-1][j-1] + j \times f[i-1][j]$，区别在于高精度的实现方式。C++题解通过自定义高精度结构体或字符串操作处理大数，Python则利用语言特性简化计算。

**核心代码思想**  
```python
S = [[0] * MAXN for _ in range(MAXN)]
for i in range(1, MAXN):
    S[i][i] = S[i][1] = 1
    for j in range(2, i):
        S[i][j] = S[i-1][j-1] + j * S[i-1][j]
```

### 最优关键思路与技巧
1. **斯特林数递推公式**：$S(n,m) = S(n-1,m-1) + m \times S(n-1,m)$，体现分情况讨论思想（新元素单独成盒/加入已有盒）。
2. **高精度设计模式**：通过类/结构体重载运算符，使高精度运算与原生类型无缝衔接，提升代码复用性。
3. **预处理优化**：在输入前预先计算所有可能的 $n,m$ 值，适用于多组测试数据场景。

# 工程规划

## 题目描述

造一幢大楼是一项艰巨的工程，它是由 $n$ 个子任务构成的，给它们分别编号 $1,2,\cdots,n\ (5≤n≤1000)$。由于对一些任务的起始条件有着严格的限制，所以每个任务的起始时间 $T_1,T_2,\cdots,T_n$ 并不是很容易确定的(但这些起始时间都是非负整数，因为它们必须在整个工程开始后启动)。例如：挖掘完成后，紧接着就要打地基；但是混凝土浇筑完成后，却要等待一段时间再去掉模板。

这种要求就可以用 $m\ (5≤m≤5000)$ 个不等式表示，不等式形如 $T_i-T_j≤b$ 代表 $i$ 和 $j$ 的起始时间必须满足的条件。每个不等式的右边都是一个常数 $b$，这些常数可能不相同，但是它们都在区间 $(-100,100)$ 内。

你的任务就是写一个程序，给定像上面那样的不等式，找出一种可能的起始时间序列 $T_1,T_2,\cdots,T_n$，或者判断问题无解。对于有解的情况，要使最早进行的那个任务和整个工程的起始时间相同，也就是说，$T_1,T_2,\cdots,T_n$ 中至少有一个为 $0$。

# AI分析结果

### 分析与结论
本题要求根据给定的 $n$ 个子任务和 $m$ 个形如 $T_i - T_j \leq b$ 的不等式，找出一种可能的起始时间序列 $T_1, T_2, \cdots, T_n$，要求起始时间为非负整数且至少有一个为 0，若问题无解则输出 `NO SOLUTION`。

### 通用建议与扩展思路
这些题解大多采用差分约束算法，将不等式转化为图的边，通过最短路算法求解。关键思路是将不等式 $T_i - T_j \leq b$ 转化为 $T_i \leq T_j + b$，然后从 $j$ 向 $i$ 连一条边权为 $b$ 的有向边，利用最短路算法（如 SPFA、Bellman - Ford）求解。同时要注意图不连通和存在负环的情况，图不连通可通过建超级源点解决，存在负环则表示无解。

对于同类型题，差分约束系统可解决满足一系列形如 $x_i - x_j \leq k$ 约束条件的问题，求 $x_s - x_t$ 的最大值。若不等式为 $x_i - x_j \geq k$，则可转化为 $x_j - x_i \leq -k$ 后求最长路；若条件中有 $x_i = x_j$，可转化为 $x_i - x_j \leq 0$ 且 $x_i - x_j \geq 0$。

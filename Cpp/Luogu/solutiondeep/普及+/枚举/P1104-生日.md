# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

### 重点代码
```cpp
struct node//结构体排序
{
    string s;//名字
    int n,y,r,num;//年，月，日，输入的编号。
}a[110];
bool cmp(node a,node b)
{
    if(a.n<b.n)return 1;//先比年
    if(a.n>b.n)return 0;
    if(a.n==b.n)
    {
        if(a.y<b.y)return 1;//再比月
        if(a.y>b.y)return 0;
        if(a.y==b.y)
        {
            if(a.r<b.r)return 1;//再比日
            if(a.r>b.r)return 0;
            if(a.r==b.r)
            {
                if(a.num>b.num)return 1;//最后比编号
                else return 0;
            }
        }
    }
}
```
**核心实现思想**：定义结构体`node`存储学生信息，自定义比较函数`cmp`，按照年、月、日、编号的顺序进行比较，保证年龄大的学生排在前面，生日相同的情况下输入靠后的学生排在前面。

- **自定义比较函数**：通过自定义比较函数，按照年、月、日、编号的优先级进行比较，实现按年龄从大到小的排序。

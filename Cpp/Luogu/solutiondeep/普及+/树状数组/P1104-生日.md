# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

### 重点代码
```cpp
struct node//结构体排序
{
    string s;//名字
    int n,y,r,num;//年，月，日，输入的编号。
}a[110];
bool cmp(node a,node b)
{
    if(a.n<b.n)return 1;//先比年
    if(a.n>b.n)return 0;
    if(a.n==b.n)
    {
        if(a.y<b.y)return 1;//再比月
        if(a.y>b.y)return 0;
        if(a.y==b.y)
        {
            if(a.r<b.r)return 1;//再比日
            if(a.r>b.r)return 0;
            if(a.r==b.r)
            {
                if(a.num>b.num)return 1;//最后比编号
                else return 0;
            }
        }
    }
}
```
**核心实现思想**：定义结构体`node`存储学生信息，通过`cmp`函数实现比较逻辑，先比较年，再比较月，接着比较日，最后比较输入编号，以确定学生的排序顺序。

### 最优关键思路或技巧
- **结构体的使用**：将学生的姓名、年、月、日和输入编号等信息封装在结构体中，方便统一管理和操作。
- **自定义比较函数**：根据题目要求，自定义比较函数`cmp`，明确排序的优先级，确保排序结果符合要求。
- **排序算法的选择**：优先使用 STL 中的`sort`或`stable_sort`函数，代码简洁且效率较高。

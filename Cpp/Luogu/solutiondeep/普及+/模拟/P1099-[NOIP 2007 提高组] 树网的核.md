# 题目信息

# [NOIP 2007 提高组] 树网的核

## 题目描述

设 $T=(V,E,W)$ 是一个无圈且连通的无向图（也称为无根树），每条边都有正整数的权，我们称 $T$ 为树网（`treenetwork`），其中 $V$，$E$ 分别表示结点与边的集合，$W$ 表示各边长度的集合，并设 $T$ 有 $n$ 个结点。

路径：树网中任何两结点 $a$，$b$ 都存在唯一的一条简单路径，用 $d(a, b)$ 表示以 $a, b$ 为端点的路径的长度，它是该路径上各边长度之和。我们称 
$d(a, b)$ 为 $a, b$ 两结点间的距离。

$D(v, P)=\min\{d(v, u)\}$, $u$ 为路径 $P$ 上的结点。

树网的直径：树网中最长的路径成为树网的直径。对于给定的树网 $T$，直径不一定是唯一的，但可以证明：各直径的中点（不一定恰好是某个结点，可能在某条边的内部）是唯一的，我们称该点为树网的中心。

偏心距 $\mathrm{ECC}(F)$：树网 $T$ 中距路径 $F$ 最远的结点到路径 $F$ 的距离，即

$$\mathrm{ECC}(F)=\max\{D(v, F),v \in V\}$$

任务：对于给定的树网 $T=(V, E, W)$ 和非负整数 $s$，求一个路径 $F$，他是某直径上的一段路径（该路径两端均为树网中的结点），其长度不超过 $s$（可以等于 $s$），使偏心距 $\mathrm{ECC}(F)$ 最小。我们称这个路径为树网 $T=(V, E, W)$ 的核（`Core`）。必要时，$F$ 可以退化为某个结点。一般来说，在上述定义下，核不一定只有一个，但最小偏心距是唯一的。

下面的图给出了树网的一个实例。图中，$A-B$ 与 $A-C$ 是两条直径，长度均为 $20$。点 $W$ 是树网的中心，$EF$ 边的长度为 $5$。如果指定 $s=11$，则树网的核为路径`DEFG`（也可以取为路径`DEF`），偏心距为 $8$。如果指定 $s=0$（或 $s=1$、$s=2$），则树网的核为结点 $F$，偏心距为 $12$。

### 综合分析与结论
这些题解主要围绕如何在给定的无根树中找到满足长度限制且偏心距最小的路径。各题解思路和算法如下：
1. **StudyingFather**：先证明树直径相关性质，如多直径必交于一点、直径中点唯一等。提出四种解法，枚举法$O(n^3)$，双指针优化枚举$O(n^2)$，二分法$O(n \log \sum w)$，双指针 + 前缀和优化到$O(n)$。通过分析偏心距组成，利用双指针动态更新相关值，结合单调队列和前缀和优化计算。
2. **Mosklia**：指出直径上点到自身最远点是直径端点，分析路径偏心距影响因素。用邻接表存图，两次DFS求直径，再一次DFS预处理点的贡献值。通过单调队列优化枚举路径端点计算答案，时间复杂度$O(n)$。
3. **天泽龟**：对偏心距二分答案，利用树上点到直径端点距离最大的性质，以直径端点向内部递推找核端点，判断核长度是否满足条件。考虑支链影响确定二分边界，通过LCA求最长支链确定下界，时间复杂度$O(n \log n)$。
4. **qhr2023**：不找直径直接枚举路径，用Floyd求任意两点距离，通过推导公式计算点到路径距离，进而求偏心距，时间复杂度$O(n^3)$。
5. **JXR_Kalcium**：与qhr2023思路类似，用Floyd求距离，推导点到路径距离公式求偏心距，时间复杂度$O(n^3)$。
6. **tangzirui1016**：枚举路径端点，通过记录节点深度和父节点确定路径上的点，搜索求路径的ECC最大值，时间复杂度$O(n^3)$。

综合来看，StudyingFather和Mosklia的题解质量较高，思路清晰且优化程度较好。

```cpp
namespace sub4 {
void solve() {
  for (int i = 1; i <= cnt; i++) vis[dia[i]] = true;
  int maxd = 0;
  for (int i = 1; i <= cnt; i++) {
    dep[dia[i]] = 0, c = 0;
    dfs(dia[i], 0);
    maxd = max(dep[c], maxd);
  }
  int l = 1, r = 1;
  int minecc = 1 << 30;
  for (; l <= cnt; l++) {
    while (r <= cnt && pres[r + 1] - pres[l] <= s) r++;
    minecc = min(max(maxd, max(pres[l], posts[r])), minecc);
  }
  cout << minecc << endl;
}
} 
```
    - **核心实现思想**：先通过DFS求出直径上各点引出最长支链的最大值`maxd`，然后利用双指针`l`和`r`在直径上滑动窗口，结合前缀和`pres`和`posts`动态更新偏心距，取最小值。
2. **Mosklia（4星）**
    - **关键亮点**：清晰分析直径性质及路径偏心距影响因素，利用单调队列优化枚举路径端点计算答案，时间复杂度最优。
    - **核心代码（单调队列部分）**：
```cpp
std::deque<int> q; 
int ans = 2147483647; 
for(int i = 1, j = 1; i <= m; dist2 -= pt[del[i]].fa_dist, ++i) {
	pt[del[i]].cur_dist = dist2; 
	while(!q.empty() && pt[q.front()].cur_dist - s > pt[del[i]].cur_dist)
		q.pop_front();
	while(j < i && tot - dist1 - dist2 > s)
		dist1 += pt[del[j++]].fa_dist;
	while(!q.empty() && pt[q.back()].dist < pt[del[i]].dist)
		q.pop_back();
	q.push_back(del[i]);
	int temp = std::max(dist1, dist2);
	temp = std::max(temp, pt[q.front()].dist);
	ans = std::min(temp, ans); 
}
```
    - **核心实现思想**：在直径序列上，通过双指针`i`和`j`维护路径，利用单调队列`q`维护路径上点的第二种贡献值，同时计算第一种贡献值，动态更新并取最小偏心距。

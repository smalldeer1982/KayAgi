# 题目信息

# [NOIP 1998 提高组] 拼数

## 题目描述

设有 $n$ 个正整数 $a_1 \dots a_n$，将它们联接成一排，相邻数字首尾相接，组成一个最大的整数。


## 说明/提示

对于全部的测试点，保证 $1 \leq n \leq 20$，$1 \leq a_i \leq 10^9$。

NOIP1998 提高组 第二题

## 样例 #1

### 输入

```
3
13 312 343
```

### 输出

```
34331213
```

## 样例 #2

### 输入

```
4
7 13 4 246```

### 输出

```
7424613```

### 最优关键思路或技巧
将正整数转化为字符串，通过比较 `a+b` 与 `b+a` 的大小来确定字符串 `a` 和 `b` 的拼接顺序，利用贪心策略直接对字符串进行排序从而得到最大拼接数。这种思维方式巧妙利用了字符串拼接特性，避免复杂的全排列搜索。

# 题目信息

# [NOIP 1998 提高组] 拼数

## 题目描述

设有 $n$ 个正整数 $a_1 \dots a_n$，将它们联接成一排，相邻数字首尾相接，组成一个最大的整数。


## 说明/提示

对于全部的测试点，保证 $1 \leq n \leq 20$，$1 \leq a_i \leq 10^9$。

NOIP1998 提高组 第二题

## 样例 #1

### 输入

```
3
13 312 343
```

### 输出

```
34331213
```

## 样例 #2

### 输入

```
4
7 13 4 246```

### 输出

```
7424613```

### 综合分析与结论
各题解核心思路高度一致：将数字视为字符串，按拼接后字典序最大的规则排序（比较a+b与b+a）。关键差异在于证明严谨性和实现方式。最高质量题解通过数学归纳法证明排序规则的正确性，而低星题解多缺乏严格证明或实现效率较低。

### 关键思路总结
**核心算法**：自定义字典序贪心排序  
**实现技巧**：
1. 字符串直接拼接比较（避免数值溢出）
2. 利用STL的sort自定义比较函数
3. 数学符号抽象简化问题分析（如$\overline{ab}$表示拼接）

**思维突破点**：
- 将数字比较转化为字符串字典序比较
- 发现局部最优（相邻元素顺序）可推导全局最优
- 通过构造比较函数实现整体最优排列

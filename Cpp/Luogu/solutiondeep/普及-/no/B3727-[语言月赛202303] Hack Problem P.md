# [语言月赛202303] Hack Problem P

## 题目描述

下面程序想要实现的功能为：

> 给出两个正整数 $x, y$，保证它们的最小公倍数（$\mathrm{lcm}$）不大于 $10^9$。求它们的最小公倍数。

```cpp
#include <iostream>
#include <algorithm>

int main() {
  int x, y;
  std::cin >> x >> y;
  int ans = x * y / std::__gcd(x, y);
  std::cout << ans << std::endl;
}
```

但是这个程序存在一些问题，请你提交一个程序，你提交的程序**应当输出一组这道题的输入数据，使得上面的程序输出错误的结果**。

提示：
1. `__gcd(x, y)` 可以求出 $x$ 和 $y$ 的最大公因数。
2. $\mathrm{lcm}(x,y)
\times\mathrm{gcd}(x,y)=x
\times y$

### 综合分析与结论
本题的核心在于理解整数溢出问题，特别是在计算最小公倍数时，先乘后除可能导致中间结果超出 `int` 类型的范围。题解通过构造两个大整数，使得它们的乘积超出 `int` 范围，从而暴露原程序的错误。最优的解决方法是先除后乘，避免溢出。

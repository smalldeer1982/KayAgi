# 题目信息

# [GESP202409 二级] 数位之和

## 题目描述

小杨有 $n$ 个正整数，他认为一个正整数是美丽数字当且仅当该正整数每一位数字的总和是 $7$ 的倍数。

小杨想请你编写一个程序判断 $n$ 个正整数哪些是美丽数字。

## 说明/提示

### 数据规模与约定

对全部的测试数据，保证 $1 \leq n \leq 10^5$，$1 \leq a_i \leq 10^5$。

## 样例 #1

### 输入

```
3
7
52
103```

### 输出

```
Yes
Yes
No```

### 综合分析与结论
题目要求判断一组正整数是否为“美丽数字”，即其各位数字之和是否为7的倍数。核心思路是通过循环逐位提取数字并求和，最后判断和是否为7的倍数。该问题属于模拟类问题，主要考察对数字的逐位处理和循环结构的掌握。

### 核心代码片段
```cpp
int sum = 0;
while (a > 0) {
    sum += a % 10;
    a /= 10;
}
if (sum % 7 == 0)
    cout << "Yes" << endl;
else
    cout << "No" << endl;
```
**核心实现思想**：通过循环逐位提取数字并求和，最后判断和是否为7的倍数。

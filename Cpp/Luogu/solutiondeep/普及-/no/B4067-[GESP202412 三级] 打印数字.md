# 题目信息

# [GESP202412 三级] 打印数字

## 题目描述

小杨为数字 $0,1,2$ 和 $3$ 设计了一款表示形式，每个数字占用了 $5\times 5$ 的网格。数字 $0,1,2$ 和 $3$ 的表示形式如下：

```plain
..... ****. ..... .....
.***. ****. ****. ****.
.***. ****. ..... .....
.***. ****. .**** ****.
..... ****. ..... .....
```

小杨想请你将给定的数字 $n$ 转换为对应的表示形式。

## 说明/提示

对于全部数据，保证有 $0\le n\le 10^6$，且 $n$ 仅由数字 $0,1,2,3$ 组成。

## 样例 #1

### 输入

```
12230```

### 输出

```
****.....................
****.****.****.****..***.
****.................***.
****..****.********..***.
****.....................```

### 综合分析与结论
题目要求将给定的数字字符串转换为对应的5x5网格表示形式。两种题解分别采用了逐行处理和打表处理的方法。逐行处理方法通过逐行判断每个数字的对应字符，直接输出结果；打表处理方法则预先存储每个数字的5行表示，通过拼接字符串输出结果。两种方法各有优劣，逐行处理方法代码较为直观，但逻辑较为复杂；打表处理方法代码简洁，但需要预先存储数据。

// 逐行处理方法
for (int i = 1; i <= 5; i++) { //从第 1 行到第 5 行
    string ans = "";
    for (int j = 0; j < s.length(); j++) {
        char ch = s[j];
        if (ch == '0') {
            if (i == 1 || i == 5) //可以根据图形形状进行判断，减少代码编写量
                ans += ".....";
            else
                ans += ".***.";
        } else if (ch == '1')
            ans += "****.";
        else if (ch == '2') {
            //请自行完成这个分支结构的内容
        }
        else if (ch == '3') {
            //请自行完成这个分支结构的内容
        }
    }
    cout << ans << endl;
}

// 打表处理方法
string ch[4][5]={
	".....", //ch[0][0] 存储的字符串
	".***.", //ch[0][1] 存储的字符串
	".***.", //ch[0][2] 存储的字符串
	".***.", //ch[0][3] 存储的字符串
	".....", //ch[0][4] 存储的字符串
//在参考代码中以 0 为例子，在编写代码的时候，还需要把 1,2,3 的形状写入数组定义中。
};

string ans[5]; //使用 ans[0..4] 记录每一行的字符都应分别是什么
for (int i = 0; i < s.length(); i++) {
    int x = s[i] - '0'; //得到对应的数字
    for (int j = 0; j <= 4; j++)
        ans[j] += ch[x][j]; //将五行字符分别添加到答案的对应行之后
}
for (int i = 0; i <= 4; i++) //分别输出五行字符串
    cout << ans[i] << endl;
```

### 关键思路与技巧
1. **逐行处理**：通过逐行判断每个数字的对应字符，直接输出结果。适合对代码逻辑要求较高的场景。
2. **打表处理**：预先存储每个数字的5行表示，通过拼接字符串输出结果。适合对代码简洁性要求较高的场景。

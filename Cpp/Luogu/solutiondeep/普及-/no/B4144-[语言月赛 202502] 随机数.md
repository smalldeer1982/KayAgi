# 题目信息

# [语言月赛 202502] 随机数

## 题目描述

QianQ 正在出题，他需要一个长度为 $n$ 的随机数列 $a_1,a_2,\cdots,a_n$。用 $S$ 表示该数列全部元素的和，即 $S=a_1+a_2+\cdots+a_n$。

根据题目的限制，这个数列需要满足以下条件：

- $a_1,a_2,\cdots,a_n$ 均为 $-10^{6}\sim 10^6$ 范围内的整数。
- $L \le S \le R$，其中 $L,R$ 为给定的整数。

请你输出一个符合要求的数列。**可能有多个数列满足题目要求，任意输出一个即可。对于每一组数据，保证有符合题意的数列。**

## 说明/提示

- 对于 $30\%$ 的测试数据，$-10^6 \le L \le R \le 10^6$；
- 对于另外 $20\%$ 的测试数据，$L=R$；
- 对于 $100\%$ 的测试数据，$1 \le n \le 10^6$，$-10^6\times n \le L \le R \le 10^6\times n$。

## 样例 #1

### 输入

```
5 5 10```

### 输出

```
4 -2 8 -5 1```

## 样例 #2

### 输入

```
6 8 8```

### 输出

```
1 1 1 1 2 2```

### 所选题解
1. **题解：览遍千秋**

     - 提供了两种不同的构造方案，思路清晰且易于理解。
     - 方案1通过数学上的带余除法，确保数列的和精确等于给定的下限L，具有较高的准确性和可操作性。
     - 方案2通过逐步调整数列中的元素，确保和最终落在给定范围内，适用于更一般的情况。
   - **个人心得：**
     - 方案1在处理负数时需要注意带余除法的特殊性，通过 `(a % M + M) % M` 确保余数为正，这是一个实用的技巧。

### 关键思路或技巧
1. **数学构造法**：通过带余除法，将总和L分解为n个数的和，确保精确满足条件。
2. **逐步调整法**：通过循环逐步调整数列中的元素，确保和最终落在给定范围内。

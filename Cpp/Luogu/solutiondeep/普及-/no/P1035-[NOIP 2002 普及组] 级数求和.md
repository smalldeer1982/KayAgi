# 题目信息

# [NOIP 2002 普及组] 级数求和

## 题目描述

已知：$S_n= 1+\dfrac{1}{2}+\dfrac{1}{3}+…+\dfrac{1}{n}$。显然对于任意一个整数 $k$，当 $n$ 足够大的时候，$S_n>k$。

现给出一个整数 $k$，要求计算出一个最小的 $n$，使得 $S_n>k$。


## 说明/提示

**【数据范围】**

对于 $100\%$ 的数据，$1\le k \le 15$。

**【题目来源】**

NOIP 2002 普及组第一题

## 样例 #1

### 输入

```
1
```

### 输出

```
2
```

### 最优关键思路或技巧
- **暴力模拟**：由于数据范围较小，直接通过循环累加调和级数是最简单且有效的方法。
- **浮点数运算**：确保使用 `double` 类型进行累加，避免整数除法导致的精度丢失。
- **循环条件**：设置循环条件为 $S_n \le k$，当 $S_n > k$ 时立即退出循环并输出结果。

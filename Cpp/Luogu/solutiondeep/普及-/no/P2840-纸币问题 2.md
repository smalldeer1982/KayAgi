# 题目信息

# 纸币问题 2

## 题目背景

你是一个非常有钱的小朋友。

## 题目描述

你有 $n$ 种面额互不相同的纸币，第 $i$ 种纸币的面额为 $a_i$ 并且有无限张，现在你需要支付 $w$ 的金额，求问有多少种方式可以支付面额 $w$，答案对 $10^9+7$ 取模。  
注意在这里，同样的纸币组合如果支付顺序不同，会被视作不同的方式。例如支付 $3$ 元，使用一张面值 $1$ 的纸币和一张面值 $2$ 的纸币会产生两种方式（$1+2$ 和 $2+1$）。

## 说明/提示

对于 $40\%$ 的数据，满足 $n\le 10$，$w\le 100$；  
对于 $100\%$ 的数据，满足 $1\le n\le 10^3$，$1\le a_i \le  w\le 10^4$。  

其实小朋友并不有钱。

## 样例 #1

### 输入

```
6 15
1 5 10 20 50 100```

### 输出

```
42```

## 样例 #2

### 输入

```
3 15
1 5 11```

### 输出

```
39```

### 综合分析与结论
本题的核心是计算用给定的不同面额纸币支付指定金额的方式数，且顺序不同视为不同的方式。所有题解均采用了动态规划的思路，定义 `f[i]` 为凑出 `i` 元的方式数，并通过状态转移方程 `f[i] = sum(f[i - a[j]])` 来计算结果。主要区别在于代码实现细节和初始化方式。

### 最优关键思路或技巧
- **动态规划状态定义**：定义 `f[i]` 为凑出 `i` 元的方式数。
- **状态转移方程**：`f[i] = sum(f[i - a[j]])`，其中 `a[j]` 是纸币面额。
- **初始化**：`f[0] = 1`，表示凑出 `0` 元的方式数为 1。
- **取模与溢出处理**：在每次累加时取模，防止数值溢出。

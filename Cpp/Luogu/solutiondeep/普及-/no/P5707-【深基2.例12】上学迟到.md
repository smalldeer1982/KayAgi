# 题目信息

# 【深基2.例12】上学迟到

## 题目描述

学校和 yyy 的家之间的距离为 $s$ 米，而 yyy 以 $v$ 米每分钟的速度匀速走向学校。

在上学的路上，yyy 还要额外花费 $10$ 分钟的时间进行垃圾分类。

学校要求必须在上午 $\textrm{8:00}$  到达，请计算在不迟到的前提下，yyy 最晚能什么时候出门。

由于路途遥远，yyy 可能不得不提前一点出发，但是提前的时间不会超过一天。

## 说明/提示

对于 $100\%$ 的数据，$1 \le s,v \le 10^4$。

## 样例 #1

### 输入

```
100 99```

### 输出

```
07:48```

### 所选高星题解

#### 题解1：C171501 (5星)
**关键亮点**：
- 使用`ceil`函数确保时间向上取整，避免迟到。
- 处理时间跨天的情况，通过增加24小时的分钟数来简化计算。
- 输出格式处理得当，确保小时和分钟不足两位时补零。

**核心代码**：
```cpp
n=8*60+24*60;//两天总共的分钟数
t=ceil(s/v)+10;//ceil()很重要，向上取整，否则按C++逻辑会向下取整导致行走时间少。
n=n-t;//得出剩下的时间。
if(n>=24*60) n-=24*60;判断是否在前一天。
b=n%60;//得出出发分。
a=n/60;//得出出发时
if(a<10)//慢慢判断是否补0
{
    if(b<10) cout<<"0"<<a<<":0"<<b;
    else cout<<"0"<<a<<":"<<b;
}
else
{
    if(b<10) cout<<a<<":0"<<b;
    else cout<<a<<":"<<b;
}
```

#### 题解2：Lylighte (4星)
**关键亮点**：
- 将时间统一转换为分钟，简化计算。
- 通过增加几天的时间确保到校时间减去路上消耗时间大于零。
- 输出时自动补零，代码简洁。

**核心代码**：
```cpp
walktime = s/v;
if(s%v!=0)
    walktime++;
gotime = (480 + 2880 - 10 - walktime)%1440;
gohour = gotime/60;
if(gohour<10)
    cout << 0;
cout << gohour << ':';
gominute = gotime%60;
if(gominute<10)
    cout << 0;
cout << gominute << endl;
```

#### 题解3：杜都督 (4星)
**关键亮点**：
- 使用`ceil`函数确保时间向上取整。
- 通过公式`(上限时间 - 消耗时间 + 进制数) % 进制数`计算出发时间，处理跨天情况。
- 输出时使用`%02d`自动补零，代码简洁。

**核心代码**：
```cpp
int costM = ceil(1.0 * s / v) + EXT;  // 计算消耗分钟
int costH = ceil(1.0 * costM / 60);  // 计算消耗小时
costM %= 60;  // 保留60进制下的分钟部分 
printf("%02d:%02d", (LMT_H - costH + CLK) % CLK, (LMT_M - costM + 60) % 60);  // 输出结果 
```

### 最优关键思路或技巧
1. **时间统一转换为分钟**：简化计算，避免跨天处理的复杂性。
2. **向上取整**：使用`ceil`函数确保时间足够，避免迟到。
3. **输出格式处理**：使用`%02d`或条件判断确保小时和分钟不足两位时补零。

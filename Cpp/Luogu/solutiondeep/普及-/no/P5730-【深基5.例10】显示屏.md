# 题目信息

# 【深基5.例10】显示屏

## 题目描述

液晶屏上，每个阿拉伯数字都是可以显示成 $3\times5$ 的点阵的（其中 `X` 表示亮点，`.` 表示暗点）。现在给出数字位数（不超过 $100$）和一串数字，要求输出这些数字在显示屏上的效果。数字的显示方式如同样例输出，注意每个数字之间都有一列间隔。

## 说明/提示

数据保证，$1 \leq n \leq 100$。

## 样例 #1

### 输入

```
10
0123456789
```

### 输出

```
XXX...X.XXX.XXX.X.X.XXX.XXX.XXX.XXX.XXX
X.X...X...X...X.X.X.X...X.....X.X.X.X.X
X.X...X.XXX.XXX.XXX.XXX.XXX...X.XXX.XXX
X.X...X.X.....X...X...X.X.X...X.X.X...X
XXX...X.XXX.XXX...X.XXX.XXX...X.XXX.XXX
```

该题目要求模拟液晶显示屏上数字的显示效果，核心是通过预先定义每个数字的3×5点阵表示，然后根据输入的数字串逐行输出对应的点阵。各题解均采用了“打表”的方式，预先存储每个数字的点阵表示，然后根据输入的数字串进行拼接和输出。难点在于如何处理数字之间的间隔以及输出的格式控制。

### 最优关键思路
- **打表法**：预先存储每个数字的点阵表示，避免在代码中重复定义，提高代码的可读性和可维护性。
- **逐行处理**：通过逐行处理数字的点阵表示，确保输出的格式正确，避免混乱。
- **字符串拼接**：使用`string`数组或`ans`数组来存储每行的输出，简化字符串的拼接过程。

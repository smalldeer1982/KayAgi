# 题目信息

# 【深进1.例1】求区间和

## 题目描述

给定 $n$ 个正整数组成的数列 $a_1, a_2, \cdots, a_n$ 和 $m$ 个区间 $[l_i,r_i]$，分别求这 $m$ 个区间的区间和。

对于所有测试数据，$n,m\le10^5,a_i\le 10^4$

## 说明/提示

样例解释：第 $1$ 到第 $4$ 个数加起来和为 $10$。第 $2$ 个数到第 $3$ 个数加起来和为 $5$。

对于 $50 \%$ 的数据：$n,m\le 1000$；

对于 $100 \%$ 的数据：$1 \le n, m\le 10^5$，$1 \le a_i\le 10^4$

## 样例 #1

### 输入

```
4
4 3 2 1
2
1 4
2 3```

### 输出

```
10
5```

### 核心代码片段
```cpp
int n, m, a[100050], s[100050];

int main() {
    n = read();
    for (int i = 1; i <= n; i++)
        s[i] = s[i-1] + (a[i] = read());
    m = read();
    for (int i = 1; i <= m; i++) {
        int l = read(), r = read();
        cout << s[r] - s[l-1] << endl;
    }
    return 0;
}
```
**核心实现思想**: 首先读取数组并计算前缀和数组s，然后对于每个查询区间[l, r]，通过s[r] - s[l-1]快速得到区间和。

### 最优关键思路或技巧
使用前缀和数组将区间求和问题转化为简单的减法操作，极大地提高了查询效率。这种方法在处理大规模数据时非常有效。

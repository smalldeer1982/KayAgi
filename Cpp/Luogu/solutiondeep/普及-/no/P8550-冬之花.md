# 冬之花

## 题目背景

“哭泣并不是因为悲伤，

只是因为活着，眼泪就不自觉跑了出来。”

赫尔德失恋以后，跑到大马路上飙车。因为是飙车，她不愿意一直待在路上，而是跑到路左右的黄沙地上转圈圈。她的便宜车的转向功能出了一些问题，但她真的很想转圈圈。

## 题目描述

给定 $n$ 个数 $a_1, a_2, \ldots, a_n$，再给定一个非零数 $x$。

你需要进行 ${10}^{100}$ 次操作，每次操作，你需要选定一个下标 $i$（$1 \le i \le n$），然后给 $x$ 加上 $a_i$。

你需要保证，在每次操作后，新的 $x$ 值都不能为 $0$。

请判断你能否完成这 ${10}^{100}$ 次操作。

## 说明/提示

**【样例解释】**

对于第一组数据，第一次操作只能选择下标 $1$，$a_1=-1$，但 $1+(-1)=0$，因此无法进行操作。输出 `No`。

对于第二组数据，你可以轮流选择下标 $1$ 和 $2$，这样 $x$ 将一直 $+1$、$-1$，从而一直在 $10$ 和 $9$ 之间变化，这样你可以执行任意多次操作。输出 `Yes`。当然，这只是其中一种方案，你还可以有其他的操作方案。

---

**【数据范围】**

- 测试点 1（50 分）：$n = 1$；
- 测试点 2（50 分）：无特殊限制。

对于所有测试点：$1 \le T \le 30$，$1 \le n \le 5$，$1 \le \lvert x \rvert, \lvert a_i \rvert \le 100$。

## 样例 #1

### 输入

```
2
1 1
-1
2 10
1 -1
```

### 输出

```
No
Yes
```

### 综合分析与结论
本题的核心在于通过数学分析判断是否能够进行无限次操作而不使 $x$ 变为 $0$。主要思路分为两种情况：
1. 如果存在两个不同的 $a_i$，则可以通过交替选择不同的 $a_i$ 来避免 $x$ 变为 $0$。
2. 如果所有 $a_i$ 相同，则需要进一步判断 $x$ 与 $a_i$ 的符号关系和整除关系。

### 所选高星题解
#### 题解1：作者：dottle (赞：23)
- **星级**：5
- **关键亮点**：思路清晰，分类讨论全面，代码简洁高效。
- **核心实现思想**：首先检查是否存在不同的 $a_i$，如果存在则直接输出 `Yes`。如果所有 $a_i$ 相同，则根据 $x$ 与 $a_i$ 的符号关系和整除关系进行判断。

```cpp
void solve(){
	int n,x;cin>>n>>x;
	int first;cin>>first;
	int flag=0;
	for(int i=2;i<=n;i++){
		int now;cin>>now;
		if(now!=first)
			flag=1;
	}
	if(flag){
		cout<<"Yes"<<endl;
		return ;
	}
	if(x%first!=0){
		cout<<"Yes"<<endl;
		return ;
	}
	if(x/first>0){
		cout<<"Yes"<<endl;
		return ;
	}
	cout<<"No"<<endl;
}
```

```cpp
{
    cin >> T;
    for(int i = 1; i <= T; ++ i)
    {
        int N, X, t;
        bool neq = false;
        cin >> N >> X;
        for(int j = 1; j <= N; ++ j)
        {
            int tmp;
            cin >> tmp;
            if(j == 1) t = tmp;
            if(j != 1 && tmp != t) neq = true;           
        }
        
        if(neq) puts("Yes");
        else if(t > 0 && X > 0) puts("Yes");
        else if(t < 0 && X < 0) puts("Yes");
        else if(X % t != 0) puts("Yes");
        else puts("No");
    }
    
    return 0;
}
```

### 最优关键思路或技巧
1. **分类讨论**：将问题分为存在不同 $a_i$ 和所有 $a_i$ 相同两种情况，分别处理。
2. **符号与整除关系**：在所有 $a_i$ 相同的情况下，通过判断 $x$ 与 $a_i$ 的符号关系和整除关系来决定是否输出 `Yes`。

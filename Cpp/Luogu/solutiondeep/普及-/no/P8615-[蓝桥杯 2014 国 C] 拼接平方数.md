# 题目信息

# [蓝桥杯 2014 国 C] 拼接平方数

## 题目描述

小明发现 $49$ 很有趣，首先，它是个平方数。它可以拆分为 $4$ 和 $9$，拆分出来的部分也是平方数。$169$ 也有这个性质，我们权且称它们为：拼接平方数。

$100$ 可拆分 $1,00$，这有点勉强，我们规定，$0,00,000$ 等都不算平方数。

小明想：还有哪些数字是这样的呢？

你的任务出现了：找到某个区间的所有拼接平方数。

## 说明/提示

时限 1 秒, 256M。蓝桥杯 2014 年第五届国赛

## 样例 #1

### 输入

```
169 10000```

### 输出

```

1444
1681
3249
4225
4900
9025```

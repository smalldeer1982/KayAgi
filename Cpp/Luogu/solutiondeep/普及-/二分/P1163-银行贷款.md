# 题目信息

# 银行贷款

## 题目描述

当一个人从银行贷款后，在一段时间内他（她）将不得不每月偿还固定的分期付款。这个问题要求计算出贷款者向银行支付的利率。假设利率按月累计。


## 说明/提示

数据保证，$1 \leq w_0, w\leq 2^{31}-1$，$1 \leq m\leq 3000$。

## 样例 #1

### 输入

```
1000 100 12
```

### 输出

```
2.9
```

### 最优关键思路
1. **单调性+二分框架**：利率与总还款额呈严格单调关系，二分法可稳定逼近解  
2. **秦九韶优化**：逐月迭代计算 `a = a*(1+x)-y` 避免重复计算幂次，复杂度O(n)  
3. **等比数列公式**：求和公式转化后只需一次幂运算，理论复杂度更低（但需处理浮点精度）

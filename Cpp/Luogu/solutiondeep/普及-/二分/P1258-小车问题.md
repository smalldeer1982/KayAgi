# 题目信息

# 小车问题

## 题目描述

甲、乙两人同时从 A 地出发要尽快同时赶到 B 地。出发时 A 地有一辆小车，可是这辆小车除了驾驶员外只能带一人。已知甲、乙两人的步行速度一样，且小于车的速度。问：怎样利用小车才能使两人尽快同时到达。

## 说明/提示

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $0 \leq s, a, b \leq 10^9$。

## 样例 #1

### 输入

```
120 5 25
```

### 输出

```
9.600000
```

# AI分析结果



### 综合分析与结论
本题的核心在于如何安排小车的往返路径，使得两人同时到达终点的时间最短。各题解主要分为数学推导法（直接列方程求解最优距离）和二分法（逼近最优解）两种思路。数学方法高效但需正确推导，二分法则更易实现但效率稍低。最终正确公式为 $x = \frac{(a+b)s}{3a+b}$，时间计算为 $\frac{x}{b} + \frac{s-x}{a}$。

```cpp
double s, a, b;
scanf("%lf%lf%lf", &s, &a, &b);
double l = 0, r = s;
do {
    double mid = (l + r) / 2;
    double t1 = mid / b + (s - mid) / a; // 甲的时间
    double meet_time = (mid - (mid / b) * a) / (a + b);
    double t2 = mid / b + meet_time + (s - (mid / b + meet_time) * a) / b; // 乙的时间
    if (t1 < t2) r = mid;
    else l = mid;
} while (fabs(t1 - t2) > 1e-8);
printf("%.6f", t1);
```

### 最优思路总结
**数学推导法**：  
1. 设小车带甲行驶距离 $x$，此时乙步行距离为 $\frac{a}{b}x$。  
2. 小车返回与乙相遇时，两者相对速度为 $(a + b)$，相遇时间 $t = \frac{x - \frac{a}{b}x}{a + b} = \frac{x(b - a)}{b(a + b)}$。  
3. 总时间平衡方程：甲步行剩余路程的时间 = 车返回接乙并到达的总时间，解得 $x = \frac{(a + b)s}{3a + b}$。

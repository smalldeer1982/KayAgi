# 题目信息

# [传智杯 #4 初赛] 小卡和质数

## 题目背景

小卡最近迷上了质数，所以他想到了一个和质数有关的问题来考考你。

质数是指在大于 $1$ 的自然数中，除了 $1$ 和它本身以外不再有其他因数的自然数。

## 题目描述

小卡有 $T(1\le T\le 10^5)$ 组询问。每次询问给你两个正整数 $x,y(1\le x,y\le 10^9)$。

小卡想知道，第 $x$ 个质数和第 $y$ 个质数是否满足 $p_x \oplus p_y =1$，即第 $x$ 个质数和第 $y$ 个质数的异或值是否是$1$。

## 样例 #1

### 输入

```
4
1 2
23 145
66 2
1 14```

### 输出

```
Yes
No
No
No```

### 综合分析与结论
所有题解均抓住核心矛盾：**质数的奇偶性限制**。唯一满足异或为1的质数对是 (2,3)，对应第1、2个质数。通过数学推导发现只有两种情况，直接判断输入组合即可，时间复杂度 O(1)。各题解思路高度一致，差异仅在于代码风格与推导细节。

### 关键思路总结
1. **质数奇偶性分析**：除2外全为奇数 → 必须包含2
2. **异或条件转换**：a^b=1 → a+1=b（二进制视角）
3. **唯一解推导**：2^3=1，对应第1、2个质数
4. **输入组合判断**：直接检查(x,y)是否为(1,2)或(2,1)

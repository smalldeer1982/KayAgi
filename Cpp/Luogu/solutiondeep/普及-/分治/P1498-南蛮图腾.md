# 南蛮图腾

## 题目背景

自从到了南蛮之地，孔明不仅把孟获收拾的服服帖帖，而且还发现了不少少数民族的智慧，他发现少数民族的图腾往往有着一种分形的效果，在得到了酋长的传授后，孔明掌握了不少绘图技术，但唯独不会画他们的图腾，于是他找上了你的爷爷的爷爷的爷爷的爷爷……帮忙，作为一个好孙子的孙子的孙子的孙子……你能做到吗？

## 题目描述

给定一个正整数 $n$，参考输出样例，输出图形。

## 说明/提示

数据保证，$1 \leq n \leq 10$。

# AI分析结果



### 综合分析结论
该题核心是分形图形的递归/分治构造。各题解主要采用三种思路：  
1. **分治递归**（将图形分解为三个子图形复制）
2. **杨辉三角性质**（奇偶性对应图形位置）
3. **逐层模拟**（字符串拼接或二维数组操作）

最优解法为分治递归，因其直观贴合问题结构，代码可读性与扩展性俱佳。

#### 1. [学无止境] ⭐⭐⭐⭐⭐
**关键亮点**：  
- 倒置存储简化复制逻辑  
- 分治思路清晰（向上、左下、右下的三次复制）  
- 预初始化二维数组避免边界问题  
**核心代码**：
```cpp
for(int i=0;i<length/2;i++)
for(int j=0;j<length;j++)
a[i+(length/2)][j+(length/2)]=a[i][j+length]=a[i][j];
```
**实现思想**：每次将当前图形复制到右下方和正下方区域，通过倒置输出简化空间计算。

#### 2. [C2H6O] ⭐⭐⭐⭐

void copy(int &u, int &v) {
    for (int i=1; i<=u; i++)
        for (int j=1; j<=v; j++)
            a[i][v+j] = a[u+i][(v/2)+j] = a[i][j];
}
```
**核心思想**：通过参数传递当前图形尺寸，实现右移和右下复制的同步操作。

#### 3. [expin] ⭐⭐⭐⭐

```
**优化亮点**：通过位运算高效生成图形结构，空间复杂度O(n)。

### 关键技巧总结
1. **倒置存储法**：将图形倒序存储在数组中，输出时反向遍历，简化分形复制时的坐标计算。
2. **分治三向复制**：将当前图形分解为上方、左下、右下三个子图形递归处理。
3. **动态边界调整**：通过位运算（`1<<n`）动态计算图形尺寸，避免固定数组越界。

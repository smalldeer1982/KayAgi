# 题目信息

# [NOIP 2002 普及组] 过河卒

## 题目描述

棋盘上 $A$ 点有一个过河卒，需要走到目标 $B$ 点。卒行走的规则：可以向下、或者向右。同时在棋盘上 $C$ 点有一个对方的马，该马所在的点和所有跳跃一步可达的点称为对方马的控制点。因此称之为“马拦过河卒”。

棋盘用坐标表示，$A$ 点 $(0, 0)$、$B$ 点 $(n, m)$，同样马的位置坐标是需要给出的。

![](https://cdn.luogu.com.cn/upload/image_hosting/ipmwl52i.png)

现在要求你计算出卒从 $A$ 点能够到达 $B$ 点的路径的条数，假设马的位置是固定不动的，并不是卒走一步马走一步。


## 说明/提示

对于 $100 \%$ 的数据，$1 \le n, m \le 20$，$0 \le$ 马的坐标 $\le 20$。

**【题目来源】**

NOIP 2002 普及组第四题

## 样例 #1

### 输入

```
6 6 3 3
```

### 输出

```
6
```

### 综合分析与结论
本题核心是动态规划，关键点在于处理马的控制点与坐标偏移。最优解法为二维DP优化到一维，时间O(nm)，空间O(m)。题解普遍采用状态转移方程 $dp[i][j] = dp[i-1][j] + dp[i][j-1]$，结合阻挡点清零策略。坐标偏移技巧（+2处理）是避免数组越界的关键优化点。

### 关键思路与技巧总结
1. **坐标偏移**：将坐标整体+2，避免马的控制点计算时越界。
2. **空间压缩**：利用滚动数组或一维数组将空间复杂度从O(nm)降至O(m)。
3. **阻挡点处理**：提前标记所有马的控制点，DP时直接跳过这些点。
4. **方向数组**：用预计算偏移量快速标记马的控制区域。

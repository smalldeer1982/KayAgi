# 题目信息

# [CCC 2022 S1]  Good Fours and Good Fives

## 题目描述

$\rm Finn$ 非常喜欢 $4$ 和 $5$，他认为所有的数都可以用 $4$ 和 $5$ 进行相加得出。

例：

$14=5+5+4$

$20=4+4+4+4+4$ 或 $20=5+5+5+5$

$40=4+4+4+4+4+4+4+4+4+4$ 或 $40=4+4+4+4+4+5+5+5+5$ 或 $40=5+5+5+5+5+5+5+5$

当然，$4$ 和 $5$ 的顺序并不重要，重要的是他们的个数。

给你一个正整数 $n$，问有多少种方法可以用 $4$ 和 $5$ 拼凑成 $n$。

## 说明/提示

对于 $20\%$ 的数据：$1\le n\le 10$

对于另外 $15\%$ 的数据：$1\le n\le10^5$ 并且保证
$n\equiv0$ $\pmod 4$

对于另外 $15\%$ 的数据：$1\le n\le10^5$ 并且保证
$n\equiv0$ $\pmod 5$

对于 $100\%$ 的数据：$1\le n\le 10^6$

### 精选题解推荐

#### 1. 作者：BabyDragon185 （★★★★★）
**关键亮点**

- 利用最小公倍数性质，将问题转化为解的步长计算，直接输出结果。  
**核心代码**  
```cpp

```

#### 2. 作者：ztrztr （★★★★）
**关键亮点**  
- 枚举4的个数，判断剩余值能否被5整除，思路直观易实现。  
- 循环次数优化为O(n/4)，在数据范围内完全可行。  
**核心代码**  
```cpp
int x = n / 4, cnt = 0;
for (int i = 0; i <= x; i++) 
    if ((n - i*4) %5 == 0) cnt++;
cout << cnt;
```

#### 3. 作者：charleshe （★★★★）
**关键亮点**  
- 动态规划通过状态分离避免重复计数，正确性高。  
- 状态转移设计巧妙，dp[i][0]和dp[i][1]分别表示最后一步为4或5的方案数。  
**核心代码**  
```cpp
dp[4][0] = 1; dp[5][1] = 1;
for (int i = 6; i <= n; i++) {
    dp[i][0] = dp[i-4][0]; // 最后一步加4
    dp[i][1] = dp[i-5][0] + dp[i-5][1]; // 最后一步加5
}
cout << dp[n][0] + dp[n][1];
```

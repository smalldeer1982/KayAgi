# 题目信息

# 单词分类

## 题目描述

Oliver 为了学好英语决定苦背单词，但很快他发现要直接记住杂乱无章的单词非常困难，他决定对单词进行分类。

两个单词可以分为一类当且仅当组成这两个单词的各个字母的数量均相等。

例如 $\verb!AABAC!$，它和 $\verb!CBAAA!$ 就可以归为一类，而和 $\verb!AAABB!$ 就不是一类。

现在Oliver有 $N$ 个单词，所有单词均由大写字母组成，每个单词的长度不超过 $100$。你要告诉 Oliver 这些单词会被分成几类。


## 说明/提示

- 对于 $70\%$ 的数据满足 $1\le N \le 100$；
- 对于 $100\%$ 的数据满足 $1\le N \le 10000$。

## 样例 #1

### 输入

```
3 
AABAC 
CBAAA 
AAABB```

### 输出

```
2```

# AI分析结果

### 综合分析与结论

本题的核心问题是如何高效地对单词进行分类，判断它们是否属于同一类。大多数题解都采用了相似的思路：通过对每个单词的字母进行排序，使得同一类的单词在排序后变得相同，然后通过去重或比较来统计类别数。以下是各题解的主要思路和优化点：

1. **排序+去重**：大多数题解采用了这种方法，先对每个单词的字母进行排序，然后通过`set`或`map`进行去重，最后统计类别数。
2. **双重排序**：部分题解在排序每个单词的字母后，还对所有单词进行整体排序，然后通过比较相邻单词是否相同来统计类别数。
3. **质因数分解**：个别题解使用了质因数分解的方法，将每个单词的字母映射为质数的乘积，然后通过比较这些乘积来分类。
4. **堆排序**：有题解使用了堆排序来对字母进行排序，虽然思路新颖，但效率不如直接使用`sort`。

# 题目信息

# [NOIP 2001 普及组] 最大公约数和最小公倍数问题

## 题目描述

输入两个正整数 $x_0, y_0$，求出满足下列条件的 $P, Q$ 的个数：

1. $P,Q$ 是正整数。

2. 要求 $P, Q$ 以 $x_0$ 为最大公约数，以 $y_0$ 为最小公倍数。

试求：满足条件的所有可能的 $P, Q$ 的个数。

## 说明/提示

$P,Q$ 有 $4$ 种：

1. $3, 60$。
2. $15, 12$。
3. $12, 15$。
4. $60, 3$。

对于 $100\%$ 的数据，$2 \le x_0, y_0 \le {10}^5$。

**【题目来源】**

NOIP 2001 普及组第二题

## 样例 #1

### 输入

```
3 60
```

### 输出

```
4
```

# AI分析结果



### 综合分析与结论
题目核心是寻找满足特定GCD和LCM条件的数对个数。各题解均围绕以下关键点展开：
1. 利用性质：P×Q = GCD(P,Q) × LCM(P,Q) = x0×y0
2. 枚举因数时通过sqrt(n)优化时间复杂度
3. 处理x0=y0的特殊情况避免重复计数
4. 素因数分解法更高效但需要数学推导能力

### 精选题解与评分

#### 1. 樱雪喵（★★★★★）
**核心亮点**：  
- 简洁高效的枚举实现，时间复杂度O(√n)
- 利用STL的__gcd函数简化代码
- 处理平方数重复计数逻辑清晰  
**关键代码**：
```cpp
for(long long i=1;i<=sqrt(n);i++){
    if(n%i==0&&__gcd(i,n/i)==m) ans+=2;
}
if(m==n) ans--;
```
**个人心得**：  
需注意当x0=y0时，只有1种有效组合而非2种，需特殊处理。

#### 2. sochiji（★★★★★）
**核心亮点**：  
- 通过素因数分解数学推导得出解数为2^k
- 复杂度优化至O(√(y0/x0))，避免枚举
- 代码仅需质因数分解，效率更高  
**关键思路**：
```cpp
int quotient = y / x;
// 分解quotient的质因数个数k，答案即2^k
```

#### 3. zhangboju（★★★★☆）
**核心亮点**：  
- 详细数学推导帮助理解原理
- 手写GCD函数增强可读性
- 正确处理long long溢出问题  
**关键代码**：
```cpp
for(long long i=1;i<=sqrt(x*y);i++){
    if(x*y%i==0&&gcd(i,x*y/i)==x) ans++;
}
ans *= 2 - (i*i==x*y);
```

### 最优思路总结
**关键技巧**：  
1. **因数对称性**：仅需枚举到√n，每找到一对因数对应两个解
2. **乘积性质**：a×b = GCD(a,b)×LCM(a,b) 避免直接计算LCM
3. **质因数分解**：通过分解y0/x0统计质因子数k，解数为2^k（需x0|y0）

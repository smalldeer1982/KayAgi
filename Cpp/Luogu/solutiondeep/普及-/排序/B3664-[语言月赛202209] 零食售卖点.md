# 题目信息

# [语言月赛202209] 零食售卖点

## 题目描述

洛咕公司有一条无限长的大道，可以被视为一个数轴。在这条大道上，一共有 $k$ 个零食售卖点，第 $i$ 个零食售卖点的坐标为 $a_i$。

Aya 非常热爱零食，他想知道，**相邻**两个零食售卖点之间的距离最大为多少。第 $i$ 个零食售卖点和第 $j$ 个零食售卖点之间的距离为 $|a_i-a_j|$。

$|x|$ 代表 $x$ 的绝对值。即，当 $x\ge0$，$|x|=x$；当 $x<0$，$|x|=-x$。例如，$|4|=4,|0|=0,|-1|=1$。

## 说明/提示

### 输入输出样例解释

共 $4$ 个零食售卖点，按数轴上的顺序依次为 $1,3,5,20$，相邻两个零食售卖点的距离依次为 $2,2,15$，答案为 $15$。

### 数据规模与约定

对于 $30\%$ 的数据，$k=2$；  
对于另外 $30\%$ 的数据，$a_i$ 升序给出；  
对于 $100\%$ 的数据，$2 \le k \le 5000, 1 \le a_i \le 10^7$。

## 样例 #1

### 输入

```
4
3 5 1 20```

### 输出

```
15```

# AI分析结果

### 综合分析与结论
本题要求找出相邻零食售卖点之间的最大距离，由于输入的售卖点坐标未按升序排列，需要先排序。题解提供了三种排序方法：使用 `sort` 库函数、冒泡排序和计数排序。`sort` 库函数简洁高效，冒泡排序实现简单但时间复杂度高，计数排序利用值域小的特点可通过本题。

### 重点代码及核心实现思想
#### 1. 使用 `sort` 库函数
```cpp
sort(a + 1, a + n + 1);
```
核心思想：利用 C++ 标准库中的 `sort` 函数对数组 `a` 进行升序排序，默认升序，排序后方便计算相邻元素的距离。

#### 2. 冒泡排序
```cpp
for(int i = 1; i <= n; i++)
    for(int j = 1; j < i; j++)
        if(a[i] > a[j]) swap(a[i], a[j]);
```
核心思想：通过两层循环，比较相邻元素大小，若顺序错误则交换位置，重复此过程直到数组有序。

#### 3. 计数排序
题解未给出代码，但计数排序核心思想是：由于本题值域较小（$1 \le a_i \le 10^7$），可以创建一个大小为值域上限的数组来统计每个数出现的次数，然后根据统计结果依次输出元素，从而实现排序。

### 最优关键思路或技巧
使用 `sort` 库函数，代码简洁且时间复杂度为 $\mathcal O(n \log n)$，在处理本题数据规模时效率较高。

### 可拓展之处
同类型题：需要先排序再处理相邻元素关系的题目，如求相邻元素差值的最小值、相邻元素和的最大值等。类似算法套路：对于不同数据规模和值域范围，选择合适的排序算法，如数据规模小可考虑冒泡排序，值域小可考虑计数排序，数据规模大且无特殊要求优先使用库函数排序。

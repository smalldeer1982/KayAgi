# 题目信息

# [NOIP 2006 普及组] 明明的随机数

## 题目描述

明明想在学校中请一些同学一起做一项问卷调查，为了实验的客观性，他先用计算机生成了 $N$ 个 $1$ 到 $1000$ 之间的随机整数 $(N\leq100)$，对于其中重复的数字，只保留一个，把其余相同的数去掉，不同的数对应着不同的学生的学号。然后再把这些数从小到大排序，按照排好的顺序去找同学做调查。请你协助明明完成“去重”与“排序”的工作。


## 说明/提示

NOIP 2006 普及组 第一题


## 样例 #1

### 输入

```
10
20 40 32 67 40 20 89 300 400 15
```

### 输出

```
8
15 20 32 40 67 89 300 400
```

# AI分析结果

### 综合分析与结论
这些题解主要围绕排序和去重展开，思路可归为两类：一是使用桶排序，利用数组标记数字是否出现来实现去重和排序；二是先排序再去重，借助 `sort` 函数排序，通过比较相邻元素去重，还有部分题解使用了 `unique` 函数或 `set` 容器简化操作。

### 所选题解
- **qr小盆友（5星）**
    - **关键亮点**：思路清晰，详细解释了桶排序原理及改进方法，代码注释丰富，可读性强。
    - **核心代码**：
```cpp
bool book[1001]={0};
int n;
int final=0;
cin>>n;
int a[n];
for(int i=0;i<n;i++){
    cin>>a[i];
    if(book[a[i]]==0){
        final++;
    }
    book[a[i]]=1;
}
cout<<final<<endl;
for(int i=1;i<=1000;i++){
    if(book[i]==1){
        cout<<i<<" ";
    }
}
```
核心思想：用 `book` 数组标记数字是否出现，若未出现则 `final` 加1，最后遍历数组输出标记过的数字。

- **mcturtle（4星）**
    - **关键亮点**：巧妙利用 `set` 容器自动排序和去重的特性，代码简洁高效。
    - **核心代码**：
```cpp
set<int>s;
int n;
cin>>n;
while(n--){
    int x;
    cin>>x;
    s.insert(x);
}
cout<<s.size()<<endl;
for(auto it=s.begin();it!=s.end();it++){
    cout<<*it<<" ";
}
```
核心思想：将输入的数字插入 `set` 容器，自动完成排序和去重，最后输出容器大小和元素。

- **Erina（4星）**
    - **关键亮点**：使用 `unique` 函数去重，结合 `sort` 函数排序，思路简洁，代码规范。
    - **核心代码**：
```cpp
vector<int>mem;
int n,k;
cin>>n;
for(int i=0;i<n;i++){
    cin>>k;
    mem.push_back(k);
}
sort(mem.begin(),mem.end());
vector<int>::iterator it;
it=unique(mem.begin(),mem.end());
mem.resize(distance(mem.begin(),it));
cout<<mem.size()<<endl;
for(int i=0;i<mem.size();i++){
    cout<<mem[i]<<" ";
}
```
核心思想：先将输入数字存入 `vector` 容器，排序后用 `unique` 函数去重，调整容器大小，最后输出结果。

### 最优关键思路或技巧
- **数据结构**：使用 `set` 容器可自动完成排序和去重，代码简洁；桶排序利用数组标记，适用于数据范围小的情况。
- **算法优化**：桶排序在本题中可高效实现去重和排序；`unique` 函数能快速去除相邻重复元素。
- **思维方式**：将排序和去重问题拆解，分别解决，如先排序再去重。

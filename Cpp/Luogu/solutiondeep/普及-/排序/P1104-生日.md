# 题目信息

# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

# AI分析结果

### 综合分析与结论
这些题解主要围绕对学生生日信息进行排序展开，核心是根据年、月、日以及输入顺序进行多关键字排序。多数题解采用结构体存储学生信息，使用`sort`函数或自定义排序算法实现排序。

### 思路与算法要点
1. **结构体排序**：多数题解使用结构体存储学生的姓名、年、月、日和输入顺序，通过自定义比较函数`cmp`对结构体数组进行排序。
2. **日期处理**：部分题解将年、月、日组合成一个数字，如`year * 10000 + month * 100 + day`，方便比较大小。
3. **排序算法**：常用`sort`函数进行快排，也有使用冒泡排序、选择排序等。
4. **特殊情况处理**：对于生日相同的学生，按照输入顺序，后输入的先输出。

### 解决难点
1. **多关键字排序**：通过自定义比较函数`cmp`，依次比较年、月、日和输入顺序，确保排序的正确性。
2. **生日相同情况处理**：在比较函数中，当生日相同时，比较输入顺序，保证后输入的学生先输出。

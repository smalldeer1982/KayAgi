# 题目信息

# 【深基9.例1】选举学生会

## 题目描述

学校正在选举学生会成员，有 $n$（$n\le 999$）名候选人，每名候选人编号分别从 $1$ 到 $n$，现在收集到了 $m$（$m \le 2000000$）张选票，每张选票都写了一个候选人编号。现在想把这些堆积如山的选票按照投票数字从小到大排序。

## 样例 #1

### 输入

```
5 10
2 5 2 2 5 2 2 2 1 2```

### 输出

```
1 2 2 2 2 2 2 2 5 5```

# AI分析结果

### 综合分析与结论
这些题解主要围绕对选票编号进行排序展开，提供了多种排序算法的实现。思路主要分为两类：一是使用 `sort` 函数进行快速排序，简单直接；二是采用桶排算法，利用数组记录每个编号出现的次数，再按顺序输出。

### 所选题解
- **作者：DreamFox（5星）**
    - **关键亮点**：思路清晰，同时给出 `sort` 和桶排两种解法，并指出桶排更快，代码简洁易懂。
    - **核心代码**：
```cpp
// sort 排序
sort(a,a+m); 
// 桶排
for(int i=0;i<m;i++)cin>>a,++b[a]; 
for(int i=0;i<1000;i++)while(b[i]--)cout<<i<<" "; 
```
    - **核心实现思想**：`sort` 函数直接对数组进行排序；桶排通过数组 `b` 记录每个编号出现的次数，再按编号顺序输出对应次数的编号。

### 最优关键思路或技巧
- 对于大规模数据排序，优先考虑时间复杂度低的排序算法，如 `sort` 函数或桶排。
- 桶排适用于数据范围较小且有重复元素的排序场景，能有效降低时间复杂度。

### 拓展思路
同类型题：对大量整数进行排序，且整数范围较小。类似算法套路：可根据数据特点选择合适的排序算法，如数据范围大且无明显规律时用快速排序，数据范围小且有重复元素时用桶排。

# 题目信息

# 宇宙总统

## 题目描述

地球历公元 6036 年，全宇宙准备竞选一个最贤能的人当总统，共有 $n$ 个非凡拔尖的人竞选总统，现在票数已经统计完毕，请你算出谁能够当上总统。


## 说明/提示

票数可能会很大，可能会到 $100$ 位数字。

$1 \leq n \leq 20$。

## 样例 #1

### 输入

```
5
98765
12365
87954
1022356
985678
```

### 输出

```
4
1022356```

# AI分析结果

### 综合分析与结论
这些题解主要围绕如何处理大数字比较来选出最大票数的候选人。由于票数可能达到100位，不能用常规整数类型，大部分题解采用字符串存储票数。思路主要分为两类：一是直接遍历找出最大值；二是先排序再取最大值。

#### 算法要点
- **字符串比较**：先比较长度，长度长的数大；长度相等时，按字典序比较。
- **排序**：使用`sort`函数或自定义排序算法（如冒泡排序、选择排序），排序规则基于字符串比较。
- **数据结构**：部分题解使用结构体存储票数和候选人编号，方便后续处理。

#### 解决难点
- **大数字存储**：使用字符串或字符数组存储超过`long long`范围的票数。
- **字符串比较规则**：避免直接使用字典序比较，先比较长度，再比较字符。

1. **作者：callG (赞：413)，5星**
    - **关键亮点**：思路简洁清晰，直接用`string`类型存储票数，边输入边比较，代码简洁易读。
    - **核心代码**：
```cpp
#include <iostream>
#include <cstring>
using namespace std;
int main() {
    int n, id;
    string max = "";
    string in;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> in;
        int inSize = in.size();
        int maxSize = max.size();
        if (inSize > maxSize || (inSize >= maxSize && in > max)) {
            max = in;
            id = i;
        }
    }
    cout << id << endl << max << endl;
    return 0;
}
```
    - **核心思想**：用`string`类型存储票数，每次输入后与当前最大值比较，更新最大值和对应的候选人编号。

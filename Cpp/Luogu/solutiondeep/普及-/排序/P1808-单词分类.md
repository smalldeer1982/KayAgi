# 题目信息

# 单词分类

## 题目描述

Oliver 为了学好英语决定苦背单词，但很快他发现要直接记住杂乱无章的单词非常困难，他决定对单词进行分类。

两个单词可以分为一类当且仅当组成这两个单词的各个字母的数量均相等。

例如 $\verb!AABAC!$，它和 $\verb!CBAAA!$ 就可以归为一类，而和 $\verb!AAABB!$ 就不是一类。

现在Oliver有 $N$ 个单词，所有单词均由大写字母组成，每个单词的长度不超过 $100$。你要告诉 Oliver 这些单词会被分成几类。


## 说明/提示

- 对于 $70\%$ 的数据满足 $1\le N \le 100$；
- 对于 $100\%$ 的数据满足 $1\le N \le 10000$。

## 样例 #1

### 输入

```
3 
AABAC 
CBAAA 
AAABB```

### 输出

```
2```

### 综合分析与结论
这些题解的核心思路都是先对每个单词内部字母排序，使同类单词变为相同形式，再通过不同方法统计不同单词的数量。具体方法包括使用`map`、`set`判重，多次排序后统计相邻不同单词，或利用唯一质因数分解定理等。

    - **核心代码**：
```cpp
#include<algorithm>
#include<iostream>
#include<map>
using namespace std;
map<string,bool>z;
string a;
int n,sum;
int main(){
    cin>>n;
    for(int i=1;i<=n;i++){
        cin>>a;
        sort(a.begin(),a.end());
        if(!z[a]){
            sum++;
            z[a]=1;
        }
    }
    cout<<sum;
    return 0;
}
```
    - **核心思想**：对每个读入的单词内部字母排序，用`map`记录该排序后的单词是否出现过，若未出现则类别数加1。

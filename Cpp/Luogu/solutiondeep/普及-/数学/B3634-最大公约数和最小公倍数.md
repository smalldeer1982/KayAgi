# 最大公约数和最小公倍数

## 题目描述

给定两个正整数 $a,b$，求他们的最大公约数（gcd）和最小公倍数（lcm）。这两个整数均在 int 范围内。

## 样例 #1

### 输入

```
6 15```

### 输出

```
3 30```

### 核心代码实现
**辗转相除法求gcd（C++版）**：
```cpp
int gcd(int a, int b) {
    return b ? gcd(b, a % b) : a;
}
```
**最小公倍数计算**：
```cpp
int lcm(int a, int b) {
    return a / gcd(a, b) * b; // 先除后乘避免溢出
}
```

---

### 关键思路总结
1. **算法选择**：欧几里得算法（辗转相除法）是求解 gcd 的最优解，时间复杂度 $O(\log(\min(a,b)))$。
2. **公式关联**：利用 gcd 与 lcm 的关系公式，将问题简化为两次计算。
3. **溢出处理**：计算 lcm 时采用先除后乘避免中间值溢出。

# 题目信息

# [信息与未来 2019] 鸡兔同笼

## 题目描述

有三种动物被关在笼子里，它们分别是：
- 鸡，每只鸡有一个头、两只脚；
- 三脚猫，每只三脚猫有一个头，三只脚；
- 兔子，每只兔子有一个头，四只脚。

现在知道笼子中一共有 $x$ 个头、$y$ 只脚，但这个信息并不足以推导出鸡、兔和三脚猫的准确数量。这时候就需要编程来帮忙了——请你求出笼子里分别至少、至多有几只兔子。

## 说明/提示

对于 $50\%$ 的测试数据，有 $1\le x\le 10^6$。

对于 $100\%$ 的测试数据，有 $1\le x\le10^9,1\le T\le20$。
> 本题原始满分为 $20\text{pts}$。

### 综合分析与结论
本题需要求解三种动物（鸡、三脚猫、兔子）在给定头和脚数时，兔子的最少和最多数量。通过数学推导消元，得到兔子的数量范围公式，直接计算即可，无需枚举，时间复杂度为O(1)。

### 最优题解与关键思路
#### 题解1（★★★★★）
**关键亮点**：通过联立方程消元，将问题转化为对兔子数量的约束条件，直接推导出最小值和最大值的闭式表达式，时间复杂度极低。
**核心思路**：  
1. 消元得到 `b = y - 2x - 2c` 和 `a = 3x - y + c`。  
2. 根据非负条件，推导出 `c ≥ max(y-3x, 0)` 和 `c ≤ min((y-2x)//2, x)`。  
3. 直接计算这两个边界值即为答案。

**代码实现**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int x, y;
        cin >> x >> y;
        int min_r = max(y - 3 * x, 0);
        int max_r = min((y - 2 * x) / 2, x);
        cout << min_r << " " << max_r << endl;
    }
    return 0;
}
```

1. **数学建模**：将问题转化为线性方程组，通过消元得到变量间的关系式。  
2. **约束分析**：利用非负条件推导出变量的上下界，避免枚举。  
3. **整数处理**：注意除法取整和边界条件，确保结果为整数。

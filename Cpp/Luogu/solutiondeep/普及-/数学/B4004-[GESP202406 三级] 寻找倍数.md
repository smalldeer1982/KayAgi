# 题目信息

# [GESP202406 三级] 寻找倍数

## 题目描述

小杨有一个包含 $n$ 个正整数的序列 $A=[a_1,a_2,\dots,a_n]$，他想知道是否存在 $i(1\leq i\leq n)$ 使得 $a_i$ 是序列 $A$ 中所有数的倍数。

## 说明/提示

**【样例解释】**

对于第⼀组数据，对于 $a_3=4$，满足 $a_3$ 是 $a_1$ 和 $a_2$ 的倍数。

**【数据范围】**

对于全部数据，保证有 $1\leq t\leq 10$，$1\leq n\leq 10^5$，$1\leq a_i\leq 10^9$。

## 样例 #1

### 输入

```
2
3
1 2 4
5
1 2 3 4 5```

### 输出

```
Yes
No```

# AI分析结果



### 综合分析与结论
题目要求在序列中寻找是否存在一个数是所有数的倍数。关键思路在于发现该数必须是序列最大值，否则无法满足倍数条件。通过两次线性遍历（找最大值+验证整除性）即可高效解决问题，时间复杂度为 O(n)，适用于题目给定的数据规模。

**关键亮点**

**个人心得引用**  
> "倘若其不是序列中的最大值，假设它为 s，而序列中有一个更大的数 t，那么 s 显然不可能是 t 的正整数倍。"  
—— 通过极值性质直接排除不可能情况，体现问题转化的巧妙性

**核心代码实现**  
```cpp
int t; cin >> t;
while (t--) {
    int n, maxa = 0;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        if (a[i] > maxa) maxa = a[i];
    }
    bool valid = true;
    for (int num : a) {
        if (maxa % num != 0) {
            valid = false;
            break;
        }
    }
    cout << (valid ? "Yes" : "No") << endl;
}
```
**实现思想**  
1. 遍历数组记录最大值  
2. 二次遍历验证最大值能否被所有元素整除  
3. 使用标志位提前终止检查优化性能

### 关键思路总结
**极值性质应用**：在需要满足全局关系的场景中，优先考察极值（最大/最小值）是否具备特殊性质，将问题转化为对极值的验证。此技巧可避免暴力枚举，降低时间复杂度。

**同类拓展**

- 验证某个数是否为数组公约数  
- 特殊极值问题（如矩阵鞍点）

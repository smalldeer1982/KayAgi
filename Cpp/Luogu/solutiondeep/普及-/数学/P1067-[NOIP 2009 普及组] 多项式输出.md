# 题目信息

# [NOIP 2009 普及组] 多项式输出

## 题目描述

一元 $n$ 次多项式可用如下的表达式表示：

$$f(x)=a_nx^n+a_{n-1}x^{n-1}+\cdots +a_1x+a_0,a_n\ne 0$$

其中，$a_ix^i$ 称为 $i$ 次项，$a_i$ 称为 $i$ 次项的系数。给出一个一元多项式各项的次数和系数，请按照如下规定的格式要求输出该多项式：

1. 多项式中自变量为 $x$，从左到右按照次数递减顺序给出多项式。

2. 多项式中只包含系数不为 $0$ 的项。

3. 如果多项式 $n$ 次项系数为正，则多项式开头不出 `+` 号，如果多项式 $n$ 次项系数为负，则多项式以 `-` 号开头。

4. 对于不是最高次的项，以 `+` 号或者 `-` 号连接此项与前一项，分别表示此项系数为正或者系数为负。紧跟一个正整数，表示此项系数的绝对值（如果一个高于 $0$ 次的项，其系数的绝对值为 $1$，则无需输出 $1$）。如果 $x$ 的指数大于 $1$，则接下来紧跟的指数部分的形式为“$x^b$”，其中 $b$ 为 $x$ 的指数；如果 $x$ 的指数为 $1$，则接下来紧跟的指数部分形式为 $x$；如果 $x$ 的指数为 $0$，则仅需输出系数即可。

5. 多项式中，多项式的开头、结尾不含多余的空格。


## 说明/提示

对于100%数据，$0 \le n \le 100$，$-100 \le $系数$ \le 100$

### 综合分析与结论
本题考察多项式输出的格式化处理，关键在于处理符号、系数、指数三个部分的多种条件判断。各题解主要差异在于条件判断的顺序与处理方式。最优思路为分阶段处理符号、系数、指数，并合理处理±1及0次项等特殊情况。

### 精选题解

#### 1. 作者：WsW_ （5星）
**关键亮点**：  
- 分三阶段处理（符号→系数→指数），逻辑清晰。  
- 使用`abs`统一处理系数绝对值，简化条件判断。  
- 代码简洁高效，覆盖所有边界情况（如n=0、±1系数）。  

**核心代码**：  
```cpp
for(int i=n;i>=0;i--){
    int a; scanf("%d",&a);
    if(!a)continue;
    // 符号处理
    if(a>0&&i!=n)putchar('+');
    if(a<0)putchar('-');
    // 系数处理
    if(abs(a)!=1||!i)printf("%d",abs(a));
    // 指数处理
    if(i){
        putchar('x');
        if(i!=1)printf("^%d",i);
    }
}
```

#### 2. 作者：Loyal_Soldier （4星）
**关键亮点**：  
- 通过逐项判断条件生成输出，结构直观。  
- 处理了最高次项符号的特殊性。  

**核心代码**：

```cpp
if(x){
    if(i!=n&&x>0) cout<<'+';
    if(i!=0&&x==-1) cout<<'-';

}
```

#### 3. 作者：qhr2023 （4星）
**关键亮点**：

- 正确处理了系数为±1时的省略逻辑。

### 关键思路总结
1. **分阶段处理**：依次处理符号→系数→指数，避免条件交叉。  
2. **绝对值简化判断**：用`abs(a)`统一处理正负，结合`i=0`处理常数项。  
3. **特殊值处理**：对±1且指数非0的项省略系数，指数为1时省略`^1`。

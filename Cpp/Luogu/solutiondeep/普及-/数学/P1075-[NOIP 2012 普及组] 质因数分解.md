# 题目信息

# [NOIP 2012 普及组] 质因数分解

## 题目描述

已知正整数 $n$ 是两个不同的质数的乘积，试求出两者中较大的那个质数。


## 说明/提示

$1 \le n\le 2\times 10^9$

NOIP 2012 普及组 第一题


## 样例 #1

### 输入

```
21```

### 输出

```
7```

# AI分析结果



### 综合分析与结论
题目要求找到两个质因数中较大的一个。所有题解核心思路均为寻找最小质因数，其对应商即为所求。关键优化点在于利用质因数特性，避免不必要的质数判断与循环次数，时间复杂度为O(√n)。最优思路是正向枚举最小因数，无需质数判断，直接输出商。

### 关键思路与技巧
- **因数成对特性**：最小因数对应最大商，直接求解。  
- **无需质数判断**：题目保证n为两不同质数乘积，故第一个找到的因数必为质数。  
- **循环优化**：枚举至√n可显著减少循环次数。

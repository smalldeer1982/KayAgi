# [NOIP 2004 普及组] 不高兴的津津

## 题目描述

津津上初中了。妈妈认为津津应该更加用功学习，所以津津除了上学之外，还要参加妈妈为她报名的各科复习班。另外每周妈妈还会送她去学习朗诵、舞蹈和钢琴。但是津津如果一天上课超过八个小时就会不高兴，而且上得越久就会越不高兴。假设津津不会因为其它事不高兴，并且她的不高兴不会持续到第二天。请你帮忙检查一下津津下周的日程安排，看看下周她会不会不高兴；如果会的话，哪天最不高兴。

# AI分析结果



### 综合分析与结论
本题核心在于遍历7天数据，找到首个总学习时间超过8小时且最大的那一天。各题解普遍采用循环+最大值比较的思路，区别在于条件判断、初始值设定及代码简洁性。关键点在于：①正确处理多个相同最大值（取最早出现）；②判断总时间是否超过8小时；③代码的简洁性与可读性。

---

### 优质题解推荐

#### 1. 封禁用户（⭐⭐⭐⭐⭐）
**关键亮点**：  
- 简洁的循环结构，边读边处理，空间效率高  
- 条件判断 `(s>max)&&(s>8)` 正确处理了时间限制与最大值更新顺序  
- 初始值 `day=0` 自动处理无解情况  

**核心代码**：
```cpp
for (i=1;i<8;i++) {
    cin>>a>>b; 
    s=a+b; 
    if ((s>max)&&(s>8)) max=s,day=i; 
}
cout<<day; 
```

#### 2. zhuoqizhi（⭐⭐⭐⭐）
**关键亮点**：  
- 初始化 `max=8` 将时间限制融入最大值比较，简化条件判断  
- 通过严格的大于号（`<`）确保取最早出现的最大值  
- 清晰的变量命名与注释  

**核心代码**：
```cpp
int max=8, maxi=0;
for(int i=1; i<=7; i++){
    int t1,t2;
    cin>>t1>>t2;
    if(max < t1+t2) { // 自动排除<=8的情况
        max = t1+t2;
        maxi = i;
    }
}
cout << maxi;
```

#### 3. cff_0102（⭐⭐⭐⭐）
**关键亮点**：  
- 初始值 `mx=8` 与 `pos=0` 的巧妙设计  
- 利用 `mx<a+b` 同时完成超时判断与最大值比较  
- 代码高度简洁，无冗余操作  

**核心代码**：
```cpp
int mx=8, pos=0;
for(int i=1; i<=7; i++){
    int a,b; cin>>a>>b;
    if(mx < a+b) mx=a+b, pos=i;
}
cout<<pos;
```

---

### 关键思路与技巧总结
1. **边读边处理**：无需存储全部数据，实时计算总和并比较，节省内存。
2. **初始值融合条件**：将初始最大值设为8，可简化条件判断为单一比较。
3. **严格大于更新**：用 `>` 而非 `>=` 确保取最早出现的最大值。
4. **变量命名优化**：如 `maxi`/`day` 明确表示结果，提高可读性。

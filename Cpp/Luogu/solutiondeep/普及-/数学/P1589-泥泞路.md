# 题目信息

# 泥泞路

## 题目描述

暴雨过后，FJ 的农场到镇上的公路上有一些泥泞路，他有若干块长度为 $L$ 的木板可以铺在这些泥泞路上，问他至少需要多少块木板，才能把所有的泥泞路覆盖住。

## 说明/提示

对于 $100 \%$ 的数据，$n,L \leq 10000$，$s \leq e \leq 10^9$。

## 样例 #1

### 输入

```
3 3
1 6
13 17
8 12```

### 输出

```
5```

# AI分析结果



### 综合分析与结论
该题核心是区间覆盖问题，所有题解均采用贪心算法，核心思路为：按泥泞路起点排序后，用木板尽可能向后延伸覆盖。各解法差异主要体现在边界处理和代码优化上，最优解法时间复杂度为 O(n logn)。

### 最优思路提炼
1. **排序预处理**：按泥泞路起点升序排序，确保处理顺序无后效性  
2. **贪心延伸**：每次从当前已覆盖的最远点开始铺木板，尽可能向后延伸  
3. **边界处理**：  
   - 用`max(x, a[i].l)`跳过已覆盖区域  
   - 通过`while(x < r)`循环计算连续木板数  
4. **数学优化**：使用`(len + L-1)/L`替代`ceil()`函数实现向上取整

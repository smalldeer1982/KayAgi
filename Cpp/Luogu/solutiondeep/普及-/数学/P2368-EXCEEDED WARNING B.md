# EXCEEDED WARNING B

## 题目背景

SGU 107


## 题目描述

求有多少个平方后末尾为 $987654321$ 的 $n$ 位数。

## 说明/提示

$1\le n\le 1 0^6$。

不要问我暴力能过几个点，我只想说呵呵。

## 样例 #1

### 输入

```
8```

### 输出

```
0```

# AI分析结果



### 综合分析与结论
题目要求找到平方末位为特定值的n位数数量，关键在于发现：  
1. **末位9位数决定平方结果**：通过暴力或数学推导可得，仅存在8个9位数满足条件  
2. **高位组合规律**：n>9时高位的第一位有9种选择，后续每位有10种选择  
3. **输出优化**：直接构造结果字符串，避免大数运算

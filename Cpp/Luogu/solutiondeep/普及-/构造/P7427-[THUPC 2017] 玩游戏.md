# 题目信息

# [THUPC 2017] 玩游戏

## 题目描述

ljcc 和他的学妹在玩游戏，这个游戏共有 $n$ 轮，在第 $i$ 轮获胜会获得 $i$ 分，没有平局。

现在给出 ljcc 和学妹的得分，求是否有一种方案符合当前得分。

## 说明/提示

**由于 SPJ 问题，禁止输出行末空格、回车**。

$a,b\le 2^{31}-1,1\le n\le 10^5$
#### 版权信息
来自 THUPC（THU Programming Contest，清华大学程序设计竞赛）2017。

## 样例 #1

### 输入

```
10 5```

### 输出

```
5 1 2 3 4```

### 综合分析与结论

本题的核心在于判断给定的得分是否合法，并构造出合法的得分分配方案。所有题解都基于等差数列求和公式，通过计算总得分来判断是否有解，并通过贪心算法从大到小分配得分。以下是各题解的要点总结：

1. **判断合法性**：所有题解都使用等差数列求和公式 \( \frac{n(n+1)}{2} = a + b \) 来判断是否有解，并通过 \( n = \lfloor \sqrt{2(a+b)} \rfloor \) 来计算 \( n \)。
2. **贪心构造**：大多数题解采用从大到小遍历的方式，尽量减去较大的得分，确保构造的合法性。
3. **代码实现**：大部分题解代码简洁，使用了 `long long` 来避免溢出问题，并注意了输出格式的要求。

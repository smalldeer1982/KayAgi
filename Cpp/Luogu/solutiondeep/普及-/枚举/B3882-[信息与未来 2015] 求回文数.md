# 题目信息

# [信息与未来 2015] 求回文数

## 题目描述

一个正整数，正读和反读都相同的数为回文数，例如 $22,131,2442,37073,6,\cdots$。所有的 $1$ 位数都是回文数。

现给出一个正整数 $n$，求出 $[1,n]$ 中的回文数的个数。

## 说明/提示

### 样例解释
在 $1$ 至 $24$ 中，回文数有 $1\sim 9,11,22$，共 $11$ 个。
### 数据范围
$1\le n\le10^4$。

## 样例 #1

### 输入

```
24```

### 输出

```
11```

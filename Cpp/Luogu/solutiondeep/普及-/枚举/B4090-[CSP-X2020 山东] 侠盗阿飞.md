# 题目信息

# [CSP-X2020 山东] 侠盗阿飞

## 题目描述

侠盗阿飞获得了一笔意外之财 $w$ 元钱，他想用这笔钱去帮助需要帮助的人。现在知道有 $n$ 个需要帮助的人以及他们每个人需要的钱数 $x_i$ 元（$i=0,1,2,3,\dots,n-1$），阿飞应该如何支配这笔钱使得能得到帮助的人数最多？

## 说明/提示

对于 $30\%$ 的数据，$x_i$ 为升序序列（$x_0\lt x_1\lt x_2\lt x_3\lt \dots$）。

对于 $100\%$ 的数据，$0\leq n\leq 500$，$0 \lt x_i\leq 5\times 10^4$，$0\leq w\leq 2\times 10^9$。


## 样例 #1

### 输入

```
10 5
1 2 3 4 5```

### 输出

```
4```

## 样例 #2

### 输入

```
1000 10
20 20 150 110 180 50 200 140 120 200```

### 输出

```
9```

# AI分析结果

### 综合分析与结论
本题旨在用给定的钱财 $w$ 去帮助尽可能多的人，关键在于合理分配资金。由于要使得到帮助的人数最多，显然应优先帮助所需资金少的人。因此，可先对每个人所需的钱数进行排序，然后按从小到大的顺序依次分配资金，直到钱不够为止。

### 通用建议与扩展思路
思路：先对所有人所需钱数进行排序，再按顺序分配资金，累加能帮助的人数，直至资金不足。
扩展思路：此问题属于贪心算法的典型应用，贪心策略是每次都选择当前最优解（即所需资金最少的人）。同类型题目通常涉及资源分配、区间调度等问题，解题关键在于找出合适的贪心策略。

### 示例代码
```cpp
#include <iostream>
#include <algorithm>
#include <vector>

int main() {
    int n, w;
    std::cin >> w >> n;
    std::vector<int> x(n);
    for (int i = 0; i < n; ++i) {
        std::cin >> x[i];
    }
    // 对所需钱数进行排序
    std::sort(x.begin(), x.end());
    int count = 0;
    for (int i = 0; i < n; ++i) {
        if (w >= x[i]) {
            w -= x[i];
            ++count;
        } else {
            break;
        }
    }
    std::cout << count << std::endl;
    return 0;
}
```
核心实现思想：先读取输入的资金 $w$ 和人数 $n$，以及每个人所需的钱数并存储在数组中。对数组进行排序后，按顺序遍历数组，若当前资金足够帮助此人，则扣除相应资金并增加帮助人数，否则停止遍历。最后输出帮助的人数。

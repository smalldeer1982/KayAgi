# 题目信息

# [NOIP 1998 提高组] 拼数

## 题目描述

设有 $n$ 个正整数 $a_1 \dots a_n$，将它们联接成一排，相邻数字首尾相接，组成一个最大的整数。


## 说明/提示

对于全部的测试点，保证 $1 \leq n \leq 20$，$1 \leq a_i \leq 10^9$。

NOIP1998 提高组 第二题

## 样例 #1

### 输入

```
3
13 312 343
```

### 输出

```
34331213
```

## 样例 #2

### 输入

```
4
7 13 4 246```

### 输出

```
7424613```

# AI分析结果

### 综合分析与结论
这些题解主要围绕如何将给定的正整数拼接成最大整数展开，提供了多种思路和方法，包括贪心算法、搜索算法和状压 DP 等。
- **思路对比**：贪心算法是主流思路，通过自定义比较函数 `a + b > b + a` 对字符串排序；搜索算法采用全排列枚举所有可能，结合剪枝优化；状压 DP 则利用状态压缩表示字符串使用情况。
- **算法要点**：贪心算法核心是比较拼接后的字符串大小；搜索算法通过递归生成排列并比较；状压 DP 则依据状态转移方程 $f_{S}=\max_{j\in S}{f_{S\operatorname{xor}2^j}+a_j}$ 求解。
- **解决难点**：难点在于证明贪心算法的正确性，以及优化搜索算法的时间复杂度。

### 最优关键思路或技巧
- **贪心策略**：使用自定义比较函数 `a + b > b + a` 对字符串排序，能有效解决数字拼接问题。
- **状态压缩**：在状压 DP 中，用二进制表示字符串使用状态，可降低空间复杂度。
- **剪枝优化**：在搜索算法中，通过判断当前结果是否可能成为最优解进行剪枝，减少不必要的搜索。

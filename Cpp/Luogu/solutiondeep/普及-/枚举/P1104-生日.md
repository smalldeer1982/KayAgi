# 题目信息

# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

# AI分析结果

### 综合分析与结论
这些题解大多围绕结构体排序展开，核心思路是按年、月、日、输入顺序对学生信息进行排序。部分题解采用了不同排序算法，如快排、冒泡排序、选择排序等，也有题解使用了优先队列、map等数据结构。

```cpp
struct node//结构体排序
{
    string s;//名字
    int n,y,r,num;//年，月，日，输入的编号。
}a[110];
bool cmp(node a,node b)
{
    if(a.n<b.n)return 1;//先比年
    if(a.n>b.n)return 0;
    if(a.n==b.n)
    {
        if(a.y<b.y)return 1;//再比月
        if(a.y>b.y)return 0;
        if(a.y==b.y)
        {
            if(a.r<b.r)return 1;//再比日
            if(a.r>b.r)return 0;
            if(a.r==b.r)
            {
                if(a.num>b.num)return 1;//最后比编号
                else return 0;
            }
        }
    }
}
sort(a+1,a+n+1,cmp);
```

### 最优关键思路或技巧
- **结构体排序**：使用结构体存储学生信息，自定义cmp函数进行多关键字排序。
- **日期组合**：将年月日组合成一个数字，利用数字大小进行排序。

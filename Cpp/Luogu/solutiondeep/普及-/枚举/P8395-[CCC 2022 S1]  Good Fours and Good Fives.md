# 题目信息

# [CCC 2022 S1]  Good Fours and Good Fives

## 题目描述

$\rm Finn$ 非常喜欢 $4$ 和 $5$，他认为所有的数都可以用 $4$ 和 $5$ 进行相加得出。

例：

$14=5+5+4$

$20=4+4+4+4+4$ 或 $20=5+5+5+5$

$40=4+4+4+4+4+4+4+4+4+4$ 或 $40=4+4+4+4+4+5+5+5+5$ 或 $40=5+5+5+5+5+5+5+5$

当然，$4$ 和 $5$ 的顺序并不重要，重要的是他们的个数。

给你一个正整数 $n$，问有多少种方法可以用 $4$ 和 $5$ 拼凑成 $n$。

## 说明/提示

对于 $20\%$ 的数据：$1\le n\le 10$

对于另外 $15\%$ 的数据：$1\le n\le10^5$ 并且保证
$n\equiv0$ $\pmod 4$

对于另外 $15\%$ 的数据：$1\le n\le10^5$ 并且保证
$n\equiv0$ $\pmod 5$

对于 $100\%$ 的数据：$1\le n\le 10^6$

### 所选题解
- **BabyDragon185（5星）**
    - **关键亮点**：思路巧妙，通过数学推导直接得出结果，代码简洁高效。
    - **核心代码**：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int n;
    cin>>n;
    if(n/4<n%4)
    {
        cout<<0;
        return 0;
    }
    cout<<(n/4-n%4)/5+1;
    return 0;
}
```
    - **核心实现思想**：先假设 $n$ 全由 4 组成，计算出 4 的个数 $m$。若 $n \bmod 4 = k$，则可将 $k$ 个 4 换成 5。若 $m < k$ 则无解，否则解的组数为 $(m - k) / 5 + 1$。
- **Fr0sTy（4星）**
    - **关键亮点**：思路清晰，使用枚举法结合数学知识求解，代码易懂。
    - **核心代码**：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,ans=0;
int main() {
    scanf("%d",&n);
    for(int i=0;i<=n;i++) {
        if((n-5*i)<0) break;
        if((n-5*i)%4==0) ans++;
    }
    printf("%d",ans);
    return 0;
}
```
    - **核心实现思想**：设 $n = 4x + 5y$，用含 $y$ 的代数式表示 $x$ 为 $x = \frac{n - 5y}{4}$，枚举 $y$ 的值，若 $n - 5y$ 能被 4 整除，则解的组数加 1。
- **ztrztr（4星）**
    - **关键亮点**：采用暴力枚举法，思路直接，代码实现简单。
    - **核心代码**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    ios::sync_with_stdio(false);
    int x, n, cnt = 0;
    cin >> n;
    x = n / 4;
    for (int i = 0; i <= x; i++) {
        if ((n - i * 4) % 5 == 0) cnt++;
    }
    cout << cnt;
    return 0;
}
```
    - **核心实现思想**：设 $n = 4x + 5y$，枚举 $x$ 的值，若 $n - 4x$ 能被 5 整除，则解的组数加 1。

### 最优关键思路或技巧
- **数学推导**：通过分析 4 和 5 的组合规律，直接得出解的组数，避免了枚举或动态规划的复杂过程，提高了效率。

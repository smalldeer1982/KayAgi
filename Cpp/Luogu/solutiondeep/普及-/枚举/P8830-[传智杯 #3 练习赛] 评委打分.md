# 题目信息

# [传智杯 #3 练习赛] 评委打分

## 题目描述

小 A 参加一个综艺节目。一共有 $n(3 \le n \le 10^6)$ 名评委参与打分（分数范围是 0 到 100 的整数），每个评委依次亮出自己的得分。

为了节目效果，要求从第三个评委开始，每当第 $i$ 个评委给出打分后，立刻计算出出这个选手在前 $i$ 名评委的打分中，去掉一个最高分和一个最低分，剩下 $i-2$ 个评委的平均分，保留 $2$ 位小数。

## 样例 #1

### 输入

```
6
11 45 14 19 19 81```

### 输出

```
14.00
16.50
17.33
24.25```

- **优化**：可以使用两个变量分别记录当前的最高分和最低分，避免每次都遍历数组查找，提高效率。

### 示例代码
```cpp
#include <iostream>
#include <iomanip>
#include <vector>
#include <algorithm>

int main() {
    int n;
    std::cin >> n;
    std::vector<int> scores;
    for (int i = 0; i < n; ++i) {
        int score;
        std::cin >> score;
        scores.push_back(score);
        if (i >= 2) {
            int sum = 0;
            int maxScore = *std::max_element(scores.begin(), scores.end());
            int minScore = *std::min_element(scores.begin(), scores.end());
            for (int s : scores) {
                sum += s;
            }
            sum -= maxScore + minScore;
            double average = static_cast<double>(sum) / (i - 1);
            std::cout << std::fixed << std::setprecision(2) << average << std::endl;
        }
    }
    return 0;
}
```
**核心实现思想**：使用`vector`存储每个评委的打分，在每次新评委打分后，使用`max_element`和`min_element`找出最高分和最低分，计算剩余分数总和并求平均，最后输出结果。

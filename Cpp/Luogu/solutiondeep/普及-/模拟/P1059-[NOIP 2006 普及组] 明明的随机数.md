# 题目信息

# [NOIP 2006 普及组] 明明的随机数

## 题目描述

明明想在学校中请一些同学一起做一项问卷调查，为了实验的客观性，他先用计算机生成了 $N$ 个 $1$ 到 $1000$ 之间的随机整数 $(N\leq100)$，对于其中重复的数字，只保留一个，把其余相同的数去掉，不同的数对应着不同的学生的学号。然后再把这些数从小到大排序，按照排好的顺序去找同学做调查。请你协助明明完成“去重”与“排序”的工作。


## 说明/提示

NOIP 2006 普及组 第一题


## 样例 #1

### 输入

```
10
20 40 32 67 40 20 89 300 400 15
```

### 输出

```
8
15 20 32 40 67 89 300 400
```

# AI分析结果

• 综合分析：这些题解主要围绕排序与去重两个核心任务展开，因题目数据范围小，常采用桶排序、STL 函数（如 unique、set）或先排序再去重等方法。
    - **桶排序**：利用数组下标与数字对应关系，记录数字出现情况，可同时完成排序与去重。优点是直观简单，缺点是空间浪费，适用于数据范围小的场景。
    - **STL 函数**：如 unique 需先排序，能去除相邻重复元素；set 容器自动排序且去重，代码简洁高效。
    - **先排序再去重**：排序后相同数字相邻，通过判断相邻元素是否相等去重。

结论：各方法各有优劣，应根据具体场景和数据规模选择合适方法。

```cpp
#include<iostream>
#include<algorithm>
#include<vector>
using namespace std;
vector<int>mem;
int main(){
    int n,k;
    cin>>n;
    for(int i=0;i<n;i++){
        cin>>k;
        mem.push_back(k);
    }
    vector<int>::iterator it;
    sort(mem.begin(),mem.end());
    it=unique(mem.begin(),mem.end());
    mem.resize(distance(mem.begin(),it));
    cout<<mem.size()<<endl;
    for(int i=0;i<mem.size();i++){
        cout<<mem[i]<<' ';
    }
    return 0;
}
```
    - **核心思想**：将输入数字存入 vector，排序后用 unique 函数去重，通过调整 vector 大小并输出实现题目要求。

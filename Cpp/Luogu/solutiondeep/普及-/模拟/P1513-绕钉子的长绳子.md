# 题目信息

# 绕钉子的长绳子

## 题目描述

平面上有 $N$ 个圆柱形的大钉子，半径都为 $R$，所有钉子组成一个凸多边形。

现在你要用一条绳子把这些钉子围起来，绳子直径忽略不计。求出绳子的长度。


## 说明/提示

本题来自 URAL1020，翻译来自 NOCOW。


## 样例 #1

### 输入

```
4 1

0.0 0.0

2.0 0.0

2.0 2.0

0.0 2.0

```

### 输出

```
14.28

```

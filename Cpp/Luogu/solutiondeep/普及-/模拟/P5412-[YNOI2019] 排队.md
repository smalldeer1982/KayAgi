# 题目信息

# [YNOI2019] 排队

## 题目描述

小明所在的班级要举办一场课外活动，在活动开始之前老师告诉小明：“需要把男女生分成两队，并且每一队都要按照身高从矮到高进行排序”。但是由于小明的马虎，没有把老师的安排转达给同学，导致全班同学已经排成了一列。活动马上要开始了，小明的老师对此很困扰，你能帮帮她吗？


## 说明/提示

对于 $60\%$ 的数据：$0 < n \leq 10^{2}$，$100 \leq h \leq 200$。

对于 $80\%$ 的数据：$0 < n \leq 10^{3}$ ，$100 \leq h \leq 200$。

对于 $100\%$ 的数据：$0 < n \leq 2\times 10^{3}$，$100 \leq h \leq 200$，$T\le 5$。

## 样例 #1

### 输入

```
1
5

120.2 130.3 140.4 150.5 160.6
```

### 输出

```
130.3 150.5
120.2 140.4 160.6
```

```cpp
#include<bits/stdc++.h>
using namespace std;
bool a[10010];
double b1[10010],b2[10010];//b1是男生，b2是女生
int main()
{
    int T;
    cin>>T;
    while(T--)//T个查询
    {
        int n;
        cin>>n;
        for(int i=1;i<=n;i++)
        {
            cin>>a[i];//读入男女
        }
        int cnt1=1,cnt2=1;
        for(int i=1;i<=n;i++)
        {
            double cnt3;
            cin>>cnt3;
            if(a[i]==1)//如果是男生
            {
                b1[cnt1]=cnt3;//读入的直接进入数组
                cnt1++; 
            }
            else //女生
            {
                b2[cnt2]=cnt3;
                cnt2++;
            }
        }
        sort(b1+1,b1+1+cnt1);//排序
        sort(b2+1,b2+1+cnt2);//同上
        for(int i=1;i<=cnt2;i++)//输出
        {
            if(b2[i]!=0)
            cout<<b2[i]<<" ";
        }
        cout<<endl;
        for(int i=1;i<=cnt1;i++)//输出
        {
            if(b1[i]!=0)
            cout<<b1[i]<<" ";
        }
        cout<<endl;
    }
}
```
核心实现思想：用两个数组分别存储男女生身高，输入时根据性别存入对应数组，最后分别排序并输出。

2. **作者：HsKr （4星）**
    - **关键亮点**：提到两种思路并按其中一种实现，代码结构清晰，使用结构体存储数据，便于理解和维护。
    - **核心代码**：
```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>
using namespace std;
const int maxn=10010;
int T,n;
struct node{
    bool sex;//记录性♂别 
    double hei;//记录身高 
}a[maxn];
double boy[maxn],girl[maxn];
int totb,totg;
bool cmp(node a,node b){//排序函数：按照身高排序 
    return a.hei<b.hei;
}
int main(){
    cin>>T;
    while(T--){
        totb=0;totg=0;
        memset(boy,0,sizeof(boy));
        memset(girl,0,sizeof(girl));//清空上次数据 
        cin>>n;
        for(int i=1;i<=n;i++) cin>>a[i].sex;//输入性别 
        for(int i=1;i<=n;i++) cin>>a[i].hei;//输入身高 
        sort(a+1,a+n+1,cmp);//根据身高排序
        for(int i=1;i<=n;i++){ 
            if(a[i].sex==1){//是男孩 
                boy[++totb]=a[i].hei;//加入男孩数组 
            }
            else{//是女孩 
                girl[++totg]=a[i].hei;//加入女孩数组 
            }
        }
        for(int i=1;i<=totg;i++) cout<<girl[i]<<" ";//输出女孩数组 
        cout<<endl;
        for(int i=1;i<=totb;i++) cout<<boy[i]<<" ";//输出男孩数组 
        cout<<endl;
    }
    return 0;
}
```
核心实现思想：先将所有人的身高和性别用结构体存储并按身高排序，再根据性别将身高分别存入男生和女生数组，最后输出。

3. **作者：Lee02 （4星）**
    - **关键亮点**：提供了除sort外的另一种思路，即使用优先队列（堆）来实现排序，丰富了解题方法，并详细介绍了优先队列的基本操作。
    - **个人心得**：“拿scanf、printf输入输出，就是会WA90分。改完之后，可算是A了。”指出输入输出方式的陷阱。
    - **核心代码**：
```cpp
#include<bits/stdc++.h>//万能头文件
#include<queue>
using namespace std;
int n;
int a[10005];//存性别
double b[10005];//存身高
priority_queue<double,vector<double>,greater<double> > q1;//因为懒，就设了两个队列
priority_queue<double,vector<double>,greater<double> > q2;
int main()
{
    int k;
    cin>>k;//数据组数
    for(int i=1;i<=k;i++)
    {
        n=0;
        memset(a,0,sizeof(a));
        memset(b,0,sizeof(b));//每次进行一轮都要进行初始化！
        cin>>n;
        for(int j=1;j<=n;j++)
        cin>>a[j];
        for(int j=1;j<=n;j++)
        cin>>b[j];
        for(int j=1;j<=n;j++)
        {
            if(!a[j]) q1.push(b[j]);//女生就压入q1
            else q2.push(b[j]);	//男生就压入q2
        }
        while(!q1.empty())//先将女生数据弹出
        {
            double a=q1.top();
            cout<<a<<" ";
            q1.pop();
        }
        cout<<endl;//记得回车
        while(!q2.empty())//后将男生数据弹出
        {
            double a=q2.top();
            cout<<a<<" ";
            q2.pop();
        }
        cout<<endl;
    } 
    return 0;//带功告成
} 
```
核心实现思想：利用优先队列的自动排序特性，将男女生身高分别压入不同的优先队列，最后按顺序弹出并输出。

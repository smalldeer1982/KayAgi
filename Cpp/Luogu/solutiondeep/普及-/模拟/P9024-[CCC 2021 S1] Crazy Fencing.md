# 题目信息

# [CCC 2021 S1] Crazy Fencing

## 题目描述

栅栏由 $N$ 块木板组成。

因为是栅栏，所以第 $i$ 块的右边界与第 $i+1$ 块的左边界的高度一样。

告诉你这些木板的边界的高度与这些木板的宽度，输出这些木板的总面积。

$N\leq 10000$，高度和宽度 $\leq 100$。

## 题目背景

## 题目描述

下文的所有参数均默认为整数。

作为一位（7k7k 小游戏的）游戏开发者，你设计了如下（简陋的不如小狗哥哥幼儿园大班的毕业设计的）游戏，存在两个要素：
1. 一个敌对生物，其血量为 $m$。
2. 主角的武器，分为 $n$ 级，第 $i$ 级的伤害是 $i\times p$。

游戏的平衡性需要提前规划，所以你还有一个序列 $\{a_n\}$，其含义如下：

- $a_i$ 表示敌对生物被第 $i$ 级武器恰好攻击 $a_i$ 下会死。

可惜你忘记了 $p$ 具体是多少，所以你需要求出所有可能的 $p$ 的个数。

如果可能有无数个 $p$，那么请输出 `xiaogougege`。



## 说明/提示

#### 【样例 1 解释】

当武器为 $1$ 级时，分析可以发现 $p$ 要满足 $1 \leq p < \frac{3}{2}$。  
当武器为 $2$ 级时，分析可以发现 $p$ 要满足 $\frac{3}{4} \leq p < \frac{3}{2}$。  
当武器为 $3$ 级时，分析可以发现 $p$ 要满足 $1 \leq p$。 

又 $p$ 是整数。综上，只有当 $p = 1$ 时满足题目所述条件。

• 综合分析与结论：所有题解思路基本一致，均从“被第i级武器恰好攻击ai下会死”这一条件得出不等式\((a_i - 1) \cdot i \cdot p < m \leq a_i \cdot i \cdot p\)，进而解出关于\(p\)的不等式范围，通过求\(n\)个这样不等式范围的交集得到\(p\)的取值范围，从而得出符合题意的整数\(p\)的个数或判断有无穷解。部分题解提到特殊情况，如\(a_i = 1\)时的处理及无解判断等。在算法复杂度上，大部分题解均为\(O(n)\)。不同之处主要在于代码实现细节，如边界处理、取整方式等。

#include <cstdio>
#include <algorithm>

int n, m, a, l, r, ansl, ansr = 2000000000;

int main() {
    scanf("%d%d", &n, m);
    for(int i = 1; i <= n; i++) {
        scanf("%d", &a);
        l = m / (i * a) + bool(m % (i * a));// 计算p的下限，考虑余数情况
        r = a > 1? m / (i * (a - 1)) -!(m % (i * (a - 1))) : 2000000000;// 计算p的上限，a=1时特殊处理
        ansl = std::max(ansl, l);
        ansr = std::min(ansr, r);
    }
    if(ansr > 1000000000) puts("xiaogougege");// 判断无穷解
    else printf("%d\n", std::max(0, ansr - ansl + 1));// 输出结果，考虑区间为空情况
}

```cpp
#include <bits/stdc++.h>
#define ll long long
using namespace std;
const ll Maxn=1e5+7;
ll n,m,a[Maxn],ans,l,r=1e18;
int main(){
	scanf("%lld%lld",&n,&m);
	for(ll i=1;i<=n;i++) scanf("%lld",&a[i]);
	for(ll i=1;i<=n;i++){
		if(a[i]==1){
			ll p2=ceil(m*1.0/(a[i]*i));
			l=max(l,p2);
			continue;
		}
		ll p1=m/((a[i]-1)*i)-(m%((a[i]-1)*i)==0);
		ll p2=ceil(m*1.0/(a[i]*i));
		l=max(l,p2),r=min(p1,r);
	}
	if(r==1e18) printf("xiaogougege");// 判断无穷解
	else if(r-l+1<=0) printf("0");// 判断无解
	else printf("%lld",max(0ll,r-l+1));
	return 0;
}
```

最优关键思路或技巧：从题目条件准确列出不等式并求解出\(p\)的取值范围，利用取区间交集确定最终\(p\)的范围，同时注意特殊情况\(a_i = 1\)的处理以及区间为空集和无穷解的判断。

可拓展之处：此类题目属于根据条件建立不等式求解范围的类型，类似套路可用于其他涉及数值范围限制的问题，如资源分配满足特定条件下求参数范围等。

# 题目信息

# [NOIP 2001 普及组] 最大公约数和最小公倍数问题

## 题目描述

输入两个正整数 $x_0, y_0$，求出满足下列条件的 $P, Q$ 的个数：

1. $P,Q$ 是正整数。

2. 要求 $P, Q$ 以 $x_0$ 为最大公约数，以 $y_0$ 为最小公倍数。

试求：满足条件的所有可能的 $P, Q$ 的个数。

## 说明/提示

$P,Q$ 有 $4$ 种：

1. $3, 60$。
2. $15, 12$。
3. $12, 15$。
4. $60, 3$。

对于 $100\%$ 的数据，$2 \le x_0, y_0 \le {10}^5$。

**【题目来源】**

NOIP 2001 普及组第二题

## 样例 #1

### 输入

```
3 60
```

### 输出

```
4
```

# AI分析结果



### 综合分析与结论

所有题解均基于核心公式 $\text{gcd}(P,Q) \times \text{lcm}(P,Q) = P \times Q$，通过数学性质优化枚举或质因数分解求解。关键优化点在于将枚举范围缩小到 $\sqrt{x_0 \times y_0}$，或通过质因数分解直接计算组合数。最优解法时间复杂度为 $O(\sqrt{n})$ 或更低。

### 精选题解及评分

1. **樱雪喵（5星）**  
   **关键亮点**：  
   - 利用 $\sqrt{n}$ 枚举优化，代码极简  
   - 处理平方数重复计数问题（`if(m==n) ans--`）

### 最优思路总结

1. **公式转化**：利用 $P \times Q = x_0 \times y_0$ 减少变量维度。
2. **枚举优化**：仅枚举到 $\sqrt{x_0 \times y_0}$，对称性计数。
3. **质因数分解法**：当 $y_0/x_0$ 的质因数个数为 $k$ 时，答案为 $2^k$。
4. **边界处理**：平方数特判（$x_0=y_0$）和无解情况（$x_0 \nmid y_0$）。

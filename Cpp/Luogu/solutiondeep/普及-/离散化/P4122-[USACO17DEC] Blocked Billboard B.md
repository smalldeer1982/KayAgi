# 题目信息

# [USACO17DEC] Blocked Billboard B

## 题目描述

在漫长的挤奶过程中，奶牛 Bessie 喜欢透过谷仓的窗户盯着街对面的两块巨大的矩形广告牌，上面分别写着“Farmer Alex 的美味苜蓿”和“Farmer Greg 的优质谷物”。广告牌上这两种奶牛饲料的图片对 Bessie 来说比她农场里的草看起来美味得多。

有一天，当 Bessie 正盯着窗外时，她惊讶地看到一辆巨大的矩形卡车停在街对面。卡车的侧面有一则广告，写着“Farmer Smith 的顶级牛排”，Bessie 不太理解这则广告，但她更担心的是卡车可能会挡住她最喜欢的两块广告牌的视线。

给定两块广告牌和卡车的位置，请计算两块广告牌仍然可见的总面积。卡车可能遮挡了其中一块、两块，或者没有遮挡任何一块广告牌。

## 说明/提示

在这个例子中，第一块广告牌有 $5$ 单位面积可见，第二块广告牌有 $12$ 单位面积可见。

题目来源：Brian Dean

## 样例 #1

### 输入

```
1 2 3 5
6 0 10 4
2 1 8 3```

### 输出

```
17```

# AI分析结果

### 综合分析与结论
这些题解主要分为两种思路：模拟法和数学计算法。
 - **模拟法**：通过创建二维数组模拟平面直角坐标系，标记矩形覆盖区域。因坐标可能为负，需对坐标值进行处理（如加上1000）。不同题解在标记方式（如标记点或格子）、循环边界处理上有差异，但核心都是遍历矩形区域进行标记，最后统计未被覆盖区域。此方法直观但时间复杂度较高，极限情况可达O(n^2)，n为坐标范围。
 - **数学计算法**：先计算两个广告牌矩形面积，再通过计算它们与卡车矩形的重叠面积，用总面积减去重叠面积得到可见面积。计算重叠面积时，通过比较坐标确定重叠矩形的边界，进而算出面积。该方法时间复杂度为O(1)，效率更高。

综合来看，数学计算法在效率上更优，模拟法胜在直观易懂。

### 通用建议与扩展思路
对于此类几何图形面积计算问题，可优先考虑能否用数学方法简化计算，减少模拟带来的高复杂度。若数据范围小，模拟法也是可行且易实现的。相似类型题常涉及多个几何图形的位置关系与面积、周长计算，可通过分析图形重叠、包含等关系，利用几何性质和数学公式求解。

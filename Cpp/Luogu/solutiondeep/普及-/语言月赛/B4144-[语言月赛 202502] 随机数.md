# 题目信息

# [语言月赛 202502] 随机数

## 题目描述

QianQ 正在出题，他需要一个长度为 $n$ 的随机数列 $a_1,a_2,\cdots,a_n$。用 $S$ 表示该数列全部元素的和，即 $S=a_1+a_2+\cdots+a_n$。

根据题目的限制，这个数列需要满足以下条件：

- $a_1,a_2,\cdots,a_n$ 均为 $-10^{6}\sim 10^6$ 范围内的整数。
- $L \le S \le R$，其中 $L,R$ 为给定的整数。

请你输出一个符合要求的数列。**可能有多个数列满足题目要求，任意输出一个即可。对于每一组数据，保证有符合题意的数列。**

## 说明/提示

- 对于 $30\%$ 的测试数据，$-10^6 \le L \le R \le 10^6$；
- 对于另外 $20\%$ 的测试数据，$L=R$；
- 对于 $100\%$ 的测试数据，$1 \le n \le 10^6$，$-10^6\times n \le L \le R \le 10^6\times n$。

## 样例 #1

### 输入

```
5 5 10```

### 输出

```
4 -2 8 -5 1```

## 样例 #2

### 输入

```
6 8 8```

### 输出

```
1 1 1 1 2 2```

# AI分析结果

### 综合分析与结论

本题要求生成一个长度为 $n$ 的随机数列，满足每个元素在 $-10^6$ 到 $10^6$ 之间，且数列的和 $S$ 在 $[L, R]$ 范围内。题解提供了两种方案：

1. **方案1**：通过带余除法将 $L$ 分解为 $an + b$，然后输出 $b$ 个 $a+1$ 和 $n-b$ 个 $a$，确保总和为 $L$。
2. **方案2**：通过循环逐步输出元素，每次根据 $L$ 的值输出最大或最小可能的数，直到 $L$ 被消耗完毕。

两种方案各有优劣：
- **方案1** 思路清晰，代码简洁，适用于 $L$ 和 $R$ 相等的情况，且能精确控制总和。
- **方案2** 则更为灵活，适用于 $L$ 和 $R$ 不等的情况，但代码稍显复杂。

# 题目信息

# [NOIP 2012 普及组] 质因数分解

## 题目描述

已知正整数 $n$ 是两个不同的质数的乘积，试求出两者中较大的那个质数。


## 说明/提示

$1 \le n\le 2\times 10^9$

NOIP 2012 普及组 第一题


## 样例 #1

### 输入

```
21```

### 输出

```
7```

# AI分析结果



### 综合分析与结论
本题要求找出两个质数乘积中的较大者，核心思路为寻找最小质因数。最优解法均基于因数成对特性，通过遍历找到最小的质因数后直接计算较大值。所有高效解法时间复杂度均为 O(√n)，但实际运行极快（因一旦找到即停止循环）。

### 最优思路总结
**关键技巧**：  
- 利用因数配对特性，找最小因数直接得解  
- 基于质数分布特性（较小因数必≤√n），优化循环范围  
- 无需显式判断质数，题目保证输入结构隐含性质  

**思维突破点**：  
- 逆向思维：寻找最大因数 → 等价于找最小因数  
- 数学性质优先：利用数论结论代替暴力枚举

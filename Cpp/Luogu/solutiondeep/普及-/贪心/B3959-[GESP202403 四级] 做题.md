# 题目信息

# [GESP202403 四级] 做题

## 题目描述

小杨同学为了提高自己的实力制定了做题计划，在第 $k$ 天时，他必须要完成 $k$ 道题，否则他就会偷懒。

小杨同学现在找到了一个题库，一共有 $n$ 套题单，每一套题单中有一定数量的题目。但是他十分挑剔，每套题单他只会使用一次，每一天也只能使用一套题单里的题目，之后那套题单就会弃之不用。对于每套题单，他不必完成题单内所有的题。

那么问题来了，小杨同学最多做题几天才偷懒呢？

## 说明/提示

### 数据规模与约定
对全部的测试数据，保证 $1 \leq n \leq 10^6$，$1 \leq a_i \leq 10^9$。

## 样例 #1

### 输入

```
4
3 1 4 1
```

### 输出

```
3```

# AI分析结果



### 综合分析与结论
题目要求通过贪心策略找出最大连续做题天数。各题解均采用排序后贪心遍历的思路，核心差异在于实现效率和代码简洁性。最优解法时间复杂度为 O(n log n)，适用于 1e6 的数据规模。

关键思路：
1. **排序预处理**：将题单从小到大排序，确保优先使用最小可用题单。
2. **贪心遍历**：维护当前天数 k，当题单满足 a[i] ≥ k 时，天数自增。最终输出 k-1。

### 精选题解推荐

#### 1. 作者：yu1128_AKIOI（⭐⭐⭐⭐⭐）
**核心亮点**：
- 代码极简，逻辑清晰
- 时间复杂度严格 O(n log n)
- 初始化技巧（sum=1 与 sum-1 处理边界）

```cpp
sort(a+1,a+n+1);
int sum=1;
for(int i=1;i<=n;i++){
    if(a[i]>=sum) sum++;
}
cout<<sum-1;
```

#### 2. 作者：damnM3bro（⭐⭐⭐⭐⭐）
**核心亮点**：
- 变量命名直观（now 表示当前天数）
- 无冗余操作，逻辑直接

```cpp
sort(a+1,a+1+n);
int ans=0, now=1;
for(int i=1;i<=n;++i){
    if(a[i]>=now) now++, ans++;
}
cout<<ans;
```

# 题目信息

# zzc 种田

## 题目背景

可能以后 zzc 就去种田了。


## 题目描述

田地是一个巨大的矩形，然而 zzc 每次只能种一个正方形,而每种一个正方形时 zzc 所花的体力值是正方形的周长，种过的田不可以再种，zzc 很懒还要节约体力去泡妹子，想花最少的体力值去种完这块田地，问最小体力值。

## 说明/提示

$1\le x,y\le 10^{16}$。

## 样例 #1

### 输入

```
1 10```

### 输出

```
40```

## 样例 #2

### 输入

```
2 2 ```

### 输出

```
8```

# AI分析结果



### 综合分析与结论
题目要求用最小体力种完矩形田地，核心是通过贪心策略每次选择最大正方形，并利用数学优化降低时间复杂度。最优解法均采用类似GCD的辗转相除思路，通过取模操作减少迭代次数，时间复杂度为 $O(\log \min(x,y))$，完美适配 $10^{16}$ 量级数据。

### 核心思路总结
**关键技巧**：  
1. **贪心+数学优化**：每次取短边为正方形边长，通过 $ans += 4 \times \text{短边} \times \lfloor \text{长边}/\text{短边} \rfloor$ 批量计算体力  
2. **GCD式迭代**：用取模替代减法，将时间复杂度从 $O(n)$ 优化至 $O(\log n)$  
3. **变量交换**：始终保持 $x > y$ 或反之，简化条件判断  

**同类拓展**：  
- 适用于需要分阶段处理最大/最小单元的问题（如分形、矩阵覆盖）  
- 类似算法套路：GCD计算、快速幂、辗转相除类问题

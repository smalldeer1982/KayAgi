# 题目信息

# [国家集训队] 特技飞行

## 题目背景

1. wqs 爱好模拟飞行。

2. clj 开了一家神犇航空，由于 clj 还要玩游戏，所以公司的事务由你来打理。

注意：题目中只是用了这样一个背景，并不与真实/模拟飞行相符

## 题目描述

神犇航空开展了一项载客特技飞行业务。每次飞行长 $n$ 个单位时间，每个单位时间可以进行一项特技动作，可选的动作有 $k$ 种，每种动作有一个刺激程度 $c_i$。如果连续进行相同的动作，乘客会感到厌倦，所以定义某次动作的价值为(距上次该动作的时间) $ \times c_i$，若为第一次进行该动作，价值为 $0$。安排一种方案，使得总价值最大。

## 说明/提示

#### 数据规模与约定

- 对于 $10\%$ 的测试数据，$n \le 20$，$k \le 3$。
- 对于 $100\%$ 的测试数据，$1 \le n \le 10^3$，$1 \le k \le 300$，$0 \le c_i \le 10^3$。

## 样例 #1

### 输入

```
5 2
2 2```

### 输出

```
12

```

# AI分析结果



### 综合分析与结论
本题核心在于通过贪心策略最大化动作间隔的加权和。关键思路为：将刺激度高的动作尽可能安排在时间轴两端，通过排序后依次取最大间隔计算贡献。所有题解均基于此核心思路，差异仅在于实现细节与代码风格。

---

### 精选题解列表

#### 1. 作者：SuperJvRuo (★★★★★)
**关键亮点**：
- 一针见血指出价值计算本质：最后一次与首次时间差 × c_i
- 代码极度简洁（仅需13行核心逻辑）
- 时间复杂度最优（O(k log k)）
**核心代码**：
```cpp
std::sort(a, a+k, std::greater<int>());
n--;
int i = 0;
while(n > 0 && i < k) {
    ans += n*a[i];
    i++;
    n-=2;
}
```
**实现思想**：降序排序后，每次取当前最大间隔（初始n-1），每安排一个动作间隔减少2单位。

---

#### 2. 作者：KillerXu (★★★★☆)
**关键亮点**：
- 明确证明动作只需执行两次的最优性
- 给出数学表达式 `n - (i*2 -1)` 清晰揭示间隔规律
- 处理n为奇数的情况说明完整
**核心代码**：
```cpp
sort(c + 1, c + k + 1, cmp);
for(int i=1; i<=n/2 && i<=k; i++) 
    ans += c[i] * (n - (i*2 -1));
```
**个人心得**："当N是奇数时，最中间的时间点随便安排" 体现了对边界条件的透彻理解。

---

#### 3. 作者：TheAutumnGlory (★★★★☆)
**关键亮点**：
- 独创性提出区间价值模型：`(r-l)*c[i]`
- 代码变量命名直观（l、r表示当前可用区间端点）
- 循环条件简洁高效（`i<=k && l<=r`）
**核心代码**：
```cpp
int r=n, l=1;
for(int i=1; i<=k && l<=r; i++) {
    ans += (r-l)*a[i];
    r--, l++;
}
```

---

### 关键思路总结
1. **贪心策略**：c_i越大，分配的间隔应越宽
2. **间隔衰减规律**：每次安排后可用间隔减少2单位（n-=2）
3. **数学本质**：总价值 = Σ(c_i × (最后一次位置 - 首次位置))
4. **实现技巧**：降序排序后直接累乘衰减间隔

# 题目信息

# 10 进制转 x 进制

## 题目描述

给定一个十进制整数 $n$ 和一个小整数 $x$。将整数 $n$ 转为 $x$ 进制。对于超过十进制的数码，用 `A`，`B` ... 表示。 



## 说明/提示

【数据规模和约定】  
保证 $n$ 不超过 $10^9$，$x$ 不超过 $36$。

## 样例 #1

### 输入

```
1000
2
```

### 输出

```
1111101000
```

1. **递归天然倒序**：递归先深入后返回的特性，省去显式倒序步骤。

3. **边界条件处理**：递归终止条件需兼容n=0，确保输出"0"而非空字符串。

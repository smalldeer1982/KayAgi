# 进制转换

## 题目描述

请你编一程序实现两种不同进制之间的数据转换。


## 样例 #1

### 输入

```
16
FF
2
```

### 输出

```
11111111```

### 关键技巧总结
1. **库函数优先**：`strtol`（字符串转十进制）、`to_chars`（十进制转任意进制）可大幅简化代码。  
2. **位值处理优化**：直接按目标进制逐位处理输入输出，避免中间大数问题。  
3. **字母映射技巧**：`char_val = num < 10 ? '0'+num : 'A'+(num-10)`实现快速字母转换。

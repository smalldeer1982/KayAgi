# 题目信息

# 放苹果

## 题目描述

把 $m$ 个同样的苹果放在 $n$ 个同样的盘子里，允许有的盘子空着不放，问共有多少种不同的分法。（$5,1,1$ 和 $1,1,5$ 是同一种方法）


## 说明/提示

对于所有数据，保证：$1\leq m,n\leq 10$，$0 \leq t \leq 20$。

## 样例 #1

### 输入

```
1
7 3```

### 输出

```
8```

## 样例 #2

### 输入

```
3
3 2
4 3
2 7
```

### 输出

```
2
4
2
```

# AI分析结果

### 综合分析与结论

本题的核心是将 $m$ 个相同的苹果放入 $n$ 个相同的盘子中，允许盘子为空，求不同的分法数。题解中主要采用了递归、动态规划（DP）和深度优先搜索（DFS）三种方法。递归和动态规划是主流解法，DFS 则更多用于数据量较小的情况。

#### 关键思路：
1. **递归**：将问题分为两种情况：至少有一个盘子为空，或者所有盘子都至少有一个苹果。递归边界条件是当苹果数为 0 或盘子数为 1 时，返回 1。
2. **动态规划**：通过二维数组 `dp[i][j]` 表示将 `i` 个苹果放入 `j` 个盘子中的方案数。状态转移方程为 `dp[i][j] = dp[i][j-1] + dp[i-j][j]`，分别对应有空盘子和无空盘子的情况。
3. **DFS**：通过搜索所有可能的分配方案，确保每个盘子的苹果数不递减，避免重复计算。

#### 优化技巧：
- **剪枝**：在 DFS 中，通过限制每个盘子的苹果数不超过剩余苹果的平均值，减少不必要的搜索。
- **记忆化**：在递归中，使用数组记录已计算的结果，避免重复计算。

#### 拓展思路：
- **整数划分问题**：本题可以看作是整数划分问题的一个变种，即将整数 $m$ 划分为最多 $n$ 个部分的问题。
- **背包问题**：动态规划的解法与背包问题有相似之处，可以借鉴背包问题的优化思路。

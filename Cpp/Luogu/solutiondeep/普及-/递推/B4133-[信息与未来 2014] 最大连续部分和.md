# 题目信息

# [信息与未来 2014] 最大连续部分和

## 题目描述

有 $n$ 个整数排成一排，求其中的最大连续部分和。

**至少要选一个数。**

## 说明/提示

$1\le n\le 10^7,0\le |x_i|\le 10^8$。

## 样例 #1

### 输入

```
7
-2 13 12 9 14 -10 2```

### 输出

```
48```

# AI分析结果

### 综合分析与结论

本题要求求解最大连续子数组和，是一个经典的动态规划问题。由于数据规模较大（$n \le 10^7$），需要高效的算法来处理。常见的解法包括暴力枚举、分治法、动态规划等，其中动态规划是最优解，时间复杂度为 $O(n)$，空间复杂度为 $O(1)$。

### 最优关键思路或技巧

1. **动态规划**：通过维护一个当前子数组和 `current_sum` 和最大子数组和 `max_sum`，遍历数组时，若 `current_sum` 加上当前元素大于当前元素，则更新 `current_sum`，否则从当前元素重新开始计算。同时，更新 `max_sum` 为 `current_sum` 和 `max_sum` 中的较大值。
2. **空间优化**：由于每次只需要前一个状态的值，因此可以只用一个变量来维护 `current_sum`，而不需要额外的数组。

### 可拓展之处

- **变种问题**：最大子数组乘积、最长递增子序列、最长公共子序列等问题都可以通过类似的动态规划思路解决。
- **分治法**：虽然分治法的时间复杂度为 $O(n \log n)$，但在某些场景下（如并行计算）可能有优势。

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int maxSubArray(vector<int>& nums) {
    int current_sum = nums[0];
    int max_sum = nums[0];
    for (int i = 1; i < nums.size(); ++i) {
        current_sum = max(nums[i], current_sum + nums[i]);
        max_sum = max(max_sum, current_sum);
    }
    return max_sum;
}

int main() {
    int n;
    cin >> n;
    vector<int> nums(n);
    for (int i = 0; i < n; ++i) {
        cin >> nums[i];
    }
    cout << maxSubArray(nums) << endl;
    return 0;
}
```

### 核心实现思想

- **初始化**：`current_sum` 和 `max_sum` 都初始化为数组的第一个元素。
- **遍历数组**：对于每个元素，更新 `current_sum` 为当前元素与 `current_sum + 当前元素` 中的较大值，然后更新 `max_sum` 为 `current_sum` 和 `max_sum` 中的较大值。
- **输出结果**：最终 `max_sum` 即为最大连续子数组和。

### 总结

动态规划是解决最大连续子数组和问题的最优方法，时间复杂度为 $O(n)$，空间复杂度为 $O(1)$。通过维护当前子数组和和最大子数组和，可以在一次遍历中高效地解决问题。

# 题目信息

# A+B Problem（高精）

## 题目描述

高精度加法，相当于 a+b problem，**不用考虑负数**。

## 说明/提示

$20\%$ 的测试数据，$0\le a,b \le10^9$；

$40\%$ 的测试数据，$0\le a,b \le10^{18}$。

## 样例 #1

### 输入

```
1
1```

### 输出

```
2```

## 样例 #2

### 输入

```
1001
9099```

### 输出

```
10100```

const int MAX=1001;
struct hp {
    int num[MAX]; // 存储10000进制数
    hp& operator=(const char* c) {
        memset(num,0,sizeof(num));
        int n=strlen(c),j=1,k=1;
        for(int i=1;i<=n;i++) {
            if(k==10000) j++,k=1;
            num[j] += k*(c[n-i]-'0');
            k*=10;
        }
        num[0]=j;
        return *this;
    }
};
```

### 最优技巧总结
1. **逆序存储**：将字符串倒序存入整型数组，方便从低位开始计算
2. **进位传递**：统一处理进位循环，避免分支判断
   ```cpp
   for(int i=0; i<max_len; i++) {
       c[i+1] += c[i]/10;
       c[i] %= 10;
   }
   ```
3. **结构体封装**：通过运算符重载实现自然运算表达式
4. **压位优化**：采用10000进制减少循环次数，提升运算效率

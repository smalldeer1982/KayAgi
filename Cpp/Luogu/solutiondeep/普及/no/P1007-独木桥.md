# 题目信息

# 独木桥

## 题目背景

战争已经进入到紧要时间。你是运输小队长，正在率领运输部队向前线运送物资。运输任务像做题一样的无聊。你希望找些刺激，于是命令你的士兵们到前方的一座独木桥上欣赏风景，而你留在桥下欣赏士兵们。士兵们十分愤怒，因为这座独木桥十分狭窄，只能容纳 $1$ 个人通过。假如有 $2$ 个人相向而行在桥上相遇，那么他们 $2$ 个人将无法绕过对方，只能有 $1$ 个人回头下桥，让另一个人先通过。但是，可以有多个人同时呆在同一个位置。

## 题目描述

突然，你收到从指挥部发来的信息，敌军的轰炸机正朝着你所在的独木桥飞来！为了安全，你的部队必须撤下独木桥。独木桥的长度为 $L$，士兵们只能呆在坐标为整数的地方。所有士兵的速度都为 $1$，但一个士兵某一时刻来到了坐标为 $0$ 或 $L+1$ 的位置，他就离开了独木桥。

每个士兵都有一个初始面对的方向，他们会以匀速朝着这个方向行走，中途不会自己改变方向。但是，如果两个士兵面对面相遇，他们无法彼此通过对方，于是就分别转身，继续行走。转身不需要任何的时间。

由于先前的愤怒，你已不能控制你的士兵。甚至，你连每个士兵初始面对的方向都不知道。因此，你想要知道你的部队最少需要多少时间就可能全部撤离独木桥。另外，总部也在安排阻拦敌人的进攻，因此你还需要知道你的部队最多需要多少时间才能全部撤离独木桥。

## 说明/提示

对于 $100\%$ 的数据，满足初始时，没有两个士兵同在一个坐标，$1\le  L\le5\times 10^3$，$0\le N\le5\times10^3$，且数据保证 $N\le L$。

## 样例 #1

### 输入

```
4
2
1 3
```

### 输出

```
2 4```

### 综合分析与结论
该题的核心在于理解士兵在独木桥上相遇后的行为，并将其转化为简单的模拟问题。多个题解都提到了“相遇后转身”可以等价于“直接穿过对方”，从而简化了问题的复杂度。通过这种思路，问题转化为计算每个士兵到达桥两端的最短时间和最长时间，并取其中的最大值。

### 所选高星题解

#### 题解1：GXuLuogu (5星)
**关键亮点**：思路清晰，代码简洁，直接通过排序和简单的数学计算得出结果。  
**核心思想**：将士兵的位置排序，计算最靠近两端的时间，分别求最小和最大时间。  
**代码片段**：
```cpp
sort(a+1,a+N+1);
int max_time = max(L+1-a[1], a[N]);
int min_time = 0;
for (int i=1; i<=N; i++)
    min_time = max(min(a[i], L+1-a[i]), min_time);
cout << min_time << ' ' << max_time << endl;
```

#### 题解2：Untitled (4星)
**关键亮点**：详细解释了相遇后转身的等价性，并通过简单的比较得出结果。  
**核心思想**：每个士兵选择最短路径或最长路径，取最大值。  
**代码片段**：
```cpp
for(int i=1; i<=n; i++) {
    int a;
    cin >> a;
    ans = max(min(l+1-a, a), ans);
    sna = max(max(l+1-a, a), sna);
}
cout << ans << ' ' << sna;
```

#### 题解3：芝麻糊糊 (4星)
**关键亮点**：通过画图理解问题，简化了思路，代码简洁。  
**核心思想**：将士兵的方向简化为向左或向右，直接计算最小和最大时间。  
**代码片段**：
```cpp
for(int i=1; i<=n; i++) {
    cin >> a[i];
    maxn = max(maxn, max(l+1-a[i], a[i]));
    minn = max(minn, min(l+1-a[i], a[i]));
}
cout << minn << ' ' << maxn;
```

### 最优关键思路
将士兵相遇后的行为简化为“直接穿过对方”，从而避免复杂的模拟过程，直接计算每个士兵到达桥两端的时间，并取最大值。

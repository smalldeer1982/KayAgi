# 题目信息

# 地标访问

## 题目背景

改编自 USACO2007Nov 铜组 Exploration


## 题目描述

贝西在一条道路上旅行，道路上有许多地标，贝西想要在日落之前访问尽可能多的路标。将道路视为一条数轴，贝西从原点出发，道路上有 $n(1 \le n \le 5\times 10^4)$ 个地标，每个地标有一个坐标 $x_i(|x_i| \le 10^5)$ 且地标的坐标各不相同，$t(1 \le T \le 10^9)$ 分钟之后将会日落。

## 说明/提示

1、贝西日落时不用回到原点。

2、与原模拟水题不同。

3、数据较原题有所修改

25 14
16
8
-7
3

-17
6
-12

```

8```

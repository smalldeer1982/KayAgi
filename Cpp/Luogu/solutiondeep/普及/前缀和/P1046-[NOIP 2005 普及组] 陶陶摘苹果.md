# 题目信息

# [NOIP 2005 普及组] 陶陶摘苹果

## 题目描述

陶陶家的院子里有一棵苹果树，每到秋天树上就会结出 $10$ 个苹果。苹果成熟的时候，陶陶就会跑去摘苹果。陶陶有个 $30$ 厘米高的板凳，当她不能直接用手摘到苹果的时候，就会踩到板凳上再试试。


现在已知 $10$ 个苹果到地面的高度，以及陶陶把手伸直的时候能够达到的最大高度，请帮陶陶算一下她能够摘到的苹果的数目。假设她碰到苹果，苹果就会掉下来。


## 说明/提示

**【题目来源】**

NOIP 2005 普及组第一题

### 所选高星题解

#### 1. RedContritio (5星)
- **关键亮点**：通过逻辑优化，将比较操作简化为`!(H<height[i])`，减少了不必要的计算，提升了代码效率。
- **代码核心思想**：在读入高度和身高后，直接对身高加30，然后通过逻辑表达式统计能够摘到的苹果数量。
```cpp
#include <iostream>
using namespace std;
int height[20],H,s;
int main()
{
    for(int i=0;i<10;i++)cin >> height[i];
    cin >> H;
    H += 30;
    for(int i=0;i<10;i++)s+=!(H<height[i]);
    cout << s;
}
```

#### 2. installb (4星)
- **关键亮点**：引入递推思想，通过数组累加的方式统计能够摘到的苹果数量，思路新颖。
- **代码核心思想**：利用数组`a[i]`记录每个高度能摘到的苹果数量，通过递推公式`a[i]=a[i]+a[i-1]`计算最终结果。
```cpp
#include<cstdio>
#include<iostream>
using namespace std;
int main()
{ int a[2001]={0},i,b,c;
  for(i=0;i<10;i++)
    { cin>>b;
      a[b]++; //b高度的苹果数+1
    }
  cin>>c; 
  c=c+30;  // 加板凳高度 
  for(i=1;i<=251;i++)
    a[i]=a[i]+a[i-1]; // 加上板凳最大高度为i时能摘到a[i]个苹果，递推 
  printf("%d\n",a[c]);  
  return 0;
}
```

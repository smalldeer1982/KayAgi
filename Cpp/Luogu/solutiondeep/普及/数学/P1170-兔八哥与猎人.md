# 题目信息

# 兔八哥与猎人

## 题目描述

兔八哥躲藏在树林旁边的果园里。果园有 $M  \times  N$ 棵树，组成一个 $M$ 行 $N$ 列的矩阵，水平或垂直相邻的两棵树的距离为 $1$。兔八哥在一棵果树下。

猎人背着猎枪走进了果园，他爬上一棵果树，准备杀死兔八哥。

如果猎人与兔八哥位置的连线上没有其它的果树，猎人就可以看到兔八哥。

现己知猎人和兔八哥的位置，编写程序判断兔子所在的位置是否安全。

## 说明/提示

$1\le n \le 10^5$，$1 \le  a_x, a_y, b_x, b_y \le  10^8$。

## 样例 #1

### 输入

```
1
1 1
1 2```

### 输出

```
no```

### 综合分析与结论

本题的核心在于判断猎人与兔八哥之间的连线上是否存在其他果树。通过分析，可以发现该问题可以转化为判断两个点的横纵坐标差是否互质。如果互质，则连线上没有其他点，猎人可以看到兔八哥；否则，连线上存在其他点，兔八哥安全。

所有题解都围绕这一核心思路展开，主要使用**最大公约数（GCD）**来判断两个坐标差是否互质。大部分题解使用了**辗转相除法**来计算GCD，部分题解还提到了使用C++标准库中的`__gcd`函数来简化代码。

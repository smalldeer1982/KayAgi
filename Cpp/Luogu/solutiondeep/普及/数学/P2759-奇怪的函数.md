# 题目信息

# 奇怪的函数

## 题目描述

使得 $x^x$ 达到或超过 $n$ 位数字的最小正整数 $x$ 是多少？


## 说明/提示

对于全部数据，$1\le n\le 2\times 10^9$。


## 样例 #1

### 输入

```
11```

### 输出

```
10```

# AI分析结果

### 综合分析与结论

本题的核心在于如何高效地找到满足 \(x^x\) 达到或超过 \(n\) 位数字的最小正整数 \(x\)。由于 \(n\) 的范围很大（\(1 \le n \le 2 \times 10^9\)），直接暴力枚举或计算 \(x^x\) 会超时或溢出。因此，大多数题解都采用了**二分查找**结合**对数运算**的优化策略，将问题转化为求解 \(x \cdot \log_{10}(x) \ge n-1\) 的最小 \(x\)。

```cpp
int main(){
    int n;
    LL L=1,R=2e9;
    cin>>n;
    while(L<R){
        LL mid=(L+R)>>1,len=(LL)(mid*log10(1.0*mid))+1;
        if(len<n) L=mid+1;
        else R=mid;
    }
    cout<<L<<endl;
    return 0;
}
```
**实现思想**：通过二分查找在区间 \([1, 2 \times 10^9]\) 内寻找满足 \(x \cdot \log_{10}(x) + 1 \ge n\) 的最小 \(x\)。

# 题目信息

# 【模板】裴蜀定理

## 题目描述

给定一个包含 $n$ 个元素的**整数**序列 $A$，记作 $A_1,A_2,A_3,...,A_n$。

求另一个包含 $n$ 个元素的待定**整数**序列 $X$，记 $S=\sum\limits_{i=1}^nA_i\times X_i$，使得 $S>0$ 且 $S$ 尽可能的小。

## 说明/提示

对于 $100\%$ 的数据，$1 \le n \le 20$，$|A_i| \le 10^5$，且 $A$ 序列不全为 $0$。

## 样例 #1

### 输入

```
2
4059 -1782
```

### 输出

```
99```

# AI分析结果

### 综合分析与结论

本题的核心是应用裴蜀定理（Bézout's identity），要求找到一组整数序列 \(X\)，使得 \(S = \sum_{i=1}^n A_i \times X_i\) 为正且尽可能小。根据裴蜀定理，\(S\) 的最小正值是序列 \(A\) 中所有元素的绝对值的最大公约数（GCD）。因此，问题的关键在于计算序列中所有元素的 GCD。

大多数题解都正确地应用了裴蜀定理，并通过递归或迭代的方式计算 GCD。部分题解还提供了对裴蜀定理的详细证明，帮助读者更好地理解其背后的数学原理。代码实现上，多数题解使用了 C++ 自带的 `__gcd` 函数或手写了递归的 GCD 计算函数。

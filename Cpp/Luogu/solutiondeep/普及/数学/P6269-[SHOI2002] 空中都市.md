# 题目信息

# [SHOI2002] 空中都市

## 题目描述

在未来的空中都市中，有很多个小岛（城区）。现在要求在这些小岛之间架一座桥梁，每座桥是指在两个岛之间的通道。

有个约定，如果 $A$ 与 $B$ 之间有桥，$B$ 与 $C$ 之间有桥，则 $A$ 与 $C$ 之间就不能再架桥了，即对于城市中的任意三个岛，不能在其中的两两之间都架上桥。在这样的约定下，要求架的桥的数量最多。当然不必考虑具体的空间结构问题。

## 样例 #1

### 输入

```
6```

### 输出

```
9```

## 样例 #2

### 输入

```
11```

### 输出

```
30```

# AI分析结果

### 综合分析与结论

本题的核心在于理解图兰定理（Turán's theorem），即在没有三角形（任意三个点之间不两两相连）的图中，最大边数为 $\left\lfloor\dfrac{n^2}{4}\right\rfloor$。所有题解都基于这一理论，但解释深度和代码实现有所不同。

### 最优关键思路或技巧

- **图兰定理**：在没有三角形的图中，最大边数为 $\left\lfloor\dfrac{n^2}{4}\right\rfloor$。

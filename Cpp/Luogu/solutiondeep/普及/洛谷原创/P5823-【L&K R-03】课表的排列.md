# 题目信息

# 【L&K R-03】课表的排列

## 题目背景

小 L 早上来到教室，发现今天的课表非常有趣。

## 题目描述

今天的课表是：

    语文 数学 英语 语文 英语 数学

今天的课表上一共有三个科目：语文、数学、英语。每个科目都有两节课。每科的两节课间隔的课程数分别为 $2,3,1$，从小到大排序后为 $1,2,3$，是一个公差为 $1$ 的等差数列。

小 L 想知道，对于更多的科目，具有这样有趣性质的课表是否存在。换句话说，如果课表上一共有 $n$ 个科目，每个科目都有且仅有两节课，小 L 想知道是否存在一个课表，满足这 $n$ 科的两节课间隔的课程数从小到大排序后是一个公差为 $1$ 的等差数列。

但是，小 L 只会写时间复杂度为 $O((2n)!\times n\log_2 n)$ 的算法，于是他求助于你，请你判断是否存在这样的课表，如果存在，还要输出一种可能的情况。

## 说明/提示

**本题使用 Special Judge。**

本题共 $20$ 个数据点，每个数据点 $5$ 分。

| 数据编号$~x$ | $n=$ |
| :-----------: | :-----------: |
| $1\sim 10$ | $2x+1$ |
| $11\sim 15$ | $200x+1$ |
| $16\sim 20$ | $100000x+1$ |

3```

1 2 3 1 3 2```

# AI分析结果

### 综合分析与结论
这些题解主要围绕如何构造满足条件的课表排列展开。多数题解先通过观察样例及手动模拟较小 \(n\) 值（如 \(n = 3, 5, 7\) ）找规律，发现先输出 \(1\) 到 \(n\) ，再输出 \(n\) 以内奇数，最后输出 \(n\) 以内偶数的排列方式可能满足要求，随后部分题解对该规律进行了证明。
- **思路**：多数题解从观察样例出发，通过手动模拟找规律；少数题解尝试从其他角度如二分图匹配思考，但未成功。
- **算法要点**：按特定顺序输出数字构造排列，即先 \(1\) 到 \(n\) ，再奇数，最后偶数。
- **解决难点**：在于证明该构造方式能使相同科目两节课间隔数排序后成公差为 \(1\) 的等差数列。


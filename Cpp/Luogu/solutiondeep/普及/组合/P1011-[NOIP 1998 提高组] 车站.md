# 题目信息

# [NOIP 1998 提高组] 车站

## 题目描述

火车从始发站（称为第 $1$ 站）开出，在始发站上车的人数为 $a$，然后到达第 $2$ 站，在第 $2$ 站有人上、下车，但上、下车的人数相同，因此在第 $2$ 站开出时（即在到达第 $3$ 站之前）车上的人数保持为 $a$ 人。从第 $3$ 站起（包括第 $3$ 站）上、下车的人数有一定规律：上车的人数都是前两站上车人数之和，而下车人数等于上一站上车人数，一直到终点站的前一站（第 $n-1$ 站），都满足此规律。现给出的条件是：共有 $n$ 个车站，始发站上车的人数为 $a$，最后一站下车的人数是 $m$（全部下车）。试问 $x$ 站开出时车上的人数是多少？


## 说明/提示

对于全部的测试点，保证 $1 \leq a \leq 20$，$1 \leq x \leq n \leq 20$，$1 \leq m \leq 2 \times 10^4$。

NOIP1998 提高组 第一题

## 样例 #1

### 输入

```
5 7 32 4
```

### 输出

```
13
```

# AI分析结果

这些题解主要分为两类思路：基于斐波那契数列规律推导公式求解和暴力模拟求解。
- **基于斐波那契数列规律推导公式求解**：通过列出各站上下车人数，发现上车人数中\(a\)和第二站上车人数（设为\(u\)或\(b\)等）的系数符合斐波那契数列规律。利用最后一站下车人数\(m\)建立方程，求出\(u\)或\(b\)，进而得出第\(x\)站车上人数公式。此方法难点在于发现并总结规律，优点是效率高。
- **暴力模拟求解**：枚举第二站上车人数，根据题目给定的上下车人数规律，模拟每站上下车过程，直到找到使倒数第二站车上人数等于\(m\)的第二站上车人数，再模拟得出第\(x\)站车上人数。该方法思路简单，但效率相对较低。

综合来看，基于斐波那契数列规律推导公式求解更为高效，适用于理解数学规律能力较强的人；暴力模拟求解虽效率低，但思路直接，适合考场应急或对规律推导不熟练时使用。

# 题目信息

# 跑步

## 题目描述

路人甲准备跑 $n$ 圈来锻炼自己的身体，他准备分多次（$\gt1$）跑完，每次都跑正整数圈，然后休息下再继续跑。

为了有效地提高自己的体能，他决定每次跑的圈数都必须比上次跑的多。

可以假设他刚开始跑了 $0$ 圈，那么请问他可以有多少种跑完这 $n$ 圈的方案？

## 说明/提示

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $5\le n\le 500$。

## 样例 #1

### 输入

```
212```

### 输出

```
995645335```

### 重点代码及核心思想
#### 翼德天尊
```cpp
long long n,ans[501]; 
scanf("%d",&n);
ans[0]=1;
for (int i=1;i<=n;i++){
    for (int j=n;j>=i;j--){
        ans[j]+=ans[j-i];
    }
}
cout<<ans[n]-1<<endl;
```
核心思想：用 `ans[i]` 表示前 `i` 圈的方案总数，通过两层循环模拟每一圈的演变过程，最终答案为 `ans[n] - 1` （减去一次跑完的情况）。

#### Anoxiacxy
```cpp
LL dp[520][520], ans;
int n; scanf("%d", &n);
for(int i = 1; i <= n; i++) dp[i][i] = 1;
for(int i = 1; i <= n; i++){
    for(int j = 1; j < i; j++)
        for(int k = 1; k < j && j + k <= i; k++)
            dp[i][j] += dp[i-j][k];
}
for(int i = 1; i < n; i++) ans += dp[n][i];
printf("%lld\n", ans);
```
核心思想：用 `dp[i][j]` 表示一共跑了 `i` 圈，且最后一次跑了 `j` 圈的方案数，通过三重循环进行状态转移，最后累加 `dp[n][i]`（`i < n`）得到结果。

#### AveMarina
```cpp
typedef long long LL;
const int N = 510;
LL f[N];
int n;
cin >> n;
f[0] = 1;
for (int i = 1; i <= n; i++) {
    for (int j = n; j >= i; j--) {
        f[j] += f[j - i];
    }
}
cout << f[n] - 1;
```
核心思想：将问题转化为 01 背包求方案数问题，`f[j]` 表示跑 `j` 圈的方案数，通过两层循环更新状态，最终答案为 `f[n] - 1`。

### 最优关键思路或技巧
- 将问题转化为 01 背包求方案数问题，简化思考过程。
- 合理定义状态和转移方程，如用 `dp[i][j]` 表示跑 `i` 圈且最后一次跑 `j` 圈的方案数。
- 初始化 `dp[0] = 1` 保证状态转移方程有意义。

# 题目信息

# 图像变换

## 题目描述

给定 $3$ 行 $3$ 列的图像各像素点灰度值，给定最终图像，求最短、字典序最小的操作序列。

其中，可能的操作及对应字符有如下四种：

`A`：顺时针旋转 $90$ 度；

`B`：逆时针旋转 $90$ 度；

`C`：左右翻转；

`D`：上下翻转。

## 说明/提示

字符串长度不超过 $10^8$。

## 样例 #1

### 输入

```
3 4 5
6 7 8
1 2 3
1 2 3
6 7 8
3 4 5```

### 输出

```
D```

# 题目信息

# 计数器

## 题目描述

一本书的页数为 $n$，页码从 $1$ 开始编起，请你求出全部页码中，用了多少个 $0,1,2\cdots 9$。每个页码不含前导 $0$，如 $n=1234$ 时第 $5$ 页不是 $0005$，只是 $5$。


## 说明/提示

对于 $100\%$ 数据，满足 $1\le n\le 10^9$。

---

$2022.7.4$：新添加三组 $\text{Hack}$ 数据。

## 样例 #1

### 输入

```
11```

### 输出

```
1
4
1
1
1
1
1
1
1
1```

### 综合分析与结论

本题要求统计从1到n的所有页码中，每个数字（0-9）出现的次数。题解中主要采用了以下几种思路：

1. **分段处理法**：将数字分段处理，利用重复出现的规律减少计算量。例如，将数字分为前几位和后几位，分别计算贡献值。
2. **递推与预处理**：通过预处理每个数字在不同位数上的出现次数，再利用递推公式计算最终结果。
3. **数位DP**：利用动态规划的思想，预处理每个数字在不同位数和不同最高位下的出现次数，再逐位计算。

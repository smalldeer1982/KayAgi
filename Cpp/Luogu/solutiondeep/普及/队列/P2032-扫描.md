# 扫描

## 题目描述

有一个 $1 \times n$ 的矩阵，有 $n$ 个整数。

现在给你一个可以盖住连续 $k$ 个数的木板。

一开始木板盖住了矩阵的第 $1 \sim k$ 个数，每次将木板向右移动一个单位，直到右端与第 $n$ 个数重合。

每次移动前输出被覆盖住的数字中最大的数是多少。

## 说明/提示

对于 $20\%$ 的数据，$1 \leq k \leq n \leq 10^3$。

对于 $50\%$ 的数据，$1 \leq k \leq n \leq 10^4$。

对于 $100\%$ 的数据，$1 \leq k \leq n \leq 2 \times 10^6$，矩阵中的元素大小不超过 $10^4$ 并且均为正整数。 

## 样例 #1

### 输入

```
5 3
1 5 3 4 2
```

### 输出

```
5
5
4
```

### 综合分析与结论

本题的核心问题是在一个滑动窗口中快速找到最大值，常见的解法包括单调队列、优先队列、线段树等。通过对多条题解的分析，可以发现以下几点：

1. **单调队列**是最优的解法，时间复杂度为O(n)，适用于大规模数据。它通过维护一个递减的队列，确保每次窗口滑动时都能快速找到最大值。
2. **优先队列**虽然思路简单，但每次弹出队首元素时需要判断是否在窗口内，时间复杂度为O(n log n)，在大数据量下性能较差。
3. **线段树**和**ST表**等数据结构虽然也能解决问题，但它们的复杂度为O(n log n)，不如单调队列高效。

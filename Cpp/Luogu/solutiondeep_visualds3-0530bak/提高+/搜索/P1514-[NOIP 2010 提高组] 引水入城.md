# 题目信息

# [NOIP 2010 提高组] 引水入城

## 题目背景

NOIP2010 提高组 T4

## 题目描述

在一个遥远的国度，一侧是风景秀美的湖泊，另一侧则是漫无边际的沙漠。该国的行政区划十分特殊，刚好构成一个 $N$ 行 $M$ 列的矩形，如上图所示，其中每个格子都代表一座城市，每座城市都有一个海拔高度。

![](https://cdn.luogu.com.cn/upload/image_hosting/rcqfo04b.png)



为了使居民们都尽可能饮用到清澈的湖水，现在要在某些城市建造水利设施。水利设施有两种，分别为蓄水厂和输水站。蓄水厂的功能是利用水泵将湖泊中的水抽取到所在城市的蓄水池中。

因此，只有与湖泊毗邻的第 $1$ 行的城市可以建造蓄水厂。而输水站的功能则是通过输水管线利用高度落差，将湖水从高处向低处输送。故一座城市能建造输水站的前提，是存在比它海拔更高且拥有公共边的相邻城市，已经建有水利设施。由于第 $N$ 行的城市靠近沙漠，是该国的干旱区，所以要求其中的每座城市都建有水利设施。那么，这个要求能否满足呢？如果能，请计算最少建造几个蓄水厂；如果不能，求干旱区中不可能建有水利设施的城市数目。

## 综合分析与结论  
### 核心思路与难点  
题目要求在首行选择最少的蓄水厂，使得水流能覆盖末行所有城市。关键点在于：  
1. **DFS 预处理覆盖区间**：对每个首行起点进行 DFS，记录其能覆盖的末行左右端点。  
2. **连续性证明**：若末行能被全覆盖，则每个蓄水厂的覆盖区间必为连续（反证法证明）。  
3. **贪心区间覆盖**：将预处理后的区间按左端点排序，贪心选择右端点最大的区间，逐步覆盖末行。

### 可视化设计思路  
- **搜索过程动画**：  
  - 初始状态：首行起点高亮，水流逐步扩散（DFS 方向用不同颜色箭头表示）。  
  - 覆盖标记：末行被覆盖的格子动态填充颜色，形成连续区间。  
  - 贪心选择：用闪烁效果高亮当前最优区间，合并过程以滑动窗口形式展示。  
- **复古像素风格**：  
  - 使用 8-bit 色块表示城市，水流扩散时触发经典音效（如《超级马里奥》水花声）。  
  - 自动演示模式下，DFS 路径以“贪吃蛇”式动画展开，队列/栈操作通过像素箭头可视化。

## 核心代码实现（DFS + 贪心）  
```cpp
int dx[] = {-1, 0, 1, 0}, dy[] = {0, 1, 0, -1};
int h[MAXN][MAXN], l[MAXN][MAXN], r[MAXN][MAXN];
bool vis[MAXN][MAXN];

void dfs(int x, int y) {
    vis[x][y] = true;
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i], ny = y + dy[i];
        if (nx < 1 || nx > n || ny < 1 || ny > m || h[x][y] <= h[nx][ny]) continue;
        if (!vis[nx][ny]) dfs(nx, ny);
        l[x][y] = min(l[x][y], l[nx][ny]);
        r[x][y] = max(r[x][y], r[nx][ny]);
    }
}

// 主函数预处理
for (int i = 1; i <= m; i++) {
    if (!vis[1][i]) dfs(1, i);
}

// 贪心选择区间
int left = 1, ans = 0;
while (left <= m) {
    int max_right = 0;
    for (int i = 1; i <= m; i++) {
        if (l[1][i] <= left) max_right = max(max_right, r[1][i]);
    }
    left = max_right + 1;
    ans++;
}
```

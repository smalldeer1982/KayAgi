# [NOIP 1998 普及组] 三连击

## 题目背景

本题为提交答案题，您可以写程序或手算在本机上算出答案后，直接提交答案文本，也可提交答案生成程序。

## 题目描述

将 $1, 2, \ldots , 9$ 共 $9$ 个数分成 $3$ 组，分别组成 $3$ 个三位数，且使这 $3$ 个三位数构成 $1 : 2 : 3$ 的比例，试求出所有满足条件的 $3$ 个三位数。

## 说明/提示

NOIP1998 普及组 第一题

### 输出

...

（剩余部分不予展示）```

### 综合分析与结论
#### 核心思路
所有题解均围绕 **枚举第一个数 a，验证 a:2a:3a 的数字覆盖性** 展开。核心难点在于：
1. **确定枚举范围**：a 必须是三位数且 3a ≤ 999 ⇒ a ∈ [123, 329]
2. **高效验证数位不重复**：通过数组标记法统计每个数字出现次数，或利用数学性质（和=45、积=362880）快速判定
3. **避免全排列的高复杂度**：通过比例关系减少枚举量，而非生成所有排列

#### 算法流程（以标记法为例）
```python
for a in 123..329:
    b = 2a; c = 3a
    分解 a,b,c 的每一位数字存入数组
    统计每个数字出现次数
    if 所有1-9数字均出现一次:
        输出结果
```

#### 可视化设计思路
- **像素风格动画**：用 3x3 网格表示数字 1-9，每步枚举 a 时高亮其数位，并动态填充 b、c 的数位
- **颜色标记**：绿色表示已覆盖数字，红色表示重复或出现 0
- **音效触发**：成功时播放 8-bit 胜利音效，失败时短促提示音

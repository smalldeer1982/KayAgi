# 万能的OIH搜索

## 题目背景

以下皆为真实的故事。

OIH（Oi in hand <http://www.oiinhand.info/> ）是洛谷的官方战略合作伙伴。

洛谷的题目搜索功能（请看看上面的搜索框）、在线 IDE 功能均使用 OIH 的服务。

OIH 的搜索功能很强大，其庞大的数据库拥有 $8000+$ 道中外主流 OJ 的题目。只需要只言片语（题目名称、描述、甚至是测试数据）就可以很快的搜出这道题的出处（“改编题”的杀手）。

以后，OIH 还将收录全网上的题解（数量是万级别的），只要通过题目名称，就可以搜到这个题目的题解。

OIH 的目标是制作出一个适合 OIers 的 app。虽然 OIH 不是 OJ，但是 OIH 的辅助功能能够给 OIers 带来很大的便利。如果您有任何的 idea，请 @soha（oih 的作者）。

## 题目描述

经过多次谈判，kkksc03 终于取得了 OIH 搜索的接口，现在向 OIH 服务器发送了一个查询请求，返回了一道题目。

查询关键词是一句话，由汉字（我们用它的拼音代替）、空格（每个汉字之间都有空格）、半角标点组成。返回的题目也是这样的。kkksc03 希望将题目中的某些词语加粗。所谓词语，就是连续两个或以上汉字。比如说 `tao tao`,`zhai ping guo` 都是词语，而 `hao`,`ni,hao` 不是词语。如果一个在题目中出现的词语也在关键词中出现的词语，那么加粗。加粗的方法是在词语前加上 `<b>`，在词语后面加上一个 `</b>`。至于标点符号么加不加粗无所谓。kkksc03 希望添加标记（`<b>` `</b>`对）的数量尽可能少，而且每个标记之间的的字符数最少。

## 说明/提示

每个汉字不超过 $7$ 个英文字符。

关键字不超过 $100$ 个汉字。

返回的题目不超过 $1000$ 个汉字。

## 题解思路与核心难点分析

### 核心逻辑
1. **词素拆分**：将输入字符串按汉字（连续字母）与非汉字（空格/标点）拆分成交替序列
2. **词语提取**：从 S1 中提取所有由两个连续汉字（中间无标点）组成的词语，建立哈希表
3. **标记匹配**：扫描 S2 中所有可能的二字词语，若存在于哈希表则标记起止位置
4. **区间合并**：将相邻或重叠的标记区间合并为极小区间，保证 `<b>` 标签最少

### 难点对比
| 题解特点               | rzh123 题解                | intconstlee 题解          |
|----------------------|---------------------------|--------------------------|
| 数据结构               | 结构体存储汉字+分隔符属性       | Map 存储二字词语            |
| 标点处理               | 显式检查分隔符中的标点         | 非字母字符自动分割词语         |
| 匹配方式               | 双循环暴力匹配              | 哈希表 O(1) 查询           |
| 合并策略               | 实时状态机合并             | 预标记数组后处理合并         |
| 代码复杂度             | 较高（结构体+多次遍历）       | 较低（线性扫描+位标记）       |

### 精炼结论
两个题解均采用**预处理-匹配-合并**的三段式策略。intconstlee 的解法通过哈希表优化匹配效率，通过位标记实现区间合并，更简洁高效。

## 最优思路提炼
1. **双指针词素拆分**  
   使用双指针扫描字符串，分离汉字与分隔符：
   ```python
   while i < len(s):
       if s[i]是字母: 记录至当前词素
       else: 记录至当前分隔符
   ```

2. **哈希表加速匹配**  
   将 S1 的合法二字词存入哈希表，匹配时直接查询：
   ```cpp
   map<string, int> mp;
   mp[lowercase(word1 + word2)] = 1;
   ```

3. **位标记合并算法**  
   使用标记数组记录词语起止点，通过位运算合并区间：
   ```cpp
   tag[start] += 1;  // 起始标记
   tag[mid] += 3;     // 中间标记
   tag[end] += 2;     // 结束标记
   ```

## 核心代码实现（intconstlee 题解关键片段）
```cpp
// 哈希表构建
for (遍历S1) {
    if (当前是汉字 && 前一词存在) 
        mp[小写拼接词] = 1;
}

// 标记匹配
for (遍历S2所有二字词) {
    if (mp中存在该词) 
        tag[start]++, tag[mid] +=3, tag[end]++;
}

// 标签合并输出
bool inTag = false;
for (每个词素) {
    if (tag[start] && !inTag) {
        输出 <b>;
        inTag = true;
    }
    输出词素;
    if (tag[end] && inTag) {
        输出 </b>;
        inTag = false;
    }
}
```

# 题目信息

# 欧几里德的游戏

## 题目描述

欧几里德的两个后代 Stan 和 Ollie 正在玩一种数字游戏，这个游戏是他们的祖先欧几里德发明的。给定两个正整数 $M$ 和 $N$，从 Stan 开始，从其中较大的一个数，减去较小的数的正整数倍，当然，得到的数不能小于 $0$。然后是 Ollie，对刚才得到的数，和 $M,N$ 中较小的那个数，再进行同样的操作……直到一个人得到了 $0$，他就取得了胜利。下面是他们用 $(25,7)$ 两个数游戏的过程：

- 初始：$(25,7)$；
- Stan：$(11,7)$；
- Ollie：$(4,7)$；
- Stan：$(4,3)$；
- Ollie：$(1,3)$；
- Stan：$(1,0)$。

Stan 赢得了游戏的胜利。

现在，假设他们完美地操作，谁会取得胜利呢？

## 说明/提示

$1 \leq C \leq 6$。

## 样例 #1

### 输入

```
2
25 7
24 15
```

### 输出

```
Stan wins
Ollie wins
```

## 综合分析与结论  
**核心思路**：博弈论与辗转相减法结合，通过分析两数之商判断必胜态。  
- **关键公式**：设两数为 `x > y`，若 `x/y >= 2`，当前玩家必胜；否则递归处理 `(y, x%y)` 并交换胜负。  
- **数学证明**：当 `x >= 2y` 时，玩家可通过调整操作将必败态转移给对手；否则胜负由递归奇偶性决定。  
- **可视化设计**：  
  - **动画流程**：展示两数相减过程，高亮当前商 `k = x/y`。若 `k >= 2` 时显示“必胜”，否则标记递归箭头。  
  - **复古像素风**：用 8-bit 方块表示数值，音效提示关键步骤（如商≥2时播放胜利音效）。

## 最优思路提炼  
**关键技巧**：  
1. **商≥2必胜**：若 `x >= 2y`，直接判定当前玩家胜。  
2. **奇偶性递归**：商=1时胜负由后续递归深度的奇偶性决定。  

**代码片段**（Special_zyy解法）：  
```cpp
bool solve(int x, int y) {
    if (x < y) swap(x, y);
    if (y == 0 || x / y >= 2) return true;
    return !solve(y, x % y);
}
```

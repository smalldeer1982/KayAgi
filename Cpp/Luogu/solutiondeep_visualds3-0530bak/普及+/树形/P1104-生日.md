# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

## 【综合分析与结论】  
**核心思路**：  
1. 使用结构体存储学生信息（姓名、年、月、日、输入顺序）。  
2. 自定义排序规则：先按年升序，年相同按月升序，月相同按日升序，日期相同则按输入顺序降序。  
3. 通过 `sort` 函数实现高效排序，时间复杂度为 O(n log n)。  

**关键难点**：  
- 多级条件比较的优先级顺序（年 > 月 > 日 > 输入顺序）。  
- 处理生日相同时需保证后输入者优先输出（通过记录输入顺序并逆序比较）。  

**可视化设计**：  
- **像素风格**：用不同颜色方块表示学生，方块内显示姓名和生日（如 "Luowen 1991-08-01"）。  
- **高亮比较**：当前比较的两个元素用闪烁边框标记，显示比较的关键字（如黄色高亮年份）。  
- **音效触发**：  
  - 比较时播放“滴”声，交换位置时播放“咔”声。  
  - 完成排序后播放 8-bit 胜利音效。  
- **自动演示**：AI 自动执行排序步骤，速度可调节（慢速观察细节或快速查看结果）。

## 【核心代码实现】  
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Student {
    string name;
    int year, month, day, idx;
};

bool cmp(const Student &a, const Student &b) {
    if (a.year != b.year) return a.year < b.year;
    if (a.month != b.month) return a.month < b.month;
    if (a.day != b.day) return a.day < b.day;
    return a.idx > b.idx; // 生日相同时后输入者优先
}

int main() {
    int n;
    cin >> n;
    Student stu[n];
    for (int i = 0; i < n; i++) {
        cin >> stu[i].name >> stu[i].year >> stu[i].month >> stu[i].day;
        stu[i].idx = i; // 记录输入顺序
    }
    sort(stu, stu + n, cmp);
    for (int i = 0; i < n; i++) 
        cout << stu[i].name << endl;
    return 0;
}
```

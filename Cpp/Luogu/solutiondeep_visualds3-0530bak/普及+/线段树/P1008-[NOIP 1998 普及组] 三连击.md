## 题目描述

将 $1, 2, \ldots , 9$ 共 $9$ 个数分成 $3$ 组，分别组成 $3$ 个三位数，且使这 $3$ 个三位数构成 $1 : 2 : 3$ 的比例，试求出所有满足条件的 $3$ 个三位数。

## 说明/提示

NOIP1998 普及组 第一题

## 题解思路与核心难点分析

### 核心算法流程
所有题解的核心逻辑均可归纳为：
1. **枚举第一个数**：根据比例关系，确定第一个数的范围（123 ≤ x ≤ 329）
2. **生成后两个数**：通过x*2和x*3得到后两个三位数
3. **数字查重**：验证三个数共用1-9且无重复

### 算法优化与难点对比
- **枚举范围推导**：通过数学推导缩小范围（x≤329确保x*3≤987）
- **查重方式**：
  - 数组标记法（O(1)空间）：通过长度为10的bool数组快速判断重复
  - 数学特性法：利用1-9的和=45、积=362880快速验证
- **数据生成**：
  - 直接计算各数位（/100、%10等操作）
  - 字符串转换后遍历字符

### 最优思路提炼
**Kelin的题解实现**：
1. 遍历第一个数i的范围192~327
2. 用a数组标记i、i*2、i*3三个数的所有数位
3. 统计1-9的出现次数，若总计9次则输出

## 最优代码实现
```cpp
#include<cstdio>
#include<cstring>
int main() {
    for(int i=192; i<=327; i++) {
        bool a[10]={0}; int v=0;
        a[i%10] = a[i/10%10] = a[i/100] 
            = a[i*2%10] = a[i*2/10%10] = a[i*2/100] 
            = a[i*3%10] = a[i*3/10%10] = a[i*3/100] = 1;
        for(int j=1;j<=9;j++) v += a[j];
        if(v==9) printf("%d %d %d\n",i,i*2,i*3);
    }
    return 0;
}
```

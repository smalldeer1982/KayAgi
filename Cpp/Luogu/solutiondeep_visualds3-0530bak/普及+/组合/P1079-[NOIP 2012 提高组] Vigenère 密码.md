# [NOIP 2012 提高组] Vigenère 密码

## 题目描述

16 世纪法国外交家 Blaise de Vigenère 设计了一种多表密码加密算法 Vigenère 密码。Vigenère 密码的加密解密算法简单易用，且破译难度比较高，曾在美国南北战争中为南军所广泛使用。

 
在密码学中，我们称需要加密的信息为明文，用 $M$ 表示；称加密后的信息为密文，用 $C$ 表示；而密钥是一种参数，是将明文转换为密文或将密文转换为明文的算法中输入的数据，记为 $k$。在 Vigenère 密码中，密钥 $k$ 是一个字母串，$k=k_1,k_2,…,k_n$。当明文 $M=m_1,m_2,…,m_n$ 时，得到的密文 $C=c_1,c_2,…,c_n$，其中 $c_i$=$m_i \operatorname{\circledR} k_i$，运算 $\circledR$ 的规则如下表所示：

Vigenère 加密在操作时需要注意：

1. $\circledR$ 运算忽略参与运算的字母的大小写，并保持字母在明文 $M$ 中的大小写形式；
2. 当明文 $M$ 的长度大于密钥 $k$ 的长度时，将密钥 $k$ 重复使用。

例如，明文 $M=\texttt{Helloworld}$，密钥 $k=\texttt{abc}$ 时，密文 $C=\texttt{Hfnlpyosnd}$。

$$

\textsf{密文} & \tt H & \tt f & \tt n & \tt l & \tt p & \tt y & \tt o & \tt s & \tt n & \tt d \\  \hline
\end{array}
$$

## 综合分析与结论

### 核心算法流程
1. **密钥循环匹配**：通过 `i % key_len` 实现密钥的循环使用
2. **字符转数值**：统一将字符转换为 0-25 的数值（`k[i]&31` 或 `toupper(c)-'A'`）
3. **位移计算**：明文 = (密文字符 - 密钥位移 + 26) % 26（+26 防止负数）
4. **大小写保留**：根据密文字符的原始大小写决定输出形式

### 解决难点
- **循环密钥处理**：通过取模运算简化密钥重复逻辑
- **跨大小写计算**：用位运算 `&31` 或统一转大写处理字母顺序
- **负数位移处理**：通过 `+26` 后取模保证结果在 0-25 范围内

### 可视化设计
- **动画方案**：  
  ![Vigenère 解密动画示意图](https://i.imgur.com/0wGkKkZ.gif)
  1. 显示密文、密钥两行字符，用不同颜色高亮当前处理位
  2. 展示字符转数值过程（如 `H→7`, `a→0`）
  3. 动态绘制 `(7-0)%26=7 → H` 的数值计算过程
  4. 循环指针 `i%key_len` 以红框在密钥行移动
- **复古风格**：  
  用 8-bit 像素字体，背景加载老式终端绿色网格，密钥指针移动时播放「滴滴」音效，每次解密成功播放「叮」的提示音

## 最优思路与技巧提炼

### 关键公式
**`明文 = (密文数值 - 密钥数值 + 26) % 26`**  
- **+26 的作用**：确保减法结果非负（如 `a(0)-b(1)=-1 → 25`）
- **%26 的作用**：处理超过 Z 的循环（如 `z(25)+1 → 0(a)`）

### 高效位运算
- **`char & 31`**：获取字母表位置（A-Z/a-z 后 5 位相同）
- **示例**：`'C' & 31 = 3`, `'c' & 31 = 3`

### 大小写处理
- **保留原始大小写**：先判断 `isupper(c)`，结果用 `'A'`/`'a'` 偏移量还原
- **统一计算**：转大写计算数值，最后恢复原始大小写

## 核心代码实现

### 极简版（ingu 思路）
```cpp
#include <iostream>
using namespace std;
int main() {
    string k, c;
    cin >> k >> c;
    for (int i = 0; i < c.length(); i++) {
        int t = (k[i % k.length()] & 31) - 1;
        c[i] = (c[i] & 31) - t > 0 ? c[i] - t : c[i] - t + 26;
    }
    cout << c << endl;
    return 0;
}
```

### 清晰版（Sinwind 思路）
```cpp
char decryption(char c, char k) {
    char temp_c = toupper(c);
    char temp_k = toupper(k);
    char base = isupper(c) ? 'A' : 'a';
    return base + (temp_c - temp_k + 26) % 26;
}
```

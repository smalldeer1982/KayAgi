# 题目信息

# 删数问题

## 题目描述

键盘输入一个高精度的正整数 $n$（不超过 $250$ 位），去掉其中任意 $k$ 个数字后剩下的数字按原左右次序将组成一个新的非负整数。编程对给定的 $n$ 和 $k$，寻找一种方案使得剩下的数字组成的新数最小。

## 说明/提示

用 $\operatorname{len}(n)$ 表示 $n$ 的**位数**，保证 $1 \leq k < \operatorname{len}(n) \leq 250$。

## 样例 #1

### 输入

```
175438 
4
```

### 输出

```
13```

### 综合分析与结论
#### 核心思路
各题解均采用贪心策略，核心思路为 **局部最优推导全局最优**：
1. **删除策略**：从左往右遍历，优先删除高位较大的数字（如"175438"中7>5，优先删7）
2. **递减区间处理**：若存在高位>低位的情况（如"365"中6>5），删除高位
3. **前导零处理**：最终结果需去除前导零，但保留至少一个零

#### 难点与解决方案
- **难点1**：如何高效找到应删除的位置
  - 解法：维护单调递增序列，遇到递减时删除前一个元素（如题解xh39的代码）
- **难点2**：处理全零或剩余零的情况
  - 解法：输出前跳过前导零，若全零则强制输出0（如题解KillerXu的调试经历）

### 最优思路提炼
1. **贪心选择**：在允许范围内（`当前位 + 剩余可删次数`）选最小数字
   ```python
   # 伪代码示例
   for i in 0 to len(num)-k:
       min_pos = find_min(num, start=i, end=i+k)
       result += num[min_pos]
       k -= (min_pos - i)
       i = min_pos + 1
   ```
2. **单调栈优化**：维护递增栈，遇到更小元素时弹出栈顶（时间复杂度O(n)）
   ```cpp
   stack<char> st;
   for (char c : num) {
       while (!st.empty() && st.top() > c && k > 0) {
           st.pop(); k--;
       }
       st.push(c);
   }
   ```

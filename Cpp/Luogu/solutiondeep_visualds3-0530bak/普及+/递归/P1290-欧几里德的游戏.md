# 题目信息

# 欧几里德的游戏

## 题目描述

欧几里德的两个后代 Stan 和 Ollie 正在玩一种数字游戏，这个游戏是他们的祖先欧几里德发明的。给定两个正整数 $M$ 和 $N$，从 Stan 开始，从其中较大的一个数，减去较小的数的正整数倍，当然，得到的数不能小于 $0$。然后是 Ollie，对刚才得到的数，和 $M,N$ 中较小的那个数，再进行同样的操作……直到一个人得到了 $0$，他就取得了胜利。下面是他们用 $(25,7)$ 两个数游戏的过程：

- 初始：$(25,7)$；
- Stan：$(11,7)$；
- Ollie：$(4,7)$；
- Stan：$(4,3)$；
- Ollie：$(1,3)$；
- Stan：$(1,0)$。

Stan 赢得了游戏的胜利。

现在，假设他们完美地操作，谁会取得胜利呢？

## 说明/提示

$1 \leq C \leq 6$。

## 样例 #1

### 输入

```
2
25 7
24 15
```

### 输出

```
Stan wins
Ollie wins
```

## 题解思路与核心难点
### 核心思路
1. **必胜态判定**：若当前较大的数是较小数的两倍以上（`max ≥ 2*min`），则当前操作者必胜。因为可以通过调整策略将必败态转移给对手。
2. **辗转相减过程**：若无法直接判定必胜态，则进入相减循环，胜负由步骤数的奇偶性决定。

### 解决难点
- **博弈树剪枝**：当出现 `max ≥ 2*min` 时，无需继续递归，直接判定胜负。
- **奇偶性判断**：若所有步骤中未触发必胜态，胜负由操作次数的奇偶性决定。

## 代码实现（递归法）
```cpp
#include <iostream>
using namespace std;

bool solve(int x, int y) {
    if (x < y) swap(x, y);
    if (y == 0) return false;
    if (x / y >= 2) return true;
    return !solve(y, x % y);
}

int main() {
    int T, m, n;
    cin >> T;
    while (T--) {
        cin >> m >> n;
        cout << (solve(m, n) ? "Stan wins" : "Ollie wins") << endl;
    }
    return 0;
}
```

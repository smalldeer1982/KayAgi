# 题目信息

# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

## 题解思路与难点分析
### 核心思路
1. **数据结构选择**：使用结构体存储姓名、年、月、日及输入序号
2. **比较规则**：
   - 优先比较年份（数值越小年龄越大）
   - 年份相同比较月份
   - 年月相同比较日期
   - 生日相同则比较输入顺序（后输入的优先）
3. **排序实现**：利用STL的sort函数配合自定义比较规则

### 解决难点
1. **多级比较**：通过嵌套条件判断实现年→月→日的优先级顺序
2. **稳定排序**：通过显式比较输入序号而非依赖稳定排序，保证相同生日时的正确顺序
3. **输入处理**：在读取数据时记录输入顺序作为最终比较依据

## 最优思路提炼
**关键技巧**：  
1. **四元组比较法**：`year*1e4 + month*1e2 + day`生成可比较整数  
2. **输入序号倒序**：在相同生日时用`输入序号 >`实现后输入优先  
3. **稳定排序优化**：当数据量大时改用stable_sort保持顺序稳定性

**代码实现示例**：
```cpp
struct Student {
    string name;
    int y, m, d, id;
};

bool cmp(const Student& a, const Student& b) {
    if(a.y != b.y) return a.y < b.y;
    if(a.m != b.m) return a.m < b.m; 
    if(a.d != b.d) return a.d < b.d;
    return a.id > b.id; // 后输入优先
}

// 使用示例
sort(students, students+n, cmp);
```

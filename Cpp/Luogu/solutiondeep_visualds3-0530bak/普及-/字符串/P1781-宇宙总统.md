# 宇宙总统

## 题目描述

地球历公元 6036 年，全宇宙准备竞选一个最贤能的人当总统，共有 $n$ 个非凡拔尖的人竞选总统，现在票数已经统计完毕，请你算出谁能够当上总统。


## 说明/提示

票数可能会很大，可能会到 $100$ 位数字。

$1 \leq n \leq 20$。

## 样例 #1

### 输入

```
5
98765
12365
87954
1022356
985678
```

### 输出

```
4
1022356```

# 宇宙总统题解分析

## 算法分类
字符串比较 / 自定义排序规则

## 题解核心思路与难点对比
### 核心逻辑
1. **字符串长度优先**：位数多的数字更大
2. **字典序比较**：位数相同时，逐字符比较ASCII码
3. **实时更新最大值**：遍历过程中维护当前最大票数和对应编号

### 关键难点
1. **字典序陷阱**：字符串比较时，"9" > "89"（需先保证长度相同）
2. **结构体排序**：需同时存储编号和长度信息
3. **空间优化**：部分解法误用全局变量导致命名冲突

## 高星题解推荐（≥4★）
1. **callG（5★）**  
   - 实时维护最大值，O(n)时间复杂度
   - 单次遍历无需排序，空间复杂度O(1)
   - 代码示例：
     ```cpp
     if (inSize > maxSize || (inSize == maxSize && in > max)) {
         max = in;
         id = i;
     }
     ```

2. **right_cat（4.5★）**  
   - 结构体存储长度和编号
   - 自定义排序函数，逻辑清晰
   - 代码示例：
     ```cpp
     bool cmp(node a,node b) {
         if(a.lenx > b.lenx) return 1;
         if(a.lenx == b.lenx && a.x > b.x) return 1;
         return 0;
     }
     ```

3. **wuliang1（4★）**  
   - 最短实现（仅15行）
   - 直接操作全局变量，牺牲可读性换简洁
   - 代码示例：
     ```cpp
     if((strlen(max)<strlen(a))||(strlen(a)==strlen(max)&&strcmp(max,a)<0))
     ```

## 最优思路提炼
1. **双条件筛选**：先比长度再比字典序
2. **字符串特性**：利用string类的`size()`和`operator>`直接比较
3. **结构体优化**：存储预处理长度提升比较效率

## 核心代码实现
```cpp
// callG解法核心片段
string max = "";
for (int i = 1; i <= n; i++) {
    cin >> in;
    if (in.size() > max.size() || 
       (in.size() == max.size() && in > max)) {
        max = in;
        id = i;
    }
}
```

## 扩展思考
当数据量增大到10^5时，基数排序（如arfa的解法）时间复杂度O(nk)可能更优，其中k为最大位数。但对于本题n≤20的规模，O(n)遍历是最佳选择。

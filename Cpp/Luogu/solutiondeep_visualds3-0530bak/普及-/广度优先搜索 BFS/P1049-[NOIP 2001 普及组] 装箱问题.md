# 题目信息

# [NOIP 2001 普及组] 装箱问题

## 题目描述

有一个箱子容量为 $V$，同时有 $n$ 个物品，每个物品有一个体积。


现在从 $n$ 个物品中，任取若干个装入箱内（也可以不取），使箱子的剩余空间最小。输出这个最小值。

## 说明/提示

对于 $100\%$ 数据，满足 $0<n \le 30$，$1 \le V \le 20000$。

**【题目来源】**

NOIP 2001 普及组第四题

## 样例 #1

### 输入

```
24
6
8
3
12
7
9
7
```

### 输出

```
0
```

## 综合分析与结论
### 核心算法流程
所有题解的核心思路均为将问题转化为 01 背包模型，核心算法流程如下：
1. **状态定义**：`f[j]` 表示容量为 `j` 的箱子能装入的最大体积
2. **状态转移**：`f[j] = max(f[j], f[j-w[i]]+w[i])`
3. **逆向遍历**：从 `V` 到 `w[i]` 逆序更新，防止重复计算
4. **最终答案**：`V - f[V]`

### 可视化设计思路
1. **动态规划过程动画**：在网格中展示 `f[]` 数组更新过程，高亮当前处理的物品和正在更新的容量值
2. **颜色标记**：
   - 红色：当前正在更新的 `f[j]`
   - 黄色：被引用的 `f[j-w[i]]`
   - 绿色：已确定的最优解
3. **步进控制**：允许单步执行物品循环和容量循环，观察每个物品对 `f[]` 的影响

## 最优思路与代码实现
### 核心代码片段
```cpp
int f[20010];
for(int i=1; i<=n; i++){
    for(int j=V; j>=w[i]; j--){
        f[j] = max(f[j], f[j-w[i]]+w[i]);
    }
}
cout << V - f[V];
```

### 完整代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int V, n, w[35], f[20005];

int main() {
    cin >> V >> n;
    for(int i=1; i<=n; i++) cin >> w[i];
    for(int i=1; i<=n; i++)
        for(int j=V; j>=w[i]; j--)
            f[j] = max(f[j], f[j-w[i]]+w[i]);
    cout << V - f[V];
    return 0;
}
```

# 题目信息

# [语言月赛202209] 零食售卖点

## 题目描述

洛咕公司有一条无限长的大道，可以被视为一个数轴。在这条大道上，一共有 $k$ 个零食售卖点，第 $i$ 个零食售卖点的坐标为 $a_i$。

Aya 非常热爱零食，他想知道，**相邻**两个零食售卖点之间的距离最大为多少。第 $i$ 个零食售卖点和第 $j$ 个零食售卖点之间的距离为 $|a_i-a_j|$。

$|x|$ 代表 $x$ 的绝对值。即，当 $x\ge0$，$|x|=x$；当 $x<0$，$|x|=-x$。例如，$|4|=4,|0|=0,|-1|=1$。

## 说明/提示

### 输入输出样例解释

共 $4$ 个零食售卖点，按数轴上的顺序依次为 $1,3,5,20$，相邻两个零食售卖点的距离依次为 $2,2,15$，答案为 $15$。

### 数据规模与约定

对于 $30\%$ 的数据，$k=2$；  
对于另外 $30\%$ 的数据，$a_i$ 升序给出；  
对于 $100\%$ 的数据，$2 \le k \le 5000, 1 \le a_i \le 10^7$。

## 样例 #1

### 输入

```
4
3 5 1 20```

### 输出

```
15```

## 题解分析与结论

### 核心思路与难点
题目要求找到排序后相邻零食点的最大间隔。核心步骤：
1. **正确排序**：将无序输入的坐标转化为有序序列
2. **遍历比较**：计算相邻元素差值的最大值

难点在于：
- 排序方法的效率选择（数据规模 5,000）
- 边界处理（所有点相同的情况）

### 题解对比与评分
**题解评分**（仅展示≥4星）：
1. 库函数sort法 ★★★★★
   - 直接调用STL快速排序，时间复杂度O(n log n)
   - 代码简洁高效，3行核心逻辑完成排序与遍历
   - 完整代码示例：
     ```cpp
     #include <iostream>
     #include <algorithm>
     using namespace std;
     int a[5005], n, maxd;
     int main() {
         cin >> n;
         for(int i=0; i<n; i++) cin >> a[i];
         sort(a, a+n);
         for(int i=1; i<n; i++)
             maxd = max(maxd, a[i]-a[i-1]);
         cout << maxd;
         return 0;
     }
     ```

## 核心代码实现
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int k, a[5005], max_diff = 0;
    cin >> k;
    for(int i=0; i<k; ++i) cin >> a[i];
    sort(a, a+k);
    for(int i=1; i<k; ++i)
        max_diff = max(max_diff, a[i] - a[i-1]);
    cout << max_diff;
    return 0;
}
```
关键点：利用STL sort快速排序，线性遍历维护最大值。

# 题目信息

# [CSP-X2021 山东] 成绩排名

## 题目描述

达米奇同学的学校组织信息学竞赛模拟，发现了成绩并列如何排名是个难题。例如有四名选手的成绩分别为 $50,80,50,30$ 分，则 $80$ 分的选手为第一名，$50$ 分的两名选手均为第二名，$30$ 分的选手为第四名。老师请达米奇同学编写一个程序，计算一个选手在这种排名方式之下的名次（分数高的选手排前面）。

## 说明/提示

【数据范围和限制】

对于 $30\%$ 的数据，$N\leq 100$。

对于 $60\%$ 的数据，$N\leq 1000$。

对于 $100\%$ 的数据，$N\leq 10^4$。

对于 $100\%$ 的数据，每名选手的分数范围为 $[0,100]$。

## 样例 #1

### 输入

```
4
50 80 50 30
50```

### 输出

```
2```

### **综合分析与结论**
**核心思路**  
题目要求计算特定分数在并列排名中的名次。核心在于确定所有分数中比目标分数大的不同分数的数量，加1即为名次。例如，目标分数为50，比它大的分数只有80，则名次为2。

**算法流程**  
1. **降序排序**：将所有分数降序排列，使高分在前。  
2. **遍历统计**：遍历排序后的数组，统计不同分数的数量。每遇到新分数，名次加1。  
3. **匹配目标**：当遍历到目标分数时，当前名次即为最终结果。

**关键难点**  
- **重复分数处理**：相同分数共享同一名次，需跳过重复项统计不同分数。  
- **高效定位名次**：通过排序和遍历，直接统计不同分数，无需额外去重操作。

**可视化设计**  
1. **排序动画**：将输入的分数降序排列，以不同颜色块区分不同分数段。  
2. **遍历过程**：用箭头或高亮标记当前遍历的元素，动态更新名次计数器。  
3. **名次反馈**：当遇到目标分数时，名次高亮显示，并触发音效。  
- **复古像素风格**：用16色调色板渲染分数块，箭头移动时播放“步进”音效，命中目标时播放经典8位胜利音效。

### **最优代码与核心逻辑**
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int n, x;
    cin >> n;
    vector<int> scores(n);
    for (int i = 0; i < n; ++i) cin >> scores[i];
    cin >> x;
    
    sort(scores.begin(), scores.end(), greater<int>());
    int rank = 0, prev = -1;
    for (int num : scores) {
        if (num != prev) {
            rank++;
            prev = num;
        }
        if (num == x) {
            cout << rank;
            return 0;
        }
    }
    return 0;
}
```
**核心逻辑**：排序后遍历，动态更新名次计数器，命中目标时终止。

# 题解思路与难点分析  
## 核心逻辑  
题目本质是 **遍历数组找最大值**，附带条件筛选（总和>8）和相同值优先取最前项。所有题解均采用线性遍历而非传统 BFS/DFS，但属于广义的暴力搜索范畴。

## 搜索算法核心要点  
1. **遍历顺序**：按周一到周日的顺序扫描，确保相同最大值时记录最早出现的天数。  
2. **剪枝条件**：仅当 `a+b > 8` 时才参与比较，且需严格大于当前最大值才更新记录。  
3. **数据结构**：仅需两个变量 `max_time` 和 `result_day`，无需复杂结构。

## 难点对比  
| 题解特点               | 封禁用户题解            | cff_0102题解             | zhuoqizhi题解           |
|-----------------------|-------------------------|--------------------------|-------------------------|
| 初始最大值设定         | 0（需额外条件判断）     | 8（直接过滤<=8的情况）   | 8（逻辑更直接）         |
| 更新条件               | `s>max && s>8`          | `mx < a+b`               | `max < t1+t2`           |
| 相同值处理             | 自然保留最早天数        | 自然保留最早天数         | 自然保留最早天数        |
| 空间复杂度            | O(1)                   | O(1)                    | O(1)                   |

# 最优思路提炼  
1. **初始化技巧**：将 `max` 初始化为8，避免无效数据的干扰。  
2. **严格大于更新**：仅当新值严格大于当前 `max` 时更新，保证最早出现的天数被保留。  
3. **空间优化**：无需存储全部数据，实时计算并比较。

# 核心代码实现  
**最优解（cff_0102 题解）**  
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int mx=8, pos=0;
    for(int i=1; i<=7; i++){
        int a,b; cin>>a>>b;
        if(mx < a+b) mx = a+b, pos = i;
    }
    cout << pos;
    return 0;
}
```

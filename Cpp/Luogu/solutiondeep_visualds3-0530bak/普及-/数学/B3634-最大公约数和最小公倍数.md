# 最大公约数和最小公倍数

## 题目描述

给定两个正整数 $a,b$，求他们的最大公约数（gcd）和最小公倍数（lcm）。这两个整数均在 int 范围内。

## 样例 #1

### 输入

```
6 15```

### 输出

```
3 30```

### **题解思路与核心难点**  
**核心思路**  
1. **最大公约数 (gcd)**：采用欧几里得算法（辗转相除法），基于公式 $\gcd(a, b) = \gcd(b, a \bmod b)$，递归或迭代求解。  
2. **最小公倍数 (lcm)**：利用公式 $\text{lcm}(a, b) = \frac{a \times b}{\gcd(a, b)}$，依赖已求得的 gcd 值。  

**解决难点**  
- **公式证明**：需严格证明辗转相除法的正确性（如公约数集合不变性）和 lcm 公式的合理性（如短除法分解质因数）。  
- **实现优化**：避免中间计算溢出（如先除后乘），并处理输入为 $0$ 的特殊情况。

### **代码实现**  
```cpp
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

int lcm(int a, int b) {
    return a / gcd(a, b) * b;  // 先除后乘避免溢出
}

int main() {
    int a, b;
    cin >> a >> b;
    int g = gcd(a, b);
    cout << g << " " << lcm(a, b) << endl;
    return 0;
}
```

### **个人心得**  
- **调试经验**：若先乘后除，$a \times b$ 可能溢出，需用 `long long` 或调整计算顺序。  
- **思维误区**：无需预先判断 $a > b$，因第一次取模会自动交换（如 $a=3$, $b=6$ 时，$3 \bmod 6 = 3$，下一轮变为 $\gcd(6, 3)$）。

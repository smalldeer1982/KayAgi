# 题目信息

# [信息与未来 2019] 鸡兔同笼

## 题目描述

有三种动物被关在笼子里，它们分别是：
- 鸡，每只鸡有一个头、两只脚；
- 三脚猫，每只三脚猫有一个头，三只脚；
- 兔子，每只兔子有一个头，四只脚。

现在知道笼子中一共有 $x$ 个头、$y$ 只脚，但这个信息并不足以推导出鸡、兔和三脚猫的准确数量。这时候就需要编程来帮忙了——请你求出笼子里分别至少、至多有几只兔子。

## 说明/提示

对于 $50\%$ 的测试数据，有 $1\le x\le 10^6$。

对于 $100\%$ 的测试数据，有 $1\le x\le10^9,1\le T\le20$。
> 本题原始满分为 $20\text{pts}$。

## 综合分析与结论
题目通过建立线性方程组，结合变量的非负约束条件，推导出兔子的数量范围。核心公式：

- 最小兔子数：r_min = max(y - 3x, 0)
- 最大兔子数：r_max = (y - 2x) // 2

**关键难点**在于处理整数解约束与不等式条件。通过消元法将三变量问题简化为单变量的边界分析，结合数学推导得出公式。

## 题解思路与代码实现
### 核心数学逻辑
1. **变量消元**：通过联立方程组消去鸡和三脚猫的数量，得到仅含兔子数量的约束条件。
2. **边界推导**：利用非负性条件确定兔子的上下界。
3. **范围验证**：检查总脚数是否在合理区间（2x ≤ y ≤4x），否则无解。

### 代码片段
```python
T = int(input())
for _ in range(T):
    x, y = map(int, input().split())
    if y < 2 * x or y > 4 * x:
        print(0, 0)
    else:
        r_min = max(y - 3 * x, 0)
        r_max = (y - 2 * x) // 2
        print(r_min, r_max)
```

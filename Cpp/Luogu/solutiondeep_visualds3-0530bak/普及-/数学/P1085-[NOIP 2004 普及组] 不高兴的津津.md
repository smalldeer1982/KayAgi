### 题解思路与算法要点  
**核心逻辑**：遍历 7 天的数据，维护两个变量：  
1. **最大值**：记录当前超过 8 小时的最大总时间  
2. **天数**：记录该最大值对应的最早出现天数  

**关键点**：  
- 只有总时间 **严格大于 8** 时才考虑  
- 遍历顺序必须为 **正序**（周一至周日），确保最早出现的最大值被记录  
- 判断条件为 **总时间 > 当前最大值**（严格大于），避免后续相同值覆盖天数  

**解决难点**：  
- 需要正确处理多个相同最大值的情况（取最早一天）  
- 初始化最大值的阈值（如设为 8），避免无效更新

### 最优思路与技巧提炼  
1. **初始化技巧**：将最大值初始化为 8，直接跳过总时间 ≤8 的天数  
2. **正序遍历**：确保最早出现的最大值被记录  
3. **严格比较**：仅当总时间严格大于当前最大值时更新，保证最早性  
4. **变量设计**：用 `day` 或 `pos` 记录天数，避免数组存储全部数据  

**代码片段**：  
```cpp
int max_val = 8, day = 0;
for (int i = 1; i <= 7; i++) {
    int a, b;
    cin >> a >> b;
    if (a + b > max_val) {
        max_val = a + b;
        day = i;
    }
}
cout << day;
```

### 个人心得摘录  
- **严格比较陷阱**：若用 `>=` 更新最大值，会覆盖为最后一天，必须用 `>`

- **输入顺序**：正序处理天数，避免倒序逻辑错误

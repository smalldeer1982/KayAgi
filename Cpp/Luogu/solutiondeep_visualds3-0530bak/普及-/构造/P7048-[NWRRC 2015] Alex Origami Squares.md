# 题目信息

# [NWRRC 2015] Alex Origami Squares

## 题目描述

有一张矩形纸，要求将其裁成三个全等的正方形，且每一个正方形的侧面都与纸张的侧面平行，问裁成的小正方形边长最大是多少。

## 样例 #1

### 输入

```
210 297
```

### 输出

```
105.0
```

## 样例 #2

### 输入

```
250 100
```

### 输出

```
83.333333
```

### **综合分析与结论**
#### **核心思路**
题目要求将矩形分割为三个全等的正方形，求最大边长。通过分析矩形长宽比例，确定三种排列方式对应的最大边长：
1. **长 ≥ 3×宽**：正方形边长为宽，三个并排。
2. **1.5×宽 ≤ 长 < 3×宽**：边长为长/3，三个并排但长边被充分利用。
3. **长 < 1.5×宽**：边长为宽/2，两行排列（如两个在左列，一个在右侧）。

#### **关键难点**
- 推导不同长宽比例下的最优排列方式。
- 确保代码正确处理所有边界条件（如交换长宽、浮点数精度）。

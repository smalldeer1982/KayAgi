# 「DROI」Round 2 构造与取模

## 题目背景

与其编写苍白无力的背景，不如出更有质量的题。



## 题目描述

对于整数 $n,k$，若存在**非负整数** $x$ 与 **正整数** $y$ 满足：

$$x + y = n\ \land\ x \bmod y =k$$

则我们称有序数对 $(x,y)$ 是 $n$ 的一个 **优秀拆分**（其中 $\land$  是**并且**的意思）。

现给定非负整数 $n,k$，请你构造**任意**一组 $n$ 的优秀拆分，并分别输出你构造方案中的 $x$ 和 $y$。特殊地，若不存在这样的拆分，则输出 `-1`。

## 说明/提示

#### 样例解释

对于第一组数据，只存在唯一的构造方式。

对于第二组数据，$(3,10)$ 也是一组合法的构造。

对于第三组数据，可以证明不存在一组合法的构造。

------------

#### 数据范围

**「本题采用捆绑测试」**

- $\operatorname{Subtask} 1(20\%)$：$n \leq 10^6$。

- $\operatorname{Subtask} 2(40\%)$：$n \leq 10^{12}$。

- $\operatorname{Subtask} 3(40\%)$：无特殊限制。

对于 $100\%$ 的数据：$T \leq 5$，$0 \leq n,k \leq 10^{18}$。

## **综合分析与结论**
**核心思路**  
通过数学推导构造满足条件的x和y，关键在于发现当且仅当n > 2k时，x = k且y = n - k的解满足条件。此时余数k必小于除数y，且x + y = n。若n ≤ 2k，则无解。

**难点与解决**  
- **难点**：如何快速判断解的存在性并构造解。  
- **解决**：利用模运算性质，推导出解存在的充要条件是n > 2k，直接构造x=k和y=n-k。

**可视化设计**  
- **动画方案**：显示n和k的输入值，动态计算并高亮条件n > 2k的判断过程。若成立，展示x=k和y=n-k的构造结果；否则显示-1。  
- **复古像素风格**：用8位像素字体显示数值，条件判断时使用闪烁特效，成功构造时播放清脆音效，失败则低音提示。

## **最优思路提炼**
**关键技巧**  
- **构造法**：直接令x=k，y=n-k，利用模运算性质（余数k必小于除数y）推导出解存在的条件n > 2k。  
- **数学优化**：将问题转化为简单的不等式判断，时间复杂度O(1)。

**代码实现**  
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    scanf("%d", &T);
    while (T--) {
        long long n, k;
        scanf("%lld%lld", &n, &k);
        if (n > 2 * k) printf("%lld %lld\n", k, n - k);
        else printf("-1\n");
    }
    return 0;
}
```

# 题目信息

# [NOIP 2015 提高组] 神奇的幻方

## 题目背景

NOIp2015 提高组 Day1T1


## 题目描述

幻方是一种很神奇的 $N\times N$ 矩阵：它由数字 $1,2,3,\cdots \cdots ,N \times N$ 构成，且每行、每列及两条对角线上的数字之和都相同。

当 $N$ 为奇数时，我们可以通过下方法构建一个幻方：

首先将 $1$ 写在第一行的中间。

之后，按如下方式从小到大依次填写每个数 $K \ (K=2,3,\cdots,N \times N)$ ：

1. 若 $(K-1)$ 在第一行但不在最后一列，则将 $K$ 填在最后一行， $(K-1)$ 所在列的右一列；
2. 若 $(K-1)$ 在最后一列但不在第一行，则将 $K$ 填在第一列， $(K-1)$ 所在行的上一行；
3. 若 $(K-1)$ 在第一行最后一列，则将 $K$ 填在 $(K-1)$ 的正下方；
4. 若 $(K-1)$ 既不在第一行，也不在最后一列，如果 $(K-1)$ 的右上方还未填数，则将 $K$ 填在 $(K-1)$ 的右上方，否则将 $K$ 填在 $(K-1)$ 的正下方。

现给定 $N$ ，请按上述方法构造 $N \times N$ 的幻方。

## 说明/提示

对于 $100\%$ 的数据，对于全部数据， $1 \leq N \leq 39$ 且 $N$ 为奇数。

## **题解思路、算法要点与解决难点**  

### **核心思路**  
所有题解均基于题目描述的**规则模拟**，直接按照四种条件判断当前数字的位置并更新坐标。关键在于**高效处理条件顺序**与**避免重复查找K-1的位置**。  

### **实现对比**  
1. **坐标跟踪 vs 暴力查找**  
   - **高效解法（如 GSQ0829、ClV_Csy）**：维护当前坐标 `(x,y)`，直接根据规则更新坐标，时间复杂度为 $O(N^2)$。  
   - **低效解法（如 Kawaii_qiuw、prh_rpjiajia）**：每次通过遍历二维数组查找 `K-1` 的位置，时间复杂度为 $O(N^3)$，但通过优化改进为 $O(N^2)$。  

2. **条件判断顺序**  
   - 高赞题解优先处理**“右上方是否填数”**的条件（如 `if (a[x-1][y+1])`），避免冗余判断。  
   - 部分题解（如 JCT_addyi）因条件顺序错误导致逻辑漏洞。  

3. **边界处理**  
   - 多数解法通过 `x = (x-1 + n) % n` 或显式赋值（如 `x = n`）处理越界，保证坐标在合法范围内。  

### **解决难点**  
- **坐标更新的无回溯性**：需确保每次更新后的坐标唯一且正确，避免重复填充或遗漏。  
- **右上方判断的优先级**：若先处理其他条件，可能无法正确触发“右上方未填数”的分支。

## **最优思路或技巧提炼**  

### **核心技巧**  
1. **坐标跟踪法**：始终维护当前数字的坐标 `(x,y)`，避免暴力查找。  
2. **条件优先级**：优先判断“右上方是否填数”，再处理其他边界条件。  
3. **越界处理**：通过赋值而非取模处理越界（如 `x = n`），更直观高效。  

### **代码片段**  
```cpp
int x = 1, y = (n + 1) / 2;
for (int k = 1; k <= n*n; k++) {
    a[x][y] = k;
    if (x == 1 && y == n) x++;
    else if (a[x-1][y+1]) x++;
    else if (x == 1) { x = n; y++; }
    else if (y == n) { x--; y = 1; }
    else { x--; y++; }
}
```

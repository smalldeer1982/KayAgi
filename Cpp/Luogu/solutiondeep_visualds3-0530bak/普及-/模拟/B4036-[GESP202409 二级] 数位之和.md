# 题目信息

# [GESP202409 二级] 数位之和

## 题目描述

小杨有 $n$ 个正整数，他认为一个正整数是美丽数字当且仅当该正整数每一位数字的总和是 $7$ 的倍数。

小杨想请你编写一个程序判断 $n$ 个正整数哪些是美丽数字。

## 说明/提示

### 数据规模与约定

对全部的测试数据，保证 $1 \leq n \leq 10^5$，$1 \leq a_i \leq 10^5$。

## 样例 #1

### 输入

```
3
7
52
103```

### 输出

```
Yes
Yes
No```

2. **综合分析与结论**

**关键难点**在于正确实现数字分解逻辑（如处理连续取余/整除的边界条件），以及处理大规模数据时的效率问题（需保证O(1)空间复杂度）。

4. **核心代码实现**  
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, a;
    cin >> n;
    while (n--) {
        cin >> a;
        int sum = 0;
        while (a > 0) { // 分解数位核心逻辑
            sum += a % 10;
            a /= 10;
        }
        cout << (sum % 7 ? "No" : "Yes") << endl;
    }
    return 0;
}
```

7. **可视化 JS 核心逻辑**

class DigitAnimator {
    constructor(canvas) {
        this.grid = new PixelGrid(canvas, 16); // 16x16像素风格
        this.speed = 500;
    }

    async animate(num) {
        let sum = 0;
        let step = 0;
        while (num > 0) {
            const digit = num % 10;
            this.highlightDigit(step, digit); // 高亮当前数位
            sum += digit;
            num = Math.floor(num / 10);
            await this.delay(this.speed);
            step++;
        }
        this.showResult(sum % 7 === 0);
    }

    highlightDigit(step, digit) {
        this.grid.drawBlock(step, 0, COLORS.HIGHLIGHT); // 步数作为x坐标
        this.grid.drawText(step, 1, `${digit}`, COLORS.TEXT);
    }
}
```

9. **教学建议**

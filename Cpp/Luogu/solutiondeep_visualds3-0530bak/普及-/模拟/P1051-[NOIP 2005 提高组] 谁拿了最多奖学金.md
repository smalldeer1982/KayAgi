# 题目信息

# [NOIP 2005 提高组] 谁拿了最多奖学金

## 题目描述

某校的惯例是在每学期的期末考试之后发放奖学金。发放的奖学金共有五种，获取的条件各自不同：

1. 院士奖学金，每人 $8000$ 元，期末平均成绩高于 $80$ 分（$>80$），并且在本学期内发表$1$篇或$1$篇以上论文的学生均可获得；
2. 五四奖学金，每人 $4000$ 元，期末平均成绩高于 $85$ 分（$>85$），并且班级评议成绩高于 $80$ 分（$>80$）的学生均可获得；
3. 成绩优秀奖，每人 $2000$ 元，期末平均成绩高于 $90$ 分（$>90$）的学生均可获得；
4. 西部奖学金，每人 $1000$ 元，期末平均成绩高于 $85$ 分（$>85$）的西部省份学生均可获得；
5. 班级贡献奖，每人 $850$ 元，班级评议成绩高于 $80$ 分（$>80$）的学生干部均可获得；

只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金。例如姚林的期末平均成绩是 $87$ 分，班级评议成绩 $82$ 分，同时他还是一位学生干部，那么他可以同时获得五四奖学金和班级贡献奖，奖金总数是 $4850$ 元。

现在给出若干学生的相关数据，请计算哪些同学获得的奖金总数最高（假设总有同学能满足获得奖学金的条件）。


## 说明/提示

**【数据范围】**

对于 $100\%$ 的数据，满足 $1 \le  N \le  100$。

## 综合分析与结论

### 核心思路  
题目要求根据多个条件计算每位学生的奖学金总额，核心难点在于：  
1. **多条件独立判断**：五个奖学金条件相互独立，需用多个独立的 `if` 判断而非 `else if`  
2. **数据存储与比较**：需记录最高奖学金学生的姓名（若并列取输入顺序最早者）及总奖学金  
3. **输入格式处理**：字符串与混合类型数据的正确读取  

### 最优解法要点  
1. **打擂台法**：遍历时直接维护最大值及其姓名，无需排序（时间复杂度 O(n)）  
2. **条件表达式优化**：将每个条件转化为逻辑表达式，直接参与数值计算（如 `(a>80&&e)*8000`）  
3. **输入处理简化**：题目数据规模小，用 `cin` 即可，无需复杂输入优化  

### 可视化设计思路  
1. **动画流程**：  
   - **学生卡片**：像素风格卡片展示姓名、成绩、干部/西部身份、论文数  
   - **条件高亮**：逐个高亮奖学金条件，满足条件时触发金币掉落动画  
   - **累加过程**：右侧面板实时更新当前学生各项奖学金金额及总额  
   - **擂台比较**：顶部展示当前最高奖学金学生，当新学生超越时触发像素爆炸特效  
2. **复古交互**：  
   - **音效**：满足条件时播放 8-bit 金币音效，更新最高纪录时播放升级音效  
   - **像素风格**：使用 16 色调色板，界面元素以方块拼接（如 FC 游戏《勇者斗恶龙》物品栏）

## 核心代码实现

### 打擂台法实现（qhr2023 优化版）
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, total = 0, max_sum = -1;
    string max_name;
    cin >> n;
    
    while (n--) {
        string name;
        int score, class_score, papers;
        char leader, west;
        cin >> name >> score >> class_score >> leader >> west >> papers;
        
        int current = (score > 80 && papers >= 1) * 8000 +
                     (score > 85 && class_score > 80) * 4000 +
                     (score > 90) * 2000 +
                     (score > 85 && west == 'Y') * 1000 +
                     (class_score > 80 && leader == 'Y') * 850;
        
        if (current > max_sum) {
            max_sum = current;
            max_name = name;
        }
        total += current;
    }
    
    cout << max_name << "\n" << max_sum << "\n" << total;
    return 0;
}
```

# 题目信息

# 【深基7.例9】最厉害的学生

## 题目描述

现有 $N$ 名同学参加了期末考试，并且获得了每名同学的信息：姓名（不超过 $8$ 个字符的仅有英文小写字母的字符串）、语文、数学、英语成绩（均为不超过 $150$ 的自然数）。总分最高的学生就是最厉害的，请输出最厉害的学生各项信息（姓名、各科成绩）。如果有多个总分相同的学生，输出靠前的那位。

## 说明/提示

数据保证，$1 \leq N \leq 1000$，姓名为长度不超过 $8$ 的字符串，语文、数学、英语成绩均为不超过 $150$ 的自然数。

## 样例 #1

### 输入

```
3
senpai 114 51 4
lxl 114 10 23
fafa 51 42 60```

### 输出

```
senpai 114 51 4```

## 综合分析与结论

### 核心思路
题目要求找出总分最高且最早出现的同学。所有题解均采用线性扫描策略，核心差异在于数据存储方式（结构体 vs 数组）和遍历方式（全存储后扫描 vs 边输入边比较）。

### 算法流程
1. **数据存储**：使用结构体或数组保存姓名及三科成绩
2. **总分计算**：遍历时累加三科成绩
3. **最大值维护**：每次遇到更高总分时更新最大值及其索引
4. **结果输出**：直接输出最高分对应学生的原始数据

### 可视化设计
- **像素动画**：在 Canvas 网格中，每个学生显示为像素方块，包含姓名缩写和总分。当前处理的学生用黄色边框，最高分用绿色填充。
- **步进演示**：通过方向键控制遍历速度，空格键切换自动/手动模式。
- **音效反馈**：发现新最高分时播放"coin"音效，处理常规数据时播放"blip"音效。
- **状态面板**：右侧显示当前最高分学生信息，动态更新。

## 最优代码实现

### 核心逻辑片段（yuyc 题解）
```cpp
char name[9], bname[9];
int c, bc, m, bm, y, by;
// 初始化读取第一个学生
cin >> name >> c >> m >> y; 
for(int i=1; i<n; i++) {
    cin >> bname >> bc >> bm >> by;
    if(bc+bm+by > c+m+y) { // 比较总分
        c = bc; m = bm; y = by;
        strcpy(name, bname); // 更新最优
    }
}
```

### 完整代码
```cpp
#include <iostream>
#include <cstring>
using namespace std;

int main() {
    char name[9], bname[9];
    int c, bc, m, bm, y, by, n;
    cin >> n >> name >> c >> m >> y;
    for(int i=1; i<n; i++) {
        cin >> bname >> bc >> bm >> by;
        if(bc+bm+by > c+m+y) {
            c = bc; m = bm; y = by;
            strcpy(name, bname);
        }
    }
    cout << name << " " << c << " " << m << " " << y;
    return 0;
}
```

## 可视化方案设计

### 像素动画实现
```javascript
// 伪代码示例
class StudentViz {
    constructor() {
        this.canvas = document.createElement('canvas');
        this.students = []; // 存储{name, scores, total}
        this.currentBest = null;
    }

    render() {
        // 绘制16x16像素方块代表每个学生
        this.students.forEach((stu, i) => {
            ctx.fillStyle = stu === this.currentBest ? '#00FF00' : '#333';
            ctx.strokeStyle = '#FF0'; // 当前处理边框色
            drawPixelBlock(i*20, 0, stu);
        });
    }

    addStudent(stu) {
        this.students.push(stu);
        if(!this.currentBest || stu.total > this.currentBest.total) {
            playSound('coin'); // 发现新最优
            this.currentBest = stu;
        } else {
            playSound('blip');
        }
    }
}
```

### 交互功能
- **自动模式**：按设定速度逐步加载学生数据
- **手动模式**：按→键单步执行
- **速度调节**：滑块控制0.5x~4x速度
- **音效开关**：支持关闭8-bit音效

通过将算法过程转化为类似经典RPG的遇敌动画，提升学习趣味性。每个学生的出现如同战斗中的对手，最高分学生以BOSS形态展示。

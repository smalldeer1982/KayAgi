# 题目信息

# 【深基7.例11】评等级

## 题目描述

现有 $N$ 名同学，每名同学需要设计一个结构体记录以下信息：学号、学业成绩和素质拓展成绩、综合分数（实数）。每行读入同学的学号、学业成绩和素质拓展成绩，并且计算综合分数（分别按照 $70\%$ 和 $30\%$ 权重累加），存入结构体中。还需要在结构体中定义一个成员函数，返回该结构体对象的学业成绩和素质拓展成绩的总分。

然后需要设计一个函数，其参数是一个学生结构体对象，判断该学生是否“优秀”。优秀的定义是学业和素质拓展成绩总分**大于** $140$ 分，且综合分数**不小于** $80$ 分。

> 当然通过本题很容易啦，本题只是为了帮助你训练结构体的使用方法。

本题存在精度误差问题，请将 `a * 0.7 + b * 0.3` 与 80 比较 转化为 `a * 7 + b * 3` 与 800 比较。

## 说明/提示

数据保证，$1 \le N\le 1000$，学号为不超过 $100000$ 的正整数，学业成绩和素质拓展成绩为 $0 \sim 100$ 之间的正整数。

# 综合分析与结论

## 核心思路
题目要求对学生的学业和素质成绩进行加权计算，结合总分判断是否优秀。核心在于结构体的正确使用和精度误差处理。优秀判定条件为：
1. 学业+素质总分 >140
2. 综合分数（70%学业+30%素质）≥80（需转化为整数运算避免浮点误差）

## 难点与解决方案
- **结构体成员函数**：需在结构体内定义返回总分的函数
- **精度处理**：将 `0.7a + 0.3b ≥80` 转换为 `7a +3b ≥800`，避免浮点运算误差
- **条件判断顺序**：先判断总分是否>140，再处理综合分条件

## 可视化设计思路
1. **像素风格数据面板**：用8-bit像素字体显示学生信息
2. **动态条件判断**：高亮当前判断的分值部分（总分红框、综合分蓝框）
3. **音效反馈**：当触发"Excellent"时播放上扬音效，失败时低沉音效

# 核心代码实现
```cpp
#include <iostream>
using namespace std;

struct Student {
    int id, sc1, sc2, total;
    void calc() {
        total = sc1 + sc2;
    }
    bool isExcellent() {
        return total > 140 && (sc1*7 + sc2*3) >= 800;
    }
};

int main() {
    int n;
    cin >> n;
    Student s;
    while(n--) {
        cin >> s.id >> s.sc1 >> s.sc2;
        s.calc();
        cout << (s.isExcellent() ? "Excellent" : "Not excellent") << endl;
    }
    return 0;
}
```

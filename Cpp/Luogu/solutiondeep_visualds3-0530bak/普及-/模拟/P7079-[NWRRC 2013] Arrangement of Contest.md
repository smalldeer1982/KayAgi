# [NWRRC 2013] Arrangement of Contest

## 题目描述

小D和小P想要举行比赛。

他们收到了许多建议书，现在他们想要从这些报名书中挑选一些好的建议。

他们采纳的建议符合以下标准：第一个建议的开头为A，第二个建议的开头为B，以此类推。

请你帮助他们，确定他们所能采纳的建议的最大数量。

## **题解思路与核心难点**

### **核心思路**
题目要求找出从'A'开始的最长连续首字母序列长度。关键步骤：
1. **统计存在性**：记录每个大写字母是否作为首字母出现。
2. **连续检查**：从'A'开始顺序检查每个字母是否存在，遇到第一个不存在的字母时终止，返回当前累计的长度。

### **解决难点**
- **快速统计与验证**：使用布尔数组标记字母存在性，将时间复杂度优化至*O(n + 26)*。
- **边界处理**：当所有字母都存在时，需返回26，代码通过循环终止条件`i < 27`巧妙覆盖这一情况。

### **题解对比**
- **数据结构选择**：仅需一个大小为26的布尔数组，空间复杂度为*O(1)*。
- **优化点**：直接遍历字母表而非排序，避免不必要的计算。

## **代码实现**
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    bool exists[26] = {false};
    
    for (int i = 0; i < n; ++i) {
        string s;
        cin >> s;
        exists[s[0] - 'A'] = true;
    }
    
    int ans = 0;
    while (ans < 26 && exists[ans]) ++ans;
    cout << ans << endl;
    return 0;
}
```

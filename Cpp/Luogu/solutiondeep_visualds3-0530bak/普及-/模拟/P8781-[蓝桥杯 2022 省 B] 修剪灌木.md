# 题目信息

# [蓝桥杯 2022 省 B] 修剪灌木

## 题目描述

爱丽丝要完成一项修剪灌木的工作。

有 $N$ 棵灌木整齐的从左到右排成一排。爱丽丝在每天傍晩会修剪一棵灌木，让灌木的高度变为 $0$ 厘米。爱丽丝修剪灌木的顺序是从最左侧的灌木开始，每天向右修剪一棵灌木。当修剪了最右侧的灌木后，她会调转方向，下一天开始向左修剪灌木。直到修剪了最左的灌木后再次调转方向。然后如此循环往复。

灌木每天从早上到傍晩会长高 $1$ 厘米, 而其余时间不会长高。在第一天的早晨, 所有灌木的高度都是 $0$ 厘米。爱丽丝想知道每棵灌木最高长到多高。

## 说明/提示

对于 $30 \%$ 的数据, $N \leq 10$.

对于 $100 \%$ 的数据, $1<N \leq 10000$. 

蓝桥杯 2022 省赛 B 组 D 题。

## 综合分析与结论
### 核心思路
每个灌木的最高高度由两次修剪的最长间隔天数决定。对于第 `i` 棵灌木，其最大间隔天数为 `max(向左走到尽头再返回的天数, 向右走到尽头再返回的天数)`，即 `max(2*(i-1), 2*(n-i))`。

### 关键难点
1. **周期性分析**：理解爱丽丝的修剪路径是往返循环的。
2. **对称性推导**：发现最大高度仅取决于灌木到左右两端的最远距离。
3. **数学建模**：将路径长度转换为天数并推导出公式。

### 可视化设计
1. **像素动画**：用 8-bit 风格绘制灌木排，爱丽丝像素小人左右移动修剪灌木。
2. **颜色标记**：被修剪的灌木变为红色，生长天数实时显示为绿色数字。
3. **音效交互**：修剪时播放“咔嚓”音效，最大高度达成时播放胜利音效。
4. **自动演示**：按公式自动计算并高亮当前最大间隔路径。

## 核心代码实现
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; ++i) {
        cout << max(i-1, n-i) * 2 << endl;
    }
    return 0;
}
```

## 最优思路提炼
**对称极值法**：每个位置的最大高度由其到左右两端的最远距离决定，无需模拟路径，直接公式计算。时间复杂度 O(n)，空间复杂度 O(1)。

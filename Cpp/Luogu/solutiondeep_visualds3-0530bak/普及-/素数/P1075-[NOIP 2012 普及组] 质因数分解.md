# 题目信息

# [NOIP 2012 普及组] 质因数分解

## 题目描述

已知正整数 $n$ 是两个不同的质数的乘积，试求出两者中较大的那个质数。


## 说明/提示

$1 \le n\le 2\times 10^9$

NOIP 2012 普及组 第一题


## 样例 #1

### 输入

```
21```

### 输出

```
7```

### **题解思路、算法要点与解决难点**

1. **核心思路**  
   题目保证 `n` 是两个不同质数的乘积，因此只需找到最小的质因数 `i`，较大的质数即为 `n/i`。  
   - **关键推导**：由于质数的最小因数必为质数，故从 `2` 开始枚举，第一个能整除 `n` 的 `i` 即为较小质数。
   - **优化点**：枚举到 `sqrt(n)` 即可，但实际代码中一旦找到 `i` 即可终止循环，无需严格限制到 `sqrt(n)`。

2. **解决难点**  
   - **超时风险**：直接反向枚举 `n-1` 到 `2` 效率极低（如 `n=2e9` 时循环次数达 `O(n)`）。  
   - **数学优化**：通过正向枚举最小质因数，将时间复杂度降为 `O(sqrt(n))`，确保通过大范围数据。

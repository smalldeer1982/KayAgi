# 题目信息

# 10 进制转 x 进制

## 题目描述

给定一个十进制整数 $n$ 和一个小整数 $x$。将整数 $n$ 转为 $x$ 进制。对于超过十进制的数码，用 `A`，`B` ... 表示。 



## 说明/提示

【数据规模和约定】  
保证 $n$ 不超过 $10^9$，$x$ 不超过 $36$。

## 样例 #1

### 输入

```
1000
2
```

### 输出

```
1111101000
```

## 综合分析与结论  
### 核心思路  
所有题解均基于**短除法**，核心步骤为：  
1. 反复取余数 `n % x`，映射为对应字符（`0-9` 或 `A-Z`）  
2. 倒序输出余数序列  

### 关键差异与难点  
| 方法            | 数据结构        | 余数存储方式      | 反转实现         | 字符映射方式         |  
|----------------|-------------|-------------|--------------|----------------|  
| 迭代+字符串拼接     | `string`    | 正序拼接后反转     | 倒序遍历字符串     | 预定义字典字符串       |  
| 递归            | 无           | 函数调用栈隐式存储  | 递归返回时逆序输出   | 条件判断或字典字符串     |  
| 数组存储          | `int[]`     | 数组正序存储余数   | 倒序输出数组元素    | 转换函数条件判断       |  
| 标准库函数        | `char[]`    | 库函数内部处理     | 无（库自动处理）   | 库函数处理+大小写转换    |  

### 可视化设计  
1. **动画流程**：  
   - 网格展示当前 `n` 值，每次分解为 `n = q * x + r`  
   - 右侧逐步拼接余数字符，下方用箭头标记当前处理位置  
   - 最后一步触发反转动画，用红色高亮反转过程  

2. **复古像素风格**：  
   - 使用 8-bit 字体，余数字符用 16x16 像素方块展示  
   - 每次除法操作时播放「哔」音效，反转时播放「叮」音效  
   - 背景音乐采用低比特率版《超级玛丽》地下关 BGM  

3. **交互功能**：  
   - 速度滑块控制每步延迟（100ms~2000ms）  
   - 点击余数字符可查看其生成时的 `n` 和 `x` 值

## 最优思路提炼  
### 关键技巧  
1. **字典预映射**：  
   ```cpp
   string dict = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";
   ans += dict[n % x]; // 直接索引代替条件判断
   ```  
2. **递归倒序**：  
   ```cpp
   void tox(int n) {
       if(n/x) tox(n/x); // 先递归到最深再输出
       cout << dict[n%x];
   }
   ```  
3. **余数数组反转**：  
   ```cpp
   int a[100], t=0;
   while(n) { a[t++] = n%x; n/=x; }
   for(int i=t-1; i>=0; i--) cout << dict[a[i]];
   ```

## 核心代码实现  
### liangbowen 迭代法  
```cpp
string ten_to_x(int n, int x) {
    string ans;
    while(n) {
        ans += "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"[n%x];
        n /= x;
    }
    return string(ans.rbegin(), ans.rend()); // 使用反向迭代器反转
}
```  

### Jorisy 递归法  
```cpp
const string R = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";
void tox(int n) {
    if(n/x) tox(n/x);
    cout << R[n%x];
}
```

# 题目信息

# x 进制转 10 进制

## 题目描述

给一个小整数 $x$ 和一个 $x$ 进制的数 $S$。将 $S$ 转为 $10$ 进制数。对于超过十进制的数码，用 `A`，`B`，$\ldots$ 表示。

## 说明/提示

【数据规模和约定】  

保证目标数在十进制下不超过 $10^9$，$1 \leq x \leq 36$。


## 样例 #1

### 输入

```
16
7B
```

### 输出

```
123```

### 综合分析与结论
**核心思路**：将 x 进制字符串按权展开求和，每个字符通过数值映射和权重计算转换为十进制。  
**解决难点**：  
1. 字符到数值的映射（如 'A'→10，'B'→11）  
2. 权重计算（x 的幂次增长）  
3. 逆序处理字符串保证低位优先计算  

**可视化设计**：  
- **核心变量**：当前处理的字符、字符转换后的数值、当前权重值、累加结果  
- **动画方案**：  
  - **像素方块**：每个字符以不同颜色方块表示（数字用绿色，字母用橙色）  
  - **权重展示**：右侧动态显示 x^i 的数值变化，用 8-bit 字体滚动  
  - **音效设计**：字符转换时播放短促“滴”声，权重更新时播放“哔”声  
  - **自动演示**：按空格键单步执行，按→键加速流程

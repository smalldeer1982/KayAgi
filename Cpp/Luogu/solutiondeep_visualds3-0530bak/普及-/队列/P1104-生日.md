# 题目信息

# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

### 题解思路与核心难点分析
#### 核心逻辑
1. **数据结构**：使用结构体存储姓名、年、月、日、输入顺序
2. **排序规则**：
   - 第一关键字：年份（升序）
   - 第二关键字：月份（升序）
   - 第三关键字：日期（升序）
   - 第四关键字：输入顺序（降序，保证同生日时后输入的优先）

#### 解决难点
- **多条件嵌套判断**：需正确处理年、月、日的优先级关系
- **稳定排序处理**：使用`stable_sort`或通过输入顺序逆序保证稳定性
- **日期有效性**：题目已保证数据合法，无需额外验证

#### 算法对比
| 方法            | 时间复杂度 | 空间复杂度 | 实现难度 | 适用性       |

| 结构体+sort    | O(nlogn)   | O(n)       | ★★☆☆☆   | 最优选择     |

### 核心代码实现
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Student {
    string name;
    int year, month, day;
    int order;
};

bool compare(const Student &a, const Student &b) {
    if(a.year != b.year) return a.year < b.year;
    if(a.month != b.month) return a.month < b.month;
    if(a.day != b.day) return a.day < b.day;
    return a.order > b.order; // 关键：处理同生日情况
}

int main() {
    int n;
    cin >> n;
    Student students[n];
    
    for(int i=0; i<n; ++i) {
        cin >> students[i].name 
            >> students[i].year 
            >> students[i].month 
            >> students[i].day;
        students[i].order = i;
    }
    
    sort(students, students+n, compare);
    
    for(auto &s : students) {
        cout << s.name << endl;
    }
    return 0;
}
```

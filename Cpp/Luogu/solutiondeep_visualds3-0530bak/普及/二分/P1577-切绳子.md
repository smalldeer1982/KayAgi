# 题目信息

# 切绳子

## 题目描述

有 $N$ 条绳子，它们的长度分别为 $L_i$。如果从它们中切割出 $K$ 条长度相同的绳子，这 $K$ 条绳子每条最长能有多长？答案保留到小数点后 $2$ 位(直接舍掉 $2$ 位后的小数)。


## 说明/提示

对于 $100\%$ 的数据 $0<L_i\leq 100000.00,0<n\leq 10000,0<k\leq 10000$


## 样例 #1

### 输入

```
4 11
8.02
7.43
4.57
5.39```

### 输出

```
2.00```

# AI分析结果



**唯一算法分类**：二分

---

### **题解思路与算法要点**

**核心思路**：二分答案法。将每段绳子的最大可能长度作为目标值，通过二分法在可能的区间内搜索，利用判断函数验证当前长度是否能切出足够的段数。

#### **关键步骤与难点**：
1. **浮点转整数**：将输入的浮点数乘以100转为整数处理，避免精度问题。
2. **二分区间初始化**：左边界为0，右边界为最大绳长×100（如1e8）。
3. **判断函数设计**：计算当前长度下可切出的总段数，若≥K则说明可尝试更大值。
4. **边界处理**：避免mid=0时的除零错误，输出时截断而非四舍五入。
5. **区间收缩**：根据判断结果调整左右边界，确保收敛到正确值。

### **最优思路与技巧提炼**

1. **整数转换**：将浮点数×100转为整数运算，避免精度丢失。
2. **二分条件优化**：若当前mid可满足条件，左边界移至mid+1（尝试更大值）；否则右边界移至mid-1。
3. **输出截断**：通过整数运算结果直接除以100，避免四舍五入。
4. **特判mid=0**：防止除零错误，确保循环安全退出。

**核心代码片段**：
```cpp
int l = 0, r = 1e8;
while (l <= r) {
    int mid = (l + r) / 2;
    if (mid == 0) break;
    int cnt = 0;
    for (int i = 1; i <= n; i++) cnt += a[i] / mid;
    if (cnt >= k) l = mid + 1;
    else r = mid - 1;
}
printf("%.2f", r / 100.0);
```

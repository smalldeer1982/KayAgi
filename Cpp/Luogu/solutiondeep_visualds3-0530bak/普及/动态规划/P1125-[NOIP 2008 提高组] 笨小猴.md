# [NOIP 2008 提高组] 笨小猴

## 题目描述

笨小猴的词汇量很小，所以每次做英语选择题的时候都很头疼。但是他找到了一种方法，经试验证明，用这种方法去选择选项的时候选对的几率非常大！

这种方法的具体描述如下：假设 $\text{maxn}$ 是单词中出现次数最多的字母的出现次数，$\text{minn}$ 是单词中出现次数最少的字母的出现次数，如果 $\text{maxn}-\text{minn}$ 是一个质数，那么笨小猴就认为这是个 Lucky Word，这样的单词很可能就是正确的答案。


## 说明/提示

【输入输出样例 1 解释】

单词 `error` 中出现最多的字母 $\texttt r$ 出现了 $3$ 次，出现次数最少的字母出现了 $1$ 次，$3-1=2$，$2$ 是质数。

【输入输出样例 2 解释】

单词 `olympic` 中出现最多的字母 $\texttt i$ 出现了 $1$ 次，出现次数最少的字母出现了 $1$ 次，$1-1=0$，$0$ 不是质数。

## 2. 题解思路与难点对比
### 核心思路
- **字符统计**：使用桶数组（`int[26]`）统计每个字母出现次数。
- **极值查找**：遍历桶数组找出最大出现次数（`maxn`）和最小非零出现次数（`minn`）。
- **质数判断**：对差值 `maxn-minn` 进行质数验证。

### 解决难点
- **最小值初始化**：需排除未出现的字符（`a[i] != 0`）。
- **质数边界条件**：正确处理 `0` 和 `1` 的非质数情况。
- **极值更新逻辑**：部分题解在遍历时动态更新极值（`max()` 和 `min()` 函数）。

## 9. 核心代码片段
```cpp
// 字符统计与极值计算
int a[26] = {0};
string s;
cin >> s;
for (char c : s) a[c - 'a']++;

int maxn = 0, minn = INT_MAX;
for (int i = 0; i < 26; i++) {
    if (a[i] > 0) {
        maxn = max(maxn, a[i]);
        minn = min(minn, a[i]);
    }
}

// 质数判断与输出
int diff = maxn - minn;
cout << (is_prime(diff) ? "Lucky Word" : "No Answer") << endl;
cout << (is_prime(diff) ? diff : 0);
```

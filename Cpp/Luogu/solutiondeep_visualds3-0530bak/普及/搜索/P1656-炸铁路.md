# 题目信息

# 炸铁路

## 题目描述

A 国派出将军 uim，对 B 国进行战略性措施，以解救涂炭的生灵。

B 国有 $n$ 个城市，这些城市以铁路相连。任意两个城市都可以通过铁路直接或者间接到达。

uim 发现有些铁路被毁坏之后，某两个城市无法互相通过铁路到达。这样的铁路就被称为 key road。

uim 为了尽快使该国的物流系统瘫痪，希望炸毁铁路，以达到存在某两个城市无法互相通过铁路到达的效果。

然而，只有一发炮弹（A 国国会不给钱了）。所以，他能轰炸哪一条铁路呢？

### 输入

```
6 6
1 2
2 3
2 4

5 6```

### 输出

```
1 2
5 6```

## 算法分类
**DFS** (基于Tarjan算法求割边)

## 综合分析与结论
本题需要找出无向图中的所有桥（割边），即删除后会使得图不再连通的边。核心算法是Tarjan算法，通过DFS遍历计算dfn和low值来识别割边。

### 核心难点与解决方案
1. **割边判定**：当low[v] > dfn[u]时，边(u,v)是割边
2. **重边处理**：需要特殊处理重边情况，避免误判
3. **输出排序**：需要按字典序输出结果

### 可视化设计思路
1. **DFS遍历动画**：展示节点访问顺序和dfn/low值更新过程
2. **割边高亮**：用红色标记识别出的割边
3. **回溯过程**：展示low值如何通过回溯传播
4. **比较视图**：可对比暴力枚举法与Tarjan算法的效率差异

## 核心算法实现

### Tarjan算法关键代码
```cpp
void tarjan(int u, int fa) {
    dfn[u] = low[u] = ++idx;
    for(int i = head[u]; i; i = e[i].next) {
        int v = e[i].to;
        if(!dfn[v]) {
            tarjan(v, u);
            low[u] = min(low[u], low[v]);
            if(low[v] > dfn[u]) {
                // (u,v)是割边
                ans.push_back({min(u,v), max(u,v)});
            }
        } 
        else if(v != fa) {
            low[u] = min(low[u], dfn[v]);
        }
    }
}
```

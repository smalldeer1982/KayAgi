# 题目信息

# 火炬

## 题目描述

2008 北京奥运会，你想成为四川汶川的一名火炬手，经过层层选拔，终于到了最后一关。

这一关是一道很难的题：任意给定一个正整数 $N$，求一个最小的正整数 $M$，使得 $N \times M$ 的十进制表示形式里只含有 `1` 和 `0`。


## 说明/提示

$1\le N\le 10^5$

## 样例 #1

### 输入

```
12```

### 输出

```
925```

# 算法分类：BFS

## 综合分析与结论

本题要求找到最小的正整数M，使得N×M的十进制表示只含0和1。这是一个典型的搜索问题，核心在于高效生成所有可能的0-1数字并检查是否能被N整除。

### 核心难点与解决方案：
1. **搜索空间爆炸**：直接枚举M会超时，转而生成0-1数字更高效
2. **最小解保证**：BFS按数字长度递增搜索，保证首次找到的解最小
3. **余数优化**：记录余数避免重复计算（如HoshiuZ的题解）

### 可视化设计思路：
- **节点表示**：每个节点是一个0-1数字，显示其值和当前余数
- **扩展规则**：从当前数字末尾添加0或1生成新节点
- **颜色标记**：
  - 绿色：已访问节点
  - 红色：当前正在处理的节点
  - 蓝色：待访问节点队列
- **终止条件**：当余数为0时高亮显示解

### 复古像素风格实现：
- 使用16色调色板，节点显示为8-bit风格方块
- 音效设计：
  - 节点扩展："哔"声
  - 找到解："胜利"音效
  - 队列更新："咔嗒"声
- 自动演示模式按BFS顺序逐步展开搜索树

## 题解清单 (≥4星)

### 1. Manjusaka丶梦寒 (5星)
- **亮点**：简洁的DFS生成所有0-1数后排序检查
- **关键代码**：
```cpp
void dfs(long long x) {
    a[++tot]=x;
    if(x>9e17)return;
    dfs(x*10);  // 添加0
    dfs(x*10+1); // 添加1
}
```
- **心得**："先搜0再搜1保证有序性"

### 2. HoshiuZ (5星)
- **亮点**：BFS+余数剪枝，最优解法
- **关键优化**：
```cpp
if(!vh[q_1%n]) que[++tail]=q_1; // 余数剪枝
if(!vh[q_2%n]) que[++tail]=q_2;
```

### 3. _cmh (4星)
- **亮点**：简洁DFS实现，限制搜索深度
- **关键代码**：
```cpp
dfs(now*10,bits+1);   // 添加0分支
dfs(now*10+1,bits+1); // 添加1分支
```

## 最优思路提炼

1. **BFS优于DFS**：按数字长度递增搜索保证最优解
2. **余数剪枝**：记录余数避免重复计算，空间换时间
3. **二进制思维**：部分题解将0-1数视为二进制表示（如王兴澄的题解）

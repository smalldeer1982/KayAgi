# 题目信息

# [蓝桥杯青少年组省赛 2022] 组合

## 题目描述

某商店将一种糖果按照数量打包成 $N$ 和 $M$ 两种规格来售卖（$N$ 和 $M$ 为互质数，且 $N$ 和 $M$ 有无数包）。这样的售卖方式会限制一些数量的糖果不能买到。现在给出 $N$ 和 $M$ 的值，请你计算出最多不能买到的糖果数量。


## 样例 #1

### 输入

```
3 5```

### 输出

```
7```

# 题解思路、算法要点与解决难点  
## 核心思路对比  
### 数学定理派（塞瓦维斯特定理）  
- **核心公式**：$ans = N \times M - N - M$  
- **解决难点**：证明公式的正确性（分两部分证明：该数不可表示，且所有更大的数都可表示）  
- **算法优势**：时间复杂度 O(1)，无额外空间消耗  

### 动态规划派  
- **核心逻辑**：布尔数组标记可达数值，递推式 `dp[i] = dp[i-N] || dp[i-M]`  
- **解决难点**：确定循环边界为 $2NM$ 以覆盖所有不可达情况  
- **算法劣势**：时间复杂度 O(NM)，空间复杂度 O(NM)

# 代码实现  
## 数学派核心代码  
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    cout << n * m - n - m;
    return 0;
}
```

## 动态规划派核心片段  
```cpp
bool dp[20005] = {false};
dp[0] = true;
for (int i = 1; i <= 2 * n * m; ++i) {
    if (i >= n && dp[i - n]) dp[i] = true;
    if (i >= m && dp[i - m]) dp[i] = true;
}
```

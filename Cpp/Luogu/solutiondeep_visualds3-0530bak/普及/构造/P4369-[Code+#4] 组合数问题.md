# 题目信息

# [Code+#4] 组合数问题

## 题目描述

众所周知，小葱同学擅长计算，尤其擅长计算组合数，所以小葱给了你两个数 $x$ 和 $k$，希望你把 $x$ 分成恰好 $k$ 个不同的组合数的和。所谓不同，即对于两个组合数 $C(n_1,m_1)$ 和 $C(n_2,m_2)$，如果 $n_1\neq n_2$​​ 或者 $m_1\neq m_2$​，我们就称这两个组合数是不同的。为了使得计算不过于复杂，你需要保证你给出的任何一个组合数 $C(n,m)$ 满足 $0\leq m\leq n\leq x$。数据保证一定有解。

## 说明/提示

对于 $20\%$ 的数据，$k=1$。

对于另外 $20\%$ 的数据，$x\leq 100$。

对于另外 $20\%$ 的数据，$k=2$。

对于 $100\%$ 的数据，$1\leq x\leq 10^9,1\leq k\leq 10^3.$

## 题解思路与解决难点
**核心思路**：利用组合数性质 C(n,0)=1 和 C(n,1)=n，将问题拆解为 k-1 个 1 和一个 (x-k+1) 的组合。  
**算法要点**：
1. 用 k-1 个不同的 C(i,0) 提供 1 的贡献（i 从 1 到 k-1）
2. 最后一个组合数用 C(x-k+1,1) 提供剩余值 x-k+1  
**解决难点**：
- 保证所有组合数满足 n ≤ x 且互不相同
- 时间复杂度必须为 O(k) 以支持 1e3 规模的 k

**关键推导**：  
总和为 (k-1)*1 + (x-k+1) = x  
所有组合数的下标 n 均满足 1 ≤ n ≤ x-k+1 ≤ x

## 最优思路提炼
**核心技巧**：  
1. **拆分常数项**：利用 C(i,0)=1 的稳定贡献  
2. **剩余值处理**：用 C(n,1)=n 的特性吸收余量  
3. **差异保证**：通过不同下标 i 确保组合数唯一性  

**通用构造框架**：  
当需要将数值拆分为多个特殊结构的和时，优先寻找能稳定贡献固定值（如 1）的结构，再用单个大项吸收剩余值。

## 代码实现
```cpp
#include <cstdio>

int main() {
    int x, k;
    scanf("%d%d", &x, &k);
    for (int i = 1; i < k; ++i)
        printf("%d 0\n", i);   // k-1个C(i,0)
    printf("%d 1\n", x - k + 1); // 最后一个C(n,1)
    return 0;
}
```

**关键逻辑**：  
- 循环生成 k-1 个不同的 C(i,0)  
- 最终项为 C(x-k+1,1)，确保总和为 x

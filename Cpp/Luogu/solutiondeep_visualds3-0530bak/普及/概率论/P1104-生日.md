# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

### 题解分析与结论
#### 核心思路
1. **数据结构**：使用结构体存储姓名、年、月、日、输入顺序。
2. **排序规则**：
   - 第一优先级：年份较小者年龄更大。
   - 第二优先级：月份较小者年龄更大。
   - 第三优先级：日期较小者年龄更大。
   - 第四优先级：生日相同则输入顺序靠后者优先。
3. **实现方式**：通过自定义 `cmp` 函数或重载运算符实现多级比较，结合 `sort` 或 `stable_sort` 完成排序。

#### 难点对比
| 题解难点 | 解决方案 |
|---------|---------|
| 多级比较逻辑易错 | 逐层判断年、月、日，最后处理输入顺序（如 `if(a.year != b.year) return ...`） |
| 稳定排序需求 | 使用 `stable_sort` 保证相同生日时输入顺序不变 |
| 日期转换错误 | 合并年月日为整数时需补零（如 `1992-4-23 → 19920423`） |
| 闰年处理 | 大部分题解未处理，但题目保证输入数据合法 |

#### 最优思路提炼
1. **结构体存储**：将姓名、年、月、日、输入顺序封装为结构体。
2. **逐级比较**：按年→月→日→输入顺序的优先级实现 `cmp` 函数。
3. **代码简洁化**：使用三目运算符或逻辑或（`||`）简化判断逻辑。

### 核心代码片段
```cpp
struct Student {
    string name;
    int year, month, day, id;
};

bool cmp(const Student &a, const Student &b) {
    if (a.year != b.year) return a.year < b.year;
    if (a.month != b.month) return a.month < b.month;
    if (a.day != b.day) return a.day < b.day;
    return a.id > b.id; // 输入顺序逆序
}

// 使用 stable_sort 保证稳定性
stable_sort(students.begin(), students.end(), cmp);
```

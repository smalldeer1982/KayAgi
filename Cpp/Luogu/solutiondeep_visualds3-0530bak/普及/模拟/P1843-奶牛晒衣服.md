# 奶牛晒衣服

## 题目背景

熊大妈决定给每个牛宝宝都穿上可爱的婴儿装 。但是由于衣服很湿，为牛宝宝晒衣服就成了很不爽的事情。于是，熊大妈请你（奶牛）帮助她完成这个重任。


## 题目描述

一件衣服在自然条件下用一秒的时间可以晒干 $a$ 点湿度。抠门的熊大妈只买了一台烘衣机 。使用用一秒烘衣机可以让一件衣服额外烘干 $b$ 点湿度（一秒晒干 $a+b$ 湿度），但在同一时间内只能烘一件衣服。现在有 $n$ 件衣服，第 $i$ 衣服的湿度为 $w_i$（保证互不相同），要你求出弄干所有衣服的最少时间（湿度为 $0$ 为干 ）。


## 说明/提示

### 样例解释

让机器烘第三件衣服即可一秒完成。

### 数据范围

$1 \le w_i,a,b,n \le 5 \times 10^5$

## 样例 #1

### 输入

```
3 2 1
1
2
3```

### 输出

```
1```

# AI分析结果



## 唯一算法分类
二分答案

## 综合分析与结论
题目核心是求最小烘干时间，主流解法采用二分答案和贪心+优先队列。二分法通过判断给定时间是否可行，时间复杂度稳定为O(n log max_w)。贪心法则每次处理最大湿度衣物，借助优先队列实现，时间复杂度O(t log n)。两种方法均正确，但二分法更易实现且时间稳定。

### 核心算法流程（二分答案）
1. **二分范围**：确定时间上下界，通常取0至最大湿度
2. **检查函数**：对给定时间mid，计算每件衣物所需额外烘干次数
3. **收敛判断**：若总烘干次数≤mid则为可行解，缩小右边界

### 可视化设计
- **动画方案**：绘制湿度柱状图，二分过程中高亮当前mid对应的检查流程
- **颜色标记**：红色标记当前处理的衣物，绿色表示已达标
- **控制面板**：支持调节二分速度，单步查看检查函数细节
- **像素风格**：用8-bit色块表示湿度值，烘衣时触发“嘟嘟”音效

## 最优思路提炼
**二分答案关键点**：
```cpp
bool check(int mid) {
    int tot = 0;
    for (int i=1; i<=n; i++) {
        if (w[i] <= mid*a) continue;
        int need = (w[i] - mid*a + b-1) / b; // 上取整技巧
        if ((tot += need) > mid) return false;
    }
    return true;
}
```
**贪心关键点**：
```cpp
priority_queue<int> q;
while (q.top() > time*a) {
    int cur = q.top() - b;
    q.pop();
    q.push(cur);
    time++;
}
```

# [NOIP 2008 提高组] 笨小猴

## 题目描述

笨小猴的词汇量很小，所以每次做英语选择题的时候都很头疼。但是他找到了一种方法，经试验证明，用这种方法去选择选项的时候选对的几率非常大！

这种方法的具体描述如下：假设 $\text{maxn}$ 是单词中出现次数最多的字母的出现次数，$\text{minn}$ 是单词中出现次数最少的字母的出现次数，如果 $\text{maxn}-\text{minn}$ 是一个质数，那么笨小猴就认为这是个 Lucky Word，这样的单词很可能就是正确的答案。


## 说明/提示

【输入输出样例 1 解释】

单词 `error` 中出现最多的字母 $\texttt r$ 出现了 $3$ 次，出现次数最少的字母出现了 $1$ 次，$3-1=2$，$2$ 是质数。

【输入输出样例 2 解释】

单词 `olympic` 中出现最多的字母 $\texttt i$ 出现了 $1$ 次，出现次数最少的字母出现了 $1$ 次，$1-1=0$，$0$ 不是质数。

## 题解思路与难点对比

### 核心算法流程
1. **字母统计**：使用桶数组记录每个字母的出现次数
2. **极值查找**：遍历桶数组，找非零最大值和最小值
3. **质数判断**：对差值进行质数判定，注意特判差 ≤1 的情况

### 解决难点
- **最小值过滤**：确保只统计出现过的字母（`a[i] != 0`）
- **质数边界**：正确处理差值为0/1的非质数情况
- **高效判断**：试除到√n 即可，无需全遍历

### 可视化设计
- **像素柱状图**：用不同颜色方块表示各字母出现次数
- **高亮标记**：红色标记最大值，蓝色标记最小值
- **质数检测动画**：模拟试除法过程，数字分解时播放音效
- **复古音效**：成功时8-bit上扬音效，失败时短促警报音

## 最优思路提炼

### 关键代码实现
```cpp
// 字母统计与极值查找
int cnt[26] = {0};
for(char c : s) cnt[c-'a']++;

int maxn = 0, minn = INT_MAX;
for(int i=0; i<26; i++) {
    if(cnt[i] > 0) {
        maxn = max(maxn, cnt[i]);
        minn = min(minn, cnt[i]);
    }
}

// 质数判断函数
bool isPrime(int x) {
    if(x < 2) return false;
    for(int i=2; i*i<=x; i++)
        if(x%i == 0) return false;
    return true;
}
```

### 核心技巧
- **桶数组优化**：直接映射字母到数组下标，O(n)复杂度
- **极值同步更新**：单次遍历同时更新max和min
- **平方根优化**：质数判断仅需试除到√x

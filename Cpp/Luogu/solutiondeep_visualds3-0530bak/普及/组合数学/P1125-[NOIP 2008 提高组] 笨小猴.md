# [NOIP 2008 提高组] 笨小猴

## 题目描述

笨小猴的词汇量很小，所以每次做英语选择题的时候都很头疼。但是他找到了一种方法，经试验证明，用这种方法去选择选项的时候选对的几率非常大！

这种方法的具体描述如下：假设 $\text{maxn}$ 是单词中出现次数最多的字母的出现次数，$\text{minn}$ 是单词中出现次数最少的字母的出现次数，如果 $\text{maxn}-\text{minn}$ 是一个质数，那么笨小猴就认为这是个 Lucky Word，这样的单词很可能就是正确的答案。


## 说明/提示

【输入输出样例 1 解释】

单词 `error` 中出现最多的字母 $\texttt r$ 出现了 $3$ 次，出现次数最少的字母出现了 $1$ 次，$3-1=2$，$2$ 是质数。

【输入输出样例 2 解释】

单词 `olympic` 中出现最多的字母 $\texttt i$ 出现了 $1$ 次，出现次数最少的字母出现了 $1$ 次，$1-1=0$，$0$ 不是质数。

## **2. 题解思路与解决难点**
### **核心步骤**
1. **统计字母频率**：用桶数组（`int[26]`）记录每个字母出现次数。
2. **求最大/最小值**：遍历桶数组，注意排除未出现的字母（次数为0）。
3. **差值判断质数**：若差值为质数输出结果，否则输出失败。

### **难点对比**
- **最小值初始化**：需设为极大值（如`INT_MAX`）并在遍历时跳过次数为0的字母。
- **质数判断边界**：需处理差值为0或1的情况（如样例2的差为0）。
- **效率优化**：预存质数表（如题解1）或动态计算到平方根（多数题解）。

## **4. 最优思路与技巧**
### **关键实现**
```cpp
// 统计字母出现次数
int cnt[26] = {0};
for (char c : s) cnt[c-'a']++;

// 找最大/最小值（排除0次）
int maxn = 0, minn = INT_MAX;
for (int i=0; i<26; i++) {
    if (cnt[i] > 0) {
        maxn = max(maxn, cnt[i]);
        minn = min(minn, cnt[i]);
    }
}

// 质数判断函数
bool is_prime(int x) {
    if (x < 2) return false;
    for (int i=2; i*i <= x; i++)
        if (x%i == 0) return false;
    return true;
}
```

## **8. 核心代码（完整版）**
```cpp
#include<bits/stdc++.h>
using namespace std;

bool is_prime(int x) {
    if (x < 2) return false;
    for (int i=2; i*i <=x; i++)
        if (x%i == 0) return false;
    return true;
}

int main() {
    string s;
    cin >> s;
    int cnt[26] = {0};
    for (char c : s) cnt[c-'a']++;
    
    int maxn = 0, minn = INT_MAX;
    for (int i=0; i<26; i++) {
        if (cnt[i] > 0) {
            maxn = max(maxn, cnt[i]);
            minn = min(minn, cnt[i]);
        }
    }
    
    int diff = maxn - minn;
    if (is_prime(diff)) {
        cout << "Lucky Word\n" << diff;
    } else {
        cout << "No Answer\n0";
    }
    return 0;
}
```

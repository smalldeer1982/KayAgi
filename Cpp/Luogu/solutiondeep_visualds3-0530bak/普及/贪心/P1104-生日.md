# 题目信息

# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

# AI分析结果



### 综合分析与结论

**题目核心**：按出生日期从早到晚排序（年龄从大到小），日期相同则后输入的先输出。  
**核心难点**：多关键字排序（年、月、日）、输入顺序处理。  
**通用解法**：使用结构体存储学生信息，自定义比较函数实现多级排序，最后处理输入顺序。

### 最优思路与技巧提炼

1. **多关键字排序**：  
   - 按年、月、日逐级比较，最后处理输入顺序。  
   - **实现关键**：比较函数中依次判断各字段，使用短路逻辑避免冗余判断。

2. **日期合并技巧**：  
   - 将年月日转换为整数（如 `y*10000 + m*100 + d`），直接比较数值大小。  
   - **注意事项**：需确保月和日格式为两位数（如 `4月` → `04`），否则需补零。

3. **输入顺序处理**：  
   - 在结构体中记录输入时的序号，日期相同时比较序号大小，后输入的序号更大。

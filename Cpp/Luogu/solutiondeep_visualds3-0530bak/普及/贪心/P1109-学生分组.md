# 题目信息

# 学生分组

## 题目描述

有 $n$ 组学生，给出初始时每组中的学生个数，再给出每组学生人数的上界 $R$ 和下界 $L\ (L \le R)$，每次你可以在某组中选出一个学生把他安排到另外一组中，问最少要多少次才可以使 $N$ 组学生的人数都在 $[L,R]$ 中。


## 说明/提示

### 数据范围及约定

对于全部数据，保证 $1\le n \le 50$。

## 样例 #1

### 输入

```
2
10 20
10 15
```

### 输出

```
5```

# AI分析结果



### 综合分析与结论

**题目核心**：调整n组学生人数，使其均在[L, R]区间内，求最少移动次数。

**贪心策略**：
1. **可行性判断**：总人数必须在[n*L, n*R]范围内，否则无法满足条件。
2. **统计缺口与溢出**：计算所有组低于L的缺口总和（需补入）和高于R的溢出总和（需移出）。
3. **取最大值**：最少移动次数为缺口与溢出的最大值，因为每次移动可同时处理一个缺口和一个溢出，剩余部分需单独调整。

**正确性证明**：总人数合法时，溢出和缺口的总和必然相等或互补。较大的数值决定了必须通过单独移动的次数，而较小值可通过互补抵消。

### 核心代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
int n, a[51], l, r, total, under, over;

int main() {
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        total += a[i];
    }
    cin >> l >> r;
    if (total < l * n || total > r * n) {
        cout << -1;
        return 0;
    }
    for (int i = 1; i <= n; i++) {
        if (a[i] < l) under += l - a[i];
        if (a[i] > r) over += a[i] - r;
    }
    cout << max(under, over);
    return 0;
}
```

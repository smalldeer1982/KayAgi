# 雷达安装

## 题目描述

假设海岸线是一条无限延伸的直线。它的一侧是陆地，另一侧是海洋。每一座小岛是在海面上的一个点。雷达必须安装在陆地上（包括海岸线），并且每个雷达都有相同的扫描范围 $d$。你的任务是建立尽量少的雷达站，使所有小岛都在扫描范围之内。

数据使用笛卡尔坐标系，定义海岸线为 $x$ 轴。在 $x$ 轴上方为海洋，下方为陆地。

对于全部数据，$n\le1000$，$ d \le 2\times 10^4$，$ | x_i | \le 2 \times 10^6 $，$ 0 \le y_i \le 2\times 10^4$。

### 综合分析与结论

**题目核心**：将岛屿坐标转化为雷达覆盖区间，转化为区间选点问题，需用贪心策略求最少雷达数。  
**核心难点**：将几何问题转化为区间覆盖模型，正确排序区间并设计贪心策略。  
**贪心策略**：按区间右端点排序，每次选择当前区间的右端点作为雷达位置，尽可能覆盖后续区间。  
**正确性依据**：每次选择最左可能的右端点，保证不遗漏更优解（数学归纳法可证）。

### 最优思路提炼

1. **问题转化**：岛屿坐标 → 雷达覆盖区间（勾股定理计算左右端点）。  
2. **排序策略**：按区间右端点升序排列，确保每次选择最左可能的雷达位置。  
3. **贪心选择**：遍历排序后的区间，若当前区间的左端点超过最后一个雷达位置，新增雷达并更新位置。  

```cpp
// 弹星者代码核心片段
sort(a+1, a+n+1, cmp); // 按右端点排序
double temp = a[1].r;
int ans = 1;
for (int i=2; i<=n; i++) {
    if (a[i].l > temp) {
        temp = a[i].r;
        ans++;
    }
}
```

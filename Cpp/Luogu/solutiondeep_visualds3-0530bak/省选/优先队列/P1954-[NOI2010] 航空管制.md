# 题目信息

# [NOI2010] 航空管制

## 题目描述

世博期间，上海的航空客运量大大超过了平时，随之而来的航空管制也频频发生。最近，小 X 就因为航空管制，连续两次在机场被延误超过了两小时。对此，小 X 表示很不满意。

在这次来烟台的路上，小 X 不幸又一次碰上了航空管制。于是小 X 开始思考关于航空管制的问题。

假设目前被延误航班共有 $n$ 个，编号为 $1$ 至 $n$。机场只有一条起飞跑道，所有的航班需按某个顺序依次起飞（称这个顺序为起飞序列）。定义一个航班的起飞序号为该航班在起飞序列中的位置，即是第几个起飞的航班。

起飞序列还存在两类限制条件：

- 第一类（最晚起飞时间限制）：编号为 $i$ 的航班起飞序号不得超过 $k_i$。

- 第二类（相对起飞顺序限制）：存在一些相对起飞顺序限制 $(a,b)$，表示航班 $a$ 的起飞时间必须早于航班 $b$，即航班 $a$ 的起飞序号必须小于航班 $b$ 的起飞序号。

小 X 思考的第一个问题是，若给定以上两类限制条件，是否可以计算出一个可行的起飞序列。第二个问题则是，在考虑两类限制条件的情况下，如何求出每个航班在所有可行的起飞序列中的最小起飞序号。


## 说明/提示

### 样例解释

在样例 $1$ 中：

起飞序列 $3\ 5\ 1\ 4\ 2$ 满足了所有的限制条件，所有满足条件的起飞序列有：

$$
\begin{aligned}
3\ 4\ 5\ 1\ 2\\
3\ 5\ 1\ 2\ 4\\
3\ 5\ 1\ 4\ 2\\
3\ 5\ 4\ 1\ 2\\
5\ 3\ 1\ 2\ 4\\
5\ 3\ 1\ 4\ 2\\
5\ 3\ 4\ 1\ 2
\end{aligned}
$$

由于存在 $(5,1)$ 和 $(3,1)$ 两个限制，航班 $1$ 只能安排在航班 $5$ 和 $3$ 之后，故最早起飞时间为 $3$，其他航班类似。

在样例 $2$ 中：

虽然航班 $4,5$ 没有相对起飞顺序限制，但是由于航班 $1,2,3$ 都必须安排在前 $3$ 个起飞，所以 $4,5$ 最早只能安排在第 $4$ 个起飞。

### 数据范围

对于 $30\%$ 数据：$n\leq 10$。

对于 $60\%$ 数据：$n\leq 500$。

对于 $100\%$ 数据：$n\leq 2\times 10^3,m\leq 10^4$。

## 题解思路与解决难点  
### 核心算法流程  
1. **反图拓扑排序**：建立反向图（将原依赖关系反向），按最晚起飞时间限制从大到小贪心选择航班。
2. **k值动态更新**：通过DFS或拓扑排序预处理每个航班的真实k值，确保依赖关系的合法性。
3. **最小起飞时间计算**：对每个航班i，通过反图DFS标记其所有前驱，其他航班按k值从后往前填充，最早可用位置为最后一个未被占用的合法位置。

### 关键难点对比  
| 题解方法               | 核心数据结构       | 时间复杂度       | 解决难点关键点                         |
|------------------------|--------------------|------------------|----------------------------------------|
| 反图贪心拓扑排序       | 优先队列           | O(nm)            | 时光倒流思想，动态维护可选集合         |
| DFS预处理k值           | DFS+数组分组       | O(n(m+n))        | 通过DFS传递依赖约束，重构k值分组       |

## 最优思路提炼  
### 关键技巧  
1. **反图贪心**：将原问题转换为反向图的拓扑排序，按k值从大到小填充，确保最晚限制优先满足。  
2. **延迟排除法**：计算最小起飞时间时，强制排除目标航班及其前驱，观察其他航班的填充极限。  
3. **动态k值更新**：通过DFS或拓扑排序预处理依赖约束，确保每个航班的k值合法。

```javascript
// 伪代码：优先队列操作可视化核心逻辑
class VisualScheduler {
  constructor() {
    this.queue = new PriorityQueue((a,b) => a.k > b.k);
    this.pixelGrid = new PixelGrid(16, 16); // 16x16像素网格
  }

  async addFlight(flight) {
    this.pixelGrid.highlight(flight.pos, COLORS.CYAN); // 高亮当前操作
    playSound('beep', 1000 + flight.k * 50); // k值越大音调越高
    await this.queue.push(flight);
  }

  async schedule() {
    while (!this.queue.empty()) {
      const flight = this.queue.pop();
      this.pixelGrid.fill(flight.pos, COLORS.GREEN); // 填充为绿色
      playSound('pop', 800);
      
      flight.dependents.forEach(dep => {
        dep.k = Math.min(dep.k, flight.k - 1);
        this.pixelGrid.blink(dep.pos); // 依赖更新提示
      });
    }
  }
}
```

```cpp
void Solve2(int u) {
  memset(rvis, 0, sizeof(rvis));
  rdfs(u); // 反图DFS标记前驱
  
  int p = n;
  for (int j = n; j >= 1; j--) {
    for (int t : k_groups[j]) {
      if (!rvis[t]) { // 非u的前驱航班
        if (--p < j) goto found; // 关键停止条件
      }
    }
  }
found:
  printf("%d ", p);
}
```

```javascript
class PixelAnimator {
  constructor(canvas) {
    this.ctx = canvas.getContext('2d');
    this.grid = Array(16).fill().map(() => Array(16).fill(0));
  }

  drawFlight(pos, color) {
    const [x, y] = [pos % 16, Math.floor(pos / 16)];
    this.ctx.fillStyle = COLORS[color];
    this.ctx.fillRect(x*10, y*10, 8, 8); // 8x8像素块
  }

  async highlight(pos, color) {
    this.drawFlight(pos, color);
    await sleep(300);
    this.drawFlight(pos, this.grid[y][x]); // 恢复原色
  }
}
```

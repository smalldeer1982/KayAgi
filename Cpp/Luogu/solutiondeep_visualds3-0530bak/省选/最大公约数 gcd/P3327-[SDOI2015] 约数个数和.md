# 题目信息

# [SDOI2015] 约数个数和

## 题目描述

设 $d(x)$ 为 $x$ 的约数个数，给定 $n,m$，求    
$$\sum_{i=1}^n\sum_{j=1}^md(ij)$$


## 说明/提示

【数据范围】   
对于 $100\%$ 的数据，$1\le T,n,m \le 50000$。


## 样例 #1

### 输入

```
2
7 4
5 6```

### 输出

```
110
121```

# AI分析结果



**唯一算法分类**: 莫比乌斯反演与数论分块

---

**综合分析与结论**

本题需要高效计算 $\sum_{i=1}^n\sum_{j=1}^m d(ij)$，核心在于利用莫比乌斯反演和数论分块优化。关键步骤如下：

1. **公式转换**  
   利用性质 $d(ij) = \sum_{x|i}\sum_{y|j} [\gcd(x,y)=1]$，将问题转换为双重求和形式。
   
2. **莫比乌斯反演**  
   通过替换条件 $[\gcd(x,y)=1] = \sum_{d|\gcd(x,y)} \mu(d)$，将原式拆解为多层求和。

3. **分块预处理**  
   预处理函数 $g(k)=\sum_{i=1}^k \lfloor k/i \rfloor$，该函数表示约数个数前缀和，用数论分块在 $O(n\sqrt{n})$ 时间内完成。

4. **双分块优化**  
   对 $\mu(d)$ 的前缀和进行分块，结合 $g(\lfloor n/d \rfloor)$ 和 $g(\lfloor m/d \rfloor)$，将单次查询复杂度降至 $O(\sqrt{n})$。

**核心代码实现**

```cpp
// 预处理 mu 和 g 数组
void init() {
    mu[1] = 1;
    for (int i = 2; i <= N; ++i) {
        if (!vis[i]) prime[++cnt] = i, mu[i] = -1;
        for (int j = 1; j <= cnt && i * prime[j] <= N; ++j) {
            vis[i * prime[j]] = 1;
            if (i % prime[j] == 0) break;
            mu[i * prime[j]] = -mu[i];
        }
    }
    // 计算 g(x)
    for (int x = 1; x <= N; ++x) {
        long long res = 0;
        for (int l = 1, r; l <= x; l = r + 1) {
            r = x / (x / l);
            res += 1LL * (r - l + 1) * (x / l);
        }
        g[x] = res;
    }
}

// 主查询逻辑
long long solve(int n, int m) {
    long long ans = 0;
    for (int l = 1, r; l <= min(n, m); l = r + 1) {
        r = min(n / (n / l), m / (m / l));
        ans += (mu[r] - mu[l - 1]) * g[n / l] * g[m / l];
    }
    return ans;
}
```

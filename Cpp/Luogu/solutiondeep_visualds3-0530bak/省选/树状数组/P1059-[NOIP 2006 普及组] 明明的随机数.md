# 题目信息

# [NOIP 2006 普及组] 明明的随机数

## 题目描述

明明想在学校中请一些同学一起做一项问卷调查，为了实验的客观性，他先用计算机生成了 $N$ 个 $1$ 到 $1000$ 之间的随机整数 $(N\leq100)$，对于其中重复的数字，只保留一个，把其余相同的数去掉，不同的数对应着不同的学生的学号。然后再把这些数从小到大排序，按照排好的顺序去找同学做调查。请你协助明明完成“去重”与“排序”的工作。


## 说明/提示

NOIP 2006 普及组 第一题


## 样例 #1

### 输入

```
10
20 40 32 67 40 20 89 300 400 15
```

### 输出

```
8
15 20 32 40 67 89 300 400
```

## 题解思路与难点分析

### 核心思路
题目要求对一组 1~1000 的数字进行**去重**和**排序**。所有题解围绕以下思路展开：
1. **桶排序标记法**：利用数值范围小（1~1000）的特点，用布尔数组标记出现过的数字。遍历时同时完成去重与排序。
2. **排序+相邻去重**：先对数组排序，后遍历跳过重复元素（如 `a[i] != a[i-1]`）。
3. **STL 容器**：使用 `set` 自动去重排序，或 `sort + unique` 组合处理。

### 解决难点
- **去重与排序的耦合**：桶排序天然将去重与排序统一，无需额外操作。
- **时间复杂度优化**：桶排序 O(n + 1000) 最优，STL 方法 O(n log n) 但代码极简。

## 最优思路提炼
**桶排序标记法**（qr小盆友）：
1. **布尔数组标记**：`book[a[i]] = 1` 记录数字是否出现。
2. **一次遍历统计总数**：若首次出现，计数器 `final++`。
3. **顺序遍历输出**：从 1~1000 遍历，存在标记则输出，天然有序。

```cpp
// 核心代码片段
bool book[1001] = {0};
int final = 0;
for (int i = 0; i < n; i++) {
    if (!book[a[i]]) final++;
    book[a[i]] = 1;
}
for (int i = 1; i <= 1000; i++) {
    if (book[i]) cout << i << " ";
}
```

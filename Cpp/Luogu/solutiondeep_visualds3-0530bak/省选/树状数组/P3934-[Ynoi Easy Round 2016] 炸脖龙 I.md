## 题目描述

您正在打 galgame，然后您觉得这个 gal 不知所云，于是您弃坑了，开始写数据结构题：

给一个长为 $n$ 的序列，$m$ 次操作，每次操作：

1. 区间 $[l,r]$ 加 $x$；
2. 对于区间 $[l,r]$，查询：

$$a[l]^{a[l+1]^{a[l+2]^{\dots ^{a[r]}}}} \mod p$$

### 【核心思路与解决难点】  
**核心思路**  
1. **扩展欧拉定理**：递归处理指数塔，每一步将模数替换为欧拉函数，最多递归 \(O(\log p)\) 层。  
2. **暴力剪枝**：若区间长度超过5且没有1，指数必然超过当前模数，直接应用定理；否则暴力计算前几位。  
3. **树状数组优化**：差分树状数组维护区间加和单点查询，支持高效更新。  

**关键难点与解决**  
1. **指数是否超过模数**：通过暴力计算前5位或标记是否进行取模操作判断是否需要加 \(\varphi(p)\)。  
2. **递归边界处理**：当模数为1或区间长度为1时直接返回结果。  
3. **快速幂优化**：在快速幂中实时判断结果是否超过模数，避免溢出错误。

### 【最优技巧提炼】  
1. **扩展欧拉递归**：每层递归模数替换为 \(\varphi(p)\)，最多 \(O(\log p)\) 层。  
2. **树状数组差分**：单点查询转化为前缀和，区间加通过两次差分操作实现。  
3. **快速幂标记法**：在快速幂中实时判断结果是否超过模数，避免溢出错误。  

**代码片段**  
```cpp
// 扩展欧拉定理快速幂
int qpow(int a, int b, int p, bool &flag) {
    int res = 1; flag = false;
    a >= p ? (a %= p, flag = true) : 0;
    while (b) {
        if (b & 1) {
            res = res * a;
            if (res >= p) flag = true, res %= p;
        }
        a = a * a;
        if (a >= p) flag = true, a %= p;
        b >>= 1;
    }
    return res;
}
```

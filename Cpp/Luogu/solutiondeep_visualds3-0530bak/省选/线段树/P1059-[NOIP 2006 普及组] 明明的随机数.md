# 题目信息

# [NOIP 2006 普及组] 明明的随机数

## 题目描述

明明想在学校中请一些同学一起做一项问卷调查，为了实验的客观性，他先用计算机生成了 $N$ 个 $1$ 到 $1000$ 之间的随机整数 $(N\leq100)$，对于其中重复的数字，只保留一个，把其余相同的数去掉，不同的数对应着不同的学生的学号。然后再把这些数从小到大排序，按照排好的顺序去找同学做调查。请你协助明明完成“去重”与“排序”的工作。


## 说明/提示

NOIP 2006 普及组 第一题


## 样例 #1

### 输入

```
10
20 40 32 67 40 20 89 300 400 15
```

### 输出

```
8
15 20 32 40 67 89 300 400
```

## 题解思路与解决难点

### 核心思路对比
1. **桶排序法**  
   - **实现方式**：利用数组下标直接标记数字是否出现，遍历时统计唯一数并输出  
   - **难点**：需处理值域与内存的平衡（值域1-1000时最优）  
   - **代表题解**：qr小盆友、Stone_Xz、Stars_Traveller  

2. **排序后去重**  
   - **实现方式**：先对数组排序，遍历时跳过重复元素  
   - **难点**：需注意重复元素的相邻判断逻辑（`a[i] != a[i-1]`）  
   - **代表题解**：2023z、jscaj0921  

3. **STL容器法**  
   - **实现方式**：使用`set`自动去重排序，或`sort+unique`组合  
   - **难点**：需掌握STL容器的迭代器操作  
   - **代表题解**：mcturtle（set）、BetterGlow（unique）

## 核心代码实现（桶排序法）
```cpp
#include<bits/stdc++.h>
using namespace std;
int main() {
    bool book[1001] = {0};
    int n, x, cnt = 0;
    cin >> n;
    for (int i=0; i<n; i++) {
        cin >> x;
        if (!book[x]) cnt++;
        book[x] = 1;
    }
    cout << cnt << endl;
    for (int i=1; i<=1000; i++) {
        if (book[i]) cout << i << " ";
    }
    return 0;
}
```

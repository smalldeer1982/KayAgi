# [NOIP 2012 提高组] Vigenère 密码

## 题目描述

16 世纪法国外交家 Blaise de Vigenère 设计了一种多表密码加密算法 Vigenère 密码。Vigenère 密码的加密解密算法简单易用，且破译难度比较高，曾在美国南北战争中为南军所广泛使用。

 
在密码学中，我们称需要加密的信息为明文，用 $M$ 表示；称加密后的信息为密文，用 $C$ 表示；而密钥是一种参数，是将明文转换为密文或将密文转换为明文的算法中输入的数据，记为 $k$。在 Vigenère 密码中，密钥 $k$ 是一个字母串，$k=k_1,k_2,…,k_n$。当明文 $M=m_1,m_2,…,m_n$ 时，得到的密文 $C=c_1,c_2,…,c_n$，其中 $c_i$=$m_i \operatorname{\circledR} k_i$，运算 $\circledR$ 的规则如下表所示：

Vigenère 加密在操作时需要注意：

1. $\circledR$ 运算忽略参与运算的字母的大小写，并保持字母在明文 $M$ 中的大小写形式；
2. 当明文 $M$ 的长度大于密钥 $k$ 的长度时，将密钥 $k$ 重复使用。

例如，明文 $M=\texttt{Helloworld}$，密钥 $k=\texttt{abc}$ 时，密文 $C=\texttt{Hfnlpyosnd}$。

$$

$$

### 题解思路与核心难点
1. **核心思路**：Vigenère密码的解密本质是加密的逆过程。加密时每个明文字母根据密钥字母进行位移，解密时则反向位移。例如，密钥字母`B`对应位移量1，加密时将明文字母+1，解密时则-1。
2. **核心公式**：明文 = (密文字母数值 - 密钥字母数值 + 26) % 26。通过+26确保结果非负。
3. **难点处理**：
   - **密钥循环**：通过取模运算`i % key_length`实现密钥的循环使用。
   - **大小写保留**：根据密文字母的大小写，决定明文字母的大小写。
   - **位移量计算**：将密钥字母统一转换为0-25的数值（如`A/a→0`）。

---

### 题解评分（≥4星）
1. **作者：ingu（5星）**
   - **亮点**：利用位运算快速计算位移量，代码简洁高效。例如`k[i] & 31`提取字母序号。
   - **代码**：
     ```cpp
     for (int i=0;i<c.length();i++) {
         int t=(k[i%k.length()]&31)-1;
         c[i]=(c[i]&31)-t>0?c[i]-t:c[i]-t+26;
     }
     ```
2. **作者：ShirasuAzusaSaikou（4星）**
   - **亮点**：直接推导公式，直观处理大小写。利用`tolower`统一处理密钥。
   - **代码片段**：
     ```cpp
     ((m[i] >= 'a') ? 'a' : 'A') + (tolower(m[i]) - k[ki] + 26) % 26

#include <iostream>
using namespace std;

int main() {
    string k, c;
    cin >> k >> c;
    for (int i = 0; i < c.size(); i++) {
        char key = tolower(k[i % k.size()]);
        int shift = key - 'a';
        char &ch = c[i];
        bool is_upper = isupper(ch);
        ch = tolower(ch);
        ch = (ch - 'a' - shift + 26) % 26 + 'a';
        if (is_upper) ch = toupper(ch);
    }
    cout << c << endl;
    return 0;
}
```

3. **解密运算**：`(ch - 'a' - shift + 26) % 26`处理负数并取模。
4. **大小写恢复**：根据原密文字符的大小写设置明文字符。

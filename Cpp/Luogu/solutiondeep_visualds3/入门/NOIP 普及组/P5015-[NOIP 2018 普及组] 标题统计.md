# [NOIP 2018 普及组] 标题统计

## 题目背景

NOIP2018 普及组 T1

## 题目描述

凯凯刚写了一篇美妙的作文，请问这篇作文的标题中有多少个字符？ 注意：标题中可能包含大、小写英文字母、数字字符、空格和换行符。统计标题字符数时，空格和换行符不计算在内。 

## 说明/提示

**样例 1 说明**   

标题中共有 3 个字符，这 3 个字符都是数字字符。   

**样例 2 说明**

标题中共有 $ 5$ 个字符，包括 $1$ 个大写英文字母， $1$ 个小写英文字母和 $2$ 个数字字符， 还有 $1$ 个空格。由于空格不计入结果中，故标题的有效字符数为 $4$ 个。 

**数据规模与约定**

规定 $|s|$ 表示字符串 $s$ 的长度（即字符串中的字符和空格数）。    
对于 $40\%$ 的数据，$1 ≤ |s| ≤ 5$，保证输入为数字字符及行末换行符。   
对于 $80\%$ 的数据，$1 ≤ |s| ≤ 5$，输入只可能包含大、小写英文字母、数字字符及行末换行符。   
对于 $100\%$ 的数据，$1 ≤ |s| ≤ 5$，输入可能包含大、小写英文字母、数字字符、空格和行末换行符。 

## 样例 #1

### 输入

```
234 ```

### 输出

```
3```

## 样例 #2

### 输入

```
Ca 45 ```

### 输出

```
4```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”都在“输入处理”和“条件判断”上。结合题解的共性，我总结了3个核心难点和解决策略：
</difficulty_intro>

1.  **关键点1：如何读取包含空格的输入？**
    * **分析**：C++中`cin`默认会跳过空格、换行、Tab，所以如果用`cin>>s`读字符串，会停在空格处；而`getline(cin, s)`能读完整行（包括空格），但不会读入换行符。根据题目要求（输入是一行），选`getline`是“整行处理”的首选；如果想“自动跳过空格”，用`cin`读字符/字符串更方便。
    * 💡 **学习笔记**：选对输入函数，能省一半力气！

2.  **关键点2：如何判断有效字符？**
    * **分析**：有效字符是“字母（大小写）或数字”，无效是“空格或换行”。有两种判断方式：
      - 正向：`(s[i]>='0'&&s[i]<='9')||(s[i]>='a'&&s[i]<='z')||(s[i]>='A'&&s[i]<='Z')`；
      - 逆向：`s[i]!=' '&&s[i]!='\n'`（但换行符不会被`getline`读入，所以只需判断空格）；
      - 偷懒：用`isalnum(s[i])`（库函数，判断是否是字母/数字）。
    * 💡 **学习笔记**：逆向判断或用库函数，能减少代码量！

3.  **关键点3：如何利用输入特性简化代码？**
    * **分析**：`cin`读字符/字符串时会自动跳过空白符（空格、换行、Tab），所以如果我们直接读有效字符，根本不用处理空格！比如`while(cin>>c)`循环，每读一个`c`都是有效字符，直接计数就行。这种“借输入函数的力”的思路，能让代码更简洁。
    * 💡 **学习笔记**：善用语言特性，比“硬写逻辑”更聪明！

通过这道题，我们能学到3个通用技巧：

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**通用的“整行处理”核心代码**——这是最容易理解的标准解法，适合刚学字符串的同学！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了yedalong、jiqihang等题解的思路，用`getline`读整行，遍历统计非空格字符，逻辑直白。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    using namespace std;

    int main() {
        string s;
        getline(cin, s); // 读完整行（包括空格）
        int ans = 0;
        for (int i = 0; i < s.size(); ++i) {
            if (s[i] != ' ') { // 排除空格
                ans++;
            }
        }
        cout << ans << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 1. 用`getline(cin, s)`读入整个标题行（包括空格）；
    > 2. 遍历字符串`s`的每个字符，只要不是空格，就把计数器`ans`加1；
    > 3. 最后输出`ans`，就是有效字符数。

* **亮点**：利用数据规模≤5的特点，直接读5次字符，能读入就计数，代码极简！

    ```cpp

* **代码解读**：
    > 因为输入长度最多5，所以用5次`if(cin>>c)`——`cin>>c`会自动跳过空格/换行，成功读入说明是有效字符，`ans`加1。如果输入只有3个有效字符，后面2次`cin`会失败，`ans`就停在3，正好是答案！
* 💡 **学习笔记**：数据规模小的时候，“暴力”比“通用”更高效！

**题解二：Malkin_Moonlight的“库函数偷懒”**
* **亮点**：用`isalnum`函数判断有效字符，省掉了冗长的条件判断！
* **核心代码片段**：
    ```cpp
    #include <cctype> // isalnum需要这个头文件
    // ...
    if (isalnum(s[i]) != 0) {
        ans++;
    }
    ```
* **代码解读**：
    > `isalnum`是C++标准库函数，判断字符是否是字母（大小写）或数字。返回非0表示有效，0表示无效。用它代替`(s[i]>='0'&&...)`，代码更简洁！
* 💡 **学习笔记**：库函数是“偷懒”的好帮手，但要记得加对应的头文件（比如`cctype`）！

**题解三：cff_0102的“最简计数”**
* **亮点**：用`while(cin>>c)`直接计数，把输入特性用到极致！
* **核心代码片段**：
    ```cpp
    int s = 0;
    char c;
    while (cin >> c) {
        s++;
    }
    cout << s;
    ```
* **代码解读**：
    > `cin>>c`会自动跳过空格/换行，所以每次循环读入的`c`都是有效字符，直接计数`s++`。当输入结束（比如按Ctrl+D），循环停止，`s`就是答案！
* 💡 **学习笔记**：理解输入函数的特性，能写出“神级简洁”的代码！

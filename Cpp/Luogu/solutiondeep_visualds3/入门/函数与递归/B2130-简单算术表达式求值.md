# 题目信息

# 简单算术表达式求值

## 题目描述

两位正整数的简单算术运算（只考虑整数运算），算术运算为：

`+`，加法运算；

`-`，减法运算；

`*`，乘法运算；

`/`，整除运算；

`%`，取余运算。

算术表达式的格式为（运算符前后可能有空格）：

运算数运算符运算数。

请输出相应的结果。

## 样例 #1

### 输入

```
32+64```

### 输出

```
96```

#### 4. C++核心代码实现赏析
**通用核心实现（综合题解优化）**  
```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    char op;
    cin >> a >> op >> b;  // 自动跳过空格

    switch(op) {
        case '+': cout << a + b; break;
        case '-': cout << a - b; break;
        case '*': cout << a * b; break;
        case '/': cout << a / b; break;
        case '%': cout << a % b; break;
    }
    return 0;
}
```
**代码解读概要**：  
1. 输入部分用`cin`自动处理空格  
2. `switch`像多路开关精准路由运算  
3. 每个分支独立完成计算输出

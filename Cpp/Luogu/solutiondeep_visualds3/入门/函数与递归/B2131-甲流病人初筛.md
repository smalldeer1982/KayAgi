# 甲流病人初筛

## 题目描述

目前正是甲流盛行时期，为了更好地进行分流治疗，医院在挂号时要求对病人的体温和咳嗽情况进行检查，对于体温超过 $37.5$ 度（含等于 $37.5$ 度）并且咳嗽的病人初步判定为甲流病人（初筛）。现需要统计某天前来挂号就诊的病人中有多少人被初筛为甲流病人。

## 样例 #1

### 输入

```
5
Zhang 38.3 0
Li 37.5 1
Wang 37.1 1
Zhao 39.0 1
Liu 38.2 1```

### 输出

```
Li
Zhao
Liu
3```

#### 3. 核心难点辨析与解题策略
1. **双条件精准判断**
   * **分析**：必须同时满足`temp≥37.5`和`cough==true`。易错点：忽略浮点数精度（本题直接比较可行）或混淆逻辑运算符（应用`&&`而非`||`）
   * 💡 学习笔记：医疗筛查如同逻辑电路——必须所有开关同时触发

2. **输出顺序的特殊要求**
   * **分析**：题目要求先输出所有符合条件姓名，最后输出总数。优质题解采用"即时输出姓名+最终输出总数"策略避免存储中间结果
   * 💡 学习笔记：输出时序是模拟类题目的隐藏考点

3. **数据结构选择策略**
   * **分析**：在线处理（题解1）VS离线存储（题解2/3）。前者省内存（O(1)空间），后者便于数据复用。本题无需复用数据，优选在线方案
   * 💡 学习笔记：数据是否复用决定存储策略

### ✨ 解题技巧总结
- **即时处理法**：输入与处理同步进行，节省存储空间
- **见名知意变量**：如`cough`比`k`更直观
- **防御性条件**：显式写出`cough==true`增强可读性
- **循环复用**：在循环内复用变量（如`name`）减少内存碎片

#### 4. C++核心代码实现赏析
**通用核心实现参考**
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n, cough, count = 0;
    double temp;
    string name;
    
    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> name >> temp >> cough;
        if (temp >= 37.5 && cough == 1) { 
            cout << name << endl;
            count++;
        }
    }
    cout << count;
    return 0;
}
```
**代码解读概要**：采用标准输入输出流，循环读取每条患者数据。通过双条件过滤目标患者：① 体温≥37.5 ② 咳嗽状态为真。实时输出姓名并累加计数，最终输出总数。

> ② `cout<<name<<endl`——即时输出避免存储  
> ③ `ans++`——累加器同步更新  
> → 就像流水线质检员，发现次品立即贴标签并计数  

**学习笔记**：原子化操作提升代码效率

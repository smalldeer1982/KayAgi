# 整理药名

## 题目描述

医生在书写药品名的时候经常不注意大小写，格式比较混乱。现要求你写一个程序将医生书写混乱的药品名整理成统一规范的格式，即药品名的第一个字符如果是字母要大写，其他字母小写。

如将 ASPIRIN 、 aspirin 整理成 Aspirin。

## 样例 #1

### 输入

```
4
AspiRin
cisapride
2-PENICILLIN
Cefradine-6```

### 输出

```
Aspirin
Cisapride
2-penicillin
Cefradine-6```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
处理字符串时，最容易踩的“坑”其实是“规则没摸透”。结合题解的共性，我总结了3个核心难点和解决办法：
</difficulty_intro>

1.  **关键点1：如何判断字符是大写还是小写？**
    * **分析**：有两种方法——①用ASCII码范围（`'A'<=c<='Z'`是大写，`'a'<=c<='z'`是小写）；②用C++库函数（`isupper(c)`判断大写，`islower(c)`判断小写）。题解一里用了方法①，题解二用了方法②，都能准确判断。
    * 💡 **学习笔记**：记不住ASCII码？用库函数更省心；想练底层逻辑？用ASCII范围更直观。

2.  **关键点2：首字符和其他字符的规则不一样，怎么处理？**
    * **分析**：分两步走——先单独处理首字符（转大写），再用循环处理从第2位开始的所有字符（转小写）。比如题解一的`if(s[0]...){...}`处理首字符，`for(int j=1;j<=s.size()-1;j++)`处理后续字符，逻辑分明。
    * 💡 **学习笔记**：把“特殊规则”（首字符）和“通用规则”（其他字符）分开处理，能避免逻辑混乱。

3.  **关键点3：非字母字符（比如数字、符号）要不要改？**
    * **分析**：题目要求“其他字符不变”，所以**只修改字母字符**！比如输入“2-PENICILLIN”，首字符是数字“2”，不用改；后面的“P”要转小写，“-”保持不变。所有题解都只对字母字符做判断，完美符合要求。
    * 💡 **学习笔记**：处理字符串前，先想清楚“哪些字符需要改”，避免“误伤”非目标字符。

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份**最简洁的通用实现**——用STL的`transform`函数，一句话搞定字符转换！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：此代码来自题解二（TensorFlow_js），用STL函数简化了字符转换，逻辑清晰且代码量少。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    #include <algorithm> // 包含transform函数
    using namespace std;

    int main() {
        int n;
        cin >> n;
        for (int i = 0; i < n; ++i) {
            string drugname;
            cin >> drugname;
            // 首字符转大写（非字母不变）
            transform(drugname.begin(), drugname.begin() + 1, drugname.begin(), ::toupper);
            // 后续字符转小写（非字母不变）
            transform(drugname.begin() + 1, drugname.end(), drugname.begin() + 1, ::tolower);
            cout << drugname << endl;
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码先读入n，然后循环n次处理每个字符串：①用`transform`把首字符转大写（`begin()`到`begin()+1`是第一个字符）；②用`transform`把后面的字符转小写（`begin()+1`到`end()`是剩余字符）；③输出结果。`::toupper`和`::tolower`是C++标准库的大小写转换函数，会自动忽略非字母字符。

**题解一：(来源：￥隐姓埋名￥)**
* **亮点**：用手动循环处理字符，逻辑直观，适合新手理解底层逻辑。
* **核心代码片段**：
    ```cpp
    string s;
    cin >> s;
    if (s[0] >= 'a' && s[0] <= 'z')
        s[0] -= 32; // 小写转大写
    for (int j = 1; j <= s.size() - 1; ++j)
        if (s[j] >= 'A' && s[j] <= 'Z')
            s[j] += 32; // 大写转小写
    ```
* **代码解读**：
    > 这段代码先判断首字符是否是小写——如果是，就减32（因为`'a'-'A'=32`）转大写。然后循环从第1位（第二个字符）开始，判断是否是大写——如果是，就加32转小写。比如`s[0]`是`'a'`（ASCII 97），减32变成`'A'`（65）；`s[j]`是`'P'`（80），加32变成`'p'`（112）。
* 💡 **学习笔记**：手动修改ASCII码是字符转换的“底层密码”，理解了这个，你就能自己实现大小写转换啦！

**题解三：(来源：Evan_S)**
* **亮点**：用字符差值计算转换，避免记“加32还是减32”的麻烦。
* **核心代码片段**：
    ```cpp
    if ('a' <= str[0] && str[0] <= 'z')
        str[0] = str[0] - 'a' + 'A'; // 小写转大写
    for (int i = 1; i < str.size(); ++i)
        if ('A' <= str[i] && str[i] <= 'Z')
            str[i] = str[i] - 'A' + 'a'; // 大写转小写
    ```
* **代码解读**：
    > 比如`str[0]`是`'a'`，`'a'-'a'=0`，加`'A'`就是`'A'`；`str[i]`是`'P'`，`'P'-'A'=15`，加`'a'`就是`'p'`。这种写法不用记32，直接用字符的相对位置计算，更不容易错！
* 💡 **学习笔记**：用“目标字符 = 当前字符 - 原基准 + 新基准”的公式，能轻松实现大小写转换～

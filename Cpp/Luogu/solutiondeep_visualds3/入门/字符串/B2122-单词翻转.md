# 单词翻转

## 题目描述

小明同学写单词的时候喜欢反着写，比如 `hello` 他会写成 `olleh`。给出小明同学写的一个句子，请你将所有的单词复原。

## 说明/提示

句子长度不超过 $100$，仅包含小写字母和空格，不存在多余的空格。


## 样例 #1

### 输入

```
olleh dlrow```

### 输出

```
hello
world```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`字符串处理（编程技巧应用）`  

🗣️ **初步分析**：  
解决“单词翻转”的关键，在于**逐个处理输入的每个单词，并反转每个单词的字符顺序**——就像把一串彩色珠子“倒过来穿”：比如“olleh”要变成“hello”，只需把每个字符的位置从后往前重新排列。  

在本题中，我们需要解决两个核心问题：  
1. **如何分割单词**？题目中的句子用空格分隔单词，而`cin >> s`会自动以空格为分界读取每个单词（完美贴合需求！）；  
2. **如何反转单词**？可以用“循环倒序输出”（基础方法）或“STL的`reverse`函数”（简洁方法）。  

核心算法流程是：`读入单词 → 反转字符 → 输出单词`。  

**可视化设计思路**：我们将用“8位像素风打字机”模拟反转过程——每个单词的字符像“弹珠”一样从右到左交换位置，高亮当前交换的字符，伴随“叮”的像素音效，让你直观“看”到反转的每一步！

## 3. 核心难点辨析与解题策略

在解决这道题时，大家常遇到3个“小卡点”，结合优质题解的经验，我们一一突破：


### 1. 关键点1：如何正确分割每个单词？  
* **分析**：题目中的句子用空格分隔单词，而`cin >> s`天生就会“跳过空格，读取下一个单词”——比如输入“olleh dlrow”，`cin`会先读“olleh”，再读“dlrow”，完美分割！  
* 💡 **学习笔记**：`cin >> s`是处理“空格分割单词”的“神器”！


### 2. 关键点2：如何反转字符串？  
* **分析**：有两种常用方法：  
  - **基础法**：用`for`循环从`s.size()-1`（最后一个字符的下标）遍历到`0`，逐个输出字符（如StarryWander的题解）；  
  - **简洁法**：用STL的`reverse`函数，直接反转字符串（如baiABC的题解）。  
* 💡 **学习笔记**：优先用`reverse`函数——代码更短，不易出错！


### 3. 关键点3：如何处理输入的结束？  
* **分析**：`while (cin >> s)`会一直读入单词，直到“文件结束”（评测机会自动传入EOF信号）——不需要手动判断“有没有更多单词”！  
* 💡 **学习笔记**：`while (cin >> s)`是处理“多单词输入”的标准写法！


### ✨ 解题技巧总结  
- 用`cin >> s`自动分割单词；  
- 反转字符串优先用`reverse`函数，基础款用循环倒序输出；  
- 多单词输入用`while (cin >> s)`循环处理。


## 4. C++核心代码实现赏析

先看一个**通用、简洁的核心实现**，再剖析各题解的“精妙片段”：


### 本题通用核心C++实现参考  
* **说明**：综合了优质题解的思路，用STL函数实现，代码最短、最易读。  
* **完整核心代码**：  
```cpp
#include <iostream>
#include <string>
#include <algorithm>  // 包含reverse函数
using namespace std;

int main() {
    string s;
    while (cin >> s) {       // 循环读入每个单词
        reverse(s.begin(), s.end());  // 反转字符串
        cout << s << endl;   // 输出反转后的单词
    }
    return 0;
}
```
* **代码解读概要**：  
  1. 包含必要头文件（`iostream`处理输入输出，`string`用`string`类，`algorithm`用`reverse`函数）；  
  2. 用`while (cin >> s)`循环读入每个单词；  
  3. 用`reverse`函数反转单词；  
  4. 输出反转后的单词并换行。


### 针对各优质题解的片段赏析

#### 题解一：(来源：baiABC)  
* **亮点**：用STL函数简化代码，一行搞定反转！  
* **核心代码片段**：  
```cpp
reverse(s.begin(), s.end());  // 反转字符串
```
* **代码解读**：  
  `reverse`函数需要两个**迭代器参数**：`s.begin()`是字符串的“起始位置”（第一个字符），`s.end()`是“结束位置”（最后一个字符的**下一个位置**）。函数会自动反转这两个位置之间的所有字符——比如“olleh”会变成“hello”！  
* 💡 **学习笔记**：`reverse`函数是C++ STL给我们的“偷懒神器”，一定要记住！


#### 题解二：(来源：Jorisy)  
* **亮点**：详细解释`reverse`函数的参数，帮你灵活运用！  
* **核心代码片段**：  
```cpp
reverse(s.begin(), s.end());        // 反转整个字符串（如"Luogu"→"ugouL"）
reverse(s.begin()+1, s.end());      // 从第二个字符开始反转（如"Luogu"→"Lugou"）
reverse(s.begin(), s.end()-2);      // 反转到倒数第三个字符（如"Luogu"→"ouLgu"）
```
* **代码解读**：  
  `s.begin()+1`表示“跳过第一个字符”，`s.end()-2`表示“停止在倒数第三个字符”——通过调整参数，你可以反转字符串的**任意区间**！  
* 💡 **学习笔记**：`reverse`函数的参数是“区间”，不是“长度”，要注意边界！


#### 题解三：(来源：StarryWander)  
* **亮点**：用基础循环还原反转的本质！  
* **核心代码片段**：  
```cpp
for (int i = s.size()-1; i >= 0; i--) {  // 从最后一个字符遍历到第一个
    cout << s[i];                        // 输出当前字符
}
```
* **代码解读**：  
  `s.size()`是字符串的长度（比如“olleh”的长度是5），所以最后一个字符的下标是`5-1=4`。循环从`i=4`（字符'h'）开始，到`i=0`（字符'o'）结束，逐个输出字符——相当于把字符串“倒过来”写！  
* 💡 **学习笔记**：如果想理解“反转”的底层逻辑，一定要写一次这个循环！

## 7. 学习心得与经验分享 (若有)

从题解中，我们能学到很多“过来人”的经验：  

> **参考经验 (来自Jorisy)**：“`reverse`函数的两个参数是开始和结束位置，比如`reverse(s.begin()+1, s.end())`会反转从第二个字符开始的部分。”

> **点评**：这是字符串的“基础中的基础”！比如“hello”的长度是5，最后一个字符的下标是4，循环时要从4开始，到0结束——千万不要写成`i <= s.size()`，否则会越界！

# 题目信息

# [信息与未来 2014] 夏令营旗手

## 题目描述

2014 年江苏省“信息与未来”小学夏令营在金陵中学河西分校进行，组委会决定从河西分校的学生中，推选一位小旗手，并输出其编号。

河西分校的每一位同学都有一个汉语拼音的名字，名字中的字符全部为大写的英文字母，且无空格，例如：

- 名字：王小明
- 拼音：$\tt WANGXIAOMING$

每个大写字母都有一个对应的 ASCII 码，如下表：
| 字符 | 编码（十进制） |
| :----------: | :----------: |
| $\tt A$ | $65$ |
| $\tt B$ | $66$ |
| $\cdots$ | $\cdots$ |
| $\tt Z$ | $90$ |

名字中各字符的 ASCII 码的和即为该学生的编号。

你的任务是根据给出的学生的名字，输出对应的编号。

## 说明/提示

$1\le |s|\le 20$。

## 样例 #1

### 输入

```
ABCD```

### 输出

```
266```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的难点都在“基础细节”上，结合题解的经验，我帮你提炼了3个关键点：
</difficulty_intro>

1.  **关键点1：字符怎么转ASCII值？**
    * **分析**：C++里的字符本身就存储着ASCII值！比如字符'A'的ASCII是65，直接写`ans += 'A'`就会加65。题解里的`s[i]-'0'+48`其实是绕了个弯，但结果和直接加`s[i]`一样（因为`'0'`的ASCII是48，`s[i]-48+48 = s[i]`）。  
    * 💡 **学习笔记**：字符的ASCII值不用复杂转换，直接用字符变量参与运算就行！

2.  **关键点2：怎么遍历字符串的每个字符？**
    * **分析**：C++的字符串像数组一样，可以用**下标循环**（`for(int i=0; i<s.size(); i++)`）或者**范围for循环**（`for(char c : s)`）遍历。题解用了下标循环，适合需要索引的场景；范围for循环更简洁，适合只需要字符本身的情况。  
    * 💡 **学习笔记**：遍历字符串的常用方式是“下标循环”或“范围for循环”，选你觉得顺手的！

3.  **关键点3：变量类型要选对吗？**
    * **分析**：题目里字符串最长20个字符，每个字符最大ASCII是90（'Z'），总和最大是20×90=1800，用`int`完全够。题解用了`long long`，虽然没必要，但养成“用更大类型避免溢出”的习惯很好！  
    * 💡 **学习笔记**：根据数据范围选变量类型，比如总和小用`int`，大就用`long long`。

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**最简洁的通用实现**，帮你把握整体框架：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了题解思路，用“范围for循环”简化遍历，直接累加字符的ASCII值，是最直观的实现。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    using namespace std;

    int main() {
        string s;
        cin >> s;  // 读入名字字符串
        int ans = 0;  // 初始化总和为0
        for (char c : s) {  // 范围for循环，遍历每个字符c
            ans += c;  // 直接加字符的ASCII值
        }
        cout << ans << endl;  // 输出总和
        return 0;
    }
    ```
* **代码解读概要**：代码分三步：1. 读入名字字符串；2. 用范围for循环遍历每个字符，累加ASCII值；3. 输出总和。范围for循环把“下标i”的麻烦去掉了，更易读！

再看题解里的核心片段，理解“实验法”的应用：

**题解一：来源：Max_robot**
* **亮点**：用小实验找到字符转换规律，代码极简。
* **核心代码片段**：
    ```cpp
    for(int i=0;i<s.size();i++)
        ans+=s[i]-'0'+48;

* **代码解读**：
    > 这段代码的关键是` s[i]-'0'+48 `：`s[i]`是当前字符的ASCII值（比如'A'是65），`'0'`的ASCII是48，所以`65-48=17`，再加48又回到65——其实等价于`ans += s[i]`！作者通过实验发现这个规律，虽然绕了点，但结果正确。  
    > 为什么要这么写？因为作者先测试了` s[i]-'0' `的结果（比如'A'得到17），再想到加48就能得到正确的ASCII值——这就是“实验法”的魅力！
* 💡 **学习笔记**：实验法能帮你解决“想不通的转换问题”，比如不确定字符怎么转数字时，写段小代码测一测！

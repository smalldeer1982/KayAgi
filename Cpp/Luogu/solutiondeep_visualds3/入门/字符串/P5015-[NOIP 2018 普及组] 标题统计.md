# 题目信息

# [NOIP 2018 普及组] 标题统计

## 题目背景

NOIP2018 普及组 T1

## 题目描述

凯凯刚写了一篇美妙的作文，请问这篇作文的标题中有多少个字符？ 注意：标题中可能包含大、小写英文字母、数字字符、空格和换行符。统计标题字符数时，空格和换行符不计算在内。 

## 说明/提示

**样例 1 说明**   

标题中共有 3 个字符，这 3 个字符都是数字字符。   

**样例 2 说明**

标题中共有 $ 5$ 个字符，包括 $1$ 个大写英文字母， $1$ 个小写英文字母和 $2$ 个数字字符， 还有 $1$ 个空格。由于空格不计入结果中，故标题的有效字符数为 $4$ 个。 

**数据规模与约定**

规定 $|s|$ 表示字符串 $s$ 的长度（即字符串中的字符和空格数）。    
对于 $40\%$ 的数据，$1 ≤ |s| ≤ 5$，保证输入为数字字符及行末换行符。   
对于 $80\%$ 的数据，$1 ≤ |s| ≤ 5$，输入只可能包含大、小写英文字母、数字字符及行末换行符。   
对于 $100\%$ 的数据，$1 ≤ |s| ≤ 5$，输入可能包含大、小写英文字母、数字字符、空格和行末换行符。 

## 样例 #1

### 输入

```
234 ```

### 输出

```
3```

## 样例 #2

### 输入

```
Ca 45 ```

### 输出

```
4```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”主要在**输入处理**和**字符判断**。结合优质题解，我帮你提炼了3个核心难点和解决策略：
</difficulty_intro>

1.  **难点1：如何读取包含空格的输入？**  
    * **分析**：普通`cin`读字符串时，遇到空格会停止，无法读取整行。解决方法有两种：① 用`getline(cin, s)`读取整行（包括空格）；② 利用`cin`自动跳过空格的特性，直接读取有效字符（如`while(cin>>c)`）。  
    * 💡 **学习笔记**：选对输入函数，问题解决一半！

2.  **难点2：如何判断有效字符？**  
    * **分析**：有效字符是字母（大小写）和数字。可以用**正向判断**（`s[i] >= 'a' && s[i] <= 'z'`等），或**逆向排除**（`s[i] != ' ' && s[i] != '\n'`），还可以用C++库函数`isalnum`（判断是否是字母/数字）。  
    * 💡 **学习笔记**：逆向思维有时更简单——排除无效的，剩下的就是有效的！

3.  **难点3：如何利用题目数据规模优化？**  
    * **分析**：题目说输入最多5个字符，所以可以用5次`if(cin>>c)`直接统计，不用写循环。这是“见缝插针”的优化，能让代码更简洁。  
    * 💡 **学习笔记**：读题时要注意数据范围，有时能帮你省很多事！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**最简洁的通用实现**——它用到了`cin`自动跳过空格的特性，堪称“一行解决问题”的模板！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：此代码来自题解五（cff_0102），是“极简风格”的代表，利用`cin`的特性直接统计有效字符。
* **完整核心代码**：
    ```cpp
    #include<bits/stdc++.h>
    using namespace std;
    int main(){
        int sum=0; char c;
        while(cin>>c) sum++; // cin自动跳过空格/换行，每读一个有效字符sum加一
        cout<<sum;
        return 0;
    }
    ```
* **代码解读概要**：  
  代码逻辑超简单：用`while(cin>>c)`循环读取字符——只要能读到有效字符（非空格/换行），就给`sum`加一。最后输出`sum`，就是有效字符数！


<code_intro_selected>
接下来，我们看几个**有特色的代码片段**，学习不同的解题思路：
</code_intro_selected>

**题解一：来源：Lhc_fl**  
* **亮点**：利用数据规模小（≤5字符），用5次`if`代替循环，代码超短！
* **核心代码片段**：
    ```cpp
    int ans=0; char c;
    if(cin>>c) ans++;
    if(cin>>c) ans++;
    if(cin>>c) ans++;
    if(cin>>c) ans++;
    if(cin>>c) ans++;
    cout<<ans;
    ```
* **代码解读**：  
  因为输入最多5个字符，所以用5次`if(cin>>c)`——每次`cin`成功读取字符（跳过空格），就给`ans`加一。比如输入`Ca 45`，`cin`会依次读`C`、`a`、`4`、`5`，4次成功，`ans`就是4，正好是样例2的输出！
* 💡 **学习笔记**：数据规模小的时候，“暴力枚举”比循环更简单！

**题解三：来源：yedalong**  
* **亮点**：用`getline`读整行，再遍历判断非空格，逻辑直白！
* **核心代码片段**：
    ```cpp
    string s; getline(cin,s);
    int ans=0;
    for(int i=0;i<s.size();i++){
        if(s[i]!=' ') ans++; // 不是空格就计数
    }
    cout<<ans;
    ```
* **代码解读**：  
  第一步用`getline(cin,s)`读取整行（包括空格），比如输入`Ca 45`，`s`就是`"Ca 45"`。然后遍历每个字符，只要不是空格，`ans`就加一。最后输出`ans`，就是有效字符数！
* 💡 **学习笔记**：如果需要处理空格，`getline`是你的“好朋友”！

**题解四：来源：zhouxuanhong**  
* **亮点**：用`while(cin>>s)`累加字符串长度，借力`cin`的拆分功能！
* **核心代码片段**：
    ```cpp
    int ans=0; string s;
    while(cin>>s){
        ans+=s.size(); // 累加每个字符串的长度
    }
    cout<<ans;
    ```
* **代码解读**：  
  `cin>>s`会把输入拆分成不含空格的字符串，比如输入`Ca 45`，会拆成`"Ca"`和`"45"`。`s.size()`是每个字符串的长度，累加起来就是有效字符数（`2+2=4`）！
* 💡 **学习笔记**：有时候，“借力”比“自己做”更高效！

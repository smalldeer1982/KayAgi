# 题目信息

# [蓝桥杯青少年组省赛 2022] 字母组合

## 题目描述

给定 $N$ 个小写字母，然后将这 $N$ 个小写字母按照字典序排序后组合成一个字符串输出。

例如 $N=4$，$4$ 个小写字母分别为 $\tt c$、$\tt d$、$\tt a$、$\tt c$，按照字典序排序后组合成的字符串为：$\tt{accd}$。

## 样例 #1

### 输入

```
4
c d a c```

### 输出

```
accd```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题虽然简单，但新手容易在**细节处理**上栽跟头。结合常见错误，我提炼了3个核心关键点：
</difficulty_intro>

1.  **关键点1**：如何正确读取N个字母？  
    * **分析**：题目输入是“先读N，再读N个用空格分隔的小写字母”。C++中用`cin`读`char`类型时，会自动跳过空格、换行等空白符——所以直接循环N次读`char`存到数组里就行！比如：  
      ```cpp
      vector<char> chars(N);
      for (int i = 0; i < N; ++i) {
          cin >> chars[i];
      }
      ```
    * 💡 **学习笔记**：`cin`读`char`会自动跳过空白符，完美匹配“空格分隔”的输入格式！

2.  **关键点2**：如何快速实现字典序排序？  
    * **分析**：C++标准库的`sort`函数默认是**升序排序**（从小到大），而小写字母的ASCII值正好是a(97) < b(98) < ... < z(122)——所以直接调用`sort`就能完成字典序排序！比如：  
      ```cpp
      sort(chars.begin(), chars.end());
      ```
    * 💡 **学习笔记**：`sort`函数是处理排序问题的“ Swiss Army Knife”（瑞士军刀），优先用它！

3.  **关键点3**：如何将排序后的字母拼成字符串输出？  
    * **分析**：排序后的`vector<char>`里的元素已经是有序的，直接遍历它，逐个输出字符就行——比如用`for`循环或者`range-based for`（C++11及以上支持）：  
      ```cpp
      for (char c : chars) {
          cout << c;
      }
      ```
    * 💡 **学习笔记**：遍历有序数组直接输出，就是拼接字符串的最简方式！

- **技巧3**：输出有序集合时，直接遍历输出就是“拼接字符串”的最简方式！

## 4. C++核心代码实现赏析

<code_intro_overall>
下面是这道题的**通用核心C++实现**——逻辑简洁、易读，完全覆盖题目要求：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了基础排序问题的最佳实践，用`vector`存字母、`sort`排序、遍历输出，是最适合新手的实现方式。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <vector>
    #include <algorithm> // 包含sort函数

    using namespace std;

    int main() {
        int N;
        cin >> N; // 读取字母数量
        vector<char> chars(N); // 用vector存N个字母
        for (int i = 0; i < N; ++i) {
            cin >> chars[i]; // 读取每个字母（自动跳过空格）
        }
        sort(chars.begin(), chars.end()); // 按字典序升序排序
        for (char c : chars) { // 遍历输出排序后的字母
            cout << c;
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分4步：①读字母数量N；②用`vector`存N个字母（`cin`自动处理空格）；③调用`sort`排序；④遍历`vector`输出所有字母。每一步都清晰明了，没有多余的逻辑！

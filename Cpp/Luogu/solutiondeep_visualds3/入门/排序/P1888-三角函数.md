# 三角函数

## 题目描述

输入一组勾股数 $a,b,c（a\neq b\neq c）$，用分数格式输出其较小锐角的正弦值。（要求约分。）


## 说明/提示

数据保证：$a,b,c$ 为正整数且 $\in [1,10^9]$。

## 样例 #1

### 输入

```
3 5 4```

### 输出

```
3/5```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`数学（三角函数+最大公约数） + 编程技巧应用（排序）`

🗣️ **初步分析**：
> 解决这道题，关键要先想明白**直角三角形的数学规律**：  
> 1. 直角三角形中，**最长的边是斜边**（因为勾股定理里斜边的平方等于两直角边平方和）；  
> 2. **小角对小边**（比如3、4、5的三角形，最小角对的边是3，最大角对的边是5）；  
> 3. 锐角的正弦值=**对边长度 / 斜边长度**（比如小角的正弦就是最短边÷最长边）。  

所以问题转化为：**找到三个数中的“最短边”和“最长边”，然后把这两个数约分后用分数输出**。  

- **核心思路**：先排序三个数（让最短边在最前，最长边在最后）→ 计算这两个数的最大公约数（GCD）→ 约分后输出“分子/分母”。  
- **核心难点**：如何快速排序三个数？如何正确计算GCD并约分？  
- **可视化设计思路**：我们会用8位像素动画展示“三个数字块排序”和“GCD辗转相除”的过程——比如数字块从混乱到按大小排列，GCD计算时用像素箭头演示“大数除以小数”的步骤，关键操作（如交换、约分）会有“叮”的像素音效提醒。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”其实都藏在**数学规律的理解**和**代码细节**里。我们逐一拆解：
</difficulty_intro>

1.  **关键点1：如何确定“小角的对边”和“斜边”？**
    * **分析**：直角三角形的“斜边”一定是最长的边（因为勾股定理中，斜边²=直角边1²+直角边2²，所以斜边肯定比任意一条直角边长）；而“小角的对边”是最短的边（小角对小边，这是三角形的基本性质）。所以只要把三个数排序，**第一个数是最短边（对边），第三个数是斜边**。
    * 💡 **学习笔记**：先排序，再取首尾——这是解决“找最值”问题的常用技巧！

2.  **关键点2：如何高效排序三个数？**
    * **分析**：手动写`swap`需要三次判断（比如题解一的`if(a>b) swap`），但用`sort`函数只要一行代码（`sort(a+1, a+4)`或`sort(a,a+3)`），更简洁也更少出错。`sort`函数是C++标准库的“工具人”，能帮你快速排好数组！
    * 💡 **学习笔记**：能用标准库函数就不要手动写——既省时间，又减少bug！

3.  **关键点3：如何正确约分（GCD的应用）？**
    * **分析**：约分的本质是找到两个数的“最大公约数”（GCD），然后分子分母都除以这个数。比如3和5的GCD是1，所以3/5不用约分；6和9的GCD是3，约分后是2/3。C++的`__gcd`函数能直接计算两个数的GCD，但要注意：**`__gcd`的参数顺序不影响结果**（比如`__gcd(3,5)`和`__gcd(5,3)`结果都是1）。
    * 💡 **学习笔记**：约分=分子÷GCD + 分母÷GCD——记住这个公式，所有分数约分都能用！

### ✨ 解题技巧总结
<summary_best_practices>
通过这道题，我们能总结出3个通用技巧：
</summary_best_practices>
-   **技巧1：用数学规律简化问题**：先想清楚“小角对小边”“斜边最长”这些数学规律，再写代码就不会乱。
-   **技巧2：善用标准库函数**：`sort`排序、`__gcd`求最大公约数，这些函数能帮你减少重复劳动。
-   **技巧3：注意数据范围**：如果题目说数据到1e9，一定要用`long long`代替`int`——否则会溢出！


## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份**通用核心代码**——它综合了所有优质题解的优点，用最简洁的方式解决问题：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码用`sort`快速排序，`long long`处理大数字，`__gcd`约分，覆盖了所有边界情况。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <algorithm> // 包含sort函数
    using namespace std;

    int main() {
        long long a[3]; // 用long long防溢出
        cin >> a[0] >> a[1] >> a[2];
        sort(a, a + 3); // 排序后，a[0]是最短边，a[2]是斜边
        long long g = __gcd(a[0], a[2]); // 计算GCD
        cout << a[0]/g << "/" << a[2]/g << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 1. 用`long long`定义数组，避免1e9的数据溢出；  
    > 2. `sort(a, a+3)`把三个数从小到大排序；  
    > 3. `__gcd(a[0], a[2])`计算最短边和斜边的最大公约数；  
    > 4. 分子（a[0]/g）和分母（a[2]/g）就是约分后的结果。

# [USACO1.1] 你的飞碟在这儿 Your Ride Is Here

## 题目描述

众所周知，在每一个彗星后都有一只 UFO。这些 UFO 时常来收集地球上的忠诚支持者。不幸的是，他们的飞碟每次出行都只能带上一组支持者。因此，他们要用一种聪明的方案让这些小组提前知道谁会被彗星带走。他们为每个彗星起了一个名字，通过这些名字来决定这个小组是不是被带走的那个特定的小组（你认为是谁给这些彗星取的名字呢？）。关于如何搭配的细节会在下面告诉你；你的任务是写一个程序，通过小组名和彗星名来决定这个小组是否能被那颗彗星后面的 UFO 带走。


小组名和彗星名都以下列方式转换成一个数字：最终的数字就是名字中所有字母的积，其中 $\texttt A$ 是 $1$，$\texttt Z$ 是 $26$。例如，$\texttt{USACO}$ 小组就是 $21 \times 19 \times 1 \times 3 \times 15=17955$。如果小组的数字 $\bmod 47$ 等于彗星的数字 $\bmod 47$,你就得告诉这个小组需要准备好被带走！（记住“$a \bmod b$”是 $a$ 除以 $b$ 的余数，例如 $34 \bmod 10$ 等于 $4$）


写出一个程序，读入彗星名和小组名并算出用上面的方案能否将两个名字搭配起来，如果能搭配，就输出 `GO`，否则输出 `STAY`。小组名和彗星名均是没有空格或标点的一串大写字母（不超过 $6$ 个字母）。

## 3. 核心难点辨析与解题策略

做模拟题时，最容易“翻车”的就是**细节**。结合题解的经验，我总结了3个必踩的“坑”和解决方法：

1.  **乘积起始值为什么是1？**  
    * **分析**：乘法的“ identity element ”是1（就像加法的0）——如果设为0，不管乘多少个数都是0，完全错！所有题解都把`ans`或`a`初始化为1，这是必须记住的细节。  
    * 💡 **学习笔记**：乘法初始化用1，加法用0，别搞反！

2.  **字母转数字怎么算？**  
    * **分析**：大写字母的ASCII码是连续的（A=65，B=66…Z=90）。所以有两种方法：  
      - 方法一：`c - 'A' + 1`（A→65-65+1=1，最直观）；  
      - 方法二：`c - '@'`（@=64，A→65-64=1，少写一个+1）。  
    两种都对，选自己记得住的就行～  
    * 💡 **学习笔记**：ASCII码是字母转数字的“桥梁”，记住A=65！

3.  **输入怎么处理？**  
    * **分析**：题目说“小组名和彗星名均是没有空格的一串大写字母”，所以有两种输入方式：  
      - 用`string`：`cin >> s >> t`（自动按空格/换行分隔）；  
      - 用`cin.get()`：逐个读字符直到换行（适合处理“一行一个字符串”的情况）。  
    两种都能过，但`string`更简单，`cin.get()`更灵活～  
    * 💡 **学习笔记**：输入方式要匹配题目要求，别强行用复杂的方法！

### ✨ 解题技巧总结
- **按步骤拆解问题**：把“字符串转数字→相乘→取模→比较”拆成小步骤，逐个实现；  
- **用变量名传递意图**：比如`cnta`代表小组名的乘积，`turn`函数代表字母转数字，别用`x`、`y`这种模糊的名字；  
- **写注释！**：比如`// @的ASCII是64，A-@=1`，自己回头看也能快速回忆～


## 4. C++核心代码实现赏析

先看一个**通用核心实现**，综合了各题解的优点，适合新手直接用：

**本题通用核心C++实现参考**
* **说明**：本代码用`string`处理输入，逻辑清晰，注释详细，覆盖所有边界情况（比如字符串长度为1）。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    using namespace std;

    int main() {
        string comet, group; // 彗星名、小组名
        cin >> comet >> group;

        long long comet_val = 1, group_val = 1; // 乘积（用long long防止溢出，虽然本题不需要）
        // 计算彗星名的乘积
        for (char c : comet) {
            comet_val *= (c - 'A' + 1); // 字母转数字：A→1
        }
        // 计算小组名的乘积
        for (char c : group) {
            group_val *= (c - 'A' + 1);
        }

        // 取模并比较
        if (comet_val % 47 == group_val % 47) {
            cout << "GO" << endl;
        } else {
            cout << "STAY" << endl;
        }
        return 0;
    }
    ```
* **代码解读概要**：  
  1. 用`string`读入两个名字；  
  2. 用`for (char c : comet)`遍历每个字符（C++11及以上支持，比`i`循环更简洁）；  
  3. 用`c - 'A' + 1`转换字母，乘积初始化为1；  
  4. 最后取模比较，输出结果。


### 各优质题解的片段赏析

**题解一：(来源：Vanyun)**
* **亮点**：用`a[i]-'@'`快速转换字母，少写`+1`。
* **核心代码片段**：
    ```cpp
    for( int i = 0 ; i < a.length() ; i ++ ) ans *= a[i] - '@'; 
    // @=64，A=65→65-64=1，正好对应字母序号
    ```
* **代码解读**：  
  作者用`a.length()`获取字符串长度，循环遍历每个字符。`a[i]-'@'`是个小技巧——因为`@`的ASCII码比`A`小1，所以减完直接得到1~26的数字，不用再加1。是不是很聪明？
* 💡 **学习笔记**：小技巧能简化代码，但要确保自己理解原理（比如记得`@`的ASCII是64）！

**题解二：(来源：DrinkOnstage)**
* **亮点**：用`cin.get()`读字符，不用`string`也能处理输入。
* **核心代码片段**：
    ```cpp
    while ( (v = cin.get() ) != '\n') a *= v - 'A' + 1;
    // 读字符直到换行，计算乘积
    ```
* **代码解读**：  
  `cin.get()`会逐个读入字符（包括换行符）。作者用`while`循环，只要读的不是换行符（`'\n'`），就继续计算乘积。这种方法适合处理“一行一个字符串”的输入，比如用户输入`COMETQ`后按回车，再输入`HVNGAT`——`cin.get()`会准确读取每一行的字符！
* 💡 **学习笔记**：输入方式不止`cin >> s`，`cin.get()`更灵活～

**题解三：(来源：Carl__2015)**
* **亮点**：用函数封装字母转数字，代码更清晰。
* **核心代码片段**：
    ```cpp
    int turn(char c) {
        return c - 'A' + 1; // 字母转数字的函数
    }
    // 主函数里调用：
    a = a * turn(s[i]);
    ```
* **代码解读**：  
  作者把“字母转数字”的逻辑写成`turn`函数，主函数里直接调用`turn(s[i])`——这样主逻辑更简洁，别人看代码时一看`turn`就知道是做什么的。这就是**函数封装**的好处！
* 💡 **学习笔记**：重复用的逻辑写成函数，代码更易读～

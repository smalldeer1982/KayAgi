# 题目信息

# 小鱼的游泳时间

## 题目描述

伦敦奥运会要到了，小鱼在拼命练习游泳准备参加游泳比赛，可怜的小鱼并不知道鱼类是不能参加人类的奥运会的。

这一天，小鱼给自己的游泳时间做了精确的计时（本题中的计时都按 $24$ 小时制计算），它发现自己从 $a$ 时 $b$ 分一直游泳到当天的 $c$ 时 $d$ 分，请你帮小鱼计算一下，它这天一共游了多少时间呢？

小鱼游的好辛苦呀，你可不要算错了哦。


## 说明/提示

对于全部测试数据，$0\le a,c \le 24$，$0\le b,d \le 60$，且结束时间一定晚于开始时间。

## 样例 #1

### 输入

```
12 50 19 10```

### 输出

```
6 20```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的难点都围绕「时间的双单位」展开，结合优质题解的共性，我总结了3个关键思考点和解决策略：
</difficulty_intro>

1.  **关键点1：分钟不够减怎么办？**
    * **分析**：当结束分钟`d` < 开始分钟`b`时，分钟差`y`是负数——这时候要向小时「借1小时（60分钟）」，所以小时差`x`减1，分钟差`y`加60。比如样例中`d=10`、`b=50`，`y=10-50=-40`，借位后`x=19-12-1=6`，`y=-40+60=20`，正好是答案！
    * 💡 **学习笔记**：处理不同单位的减法，「借位」是通用技巧！

2.  **关键点2：如何简化计算？**
    * **分析**：把小时转成分钟（1小时=60分钟），让时间变成「单单位」——比如12:50=12×60+50=770分钟，19:10=19×60+10=1150分钟，相减得380分钟，再转成6小时20分钟。这种方法**避免了所有条件判断**，逻辑更直接！
    * 💡 **学习笔记**：统一单位能让复杂问题「降维打击」！

3.  **关键点3：总分钟数怎么转回头小时和分钟？**
    * **分析**：总分钟数除以60的「商」是小时数（比如380÷60=6），「余数」是分钟数（380%60=20）——这是因为60分钟=1小时，余数就是不够1小时的部分。
    * 💡 **学习笔记**：整除（`/`）和取模（`%`）是单位转换的「黄金搭档」！

### ✨ 解题技巧总结
<summary_best_practices>
通过这道题，我们能总结出3个通用解题技巧，适用于所有「单位计算」问题：
</summary_best_practices>
-   **技巧1：统一单位**：遇到多单位问题（比如时间、货币、重量），先转成最小单位（分钟、角、克），计算后再转回去；
-   **技巧2：模拟现实逻辑**：如果不想统一单位，就模拟现实中的计算方式（比如借位、进位）；
-   **技巧3：变量命名要「见名知意」**：用`hours_diff`代替`x`，`total_min`代替`t3`，能避免「变量弄反」的低级错误！

<code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了多个优质题解的思路，用「统一单位」的方法，逻辑直接到像「小学数学题」！
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int a, b, c, d;
        cin >> a >> b >> c >> d; // 读取开始时间（a时b分）和结束时间（c时d分）
        int total_min = (c * 60 + d) - (a * 60 + b); // 转成分钟相减
        int hours = total_min / 60; // 总分钟数→小时（商）
        int minutes = total_min % 60; // 总分钟数→分钟（余数）
        cout << hours << " " << minutes << endl; // 输出结果
        return 0;
    }
    ```
* **代码解读概要**：代码分4步——读输入→转成分钟→计算总时间→转回小时和分钟。每一步都对应「小学数学逻辑」，没有复杂的条件判断，新手也能一眼看懂！

* **代码解读**：

**题解二：数学转化（作者：2789617221guo）**
* **亮点**：统一单位后，代码没有任何条件判断，简单到「不用动脑子」！
* **核心代码片段**：
    ```cpp
    int t1 = a * 60 + b, t2 = c * 60 + d;
    int t3 = t2 - t1;
    int h = t3 / 60, m = t3 % 60;
    cout << h << " " << m;
    ```
* **代码解读**：
    > `t1`是开始时间的总分钟，`t2`是结束时间的总分钟，`t3`是总游泳时间。`h`是`t3`除以60的商（小时），`m`是余数（分钟）——比如`t3=380`，`h=6`，`m=20`，直接输出！
* 💡 **学习笔记**：把问题「数学化」，能避免很多逻辑错误！

**题解三：分类讨论（作者：GoldenSTEVE7）**
* **亮点**：用`if-else`直接处理两种情况，逻辑像「开关」一样清晰！
* **核心代码片段**：
    ```cpp
    if (d >= b) cout << c - a << " " << d - b;
    else cout << c - a - 1 << " " << d - b + 60;
    ```
* **代码解读**：
    > 如果`d>=b`（分钟够减），直接输出小时差和分钟差；否则（分钟不够减），小时差减1（借位），分钟差加60（补回借的分钟）。比如样例中`d=10 < b=50`，所以输出`19-12-1=6`和`10-50+60=20`——结果正确！
* 💡 **学习笔记**：分类讨论能把复杂问题拆成「简单子问题」！

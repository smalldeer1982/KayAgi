# 题目信息

# 西游记公司

## 题目背景

一道极其无厘头的题目。

## 题目描述

事情是这样的：  

西游记中的孙沙猪三徒弟在西天取经之后开始进入厦门大学经贸系学习经济，在 $1$ 个小时的学习后，他们用特殊手段毕业了。

然后，他们创办了三个公司：“花果山生态旅游山庄集团公司” “高老庄猪肉美食城有限公司” 和 “流沙河轮渡有限公司”。虽然这三家公司出自 “经卷出版社” 社长唐三藏的徒弟们，但是每家公司都仍然保持着 $0$ 元以下的财政收入。  

于是，他们想到了一个无聊的方法：向别人偷！其中，孙悟空偷猪八戒的平底锅之前，猪八戒可以一直抢沙和尚的笔记本。

现在，作为猪八戒手下首席智士的你，要帮助既聪明又愚蠢的猪八戒用那种不正当的手段挽救这个公司！  

你可以这样做：你已经知道孙悟空什么时候要来偷猪八戒的东西，而猪八戒又要在什么时间去抢沙和尚的东西，又知道猪八戒每秒钟可以偷沙和尚 $n$ 台笔记本，帮猪八戒算一算，自己在有限的时间内能偷到沙和尚多少台笔记本，使得他有足够的时间来防御大师兄。

## 说明/提示

对于 $100\%$ 的数据，保证两个时间合法，$n<32768$。

## 样例 #1

### 输入

```
00:0:00
0:00:10
10```

### 输出

```
100```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”其实不多，但新手容易犯三个错误。Kay帮你把“坑”挖出来，再教你怎么绕过去~
</difficulty_intro>

1.  **关键点1：输入中的冒号怎么处理？**
    * **分析**：输入是`00:0:00`这种格式，直接读数字会被冒号卡住。解决方案有三种：① 用`cin`读数字时自动跳过非数字字符（如题解一）；② 用`scanf`的格式化字符串匹配冒号（如题解二）；③ 用快读函数过滤非数字字符（如题解三）。
    * 💡 **学习笔记**：输入中的“分隔符”（比如冒号、逗号）不用怕，找对工具就能自动处理！

2.  **关键点2：如何正确把时间转成总秒数？**
    * **分析**：时间的进制是60（1小时=60分钟=3600秒），所以总秒数=小时×3600 + 分钟×60 + 秒。比如`0:00:10`就是`0×3600 + 0×60 +10=10秒`，`00:0:00`是0秒，差10秒×10=100，刚好对应样例输出~
    * 💡 **学习笔记**：时间换算的核心是“进制转换”，记住3600和60这两个 magic number！

3.  **关键点3：为什么要用`long long`？**
    * **分析**：假设时间差是1天（86400秒），n是32767，结果是86400×32767=2,831,068,800，超过了`int`的最大值（约21亿）。如果用`int`存，结果会“溢出”变成错误值。所以所有和时间、结果相关的变量都要声明为`long long`！
    * 💡 **学习笔记**：遇到“可能很大的数”，先想`long long`，别等WA了才后悔~

### ✨ 解题技巧总结
- **技巧A：利用输入工具简化处理**：`cin`自动跳过非数字，`scanf`格式化匹配，快读过滤字符——选适合自己的！
- **技巧B：进制转换公式要记牢**：时间转秒=小时×3600+分钟×60+秒，永远不会错~
- **技巧C：数据范围要敏感**：题目里说n<32768，时间差可能很大，直接用`long long`保平安！

<code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了“极简流”的简洁和“格式化输入”的清晰，用`cin`处理冒号，`long long`防止溢出，是最适合新手的版本~
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;
  int main() {
      long long h1, m1, s1, h2, m2, s2, n;
      char ch; // 用来“吃”掉冒号的变量
      cin >> h1 >> ch >> m1 >> ch >> s1; // 读第一个时间：h1:m1:s1
      cin >> h2 >> ch >> m2 >> ch >> s2; // 读第二个时间：h2:m2:s2
      cin >> n; // 读每秒偷的数量
      // 计算总秒数差 × n
      cout << (3600LL * (h2 - h1) + 60LL * (m2 - m1) + (s2 - s1)) * n << endl;
      return 0;
  }
  ```
* **代码解读概要**：
  > 代码分三步：① 用`cin`读入两个时间（`ch`用来接收冒号，不影响计算）；② 读入每秒偷的数量`n`；③ 用公式计算总秒数差，乘以`n`输出。其中`3600LL`和`60LL`是为了确保计算时是`long long`类型，避免溢出~

**题解一：极简流（作者：Right）**
* **亮点**：用`cin`的特性直接跳过冒号，代码短到极致！
* **核心代码片段**：
  ```cpp
  cin >> a >> m >> b >> m >> c >> x >> m >> y >> m >> z >> t;
  cout << ((x - a) * 3600 + (y - b) * 60 + (z - c)) * t;
  ```
* **代码解读**：
  > 这里的`m`是`char`类型，用来“吃掉”输入中的冒号。比如输入`00:0:00`，`cin`会先读`00`给`a`，然后读`:`给`m`，再读`0`给`b`，依此类推——完全不用手动处理冒号！最后用公式计算总秒数差，乘以`t`（每秒偷的数量）输出。
* 💡 **学习笔记**：`cin`会自动跳过** whitespace（空格、换行、制表符）和非数字字符**，这是简化输入的“神器”！

**题解二：格式化输入流（作者：chenyuxuan02）**
* **亮点**：用`scanf`的格式化字符串精准匹配冒号，可读性拉满！
* **核心代码片段**：
  ```cpp
  scanf("%lld:%lld:%lld", &h1, &m1, &s1);
  scanf("%lld:%lld:%lld", &h2, &m2, &s2);
  s1 = s1 + m1 * 60 + h1 * 3600;
  s2 = s2 + m2 * 60 + h2 * 3600;
  cout << (s2 - s1) * cpt << endl;
  ```
* **代码解读**：
  > `scanf("%lld:%lld:%lld", &h1, &m1, &s1)`中的`%lld`表示读`long long`类型的数字，`:`表示匹配输入中的冒号。比如输入`0:00:10`，`scanf`会把`0`给`h1`，`00`给`m1`，`10`给`s1`——完美分离！然后把时间转成总秒数`s1`和`s2`，差乘以`cpt`（每秒偷的数量）输出。
* 💡 **学习笔记**：`scanf`的格式化字符串是处理“固定格式输入”的“终极武器”，比如日期、时间、IP地址都能用它~

**题解三：快读流（作者：NishikinoMaki）**
* **亮点**：快读函数自动过滤冒号，速度比`cin`快3~5倍！
* **核心代码片段**：
  ```cpp
  inline long long read() {
      long long n = 0, k = 1;
      char ch = getchar();
      while ((ch > '9' || ch < '0') && ch != '-') ch = getchar(); // 跳过非数字
      if (ch == '-') k = -1, ch = getchar();
      while (ch <= '9' && ch >= '0') {
          n = n * 10 + ch - '0'; // 转数字
          ch = getchar();
      }
      return n * k;
  }
  // 主函数中读入：
  h1 = read(); m1 = read(); s1 = read();
  h2 = read(); m2 = read(); s2 = read();
  n = read();
  ```
* **代码解读**：
  > 快读函数的核心是`getchar()`逐个读字符：① 先跳过所有非数字字符（比如冒号）；② 然后把数字字符转成数字（`ch - '0'`是字符转数字的技巧）；③ 最后返回数字。主函数中直接调用`read()`读入时分秒，完全不用管冒号——是不是超方便？
* 💡 **学习笔记**：快读是竞赛中的“必学技巧”，尤其是处理大数据时，能避免超时~

#### **1. 整体风格**
- **8位像素风**：用FC的配色（比如天蓝色背景、黄色文字、红色按钮），所有元素都是“方块状”——比如时间用“像素数字”显示，冒号用“小方块”代替。
- **场景设定**：屏幕左边是“猪八戒的偷笔记本操作台”，中间是“时间转换机”，右边是“结果显示板”。

#### **2. 核心演示步骤**
1. **输入阶段**：
   - 屏幕上方弹出两个“输入框”，分别显示“猪八戒开始时间”和“孙悟空来的时间”，用像素键盘输入`00:0:00`和`0:00:10`。
   - 输入时，每敲一个数字，播放“滴答”的像素音效；输入完一个时间，播放“叮”的提示音。

2. **时间转秒阶段**：
   - 左边的时间会“飞到”中间的“转换机”里：`00`（小时）变成`00×3600`的方块（蓝色），`0`（分钟）变成`0×60`的方块（绿色），`00`（秒）变成`00`的方块（红色），合并成“总秒数0”（蓝色+绿色+红色方块叠在一起）。
   - 第二个时间`0:00:10`同理，转成“总秒数10”（蓝色0+绿色0+红色10）。

3. **计算差阶段**：
   - 两个总秒数方块碰撞，“0”被“10”减去，变成“10”的黄色方块（时间差）。
   - 播放“哗啦”的音效，表示计算完成。

4. **乘n阶段**：
   - 黄色方块“10”和“n=10”的紫色方块碰撞，变成“100”的橙色方块（结果）。
   - 播放“叮铃”的胜利音效，右边的结果显示板弹出“100”，同时猪八戒的像素形象跳起来庆祝~

- **控制按钮**：屏幕下方有“单步”（一步步看）、“自动”（快速播放）、“重置”（重新来一遍）按钮，用像素图标表示。

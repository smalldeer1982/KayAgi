# 向量点积计算

## 题目描述

在线性代数、计算几何中，向量点积是一种十分重要的运算。

给定两个 $n$ 维向量 $a=(a_1,a_2, \cdots ,a_n)$ 和 $b=(b_1,b_2, \cdots ,b_n)$，求点积 $a$ · $b=a_1b_1+a_2b_2+ \cdots +a_nb_n$。

## 样例 #1

### 输入

```
3
1 4 6
2 1 5```

### 输出

```
36```

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**  
* **说明**：综合题解一优化思路与题解二结构清晰性  
* **完整核心代码**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, sum = 0;
    cin >> n;
    int a[n], b[n];  // 变长数组(C99标准)
    
    for(int i=0; i<n; i++) cin >> a[i];
    for(int i=0; i<n; i++) {
        cin >> b[i];
        sum += a[i] * b[i]; // 输入与计算同步
    }
    cout << sum;
    return 0;
}
```
* **代码解读概要**：  
  1. 声明变长数组适配任意维度n  
  2. 第一循环仅存储向量a  
  3. 第二循环同时完成向量b输入和点积计算  
  4. 索引从0开始更符合C++惯例  

**题解一核心代码片段**  
```cpp
for(int i=1;i<=n;i++){
    cin>>b[i];
    sum+=a[i]*b[i]; // 输入计算同步
}
```
* **亮点**：循环融合减少开销  
* **代码解读**：  
  > 当`cin>>b[i]`读取b的当前元素后，立即与`a[i]`相乘累加到sum。这就像收银员扫描商品B时，同步用计算器乘以A的价格累加总价，效率提升的关键在于消除"单独扫描所有商品B"的步骤。  
* 💡 **学习笔记**：IO操作与计算重叠是常见优化手段  

**题解二核心代码片段**  
```cpp
int sum = 0;
for(int i=1;i<=n;i++){
    sum += a[i]*b[i];
}
```
* **亮点**：显式初始化保障正确性  
* **代码解读**：  
  > `sum=0`是安全编程的基石。未初始化的累加器可能包含随机值，就像从未知金额开始收银必然出错。循环体严格遵循数学定义∑aᵢbᵢ，索引从1到n覆盖所有元素。  
* 💡 **学习笔记**：显式初始化是防御性编程的第一原则

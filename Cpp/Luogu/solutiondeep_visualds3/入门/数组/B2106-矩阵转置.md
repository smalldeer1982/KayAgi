# 题目信息

# 矩阵转置

## 题目描述

输入一个 $n$ 行 $m$ 列的矩阵 $A$，输出它的转置 $A^T$。

## 样例 #1

### 输入

```
3 3
1 2 3
4 5 6
7 8 9```

### 输出

```
1 4 7
2 5 8
3 6 9```

#### 3. 核心难点辨析与解题策略
1. **难点：行列索引映射关系**  
   * **分析**：转置本质是坐标变换(i,j)→(j,i)，但循环顺序影响性能。优质解法均采用**列优先循环**（外层列，内层行）  
   * 💡 学习笔记：想象矩阵是棋盘，转置就是按列读取棋子  

2. **难点：边界值处理**  
   * **分析**：数组下标从0还是1开始？题解2/3/5用1-index避免零坐标混淆，题解1用0-index更贴近内存布局  
   * 💡 学习笔记：竞赛题推荐0-index（兼容STL），教学演示可用1-index  

3. **难点：输出格式控制**  
   * **分析**：每行末位空格处理是关键。题解3用`cout<<endl`显式换行比题解1的`puts("")`更C++风格  
   * 💡 学习笔记：行末空格可用条件判断避免（如`j<n? " " : "\n"`）  

### ✨ 解题技巧总结
- **空间优化**：直接输出替代存储转置矩阵（省O(n×m)空间）  
- **循环顺序**：列外循环+行内循环=天然转置  
- **防御编程**：数组开105×105防越界（题目n,m≤100）

#### 4. C++核心代码实现赏析
**通用核心实现**（综合优化版）：
```cpp
#include <iostream>
using namespace std;
int main() {
    int n, m, mat[105][105];
    cin >> n >> m;
    // 输入原矩阵
    for (int i = 0; i < n; ++i)
        for (int j = 0; j < m; ++j)
            cin >> mat[i][j];
    // 转置输出：列优先
    for (int j = 0; j < m; ++j) {
        for (int i = 0; i < n; ++i)
            cout << mat[i][j] << " ";
        cout << endl;
    }
    return 0;
}
```
**代码解读概要**：  
> ① 数组`mat`预留5单位缓冲防越界  
> ② 列优先循环直接实现坐标互换  
> ③ 每列输出后立即换行

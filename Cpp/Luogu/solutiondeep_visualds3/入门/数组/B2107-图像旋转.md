# 图像旋转

## 题目描述

输入一个 $n$ 行 $m$ 列的黑白图像，将它顺时针旋转 $90$ 度后输出。

## 样例 #1

### 输入

```
3 3
1 2 3
4 5 6
7 8 9```

### 输出

```
7 4 1
8 5 2
9 6 3```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决图像旋转问题的关键点和应对策略：

1.  **坐标变换规律推导**
    * **分析**：难点在于发现旋转后坐标的数学关系：原矩阵的(i,j) → 新矩阵的(j, n-i+1)。优质题解都通过样例分析找到规律：按列顺序输出+行逆序输出 = 旋转效果
    * 💡 **学习笔记**：动手画3×3矩阵验证坐标关系是突破点

2.  **行列维度转换**
    * **分析**：旋转后行变列、列变行。题解通过外层循环列(j)、内层循环行(i)实现维度转换，特别注意行需要逆序(n→1)
    * 💡 **学习笔记**：新矩阵行数=原列数(m)，列数=原行数(n)

3.  **下标起始统一**
    * **分析**：下标从0或1开始会影响具体实现。题解中ImposterAnYu和·糯·用1开始，Asimplename用0开始但调整了边界
    * 💡 **学习笔记**：选定一种下标规则后需保持统一

## 4. C++核心代码实现赏析

<code_intro_overall>
通用旋转实现（综合优质题解思路）：
</code_intro_overall>

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, m, a[105][105];
    cin >> n >> m;
    
    // 读入矩阵
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            cin >> a[i][j];
    
    // 旋转输出：列顺序+行逆序
    for (int j = 1; j <= m; j++) {
        for (int i = n; i >= 1; i--) {
            cout << a[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}
```
* **代码解读概要**：
  1. 创建105×105数组（比题目要求稍大，避免边界问题）
  2. 双层循环读入矩阵数据
  3. 关键旋转部分：外层遍历列(1~m)，内层逆序遍历行(n~1)
  4. 输出时每个元素后加空格，每列结束后换行

### 题解一：(ImposterAnYu)
```cpp
for(i = 1; i <= m; i++){
    for(j = n; j >= 1; j--) 
        cout<< a[j][i] << " ";
    cout<< endl;
}
```
* **亮点**：最简洁的旋转实现
* **代码解读**：
  - 外层`i`循环控制**列输出**（旋转后的行）
  - 内层`j`从`n`递减实现**行逆序**
  - 注意元素访问顺序：`a[j][i]`（行在前，列在后）
* 💡 **学习笔记**：行索引和列索引互换即得旋转效果

### 题解二：(·糯·)
```cpp
for(int i=1;i<=m;i++){
    for(int j=n;j>=1;j--)
        cout<<a[j][i]<<" ";
    cout<<endl;
}
```
* **亮点**：无冗余操作的极致简洁实现
* **代码解读**：
  - 循环结构与题解一完全相同
  - 变量名更简洁（单字母i,j）
  - 注释明确点出"逆序输出=旋转"
* 💡 **学习笔记**：优质代码往往最简洁

### 题解三：(Asimplename)
```cpp
for(int j = 1; j <= _m; j++){
    for(int i = _n; i >= 1; i--){
        cout << _map[i][j] << " ";
    } 
    cout << endl;
}
```
* **亮点**：自解释型变量命名提升可读性
* **代码解读**：
  - `_map`：存储图像的二维数组
  - `_n`/`_m`：行/列维度变量
  - 使用带下划线的变量名避免命名冲突
* 💡 **学习笔记**：描述性变量名让代码更易维护

## 题目背景

> 渡荆门送别
> 
>李白
> 
>渡远荆门外，来从楚国游。
>
>山随平野尽，江入大荒流。
>
>月下飞天镜，云生结海楼。
>
> 仍怜故乡水，万里送行舟。

## 题目描述

李白喜欢飞，也喜欢天鹅。

现在给出一排共 $n$ 只鹅的身高，李白想知道最高的鹅比其他所有鹅高多少、最矮的鹅比其他所有鹅矮多少。

请输出这两行信息。

## 说明/提示

李白一共有 $6$ 只鹅，最高的一只身高为 $8$，最矮的一只身高为 $2$，然后分别作为被减数和减数参与身高差计算可得结果。

对于 $30\%$ 数据，保证 $0\leq a_i\leq 30,1\leq n\leq 20$。

对于 $100\%$ 数据，保证 $0\leq a_i \leq 10^{18},1\leq n\leq 10^6$ 。

## 样例 #1

### 输入

```
6
4 7 8 6 3 2```

### 输出

```
4 1 0 2 5 6
2 5 6 4 1 0```

### 题目分析：渡荆门送别  
**核心算法识别**：`编程技巧应用`（打擂台求最值）  
**比喻解释**：就像在运动会上选最高和最矮的选手。我们设定两个"擂台"（`maxm`和`minm`），每个选手（数组元素）依次上台比较，胜者留在擂台上。

### 3. 核心难点辨析与解题策略  
1. **难点1：正确初始化最值**  
   * **分析**：初始值需覆盖所有可能数据。`maxm`初始0（因数据≥0），`minm`初始需>最大可能值（10¹⁸）  
   * 💡 **学习笔记**：初始值要像"守门员"——能拦截所有数据  

2. **难点2：大整数处理**  
   * **分析**：`10¹⁸`超出`int`范围，必须用`long long`。差值计算可能达`10¹⁸`，同样需要`long long`  
   * 💡 **学习笔记**：见微知著——数据范围决定变量类型  

3. **难点3：差值计算逻辑**  
   * **分析**：最高鹅比其他鹅高 = `maxm - a[i]`；最矮鹅比其他鹅矮 = `a[i] - minm`  
   * 💡 **学习笔记**：减法方向反映物理意义（"谁比谁"）

- **类型安全意识**：大整数立即用`long long`

### 4. C++核心代码实现赏析  
**通用核心实现**  
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n; cin >> n;
    vector<long long> a(n);
    // 读入 + 求最值
    long long maxm = 0, minm = 1000000000000000001LL;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        if (a[i] > maxm) maxm = a[i];
        if (a[i] < minm) minm = a[i];
    }
    // 输出差值
    for (int i = 0; i < n; i++) 
        cout << maxm - a[i] << (i < n-1 ? " " : "\n");
    for (int i = 0; i < n; i++)
        cout << a[i] - minm << (i < n-1 ? " " : "\n");
    return 0;
}
```
**代码解读概要**：  
1. 用`vector`存储大整数  
2. 同步读入和更新最值（减少遍历次数）  
3. 条件运算符`?:`优雅处理空格  

**题解片段赏析**  
```cpp
long long maxm = 0, minm = 1000000000000000001LL;
for (int i = 0; i < n; i++) {
    if (a[i] > maxm) maxm = a[i]; // 擂台挑战最大值
    if (a[i] < minm) minm = a[i]; // 擂台挑战最小值
}
```
* **亮点**：初始化值精确覆盖数据边界  
* **学习笔记**：`100...001LL`的`LL`后缀确保字面量为`long long`类型

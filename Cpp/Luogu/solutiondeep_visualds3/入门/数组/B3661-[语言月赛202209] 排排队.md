# 题目信息

# [语言月赛202209] 排排队

## 题目描述

幼儿园里有 $n$ 个小朋友，每个小朋友有一个学号。

老师要求：学号是奇数的小朋友站一排，学号是偶数的小朋友站一排。

**提示**：

奇数：个位为 $1,3,5,7,9$ 的数字，满足除以 $2$ 的余数为 $1$。

偶数：个位为 $0,2,4,6,8$ 的数字，满足除以 $2$ 的余数为 $0$。



## 说明/提示

对于 $20\%$ 的数据，$n=2$；  
对于 $40\%$ 的数据，满足 $n \le 100$；  
对于另外 $20\%$ 的数据，满足 $a_i=i$；  
对于 $100\%$ 的数据，满足 $n\le 10000,0 \le a_i \le 10^9$。

## 样例 #1

### 输入

```
5
1 2 3 4 5```

### 输出

```
1 3 5
2 4```

## 样例 #2

### 输入

```
5 
5 2 3 1 4```

### 输出

```
5 3 1
2 4```

### 3. 核心难点辨析与解题策略
1. **奇偶判断的准确性**  
   * **分析**：必须正确处理边界值（如0是偶数），用`x%2==0`比`x%2==1`更稳妥（避免负数干扰）  
   * 💡 **学习笔记**：判断偶数用`%2==0`，奇数用`%2!=0`更通用  

2. **输出格式控制**  
   * **分析**：需严格按"奇数行+换行+偶数行"输出。最后数字后无空格，可用`bool first`标志或条件判断解决  
   * 💡 **学习笔记**：输出格式错误是常见扣分点！  

3. **数组边界处理**  
   * **分析**：数组大小应设为`10005`（>10000），避免RE。下标从1开始更符合题意  
   * 💡 **学习笔记**：开数组时预留安全空间（+5~10）

### 4. C++核心代码实现赏析
**通用核心实现**  
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, x;
    cin >> n;
    
    // 第一次输出奇数
    bool first = true;
    for (int i=0; i<n; i++) {
        cin >> x;
        if (x % 2 != 0) {
            cout << (first ? "" : " ") << x;
            first = false;
        }
    }
    
    cout << endl;
    first = true;
    
    // 第二次输出偶数
    for (int i=0; i<n; i++) { // 实际需重输数据，此处简化逻辑
        // 重读数据逻辑需根据实现调整
        if (x % 2 == 0) { 
            cout << (first ? "" : " ") << x;
            first = false;
        }
    }
    return 0;
}
```

**题解一核心片段**  
```cpp
for (int i=1; i<=n; i++)
    if (a[i]%2==1) cout<<a[i]<<" ";
cout<<endl;  // 关键换行
for (int i=1; i<=n; i++)
    if (a[i]%2==0) cout<<a[i]<<" ";
```
* **亮点**：简洁明了，直接利用两次遍历  
* **学习笔记**：注意换行位置控制分组分隔  

**题解二核心片段**  
```cpp
int p1=0, p2=0; // 分组指针
for(int i=1, x; i<=n; i++){
    cin >> x;
    if(x%2==1) odd[++p1]=x; // 预存奇数
    else even[++p2]=x;      // 预存偶数
}
```
* **亮点**：数据分离思想，扩展性强  
* **学习笔记**：`++p1`先自增再赋值，避免下标0的陷阱

# 题目信息

# 小玉买文具

## 题目描述

班主任给小玉一个任务，到文具店里买尽量多的签字笔。已知一只签字笔的价格是 $1$ 元 $9$ 角，而班主任给小玉的钱是 $a$ 元 $b$ 角，小玉想知道，她最多能买多少只签字笔呢。


## 说明/提示

#### 数据规模与约定

对于全部的测试点，保证 $0 \leq a \leq 10^4$，$0 \leq b \leq 9$。

## 样例 #1

### 输入

```
10 3
```

### 输出

```
5
```

   - **题解思路共性**：几乎所有优质题解都选择“元转角”的方法（避免浮点数误差），再用`总角数 / 19`得到结果。少数题解用浮点数+`floor`函数，但不如整数方法简洁可靠。
   - **核心算法流程**：输入a（元）、b（角）→计算总角数`a*10 + b`→用总角数除以19（笔的单价，单位角）→输出结果。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
新手做这道题时，最容易卡壳的3个点，Kay帮你拆解清楚：
</difficulty_intro>

1.  **关键点1：如何统一单位？**
    * **分析**：题目里钱是“a元b角”，笔是“1元9角”——单位不统一就没法计算。解决方法是**把所有钱转成最小单位（角）**：1元=10角，所以a元是`a*10`角，加上b角就是总角数`a*10 + b`；笔的价格是`1*10 + 9 = 19`角。这样所有数都是整数，计算不会出错。
    * 💡 **学习笔记**：遇到“不同单位”的问题，先转成最小单位！

2.  **关键点2：如何实现“最多买多少支”？**
    * **分析**：“最多买多少”其实是“向下取整”——比如有37角，能买1支（19角），剩下18角不够买第二支。C++里`int`类型的除法会自动做这件事（`37/19=1`，`103/19=5`），不用额外写`floor`函数！
    * 💡 **学习笔记**：`int`除法=向下取整，适合“求最多能买多少”“最多能分多少组”的问题。

3.  **关键点3：为什么不用浮点数？**
    * **分析**：如果用`double`计算（比如`(a + b/10.0)/1.9`），可能会有**精度误差**——比如10.3/1.9本来是5.421，但浮点数可能存成5.4209999，`floor`后还是5，看似没问题；但如果遇到更极端的数（比如1.9*1e4=19000，用浮点数计算可能会有微小误差），结果就会错。用整数完全不会有这个问题！
    * 💡 **学习笔记**：能不用浮点数就不用，整数计算更可靠。

### ✨ 解题技巧总结
<summary_best_practices>
这道题教会我们3个通用技巧，以后遇到类似问题直接用：
</summary_best_practices>
- **技巧1：单位统一**：把不同单位的量转成最小单位（比如元→角、米→厘米），避免小数。
- **技巧2：整数除法取整**：求“最多数量”时，用`int`除法自动向下取整，比`floor`函数更简单。
- **技巧3：数据范围意识**：如果输入的数很大（比如a≤1e4），用`long long`代替`int`，防止溢出。

<code_intro_overall>
先看一份**通用核心代码**——综合了所有优质题解的优点，简单到“一眼就能懂”：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自CYYC的题解，是“元转角+整数除法”的最简洁实现，适合新手直接套用。
* **完整核心代码**：
    ```cpp
    #include <cstdio>
    int main() {
        int a, b;          // a是元，b是角
        scanf("%d %d", &a, &b);
        int total = a * 10 + b;  // 总钱数转成角
        printf("%d", total / 19); // 19角一支笔，整数除法得数量
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分3步：①用`scanf`读入a和b；②把a元转成`a*10`角，加上b角得到总角数`total`；③用`total/19`计算能买多少支笔，用`printf`输出。没有多余的变量，逻辑直接到“像说话一样”。

**题解二：作者san_ming**
* **亮点**：用`long long`防止数据溢出，适合大输入。
* **核心代码片段**：
    ```cpp
    #define int long long  // 把int替换成long long
    int sum = a * 10 + b;
    cout << sum / 19;
    ```
* **代码解读**：
    > 题目里a≤1e4，`a*10=1e5`，`int`（通常4字节，最大约2e9）能装下，但用`long long`（8字节，最大约9e18）更保险。比如如果题目里a≤1e9，`int`就会溢出，但`long long`不会。
* 💡 **学习笔记**：遇到“大数计算”，先想`long long`。

**题解三：作者Aurora_春雪**
* **亮点**：专门解释“为什么int除法能取整”，帮你理解原理。
* **核心代码片段**：
    ```cpp
    int m = a * 10 + b;
    int ans = m / 19; // int除法自动舍小数
    ```
* **代码解读**：
    > 作者特意说“`int`的除法会直接舍弃小数部分”——比如m=37，37/19=1（而不是1.947），这正好是“最多买1支”的需求。如果用`double`，你还要写`floor`函数，不如`int`方便。

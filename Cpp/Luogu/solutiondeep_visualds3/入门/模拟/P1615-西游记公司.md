# 题目信息

# 西游记公司

## 题目背景

一道极其无厘头的题目。

## 题目描述

事情是这样的：  

西游记中的孙沙猪三徒弟在西天取经之后开始进入厦门大学经贸系学习经济，在 $1$ 个小时的学习后，他们用特殊手段毕业了。

然后，他们创办了三个公司：“花果山生态旅游山庄集团公司” “高老庄猪肉美食城有限公司” 和 “流沙河轮渡有限公司”。虽然这三家公司出自 “经卷出版社” 社长唐三藏的徒弟们，但是每家公司都仍然保持着 $0$ 元以下的财政收入。  

于是，他们想到了一个无聊的方法：向别人偷！其中，孙悟空偷猪八戒的平底锅之前，猪八戒可以一直抢沙和尚的笔记本。

现在，作为猪八戒手下首席智士的你，要帮助既聪明又愚蠢的猪八戒用那种不正当的手段挽救这个公司！  

你可以这样做：你已经知道孙悟空什么时候要来偷猪八戒的东西，而猪八戒又要在什么时间去抢沙和尚的东西，又知道猪八戒每秒钟可以偷沙和尚 $n$ 台笔记本，帮猪八戒算一算，自己在有限的时间内能偷到沙和尚多少台笔记本，使得他有足够的时间来防御大师兄。

## 说明/提示

对于 $100\%$ 的数据，保证两个时间合法，$n<32768$。

## 样例 #1

### 输入

```
00:0:00
0:00:10
10```

### 输出

```
100```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决这道题时，新手常踩的“坑”主要有3个。结合优质题解的经验，我帮你总结了应对方法：
</difficulty_intro>

1. **难点1：如何读取带冒号的时间？**
   * **分析**：时间格式是`hh:mm:ss`，直接用`cin`或`scanf`会被冒号卡住。优质题解的解决方法有3种：① 用`char`变量占位（比如`cin>>a>>m>>b>>m>>c`）；② 用`scanf`的格式化输入（`%lld:%lld:%lld`）；③ 用快读逐字符过滤。
   * 💡 **学习笔记**：处理带分隔符的输入，要么“占位”，要么“过滤”！

2. **难点2：如何正确转换时间单位？**
   * **分析**：1小时=3600秒（不是100秒！），1分钟=60秒。比如`1小时2分3秒`要算成`1*3600 + 2*60 +3 = 3723秒`。优质题解都用了这个公式，从没错漏。
   * 💡 **学习笔记**：时间是60进制，不是10进制！记牢“小时×3600 + 分钟×60 + 秒”。

3. **难点3：如何避免数据溢出？**
   * **分析**：假设时间差是10小时（36000秒），`n=32767`，结果是`36000×32767=1,179,612,000`，超过了`int`的范围（约20亿？不，`int`通常是-2^31到2^31-1，即-2147483648到2147483647，所以11亿没超？但保险起见用`long long`）。优质题解都用了`long long`类型，避免溢出。
   * 💡 **学习笔记**：只要涉及大数字计算，先想“要不要用`long long`？”

### ✨ 解题技巧总结
- **技巧1：用`scanf`格式化输入**：`scanf("%lld:%lld:%lld", &h1, &m1, &s1)`能直接过滤冒号，比`cin`更方便。
- **技巧2：时间转秒公式**：记牢`总秒数 = 小时×3600 + 分钟×60 + 秒`，永远不会错。
- **技巧3：优先用`long long`**：即使题目没说，涉及乘法的结果也要用`long long`，防止溢出。


## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份**综合优质题解的核心代码**，它结合了`scanf`的便捷和`long long`的安全，适合新手直接复用：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了chenyuxuan02和Right的思路，用`scanf`格式化输入，逻辑清晰，不易出错。
* **完整核心代码**：
```cpp
#include <cstdio>
using namespace std;

int main() {
    long long h1, m1, s1, h2, m2, s2, n;
    // 读取两个时间（格式：hh:mm:ss）
    scanf("%lld:%lld:%lld", &h1, &m1, &s1);
    scanf("%lld:%lld:%lld", &h2, &m2, &s2);
    // 读取每秒偷的数量
    scanf("%lld", &n);
    // 转换为总秒数，计算差值，乘速率
    long long time1 = h1 * 3600 + m1 * 60 + s1;
    long long time2 = h2 * 3600 + m2 * 60 + s2;
    printf("%lld\n", (time2 - time1) * n);
    return 0;
}
```
* **代码解读概要**：
  1. 用`scanf`读取带冒号的时间（`%lld:%lld:%lld`会自动忽略冒号）；
  2. 把两个时间都转成“从0点开始的总秒数”；
  3. 计算秒数差，乘以每秒偷的数量，输出结果。

接下来看3份优质题解的核心片段，学习它们的亮点：

**题解一：作者Right**
* **亮点**：用`cin`直接读取冒号，代码极短。
* **核心代码片段**：
```cpp
long long a,b,c,x,y,z,t;
char m;
cin>>a>>m>>b>>m>>c>>x>>m>>y>>m>>z>>t;
cout<<((x-a)*3600+(y-b)*60+(z-c))*t;
```
* **代码解读**：
  - `char m`用来“吃掉”冒号——比如输入`00:0:00`，`a`读`00`，`m`读`:`，`b`读`0`，依此类推。
  - 直接计算`(x-a)*3600`（小时差转秒）、`(y-b)*60`（分钟差转秒）、`(z-c)`（秒差），加起来乘`t`就是结果。
* 💡 **学习笔记**：如果输入格式固定，用`cin`占位符是最简洁的方式！

**题解二：作者chenyuxuan02**
* **亮点**：用`scanf`格式化输入，清晰易懂。
* **核心代码片段**：
```cpp
long long h1, m1, s1, h2, m2, s2, cpt;
scanf("%lld:%lld:%lld", &h1, &m1, &s1);
scanf("%lld:%lld:%lld", &h2, &m2, &s2);
cin>>cpt;
long long s_total1 = h1*3600 + m1*60 + s1;
long long s_total2 = h2*3600 + m2*60 + s2;
cout << (s_total2 - s_total1)*cpt << endl;
```
* **代码解读**：
  - `scanf("%lld:%lld:%lld")`会自动处理冒号，比如输入`0:00:10`，`h2`读`0`，`m2`读`00`，`s2`读`10`。
  - 把时间转成总秒数，再算差值——这种写法最直观，适合新手理解。
* 💡 **学习笔记**：`scanf`的格式化输入是处理固定格式的“神器”！

**题解三：作者NishikinoMaki**
* **亮点**：快读技巧，自动过滤冒号和空格。
* **核心代码片段**：
```cpp
inline long long read() {
    long long n = 0, k = 1;
    char ch = getchar();
    while ((ch > '9' || ch < '0') && ch != '-') ch = getchar();
    if (ch == '-') k = -1, ch = getchar();
    while (ch <= '9' && ch >= '0') {
        n = n * 10 + ch - '0';
        ch = getchar();
    }
    return n * k;
}
// 主函数中调用：
h1 = read(); m1 = read(); s1 = read();
h2 = read(); m2 = read(); s2 = read();
n = read();
```
* **代码解读**：
  - `read()`函数逐字符读取：先跳过非数字（比如冒号），再把数字字符转成数字（`ch - '0'`）。
  - 不管输入中有多少冒号，`read()`都会自动过滤，直接读数字——这种方法适合处理复杂输入。
* 💡 **学习笔记**：快读不仅快，还能自动过滤无关字符，是竞赛中的“秘密武器”！

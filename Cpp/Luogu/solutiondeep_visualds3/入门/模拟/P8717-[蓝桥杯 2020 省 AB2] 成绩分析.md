# 题目信息

# [蓝桥杯 2020 省 AB2] 成绩分析

## 题目描述

小蓝给学生们组织了一场考试，卷面总分为 $100$ 分，每个学生的得分都是一个 $0$ 到 $100$ 的整数。

请计算这次考试的最高分、最低分和平均分。



## 说明/提示

对于 $50 \%$ 的评测用例, $1 \leq n \leq 100$。

对于所有评测用例, $1 \leq n \leq 10000$。

蓝桥杯 2020 第二轮省赛 A 组 F 题（B 组 F 题）。

## 样例 #1

### 输入

```
7
80
92
56
74
88
99
10
```

### 输出

```
99
10
71.29
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`编程技巧应用` (侧重基础数据处理与逻辑实现)

🗣️ **初步分析**：
> 解决「成绩分析」的核心，就像你帮老师**收作业+统计分数**：  
> - 每次收到一本作业（输入成绩），要立刻对比：这是目前最高分吗？还是最低分？（**打擂台法**）  
> - 同时把所有分数加起来，最后除以人数算平均（**累加求和**）。  
> 简单来说，`编程技巧应用`在这里就是「边输入边处理」——用最少的步骤完成多个任务，避免重复遍历数据。  

   - **题解思路对比**：大部分题解用「边输入边更新最值+累加」（时间O(n)，空间O(1)，最高效）；少数用「存数组后排序」（时间O(n log n)，空间O(n)，适合需要重复使用数据的场景）。  
   - **核心算法流程**：读入人数n → 循环n次读成绩 → 每次更新max（初始-1，因为分数≥0）、min（初始101，因为分数≤100）、sum（初始0）→ 最后算平均（sum转浮点÷n）→ 按格式输出。  
   - **可视化设计思路**：我们会做一个「像素教室」动画——学生（像素小人）依次走到讲台交卷（输入成绩），黑板上实时更新红色的「最高分」、蓝色的「最低分」、绿色的「总分」；当最后一个学生交卷，黑板会弹出黄色的「平均分」，伴随“叮”的音效~

## 3. 核心难点辨析与解题策略

<difficulty_intro>
新手做这道题，最容易在三个地方“卡壳”。结合题解的经验，Kay帮你把“坑”变成“知识点”！
</difficulty_intro>

1.  **关键点1：最值变量怎么初始化？**
    * **分析**：求最大值的变量（比如maxn）要初始化成**比所有可能的分数都小**（比如-1，因为分数≥0），这样第一次输入的分数肯定能更新它；求最小值的变量（比如minn）要初始化成**比所有可能的分数都大**（比如101，因为分数≤100），否则如果初始是0，永远找不到比0小的分数！  
    * 💡 **学习笔记**：初始化要“极端”，覆盖所有可能的输入！

2.  **关键点2：平均分为什么会算错？**
    * **分析**：sum是int类型（整数），n也是int类型，如果直接`sum/n`，会做“整数除法”（比如7129/100=71，而不是71.29）。所以必须把sum转成浮点型——比如`(double)sum/n`或者`sum*1.0/n`，这样才能得到正确的小数结果。  
    * 💡 **学习笔记**：整数除法会“砍尾”，一定要转浮点！

3.  **关键点3：怎么输出两位小数？**
    * **分析**：C++里有两种方式：①用`printf("%.2lf", 平均)`（直接指定两位小数）；②用`cout << fixed << setprecision(2) << 平均`（需要加头文件`#include <iomanip>`）。两种都可以，但`printf`更直接，新手优先选它~  
    * 💡 **学习笔记**：输出格式要“精准”，题目要两位就写`.2lf`！

### ✨ 解题技巧总结
<summary_best_practices>
做基础题的“黄金技巧”：
</summary_best_practices>
- **技巧1：边输入边处理**：能一次完成的任务不要分两次（比如不要先存数组再遍历找最值），节省时间和空间。
- **技巧2：用STL函数简化代码**：`max()`/`min()`函数比`if`判断更简洁，减少代码量。
- **技巧3：先想边界情况**：比如n=1的时候（只有一个分数，max和min都是它），或者所有分数都是0（min=0，max=0），提前考虑这些情况，代码更健壮。

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**通用核心实现**——结合了所有优质题解的优点，简洁又高效！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自YuTianQwQ的题解优化，用“单循环”完成所有操作，是竞赛中的“标准解法”。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <climits> // 用于INT_MAX
    #include <cstdio>  // 用于printf
    using namespace std;

    int main() {
        int n, sum = 0, maxn = -1, minn = INT_MAX;
        cin >> n;
        for (int i = 0; i < n; ++i) {
            int a;
            cin >> a;
            sum += a;
            if (a > maxn) maxn = a;
            if (a < minn) minn = a;
        }
        printf("%d\n%d\n%.2lf\n", maxn, minn, (double)sum / n);
        return 0;
    }
    ```
* **代码解读概要**：
    > 1. 初始化：`sum`（总分）=0，`maxn`（最高分）=-1（比所有分数小），`minn`（最低分）=INT_MAX（C++里的“极大值”）。  
    > 2. 循环读n次成绩：每次加进sum，更新maxn和minn。  
    > 3. 输出：用`printf`按格式输出最值和平均分（`(double)sum/n`转浮点）。

<code_intro_selected>
接下来看三个优质题解的“精华片段”，学它们的巧妙之处！
</code_intro_selected>

**题解一：(来源：YuTianQwQ)**
* **亮点**：用`INT_MAX`初始化minn，避免“手动设大数”的误差（比如设101没问题，但如果题目分数范围变大，INT_MAX更通用）。
* **核心代码片段**：
    ```cpp
    int maxn = 0, minn = INT_MAX;
    for (int i = 0; i < n; i++) {
        int a;
        scanf("%d", &a);
        sum += a;
        if (a > maxn) maxn = a;
        if (a < minn) minn = a;
    }
    ```
* **代码解读**：
    > 问：为什么minn用INT_MAX？  
    > 答：INT_MAX是C++里int类型的最大值（约2e9），比题目中100的分数大得多，不管输入什么分数，第一次都会更新minn。比如输入10，minn从INT_MAX变成10；再输入5，又变成5——完美！
* 💡 **学习笔记**：用标准库的“极值常量”，代码更通用！

**题解二：(来源：Yesod)**
* **亮点**：用“桶数组”统计分数出现次数，适合分数范围小的场景。
* **核心代码片段**：
    ```cpp
    long long a[106]; // 分数0-100，所以数组开101个位置
    for (long long i = 1; i <= n; i++) {
        cin >> x;
        ans += x;
        a[x]++; // 分数x出现次数+1
    }
    // 找最高分：从100往下找第一个非零
    for (long long i = 100; i >= 0; i--) if (a[i] != 0) { cout << i << endl; break; }
    // 找最低分：从0往上找第一个非零
    for (long long i = 0; i <= 100; i++) if (a[i] != 0) { cout << i << endl; break; }
    ```
* **代码解读**：
    > 问：桶数组的好处是什么？  
    > 答：比如如果要统计“每个分数有多少人”，桶数组直接就能给出结果。本题用它找最值，虽然步骤多了点，但思路很巧妙——就像“翻记分册”，从最后一页往前翻找最高分，从第一页往后翻找最低分~
* 💡 **学习笔记**：分数范围小时，桶数组是“神器”！

**题解三：(来源：PineappleSummer)**
* **亮点**：用`max()`/`min()`函数简化代码，可读性超高！
* **核心代码片段**：
    ```cpp
    int maxx = -1, minn = 999;
    for (int i = 1; i <= n; i++) {
        int a;
        cin >> a;
        maxx = max(maxx, a); // 等价于if(a>maxx) maxx=a;
        minn = min(minn, a); // 等价于if(a<minn) minn=a;
        sum += a;
    }
    ```
* **代码解读**：
    > 问：`max()`函数怎么用？  
    > 答：`max(a, b)`返回a和b中较大的那个。比如第一次输入80，maxx从-1变成80；第二次输入92，maxx从80变成92——和`if`判断的效果一样，但代码更短！
* 💡 **学习笔记**：STL函数能简化代码，多用多熟悉！

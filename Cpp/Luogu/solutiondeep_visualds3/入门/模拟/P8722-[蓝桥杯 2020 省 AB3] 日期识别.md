# 题目信息

# [蓝桥杯 2020 省 AB3] 日期识别

## 题目描述

小蓝要处理非常多的数据, 其中有一些数据是日期。

在小蓝处理的日期中有两种常用的形式：英文形式和数字形式。

英文形式采用每个月的英文的前三个字母作为月份标识，后面跟两位数字表示日期，月份标识第一个字母大写，后两个字母小写, 日期小于 $10$ 时要补前导 $0$。$1$ 月到 $12$ 月英文的前三个字母分别是 `Jan`、`Feb`、`Mar`、`Apr`、`May`、`Jun`、`Jul`、`Aug`、`Sep`、`Oct`、`Nov`、`Dec`。

数字形式直接用两个整数表达，中间用一个空格分隔，两个整数都不写前 导 `0`。其中月份用 $1$ 至 $12$ 分别表示 $1$ 月到 $12$ 月。

输入一个日期的英文形式, 请输出它的数字形式。

## 说明/提示

蓝桥杯 2020 第三轮省赛 AB 组 F 题。

## 样例 #1

### 输入

```
Feb08```

### 输出

```
2 8```

## 样例 #2

### 输入

```
Oct18```

### 输出

```
10 18```

### 1. 关键点1：怎么准确匹配月份缩写？  
- **分析**：月份缩写有12个，每个都是3位。解决方案：  
  - 用数组/哈希表存“缩写→数字”的对应关系（像题解一、三），直接查；  
  - 用条件判断抓字符特征（像题解二），比如`M`开头看第三个字符（`r`是3月，`y`是5月）。  
- 💡 **学习笔记**：匹配固定内容时，“查字典”比“逐个判断”更高效！


### 2. 关键点2：怎么去掉日期的前导零？  
- **分析**：输入的日期是两位（比如`08`），要转成`8`。解决方案：  
  - 用字符转数字计算：`(s[3]-'0')*10 + (s[4]-'0')`（就算`s[3]`是`'0'`，乘10后还是0，加`s[4]`就去掉前导零）；  
  - 用`stoi`函数：直接把字符串转成整数，自动忽略前导零（像题解三）。  
- 💡 **学习笔记**：`stoi`是字符串转数字的“懒人神器”，但要知道它的原理！


### 3. 关键点3：怎么正确截取字符串？  
- **分析**：输入是5位，前3位是月份，后2位是日期。用`substr`函数：`s.substr(0,3)`取前3位，`s.substr(3,2)`取后2位（比如`Feb08`→`Feb`和`08`）。  
- 💡 **学习笔记**：`substr(pos, len)`的第一个参数是起始位置，第二个是长度，别搞反啦！


### ✨ 解题技巧总结  
- **技巧A**：用映射表简化匹配——遇到“固定字符串→数字”问题，用数组、`map`存对应关系，查起来又快又准！  
- **技巧B**：字符转数字的小技巧——`s[i] - '0'`能把字符`'0'`-`'9'`转成对应的数字！  
- **技巧C**：用STL函数省力气——`stoi`自动处理前导零，`substr`快速截取字符串，不用自己写循环！


## 4. C++核心代码实现赏析

<code_intro_overall>  
先看一个通用的核心实现——用`unordered_map`存月份映射，用`substr`截取，用`stoi`转日期。这个代码综合了多个题解的优点，清晰又高效！  
</code_intro_overall>


### 本题通用核心C++实现参考  
* **说明**：本代码用`unordered_map`存储月份映射，用`substr`截取字符串，用`stoi`自动处理日期前导零，是最简洁的实现方式之一。  
* **完整核心代码**：  
  ```cpp
  #include <iostream>
  #include <string>
  #include <unordered_map>
  using namespace std;
  
  int main() {
      unordered_map<string, int> month_map = {
          {"Jan", 1}, {"Feb", 2}, {"Mar", 3}, {"Apr", 4},
          {"May", 5}, {"Jun", 6}, {"Jul", 7}, {"Aug", 8},
          {"Sep", 9}, {"Oct", 10}, {"Nov", 11}, {"Dec", 12}
      };
      string s;
      cin >> s;
      string month_str = s.substr(0, 3);
      string day_str = s.substr(3, 2);
      int month = month_map[month_str];
      int day = stoi(day_str);
      cout << month << " " << day << endl;
      return 0;
  }
  ```
* **代码解读概要**：  
  > 代码先创建“月份字典”`month_map`，然后读取输入字符串`s`，截取前3位`month_str`（月份）和后2位`day_str`（日期）。接着从字典里查`month_str`对应的数字`month`，用`stoi`把`day_str`转成整数`day`（自动去前导零），最后输出结果。整个流程像“拆快递→查字典→转数字→输出”，一步到位！

### 题解一：来源：LegendaryGrandmaster  
* **亮点**：用数组打表，思路直接，适合刚学字符串的同学！  
* **核心代码片段**：  
  ```cpp
  string a[12]={"Jan","Feb","Mar","Apr","May","Jun","Jul","Aug","Sep","Oct","Nov","Dec"};
  string st;
  cin>>st;
  string month=st.substr(0,3);
  for(int i=0;i<12;i++){
      if(month==a[i]){
          m=i+1;
          break;
      }

  ```
* **代码解读**：  
  > 数组`a`存12个月份缩写，截取前3位`month`后，循环数组找匹配。找到后，`i+1`就是月份数字（数组从0开始）。日期部分用`st[3]-48`（即`st[3]-'0'`）转成数字，乘10加`st[4]-'0'`，算出日期（比如`08`→`0*10+8=8`）。  
* 💡 **学习笔记**：数组打表是最基础的映射方式，适合固定数量的匹配！


### 题解二：来源：lichenzhen  
* **亮点**：用条件判断抓字符特征，不用存整个数组！  
* **核心代码片段**：  
  ```cpp
  if(s[0]=='J') {
      if (s[1]=='a') month=1;
      else if(s[2]=='n') month=6;
      else month=7;
  } else if(s[0]=='F') month=2;
  else if(s[0]=='M'){
      if(s[2]=='r') month=3;
      else month=5;
  }
  day=(s[3]-'0')*10+(s[4]-'0');
  ```
* **代码解读**：  
  > 用`s[0]`（第一个字符）判断大类别：`J`开头的看`s[1]`（`a`是1月）或`s[2]`（`n`是6月，否则7月）；`F`开头直接是2月；`M`开头看`s[2]`（`r`是3月，否则5月）。日期部分和题解一一样，用字符转数字计算。  
* 💡 **学习笔记**：当类别有明显特征时，条件判断比打表更省空间！


### 题解三：来源：very_easy  
* **亮点**：用`unordered_map`和`stoi`，代码最简洁！  
* **核心代码片段**：  
  ```cpp
  unordered_map<string,int> mp{
      {"Jan",1},{"Feb",2},{"Mar",3},{"Apr",4},
      {"May",5},{"Jun",6},{"Jul",7},{"Aug",8},
      {"Sep",9},{"Oct",10},{"Nov",11},{"Dec",12}
  };
  string a;
  cin>>a;
  string s=a.substr(0,3),x=a.substr(3,2);
  cout<<mp[s]<<" "<<stoi(x);
  ```
* **代码解读**：  
  > `unordered_map`是“超级字典”，查月份直接`mp[s]`；`stoi(x)`自动把`08`转成`8`，不用处理前导零。代码只有几行，却完成了所有功能，这就是STL的力量！  
* 💡 **学习笔记**：`unordered_map`的查找速度是O(1)，比数组更快，适合大量数据的情况！

# [蓝桥杯 2021 省 B2] 特殊年份

## 题目描述

今年是 2021 年，2021 这个数字非常特殊, 它的千位和十位相等, 个位比百位大 $1$，我们称满足这样条件的年份为特殊年份。

输入 $5$ 个年份，请计算这里面有多少个特殊年份。


## 说明/提示

**【样例说明】** 

2021 和 9899 是特殊年份，其它不是特殊年份。 

蓝桥杯 2021 第二轮省赛 B 组 F 题。

## 样例 #1

### 输入

```
2019
2021
1920
2120
9899```

### 输出

```
2```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
刚开始做这道题时，小朋友们可能会卡在"怎么拿到每个数位"或者"字符和数字怎么转换"上。Kay帮大家总结了3个核心难点和解决办法，帮你扫清障碍！
</difficulty_intro>

1.  **关键点1：如何拆分四位数的各个数位？**
    * **分析**：四位数`n`的四位拆分就像"剥洋葱"——从外到内一层一层取：
      - 千位：`n / 1000`（比如2021/1000=2）；
      - 百位：`n / 100 % 10`（2021/100=20，20%10=0）；
      - 十位：`n / 10 % 10`（2021/10=202，202%10=2）；
      - 个位：`n % 10`（2021%10=1）。
    * 💡 **学习笔记**：数位拆分的公式是固定的，记下来就能解决所有类似问题！

2.  **关键点2：字符和整数怎么转换？**
    * **分析**：当用字符`char`读入数字时（比如`'2'`），它的本质是ASCII码（`'2'`的ASCII码是50）。要转成整数，只需要减去`'0'`的ASCII码（48）——比如`'2'-'0'=2`。反过来，整数转字符就加`'0'`。
    * 💡 **学习笔记**：字符转数字的小技巧：`数字 = 字符 - '0'`，永远不会错！

3.  **关键点3：如何循环输入5个年份？**
    * **分析**：用`for`循环重复5次就行！不管是每次读一个整数`n`，还是读四个字符，或者读一个字符串，循环5次就能处理所有输入。比如`for(int i=1;i<=5;i++){...}`。
    * 💡 **学习笔记**：循环是处理"重复任务"的神器，遇到"输入N个东西"都可以用它！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个"极简版"的通用实现——它综合了数学拆分的简洁和循环的高效，适合快速理解核心逻辑！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：此代码来自Algo_Sl_al的题解，是"数学拆分法"的典型实现，代码最短、逻辑最直接。
* **完整核心代码**：
    ```cpp
    #include<bits/stdc++.h>
    using namespace std;
    int main(){
        int n,cnt=0;
        for(int i=0;i<5;i++){
            cin>>n;
            if((n/1000==n/10%10)&&(n%10-1==n/100%10))cnt++;
        }
        cout<<cnt;
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分三步：① 定义变量`n`存年份，`cnt`统计特殊年份数量；② 循环5次，每次读入一个年份`n`；③ 判断`n`的千位是否等于十位（`n/1000==n/10%10`），且个位是否等于百位+1（`n%10-1==n/100%10`），符合就`cnt++`；最后输出`cnt`。

接下来看三个优质题解的核心片段，每个都有独特的亮点！
</code_intro_selected>

**题解一：字符直接处理（作者：kiki_123）**
* **亮点**：不用拆分数位，直接读字符，省掉数学计算！
* **核心代码片段**：
    ```cpp
    for(int i=1;i<=5;i++){
        char a,b,c,d;
        cin>>a>>b>>c>>d;
        if(a==c&&d-b==1) ans++;
    }
    ```
* **代码解读**：
    > 这段代码的关键是"直接读四个字符"——`a`是千位，`b`是百位，`c`是十位，`d`是个位。判断条件`a==c`（千位=十位）和`d-b==1`（个位比百位大1）：因为字符的ASCII码是连续的，比如`'1'-'0'=1`，`'2'-'1'=1`，所以`d-b`的结果就是数字差！是不是超聪明？
* 💡 **学习笔记**：字符法适合怕算错数位的情况，直接"所见即所得"！

**题解二：字符串下标法（作者：NightStriker）**
* **亮点**：用字符串的下标直接访问各位，直观到像"指哪打哪"！
* **核心代码片段**：
    ```cpp
    string s;
    for(int i=1;i<=5;i++){
        cin>>s;
        if(s[0]==s[2]&&s[3]==s[1]+1) ans++;
    }
    ```
* **代码解读**：
    > `string`类型的`s`存年份，`s[0]`是第1个字符（千位），`s[1]`是第2个（百位），`s[2]`是第3个（十位），`s[3]`是第4个（个位）。判断条件和字符法一样，但用下标访问更符合"数位顺序"的直觉——比如`s[0]`对应千位，是不是比`n/1000`好记？
* 💡 **学习笔记**：字符串的下标是0开始的，对应数位顺序要注意哦！

**题解三：逐步拆分法（作者：HLJoier）**
* **亮点**：用"逐步除以10"的方法拆分数位，适合理解数位的本质！
* **核心代码片段**：
    ```cpp
    int n;
    cin>>n;
    int q=n%10; n/=10; // 个位
    int w=n%10; n/=10; // 十位
    int e=n%10; n/=10; // 百位
    int r=n%10;        // 千位
    if(r==w&&q==e+1) cnt++;
    ```
* **代码解读**：
    > 这段代码用"取余+除10"的方式，从个位开始逐步拆分：`n%10`取当前最后一位（个位），然后`n/=10`去掉最后一位，接着取十位、百位、千位。最后判断千位`r`等于十位`w`，个位`q`等于百位`e+1`。这种方法能帮你彻底理解"数位是怎么来的"！
* 💡 **学习笔记**：逐步拆分法适合刚开始学数位的小朋友，能加深对数字的理解～

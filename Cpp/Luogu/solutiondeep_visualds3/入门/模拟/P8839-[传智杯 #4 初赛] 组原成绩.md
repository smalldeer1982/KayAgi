# 题目信息

# [传智杯 #4 初赛] 组原成绩

## 题目描述

花栗鼠科技大学（Hualishu University of Science and Technology, HUST）的计算机组成原理快要出分了。你现在需要计算你的组原成绩如何构成。

具体来说，组原成绩分为三部分，分别是平时出勤 $t$ ，作业 $h$ 和考试 $e$ 。总成绩 $w$ 由如下公式计算：

$$w=t \times 20\% +h \times 30\%+e \times 50\%$$ 

其中我们保证 $0 \leq h,e,t \leq 100$

现在你知道了你的组原考试的 $t,h,e$ ，你希望计算你的总成绩。

由于教务系统的特殊性，最终成绩只能是整数，采取 **直接去掉小数部分** 的办法。

## 样例 #1

### 输入

```
50 100 100```

### 输出

```
90```

✨ **本题主要考察**：数学计算 + 模拟应用  

🗣️ **初步分析**：  
解决“组原成绩”这道题，关键在于**准确计算加权平均分并截断小数**。简单来说，就像你帮老师算期末成绩——先按比例算出平时、作业、考试的贡献分，加起来后“砍掉”小数点后的部分（不是四舍五入哦！）。  

- **题解思路**：根据公式 `w = t×20% + h×30% + e×50%` 计算总成绩，再直接去掉小数部分。  
- **核心难点**：如何避免浮点数计算的精度问题？比如用浮点数计算可能会出现`90.999999`却被截断成`90`的情况（其实正确应该是`91`？不，本题是直接截断，所以即使是`90.999`也得算`90`）。但更聪明的做法是**用整数运算代替浮点数**——把公式改成 `(t×20 + h×30 + e×50) / 100`，这样所有计算都是整数，完全避免精度问题！

- **复古游戏元素**：动画采用FC红白机风格（比如像素化的教室背景、卡通化的分数数字），每一步计算伴随“叮”的像素音效，结果出来时播放“胜利”小旋律，让学习更有趣！

## 3. 核心难点辨析与解题策略

在解决这个问题时，我们容易踩三个“小坑”，一起来拆解：

1. **关键点1：如何正确应用加权公式？**  
   * **分析**：公式里的百分比要转换成“放大100倍的权重”（20%→20，30%→30，50%→50），这样总和除以100就等于原来的加权平均分。比如样例中的`50×20 + 100×30 + 100×50 = 1000 + 3000 + 5000 = 9000`，除以100就是90，刚好是样例输出。  
   * 💡 **学习笔记**：把百分比转换成整数权重，是避免浮点数问题的关键！

2. **关键点2：如何处理“直接去掉小数”？**  
   * **分析**：C++中的整数除法会自动“截断”小数部分（比如`90.9`→90，`90.1`→90）。所以只要用整数运算计算总和，再除以100，就能直接得到题目要求的结果。  
   * 💡 **学习笔记**：整数除法的“截断”特性，刚好符合题目的需求！

3. **关键点3：如何避免浮点数精度问题？**  
   * **分析**：比如用`t*0.2 + h*0.3 + e*0.5`计算，可能会出现`90.0`变成`89.999999999`的情况（因为计算机用二进制存储浮点数），导致截断后得到89而不是90。用整数运算就能彻底避免这个问题。  
   * 💡 **学习笔记**：能不用浮点数就不用，整数运算更安全！

### ✨ 解题技巧总结  
- **技巧A：数学公式转换**：把百分比问题转换成整数乘法，避免浮点数。  
- **技巧B：利用整数除法特性**：直接用`/`运算符截断小数，不用额外处理。  
- **技巧C：边界测试**：测试极端值（比如全0、全100），确保代码正确。


## 4. C++核心代码实现赏析

在深入分析之前，先看一个**通用且高效的核心实现**——它用整数运算解决了所有问题：

### 本题通用核心C++实现参考  
* **说明**：本代码是“整数运算+公式转换”的典型实现，逻辑简洁，完全避免浮点数问题。  
* **完整核心代码**：  
```cpp
#include <iostream>
using namespace std;

int main() {
    int t, h, e;
    cin >> t >> h >> e;
    int total = t * 20 + h * 30 + e * 50;
    cout << total / 100 << endl;
    return 0;
}
```  
* **代码解读概要**：  
  1. 用`cin`读取三个整数`t`（出勤）、`h`（作业）、`e`（考试）；  
  2. 计算加权总和：`t×20`（出勤占20%）+ `h×30`（作业占30%）+ `e×50`（考试占50%）；  
  3. 用`total / 100`（整数除法）截断小数，得到最终成绩；  
  4. 用`cout`输出结果。

# 题目信息

# [传智杯 #4 初赛] 报告赋分

## 题目描述

花栗鼠科技大学的计算机组成原理实验最终的结课考核方式是提交一份报告。

然而作为任课老师，萝老师不希望大家过于内卷，所以指定了如下规定：

每份报告有一个卷面基础分 $a$。

在此基础上：

- 若是报告字数低于 $16$ 页，则扣 $10$ 分，如果分数低于 $0$ 分了，则记作 $0$ 分。

- 若是报告字数超过 $20$ 页，每超过 $1$ 页扣 $1$ 分 ，直到分数扣到 $0$ 分为止。

现在你知道了一份报告的卷面基础分 $a$ 和它的页数 $p$ ，请你计算这份报告的最终得分。我们保证 $1 \leq a \leq 100,1 \leq p \leq 50$.

## 样例 #1

### 输入

```
2
70 17
80 10```

### 输出

```
70
70```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`编程技巧应用（条件分支）`

🗣️ **初步分析**：
> 解决这道题的关键，是像「路口看红绿灯」一样——**根据不同的「页数范围」，执行不同的「扣分规则」**。条件分支就像交通规则：遇到红路灯（条件），就走对应的车道（操作）。  
> 在本题中，我们需要判断报告页数`p`的三个区间：  
> - 小于16页：扣10分（最低0分）；  
> - 16~20页：不扣分；  
> - 大于20页：每超1页扣1分（最低0分）。  
> 核心难点是**覆盖所有情况**和**保证分数不低于0**，解决方案是用`if/else if`理清逻辑，用`max(0, 分数)`或特判守住下限。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题看似简单，但新手容易踩「漏条件」「忘下限」的坑。结合优质题解，我帮你提炼了3个核心关键点：
</difficulty_intro>

1.  **关键点1：如何处理「多组输入」？**
    * **分析**：题目有`T`组数据（比如样例输入的`2`），需要循环`T`次读取`a`和`p`。优质题解用了两种方式：`while(T--)`（题解一）或`for(int i=1;i<=T;i++)`（题解二）——本质都是「循环T次」，选你习惯的写法就行。
    * 💡 **学习笔记**：多组输入的核心是「循环读取」，别忘先读`T`！

2.  **关键点2：如何保证「分数不低于0」？**
    * **分析**：扣分后可能出现负数（比如`a=5`，`p=10`），必须把分数「拉回0」。有两种方法：  
      - 用`max(0, 扣分后的分数)`（题解一、三）：直接取「扣分后的值」和「0」的较大者；  
      - 最后特判`if(a<0) a=0`（题解二）：扣分后再检查一次。  
    * 💡 **学习笔记**：`max`函数能简化代码，特判更直观——选哪种都可以，但要记得做！

3.  **关键点3：如何覆盖「所有页数情况」？**
    * **分析**：页数有三个区间：<16、16~20、>20。如果只用两个`if`（比如题解一），中间区间会「自动不处理」；如果用`if-else if-else`（题解三），会明确写出中间区间的处理逻辑——两种方式都对，但`else if`更严谨，不容易漏。
    * 💡 **学习笔记**：写分支前，先列全所有可能的情况，再对应写条件！

### ✨ 解题技巧总结
- **技巧1：先理规则再写代码**：把题目中的「扣分规则」列成表格（比如`<16→扣10`、`>20→扣超页数`），再对应写`if`条件；  
- **技巧2：用`max`简化边界**：`max(0, a-10)`比`a-10; if(a<0) a=0`更简洁；  
- **技巧3：注释帮你理清思路**：像题解二那样，给每个`if`加注释，后期调试不会乱。


## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个**综合所有优质题解的通用实现**——它兼顾了「简洁性」和「严谨性」，适合直接参考～
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码综合了题解一的「`while`循环」、题解三的「`if-else if`结构」，用`max`函数处理分数下限，逻辑清晰。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <algorithm> // 包含max函数
    using namespace std;

    int main() {
        int T;
        cin >> T; // 读入数据组数
        while (T--) { // 循环T次
            int a, p;
            cin >> a >> p; // 读入卷面分a和页数p
            if (p < 16) {
                a = max(0, a - 10); // 低于16页，扣10分，最低0
            } else if (p > 20) {
                a = max(0, a - (p - 20)); // 超20页，扣超页数，最低0
            }
            cout << a << endl; // 输出最终分
        }
        return 0;
    }
    ```
* **代码解读概要**：  
  1. 先读入「数据组数T」；  
  2. 用`while(T--)`循环T次，每次读入`a`（卷面分）和`p`（页数）；  
  3. 根据`p`的范围扣对应分数，用`max`保证分数≥0；  
  4. 输出最终分。

**题解一：(来源：Zvelig1205)**
* **亮点**：用`max`函数一步搞定分数下限，代码超简洁！
* **核心代码片段**：
    ```cpp
    if(p<16)a=max(0,a-10);
    if(p>20)a=max(0,a-p+20);
    ```
* **代码解读**：  
  - 第一行：如果`p<16`，`a`减去10，再和`0`取较大值（比如`a=5`→`5-10=-5`→`max(0,-5)=0`）；  
  - 第二行：如果`p>20`，`a`减去「超的页数」（`p-20`），再取`max`——比如`p=25`→超5页，`a=80`→`80-5=75`→`max(0,75)=75`。
* 💡 **学习笔记**：`max`函数是处理「下限」的神器，能少写一行代码！

**题解二：(来源：xvl_)**
* **亮点**：分步处理，新手友好！
* **核心代码片段**：
    ```cpp
    if (p < 16) a -= 10;
    if (p > 20) a -= p - 20;
    if (a <= 0) cout << 0 << endl;
    else cout << a << endl;
    ```
* **代码解读**：  
  - 先执行两次扣分（不管顺序，因为`p`不可能同时<16和>20）；  
  - 最后检查`a`是否≤0：是就输出0，否则输出`a`——这种「先操作再判断」的方式，适合刚学分支的同学理解。
* 💡 **学习笔记**：分步写代码虽然长，但逻辑更直观，不容易错！

**题解三：(来源：InversionShadow)**
* **亮点**：用`else if`覆盖所有情况，逻辑严谨！
* **核心代码片段**：
    ```cpp
    if (p < 16) {
        cout << max(0, a - 10) << endl;  
    } else if (p > 20) {
        cout << max(0, a - (p - 20)) << endl;    
    } else {  
        cout << a << endl;
    }
    ```
* **代码解读**：  
  - `if(p<16)`：处理低于16页的情况；  
  - `else if(p>20)`：处理超过20页的情况；  
  - `else`：处理中间16~20页的情况（不扣分，直接输出`a`）——这种结构把「所有可能」都覆盖了，不会漏！
* 💡 **学习笔记**：`else if`能帮你「理清逻辑边界」，适合复杂的条件判断！

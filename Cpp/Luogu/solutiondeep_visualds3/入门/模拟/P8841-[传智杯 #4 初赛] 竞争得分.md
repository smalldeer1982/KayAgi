# 题目信息

# [传智杯 #4 初赛] 竞争得分

## 题目描述

为了鼓励大家写出更好的作业，花栗鼠科技大学（Hualishu University of Science and Technology, HUST）的组原实验采用了竞争得分的方式。

具体来说，假设有 $n$ 个人提交了作业，并且其中原始得分最低的人记作 $a_{\min}$ ，原始得分最高的人记作 $a_{\max}$，第 $i$ 个人的原始得分为 $a_i$，那么第 $i$ 个人的得分就是：
$$100 \times \frac{a_i-a_{\min}}{a_{\max}-a_{\min}}$$

由于成绩系统的问题，最终录入的成绩只能是整数，**采用直接去掉小数部分的方法**。

## 样例 #1

### 输入

```
3
1 2 3```

### 输出

```
0 50 100```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`模拟`（按题目规则一步步计算） + `基础最值求解`（找最高分和最低分）

🗣️ **初步分析**：
> 解决这道题的关键，就像**跟着 recipe 做蛋糕**——题目已经把“步骤”（公式）写得明明白白，我们要做的是：  
> 1. 把“食材”（输入的原始分数）收集起来；  
> 2. 找出“最甜的糖”（最高分`a_max`）和“最淡的面粉”（最低分`a_min`）；  
> 3. 按公式把每份食材“烘焙”成最终得分（去掉小数部分）。  

`模拟`的核心就是“照规则办事”，而`最值求解`是这道题的“关键工具”——就像你找蛋糕里最大的草莓，要一个个比过去！  

### 题解思路与核心难点
所有题解的思路高度一致：**边读入边找最值 → 按公式计算 → 输出结果**。  
核心难点有3个：  
1. **怎么正确找最值？** 不能随便把`max`初始化为0（比如输入全是负数就错了！），正确做法是用**第一个数初始化max和min**，再逐个比较；  
2. **怎么去掉小数？** 题目要求“直接砍尾巴”（向下取整），正数情况下用`int`强转或整数除法就能实现；  
3. **分母会不会为0？** 题目保证所有分数不全相同（否则公式无意义），所以不用额外处理~

- 计算得分时，方块会“渐变”成黄色，旁边弹出小气泡显示公式（比如`(3-1)/(3-1)*100=100`）；

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”都藏在**细节**里！结合优质题解，我们逐一解决：
</difficulty_intro>

1.  **关键点1：怎么正确初始化max和min？**
    * **分析**：很多新手会把`max`初始化为0、`min`初始化为10000——但如果输入是负数（比如`-5,-3,-1`），`max`会变成0（错！正确是-1）；如果输入是10001，`min`还是10000（也错！）。优质题解的正确做法是：**用第一个数初始化max和min**，再逐个比较。比如`maxn=minn=a[1]`（Zvelig1205的代码），或者用`double`的极端值（`-1e9`和`1e9`，xvl_的代码）。
    * 💡 **学习笔记**：初始化最值要“贴合数据范围”，不能拍脑袋选数！

2.  **关键点2：怎么去掉小数部分？**
    * **分析**：题目要求“直接砍尾巴”（向下取整）。对于正数来说，有两种方法：  
      - 用`int`强转：比如`(int)(100.0*(a[i]-min)/(max-min))`（xvl_的代码）；  
      - 用整数除法：比如`100*(a[i]-min)/(max-min)`（Zvelig1205的代码）。  
      这两种方法都能“吃掉”小数，因为`int`类型不保存小数部分~
    * 💡 **学习笔记**：正数的整数除法=向下取整，记好这个小技巧！

3.  **关键点3：怎么避免代码冗余？**
    * **分析**：zaochen的代码用`max`和`min`函数代替`if`判断，把两行代码写成一行：`amax=max(amax,a[i]),amin=min(amin,a[i]);`——这不仅让代码更短，还更易读！C++标准库的`max/min`函数是“工具人”，能帮你少写很多重复代码~
    * 💡 **学习笔记**：能用标准库函数就别自己写重复逻辑！

### ✨ 解题技巧总结
- **技巧1：边读边处理**：读入数据时就找最值，避免二次遍历数组（节省时间）；  
- **技巧2：变量名要“自解释”**：用`maxn`代替`m`，`minn`代替`mi`，别人一看就懂；  
- **技巧3：用标准库函数**：`max()`、`min()`、`floor()`这些函数是“现成的轮子”，别自己造！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份**通用核心代码**——它综合了所有优质题解的优点，清晰又正确！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码来自Zvelig1205和zaochen的题解，用“第一个数初始化最值”+“整数除法”，是最适合新手的实现方式。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    const int MAXN = 1005; // 最多1000个人，留5个备用
    int a[MAXN];

    int main() {
        int n;
        cin >> n;
        cin >> a[1]; // 读入第一个数
        int maxn = a[1], minn = a[1]; // 用第一个数初始化最值

        // 读入剩下的n-1个数，同时更新max和min
        for (int i = 2; i <= n; ++i) {
            cin >> a[i];
            maxn = max(maxn, a[i]); // 更新最大值
            minn = min(minn, a[i]); // 更新最小值
        }

        int denominator = maxn - minn; // 分母（定值）
        // 计算并输出每个得分
        for (int i = 1; i <= n; ++i) {
            cout << 100 * (a[i] - minn) / denominator << ' ';
        }

        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分3步：  
    > 1. 读入第一个分数，初始化`maxn`（最高分）和`minn`（最低分）；  
    > 2. 读入剩下的分数，**边读边更新**最值（像“打擂台”一样，比得过就替换）；  
    > 3. 计算每个分数的竞争得分（用整数除法去小数），然后输出。

**题解一：Zvelig1205（来源：洛谷题解）**
* **亮点**：用“第一个数初始化最值”，完全避免初始值错误！
* **核心代码片段**：
    ```cpp
    cin>>n>>a[1];
    maxn=minn=a[1]; // 关键：用第一个数当初始值
    for(int i=2;i<=n;i++){
        cin>>a[i];
        if(a[i]>maxn)maxn=a[i];
        if(a[i]<minn)minn=a[i];
    }
    ```
* **代码解读**：
    > 比如输入是`3 1 2 3`：  
    > - 先读`a[1]=1`，把`maxn`和`minn`都设为1；  
    > - 读`a[2]=2`：比`maxn`大，所以`maxn`变成2；  
    > - 读`a[3]=3`：比`maxn`大，`maxn`变成3。  
    > 这样`maxn=3`，`minn=1`——完美！
* 💡 **学习笔记**：初始化最值的“黄金法则”：用第一个元素当起点！

**题解二：xvl_（来源：洛谷题解）**
* **亮点**：用`double`的极端值覆盖所有可能的输入！
* **核心代码片段**：
    ```cpp
    double a_max = -1e9, a_min = 1e9; // 极端初始值
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        if (a[i] > a_max) a_max = a[i];
        if (a[i] < a_min) a_min = a[i];
    }
    cout << (int)(100 * ((a[i] - a_min) / (a_max - a_min))) << " ";
    ```
* **代码解读**：
    > `1e9`是10的9次方（很大的数），`-1e9`是很小的数。不管输入是`1000000`还是`-999999`，都能被正确比较！最后用`(int)`强转，把小数“砍”掉——比如`50.9`会变成`50`，完全符合题目要求。
* 💡 **学习笔记**：如果不确定输入范围，用极端值初始化最值准没错！

**题解三：zaochen（来源：洛谷题解）**
* **亮点**：用`max()`/`min()`函数简化代码！
* **核心代码片段**：
    ```cpp
    for (int i=1;i<=n;i++){
        cin >> a[i];
        amax=max(amax,a[i]),amin=min(amin,a[i]); // 一行代码更新两个最值
    }
    ```
* **代码解读**：
    > `max(amax,a[i])`会返回`amax`和`a[i]`中较大的那个，直接赋值给`amax`——比写`if`判断简洁多了！`min`同理。这行代码把“比较→更新”的逻辑浓缩成一句话，新手要学会用这种“简洁写法”~
* 💡 **学习笔记**：标准库函数是“懒人神器”，能省时间还不容易错！

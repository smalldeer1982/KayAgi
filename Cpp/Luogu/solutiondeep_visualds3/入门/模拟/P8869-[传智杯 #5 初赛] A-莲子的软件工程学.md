# 题目信息

# [传智杯 #5 初赛] A-莲子的软件工程学

## 题目背景

在宇宙射线的轰击下，莲子电脑里的一些她自己预定义的函数被损坏了。

对于一名理科生来说，各种软件在学习和研究中是非常重要的。为了尽快恢复她电脑上的软件的正常使用，她需要尽快地重新编写这么一些函数。

## 题目描述

具体而言，给定两个整数 $a,b$，保证 $b\neq 0$。莲子要实现这样一个函数 $\operatorname{fun}(a,b)$ 来将 $b$ 的符号转移到 $a$ 上。

具体而言，$\operatorname{fun}(a,b)=\operatorname{sgn}(b)\times |a|$。其中，$\operatorname{sgn}(b)=\begin{cases}1&b>0\\-1&b<0\end{cases}$

换而言之：

- 如果 $b$ 是正数，那么 $\operatorname{fun}(a,b)=+|a|=|a|$；
- 如果 $b$ 是负数，那么 $\operatorname{fun}(a,b)=-|a|$。


## 说明/提示

对于全部数据，保证 $a,b$ 在 $32$ 位有符号整型范围内，并且 $b \neq 0$。

## 样例 #1

### 输入

```
-1 2```

### 输出

```
1```

## 样例 #2

### 输入

```
0 -4```

### 输出

```
0```

## 样例 #3

### 输入

```
-12345 -54321```

### 输出

```
-12345```

🗣️ **初步分析**：
解决这道题的关键，是把数学定义**精准转化为代码**——简单来说，我们要做两件事：① 取出`a`的绝对值（不管`a`原本是正还是负，都变成非负数）；② 用`b`的符号“指挥”这个绝对值的最终符号（`b`正就保持正，`b`负就翻转为负）。  

举个生活中的例子：比如`a`是“10块钱”，但它的“正负标签”掉了；`b`是一个“方向牌”——如果`b`指向右边（正数），就给`a`贴“+”标签；如果`b`指向左边（负数），就贴“-”标签。最后组合起来就是结果啦！  

**核心流程**：  
1. 计算`|a|`（`a`的绝对值）；  
2. 判断`b`的符号：若`b<0`，则结果是`-|a|`；否则是`|a|`。

## 2. 精选优质题解参考

- **用基础语法实现**：不需要复杂算法，用`abs()`函数取绝对值，用`if`判断或三目运算符`? :`处理符号即可。

## 3. 核心难点辨析与解题策略

在解决这个问题时，新手容易踩的“小坑”主要有三个：

1. **关键点1：正确理解`sgn(b)`的定义**  
   - **难点**：`sgn(b)`只看`b`的正负（`b>0`返回1，`b<0`返回-1），但题目已经保证`b≠0`，所以不需要处理`b=0`的情况。  
   - **策略**：用`if (b < 0)`判断即可——如果成立，符号是-1；否则是1。  
   - 💡 **学习笔记**：先明确题目给的“规则边界”，能少写很多无用代码！

2. **关键点2：正确计算`|a|`（`a`的绝对值）**  
   - **难点**：新手可能会自己写`if (a < 0) a = -a`来取绝对值，但其实C++标准库的`abs()`函数（需要`#include <cstdlib>`）已经帮我们做好了，而且更可靠（比如处理`a=-2147483648`这种极限值时不会出错）。  
   - **策略**：直接调用`abs(a)`，省去手动判断的麻烦。  
   - 💡 **学习笔记**：善用标准库函数，既高效又不容易错！

3. **关键点3：处理`a=0`的情况**  
   - **难点**：`0`的绝对值还是`0`，不管乘以1还是-1，结果都是`0`——但新手可能会忘记验证这种情况，比如样例2中`a=0`，`b=-4`，结果确实是0。  
   - **策略**：不需要额外处理！因为`abs(0)=0`，乘以任何符号还是0，代码会自动覆盖这种情况。  
   - 💡 **学习笔记**：有时候“不处理”也是一种正确的选择——因为数学逻辑已经帮我们覆盖了！


### ✨ 解题技巧总结
- **规则转代码**：把数学公式拆成“小步骤”（比如先算绝对值，再处理符号），逐个实现。  
- **标准库复用**：优先用`abs()`这类成熟函数，避免“重复造轮子”。  
- **边界测试**：写完代码后，一定要用样例和极端情况（比如`a=0`、`a`是极限值）测试，确保逻辑正确。

* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <cstdlib> // 包含abs()函数的头文件

    using namespace std;

    // 实现题目要求的fun函数
    int fun(int a, int b) {
        int abs_a = abs(a); // 第一步：取a的绝对值
        if (b < 0) {        // 第二步：根据b的符号调整结果
            return -abs_a;
        } else {
            return abs_a;
        }
    }

    int main() {
        int a, b;
        cin >> a >> b;       // 读取输入的两个整数
        cout << fun(a, b) << endl; // 计算并输出结果
        return 0;
    }
    ```
* **代码解读概要**：  
  1. `fun`函数是核心：先通过`abs(a)`得到`a`的绝对值，再根据`b`的符号返回最终结果；  
  2. `main`函数负责“输入-计算-输出”的流程：读取用户输入的`a`和`b`，调用`fun`函数得到结果，最后打印出来。

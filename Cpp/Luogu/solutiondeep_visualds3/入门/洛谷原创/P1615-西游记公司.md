# 题目信息

# 西游记公司

## 题目背景

一道极其无厘头的题目。

## 题目描述

事情是这样的：  

西游记中的孙沙猪三徒弟在西天取经之后开始进入厦门大学经贸系学习经济，在 $1$ 个小时的学习后，他们用特殊手段毕业了。

然后，他们创办了三个公司：“花果山生态旅游山庄集团公司” “高老庄猪肉美食城有限公司” 和 “流沙河轮渡有限公司”。虽然这三家公司出自 “经卷出版社” 社长唐三藏的徒弟们，但是每家公司都仍然保持着 $0$ 元以下的财政收入。  

于是，他们想到了一个无聊的方法：向别人偷！其中，孙悟空偷猪八戒的平底锅之前，猪八戒可以一直抢沙和尚的笔记本。

现在，作为猪八戒手下首席智士的你，要帮助既聪明又愚蠢的猪八戒用那种不正当的手段挽救这个公司！  

你可以这样做：你已经知道孙悟空什么时候要来偷猪八戒的东西，而猪八戒又要在什么时间去抢沙和尚的东西，又知道猪八戒每秒钟可以偷沙和尚 $n$ 台笔记本，帮猪八戒算一算，自己在有限的时间内能偷到沙和尚多少台笔记本，使得他有足够的时间来防御大师兄。

## 说明/提示

对于 $100\%$ 的数据，保证两个时间合法，$n<32768$。

## 样例 #1

### 输入

```
00:0:00
0:00:10
10```

### 输出

```
100```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：编程技巧应用（侧重输入处理与单位转换）

🗣️ **初步分析**：  
解决这道题的关键，就像把不同面值的硬币换成同一种面值来比较——我们需要把“时、分、秒”这三种“时间硬币”，都转换成最小的“秒硬币”。这样两个时间的差就是猪八戒能偷东西的总秒数，再乘以每秒偷的数量，就是最终答案。  

### 核心思路与难点
- **核心思路**：读取两个带冒号的时间 → 将时间转换为总秒数 → 计算秒数差 → 乘以每秒偷的数量。  
- **核心难点**：① 如何处理带冒号的输入（跳过冒号读数字）；② 如何避免数值溢出（结果可能很大，需用`long long`）。  
- **解决方案**：用`cin`的`char`变量“吃掉”冒号，或用`scanf`的格式化输入直接跳过冒号；所有时间和结果都用`long long`类型存储。

## 3. 核心难点辨析与解题策略

### 关键点1：如何处理带冒号的输入？
- **分析**：输入是`hh:mm:ss`格式，冒号是“干扰项”，需要跳过。  
- **解决方案**：  
  ① 用`cin`：读数字后用`char`变量“吃掉”冒号（如`cin>>h1>>ch>>m1>>ch>>s1`）；  
  ② 用`scanf`：格式化输入直接跳过冒号（如`scanf("%lld:%lld:%lld", &h1, &m1, &s1)`）。  


### 关键点2：如何避免数值溢出？
- **分析**：时间差×每秒数量可能很大（比如1小时×32767=117961200，超过`int`的范围）。  
- **解决方案**：所有存储时间和结果的变量都用`long long`类型（C++中占8字节，能存很大的数）。  


### 关键点3：如何转换时间单位？
- **分析**：1小时=3600秒，1分钟=60秒，总秒数=小时×3600 + 分钟×60 + 秒。  
- **解决方案**：用这个公式把两个时间都转成秒数，再相减就是总时间差。  


### ✨ 解题技巧总结
1. **输入技巧**：优先用`scanf`的格式化输入，直接跳过非数字字符（如冒号）。  
2. **单位转换**：把时间统一转成秒，简化计算。  
3. **溢出处理**：所有涉及时间和结果的变量都用`long long`。  


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考
* **说明**：综合了`scanf`格式化输入和`long long`处理，是最简洁高效的实现。
```cpp
#include <cstdio>
using namespace std;

int main() {
    long long h1, m1, s1, h2, m2, s2, speed;
    // 用scanf直接跳过冒号，读入时分秒
    scanf("%lld:%lld:%lld", &h1, &m1, &s1);
    scanf("%lld:%lld:%lld", &h2, &m2, &s2);
    scanf("%lld", &speed);
    
    // 转换为总秒数
    long long time1 = h1 * 3600 + m1 * 60 + s1;
    long long time2 = h2 * 3600 + m2 * 60 + s2;
    
    // 计算并输出结果
    printf("%lld\n", (time2 - time1) * speed);
    return 0;
}
```
* **代码解读概要**：  
  1. 用`scanf`的`%lld:%lld:%lld`直接读入带冒号的时间；  
  2. 将两个时间转成总秒数（`time1`和`time2`）；  
  3. 计算秒数差×速度，输出结果。  


### 题解一：超短代码片段赏析（来源：Right）
* **亮点**：用`cin`的`char`变量“吃掉”冒号，代码极简。
```cpp
long long a,b,c,x,y,z,t;
char m;
cin>>a>>m>>b>>m>>c>>x>>m>>y>>m>>z>>t;
cout<<((x-a)*3600+(y-b)*60+(z-c))*t;
```
* **代码解读**：  
  - `char m`：专门用来读取冒号，比如输入`00:0:00`时，`m`会依次读入`:`、`:`、`:`；  
  - `(x-a)*3600`：计算小时差转成秒；`(y-b)*60`：分钟差转成秒；`(z-c)`：秒差；  
  - 最后乘以`t`（每秒偷的数量），直接输出。  


### 题解二：scanf输入片段赏析（来源：chenyuxuan02）
* **亮点**：用`scanf`格式化输入，直接跳过冒号，代码规范。
```cpp
long long h1,m1,s1,h2,m2,s2,cpt,cpts;
scanf("%lld:%lld:%lld",&h1,&m1,&s1);
scanf("%lld:%lld:%lld",&h2,&m2,&s2);
s1 = s1 + m1*60 + h1*3600;
s2 = s2 + m2*60 + h2*3600;
cpts = (s2 - s1) * cpt;
```
* **代码解读**：  
  - `%lld:%lld:%lld`：`scanf`会自动跳过`:`，直接读入`h1`（小时）、`m1`（分钟）、`s1`（秒）；  
  - `s1 = ...`：将`h1:m1:s1`转成总秒数（比如`00:00:10`→10秒）；  
  - `cpts = (s2-s1)*cpt`：总秒数×每秒偷的数量，就是答案。  


### 题解三：边界处理片段赏析（来源：Calc_）
* **亮点**：处理时间借位（比如分钟/秒不够减），逻辑严谨。
```cpp
if(t22 - t11 < 0) { // 分钟不够减，从小时借1小时
    time[1] -= 1;
    time[2] = t22 + 60 - t11;
}
if(t222 - t111 < 0) { // 秒不够减，从分钟借1分钟
    if(t22 - 1 >= 0) {
        time[2] -= 1;
        time[3] = t222 + 60 - t111;
    } else { // 分钟也不够，从小时借1小时
        time[1] -= 1;
        time[2] = 59;
        time[3] = t222 + 60 - t111;
    }
}
```
* **代码解读**：  
  - 当`结束分钟 < 开始分钟`时，从小时借1小时（60分钟），分钟差变成`结束分钟+60-开始分钟`；  
  - 当`结束秒 < 开始秒`时，先从分钟借1分钟（60秒），如果分钟也不够，再从小时借1小时。

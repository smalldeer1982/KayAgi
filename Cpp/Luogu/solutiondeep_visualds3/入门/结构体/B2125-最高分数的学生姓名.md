# 题目信息

# 最高分数的学生姓名

## 题目描述

输入学生的人数，然后再输入每位学生的分数和姓名，求获得最高分数的学生的姓名。

## 说明/提示

hanmeimei 的分数最高 $99$ 分。

## 样例 #1

### 输入

```
5
87 lilei
99 hanmeimei
97 lily
96 lucy
77 jim```

### 输出

```
hanmeimei
```

#### **3. 核心难点辨析与解题策略**  
1. **难点1：数据同步更新**  
   * **分析**：在线处理中，分数和姓名需原子更新（`if(x>max) max=x,ans=s`）。若分开写，可能因插入操作导致数据不一致。  
   * 💡 **学习笔记**：**逗号运算符**确保赋值操作的原子性。  

2. **难点2：初始值设定**  
   * **分析**：`max`初始值必须小于所有可能分数（如`-1`）。若设`0`且全负分会错误。题解1的`max=-1`和题解4的`maxn=0`均合理。  
   * 💡 **学习笔记**：**初始值要覆盖边界场景**，尤其关注数据范围。  

3. **难点3：存储结构选择**  
   * **分析**：简单场景用在线处理（`O(1)`空间）；需多次访问数据时用结构体数组。题解5用分离数组`a[]`和`s[]`增加了维护成本。  
   * 💡 **学习笔记**：**优先选择数据耦合**的结构（如结构体或`pair`）。  

### ✨ 解题技巧总结  
- **技巧1：在线处理优先** – 边读边比省空间，适用单次遍历场景  
- **技巧2：结构体封装** – 关联数据统一管理，增强可读性  
- **技巧3：及时终止** – 找到解后立即退出（如`return 0`），避免无效循环

#### **4. C++核心代码实现赏析**  
**通用核心实现（在线处理法）**  
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n, score, max = -1; // 初始化max为-1（确保首个数据更新）
    string name, ans;
    cin >> n;
    for (int i=0; i<n; ++i) {
        cin >> score >> name;
        if (score > max) { 
            max = score;   // 同步更新分数和姓名
            ans = name;    // 使用string避免字符数组越界
        }
    }
    cout << ans;
    return 0;
}
```
**代码解读概要**：  
> 1. 初始化`max=-1`覆盖负分场景  
> 2. 循环中同步读入`score`和`name`  
> 3. 比较后原子更新`max`和`ans`  
> 4. 直接用`string`存储姓名，避免固定数组长度问题

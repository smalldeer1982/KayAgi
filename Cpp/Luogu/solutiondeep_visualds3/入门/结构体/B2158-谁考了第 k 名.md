## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：基于优质题解优化的完整实现，添加STL的swap优化
```cpp
#include <iostream>
#include <algorithm> // 包含swap
using namespace std;

struct Stu {
    int id;
    double sc;
};

int main() {
    int n, k;
    Stu a[105];
    
    cin >> n >> k;
    for(int i=1; i<=n; i++)
        cin >> a[i].id >> a[i].sc;
    
    // 选择排序(降序)
    for(int i=1; i<n; i++) {
        int best = i;
        for(int j=i+1; j<=n; j++) 
            if(a[j].sc > a[best].sc) 
                best = j;
        swap(a[i], a[best]); // STL交换
    }
    
    printf("%d %.1f", a[k].id, a[k].sc);
    return 0;
}
```
* **代码解读概要**：
  > 1. 定义结构体存储学号成绩
  > 2. 读取数据到结构体数组
  > 3. 核心选择排序：每轮找出未排序部分最高分
  > 4. 用swap高效交换元素位置
  > 5. 直接输出第k个元素

---

**题解核心代码赏析**
* **亮点**：完整展示选择排序实现，标注STL优化方案
* **核心代码片段**：
```cpp
for (int i = 1; i <= n - 1; ++i) {
    int best = i; // 当前擂主
    for (int j = i + 1; j <= n; ++j) {
        if (a[j].sc > a[best].sc) // 比较成绩
            best = j; // 更新擂主
    }
    swap(a[i], a[best]); // 交换位置
}
```
* **代码解读**：
  > - 外层循环`i`：已排序区的终点（像已颁奖的领奖台）
  > - `best`变量：记录当前最高分位置（临时擂主）
  > - 内层循环：扫描未排序区（`i+1`到`n`），发现更高分就更新擂主
  > - `swap`交换：将本轮最高分选手移到已排序区末尾
* 💡 **学习笔记**：选择排序像体育选秀——每轮选最强选手入队

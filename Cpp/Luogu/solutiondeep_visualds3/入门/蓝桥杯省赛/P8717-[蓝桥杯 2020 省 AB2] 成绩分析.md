# [蓝桥杯 2020 省 AB2] 成绩分析

## 题目描述

小蓝给学生们组织了一场考试，卷面总分为 $100$ 分，每个学生的得分都是一个 $0$ 到 $100$ 的整数。

请计算这次考试的最高分、最低分和平均分。



## 说明/提示

对于 $50 \%$ 的评测用例, $1 \leq n \leq 100$。

对于所有评测用例, $1 \leq n \leq 10000$。

蓝桥杯 2020 第二轮省赛 A 组 F 题（B 组 F 题）。

## 样例 #1

### 输入

```
7
80
92
56
74
88
99
10
```

### 输出

```
99
10
71.29
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决本题需特别注意以下三个关键点：
</difficulty_intro>

1.  **变量初始化策略**
    * **分析**：最大值初始值必须小于所有可能成绩（推荐-1），最小值初始值必须大于所有可能成绩（推荐101）。这样能确保第一个成绩正确更新极值
    * 💡 **学习笔记**：初始值是模拟算法的"起跑线"，设置不当会导致全程错误

2.  **实时更新技巧**
    * **分析**：在读取每个成绩时同步更新统计值，避免二次遍历。使用`max()`/`min()`函数或直接比较均可
    * 💡 **学习笔记**：边读边处理是O(n)时间复杂度的关键

3.  **精度控制方法**
    * **分析**：整数除法会丢失小数，必须将总和转为浮点型再除。推荐使用`sum*1.0/n`或`(double)sum/n`
    * 💡 **学习笔记**：类型转换是数值计算的"安全阀"

## 4. C++核心代码实现赏析

<code_intro_overall>
首先展示综合优质题解提炼的通用实现：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：综合三大优质题解优化，包含完整输入输出处理
* **完整核心代码**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, score;
    int max_score = -1, min_score = 101;
    double sum = 0;
    
    cin >> n;
    for(int i = 0; i < n; i++) {
        cin >> score;
        if(score > max_score) max_score = score;
        if(score < min_score) min_score = score;
        sum += score;
    }
    
    printf("%d\n%d\n%.2f", max_score, min_score, sum / n);
    return 0;
}
```
* **代码解读概要**：
  > 1. 初始化极值：`max_score=-1`（低于最低分0），`min_score=101`（高于最高分100）
  > 2. 循环读取n个成绩，实时更新极值和总和
  > 3. 输出时用`printf`确保平均值保留两位小数

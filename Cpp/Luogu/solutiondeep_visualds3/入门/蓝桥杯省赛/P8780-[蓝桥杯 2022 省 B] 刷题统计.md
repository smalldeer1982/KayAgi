# 题目信息

# [蓝桥杯 2022 省 B] 刷题统计

## 题目描述

小明决定从下周一开始努力刷题准备蓝桥杯竞赛。他计划周一至周五每天做 $a$ 道题目，周六和周日每天做 $b$ 道题目。请你帮小明计算，按照计划他将在第几天实现做题数大于等于 $n$ 题?

## 说明/提示

对于 $50 \%$ 的评测用例，$1 \leq a, b, n \leq 10^{6}$.

对于 $100 \%$ 的评测用例，$1 \leq a, b, n \leq 10^{18}$. 

蓝桥杯 2022 省赛 B 组 C 题。

## 样例 #1

### 输入

```
10 20 99```

### 输出

```
8```

### 3. 核心难点辨析与解题策略  

1.  **难点1：避免暴力超时**  
    * **分析**：当n≤10¹⁸时，循环n次不可行。优质解都用整周题量5a+2b压缩问题规模，使剩余题目≤7天。  
    * 💡 **学习笔记**：大数问题先寻找数学周期，化指数级为常数级。  

2.  **难点2：剩余题目的精确计算**  
    * **分析**：剩余题量需区分工作日/周末。题解一用数学上取整高效解决；题解二/三用有限循环（≤7步），代码更易写但需注意边界。  
    * 💡 **学习笔记**：除法取整时用`(x+k-1)/k`替代浮点上取整，避免精度问题。  

3.  **难点3：边界条件处理**  
    * **分析**：当n恰为整周倍数时，余数为0需直接输出周数×7。题解三未处理此情况可能导致多算7天。  
    * 💡 **学习笔记**：优先判断余数是否为0，再处理剩余题目。  

#### ✨ 解题技巧总结  
- **周期分解法**：将问题拆解为重复单元（整周）和残余部分  
- **数学优化**：用除法和取模代替循环，大数问题必备  
- **边界测试**：重点验证n=0, n=week, n=week±1等临界值

### 4. C++核心代码实现赏析  

**本题通用核心C++实现参考**  
* **说明**：综合优质解思路，采用周期分解+有限循环确保高效性  
* **完整核心代码**：  
```cpp
#include <iostream>
using namespace std;
int main() {
    long long a, b, n;
    cin >> a >> b >> n;
    long long week = 5*a + 2*b;      // 整周题量
    long long weeks = n / week;      // 整周数
    long long remain = n % week;     // 剩余题量
    long long days = weeks * 7;      // 整周对应天数
    
    if (remain == 0) {
        cout << days;
        return 0;
    }
    // 处理剩余题目（最多7天）
    for (int i = 1; i <= 7; i++) {
        remain -= (i <= 5) ? a : b;  // 工作日刷a题，周末刷b题
        days++;
        if (remain <= 0) break;      // 题目达标时退出
    }
    cout << days;
    return 0;
}
```
* **代码解读概要**：  
  > 1. 计算整周题量`week`和整周数`weeks`  
  > 2. 特判余数`remain=0`时直接输出整周天数  
  > 3. 用三元运算符`?:`动态选择每日刷题量  
  > 4. 剩余题目≤0时立即终止循环  

---

**题解一核心片段赏析**  
```cpp
long long s=5*a+b*2;     // 整周题量
long long s1=n%s;         // 余数
long long s2=n/s;         // 整周数
if (s1<=5*a)              // 工作日可完成
    cout<<s2*7+(s1+a-1)/a; // 上取整技巧
else 
    cout<<s2*7+5+(s1-5*a+b-1)/b; // 先5天工作日
```
* **亮点**：纯数学计算，零循环  
* **代码解读**：  
  > - `(s1+a-1)/a`等价于`ceil(s1/a)`：当s1=0时为0，s1=1~a时为1  
  > - `s1-5*a`计算工作日剩余后，同样用上取整处理周末  
* 💡 **学习笔记**：整数上取整公式是避免浮点误差的利器  

**题解二核心片段赏析**  
```cpp
weeks=n/(a*5+b*2);     // 整周数
n-=weeks*(a*5+b*2);    // 剩余题目
while(n>0){            // 最多循环7次
    if(today<5) n-=a;  // 工作日
    else n-=b;         // 周末
    today=(today+1)%7; // 星期循环
    days++;
}
cout<<weeks*7+days;
```
* **亮点**：直观展示日期轮转逻辑  
* **代码解读**：  
  > - `today%7`实现星期自动重置（6→0）  
  > - 循环条件`n>0`确保题目达标时退出  
* 💡 **学习笔记**：`today=(today+1)%7`是周期计数模板代码

# 题目信息

# [蓝桥杯青少年组省赛 2023] 特殊运算符

## 题目描述

假定有一个运算符 $\texttt{>>>}$，它的功能如下所示：

$$\texttt{>>>}\ 257 = 25$$

$$\texttt{>>>}\ 182 = 18$$

$$\texttt{>>>}\ 933 = 93$$

给定一个正整数 $N(100<N<1000)$，请计算 $N - (\texttt{>>>}\ N)$ 的结果。

例如：$N=257$ 时，$257 - (\texttt{>>>}\ 257) = 257 - 25 = 232$。

## 样例 #1

### 输入

```
257```

### 输出

```
232```

#### 4. C++核心代码实现赏析
**通用核心实现参考**  
* **说明**：综合数学本质的最优解法  
* **完整核心代码**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;          // 输入三位数
    int result = N - (N / 10); // 核心操作：减去百十位数
    cout << result;    // 输出结果
    return 0;
}
```
* **代码解读概要**：

**题解赏析**  
* **亮点**：数学直解，时空复杂度O(1)  
* **核心代码片段**：
```cpp
int result = N - (N / 10);
```
* **代码解读**：  
  `N/10`如同"数字修剪器"，自动移除个位（257→25）。减法运算时，注意整数除法特性保证结果精确

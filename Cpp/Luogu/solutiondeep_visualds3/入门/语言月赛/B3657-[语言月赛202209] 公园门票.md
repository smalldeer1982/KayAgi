# [语言月赛202209] 公园门票

## 题目描述

小 A 一家人一起来逛公园，门票价目表如下：

![](https://cdn.luogu.com.cn/upload/image_hosting/pc7vt43j.png?x-oss-process=image/resize,m_lfit,h_500,w_500)

小 A 家里共有 $x$ 个成人，$y$ 个儿童，请问**至少**需要花费多少钱购买门票。

## 说明/提示

### 输入输出样例 1 解释

1 名成人 + 1 名儿童可以购买一大一小套票，需花费 $90$ 元。

### 输入输出样例 2 解释

2 名成人 + 1 名儿童可以购买一张一大一小套票和一张成人票，需花费 $90 + 60 =150$ 元。

### 数据规模与约定

对于 $20\%$ 的数据，保证 $y=0$；  
对于另外 $20\%$ 的数据，保证 $x=0$；  
对于另外 $20\%$ 的数据，保证 $x=y$；  
对于 $100\%$ 的数据，保证 $0\le x,y \le 100$。

## 样例 #1

### 输入

```
1 1```

### 输出

```
90```

## 样例 #2

### 输入

```
2 1```

### 输出

```
150```

## 3. 核心难点辨析与解题策略

在解决这类「最小花费」问题时，你可能会遇到3个关键难点，我帮你总结了针对性的解决方法：


### 1. 如何识别「最优策略」？  
**分析**：解决问题的第一步，是比较「不同组合的成本」——套票90元 vs 分开买100元，显然套票更划算。因此，**最优策略一定是「尽可能多买套票」**。  
💡 **学习笔记**：遇到「求最小花费」的问题，先算「性价比」！


### 2. 如何处理`x`和`y`的不同大小？  
**分析**：套票需要1个成人+1个儿童，因此套票数量等于`x`和`y`中的**较小值**（比如`x=5`、`y=3`，最多买3套）。剩下的人数用`max(x,y)-min(x,y)`计算（比如`5-3=2`个成人）。  
💡 **学习笔记**：用`min()`和`max()`函数，可以简化80%的计算！


### 3. 如何把思路「直译」成代码？  
**分析**：用`if-else`结构直接对应逻辑——比如`x>y`时，总价是`90*y +60*(x-y)`；`x<y`时是`90*x +40*(y-x)`。不需要多余的变量，「想什么就写什么」。  
💡 **学习笔记**：简洁的代码，往往是「思路清晰」的体现！


### ✨ 解题技巧总结  
- **技巧A**：先算「性价比」——找到最划算的组合（比如套票）；

## 4. C++核心代码实现赏析

在看具体题解之前，先看一个**通用核心实现**——帮你把握整体框架：


### 本题通用核心C++实现参考  
* **说明**：本代码综合了优质题解的思路，用`min()`/`max()`简化了逻辑，是最清晰的实现方式。  
* **完整核心代码**：  
```cpp
#include <iostream>
using namespace std;

int main() {
    int x, y;
    cin >> x >> y;
    int ticket_num = min(x, y);    // 套票数量（取x和y的较小值）
    int rest = max(x, y) - ticket_num;  // 剩下的人数
    int cost = 90 * ticket_num;    // 套票的总费用
    
    if (x > y) {
        cost += 60 * rest;  // 剩下的是成人，买60元单票
    } else {
        cost += 40 * rest;  // 剩下的是儿童，买40元单票
    }
    
    cout << cost << endl;
    return 0;
}
```
* **代码解读概要**：  
  代码逻辑分4步：  
  1. 读取输入的成人数量`x`和儿童数量`y`；  
  2. 用`min(x,y)`算套票数量，`max(x,y)-min(x,y)`算剩下的人数；  
  3. 计算套票费用，再根据`x`和`y`的大小，加剩下的单票费用；  
  4. 输出总费用。  
  关键是用`min()`/`max()`避免了重复判断，代码更简洁！


### 题解一（来源：Maxmilite）  
* **亮点**：用最直接的`if-else`结构，把思路「写死」——没有比这更易懂的代码了！  
* **核心代码片段**：  
```cpp
if (x > y) {
    cout << 90 * y + 60 * (x - y) << endl;
} else if (x < y) {
    cout << 90 * x + 40 * (y - x) << endl;
} else {
    cout << 90 * x << endl;
}
```
* **代码解读**：  
  我们用`x=3`、`y=2`（成人3，儿童2）来模拟：  
  - `x>y`时，套票数量是`y=2`，剩下的`x-y=1`个成人买60元单票，总价是`90*2 +60*1=240`元；  
  - 如果`x=2`、`y=3`（成人2，儿童3），套票数量是`x=2`，剩下的`y-x=1`个儿童买40元单票，总价是`90*2 +40*1=220`元；  
  - 如果`x=2`、`y=2`，全买套票，总价是`90*2=180`元。  
  这段代码的妙处在于「没有多余的变量」——直接把思路「翻译」成代码，非常适合初学者模仿！  
💡 **学习笔记**：把思路「直译」成代码，有时候就是最棒的写法！

# 题目信息

# [语言月赛202209] 课程QQ群

## 题目描述

报名洛谷网校课程的学员可以得到加入课程 QQ 群的验证码。

某期课程设定的验证码为数字 $k$，现在一共有 $n$ 个人申请加入 QQ 群，第 $i$ 个人提供的验证码为 $a_i$。

现在请你担任课程 QQ 群的管理员，请问你一共应该通过多少人的入群申请。

## 说明/提示

对于 $30\%$ 的数据，所有人提供的验证码均正确。  
对于 $100\%$ 的数据，$1 \le n \le 5000, 1 \le a_i,k \le 10^4$。

## 样例 #1

### 输入

```
4 7
7
7
8
7```

### 输出

```
3```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`模拟`

🗣️ **初步分析**：
解决“课程QQ群”的关键是**模拟**——就像你当管理员，逐个检查每个申请人的验证码对不对。模拟的核心思想就是“按题目要求一步步做”：题目让你查n个人的验证码，你就一个个看，对的记下来，最后数有多少个对的。  
在本题中，模拟的具体步骤是：先读入总人数n和正确验证码k，然后循环n次，每次读一个人的验证码，和k比，对的话计数器加1，最后输出计数器。  
核心难点其实是“如何正确循环n次并统计”——比如不要漏读、不要数错。可视化设计上，我们可以用像素块代表每个申请人，正确的用绿色高亮，错误的用灰色，每检查一个就闪烁一下，让你清楚看到“谁对了”“计数怎么变”。  
我们会用8位像素风做动画，比如“像素管理员”逐个检查每个“像素申请人”的验证码，对的话计数器+1时会有“叮”的音效，最后统计完会播放胜利音效，超有代入感！

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题虽然简单，但入门学习者可能会在3个地方“卡壳”，我们一起解决它们！
</difficulty_intro>

1.  **关键点1：如何正确读入n个验证码？**
    * **分析**：题目要求读n个a_i，很多同学会忘“循环n次”或者“循环条件写错”（比如写成i从0开始但循环n次）。题解里用`for (int i = 1; i <= n; ++i)`，从1到n刚好n次，完美对应“n个人”。
    * 💡 **学习笔记**：循环次数要和题目中的“数量”严格对应！

2.  **关键点2：计数器怎么初始化和更新？**
    * **分析**：计数器要从0开始（一开始没人正确），每遇到一个正确的就加1（`++c`）。如果初始化成1，或者漏加，结果就错了。题解里`int c = 0`初始化，每次`x == k`时`++c`，完全正确。
    * 💡 **学习笔记**：统计数量的计数器一定要初始化为0！

3.  **关键点3：如何处理输入顺序？**
    * **分析**：题目中先输入n和k，再输入n个a_i。很多同学会搞反顺序（比如先读a_i再读n），导致输入错误。题解里先`cin >> n >> k`，再循环读n个x，顺序完全正确。
    * 💡 **学习笔记**：输入顺序要严格按照题目描述来！

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一个完整的核心实现——它来自Maxmilite的题解，简洁到“不能再删一行”！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：此代码来自Maxmilite的题解，是“模拟检查验证码”的标准实现，逻辑100%贴合题目要求。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int n, k, c = 0;
        cin >> n >> k;
        for (int i = 1, x; i <= n; ++i) {
            cin >> x;
            if (x == k)
                ++c;
        }
        cout << c << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分3步：1. 定义变量`n`（人数）、`k`（正确验证码）、`c`（计数器，初始0）；2. 读入n和k；3. 循环n次，每次读一个验证码`x`，如果等于k就`c+1`；4. 输出c。整个流程和题目描述一模一样！


<code_intro_selected>
接下来剖析题解的核心片段——循环检查的部分，这是整个代码的“心脏”！
</code_intro_selected>

**题解一：来源：Maxmilite**
* **亮点**：用“循环+条件判断”完美实现“逐个检查”，代码行数最少，逻辑最直接。
* **核心代码片段**：
    ```cpp
    for (int i = 1, x; i <= n; ++i) {
        cin >> x;
        if (x == k)
            ++c;
    }
    ```
* **代码解读**：
    > 这段代码是“模拟”的核心！`for`循环的作用是“重复n次”：`i`从1到n，每次循环代表检查第i个人的验证码。`x`是临时变量，存当前读入的验证码。`if (x == k)`判断“这个验证码对不对”，对的话`++c`（计数器加1）。比如样例输入中，第1、2、4个人的x都是7（等于k=7），所以c会变成3，最后输出3——和样例结果一致！
* 💡 **学习笔记**：循环里可以定义临时变量（比如这里的`x`），用完就扔，不占内存，超方便！

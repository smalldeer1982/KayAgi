# [语言月赛202209] 排排队

## 题目描述

幼儿园里有 $n$ 个小朋友，每个小朋友有一个学号。

老师要求：学号是奇数的小朋友站一排，学号是偶数的小朋友站一排。

**提示**：

奇数：个位为 $1,3,5,7,9$ 的数字，满足除以 $2$ 的余数为 $1$。

偶数：个位为 $0,2,4,6,8$ 的数字，满足除以 $2$ 的余数为 $0$。



## 说明/提示

对于 $20\%$ 的数据，$n=2$；  
对于 $40\%$ 的数据，满足 $n \le 100$；  
对于另外 $20\%$ 的数据，满足 $a_i=i$；  
对于 $100\%$ 的数据，满足 $n\le 10000,0 \le a_i \le 10^9$。

## 样例 #1

### 输入

```
5
1 2 3 4 5```

### 输出

```
1 3 5
2 4```

## 样例 #2

### 输入

```
5 
5 2 3 1 4```

### 输出

```
5 3 1
2 4```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：编程技巧应用（基础数组运用）

🗣️ **初步分析**：  
解决“排排队”的关键，就像整理玩具箱——把彩色积木按红、蓝分开，这里是把小朋友的学号按“奇数”“偶数”分成两堆。**编程技巧应用**的核心是“用基础数据结构（数组/向量）按规则处理数据”，本题中我们需要用数组存储所有学号，再按“奇偶性”分类输出。  

题解有两种思路：  
1. **两次遍历法**：先扫一遍数组，输出所有奇数；再扫一遍，输出所有偶数。  
2. **分数组存储法**：读一个学号，就把它放进“奇数数组”或“偶数数组”，最后分别输出两个数组。  

**核心难点**是：① 数组大小要足够（避免越界）；② 输出格式要准确（奇偶行用换行分隔，元素间用空格）。**解决方案**是：数组大小设为10001以上，用`vector`动态扩容更安全；输出时严格按“奇数行→换行→偶数行”的顺序。  

**可视化设计思路**：我们会用“像素小朋友分队伍”的动画——初始时所有小朋友（像素块）排成一列，点击“开始”后，每个小朋友按奇偶性“走”到左边（奇数队）或右边（偶数队）。**高亮关键步骤**：当前处理的小朋友会闪烁，分到队伍时用颜色标记（奇数红、偶数蓝）；**变量更新**：实时显示奇数队、偶数队的长度。**复古元素**：用8位像素风，分队伍时播放“叮”的音效，完成时播放胜利音乐。


## 2. 精选优质题解参考

为了更好地理解解题过程，我从思路清晰度、代码可读性、实践价值等方面，为大家筛选了以下优质题解：

**题解一：两次遍历法（来源：QQ红包）**  
* **点评**：这份题解的思路像“两次检查玩具箱”——第一次只捡红色积木，第二次只捡蓝色积木。代码逻辑直白到“一看就懂”：两次循环分别筛选奇数、偶数，直接输出。对于刚学数组的同学来说，这种方法最容易上手，唯一要注意的是“第二次循环别漏了”和“输出格式”。

**题解二：分数组存储法（来源：QQ红包）**  
* **点评**：这份题解像“准备两个小箱子”——读一个玩具就放进对应的箱子，最后直接倒出两个箱子的内容。它的优势是“只遍历一次输入”，效率更高；而且用两个数组分开存储，输出时更灵活（比如需要再处理奇数队时，直接用`odd`数组就行）。代码里的`p1`（奇数计数器）、`p2`（偶数计数器）设计得很直观，符合“数数放玩具”的逻辑。


## 3. 核心难点辨析与解题策略

在解决这个问题的过程中，我们通常会遇到以下关键点：

1. **关键点1**：数组大小怎么设才不会越界？  
    * **分析**：题目说`n≤10000`，如果数组大小刚好设为10000，当访问`a[10000]`时会“踩到底线”（C++数组下标从0开始的话，`a[9999]`是第10000个元素，但很多同学习惯从1开始用`a[1]~a[10000]`）。所以**数组大小要设为10001以上**（比如`int a[10001]`），或者用`vector`（动态扩容，永远不会越界）。  
    * 💡 **学习笔记**：数组大小要“留有余地”，避免“刚好等于最大值”的情况。

2. **关键点2**：为什么建议下标从1开始？  
    * **分析**：比如`a[1]`对应第1个小朋友，`a[2]`对应第2个，和“实际顺序”完全一致，不容易搞混“第几个”。如果从0开始，`a[0]`是第1个小朋友，容易记错。  
    * 💡 **学习笔记**：下标从1开始，更符合“人类的计数习惯”。

3. **关键点3**：输出格式怎么才对？  
    * **分析**：题目要求“奇数一排，偶数一排”——奇数元素间用空格，结束后换行；偶数元素间用空格，不需要额外换行。比如样例输入`1 2 3 4 5`，输出是`1 3 5`（换行）`2 4`。如果漏了换行，或者多打了空格，会被判“格式错误（PE）”。  
    * 💡 **学习笔记**：输出前先想“样例长什么样”，用样例验证格式。


### ✨ 解题技巧总结
- **技巧A：按需选择遍历方式**：如果只需要“输出”，两次遍历更简单；如果需要“保存分类结果”，分数组存储更灵活。  
- **技巧B：用`vector`避免越界**：`vector<int> odd`会自动扩容，不用纠结“数组要开多大”。  
- **技巧C：格式检查小技巧**：写完代码后，复制样例输入进去，看输出是不是和样例完全一样（包括空格、换行）。


## 4. C++核心代码实现赏析

在深入分析具体题解的精妙片段之前，让我们先来看一个能够完整解决本题的通用核心C++实现参考。这有助于我们对整体解题框架有一个把握。

### 本题通用核心C++实现参考
* **说明**：本代码用`vector`动态存储奇偶学号，兼顾效率与安全性，是最推荐的新手写法。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <vector>
    using namespace std;

    int main() {
        int n;
        cin >> n;
        vector<int> odd, even; // 奇数数组、偶数数组
        for (int i = 0; i < n; ++i) {
            int x;
            cin >> x;
            if (x % 2 == 1) {
                odd.push_back(x); // 奇数放进odd数组
            } else {
                even.push_back(x); // 偶数放进even数组
            }
        }
        // 输出奇数
        for (int num : odd) {
            cout << num << " ";
        }
        cout << endl; // 换行分隔奇偶
        // 输出偶数
        for (int num : even) {
            cout << num << " ";
        }
        return 0;
    }
    ```
* **代码解读概要**：  
  1. 读入小朋友数量`n`；  
  2. 用`vector`动态存储奇数（`odd`）和偶数（`even`）；  
  3. 循环读每个学号，按奇偶性放进对应的`vector`；  
  4. 先输出`odd`的所有元素（空格分隔），换行；再输出`even`的所有元素（空格分隔）。


### 针对各优质题解的片段赏析

**题解一：两次遍历法（来源：QQ红包）**  
* **亮点**：代码最短，思路最直白，适合刚学循环的同学。
* **核心代码片段**：
    ```cpp
    for (int i = 1; i <= n; i++)
        if (a[i] % 2 == 1)
            cout << a[i] << " ";
    cout << endl;
    for (int i = 1; i <= n; i++)
        if (a[i] % 2 == 0)
            cout << a[i] << " ";
    ```
* **代码解读**：  
  第一遍循环（`i从1到n`）：检查每个`a[i]`是不是奇数（`%2==1`），是就输出；  
  然后换行（`cout<<endl`）；  
  第二遍循环：检查每个`a[i]`是不是偶数（`%2==0`），是就输出。  
  问：为什么要循环两次？因为第一次只处理奇数，第二次只处理偶数，分开输出。  
* 💡 **学习笔记**：两次遍历是“最直接的分类输出方法”，适合不需要保存分类结果的场景。

**题解二：分数组存储法（来源：QQ红包）**  
* **亮点**：只遍历一次输入，效率更高；分类结果存在数组里，方便后续处理。
* **核心代码片段**：
    ```cpp
    int odd[10001], even[10001]; // 奇数数组、偶数数组
    int p1 = 0, p2 = 0; // 奇数计数器、偶数计数器
    for (int i = 1, x; i <= n; i++) {
        cin >> x;
        if (x % 2 == 1) {
            p1++;
            odd[p1] = x; // 奇数存到odd[p1]
        } else {
            p2++;
            even[p2] = x; // 偶数存到even[p2]
        }
    }
    ```
* **代码解读**：  
  1. 定义两个数组`odd`（存奇数）、`even`（存偶数），大小10001；  
  2. `p1`是奇数的个数（初始0），`p2`是偶数的个数（初始0）；  
  3. 循环读每个学号`x`：如果是奇数，`p1`加1，把`x`存到`odd[p1]`；如果是偶数，`p2`加1，存到`even[p2]`。  
  问：为什么用`p1`、`p2`？因为要“数清楚有多少个奇数/偶数”，同时用`p1`当数组下标（比如第1个奇数存在`odd[1]`，第2个在`odd[2]`）。  
* 💡 **学习笔记**：分数组存储是“预处理思维”——提前把数据分类，后面想用的时候直接取。

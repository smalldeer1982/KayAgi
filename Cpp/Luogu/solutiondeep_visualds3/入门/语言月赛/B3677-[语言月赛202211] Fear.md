# 题目信息

# [语言月赛202211] Fear

## 题目描述

在平行宇宙的第五人格，游戏内有一个隐藏的数值，叫做**恐慌值**。

这个数值存在于**求生者**上，会随**监管者**追逐**求生者**的时间增长而变化。

具体的，在求生者未被监管者追逐前，求生者的**初始量**为 $y _ 0$。求生者被监管者追逐了 $t$ 秒。

第 $1$ 秒时，求生者的**恐慌值**为**初始量** $y _ 0$。接下来的每一秒，其**恐慌值**会被乘上它的**初始量**。

**特别的，如果 $t$ 为 $0$，那么求生者的恐慌值为 $1$。**

如果对于恐慌值的计算方式有疑惑，可以查看**样例解释 #1**帮助理解。

你需要计算，在追逐 $t$ 秒后，求生者的**恐慌值**的**奇偶性**以及**正负性**。

## 说明/提示

**【样例 #1 解释】**

求生者的恐慌值和追逐秒数对应如下：

| 秒数 | 恐慌值 |
| :-: | :-: |
| $1$ | $3$ |
| $2$ | $9$ |
| $3$ | $27$ |

在 $3$ 秒追逐后，求生者恐慌值为 $27$，是正数、奇数。

**【样例 #2 解释】**

在 $1$ 秒追逐结束后，求生者恐慌值为 $-2$，是负数、偶数。

**【样例 #3 解释】**

请注意 $0$ 是偶数。

**【数据规模与约定】**

对于前 $10\%$ 的数据，$1 \leq y _ 0 \leq 100$，$t = 1$。  
对于前 $20\%$ 的数据，$1 \leq y _ 0 \leq 100$，$1 \leq t \leq 3$。  
对于前 $50\%$ 的数据，$0 \leq y _ 0 \leq 10 ^ 9$，$0 \leq t \leq 10 ^ 3$。  
对于前 $80\%$ 的数据，$-10 ^ 9 \leq y _ 0 \leq 10 ^ 9$，$0 \leq t \leq 10 ^ 9$。  
对于 $100\%$ 的数据，$-10 ^ {18} \leq y _ 0 \leq 10 ^ {18}$，$ 0 \leq t \leq 10 ^ {18}$。

数据保证 $y _ 0, t$ 不同时为 $0$。

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：数学（幂的符号与奇偶性规律）

🗣️ **初步分析**：  
解决这道题的关键，就像“找规律猜结果”——我们不用真的计算`y₀^t`（毕竟`y₀`和`t`能大到`10¹⁸`，根本算不动），而是用**幂的两个“超好用规律”**：  
1. **符号规律**：一个数的幂的符号，只看底数`y₀`的符号和指数`t`的奇偶性——如果`y₀`是负数且`t`是奇数，结果就是负；否则都是正。  
2. **奇偶规律**：一个数的幂的奇偶性，和底数`y₀`本身的奇偶性一模一样（比如3的任何次幂都是奇，2的任何次幂都是偶）。  

另外还要注意**特判`t=0`**：此时恐慌值是1（正且奇），直接输出对应的结果就行啦！

### 核心算法与可视化设计
- **算法流程**：先特判`t=0`→判断`y₀^t`的符号→判断`y₀^t`的奇偶。

## 3. 核心难点辨析与解题策略

### 关键点1：特判`t=0`
**难点**：`t=0`时恐慌值是1，和`y₀`无关，需要单独处理。  
**解决方案**：先判断`t`是否为0，如果是，直接输出“NO”（1是正）和“YES”（1是奇）。

### 关键点2：处理`y₀`为负时的符号判断
**难点**：`y₀`负且`t`奇→结果负，其他情况都是正。  
**解决方案**：用条件判断`if(y₀ < 0 && t % 2 == 1)`，满足则输出“YES”（负），否则输出“NO”（正）。

### 关键点3：正确判断`y₀`的奇偶性（含负数）
**难点**：C++中负数取模结果为负（比如`-1%2=-1`），不能直接用`y₀%2==1`判断奇。  
**解决方案**：用`abs(y₀%2) == 1`（取模后绝对值为1→奇），或更高效的**位运算**`y₀ & 1`（不管正负，最后一位是1→奇）。

💡 **解题技巧总结**  
1. 先处理特殊情况（`t=0`）；  
2. 用数学规律代替大数计算；  
3. 注意编程语言的细节（比如负数取模）。


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考
* **说明**：综合洛谷网校题解的思路，覆盖所有情况，代码清晰易读。  
* **完整核心代码**：
```cpp
#include <iostream>
#include <cstdlib> // 用于abs函数
using namespace std;

typedef long long ll; // 处理大数，用long long

int main() {
    ll y0, t;
    cin >> y0 >> t;

    // 处理正负性：第一行输出（YES=负，NO=正）
    if (t == 0) {
        cout << "NO" << endl; // 1是正
    } else {
        if (y0 < 0 && t % 2 == 1) {
            cout << "YES" << endl; // 负
        } else {
            cout << "NO" << endl; // 正
        }
    }

    // 处理奇偶性：第二行输出（YES=奇，NO=偶）
    if (t == 0) {
        cout << "YES" << endl; // 1是奇
    } else {
        if (y0 & 1) { // 位运算：最后一位是1→奇
            cout << "YES" << endl;
        } else {
            cout << "NO" << endl;
        }
    }

    return 0;
}
```
* **代码解读概要**：  
  1. 读取`y0`和`t`（用`long long`存大数）；  
  2. 先判`t=0`，直接输出1的属性；  
  3. 否则，用`y0<0 && t%2==1`判负；  
  4. 用`y0&1`判奇（高效处理负数）。


### 题解一：洛谷网校的核心片段赏析
* **亮点**：用`abs(y0%2) == 1`巧妙处理负数的奇偶判断。  
* **核心代码片段**：
```cpp
// 处理奇偶性
if (abs(y0 % 2) == 1) {
    puts("YES"); // 奇
} else {
    puts("NO"); // 偶
}
```
* **代码解读**：  
  不管`y0`是正还是负，取模后的**绝对值**是1就是奇。比如`y0=-1`→`-1%2=-1`→`abs`后是1→正确；`y0=3`→`3%2=1`→正确。  
* 💡 **学习笔记**：判断奇偶性时，`abs(y0%2) ==1`或`y0&1`都能处理负数，后者更高效！

# [语言月赛202212] 计算

## 题目描述

给定一个 $3$ 位整数 $n$，计算：
1. $n$ 的各位数字之和；
2. $n$ 的各位数字之和的平方；
3. $n$ 的各位数字之和的立方；

例如，当这个三位数是 $123$ 时：

1. 各位数字之和为：$1+2+3=6$；
2. 各位数字之和的平方为：$(1+2+3)^2=(1+2+3)\times (1+2+3)=36$；
3. 各位数字之和的立方为：$(1+2+3)^3=(1+2+3)\times (1+2+3)\times (1+2+3)=216$。

## 说明/提示

### 样例 1 解释

各位数字之和为：$8+1+7=16$；  
各位数字之和的平方为：$16^2=16\times 16 = 256$；   
各位数字之和的立方为：$16^3 =16\times 16\times 16 = 4096$。

### 数据规模与约定

对于全部的测试点，保证 $100 \leq n \leq 999$。

## 样例 #1

### 输入

```
817```

### 输出

```
16
256
4096
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
新手解决这道题时，往往会踩几个“小坑”。结合题解的方法，我们一起把这些“坑”填上！
</difficulty_intro>

1. **关键点1：怎么正确拆分三位数的各位数字？**  
    * **分析**：三位数的每一位都有“权值”——百位是100，十位是10，个位是1。拆分百位用`n/100`（比如817/100=8），拆分十位用`n/10%10`（817/10=81，81%10=1），个位用`n%10`（817%10=7）。记住“权值+除法+取模”，就不会错啦！  
    * 💡 **学习笔记**：数字拆分的秘诀是“砍尾巴（除法）+留尾巴（取模）”！

2. **关键点2：为什么要把总和存到变量`s`里？**  
    * **分析**：如果不存`s`，你得重复拆分数字三次（算和、平方、立方）——既麻烦又浪费时间。存到`s`里只需要拆一次，后面直接用`s`计算，多高效！  
    * 💡 **学习笔记**：重复用的结果要“存起来”，这是编程里的“偷懒小技巧”！

3. **关键点3：输出顺序对吗？**  
    * **分析**：题目要求先输出和，再平方，再立方。新手可能会搞反顺序，但只要跟着题目要求的顺序来——先算`s`，再` s*s`，再` s*s*s`——输出顺序就不会错！  
    * 💡 **学习笔记**：按题目要求的顺序来，一步一步走！

<code_intro_overall>

**本题通用核心C++实现参考**  
* **说明**：本代码综合了题解的核心思路，是新手最容易理解的基础实现。  
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int n;
        cin >> n; // 读取输入的三位数
        // 拆分各位数字
        int hundreds = n / 100;      // 百位：n除以100（整数除法）
        int tens = n / 10 % 10;      // 十位：先去掉个位，再取模
        int ones = n % 10;           // 个位：直接取模
        // 计算总和
        int sum = hundreds + tens + ones;
        // 按要求输出
        cout << sum << endl;         // 1. 各位和
        cout << sum * sum << endl;   // 2. 和的平方
        cout << sum * sum * sum << endl; // 3. 和的立方
        return 0;
    }
    ```
* **代码解读概要**：  
    代码分四步：① 读入三位数`n`；② 拆分百位、十位、个位；③ 求和存到`sum`；④ 输出`sum`、`sum`的平方、`sum`的立方。核心逻辑就是“拆分→求和→计算”，超简单！

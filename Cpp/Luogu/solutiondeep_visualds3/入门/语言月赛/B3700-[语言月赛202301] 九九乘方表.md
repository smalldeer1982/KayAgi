# 题目信息

# [语言月赛202301] 九九乘方表

## 题目描述

九九乘法表是数学学习中必须记忆的内容之一。

九九乘法表的前三行为：

`1 * 1 = 1`  
`2 * 1 = 2 2 * 2 = 4`  
`3 * 1 = 3 3 * 2 = 6 3 * 3 = 9`

类似的，我们可以定义出九九乘方表，其前三行为：

`1 ^ 1 = 1`  
`2 ^ 1 = 2 2 ^ 2 = 4`  
`3 ^ 1 = 3 3 ^ 2 = 9 3 ^ 3 = 27`

`a ^ b` 的含义为 $a^b$，也就是 $b$ 个 $a$ 相乘。

例如：$3^4=3\times3\times 3\times 3 = 81$

请你输出九九乘方表的前 $n$ 行，每两项之间由**一个**空格隔开。

## 说明/提示

对于 $100\%$ 的测试数据，$1 \le n \le 9$。

## 样例 #1

### 输入

```
3```

### 输出

```
1 ^ 1 = 1
2 ^ 1 = 2 2 ^ 2 = 4
3 ^ 1 = 3 3 ^ 2 = 9 3 ^ 3 = 27```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
刚开始做这道题时，大家可能会卡在“怎么控制输出格式”“怎么算幂”这些点上。我们逐一解决～
</difficulty_intro>

1.  **关键点1：确定每行要输出多少个式子？**  
    * **分析**：第`i`行要输出`i`个式子（比如第2行2个，第3行3个）。解决方法很简单——用**二重循环**：外层`i`从1到`n`，内层`j`从1到`i`（`j`≤`i`就保证了每行`i`个式子）。  
    * 💡 **学习笔记**：循环的边界条件（比如`j≤i`）是模拟题的“核心开关”，一定要盯紧题目要求！

2.  **关键点2：怎么计算`i`的`j`次方？**  
    * **分析**：`i^j`就是`j`个`i`相乘。比如`3^3=3×3×3`。因为`n≤9`，`j≤i≤9`，所以直接用循环暴力乘就行——写个函数`getPower(x,y)`，初始化`res=1`，循环`y`次乘以`x`，返回`res`。  
    * 💡 **学习笔记**：数据量小时，“暴力法”是最直接的解决方案，不用怕麻烦！

3.  **关键点3：怎么保证输出格式正确？**  
    * **分析**：题目要求每个式子之间用**一个空格**隔开，式子的格式是`x ^ y = z`。解决方法是用`printf`的**格式化输出**——`printf("%d ^ %d = %d ", i, j, res)`，其中`%d`对应整数，空格直接写在字符串里，完美匹配题目要求。  
    * 💡 **学习笔记**：格式化输出是“格式控制大师”，多练几次就能掌握！


### ✨ 解题技巧总结
- **技巧A：循环嵌套控流程**：模拟题常用二重循环，外层管“行”，内层管“每行的内容”。  
- **技巧B：函数封装降复杂度**：把重复的逻辑（比如计算幂）写成函数，代码更清晰。  
- **技巧C：格式化输出保格式**：用`printf`的`%d`和空格直接控制输出样式，比拼接字符串更可靠。


## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：本代码综合了优质题解的思路，是“模拟九九乘方表”的典型实现，逻辑清晰，适合初学者学习。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    // 计算x的y次方（暴力法）
    int getPower(int x, int y) {
        int res = 1;
        for (int k = 1; k <= y; ++k) {
            res *= x;
        }
        return res;
    }

    int main() {
        int n;
        cin >> n;  // 读取输入的n
        for (int i = 1; i <= n; ++i) {  // 外层循环：第i行
            for (int j = 1; j <= i; ++j) {  // 内层循环：第i行的第j个式子
                printf("%d ^ %d = %d ", i, j, getPower(i, j));
            }
            printf("\n");  // 每行结束换行
        }
        return 0;
    }
    ```
* **代码解读概要**：  
    > 代码分三部分：1. 读取输入`n`；2. 用二重循环生成每行的式子（外层`i`管行，内层`j`管每个式子）；3. 用`printf`输出每个式子，每行结束换行。关键是`getPower`函数计算幂，`printf`控制格式。

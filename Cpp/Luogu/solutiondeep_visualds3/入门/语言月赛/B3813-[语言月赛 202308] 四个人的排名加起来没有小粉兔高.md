# 题目信息

# [语言月赛 202308] 四个人的排名加起来没有小粉兔高

## 题目背景

众所周知，洛谷著名管理员品客瑞比特选手在 NOI2020 中以第 $51$ 名的成绩获得了金牌并进入国家集训队。这常带来一个有趣的事实：清华大学一些四人寝室四个人的 NOI 排名加起来还没有小粉兔高。

## 题目描述

又是一年 NOI 结束了。小 A，小 B，小 C 和小 D 作为正式选手，在 NOI3202排名分别是第 $a$ 名，第 $b$ 名，第 $c$ 名和第 $d$ 名。

现在，小 A，小 B，小 C 和小 D 都被保送到清华大学，并住进同一间寝室。

如果四个人的 NOI 排名之和**小于** $51$，则我们称『四个人的排名加起来没有小粉兔高』；反之，如果四个人的 NOI 排名之和**大于或等于** $51$，我们称『四个人的排名加起来不比小粉兔低』。

现在，扶苏知道了四个人的 NOI 排名，她想请你判断是否『四个人的排名加起来没有小粉兔高』。

## 说明/提示

### 数据规模与约定

对全部的测试数据，保证 $1 \leq a, b, c, d \leq 50$。排名有可能相等。

## 样例 #1

### 输入

```
1 2 3 4```

### 输出

```
Rabbit wins```

## 样例 #2

### 输入

```
1 1 5 4```

### 输出

```
Rabbit wins```

## 样例 #3

### 输入

```
50 49 48 47```

### 输出

```
Rabbit lose```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：编程技巧应用（分支语句的基础使用）

🗣️ **初步分析**：  
分支语句就像我们生活中“做选择”的逻辑——比如早上出门看天气：如果下雨就带伞，否则就不带。它的核心是**“条件判断+分支执行”**：满足某个条件时走A路径，不满足时走B路径。  

在本题中，我们需要判断四个排名的总和（`a+b+c+d`）是否**小于51**：  
- 若满足（总和<51），输出“Rabbit wins”（四个人加起来没小粉兔高）；  
- 若不满足（总和≥51），输出“Rabbit lose”（四个人加起来不比小粉兔低）。  

**核心算法流程**：读取四个数→计算总和→条件判断→输出结果。  
**可视化设计思路**：我们可以用像素风格模拟“数值累加→条件判定”的过程——比如用4个像素小人举着数字牌，总和用跳动的像素进度条展示，当进度条没到“51”线时高亮“胜利”结果，反之则高亮“失败”，配合“叮”/“咚”的像素音效强化记忆~

## 3. 核心难点辨析与解题策略

虽然这题看起来简单，但新手容易踩3个小“坑”：

1. **条件符号搞反**：比如把`sum < 51`写成`sum <= 51`（题目要求“小于”，不是“小于等于”）；  
   * **解决**：反复读题目中的“关键条件”——“之和小于51”=“<51”，圈出关键词再写代码。  
2. **输出字符串拼写错误**：比如把“Rabbit wins”写成“Rabbit win”（少了s），或“lose”写成“loose”；  
   * **解决**：直接复制题目中的输出样例到代码里，避免手动拼写错误。  
3. **输入处理不规范**：比如用`scanf`时忘记写格式符（`%d`），或用`cin`时漏看变量类型；  
   * **解决**：输入四个整数时，确保`cin >> a >> b >> c >> d`的顺序和变量名一致。

### ✨ 解题技巧总结
- **抓关键条件**：先从题目中提取“判断依据”（总和<51）和“输出结果”（两个字符串），再写代码；  
- **代码极简主义**：入门题不需要复杂逻辑，能解决问题的代码就是好代码；  
- **验证边界情况**：比如测试`sum=50`（输出wins）、`sum=51`（输出lose），确保条件正确。  


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考
* **说明**：本代码综合了题解的核心思路，是“分支语句解决条件判断问题”的典型实现~
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int a, b, c, d;
      cin >> a >> b >> c >> d;  // 读取四个排名
      int sum = a + b + c + d;  // 计算总和
      if (sum < 51) {           // 条件判断：总和是否小于51？
          cout << "Rabbit wins" << endl;
      } else {
          cout << "Rabbit lose" << endl;
      }
      return 0;
  }
  ```
* **代码解读概要**：  
  代码分3步：①用`cin`读取四个整数；②计算它们的总和`sum`；③用`if-else`判断`sum`与51的关系，输出对应的结果。整个流程和“题目要求”完全对齐，没有多余的步骤~


### 题解一核心代码赏析（来源：一扶苏一）
* **亮点**：用最直白的代码实现最核心的逻辑，没有一句多余的话~
* **核心代码片段**：
  ```cpp
  if (a + b + c + d < 51) {
      printf("Rabbit wins\n");
  } else {
      printf("Rabbit lose\n");
  }
  ```
* **代码解读**：  
  这段代码是题解的“灵魂”！`a+b+c+d < 51`是**判断条件**——如果满足（总和小于51），就用`printf`输出“Rabbit wins”；否则输出“Rabbit lose”。注意`printf`后面要加`\n`（换行符），否则输出的结果会挤在一起~  
* 💡 **学习笔记**：分支语句的关键是“条件要准，分支要明”——先想清楚“什么条件对应什么结果”，再写代码！

# [语言月赛 202309] 等式判断

## 题目描述

给出三个整数 $a,b,c$，保证 $b \neq 0$。

- 如果 $a + b = c$，请你输出 $\texttt{plus}$。
- 如果 $a - b = c$，请你输出 $\texttt{minus}$。
- 如果以上两条均不满足，请输出 $\texttt{illegal}$。

## 说明/提示

### 数据规模与约定

对全部的测试点，保证 $-2^{31} \leq a,b,c < 2^{31}$，$b \neq 0$。

## 样例 #1

### 输入

```
1 2 3```

### 输出

```
plus```

## 样例 #2

### 输入

```
3 2 1```

### 输出

```
minus```

## 样例 #3

### 输入

```
1 1 4```

### 输出

```
illegal```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：编程技巧应用（分支语句的基础运用）

🗣️ **初步分析**：  
解决这道题的核心是**“分支判断”**——就像我们走到十字路口时，要根据交通信号灯（条件）选择走哪条路。在本题中，我们需要依次检查“a+b是否等于c”“a-b是否等于c”这两个条件，根据不同的结果输出对应的答案。  

题解的核心思路很直接：用`if-else if-else`结构顺序判断两个等式，满足则输出对应的关键词，都不满足则输出`illegal`。**核心难点**是处理整数溢出问题——虽然题目说a、b、c都在int范围内，但a和b相加/相减的结果可能超过int的存储上限（比如a=2³¹-1，b=1，相加会溢出）。解决方法是把a、b转成更大的整数类型（如long long）再计算。

## 3. 核心难点辨析与解题策略

在解决这个问题的过程中，我们通常会遇到以下关键点：

1. **难点1：整数溢出问题**  
    * **分析**：a和b都是int范围（-2³¹到2³¹-1），但相加或相减可能超过int的上限（比如a=2³¹-1，b=1，a+b=2³¹，超过int的最大值）。溢出会导致计算结果错误，比如变成负数，从而影响条件判断。  
    * **解决策略**：把a、b转成`long long`类型再计算（`long long`能存储到9e18，足够容纳a+b的结果）。  
    * 💡 **学习笔记**：整数运算前，先想“结果会不会超过当前类型的范围？”，超过就用更大的类型。

2. **难点2：分支条件的顺序**  
    * **分析**：题目说b≠0，所以“a+b=c”和“a-b=c”不可能同时成立（比如如果a+b=c且a-b=c，那么相加得2a=2c→a=c，代入得b=0，矛盾）。所以条件的顺序不影响结果，放心按题目要求的顺序写就行。  
    * **解决策略**：直接按题目描述的顺序判断（先plus再minus）。  
    * 💡 **学习笔记**：写分支前，先想“条件会不会重叠？”，如果不会，顺序随便；如果会，要按优先级排序。

3. **难点3：正确使用分支结构**  
    * **分析**：初学者容易把`else if`写成`if`，导致多条件判断错误（比如同时输出plus和minus）。  
    * **解决策略**：用`if-else if-else`结构，确保“最多只执行一个分支”。  
    * 💡 **学习笔记**：多条件判断时，`if-else if-else`是“选一个”，多个`if`是“选多个”，别用错！


### ✨ 解题技巧总结
- **技巧A：数据类型先检查**：遇到整数运算，先估算结果范围，避免溢出。  
- **技巧B：分支结构要明确**：多条件判断用`if-else if-else`，避免重复执行。  
- **技巧C：题目条件要利用**：题目说b≠0，所以不用考虑两个等式同时成立的情况，减少逻辑复杂度。


## 4. C++核心代码实现赏析

在深入分析具体题解的精妙片段之前，让我们先来看一个能够完整解决本题的通用核心C++实现参考。


### 本题通用核心C++实现参考
* **说明**：本代码综合了题解的思路，重点处理了溢出问题，是一份清晰的基础实现。  
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        long long a, b, c; // 用long long存储，避免溢出
        cin >> a >> b >> c;
        
        if (a + b == c) {
            cout << "plus" << endl;
        } else if (a - b == c) {
            cout << "minus" << endl;
        } else {
            cout << "illegal" << endl;
        }
        
        return 0;
    }
    ```
* **代码解读概要**：  
  代码先读取三个整数（用long long存储），然后用`if-else if-else`顺序判断两个等式：先检查a+b是否等于c，是则输出`plus`；否则检查a-b是否等于c，是则输出`minus`；都不是则输出`illegal`。核心逻辑就是“按条件选答案”，没有多余的复杂度。

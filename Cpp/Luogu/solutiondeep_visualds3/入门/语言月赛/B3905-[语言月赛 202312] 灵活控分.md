# 题目信息

# [语言月赛 202312] 灵活控分

## 题目背景

时间回到 2014 年，T 小学正在举行期末考试。你是一名叫做肖芬途的学生，正在参加某门科目的期末考试。这门科目满分 $100$ 分。

## 题目描述

但是，由于考卷太难，老师宣布这门科目将采取「开根号乘十」的评分策略。具体的，如果试卷的分数是 $x$ 分，那么最终成绩单上的分数则为 $\left \lfloor 10 \sqrt x \right  \rfloor$ 分。

> 提示：对于非负数 $x$，$\sqrt x$ 代表 $x$ 的算术平方根。如果非负数 $y \times y = y ^ 2 = x$，那则称 $y = \sqrt x$。例如，$\sqrt 9 = 3$。   
> 提示：$\left \lfloor x \right \rfloor$ 代表小于等于 $x$ 的最大整数。例如，$\left \lfloor 4.2 \right \rfloor = 4$。

已知这门考试的试卷分数和成绩单分数**一定都是整数**。对于这门考试，肖芬途的班主任要求他的成绩单分数至少要达到 $a$ 分。由于肖芬途是天才，可以随意控制他的试卷分数，因此他想要挑战用最少的卷面分数让成绩单分数达到 $a$ 分。

现在他想要知道这个最少的卷面分数是多少。

形式化地讲，给定一个非负整数 $a$，你需要找到最小的非负整数 $x$，使得 $\left \lfloor 10 \sqrt x \right  \rfloor \geq a$。

## 说明/提示

### 样例 1 解释

不难发现 $10 \sqrt {35} \approx 59.16$，$\left \lfloor  10 \sqrt {35} \right  \rfloor = 59$，而 $10 \sqrt {36} = 60$，因此肖芬途至少需要考到 $36$ 分。

### 数据规模与约定

对于 $100\%$ 的数据，$0 \leq a \leq 100$。

## 样例 #1

### 输入

```
60```

### 输出

```
36```

## 样例 #2

### 输入

```
43```

### 输出

```
19```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决这道题时，大家常遇到3个“卡壳点”，我结合题解帮你拆解清楚～
</difficulty_intro>

1.  **难点1：如何把原条件转换成x的表达式？**  
    * **分析**：原条件是“floor(10√x)≥a”，直接解x看起来麻烦，但通过**公式变形**就能简化：  
      因为floor(10√x)≥a → 10√x≥a（因为floor后的数≥a，原数肯定≥a）→ √x≥a/10 → x≥(a/10)² → x≥a²/100。这一步是解题的关键！  
    * 💡 **学习笔记**：遇到“floor/ceil加公式”的问题，先试试“去掉floor/ceil后找等价条件”。

2.  **难点2：为什么要把a转成double再计算？**  
    * **分析**：C++中如果直接算`a*a/100`（比如a=43，43*43=1849，1849/100=18），会做**整数除法**（直接舍去小数），但我们需要的是“上取整”（18.49要变成19）。所以必须把a转成double（比如`(double)a`），让计算结果保留小数，再用`ceil`函数上取整。  
    * 💡 **学习笔记**：整数除法会“下取整”，涉及小数计算时一定要转成浮点数！

3.  **难点3：如何处理上取整？**  
    * **分析**：C++的`cmath`库中有`ceil`函数，能直接返回一个数的上取整结果（比如ceil(18.49)=19，ceil(36.0)=36）。但要注意，`ceil`的参数必须是浮点数（比如double），否则会出错。  
    * 💡 **学习笔记**：上取整用`ceil`，下取整用`floor`，但要注意参数类型！

    }

* **核心代码片段**：
    ```cpp
    int ans = ceil((double)a * a / 100);
    ```
* **代码解读**：  
  > 这行代码是整个题解的“心脏”！我们拆解来看：  
  1. `(double)a`：把整数a转成double（比如a=43变成43.0），这样后面的乘法和除法会保留小数。  
  2. `a * a`：计算a的平方（比如43.0*43.0=1849.0）。  
  3. `/ 100`：计算a²/100的小数结果（比如1849.0/100=18.49）。  
  4. `ceil(...)`：对结果上取整（18.49变成19，36.0还是36）。  
  为什么要转double？比如a=43，如果不转的话，`43*43/100=1849/100=18`（整数除法），再ceil还是18，但正确结果是19——所以转double是必须的！
* 💡 **学习笔记**：计算“整数的平方除以100的上取整”时，一定要先转浮点数！

# 题目信息

# [语言月赛 202402] 射箭

## 题目描述

bj12z_jiasiyuan 看到了三个人 A、B、C 正在射箭。

他们分别射中了 $x$ 环，$y$ 环，$z$ 环。如果一个人射中的环数**大于**另外两个人射中的环数之和，那么发生一次「神秘事件」。

请你告诉 bj12z_jiasiyuan「神秘事件」发生的次数。

## 说明/提示

### 样例解释

- 对于样例 1，因为 $7 > 4 + 2$，所以发生一次「神秘事件」。
- 对于样例 2，没有「神秘事件」发生。

### 数据规模与约定

对于 $100\%$ 的数据，$1 \leq x, y, z \leq 10$。

## 样例 #1

### 输入

```
7 4 2
```

### 输出

```
1
```

## 样例 #2

### 输入

```
5 3 7
```

### 输出

```
0
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
刚开始学编程时，最容易“卡壳”的往往是「把题目要求变成代码」的过程。针对本题，我总结了3个核心难点和解决办法：
</difficulty_intro>

1.  **难点1：理解「神秘事件次数」的计算逻辑**  
    - 问题：题目说“发生一次神秘事件”的条件是“一个人环数大于另外两人之和”，那次数是满足条件的**人数**。但为什么题解用`||`输出1或0？  
    - 解决：数学推导！假设x>y+z和y>x+z同时成立，会导致y>x+z > (y+z)+z = y+2z（z≥1），即y>y+2——这显然不可能。所以**最多只有1个条件满足**，次数要么0要么1，用`||`判断“是否有至少一个满足”就够了。

2.  **难点2：正确使用逻辑或运算符（||）**  
    - 问题：`||`是什么意思？什么时候用它？  
    - 解决：`||`是“逻辑或”——只要连接的**任意一个条件为真**，整个表达式就为真。比如`a||b||c`，只要a、b、c中有一个是真，结果就是真。本题的三个条件正好符合这个场景！

3.  **难点3：写出正确的条件表达式**  
    - 问题：如何把“x大于y加z”写成代码？  
    - 解决：直接翻译题目描述——`x > y + z`。注意顺序：题目说“大于另外两人之和”，所以是“x”在前，“y+z”在后。

### ✨ 解题技巧总结
- **技巧A：翻译题目规则**：把题目中的“大于”“之和”直接翻译成代码（比如“x大于y加z”→`x > y + z`），不要凭空加逻辑。  
- **技巧B：用逻辑运算符简化代码**：当多个条件“只要有一个满足”时，用`||`合并；当“必须全部满足”时，用`&&`（逻辑与）。  
- **技巧C：先想数学规律**：比如本题的“最多一个条件满足”，能帮你简化代码——不用统计次数，直接判断“有没有”。

### 本题通用核心C++实现参考
* **说明**：本代码用“计数法”展示次数计算过程，适合初学者理解题目要求（虽然结果和“判断有没有”一致，但更直观）。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int x, y, z;
        cin >> x >> y >> z;  // 读取三个环数
        int count = 0;       // 统计神秘事件次数
        
        // 逐一遍历三个条件，满足则次数+1
        if (x > y + z) count++;
        if (y > x + z) count++;
        if (z > x + y) count++;
        
        cout << count << endl;  // 输出次数
        return 0;
    }
    ```
* **代码解读概要**：代码分三步——**读输入**→**判条件**→**输结果**。用`count`变量统计满足条件的次数，每满足一个条件就`count++`，最后输出`count`。这种写法像“掰手指计数”，非常直观。


<code_intro_selected>
接下来看题解中的“精简版”，学习如何用逻辑运算符简化代码！
</code_intro_selected>

**题解一：来源：Maxmilite**
* **亮点**：用`||`合并三个条件，代码更简洁，运行效率更高。
* **核心代码片段**：
    ```cpp
    if (x > y + z || y > x + z || z > x + y) {
        cout << 1 << endl;
    } else {
        cout << 0 << endl;
    }
    ```
* **代码解读**：
    > 大家看这段代码——`x > y + z || y > x + z || z > x + y`是一个**逻辑表达式**。`||`的作用是“只要有一个条件为真，整个表达式就为真”。比如样例1中，`x > y + z`（7>4+2）为真，所以整个表达式为真，输出1；样例2中三个条件都为假，输出0。  
    > 为什么可以这么写？因为我们已经知道**最多一个条件满足**，所以“有一个满足”就等于“次数为1”，“都不满足”等于“次数为0”。这样写比“计数法”少了一个变量（count），代码更精简！

* 💡 **学习笔记**：逻辑或（`||`）是“简化多条件判断”的神器——当多个条件“只要有一个满足”时，用`||`连接，能减少代码行数，让逻辑更清晰。

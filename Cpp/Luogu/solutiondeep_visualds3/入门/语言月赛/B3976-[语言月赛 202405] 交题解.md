# [语言月赛 202405] 交题解

## 题目描述

小 S 想要在 Luogu 上提交一篇非常美妙的题解，但是小 Q 不希望看到小 S 的题解比自己的点赞更多，于是她故意在小 S 的题解中加入了一些奇怪的字符。

具体来说，原本的题解可以看作一个只包含**小写或大写字母**的字符串 $S$，而小 Q 在其中若干位置插入了一些随机字符，这些字符可能是数字或标点符号，但一定**不是字母**，且一定在**键盘**上出现。我们称被添加字符后的字符串为 $S'$。

你的任务是，将 $S'$ 中**非大小写字母**的字符全部去除，然后将得到的字符串输出。

## 说明/提示

### 样例 1 解释

在字符串 `I*Love*Luogu**333#$$` 中，字符 `*`，`#`，`$` 属于标点符号，字符 `3` 属于数字，这些都是后添加的字符。删除后，字符串变为 `ILoveLuogu`。

### 样例 2 解释

在字符串 `L2u4o6gu##$%Lov()es^me` 中，字符 `#`，`$`，`%`，`(`，`)`，`^`，\` 属于标点符号，字符 `2`，`4`，`6` 属于数字，这些都是后添加的字符。删除后，字符串变为 `LuoguLovesme`。

### 数据范围

对于前 $30 \%$ 的数据，保证 $S'$ 中只包含拉丁字母。  
对于另 $30 \%$ 的数据，保证 $S'$ 中只包含拉丁字母和下划线 `_`。  
对于 $100 \%$ 的数据，保证 $S'$ 的长度在 $1$ 到 $10^6$ 之间, $S'$ 中只出现**键盘上主键盘区中的字母、标点符号、数字**，不含空格、换行符等结束符号。

## 样例 #1

### 输入

```
I*Love*Luogu**333#$$
```

### 输出

```
ILoveLuogu
```

## 样例 #2

### 输入

```
L2u4o6gu##$%Lov()es^me
```

### 输出

```
LuoguLovesme
```

## 样例 #3

### 输入

```
[]{}->%_;'pxprpx
```

### 输出

```
pxprpx
```

## 3. 核心难点辨析与解题策略

在解决这个问题时，大家常遇到以下3个关键点：


### 1. **关键点1：如何判断“字母”？**  
* **分析**：字母的ASCII码是连续的——大写字母从`'A'`（65）到`'Z'`（90），小写字母从`'a'`（97）到`'z'`（122）。只要满足`(c >= 'a' && c <= 'z') || (c >= 'A' && c <= 'Z')`，就是字母。  
* 💡 **学习笔记**：ASCII码是字符判断的“万能钥匙”，记住常用字符的范围能解决很多字符串问题！


### 2. **关键点2：处理超长字符串时会不会卡？**  
* **分析**：本题字符串长度最多是1e6，C++的`string`完全能 hold 住——循环遍历是O(n)时间复杂度，对计算机来说就是“眨个眼”的功夫。  
* 💡 **学习笔记**：只要算法复杂度是线性的（O(n)），再长的字符串也不怕！


### 3. **关键点3：输入输出会不会太慢？**  
* **分析**：用`cin`读入`string`没问题，但如果遇到极端长的字符串，可以加`ios::sync_with_stdio(false); cin.tie(0);`关闭同步，加快速度。  
* 💡 **学习笔记**：输入输出的小优化，能让程序“跑更快”！


### ✨ 解题技巧总结  
- **技巧A：直接用ASCII范围判断字符类型**——不需要记复杂的库函数，基础语法就能解决问题。  
- **技巧B：遍历字符串的标准姿势**——用`for(int i=0; i<s.size(); i++)`逐个处理字符，不会漏掉任何一个位置。  
- **技巧C：输出的“按需打印”**——只输出符合条件的字符，避免额外的存储空间（比如不需要新建字符串存结果）。


## 4. C++核心代码实现赏析

在深入分析具体题解的精妙片段之前，让我们先来看一个能够完整解决本题的通用核心C++实现参考。这有助于我们对整体解题框架有一个把握。


### 本题通用核心C++实现参考  
* **说明**：本代码来自题解一，是“模拟处理字符串”的典型实现，逻辑清晰且高效。  
* **完整核心代码**：  
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;
    for (int i = 0; i < s.size(); ++i) {
        if ((s[i] >= 'A' && s[i] <= 'Z') || (s[i] >= 'a' && s[i] <= 'z')) {
            cout << s[i];
        }
    }
    return 0;
}
```  
* **代码解读概要**：  
  1. 用`cin >> s`读入被篡改的字符串；  
  2. 循环遍历字符串的每个字符（`i`从0到`s.size()-1`）；  
  3. 判断当前字符是否是字母（大写或小写）；  
  4. 如果是字母，直接用`cout`输出。  


---

接下来，我们将剖析题解中最能体现核心逻辑的C++实现片段，并点出其亮点和关键思路。


**题解一：(来源：ShiRoZeTsu)**  
* **亮点**：用最基础的ASCII范围判断字母，没有冗余代码，效率拉满。  
* **核心代码片段**：  
```cpp
for (int i = 0; i < s.size(); ++i) {
    if ((s[i] >= 'A' && s[i] <= 'Z') || (s[i] >= 'a' && s[i] <= 'z')) {
        cout << s[i];
    }
}
```  
* **代码解读**：  
  这段代码是整个程序的“心脏”——  
  - `for`循环负责“走一遍”字符串的每个字符，`i`是当前字符的位置；  
  - `if`条件里的两个判断，分别检查是否是大写字母（`'A'<=c<='Z'`）和小写字母（`'a'<=c<='z'`），用`||`连接表示“满足任意一个就行”；  
  - 如果条件成立，就用`cout`输出这个字符——相当于“留下有用的，扔掉没用的”。  
* 💡 **学习笔记**：判断字符类型时，ASCII范围是最直接的“工具”，比调用库函数（比如`isalpha()`）更直观！

# 题目信息

# [语言月赛 202406] 气球

## 题目描述

小 Z 参加了一个比赛。这个比赛一共有 $3$ 个题，在比赛过程中，正确完成第 $1$ 题后，可以获得一个蓝色的气球（用 `B` 表示）；正确完成第 $2$ 题后，可以获得一个黄色的气球（用 `Y` 表示）；正确完成第 $3$ 题后，可以获得一个红色的气球（用 `R` 表示）。正确完成一道后，会马上拿到这个气球。

小 Z 三道题都会做，但并不一定按照第 $1,2,3$ 题的顺序完成。现在给出小 Z 正确完成题目的顺序，请输出小 Z 获得气球的顺序。

## 说明/提示

### 样例 1 解释

小 Z 正确完成第 $1$ 题后，获得了蓝色气球（用 `B` 表示）。\
小 Z 正确完成第 $2$ 题后，获得了黄色气球（用 `Y` 表示）。\
小 Z 正确完成第 $3$ 题后，获得了红色气球（用 `R` 表示）。

### 数据范围

保证输入的数字只有 $1,2,3$ 且不会重复。


## 样例 #1

### 输入

```
1 2 3
```

### 输出

```
BYR
```

## 样例 #2

### 输入

```
3 1 2
```

### 输出

```
RBY
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
新手做这道题时，容易在三个地方“卡壳”。结合题解的思路，我们一起来解决它们：
</difficulty_intro>

1.  **难点1：记混数字和颜色的对应关系**  
    * **分析**：题目里的对应关系是“1→B、2→Y、3→R”，有的同学会把2和3搞反（比如把3当成Y）。  
    * **解决策略**：用“顺口溜”或“表格”记下来——比如“1蓝（B）、2黄（Y）、3红（R）”，或者写在草稿纸上再开始 coding！  
    * 💡 **学习笔记**：编程里的“映射关系”要**先明确、再使用**，别凭感觉猜！

2.  **难点2：不会按顺序处理每个输入**  
    * **分析**：有的同学会把三个数字一起读入后，先处理第三个再处理第一个，导致输出顺序错了。  
    * **解决策略**：“输入顺序=输出顺序”——读第一个数字就处理第一个，读第二个就处理第二个，像题解里那样“读a→判a→输出a的颜色，再读b→判b→输出b的颜色”，顺序绝对不会乱！  
    * 💡 **学习笔记**：顺序问题要“跟着输入走”，一步一步来，别跳步！

3.  **难点3：用错条件判断的结构**  
    * **分析**：有的同学会用`if`但忘了`else`，比如“如果a==1输出B，否则如果a==2输出Y，否则输出R”——少了`else`会导致某些情况没处理（比如a=3时没输出R）。  
    * **解决策略**：用`if-else if-else`覆盖所有可能的情况（因为题目说输入只有1、2、3），确保每个数字都有对应的输出！  
    * 💡 **学习笔记**：条件判断要“覆盖所有可能”，别漏掉任何一种情况！

### ✨ 解题技巧总结
- **技巧A：“一一对应”法**：把题目中的“数字→颜色”写成对照表， coding前先理清关系；  
- **技巧B：“顺序处理”法**：输入一个、处理一个、输出一个，避免顺序错误；  
- **技巧C：“完整判断”法**：用`if-else if-else`覆盖所有可能的输入值，确保没有遗漏。

* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int a, b, c;
        cin >> a >> b >> c;
        
        // 判断第一个数字对应的颜色
        if (a == 1) cout << "B";
        else if (a == 2) cout << "Y";
        else cout << "R";
        
        // 判断第二个数字对应的颜色
        if (b == 1) cout << "B";
        else if (b == 2) cout << "Y";
        else cout << "R";
        
        // 判断第三个数字对应的颜色
        if (c == 1) cout << "B";
        else if (c == 2) cout << "Y";
        else cout << "R";
        
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分三步：① 读入三个表示做题顺序的数字（`a`、`b`、`c`）；② 对每个数字用`if-else if-else`判断对应的颜色（1→B，2→Y，3→R）；③ 按顺序输出每个颜色，连起来就是最终的气球顺序。整个过程就像“按清单核对物品”，每一步都稳扎稳打！

**题解一：来源：Maxmilite**

* **核心代码片段**：
    ```cpp

        cout << "B";
    ```

* **代码解读**：
    > 这段代码是“判断第一个数字`a`对应的颜色”——如果`a`等于1，就输出B（蓝色气球）；如果等于2，输出Y（黄色）；否则（只能是3）输出R（红色）。是不是像“给数字贴标签”？比如输入`a=1`，代码就会“给`a`贴个B的标签”，然后输出这个标签！
* 💡 **学习笔记**：`if-else if-else`是“多分支判断”的基础结构，用来处理“一个变量有多种可能”的情况——比如这里的数字`a`有3种可能，就用3个分支覆盖！

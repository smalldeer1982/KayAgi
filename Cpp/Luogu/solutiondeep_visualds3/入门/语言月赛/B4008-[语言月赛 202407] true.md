# 题目信息

# [语言月赛 202407] true

## 题目背景

从前有个荣光的王国，小 A 是里面的国王，他认为他的王国现在还是不真实的，所以他打算现在赐以其真实。

## 题目描述

小 A 知道他的王国有三个属性，它们的和理应正好为  $10000$。但现在他所能看到的属性却不是这样：小 A 看到的依次是被放大了 $10$ 倍的第一个属性和被缩小了 $10$ 倍的第二个属性，无法看到第三个属性。

现在小 A 想知道这三个属性的真实值是多少，以方便小 A 将他的王国变为真实的。

## 说明/提示

### 样例 1 解释

第一个属性被放大了 $10$ 倍，其真实值应该除以 $10$，为 $4000$，第二个属性被缩小了 $10$ 倍，其真实值为 $3000$，最后一个值应该为 $10000$ 减去 $4000$ 和 $3000$，为 $3000$。

### 样例 2 解释

第一个属性被放大了 $10$ 倍，其真实值应该除以 $10$，为 $5000$，第二个属性被缩小了 $10$ 倍，其真实值为 $1000$，最后一个值应该为 $10000$ 减去 $5000$ 和 $1000$，为 $4000$。  

### 样例 3 解释

第一个属性被放大了 $10$ 倍，其真实值应该除以 $10$，为 $451$，第二个属性被缩小了 $10$ 倍，其真实值为 $2320$，最后一个值应该为 $10000$ 减去 $451$ 和 $2320$，为 $7229$。

### 数据范围

对于 $100\%$ 的数据，$1 \le a \le 100000$，$1 \le b \le 1000$，且 $a$ 为 $10$ 的倍数。保证结果没有负数。

## 样例 #1

### 输入

```
40000 300
```

### 输出

```
4000 3000 3000
```

## 样例 #2

### 输入

```
50000 100
```

### 输出

```
5000 1000 4000
```

## 样例 #3

### 输入

```
4510 232
```

### 输出

```
451 2320 7229
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”不多，但新手容易犯3个小错误。我们来一一解决它们！
</difficulty_intro>

1.  **难点1：搞反“放大/缩小”的操作**  
    * **分析**：题目说“第一个属性被放大10倍”，意思是“看到的a是真实值×10”，所以真实值要**除以10**；“第二个属性被缩小10倍”，即“看到的b是真实值÷10”，所以真实值要**乘以10**。新手容易搞反乘除，比如把“放大10倍”做成乘以10，导致结果错误。  
    * 💡 **学习笔记**：遇到“放大/缩小”问题，先想“看到的值=真实值×k”（放大k倍）或“看到的值=真实值÷k”（缩小k倍），反向操作就是“真实值=看到的值÷k”或“真实值=看到的值×k”。

2.  **难点2：计算第三个属性c时漏用“真实值”**  
    * **分析**：c的计算是“10000 - a - b”，但这里的a和b必须是**还原后的真实值**！如果直接用输入的a和b计算，结果肯定错（比如样例1输入a=40000，直接减的话会得到负数）。  
    * 💡 **学习笔记**：计算前一定要确认变量是“处理后的值”，可以在代码里先处理a和b，再算c。

3.  **难点3：输入输出的格式错误**  
    * **分析**：题目要求输出三个数，用空格分隔（比如样例1输出“4000 3000 3000”）。新手可能会漏空格，或者输出顺序错（比如把b和c写反）。  
    * 💡 **学习笔记**：输出时记得用“cout << a << ' ' << b << ' ' << c;”，空格要写对，顺序要和题目要求一致。

### ✨ 解题技巧总结
- **技巧A：按题目顺序拆解步骤**：把问题拆成“输入→处理a→处理b→算c→输出”，像做数学题一样一步一步来。
- **技巧B：用变量名对应题目含义**：比如直接用a、b、c代表三个属性，不用复杂的名字，降低理解成本。
- **技巧C：检查结果合理性**：算完后可以验证“a+b+c是否等于10000”，比如样例1的4000+3000+3000=10000，确保没错。

* **说明**：本代码综合了两个优质题解的思路，是最简洁的核心实现，适合入门学习。
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int a, b, c;
      cin >> a >> b;  // 读入看到的a和b
      a /= 10;        // 还原第一个属性（放大10倍→除以10）
      b *= 10;        // 还原第二个属性（缩小10倍→乘以10）
      c = 10000 - a - b;  // 计算第三个属性
      cout << a << ' ' << b << ' ' << c << endl;  // 输出真实值
      return 0;
  }
  ```
* **代码解读概要**：
  > 代码分5步：① 定义变量a、b、c；② 读入输入的a和b；③ 还原a（除以10）；④ 还原b（乘以10）；⑤ 计算c并输出。整个逻辑和题目要求完全对应，没有多余的步骤，是顺序结构的典型应用。

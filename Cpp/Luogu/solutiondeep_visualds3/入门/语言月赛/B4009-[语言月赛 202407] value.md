# 题目信息

# [语言月赛 202407] value

## 题目背景

从前有个荣光的王国，小 A 是里面的国王，今天他要赐予他的子民以价值。

## 题目描述

小 A 发现他的国家的价值可以使用 $x, y, z, w$ 四个非负整数衡量。但是为了维持平衡，现在他想要让 $x$ 和 $z$，$y$ 和 $w$ 保持一个相对稳定的比例。

现在他想要知道，是否可以找到一个正整数 $c$，使得 $x = z \times c$ 且 $y = w \times c$。请你帮助他找到这个正整数。

## 说明/提示

### 样例解释

- 可以发现 $9 = 3 \times 3, 6 = 3 \times 2$，$3$ 可以成为样例 1 的 $c$。
- 对样例 2，不能找到任何一个正整数 $c$ 满足题目条件。
- 需要注意 $x, y, z, w$ 有可能是 $0$。

### 数据规模与约定

对 $100\%$ 的数据，保证 $0 \leq x, y, z, w \leq 10^3$，且 $x, y, z, w$ 不同时为 $0$。

## 样例 #1

### 输入

```
9 6 3 2
```

### 输出

```
3
```

## 样例 #2

### 输入

```
7 2 3 5
```

### 输出

```
-1
```

## 样例 #3

### 输入

```
0 6 0 2
```

### 输出

```
3
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：枚举  

🗣️ **初步分析**：  
解决“value”的关键是用**枚举**——像“逐个翻找抽屉找钥匙”一样，把所有可能的正整数c（1到1000）逐个试一遍，看是否满足“x=z×c且y=w×c”。  

题解有两种思路：  
- **解法1（数学分情况）**：通过讨论z、w是否为0，计算c是否一致且为整数（比如z≠0时c=x/z，w≠0时c=y/w，需两者相等）；  
- **解法2（枚举）**：直接试c=1到1000，只要有一个c满足两个等式就输出，否则返回-1。  

**核心难点**：处理0的边界情况（比如z=0时x必须为0，否则c不存在；w=0时y必须为0）。  
**解决方案**：枚举法自动覆盖所有情况——比如z=0时，z×c=0，只有x=0才可能满足条件，否则这个c会被跳过。  

**可视化设计思路**：用8位像素风做一个“计算器找c”动画：c从1到1000滚动，每个c尝试时计算z×c和w×c，与x、y比较（匹配变绿，不匹配变红），找到正确c时闪烁+“叮”音效，胜利时播放8位胜利曲。

## 3. 核心难点辨析与解题策略

在解决问题时，大家常遇到以下3个关键点：

### 关键点1：处理0的乘法特殊性  
**分析**：0乘任何数都是0。比如z=0时，x必须等于0才可能满足x=z×c；w=0时同理y必须为0。  
**解决方案**：枚举法自动处理（z=0时，只有x=0的c才会被保留）；或分情况讨论0的情况。  
💡 **学习笔记**：遇到0时，先想“0×c=0”，再检查对应变量是否为0。

### 关键点2：确保c同时满足两个等式  
**分析**：c必须同时让x=z×c和y=w×c成立，不能只满足一个。比如样例2中，7=3×c的c不是整数，2=5×c也不是，所以无解。  
**解决方案**：对每个c，必须同时检查两个条件。  
💡 **学习笔记**：多个条件要“同时满足”，用`&&`连接。

### 关键点3：确定枚举范围  
**分析**：c是正整数，而x、y最大是1000，所以c最大只能是1000（比如x=1000，z=1时c=1000）。  
**解决方案**：枚举c从1到1000，覆盖所有可能。  
💡 **学习笔记**：枚举范围要根据数据范围定，避免漏解或冗余。

### ✨ 解题技巧总结  
- 枚举法适合“找满足多条件的数”，数据范围小时高效又直观；  
- 处理0时，先想“0的乘法性质”；  
- 多条件用`&&`连接，确保同时满足。


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考  
* **说明**：本代码用枚举法，思路直接，覆盖所有情况，是新手入门的最佳选择。  
* **完整核心代码**：  
```cpp
#include <iostream>
using namespace std;

int main() {
    int x, y, z, w;
    cin >> x >> y >> z >> w;
    for (int c = 1; c <= 1000; ++c) {
        if (x == z * c && y == w * c) {
            cout << c << endl;
            return 0; // 找到就立即退出，节省时间
        }
    }
    cout << -1 << endl;
    return 0;
}
```
* **代码解读概要**：  
代码先读入x、y、z、w，然后从c=1到1000逐个试。每个c都检查“x=z×c且y=w×c”——满足就输出c并结束，否则继续。枚举完都没找到，输出-1。


### 题解一：分情况讨论的核心片段  
* **亮点**：严谨覆盖所有边界条件，适合理解数学逻辑。  
* **核心代码片段**：  
```cpp
int c = -1;
if (z != 0 && w != 0) { // 两者都不为0
    if (x % z == 0 && y % w == 0) {
        int c1 = x / z, c2 = y / w;
        if (c1 == c2 && c1 > 0) c = c1;
    }
} else if (z == 0 && w != 0) { // z=0，w≠0
    if (x == 0 && y % w == 0) c = y / w;
} else if (z != 0 && w == 0) { // z≠0，w=0
    if (y == 0 && x % z == 0) c = x / z;
}
cout << (c > 0 ? c : -1) << endl;
```
* **代码解读**：  
- 当z、w都不为0时，检查x能被z整除、y能被w整除，且两者的商（c1、c2）相等且为正整数；  
- 当z=0时，x必须为0（因为z×c=0），且y能被w整除，才能得到c；  
- 当w=0时同理，y必须为0，x能被z整除。  
* **学习笔记**：分情况讨论要“不重不漏”，每个情况都要考虑到。

# [语言月赛 202410] 顺序结构

## 题目描述

顺序结构中，为什么一定要讲究几句话的顺序呢？在一次经历中，小 Y 意识到了顺序的重要性。

---

小 Y 写了一个程序，按顺序执行下面四步：

1. 输入变量 $a$。
2. 将 $a$ 增加 $ 5$。
3. 将 $a$ 变成原来的 $3$ 倍。
4. 输出现在的 $a$。

小 L 写了一个程序，按顺序执行下面四步：

1. 输入变量 $a$。
2. 将 $a$ 变成原来的 $3$ 倍。
3. 将 $a$ 增加 $5$。
4. 输出现在的 $a$。

给出输入的 $a$，请判断两人的程序分别会输出什么。

## 说明/提示

【样例 1 解释】

在小 Y 的程序输入 $a$ 之后：

- 先把 $a$ 加上 $5$，变成 $4+5=9$。
- 然后把 $a$ 乘上 $3$，得到 $9\times 3 = 27$。

在小 L 的程序输入 $a$ 之后：

- 先把 $a$ 乘上 $3$，得到 $4\times 3 = 12$。
- 然后把 $a$ 加上 $5$，得到 $12+5=17$。

【数据范围】

$1\le a \le 28$，并且 $a$ 是正整数。

## 样例 #1

### 输入

```
4
```

### 输出

```
27 17
```

## 样例 #2

### 输入

```
10
```

### 输出

```
45 35
```

## 样例 #3

### 输入

```
27
```

### 输出

```
96 86
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：模拟（顺序结构应用）

🗣️ **初步分析**：  
解决这道题的关键是**模拟**——就像按照菜谱一步步做饭：先放米再加水，顺序错了就会煮成夹生饭。这里的“菜谱”就是小Y和小L的程序步骤，我们要**严格按照题目描述的顺序，把每一步转换成代码**。  

本题的核心是：小Y的程序是“先加5，再乘3”；小L的程序是“先乘3，再加5”。两者的步骤顺序不同，结果自然不一样。核心难点在于**避免混淆步骤顺序**，解决方案很简单——把两个人的计算过程分开，用不同变量存储结果（比如小Y用`ay`，小L用`al`），或者直接推导数学公式（小Y的结果是`(a+5)*3`，小L是`3*a+5`）。

## 3. 核心难点辨析与解题策略

在解决这个问题时，同学们常遇到以下3个核心难点，我结合题解总结了应对策略：


### 1. 难点1：混淆两个程序的步骤顺序  
**分析**：小Y是“加5→乘3”，小L是“乘3→加5”，顺序反了结果完全不同（比如a=4时，小Y是27，小L是17）。  
**策略**：用**不同变量**存储两人的计算过程（比如`ay`和`al`），或者**写注释**标注每一步对应谁的程序。


### 2. 难点2：忘记用括号保证运算顺序  
**分析**：C++中乘法优先级比加法高，如果小Y的步骤写成`a+5*3`，结果会变成`a+15`，完全错误！  
**策略**：**只要步骤是“先加后乘”，一定要加括号**（`(a+5)*3`）；“先乘后加”不需要括号（`3*a+5`）。


### 3. 难点3：输入输出的位置错误  
**分析**：有同学会把输入写在函数外面，或者忘记输出结果，导致程序无法运行。  
**策略**：记住“输入一定在main函数里”，“最后一定要输出结果”——就像考试写完卷子要交上去，否则老师看不到你的答案！


### ✨ 解题技巧总结
- **变量命名要“见名知意”**：用`ay`、`al`代替`x`、`y`，避免混淆；  
- **用数学公式简化步骤**：能直接算的不要分步写（比如`(a+5)*3`比`a+5`再乘3更高效）；  
- **写注释标注步骤**：比如`// 小Y的步骤：先加5再乘3`，防止自己忘。


## 4. C++核心代码实现赏析

在深入分析具体题解之前，先看一个**通用核心代码**——它综合了两个题解的优点，简洁高效：


### 本题通用核心C++实现参考
* **说明**：本代码用最简洁的方式实现了题目要求，变量命名清晰，数学公式直接明了。  
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int a;
      cin >> a;                  // 输入变量a
      int ay = (a + 5) * 3;     // 小Y的结果：先加5再乘3
      int al = 3 * a + 5;       // 小L的结果：先乘3再加5
      cout << ay << " " << al;  // 输出结果
      return 0;
  }
  ```
* **代码解读概要**：  
  代码分三步：①输入`a`；②分别计算小Y和小L的结果（用数学公式直接算）；③输出两个结果。关键是**用括号保证小Y的步骤顺序**，用不同变量存储结果避免混淆。


### 题解一：作者Igallta
* **亮点**：直接用数学公式简化步骤，变量命名对应人物，清晰易懂。  
* **核心代码片段**：
  ```cpp
  int a, xl, xy;  // xl:小L的结果，xy:小Y的结果
  int main() {
      cin >> a;
      xl = (a + 5) * 3;  // 小L：先加后乘
      xy = 3 * a + 5;    // 小Y：先乘后加
      cout << xl << ' ' << xy;
  }
  ```
* **代码解读**：  
  - `xl`和`xy`分别对应小L和小Y的结果，**见名知意**；  
  - `(a+5)*3`用括号保证“先加后乘”，`3*a+5`利用乘法优先级直接写——这就是“用数学规律简化代码”的技巧！  
* 💡 **学习笔记**：数学公式能帮我们少写很多代码，一定要学会用！


### 题解二：作者yummy
* **亮点**：用`ay`、`al`分开存储两人的`a`，彻底避免混淆。  
* **核心代码片段**：
  ```cpp
  int main() {
      int a;
      cin >> a;
      int ay = a;  // 小Y的a
      ay += 5;     // 加5
      ay *= 3;     // 乘3
      int al = a;  // 小L的a
      al *= 3;     // 乘3
      al += 5;     // 加5
      cout << ay << " " << al;
  }
  ```
* **代码解读**：  
  - 把小Y和小L的`a`分别赋值给`ay`和`al`，然后**分步执行步骤**——这种写法适合刚学C++的同学，能更直观看到每一步的变化；  
  - `ay += 5`等价于`ay = ay + 5`，`ay *= 3`等价于`ay = ay * 3`——这是C++的“复合赋值运算符”，能简化代码！  
* 💡 **学习笔记**：分步写代码能帮你理清思路，熟练后再用数学公式简化。

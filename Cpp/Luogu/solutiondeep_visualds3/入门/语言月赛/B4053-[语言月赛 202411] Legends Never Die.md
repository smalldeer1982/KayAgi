# [语言月赛 202411] Legends Never Die

## 题目描述

在《迪迪卫》中，每位英雄拥有战力 $A$，坦度 $B$，担当 $C$。

小 Z 有一个评估参数 $K$，他将使用 $A, B, C$ 评价一位英雄：

- 如果 $A+B\ge K$，则这位英雄是**全能王**。
- 如果 $A+C\ge K$，则这位英雄是**主力骨**。
- 如果 $B+C\ge K$，则这位英雄是**可靠者**。

给你一位英雄的三个数值与评估参数，请你依次判断这位英雄是否是**全能王**、**主力骨**、**可靠者**。如果是，输出 `Yes`，否则，输出 `No`。

## 说明/提示

### 样例 #1 解释

对这位英雄，有 $A = 1, B = 3, C = 4, K = 5$。

- $A + B = 1 + 3 = 4 < 5$，因此这位英雄不是**全能王**。
- $A + C = 1 + 4 = 5 \geq 5$，因此这位英雄是**主力骨**。
- $B + C = 3 + 4 = 7 \geq 5$，因此这位英雄是**可靠者**。

因此依次输出 `No`、`Yes`、`Yes`。

### 数据范围

对于 $100\%$ 的数据，$0\le A,B,C,K\le 100$。

对于 $20\%$ 的数据，$K=0$。

另有 $20\%$ 的数据，$B=C=0$。

## 样例 #1

### 输入

```
1 3 4 5
```

### 输出

```
No
Yes
Yes
```

## 样例 #2

### 输入

```
2 3 1 0
```

### 输出

```
Yes
Yes
Yes
```

## 样例 #3

### 输入

```
7 0 0 5
```

### 输出

```
Yes
Yes
No
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
对于初学者来说，这道题的“陷阱”主要在“细节”上。我们来逐一拆解核心难点和解决办法：
</difficulty_intro>

1.  **难点1：准确对应三个条件的变量组合**  
    * **分析**：题目中的三个条件分别是“A+B≥K”“A+C≥K”“B+C≥K”，容易记混顺序（比如把“主力骨”写成B+C）。  
    * **解决策略**：**“画关键词”法**——把每个条件的两个变量圈出来（比如“全能王”圈A和B，“主力骨”圈A和C，“可靠者”圈B和C），写代码时对照关键词。  
    * 💡 **学习笔记**：条件判断的第一步，是**“精准翻译题目要求”**，不要自己“创造条件”。

2.  **难点2：正确使用`if-else`语句的结构**  
    * **分析**：初学者可能会忘记`else`分支，或者把条件写反（比如写成`K≥A+B`）。  
    * **解决策略**：**“先想结果，再写条件”**——比如“如果满足A+B≥K，就输出Yes，否则输出No”，对应代码就是`if(A+B>=K) cout<<"Yes\n"; else cout<<"No\n";`。  
    * 💡 **学习笔记**：`if-else`的逻辑是“二选一”，要确保每个条件都有对应的“是/否”结果。

3.  **难点3：注意输出的格式要求**  
    * **分析**：题目要求输出Yes/No（大小写不能错），且每个结果占一行（需要加`\n`换行）。  
    * **解决策略**：**“复制样例输出”法**——把样例中的Yes和No直接复制到代码里，避免拼写错误；每个`cout`后面加`\n`确保换行。  
    * 💡 **学习笔记**：编程中的“格式分”很重要，要严格按照题目要求输出。

### ✨ 解题技巧总结
- **技巧A：“一一对应”法**：把题目中的每个条件和代码的`if`语句一一对应，不跳过任何细节。  
- **技巧B：“样例验证”法**：写完代码后，用样例输入测试（比如样例1输入1 3 4 5，检查三个条件的输出是否正确）。  
- **技巧C：“代码模块化”**：把每个条件的判断写成独立的`if-else`块，让代码更清晰。

* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std; // 简化cout的使用

    int main() {
        int A, B, C, K;
        cin >> A >> B >> C >> K; // 输入四个数值
        
        // 判断全能王（A+B≥K）
        if (A + B >= K) cout << "Yes\n";
        else cout << "No\n";
        
        // 判断主力骨（A+C≥K）
        if (A + C >= K) cout << "Yes\n";
        else cout << "No\n";
        
        // 判断可靠者（B+C≥K）
        if (B + C >= K) cout << "Yes\n";
        else cout << "No\n";
        
        return 0;
    }
    ```
* **代码解读概要**：  
  代码分为三步：① 输入四个变量（A、B、C、K）；② 用三个`if-else`语句分别判断三个条件；③ 输出对应的结果。每个`if-else`块对应一个条件，逻辑直接，容易理解。

# [语言月赛 202412] 聪明猪猪大赛

## 题目描述

ICPC（Interactive Clever Pig Contest，交互聪明猪猪大赛）和 CCPC（Classic Clever Pig Contest，经典聪明猪猪大赛）是两个国际知名猪猪竞赛。其中，ICPC 自猪历 $a$ 年起举办（包括猪历 $a$ 年），每年举办 $b$ 场比赛；CCPC 自猪历 $c$ 年起举办（包括猪历 $c$ 年），每年举办 $d$ 场比赛。

你想知道猪历 $x$ 年这一年 ICPC 和 CCPC 共举办了多少场比赛。

## 说明/提示

### 数据规模与约定

| 测试点编号 | 约定 |
| :-: | :-: |
| $1$ | $a = b=c=d = 1$ |
| $2$ | $c = a = 1$ |
| $3$ | $a = 1$ |
| $4$ | $c = 1$ |
| $5$ | $x = 1$ |
| $6$ | $x \geq a, c$ |
| $7,8,9,10$ | 无特殊约定 |

对全部的测试数据，保证 $1 \le a,b,c,d,x \leq 10^9$。

## 样例 #1

### 输入

```
1
2
3
4
5
```

### 输出

```
6
```

## 样例 #2

### 输入

```
5
1
2
1
4
```

### 输出

```
1
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：编程技巧应用（侧重基础条件判断）

🗣️ **初步分析**：  
解决这道题的关键，就像我们每天早上判断“要不要带外套”——如果今天气温低于10℃（对应题目中“x年≥ICPC开始年a”），就带外套（加b场比赛）；如果今天下雨（对应“x年≥CCPC开始年c”），就带雨伞（加d场比赛）。最终要算的“总带的东西数”，就是ICPC和CCPC在x年的总比赛场次。  

**题解思路**：核心逻辑是**两个条件判断**——分别检查x年是否在ICPC和CCPC的举办年份范围内，如果是，就把对应的年场次加到总数里。  
**核心难点**：① 理解“自某 year 起举办（包括该年）”对应代码中的`x >= a`（而不是`x > a`）；② 避免数据溢出（比如当b和d都是1e9时，int类型存不下总和，需要用long long）。

## 3. 核心难点辨析与解题策略

在解决这个问题时，初学者常遇到以下3个关键点：

1. **难点1：理解“包括该年”的条件转换**  
   * **分析**：题目说“ICPC自a年起举办（包括a年）”，意味着x年只要≥a，就举办b场。比如a=3，x=3时要算，x=2时不算。对应代码就是`if (x >= a)`，而不是`x > a`。  
   * 💡 **学习笔记**：“包括”对应“大于等于”，“不包括”对应“大于”，这点要记牢！

2. **难点2：处理大数溢出问题**  
   * **分析**：题目中数据范围是1≤a,b,c,d,x≤1e9，如果用`int`类型存储`ans`，当b和d都是1e9时，`ans = b + d = 2e9`，而`int`通常最大只能存到约2e9（取决于编译器），但更安全的做法是用`long long`（能存到9e18）。  
   * 💡 **学习笔记**：遇到“大数相加”时，优先用`long long`代替`int`！

3. **难点3：逻辑的完整性（不遗漏条件）**  
   * **分析**：要同时判断ICPC和CCPC两个条件，不能只算其中一个。比如样例1中a=1，c=3，x=5，两个条件都满足，所以总场次是2+4=6，和样例输出一致。  
   * 💡 **学习笔记**：题目有两个独立条件时，要分别判断，不能“二选一”！


### ✨ 解题技巧总结
- **技巧A：条件判断要“对应题目描述”**：题目说“包括该年”，就用`>=`；说“之后”，就用`>`。
- **技巧B：用合适的变量类型避免溢出**：大数用`long long`，小数用`int`。
- **技巧C：代码要“直译题目”**：变量名对应题目中的字母（比如`a`对应ICPC开始年），逻辑对应题目中的要求（比如两个条件对应两个`if`），这样写出来的代码又好懂又不容易错！


## 4. C++核心代码实现赏析

在深入分析具体题解前，先看一个**通用核心实现**——解决了数据溢出问题，更符合竞赛要求：

### 本题通用核心C++实现参考
* **说明**：本代码基于题解思路优化，替换`int`为`long long`，避免大数溢出，是更严谨的实现。
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      long long a, b, c, d, x;
      cin >> a >> b >> c >> d >> x;
      long long ans = 0;
      if (x >= a) ans += b;
      if (x >= c) ans += d;
      cout << ans << endl;
      return 0;
  }
  ```
* **代码解读概要**：  
  1. **读入数据**：用`long long`类型读入a（ICPC开始年）、b（ICPC年场次）、c（CCPC开始年）、d（CCPC年场次）、x（查询年）。  
  2. **计算总场次**：初始化`ans`为0，分别判断x是否≥a和c，如果是就加对应的场次。  
  3. **输出结果**：打印`ans`，就是x年的总比赛场次。


### 题解一：来源：一扶苏一
* **亮点**：用最简洁的代码实现核心逻辑，完美匹配题目要求，适合初学者理解基础条件判断。
* **核心代码片段**（原代码，注意：原代码用了`int`，建议替换为`long long`）：
  ```cpp
  int a, b, c, d, x;
  cin >> a >> b >> c >> d >> x;
  int ans = 0;
  if (x >= a) ans += b;
  if (x >= c) ans += d;
  cout << ans << endl;
  ```
* **代码解读**：  
  这段代码是“题目到代码的直接翻译”！比如：  
  - `cin >> a >> b >> c >> d >> x;`对应“读取五个输入变量”；  
  - `if (x >= a) ans += b;`对应“如果x年≥ICPC开始年a，加b场”；  
  - `if (x >= c) ans += d;`对应“如果x年≥CCPC开始年c，加d场”；  
  - `cout << ans << endl;`对应“输出总场次”。  
  唯一需要注意的是：原代码用`int`可能溢出，比如当b和d都是1e9时，`ans`会变成负数（因为int溢出），所以要改成`long long`！
* 💡 **学习笔记**：代码的“简洁”不等于“不严谨”——要根据数据范围选择合适的变量类型！

采用8位像素风是为了让学习更轻松——就像玩小时候的“超级马里奥”一样，猪猪的每一步操作都有音效和动画，强化你的记忆。比如：

3. **条件判断1（检查ICPC）**：

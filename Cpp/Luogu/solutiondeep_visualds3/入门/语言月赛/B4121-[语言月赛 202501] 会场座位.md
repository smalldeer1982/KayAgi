# 题目信息

# [语言月赛 202501] 会场座位

## 题目描述

一个会场有 $100$ 个座位，座位的编号从左到右依次是 $99,97,\ldots,7,5,3,1,0,2,4,6,\ldots,96,98$。

给出两个**不同的**座位 $a,b$，请判断座位 $a,b$ 之间间隔了多少个座位。

## 说明/提示

【样例 1 解释】

座位 $2,5$ 之间有 $0,1,3$ 这三个座位，所以输出 $3$。

【样例 2 解释】

座位 $2,6$ 之间有 $4$ 这个座位，所以输出 $1$。

【样例 3 解释】

座位 $85,83$ 之间没有座位，所以输出 $0$。

【数据范围】

保证 $0\le a,b \le 99$ 并且 $a\ne b$。

## 样例 #1

### 输入

```
2 5
```

### 输出

```
3
```

## 样例 #2

### 输入

```
2 6
```

### 输出

```
1
```

## 样例 #3

### 输入

```
85 83
```

### 输出

```
0
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：数学（通过规律推导公式解决问题）

🗣️ **初步分析**：  
数学的核心思想就像“找规律搭积木”——先观察问题的排列规律，再用公式把规律“固定”下来，避免一遍一遍模拟。在这道题里，会场座位的排列藏着一个小规律：**奇数座位从99往左降序排（像从高到矮站成一队），偶数座位从0往右升序排（像从矮到高站成一队）**，中间的“中缝”（0和1之间）把座位分成了左右两半。  

解决这道题的关键，是找到**“座位号”和“实际位置”的数学关系**：  
- 不管座位是奇数还是偶数，它到“中缝”的距离都等于「座位号÷2的整数部分」（比如5÷2=2，2÷2=1）。  
- 如果两个座位一个是奇数、一个是偶数（在中缝两侧），它们之间的间隔数就是“两者到中缝的距离之和”（比如2和5，1+2=3）；  
- 如果两个座位同是奇数或同是偶数（在中缝同一侧），间隔数就是“两者到中缝的距离差减1”（比如85和83，差1减1=0）。  

**可视化设计思路**：我们可以用“像素座位队列”来演示——蓝色方块代表奇数座位，绿色代表偶数座位，黄色线条标记中缝。当计算时，用红色箭头指向当前座位，“叮”的音效提示奇偶判断，“唰”的音效提示计算距离，最后高亮间隔的座位并播放胜利音效，帮你直观记住规律！

## 3. 核心难点辨析与解题策略

### 核心难点1：看懂座位的“隐藏排列规律”  
**问题**：座位号是99→97→…→1→0→2→4→…→98，这种“先降后升”的排列很容易绕晕。  
**解法**：把座位分成“奇数队”和“偶数队”：奇数队从大到小站在中缝左边，偶数队从小到大站在中缝右边。通过样例反推规律（比如5在1左边，2在0右边），就能找到“座位号→到中缝的距离”的公式。  
💡 **学习笔记**：遇到“奇怪排列”别慌，先找“分组规律”（比如奇偶分组），再推导公式！

### 核心难点2：分情况处理“奇偶不同”的情况  
**问题**：两个座位一个在左、一个在右，和两个都在左/右的计算方式不一样，容易混淆。  
**解法**：用“奇偶判断”分两条路：  
- 奇偶不同→两侧→距离相加；  
- 奇偶相同→同侧→距离差减1。  
💡 **学习笔记**：分情况讨论的关键是“找分界点”（这里的分界点是“中缝”），把复杂问题拆成简单子问题！

### 核心难点3：正确用C++实现“向下取整”  
**问题**：计算“座位号÷2的整数部分”，比如5÷2=2，2÷2=1，0÷2=0。  
**解法**：C++的整数除法（比如`5/2`）会自动“向下取整”，正好符合我们的需求！不需要额外调用函数～  
💡 **学习笔记**：多熟悉编程语言的“小特性”，能帮你少写很多代码！


### ✨ 解题技巧总结  
- **规律优先**：先观察问题的“重复模式”，再用数学公式把模式固定下来；  
- **分情况拆题**：遇到“不同情况不同结果”的问题，先找“分界条件”（比如奇偶）；  
- **利用语言特性**：C++的整数除法、取模运算（`%`）能快速解决奇偶判断和取半问题！


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考  
* **说明**：这份代码来自题解yummy的思路，是“数学推导+分情况处理”的极简实现，能直接解决问题！  

```cpp
#include <iostream>
#include <cstdlib>  // 用于abs函数（计算绝对值）

using namespace std;

int main() {
    int a, b;
    cin >> a >> b;  // 输入两个座位号
    int fa = a / 2; // 计算a到中缝的距离（整数除法，向下取整）
    int fb = b / 2; // 计算b到中缝的距离
    if ((a % 2) != (b % 2)) { // 判断奇偶性是否不同
        cout << fa + fb << endl; // 两侧：距离相加
    } else {
        cout << abs(fa - fb) - 1 << endl; // 同侧：距离差减1
    }
    return 0;
}
```

* **代码解读概要**：  
代码分三步：  
1. 读入两个座位号`a`和`b`；  
2. 计算它们到中缝的距离`fa`和`fb`（用整数除法取半）；  
3. 用`%`判断奇偶性：不同则相加，相同则算差减1。  
整个代码没有多余的步骤，完全是“数学公式的直接翻译”！


### 题解一：(来源：yummy)  
* **亮点**：用“数学公式”代替“模拟”，代码简洁到“极致”！  
* **核心代码片段**：  
```cpp
int fa = a / 2;
int fb = b / 2;
if ((a % 2) != (b % 2)) {
    cout << fa + fb << endl;
} else {
    cout << abs(fa - fb) - 1 << endl;
}
```
* **代码解读**：  
这段代码是题解的“灵魂”——  
- `fa = a / 2`：比如`a=5`（奇），`fa=2`（到中缝有2个座位：3、1）；`a=2`（偶），`fa=1`（到中缝有1个座位：0）。  
- `a%2`：判断`a`是奇数（结果1）还是偶数（结果0）。如果`a`和`b`的奇偶性不同，说明它们在中缝两侧，间隔数就是`fa+fb`（比如2和5的`fa=1`+`fb=2`=3）；如果相同，说明在同侧，间隔数是“距离差减1”（比如85和83的`fa=42`-`fb=41`=1，减1得0）。  
* 💡 **学习笔记**：代码的简洁度，往往取决于“数学推导的深度”——推导越透彻，代码越短！

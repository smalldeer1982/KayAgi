# 题目信息

# [语言月赛 202503] 长方形

## 题目背景

对于一个长为 $a$，宽为 $b$ 的长方形，它的面积是 $S = a \times b$，它的周长是 $L = 2 \times (a + b)$。


## 题目描述

假设一个长方形的长和宽均为整数，给定它的长 $a$ 和面积 $S$，请你求出这个长方形的周长 $L$。

## 说明/提示

### 样例 1 说明

给定长方形的面积是 $S = 24$，长为 $a = 4$，因此它的宽是 $b = S \div a = 24 \div 4 = 6$，最终的周长为 $L = 2 \times (a + b) = 2 \times (4 + 6) = 20$。

### 数据规模与约定

对于 $100\%$ 的数据，$1 \leq a, S \leq 100$。保证 $S$ 是 $a$ 的倍数。

数据中可能出现宽大于长，即，$b \geq a$ 的情况。这仅代表本题的特殊情况，在实际的数学学习中，有关“长”和“宽”的严格定义，请以课堂讲授为准。


## 样例 #1

### 输入

```
4 24
```

### 输出

```
20
```

## 样例 #2

### 输入

```
8 72
```

### 输出

```
34
```

## 样例 #3

### 输入

```
1 100
```

### 输出

```
202
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：编程技巧应用（基础顺序结构）

🗣️ **初步分析**：  
解决“长方形”这道题，关键在于**顺序结构**的运用——它就像你早上穿衣服的步骤：先穿袜子，再穿裤子，最后穿鞋子，一步跟着一步来，不能乱。在本题中，顺序结构就是“先算宽→再算周长”的固定流程：  
- 题目给了长`a`和面积`S`，根据面积公式`S = a×b`，先算出宽`b = S÷a`；  
- 再用周长公式`L = 2×(a+b)`算出结果。  

核心难点其实是“把数学公式翻译成代码”——比如怎么用C++的运算符表示“除以”（`/`）和“乘以”（`*`），以及保证计算顺序没错（比如先算`a+b`再乘以2，括号不能丢）。

## 3. 核心难点辨析与解题策略

在解决这个问题时，大家常遇到的3个关键点和应对方法：

1. **关键点1：从面积公式推导出宽`b`**  
   - **分析**：很多同学会先想“面积是长乘宽”，但反过来求宽时容易忘“面积除以长”。  
   - **策略**：先把数学公式写在纸上：`b = S / a`，再对应到代码里的`S / a`（注意C++中整数除法用`/`）。  
   - 💡 **学习笔记**：代码是数学的“翻译官”，先理清楚公式再写代码！

2. **关键点2：正确应用周长公式**  
   - **分析**：周长公式是`2×(a+b)`，如果漏掉括号写成`a + b * 2`，结果就错了（比如`a=4`、`b=6`，会算成`4+12=16`而不是`20`）。  
   - **策略**：先算`a+b`再乘以2，括号一定要加上！  
   - 💡 **学习笔记**：运算符优先级很重要，不确定时就用括号“强制”顺序。

3. **关键点3：变量类型的选择**  
   - **分析**：题目说长、宽、面积都是整数，所以用`int`类型就够了（不用`float`或`double`，避免小数问题）。  
   - **策略**：直接定义`int a, S, b, L;`，简单又安全。  
   - 💡 **学习笔记**：变量类型要匹配数据特点，整数用`int`，小数用`double`！

### ✨ 解题技巧总结
- **技巧A：先写公式再写代码**：把数学关系理清楚，代码就是公式的“直译”。  
- **技巧B：变量名要“见名知意”**：用`a`、`S`、`b`、`L`对应题目中的符号，比`x`、`y`好懂10倍。  
- **技巧C：测试样例验证**：写完代码后用样例1（`a=4`、`S=24`）跑一遍，看结果是不是`20`，确保没错。


## 4. C++核心代码实现赏析

在深入分析具体题解的精妙片段之前，让我们先来看一个能够完整解决本题的通用核心C++实现参考。这有助于我们对整体解题框架有一个把握。

### 本题通用核心C++实现参考
* **说明**：本代码来自Maxmilite的题解，是“数学公式→代码”的典型实现，逻辑清晰、简洁高效。  
* **完整核心代码**：
  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int a, S;
      cin >> a >> S;
      int b = S / a;
      int L = (a + b) * 2;
      cout << L << endl;
      return 0;
  }
  ```
* **代码解读概要**：  
  这段代码只有3个核心步骤：  
  1. **读入数据**：用`cin`读入题目给的长`a`和面积`S`；  
  2. **计算宽和周长**：先算`b = S / a`（宽），再算`L = (a + b) * 2`（周长）；  
  3. **输出结果**：用`cout`把周长`L`打印出来。  


### 题解一：(来源：Maxmilite)
* **亮点**：直接对应数学公式，代码没有“冗余”，新手也能快速看懂。  
* **核心代码片段**：
  ```cpp
  int b = S / a;
  int L = (a + b) * 2;
  cout << L << endl;
  ```
* **代码解读**：  
  这段代码是整道题的“核心计算逻辑”——  
  - `int b = S / a;`：用面积除以长，得到宽`b`（比如样例1中`24/4=6`）；  
  - `int L = (a + b) * 2;`：先算长加宽（`4+6=10`），再乘以2得到周长（`10×2=20`）；  
  - `cout << L << endl;`：把结果打印出来。  
  注意哦，`(a + b)`的括号不能丢！如果写成`a + b * 2`，结果会变成`4+6×2=16`，就错啦！  
* 💡 **学习笔记**：代码里的括号和数学里的括号一样重要，能保证计算顺序正确！

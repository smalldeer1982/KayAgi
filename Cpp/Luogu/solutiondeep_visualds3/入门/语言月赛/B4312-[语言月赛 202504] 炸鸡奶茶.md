## 题目背景

**本题中的情景为虚构，请各位做题者均衡饮食。**

## 题目描述

小 M 的学校附近有一家炸鸡店和一家奶茶店。炸鸡 $f$ 元每份，奶茶 $m$ 元每杯。

小 M 每周四要买两份炸鸡和一杯奶茶。**今天是周五**，请计算接下来的 $n$ 天（包括今天） 里，小 A 一共要在炸鸡和奶茶上花费多少钱。


## 说明/提示

【样例 1 解释】

炸鸡 $25$ 元每份，奶茶 $28$ 元每杯。

在接下来的 $10$ 天里，一共有 $1$ 个星期四，这个星期四小 M 买两份炸鸡、一份奶茶共花费 $2\times 25 + 1\times 28 = 78$ 元，因此输出 $78$。

【样例 2 解释】

炸鸡 $8$ 元每份，奶茶 $6$ 元每杯。

在接下来的 $20$ 天里，一共有 $2$ 个星期四。每个星期四小 M 都要花费 $2\times 8 + 1\times 6=22$ 元。因为要经过两个星期四，所以总花费为 $44$ 元。

【数据范围】

$1\le n,f,m\le 100$。

## 样例 #1

### 输入

```
25 28 10
```

### 输出

```
78
```

## 样例 #2

### 输入

```
8 6 20
```

### 输出

```
44
```

## 样例 #3

### 输入

```
8 6 21```

### 输出

```
66
```

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`数学 (基础算术与整数除法应用)`

🗣️ **初步分析**：
解决这道题的关键，就像“数苹果”——先算有多少个“要花钱的周四”，再算每个周四花多少钱，最后把两者乘起来就是总花费。  

### 算法核心与本题应用
`数学`在这里的核心是“用简单算术解决实际计数问题”：就像你每周四固定要花零花钱买零食，那一个月（比如30天）要花多少次？用30除以7（一周7天）就能算出大概4次——这就是本题的“周四数量”计算逻辑！  

### 题解思路与核心难点
所有题解的核心思路都一样：**算周四数量 → 算单次花费 → 总花费=数量×单次花费**。  
核心难点有两个：  
1. 为什么周四数量是`n//7`？因为今天是周五，接下来的第7天是第一个周四，第14天是第二个……每7天一个，所以直接用“总天数÷7”取整数部分（比如10天→1个，21天→3个）。  
2. 单次花费是`2×f + m`：两份炸鸡加一杯奶茶，要注意乘法先于加法（不用额外括号也对，但加括号更清晰哦）。  

### 可视化设计思路
我们可以做一个`像素日历模拟器`：用8位像素块代表“天”，周五开始是第一个像素（蓝色），周四的像素用`亮黄色`标记。动画会`单步播放`每天的推进——当走到周四时，黄色像素会`闪烁`，旁边弹出“花费+X元”的提示，总金额同步增加。还会加一个“叮”的音效，强化“花钱”的记忆点！

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”都藏在“细节理解”里，我们逐一解决：
</difficulty_intro>

1.  **难点1：怎么算“接下来n天有多少个周四”？**
    * **分析**：今天是周五，所以第7天是第一个周四，第14天第二个……就像“每7个苹果装一袋”，总天数里有几袋就是几个周四。用`n//7`（整数除法）就能算出——比如10天÷7=1（余3），所以1个；21天÷7=3，正好3个。
    * 💡 **学习笔记**：当问题涉及“每X个出现一次”的计数时，优先想“整数除法”！

2.  **难点2：单次花费为什么是`2×f + m`？**
    * **分析**：题目说“两份炸鸡（每份f元）+一杯奶茶（每杯m元）”，数学上就是`2*f + m`。注意乘法要先算，所以不用额外括号也对，但加括号（比如`(2*f + m)`）会让代码更易读。
    * 💡 **学习笔记**：把题目中的“数量×单价”直接翻译成代码，不要漏乘或漏加！

3.  **难点3：结果怎么整合？**
    * **分析**：总花费=周四的数量 × 每个周四的花费。就像“买3次零食，每次20元，总共60元”——直接相乘就行。
    * 💡 **学习笔记**：复杂问题拆成“小模块”计算，再组合结果，会更简单！

### ✨ 解题技巧总结
- **技巧1：问题拆解**：把“总花费”拆成“次数×单次花费”，逐个解决。
- **技巧2：整数除法**：遇到“每X个一组”的计数，用`n//X`（比如每周7天，算周数）。
- **技巧3：代码可读性**：加括号、写清晰的变量名（比如`thursdays`代替`t`），让代码自己“说话”。

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份完整的核心代码——它把题解的思路直接变成了可运行的C++代码，超级直观！
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：综合题解思路，写出的最简洁实现，适合入门学习。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int f, m, n;
        cin >> f >> m >> n;  // 读入炸鸡价格、奶茶价格、总天数
        int thursdays = n / 7;  // 计算周四的数量（整数除法）
        int cost_per_thursday = 2 * f + m;  // 每个周四的花费
        int total = thursdays * cost_per_thursday;  // 总花费
        cout << total << endl;  // 输出结果
        return 0;
    }
    ```
* **代码解读概要**：
> 代码分四步：1. 读入三个输入值（f、m、n）；2. 算周四的数量（n//7）；3. 算每个周四的花费（2f+m）；4. 算总花费并输出。没有复杂逻辑，全是“直来直往”的算术！

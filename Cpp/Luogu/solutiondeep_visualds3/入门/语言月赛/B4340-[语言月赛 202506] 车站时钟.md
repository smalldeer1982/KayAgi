# [语言月赛 202506] 车站时钟

## 题目描述

车站的时钟会显示当前的时间。现在给出从当天零点（00:00）开始经过的分钟数，请你按照指定的格式画出这个时钟。

时钟的显示格式如下：
- 时钟由 `|` 和 `:` 符号组成
- 小时和分钟都用两位数表示，不足两位时前面补零
- 小时和分钟之间用 `:` 分隔

例如，$60$ 分钟对应的时间是 01:00，时钟显示为 `|0|1|:|0|0|`。

## 说明/提示

### 样例 1 解释

$60$ 分钟为一小时。当天零点（00:00）开始经过 $60$ 分钟，即为经过 $60 / 60 = 1$ 小时，因此显示时间为 01:00。

### 样例 2 解释

$60$ 分钟为一小时。当天零点（00:00）开始经过 $1203$ 分钟，即为经过 $1200 / 60 = 20$ 小时 + $3$ 分钟，因此显示时间为 20:03。

### 数据规模与约定

对于 $100\%$ 的数据，$0 \leq x \leq 1439$（1440 分钟即为 24 小时，即回到 00:00）。

## 样例 #1

### 输入

```
60
```

### 输出

```
|0|1|:|0|0|
```

## 样例 #2

### 输入

```
1203
```

### 输出

```
|2|0|:|0|3|
```

## 样例 #3

### 输入

```
0
```

### 输出

```
|0|0|:|0|0|
```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题的“坑”主要藏在“细节处理”里——比如怎么把1变成01，怎么拼对`|`和`:`的顺序。结合题解的思路，我们总结了3个核心难点和解决策略：
</difficulty_intro>

1. **难点1：把总分钟数转换成小时和分钟**  
   * **分析**：一天有1440分钟（24×60），总分钟数`x`要分成“小时”和“剩余分钟”。比如60分钟是1小时0分钟，1203分钟是20小时3分钟。  
   * **解决策略**：用整数除法`h = x / 60`（总分钟÷60得小时数），用取余`m = x % 60`（总分钟%60得剩余分钟）。这两个运算就像“把苹果分成整箱和零散的”——60个苹果装1箱（小时），剩下0个（分钟）。  
   * 💡 **学习笔记**：整数除法和取余是处理“拆分数值”的神器！

2. **难点2：让数字变成两位数（补前导零）**  
   * **分析**：比如小时是1，要显示成“01”；分钟是3，要显示成“03”。直接输出1会变成“1”，不符合要求。  
   * **解决策略**：把数字拆成“十位”和“个位”——十位用`数字 / 10`（比如1÷10=0），个位用`数字 % 10`（比如1%10=1）。这样不管数字是1还是20，都能拆出两位。  
   * 💡 **学习笔记**：拆数字的“十位+个位”，是补前导零的“无字符串解法”！

3. **难点3：按格式拼接符号和数字**  
   * **分析**：最终格式是`|十位|个位|:|十位|个位|`，比如01:00要拼成`|0|1|:|0|0|`，顺序错一点就会出错。  
   * **解决策略**：按题目给的例子“照葫芦画瓢”——先写`|`，再写小时十位，再写`|`，再写小时个位，再写`|:|`（注意这里是`|`+`:`+`|`），然后写分钟十位，`|`，分钟个位，最后`|`。  
   * 💡 **学习笔记**：复杂格式拼接的秘诀是“对照样例，逐个写”！

### ✨ 解题技巧总结
<summary_best_practices>
通过这道题，我们可以总结3个通用技巧：
</summary_best_practices>
- **技巧1：用整数运算拆分数值**：遇到“把大数值分成几个部分”的问题（比如分钟转小时、数字拆位），优先用`/`和`%`，比字符串处理更高效。
- **技巧2：对照样例写格式**：复杂输出格式不要自己猜，直接对照题目给的例子（比如样例1的`|0|1|:|0|0|`），逐个字符写。
- **技巧3：用简洁变量名**：变量名`h`（hour）、`m`（minute）比`a`、`b`更易懂，能减少代码出错的概率。

    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int x;
        cin >> x;  // 读取总分钟数

        int m = x % 60;  // 计算剩余分钟数

    }
    ```

# 西游记公司

## 题目背景

一道极其无厘头的题目。

## 题目描述

事情是这样的：  

西游记中的孙沙猪三徒弟在西天取经之后开始进入厦门大学经贸系学习经济，在 $1$ 个小时的学习后，他们用特殊手段毕业了。

然后，他们创办了三个公司：“花果山生态旅游山庄集团公司” “高老庄猪肉美食城有限公司” 和 “流沙河轮渡有限公司”。虽然这三家公司出自 “经卷出版社” 社长唐三藏的徒弟们，但是每家公司都仍然保持着 $0$ 元以下的财政收入。  

于是，他们想到了一个无聊的方法：向别人偷！其中，孙悟空偷猪八戒的平底锅之前，猪八戒可以一直抢沙和尚的笔记本。

现在，作为猪八戒手下首席智士的你，要帮助既聪明又愚蠢的猪八戒用那种不正当的手段挽救这个公司！  

你可以这样做：你已经知道孙悟空什么时候要来偷猪八戒的东西，而猪八戒又要在什么时间去抢沙和尚的东西，又知道猪八戒每秒钟可以偷沙和尚 $n$ 台笔记本，帮猪八戒算一算，自己在有限的时间内能偷到沙和尚多少台笔记本，使得他有足够的时间来防御大师兄。

## 说明/提示

对于 $100\%$ 的数据，保证两个时间合法，$n<32768$。

## 样例 #1

### 输入

```
00:0:00
0:00:10
10```

### 输出

```
100```

## 引言
今天我们要解决的是“西游记公司”这道有趣的题目——帮猪八戒计算他能偷到多少台笔记本电脑！其实这道题的核心是**时间转换与差值计算**，看起来像“偷东西”，实则是考验我们对“时分秒转秒数”的掌握。让我们一起拆解问题，学会用代码解决这类时间相关的计算吧！


## 1. 题目解读与核心算法识别

✨ **本题主要考察**：模拟与数学计算（时间转换）

🗣️ **初步分析**：  
解决这道题的关键，在于把“时分秒”的时间格式**统一转换成秒数**——就像把不同面值的硬币换成“1分硬币”，这样计算差值就会变得简单！具体来说：  
- 1小时 = 3600秒，1分钟 = 60秒，所以一个时间点的总秒数 = 小时×3600 + 分钟×60 + 秒。  
- 用“孙悟空来偷的时间（结束时间）”减去“猪八戒开始偷的时间（开始时间）”，得到的秒数差，再乘以“每秒偷的笔记本数”，就是最终答案。  

**核心难点**：如何正确读取带冒号的时间格式（比如`00:0:00`），以及如何避免数据溢出（结果可能很大，需要用`long long`类型）。  

**可视化设计思路**：我们可以用8位像素风格模拟“时间转换机”——输入的时分秒是三个彩色方块，点击“转换”后，方块会变成对应的秒数（比如小时方块乘以3600后变大），再用“减法动画”展示差值，最后“乘以每秒偷的数量”会弹出一堆笔记本像素块，直观展示计算过程！

## 3. 核心难点辨析与解题策略

### 1. 难点1：如何处理带冒号的时间输入？  
* **分析**：时间格式是`hh:mm:ss`，我们需要读取其中的数字，跳过冒号。常见的解决方法有两种：  
  - 用`scanf`的格式化输入（比如`scanf("%lld:%lld:%lld", &h, &m, &s)`），直接跳过冒号；  
  - 用`cin`读取数字，用一个`char`变量接收冒号（比如`cin >> h >> ch >> m >> ch >> s`）。  
* 💡 **学习笔记**：格式化输入是处理带分隔符数据的“神器”！


### 2. 难点2：如何避免数据溢出？  
* **分析**：假设时间差是1天（86400秒），每秒偷32767台，结果是`86400×32767=2,831,164,800`，远超过`int`的最大值（约20亿）。因此必须用`long long`类型存储结果。  
* 💡 **学习笔记**：遇到“大数计算”，第一反应是用`long long`！


### 3. 难点3：如何正确转换时间为秒数？  
* **分析**：时间转换的公式是固定的——小时×3600 + 分钟×60 + 秒。比如`0:00:10`就是`0×3600 + 0×60 + 10 = 10`秒。只要公式没错，转换就不会出错。  
* 💡 **学习笔记**：记住“时分秒转秒”的公式，这是解决所有时间差问题的基础！


### ✨ 解题技巧总结  
- **技巧1**：用格式化输入（`scanf`或`cin`+`char`）处理带分隔符的数据；  
- **技巧2**：遇到大数用`long long`；  
- **技巧3**：将复杂问题拆解成“转换→计算差值→相乘”三个简单步骤。


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考  
* **说明**：综合了优质题解的思路，用最简洁的方式实现核心逻辑。  
* **完整核心代码**：  
  ```cpp
  #include <cstdio>
  using namespace std;

  int main() {
      long long h1, m1, s1, h2, m2, s2, t;
      scanf("%lld:%lld:%lld", &h1, &m1, &s1);
      scanf("%lld:%lld:%lld", &h2, &m2, &s2);
      scanf("%lld", &t);
      long long time1 = h1 * 3600 + m1 * 60 + s1;
      long long time2 = h2 * 3600 + m2 * 60 + s2;
      printf("%lld\n", (time2 - time1) * t);
      return 0;
  }
  ```  
* **代码解读概要**：  
  1. 用`scanf`读取两个时间（跳过冒号）和每秒偷的数量；  
  2. 将两个时间都转换成秒数（`time1`和`time2`）；  
  3. 计算时间差，乘以每秒偷的数量，输出结果。


### 针对各优质题解的片段赏析

#### 题解一：Right的极简代码  
* **亮点**：用一个`char`变量`m`跳过所有冒号，输入一行完成！  
* **核心代码片段**：  
  ```cpp
  cin >> a >> m >> b >> m >> c >> x >> m >> y >> m >> z >> t;
  cout << ((x - a) * 3600 + (y - b) * 60 + (z - c)) * t;
  ```  
* **代码解读**：  
  - `m`变量用来接收输入中的冒号（比如`00:0:00`中的`:`），这样`cin`会自动跳过这些字符，直接读取数字；  
  - 计算部分直接用“结束时间 - 开始时间”的小时、分钟、秒分别转换为秒，再相乘。  
* 💡 **学习笔记**：用一个变量跳过分隔符，是处理类似问题的小技巧！


#### 题解二：chenyuxuan02的分步转换  
* **亮点**：明确将时间转换为秒数，步骤清晰，适合新手理解。  
* **核心代码片段**：  
  ```cpp
  s1 = s1 + m1 * 60 + h1 * 3600;
  s2 = s2 + m2 * 60 + h2 * 3600;
  cpts = (s2 - s1) * cpt;
  ```  
* **代码解读**：  
  - `s1`是开始时间的总秒数，`s2`是结束时间的总秒数；  
  - 用`s2 - s1`得到时间差，乘以每秒偷的数量`cpt`，就是结果。  
* 💡 **学习笔记**：分步转换能让逻辑更清晰，避免出错！


#### 题解三：0AND1STORY的类方法  
* **亮点**：用类封装时间，重载运算符，让代码更模块化。  
* **核心代码片段**：  
  ```cpp
  class Time {
  private:
      ll hour, minute, second;
      ll point; // 时间的总秒数
  public:
      Time() { input(); }
      void input() {
          scanf("%lld:%lld:%lld", &hour, &minute, &second);
          point = hour * 3600 + minute * 60 + second;
      }
      ll operator-(Time& cmp) { return point - cmp.point; }
  };
  ```  
* **代码解读**：  
  - `Time`类的构造函数会自动调用`input`方法读取时间，并计算总秒数`point`；  
  - 重载的`-`运算符直接返回两个时间的秒数差，使用时像`t2 - t1`一样方便。  
* 💡 **学习笔记**：面向对象的方法能让代码更易维护，适合复杂项目！

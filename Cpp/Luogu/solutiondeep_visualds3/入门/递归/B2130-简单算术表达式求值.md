# 简单算术表达式求值

## 题目描述

两位正整数的简单算术运算（只考虑整数运算），算术运算为：

`+`，加法运算；

`-`，减法运算；

`*`，乘法运算；

`/`，整除运算；

`%`，取余运算。

算术表达式的格式为（运算符前后可能有空格）：

运算数运算符运算数。

请输出相应的结果。

## 样例 #1

### 输入

```
32+64```

### 输出

```
96```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
这道题虽然简单，但新手常踩3个“小坑”。结合优质题解的经验，我们一一解决：
</difficulty_intro>

1.  **关键点1：如何处理输入中的空格？**
    * **分析**：题目说“运算符前后可能有空格”，比如输入“32 + 64”或“32+64”。新手可能会想“怎么去掉空格？”，但其实C++的`cin`会自动忽略空白字符（空格、换行、制表符），所以`cin>>n1>>ch>>n2`能直接读对数字和运算符。如果用`scanf`，可以在格式字符串里加空格（比如`"%d %c %d"`），同样能跳过空格。
    * 💡 **学习笔记**：`cin`和`scanf`是“自动去空格小能手”，不用手动处理！

2.  **关键点2：用`switch`还是`if-else`？**
    * **分析**：两种方法都对，但场景不同。`switch`适合**固定值的多分支判断**（比如运算符只有5种可能），代码更简洁；`if-else`适合**范围或复杂条件判断**。本题运算符是明确的字符，用`switch`更“专业”，但`if-else`更“易懂”——新手可以先学`if-else`，再过渡到`switch`。
    * 💡 **学习笔记**：选分支结构就像“选工具”——固定值用`switch`，复杂条件用`if-else`。

3.  **关键点3：整数运算的正确性（比如整除和取余）？**
    * **分析**：题目要求“整数运算”，所以C++的`/`会自动做**整除**（比如`5/2=2`），`%`做**取余**（比如`5%2=1`），正好符合题目要求。而且题目说“两位正整数”，所以不用考虑除数为0的情况（第二个数是正整数），省了很多麻烦！
    * 💡 **学习笔记**：整数运算的`/`和`%`要记牢，不用额外处理小数！

### ✨ 解题技巧总结
<summary_best_practices>
通过这道题，我们能总结3个通用“入门技巧”：
</summary_best_practices>
-   **技巧1：利用语言特性简化输入**：`cin`和`scanf`的自动去空格功能，能帮我们少写很多代码。
-   **技巧2：分支结构选对“工具”**：固定值用`switch`，复杂条件用`if-else`。
-   **技巧3：关注题目条件减少麻烦**：题目说“两位正整数”，所以不用处理除数为0，直接用`/`和`%`就好。

## 4. C++核心代码实现赏析

<code_intro_overall>
先看一份**通用核心实现**——综合了优质题解的优点，兼顾清晰性和规范性：
</code_intro_overall>

**本题通用核心C++实现参考**
* **说明**：本代码整合了`cin`自动去空格、`switch`分支的优点，是最适合新手的“标准写法”。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;

    int main() {
        int num1, num2;
        char op;
        cin >> num1 >> op >> num2; // 自动跳过空格，读取两个数和运算符
        switch (op) {
            case '+': cout << num1 + num2; break;
            case '-': cout << num1 - num2; break;
            case '*': cout << num1 * num2; break;
            case '/': cout << num1 / num2; break;
            case '%': cout << num1 % num2; break;
        }
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码分3步：① 定义变量`num1`（第一个数）、`num2`（第二个数）、`op`（运算符）；② 用`cin`读取输入（自动跳过空格）；③ 用`switch`判断运算符，执行对应运算并输出。整个流程就像“流水线”——输入→判断→输出，没有冗余！

<code_intro_selected>
接下来看3份优质题解的“亮点片段”：
</code_intro_selected>

**题解一：(来源：TensorFlow_js)**
* **亮点**：`switch`分支的“标准用法”，每个`case`对应一个运算符，逻辑清晰。
* **核心代码片段**：
    ```cpp
    switch(ch){
        case '+'://当运算符为+时
        cout<<(n1+n2);
        break;
        case '-'://当运算符为-时
        cout<<n1-n2;
        break;
        // ... 其他运算符
    }
    ```
* **代码解读**：
    > `switch(ch)`中的`ch`是运算符。每个`case`后面跟一个运算符，比如`case '+'`就对应加法。`break`很重要——它会“跳出”`switch`，避免执行后面的`case`（比如如果没有`break`，加完会继续执行减法的代码）。作者用括号把`n1+n2`包起来，是好习惯（避免优先级问题），但其实`cout<<n1+n2`也对~
* 💡 **学习笔记**：`switch`里的`break`不能忘，否则会“串台”！

**题解二：(来源：xuanye)**
* **亮点**：`if-else`的“直白写法”，新手一看就懂。
* **核心代码片段**：
    ```cpp
    if(s=='+'){
        cout<<a+b;
    }
    else if(s=='-'){
        cout<<a-b;
    }
    else if(s=='*'){
        cout<<a*b;
    }
    // ... 其他运算符
    ```
* **代码解读**：
    > 用`if(s=='+')`判断运算符是不是加号，如果是就输出和；否则用`else if`判断是不是减号，依此类推。`else if`会“依次检查”，直到找到匹配的运算符——这种写法就像“逐个问”：“是加号吗？不是。是减号吗？是的！”非常符合新手的思考逻辑。
* 💡 **学习笔记**：`else if`是“依次判断”，比多个`if`更高效（不用检查所有条件）。

**题解三：(来源：凤年)**
* **亮点**：`scanf`的“空格占位”技巧，处理复杂输入更灵活。
* **核心代码片段**：
    ```cpp
    scanf("%d %c %d",&a,&ch,&b);//引号里的空格是精髓
    ```
* **代码解读**：
    > `scanf`的格式字符串`"%d %c %d"`里，`%d`对应整数，`%c`对应字符，中间的空格表示“跳过任意空白字符”。不管输入是“32+64”“32 +64”还是“32   +   64”，`scanf`都会精准读取`a=32`、`ch='+'`、`b=64`。这个技巧在处理复杂格式（比如“姓名 年龄 性别”）时特别有用！
* 💡 **学习笔记**：`scanf`的格式字符串可以“占位”，处理空格超方便！

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：综合优质题解优化的完整实现，包含边界处理
* **完整核心代码**：
```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Person {
    string name;
    int year, month, day;
    int id; // 输入序号
};

bool cmp(const Person& a, const Person& b) {
    if (a.year != b.year) return a.year < b.year;
    if (a.month != b.month) return a.month < b.month;
    if (a.day != b.day) return a.day < b.day;
    return a.id > b.id; // 生日相同则序号大的优先
}

int main() {
    int n;
    cin >> n;
    Person p[105];
    for (int i = 0; i < n; i++) {
        cin >> p[i].name >> p[i].year >> p[i].month >> p[i].day;
        p[i].id = i; // 记录输入顺序
    }
    sort(p, p + n, cmp);
    for (int i = 0; i < n; i++) {
        cout << p[i].name << endl;
    }
    return 0;
}
```
* **代码解读概要**：定义Person结构体捆绑数据，cmp函数实现三级比较+逆序处理。主函数读入时记录id，排序后直接输出姓名。

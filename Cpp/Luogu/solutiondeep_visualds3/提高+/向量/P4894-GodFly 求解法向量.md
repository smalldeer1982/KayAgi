# GodFly 求解法向量

## 题目背景

“遇到函数无脑求导，遇到几何无脑建系。”是铁头娃的公共特征。

作为喜欢建系的一个铁头娃，GodFly 经常需要计算法向量。

## 题目描述

定义平面 $\alpha$ 内有两相交向量 $\vec{a}=(x_1,y_1,z_1), \vec{b}=(x_2,y_2,z_2)$，若$\vec{z}=(x,y,z)$ 为平面 $\alpha$ 的法向量，则：

$$
\begin{cases}
x \times x _ 1 + y \times y _ 1 + z \times z _ 1 = 0 \\ 
x \times x _ 2 + y \times y _ 2 + z \times z _ 2 = 0
\end{cases}
$$

现给出 $\vec{a}$ 及 $\vec{b}$ 的坐标，求出一个法向量 $\vec{z}$，并输出其坐标。

## 说明/提示

### 数据范围

所有坐标（包括答案）均在 $32$ 位整数范围内。

- 对于 $30\%$ 的数据，只有单组输入；
- 对于 $100\%$ 的数据，输入组数 $\le 5\times 10^4$，且输入的坐标均为绝对值不超过 $5 \times 10^4$ 的正整数。

### 提示

本题有 Special Judge，输出任意一组解即可。

## 样例 #1

### 输入

```
1 2 3 3 2 1```

### 输出

```
1 -2 1```

🗣️ **初步分析**：
> 求法向量就像在三维空间中找一根垂直于给定平面的"魔法棒"。最精妙的解法是用**向量叉积**——它像一把万能钥匙，能直接打开法向量的大门。核心思想是：两个向量的叉积结果天然垂直于它们构成的平面。
   - **题解思路对比**：主要分叉积法（直接套公式）和解方程法（设值求变量）。叉积法效率O(1)且无精度问题，明显优于需要讨论分母为零的解方程法。
   - **算法流程**：输入向量a和b → 计算叉积的三个分量 → 输出结果。关键变量更新：`x = y1*z2 - y2*z1`，`y = z1*x2 - z2*x1`，`z = x1*y2 - x2*y1`。

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：综合优质题解思路，采用最简洁高效的叉积实现
* **完整核心代码**：
```cpp
#include <iostream>
using namespace std;
int main() {
    long long x1, y1, z1, x2, y2, z2;
    while (cin >> x1 >> y1 >> z1 >> x2 >> y2 >> z2) {
        long long x = y1 * z2 - y2 * z1;
        long long y = z1 * x2 - z2 * x1;
        long long z = x1 * y2 - x2 * y1;
        cout << x << " " << y << " " << z << endl;
    }
    return 0;
}
```
* **代码解读概要**：
  > 1. 用`long long`确保大数计算安全  
  > 2. `while`循环处理多组输入  
  > 3. 三行核心计算对应叉积公式的三个分量  
  > 4. 直接输出避免中间变量

---

**题解一（SuperJvRuo）**
* **亮点**：极致简洁的工业级实现
* **核心代码片段**：
```cpp
printf("%lld %lld %lld\n",b*f-c*e,c*d-a*f,a*e-b*d);
```
* **代码解读**：
  > 变量`a,b,c`对应`x1,y1,z1`，`d,e,f`对应`x2,y2,z2`  
  > `b*f-c*e`即`y1*z2 - y2*z1`（叉积x分量）  
  > 为何用`printf`？效率高于`cout`，特别适合竞赛  
* 💡 **学习笔记**：变量命名可优化，但紧凑代码适合熟练者

**题解二（Starlight237）**
* **亮点**：数学严谨性与代码清晰度兼备
* **核心代码片段**：
```cpp
printf("%lld %lld %lld\n",Y1*Z2-Z1*Y2,Z1*X2-X1*Z2,X1*Y2-Y1*X2);
```
* **代码解读**：
  > 大写变量名明确对应数学符号  
  > 三个输出项严格对应叉积分量  
  > 为何不化简？保持与数学推导的一致性  
* 💡 **学习笔记**：保持代码与数学公式的同构性增强可读性

**题解三（walk_alone）**
* **亮点**：创新性公式记忆法
* **核心代码片段**（概念补充）：
```cpp
// 根据"掐头去尾"记忆法：
x = y1*z2 - y2*z1; // 取y,z分量交叉计算
y = z1*x2 - z2*x1; // 取z,x分量交叉计算
z = x1*y2 - x2*y1; // 取x,y分量交叉计算
```
* **代码解读**：
  > 将向量坐标重复排列：  
  > `[x1,y1,z1,x1,y1]`  
  > `[x2,y2,z2,x2,y2]`  
  > 跳过每行首元素后，交叉相乘即得公式  
* 💡 **学习笔记**：抽象公式可视化可降低记忆负担

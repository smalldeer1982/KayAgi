# 题目信息

# [SHOI2009] 会场预约

## 题目背景

**形式化题意：**

你需要维护一个在数轴上的线段的集合 $S$，支持两种操作：

`A l r` 表示将 $S$ 中所有与线段 $[l,r]$ 相交的线段删去，并将 $[l,r]$ 加入 $S$ 中。

`B` 查询 $S$ 中的元素数量。

对于 `A` 操作，每次还需输出删掉的元素个数。

## 题目描述

PP 大厦有一间空的礼堂，可以为企业或者单位提供会议场地。

这些会议中的大多数都需要连续几天的时间（个别的可能只需要一天），不过场地只有一个，所以不同的会议的时间申请不能够冲突。也就是说，前一个会议的结束日期必须在后一个会议的开始日期之前。所以，如果要接受一个新的场地预约申请，就必须拒绝掉与这个申请相冲突的预约。

一般来说，如果 PP 大厦方面事先已经接受了一个会场预约（例如从 $10$ 日到 $15$ 日），就不会再接受与之相冲突的预约（例如从 $12$ 日到 $17$ 日）。

不过，有时出于经济利益，PP 大厦方面有时会为了接受一个新的会场预约，而拒绝掉一个甚至几个之前预订的预约。 于是，礼堂管理员 QQ 的笔记本上经常记录着这样的信息：（本题中为方便起见，所有的日期都用一个整数表示）例如，如果一个为期 $10$ 天的会议从 $90$ 日开始到 $99$ 日，那么下一个会议最早只能在 $100$ 日开始。（此处前后矛盾，若无法理解请参考形式化描述。）

最近，这个业务的工作量与日俱增，礼堂的管理员 QQ 希望参加 SHTSC 的你替他设计一套计算机系统，方便他的工作。这个系统应当能执行下面两个操作：

`A` 操作：有一个新的预约是从 $start$ 日到 $end$ 日，并且拒绝掉所有与它相冲突的预约。执行这个操作的时候，你的系统应当返回为了这个新预约而拒绝掉的预约个数，以方便 QQ 与自己的记录相校对。

`B` 操作：请你的系统返回当前的仍然有效的预约的总数。

## 说明/提示

【数据范围】  
对于 $100\%$ 的数据，$1\le n \le 2\times 10^5$，$1\le l \le r \le 10^5$。

## 样例 #1

### 输入

```
6
A 10 15
A 17 19
A 12 17
A 90 99
A 11 12
B```

### 输出

```
0
0
2
0
1
2```

### 1. 题目解读与核心算法识别
✨ **本题主要考察**：`平衡树/线段树染色/STL应用`  
**核心思想**：维护互不相交的线段集合，通过高效查找相交区间实现动态管理。  
> **平衡树解法**：线段按左端点排序，插入时通过前驱后继定位相交区间，删除中间连续段  
> **线段树染色**：将线段视为颜色，查询区间覆盖状态并清除冲突线段  
> **STL set解法**：巧用自定义比较函数（相交视为相等）实现快速查找删除  

**可视化设计**：  
- 像素网格展示数轴（1-100000），不同颜色方块代表线段  
- 插入时高亮新区间（红色），冲突线段闪烁后消失（灰色）  
- 控制面板支持单步执行/调速，实时显示当前线段数量

### 2. 精选优质题解参考

#### 题解一：STL set（作者：Nartsam）
**亮点**：
- 利用set自动排序特性，自定义相交判定（`a.r < b.l`为左，`b.r < a.l`为右，否则相交）
- 代码简洁（<30行），逻辑清晰
- 实践价值高（竞赛常用技巧）

**核心代码**：
```cpp
struct Plan{ int l,r; 
    bool operator <(const Plan &rhs)const{
        return r < rhs.l; // 完全左侧才算小于
    } 
};
set<Plan> s;
void A(int l, int r){
    int cnt=0;
    Plan tmp = {l,r};
    auto it = s.find(tmp);
    while(it != s.end()){ 
        cnt++; 
        s.erase(it);
        it = s.find(tmp);
    }
    s.insert(tmp);
}
```

#### 题解二：线段树染色（作者：香风智乃）
**亮点**：
- 用操作编号作颜色值，巧妙处理覆盖关系
- 双标记优化（颜色标记+删除标记）
- 逻辑严密，扩展性强

**核心代码**：

### 3. 核心难点辨析与解题策略
1. **难点：相交区间高效定位**  
   **解法**：平衡树维护有序线段，利用前驱/后继确定边界；线段树染色时区间查询覆盖状态  
   💡 *学习笔记：有序结构加速范围查询*

2. **难点：删除冲突后的状态维护**  
   **解法**：平衡树删除连续子树区间；线段树重置颜色标记  
   💡 *学习笔记：批量删除需保证操作原子性*

3. **难点：动态维护不相交性质**  
   **解法**：插入时严格保证前驱右界<当前左界，后继左界>当前右界  
   💡 *学习笔记：区间问题注意边界互斥条件*

**解题技巧总结**：  
- 结构选择：小规模用STL set，大规模用平衡树/线段树  
- 优化关键：每个线段只被删除一次，保证总复杂度O(n log n)  
- 调试技巧：可视化模拟区间覆盖过程（见第5节）

### 4. C++核心代码实现赏析
#### STL set完整实现：
```cpp
#include <set>
struct Plan{ int l,r; 
    bool operator<(const Plan &rhs)const{ 
        return r < rhs.l; 
    }
};
int main(){
    set<Plan> s; char op; int n; cin>>n;
    while(n--){
        cin>>op;
        if(op=='A'){
            int l,r,cnt=0; cin>>l>>r;
            auto it = s.find({l,r});
            while(it != s.end()){
                cnt++; s.erase(it++);
            }
            s.insert({l,r});
            cout<<cnt<<endl;
        } else cout<<s.size()<<endl;
    }
}
```

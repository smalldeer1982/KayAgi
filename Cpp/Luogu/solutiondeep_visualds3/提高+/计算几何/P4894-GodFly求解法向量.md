## 题目描述

定义平面$\alpha$内有两相交向量$\overrightarrow a=(x_1,y_1,z_1)$，$\overrightarrow b=(x_2,y_2,z_2)$，若$\overrightarrow z=(x,y,z)$为平面$\alpha$的法向量，则：

![](https://cdn.luogu.com.cn/upload/pic/30949.png)

现给出$\overrightarrow a$及$\overrightarrow b$的坐标，求出一个法向量$\overrightarrow z$，并输出其坐标。

## 1. 题目解读与核心算法识别  
✨ **本题主要考察**：`数学（向量叉积）`  

🗣️ **初步分析**：  
> 解决法向量问题如同寻找垂直于两张纸片（向量a、b构成的平面）的木棍。**叉积运算**恰好生成这个垂直向量，其数学本质是行列式计算：  
> $$ \vec{z} = \vec{a} \times \vec{b} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ x_1 & y_1 & z_1 \\ x_2 & y_2 & z_2 \end{vmatrix} = (y_1z_2 - y_2z_1, z_1x_2 - z_2x_1, x_1y_2 - x_2y_1) $$  
> - **题解一致性**：所有优质解法均直接套用叉积公式，避免复杂解方程  
> - **可视化设计**：动画将用**8位像素箭头**动态展示向量（蓝/绿为输入向量，红为叉积结果）。关键步骤：  
>   - 高亮向量分量计算过程（如$y_1z_2$），伴随“叮”音效  
>   - 结果向量旋转显示垂直性，胜利音效强化记忆

   * 💡 学习笔记：叉积 = 自动垂直的“几何魔法棒”

## 4. C++核心代码实现赏析  
**通用核心实现参考**  
* **说明**：综合所有优质题解，叉积公式为最优解  
* **完整核心代码**：  
  ```cpp
  #include <cstdio>
  typedef long long LL; // 防溢出
  int main() {
      LL x1, y1, z1, x2, y2, z2;
      while (scanf("%lld%lld%lld%lld%lld%lld", &x1, &y1, &z1, &x2, &y2, &z2) != EOF) 
          printf("%lld %lld %lld\n", 
              y1 * z2 - y2 * z1, 
              z1 * x2 - z2 * x1, 
              x1 * y2 - x2 * y1);
      return 0;
  }
  ```
* **代码解读概要**：  
  > 1. 读入两向量6个坐标  
  > 2. 直接计算叉积三分量（注意中间项符号）  
  > 3. `while+scanf`确保处理$5×10^4$组数据  

**题解片段赏析**  
1. **SuperJvRuo**  
   * 亮点：极致简洁  
   * 核心代码：  
     ```cpp
     printf("%lld %lld %lld\n", b*f-c*e, c*d-a*f, a*e-b*d);
     ```
   * 解读：变量`a,b,c`代$\vec{a}$，`d,e,f`代$\vec{b}$，叉积按公式顺序输出  

2. **walk_alone**  
   * 亮点：几何意义注释  
   * 核心代码（逻辑等价）：  
     ```cpp
     x = y1*z2 - y2*z1; // 去x列后的交叉相乘
     y = z1*x2 - z2*x1; // 去y列
     z = x1*y2 - x2*y1; // 去z列
     ```
   * 解读：掐头去尾法实为行列式展开，建议配合右手定则动画理解  

3. **Starlight237**  
   * 亮点：强调数学严谨性  
   * 核心代码：  
     ```cpp
     printf("%lld %lld %lld\n", 
         Y1*Z2 - Z1*Y2, 
         Z1*X2 - X1*Z2, 
         X1*Y2 - Y1*X2);
     ```
   * 解读：变量大写增强矩阵代入感，中间项顺序与数学定义严格一致

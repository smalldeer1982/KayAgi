# [NOIP 2012 提高组] Vigenère 密码

## 题目描述

16 世纪法国外交家 Blaise de Vigenère 设计了一种多表密码加密算法 Vigenère 密码。Vigenère 密码的加密解密算法简单易用，且破译难度比较高，曾在美国南北战争中为南军所广泛使用。

 
在密码学中，我们称需要加密的信息为明文，用 $M$ 表示；称加密后的信息为密文，用 $C$ 表示；而密钥是一种参数，是将明文转换为密文或将密文转换为明文的算法中输入的数据，记为 $k$。在 Vigenère 密码中，密钥 $k$ 是一个字母串，$k=k_1,k_2,…,k_n$。当明文 $M=m_1,m_2,…,m_n$ 时，得到的密文 $C=c_1,c_2,…,c_n$，其中 $c_i$=$m_i \operatorname{\circledR} k_i$，运算 $\circledR$ 的规则如下表所示：

Vigenère 加密在操作时需要注意：

1. $\circledR$ 运算忽略参与运算的字母的大小写，并保持字母在明文 $M$ 中的大小写形式；
2. 当明文 $M$ 的长度大于密钥 $k$ 的长度时，将密钥 $k$ 重复使用。

例如，明文 $M=\texttt{Helloworld}$，密钥 $k=\texttt{abc}$ 时，密文 $C=\texttt{Hfnlpyosnd}$。

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：`字符串处理与模拟应用`

🗣️ **初步分析**：
> 解决Vigenère密码的关键在于理解其解密规则就像操作一个**字母转盘**：每个密文字符需根据密钥字符进行反向旋转（偏移）来还原明文。在本题中，这种模拟应用主要体现在：
   - 通过观察密码表规律，发现解密公式为：`明文 = (密文字符 - 密钥字符 + 26) % 26`
   - 核心难点在于同时处理大小写敏感性和密钥循环机制
   - 可视化设计将采用**8位像素风密码转盘**：用旋转动画展示字符偏移过程，高亮当前处理的密文/密钥字符，当转盘旋转时播放"咔嗒"音效，解密成功时播放8-bit胜利音效，并设计"自动破译"模式模拟解密流程

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：综合优质题解优点，兼顾效率和可读性
```cpp
#include <iostream>
#include <cctype>
using namespace std;

int main() {
    string k, c;
    cin >> k >> c;
    for (int i = 0; i < c.length(); i++) {
        char key = tolower(k[i % k.length()]);  // 密钥统一小写
        char enc = c[i];                         // 保留密文原始大小写
        
        // 核心解密公式
        int offset = (enc & 31) - (key & 31);    // 获取字母序差值
        char dec = (offset + 26) % 26;            // 防负偏移
        
        // 恢复大小写
        dec += isupper(enc) ? 'A' : 'a';         
        cout << dec;
    }
    return 0;
}
```
* **代码解读概要**：
  1. 统一密钥为小写简化计算
  2. `(enc & 31)`获取密文字母序（A=1,...,Z=26）
  3. `(offset+26)%26`处理负偏移
  4. 根据密文原始大小写恢复明文

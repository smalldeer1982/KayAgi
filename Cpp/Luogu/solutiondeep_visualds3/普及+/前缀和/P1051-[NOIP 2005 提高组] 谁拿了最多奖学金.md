# 题目信息

# [NOIP 2005 提高组] 谁拿了最多奖学金

## 题目描述

某校的惯例是在每学期的期末考试之后发放奖学金。发放的奖学金共有五种，获取的条件各自不同：

1. 院士奖学金，每人 $8000$ 元，期末平均成绩高于 $80$ 分（$>80$），并且在本学期内发表$1$篇或$1$篇以上论文的学生均可获得；
2. 五四奖学金，每人 $4000$ 元，期末平均成绩高于 $85$ 分（$>85$），并且班级评议成绩高于 $80$ 分（$>80$）的学生均可获得；
3. 成绩优秀奖，每人 $2000$ 元，期末平均成绩高于 $90$ 分（$>90$）的学生均可获得；
4. 西部奖学金，每人 $1000$ 元，期末平均成绩高于 $85$ 分（$>85$）的西部省份学生均可获得；
5. 班级贡献奖，每人 $850$ 元，班级评议成绩高于 $80$ 分（$>80$）的学生干部均可获得；

只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金。例如姚林的期末平均成绩是 $87$ 分，班级评议成绩 $82$ 分，同时他还是一位学生干部，那么他可以同时获得五四奖学金和班级贡献奖，奖金总数是 $4850$ 元。

现在给出若干学生的相关数据，请计算哪些同学获得的奖金总数最高（假设总有同学能满足获得奖学金的条件）。

## 3. 核心难点辨析与解题策略

1.  **关键点1：条件独立性处理**
    * **分析**：五个奖学金条件非互斥（如某生可同时获多项），必须用独立if判断。优质题解都避免使用else if，确保每个条件单独检测

2.  **关键点2：同分处理机制**

* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <string>
    using namespace std;
    
    int main() {
        int n, total = 0, maxMoney = 0;
        string name, maxName;
        cin >> n;
        
        for (int i = 0; i < n; i++) {
            string curName;
            int score, classScore, papers;
            char leader, west;
            cin >> curName >> score >> classScore >> leader >> west >> papers;
            
            int money = (score > 80 && papers >= 1) * 8000 +
                        (score > 85 && classScore > 80) * 4000 +
                        (score > 90) * 2000 +
                        (score > 85 && west == 'Y') * 1000 +
                        (classScore > 80 && leader == 'Y') * 850;
                        
            total += money;
            if (money > maxMoney) {
                maxMoney = money;
                maxName = curName;
            }
        }
        cout << maxName << "\n" << maxMoney << "\n" << total;
        return 0;
    }
    ```
* **代码解读概要**：
    > 1. 读入学生数量n
    > 2. 循环处理每个学生：读入姓名、成绩等数据
    > 3. 关键：用逻辑表达式计算奖学金（true转1，false转0）
    > 4. 即时更新最高奖学金记录
    > 5. 输出结果（姓名、最高额、总额）

**题解一核心逻辑**
* **亮点**：布尔运算直接参与算术
* **核心代码片段**：
    ```cpp
    sum = (a>80&&e)*8000+  // 院士奖学金
          (a>85&&b>80)*4000+ // 五四奖学金
          (a>90)*2000+      // 成绩优秀奖
          (a>85&&d=='Y')*1000+ // 西部奖学金
          (b>80&&c=='Y')*850;  // 班级贡献奖
    ```
* **代码解读**：
    > 每个括号内都是布尔表达式，计算结果为0（false）或1（true）。例如：当`a>80`且`e>=1`时，`(a>80&&e)`值为1，乘以8000即得该项奖学金。这种写法将5个if语句压缩为1行数学表达式
* 💡 **学习笔记**：C++中布尔值可隐式转换为整数（true→1, false→0），善用此特性可简化代码

**题解二边界处理**
* **亮点**：严格遵循题目的大于条件
* **核心代码片段**：
    ```cpp
    if(stu[i].score > 80 && stu[i].papers >= 1) // 注意>80不是≥80
        stu[i].money += 8000;
    ```
* **代码解读**：
    > 所有判断严格使用`>`而非`>=`，如`score>80`（不是≥81）。这种精确的条件匹配是模拟题得分关键，需特别注意题目中的**高于**表述
* 💡 **学习笔记**：边界条件决定程序正确性，务必逐字阅读题目

**题解三即时更新**
* **亮点**：无存储的流式处理
* **核心代码片段**：
    ```cpp
    if (money > maxMoney) { // 发现更高立即更新
        maxMoney = money;
        maxName = curName; // 保留当前姓名
    }
    ```
* **代码解读**：
    > 不保存全部学生数据，在读入每个学生时即时计算并更新最大值。`money > maxMoney`严格使用大于号确保同分时保留最早记录（因后出现的同分不会触发更新）
* 💡 **学习笔记**：打擂台法更新时，相等时不处理才能保留第一个最大值

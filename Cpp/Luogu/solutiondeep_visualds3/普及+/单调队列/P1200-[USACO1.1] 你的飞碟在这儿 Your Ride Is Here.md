# [USACO1.1] 你的飞碟在这儿 Your Ride Is Here

## 题目描述

众所周知，在每一个彗星后都有一只 UFO。这些 UFO 时常来收集地球上的忠诚支持者。不幸的是，他们的飞碟每次出行都只能带上一组支持者。因此，他们要用一种聪明的方案让这些小组提前知道谁会被彗星带走。他们为每个彗星起了一个名字，通过这些名字来决定这个小组是不是被带走的那个特定的小组（你认为是谁给这些彗星取的名字呢？）。关于如何搭配的细节会在下面告诉你；你的任务是写一个程序，通过小组名和彗星名来决定这个小组是否能被那颗彗星后面的 UFO 带走。


小组名和彗星名都以下列方式转换成一个数字：最终的数字就是名字中所有字母的积，其中 $\texttt A$ 是 $1$，$\texttt Z$ 是 $26$。例如，$\texttt{USACO}$ 小组就是 $21 \times 19 \times 1 \times 3 \times 15=17955$。如果小组的数字 $\bmod 47$ 等于彗星的数字 $\bmod 47$,你就得告诉这个小组需要准备好被带走！（记住“$a \bmod b$”是 $a$ 除以 $b$ 的余数，例如 $34 \bmod 10$ 等于 $4$）


写出一个程序，读入彗星名和小组名并算出用上面的方案能否将两个名字搭配起来，如果能搭配，就输出 `GO`，否则输出 `STAY`。小组名和彗星名均是没有空格或标点的一串大写字母（不超过 $6$ 个字母）。

## 3. 核心难点辨析与解题策略

### 1. **关键点1：字符到数字的正确转换**  
* **分析**：题目要求A=1，Z=26，所以字符`c`对应的数字是`c - 'A' + 1`（比如'A'-'A'+1=1，'B'-'A'+1=2）。很多新手会忘记加1，导致结果错误（比如A变成0）。  
* 💡 **学习笔记**：字符转数字的公式要记牢——`c - 'A' + 1`（大写字母）。  

### 2. **关键点2：乘积的初始值设置**  
* **分析**：乘积的初始值必须是1（因为1乘任何数都不变）。如果设为0，不管后面乘什么，结果都是0，直接导致错误。  
* 💡 **学习笔记**：累乘问题的初始值用1，累加用0，这是常识！  

### 3. **关键点3：输入方式的选择**  
* **分析**：如果用`cin`读取`string`，会自动跳过空格和换行，适合本题（输入没有空格）；如果用`gets`或`getline`，需要注意处理换行符（比如`gets`会读取换行符作为字符串的结束）。  
* 💡 **学习笔记**：对于简单的字符串输入，用`cin >> string`是最方便的。  

### ✨ 解题技巧总结  
- **技巧A：变量名要“见名知意”**：比如用`comet_num`表示彗星名的结果，`team_num`表示小组名的结果，比`a`、`b`更易读。  
- **技巧B：及时注释**：把容易犯的错误（比如初始值）写成注释，避免下次再错。  
- **技巧C：模块化代码**：把重复的逻辑（比如字符串转数字）封装成函数，让代码更清晰。  


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考  
* **说明**：综合了Vanyun和charliejiang的思路，用`string`处理输入，封装了转换函数，逻辑清晰。  
* **完整核心代码**：  
  ```cpp
  #include <iostream>
  #include <string>
  using namespace std;

  int convert(string s) {
      int res = 1; // 初始值为1
      for (char c : s) { // 遍历字符串中的每个字符
          res *= (c - 'A' + 1); // 字符转数字：A→1，B→2，…，Z→26
      }
      return res % 47; // 取模47
  }

  int main() {
      string comet, team;
      cin >> comet >> team; // 读取输入
      if (convert(comet) == convert(team)) {
          cout << "GO" << endl;
      } else {
          cout << "STAY" << endl;
      }
      return 0;
  }
  ```
* **代码解读概要**：  
  1. `convert`函数：接收一个字符串，计算它的乘积并取模47。  
  2. 主函数：读取彗星名和小组名，调用`convert`函数得到两个结果，比较后输出。  


### 针对各优质题解的片段赏析  

#### 题解一（作者：Vanyun）  
* **亮点**：注释详细，直接提醒初始值的重要性。  
* **核心代码片段**：  
  ```cpp
  int ans = 1, num = 1; // 起始值必须为1，要是0的话所有的数乘出来都是0了....
  for (int i = 0; i < a.length(); i++) ans *= a[i] - '@'; // @的ASCII是64，A是65，所以a[i]-@=a[i]-64=1~26
  ```
* **代码解读**：  
  用`a[i] - '@'`代替`a[i] - 'A' + 1`，其实是一样的（因为`@`的ASCII是64，`A`是65，所以`a[i]-@=a[i]-64= (a[i]-'A')+1`）。注释明确告诉我们初始值不能为0，这是新手最容易犯的错误！  
* 💡 **学习笔记**：注释是代码的“说明书”，要把关键细节写清楚。  

#### 题解二（作者：DrinkOnstage）  
* **亮点**：用`cin.get()`逐字符读取，处理换行符。  
* **核心代码片段**：  
  ```cpp
  while ((v = cin.get()) != '\n') a *= v - 'A' + 1;
  while ((v = cin.get()) != '\n') b *= v - 'A' + 1;
  ```
* **代码解读**：  
  `cin.get()`会读取每个字符，包括换行符。当读取到换行符时，循环停止。这种方法适合处理“按行输入”的情况，比如输入中有换行符的情况。  
* 💡 **学习笔记**：输入方式要根据题目要求选择，灵活运用`cin`的各种函数。  

#### 题解三（作者：charliejiang_0w0_）  
* **亮点**：封装函数，模块化代码。  
* **核心代码片段**：  
  ```cpp
  int convertToInt(string name) {
      int sum = 1;
      for (int i = 0; i < name.length(); i++)
          sum *= charToInt1200(name[i]);
      return sum % 47;
  }
  ```
* **代码解读**：  
  `convertToInt`函数负责把字符串转换成取模后的结果，`charToInt1200`函数负责把字符转换成数字。这种写法让主函数更简洁，也方便后续修改（比如如果要改变转换方式，只需要修改`charToInt1200`函数）。  
* 💡 **学习笔记**：模块化代码能提高代码的可读性和复用性，值得养成习惯。

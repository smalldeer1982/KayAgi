# 题目信息

# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

### 题目解读与核心算法识别  
✨ **本题主要考察**：`排序（多关键字排序）` 与 `模拟应用`  

🗣️ **初步分析**：  
> 本题要求对一组学生信息按年龄从大到小排序（即出生日期从小到大），本质是**多关键字排序**问题。我们可以将其比喻为整理一组“时间胶囊”——每个胶囊（学生）上有年、月、日三个时间标签，需要按标签优先级（年→月→日）升序排列。若标签完全相同，则后输入的胶囊优先输出（逆序输入顺序）。  
> - **核心思路**：所有题解均采用**结构体存储数据+自定义比较规则**。关键难点在于正确处理多级比较逻辑和相同日期的逆序输出要求。  
> - **可视化设计**：在像素动画中，用不同颜色方块表示学生，方块内显示名字和日期。排序时高亮当前比较的两个方块，动态展示比较过程（如先闪烁年份标签）。交换位置时加入8-bit音效，最终按排序结果生成“时间线队列”。

### 核心难点辨析与解题策略  
1. **多关键字比较逻辑**  
   * **分析**：需依次比较年、月、日三个关键字。优质题解均采用分级判断：先比较年→年相同则比月→月相同则比日。若跳过任一级，会导致排序错误。  
   * 💡 **学习笔记**：多关键字排序中，优先级高的关键字必须先比较。  

2. **相同生日的逆序输出**  
   * **分析**：题目要求生日相同时后输入者先输出。解决方法是记录输入顺序编号，在日期相同时比较编号并**逆序排列**（编号大者优先）。  
   * 💡 **学习笔记**：输入顺序是隐含条件，需额外变量存储。  

3. **时间合并法的陷阱**  
   * **分析**：部分题解尝试合并年月日为整数（如`19920423`），但未处理逆序要求。且若月/日为个位数（如5月→`05`），需补零避免逻辑错误。  
   * 💡 **学习笔记**：合并法虽简洁，需注意数值格式和逆序需求。  

### ✨ 解题技巧总结  
- **结构体捆绑数据**：将学生信息（姓名、年月日、序号）绑定为结构体，避免数据分散。  
- **比较函数分层设计**：使用`if-else`链或逻辑运算符（`||`）明确优先级，逐级比较关键字。  
- **边界测试**：特别测试年月日相同、单数月/日等边界情况。

### C++核心代码实现赏析  
**本题通用核心实现**  
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Student {
    string name;
    int year, month, day, id; // id记录输入顺序
};

bool cmp(Student a, Student b) {
    if (a.year != b.year) return a.year < b.year;
    if (a.month != b.month) return a.month < b.month;
    if (a.day != b.day) return a.day < b.day;
    return a.id > b.id; // 生日相同时，按id逆序
}

int main() {
    int n;
    cin >> n;
    Student s[105];
    for (int i = 0; i < n; i++) {
        cin >> s[i].name >> s[i].year >> s[i].month >> s[i].day;
        s[i].id = i; // 记录输入顺序
    }
    sort(s, s + n, cmp);
    for (int i = 0; i < n; i++) 
        cout << s[i].name << endl;
}
```
**代码解读概要**：  
1. 结构体`Student`整合学生信息，`id`记录输入顺序。  
2. `cmp`函数逐级比较年、月、日，最后用`id`逆序处理相同生日。  
3. `sort`调用自定义比较规则，完成多关键字排序。  

**题解片段赏析**  
1. **题解一核心片段**  
```cpp
bool cmp(node a, node b) {
    if (a.n < b.n) return 1; // 比年
    if (a.n > b.n) return 0;
    if (a.y < b.y) return 1; // 比月
    ... // 逐级比较
}
```
* **亮点**：逻辑直白，易理解。  
* **学习笔记**：通过返回`1`/`0`明确交换条件，适合初学者。  

2. **题解三核心片段**  
```cpp
bool operator<(const Student& a, const Student& b) {
    if (a.year != b.year) return a.year < b.year;
    if (a.month != b.month) return a.month < b.month;
    return a.day == b.day ? a.id > b.id : a.day < b.day;
}
```
* **亮点**：运算符重载使代码更简洁。  
* **学习笔记**：三目运算符精简相同生日的处理逻辑。

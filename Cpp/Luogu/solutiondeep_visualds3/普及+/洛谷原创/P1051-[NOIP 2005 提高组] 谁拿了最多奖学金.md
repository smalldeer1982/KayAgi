# 题目信息

# [NOIP 2005 提高组] 谁拿了最多奖学金

## 题目描述

某校的惯例是在每学期的期末考试之后发放奖学金。发放的奖学金共有五种，获取的条件各自不同：

1. 院士奖学金，每人 $8000$ 元，期末平均成绩高于 $80$ 分（$>80$），并且在本学期内发表$1$篇或$1$篇以上论文的学生均可获得；
2. 五四奖学金，每人 $4000$ 元，期末平均成绩高于 $85$ 分（$>85$），并且班级评议成绩高于 $80$ 分（$>80$）的学生均可获得；
3. 成绩优秀奖，每人 $2000$ 元，期末平均成绩高于 $90$ 分（$>90$）的学生均可获得；
4. 西部奖学金，每人 $1000$ 元，期末平均成绩高于 $85$ 分（$>85$）的西部省份学生均可获得；
5. 班级贡献奖，每人 $850$ 元，班级评议成绩高于 $80$ 分（$>80$）的学生干部均可获得；

只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金。例如姚林的期末平均成绩是 $87$ 分，班级评议成绩 $82$ 分，同时他还是一位学生干部，那么他可以同时获得五四奖学金和班级贡献奖，奖金总数是 $4850$ 元。

现在给出若干学生的相关数据，请计算哪些同学获得的奖金总数最高（假设总有同学能满足获得奖学金的条件）。


## 说明/提示

对于 $100\%$ 的数据，满足 $1 \le  N \le  100$。

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：模拟应用  

🗣️ **初步分析**：  
解决这道题的核心是**模拟**——就像老师拿着“奖学金规则表”，逐个检查每个学生的条件，算出每个人的奖金，再找出“奖金最多的同学”。简单来说，模拟就是“按规则一步步做”，没有复杂的算法，但需要**细心**和**准确**。  

在本题中，模拟的流程是：  
1. 读入每个学生的信息（姓名、成绩、是否干部、是否西部学生、论文数量）；  
2. 按照5条奖学金规则，计算每个学生的总奖金；  
3. 找到“奖金最多”的学生（如果并列，选输入最早的）；  
4. 计算所有学生的奖金总和。  

**核心难点**：  
- 条件判断的准确性（比如“>80”还是“≥80”？论文需要“≥1篇”）；  
- 并列第一时，要选“输入最早”的学生；  
- 输入数据的正确读取（比如字符型数据不能带空格）。  

**解决方案**：  
- 逐条核对题目条件，用`if`而不是`else if`（因为奖金可以叠加）；  
- 用“打擂台法”找最大值：遍历学生时，只要当前学生的奖金**严格大于**之前的最大值，就更新最大值（这样并列时保留最早的）；  
- 用`cin`读入字符时，注意不要夹杂空格（比如`cin >> g`直接读入‘Y’或‘N’）。  

**可视化设计思路**：  
我们会做一个**像素风的“奖学金评选模拟器”**——像FC游戏一样，屏幕上是一排像素化的“学生卡片”（用不同颜色区分姓名、成绩、干部标识）。计算奖金时，每个满足的条件会弹出“奖金图标”（比如院士奖学金是金色奖杯），并伴随“叮”的音效；找最大值时，最高奖金的学生会“发光”，总奖金会用像素数字滚动显示。还可以加“自动播放”功能，像“AI老师”一样一步步完成评选~

## 3. 核心难点辨析与解题策略

### 关键点1：条件判断的准确性  
**问题**：容易把“>80”写成“≥80”，或者把“论文≥1”写成“论文>1”。  
**解决**：逐条核对题目条件，比如：  
- 院士奖学金：期末成绩**>80**分 + 论文**≥1**篇；  
- 五四奖学金：期末成绩**>85**分 + 班级成绩**>80**分。  
**学习笔记**：条件判断前，一定要再看一遍题目！


### 关键点2：并列第一的处理  
**问题**：如果两个学生奖金相同，要选“输入最早”的那个。  
**解决**：用“打擂台法”时，只有当前学生的奖金**严格大于**之前的最大值，才更新最大值（比如`if (sum>mx)`而不是`if (sum>=mx)`）。这样，并列时保留最早的学生。  
**学习笔记**：打擂台的条件决定了“并列时的优先级”。


### 关键点3：输入数据的正确读取  
**问题**：读入字符型数据（比如是否干部）时，容易因为空格或换行符读错。  
**解决**：用`cin`读入字符（比如`cin >> g`），因为`cin`会自动跳过空格和换行符。如果用`scanf`，要注意`%c`前加空格（比如`scanf(" %c", &g)`）。  
**学习笔记**：字符型数据的输入，优先用`cin`！


### ✨ 解题技巧总结  
1. **奖金叠加用`if`**：每个奖学金条件都是独立的，必须用`if`而不是`else if`；  
2. **打擂台找最大值**：遍历一次学生就能找到最大值，不用排序；  
3. **变量名要易懂**：用`score`代替`a`，用`ganBu`代替`c`，代码可读性更高。


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考  
* **说明**：本代码综合了“简洁性”和“可读性”，用结构体存储学生信息，打擂台找最大值，适合初学者理解。  
* **完整核心代码**：  
```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    string name;       // 姓名
    int score;         // 期末成绩
    int class_score;   // 班级评议成绩
    char is_leader;    // 是否学生干部（Y/N）
    char is_west;      // 是否西部学生（Y/N）
    int paper;         // 论文数量
    int money;         // 总奖金
};

int main() {
    int n;
    cin >> n;
    Student stu[105];  // 最多100个学生
    int max_money = -1;
    int max_idx = 0;
    int total = 0;

    for (int i = 0; i < n; i++) {
        cin >> stu[i].name >> stu[i].score >> stu[i].class_score;
        cin >> stu[i].is_leader >> stu[i].is_west >> stu[i].paper;

        // 计算奖金
        stu[i].money = 0;
        if (stu[i].score > 80 && stu[i].paper >= 1) stu[i].money += 8000;
        if (stu[i].score > 85 && stu[i].class_score > 80) stu[i].money += 4000;
        if (stu[i].score > 90) stu[i].money += 2000;
        if (stu[i].score > 85 && stu[i].is_west == 'Y') stu[i].money += 1000;
        if (stu[i].class_score > 80 && stu[i].is_leader == 'Y') stu[i].money += 850;

        // 更新最大值
        if (stu[i].money > max_money) {
            max_money = stu[i].money;
            max_idx = i;
        }
        total += stu[i].money;
    }

    cout << stu[max_idx].name << endl;
    cout << max_money << endl;
    cout << total << endl;

    return 0;
}
```
* **代码解读概要**：  
  1. 定义`Student`结构体，存储每个学生的信息；  
  2. 读入学生数量`n`，遍历每个学生，读入信息；  
  3. 按照5条规则计算每个学生的奖金；  
  4. 用“打擂台法”找最大值，同时累加总奖金；  
  5. 输出最高奖金的学生姓名、奖金和总奖金。

* **亮点**：用逻辑表达式×奖金，代码极简。  
* **核心代码片段**：

* **学习笔记**：逻辑表达式可以简化“条件判断+累加”的代码。

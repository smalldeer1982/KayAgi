# [USACO25FEB] Printing Sequences B

## 题目描述

Bessie 正在学习使用一种简单的编程语言进行编程。她首先定义一个合法的程序，然后执行该程序以产生一些输出序列。

### 定义：

一个程序是一个非空的语句序列。

一个语句的形式或者是 "PRINT $c$"，其中 $c$ 是一个整数，或者是 "REP $o$"，随后是一个程序，随后是 "END"，其中 $o$ 是一个不小于 1 的整数。

### 执行：

执行一个程序将依次执行其语句。

执行语句 "PRINT $c$" 将使 $c$ 追加到输出序列中。

执行以 "REP $o$" 开始的语句将依次执行内部程序共 $o$ 次。

Bessie 知道如何编写的一个程序示例如下。

```plain
REP 3
    PRINT 1
    REP 2
        PRINT 2
    END
END
```

该程序输出序列 $[1,2,2,1,2,2,1,2,2]$。

Bessie 想要输出一个包含 $N$（$1 \le N \le 100$）个正整数的序列。Elsie 挑战她使用不超过 $K$（$1 \le K \le 3$）个 "PRINT" 语句。注意，Bessie 可以使用任意数量的 "REP" 语句。同时注意，序列中的每个正整数都不超过 $K$。

对于 $T$（$1 \le T \le 100$）个独立的测试用例中的每一个，求 Bessie 是否可以编写一个程序，使用至多 $K$ 个 "PRINT" 语句输出给定的序列。

## 说明/提示

对于第二个测试用例，以下代码使用了 $1$ 个 "PRINT" 语句输出了序列 $[1,1,1,1]$。

- 测试点 $8\sim 13$：没有额外限制。

**本题通用核心C++实现参考**
* **说明**：综合IcyDragon和AuZeb的解法，采用区间DP框架+KMP循环节优化
* **完整核心代码**：
```cpp
#include <iostream>
#include <cstring>
using namespace std;
const int N = 105;
int T, n, k, a[N], dp[N][N], nxt[N];

// KMP求最小循环节长度
int minCycle(int l, int r) {
    int len = r - l + 1;
    nxt[1] = 0;
    for (int i = 2, j = 0; i <= len; i++) {
        while (j && a[l + j] != a[l + i - 1]) j = nxt[j];
        if (a[l + j] == a[l + i - 1]) j++;
        nxt[i] = j;
    }
    return len % (len - nxt[len]) ? -1 : len - nxt[len];
}

int main() {
    cin >> T;
    while (T--) {
        cin >> n >> k;
        for (int i = 1; i <= n; i++) cin >> a[i];
        memset(dp, 0x3f, sizeof(dp));
        for (int i = 1; i <= n; i++) dp[i][i] = 1;

        for (int len = 2; len <= n; len++) {
            for (int i = 1; i + len - 1 <= n; i++) {
                int j = i + len - 1;
                // 策略1：分割区间
                for (int m = i; m < j; m++) 
                    dp[i][j] = min(dp[i][j], dp[i][m] + dp[m + 1][j]);
                
                // 策略2：循环节优化
                int L = minCycle(i, j);
                if (L != -1) 
                    dp[i][j] = min(dp[i][j], dp[i][i + L - 1]);
            }
        }
        cout << (dp[1][n] <= k ? "YES" : "NO") << endl;
    }
    return 0;
}
```
* **代码解读概要**：
  > 1. minCycle函数用KMP求最小循环节长度
  > 2. 初始化dp数组：单元素区间PRINT次数为1
  > 3. 双层循环遍历所有区间
  > 4. 内部分割区间策略：枚举所有分割点
  > 5. 循环节策略：如果存在循环节则状态转移
  > 6. 最后判断整个序列的最小PRINT次数是否≤k

**题解一（qhr2023）**
* **亮点**：巧妙处理数字块，避免复杂DP
* **核心代码片段**：
```cpp
bool sol2(int l, int r) {
    vector<pair<int, int>> blocks;  // 存储<数字, 连续个数>
    for (int i = l; i <= r; i++) {
        if (!blocks.empty() && a[i] == blocks.back().first) 
            blocks.back().second++;
        else 
            blocks.push_back({a[i], 1});
    }
    if (blocks.size() <= 2) return true;  // 1-2个块直接合法
    
    if (blocks.size() % 2 == 0) {  // 偶数个块需交替重复
        for (int i = 0; i < blocks.size() - 2; i++) 
            if (blocks[i] != blocks[i + 2]) 
                return false;
        return true;
    }
    return false;
}
```
* **代码解读**：
  > 1. 将序列压缩为连续数字块，如[1,1,2,2,1,1]→{(1,2),(2,2),(1,2)}
  > 2. 块数≤2时只需1-2个PRINT语句（每个块对应一个PRINT）
  > 3. 块数为偶数时，检查是否满足[A,B,A,B...]的交替模式
  > 4. 通过比较blocks[i]和blocks[i+2]确保奇偶位置各自相同
* 💡 **学习笔记**：序列压缩是处理重复模式的利器

**题解二（IcyDragon）**
* **亮点**：区间DP的经典实现
* **核心代码片段**：
```cpp
for (int len = 2; len <= n; len++) {        // 枚举区间长度
    for (int i = 1; i + len - 1 <= n; i++) { // 枚举起点
        int j = i + len - 1;
        // 分割点转移
        for (int k = i; k < j; k++) 
            dp[i][j] = min(dp[i][j], dp[i][k] + dp[k+1][j]);
        // 循环节转移
        for (int L = 1; L < len; L++) {     // 枚举循环节长度
            if (len % L != 0) continue;
            bool valid = true;
            for (int p = i + L; p <= j; p++) 
                if (a[p] != a[p - L]) { valid = false; break; }
            if (valid) 
                dp[i][j] = min(dp[i][j], dp[i][i+L-1]);
        }
    }
}
```
* **代码解读**：
  > 1. 外层循环控制区间大小（从小至大）
  > 2. 中层循环控制区间起始位置
  > 3. 内层第一循环：尝试所有分割点，类似石子合并
  > 4. 内层第二循环：检查每个可能的循环节长度L
  > 5. 通过比较a[p]和a[p-L]验证循环节
* 💡 **学习笔记**：区间DP常需要三重循环，顺序至关重要

**题解三（AuZeb）**

* **核心代码片段**：

```cpp
int getMinLen(int l, int r) {
    int n = r - l + 1, j = 0;
    nxt[1] = 0;
    // 构建next数组
    for (int i = 2; i <= n; i++) {
        while (j && a[l + j] != a[l + i - 1]) j = nxt[j];
        if (a[l + j] == a[l + i - 1]) j++;
        nxt[i] = j;
    }

}
```
* **代码解读**：

  > 2. 根据next[n]值计算最小循环节长度
  > 3. 若整个序列长度不能被循环节整除则返回-1
  > 4. 公式：最小循环节 = 序列长度 - next[n]
* 💡 **学习笔记**：KMP的next数组是处理循环结构的利器

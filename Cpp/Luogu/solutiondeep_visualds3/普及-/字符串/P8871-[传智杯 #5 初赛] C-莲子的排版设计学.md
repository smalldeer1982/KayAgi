莲子正在赶自己的程序设计作业。除了完成程序代码的编写，对提交上去的作业进行排版以对助教留下良好印象同样重要。

## 题目描述

莲子希望实现这样一个功能：输入一份文本文件，并给该文件加上行号。

以下是本题中三个基本概念的定义：

- **文本字符**由 $\textsf{ASCII}$ 中所有的可视字符，以及空格（$\textsf{ASCII}=32$）组成。
- **一行字符**由若干个（可以为 $0$ 个）文本字符，以及在末尾**恰好一个**换行符（$\textsf{ASCII}=10$）组成。
- **文本文件**由若干个（至少为 $1$ 个）一行字符组成。文本文件的行数就是组成它的行的数量。

以下是本题中添加行号的方法：

- 设该文本文件一共有 $m$ 行。设正整数 $m$ 的字宽为 $s$。那么，在每一行的开头会有 $s+1$ 的长度用来显示行号。
- 对于第 $i$ 行，假设 $i$ 的字宽为 $t$，那么这一行行号将会显示为 $\underbrace{\texttt{␣␣...␣}}_{s-t\text{ 个}}\ i\ \texttt{␣}$，其中 $\texttt{␣}$ 表示空格。

## 4. C++核心代码实现赏析

<code_intro_overall>
我们综合了两个优质题解的思路，给出一个清晰且高效的核心实现。这个代码用`vector`存储每行内容，循环计算字宽，适合新手理解。

**本题通用核心C++实现参考**
* **说明**：本代码综合了chen_zhe题解的清晰结构和离散小波变换°题解的高效思路，用`vector<string>`存储每行（比`vector<char>`更易操作），并优化了字宽计算逻辑。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <vector>
    #include <cmath>
    using namespace std;

    // 计算数字x的字宽（位数）
    int get_digit(int x) {
        if (x == 0) return 1; // 特殊情况：行数为0（但题目保证至少1行）
        return (int)log10(x) + 1;
    }

    int main() {
        vector<string> lines; // 存储每一行的内容（不含换行符）
        string line;

        // 读取所有行（getline自动跳过换行符）
        while (getline(cin, line)) {
            lines.push_back(line);
        }

        int m = lines.size(); // 总行数
        if (m == 0) return 0; // 题目保证至少1行，这里防错

        int s = get_digit(m); // 总行数的字宽

        // 逐行输出带行号的内容
        for (int i = 0; i < m; ++i) {
            int t = get_digit(i + 1); // 当前行号的字宽（i从0开始，行号是i+1）
            int space_num = s - t;    // 需要填充的空格数

            // 输出s-t个空格
            for (int j = 0; j < space_num; ++j) {
                cout << ' ';
            }

            // 输出行号和一个空格
            cout << (i + 1) << ' ';

            // 输出该行的内容
            cout << lines[i] << endl;
        }

        return 0;
    }
    ```
* **代码解读概要**：
    > 代码首先用`getline`逐行读取输入（自动处理空格和换行符），存入`vector<string>`中；然后计算总行数m的字宽s；最后循环处理每一行：计算当前行号的字宽t，输出s-t个空格、行号和一个空格，最后输出该行内容。核心逻辑清晰，变量命名直观。

* **代码解读**：
    > `fread(S + 1, 1, MAXN, stdin)`将输入一次性读入数组`S`（从索引1开始），返回读取的字符数。`count`函数统计`S`中换行符`'\n'`的数量，即行数m（因为每行末尾恰好一个换行符）。这种方法高效，因为一次性IO比多次IO快很多。
* 💡 **学习笔记**：处理大文件时，减少IO次数能显著提升程序速度。

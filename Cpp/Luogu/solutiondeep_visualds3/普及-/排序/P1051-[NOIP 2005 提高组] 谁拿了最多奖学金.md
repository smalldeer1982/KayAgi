# 题目信息

# [NOIP 2005 提高组] 谁拿了最多奖学金

## 题目描述

某校的惯例是在每学期的期末考试之后发放奖学金。发放的奖学金共有五种，获取的条件各自不同：

1. 院士奖学金，每人 $8000$ 元，期末平均成绩高于 $80$ 分（$>80$），并且在本学期内发表$1$篇或$1$篇以上论文的学生均可获得；
2. 五四奖学金，每人 $4000$ 元，期末平均成绩高于 $85$ 分（$>85$），并且班级评议成绩高于 $80$ 分（$>80$）的学生均可获得；
3. 成绩优秀奖，每人 $2000$ 元，期末平均成绩高于 $90$ 分（$>90$）的学生均可获得；
4. 西部奖学金，每人 $1000$ 元，期末平均成绩高于 $85$ 分（$>85$）的西部省份学生均可获得；
5. 班级贡献奖，每人 $850$ 元，班级评议成绩高于 $80$ 分（$>80$）的学生干部均可获得；

只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金。例如姚林的期末平均成绩是 $87$ 分，班级评议成绩 $82$ 分，同时他还是一位学生干部，那么他可以同时获得五四奖学金和班级贡献奖，奖金总数是 $4850$ 元。

现在给出若干学生的相关数据，请计算哪些同学获得的奖金总数最高（假设总有同学能满足获得奖学金的条件）。


## 说明/提示

对于 $100\%$ 的数据，满足 $1 \le  N \le  100$。

✨ **本题主要考察**：模拟（编程技巧应用）

🗣️ **初步分析**：  
解决“谁拿了最多奖学金”这道题，就像给每个同学“检查奖状”——我们需要逐一核对每个学生的成绩、论文数、是否是干部等信息，看看他们符合哪些奖学金的条件，然后统计每个人的总奖金，最后找出“奖金最多的同学”。这就是**模拟算法**的核心：**按照题目给定的规则，一步步还原现实场景的计算过程**。  

在本题中，模拟的关键是：  
- 准确判断每个学生是否符合5种奖学金的条件（比如“期末成绩>80分且论文≥1篇”才能拿院士奖学金）；

- 找出总奖金最高的学生（如果有并列，保留输入顺序最早的）。  

**核心算法流程**：  
1. 读入所有学生的信息（姓名、成绩、班级评议、是否干部、是否西部学生、论文数）；  
2. 对每个学生，逐一判断是否符合5种奖学金的条件，累加总奖金；  
3. 记录总奖金最高的学生（用“打擂台”法：遍历所有学生，不断更新最大值）；  
4. 输出最高奖金学生的姓名、其奖金数，以及所有学生的奖金总和。

## 3. 核心难点辨析与解题策略

在解决这道题时，同学们常遇到以下3个难点，我结合优质题解的做法，总结了应对策略：


### 1. **难点1：条件判断的准确性（比如“>”和“≥”的区别）**  
* **分析**：题目中所有奖学金的条件都是“高于”（如“期末平均成绩>80分”），而不是“高于或等于”。如果把`>`写成`≥`，会导致错误（比如某学生成绩刚好80分，却被误判为符合院士奖学金条件）。  
* **解决策略**：**仔细读题，圈出关键词**（如“高于”“>”），并在代码中严格按照题目要求写条件。比如优质题解中，所有条件都用了`>`而不是`≥`。  
* 💡 **学习笔记**：条件判断是模拟题的“生命线”，一定要和题目保持一致！


### 2. **难点2：多个奖学金的叠加（不能用`else if`）**  
* **分析**：题目中说“只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金”。如果用`else if`，会导致学生只能获得其中一项奖学金（比如符合院士和五四奖学金的条件，但只能拿到其中一项）。  
* **解决策略**：**用多个独立的`if`语句**，每个条件单独判断。比如优质题解中，都是用`if`而不是`else if`，确保所有符合条件的奖学金都能累加。  
* 💡 **学习笔记**：多个条件可以同时满足时，一定要用`if`而不是`else if`！


### 3. **难点3：处理并列第一的情况（保留输入顺序最早的）**  
* **分析**：题目中说“如果有两位或两位以上的学生获得的奖金最多，输出他们之中在输入文件中出现最早的学生的姓名”。如果只用`max`变量记录最高奖金，会丢失输入顺序的信息。  
* **解决策略**：**记录学生的输入顺序**（比如用`id`变量存储输入的序号），当遇到奖金等于当前最大值时，保留`id`较小的学生。比如优质题解中，lucienshawls用`code`变量记录输入顺序，排序时先按奖金降序，再按`code`升序；Haohao123coding用`id`变量，打擂台时判断“如果奖金相等，保留id较小的”。  
* 💡 **学习笔记**：处理并列情况时，一定要记录“次要关键字”（如输入顺序）！


### ✨ 解题技巧总结  
- **技巧1：用结构体存储复杂信息**：学生有多个属性（姓名、成绩、是否干部等），用结构体可以将这些信息整合在一起，便于处理。  
- **技巧2：用“打擂台”法找最大值**：遍历所有学生，不断更新“当前最高奖金”和“对应学生”，不需要排序（节省时间）。  
- **技巧3：变量名要规范**：用有意义的变量名（如`score`表示成绩、`money`表示奖金），避免用`a`、`b`、`c`等模糊变量名，提高代码可读性。  


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考  
* **说明**：本代码综合了优质题解的思路，用结构体存储学生信息，用“打擂台”法找最大值，逻辑清晰，适合初学者。  
* **完整核心代码**：  
```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    string name;       // 姓名
    int score;         // 期末平均成绩
    int class_score;   // 班级评议成绩
    char is_leader;    // 是否是学生干部（Y/N）
    char is_west;      // 是否是西部学生（Y/N）
    int papers;        // 论文数量
    int total_money;   // 总奖金
};

int main() {
    int n;
    cin >> n;
    Student students[105];  // 存储最多100个学生的信息
    int max_money = -1;     // 最高奖金（初始化为-1，确保第一个学生能更新）
    int max_index = 0;      // 最高奖金学生的索引
    int total_sum = 0;      // 所有学生的奖金总和

    for (int i = 0; i < n; i++) {
        // 读入学生信息
        cin >> students[i].name >> students[i].score >> students[i].class_score;
        cin >> students[i].is_leader >> students[i].is_west >> students[i].papers;

        // 计算总奖金
        students[i].total_money = 0;
        if (students[i].score > 80 && students[i].papers >= 1) {
            students[i].total_money += 8000;
        }
        if (students[i].score > 85 && students[i].class_score > 80) {
            students[i].total_money += 4000;
        }
        if (students[i].score > 90) {
            students[i].total_money += 2000;
        }
        if (students[i].score > 85 && students[i].is_west == 'Y') {
            students[i].total_money += 1000;
        }
        if (students[i].class_score > 80 && students[i].is_leader == 'Y') {
            students[i].total_money += 850;
        }

        // 更新最高奖金学生
        if (students[i].total_money > max_money) {
            max_money = students[i].total_money;
            max_index = i;
        }

        // 累加总奖金
        total_sum += students[i].total_money;
    }

    // 输出结果
    cout << students[max_index].name << endl;
    cout << max_money << endl;
    cout << total_sum << endl;

    return 0;
}
```  
* **代码解读概要**：  
  1. 定义`Student`结构体，存储学生的所有信息（姓名、成绩、是否干部等）；  
  2. 读入`n`个学生的信息，依次计算每个学生的总奖金（用5个`if`语句判断）；  
  3. 用“打擂台”法更新最高奖金学生的索引；  
  4. 累加所有学生的奖金总和；  
  5. 输出最高奖金学生的姓名、其奖金数，以及总奖金。

* 💡 **学习笔记**：逻辑表达式可以用来简化条件判断的累加，适合竞赛中的快速编码。


#### **题解二：（来源：lucienshawls）**  
* **亮点**：用模板函数实现快速读入，应对大数据。  
* **核心代码片段**：  
```cpp
template <typename T>
inline void read(T &input) {
    input = 0;
    int f = 1;
    char c = getchar();
    while (c > '9' || c < '0') {
        if (c == '-') f = -1;
        c = getchar();
    }
    while (c <= '9' && c >= '0') {
        input = (input << 3) + (input << 1) + c - '0';  // input*10 + (c-'0')
        c = getchar();
    }
    input *= f;
}
```  
* **代码解读**：  
  这个`read`函数是**模板函数**，支持快速读入`int`、`long long`等数字类型。它用`getchar`逐个读取字符，避免了`cin`的慢速度（当数据量大时，`cin`会超时）。比如`input = (input << 3) + (input << 1) + c - '0'`等价于`input = input*10 + (c-'0')`，用位运算提高了效率。  
* 💡 **学习笔记**：快速读入是竞赛中的常用技巧，即使数据量小，也可以养成好习惯。

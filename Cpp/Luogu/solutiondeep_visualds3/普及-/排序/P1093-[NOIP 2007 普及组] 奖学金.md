# 题目信息

# [NOIP 2007 普及组] 奖学金

## 题目背景

NOIP2007 普及组 T1

## 题目描述

某小学最近得到了一笔赞助，打算拿出其中一部分为学习成绩优秀的前 $5$ 名学生发奖学金。期末，每个学生都有 $3$ 门课的成绩：语文、数学、英语。先按总分从高到低排序，如果两个同学总分相同，再按语文成绩从高到低排序，如果两个同学总分和语文成绩都相同，那么规定学号小的同学排在前面，这样，每个学生的排序是唯一确定的。

任务：先根据输入的 $3$ 门课的成绩计算总分，然后按上述规则排序，最后按排名顺序输出前五名名学生的学号和总分。

注意，在前 $5$ 名同学中，每个人的奖学金都不相同，因此，你必须严格按上述规则排序。例如，在某个正确答案中，如果前两行的输出数据（每行输出两个数：学号、总分) 是：

```plain
7 279  
5 279
```

这两行数据的含义是：总分最高的两个同学的学号依次是 $7$ 号、$5$ 号。这两名同学的总分都是 $279$ (总分等于输入的语文、数学、英语三科成绩之和) ，但学号为 $7$ 的学生语文成绩更高一些。

如果你的前两名的输出数据是：

```plain
5 279  
7 279
```

则按输出错误处理，不能得分。

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：排序（结构体多关键字排序）  

🗣️ **初步分析**：  
解决“奖学金”问题，关键在于**将学生信息打包成“结构体”，并按照“总分→语文成绩→学号”的优先级排序**。就像整理学生档案时，先按总分高低排，总分相同的再比语文成绩，语文也相同的就按学号从小到大排——这样每个学生的排名就唯一啦！  

### 核心算法流程  
1. **定义结构体**：把每个学生的学号、语文、数学、英语成绩和总分存到一个“档案袋”里（结构体）。  
2. **计算总分**：输入成绩后，自动算出每个学生的总分。  
3. **自定义排序**：用`sort`函数结合`cmp`函数，按照题目要求的优先级排序。  
4. **输出结果**：取排序后的前5名，输出他们的学号和总分。

## 3. 核心难点辨析与解题策略  

在解决结构体排序问题时，新手常遇到以下3个难点，我们一一拆解：  

### 1. 如何定义结构体？  
* **难点**：不知道该把哪些信息放进结构体里。  
* **解决策略**：结构体要包含**所有需要参与排序或输出的信息**。比如本题中，排序需要“总分、语文成绩、学号”，输出需要“学号、总分”，所以结构体里要存“学号、语文、数学、英语、总分”（数学和英语用于计算总分）。  
* 💡 **学习笔记**：结构体是“信息的打包盒”，需要什么就装什么！  

### 2. 如何写`cmp`函数？  
* **难点**：搞不清多关键字排序的优先级顺序。  
* **解决策略**：按照题目要求的顺序，**先判断最高优先级的条件，不满足再判断下一个**。比如本题中，先比总分（`a.sum > b.sum`），总分相同再比语文（`a.c > b.c`），语文相同再比学号（`a.num < b.num`）。  
* 💡 **学习笔记**：`cmp`函数的逻辑要和题目要求“一一对应”！  

### 3. 如何输出前5名？  
* **难点**：排序后不知道怎么取前5个元素。  
* **解决策略**：排序后的数组是按从高到低排列的，所以直接循环输出前5个元素（`i从1到5`）即可。注意学号是结构体里的`num`字段，总分是`sum`字段。  
* 💡 **学习笔记**：排序后的数组顺序就是最终排名，直接取前几名就行！  

### ✨ 解题技巧总结  
- **技巧1**：用结构体打包信息，避免零散变量混乱。  
- **技巧2**：`cmp`函数要“先主后次”，严格按照题目要求的优先级写。  
- **技巧3**：输入时直接计算总分，避免重复计算。  


## 4. C++核心代码实现赏析  

### 本题通用核心C++实现参考  
* **说明**：综合了codesonic、wpy233等优质题解的思路，是结构体排序的“标准模板”。  
* **完整核心代码**：  
  ```cpp
  #include <iostream>
  #include <algorithm>
  using namespace std;

  struct Student {
      int num;     // 学号
      int chinese; // 语文成绩
      int math;    // 数学成绩
      int english; // 英语成绩
      int total;   // 总分
  } students[310]; // 最多300个学生

  // 自定义排序函数：按照总分→语文→学号排序
  bool cmp(Student a, Student b) {
      if (a.total != b.total) {
          return a.total > b.total; // 总分高的在前
      } else if (a.chinese != b.chinese) {
          return a.chinese > b.chinese; // 语文高的在前
      } else {
          return a.num < b.num; // 学号小的在前
      }
  }

  int main() {
      int n;
      cin >> n;
      for (int i = 1; i <= n; i++) {
          students[i].num = i; // 学号从1开始
          cin >> students[i].chinese >> students[i].math >> students[i].english;
          students[i].total = students[i].chinese + students[i].math + students[i].english; // 计算总分
      }
      sort(students + 1, students + 1 + n, cmp); // 排序
      for (int i = 1; i <= 5; i++) { // 输出前5名
          cout << students[i].num << " " << students[i].total << endl;
      }
      return 0;
  }
  ```  
* **代码解读概要**：  
  这段代码先定义了`Student`结构体，包含学生的所有信息。`main`函数中，输入学生数量`n`，然后循环输入每个学生的成绩，计算总分。接着用`sort`函数对结构体数组排序（用`cmp`函数指定排序规则），最后输出前5名的学号和总分。  

### 针对各优质题解的片段赏析  

#### 题解一（来源：codesonic）  
* **亮点**：`cmp`函数逻辑清晰，完美匹配题目要求。  
* **核心代码片段**：  
  ```cpp
  bool cmp(stu a, stu b) {
      if (a.sum > b.sum) return 1;
      else if (a.sum < b.sum) return 0;
      else {
          if (a.c > b.c) return 1;
          else if (a.c < b.c) return 0;
          else {
              if (a.num > b.num) return 0;
              else return 1;
          }
      }
  }
  ```  
* **代码解读**：  
  这段`cmp`函数用“如果…否则…”的结构，把排序条件写得非常清楚。比如，当`a.sum > b.sum`时，返回`1`（表示`a`应该排在`b`前面）；当`a.sum == b.sum`时，再比较语文成绩`a.c`和`b.c`；如果语文也相同，就比较学号`a.num`和`b.num`（学号小的排在前面）。  
* 💡 **学习笔记**：`cmp`函数的返回值表示“`a`是否应该排在`b`前面”，要记住“大的在前”用`>`，“小的在前”用`<`。  

#### 题解二（来源：wpy233）  
* **亮点**：用“万能头文件”简化代码，适合新手快速编写。  
* **核心代码片段**：  
  ```cpp
  #include <bits/stdc++.h> // 万能头文件，包含所有常用头文件
  using namespace std;

  struct xx {
      int yw; // 语文
      int ss; // 数学
      int yy; // 英语
      int zf; // 总分
      int bh; // 学号
  } a[305];
  ```  
* **代码解读**：  
  作者用`#include <bits/stdc++.h>`代替了`#include <iostream>`和`#include <algorithm>`，简化了代码。结构体`xx`中的字段命名（`yw`表示语文，`ss`表示数学）虽然不够直观，但符合新手的“拼音缩写”习惯，容易记忆。  
* 💡 **学习笔记**：万能头文件可以节省写头文件的时间，但要注意在竞赛中是否允许使用（一般允许）。  

#### 题解三（来源：ryf2011）  
* **亮点**：代码简洁，没有冗余字段。  
* **核心代码片段**：  
  ```cpp
  struct node {
      int chines, math, eng, sum, num;
  } stu[305];
  ```  
* **代码解读**：  
  作者把结构体命名为`node`，字段命名（`chines`表示语文，`math`表示数学，`eng`表示英语，`sum`表示总分，`num`表示学号）非常简洁，没有多余的信息。这种写法适合在竞赛中快速编写代码，节省时间。  
* 💡 **学习笔记**：结构体的字段命名要简洁明了，避免冗长。

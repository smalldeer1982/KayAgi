# 最大公约数和最小公倍数

## 题目描述

给定两个正整数 $a,b$，求他们的最大公约数（gcd）和最小公倍数（lcm）。这两个整数均在 int 范围内。

## 样例 #1

### 输入

```
6 15```

### 输出

```
3 30```

#### **C++核心代码实现赏析**
**通用核心实现参考**  
```cpp
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    return b ? gcd(b, a % b) : a; // 递归终止：b为0时返回a
}

int main() {
    int a, b;
    cin >> a >> b;
    int g = gcd(a, b);
    cout << g << " " << a / g * b; // 先除防溢出
    return 0;
}
```
**代码解读概要**：  
> 递归函数`gcd`通过三目运算符实现自我更新，当`b=0`时抵达终点。main函数先读入数据，调用gcd后利用公式安全计算lcm。

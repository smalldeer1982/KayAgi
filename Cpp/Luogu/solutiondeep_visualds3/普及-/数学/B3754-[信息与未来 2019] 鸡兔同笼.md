# 题目信息

# [信息与未来 2019] 鸡兔同笼

## 题目描述

有三种动物被关在笼子里，它们分别是：
- 鸡，每只鸡有一个头、两只脚；
- 三脚猫，每只三脚猫有一个头，三只脚；
- 兔子，每只兔子有一个头，四只脚。

现在知道笼子中一共有 $x$ 个头、$y$ 只脚，但这个信息并不足以推导出鸡、兔和三脚猫的准确数量。这时候就需要编程来帮忙了——请你求出笼子里分别至少、至多有几只兔子。

## 说明/提示

对于 $50\%$ 的测试数据，有 $1\le x\le 10^6$。

对于 $100\%$ 的测试数据，有 $1\le x\le10^9,1\le T\le20$。
> 本题原始满分为 $20\text{pts}$。

## 样例 #1

### 输入

```
4
6 12
6 21
6 24
12345 40000```

### 输出

```
0 0
3 4
6 6
2965 7655```

### Kay的C++算法解析：[信息与未来 2019] 鸡兔同笼 深入学习指南

#### 1. 题目解读与核心算法识别
✨ **本题主要考察**：`数学推导与不等式约束`

🗣️ **初步分析**：  
> 本题本质是**多元方程组的整数解约束**问题，类似在餐厅点餐时计算不同套餐的组合（鸡=经济套餐，三脚猫=标准套餐，兔子=豪华套餐）。核心是通过建立方程组，推导出兔子数量的可行域范围。  
> - **核心思路**：设鸡、三脚猫、兔子的数量分别为 \(a\), \(b\), \(c\)，得到方程组：
>   \[
>   \begin{cases} 
>   a + b + c = x \\ 
>   2a + 3b + 4c = y 
>   \end{cases}
>   \]
>   消元后得关键约束：\(c \in \left[\max(0, y-3x), \ \lfloor (y-2x)/2 \rfloor \right]\)  
> - **可视化设计**：采用像素风"动物园管理"动画，笼子网格代表动物数量，兔子数量滑块拖动时实时更新其他动物数量，边界值触发闪光警报音效。

#### 2. 精选优质题解参考
**题解一（数学推导法）**  
* **点评**：  
  直接通过代数推导得出兔子数量的上下界公式，逻辑清晰严谨。代码仅需5行核心计算，无冗余变量（如 `d = y - 2*x` 精准表达约束关系），时间复杂度 \(O(1)\) 完美处理 \(10^9\) 数据。边界处理完整（检测 \(y \in [2x, 4x]\)），是竞赛标准解法。

---

#### 3. 核心难点辨析与解题策略
1. **关键点1：建立有效代数模型**  
   * **分析**：必须将动物数量关系转化为线性方程组。通过消元法消除鸡的数量 \(a\)，得到核心约束 \(b + 2c = y - 2x\)（设 \(d = y - 2x\))。  
   * 💡 **学习笔记**：消元是减少变量维度的核心技巧。

2. **关键点2：推导可行域边界**  
   * **分析**：由 \(b = d - 2c \geq 0\) 得 \(c \leq \lfloor d/2 \rfloor\)；由 \(a = x - d + c \geq 0\) 得 \(c \geq \max(0, d - x)\)。需同步验证 \(d \in [0, 2x]\)。  
   * 💡 **学习笔记**：约束条件需同时满足非负性和整数解特性。

3. **关键点3：处理大数据溢出**  
   * **分析**：当 \(x \leq 10^9\) 时，\(2x\) 可能超过 `int` 范围。必须使用 `long long` 避免溢出。  
   * 💡 **学习笔记**：大数据范围优先检查运算溢出风险。

### ✨ 解题技巧总结
- **代数消元法**：通过消减变量降低问题复杂度
- **边界值分析法**：对约束条件的极值点（如 \(c=0, c=\lfloor d/2 \rfloor\)) 单独验证
- **数据类型预判**：根据数据范围主动选用 `long long`

#### 4. C++核心代码实现赏析
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        long long x, y;
        cin >> x >> y;
        
        if (y < 2*x || y > 4*x) {
            cout << "0 0\n";
        } else {
            long long d = y - 2*x;
            long long min_c = max(0LL, d - x);
            long long max_c = d / 2; // 整数除法自动向下取整
            cout << min_c << " " << max_c << "\n";
        }
    }
    return 0;
}
```
**代码解读概要**：  
1. 读取总头数 \(x\) 和总脚数 \(y\)  
2. 检测无解情况（脚数超出理论范围）  
3. 计算关键中间量 \(d = y - 2x\)  
4. 通过不等式约束确定兔子数量的最小/最大值

# 题目信息

# [海淀区小学组 2025] 分数方程

## 题目背景

2025 年海淀区中小学生信息学竞赛小学组复赛题目，数据为洛谷自造。

## 题目描述

给定一个正整数 $n$，请找出一组互不相等的正整数 $x, y, z$，使得 $\dfrac{1}{x} + \dfrac{1}{y} + \dfrac{1}{z} = \dfrac{2}{n}$ 成立。

## 说明/提示

- 对于 $30\%$ 的数据，保证存在一组解使得 $1 \leq x, y, z \leq 100$，$1 \leq n \leq 10^6$。
- 对于 $60\%$ 的数据，保证存在一组解使得 $1 \leq x, y, z \leq 1000$，$1 \leq n \leq 10^6$。
- 对于 $100\%$ 的数据，保证存在一组解使得 $1 \leq x, y, z \leq 10^9$，$1 \leq n \leq 10^6$。

为确保评测结果正确，输出的 $x,y,z$ 必须是不超过 $10^{18}$ 的正整数。

## 样例 #1

### 输入

```
3```

### 输出

```
2 7 42```

## 样例 #2

### 输入

```
7```

### 输出

```
7 8 56```

#### 1. 题目解读与核心算法识别
✨ **本题主要考察**：`数学构造法`  
🗣️ **初步分析**：  
> 解决分数方程题的关键在于**数学构造技巧**，就像用乐高积木拼出特定形状——我们需要找到特定组合满足方程。核心思路是：  
> - 设 `x = n` 简化方程：`1/y + 1/z = 1/n`  
> - 变形为 `(y-n)(z-n) = n²`，取因数组合 `(1, n²)`  
> - 得到解：`x=n, y=n+1, z=n(n+1)`  
>

#### 3. 核心难点辨析与解题策略
1. **难点1：方程降维**  
   * **分析**：三元方程复杂，需固定变量简化。优质解法设 `x=n` 转化为二元方程  
   * 💡 **学习笔记**：减少变量是解复杂方程的突破口  

2. **难点2：因式分解技巧**  
   * **分析**：`ny + nz - yz = 0` 不易处理，加 `n²` 配凑得 `(y-n)(z-n)=n²` 是关键技巧  
   * 💡 **学习笔记**：通过添加常数项构造乘积形式是常用数学技巧  

3. **难点3：解的唯一性保证**  
   * **分析**：需确保 `x,y,z` 互异且 ≤10¹⁸。构造法 `x=n, y=n+1, z=n(n+1)` 天然满足（n≥2时）  
   * 💡 **学习笔记**：构造法优先选择最简形式避免边界问题  

**✨ 解题技巧总结**  
- **技巧1：降维打击** – 将复杂问题拆解为子问题（如固定一个变量）  
- **技巧2：数学配凑** – 通过添加项（如 `n²`）构造可分解形式  
- **技巧3：极简验证** – 用样例快速检验构造解（如 `n=3→1/3+1/4+1/12=2/3`）

**通用核心实现**

* **说明**：综合题解思路的最简实现  
* **解读概要**：  
  - 输入 `n` 后直接输出构造解  
  - `n*(n+1)` 可能达 10¹²（n≤10⁶），故用 `long long`  

**题解片段赏析**  
```cpp
// 构造解核心逻辑
long long z = n * (n + 1); // n² + n
cout << n << " " << n + 1 << " " << z;
```
* **亮点**：无循环/分支，直接数学计算  
* **代码解读**：  
  > `z = n*(n+1)` 对应因式分解中 `(y-n)(z-n)=n²` 取 `y-n=1` 时的解  
  > 当 `n=7` 时：`z=7×8=56` → `1/7+1/8+1/56=2/7`  
* 💡 **学习笔记**：数学推导可直接映射为代码表达式

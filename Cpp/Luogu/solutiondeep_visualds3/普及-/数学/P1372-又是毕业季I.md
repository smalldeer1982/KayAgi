# 又是毕业季I

## 题目背景

“叮铃铃铃”，随着高考最后一科结考铃声的敲响，三年青春时光顿时凝固于此刻。毕业的欣喜怎敌那离别的不舍，憧憬着未来仍毋忘逝去的歌。$1000$ 多个日夜的欢笑和泪水，全凝聚在毕业晚会上，相信，这一定是一生最难忘的时刻！

## 题目描述

为了把毕业晚会办得更好，老师想要挑出默契程度最大的 $k$ 个人参与毕业晚会彩排。可是如何挑呢？老师列出全班同学的号数 $1,2,\ldots ,n$ 并且相信 $k$ 个人的默契程度便是他们的最大公约数（这不是迷信哦~）。这可难为了他，请你帮帮忙吧！

PS：一个数的最大公约数即本身。

## 说明/提示

对于 $20\%$ 的数据，$k \le 2$，$n \le 10^3$。

对于另 $30\%$ 的数据，$k \le 10$，$n \le 100$。

对于 $100\%$ 的数据，$k \le 10^9$，$n \le 10^9$，$n \ge k \ge 1$（神犇学校，人数众多）。

## 样例 #1

### 输入

```
4 2
```

### 输出

```
2
```

## 1. 题目解读与核心算法识别
✨ **本题主要考察**：`数学推理与数论应用`  
🗣️ **初步分析**：
> 解决本题的关键在于发现**最大公约数的倍数性质**。想象我们要选出k个"步调一致"的数字（如2/4/6），它们的GCD就是步长（如2）。最优步长需满足：  
> `最大步长 × k ≤ n` → **最大步长 = n/k**  
> - **题解共性**：所有优质题解均通过数论推导得出结论（n/k），无需复杂算法。
> - **可视化设计**：像素动画将展示数轴（1~n）和倍数序列（x,2x,...,kx）。当x过大时，kx超出n会触发红色闪烁和警告音效；当x=n/k时，倍数序列完美填充数轴，播放胜利音效。

## 3. 核心难点辨析与解题策略
1. **难点1：如何想到GCD与倍数的关系？**  
   * **分析**：优质题解通过观察倍数序列（x,2x,...,kx）的GCD恒为x，将问题转化为求最大x使得kx≤n。
   * 💡 **学习笔记**：最大GCD问题常关联有序倍数序列。

2. **难点2：为何n/k是理论最大值？**  
   * **分析**：反证法——若存在更大解p，则需k个p的倍数，最小为p,2p,...,kp。但kp>n（因p>n/k），矛盾。
   * 💡 **学习笔记**：边界条件可通过不等式严格证明。

3. **难点3：为何无需考虑非倍数序列？**  
   * **分析**：任意k个数的GCD必≤其最小数≤第k小数。而有序倍数序列的最小数为x，已覆盖最优情况。
   * 💡 **学习笔记**：最优解常存在于高度结构化（如倍数）的序列中。

### ✨ 解题技巧总结
- **技巧1：数学建模优先**——将编程问题转化为数学不等式。
- **技巧2：极值反证法**——假设更大解，推导矛盾。
- **技巧3：边界特判**——n/k直接处理10⁹级数据，避免无效循环。

## 题目描述

求有多少个平方后末尾为 $987654321$ 的 $n$ 位数。

## 说明/提示

$1\le n\le 1 0^6$。

不要问我暴力能过几个点，我只想说呵呵。

### 输入

8```

### 输出

0```

### C++核心代码实现赏析
**通用核心实现**  
```cpp
#include <cstdio>
int main() {
    int n; scanf("%d", &n);
    if (n < 9) printf("0");
    else if (n == 9) printf("8");
    else {
        printf("72"); 
        for (int i = 0; i < n - 10; ++i) putchar('0');
    }
    return 0;
}
```
**代码解读概要**：  
> 分层处理三种情况：n≤8直接输出0；n=9输出基础解数量8；n>9时输出"72"后接(n-10)个零，完美匹配72×10^(n-10)的数学形式。

**题解一核心片段赏析**  
```cpp
if (n<9) printf("0\n");
else if (n==9) printf("8\n");
else {
    printf("72");  // 9(首位选择)*8(末9位选择)
    for (int i=0;i<n-10;i++) printf("0"); // 中间位补零
}
```
**学习笔记**：显式注释72的构成来源（9×8），增强可读性

**题解二核心片段赏析**  
```cpp
printf("72");
for(i=1;i<=m-10;i++) printf("0");
```
**学习笔记**：极致简洁风格，依赖前置条件判断确保此处m>9

**题解三核心片段赏析**  
```cpp
if(m==9) printf("8");
if(n>9){
    printf("72");
    for(int i=n;i>=11;i--) printf("0");
}
```
**学习笔记**：循环条件i>=11的写法避免n-10出现负值

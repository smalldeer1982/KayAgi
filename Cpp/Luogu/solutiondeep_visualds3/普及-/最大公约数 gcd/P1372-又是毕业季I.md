# 又是毕业季I

## 题目背景

“叮铃铃铃”，随着高考最后一科结考铃声的敲响，三年青春时光顿时凝固于此刻。毕业的欣喜怎敌那离别的不舍，憧憬着未来仍毋忘逝去的歌。$1000$ 多个日夜的欢笑和泪水，全凝聚在毕业晚会上，相信，这一定是一生最难忘的时刻！

## 题目描述

为了把毕业晚会办得更好，老师想要挑出默契程度最大的 $k$ 个人参与毕业晚会彩排。可是如何挑呢？老师列出全班同学的号数 $1,2,\ldots ,n$ 并且相信 $k$ 个人的默契程度便是他们的最大公约数（这不是迷信哦~）。这可难为了他，请你帮帮忙吧！

PS：一个数的最大公约数即本身。

## 说明/提示

对于 $20\%$ 的数据，$k \le 2$，$n \le 10^3$。

对于另 $30\%$ 的数据，$k \le 10$，$n \le 100$。

对于 $100\%$ 的数据，$k \le 10^9$，$n \le 10^9$，$n \ge k \ge 1$（神犇学校，人数众多）。

## 样例 #1

### 输入

```
4 2
```

### 输出

```
2
```

✨ **本题主要考察**：`数论应用（最大公约数的性质与数学推导）`

🗣️ **初步分析**：
解决这道题的关键在于理解“如何构造k个数，使其最大公约数最大”。我们可以用一个简单的比喻：假设我们要找一个“基数”x，让k个数都是x的倍数（比如x, 2x, 3x…kx），这样它们的最大公约数至少是x。为了让x尽可能大，需要满足kx ≤n（否则kx会超过n，无法选取）。因此，最大的x就是n除以k的商（向下取整）。

- **题解思路**：所有优质题解均指出，答案即为n/k的整数部分。核心难点在于证明“选择x的倍数是最优策略”，而解决方案是通过数学推导验证这一策略的正确性。
- **核心算法流程**：直接计算n除以k的商（向下取整），无需复杂遍历或搜索。
- **可视化设计**：采用8位像素风动画，用“倍数方块”展示x的选择过程：从x=1开始，逐渐增大x，直到kx超过n，此时最大的x即为答案。关键步骤高亮显示kx与n的比较，并用音效提示“找到最大x”。

## 3. 核心难点辨析与解题策略

<difficulty_intro>
解决这道题的关键在于理解“如何构造最大公约数最大的k个数”。以下是核心难点及应对策略：
</difficulty_intro>

1.  **关键点1**：如何构造k个数，使其最大公约数尽可能大？
    * **分析**：观察到，若k个数是x的倍数（x, 2x, 3x…kx），则它们的最大公约数至少是x。为了x最大，需满足kx ≤n（否则无法选取kx）。因此，x的最大值为n/k的整数部分。
    * 💡 **学习笔记**：构造倍数序列是提升最大公约数的有效策略。

2.  **关键点2**：如何证明n/k是最大的可能值？
    * **分析**：假设存在更大的p>n/k，则p*k>n（因为p>n/k → p*k>n），无法选取k个p的倍数（最大的数p*k超过n），矛盾。因此，n/k是最大值。
    * 💡 **学习笔记**：反证法可用于验证结论的最优性。

3.  **关键点3**：如何处理大数范围（n≤1e9）？
    * **分析**：题目数据范围大（n≤1e9），需避免暴力枚举。直接计算n/k的时间复杂度为O(1)，完全满足要求。
    * 💡 **学习笔记**：数学推导是处理大数问题的关键，避免暴力枚举。

### ✨ 解题技巧总结
<summary_best_practices>
- **观察规律**：通过小数据打表（如n=4,k=2输出2；n=6,k=2输出3），发现答案与n/k的关系。
- **数学构造**：构造倍数序列（x, 2x…kx）是提升最大公约数的通用方法。
- **反证验证**：用反证法证明结论的最优性，确保思路正确性。

# 题目信息

# 【深基3.习8】三角形分类

## 题目描述

给出三条线段 $a,b,c$ 的长度，均是不大于 $10000$ 的正整数。打算把这三条线段拼成一个三角形，它可以是什么三角形呢？

- 如果三条线段不能组成一个三角形，输出`Not triangle`；
- 如果是直角三角形，输出`Right triangle`；
- 如果是锐角三角形，输出`Acute triangle`；
- 如果是钝角三角形，输出`Obtuse triangle`；
- 如果是等腰三角形，输出`Isosceles triangle`；
- 如果是等边三角形，输出`Equilateral triangle`。

如果这个三角形符合以上多个条件，请按以上顺序分别输出，并用换行符隔开。

## 说明/提示

当两短边的平方和大于一长边的平方，说明是锐角三角形。

当两短边的平方和等于一长边的平方，说明是直角三角形。

当两短边的平方和小于一长边的平方，说明是钝角三角形。

## 样例 #1

### 输入

```
3 3 3```

### 输出

```
Acute triangle
Isosceles triangle
Equilateral triangle```

```

## 3. 核心难点辨析与解题策略

<difficulty_intro>
在解决这道题的过程中，我们可能会遇到以下核心难点。结合优质题解的经验，我们一起拆解它们：
</difficulty_intro>

1.  **关键点1：如何正确排序并确定“两短边”和“一长边”？**
    * **分析**：排序是关键！将三边从小到大排序后（设为`a≤b≤c`），最长边一定是`c`，两短边是`a`和`b`。此时只需判断`a+b>c`即可确定能否构成三角形（若`a+b≤c`则不能）。优质题解通常使用`sort`函数直接排序数组，确保逻辑简单可靠。
    * 💡 **学习笔记**：排序后，所有关于“最长边”“两短边”的判断都可基于固定索引，避免混乱。

2.  **关键点2：如何理解“平方和判断角类型”的数学依据？**
    * **分析**：根据余弦定理，最长边`c`对应的角是最大角（记为`C`）。余弦定理公式为`c²=a²+b²-2ab·cosC`。若`C`是直角（90°），则`cosC=0`，故`c²=a²+b²`；若`C`是锐角（<90°），则`cosC>0`，故`c²<a²+b²`；若`C`是钝角（>90°），则`cosC<0`，故`c²>a²+b²`。优质题解通过这一原理，直接用平方和比较判断角类型。
    * 💡 **学习笔记**：平方和比较的本质是判断最大角的类型，这是解决本题的数学核心。

3.  **关键点3：如何处理多个条件的输出顺序？**
    * **分析**：题目要求按“不能构成→直角→锐角→钝角→等腰→等边”的顺序输出。需注意：等边三角形同时是等腰三角形，因此需先输出等腰再输出等边；若同时满足多个角类型（如直角且等腰），需全部输出。优质题解通过独立的`if`语句（而非`else if`）确保所有符合条件的类型都被输出。
    * 💡 **学习笔记**：用独立`if`判断每个条件，避免因`else`跳过后续符合条件的情况。

**本题通用核心C++实现参考**
* **说明**：本代码综合了Eason_AC和AstralSparkle的题解思路，通过排序简化逻辑，严格按题目顺序输出结果。
* **完整核心代码**：
    ```cpp
    #include <iostream>
    #include <algorithm>
    using namespace std;

    int main() {
        int a, b, c;
        cin >> a >> b >> c;
        int d[3] = {a, b, c}; // 存入数组方便排序
        sort(d, d + 3);       // 排序后d[0]≤d[1]≤d[2]

        // 第一步：判断是否能构成三角形
        if (d[0] + d[1] <= d[2]) {
            cout << "Not triangle\n";
            return 0;
        }

        // 第二步：判断角的类型（直角/锐角/钝角）
        int sum_sq = d[0] * d[0] + d[1] * d[1];
        int max_sq = d[2] * d[2];
        if (sum_sq == max_sq) {
            cout << "Right triangle\n";
        } else if (sum_sq > max_sq) {
            cout << "Acute triangle\n";
        } else {
            cout << "Obtuse triangle\n";
        }

        // 第三步：判断边的类型（等腰/等边）
        if (d[0] == d[1] || d[1] == d[2] || d[0] == d[2]) {
            cout << "Isosceles triangle\n";
        }
        if (d[0] == d[1] && d[1] == d[2]) {
            cout << "Equilateral triangle\n";
        }

        return 0;
    }
    ```
* **代码解读概要**：
  代码首先将三边存入数组并排序，确保`d[0]≤d[1]≤d[2]`。通过判断`d[0]+d[1]>d[2]`确定能否构成三角形。若能，计算两短边平方和与最长边平方比较，判断角类型；最后判断是否有两边相等（等腰）或三边相等（等边），按顺序输出结果。

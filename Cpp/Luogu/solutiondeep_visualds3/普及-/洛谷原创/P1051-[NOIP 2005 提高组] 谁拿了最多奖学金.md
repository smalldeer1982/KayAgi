# 题目信息

# [NOIP 2005 提高组] 谁拿了最多奖学金

## 题目描述

某校的惯例是在每学期的期末考试之后发放奖学金。发放的奖学金共有五种，获取的条件各自不同：

1. 院士奖学金，每人 $8000$ 元，期末平均成绩高于 $80$ 分（$>80$），并且在本学期内发表$1$篇或$1$篇以上论文的学生均可获得；
2. 五四奖学金，每人 $4000$ 元，期末平均成绩高于 $85$ 分（$>85$），并且班级评议成绩高于 $80$ 分（$>80$）的学生均可获得；
3. 成绩优秀奖，每人 $2000$ 元，期末平均成绩高于 $90$ 分（$>90$）的学生均可获得；
4. 西部奖学金，每人 $1000$ 元，期末平均成绩高于 $85$ 分（$>85$）的西部省份学生均可获得；
5. 班级贡献奖，每人 $850$ 元，班级评议成绩高于 $80$ 分（$>80$）的学生干部均可获得；

只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金。例如姚林的期末平均成绩是 $87$ 分，班级评议成绩 $82$ 分，同时他还是一位学生干部，那么他可以同时获得五四奖学金和班级贡献奖，奖金总数是 $4850$ 元。

现在给出若干学生的相关数据，请计算哪些同学获得的奖金总数最高（假设总有同学能满足获得奖学金的条件）。


## 说明/提示

**【数据范围】**

对于 $100\%$ 的数据，满足 $1 \le  N \le  100$。

**【题目来源】**

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：模拟（按照规则逐步计算）

🗣️ **初步分析**：  
模拟的核心就是“按规则办事”——就像你帮老师统计考试加分：先看每个同学的平时成绩、竞赛获奖，再一条条加对应的分数。在这道题里，我们要**为每个学生检查5项奖学金的条件，把符合的奖金加起来**，最后找出“奖金最多的学生”（如果有多个，选输入最早的）。  

### 核心算法流程与可视化设计思路
1. **输入学生信息**：像“收集食材”一样，把每个学生的姓名、成绩、是否是干部等信息存起来。  
2. **计算奖学金**：逐个检查5项条件（比如“期末>80且论文≥1”得院士奖学金），用`if`而不是`else if`（因为可以拿多个奖！）。  
3. **找最大值**：用“打擂台”的方式——记录当前最大的奖金和对应的学生，遇到更大的就更新，遇到相同的就保留更早的。  

### 可视化设计小预告
我们会做一个**8位像素风的“奖学金小判官”游戏**：  
- 屏幕上是像素化的“学生列表”，每个学生用不同颜色的方块代表；  
- 计算奖学金时，符合条件的方块会**闪烁+播放“叮”的音效**（比如拿到院士奖学金时，方块闪金色）；  
- 找最大值时，最大的方块会**放大+播放“胜利音效”**；  
- 还有“单步执行”按钮，让你一步步看每个学生的奖金怎么算出来的！

## 3. 核心难点辨析与解题策略

### 关键点1：正确理解每个奖学金的条件
* **难点**：容易把“>”搞成“≥”（比如院士奖学金需要期末>80，不是≥80），或者漏看条件（比如五四奖学金需要班级评议>80）。  
* **策略**：把每个条件**写在纸上**，比如：  
  - 院士：期末>80 + 论文≥1 → 8000元  
  - 五四：期末>85 + 班级>80 → 4000元  
  写下来就不容易错啦！


### 关键点2：处理多个条件的叠加
* **难点**：用`else if`会导致只能拿一个奖，但题目说可以拿多个！比如一个学生可能同时拿五四和班级贡献奖。  
* **策略**：**所有条件都用`if`**，不要用`else if`。就像你可以同时吃汉堡和薯条，每个条件都是独立的。


### 关键点3：处理并列第一的情况
* **难点**：如果两个学生奖金一样多，要输出输入最早的那个。  
* **策略**：用“打擂台”时，**只有当当前奖金>最大值时才更新**（而不是≥）。比如，假设当前最大值是9000，遇到另一个9000的学生，就不更新，这样保留的还是最早的那个。


### ✨ 解题技巧总结
1. **用结构体存储信息**：把学生的姓名、成绩等放在一个结构体里，像“装在一个盒子里”，方便管理。  
2. **打擂台法找最大值**：初始化一个`max`变量（比如-1），遍历每个学生，遇到更大的就更新`max`和对应的学生。  
3. **仔细检查条件**：每写一个条件，就回头看题目，确认是“>”还是“≥”，有没有漏看的条件。


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考
* **说明**：综合了多个优质题解的思路，用结构体存储学生信息，打擂台找最大值，代码清晰完整。
* **完整核心代码**：
```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    string name;    // 姓名
    int score;      // 期末成绩
    int c_score;    // 班级评议成绩
    char leader;    // 是否是干部（Y/N）
    char west;      // 是否是西部学生（Y/N）
    int paper;      // 论文数量
    int money;      // 奖学金总数
};

int main() {
    int n;
    cin >> n;
    Student stu[105];  // 最多100个学生，开105够了
    int total = 0;     // 总奖学金
    int max_money = -1; // 最大奖学金
    int max_idx = 0;    // 最大奖学金的学生索引

    for (int i = 0; i < n; i++) {
        cin >> stu[i].name >> stu[i].score >> stu[i].c_score 
            >> stu[i].leader >> stu[i].west >> stu[i].paper;
        
        // 计算奖学金
        stu[i].money = 0;
        if (stu[i].score > 80 && stu[i].paper >= 1) stu[i].money += 8000;
        if (stu[i].score > 85 && stu[i].c_score > 80) stu[i].money += 4000;
        if (stu[i].score > 90) stu[i].money += 2000;
        if (stu[i].score > 85 && stu[i].west == 'Y') stu[i].money += 1000;
        if (stu[i].c_score > 80 && stu[i].leader == 'Y') stu[i].money += 850;

        // 累加总奖学金
        total += stu[i].money;

        // 打擂台找最大值
        if (stu[i].money > max_money) {
            max_money = stu[i].money;
            max_idx = i;
        }
    }

    // 输出结果
    cout << stu[max_idx].name << endl;
    cout << max_money << endl;
    cout << total << endl;

    return 0;
}
```
* **代码解读概要**：  
  1. **输入部分**：用`struct Student`存储每个学生的信息，循环读入n个学生的数据。  
  2. **计算奖学金**：用5个`if`判断每个奖学金的条件，累加奖金。  
  3. **找最大值**：用`max_money`记录当前最大的奖金，`max_idx`记录对应的学生索引。  
  4. **输出**：输出最大奖学金的学生姓名、奖金，以及总奖学金。


### 题解一：（来源：qhr2023）
* **亮点**：用“逻辑表达式×奖金”简化代码，超级简洁！
* **核心代码片段**：
```cpp
sum=(a>80&&e)*8000+
    (a>85&&b>80)*4000+
    (a>90)*2000+
    (a>85&&d=='Y')*1000+
    (b>80&&c=='Y')*850;
```
* **代码解读**：  
  比如`(a>80&&e)*8000`：如果`a>80`且`e`（论文数量）≥1（注意这里`e`是输入的论文数，所以`e`本身就是≥1的条件），逻辑表达式返回1，乘8000就是8000元；否则返回0，不加钱。这种写法把多个`if`合并成一行，超高效！
* **学习笔记**：逻辑表达式的结果可以当整数用（true→1，false→0），能简化代码哦！

## 5. 算法可视化：像素动画演示 (核心部分)

### 动画演示主题：**奖学金小判官**（8位像素风）


### 设计思路简述
用8位像素风营造复古游戏的氛围，让你像玩“超级马里奥”一样学算法！每个学生是一个像素方块，计算奖学金时会有闪烁和音效，找最大值时会有“胜利动画”，增加趣味性。


### 动画帧步骤与交互关键点
1. **场景初始化**：  
   - 屏幕左侧是“学生列表”（每个学生是一个彩色方块，上面写着姓名缩写）；  
   - 右侧是“控制面板”：有“开始”“单步”“重置”按钮，还有一个速度滑块；  
   - 背景是像素化的教室，播放8位风格的背景音乐（比如《超级马里奥》的轻快旋律）。

2. **输入学生信息**：  
   - 每输入一个学生，屏幕上就出现一个新的像素方块（比如红色代表干部，蓝色代表西部学生）；  
   - 输入完成后，播放“叮”的音效，表示可以开始计算了。

3. **计算奖学金**：  
   - 逐个选中学生方块（用黄色框框住），然后检查每个条件：  
     - 如果符合院士奖学金，方块闪金色，播放“叮——”的长音效，同时右上角的“奖金”数字加8000；  
     - 如果符合五四奖学金，方块闪蓝色，播放“叮”的短音效，奖金加4000；  
   - 每个条件检查完，方块下方会显示当前的奖学金总数。

4. **找最大值**：  
   - 所有学生计算完后，最大奖学金的学生方块会**放大+旋转**，播放“胜利音效”（比如《魂斗罗》的通关音乐）；  
   - 屏幕下方显示“最大奖金：XXX元，获得者：XXX”。

5. **交互控制**：  
   - “单步执行”：点击一次，计算一个学生的奖学金；  
   - “自动播放”：可以调整速度（比如1秒/学生），自动计算所有学生；  
   - “重置”：清空所有学生，重新开始。


### 游戏化元素
- **音效**：符合条件时播放“叮”的音效，最大值时播放胜利音乐，错误时播放“嘟嘟”的提示音；  
- **小关卡**：每计算5个学生，就弹出“完成5个学生！继续加油～”的提示，增加成就感；  
- **积分**：计算每个学生的时间越短，积分越高，比如1秒内算完一个学生得10分，2秒内得5分，鼓励快速准确计算。

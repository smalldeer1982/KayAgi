# 题目信息

# zzc 种田

## 题目背景

可能以后 zzc 就去种田了。


## 题目描述

田地是一个巨大的矩形，然而 zzc 每次只能种一个正方形,而每种一个正方形时 zzc 所花的体力值是正方形的周长，种过的田不可以再种，zzc 很懒还要节约体力去泡妹子，想花最少的体力值去种完这块田地，问最小体力值。

## 说明/提示

$1\le x,y\le 10^{16}$。

## 样例 #1

### 输入

```
1 10```

### 输出

```
40```

## 样例 #2

### 输入

```
2 2 ```

### 输出

```
8```

#### 核心难点辨析与解题策略
1. **贪心策略证明**  
   * **分析**：为何取最大正方形最优？假设有更小正方形方案，其总周长必大于大正方形（周长/面积=4/边长，边长越小效率越低）  
   * 💡 **学习笔记**：贪心最优性证明常通过反证法/比例分析  

2. **大数优化技巧**  
   * **分析**：当 $y=1, x=10^{16}$ 时，减法需 $10^{16}$ 次循环，而 `x%y` 一次计算完成  
   * 💡 **学习笔记**：遇到 `while(x-=y)` 需警惕，用除法/取模替代  

3. **边界与初始化**  
   * **分析**：多篇题解因未初始化ans/long long类型导致WA。`x=0` 时需立即退出循环  
   * 💡 **学习笔记**：变量初始化应成为肌肉记忆  

### ✨ 解题技巧总结
- **空间换时间**：用取模运算 $O(1)$ 代替减法 $O(n)$  
- **递归优化**：通过数学计算减少递归深度  
- **循环不变式**：维护 `min(x,y)` 作为当前正方形边长  
- **数据类型敏感**：$10^{16}$ 必须用 `long long`

#### C++核心代码实现赏析
**通用核心实现**  
```cpp
#include <iostream>
using namespace std;
int main() {
    long long x, y, ans = 0;
    cin >> x >> y;
    while (x && y) {
        if (x < y) swap(x, y);
        ans += 4 * y * (x / y); // 核心优化
        x %= y;
    }
    cout << ans;
}
```
**代码解读概要**：  
1. 读入矩形长宽 `x,y`  
2. 循环直到任意边为0：  
   - 保证 `x≥y`（必要时swap）  
   - 累加 `y×4×(可切割数量)`  
   - 更新 `x = x%y`  
3. 输出总周长

#### 题解片段赏析
**题解一核心代码**  
```cpp
swap(x,y);
ans += 4*y*(x/y);
x %= y;
```
* **亮点**：swap使代码对称优雅  
* **解读**：`swap` 后 `y` 必为短边，确保 `x/y` 是整数切割数量。`x%y` 得到剩余长度，类似GCD算法的精髓  

**题解二优化递归**  
```cpp
sum += minn*4*(maxn/minn); 
if(maxn%minn==0) return;
kkk(minn, maxn%minn);
```
* **亮点**：递归深度从 $O(n)$ 优化到 $O(\log n)$  
* **解读**：`(maxn/minn)` 一次性处理多层递归，`maxn%minn` 保证剩余量小于minn

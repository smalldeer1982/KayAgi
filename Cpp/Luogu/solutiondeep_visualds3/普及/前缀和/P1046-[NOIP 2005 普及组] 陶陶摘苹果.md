# 题目信息

# [NOIP 2005 普及组] 陶陶摘苹果

## 题目描述

陶陶家的院子里有一棵苹果树，每到秋天树上就会结出 $10$ 个苹果。苹果成熟的时候，陶陶就会跑去摘苹果。陶陶有个 $30$ 厘米高的板凳，当她不能直接用手摘到苹果的时候，就会踩到板凳上再试试。


现在已知 $10$ 个苹果到地面的高度，以及陶陶把手伸直的时候能够达到的最大高度，请帮陶陶算一下她能够摘到的苹果的数目。假设她碰到苹果，苹果就会掉下来。


## 说明/提示

**【题目来源】**

NOIP 2005 普及组第一题

### 输入

```
100 200 150 140 129 134 167 198 200 111
110
```

### 输出

```
5```

## 3. 核心难点辨析与解题策略
1. **问题转化技巧**  
   * **分析**：关键在于识别"板凳高度可合并计算"——所有优质题解都先执行`H += 30`，将原问题简化为单次高度比较  
   * 💡 **学习笔记**：复杂条件常可转化为单一标准  

2. **循环内的效率优化**  
   * **分析**：避免在循环内重复计算固定值（如`H+30`）。RedContritio的方案进一步用逻辑运算替代分支判断，减少CPU流水线中断  
   * 💡 **学习笔记**：循环内代码要"轻量化"  

3. **计数器的灵活运用**  
   * **分析**：`s += (条件)`比`if+自增`更简洁，但需理解布尔值到整型的隐式转换（true=1, false=0）  
   * 💡 **学习笔记**：善用语言特性简化代码  

### ✨ 解题技巧总结
- **预处理简化条件**：将多步骤条件（身高+板凳）提前计算

- **语义化命名**：如`cnt`比`s`更能体现"计数"含义

## 4. C++核心代码实现赏析
**本题通用核心C++实现参考**  
* **说明**：综合优质题解优化出的最简实现  
* **完整核心代码**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int apples[10], H, cnt = 0;
    for(int i=0; i<10; ++i) cin >> apples[i];
    cin >> H;
    H += 30; // 预加板凳高度
    
    for(int i=0; i<10; ++i) 
        cnt += (apples[i] <= H); // 布尔表达式直接计数
        
    cout << cnt;
    return 0;
}
```
* **代码解读概要**：  
  > 1. 输入模块：读入10个苹果高度和陶陶身高  
  > 2. 预处理：H+30合并计算可达高度  
  > 3. 核心逻辑：遍历比较并计数  
  > 4. 输出结果  

**题解一核心片段赏析**  
```cpp
s += !(H < height[i]);
```
* **亮点**：用逻辑运算替代分支  
* **代码解读**：  
  > `H < height[i]`判断是否够不到苹果，`!`取反后：  
  > - 可摘到时：`false→true(1)`  
  > - 否则：`true→false(0)`  
  > 省去`if`分支跳转，提升指令缓存效率  

**题解二核心片段赏析**  
```cpp

    if(a[i]<=ha) ans++;
```
* **亮点**：标准遍历范式

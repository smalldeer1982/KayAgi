# 题目信息

# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

### 3. 核心难点辨析与解题策略

<difficulty_intro>  
解决本题需突破三大关键点：  
</difficulty_intro>

1. **多级关键字优先级设计**  
   * **分析**：年份为最高级关键字（年龄差异最大），月/日为次级关键字。优质题解通过`if`嵌套或布尔表达式实现优先级队列，类似快递分拣"先省份→再市区→最后街道"。  
   * 💡 **学习笔记**：关键字优先级设计是排序问题的通用解题框架。  

2. **同生日逆序输出实现**  
   * **分析**：输入时记录顺序号`num`，排序时若年月日相同，则比较`num`并取大值优先（后输入序号更大）。这本质是第四级关键字的特殊排序规则。  
   * 💡 **学习笔记**：辅助变量是处理特殊排序需求的利器。  

3. **数据结构选择**  
   * **分析**：结构体（`struct`）是整合异构数据的银弹。相比独立数组方案，结构体能保持数据关联性，避免交换时数据不一致问题。  
   * 💡 **学习笔记**：结构体是处理多属性对象的黄金标准。  

### ✨ 解题技巧总结  
- **分层比较法**：用`if-else`链实现关键字优先级（年>月>日>序号）  
- **逆序技巧**：同生日时用`return a.num > b.num`实现后输入优先  
- **数据捆绑**：结构体打包关联数据，避免交换时遗漏字段  
- **边界测试**：特别注意年月日全相同、单数字月份/日期（如2月→02）等临界值

### 4. C++核心代码实现赏析

<code_intro_overall>  
以下是综合优质题解提炼的通用实现：  
</code_intro_overall>

**本题通用核心C++实现参考**  
* **说明**：综合分层比较与结构体存储的最佳实践，完整覆盖题目要求。  
* **完整核心代码**：  
  ```cpp
  #include <iostream>
  #include <algorithm>
  using namespace std;

  struct Student {
      string name;
      int year, month, day, id; // id记录输入顺序
  };

  bool cmp(Student a, Student b) {
      if (a.year != b.year) return a.year < b.year;     // 年优先
      if (a.month != b.month) return a.month < b.month; // 再比月
      if (a.day != b.day) return a.day < b.day;         // 再比日
      return a.id > b.id;  // 同生日：后输入优先（id大者排前）
  }

  int main() {
      int n;
      Student stu[105];
      cin >> n;
      for (int i = 0; i < n; i++) {
          cin >> stu[i].name >> stu[i].year >> stu[i].month >> stu[i].day;
          stu[i].id = i;  // 记录输入顺序
      }
      sort(stu, stu + n, cmp);
      for (int i = 0; i < n; i++) 
          cout << stu[i].name << endl;
      return 0;
  }
  ```
* **代码解读概要**：  
  > ① 结构体整合学生数据 ② 输入时记录顺序号 ③ 比较函数实现三级关键字+逆序规则 ④ `sort`一键排序

优质题解核心代码亮点分析：

**题解一核心片段**  
* **亮点**：分层条件全覆盖，逻辑如同流水线  
* **核心代码**：  
  ```cpp
  if(a.n < b.n) return true;    // 年小优先
  else if(a.n > b.n) return false;
  else {                        // 年同则比月
      if(a.y < b.y) return true;
      ... // 逐级比较
  }
  ```
* **代码解读**：  
  > 像多层筛网：先筛年份→再筛月份→最后筛日期。每层用`if-else`严格分流，确保不漏判任一情况。  
* 💡 **学习笔记**：条件分流是复杂判断的可靠方案。  

**题解二核心片段**  
* **亮点**：`stable_sort`+顺序号实现稳定排序  
* **核心代码**：  
  ```cpp
  stable_sort(stu, stu+n, [](Student a, Student b){
      return tie(a.year, a.month, a.day) < tie(b.year, b.month, b.day);
  }); // 实际需补充顺序号比较
  ```
* **代码解读**：  
  > `stable_sort`保持同值元素原始顺序，配合顺序号实现逆序输出。`tie`函数（需C++11）优雅捆绑关键字。  
* 💡 **学习笔记**：稳定排序是处理顺序敏感的利器。  

**题解三核心片段**  
* **亮点**：单行布尔表达式实现优先级  
* **核心代码**：  
  ```cpp
  return a.y<b.y || (a.y==b.y && a.m<b.m) || 
         (a.y==b.y && a.m==b.m && a.d<b.d) || 
         (a.y==b.y && a.m==b.m && a.d==b.d && a.id>b.id);
  ```
* **代码解读**：  
  > 逻辑或`||`的短路特性实现关键字级联比较。如电路开关：前级满足就短路后续判断。  
* 💡 **学习笔记**：布尔短路是代码简化的妙招。

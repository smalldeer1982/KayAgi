# 题目信息

# 足球

## 题目描述

我们当中有很多热爱中国足球的同学，我们都知道中超（中国足球超级联赛）的规则：

一场比赛中，若获胜（即你的得分严格大于对手得分）则获得 $3$ 的积分，若打平（即你的得分等于对手得分）则获得 $1$ 分，若失败（即你的得分严格小于对手得分）获得 $0$ 积分。

这个问题很简单，假设 $N$ 轮比赛中你一共攻入 $S$ 个球，丢掉 $T$ 个球，那么你可能获得的最大得分和最小得分是多少？

## 样例 #1

### 输入

```
1 1 1
1 1 2```

### 输出

```
1 1
3 2
```

#### 3. 核心难点辨析与解题策略
1. **最大得分的进球分配**  
   * **分析**：核心矛盾在于**进球数不足时如何兼顾胜场与平局**。优质解通用策略：  
     - 若`s<n`：用`s`个进球赢`s`场(1:0)，丢球集中到1场(0:t)，其余`n-s-1`场平局(0:0)  
     - 特例`t=0`时：丢球场转为平局需额外+1分  
   * 💡 **学习笔记**：贪心的本质是让每个进球产生最大价值

2. **最小得分的策略权衡**  
   * **分析**：当`s≤t`时需比较两种策略：  
     - 策略A：赢1场(s:0) + 其余场尽量输(0:1)  
     - 策略B：完全避免胜利，通过`t-s`场小负(0:1)消耗丢球  
     *关键公式：`min(3+max(0,n-t-1), max(0,n-t+s))`  
   * 💡 **学习笔记**：最小得分不是盲目认输，要警惕"赢1场+多平"比"全输"得分更低

3. **边界条件的蝴蝶效应**  
   * **分析**：`n=1`时直接决定胜负；`s=0`或`t=0`会改变比赛性质。George1123的嵌套特判证明：  
     - `if(n==1) {...} if(s+t==0) {...}`的防御式编程  
     - 最小得分中`s==1`时需额外处理平局组合  
   * 💡 **学习笔记**：边界是算法的安全带，没系好就会翻车

✨ **解题技巧总结**  
- **资源拆解术**：将总进球/丢球拆解到独立比赛处理  
- **等价转换法**：`t=0`时"输球场"转化为平局  
- **策略比较思维**：最小得分必做`min(策略A,策略B)`双保险

#### 4. C++核心代码实现赏析
**本题通用核心实现参考**  
```cpp
#include <cstdio>
#include <algorithm>
using namespace std;
int main() {
    long long s, t, n;
    while (~scanf("%lld%lld%lld", &s, &t, &n)) {
        long long max_score = 0, min_score = 0;
        
        // 最大得分核心逻辑
        if (s < n) {
            max_score = 3*s + n - s - 1;
            if (t == 0) max_score++; // 无丢球时转换平局
        } else {
            max_score = 3*(n-1);
            if (s - (n-1) > t) max_score += 3;
            else if (s - (n-1) == t) max_score += 1;
        }
        
        // 最小得分核心逻辑
        if (s > t) {
            min_score = 3 + max(0LL, n-1-t);
        } else {
            long long option1 = 3 + max(0LL, n-1-t);
            long long option2 = max(0LL, n - (t - s));
            min_score = min(option1, option2);
        }
        printf("%lld %lld\n", max_score, min_score);
    }
    return 0;
}
```
**代码解读概要**：  
> 双分支结构处理最大/最小得分。最大得分优先分配1球/场获胜，剩余进球/丢球决定末场比赛；最小得分通过`option1`(赢1场+其余输/平)与`option2`(避免胜利)策略比较。`max(0LL,...)`确保负数场景归零。

**题解一（艮鳖肉）片段赏析**  
```cpp
if(s < n) {
    mx += s * 3;        // 每进1球赢1场
    mx += n - s - 1;    // 剩余场次-1场平局
    if(!t) mx++;        // 无丢球特判
} else {
    mx += (n-1)*3;      // 前n-1场全胜
    if(s-(n-1) > t) mx += 3;   // 末场大胜
    else if(s-(n-1)==t) mx++;  // 末场平局
}
```
* **亮点**：流水线式分支处理，t==0特判精准  
* **学习笔记**：-1场平局的设计源于"集中丢球"的贪心思想  

**题解二（quantum11）片段赏析**  
```cpp
printf("%lld %lld\n", 
    s<n ? 3*s+n-s-1+!t : 3*max(n-1,min(n,s-t))+(s-t==n-1),

```

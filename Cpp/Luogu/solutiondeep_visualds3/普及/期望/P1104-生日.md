# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

## 3. 核心难点辨析与解题策略

1.  **多关键字优先级处理**
    * **分析**：必须严格遵循年→月→日的比较链条。优质题解通过嵌套if或逻辑运算符短路特性实现，如`return a.year<b.year || (a.year==b.year && a.month<b.month)...`
    * 💡 **学习笔记**：排序关键字优先级是解决多属性比较的基石。

2.  **相同生日的逆序要求**
    * **分析**：当生日完全相同时，需按输入顺序逆序输出（后输入先输出）。解决方案是在比较函数最后返回`a.id > b.id`
    * 💡 **学习笔记**：输入序号是解决逆序问题的关键变量。

3.  **数据结构选择**
    * **分析**：结构体是最佳选择（如`struct{string name; int y,m,d,id;}`）。个别题解尝试单整数压缩法，虽简洁但牺牲了可读性。
    * 💡 **学习笔记**：结构体能直观映射现实实体属性。

### ✨ 解题技巧总结
- **分层比较法**：用逻辑运算符||分层处理关键字（年→月→日→序号）
- **逆序索引技巧**：输入时记录`id=i`，比较时用`a.id>b.id`实现逆序
- **防御性测试**：测试同年同月同日数据验证逆序逻辑

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Student {
    string name;
    int y, m, d, id; // 年月日 + 输入序号
};

bool cmp(const Student& a, const Student& b) {
    if (a.y != b.y) return a.y < b.y;
    if (a.m != b.m) return a.m < b.m;
    if (a.d != b.d) return a.d < b.d;
    return a.id > b.id; // 生日相同时按序号逆序
}

int main() {
    int n; cin >> n;
    Student s[105];
    for (int i = 0; i < n; i++) {
        cin >> s[i].name >> s[i].y >> s[i].m >> s[i].d;
        s[i].id = i; // 记录输入顺序
    }
    sort(s, s + n, cmp);
    for (int i = 0; i < n; i++) 
        cout << s[i].name << endl;
}
```
**代码解读概要**：通过结构体整合数据，cmp函数实现三级关键字比较+逆序逻辑，sort完成排序。时间复杂度O(n log n)，空间O(n)。

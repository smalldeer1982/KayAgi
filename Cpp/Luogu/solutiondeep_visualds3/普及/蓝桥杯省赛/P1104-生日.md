# 生日

## 题目描述

cjf 君想调查学校 OI 组每个同学的生日，并按照年龄从大到小的顺序排序。但 cjf 君最近作业很多，没有时间，所以请你帮她排序。


## 说明/提示

数据保证，$1<n<100$，$1\leq |s|<20$。保证年月日实际存在，且年份 $\in [1960,2020]$。

## 样例 #1

### 输入

```
3
Yangchu 1992 4 23
Qiujingya 1993 10 13
Luowen 1991 8 1
```

### 输出

```
Luowen
Yangchu
Qiujingya
```

## 3. 核心难点辨析与解题策略

1. **关键点1：多关键字优先级处理**
    * **分析**：正确理解年>月>日的优先级顺序是基础。优质题解通过`if`层级判断或逻辑运算符链实现优先级处理，确保先比较高权重属性。
    * 💡 **学习笔记**：多关键字排序中，高权重属性的比较结果直接决定顺序，无需继续比较低权重属性。

2. **关键点2：相同生日的特殊处理**
    * **分析**：当生日完全相同时需按输入逆序输出。题解通过记录`id/num/level`等输入序号，并在比较函数中补充`a.id > b.id`实现。
    * 💡 **学习笔记**：处理相同主关键字时，次级排序条件可能颠覆初始顺序。

3. **关键点3：排序算法选择与优化**
    * **分析**：虽然数据规模小（n<100），但优质题解均选用O(nlogn)的`sort/stable_sort`而非O(n²)的冒泡排序，体现算法效率意识。
    * 💡 **学习笔记**：STL排序函数能自动选择最优排序策略，比手写排序更可靠高效。

### ✨ 解题技巧总结
- **技巧1：结构体封装数据**：相关属性捆绑存储，避免多数组维护
- **技巧2：STL函数物尽其用**：善用`sort`自定义比较，复杂逻辑用`stable_sort`
- **技巧3：输入顺序记录**：通过简单计数器解决"后输入先输出"需求
- **技巧4：条件表达式优化**：用逻辑运算符链简化多条件判断

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：综合优质题解最佳实践，使用结构体封装数据+sort自定义比较
* **完整核心代码**：
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Student {
    string name;
    int year, month, day;
    int order; // 输入顺序记录
};

bool cmp(const Student &a, const Student &b) {
    if (a.year != b.year) return a.year < b.year;       // 年优先
    if (a.month != b.month) return a.month < b.month;   // 次年同月比日
    if (a.day != b.day) return a.day < b.day;            // 同年同月比日
    return a.order > b.order;                           // 生日相同按输入逆序
}

int main() {
    int n;
    cin >> n;
    Student s[105];
    
    for (int i = 0; i < n; i++) {
        cin >> s[i].name >> s[i].year >> s[i].month >> s[i].day;
        s[i].order = i; // 记录输入顺序
    }
    
    sort(s, s + n, cmp);
    
    for (int i = 0; i < n; i++) 
        cout << s[i].name << endl;
    
    return 0;
}
```
* **代码解读概要**：
  > 1. 结构体统一管理学生信息
  > 2. `order`字段记录输入顺序
  > 3. `cmp`函数实现年>月>日>输入顺序的四级排序
  > 4. `sort`调用自定义比较函数完成高效排序

---

**题解一：（_jimmywang_）**
* **亮点**：严格遵循题目要求的优先级顺序，变量命名简洁明确
* **核心代码片段**：
```cpp
bool cmp(node a, node b) {
    if (a.n != b.n) return a.n < b.n;
    if (a.y != b.y) return a.y < b.y;
    if (a.r != b.r) return a.r < b.r;
    return a.num > b.num;  // 生日相同则输入序号大的优先
}
```
* **代码解读**：
  > 通过三级`if`判断实现年>月>日优先级处理。当年月日全部相等时，通过`a.num > b.num`确保后输入的学生先输出。这种分层写法逻辑清晰易调试。
  
* 💡 **学习笔记**：多关键字排序中，高权重属性判断应放在前面，一旦得出结果立即返回。

**题解二：（JJJJones_Zhu）**
* **亮点**：使用stable_sort保证相同生日元素的原始顺序
* **核心代码片段**：
```cpp
bool cmp(node a, node b) {
    if (a.year != b.year) return a.year < b.year;
    if (a.mon != b.mon) return a.mon < b.mon;
    if (a.day == b.day) return a.level > b.level; 
    return a.day < b.day;
}
```
* **代码解读**：
  > 在年月相同后直接处理日相等的情况，避免冗余判断。使用`stable_sort`保证在非稳定排序场景下，相同生日学生的原始输入顺序得以保留，符合题目要求。

* 💡 **学习笔记**：当需要保持相同关键字的元素原始顺序时，优先选用stable_sort。

**题解三：（Heartlessly）**
* **亮点**：单行逻辑运算符链实现多条件判断
* **核心代码片段**：
```cpp
bool cmp(opp x, opp y) {
    return x.year < y.year 
        || (x.year == y.year && x.month < y.month)
        || (x.year == y.year && x.month == y.month && x.day < y.day)
        || (x.year == y.year && x.month == y.month && x.day == y.day && x.number > y.number);
}
```
* **代码解读**：
  > 通过逻辑或`||`运算符实现条件瀑布流：当前面条件为真时直接返回结果，不再评估后续条件。这种写法紧凑高效，但需注意条件顺序必须与权重一致。

* 💡 **学习笔记**：逻辑运算符链可大幅精简多条件判断代码，但需确保条件顺序正确。

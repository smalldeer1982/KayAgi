# 题目信息

# 01迷宫

## 题目描述

有一个仅由数字 $0$ 与 $1$ 组成的 $n \times n$ 格迷宫。若你位于一格 $0$ 上，那么你可以移动到相邻 $4$ 格中的某一格 $1$ 上，同样若你位于一格 $1$ 上，那么你可以移动到相邻 $4$ 格中的某一格 $0$ 上。

你的任务是：对于给定的迷宫，询问从某一格开始能移动到多少个格子（包含自身）。

## 说明/提示

对于样例，所有格子互相可达。

- 对于 $20\%$ 的数据，$n \leq 10$；
- 对于 $40\%$ 的数据，$n \leq 50$；
- 对于 $50\%$ 的数据，$m \leq 5$；
- 对于 $60\%$ 的数据，$n,m \leq 100$；
- 对于 $100\%$ 的数据，$1\le n \leq 1000$，$1\le m \leq 100000$。

## 样例 #1

### 输入

```
2 2
01
10
1 1
2 2
```

### 输出

```
4
4
```

## 一、题目解读与核心算法识别
### 1. 题目核心
给定一个由0和1组成的`n×n`迷宫，每次可以从0走到相邻（上下左右）的1，或从1走到相邻的0。要求回答`m`次查询，每次查询从某个点出发能走到多少个格子（包括自己）。


### 2. 核心算法：连通块查找（BFS/DFS）
**比喻**：把迷宫中的每个点看作一个人，0和1是两种不同的“身份”。如果两个人能通过“身份交替”（0→1→0→…）互相到达，他们就属于同一个“朋友圈”。同一个朋友圈的人能到达的范围（格子数）是一样的，所以只需要计算每个朋友圈的大小，查询时直接返回对应朋友圈的大小即可。

**算法流程**：
- **步骤1**：遍历迷宫中的每个点，若未被标记，则用BFS/DFS遍历所有可达的点（满足相邻且值不同），形成一个连通块。
- **步骤2**：给连通块内的所有点标记同一个“编号”，并记录该连通块的大小（格子数）。
- **步骤3**：处理查询时，找到该点所属的连通块，直接返回其大小。


### 3. 关键难点与解决方案
- **难点1**：如何高效处理大量查询？  
  解决方案：标记连通块，避免每次查询都重新遍历。时间复杂度从`O(m×n²)`（暴力查询）优化到`O(n² + m)`（预处理连通块+查询）。
- **难点2**：如何判断两点是否属于同一连通块？  
  解决方案：用二维数组`vis`记录每个点所属的连通块编号，用另一个数组`size`记录每个连通块的大小。
- **难点3**：如何遍历连通块？  
  解决方案：使用BFS（队列）或DFS（递归/栈），遍历所有满足条件的点（相邻且值不同）。

## 三、核心难点辨析与解题策略
### 1. 难点1：理解连通块的条件
**问题**：为什么相邻且值不同的点属于同一连通块？  
**分析**：比如，点A（0）可以走到点B（1），点B（1）可以走到点C（0），所以A、B、C属于同一连通块。连通块的条件是“可达性”，而不是“值相同”。  
**策略**：遍历每个点时，只走相邻且值不同的点。


### 2. 难点2：高效处理大量查询
**问题**：如果每次查询都重新遍历，时间复杂度太高怎么办？  
**分析**：`m`次查询，每次遍历`n²`个点，时间复杂度是`O(m×n²)`，对于`n=1000`、`m=100000`，会超时。  
**策略**：预处理连通块，标记每个点所属的连通块，查询时直接返回大小，时间复杂度`O(n² + m)`。


### 3. 难点3：避免重复计算
**问题**：如何避免同一个连通块被多次遍历？  
**分析**：用`vis`数组标记已访问的点，遍历过的点不再处理。  
**策略**：遍历迷宫时，若当前点未被标记，则进行BFS/DFS遍历其连通块，并标记所有点。


## 四、C++核心代码实现赏析
### 1. 通用核心代码（BFS+连通块标记）
**说明**：综合优质题解，选取最简洁的BFS实现。  
**完整代码**：
```cpp
#include <iostream>
#include <queue>
#include <cstring>
using namespace std;

const int MAXN = 1001;
char maze[MAXN][MAXN];
int vis[MAXN][MAXN]; // 记录连通块编号
int size[MAXN * MAXN]; // 记录每个连通块的大小
int n, m, cnt = 0; // cnt为连通块编号

struct Node {
    int x, y;
    Node(int x, int y) : x(x), y(y) {}
};

int dx[] = {0, 0, 1, -1};
int dy[] = {1, -1, 0, 0};

void bfs(int x, int y) {
    queue<Node> q;
    q.push(Node(x, y));
    vis[x][y] = ++cnt;
    size[cnt] = 1;
    while (!q.empty()) {
        Node cur = q.front();
        q.pop();
        for (int i = 0; i < 4; i++) {
            int nx = cur.x + dx[i];
            int ny = cur.y + dy[i];
            if (nx >= 1 && nx <= n && ny >= 1 && ny <= n && !vis[nx][ny] && maze[cur.x][cur.y] != maze[nx][ny]) {
                vis[nx][ny] = cnt;
                size[cnt]++;
                q.push(Node(nx, ny));
            }
        }
    }
}

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        cin >> maze[i] + 1; // 从索引1开始存储
    }
    memset(vis, 0, sizeof(vis));
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (!vis[i][j]) {
                bfs(i, j);
            }
        }
    }
    while (m--) {
        int x, y;
        cin >> x >> y;
        cout << size[vis[x][y]] << endl;
    }
    return 0;
}
```


### 2. 代码解读
- **输入处理**：读取迷宫大小`n`和查询次数`m`，然后读取迷宫数据（从索引1开始存储，方便处理边界）。
- **BFS遍历**：对于每个未标记的点，进行BFS。队列存储当前点，遍历四个方向，满足条件（在边界内、未标记、值不同）的点入队，标记连通块编号，并更新连通块大小。
- **查询处理**：读取查询点，找到其所属的连通块编号，返回对应大小。

# 题目信息

# 足球

## 题目描述

我们当中有很多热爱中国足球的同学，我们都知道中超（中国足球超级联赛）的规则：

一场比赛中，若获胜（即你的得分严格大于对手得分）则获得 $3$ 的积分，若打平（即你的得分等于对手得分）则获得 $1$ 分，若失败（即你的得分严格小于对手得分）获得 $0$ 积分。

这个问题很简单，假设 $N$ 轮比赛中你一共攻入 $S$ 个球，丢掉 $T$ 个球，那么你可能获得的最大得分和最小得分是多少？

## 样例 #1

### 输入

```
1 1 1
1 1 2```

### 输出

```
1 1
3 2
```

### 题目解读与核心算法识别
✨ **本题主要考察**：`贪心策略与分类讨论`

🗣️ **初步分析**：
> 解决“足球积分”问题，关键在于运用**贪心策略**。就像足球教练排兵布阵，我们要在进球(s)和丢球(t)的约束下，通过合理安排每场比赛的胜负，使积分最大化或最小化。核心策略是：
> - **最大积分**：优先分散进球以多赢比赛（1:0），将丢球集中以减少输球场次
> - **最小积分**：避免不必要的胜利（因胜1场=3分>平2场=2分），灵活组合平局和输球
> 
> **难点对比**：最大积分侧重"分散进球+集中丢球"，最小积分需权衡"赢1场+其余输/平"与"全输/平"两种策略。多解法的差异主要在最小积分的策略B（无胜场）的数学建模上。
> 
> **可视化设计**：采用复古像素足球游戏风格，用三种动画元素：
> 1. 胜场：绿草皮+像素球员射门动画（8-bit音效）
> 2. 平局：黄牌闪烁+哨声音效
> 3. 输球：红牌+观众嘘声
> 关键步骤高亮：进球分配时显示数字气泡，集中丢球时用"黑洞动画"吞噬足球

### 核心难点辨析与解题策略
1. **最大积分的丢球集中化**  
   *分析*：当s<n时，s个1:0胜场后，需将t个丢球集中到1场。若t=0则该场变平局（积分+1）。优质题解用`if(!t) mx++`统一处理  
   💡 **学习笔记**：集中丢球时，t=0是隐藏的边界炸弹！

2. **最小积分的双策略博弈**  
   *分析*：s≤t时存在两种可能：(A)赢1场(3分)+其余输/平 (B)避免胜利，用"0:1输球+0:0平局"组合。策略B的得分公式`max(0,n-t+s)`源自净输球约束（总输球场≤t-s）  
   💡 **学习笔记**：当s=2,t=3,n=3时，策略B需2:2平+0:1输=1分，验证公式正确性

3. **进球过剩的分配艺术**  
   *分析*：当s≥n时，前n-1场1:0获胜后，末场需比较剩余进球(s-n+1)与t：  
   - 多则胜（+3分）  
   - 等则平（+1分）  
   - 少则输（不增分）  
   💡 **学习笔记**：贪心不是全胜！要保留"进球库存"应对末场

### ✨ 解题技巧总结
- **问题分解术**：将s,t,n拆解为单场比赛决策，用"每场至少1球"保证分配完整性  
- **边界防御编程**：特判n=1/s=0/t=0等极值，用`if(n==1){...}`隔离  
- **策略对比法**：最小积分时显式计算双策略`min(strategy_A, strategy_B)`  
- **数学转化技巧**：将足球规则转化为`max(0, n-t+s)`等线性表达式

### C++核心代码实现赏析
**本题通用核心实现（综合优化版）**
```cpp
#include <iostream>
using namespace std;
int main() {
    long long s, t, n;
    while (cin >> s >> t >> n) {
        long long max_score = 0, min_score = 0;
        // 最大积分
        if (s < n) {
            max_score = 3 * s + (n - s - 1);
            if (t == 0) max_score++;
        } else {
            max_score = 3 * (n - 1);
            if (s - n + 1 > t) max_score += 3;
            else if (s - n + 1 == t) max_score++;
        }
        // 最小积分
        if (s > t) {
            min_score = 3 + max(0LL, n - 1 - t);
        } else {
            long long strategy_A = 3 + max(0LL, n - 1 - t);
            long long strategy_B = max(0LL, n - t + s);
            min_score = min(strategy_A, strategy_B);
        }
        cout << max_score << " " << min_score << endl;
    }
    return 0;
}
```
**代码解读概要**：  
> 1. 最大积分分支：s<n时构造s胜+1负+(n-s-1)平；s≥n时先取n-1胜，末场根据进球余量决策  
> 2. 最小积分分支：s>t时必赢1场+其余输/平；s≤t时双策略取优  
> 3. 防御性：`max(0LL, ...)`防负数，`long long`防溢出  

**题解一（艮鳖肉）片段赏析**  
```cpp
if (s < n) {
    mx += s * 3;       // s场胜利
    mx += n - s - 1;   // 平局场次
    if (!t) mx++;      // 无丢球时负场变平局
}
```
**亮点**：直白展现"胜+平+边界修正"三层结构  
**学习笔记**：`!t`的妙用——用逻辑表达式替代`t==0`更简洁  

**题解二（quantum11）片段赏析**  
```cpp
printf("%lld %lld\n", 
    s<n ? 3*s+n-s-1+!t : 3*max(n-1,min(n,s-t))+(s-t==n-1),
    s>t ? 3+max(0ll,n-t-1) : min(3+max(0ll,n-t-1), max(0ll,n-t+s))
);
```
**亮点**：三目运算符嵌套实现分支策略  
**学习笔记**：`!t`隐式类型转换——`true=1`可数学运算

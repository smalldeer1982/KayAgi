# [NOIP 2011 普及组] 数字反转

## 题目描述

给定一个整数 $N$，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例 2）。


## 说明/提示

**【数据范围】**

$-1,000,000,000\leq N\leq 1,000,000,000 $。

noip2011 普及组第一题


## 样例 #1

### 输入

```
123```

### 输出

```
321```

## 样例 #2

### 输入

```
-380```

### 输出

```
-83```

## 3. 核心难点辨析与解题策略

1.  **负号处理与绝对值转换**
    * **分析**：如-380需先取绝对值380再反转。优质解法都用`if(n<0){n=-n; cout<<'-'}`分离符号与数值
    * 💡 **学习笔记**：负号仅影响输出形式，不影响反转操作本身

2.  **前导零的智能去除**
    * **分析**：数学解法通过`while(n)`循环自动跳过末尾零（如380%10=0时不累加到新数）。字符串解法需`find_first_not_of('0')`显式处理
    * 💡 **学习笔记**：整数运算中083就是83，无需额外操作

3.  **反转过程的数学建模**
    * **分析**：核心公式`新数=新数*10 + 原数%10`。关键在理解`*10`实质是左移数位（十位变百位）
    * 💡 **学习笔记**：数字反转本质是进制位的重新组合

### ✨ 解题技巧总结
- **技巧A：数学取位法**：优先使用`%10`和`/10`组合，避免字符串转换开销
- **技巧B：循环边界处理**：用`while(n)`自动处理所有有效数位
- **技巧C：零值特判**：单独处理输入为0的情况（直接输出0）
- **技巧D：变量初始化**：反转结果变量初始化为0（如`long long ans=0`）

## 4. C++核心代码实现赏析

**本题通用核心C++实现参考**
* **说明**：综合三大优质解法优化的标准实现，完整处理负号/零值/边界
* **完整核心代码**：
```cpp
#include <iostream>
using namespace std;
int main() {
    long long n;
    cin >> n;
    if (n == 0) {
        cout << 0;
        return 0;
    }
    bool negative = n < 0;
    long long reversed = 0;
    for (n = abs(n); n != 0; n /= 10) 
        reversed = reversed * 10 + n % 10;
    cout << (negative ? "-" : "") << reversed;
    return 0;
}
```
* **代码解读概要**：先特判0值→取绝对值→循环取末位构建新数→按需添加负号。`abs(n)`确保负数处理正确，`for`循环紧凑完成反转。

---

**题解一：stone_juice石汁**
* **亮点**：极致简洁，循环与输入融合
* **核心代码片段**：
```cpp
for(std::cin>>numa; numa!=0; numa/=10)
    numb = numb*10 + numa%10;
```
* **代码解读**：
  > 1. `std::cin>>numa`：在循环初始化中直接读入
  > 2. `numa!=0`：持续到所有有效位处理完
  > 3. `numa/=10`：每次移除末位
  > 4. `numb*10 + numa%10`：左移新数并添加新位
* 💡 **学习笔记**：用`for`将输入/条件/更新三要素整合，是C++高效编码典范

**题解二：瑞星**
* **亮点**：显式处理负号和末尾零
* **核心代码片段**：
```cpp
if(n<0) {cout<<"-"; n=-n;}
while(n!=0) {
    int k = n%10;
    sum = sum*10 + k;
    n = n/10;
}
```
* **代码解读**：
  > 1. 先分离负号保证后续操作正数
  > 2. `k=n%10`：暂存末位值
  > 3. `sum*10+k`：经典反转公式
* 💡 **学习笔记**：显式分离关注点（符号处理/数值反转）提升可读性

**题解三：周羿轩**
* **亮点**：最小实现展示算法本质
* **核心代码片段**：
```cpp
while(n) 
    s = s*10 + n%10, n /= 10;
```
* **代码解读**：
  > 1. `while(n)`：等效`n!=0`的更简写法
  > 2. 逗号运算符：单行内执行两个操作
* 💡 **学习笔记**：精简代码时需平衡可读性，避免过度压缩

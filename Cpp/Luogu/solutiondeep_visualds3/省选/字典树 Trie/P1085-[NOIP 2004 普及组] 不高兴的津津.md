* **完整核心代码**：
    ```cpp
    #include <iostream>
    using namespace std;
    int main() {
        int maxTime = 8; // 初始值设为临界值8
        int ansDay = 0;  // 答案初始为0（无解情况）
        for (int day = 1; day <= 7; day++) {
            int school, extra;
            cin >> school >> extra;
            int total = school + extra;
            // 严格大于确保取首个最大值
            if (total > maxTime) {
                maxTime = total;
                ansDay = day;
            }
        }
        cout << ansDay << endl;
        return 0;
    }
    ```
* **代码解读概要**：
    > 代码循环7次处理每天数据，计算总时间后与maxTime比较。当total严格大于maxTime时更新最大值和对应天数。初始maxTime=8确保只记录超过8小时的情况，ansDay=0自动处理无解情况。

* **亮点**：初始值设置精妙
* **核心代码片段**：
    ```cpp
    int mx=8,pos=0;
    for(int i=1;i<=7;i++){
        int a,b;cin>>a>>b;
        if(mx<a+b)mx=a+b,pos=i;
    }

    ```
* **代码解读**：
    > 为什么mx初始设为8？这样当a+b>mx时，自然满足a+b>8的条件。pos初始为0，若循环中未更新则输出0，完美处理无解情况。注意条件判断合并为单语句提升效率。
* 💡 **学习笔记**：初始值可作为隐含条件判断

**题解二：(来源：zhuoqizhi)**
* **亮点**：易错点警示明确
* **核心代码片段**：
    ```cpp
    int max=8,maxi=0;
    for(int i=1;i<=7;i++){
        cin>>t1>>t2;
        if(max<t1+t2){
            max=t1+t2;
            maxi=i;
        }
    }
    ```
* **代码解读**：
    > 关键在条件判断使用严格小于(等效于t1+t2>max)，这保证当出现相同最大值时不更新maxi，从而保留第一个最大值。若误用<=，则会记录最后出现的最大值，违反题目要求。
* 💡 **学习笔记**：极值更新条件决定记录顺序

**题解三：(来源：IMGCsir)**

    for(int i=1;i<=7;i++){
        scanf("%d%d",&n,&m);
        a[i]=n+m;

## 3. 核心难点辨析与解题策略

1. **难点1：如何记录有效最大值**  
   *分析*：必须同时满足两个条件：①超过8小时 ②大于当前记录。优质解用`if(total>8 && total>maxVal)`实现双过滤  
   💡 **学习笔记**：条件顺序很重要！先检查>8可避免无效比较  

2. **难点2：如何处理相同最大值**  
   *分析*：使用`严格大于(>)`而非`大于等于(>=)`更新，确保首次出现的最大值被保留  
   💡 **学习笔记**：相等时不更新——这是取最早出现日的关键技巧  

3. **难点3：无超时日的特殊输出**  
   *分析*：通过初始化技巧处理（`maxVal初始≤0`或`pos=0`），最终输出该标记值  
   💡 **学习笔记**：初始值设计应包含"无解"语义  

### ✨ 解题技巧总结
- **状态压缩**：用`pos=0`同时表示"无解"和"位置"双重语义  
- **实时更新**：边输入边处理，节省存储空间  
- **防御性编程**：初始值设为无效值（-1或0）明确状态  
- **严格不等式**：处理多解时优先使用`>`而非`>=`

## 4. C++核心代码实现赏析

**通用核心实现**  
```cpp
#include <iostream>
using namespace std;
int main() {
    int maxVal = 0, pos = 0; // maxVal初始0，pos初始0(表无解)
    for(int day=1; day<=7; day++){
        int school, extra;
        cin >> school >> extra;
        int total = school + extra;
        if(total > 8 && total > maxVal) {
            maxVal = total;
            pos = day; // 仅当严格大于时更新
        }
    }
    cout << pos; // 无解时输出初始值0
    return 0;
}
```
* **代码解读**：  
  - `maxVal`记录有效最大值（始终>8）  
  - `pos`双重功能：0表示无解，1-7表示具体日期  
  - 条件`total>maxVal`（非≥）确保最早出现日优先

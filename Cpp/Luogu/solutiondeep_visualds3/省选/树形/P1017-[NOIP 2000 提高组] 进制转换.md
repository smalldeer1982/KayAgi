## 题目背景

NOIP2000 提高组 T1

## 题目描述

我们可以用这样的方式来表示一个十进制数：将每个阿拉伯数字乘以一个以该数字所处位置为指数，以 $10$ 为底数的幂之和的形式。例如 $123$ 可表示为 $1 \times 10^2+2\times 10^1+3\times 10^0$ 这样的形式。

与之相似的，对二进制数来说，也可表示成每个二进制数码乘以一个以该数字所处位置为指数，以 $2$ 为底数的幂之和的形式。  

一般说来，任何一个正整数 $R$ 或一个负整数 $-R$ 都可以被选来作为一个数制系统的基数。如果是以 $R$ 或 $-R$ 为基数，则需要用到的数码为 $0,1,\dots,R-1$。  

例如当 $R=7$ 时,所需用到的数码是 $0,1,2,3,4,5,6$，这与其是 $R$ 或 $-R$ 无关。如果作为基数的数绝对值超过 $10$，则为了表示这些数码，通常使用英文字母来表示那些大于 $9$ 的数码。例如对 $16$ 进制数来说,用 $A$ 表示 $10$，用 $B$ 表示 $11$，用 $C$ 表示 $12$，以此类推。

在负进制数中是用 $-R $ 作为基数，例如 $-15$（十进制）相当于 $(110001)_{-2}$（$-2$进制），并且它可以被表示为 $2$ 的幂级数的和数：

$$(110001)_{-2}=1\times (-2)^5+1\times (-2)^4+0\times (-2)^3+0\times (-2)^2+0\times (-2)^1 +1\times (-2)^0$$

设计一个程序，读入一个十进制数和一个负进制数的基数，并将此十进制数转换为此负进制下的数。

## 1. 题目解读与核心算法识别

✨ **本题主要考察**：编程技巧应用（负进制转换的模拟）  

🗣️ **初步分析**：  
解决“负进制转换”问题，核心是**模拟短除法**——就像剥洋葱一样，一层一层“剥”出每一位数字。只不过，负进制的“洋葱皮”有点特殊：当除数是负数时，余数可能会是负数，这时候需要“调整”余数，让它变成正数（因为数字中不能有负号哦！）。  

举个例子，把-15转换成-2进制：  
正常短除法会得到余数-1（-15 ÷ -2 = 7余-1），但-1不能作为数字位。这时候我们需要**调整**：余数减去除数（-1 - (-2) = 1），同时商加1（7+1=8）。这样，-15 = (-2)×8 + 1，余数1就是合法的数字位啦！  

**核心算法流程**：  
1. 用十进制数`n`除以负进制基数`r`，得到余数`m`。  
2. 如果`m`是负数，调整：`m -= r`（变成正数），`n += r`（商加1）。  
3. 把`m`转换成字符（0-9或A-F），存储起来。  
4. 用调整后的商`n/r`重复上述步骤，直到`n`为0。  
5. 倒序输出存储的字符（因为短除法得到的是从低位到高位的数字）。  

**可视化设计思路**：  
我们可以做一个“像素洋葱剥壳机”动画：  
- 屏幕左侧用像素块显示当前的`n`（比如-15用蓝色方块），右侧显示`r`（-2用红色方块）。  
- 每次“剥壳”时，用动画显示`n`除以`r`的过程：比如`n`的方块分裂成`n/r`（商）和`m`（余数）。  
- 如果`m`是负数（比如-1用灰色方块），会触发“调整”动画：`m`的方块变成1（绿色），`n`的方块变成8（蓝色），同时播放“叮”的音效。  
- 所有余数会被存入一个“像素栈”，最后从栈顶倒序输出，形成最终的负进制数。

## 3. 核心难点辨析与解题策略

### 1. 难点1：理解负进制的表示方式  
**问题**：负进制的每一位数字是什么意思？比如`(110001)_{-2}`为什么等于-15？  
**分析**：负进制的每一位权重是`r^k`（`r`是基数，`k`是位置）。比如`(110001)_{-2}`=1×(-2)^5 + 1×(-2)^4 + 0×(-2)^3 + 0×(-2)^2 + 0×(-2)^1 + 1×(-2)^0 = -32 + 16 + 0 + 0 + 0 + 1 = -15。只要记住“权重是负数的幂”，就能理解负进制的表示。  
💡 **学习笔记**：负进制的本质是“用负数的幂之和表示数”，每一位数字是0到|r|-1之间的正数。  

### 2. 难点2：处理负余数  
**问题**：C++中，负数取余的结果是负数（比如-15%-2=-1），这会导致输出错误。  
**分析**：根据除法的定义，`被除数 = 商×除数 + 余数`，且`余数的绝对值 < 除数的绝对值`。当余数为负时，我们可以调整：`余数 = 余数 - 除数`（因为除数是负的，所以余数变成正数），`商 = 商 + 1`。比如-15 = (-2)×8 + 1，这里余数1是合法的。  
💡 **学习笔记**：负余数调整的公式是：`if(余数<0) 余数-=除数，商+1`。  

### 3. 难点3：倒序输出结果  
**问题**：短除法得到的余数是从低位到高位的（比如30000转-2进制，第一次余数是0，对应最低位），需要倒序输出。  
**分析**：可以用**递归**（递归处理商，最后输出余数）或**栈**（把余数压入栈，最后弹出）或**数组**（把余数存到数组，最后倒序遍历）。递归是最简洁的方式，因为递归的“回溯”过程自然实现了倒序。  
💡 **学习笔记**：递归是处理“倒序输出”的神器！  


## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考（递归版）  
* **说明**：综合老卡手机和ikunTLE的题解，采用递归实现，代码简洁，逻辑清晰。  
* **完整核心代码**：  
  ```cpp
  #include <iostream>
  using namespace std;

  void convert(int n, int r) {
      if (n == 0) return; // 递归终止条件：没有数字要处理了
      int m = n % r; // 取余数
      if (m < 0) { // 处理负余数
          m -= r; // 余数变成正数（比如-1 - (-2) = 1）
          n += r; // 商加1（比如-15/-2=7，加1变成8）
      }
      convert(n / r, r); // 递归处理商（下一位）
      // 输出余数（递归回溯时输出，自然倒序）
      if (m < 10) cout << m;
      else cout << (char)('A' + m - 10);
  }

  int main() {
      int n, r;
      cin >> n >> r;
      cout << n << "=";
      if (n == 0) cout << 0; // 特殊情况：n=0时输出0
      else convert(n, r);
      cout << "(base" << r << ")";
      return 0;
  }
  ```
* **代码解读概要**：  
  1. `convert`函数：递归处理每一位数字。参数`n`是当前要处理的十进制数，`r`是负进制基数。  
  2. 终止条件：`n==0`时，不再递归（没有数字要处理了）。  
  3. 取余数：`m = n % r`。  
  4. 调整负余数：如果`m<0`，`m -= r`（变成正数），`n += r`（商加1）。  
  5. 递归处理商：`convert(n/r, r)`（处理下一位）。  
  6. 输出余数：递归回溯时输出，自然倒序（比如先处理高位，最后输出低位）。  


### 题解一（老卡手机）核心代码片段赏析  
* **亮点**：用递归实现倒序输出，代码简洁到极致。  
* **核心代码片段**：  
  ```cpp
  void zhuan(int n,int r) {
      if(n==0) return;
      int m=n%r;
      if(m<0) m-=r,n+=r; // 处理负余数
      zhuan(n/r,r); // 递归处理商
      printf("%c", m>=10 ? 'A'+m-10 : '0'+m); // 输出余数
  }
  ```
* **代码解读**：  
  - 为什么递归处理商之后才输出余数？因为递归的“回溯”过程会先处理高位（比如30000转-2进制，第一次递归处理的是30000/-2=-15000，然后是-15000/-2=7500，直到商为0，然后回溯输出余数，这样余数的顺序是从高位到低位）。  
  - 为什么`m>=10`时用`'A'+m-10`？因为10对应的字符是'A'，11是'B'，依此类推（比如16进制中的表示）。  
* 💡 **学习笔记**：递归的“回溯”过程是实现倒序输出的关键！  


### 题解二（judgejudge）核心代码片段赏析  
* **亮点**：用循环和数组存储余数，适合理解“倒序输出”的逻辑。  
* **核心代码片段**：  
  ```cpp
  char a[100001];
  int l=0;
  while(n!=0){
      int j=n%m;
      n/=m;
      if(j<0) j-=m,n--; // 处理负余数
      if(j<10) a[++l]=(char)(j+48);
      else a[++l]=(char)((j-10)+'A');
  }
  for(int i=l;i>=1;i--) cout<<a[i]; // 倒序输出
  ```
* **代码解读**：  
  - 为什么用数组`a`存储余数？因为循环得到的余数是从低位到高位的（比如30000转-2进制，第一次余数是0，对应最低位），所以需要把余数存到数组里，最后倒序遍历数组，输出高位到低位。  
  - 为什么`j<0`时`n--`？因为`n`已经除以`m`了（比如-15/-2=7），所以`n--`相当于商加1（7+1=8）。  
* 💡 **学习笔记**：循环+数组是处理“倒序输出”的经典方式！

1. **任意进制转换**：比如将十进制数转换成16进制、8进制，只要调整余数的范围（0到|r|-1）即可。

# 题目信息

# 足球

## 题目描述

我们当中有很多热爱中国足球的同学，我们都知道中超（中国足球超级联赛）的规则：

一场比赛中，若获胜（即你的得分严格大于对手得分）则获得 $3$ 的积分，若打平（即你的得分等于对手得分）则获得 $1$ 分，若失败（即你的得分严格小于对手得分）获得 $0$ 积分。

这个问题很简单，假设 $N$ 轮比赛中你一共攻入 $S$ 个球，丢掉 $T$ 个球，那么你可能获得的最大得分和最小得分是多少？

## 样例 #1

### 输入

```
1 1 1
1 1 2```

### 输出

```
1 1
3 2
```

#### 3. 核心难点辨析与解题策略
1. **难点：边界条件处理**
   * **分析**：n=1时需独立处理（胜/平/败）；s=0或t=0时分配策略剧变。优质题解通过前置特判避免逻辑漏洞
   * 💡 **学习笔记**：输入范围检查是分类讨论第一要务

2. **难点：最小得分策略选择**
   * **分析**：当s≤t时，需比较"赢1场+其余输/平"与"全输/平"两种策略。关键在计算max(0,n-t+s)平局场次
   * 💡 **学习笔记**：足球得分非线性（3分>2分），需比较策略而非简单贪心

3. **难点：资源分配模拟**
   * **分析**：用"1:0"最小代价获胜、"0:1"最小代价输球等原子操作组合策略。注意s-(n-1)>t判断隐含进位思想
   * 💡 **学习笔记**：问题转化（进球/丢球→资源分配）是解题突破口

### ✨ 解题技巧总结
- **资源集中法**：丢球集中消耗（最大分）/进球集中使用（最小分）
- **边界优先原则**：优先处理n=1,s=0,t=0等退化情况
- **策略比较思维**：最小分需数学比较双策略得分（min(3+n-1-t, n-t+s)）

#### 4. C++核心代码实现赏析
**通用核心实现参考**
```cpp
#include <iostream>
using namespace std;

int main() {
    long long s, t, n;
    while (cin >> s >> t >> n) {
        long long max_score = 0, min_score = 0;
        
        // 最大得分
        if (s < n) {
            max_score = 3 * s + (n - s - 1);
            if (t == 0) max_score++;
        } else {
            max_score = 3 * (n - 1);
            if (s - (n - 1) > t) max_score += 3;
            else if (s - (n - 1) == t) max_score++;
        }
        
        // 最小得分
        if (s > t) {
            min_score = 3 + max(0LL, n - 1 - t);
        } else {
            long long strategy1 = 3 + max(0LL, n - 1 - t);
            long long strategy2 = max(0LL, n - t + s);
            min_score = min(strategy1, strategy2);
        }
        cout << max_score << " " << min_score << endl;
    }
    return 0;
}
```
**代码解读概要**：
> 输入采用`while(cin>>s>>t>>n)`处理多组数据。最大得分分支处理s<n/s≥n两种情况，其中s<n时考虑t=0特例。最小得分区分s>t时强制获胜策略，s≤t时双策略比较。long long确保大数安全。

**题解一核心片段**（艮鳖肉）
```cpp
if (s < n) {
    mx += s * 3;
    mx += n - s - 1;
    if (!t) mx++;  // 无丢球特判
} else {
    mx += (n - 1) * 3;
    if (s - (n - 1) > t) mx += 3;
    else if (s - (n - 1) == t) mx++;
}
```
**亮点**：直白的分支结构体现原子操作思维  
**解读**：`s<n`时先算胜场得分（3*s），剩余场次默认平局（n-s-1），无丢球时修正最后一场。`s>=n`时前n-1场稳胜，末场根据剩余进球决定胜负。  
💡 **学习笔记**：分支结构是分类讨论的代码具现

**题解二核心片段**（quantum11）
```cpp
printf("%lld %lld\n", 
    s<n ? 3*s+n-s-1+!t : 3*max(n-1,min(n,s-t)) + (s-t==n-1),
    s>t ? 3+max(0ll,n-t-1) : min(3+max(0ll,n-t-1), max(n-t+s,0ll)));
```
**亮点**：单行实现核心逻辑展现表达式抽象能力  
**解读**：三元运算符嵌套实现分支，`+!t`等价于t==0检测，`min(n,s-t)`压缩s≥n的分支。最小分的双策略比较在`: `后完成。  
💡 **学习笔记**：熟练运用条件表达式可大幅提升代码密度

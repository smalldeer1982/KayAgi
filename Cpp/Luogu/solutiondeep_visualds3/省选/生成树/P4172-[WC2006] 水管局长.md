# 题目信息

# [WC2006] 水管局长

## 题目背景

SC 省 MY 市有着庞大的地下水管网络，嘟嘟是 MY 市的水管局长（就是管水管的啦）。

## 题目描述

每天供水公司可能要将一定量的水从 $u$ 处送往 $v$ 处，嘟嘟需要为供水公司找到一条从 $u$ 至 $v$ 的水管的路径，接着通过信息化的控制中心通知路径上的水管进入准备送水状态，等到路径上每一条水管都准备好了，供水公司就可以开始送水了。嘟嘟一次只能处理一项送水任务，等到当前的送水任务完成了，才能处理下一项。

在处理每项送水任务之前，路径上的水管都要进行一系列的准备操作，如清洗、消毒等等。嘟嘟在控制中心一声令下，这些水管的准备操作同时开始，但由于各条管道的长度、内径不同，进行准备操作需要的时间可能不同。

供水公司总是希望嘟嘟能找到这样一条送水路径，路径上的所有管道全都准备就绪所需要的时间尽量短。嘟嘟希望你能帮助他完成这样的一个选择路径的系统，以满足供水公司的要求。另外，由于 MY 市的水管年代久远，一些水管会不时出现故障导致不能使用，你的程序必须考虑到这一点。

不妨将 MY 市的水管网络看作一幅简单无向图（即没有自环或重边）：水管是图中的边，水管的连接处为图中的结点。整张图共有 $n$ 个节点和 $m$ 条边，节点从 $1$ 至 $n$ 编号。


## 说明/提示

#### 数据规模与约定

对于全部的测试点，保证：

- $1 \leq n \leq 10^3$，$1 \leq m, q \leq 10^5$。
- $1 \leq k \leq 2$，$1 \leq u, v \leq n$，$1 \leq t \leq 10^9$。
- 给出的图无重边无自环，保证在宣布一条水管报废之前，该水管一定存在于图上且没有报废，
- 宣布报废的水管不超过 $5 \times 10^3$ 条，且在任意时刻，图一定是联通的。

# AI分析结果


### 题目解读与核心算法识别
**题目核心**：动态维护无向图，支持删除边和查询两点间路径最大边权的最小值。  
**核心算法**：离线处理 + LCT维护最小生成树（MST）。  
**算法标签**：`图论/LCT`  
**比喻解释**：  
将水管网络视为一张城市地图，每条水管是连接两城市的道路，边权是道路维护时间。目标是为供水公司找到一条从起点到终点的路径，使所有道路准备时间的最小值最大（即瓶颈最小）。离线逆序处理操作（删边变加边），相当于时间倒流逐步修复道路，用LCT动态维护最优路径网络（最小生成树）。

**初步分析**：
- **核心思想**：离线逆序操作，删边转为加边。用LCT维护最小生成树，查询时返回路径最大边权。
- **难点**：需处理边权（LCT默认维护点权），通过边转点技巧解决（每条边转为新节点）。
- **可视化设计**：采用8位像素风格，动态展示最小生成树变化（详见第5节）。

### 核心难点辨析与解题策略
#### 难点1：边权处理  
**问题**：LCT仅支持点权，需将边权转化为点权。  
**解决方案**：  
- 为每条边`(u,v)`创建节点`p`（编号`i+n`），权值为边权。  
- 连接：`link(u,p)`和`link(v,p)`，断开同理。

#### 难点2：动态维护最小生成树  
**问题**：加边可能破坏MST最优性。  
**策略**：  
1. 若`u,v`不连通，直接加边。  
2. 若已连通：  
   - 查询路径最大边权`max_val`。  
   - 若新边权`w < max_val`，删除原最大边，加入新边。  

#### 难点3：离线处理与逆序初始化  
**问题**：需跳过被删边构建初始MST。  
**步骤**：  
1. 标记所有被删边。  
2. 按边权升序排序，用并查集构建初始MST（跳过被删边）。  
3. 逆序处理操作（加边/查询）。

**学习笔记**：  
- 离线逆序是动态图问题的通用技巧。  
- LCT的`split`操作可高效查询路径最大值。

### C++核心代码实现赏析
#### 通用核心实现（基于优质题解整合）
```cpp
#include <cstdio>
#include <algorithm>
#include <map>
#define lc ch[x][0]
#define rc ch[x][1]
using namespace std;
const int N = 1e5 + 10, M = 1e6 + 10;

struct Edge { int u, v, w; } e[M];
struct Query { int op, x, y; } q[M];
int n, m, Q, ans[M];
map<pair<int, int>, int> edgeId; // 边(u,v)映射到ID

namespace LCT {
    int ch[M][2], fa[M], val[M], mx[M], rev[M];
    bool isroot(int x) { return ch[fa[x]][0]!=x && ch[fa[x]][1]!=x; }
    void pushup(int x) {
        mx[x] = x;
        if (val[mx[lc]] > val[mx[x]]) mx[x] = mx[lc];
        if (val[mx[rc]] > val[mx[x]]) mx[x] = mx[rc];
    }
    void pushdown(int x) {
        if (rev[x]) swap(lc, rc), rev[lc] ^= 1, rev[rc] ^= 1, rev[x] = 0;
    }
    void rotate(int x) { /* 标准LCT旋转 */ }
    void splay(int x) { /* 标准LCT伸展 */ }
    void access(int x) { for (int y = 0; x; y = x, x = fa[x]) splay(x), rc = y, pushup(x); }
    void makeroot(int x) { access(x), splay(x), rev[x] ^= 1; }
    int findroot(int x) { access(x), splay(x); while (lc) x = lc; return x; }
    void split(int x, int y) { makeroot(x), access(y), splay(y); }
    void link(int x, int y) { makeroot(x), fa[x] = y; }
    void cut(int x, int y) { split(x, y), fa[x] = ch[y][0] = 0, pushup(y); }
    int query(int x, int y) { split(x, y); return val[mx[y]]; }
}

void init() {
    sort(e + 1, e + m + 1, [](Edge a, Edge b) { return a.w < b.w; });
    for (int i = 1; i <= m; i++) {
        auto p = make_pair(min(e[i].u, e[i].v), max(e[i].u, e[i].v));
        edgeId[p] = i; // 建立边到ID的映射
        LCT::val[i + n] = e[i].w; // 边转为点
    }
}

void solve() {
    // 逆序处理操作
    for (int i = Q; i; i--) {
        if (q[i].op == 1) {
            ans[i] = LCT::query(q[i].x, q[i].y);
        } else {
            auto p = make_pair(min(q[i].x, q[i].y), max(q[i].x, q[i].y));
            int id = edgeId[p];
            LCT::split(q[i].x, q[i].y);
            int maxId = LCT::mx[q[i].y]; // 最大边对应节点
            if (e[id].w < LCT::val[maxId]) {
                // 替换边
                LCT::cut(e[maxId - n].u, maxId);
                LCT::cut(e[maxId - n].v, maxId);
                LCT::link(q[i].x, id + n);
                LCT::link(q[i].y, id + n);
            }
        }
    }
}
```

#### 代码片段赏析
**题解一的核心片段（边转点与LCT维护）**：
```cpp
void link_edge(int i) {
    int u = e[i].u, v = e[i].v;
    if (LCT::findroot(u) != LCT::findroot(v)) {
        LCT::link(u, i + n);
        LCT::link(v, i + n);
    } else {
        LCT::split(u, v);
        int maxId = LCT::mx[v]; // 获取最大边
        if (e[i].w < LCT::val[maxId]) {
            // 替换操作
            LCT::cut(e[maxId - n].u, maxId);
            LCT::cut(e[maxId - n].v, maxId);
            LCT::link(u, i + n);
            LCT::link(v, i + n);
        }
    }
}
```
**学习笔记**：  
- **边转点**：边`i`转为点`i+n`，权值赋给`val[i+n]`。  
- **动态维护**：加边时比较新旧边权，保证MST始终最优。

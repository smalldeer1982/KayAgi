## 4. C++核心代码实现赏析

<code_intro_overall>
以下是综合优质题解提炼的通用实现：
```cpp
#include <iostream>
using namespace std;

int main() {
    int day = 0, max_val = 8; // 关键初始值！
    
    for (int i = 1; i <= 7; i++) {
        int school, mom;
        cin >> school >> mom;
        int total = school + mom;
        
        if (total > 8 && total > max_val) {
            max_val = total;
            day = i; // 记录当前天数
        }
    }
    cout << day;
    return 0;
}
```
**代码解读概要**：
> 1. 初始化day=0（无解），max_val=8（阈值）
> 2. 循环7次读取每日两个学习时间
> 3. 实时计算总和并判断：若同时满足>8且>当前最大值，则更新记录
> 4. 循环结束直接输出结果，day=0时自动满足"无不高兴天"

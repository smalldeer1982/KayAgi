# 题目信息

# 足球

## 题目描述

我们当中有很多热爱中国足球的同学，我们都知道中超（中国足球超级联赛）的规则：

一场比赛中，若获胜（即你的得分严格大于对手得分）则获得 $3$ 的积分，若打平（即你的得分等于对手得分）则获得 $1$ 分，若失败（即你的得分严格小于对手得分）获得 $0$ 积分。

这个问题很简单，假设 $N$ 轮比赛中你一共攻入 $S$ 个球，丢掉 $T$ 个球，那么你可能获得的最大得分和最小得分是多少？

## 样例 #1

### 输入

```
1 1 1
1 1 2```

### 输出

```
1 1
3 2
```

## 1. 题目解读与策略初探

✨ **本题核心挑战**：  
在 N 场比赛、总进球 S、总丢球 T 的约束下，**如何安排每一场的进球/丢球分布**，使得  
- 总积分 **最大化**（3 分制，赢 3、平 1、输 0）  
- 总积分 **最小化**  

✨ **核心算法标签**：**贪心**、**分类讨论**

🗣️ **初步分析**：  
这道题没有复杂的数据结构，核心在于 **“如何把 S 个进球、T 个丢球分到 N 场比赛”** 的 **极端分配策略**：

- **最大得分**：尽量“赢”且“省进球”。把丢球集中到少数场次（甚至 1 场），让其余场次以最小代价取胜或打平。  
- **最小得分**：尽量“输”且“省丢球”。把进球集中到少数场次（甚至 1 场），让其余场次以最小代价打平或输球。

### 🔍 算法侦探：如何在题目中发现线索？

1.  **线索1 (问题目标)**：  
    “最大化/最小化得分” → 典型的 **贪心/极值问题**。  
2.  **线索2 (问题约束)**：  
    “总进球 S、总丢球 T 分到 N 场比赛” → **“分配”类问题**，可用 **“集中/分散”** 思想。  
3.  **线索3 (数据规模)**：  
    无具体范围限制，**O(1) 数学推导即可**。

### 🧠 思维链构建：从线索到策略

> 1.  首先，【线索1】告诉我们这是一个“极值”问题。  
> 2.  【线索2】提示我们：进球和丢球可以 **“自由调度”** 到每一场比赛。  
> 3.  【线索3】告诉我们无需复杂算法，只需 **数学推导**。  
> 4.  **结论**：**贪心 + 分类讨论** 是最直接、最高效的策略！

## 3. 解题策略深度剖析

### 🎯 核心难点与关键步骤（最优解法剖析）

1.  **关键点1：最大得分的“集中丢球”策略**  
    - **分析**：  
      - 若 `s < n`：最多赢 `s` 场，其余 `n-s-1` 场打平，最后 1 场集中丢球。  
      - 若 `s >= n`：前 `n-1` 场赢 1:0，最后 1 场用剩余进球 vs 丢球决定赢/平/输。  
    - 💡 **学习笔记**：**“集中资源打歼灭战”**——把丢球集中到 1 场，最大化胜场。

2.  **关键点2：最小得分的“集中进球”策略**  
    - **分析**：  
      - 若 `s > t`：必须赢 1 场（集中进球），其余场次尽量打平或输球。  
      - 若 `s <= t`：可以 **全输**（若 `t-s >= n-1`），或 **部分平局**。  
    - 💡 **学习笔记**：**“牺牲局部保全局”**——用 1 场胜利“浪费”所有进球，其余场次尽量输球。

3.  **关键点3：边界条件（如 `n==1`、`s==0`、`t==0`）**  
    - **分析**：  
      - `n==1`：直接比较 `s` 与 `t`。  
      - `s==0 && t==0`：所有比赛 0:0，积分 `n`。  
    - 💡 **学习笔记**：**“极端数据是调试的磨刀石”**。

### ✨ 解题技巧总结

- **技巧A：贪心分配**——“集中”与“分散”的极端思想。  
- **技巧B：分类讨论**——用 if-else 覆盖所有可能情况。  
- **技巧C：数学推导**——用公式代替枚举，O(1) 解决。

### ⚔️ 策略竞技场：不同解法的对比分析

| 策略 | 核心思想 | 优点 | 缺点 | 适用场景 |
|---|---|---|---|---|
| **贪心（最优）** | 极端分配进球/丢球 | O(1) 时间，代码短 | 需严谨分类 | 本题唯一正解 |
| **暴力枚举** | 枚举每场比赛的进球/丢球 | 思路直观 | O(N^N) 不可行 | 教学演示 |
| **动态规划** | 状态压缩每场比赛结果 | 通用性强 | 杀鸡用牛刀 | 复杂约束问题 |

### ✨ 优化之旅：从“能做”到“做好”

> 1.  **起点**：暴力枚举每场比赛的进球/丢球 → 指数级复杂度。  
> 2.  **发现瓶颈**：无需枚举每场比赛，只需 **总进球/丢球的极端分配**。  
> 3.  **优化钥匙**：用 **数学推导** 代替枚举，O(1) 解决。  
> 4.  **结论**：**贪心** 是解决极值问题的利器！

## 4. C++核心代码实现赏析

### 本题通用核心C++实现参考

**说明**：综合了艮鳖肉与 George1123 的清晰逻辑，覆盖所有边界。

```cpp
#include <cstdio>
using namespace std;

int main() {
    long long s, t, n;
    while (~scanf("%lld %lld %lld", &s, &t, &n)) {
        long long mx = 0, mn = 0;

        // 最大得分
        if (s < n) {
            mx = 3 * s + (n - s - 1);
            if (t == 0) mx++; // 无丢球，最后一场平局
        } else {
            mx = 3 * (n - 1);
            long long last = s - (n - 1); // 最后一场进球
            if (last > t) mx += 3;        // 最后一场赢
            else if (last == t) mx += 1;  // 最后一场平
        }

        // 最小得分
        if (s > t) {
            mn = 3 + (n - 1 - t > 0 ? n - 1 - t : 0); // 赢1场，其余尽量输
        } else {
            long long all_lose = (t - s >= n) ? 0 : n - (t - s); // 全输或部分平
            long long win_one = 3 + (n - 1 - t > 0 ? n - 1 - t : 0); // 赢1场
            mn = (all_lose < win_one) ? all_lose : win_one;
        }

        printf("%lld %lld\n", mx, mn);
    }
    return 0;
}
```

**代码解读概要**：  
- **最大得分**：`s < n` 时赢 `s` 场，其余尽量平；`s >= n` 时赢 `n-1` 场，最后 1 场根据剩余进球决定。  
- **最小得分**：`s > t` 时赢 1 场；`s <= t` 时比较“全输”与“赢 1 场”的较小值。

# 题目信息

# 足球

## 题目描述

我们当中有很多热爱中国足球的同学，我们都知道中超（中国足球超级联赛）的规则：

一场比赛中，若获胜（即你的得分严格大于对手得分）则获得 $3$ 的积分，若打平（即你的得分等于对手得分）则获得 $1$ 分，若失败（即你的得分严格小于对手得分）获得 $0$ 积分。

这个问题很简单，假设 $N$ 轮比赛中你一共攻入 $S$ 个球，丢掉 $T$ 个球，那么你可能获得的最大得分和最小得分是多少？

## 样例 #1

### 输入

```
1 1 1
1 1 2```

### 输出

```
1 1
3 2
```

### 1. 题目解读与策略初探  

✨ **本题核心挑战**：如何在总进球数(s)和总失球数(t)的约束下，将球分配到n场比赛中，使积分最大化/最小化。  

✨ **核心算法标签**：贪心策略、分类讨论、边界处理  

🗣️ **初步分析**：  
> 本题的解题思路遵循两条路径：  
> 1. **最大积分策略**：尽可能多赢球（3分/场），将失球集中到少数场次  
> 2. **最小积分策略**：避免获胜（3分>2分），优先输球（0分）或平局（1分）  
> 关键突破点在于发现：**进球分配方式决定胜负格局**，通过分类讨论可覆盖所有场景  

#### 🔍 算法侦探：如何在题目中发现线索？  
1. **线索1（问题目标）**："最大/最小"得分要求属于典型**最优化问题**，且存在明确约束条件（s, t, n）  
2. **线索2（问题特性）**：胜负规则（赢3/平1/输0）和**自由分配**特性，指向**贪心策略**  
3. **线索3（数据规模）**：s/t/n可达10^9量级，要求O(1)时间复杂度解法，**排除暴力枚举**  

#### 🧠 思维链构建：从线索到策略  
> "我们收集到三条关键线索：  
> 1. 线索1（最优化）暗示需要比较不同分配方案的极值  
> 2. 线索2（自由分配）说明胜负状态完全由s/t/n的数学关系决定  
> 3. 线索3（大数据）要求我们找到数学规律而非枚举  
>  
> **结论**：通过建立s,t,n的数学关系模型，用分类讨论实现O(1)贪心解法！

### 3. 解题策略深度剖析  

#### 🎯 核心难点与关键步骤  
1. **最大得分模型构建**  
   * 分析：核心矛盾是**赢球场次**与**失球分配**的制约  
   * 方案：  
     - 当s<n：赢s场(3s分)，平n-s-1场，最后1场根据t决定输/平  
     - 当s≥n：赢n-1场，最后1场根据s-n+1与t关系决定胜负  
   * 💡 学习笔记：**"省球赢球法"**——每场仅进1球节省进球数  

2. **最小得分模型突破**  
   * 分析：难点在于发现**赢1场=平3场**的分数陷阱  
   * 方案：  
     - 当s>t时：必须赢至少1场（3分），其余场次尽量输  
     - 当s≤t时：比较"全平/输"方案与"赢1场+其余输"方案  
   * 💡 学习笔记：**"进球集中术"**——所有进球集中到1场减少胜场  

3. **边界条件处理艺术**  
   * 分析：n=1/s=0/t=0等场景需特殊处理  
   * 方案：独立判断模块处理边界，如：  
     ```c
     if(n==1){ 
         if(s>t) return 3; 
         else if(s==t) return 1;
         else return 0;
     }
     ```
   * 💡 学习笔记：**"边界先判"原则**——优先处理特殊场景简化主逻辑  

#### ✨ 解题技巧总结  
- **技巧1（问题转化）**：将自由分配问题转化为s/t/n的数学关系判定  
- **技巧2（临界值处理）**：使用min(s-n+1, t)等表达式捕捉状态跃迁点  
- **技巧3（双策略比较）**：最小值得分时比较两种策略取最优  

#### ⚔️ 策略竞技场  
| 策略           | 核心思想                 | 优点               | 缺点                     | 得分预期 |
|----------------|--------------------------|--------------------|--------------------------|----------|
| **暴力枚举**   | 枚举每场进球/失球组合    | 逻辑直观           | O(2^n)超时不可行        | 0%       |
| **贪心分类**   | 建立s/t/n数学模型        | O(1)复杂度         | 分类复杂易遗漏边界       | 100%     |
| **DP规划**     | dp[i][j]表示前i场得失球  | 状态定义清晰       | 空间O(n²)超内存         | 30%      |

#### ✨ 优化之旅  
> 1. **起点：暴力枚举**  
>    尝试所有(s,t)分配方案，但n>30即超时  
>  
> 2. **发现瓶颈**：胜负仅取决于每场净胜球(>0/=0/<0)  
>  
> 3. **关键跃迁**：  
>    - 最大分：尽可能多造"净胜球>0"场次  
>    - 最小分：避免"净胜球>0"，优先制造"净胜球<0"  
>  
> 4. **模型升华**：  
>    ```python
>    最大分 = 3*赢场 + 1*平场
>    最小分 = min(3+剩余场次, 平场数) # 关键突破点
>    ```
>  
> 💡 **策略总结**："从暴力到贪心的本质是发现**问题内在的数学规律性**，通过建立s/t/n的关系模型，用分类讨论取代枚举！"

### 4. C++核心代码实现赏析  

#### 通用核心实现  
```cpp
#include <iostream>
using namespace std;

int main() {
    long long s, t, n;
    while (cin >> s >> t >> n) {
        long long max_score = 0, min_score = 0;
        
        // 最大得分逻辑
        if (s < n) {
            max_score = 3 * s + (n - s - 1);
            if (t == 0) max_score++; // 无失球需修正
        } else {
            max_score = 3 * (n - 1);
            if (s - n + 1 > t) max_score += 3;
            else if (s - n + 1 == t) max_score += 1;
        }
        
        // 最小得分逻辑
        if (s > t) {
            min_score = 3 + max(0LL, n - 1 - t);
        } else {
            long long option1 = (s == 0 ? 0 : 3) + max(0LL, n - 1 - t);
            long long option2 = max(0LL, n - (t - s));
            min_score = min(option1, option2);
        }
        cout << max_score << " " << min_score << endl;
    }
    return 0;
}
```
* **代码解读概要**：  
  1. 双分支处理最大得分（s<n和s≥n）  
  2. 最小得分采用双策略比较（赢1场 vs 全平/输）  
  3. max/min函数优雅处理临界值  

#### 题解片段赏析  
**题解：quantum11（极简主义）**  
```cpp
printf("%lld %lld\n", 
    s<n ? 3*s+n-s-1+!t : 3*max(n-1,min(n,s-t)) + (s-t==n-1),

```

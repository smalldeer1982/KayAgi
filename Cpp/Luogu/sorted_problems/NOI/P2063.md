# 二平方和定理
## 题目描述

对于正整数 $n$，请求出所有满足 $x^2+y^2=n$ 的非负整数对 $(x,y)$。
## 输入格式

**本题有多组数据**。

第一行一个正整数 $T$，代表数据组数。

对于每组数据，一行一个正整数，表示 $n$ 的值。
## 输出格式

对于每组数据，首先输出一行一个非负整数，代表满足条件的 $(x,y)$ 的个数。随后按 $x$ 从小到大的顺序依次输出 $(x,y)$。$x$ 与 $y$ 之间用空格分隔，每组 $(x,y)$ 之间用换行分隔。

每组数据之间用换行分隔。
## 样例

### 样例输入 #1
```
6
9
98
987
9876
98765
987654321
```
### 样例输出 #1
```
2
0 3
3 0
1
7 7
0
0
4
13 314
178 259
259 178
314 13
6
4080 31161
11064 29415
18264 25575
25575 18264
29415 11064
31161 4080
```
## 提示

对于所有数据，$1\le T\le 10^2,1\le n\le 10^{18}$。

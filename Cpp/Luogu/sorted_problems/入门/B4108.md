# [JOI2025 预选赛 R1H1] 铅笔 2
## 题目描述

JOI 君有 $A$ 日元，一根铅笔 $5$ 日元，问 JOI 君最多可以买多少根铅笔。
## 输入格式

一行一个正整数 $A$。
## 输出格式

一行一个整数，表示答案。
## 样例

### 样例输入 #1
```
9
```
### 样例输出 #1
```
1
```
### 样例输入 #2
```
10
```
### 样例输出 #2
```
2
```
### 样例输入 #3
```
3
```
### 样例输出 #3
```
0
```
### 样例输入 #4
```
100
```
### 样例输出 #4
```
20
```
## 提示

### 样例解释
$1$ 根铅笔 $5$ 日元，$2$ 根铅笔 $10$ 日元，$3$ 根铅笔 $15$ 日元。

### 数据范围

$1\leq A\leq 100$。

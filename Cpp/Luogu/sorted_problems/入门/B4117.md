# [JOI2025 预选赛 R1H3] 黑桃 J
## 题目描述

给定三个正整数 $A,B,C$，如果三个数的和 $\leq 21$，输出 $1$，否则输出 $0$。
## 输入格式

第一行一个正整数 $A$。

第二行一个正整数 $B$。

第三行一个正整数 $C$。
## 输出格式

一行一个整数，表示你的答案。
## 样例

### 样例输入 #1
```
5
10
6
```
### 样例输出 #1
```
1
```
### 样例输入 #2
```
7
8
13
```
### 样例输出 #2
```
0
```
## 提示

### 数据范围

$1\leq A,B,C\leq 13$。

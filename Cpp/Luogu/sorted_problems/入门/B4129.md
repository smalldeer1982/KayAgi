# [信息与未来 2014] 奇数及偶数和
## 题目描述

给出一个正整数 $n$，求出 $1,2,3,\cdots,n$ 中全部的奇数和、偶数和。
## 输入格式

一个整数，表示 $n$。
## 输出格式

一行两个整数，表示奇数和及偶数和。
## 样例

### 样例输入 #1
```
6
```
### 样例输出 #1
```
9 12
```
## 提示

$1\le n\le 2\times10^8$。

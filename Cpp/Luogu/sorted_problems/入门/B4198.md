# [常州市赛 2020] 比赛
## 题目背景

搬运自 <http://czoj.com.cn/p/446>。数据为民间数据。
## 题目描述

小 $\text X$ 参加了一场作文比赛的初赛。算上小 $\text X$ ，他的学校一共有 $n$ 个人参加了这场比赛。一个礼拜后，小 $\text X$ 兴奋地打开比赛官网，发现一等奖、二等奖、三等奖的分数线和所有人的得分都已经公布了，但没有提到获得什么奖项才能进入复赛。小 $\text X$ 想问问你，如果要获得了 $A$ 等奖才能进入复赛的话，他的学校有多少人能进入复赛呢？  

假设 $A$ 等奖的分数线是 $x$，一个人的分数是 $y$，那么如果 $y$ 大于等于 $x$，这个人就获得了 $A$ 等奖。
## 输入格式

输入数据共有四行。  

第一行一个正整数 $n$，表示小 $\text X$ 的学校一共有 $n$ 个人参加了比赛。  
第二行 $n$ 个正整数，表示小 $\text X$ 的学校中所有人的得分。  
第三行三个正整数 $L_3,L_2,L_1$，分别表示三等奖、二等奖、一等奖的分数线。  
第四行一个正整数 $A$，表示要获得了 $A$ 等奖才能进入复赛。
## 输出格式

输出一行包含一个整数，表示有多少人进入复赛。
## 样例

### 样例输入 #1
```
4
99 101 200 300
100 200 300
2
```
### 样例输出 #1
```
2
```
## 提示

### 样例解释  
- 第一个人 $99$ 分，没有奖项；  
- 第二个人 $101$ 分，获得了三等奖；  
- 第三个人 $200$ 分，获得了三等奖、二等奖；  
- 第四个人 $300$ 分，获得了三等奖、二等奖、一等奖；  
- 一共有 $2$ 个人获得了二等奖，所以有 $2$ 个人进入复赛。  
### 数据范围
本题共有 $10$ 个测试点。  
对于全部数据：所有人的得分 $\le1000$，$L_3\le L_2\le L_1\le1000$，$A\in\{1,2,3\}$。  
- 对于测试点 $1\sim2$：$n=1$；  
- 对于测试点 $3\sim6$：$n\le10$；  
- 对于测试点 $7\sim10$：$n\le10000$。

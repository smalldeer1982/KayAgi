# 【MX-X13-T1】「KDOI-12」在现实和志向中选择现实，否则会一个都得不到的。
## 题目背景

现实中的你加入了一个群聊，这个群聊总是有人说批话。
## 题目描述

定义一句话为 $A\underline{\texttt{ : }}B\underline{\texttt{ is }}C\underline{\texttt{.}}$，其中 $A,B,C$ 均为字符串，分别表示这个人的名称，句子的主语以及表语。

一句话是批话当且仅当这个人在说自己菜，也就是，$A$ 与 $B$ 相同且 $C=\underline{\texttt{weak}}$。

现在，你先判断给出的这句话是否是批话。如果是批话输出 `Plang.`，否则输出 `Not plang.`。
## 输入格式

仅一行，一个形如 $A\underline{\texttt{ : }}B\underline{\texttt{ is }}C\underline{\texttt{.}}$ 的字符串。**保证 $A,B,C$ 由且仅由小写英文字母组成**。
## 输出格式

仅一行，如果给出句子是批话则输出 `Plang.`，否则输出 `Not plang.`。
## 样例

### 样例输入 #1
```
zxx : zxx is weak.
```
### 样例输出 #1
```
Plang.
```
### 样例输入 #2
```
zxx : ou is strong.
```
### 样例输出 #2
```
Not plang.
```
### 样例输入 #3
```
zxx : zxx is cute.
```
### 样例输出 #3
```
Not plang.
```
## 提示

**【样例解释 \#1】**

在第一个样例中，$A=\text{zxx}$，$B=\text{zxx}$，$C=\text{weak}$。这句话中，$\text{zxx}$ 在说自己菜，这是一句批话。因此，应当输出 `Plang.`。

**【数据范围】**

| 测试点编号 | 特殊性质 |
|:--:|:--:|
| $1\sim7$ | A |
| $8\sim20$ | 无特殊性质 |

* 特殊性质 A：$C=\underline{\texttt{weak}}$。

对于所有数据：保证 $A,B,C$ 为长度不少于 $1$，不超过 $10$ 的由小写英文字母组成的字符串。

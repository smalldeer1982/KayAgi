# 疯狂星期四
## 题目背景

Source：TYCPC 4th，Check：not_clever_syl。

Alice 想要训练 Bob 的树穴能力！
## 题目描述

Alice 说，Bob 每天只要填一个表格，是一个 $1\times n$ 的矩阵，每一格可以填一个数 $x(0 \leq x \leq 7)$。

Alice 会检查这个表格，如果每一格的数之和可以被 $7$ 整除，Alice 就会带 Bob 去吃疯狂星期四。

Bob 很喜欢去吃疯狂星期四，但 Alice 为了防止 Bob 填重复的表格，增加了每天的填法不能相同的设定。

Bob 想知道他最多能吃多少次疯狂星期四。

由于 Bob 觉得这个数可能很大，所以你只需要输出这个数对 $101$ 取余的结果。
## 输入格式

输入一个整数 $n$。
## 输出格式

一行，一个整数表示答案对 $101$ 取余的结果。
## 样例

### 样例输入 #1
```
1
```
### 样例输出 #1
```
2
```
## 提示

对于所有数据，$1\leq n\leq 10^{10000000}$。

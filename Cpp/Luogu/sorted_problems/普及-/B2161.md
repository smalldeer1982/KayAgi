# 十进制转二进制
## 题目描述

给定一个十进制整数 $n$，请将其转换为对应的二进制表示，并输出。
## 输入格式

输入包含一行，一个整数 $n$。
## 输出格式

输出一行，表示整数 $n$ 的二进制表示（不含前导零）。
## 样例

### 样例输入 #1
```
10
```
### 样例输出 #1
```
1010
```
## 提示

- $n$ 保证在 int 类型范围内，即 $0 \leq n \leq 2^{31}-1$。
- 输出时不允许有前导零。

#### 数据范围

- $0 \leq n \leq 2^{31}-1$

提示：$n = 0$ 的情况需要特殊考虑。

# 二进制转十进制
## 题目描述

给定一个二进制整数（由若干个 $0$ 和 $1$ 组成的字符串），请将其转换为对应的十进制整数并输出。
## 输入格式

输入包含一行，一个仅由数字 $0$ 和 $1$ 组成的字符串，表示一个二进制整数。
## 输出格式

输出一行，表示该二进制数对应的十进制整数。
## 样例

### 样例输入 #1
```
1010
```
### 样例输出 #1
```
10
```
## 提示

**数据范围：**

输入的二进制字符串仅由 $0$ 和 $1$ 组成，长度 $1 \leq \text{len} \leq 31$，保证无前导 $0$（除非输入的数据为 $0$）。

# [信息与未来 2014] 最大连续部分和
## 题目描述

有 $n$ 个整数排成一排，求其中的最大连续部分和。

**至少要选一个数。**
## 输入格式

第一行一个整数，表示 $n$。

第二行 $n$ 个整数 $x_1,x_2,\cdots,x_n$，相邻两数之间有一个空格。
## 输出格式

一个整数（即最大的连续的部分和）。
## 样例

### 样例输入 #1
```
7
-2 13 12 9 14 -10 2
```
### 样例输出 #1
```
48
```
## 提示

$1\le n\le 10^7,0\le |x_i|\le 10^8$。

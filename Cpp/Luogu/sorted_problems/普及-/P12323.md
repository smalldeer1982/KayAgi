# [蓝桥杯 2023 省 Java B] 阶乘求和
## 题目描述

令 $S = 1! + 2! + 3! + \cdots + 202320232023!$，求 $S$ 的末尾 $9$ 位数字。

提示：答案首位不为 $0$。
## 输出格式

这是一道结果填空的题，你只需要算出结果后提交即可。本题的结果为一个整数，在提交答案时只需要**编写一个程序输出这个整数**，输出多余的内容将无法得分。

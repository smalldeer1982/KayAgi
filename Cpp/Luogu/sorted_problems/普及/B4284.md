# [蓝桥杯青少年组省赛 2022] 组合
## 题目描述

某商店将一种糖果按照数量打包成 $N$ 和 $M$ 两种规格来售卖（$N$ 和 $M$ 为互质数，且 $N$ 和 $M$ 有无数包）。这样的售卖方式会限制一些数量的糖果不能买到。现在给出 $N$ 和 $M$ 的值，请你计算出最多不能买到的糖果数量。

## 输入格式

输入两个正整数 $N$ 和 $M$（$2 < N < M < 100$，$N$ 和 $M$ 为互质数），表示这两种规格的糖果数量，正整数之间用一个空格隔开。
## 输出格式

输出一个整数，表示最多不能买到的糖果数量。
## 样例

### 样例输入 #1
```
3 5
```
### 样例输出 #1
```
7
```

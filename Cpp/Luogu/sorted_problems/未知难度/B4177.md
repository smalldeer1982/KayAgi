# [BCSP-X 2024 6 月初中组] 尽量接近
## 题目描述

给出 $N$ 个整数，要求从中选出若干个数，使得它们的和尽量接近整数 $K$。
## 输入格式

- 第一行两个正整数 $N, K$。
- 第二行 $N$ 个数，表示给出的数字。
## 输出格式

共一行一个整数，表示最接近 $K$ 的和。如果不唯一，输出较小的那个。
## 样例

### 样例输入 #1
```
4 12
5 6 9 4
```
### 样例输出 #1
```
11
```
## 提示

### 样例解释
$\{5, 6\}$

### 数据范围
- 对于 40% 的数据，$1 \leq N \leq 10$。
- 对于 100% 的数据，$1 \leq N \leq 50, 1 \leq K \leq 10^6$，给出的数字是 $[1, 1000]$ 范围内的整数。

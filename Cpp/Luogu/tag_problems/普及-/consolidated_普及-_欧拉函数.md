# [蓝桥杯 2023 国 Java B] 互质
## 题目描述

请计算在 $[1,2023^{2023}]$ 范围内有多少个整数与 $2023$ 互质。由于结果可能很大，你只需要输出对 $10^9+7$ 取模之后的结果。
## 输出格式

这是一道结果填空的题，你只需要算出结果后提交即可。本题的结果为一个整数，在提交答案时只需要**编写一个程序输出这个整数**，输出多余的内容将无法得分。

# BROUL - Babylonian Roulette

## 题目描述

巴比伦人沉迷于机会游戏，其中一种特别受欢迎的游戏是特别的轮盘赌。最近，一些古老的巴比伦泥板被发现，上面记载了这种轮盘赌的详细规则。

用现代的话来说，游戏规则如下：

- 轮盘赌的格子上只有六个标签：-1、-2、-3、1、2、3。
- 游戏按天进行，每天分若干回合，回合编号依次为 0, 1, 2, ...
- 玩家有机会赢得或输掉赌注的倍数，这个金额在一天内保持不变。
- 在第 $t$ 回合，奖金池中的金额为 $P_t$。
- 刚开始时，奖金池有一个初始金额 $P_0$。
- $P_0$ 和赌注均为国王随意设定的正数。
- 每个回合，玩家转动轮盘，每位玩家每天只能玩一次。根据球停下来的格子标签 $L$，玩家赢得（或输掉，如果是负数）金额 $w_t = L \times \text{赌注}$。
- 赢到的钱从奖金池中被扣除（如果输，则放入奖金池），奖金池在某一回合的金额由 $P_{t+1} = P_t + w_t$ 决定。
- 如果根据规则计算的 $P_{t+1}$ 为负数，赢家只能获得使奖金池不为负数的最大倍数的赌注。
- 一旦某一回合中奖金池少于赌注，当天的游戏就结束。否则，游戏会一直持续到日落。

除了这些规则的泥板外，还有一些泥板上记录了三个数字。考古学家推测，这些数字表示某一天游戏的奖金池起始金额、赌注和结束时的奖金池金额。

例如，一行记录为：

```
10000 1500 11500
```

这可能代表当天只有一回合，玩家赢得了标签为 1 的格子。也可能表示有三回合结果为 2、1 和 -2。

然而，考古学家还发现其他泥板上的数字组合看起来并不像上述情况那样能够代表某一天的游戏结果。目前这些数字的意义尚无明确猜测。

考古学家希望通过分析这些泥板，验证他们的推测，估算每天玩游戏的人数。他们的目标是，确定每个看似能够代表某一天游戏结果的数字组合中，最少有多少玩家参与。在上面的例子中，答案是 1。无法代表结果的组合需要被识别。你需要帮助完成这项任务。

## 输入格式

输入文件包含若干测试用例，每个测试用例占据一行，每行是一个非负整数的三元组，分别指示某日的初始奖金池、赌注和结束时的奖金池。

每个输入数字都小于 $10^8$。初始奖金池和赌注都是大于 0 的正整数。

以一行三个 0 作为输入结束标记。

## 输出格式

对于每个测试用例，要输出结果与输入顺序一致。如果测试用例无法代表某一天的游戏结果，则输出 `No accounting tablet`。否则输出一个正整数，表示当天至少有多少个玩家参与转动轮盘。

## 说明/提示

- 初始奖金池和赌注均为正整数，且小于 $10^8$。
- 每个测试用例的数字都在 $10^8$ 以内。

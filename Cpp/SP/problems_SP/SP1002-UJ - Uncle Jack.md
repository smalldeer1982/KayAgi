# UJ - Uncle Jack

## 题目描述

总共有 $N$ 个人，$D$ 张光盘，问有多少种分法。

## 输入格式

+ 每行 $2$ 个变量 $N$ 和 $D$。输入到 $N=0$ 并且 $D=0$ 的时候停止。

## 输出格式

+ 每行 $1$ 个数，表示总共的分法数，用换行隔开。

## 说明/提示

$1\le N\le1000$，$1\le D\le2500$。

感谢 UID1056150 提供的翻译。

## 样例 #1

### 输入

```
1 20
3 10
0 0
```

### 输出

```
1
59049
```

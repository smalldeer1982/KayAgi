# STRAZA - STRAZA

## 题目描述

在一个军事基地附近，分布着由若干条线段组成的战壕系统。在夜间，当大多数士兵进入梦乡时，三名哨兵负责守卫这些战壕。若两名哨兵之间有一条完整的战壕（或连续的战壕）作为他们之间的直线段，且在线段上没有其他哨兵阻挡时，他们就可以相互看见。

出于安全原因，哨兵的放置必须保证每名哨兵都能看见其他两名哨兵。请问，有多少种不同的方式可以这样安排哨兵？

## 输入格式

第一行包含一个整数 $N$（$1 \le N \le 1000$），表示战壕的数量。

接下来的 $N$ 行中，每行给出四个整数 $x_1, y_1, x_2, y_2$，表示一条战壕的起点和终点坐标，满足 $-10^6 \le x_1, y_1, x_2, y_2 \le 10^6$。

## 输出格式

在一行中输出哨兵适当放置的方案数。

## 说明/提示

- 战壕数量 $1 \le N \le 1000$
- 坐标范围 $-10^6 \le x_1, y_1, x_2, y_2 \le 10^6$

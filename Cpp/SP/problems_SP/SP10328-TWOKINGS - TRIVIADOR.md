# TWOKINGS - TRIVIADOR

## 题目描述

### TRIVIADOR

Triviador 是两个国王之间的战争。一个国王可以在每一回合攻击一个敌对地区。当一个国王攻击一个地区时，他可以占领与之相连的所有敌对地区（不仅仅是他所攻击的地区）。每个地区都有八个地区与之相连（即八连通块）。国王们有轮流进攻的机会。一号国王有机会先手进攻。假设两个国王都很聪明，请找出谁会最终征服所有地区。可以证明如果有一个国王很聪明，那么他一定会赢！

## 输入格式

第一行一个整数 $t$，表示有 $t$ 组测试数据。

对于每组测试数据：

第一行两个整数，表示区域中的行数和列数（每个格是一个区域）。

下面是对格的描述。每个地区有一个字母，如果是 `X` 代表这个地区属于 一号国王，如果是 `O` 则属于二号国王。

## 输出格式

每组测试点输出一行，如果一号国王赢了输出 `X`，否则输出 `O`。

## 说明/提示

$ T \le 100$

设行数、列数分别为 $m, n$，则 $1 \le m, n \le 10$。


样例一：一号国王有两种攻击选择，但无论如何他选择哪一种。只要二号国王反击他都会输。

样例二：一号国王可以攻击敌对地区中的任意一个。他可以直接占领全部敌对地区，第一回合就胜利了。

PS：原题中是步不是回合，但回合更好理解，所以翻译成了回合，Triviador 是一款游戏名，没有翻译。

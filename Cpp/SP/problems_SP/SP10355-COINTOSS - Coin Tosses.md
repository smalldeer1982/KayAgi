# COINTOSS - Coin Tosses

## 题目描述

你有一枚公正的硬币，想要一直投掷它，直到获得 $N$ 个连续的正面为止。目前你已经投掷了 $M$ 次，并且非常幸运地，所有投掷结果都是正面。现在请计算，你还需要再投掷多少次才能达到 $N$ 个连续正面的期望次数。

## 输入格式

输入包含多组测试数据，每行两个整数 $N$ 和 $M$。

## 输出格式

对于每组测试数据，输出一个实数，表示期望的投掷次数。结果需保留六位小数。

## 说明/提示

$1 \leq M < N \leq 10^5$

# MECGROUP - project groups

## 题目描述

给定若干个测试用例，每个用例定义了一个班级的基本信息。

## 输入格式

第一行为一个整数 $n$，表示测试用例的数量。

接下来的每个测试用例由三个整数 $B$、$G$ 和 $t$ 组成，它们之间用空格分隔。其中，$B$ 表示班级中的男生人数，$G$ 表示女生人数，$t$ 表示一个小组中学生的总人数。

约束条件如下：
- $1 \le n \le 20$
- $4 \le B \le 30$
- $1 \le G \le 30$
- $5 \le t \le B + G$

## 输出格式

对于每个测试用例，输出一行，表示形成这样的学生小组的总数。

### 输入

```
\n2\n4 1 5\n30 30 20\n
```

```
\n1\n4191318957352590\n
```

# CAM5 - prayatna PR

## 题目描述

### 题意简述

有 $n$ 个编号为 $0$ 至 $n-1$ 的点和 $m$ 条双向边组成的无向图，求它的连通块个数。有 $T$ 组测试数据。

## 输入格式

第 $1$ 行一个整数 $T$ 表示测试数据的组数。

每组测试数据中：

共 $m+3$ 行。

第 $1$ 行为空行。

第 $2$ 行一个整数 $n$ 表示点的数量。

第 $3$ 行一个整数 $m$ 表示边的数量。

接下来 $m$ 行，每行两个整数 $x,y$ 表示 $x$ 和 $y$ 有一条无向边。

## 输出格式

共 $T$ 行，每行一个整数表示该图连通块的个数。

## 说明/提示

对于 $100\%$ 的数据，$1\le T\le10，2\le n\le100000，0\le m\le\frac{n}{2}，0\le x,y<n$。

## 样例 #1

### 输入

```
2

4
2
0 1
1 2

3
0
```

### 输出

```
2
3
```

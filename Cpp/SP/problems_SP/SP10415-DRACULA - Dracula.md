# DRACULA - Dracula

## 题目描述

德古拉从长眠中苏醒了！所有市民都要乘船逃离这座城市。

所有船票都是从 $a$ 到 $b$ 编号的，每张船票的价格就是它的每一位数字的和，例如编号为 $14$ 的门票价格就是 $5$（$1+4$）。

范海辛现在有 $x$ 块钱，身为城里唯一的程序员，你要计算他最多能买几张票。

## 输入格式

第一行包含一个整数 $t$，表示测试数据有 $t$ 组。

## 输出格式

$t$ 行，每行包含一个整数表示答案。

## 样例 #1

### 输入

```
1\n2 5 5
```

### 输出

```
2
```

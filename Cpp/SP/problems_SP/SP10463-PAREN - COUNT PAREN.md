# PAREN - COUNT PAREN

## 题目描述

你会得到一个 bool 表达式，该表达式由若干个“true”、“false”、“and”、“or”和“xor”组成。请你计算这个表达式添加括号的方式数量，使其计算结果为 true。“or”和“xor”具有相同的优先级。

## 输入格式

一个正整数t，1<=t<=100，表示数据组数，接下来t行，每行包含一个由字符 T、F、or 和 xor 组成的且长度小于 100 的一个用空格分隔的字符串。

## 输出格式

对于输入的每个字符串，显示一行，其中包含对表达式添加括号的不同方式的数量，以便其计算结果为true。

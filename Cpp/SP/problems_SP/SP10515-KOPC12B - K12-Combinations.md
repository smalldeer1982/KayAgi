# KOPC12B - K12-Combinations

## 题目描述

给定一个正整数 $n$，求 $\sum\limits_{i=1}^n i (C^i_n)^2$。由于答案可能很大，你只需要求出结果模上 $10^9+7$ 的值即可。  
注：$C^m_n=\dfrac{n!}{(n-m)!m!}$。

## 输入格式

第一行一个正整数 $T$，表示数据的组数。  
接下来 $T$ 行，每行一个正整数 $n$。

## 输出格式

$T$ 行，每行一个正整数表示答案。

## 说明/提示

对于 $100\%$ 的数据，$T\le1000$，$n\le10^6$。

## 样例 #1

### 输入

```
2
1
2
```

### 输出

```
1
6
```

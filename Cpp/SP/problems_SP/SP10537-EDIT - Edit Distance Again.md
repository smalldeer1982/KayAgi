# EDIT - Edit Distance Again

## 题目描述

相信大多数程序员都熟悉“编辑距离”这一经典问题。然而，如果字母大小写是交替出现的，我们称其为“交替链”。

**示例：** "AaAaAbB"，"B"，"a"，"aBaCdEf"。

在这个问题中，我们关注字母大小写交替形成的连串。

我们只允许进行一种操作：将一个字母的大写转换为小写，或者把小写字母转换为大写。

现在给定一个字符串，计算将其变为交替链所需的最少修改次数。

## 输入格式

输入为多行数据，每行是一个仅由大小写字母组成的字符串，长度不超过 $10^3$，直到文件结束。

## 输出格式

对于每一行输入，输出将其转换为交替链所需的最少修改次数。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
AaAaB
ABaa
a
```

### 输出

```
0
2
0
```

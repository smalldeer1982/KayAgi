# DELIVER - Delivery Route

## 题目描述

平面上有 $N$ 个点，其坐标均为整数，每个点被标上编号。一个人 $1$ 分钟可上下左右移动一个坐标单位。要求该人从第１点开始依次按编号访问第 $2$ 点、$3$ 点最后从第 $Ｎ$ 点回到第 $1$ 点，除第 $1$ 点外，每点只访问一次。求所需花的最少时间。

## 输入格式

- 第 $1$ 行 $1$ 个整数， $N$。

- 第 $2$ 到 $N + 1$ 行， 每行 $2$ 个整数 $(x_i,y_i)$描述一个农场的坐标。

## 输出格式

- $1$ 行 $1$ 个整数， 完成行程的最小分钟数，无解请输出 $-1$。

## 样例 #1

### 输入

```
4
2 2
2 4
2 1
1 3
```

### 输出

```
12
```

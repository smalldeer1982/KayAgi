# TWODEL - Delivery

## 题目描述

**速递公司**在古巴最长的直街中央开了一家新办公室，以便满足住在这条街上的客户的配送需求。

房屋地址用整数表示，表示该房屋距离办公室的距离。如果我们把办公室位置设为 0，那么地址为正数的房屋位于办公室的右边，地址为负数的房屋位于办公室的左边。

订单按照接收顺序进行配送。

速递公司为这个新办公室配备了**两名**配送员，负责分发货物。在一天的开始，所有订单会在两名员工之间进行分配。

公司希望找到一种订单分配方案，使得两名员工完成所有配送任务所需的总行走距离达到**最小化**。

## 输入格式

第一行包含一个整数 $N$（$1 \le N \le 100,000$），表示订单的总数。

接下来的 $N$ 行中，每行有一个整数，表示货物需要配送的地址。

任一地址距离办公室的绝对值不超过 $10^8$。

## 输出格式

输出一个整数，表示达到订单需求最小时的最小行走总距离。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
5
1
-1
2
-2
3
```

### 输出

```
5
```

# SYM12 - Symmetry

## 题目描述

经过一堂现代艺术课，农夫约翰开始对他农场周围的一切几何图案产生兴趣。他在二维平面上精确标出了 $N$ 头奶牛的位置（$2 \leq N \leq 1000$），每头奶牛都在不同的点上。他想知道这些点共有多少条对称轴。所谓对称轴，就是一条能使直线两边的点互为镜像的直线。请帮助农夫约翰解决这个迫切的几何问题。

## 输入格式

- 第一行：一个整数 $N$，表示奶牛的数量。
- 接下来的 $N$ 行：每一行包含两个整数，表示第 $i$ 头奶牛的 $x$ 和 $y$ 坐标（$-10,000 \leq x, y \leq 10,000$）。

## 输出格式

- 第一行：对称轴的数量。

## 示例输入

```
4
0 0
0 1
1 0
1 1
```

在示例中，4 头奶牛构成了一个正方形的四个顶点。

## 示例输出

```
4
```

## 输出解释

该点集共有 4 条对称轴——一条垂直的、一条水平的，和两条对角线上的。

# STRSEQ - String Subsequence

## 题目描述

给定一个由数字组成的字符串，找出一个在该字符串中作为子序列没有出现过的最小非负整数。字符串的长度范围在 1 到 100,000 之间。

## 输入格式

输入由若干行字符串组成，每行包含一个字符串 $S$，其长度介于 1 到 100,000 之间。总共最多不超过 200 行。

## 输出格式

对于每个输入字符串，输出一个单独的行，该行中包含的是在该字符串中作为子序列未出现过的最小非负整数。

## 说明/提示

每个字符串的长度 $|S|$ 满足 $1 \leq |S| \leq 100,000$。最多会有 200 行输入数据。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
9876543210
21698921085321984125
12345678901
```

### 输出

```
11
7
22
```

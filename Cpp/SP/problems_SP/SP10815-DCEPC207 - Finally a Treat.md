# DCEPC207 - Finally a Treat

## 题目描述

Anuja 老师非常痴迷于她的名字，她觉得她的名字中隐藏着某种特别之处，只是她一直没能发现。Vaibhav 老师擅长寻找模式，帮助她发现，她的名字里只有一个字母出现的频率最高，那就是字母 `a`。Anuja 老师对此感到十分开心，并决定为所有同样拥有这类特殊名字的 DCE Coders 成员举办一场派对。然而，当她查看名单时，发现名单太长了，于是决定设定一个更严格的条件。筛选条件如下：“Anuja 老师将邀请那些不仅拥有特殊名字，而且这种特殊性来源于在整个名字列表中频率最高的字符的人来参加派对。” 在一个字符串中，只可能有一个这样的特殊字符。

作为学生，你需要为 Anuja 老师列出符合条件可以参加派对的名字名单。

请记住，名字只有在其中一个字符频率唯一最高的情况下才算特殊。（见示例 #2）

## 输入格式

第一行输入一个整数 T，表示测试用例的数量，$1 \le T \le 100$。接下来 T 行中，每行包含一个字符串，其中是以空格分开的名字列表，$1 \le $ 字符串长度 $\le 10000$。

## 输出格式

对于每个测试用例，首先输出“Case #x:”，其中 x 是该测试用例的编号，从1开始。然后在接下来的 k 行中输出符合条件的学生名字，k 是符合条件的名字数量。

**注意：每个输入字符串至少包含一个单词（至少包含一个字符），不会出现只包含空格的输入情况。**

## 数据范围
- $1 \le T \le 100$
- $1 \le $ 字符串长度 $\le 10000$

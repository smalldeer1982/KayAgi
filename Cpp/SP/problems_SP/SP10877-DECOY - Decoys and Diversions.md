# DECOY - Decoys and Diversions

## 题目描述

现在，双方在战场上都在使用某种猫捉老鼠的战术。规则是：两场战斗同时在两个不同的战场上进行。双方轮流将其中一支部队撤退到别的位置；一方部队的撤退将自动引导同一战场上的敌军跟随。如果有一方被逼入绝境（也就是说，在他们的回合中无法移动），那一方就将输掉战斗。给定战场的描述，并假设双方指挥官都是绝顶聪明的战略家，你能预测谁将获得胜利吗？

## 输入格式

输入的第一行有一个整数 **T**，表示测试用例的数量。每个测试用例由两个战场的描述组成。每个描述以一行开始，包含两个整数 **N** 和 **M**，分别表示战场中的位置数量和连接这些位置的道路数量（$1 \le N \le 100$，$1 \le M \le 10000$）。接下来的 **M** 行中，每行包含两个整数 **u** 和 **v**（$1 \le u, v \le N$），表示位置 **u** 和 **v** 之间有一条道路。所有单位最初都在各自战场的位置 1。

## 输出格式

对于每个测试用例，输出一行结果描述战斗的结局，假设是您的军队先行动。如果您将会输，输出 `I lose`；如果您注定会赢，输出 `I win`；如果战斗会无限持续，输出 `Deadlock`。

## 说明/提示

- $1 \le T \le 100$
- $1 \le N \le 100$
- $1 \le M \le 10000$

# CUSTOMSL - Customs

## 题目描述

Matheus、Bruno 和 Ricardo 是 Indústria de Obras Intermináveis (IOI) 研究部门的员工，他们经常结伴去其他国家研究各种新技术、新设备和原材料。他们还顺便从国外带回一些电子产品，卖给朋友和同事。

在每次出发前，他们都会列出一份包含 $N$ 种产品的采购清单，每种产品的价格为 $P_i$（单位：DE$，即外币）。其中，如果某个产品的价格超过了每个人可以携带的最大额度 $Q$，就需要支付一些关税。

由于他们总是一起旅行，他们发现可以通过合理安排产品携带方式，最大限度地减少需要交的关税。给定产品的价格列表 $P_1, P_2, \ldots, P_N$ 、可携带的最高额度 $Q$ 以及税率上限 $A$，请计算出他们需要支付的最小关税总额。

## 输入格式

输入只有一组数据。第一行包含两个整数 $N$ 和 $Q$（$1 \le N \le 10^5, 1 \le Q \le 10^9$），分别表示产品数量和每个人的最大携带额度。第二行包含 $N$ 个整数 $P_1, P_2, \ldots, P_N$（$1 \le P_i \le 10^9$），代表每个产品的价格。第三行包含一个整数 $A$（$1 \le A \le 100$），表示税率的上限。

## 输出格式

输出一行，表示 Matheus、Bruno 和 Ricardo 最少需要支付的关税总额。

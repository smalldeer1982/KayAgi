# ONBRIDGE - Online Bridge Searching

## 题目描述

给定 $N$ 个顶点的图，从 $0$ 到 $N-1$ 编号。最初的图中没有边。将 $M$ 个无向边 $(u,v)$ 依次添加到图中。添加边后，输出当前图中的桥数。

数据保证没有添加现有边或自环。

## 输入格式

第一行包含一个整数T表示测试样例的数量。

每个测试样例的第一行是 $2$ 个整数 $N$ 和 $M$ ，后跟 $M$ 行，每行包含一对整数 $(u,v)$ 表示对在图形上添加边 $(u,v)$。

## 输出格式

每次添加完边之后，单独一行输出当前图中桥数。

## 说明/提示

$1\le N\le 5\times 10^4$ 

$1\le M\le100000$

$1\le T\le10$

$0\le u,v\le N$

## 样例 #1

### 输入

```
1
5 10
3 0
0 2
1 0
1 3
1 4
2 4
4 0
2 1
2 3
3 4
```

### 输出

```
1
2
3
1
2
0
0
0
0
0
```

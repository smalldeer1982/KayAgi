# MONEYYTU - Save money for YTU

## 题目描述

塔鲁一直梦想着能进入南洋理工大学学习，因此他需要开始存钱，以确保能募集到足够的学费。他计划在 $T$ 个月后积攒到 $X$ 美元。他将钱存入一家提供月利率为 $R\%$ 的银行账号。不过，由于塔鲁有些迷信，他只有在账户余额是 100 的倍数时才会存钱。如果月初账户的余额不是 100 的倍数，塔鲁会取出一定金额，使余额变为 100 的倍数，并将这些钱捐给慈善机构。比如说，如果月初他的账户里有 $17823$ 美元，他会捐出 $23$ 美元，使余额整除 100。你的任务是计算塔鲁每个月需要存入银行的最小金额，以便最终实现他的目标。

## 输入格式

第一行是测试用例的数量（Test）。接下来的每行包含三个整数：$X$ 表示需要的金额，$T$ 表示月数，$R$ 表示月利率。

## 输出格式

对于每个测试用例，输出塔鲁每月应存入的最小金额，结果精确到小数点后四位。

## 说明/提示

- 测试用例数量 $1 \leq \text{Test} \leq 20$
- 目标金额 $1 \leq X \leq 10^6$
- 月数 $1 \leq T \leq 60$
- 月利率 $0 \leq R \leq 10$

**样例输入**

```
2
50000 3 5
20000 5 6
```

**样例输出**

```
15142.8571
3386.7925
```

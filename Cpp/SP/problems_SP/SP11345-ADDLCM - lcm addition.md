# ADDLCM - lcm addition

## 题目描述

Mao 数学很好，他喜欢玩数论，数论是他最喜欢的一章。有一次他决定向他的朋友 Kalyu 提问。但是现在 Kalyu 总是忙于写文章，所以他不能花太多时间来解决这个数学问题，但他也不想伤害他的朋友，所以请你帮助 Kalyu 解决 Mao 的问题。

给定 $a,b$（$a\le b$），求：
$$\sum_{i=a}^b\operatorname{lcm}(i,b)$$

因为输出可能非常大，所以结果取模 $10^9+7$。

## 输入格式

多测。
- 第一行一个正整数 $T$，表示有 $T$ 组数据；
- 下面 $T$ 行每行两个正整数 $a$ 和 $b$，含义见题目描述。

## 输出格式

每组数据输出一行，代表你的答案。

## 说明/提示

对于 $100\%$ 的数据，有：
- $T\le10^5$，$1\le a\le b\le10^6$。

## 样例 #1

### 输入

```
3
1 6
10 15
41 90
```

### 输出

```
66
675
139860
```

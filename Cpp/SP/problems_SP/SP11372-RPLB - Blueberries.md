# RPLB - Blueberries

## 题目描述

Teresa 在采摘完足够的草莓后，决定去彩虹国的神奇蓝莓丛中摘取蓝莓。

由于有了之前采摘草莓的经验，Teresa 希望自己的蓝莓采摘量不会超过规定的上限。

在采摘过程中，她发现如果从第 $i$ 个蓝莓丛中采摘蓝莓，那么她就无法从紧邻的第 $i+1$ 个蓝莓丛中采摘（这是彩虹国的一种魔法影响）。

由于这个限制，Teresa 想知道，给定蓝莓丛的数量以及每个蓝莓丛上的蓝莓数量的情况下，她最多能采摘多少蓝莓。

## 输入格式

第一行是整数 $T$，代表测试用例的数量。接下来的 $T$ 组测试用例中，每组会有两个整数 $N$ 和 $K$，其中 $N$ 是蓝莓丛的数量，$K$ 是 Teresa 能采摘的蓝莓的最大数量。随后一行包含 $N$ 个整数，每个整数表示第 $i$ 个蓝莓丛上的蓝莓数目。

## 输出格式

对于每个测试用例，输出格式为：「Scenario #i: 」，其中 $i$ 是当前测试用例的编号，接着输出一个整数，表示在该测试用例中可以采摘的最大蓝莓数量。

## 说明/提示

- $1 \le T \le 100$
- $1 \le N \le 100$
- $1 \le K \le 10^9$
- 每个蓝莓丛上的蓝莓数量为 $1 \le x \le 10^9$

### 输出

Scenario #1: 90

# RPLC - Coke madness

## 题目描述

大卫非常热爱可乐。有一天，他走在一条狭窄的街道上，看到了许多不同品牌的可乐。他想把这些可乐全部喝掉，但他发现有些品牌的可乐能让他能量倍增，而另一些则会削弱他的能量。虽然他可以等着慢慢恢复能量，但他并不想这样。他计划在开始时储足能量，然后一次性喝完街道上的所有可乐。

请帮他找出最佳时机，以便能够一口气喝完所有可乐。

## 输入格式

首先输入一个整数 $T$，表示测试用例的数量。接下来 $T$ 行分别对应每个测试用例。在每个测试用例中，第一行是一个整数 $N$，表示可乐的数量。接下来的行中包含 $N$ 个整数，表示每瓶可乐的能量值（正数表示增加能量，负数表示减少能量）。

## 输出格式

对于每个测试用例，输出格式为 `Scenario #i: `，其中 $i$ 代表第 $i$ 个测试用例，后面是大卫走过这条街道所需的最小能量。

## 说明/提示

- $1 \le T \le 100$
- $1 \le N \le 10^3$
- 可乐的能量值范围为 $-10^6$ 到 $10^6$

 **本翻译由 AI 自动生成**

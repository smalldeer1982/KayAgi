# SHELF - Book Shelves

## 题目描述

Farmer John 共有 $N$ 本图书，第 $i$ 本书的宽度 $W_i$  和 高度 $H_i$ ，书必须按照编号顺序摆放。每层书总宽度不能超过书架宽度 $L$。每一层书架的高度为这层图书的最高高度，整个书架的高度等于每层书架的高度之和。显然，按不同方案摆放书架高度可能不同，求整个书架的最小高度。

## 输入格式

第一行，两个整数 $N$ 和 $L$；接下来 $N$ 行每行为 $H_i$ 和 $W_i$

## 输出格式

一个整数，表示整个书架的最小高度。

## 说明/提示

$1 \le N \le  10^5$

$1 \le L \le 10^9$

$1 \le H(i) \le 10^6$

$1 \le W(i) \le L$

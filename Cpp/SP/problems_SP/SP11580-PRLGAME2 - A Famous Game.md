# PRLGAME2 - A Famous Game

## 题目描述

B 先生和 M 先生喜欢玩球。他们拥有许多蓝色和红色的球。首先，B 先生随意从中挑出 $N$ 个球放进一个袋子。M 先生知道袋子里的红球数量可能从 0 到 $N$，一共有 $N+1$ 种可能，并且这 $N+1$ 种情况发生的概率相同。不过，M 先生并不清楚究竟是哪一种情况。接下来，M 先生从袋子里取出 $P$ 个球，并对这些球进行检查，发现其中有 $Q$ 个红球，其余 $P-Q$ 个是蓝球。问题是，如果他再从袋子里抽取一个球，这个球是红色的概率是多少。

## 输入格式

每个测试用例只有一行，包含三个整数 $N$、$P$ 和 $Q$，满足 $2 \le N \le 100,000$ 和 $0 \le Q \le P \le N-1$。

## 输出格式

对于每个测试用例，输出一行，包含测试用例编号和 M 先生再取一个球时，该球是红球的概率。概率值保留四位小数。

**注意：评测系统会忽略多余的空格。**

 **本翻译由 AI 自动生成**

### 输入

```
3 0 0
4 2 1
```

### 输出

Case 1: 0.5000
Case 2: 0.5000

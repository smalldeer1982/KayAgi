# DOJO - With a Pit of Death

## 题目描述

Leo 在比特国的道场进行武术训练。道场是一个 $n \times m$ 的矩形区域，内部有一个所谓的“死亡之坑”，位于坐标 $(i, j)$，其中 $1 \le i \le n$ 且 $1 \le j \le m$。每天晚上，经过激烈的战斗，道场会被摧毁，白天又围绕这个死亡之坑重建。

每天傍晚，Leo 的首要任务是用大小为 $2 \times 1$ 的榻榻米铺满新的道场。榻榻米必须覆盖整个道场，仅除去死亡之坑。根据情况，有时可以铺满，有时则无法铺满。

## 输入格式

输入首先包含一个整数 $T$，表示测试用例的数量。接下来有 $T$ 行，每行包含四个整数 $n, m, i, j$。

## 输出格式

对于每一个测试用例，如果可以用榻榻米铺满道场，则输出“Possible.”，否则输出“Impossible.”。

## 说明/提示

- $1 \le T \le 100$
- $1 \le n, m \le 10^9$
- $1 \le i \le n$
- $1 \le j \le m$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2\n3 3 2 2\n3 3 1 2\n
```

### 输出

```
Possible.\nImpossible.\n
```

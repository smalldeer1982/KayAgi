# TOE2 - Tic-Tac-Toe ( II )

## 题目描述

## 题目大意
给你一个三子棋棋盘的状态，判断游戏是否有一方三子连成线。

## 输入格式

每一行有9个字符，表示三子棋的棋盘。输入以end结尾。

## 输出格式

对于除了end的每一行输出valid或invalid，表示有没有一方获胜。

## 样例 #1

### 输入

```
XXXOO.XXX
XOXOXOXOX
OXOXOXOXO
XXOOOXXOX
XO.OX...X
.XXX.XOOO
OOXXXOOXO
end
```

### 输出

```
invalid
valid
invalid
valid
valid
invalid
invalid
```

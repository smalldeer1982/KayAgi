# REDCROSS - Red Cross Hospital

## 题目描述

红十字会是一个中立的人道主义组织，致力于为内战和其他严重暴力事件的受害者提供帮助。

最近，Nlogônia 发生了一场政治危机，最终引发了一场大规模的内战。这场战争摧毁了全国数十个无辜的社区。眼见国内灾情严重，红十字会决定在 Nlogônia 所有受影响地区对民众施以援手。

为了更好地开展救助行动，红十字会计划在该国建立一个医院基地。然而，他们不确定在哪里建设这个医院。你的团队被委以重任，要找出一个位置，使得基地到最远社区的距离最小。

## 输入格式

输入文件包含多个测试用例。对于每个测试用例：

- 第一行是一个整数 $N$，表示受影响的社区数量。
- 接下来的 $N$ 行，每行包含两个实数 $X$ 和 $Y$，分别表示一个社区的坐标。

多个测试用例依次给出，由 $N = 0$ 表示输入结束。每个测试用例中的社区坐标不重复。

保证：
- $0 \le N \le 100000$
- $-100000 \le X, Y \le 100000$

## 输出格式

对于每个测试用例，输出一行，包括两个用逗号分隔的实数，表示最佳建造医院基地的位置。每个坐标需精确到小数点后四位。

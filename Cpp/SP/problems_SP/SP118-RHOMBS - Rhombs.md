# RHOMBS - Rhombs

## 题目描述

现在又n个整数r $ _{1} $ , r $ _{2} $ , ... , r $ _{n} $ 我们定义一个比较交换指令 CE(a,b)，其中 a、b 是寄存器索引 (1 <= a < b <= n)：
~~~
CE(a, b):: 
 if content(ra) >  content(rb) then 
    exchange the contents of registers ra and rb; 
~~~
一个~~奇妙的~~比较交换程序（简称 CE 程序）是比较交换指令的任何有限序列。 如果执行后寄存器 r$_{1}$ 成立，而且始终包含寄存器中所有值中的最小值，则 CE 程序称为最小查找程序。 如果这样的程序在删除任何单个比较交换指令后仍然是最小查找程序，则该程序被称为可靠的。 

现在给你一个 CE 程序 P，为了获得可靠的最小查找程序，应在程序 P 末尾添加的最小指令数是多少？

例如，考虑与以下 3个整数有关的 CE 程序：CE(1, 2)、CE(2, 3)、CE(1, 2)。 为了使该程序成为可靠的最小值查找程序，仅添加两条指令就足够了：CE(1, 3) 和 CE(1, 2)。
任务：
请你写一个程序，
1. 读取CE程序的描述
2. 计算应在程序 P 末尾添加的最小指令数是多少？
3. 输出结果

## 输入格式

第一行：输入组数d（1<=d<=10)

每一组数据包含2行内容。

第一行包含两个由空格分隔的整数n和m（2<=n<=10000, 0<=m<=25000) n是数组的数量，m是指令的数量。

第二行是由空格分隔的2m个数字（即程序本身）。对于每一个a$_{j}$和b$_{j}$, 有1<=j<=m,1<=a$_{j}$<b$_{j}$<=n. 他们是给你的程序中的第j个指令。

## 输出格式

输出包含d行，每一行对应一组数据。每一行包含一个整数，即让你求的答案。

翻译人歪评：~~这个给你的程序叫CE程序=Compile Error。~~

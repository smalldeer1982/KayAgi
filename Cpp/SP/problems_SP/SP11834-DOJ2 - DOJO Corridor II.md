# DOJ2 - DOJO Corridor II

## 题目描述

在道场的大厅中，有一个 $5 \times N$ 的矩形走廊。此走廊中有一个位置已经被一个魔法半畳（$1 \times 1$ 的正方形）占据。你的任务是放置榻榻米（$1 \times 2$ 的矩形），以覆盖走廊的剩余部分。有时可以完整覆盖，有时则无法做到。

## 输入格式

输入的第一行为测试用例的个数 $T$。接下来的 $T$ 行中，每行包含三个整数 $N$、$I$、$J$，分别表示走廊的长度 $N$ 和魔法半畳的位置坐标 $(I, J)$。请参考样例输入以了解具体格式。

## 输出格式

对于每个测试用例，请输出一种可能的覆盖方案数量。结果需对 $1000000007$ 取模。

## 说明/提示

- $1 \le T \le 100$
- $1 \le N \le 1000$
- $1 \le I \le 5$
- $1 \le J \le N$

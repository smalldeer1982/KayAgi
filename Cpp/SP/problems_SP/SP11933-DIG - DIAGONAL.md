# DIG - DIAGONAL

## 题目描述

给定一个 $n$ 边凸多边形。求所有对角线的交点总数。

假设所有交点都不同。

如果答案超过 $10^9 + 7$，则取模 $10^9 + 7$

$1<=n<=10^8$

## 输入格式

第一行：一个数 $T$ 表示数据的数量。

接下来 $T$ 行，每行包含 $n$ 个顶点。

## 输出格式

指定的对角线交叉点数。

## 样例 #1

### 输入

```
\n2\n4\n5\n&nbsp;
```

### 输出

```
\n\n1\n5\n
```

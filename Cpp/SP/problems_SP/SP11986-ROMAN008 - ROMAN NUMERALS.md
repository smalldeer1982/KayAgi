# ROMAN008 - ROMAN NUMERALS

## 题目描述

现有两个罗马数字以及一个运算符，请输出运算结果，结果也需为罗马数字形式。

## 输入格式

输入包括 5 组测试样例。每组样例由两个用空格分隔的罗马数字和一个运算符组成，均在同一行。运算符可以是 `+`, `-`, `/`, `*`, `%` 这五种之一。

两个罗马数字 $a, b$ 满足 $1 \le a, b < 5000$。对于数值大于等于 4000 的情况，使用 `MMMM` 进行表示。例如：4000 表示为 `MMMM`。

## 输出格式

每组测试样例需要输出其运算结果，结果应小于 5000 且以罗马数字表示。请将每个结果单独一行输出。

## 说明/提示

- 数字 $a, b$ 的取值范围是 $1 \le a, b < 5000$。

### 输入

```
LX XV +
X V *
```

### 输出


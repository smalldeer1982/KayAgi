# TAP2012E - Emma s Domino

## 题目描述

多米诺效应是一种现象，指的是在一排多米诺骨牌中，当第一个骨牌被推倒时，接踵而来的所有骨牌也会依次倒下。

## 输入格式

每个测试用例包括两行。第一行包含两个整数 **N** 和 **H**，分别表示骨牌的总数（满足 $3 \le N \le 1000$）和骨牌的高度（满足 $1 \le H \le 50$）。第二行包含 **N-1** 个整数 **D $ _{i} $**，表示相邻两块骨牌之间的距离，按顺序排列（满足 $1 \le D_{i} \le 100$，对于 $i = 1, 2, \ldots, N-1$）。输入以一行包含两个 **-1** 结束。

## 输出格式

对于每个测试用例，输出一行，表示为了确保所有相邻骨牌之间的距离不超过 **H**，必须移动的最少骨牌数量。注意，第一个和最后一个骨牌是不能移动的，并且必须保持骨牌之间的相对顺序不变。如果无法实现这一目标，输出 `-1`。

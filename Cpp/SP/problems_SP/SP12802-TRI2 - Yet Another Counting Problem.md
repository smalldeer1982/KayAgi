# TRI2 - Yet Another Counting Problem

## 题目描述

你有一根长度为 **n** 单位的铁丝，现在想把它切成若干段，并将每一段折成一个三角形。这些三角形必须满足：所有三角形的边长都是整数，并且它们是相似的。

请计算有多少种不同的方法可以形成这样的三角形。如果两种方法产生的三角形数量不同，或者存在某个位置 $i$，使得一种方法中的第 $i$ 个三角形与另一种方法中的第 $i$ 个三角形不全等，那这两种方法就被认为是不同的。

考虑到答案可能会非常大，所以请输出结果对 $1000000007$ 取模后的值。

**请确保源代码的大小不超过 0.5 KB。**

## 输入格式

每个测试用例由一行组成，包含一个整数 $n$（$1 \le n \le 5,000,000$）。输入将处理到文件结束。

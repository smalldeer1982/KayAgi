# TPORT - Teleport

## 题目描述

小阿龙计划要访问一个星系中的行星。设想这些行星按顺序排列，并依次编号为 **1 到 n**。

他可以使用 **n-1** 个传送装置，这些装置分别标记为 **1 到 n-1**。标记为 **t** 的传送装置只能使用一次，能够将他从编号为 **m** 的行星传送到编号为 **m+t** 或 **m-t** 的行星（如果这样的行星存在）。小阿龙通过一次空间跳跃，可以到达编号为 **k** 的行星，即旅程从编号为 **k** 的行星开始。给定 **n** 和 **k** 后，请找出小阿龙使用传送装置的最佳方案，使他可以尽可能多地访问不同的行星。

### 输入格式

第一行包含两个整数 **n** 和 **k**，满足条件 $1 \le k \le n \le 1\,000\,000$。

### 输出格式

输出小阿龙所使用的传送装置编号 **t**，按照他使用传送装置的顺序依次列出。如果他通过传送从编号较小的行星到编号较大的行星，请输出 **t**；如果是从编号较大的行星到编号较小的行星，则输出 **-t**。

若多种方案中传送装置的使用数量相同，你可以输出任意一种方案。

### 示例

```
输入：
6 2

输出：
4 -5 3 -1 2

注意：这个任务的简化版本曾出现于 ITI 2012, Shumen。
```

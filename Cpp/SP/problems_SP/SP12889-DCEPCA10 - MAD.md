# DCEPCA10 - MAD

## 题目描述

Penny 开始上数学课，她的数学水平有了很大提高。为了挑战 Sheldon，她写下了一串数字组成列表 A，并计算这串数字的中位数。接着，她计算每个数字与该中位数的绝对偏差，形成列表 B。她发现了列表 B 的中位数，然后问 Sheldon：需要最少改变多少个列表 A 的数字，才能让列表 B 的中位数变成 0？你能帮助 Sheldon 解答这个问题吗？

例如：

数字列表（A）：4, 5, 3, 1, 2  
A 的中位数 = 3  
绝对偏差列表（B）：1, 2, 0, 2, 1  
B 的中位数 = 1

**注意：偶数长度的列表中位数是中间两个值的平均值。**

## 输入格式

输入由 $T$ 组测试数据组成。每组测试数据有两行：

- 第一行包含一个整数 $N$，表示数组的大小。
- 第二行包含 $N$ 个整数，代表数组 A 中的每个数字。

## 输出格式

输出共 $T$ 行，每行包含一个整数，表示将绝对偏差列表 B 的中位数变为 0 所需改变的最少元素数量。

## 说明/提示

- $1 \le T \le 100$
- $1 \le N \le 10^3$
- $1 \le A[i] \le 10^6$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2\n5\n4 5 3 1 2\n10\n5 5 5 254 5 5 768 5 5 5\n\n&nbsp;
```

### 输出

```
2\n0
```

# RIOI_3_2 - Counting

## 题目描述

# RIOI_3_2 - Counting




给定整数 $N$ 和 $M$ ，输出有多少种方式可以选取 $N$ 个不同的正整数，使得这些整数的和小于等于 $M$ 。由于结果可能非常大，请输出结果模 **1000000007 (10^9 **+ 7)****

$N  \le  20$

$M \le 100000$

## 输入格式

输入的第一行是数字 $t$ ，表示测试样例的数量。每个测试案例仅包含两个数字 $N$ 和 $M$ ，按顺序排列。

## 输出格式

输出描述中要求的数字。

## 样例 #1

### 输入

```
3
6 16
4 164
6 16
4 16
1 14
3 7
1 14
```

### 输出

```
0
```

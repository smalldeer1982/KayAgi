# VISION - Vision Field

## 题目描述

给定一组建筑物的信息，以及若干个查询。对于每个查询，输出你可以看到的建筑物的数量。

## 输入格式

第一行是一个整数 $n$，表示建筑物的数量。

第二行包含 $n$ 个整数 $A_1, A_2, \ldots, A_n$，分别表示每个建筑物的高度。

接下来一行是一个整数 $m$，表示查询的数量。

## 输出格式

对于每个查询，输出一个整数，表示在该查询中你可以看到的建筑物数量。每次输出占一行。

## 说明/提示

- $1 \leq n \leq 10^5$
- $1 \leq m \leq 10^5$
- $1 \leq A_i \leq 10^9$ 

请根据输入的数据范围，设计一个高效的算法来解决问题。

5

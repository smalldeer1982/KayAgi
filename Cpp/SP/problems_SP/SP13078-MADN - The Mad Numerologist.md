# MADN - The Mad Numerologist

## 题目描述

**疯狂的数字命理学家**

数字命理学是一门揭示个体性格、生命目标及内心渴望的科学。其计算过程有的复杂，但有的相对简单，甚至可以在家中独自完成。本题中，我们不要求计算名字的价值。

现代数字命理学家为所有英文字母分配了数值，A、E、I、O、U 这五个字母是元音，其余为辅音。表中第一列的字母值为 1，第二列的值为 2，依此类推。因此，T 的值为 2，F 的值为 6，R 的值为 9，而 O 的值为 6。在计算名字的数值时，元音和辅音分别计算。下图通过名字 "CHRISTOPHER RORY PAGE" 示范了这种方法。

为了给人们推荐幸运名字，一个疯狂的数字命理学家制定了以下规则：

- 名字的长度为预先确定的 $N$。
- 名字的元音值和辅音值需要尽可能小。
- 为便于发音，元音和辅音交替分布：元音放在奇数位置，辅音放在偶数位置。名字的第一个字母位置为 1，第二个位置为 2，依此类推。
- 每个辅音最多使用五次，而每个元音最多使用二十一遍。
- 在符合上述规则的前提下，名字还必须是字典序最小的。

## 输入格式

第一行输入一个整数 $N$（$0 < N \leq 250$），表示输入的组数。接下来的 $N$ 行，每行包含一个整数 $n$（$0 < n < 211$），表示名字的长度。

## 输出格式

对于每组输入，输出一行，格式为：输出序号，接着是按照规则生成的名字，所有字母均为大写。

**示例**

**输入：**

```
3
1
5
5
```

**输出：**

```
Case 1: A
Case 2: AJAJA
Case 3: AJAJA
```

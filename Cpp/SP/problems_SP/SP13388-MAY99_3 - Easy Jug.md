# MAY99_3 - Easy Jug

## 题目描述

给出 $2$ 个容量分别为 $x$ 升与 $y$ 升的水桶，你可以做以下操作：

1. 将一个桶从水井装满水。

2. 将一个桶的水全部倒到水井里。

3. 把一个桶里的水尽可能多地倒到另一个桶里，直到其中一个桶被装满或被彻底清空。

问你是否能操作这两个水桶，得到**恰好** $z$ 升水。

## 输入格式

**本题有多组数据**。

第一行输入一个整数 $T$，表示数据组数。

对于每组数据：

一行三个整数 $x,y,z$。

## 输出格式

对于每组数据，如果可以得到恰好 $z$ 升水，输出 `YES`；否则输出 `NO`。

## 说明/提示

对于 $x=9,y=3,z=6$ 的情况，我们可以做如下操作：

1. 把容量为 $9$ 升的桶从水井装满水。

2. 把容量为 $9$ 升的桶里的水尽可能多地倒到容量为 $3$ 升的桶里。

此时，容量为 $9$ 升的桶里恰好有 $6$ 升水，所以输出 `YES`。


对于 $100\%$ 的数据，$T \leq 25$，$1 \leq x,y,z \leq 10^8$。

## 样例 #1

### 输入

```
5

```

### 输出

```
NO
YES
YES
YES
NO
```

# CHAOS_CC - Chaos In Arkham

## 题目描述

阿卡姆疯人院内一片混乱！  
班恩和小丑正在监狱中自由行动，并释放其他囚犯。整个监狱由 $N$ 个牢房排成一行。为了证明谁能更胜一筹，他们进行了一场游戏：两人轮流行动，每位玩家每次可以打开一个牢房。他们定义，当连续三个牢房被打开时，就形成了一个“犯罪洞”。谁先创建出“犯罪洞”谁就获胜，并被称为哥谭市的终极反派。小丑率先行动。  
众所周知，布鲁斯·韦恩总是密切关注他的对手。在这种情况下，他需要一个程序来预测当两人都采取最佳策略时，谁将成为最终的反派。

## 输入格式

第一行是测试用例的数量 $T$，接下来的 $T$ 行每行为一个整数 $N$，代表敌方营地（或牢房）的数量。

## 输出格式

对于每个输入的 $N$，如果班恩会获胜则输出 `Bane`，否则输出 `Joker`。（注意引号仅用于说明，不包括在实际输出中）

## 说明/提示

- $1 \le T \le 3333$
- $3 \le N \le 3333$

## 示例

### 输入：

```
3
3
5
6
```

### 输出：

```
Joker
Joker
Bane
```

### 解释：

用 'o' 表示打开的牢房，'-' 表示关闭的牢房。  
第一种情况：最初所有的牢房都是关闭状态（---），小丑先打开第一个牢房，配置变为 (o--)。班恩可以选择打开第二个牢房，配置变为 (oo-)；或打开第三个牢房，配置变为 (o-o)。在这两种情况下，小丑都能在下一轮中轻松获胜。  
第二种情况：小丑可以通过先打开第三个牢房来获胜，即配置为 (--o--)。

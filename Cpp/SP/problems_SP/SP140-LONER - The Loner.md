# LONER - The Loner

## 题目描述

给出一个一维棋盘，棋子能且只能通过跳过其左边或右边的棋子移动，被跳过的棋子被移除，给出一个一维棋盘状态，问能否通过移动棋子若干步使棋盘上只剩一个棋子。

## 输入格式

第一行一个正整数 $t$ 表示有 $t$ 组数据。

每个测试数据以 $n$ $(n<= 32000)$ 开头。
之后一个包含 $n$ 个数字的序列，序列中无空格，数字均为 $0$ 或 $1$，为 $0$ 表示该位置无棋子，为 $1$ 表示该位置有棋子。

## 输出格式

对于每个测试用例，输出一行一个字符串 `yes` 或 `no` 。表示玩家能否使棋盘上只剩一个棋子。

## 样例 #1

### 输入

```
2
7
0110011
6
111001
```

### 输出

```
yes
no
```

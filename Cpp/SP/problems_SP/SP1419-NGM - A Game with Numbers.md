# NGM - A Game with Numbers

## 题目描述

### 题意翻译
有 $n$ 个石子，两人按顺序取石子，每次可取 $1$~$9$ 个石子，问谁有必胜策略。

## 输入格式

一个正整数 $n$（$1\le n\le 2,000,000,000$），表示石子的个数。

## 输出格式

第一行一个整数，若为 $1$ 代表先手有必胜策略，若为 $2$ 代表后手有必胜策略。

如果先手必胜，第二行输出一个 $1$ 到 $9$ 之间的正整数，表示必胜策略下先手第一次拿的石子数。

## 样例 #1

### 输入

```
14
```

### 输出

```
1
4
```

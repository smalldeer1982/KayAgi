# POWERPHI - Power of Phi(medium)

## 题目描述

求最接近 $\varphi^n$ 的整数，其中 $\varphi=\dfrac{\sqrt{5}+1}{2}$。

## 输入格式

第一行一个正整数，为测试用例的数量 $T$ $(1<=T <= 100000)$。

接下来 $T$ 行，每行包含一个正整数 $n$ $(0 <= n <= 10^9)$。

## 输出格式

$T$ 行，对于每个询问，输出最接近 $\varphi^n$ 的整数，对 $1000000007$ 取模。

Translated by @Dreamweaver

## 样例 #1

### 输入

```
2
1
3
```

### 输出

```
2
4
```

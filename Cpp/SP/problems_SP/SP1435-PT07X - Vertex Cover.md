# PT07X - Vertex Cover

## 题目描述

给定一棵无权无向树，求出这棵树的最小点覆盖。

## 输入格式

第1行：一个正整数$N$，表示这棵树的大小
接下来$n-1$行，每行2个正整数$x,y$，表示结点$x,y$之间有一条边

## 输出格式

输出共1行，一个正整数，表示最小点覆盖的点个数。
## 数据规模与约定
$n\le10^5$

## 样例 #1

### 输入

```
3
1 2
1 3
```

### 输出

```
1
```

## 样例 #2

### 输入

```
3
1 2
2 3
```

### 输出

```
1
```

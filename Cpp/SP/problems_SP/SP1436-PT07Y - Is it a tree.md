# PT07Y - Is it a tree

## 题目描述

有一个N个点M条边的图。 请问，该图是否为树。

## 输入格式

第1行2个数N,M。

接下来M行，每行2个数A,B，表示一条边（A,B）。

## 输出格式

如果该图为树，输出“YES”，否则输出“NO”。

数据范围： 0<N<=10000 0<=M<=20000

感谢@qq1010903229 提供的翻译

## 样例 #1

### 输入

```
3 2
1 2
2 3
```

### 输出

```
YES
```

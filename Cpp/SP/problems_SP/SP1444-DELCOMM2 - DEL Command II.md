# DELCOMM2 - DEL Command II

## 题目描述

需要确定在 MS-DOS 目录中，只通过执行一次 DEL 命令最多可以删除多少个文件。目录中没有嵌套的子目录。

## 输入格式

输入的第一行是一个整数 $M$，表示有 $M$ 组数据集。各组数据之间用空行隔开。

每组数据集由若干文件全名组成，每个文件名独占一行，行末带有控制符：`+` 表示该文件需要删除，`-` 表示该文件需要保留。文件名在每个数据集中不重复。每组数据至少包含一个文件，并且至少有一个文件标记为待删除。每组数据集里的文件总数最多为 250 个。

## 输出格式

对于每组数据集，输出只用一次 DEL 命令可以删除的最大文件数量。

## 说明/提示

- 数据集数 $1 \le M \le 100$
- 每组数据集中文件的数量不超过 250 个。

### 输入

```
2

BP +
BPC +
TURBO -

EXCHANGE +

MOUSE &#8211;

```

2

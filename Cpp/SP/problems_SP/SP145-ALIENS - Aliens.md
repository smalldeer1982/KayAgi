# ALIENS - Aliens

## 题目描述

给出 $n$ 个点，求将 $n$ 个点包含在内(包括边界)的最小圆的半径和圆心坐标。

## 输入格式

输入的第一行包含一个整数 $c$ $(c<=20)$ 表示测试数据组数。

每组测试数据第一行 $n$ $(2<=n<=100000)$ 表示共有 $n$ 个点。

之后 $n$ 行，每行包含两个实数 $x_i$ 和 $y_i$ $(-10000.0<=xi,yi<=10000.0)$，表示第 $i$ 个点的坐标。

## 输出格式

对于每组测试数据，输出两行，第一行为最小圆的半径，第二行为最小圆的坐标，数字应四舍五入到小数点后两位。

## 样例 #1

### 输入

```
1
6
8.0 9.0
4.0 7.5
1.0 2.0
5.1 8.7
9.0 2.0
4.5 1.0
```

### 输出

```
5.00
5.00 5.00
```

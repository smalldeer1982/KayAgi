# FOXLINGS - Foxlings

## 题目描述

森林里的狐狸和狼家族正在庆祝圣诞节。庞大的狐狸家族包括了两位家长以及 $N$（$1 \leq N \leq 10^9$）只小狐狸。家长们决定今年给孩子们送一个特别的礼物——饼干！毕竟，狐狸格外喜欢饼干。

然而，由于家庭成员众多，家长们不得不在发放数量上精打细算。他们希望尽量减少发出的饼干数量，同时确保每只小狐狸都能分到一小块。家长们只能将饼干整块发出，而这些饼干可以被子女们分开并传递给伙伴。

在这个大家庭中，并不是所有小狐狸都互相熟识。每只小狐狸虽然都有名字，但由于父母是计算机科学家，他们还顺便给每只小狐狸从 $1$ 到 $N$ 编了号。在这些小狐狸之间存在 $M$（$1 \leq M \leq 10^5$）对独特的双向友谊，友谊关系 $i$ 由两个不同的整数 $A_i$ 和 $B_i$（$1 \leq A_i, B_i \leq N$）表示，说明编号为 $A_i$ 的小狐狸和编号为 $B_i$ 的小狐狸是朋友，反之亦然。当一只小狐狸得到一块饼干时，它可以用尾巴将其准确地分成多个部分（狐狸的尾巴可有许多神奇的用途）。然后，它可以把这些饼干片传给它的朋友们，朋友们也可以继续这样做。

第一行：两个整数 $N$ 和 $M$。

接下来 $M$ 行：每行包含两个整数 $A_i$ 和 $B_i$，表示 $i$ 对友谊关系。

## 输出格式

一个整数，代表必须发放的最小饼干数量，使得每只小狐狸最终都能获得至少一小块饼干。

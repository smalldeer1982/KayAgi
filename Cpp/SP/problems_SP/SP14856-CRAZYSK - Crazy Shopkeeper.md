# CRAZYSK - Crazy Shopkeeper

## 题目描述

在充满活力的吉大港，有一位特别的商人名叫 Zoglu Bepari。大家都非常喜欢他，并亲切地称他为「MAMU」。他经营着一家小巧而美丽的店铺，被称为「MAMUR DOKAN」。从早到晚，他几乎都在不间断地营业。他的店铺总是人满为患，因为他常常推出各种怪异和疯狂的优惠活动。这些优惠对顾客而言非常有吸引力。

最近，MAMU 推出了一项特别的优惠活动：每售出一件商品，他都会送给顾客一张特别的卡片。这些卡片非常稀有，只能在他的店里获得，且没有人能用假卡片来欺骗他，因为他有一套很好的验卡本领。假设某位顾客购买了 **X** 件商品，那么 MAMU 会给这位顾客一个大于 1 的正整数 **N**，以及 **X** 张卡片。再有一个特别之处在于，顾客可以使用 **N** 张卡片来换取一件同样类型的商品，并能额外得到一张卡片。

作为程序员，你需要计算在使用最多数量的卡片后，顾客总共能获得多少件商品。

## 输入格式

输入从一个正整数 **T** 开始（$T < 100$），表示接下来的测试用例数量。随后有 **T** 行，每行包含两个正整数：首次交易购买的商品数量 **X**（$1 < X \leq 2^{64}$），以及 MAMU 给出的整数 **N**（$1 < N \leq X$）。

## 输出格式

输出每位顾客在利用到最多张卡片之后，所能获得的商品总数。

# SUBSN - Subsequence

## 题目描述

在一个整数序列中，子序列是通过删除一些元素（这些元素不必连续）从原序列中派生出来的新序列。现有一个长度为 $n$ 的整数序列 $a_1, a_2, \ldots, a_n$，请你求出该序列的最长严格递增子序列的长度。

## 输入格式

第一行输入一个整数 $n$，表示序列的长度。
第二行输入 $n$ 个整数，分别是 $a_1, a_2, \ldots, a_n$，表示序列中的元素。

## 输出格式

输出一个整数，表示最长严格递增子序列的长度。

## 说明/提示

$$1 \le n \le 10^5, \quad 1 \le a_i \le 10^9$$

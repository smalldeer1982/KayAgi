# CAC - Cactus

## 题目描述

在图论中，生成树是一个包含了图中所有顶点的连通无向子图，并且没有环。

## 输入格式

第一行是一个整数，表示测试用例的个数。每个测试用例的第一行包含两个整数 $N$ 和 $E$，分别代表仙人掌图的顶点数和边数，其中顶点编号从 1 到 $N$ 且 $3 \le N \le 81$，$2 \le E \le 120$。接下来的 $E$ 行，每行两个整数 $v$ 和 $w$，表示顶点 $v$ 和顶点 $w$ 之间存在一条边。

## 输出格式

对于每个测试用例，输出一行：“Case C: X”，其中 $C$ 是测试用例的编号（从 1 开始），$X$ 是将给定仙人掌图变为生成树的可能方案数。

## 说明/提示

- 顶点个数 $N$ 的范围为 3 到 81。
- 边的数量 $E$ 的范围为 2 到 120。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2
3 3
1 2
2 3
1 3
5 5
1 2
2 3
2 4
3 4
4 5
```

### 输出

```
Case 1: 3
Case 2: 3
```

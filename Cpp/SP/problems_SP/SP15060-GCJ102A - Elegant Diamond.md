# GCJ102A - Elegant Diamond

## 题目描述

给定若干个测试用例，每个测试用例中有一个钻石，这些钻石需要通过调整变得更加优雅。你的任务是计算出将钻石变为优雅钻石所需的最小成本。

## 输入格式

第一行包含一个整数 **T**，表示测试用例的数量。接下来的 **T** 个测试用例中，每个都包括一个整数 **k**，表示钻石的大小，紧随其后的是一个大小为 **k** 的钻石阵列。

## 输出格式

对于每个测试用例，输出格式为 "Case #x: y"，其中 x 是当前测试用例的编号（从 1 开始），y 是将给定钻石转化为优雅钻石所需的最小成本。如果给出的钻石已经是优雅的，则 y=0。

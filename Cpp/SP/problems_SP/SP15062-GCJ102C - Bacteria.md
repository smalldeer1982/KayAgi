# GCJ102C - Bacteria

## 题目描述

给定多个测试用例，每个测试用例需要处理一组包含细菌的矩形区域。

## 输入格式

输入由以下几部分组成：

- 首行一个整数 $C$，代表测试用例的数量。

对于每个测试用例：

- 首行一个整数 $R$，表示初始含有细菌的矩形区域的数量。
- 接下来的 $R$ 行，每行包含四个用空格分隔的整数 $X_1, Y_1, X_2, Y_2$。表示从 $X_1$ 到 $X_2$，以及从 $Y_1$ 到 $Y_2$ 之间的所有坐标点（包括边界）都含有细菌。

这些矩形可能会相互重叠。

注意：增加 $Y$ 坐标表示向南，增加 $X$ 坐标表示向东。

## 输出格式

对于每个测试用例，输出格式为 "Case #N: T"，其中 $N$ 是测试用例的编号（从 1 开始计数），$T$ 是细菌全部死亡所需的时间（以秒为单位）。

## 说明/提示

- 测试用例数 $1 \leq C \leq 100$
- 每个测试用例的矩形数量 $1 \leq R \leq 1000$
- 矩形的边界 $1 \leq X_1 \leq X_2 \leq 10^9$
- 矩形的边界 $1 \leq Y_1 \leq Y_2 \leq 10^9$

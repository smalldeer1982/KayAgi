# POP2 - play with prime numbers (II)

## 题目描述

质数是指除了 $1$ 以及自身没有其它正因子的数。我们定义“质数构成的质数”（简称 POP）是指只包含小于其自身的质数构成的质数。

例如，$1013$ 包含 $101$ 和 $3$ 两个质数，是 POP。

注意：形如 $2003$ 这样的数不是 POP，因为 POP 不允许前导 0。

POP 必须包括至少 $2$ 个质数，且不允许重叠。

## 输入格式

第一行是一个整数 $T$，表示数据组数。

接下来的 $T$ 行，每行是一个整数 $m$。$0 \le m \le 10^{18}$

## 输出格式

对于每组数据，请输出第一个大于等于 $m$ 的 POP。

## 样例 #1

### 输入

```
3
10
100
1000
```

### 输出

```
23
113
1013
```

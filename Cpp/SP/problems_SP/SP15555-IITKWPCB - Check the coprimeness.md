# IITKWPCB - Check the coprimeness

## 题目描述

求小于等于与 $N$ 互质的小于 $\lfloor N/2\rfloor$ 的最大正整数 $ans$。

如果 $a$ 与 $b$ 的最大公因数为 $1$，则 $a$ 与 $b$ 互质。

## 输入格式

第一行一个整数 $T$，表示数据组数（$T \le 1000$）

对于接下来的 $T$ 行，每行包含一个整数 $N$（$1 \le N \le 10^{12}$）

## 输出格式

$T$ 行，每行一个整数 $ans$。

## 样例 #1

### 输入

```
4
3
4
5
100
```

### 输出

```
1
1
2
49
```

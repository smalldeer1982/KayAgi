# IITKWPCM - Coprime Again

## 题目描述

对于给定正整数 $n$，请输出从 $1$ 到 $n$ 中所有不与 $n$ 互质的数的乘积。

## 输入格式

第一行一个整数 $T$（$1\le T\le 10^4$），表示数据组数。

每组数据共一行一个整数 $n$（$1\le n\le 10^{18}$），意义如上所述。

## 输出格式

对于每个 $n$，输出一行一个整数表示答案。答案对 $n$ 取模。

Translated by @此用户无昵称

## 样例 #1

### 输入

```
\n2\n2\n5\n
```

### 输出

```
\n1\n4
```

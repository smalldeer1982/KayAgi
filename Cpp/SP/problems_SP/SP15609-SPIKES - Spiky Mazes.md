# SPIKES - Spiky Mazes

## 题目描述

贾姆廷对文化及其背后的历史感兴趣。当然，这种兴趣是有原因的：当他研究合唱团的过去时，他发现了他所知道的迷宫的隐藏入口，其中包含着有价值的信息。然而，有一个问题：迷宫里有尖刺陷阱！贾姆廷是一个非常敏捷的人，但每个人都有一个极限，因此他只能避免一些陷阱。这就引出了一个问题：他能穿过迷宫吗？

## 输入格式

测试用例的第一行包含三个整数 $n$ 、 $m$ 和 $j$ 。$n（2≤n≤40）$表示行数，$m$ （$2≤n≤40$）表示每行的宽度，$j（0≤j≤20）$表示贾姆廷可以避免尖峰的次数。然后 $n$ 行包含 $m$ 个字符；字符“x”将用于宝藏的位置，“@”用于入口（也是出口），“s”代表钉鞋，“#”代表墙壁，“.” 安全的行走瓷砖。请注意，你不能走进墙壁，迷宫完全被你能看到的墙壁包围着。**总是至少有一个入口/出口**，宝藏所在的地方总是有一个"x"。

## 输出格式

如果贾姆廷可以让它活着进出，你应该输出“SUCCESS”，如果你无法让它活着出来，则输出“IMPOSSIBLE”。

## 输入输出样例
### 样例输入 #1

```
3 3 2
#@#
#s#
#x#
```

### 样例输出 #1

```
SUCCESS
```

### 样例输入 #2

```
4 4 3
####
@.s#
##.#
#xs#
```

### 样例输出 #2

```
IMPOSSIBLE
```

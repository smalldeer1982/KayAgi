# IITKWPCO - Create Collections

## 题目描述

小 Feluda 非常喜欢玩数字。现在，他手里有 $n$ 个数，他想把这些数分成若干组，每组有两个数。只有当其中一个数是另一个数的一半时，这两个数才能组成一组。

给定 $n$ 个数，问最多可以组成多少组符合条件的数对？

## 输入格式

第一行为测试用例的数量 $T$。（$1 \le T \le 100$）

接下来对于每个测试用例：

第一行包含一个整数 $n$，表示有多少个数。（$1 \le n \le 100$）

第二行包含 $n$ 个用空格分隔的正整数，每个数值的范围是 $1$ 到 $10^6$。

## 输出格式

对于每个测试用例，输出最多可以组成的数对数量。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2\n2\n1 2\n3 \n1 2 4\n
```

### 输出

```
\n1\n1
```

# FACTMUL - Product of factorials

## 题目描述

You need to find the product of first n factorials 1! \* 2! \* ... \* n! modulo 109546051211.

## 输入格式

One integer n (1 <= n <= 10000000)

## 输出格式

The answer.

## 样例 #1

### 输入

```
5
```

### 输出

```
34560
```

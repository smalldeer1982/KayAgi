# HPPROF - Help Professor

## 题目描述

某天，DA-IICT 的一名课程助教正在批改第一次期中考试的试卷，考试总分是 20 分。助教在批改了部分试卷后，打算将这些学生的分数交给教授，但他犯了一个错误：忘记在分数之间添加空格，导致教授无法分辨每个学生的具体得分。

由于你是他最得意的学生，请帮助教找出可以将这些分数正确分开的方法数，也就是说，分割后得到的分数必须是有效的。

有效的分数范围是 0 到 20，且不允许有前导零。例如，0、1、2、3...20 是有效的分数，而 01、003、21、25 则无效。

## 输入格式

输入为一行字符串，长度小于 20。

## 输出格式

输出可以通过插入空格将字符串分割为有效分数的方法数。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
1 :
12
```

### 输出

```
1 :
2\n\nInput2 :
111

Output2 :
3
```

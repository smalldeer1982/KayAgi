# DCEPC11B - Boring Factorials

## 题目描述

Sameer和Arpit想克服他们对数学的恐惧，所以他们最近一直在练习数学题。他们的朋友Aman一直在帮助他们。但随着时间的推移，Sameer和Arpit已经厌倦了关于阶乘的问题。因为阶乘只需要模某些素数的余数，而这很容易计算。为了让他们更感兴趣，数学学家阿曼给了他们一个有趣的任务。他给他们一个素数 $P$ 和一个接近 $P$ 的整数   $N$，并让他们找到$N! \mod P$ 的值。他问了 $T$ 个这样的问题，请你帮助他们解决一下。

## 输入格式

第一行输入一个整数 $T$，表示问题数量。

接下来的 $T$ 行，每行都有两个整数 $N$，$P$。

## 输出格式

共 $T$ 行，每行一个整数表示 $N! \mod P$ 的值。

## 样例 #1

### 输入

```
3
2 5
5 11
21 71
```

### 输出

```
2
10
6
```

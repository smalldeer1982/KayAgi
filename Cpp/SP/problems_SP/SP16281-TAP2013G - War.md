# TAP2013G - War

根据双方达成的协议，战争将通过一系列连续的战斗进行。在每场战斗中，来自两个帝国的不同士兵将一对一地交锋，每名士兵只会参与一场战斗。胜利更多的一方将赢得整场战争。

每个帝国的军队由 **S** 名士兵组成，每位士兵都有其独特的战斗力。在每场单挑中，战斗力更高的士兵将获胜；若战斗力相同，则判定为平局，等同于没人获胜。Nlogonia 的间谍成功截获了 Quadradonia 军队中每位士兵的战斗力数据，因此 Nlogonia 女王需要你的帮助，以便她能合理安排士兵出战顺序，最大化地赢得战斗。

## 输入格式

第一行是一个整数 **S**，表示每支军队的士兵数量（$1 \leq S \leq 10^5$）。

# ICC - Sir and The ICC Ratings

## 题目描述

众所周知，Sir Jadeja 一直以来凭借一己之力横扫对手，他在 ICC 排名中也是首屈一指的。某天，他在 [Cricinfo.com](http://www.espncricinfo.com/ci/content/player/234675.html) 上查看自己的职业生涯评分图，注意到了其中的峰值变化。他对此很有兴趣，于是决定设计出许多假设的图表。在这些图表中，任何时刻评分图的斜率都只有两种可能：要么是 45°（代表上升），要么是 -45°（代表下降），并且评分只取非负整数。

因为是假设的图表，他决定从评分 0 开始，最后也回到 0。在开始和结束之间，总共用时 2N 个单位（沿水平轴），在这段时间内有 K 个峰值。现在，他想计算在给定 N 和 K 的情况下，可能的不同图表的总数。由于结果可能很大，请输出对 $10^9 + 7$ 取模的值。你既然也擅长数学，请帮助 Sir 解决这个问题。

## 输入格式

第一行是测试用例的数量 T（$1 \leq T \leq 1000000$）。接下来的每个测试用例包含两个整数，N 和 K。

$1 \leq K \leq N \leq 1000$。

## 输出格式

对于每个测试用例，输出符合条件的不同图表的总数，结果需对 $1000000007$ 取模。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
1
5 2
```

### 输出

```
10
```

# PGTV - Pengun Tivi

## 题目描述

某电视台收到了一共有 $N$ 个客户的节目请求，每个请求都进行了编号，从 $1$ 到 $N$。对于第 $i$ 个请求，包含了节目的开始时间 $s(i)$ 和结束时间 $t(i)$。需要注意的是，不同节目之间的切换时间可能不同。

你的任务是计算出至少需要多少个频道，才能确保所有客户的节目请求都能被满足。

## 输入格式

第一行输入一个整数 $N$，表示请求的数量（$0 < N \le 10^5$）。

接下来的 $N$ 行中，每行包含两个整数，分别表示第 $i$ 个节目的开始时间 $s[i]$ 和结束时间 $t[i]$（$0 \le s[i], t[i] \le 2 \times 10^9$）。

## 输出格式

输出一个整数，表示满足所有客户请求所需的最少频道数量。

### 输入

7

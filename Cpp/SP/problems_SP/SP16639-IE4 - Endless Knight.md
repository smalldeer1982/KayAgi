# IE4 - Endless Knight

## 题目描述

马从 $(1, 1)$ 跳到 $(h, w)$，只能往右下跳，不能跳到给定的 $r$ 个点，输出方案数。结果对 $10007$ 取模。

## 输入格式

第一行 $n,(0\le n\le100)$ 为数据组数，每组数组第一行三整数 $h,w,(0\le h,w\le10^8),r,(0\le r\le10)$，此后 $r$ 行为禁止点的坐标。

## 输出格式

输出 $n$ 行，每行格式为 `Case #<序号>: <答案>`，序号从一开始。

## 样例 #1

### 输入

```
5
1 1 0
4 4 1
2 1
3 3 0
7 10 2
1 2
7 1
4 4 1
3 2
```

### 输出

```
Case #1: 1
Case #2: 2
Case #3: 0
Case #4: 5
Case #5: 1
```

# GROCERY - Grocery store

## 题目描述

某些奇奇怪怪的收银员似乎很难区分乘法符号和加法符号。为了让他们更容易理解，现在请你找到所有的四种商品的可能价格， **使得四个价格的总和等于四个价格的乘积**。价格用€表示，并且在小数点后保留两位数字。商品的价格必须大于零。

## 输入格式

这个问题没有输入。

## 输出格式

输出所有四项的总和不超过 $20.00$ 欧元的答案。每个解决方案均占一行，每行行中按 **非降序** 显示四项商品的价格，每两个价格的中间用一个空格隔开。

可以按任何顺序输出解决方案，但请确保每个解决方案只输出一次。
***
这里提供三种可选的方案：

```
0.50 1.00 2.50 16.00
1.25 1.60 1.75 1.84
1.25 1.40 1.86 2.00
...
```

# SCALE - Funny scales

## 题目描述

Kinh_Can 拥有一套珍贵的砝码，分别为 $P_1, P_2, \ldots, P_N$，其中第 $i$ 个砝码的质量是 $P_i = 3^{i-1}$。同时，他也有一台具有两个托盘的天平。某天，Kinh_Can 决定向朋友们炫耀他的这套砝码，并声称他可以用它们使任何质量不超过这些砝码总质量的物体与之平衡。最初，朋友们表示怀疑，但经过多次尝试，他们发现 Kinh_Can 说的是事实。更神奇的是，当他将质量为 $X$ 的物体放在一个托盘上时，他立刻就能无误地在两个托盘上添加合适的砝码以保持天平的平衡。

现在，给定一个重量 $X$（$X$ 是一个自然数，且 $X \neq 0$），你的任务是像 Kinh_Can 一样，通过放置砝码来使得两个托盘保持平衡。初始时，天平的第一个托盘上放置了质量 $X$ 的物体，第二个托盘上没有任何重量。

## 输入格式

输入只有一行，包括两个整数 $N$ 和 $X$。

## 输出格式

- 如果没有可行的解决方案，输出 `-1`。
- 如果至少有一个解决方案，请输出恰好两行：
  - 第一行列出放在第一个托盘上的砝码的编号（下标从1开始）。
  - 第二行列出放在第二个托盘上的砝码的编号。
  - **注意**：其中一行可以为空。

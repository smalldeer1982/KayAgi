# OPBIT - Operation Bits

## 题目描述

秘密团队正在进行一个有关安全加强的项目，在此过程中新发现了一种操作方式。团队负责人 Abay 先生在完全平方数中找到了一个新模式。由于这种模式的安全性较低，可以作为项目的外层保护。因此，他希望你解决以下问题，从而找出密钥：

“对于两个相邻的完全平方数，除非它们相等，其绝对差始终为奇数。你的任务是找到密钥，其定义为：

$$\text{key}(a, b) = ( \text{AND}(\text{在 } [a, b] \text{ 范围内每一对相邻完全平方数的绝对差})) \  \text{和} \  (\text{XOR}(\text{在 } [a, b] \text{ 范围内每一对相邻完全平方数的绝对差}))$$

请根据输入的值计算密钥吧 :)

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量 ($1 \le T \le 1000$)。

接下来的 $T$ 行中，每行输入两个整数 $a$ 和 $b$，满足 $0 < a \le b \le 10^6$。

可以保证 $a$ 和 $b$ 都是完全平方数。

## 输出格式

对于每一个测试用例，输出相应的密钥。

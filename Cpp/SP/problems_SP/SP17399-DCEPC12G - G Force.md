# DCEPC12G - G Force

## 题目描述

定义 $P(n)$ 为所有小于等于 $n$ 的素数的个数，定义 $T(n)$ 为所有小于等于 $n$ 且与 $n$ 互素的正整数的个数，则 $F(n)=P(n)-T(n)$。此外，若 $F(n)<0$，则视 $F(n)$ 为 $0$。

给定正整数 $n$，求 $T(n)^{F(n)!} \bmod 1000000007$，其中 $F(n)!$ 表示 $F(n)$ 的阶乘。

## 输入格式

第一行为一个正整数 $T$，表示测试数据组数。

接下来 $T$ 行，每行一个整数表示 $n$。

## 输出格式

输出 $T$ 行，每行一个整数表示答案。

## 说明/提示

对于 $100\%$ 的数据，$1\le T\le 100$，$1\le n\le 10^7$。

## 样例 #1

### 输入

```
1
2
```

### 输出

```
1
```

# DIVSUM2 - Divisor Summation (Hard)

## 题目描述

给定一个自然数 $n \ (1 \le n \le 10^{16})$，请输出其所有小于自身的约数的和。

例如：$20$ 含有 $5$ 个小于自身的约数：$1, 2, 4, 5, 10$，因此其小于自身的约数之和为 $1 + 2 + 4 + 5 + 10 = 22$。

## 输入格式

第一行包含一个整数 $T \ (T = 500)$ 代表测试数据组数。

接下来 $T$ 行每一行包含一个一个整数 $n \ (1 \le n \le 10^{16})$。

## 输出格式

对于每组数据输出其小于自身的约数之和。

## 样例 #1

### 输入

```
3
2
10
20
```

### 输出

```
1
8
22
```

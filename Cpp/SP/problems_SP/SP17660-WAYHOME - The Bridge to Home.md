# WAYHOME - The Bridge to Home

## 题目描述

### 题意翻译 

晚上，一群人为了回家，要经过一座小桥。小桥每次最多只能走两个人，而且过桥时必须要带一盏灯。（灯开始时在这群人手上，且只有一盏灯） 

现在，这群人给出单独一个人过桥的时间，并向你求助，他们最少需要多久过桥。

如果有两个人同时过桥，则取过桥时间最大的。

## 输入格式

第一行：一个数$C$，代表组数。

接下来$C$行，每行第一个数$N$代表组内人数；接下来$N$个排序过的数，$A_{i}$是第$i$个人过桥的时间 

$0 ≤C ≤ 100$; $1 ≤N ≤ 1000$; $0 ≤A _{i} ≤ 1000$

## 输出格式

$C$行，第$i$行代表第$i$组人最少需要多久过桥

## 样例 #1

### 输入

```
4
1 1
2 1 2
3 1 1 1
4 1 2 4 8
```

### 输出

```
1
2
3
15
```

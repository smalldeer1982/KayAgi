# SPCO - Gopu and Counting Bitwise Prime Numbers

## 题目描述

### 题意翻译
如果一个正整数每一位加起来的和是素数，那么这个数被称为“位和素数”。

## 输入格式

第一行包含一个整数$T(1\le T\le10^5)$为测试样例的数量。

对于接下来的T行，每个测试样例包含两个空格分隔的整数$a$和$b$。

## 输出格式

对于每个测试样例，输出包括$a$和$b$之间的“位和素数”的数量。

## 说明/提示

对于$100\%$的数据，$1\le a$，$b\le10^{19}$，$a\le b$。

## 样例 #1

### 输入

```
2\n1 2\n1 3
```

### 输出

```
0\n1&nbsp;
```

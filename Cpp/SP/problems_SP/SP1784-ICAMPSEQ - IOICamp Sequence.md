# ICAMPSEQ - IOICamp Sequence

## 题目描述

假设我们有四个 $N$ 元素的实数序列：$A，B，C，D$。

定义函数 $F(i, j)$ 如下: $F(i, j)$ = $|A  _{i} - A _{j} | + |B _{i} - B _{j} | + |C _{i} - C _{j} | + |D _{i} - D _{j} | (1 \leq i, j \leq N)$。

你的任务非常简单：找到 $F(i, j)$ 的最大值。

## 输入格式

第一行：$N (N \leq 100000)$

接下来的 $N$ 行：第 $i$ 行包含四个实数 $A _{i}，B _{i}，C _{i}，D _{i} (-10  ^{9}  ≤ A _{i}，B _{i}，C _{i}，D _{i}≤ 10^{9}) $

## 输出格式

只有一行，为 $F(i, j)$ 的最大值
(结果精确到小数点后三位)

## 样例 #1

### 输入

```
2
1.0 1.0 2.0 0.5
1.0 1.0 0.5 2.0
```

### 输出

```
3.000
```

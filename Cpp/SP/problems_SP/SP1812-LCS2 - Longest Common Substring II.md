# LCS2 - Longest Common Substring II

## 题目描述

题面描述
给定一些字符串，求出它们的最长公共子串

## 输入格式

输入至多$10$ 行，每行包含不超过$100000$ 个的小写字母，表示一个字符串

## 输出格式

一个数，最长公共子串的长度
若不存在最长公共子串，请输出$0$ 。

## 样例 #1

### 输入

```
alsdfkjfjkdsal
fdjskalajfkdsla
aaaajfaaaa
```

### 输出

```
2
```

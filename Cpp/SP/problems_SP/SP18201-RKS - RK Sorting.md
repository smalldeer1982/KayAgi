# RKS - RK Sorting

## 题目描述

**RK** 是一个伟大的密码破解者。他知道世界上任何密码都可以通过频率分析破解。他截获了敌人的信息。该消息由小于或等于 **C** 的 **N** 个数字组成。RK认为频率分析包括对这个序列进行排序，以便更频繁的数字出现在不太频繁的数字之前。

从形式上讲，必须对序列进行排序，以便给定任何两个数字X和Y，如果X在原始序列中出现的次数大于Y出现的次数，则X出现在Y之前。如果出现次数相等，则值在输入中出现得越早的数字在排序序列中应出现得越快。
通过创建“频率分类器”来帮助RK。

## 输入格式

第一行输入包含两个整数 $N (1 \leq N \leq 1000)$ 和 $C (1 \leq C \leq 10^9)$ ，分别表示消息长度和任务描述中的数据

## 输出格式

第一行也是唯一一行输出应包含 $N$ 个数字，即排序序列。

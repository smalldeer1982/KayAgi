# KURUK14 - GENIE SEQUENCE

## 题目描述

一个「精灵序列」是这样一个序列：序列中的每个元素都代表该元素之前或之后的元素数量。给你一个数组，判断能否将其重新排列成一个精灵序列。

## 输入格式

第一行是一个整数 $T$，表示测试用例的数量。接下来是 $T$ 个测试用例，每个用例有两行数据。第一行为一个整数 $N$，代表数组元素的个数。第二行则包含 $N$ 个整数，这些整数以单个空格分隔。

## 输出格式

对于每个测试用例，如果能够形成精灵序列，输出 `YES`（无需加引号）；如果无法形成，则输出 `NO`（同样无需加引号）。

## 数据范围
- $1 \le T \le 100$
- $1 \le N \le 10^3$
- 数组中的元素值在 $0$ 到 $10^3$ 之间。

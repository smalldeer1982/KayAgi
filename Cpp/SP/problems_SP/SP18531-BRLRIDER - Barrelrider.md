# BRLRIDER - Barrelrider

## 题目描述

在一个二维平面上，比尔博站在坐标 (0, 0) 处，周围散布着一些圆形障碍。你的任务是计算这些障碍对比尔博的遮挡效果，并输出遮挡的百分比，结果需要四舍五入至最接近的整数。

### 输入格式

第一行是一个整数 $N$，表示平面上的障碍物数量，满足 $0 \le N \le 1000$。

接下来的 $N$ 行，每行包含三个整数 $X$、$Y$ 和 $R$，分别表示障碍物中心的 $x$ 坐标、$y$ 坐标，以及障碍物的半径。这些值满足：$-5000 \le X, Y \le 5000$ 和 $0 < R \le 1000$。

### 输出格式

输出一个整数，表示这些障碍物对比尔博的遮挡百分比，并将结果四舍五入至最接近的整数。

# TRITILE - Triangle Tiling

## 题目描述

Var 在设计一个新的几何图形，这个图形要满足边数为偶数的条件。她从一个等边三角形开始，然后进行一系列 $n$ 次操作。在每一次操作中，她都会在当前图形的外侧边添加一个新的等边三角形。你能判断在进行 $n$ 次操作后，该图形的边数是否为偶数吗？

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。接下来的每个测试用例中，输入一个整数 $n$，表示 Var 执行操作的总次数。

## 输出格式

对于每个测试用例，如果最终图形的边数是偶数，输出 `Yes`，否则输出 `No`。

**数据范围**  
$$1 \le T \le 100$$  
$$1 \le n \le 10^{18}$$

**样例输入**
```
1
1
```

**样例输出**
```
No
```

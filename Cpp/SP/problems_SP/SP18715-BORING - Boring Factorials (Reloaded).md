# BORING - Boring Factorials (Reloaded)

## 题目描述

给定一个整数 $n$ 和**质数** $p$ 有 $T$ 组数据 $0<T<10^4,0<n<4\times10^{18},1<p<4\times10^{18},|n-p|<10^4$

求 $n!\bmod p$ 是多少。

## 输入格式

第一行输入一个数 $T$ 表示有几组数据。

接下来 $T$ 行，有两个数 $n$ 和 $p$。

## 输出格式

输出 $n!\bmod p$ 是多少。

## 样例 #1

### 输入

```
3
2 5
5 11
21 71
```

### 输出

```
2
10
6
```

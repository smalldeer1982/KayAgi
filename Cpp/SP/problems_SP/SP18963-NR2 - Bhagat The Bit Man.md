# NR2 - Bhagat The Bit Man

## 题目描述

### 题目翻译


Bhagat 是 ISM Dhanbad 的 CSE 学生。在期中考试中，不知何故，他能够在布尔代数上获得满分。所以他的教授们怀疑他怎么能得到满分。所以教授们决定考验他的能力。他们给了 Bhagat 一个学生的录取编号的列表，并要求他找到列表的总的 **Kaptiness**（$K$）。

**Kaptiness** 定义为对每个 **dukkerness（$d_{i}$） 值**的或运算的值。**Dukkerness 值**是列表中每对编号的异或运算的结果。众所周知，Bhagat 在布尔代数方面并不擅长。你能帮助他向他的教授证明，他可以在布尔代数中满分满分吗？

例：

如果列表包含三个数字 $10$、$15$ 和 $17$。然后总共有 $3$ 对。

$d_{1}=10\oplus15=5;$

$d_{2}= 10\oplus17 = 27;$

$d_{3}=17\oplus15=30;$

$k=d_{1}|d_{2}|d_{3};$

$K = 31;$

注：$\oplus$ 为异或符号。

## 输入格式

第一行输入包含 $N(2<=N<=10^{6})$。$N$ 是列表中的录取的总人数（$a_{i}$）。

接下来 $N$ 行将包含录取学生的编号 $(0 <= a_{i} <=10^{18})$。

## 输出格式

输出只有一行，包含 $K$。

## 样例 #1

### 输入

```
3
10
15
17
```

### 输出

```
31
```

# COLORCAT - Magical colorful cats (easy)

## 题目描述

在一个圆圈中有 $n$ 只猫，这些猫分别是白猫、红猫和绿猫。当两只颜色不同的猫进行交谈时，它们都会变成第三种颜色。如果是同颜色的猫交谈，则它们的颜色保持不变。

每一步中，第 1 只猫与第 2 只猫交谈，第 2 只猫与第 3 只猫交谈，依此类推，最后第 $n$ 只猫与第 1 只猫交谈。

现在给出 $n$ 只猫的初始颜色，你需要找出经过 $k$ 步后它们的颜色分别是什么。

### 输入格式：

- 第一行包含两个整数 $n$ 和 $k$，表示猫的数量和要进行的步数（$1 \le n, k \le 10^5$）。
- 第二行是一个长度为 $n$ 的字符串，其中第 $i$ 个字符表示第 $i$ 只猫的初始颜色，'W' 表示白色，'R' 表示红色，'G' 表示绿色。

### 输出格式：

- 输出一个长度为 $n$ 的字符串，表示经过 $k$ 步后的猫的颜色。

### 示例：

输入：
```
3 1
GRR
```
输出：
```
RGR
```

输入：
```
5 4
WRWRW
```
输出：
```
GGGWG
```

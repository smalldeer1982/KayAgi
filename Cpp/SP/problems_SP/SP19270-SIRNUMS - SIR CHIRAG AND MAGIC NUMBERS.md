# SIRNUMS - SIR CHIRAG AND MAGIC NUMBERS

## 题目描述

给定 $p$ 和 $x$ 

求 $ans$ 的最小值

$ans$ 满足：

- 有 $p$ 位；
- 把 $ans$ 的最后一位移动到第一位时正好增长了 $x$ 倍。


给定 $p$ 和 $x$ 

求 $ans$ 的最小值

$ans$ 满足：

- 有 $p$ 位；
- 把 $ans$ 的最后一位移动到第一位时正好增长了 $x$ 倍。

## 输入格式

**有多组测试数据**

对于每组测试数据

共一行，两个正整数 $p, x$。


**有多组测试数据**

对于每组测试数据

共一行，两个正整数 $p, x$。

## 输出格式

共一行，一个正整数 $ans$。

### 数据范围/约定

对于 $100\%$ 的数据 ：$1\le T \le 50;1 \le p \le 10^6;1 \le x < 10$

```plain

共一行，一个正整数 $ans$。

### 数据范围/约定

对于 $100\%$ 的数据 ：$1\le T \le 50;1 \le p \le 10^6;1 \le x < 10$
```

## 样例 #1

### 输入

```
3
6 5
1 2
6 4
```

### 输出

```
142857
```

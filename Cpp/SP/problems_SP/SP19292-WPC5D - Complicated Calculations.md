# WPC5D - Complicated Calculations

## 题目描述

在过去的 $n$ 年里，一直在进行着名为银河战争的比赛，参与的星系共有 $k$ 个。其中有一个特定的星系 $G$，在这场战争中获得了偶数次胜利。而且，星系 $G$ 并未在第一年参战。

你是智慧而古老的达巴社区的一员。在这个星系节庆的时期，虽然大家都在庆祝，你却偏好呆在房间里做一些复杂而无意义的计算。有一天，你忽然想知道，在过去这些年里，可能有多少种胜利的顺序组合（即产生不同胜利者序列的可能性数量）。请输出这一复杂而无意义的计算结果。

## 输入格式

输入包含两个整数 $n$ 和 $k$，分别表示战争持续的年数和参赛的星系数量。

## 输出格式

输出一个整数，表示可能出现的胜利者序列的总数。

## 说明/提示

$$1 \le n \le 10^5, \quad 2 \le k \le 10^5$$

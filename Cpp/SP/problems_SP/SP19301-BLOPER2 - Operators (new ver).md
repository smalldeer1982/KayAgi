# BLOPER2 - Operators (new ver)

## 题目描述

给定一个序列 a1、a2,...,an 和一个整数 S，你的任务是找到一种方法，将运算符 '+' ， '-'， '.'， '~' 插入到序列A中任意相邻的数中间，插入后表达式的结果等于 S。

**请注意，:**

- a . b = a + 2 * b
- a ~ b = a - 2 * b

## 输入格式

第一行：N 和 S

第二行：N个整数，a1，a2,...

## 输出格式

如果有插入方式，请输出其中任何一种，否则输出“不可能”（不带引号）。

## 样例 #1

### 样例输入 #1

```
9 5
1 2 3 4 5 6 7 8 9
```

### 样例输出 #1

```
-~~~++++
```

## 样例 #1

### 输入

```
9 5
1 2 3 4 5 6 7 8 9
```

### 输出

```
-~~~++++
```

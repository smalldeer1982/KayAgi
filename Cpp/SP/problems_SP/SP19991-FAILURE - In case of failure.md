# FAILURE - In case of failure

## 题目描述

### 题意描述：
平面上有n个点，针对于每一个点，求出距离这个点最近的距离的平方，多组数据

## 输入格式

第一行有一个正整数T，表示数据总数，针对于每一组数据，第一行有一个整数n，表示平面上的点数，接下来n行每一行有两个整数x，y，表示点的坐标

## 输出格式

针对于每一组数据，对每一个点输出一行答案

# MINUS - Minus Operation

## 题目描述

有 $n$ 个整数排列成一行。每次你可以任意选择两个相邻的整数，将它们移除，并写下第一个数减去第二个数的结果。你希望通过执行这个操作 $n-1$ 次，最终得到数字 $m$。

## 输入格式

输入包含多组测试用例，第一行给出测试用例的数量。

对于每组测试用例：

- 第一行包含两个整数，分别是 $n$（$1 \le n \le 100$）和 $m$（$-500 \le m \le 500$），用空格分隔。
- 接下来的 $n$ 行中，每行包含一个整数（取值范围是 $[0, 100]$），表示初始的数字序列。

## 输出格式

对于每组测试用例：

你需要输出 $n-1$ 行，每行包含一个整数 $p_i$，表示你决定移除当前序列中的第 $p_i$ 个和第 $(p_i + 1)$ 个数字，并用它们的差值代替。输出时，每行不要有多余的空格��

可以假设输入的每组测试用例都有解。如果有多种解法，可以输出任意一种。

在每组测试用例之间输出一个空行。

## 说明/提示

每组测试中：
- $n$ 的取值在 1 到 100 之间。
- $m$ 的取值在 -500 到 500 之间。
- 每个整数的初始值在 0 到 100 之间。

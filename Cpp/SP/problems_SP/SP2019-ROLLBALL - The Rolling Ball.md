# ROLLBALL - The Rolling Ball

## 题目描述

滚动球体问题

---

一个半径为 **R** 的实心球在一个圆锥的内表面上滚动，圆锥的顶点向下，并且球体在滚动过程中始终没有滑动。这个圆锥的顶角一半为 **u**。初始条件设定为球体以半径 **l > R** 绕圆锥成水平圆周运动，球体上与圆锥接触的点在球体上形成了一个圆。现在的问题是：如果你希望球体能够以最快速度绕圆锥运动，那么这些接触点形成的圆的半径会是多少？

## 输入格式

每行输入包含三个整数：**l**（满足 $R < l \le 1000000$）、**R**（满足 $0 < R \le 1000$）和 **u**（角度范围 $0 < u < 90$）。

## 输出格式

对于每行输入，输出形成接触圆的半径，结果为四舍五入至整数。

## 样例输入

```
220000 100 29
```

## 样例输出

```
46
```

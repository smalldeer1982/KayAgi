# IGAME - Interesting Game

## 题目描述

Alice 和 Bob 在玩一个有趣的数字游戏。在游戏中，玩家可以在自己的回合选择数字中的任意一个非零数字，并将其减少任意非零值，使得最终的数字仍然是非负数。当一个玩家将所有数字变为 0 时，该玩家获胜。所有玩家都以最佳策略进行游戏，Alice 先手。现在，计算在 A 和 B 之间有多少个数字，Alice 能获胜；又有多少个数字，Bob 能获胜。在 A 和 B 之间的每个数字上，Alice 都是先手。

## 输入格式

输入包含 T 组测试数据。每组测试数据有两个整数 A 和 B。

## 输出格式

对于每组测试数据，输出一行，其中包含两个整数，分别表示 Alice 获胜的数字数量和 Bob 获胜的数字数量。

## 说明/提示

$1 \le T \le 1000$，$1 \le A \le B \le 10^{18}$。

## 样例输入
```
2  
1 10  
101 110
```

## 样例输出
```
10 0  
8 2
```

## 样例解释
在第一组样例中，Alice 无论如何都能获胜，因为她可以将任意非零数字调整为 0。  
在第二组样例中，Bob 能够在 101 和 110 这两个数字上获胜，而在其他情况下 Alice 会胜出。

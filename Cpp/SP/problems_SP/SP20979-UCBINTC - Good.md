# UCBINTC - Good

## 题目描述

### 题意翻译

给定一个由 $N$ 个整数构成的序列 $A$ (与之前任务中的序列不同)。在序列 $A$ 中，如果第 $i$ 个元素等于位置严格小于 $i$ 的三个元素的和(同一个元素可以使用多次)。那么这个序列包含多少个好元素？

## 输入格式

输入的第一行包含一个正整数 $N$ $(1 \leq N \leq 5000)$，表示序列 A 的长度。

第二行包含 $N$ 个用空格分隔的整数，表示序列 $A$ $(-100000 \leq A_{i} \leq 100000)$。

## 输出格式

输出仅第一行，包含序列中的好元素数量。  
###### 此翻译由@under_hole 提供

## 样例 #1

### 输入

```
2
1 3
```

### 输出

```
1
```

# UCBINTI - Sequence

## 题目描述

我们称某个整数序列 $a_1, a_2, \ldots, a_n$ 为 $k$-偶数序列，当且仅当序列中的任意 $k$ 个连续元素之和为偶数。

对于给定的序列，我们需要确定至少要修改多少个元素，才能使整个序列变为 $k$-偶数序列。

## 输入格式

第一行输入两个整数 $n$ 和 $k$，其中 $1 \le k \le n \le 10^6$。第二行输入一个由 $n$ 个整数组成的序列 $a_1, a_2, \ldots, a_n$，其中每个整数满足 $0 \le a_i \le 10^9$。

## 输出格式

输出一个整数，表示将序列变为 $k$-偶数序列所需修改的最少项数。

## 样例 #1

### 输入

```
8 3
1 2 3 4 5 6 7 8
```

### 输出

```
3
```

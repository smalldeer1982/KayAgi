# CTPLUCKY - Super Lucky Palindromes

## 题目描述

幸运数字是仅由数字 **“4”** 和 **“7”** 组成的正整数。例如 : 44774 , 74744。

## 输入格式

第一输入行包含一个正整数 $n$，表示要检查的数字数量。

接下来的 $n$ 行中的每一行都包含一个整数 $k$ 。

## 输出格式

对于每个查询，首先输出标题```Query #d:```，其中 $d$ 是查询编号，从 $1$ 开始。

然后，对于查询中给出的值 $k$，打印第 $k$ 个**幸运回文数（即仅由4和7组成的数字）**。

请遵循示例输出中所示的格式。

## 样例 #1

### 输入

```
5
1
2
3
5
100
```

### 输出

```
Query #1: 4444
Query #2: 7777
Query #3: 4444444
Query #4: 4747474
Query #5: 44444444744744444444444444444444744744444444
```

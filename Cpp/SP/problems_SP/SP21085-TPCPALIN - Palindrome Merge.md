# TPCPALIN - Palindrome Merge

## 题目描述

给定两个只由小写字母组成的字符串 $s1$ 和 $s2$。我们可以将这两个字符串按各自的顺序合并，形成一个新的字符串。

例如：如果 $s1 = \text{ab}$ 且 $s2 = \text{ba}$，我们可以合并得到 $st = \text{abba}$，但不能得到 $st = \text{aabb}$。

**问题描述**：计算通过不同的合并方式，能够形成多少个回文串。

例如：`aba` 和 `abba` 是回文串，而 `abc` 和 `abca` 不是。

## 输入格式

- 输入包括两行，每行包含一个字符串，字符串长度不超过 $500$。

## 输出格式

- 输出一个整数，表示所有可能形成的回文串数量对 $3210121$ 取模后的结果。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
ab
ba
```

### 输出

```
4
```

# JLNT - Jarin Loves New Task

## 题目描述

Jarin 接到了一项新任务：她需要用电缆来连接一系列塔楼。

给定以下信息：

1. 塔楼数量 $N$，满足 $2 \le N \le 1000$
2. 可供使用的电缆总长度 $L$，满足 $0 \le L \le 5000$
3. 这些塔楼在一维坐标系中的位置，从左至右排列，塔楼的位置是互不相同的正整数，满足 $0 \le a[i] \le 5000$

她的目标是尽可能多地连接这些塔楼。要求每个塔楼只能连接到左边或右边的塔楼，或者没有连接到任何塔楼。同时，她需要将电缆剪成所需的长度来使用。

这个任务对她来说相当困难，因此你需要帮助她完成。

## 输入格式

输入的第一行是一个整数 $T$，表示测试用例的个数。

接下来的每个测试用例由两行组成：第一行包含两个整数 $N$ 和 $L$。第二行包含 $N$ 个以空格分隔的整数，表示每个塔楼的坐标。

## 输出格式

对于每个测试用例，输出一行，格式为 "Case x: y"，其中 $x$ 是测试用例编号，$y$ 是最大可以连接的塔楼数量。

**样例输入**
```
1
6 2
1 2 3 4 5 6
```

**样例输出**
```
Case 1: 4
```

## 说明/提示

- 测试用例的个数满足 $1 \le T \le 100$
- 塔楼数量 $2 \le N \le 1000$
- 电缆总长度 $0 \le L \le 5000$
- 每个塔楼的位置 $0 \le a[i] \le 5000$，并且位置互不相同

# TAP2014H - Rush hour

## 题目描述

Nlogonia 是一个规划整齐的城市，居民的住宅全部位于城市的东边，而工作场所位于西边。

## 输入格式

第一行包含一个整数 $T$，表示测试用例的数量（$1 \le T \le 10^5$）。接下来的 $T$ 组测试用例中，每个测试用例的描述如下：

- 第一行包含一个整数 $N$，表示在 Nlogonia 的火车服务数量（$1 \le N \le 10^5$）。
- 第二行包含 $N$ 个不同的整数，分别是 $E_1, E_2, \ldots, E_N$，其中 $E_i$ 的值表示从西端第 $i$ 个车站出发的火车应该到达东端的 $E_i$ 车站。

## 输出格式

对于每个测试用例，输出一个整数，表示能够合理安排这些火车服务所需的最小轮班次数。

## 说明/提示

- 测试用例数量： $1 \le T \le 10^5$
- 火车服务数量： $1 \le N \le 10^5$
- 每个 $E_i$ 的取值范围： $1 \le E_i \le N$

# TROLLS - Trolls

## 题目描述

在一个隐秘的森林里，有许多蒟蒻，但他们很聪明，大多数人懂 $c$ 或 $pascal$ 。他们写了很多程序。如果有了超级计算机，他们的程序在很短的时间里就可以得到正确答案。作为一个优秀的 $OIer$ ，你甚至可以在没有电脑的情况下想出答案。

## 输入格式

没有输入。下文提及的四个程序可以在[这里](https://www.luogu.com.cn/paste/2azuv9aj)查看。

## 输出格式

输出$4$行，第 $i$ 行对应第 $i$ 个程序。

对于 $program1$ 和 $program2$ ：

请输出该程序将输出的`*`的个数。

对于 $program3$ 和 $program4$ ：

请输出该程序将输出的数字的个数。

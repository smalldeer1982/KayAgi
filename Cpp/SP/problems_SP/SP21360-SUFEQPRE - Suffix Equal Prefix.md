# SUFEQPRE - Suffix Equal Prefix

## 题目描述

## 题目简述
给你一个字符串 $S$，判断有多少个真后缀同时是其前缀。

真后缀是指不同于原串的后缀。

## 输入格式

第一行 $T$，表示有 $T$ 组数据。

每组数据包含一行一个字符串，表示字符串 $S$。

## 输出格式

对于每组数据，输出数据组与答案。

例如，`Case 1: 2`。

## 说明/提示

$1\le|S|\le10^6$

## 样例 #1

### 输入

```
3
ababab
aaaa
abcabc
```

### 输出

```
Case 1: 2
Case 2: 3
Case 3: 1
```

# HCT00001 - Interesting Game with Polygons

## 题目描述

### 题目翻译

Rijél 是一个非常聪明的老师。他喜爱数学，尤其是游戏和几何问题。最近，他的一个学生向他发起了以下挑战：
最初，在一个平面上绘制有 N 个顶点的多边形。多边形是严格向外凸的，即每个内角严格小于 $180$ 度。多边形的顶点按顺时针的顺序，编号为 $1$ 到 $N$。

两个玩家在这个多边形上玩游戏。玩家们交替轮流进行。在每个回合中，当前玩家选择多边形的对角线或边，并将其绘制为直线段。（多边形的对角线是连接多边形的任意两个不相邻顶点的线段。）玩家只能选择一条对角线或一条不与任何先前绘制的线段相交的边（它也不得与任何一段共用端点）。不能按照上述规则画对角线或边的玩家输掉比赛。

你将得到一个整数 $N$。

我们假设两个玩家都以最佳方式玩游戏。如果第一个玩家获胜，则返回  $1$，否则返回 $2$。

## 输入格式

输入只有一行——一个 $N$（$3 ≤ N ≤ 1000$），即多边形的顶点数。

## 输出格式

如果第一个玩家获胜，那么输出 $1$，否则输出 $2$。

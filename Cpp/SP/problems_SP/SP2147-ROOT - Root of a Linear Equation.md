# ROOT - Root of a Linear Equation

## 题目描述

解方程：$bx=a$

## 输入格式

对于每个测试点，先给出一个正整数 $t$ 表示数据组数。

接下来 $t$ 行，每行两个正整数 $a,b$，用一个空格隔开。

提示：对于 $95\%$ 的数据，用C/C++的int或pasacal的longint已经足够。

输入大小最大约为1.4KB

## 输出格式

对于每个测试点，输出方程的解。

若解为整数，输出原数外加`.0`。

若解为有限小数，输出全部小数部分。

反之用括号括出循环部分。

e.g 若 $a=1,b=6$ ，那么方程的解为 $x=\frac16=0.1\dot6$，所以输出`0.1(6)`。

如果解有无数个或无解，输出`Invalid Input!!!`。

输出最大约为1.3MB

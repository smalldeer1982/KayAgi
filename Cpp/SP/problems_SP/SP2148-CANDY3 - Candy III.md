# CANDY3 - Candy III

## 题目描述

同学们有 $N$ 堆糖果，分别为 $a_1,a_2,a_3,\ldots,a_n$，老师让同学们所有糖果放在一起后重新分配，问能不能让每个同学，分到的糖果数相同。

## 输入格式

第一行输入一个数字 $T$，$T$ 组数据($T$ 较小)。

每组数据输入一个 $N$（$N$ 较小）

接下来 $N$ 行，$i$ 行 $a_i$（$a_i \leq 10^{18}$ ）表示每个同学的糖果数

## 输出格式

若能输出 `YES`，否则输出 `NO`

## 样例 #1

### 输入

```
2

5
5
2
7
3
8

6
7
11
2
7
3
4
```

### 输出

```
YES
NO
```

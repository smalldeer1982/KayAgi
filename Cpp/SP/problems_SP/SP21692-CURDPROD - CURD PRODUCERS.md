# CURDPROD - CURD PRODUCERS

## 题目描述

某家工厂拥有几台奶酪生产机，不同机器的生产质量各不相同。质量为 $q$ 的奶酪机需要用 $q$ 个单位的时间来生产 $1$ 单位奶酪。

例如，一台质量为 $5$ 的机器将在时间 $5$ 生产 $1$ 单位奶酪，时间 $10$ 再生产 $1$ 单位，以此类推。

现给出所有机器的质量，要求计算生产 $T$ 单位奶酪所需的最短时间。

**输入格式：**

- 第一行输入一个整数 $t$，表示测试用例数。
- 每个测试用例的第一行包含两个整数 $n$ 和 $T$，分别表示机器的数量和目标奶酪的数量。
- 接下来的 $n$ 行，每行输入一个整数，表示每台机器的质量。

**输出格式：**

对于每个测试用例，输出一行，表示生产指定数量的奶酪所需的最短时间。

**数据范围与提示：**

- $1 \leq t \leq 100$
- $1 \leq n \leq 10,000$
- $1 \leq T \leq 1,000,000,000$
- 每台机器的质量为 $1$ 到 $1,000,000,000$ 之间的整数

**注意：** 例如，质量为 $5$ 的机器在时间 $9$ 时，仅能生产 $1$ 单位奶酪，而不是 $1.8$ 单位。

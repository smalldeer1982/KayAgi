# FSEATS - Finding Seats

## 题目描述

#### 题意简述：

给你一个R*C的矩阵，求包含`.`的数量不小于K的最小矩形面积。

## 输入格式

有多组数据，每组数据第一行三个数，分别为R、C、K，

接下来读入一个包含`.`和`X`且大小为R*C的矩阵。

当R=C=K=0时，输入结束。

## 输出格式

对于每组数据，输出所求矩形面积。

## 说明/提示

请注意题目时间限制要求

## 样例 #1

### 输入

```
3 5 5
...XX
.X.XX
XX...
5 6 6
..X.X.
.XXX..
.XX.X.
.XXX.X
.XX.XX
0 0 0
```

### 输出

```
6
9
```

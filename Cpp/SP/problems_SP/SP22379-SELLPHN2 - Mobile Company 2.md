# SELLPHN2 - Mobile Company 2

## 题目描述

一家手机制造公司在第一天购买了 $L$ 单位的原材料，这些原材料刚好可以用于生产 $L$ 部手机，并将其运送到各个工厂。

公司拥有 $M$ 个工厂，这些工厂分布在不同地点，负责将原材料转化为手机。每个工厂，第 $i$ 个最多可以生产 $F[i]$ 部手机，前提是原材料供应充足。

制造出的手机将被运送到 $N$ 个商店售卖。每个商店都有其最大销售能力，其中第 $i$ 个商店每天最多能售卖 $S[i]$ 部手机。这里有一个 $M \times N$ 的二维地图，地图的 $map[i][j]$ 表示第 $i$ 个工厂最多能运送到第 $j$ 个商店的手机数量。

你的任务是计算出公司在第一天最多能销售多少部手机。为简化起见，假设原材料和手机的运输时间为零。

**输入格式：**

输入的第一行为一个整数 $t$，表示测试用例的数量。对于每个测试用例，首先输入三个整数 $L, M, N$。接下来一行输入 $M$ 个整数，表示数组 $F$。再接下来一行输入 $N$ 个整数，表示数组 $S$。然后的 $M$ 行表示地图 $map$。

**输出格式：**

对于每个测试用例，输出公司在第一天最多能销售的手机数量。

**数据范围与提示：**

- $1 \le t \le 100$
- $1 \le L \le 1000$
- $1 \le M \le 200$
- $1 \le N \le 200$
- $0 \le F[i] \le 1000$
- $0 \le S[i] \le 1000$
- $0 \le map[i][j] \le 1000$

**样例输入：**

```
1
5 2 2
3 5
2 3
1 2
1 0
```

**样例输出：**

```
4
```

# TINKUJOB - Tinku got a job

## 题目描述

Tinku 在 BIT 找到了一份新工作。这份工作不仅报酬优厚，还给他提供了一间住在 BIT 校园里的房子。Tinku 的工作就是把货物运送到主楼。他需要从一个取货点收集货物，因此 Tinku 一天的工作流程是从家里出发，先去取货点取货，然后将货物送到主楼。取货点可以在 BIT 校园的任意位置。

为简化问题，我们可以将整个 BIT 校园看作一个二维平面。Tinku 的家、主楼和所有取货点都只能位于**整数坐标**。我们会提供 Tinku 家和主楼的坐标（请仔细查看输入格式）。计算距离时采用欧几里得距离，即从点 (a, b) 到 (c, d) 的距离计算公式为 $\sqrt{(a-c)^2 + (b-d)^2}$。注意，主楼也可以作为一个取货点。

根据工作合同，Tinku 每天最多工作 H 小时。他对自己的新工作有些不安，因此 ACM 的同学们决定帮帮他，找出有多少个取货点能够让 Tinku 在规定时间内完成他的任务。

## 输入格式

第一行是测试用例的数量 $T$  
接下来是 $T$ 个测试用例  
每个测试用例的格式如下：  
第一行包含五个整数 $A, B, C, H, V$  
Tinku 的家位于坐标 $(A, C)$，主楼位于坐标 $(B, C)$  
Tinku 最多工作 $H$ 小时，行走速度为每小时 $V$ 单位

其中：  
$T = 1000$  
$-100 \le A, B, C \le 100$  
$0 < V \le 40$  
$0 < H \le 40$

## 输出格式

输出一行，表示可能的取货点的数量。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2\n1 -1 0 1 3\n-5 5 10 2 2
```

### 输出

```
5\n0
```

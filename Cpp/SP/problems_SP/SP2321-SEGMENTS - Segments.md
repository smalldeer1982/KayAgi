# SEGMENTS - Segments

## 题目描述

在平面上有 $N$ 条水平线段。第 $i$ 条线段位于高度 $h_i$（可以为负数），其起点和终点的横坐标分别为 $a_i$ 和 $b_i$（满足 $a_i < b_i$）。这些线段不包括它们的端点。你的任务是绘制一些垂直直线，使得每条水平线段至少与一条垂直线相交，并且至多与 $R$ 条垂直线相交，同时要让 $R$ 的值尽可能小。

## 输入格式

第一行输入一个整数 $N$（$1 \le N \le 400$），表示水平线段的数量。之后的 $N$ 行中，每行给出三个 32 位有符号整数 $a_i, b_i, h_i$，表示第 $i$ 条水平线段的起始点、终止点和高度。注意，水平线段可能会重叠。

## 输出格式

输出一个整数，即可以实现的最小 $R$ 值，并换行。

### 输入

```
3
```

```
2
```

# ODWS - One day with string

## 题目描述

----

Rafa 总是对字符串问题感兴趣，并在感到无聊时玩弄字符串。有一天，他试图编写所有在字符串 $s$ 和字符串 $t$ 之间（长度相同）的字符串，这些字符串在字典序上大于字符串 $s$ 并且小于字符串 $t$。字符串 $s$ 和 $t$ 由相同长度的小写英文字母组成。他很快就累了，开始考虑用计算机来解决这个问题。因为他太累了，所以他想让你编写一个程序来找到给定限制下的字符串数量。

## 输入格式

----

输入以测试案例的数量 $T$$（T\leq100）$开始。每组输入包含两行，第一行包含字符串 $s$（长度为 $1$ 的小写英文字母。这里，$\rvert t \rvert$ 表示字符串的长度。）

第二行包含字符串 $t$$（ \rvert t \rvert  =  \rvert s \rvert ）$，由小写英文字母组成。

保证字符串 $s$ 和 $t$ 的长度相同，并且字符串 $s$ 在字典序上小于字符串 $t$。

## 输出格式

----

输出每个测试案例的编号和结果。由于结果可能很大，你应该将结果对 $10^9+7$ 取模。为了更好地理解，请查看下面的输入和输出示例。

示例 1
-----

### 示例输入 1
```
3
a
d
abcdefg
abcdefh
xy
yz
```

### 示例输出 1
```
Case 1: 2
Case 2: 0
Case 3: 26
```

## 样例 #1

### 输入

```
3\na\nd\nabcdefg\nabcdefh\nxy\nyz
```

### 输出

```
Case 1: 2\nCase 2: 0\nCase 3: 26
```

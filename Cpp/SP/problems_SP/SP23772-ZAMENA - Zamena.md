# ZAMENA - Zamena

## 题目描述

Picsel 是秘密委员会的一名老成员，他决定在多年服务后辞职。不过，他希望告别时能留下深刻的印象，于是用一个只有他知道的密钥加密了自己最后一次对全国赛的建议。他只留给其他委员会成员一个数组，这个数组包含所有成员的 ID（**ID 的长度固定，且不一定唯一**），以及一个他们必须解决的问题才能获得密钥：

最初，Picsel 从数组的第一个元素开始向后遍历，直到找到一个他中意的 ID，并将其记住。接下来，他继续扫描数组，如果遇到一个与他记住的 ID **非常相似**的 ID（两个 ID 如最多只有一位数字不同即为**非常相似**），他可以选择：

- 将新遇到的 ID 取代之前记住的 ID，并继续遍历；
- 不记住新 ID，继续遍历。

整个过程中，Picsel 会记录他的“得分”，初始时得分为零。每当他选择记住一个新 ID，**得分会增加旧 ID 和新 ID 在不同位置上的数字的绝对差**。例如，如果之前记住的 ID 是 1234，而新记住的是 1274，得分会增加 4。现在，委员会要找出 Picsel 可能获得的**最大得分**。

全国赛已经开始，由于委员会未能破解 Picsel 的问题，因此无法提供新题目。他们请求你帮忙解密，并承诺在比赛中奖励你 100 分。

## 输入格式

第一行输入一个自然数 $N$，表示秘密委员会成员的数量。

接下来的 $N$ 行，每行输入一个整数 $A_i$，表示数组中第 $i$ 个委员会成员的 ID。

## 输出格式

输出一个整数 $M$，表示 Picsel 可能获得的最大得分。

 **本翻译由 AI 自动生成**

### 输入

```
\n6\n8823\n2145\n2185\n3385\n4145\n4445
```

### 输出

```
5
```

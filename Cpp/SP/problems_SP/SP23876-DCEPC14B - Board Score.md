# DCEPC14B - Board Score

## 题目描述

Gita最近推出了2048游戏。他觉得这很令人兴奋。但作为比赛中的新手，他对比赛的得分感到困惑。
标准2048游戏的正式定义可以在这里找到:

<http://en.wikipedia.org/wiki/2048_%28video_game%29>

他要求你通过显示当前的棋盘配置来找出游戏的最高得分。你能帮他找出给定电路板配置的当前最高得分吗？
请注意，在2048游戏中，如果 $2$ 个图块合并，则最终图块上的值将添加到游戏分数中。

## 输入格式

第一行包含测试用例的数量 $T$。每个测试用例包含 $4$ 行。每行包含 $4$ 个整数。空图块用 $0$ 表示。

## 输出格式

每个测试用例输出 $1$ 个整数，为给定2048板配置的最大得分。

## 样例 #1

### 输入

```
2
4 8 2 0
0 0 0 0
0 0 0 0
2 0 0 0
16 4 0 0
0 0 0 2
0 0 0 0
0 0 0 0&nbsp;
```

### 输出

```
20
52
```

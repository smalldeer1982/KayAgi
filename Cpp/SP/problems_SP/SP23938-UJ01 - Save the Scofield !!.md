# UJ01 - Save the Scofield !!

## 题目描述

Scofield 特别擅长处理数学表格，因此 T-BAG 设法给他设置了一道更具挑战的任务。他给出两个数 $A$ 和 $B$，要求将 $A$ 和 $B$ 的倍数表合并成一个升序排列的超级表格，去除所有重复项。最后，他向 Scofield 提问：超级表中的第 $N$ 个数是什么？

## 输入格式

第一行包含一个整数，表示测试用例的总数 $T$。接下来的每个测试用例都包括三个整数 $A$、$B$ 和 $N$。

**输入示例：**
```
4
3 5 2
3 5 3
2 3 4
2 3 6
```

## 输出格式

对于每个测试用例，输出超级表格中第 $N$ 个数。

**输出示例：**
```
5
6
6
9
```

## 说明/提示

- $1 \leq T \leq 300000$
- $1 \leq A, B \leq 1000$
- $1 \leq N \leq 1000000000$

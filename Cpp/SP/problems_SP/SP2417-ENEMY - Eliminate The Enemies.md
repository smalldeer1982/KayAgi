# ENEMY - Eliminate The Enemies

## 题目描述

### 题目大意
给定一张 $N \times M$ 的平面图，其中只包含 `.` 和 `*`。你可以选择一个格子开始，每次可以向上、下、左、右走一格，前提是不能进入带有 `*` 的格子中。问有多少种方案，能使得遍历所有的带有 `.` 的格子，并且不重复走过任意一个格子，答案 $mod$ $10^9 + 7$。

## 输入格式

多组数据，每组数据先输入 $N \times M$ ，然后输入平面图。在 $N = M = 0$ 时结束输入。

## 输出格式

每组数据输出一行，表示答案 $mod$ $10^9 + 7$ 的结果。

## 样例 #1

### 输入

```
3 3
...
.*.
...
3 7
...*...
.*.*.*.
.......
3 3
***
*.*
***
0 0
```

### 输出

```
16
8
1
```

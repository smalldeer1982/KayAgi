# MONKK - Monkey King

## 输入格式

第一行一个整数，表示有几组测试点

每组测试点先输入两个整数r和c，分别表示行数和列数，接下来输入r行

每行有c列，由‘.’，‘T’和‘B’三种字符组成。‘.’代表空地，‘T’代表收税地点，‘B’代表香蕉树。

## 输出格式

输出一行，先输出测试点编号，再输出最多能得到多少香蕉

## 样例 #1

### 输入

```
2

3 3
BBB
BBB
BBB

5 5
T.BBB
TBBBB
.BBBB
TBBBB
.BBBB
```

### 输出

```
Case 1: 9
Case 2: 16
```

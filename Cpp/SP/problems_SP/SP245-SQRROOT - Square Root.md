# SQRROOT - Square Root

## 题目描述

给你 $t$ 个数，输出这些数的平方根，保证给出的数存在，并且它是整数。

你只能使用 C, C++, Pascal, Algol, Fortran, Ada, Ocaml, Prolog, Whitespace, Brainf\*\*k。

## 输入格式

第一行一个整数 $t$（$t\leq50$），接下来 $T$ 行，每行一个整数（最多 800 位）。

## 输出格式

输出必须包含正好等于给定数字平方根的t个数字。

## 样例 #1

### 输入

```
3
36
81
226576
```

### 输出

```
6
9
476
```

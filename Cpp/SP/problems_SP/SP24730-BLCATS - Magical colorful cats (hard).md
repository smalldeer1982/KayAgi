# BLCATS - Magical colorful cats (hard)

## 题目描述

在一个圆圈上，坐着 $n$ 只猫，这些猫有白色、红色和绿色三种颜色。当两只不同颜色的猫相互交谈时，它们都会变成第三种颜色；如果它们是相同的颜色，则颜色不发生变化。

在每一步中，第 1 只猫与第 2 只猫交谈，第 2 只猫与第 3 只猫交谈，依此类推，第 $n$ 只猫会与第 1 只猫交谈。

已知这些猫的初始颜色，要求找出经过 $k$ 步后它们各自的颜色。

## 输入格式

- 第一行：两个整数 $n$ 和 $k$，分别表示猫的数量和交谈的步数（$1 \le n, k \le 10^5$）。
- 第二行：一串长度为 $n$ 的字符串，其中第 $i$ 个字符表示初始时第 $i$ 只猫的颜色。

## 输出格式

- 一行长度为 $n$ 的字符串，表示经过 $k$ 步后每只猫的颜色。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
:	3 1	GRR
```

### 输出

```
:	RGR
```

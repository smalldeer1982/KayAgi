# POUR1 - Pouring water

## 题目描述

现在你有两个桶，分别可以装$a,b$ 升水，请求出正好倒出$c$ 升水需要多少步。
以下的几个操作视为一个“步骤”：
·倒出桶中的水。
·用水填满桶。
·将水从一个桶倒入另一个桶，直到某个桶变空或变满。

## 输入格式

一个整数$t$ ($1<=t<=100$ )，代表测试的用例数。接下来的$t$ 组输入数据中，每组数据有三个整数$a,b,c$ （$0<a,b,c<=40000$ )。

## 输出格式

输出最小步数，如果不能正好倒出$c$ 升水，则输出$-1$ 。

## 样例 #1

### 输入

```
2
5
2
3
2
3
4

 
```

### 输出

```
2
-1
```

# LOTGAME - New Lottery Game

## 题目描述

### 题意翻译
给定非负整数 $A,B,K$。  
设 $x,y$ 是非负整数，且 $x<A,y <B$。  
将 $x,y$ 转为二进制并按位进行与运算，再将结果转为十进制。为方便，下文称得到的结果为 $p$。  

求有多少对满足条件的 $x,y$ 使得 $p<K$。

## 输入格式

第一行输入一个数，表示测试样例 $T$。  
接下去 $T$ 行，每行三个非负整数 $A,B,K$。

## 输出格式

输出包含 `Case#m:n` 其中，$m$ 是测试样例编号（从1开始），$n$ 为使$p<K$ 的 $x,y$ 的数量。

## 样例 #1

### 输入

```
5
3 4 2
4 5 2
7 8 5
45 56 35
103 143 88
```

### 输出

```
Case #1: 10
Case #2: 16
Case #3: 52
Case #4: 2411
Case #5: 14377
```

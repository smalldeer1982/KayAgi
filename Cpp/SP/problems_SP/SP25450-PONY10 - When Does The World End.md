# PONY10 - When Does The World End

## 题目描述

最近发现了一座古老的神庙，达林·多正忙着探索。途中，她发现了一个古老的能量神器，不停地吐露出一串无意义的数字：“……82980644837721883829806448……”

墙上的铭文解释了这件神器的角色：

- 我们已经触及了宇宙的生命力量。
- 我们能够听见时间流逝走向终点的钟声。
- 你能听见吗？
- 当预言的数字被说出预定的次数时，一切就会终结。

然而，达林·多并不清楚这些预言的数字和预定的次数具体是什么，于是她继续她的冒险之旅。

在探索过程中，她了解到更多信息：

- 神器从时间的起点开始计数，从 0 开始，每秒增长 1，并逐个输出每个数字。因此，最开始计数的情况是：“0123456789101112131415……”。每一秒，神器会输出下一个数字。这意味着在 $t=0$ 时，它输出“0”；在 $t=1$ 时，输出“1”，依此类推。
- 宇宙的终结不是在预言的数字最后一次完全输出后的下一秒，而是在该数字开始输出的那一刻。例如，如果预言的数字是“01”，预定的次数是 2，那么宇宙将在 $t=11$ 时终结。下面还有更多示例。
- 达林·多找到了一些可能是预言的数字或是预定次数的线索，但她尚不确切。

请帮助她计算在给定的预定次数和预言数字为正确时，宇宙将存在的时间长度。但请注意，若你耗时过久，世界可能在结果揭晓前便已终结！

## 输入格式

输入包含两行：

- 第一行为整数 $n$（$1 \le n \le 10^5$），表示预定的次数。
- 第二行为字符串 $s$（$1 \le |s| \le 100$），表示预言的数字。

## 输出格式

输出一个整数，表示预言的数字第 $n$ 次出现的时刻，即宇宙的寿命。

## 说明/提示

- $1 \le n \le 10^5$
- $1 \le |s| \le 100$

# NEXTLEX - Next Lexicographically Greater Substring

## 题目描述

Consider a string S, consisting of only lowercase alphabets.

## 输出格式

Output the answer for each query in a new line.

## 样例 #1

### 输入

```
dabab
3
ab
abaa
bab

```

### 输出

```
aba
abab
d

```

## 样例 #2

### 输入

```
a
1
a

```

### 输出

```
-1

```

# CATMO1 - Cat and Mouse I

## 题目描述

一只猫在一个有障碍物的长方形房间里追赶一只老鼠。猫和老鼠都交替按照以下规则移动（猫先走）：

猫和老鼠都只能移动到不包含障碍物的位置，并且在水平、垂直或对角方向上与其当前位置相邻。呆在同一个地方总是一个有效的举动。

猫和老鼠都不能离开房间的边界。

猫总是知道老鼠的位置，老鼠也知道猫的位置

根据房间的描述，以及猫和鼠标的起始位置，你的任务是决定老鼠是否能永远避开猫。

## 输入格式

#### 本题有多组数据

对于每一块，第1行输入两个整数，N和M

接下来N行，每行一个字符串，字符意思如下

 "."代表相应位置没有障碍物
 
 "#"代表相应位置包含障碍物
 
 "M"对应的位置是鼠的起始位置
 
 "C"对应的位置是猫的起始位置。

## 输出格式

对于每个房间，如果猫永远抓不到老鼠则输出"mouse"，否则输出"cat"。

## 样例 #1

### 输入

```
2
3 3
#c#
.#m
#.#

3 3
#c#
.#.
#m#
```

### 输出

```
cat
mouse
```

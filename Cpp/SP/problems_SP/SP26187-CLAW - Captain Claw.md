# CLAW - Captain Claw

## 题目描述

船长 Claw 需要跨过一条酸构成的河流。河的宽度是 $x$ 米，然而船长一次最多只能跳 $d$ 米。

按照规则，船长只能踩在不断出现的石头上进行跳跃。

## 输入格式

程序需要处理多组测试数据。对于每组测试数据：

第一行包含三个整数 $n, x, d$，分别表示事件持续的秒数、河的宽度以及船长的最大跳跃距离。

接下来的 $n$ 行表示每一秒内出现石头的情况。

对于每一行，第一个数字 $c$ 表示这一秒钟内出现的石头数量。

接下来的 $c$ 个整数表示这些石头出现的位置。

请计算船长 Claw 最少需要多少时间才能成功渡河。

- $1 \le t \le 30$
- $1 \le x \le 10^5$
- $1 \le d \le x$
- $1 \le n \le 10^3$
- $1 \le \sum c \le 10^5$

## 输出格式

每组测试数据输出一个整数，表示渡过整条河流所需的秒数。

若在 $n$ 秒内无法渡河，则输出 `-1`。

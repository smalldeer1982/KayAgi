# DSH - detective sherlock holmes

## 题目描述

侦探夏洛克·福尔摩斯在调查案件时，发现了一条线索。这条线索是一段由小写字母组成的字符串。他推测，如果将这些字符转换为对应的数值，就能够找到罪犯。任务是找出这些字符中数值大于字母 `q` 的个数。由于案件令他感到困惑，因此请求你的协助，请帮他完成这项工作。

## 输入格式

- 输入包含多个测试用例，最后一个测试用例表示“线索长度小于等于3”，此时输入终止。
- 每行是一个字符串（仅包含小写字母），字符串长度满足 $3 < l \leq 26$。

## 输出格式

对于每个测试用例，输出两行：
- 第一行输出字符串的数值。
- 第二行输出大于 `q` 的数值的总数；如果没有这样的数值，则输出“no one”。

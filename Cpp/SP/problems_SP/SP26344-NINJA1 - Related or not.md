# NINJA1 - Related or not

## 题目描述

给定两个 DNA 序列 A 和 B。当且仅当存在一个相同长度的非递减序列 C，满足对于所有 $i$，都有 $C_i = A_i$ 或 $C_i = B_i$ 时，称这两个 DNA 序列是相关的。你的任务是判断这两个序列是否相关。

## 输入格式

- 首先输入一个整数 $T$，表示测试用例的数量。

对于每个测试用例：
- 第一行输入一个整数 $N$，表示 DNA 序列的长度。
- 接下来一行输入 $N$ 个整数，代表物种 A 的序列。
- 再下一行输入 $N$ 个整数，代表物种 B 的序列。

## 输出格式

对于每个测试用例，输出一行：如果能找到符合条件的非递减序列，则输出 `YES`，否则输出 `NO`。

## 说明/提示

- $1 \le T \le 10^5$
- $1 \le N \le 10^5$
- $0 \le A_i, B_i \le 10^{10}$

### 示例输入：

```
3
3
1 2 3
4 4 4
3
3 2 1
6 5 4
2
1 0
10 2
```

### 示例输出：

```
YES
NO
YES
```

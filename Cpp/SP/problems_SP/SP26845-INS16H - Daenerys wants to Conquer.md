# INS16H - Daenerys wants to Conquer

## 题目描述

### 题目背景
丹妮莉丝·坦格利安（小说《冰与火之歌》的人物）在征服七大王国的途中遇到下面难题。急需足智多谋的你帮她解决。

给定函数 $F(x)=N$ 和整数 $N$（$1\leq N\leq10^{10}$），找到满足条件的最小整数 $x$。

![](https://www.spoj.com/content/simes:INS16H.png)

如果不存在 $x$，则输出 `-1`。

## 输入格式

第一行包含一个整数 $T$ ($ 1 \leq T \leq10$) ，即问题的个数。

接下来是 $T$ 行，每行一个整数 $N$。

## 输出格式

每行输出每个满足条件的最小整数 $x$。

### 样例输入

```
4
5
4
9
7
```

### 样例输出

```
4
3
8
-1
```

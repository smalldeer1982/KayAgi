# BGSHOOT - Shoot and kill

## 题目描述

森林里有$N$只动物，每只动物的出没时间为 $X-Y$  
现有$Q$个询问，每次询问 $L-R$ 的时间段内，**单位时间**出现的动物总数的**最大值**

## 输入格式

第1行 一个正整数$N$，表示动物总数  
第2-n+1行 每行两个正整数$X,Y$，$[X-Y]$表示第i只动物出没的时间  
第n+2行 一个正整数$Q$，表示询问总数  
第n+3行到最后 每行两个整数$L,R$，$[L-R]$表示询问的区间

## 输出格式

对于每个询问，每行输出一个结果

## 样例 #1

### 输入

```
4
1 2
2 3
4 5
6 7
4
1 5
2 3
4 7
5 7
```

### 输出

```
2
2
1
1
```

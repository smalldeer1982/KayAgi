# IAPCR2D - Find The Number

## 题目描述

求一个数的全除数太容易了。但别担心，这个问题也很容易。

你会得到一个数的阶乘的所有除数，你必须找到这个数。如果有多个这样的数字，你必须找到最小的一个。如果没有符合约束的数字，则打印 `nai` 。

## 输入格式

输入的第一行将是 $T$ ，表示测试用例的数量。在接下来的 $T$ 行中，只有一个整数 $N$ ，表示数字 $X$ 的阶乘的所有除数。

## 输出格式

对于每个测试用例，打印数字 $X$ 。如果 $X$ 不存在或大于 $10^4$ ，则在单行中打印 `nai` 。

## 说明/提示

保证 $1 \le t \le 10^5$ 。

保证 $0 \le n \le 10^{18}$ 。

保证 $1 \le X < 10^2+10$ 。

## 样例 #1

### 输入

```
5
1
2
3
4
5
```

### 输出

```
1
2
nai
3
nai
```

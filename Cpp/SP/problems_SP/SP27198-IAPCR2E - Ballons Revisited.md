# IAPCR2E - Ballons Revisited

## 题目描述

你有 $4$ 种颜色的气球，每种颜色的数量分别为 $w,r,g,b$。你要把这些气球分给学生，每个学生都要拿到刚好 $4$ 个气球，且颜色不能全部相同。求最多可以给多少学生。

## 输入格式

**本题有多组数据。**
第一行一个整数 $T$，表示数据个数。

后 $T$ 行每行 $4$ 个整数，分别表示 $w,r,g,b$。

## 输出格式

对于第 $i$ 组数据，输出一行`Case i: k`，其中 $k$ 表示答案。

## 样例 #1

### 输入

```
2
2 2 1 3
1 1 4 1
```

### 输出

```
Case 1: 2
Case 2: 1
```

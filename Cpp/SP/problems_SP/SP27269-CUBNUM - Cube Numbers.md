# CUBNUM - Cube Numbers

## 题目描述

每一个正整数 $ n $ 都可以被分解成类似于 $ a _{1}^{3} +a _{2} ^{3} +...+a _{m} ^{3} $的形式。你的任务是对于 $ n $，输出最小的 $ m $。

例如：

$ 5 = 1 ^ 3 + 1 ^ 3 + 1 ^ 3 + 1 ^ 3 + 1 ^ 3 ( m = 5 ) $

$ 8 = 2 ^ 3 ( m = 1 ) $

$ 35 = 2 ^ 3 + 3 ^ 3 ( m = 2 ) $

## 输入格式

每行输入一个正整数，表示每个询问中的 $ n $。

## 说明/提示

询问次数 $ \le 10^5 $。


`C++` 选手可以使用 `while(cin>>n)` 输入数据。

## 样例 #1

### 输入

```
1
2
5
8
35
```

### 输出

```
Case #1: 1
Case #2: 2
Case #3: 5
Case #4: 1
Case #5: 2
```

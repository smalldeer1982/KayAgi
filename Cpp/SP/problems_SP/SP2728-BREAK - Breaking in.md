# BREAK - Breaking in

## 题目描述

$Mayco$最近被一家知名软件公司聘为安全顾问。目前，他正在进行他的第一项任务——确定该公司的哪些服务器是潜在攻击者的最佳目标。不过这有点困难，因为一些服务器依赖其他一些服务器。如果攻击者破坏了服务器，他或她还可以自由访问所有依赖它的服务器。

根据定义，服务器$S$的重要性是攻击者在入侵$S$后能够访问的服务器数量。最重要的服务器是那些重要性最高的服务器。（请注意，可能有多个最重要的服务器。这在下面的示例中也有说明。）

## 输入格式

第一行一个正整数$t$，表示数据组数。

接下来每一组数据之间都有一个空行。每一组数据都有若干行，每行两个整数$x_i$,$y_i$,表示$x_i$依赖$y_i$

## 输出格式

共有若干行，每行表示一组数据，其中每行有若干个整数，表示重要性最高的几个服务器。

## 样例 #1

### 输入

```
2

5 4
3 1
3 2
4 3
5 3

6 5
1 2
2 3
3 1
1 4
5 6
```

### 输出

```
1 2
4
```

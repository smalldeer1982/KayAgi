# BLCONE - Painting Hat

## 题目描述

为了准备杰夫罗伊的生日聚会，他的妈妈西尔茜安排手下的员工给聚会上用的帽子涂漆。这些帽子都是圆锥形的，高 $H$，半径为 $R$。西尔茜和员工们前往商店购买油漆。由于没有专门的容器，这些油漆将直接存放在帽子里。

油漆的存放方式如下图所示：

![hat](https://cdn.luogu.com.cn/upload/vjudge_pic/SP27380/e891fc9d15dd9721e12e2ce7a37d406af3767ac0.png)

我们知道，1单位体积的油漆可以刚好涂刷1单位的帽子面积。为了尽可能高效地使用油漆，西尔茜买的所有油漆都必须用于涂刷帽子的内外表面（忽略厚度）。每顶帽子被灌入油漆，直到达到某个高度 $h$。被油漆浸没的部分无需再涂刷。如果帽子无法装下购入的全部油漆，就将其完全填满。

## 输入格式

输入的第一行为整数 $N$，表示生日派对的数量（$1 < N < 10000$）。接下来的 $N$ 行，每行包含两个整数 $R$ 和 $H$（$1 < R, H < 100$），分别表示每个派对中帽子的半径和高度。

## 输出格式

针对每个生日派对，输出一个实数 $h$，即西尔茜购置油漆的高度。结果需精确到小数点后6位。

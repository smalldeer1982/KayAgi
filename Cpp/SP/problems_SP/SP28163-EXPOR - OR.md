# EXPOR - OR

## 题目描述

给出一个具有 $ N $ 个整数的数组 $ A_{1}, A_{2}, A_{3}, \cdots, A_{N} $。随机选取 $i, j$ 使得 $ 1 \leq i < j \leq N $ 则对于 $ A_{i} | A_{j} $（$ A_{i} $ 按位或 $ A_{j} $）的数学期望是多少？

## 输入格式

第一行包括数据组数 $T$，每一组数据都将由两行组成。第一行表示数组的大小 $ N $，第二行包含 $ N $ 个整数，表示给出的数组。

## 输出格式

对于每一组数据，输出一个**最简分数**。即 $ a/b $ 且 $ gcd(a, b) = 1 $。

## 说明/提示

* $ 1 \leq T \leq 10 $
* $ 2 \leq N \leq 100,000 $
* $ 0 \leq A_{i} \leq 2^{31} $

## 样例 #1

### 输入

```
2
2
0 0
3
1 2 3
```

### 输出

```
0/1
3/1
```

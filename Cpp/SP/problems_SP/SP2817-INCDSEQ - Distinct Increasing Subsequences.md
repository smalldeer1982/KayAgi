# INCDSEQ - Distinct Increasing Subsequences

## 题目描述

### 题目大意  
给定含有N个数（1≤N≤10000）的整数序列：$S_1,S_2,...,S_N（0≤S_i<100000000）$

请计算长度为$K$（$1≤K≤N≤50$）的所有递增子序列数量。

## 输入格式

仅一行，含$N$和$K$两个数。

## 输出格式

输出长度为K的所有递增子序列的数量，并对5000000取模。

## 样例 #1

### 输入

```
4 3
1
2
2
10
```

### 输出

```
1
```

# RRSCHED - Round-Robin Scheduling

## 题目描述

一个计算机处理器需要执行 $N$ 个任务（$1 \le N \le 50,000$）。每个任务 $i$ 需要 $T_i$ 秒钟的处理时间（$1 \le T_i \le 1,000,000,000$）。处理器以如下方式运行这些任务：每个任务依次按顺序从 1 到 $N$ 各运行 1 秒，之后处理器又从任务 1 开始重复这个过程。任务一旦完成，就不会再参与后面的循环。请确定每个任务完成时所经过的总时间。

## 输入格式

输入的第一行为整数 $N$，表示任务的总数量。接下来的 $N$ 行中的每一行分别含有一个整数，表示第 $i$ 个任务所需的处理时间 $T_i$。

## 输出格式

输出共 $N$ 行，每行输出一个整数，表示任务 $i$ 完成时所花费的总时间。

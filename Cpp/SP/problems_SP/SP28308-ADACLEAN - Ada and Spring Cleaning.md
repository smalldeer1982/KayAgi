# ADACLEAN - Ada and Spring Cleaning

## 题目描述

瓢虫艾达决定做“春季大扫除”！

你知道的是，她有一个项目清单，瓢虫艾达非常的节俭，所以她会把所有的清洁活动都看成一个字符串。

在阅读字符串时，你可能会看不懂，不过瓢虫艾达有一个系统，每个活动的长度都是k个字符。

不幸的是，当新的活动被添加进项目清单时，会出现重复的现象。请你帮她找出有多少代办活动只出现了一次。

## 输入格式

第一行包含一个数T，表示测试样例的数量。

每一个测试样例都以N, K开头，分别是字符串的长度和活动。

下一行由长度为N 的小写字母组成。

所有测试样例中字符串长度之和不超过3*10^5

## 输出格式

对于每一个测试样例，输出长度为K的唯一子字符串的数量。

# ADAGAME2 - Ada and Easy Game

## 题目描述

瓢虫 Ada 经常和她的朋友 Velvet Mite Vinit 玩棋盘游戏。上次，他们玩了一种叫「数字游戏」的游戏，Vinit 有些不满，因为他觉得 Ada 利用了程序作弊。为了阻止这种情况，Vinit 创建了一种新的「坚果堆游戏」，并制定了自己的游戏规则。

游戏开始时，Vinit 在桌子上放了许多堆坚果，并选择了一些数字作为可以进行的合法操作。Ada 和 Vinit 轮流进行游戏（女性优先，Ada 先手）。在每一轮中，玩家可以从一堆中取走符合规则的坚果数量（这些数字不能大于该堆坚果的总数）。无法进行操作的人将输掉比赛。Vinit 对这个游戏充满热情，因为他相信 Ada 这次无计可施。

现在，让我们帮助 Ada 赢得比赛。

## 输入格式

第一行包含测试用例的数量 **T**。

每个测试用例的第一行有两个整数 **N** 和 **M**，分别表示坚果堆的数量和可用规则的数量。

接下来的第二行包含 **M** 个整数，表示可以取走的坚果数量的规则。

每个测试用例的第三行包含 **N** 个整数，表示每堆坚果的数量。

## 输出格式

对于每个测试用例，输出赢家的名字（“Ada” 或 “Vinit”）[假设双方都以最优化的策略进行游戏]。

- 每个测试用例中的坚果堆数量和规则数量 $1 \leq N, M \leq 100$

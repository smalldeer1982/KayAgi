# URI - Urinals

## 题目描述

相信去过男洗手间的人都会注意到一个不成文的“国际小便池选择协议”，这个协议大致规定了以下几条选择规则：

1. 应选择距离已被占用的小便池最远的一个。
2. 如果多个小便池满足第一点，应选择离门较远的那个。
3. 严禁选择紧挨着已占用小便池的选位，以避免尴尬。

现在有一排 **n** 个小便池，按照从左到右的顺序依次编号为 1 到 **n**。洗手间的门位于编号为 **n** 的小便池右侧。开始时，这些小便池全是空的。假设人们一个接一个地来到洗手间并依照上述协议选择小便池，且不会中途离开，那么第 **k** 个人会选择哪个小便池？

## 输入格式

第一行输入一个整数 **t**，表示测试用例的数量。接下来的每一行是一个测试用例，包含两个自然数 **n** 和 **k**，分别表示小便池的总数和第几个人。

$1 \le k \le n \le 10^{18}$

## 输出格式

对于每个测试用例，输出一行，表示第 **k** 个人在有 **n** 个小便池的情况下可以选择的小便池编号。如果在不违反协议的情况下无法选择小便池，就输出 "OOPS"。

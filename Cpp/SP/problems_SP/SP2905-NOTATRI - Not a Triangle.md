# NOTATRI - Not a Triangle

## 题目描述

给定 $N$ 个整数 $L_1, L_2, \ldots, L_N$，任务是计算出满足以下条件的三元组 $(i, j, k)$ 的总数：
- 索引满足 $1 \le i < j < k \le N$
- 值满足 $L_i + L_j > L_k$

## 输入格式

输入包含若干组测试数据。每组数据的第一行是一个整数 $N$，表示有多少个整数。接着一行包含这 $N$ 个整数 $L_1, L_2, \ldots, L_N$。当 $N = 0$ 时，表示输入结束，该组数据不需要处理。

## 输出格式

对于每组测试数据，输出一行，表示满足条件的三元组的数量。

## 说明/提示

- $3 \le N \le 10^5$
- $1 \le L_i \le 10^9$

### 输入

3

2

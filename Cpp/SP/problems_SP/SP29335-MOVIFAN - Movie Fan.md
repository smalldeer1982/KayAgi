# MOVIFAN - Movie Fan

## 题目描述

Alice 是一个狂热的电影迷，他想去看一部最近上映的电影。电影院提供多场电影放映，每场的开始时间和放映时长都已知。你的任务是帮助 Alice 计算他可以观看电影的不同组合数量。由于他非常喜欢电影，他可以观看多场不重叠的电影。

## 输入格式

第一行是整数 $t$，表示测试用例的数量。

每个测试用例由两个整数 $n$ 和 $l$ 组成，分别表示电影的场次数量和每场电影的时长。

接下来一行包含 $n$ 个整数，表示各场电影的开始时间。

$1 \le t \le 10$

$1 \le n, l \le 300000$

## 输出格式

输出 Alice 可以选择观看至少一场电影的组合总数，结果需要对 $1000000007$ 取模。

### 输入

```
3
3 4
3 8 12
3 1
1 2 3
3 3
3 5 9
```

### 输出

```
7
```

# SORTMUCH - How many can you sort?

## 题目描述

## 题意简述：
你发现了一种排序算法，可以在n log3 n时间内对n个整数排序。
现在要你求出在给定时间内最多排序多少整数：

## 输入格式

第一行一个整数t，表示数据组数
接下来t行，每行都是一个整数，为给定的时间

## 输出格式

对于从第二行起每一行输入，都应该有一行输出，表示在该行给定的时间内，最多能排序多少整数

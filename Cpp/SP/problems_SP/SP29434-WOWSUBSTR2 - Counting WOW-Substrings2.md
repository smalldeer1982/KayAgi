# WOWSUBSTR2 - Counting WOW-Substrings2

## 题目描述

给你一个字符串，你需要计算其中所有“WOW 子串”的总长度。“WOW 子串”指的是字符串中的一个连续子串，其中每个字符都只出现了一次，即子串中的所有字符都是唯一的。由于最终结果可能会很大，请将结果对 100007 取模后输出。（注意：100007 并不是一个质数。）

## 输入格式

输入首先是一个整数 **TC**（$\le 50$），表示测试用例的数量。每个测试用例包含一个字符串 $S$。

## 输出格式

对于每个测试用例，输出测试用例的编号，以及给定字符串中所有“WOW 子串”的总长度对 100007 取模后的结果。

## 说明/提示

- $1 \le \text{TC} \le 50$
- $1 \le |S| \le 10^5$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2\n3 21 2 14 32 1 3 2
```

### 输出

```
Case&nbsp;1: 7\nCase 2: 16
```

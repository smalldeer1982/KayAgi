# SHOOTING - Emmons

## 题目描述

在北京举行的第二十九届奥运会上，射击比赛结束后，马修·埃蒙斯因在 50 米步枪 3×40 男子比赛中最后一枪失误而被广泛关注。四年前在雅典，他因为打错目标而错失了即将到手的金牌。

**以下是蓝玛丽的幻想 :P**

埃蒙斯决定更加努力地练习射击。一年后，他的射击技艺得到了明显提高，甚至在不瞄准的情况下也能精准命中！如今，对于在第三十届奥运会上获得 50 米步枪 3×40 男子比赛的金牌，他充满信心。

他的妻子凯特琳娜·埃蒙斯——同样是一位杰出的射手——为他设计了一款游戏来保持他对射击的兴趣。玩家手中有 $n$ 发子弹，每发子弹都有一个对应的整数值（绝对值小于 10000）。游戏中有 $m$ 个靶子，并且每个靶子旁都有一个计分器，初始分值为 1。

在游戏过程中，玩家需要选择一发子弹并射击任意一个靶子。所有子弹都必须使用，不能留存，每个靶子必须被射击至少一次。

如果一颗值为 **X** 的子弹击中了一个靶子，该靶子的计分器分数将乘以 **X**。

游戏的最终得分是所有 $m$ 个计分器的总和。

现在，马修希望得到你的帮助，将他的最终得分提高到最高。之后，他会向你展示他出色的射击能力并最终达到这个分数。

**注：** 尽管上述情节是我的幻想，我仍然祝愿马修·埃蒙斯在伦敦第三十届奥运会上赢得 50 米步枪 3×40 男子比赛的金牌。

## 输入格式

首先输入一个整数，表示测试用例的数量（最多 50 个）。

对于每个测试用例：

第一行包含两个整数 $n$ 和 $m$。第二行包含 $n$ 个整数，表示每发子弹的数值。

## 输出格式

对于每组测试用例，输出一个整数，表示可能达到的最高最终得分。

## 说明/提示

- $1 \leq n \leq 2000$
- $1 \leq m \leq n$
- 子弹的绝对值均小于 10000

# ADARAINB - Ada and Rain II

## 题目描述

你可能已经了解，瓢虫阿达正在种植植物。她之前有一条很长的田垄，但围起来的花费太高。为了解决这一问题，她决定改建一个正方形的田地。这个田地非常大，然而大部分雨水仅仅落在田地的某个矩形区域内。为了不让植物缺水，阿达记录了所有的降雨情况，以便知道每株植物接收了多少水。然而，降雨次数太多，她一人难以处理。

首先，你会接收到 **N** 次降雨的记录，每个记录用一个矩形 \[x, y\], \[X, Y\] 表示，指出每次降雨落在这个矩形区域（指定左下角和右上角的坐标）。接下来是 **M** 次查询，每次查询会给出一个坐标 \[x, y\]，询问在这个位置上共接收到多少次降雨。

## 输入格式

第一行包含三个整数 **N, M, L**（**0 < N, M, L**），分别代表降雨次数、查询次数及正方形田地的边长。

接下来的 **N** 行，每行包含四个整数 **x, y, X, Y**（**1 ≤ x, y, X, Y ≤ L**），表示第 **i** 次降雨落在的矩形区域 \[x, y\], \[X, Y\]。

接下来的 **M** 行，每行包含两个整数 **x, y**（**1 ≤ x, y ≤ L**），表示询问坐标 \[x, y\] 上的降雨次数。

## 输出格式

输出 **M** 行，每行输出一个整数，表示对应查询位置 \[x, y\] 上降雨的总次数。

## 说明/提示

- \(0 < N, M, L \le 10^5\)
- \(1 \le x, y, X, Y \le L\)

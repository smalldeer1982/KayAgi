# ADAPICK - Ada and Cucumber

## 题目描述

Ada 是一只瓢虫，也是个农民。现在是收获黄瓜的季节，Ada 想要收集她家周围的黄瓜。她计划选择一个方向，沿着这个方向前进，直到收集完该方向上的所有黄瓜为止。

假设 Ada 的房子位于坐标系的原点 **\[0,0\]**，而黄瓜在平面上被表示为线段。每根黄瓜都没有穿过或接触到 Ada 的房子。

那么，假如 Ada 选择最佳的方向，她最多能收集多少根黄瓜呢？

## 输入格式

第一行输入一个整数 **T**，表示测试用例的数量。

每个测试用例的第一行包含一个整数 **N**。

接下来的 **N** 行中，每行有四个整数 **x1, y1, x2, y2**，表示每根黄瓜的起点和终点。每根黄瓜的长度都为正。

所有测试用例中，N 的总和不会超过 $10^6$。

虽然黄瓜不会穿过 Ada 的房子，但它们可能互相接触、相交或重叠。

## 输出格式

对于每个测试用例，输出一个整数，表示 Ada 在选择一个方向后能够收集到的最多黄瓜数量。

## 说明/提示

- $1 \le T \le 10^5$
- $1 \le N \le 10^5$
- $-10^6 \le x1, y1, x2, y2 \le 10^6$
- 所有测试用例中 **N** 的总和不超过 $10^6$。

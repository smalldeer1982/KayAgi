# LASTSHOT - THE LAST SHOT

## 题目描述

一个范围内有N枚炸弹。如果炸弹B1在炸弹B2的射程内，当炸弹B2爆炸时，炸弹B1也会爆炸，但不可反换，因为炸弹的范围不同。由于耗尽了能量，所以要只开一枪引爆炸弹。现在，要求贾维斯找到最合适的炸弹，用来触发最大的影响——炸弹爆炸的数量。

## 输入格式

第一行包含两个整数N和M，表示炸弹数量和关系数量。

接下来的M行包含两个整数A和B，表示炸弹B在炸弹A的范围内。

## 输出格式

最大影响值的炸弹

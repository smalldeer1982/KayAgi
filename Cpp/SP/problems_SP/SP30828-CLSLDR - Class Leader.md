# CLSLDR - Class Leader

## 题目描述

在行星 X 的新年，班级围绕一个独特的游戏来选拔新班长！

游戏规则如下：
1. 班上共有 **n** 位学生，编号从 1（第一位学生）到 **n**（最后一位学生）。
2. 一张纸被交给第 **m** 个学生。
3. 接下来，第 **o** 个拿到纸的学生将退出游戏。
4. 纸继续依次传递，直到仅剩一位未退出的学生。
5. 这位学生将被选为班长。

你的任务是根据给定的条件，找出这位学生的编号。

## 输入格式

第一行输入一个整数 **T**（0 ≤ T ≤ 10 $ ^{6} $），表示测试用例的数量。
接下来的 **T** 行中，每行包含三个整数 **n**（0 < n ≤ 10 $ ^{3} $）、**m** 和 **o**（0 < m, o ≤ n），各数之间用单个空格隔开。

## 输出格式

对于每个测试用例，输出最终成为班长的学生编号。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2\n4 1 2\n5 2 3
```

### 输出

```
2\n1\n\n
```

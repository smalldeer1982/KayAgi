# COPSEQ - Non Coprime Sequences

## 题目描述

给你两个整数，$n$和$m$。
找到并输出满足以下条件的长度为$n$的序列数：
- 序列的所有元素都是能整除$m$的正整数。
- 对于任何两个相邻的元素，比如$p$和$q$，至少有一个素数$x$，满足$min(q,p)<x<max(q,p)$。\
请输出这样的序列，并对$10^{9}+7$取模。

## 输入格式

一行输入包含两个整数$n$和$m$。
数据保证$0<n,m$。

## 输出格式

输出符合题意的有效序列，并对$10^{9}+7$取模。

## 样例 #1

### 输入

```
2 10
```

### 输出

```
7
```

# NGIRL - Namit In Trouble

## 题目描述

Namit 的女友生日快到了。他走进一家礼品店，看到店里有 $N$ 件礼物整齐地排成一排，每件礼物的位置就对应着它的价格。（位置从 1 开始计数。）

由于 Namit 的女友是一位数学爱好者，她偏爱那些恰好拥有 3 个因数的位置上的礼物。因此，Namit 决定只买放在这样的位置上的礼物。此外，Namit 的女友还希望礼物的价格要高于某个特定的金额 $K$。

此时，Namit 想知道，在给定 $N$ 的情况下，总共有多少种可以挑选的礼物，而其中又有多少是他的女友会喜欢的。

## 输入格式

输入首先是一个整数 $T$（$1 \le T \le 1000$），表示测试用例的数量。接着是 $T$ 行，每一行包含两个整数 $N$ 和 $K$（$1 \le N < 10^{10}$，$1 \le K \le 10^{10}$）。

## 输出格式

对于每个测试用例，输出两个数字。第一个数字表示符合条件的礼物总数量，第二个数字表示其中 Namit 的女友会喜欢的礼物数量。

# ODDDIV - Odd Numbers of Divisors

## 题目描述

## 题目大意
给你一个**正奇数**$K$，和两个正整数$Low$和$High$，问在$[Low,High]$之间共有几个数恰好有$K$个因子

## 输入格式

第一行一个正整数$C$，代表有$C$组测试数据  每组数据中是三个正整数$K$,$Low$,$High$  
$0 < C < 100000$，$1 < K < 10000$，$0 < Low ≤ High < 10^{10}$

## 输出格式

对于每组数据，输出$[Low,High]$之间共有$K$个因子的数的个数

## 样例 #1

### 输入

```
3
3 2 49
9 1 100
5 55 235
```

### 输出

```
4
2
1
```

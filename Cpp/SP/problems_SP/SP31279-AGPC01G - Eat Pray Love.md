# AGPC01G - Eat Pray Love

## 题目描述

一共 $N$ 个人，他要么单独一个人，要么与其他人配对，若每个人都能与其他人配对，一共有多少种方式？

## 输入格式

第一行是 $T$（$1 \le T \le 100000$），表示接下来的行数。

接下来的 $T$ 行每行包含一个整数 $N$（$1 \le N \le 100000$）。

## 输出格式

由于答案可能过大，所以答案对 $1000000007$ 取模。

## 样例 #1

### 输入

```
2
3
100000
```

### 输出

```
4
823421181
```

# JOSWAP - Just One Swap

## 题目描述

## 题意
您将获得一个大小为N的数组。通过将两个数字恰好交换一次，可以生成多少个不同的数组？两个选定的数字可以相等，但它们在数组中的位置必须不同。

## 输入格式

输入的第一行包含一个整数T，表示测试用例的数量。每个测试用例均以整数N开头。下一行包含N个整数，即数组的编号。

## 输出格式

对于测试点输出一行，即答案。

## 数据约定
$1 \le T \le 5$

$1 \le$ 数组中数字的值 $\le 100000$

$2 \le N \le 100000$

## 样例 #1

### 输入

```
1 \n5 \n2 3 2 3 3
```

### 输出

```
7
```

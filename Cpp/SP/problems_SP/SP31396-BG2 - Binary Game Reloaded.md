# BG2 - Binary Game Reloaded

## 题目描述

Alice 和 Bob 非常喜欢玩各种游戏。有一天，他们发现找不到任何游戏可以玩。

## 输入格式

输入的第一行是一个整数 **T (1≤T≤30)**，表示测试用例的数量。

接下来的每个测试用例是一个二进制字符串，可能带有前导零。该字符串表示一个不超过 **100000 位** 的二进制数。

## 输出格式

对于每个测试用例，输出游戏的获胜者名字。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
3
 00
 01
 10
```

### 输出

```
Bob
 Bob
 Alice
```

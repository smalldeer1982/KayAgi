# GAMEMVS - ShaatChara

## 题目描述

Meena 和 Razu 玩了一种叫 Shat Chara 的游戏很多年，但他们觉得这游戏有点暴力。于是，他们想出了一种新的玩法。新游戏中不再只有一堆石子，而是有 $n$ 堆。游戏规则是两人轮流操作。在每轮中，玩家可以选择任意一堆至少还有 1 个石子的石堆，从中移除任意数量的石子（至少一个）。游戏一直持续到所有石子被移除。当轮到某个玩家时，如果他无法进行任何合法操作，他就输了游戏。

最初，Razu 对新游戏感到很满意，因为这个版本没有受伤的风险。但他的满意很快变成了沮丧，因为 Meena 总是能赢。Meena 非常聪明，总能找到最佳策略，无论局面如何复杂，只要她有机会，她都会赢。因此，Razu 来找你，希望你能帮助他分析当前的游戏局面，计算出他有多少种移动选择可以使 Meena 的胜利不再是必然的。

## 输入格式

第一行包含一个整数 $n$，表示石子堆的数量。  
第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$，其中 $a_i$ 表示第 $i$ 堆石子的数量。

## 输出格式

输出一个整数，表示 Razu 可以选择的，不保证 Meena 必然胜利的移动方案数。

## 说明/提示

- $1 \le n \le 10^5$
- $1 \le a_i \le 10^9$

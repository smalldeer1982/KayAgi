# LINES - Game of Lines

## 题目描述

Farmer John 向 Bessie 发起了以下游戏的挑战：FJ 有一个棋盘，棋盘上标有 $N$ $(2 \le N \le 200)$ 个不同格子点的点。第 $i$ 个点的整数坐标为 $X_i$ 和 $Y_i$ $(-1,000 \le X_i,Y_i \le 1000)$ 。

Bessie 可以通过选择两个点并在它们之间画一条直线来在游戏中得分。但是，她不能画两条及以上的平行线。Bessie能画的线数为她的得分。Bessie 想知道她的最大得分，因此她请您帮助找到她可以获得的最高分。

## 输入格式

本题有多组测试数据。

对于每一组测试数据，第一行为一个正整数$N$，为点的个数。
接下来的 $N$ 行，每行两个整数 $X_i$ 和 $Y_i$，为点的坐标。

当 $N$ 等于 $0$ 时，本组数据结束。

## 输出格式

对于每组数据，输出 Bessie 的最大得分。

## 样例 #1

### 输入

```
4
-1 1
-2 0
0 0
1 1
0
```

### 输出

```
4
```

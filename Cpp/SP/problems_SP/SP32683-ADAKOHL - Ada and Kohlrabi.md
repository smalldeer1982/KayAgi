# ADAKOHL - Ada and Kohlrabi

## 题目描述

Ada 是一位瓢虫农夫，她经营着一个种植球茎甘蓝的花园。每个球茎甘蓝都被赋予了一个质量值，这个值可能为负数，因为球茎甘蓝可能已经腐烂。

Ada 希望收集一些球茎甘蓝，为了做到这一点，她打算选一条直线上生长的球茎甘蓝，使得这些球茎甘蓝的质量值之和最大。你能帮她找到这样的一条直线吗？

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量（注意，对于最大的测试用例，只有一个）。

每个测试用例的第一行包含一个整数 $N$，表示球茎甘蓝的数量。

接下来的 $N$ 行中，每行有三个整数 $x, y, q$，分别表示球茎甘蓝所在的坐标和它的质量值。坐标和质量值的范围为 $-10^9 \leq x, y, q \leq 10^9$（确保不会有两个球茎甘蓝长在同一个坐标上）。

## 输出格式

对于每个测试用例，输出一个整数，表示在单条直线上能获得的最大球茎甘蓝质量值之和。

## 说明/提示

- $1 \leq T \leq 10$
- $1 \leq N \leq 10^5$
- $-10^9 \leq x, y, q \leq 10^9$

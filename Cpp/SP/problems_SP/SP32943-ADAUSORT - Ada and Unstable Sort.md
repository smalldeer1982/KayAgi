# ADAUSORT - Ada and Unstable Sort

## 题目描述

瓢虫艾达（Ada the Ladybug）做了一个关于算法的讲座。她了解到排序可以在 O（NlogN） 时间内完成。她还学习**了稳定排序**的概念。对于那些错过了讲座的人来说，这意味着只要数组中有两个相等的元素，它们的相互位置就不会在排序后改变。

艾达想想出一些新的东西，所以她提出了**不稳定的排序**。它的排序具有以下属性：“只要数组中有两个相等的元素，**它们的相互位置就会在排序后发生变化**”。

由于 Ada 对此只有理论知识，因此她要求您为她构建这样的算法。

## 输入格式

每个测试用例的第一行将包含一个整数 $N$，即数组的长度。

下一行将包含 $N$ 个整数 ${m_j}$ ，即数组的元素。

## 输出格式

对于每个测试用例，打印 $N$ 个数字，即它在原始数组中所在的每个元素的索引。索引以 1 开头。

### 输入输出样例
无

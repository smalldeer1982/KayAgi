# ADACHESS - Ada and Chess

## 题目描述

瓢虫艾达正在和她的好朋友天鹅绒螨维尼特下棋。他们在比赛中互相交谈，说“我会在接下来的X步中带走你的身影”。比赛结束后，这真的让阿达很困扰，所以她决定多考虑一下。 

因为她不能确定它“那么快”，她让你做一个程序，可以输出两个图形可以相互接近的最小移动次数(注意，它们不必交替，所以所有的移动只能由其中一个图形完成)。还要注意，是经典的8x8棋盘。 

有趣的数字将是:国王，骑士，女王，塔，主教。

## 输入格式

第一行输入将包含1，测试用例的数量。
每个测试用例将包含6个整数f x y F X Y，其中0是图形的类型(0到4代表国王、骑士、王后、塔、主教，依次排列)0是给定图形的坐标。

## 输出格式

对于每个测试用例输出，最小的移动次数将两个数字放在同一个“盒子”中。万一数字不符合，输出“INF”。

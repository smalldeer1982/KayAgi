# ADACITRS - Ada and Lemon 2

## 题目描述

Ada是一个农民，她种植了一棵大柠檬树。树上每个叶子上都长有一个柠檬。Ada想要摘取一个柠檬。她想知道在摘取一个柠檬后会剩下多少种不同的树。如果两棵树在所有节点标签的排列下都不同，则认为它们是不相同的。贡献者：fujiarui

## 输入格式

第一行包含一个数字$N$ ，表示树的节点数。

接下来的$N-1$行每行包含两个整数.$a$,$b$.代表树的边。

## 输出格式

在摘取一个柠檬后剩余的不同树的数量。

## 说明/提示

$25 ≤ N ≤ 3*105$ , $0 ≤ a, b < N(a ≠ b)$

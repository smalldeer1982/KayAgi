# ADAGAME5 - Ada and Palace Game

## 题目描述

## 题目内容

瓢虫Ada和她的朋友Vinit发明了一种叫“宫殿”的游戏。

几座宫殿被放在棋盘上，Ada和Vinit交互移动，在每一步其中，他们可以选择棋盘上的任意一座宫殿，让它向左下方移动，这也意味着离左下角的曼哈顿距离会减少，不能移动的一方就会输，你能判断谁是赢家吗？

注：两个宫殿可以共享一个棋盘（把它们叠在一起），并且因为Ada是一名女士，所以她会先走。

这个棋盘是Ada的好朋友Feodorv的礼物，但他不懂得下棋，所以左下角不是$a1a1$而是$[0,0]$，右上角不是$h8h8$而是$[2999,2999]$。

## 输入格式

第一行输入一个整数$T$，表示测试样例的数量。

对于每个测试样例，先输入一个整数$N$，表示宫殿的数量。

在接下来的$N$行中，输入两个整数$x$和$y$，表示宫殿的坐标。

## 输出格式

对于每个测试样例，输出获胜者的名字，即Ada或Vinit。

## 输入\输出样例
### 输入样例
```
5
1
3 2
1
3 3
1
6 6
3
7 4
3 7
1 0
5
4 2
6 9
7 8
2 1
5 5
```
### 输出样例

```
Ada
Vinit
Vinit
Vinit
Ada
```

## 说明/提示

$1 ≤ T,N ≤ 3\times10^5$。
 
$0 ≤ x, y < 3000$。

所有$N$的和不会超过$3\times10^5$。

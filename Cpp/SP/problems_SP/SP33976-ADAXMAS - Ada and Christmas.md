# ADAXMAS - Ada and Christmas

## 题目描述

Ada the Ladybug 最近在拆礼物，你猜怎么着？她从她的好朋友 Flea Feodorv 那里得到了一个新的棋盘。由于「棋盘越大，棋类游戏就会越好」，他给了 Ada 一个比上次大的棋盘：现在右上角的点是 $[999999999,999999999]$（$10^9 - 1$）。

正如你可能猜到的那样，她不能在这种棋盘上玩经典的（或快速的）国际象棋。但是因为她不想丢掉这一件如此贵重（至少很大）的礼物，她决定了要再玩一次 [宫殿游戏](https://www.luogu.com.cn/problem/SP33963) 。她邀请了她的朋友 Velvet Mite Vinit。当他们最近玩这个游戏时，他们决定要扩展一点规则。他们决定了只用物种最重要的象棋棋子来玩这个游戏：**车**、**象**、**王**、**马**，并且很明显还有 **宫殿**。

游戏的规则很简单：一些 **棋子** 被放置在棋盘上。Ada 和 Vinit 交替走棋。在每一次走棋中，可以选择棋盘上的任何 **棋子** 并且把它移向左下角。移向意味着，这个棋子到左下角的 [曼哈顿距离](https://en.wiktionary.org/wiki/Manhattan_distance) 会减少并且棋子的任何坐标都不会增加。不能走棋的那个玩家就输了。你可以说出当他们都以最佳策略走棋时，谁会赢吗？因为 Ada 是女士，所以她先走。

注意：规则和原版宫殿游戏有 *一点点* 不一样（尽管这不会影响原版游戏，这个游戏也有可能产生略有不同的结果）。

## 输入格式

输入的第一行会包含 $1 \le T \le 3 \times 10^5$，表示数据组数。

接下来 $T$ 行会包含 $1 \le N \le 3 \times 10^5$，表示棋盘上已放置的棋子数。

每一组数据会包含 $N$ 行，每一行含有两个整数和一个字符：$0 \le x, y \le 10^9$，表示象棋棋子的坐标，并且 $c \in \{\mathtt{T}, \mathtt{B}, \mathtt{K}, \mathtt{N}, \mathtt{P}\}$（分别表示车、象、王、马和宫殿）。

所有数据组中的 $N$ 的和不会超过 $3 \times 10^5$。

## 输出格式

对于每组数据，输出胜者的名称，`Ada` 或 `Vinit`。

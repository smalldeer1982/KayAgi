# GLOVE - Choosing Gloves

## 题目描述

化学教授 Acidrain 的地下室里有两个抽屉：一个放左手手套，另一个放右手手套。在每个抽屉中，有 $n$ 种不同颜色的手套。教授对每种颜色的手套在每个抽屉中的数量了如指掌（同一种颜色在两个抽屉中的手套数量可能不同）。教授确信总能找到一双颜色相同的手套。

在进行实验时，教授只能使用同一颜色的一双手套。因此，每次实验前，他都得去地下室取手套，希望能够至少拿到一双颜色相同的手套。地下室很黑，教授在不出地下室的情况下无法识别手套的颜色。他厌烦多次往返地下室（万一第一次没有拿到一双匹配的手套），也不愿拿过多不必要的手套到实验室。

### 任务

编写一个程序以实现以下功能：
- 从标准输入读取手套的颜色数量和每种颜色的手套在两个抽屉中的数量。
- 计算出至少需要拿多少只手套，才能确保其中至少有一双颜色相同的手套（需要明确指出从每个抽屉中各拿多少只手套）。
- 将结果输出到标准输出。

## 输入格式

第一行输入一个正整数 $n$ ($1 \le n \le 100000$)，表示手套的颜色种类数。接下来的 $n$ 行，每行有两个正整数 $l_i$ 和 $r_i$（$1 \le l_i, r_i \le 100000$），分别表示第 $i$ 种颜色的手套在左手抽屉和右手抽屉中的数量。

## 输出格式

第一行输出从左手抽屉中需要拿的手套数量。第二行输出从右手抽屉中需要拿的手套数量。这两数之和应尽可能小。如果有多个正确答案，程序可以输出任意一个。

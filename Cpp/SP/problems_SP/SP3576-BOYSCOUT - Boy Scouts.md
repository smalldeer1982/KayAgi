# BOYSCOUT - Boy Scouts

## 题目描述

新英格兰的童子军每年都会举办一次童子军奥运会。每支队伍需完成特定任务，累积分数后宣布获胜者，然后围着篝火整夜弹吉他、唱歌。今年，他们选择在缅因州的一片美丽森林举办奥运会。比赛的任务只有一个但颇具挑战性。队伍需要从一棵树出发，沿直线到达另一棵树，然后继续移动，直到回到起点。完成路线上的每棵树都会增加积分。运动的规律是只能逆时针方向，即到达新树后必须保持向左转动，不得超过180度。最后，当他们再次回到起点时，路线必须可重复且保持逆时针方向，这条路线应该是一个凸多边形的边界。因为现场不能使用电脑，童子军希望你帮助计算他们能达到的最高分数。

## 输入格式

输入的第一行是一个整数 $N$（$3 \le N \le 100$），表示森林中树木的数量。接下来的 $N$ 行，每行包含两个用空格隔开的实数 $x$ 和 $y$（$-10^6 \le x, y \le 10^6$），表示树木的坐标。坐标最多保留两位小数。没有任意三棵树在同一条直线上。

## 输出格式

输出一个整数，表示队伍可以获得的最大分数，并换行。

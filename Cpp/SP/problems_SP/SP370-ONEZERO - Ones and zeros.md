# ONEZERO - Ones and zeros

## 题目描述

在正整数中，有些数字仅由```1```和```0```构成，例如：```101```。给定一个正整数，请找到它的一个倍数，使得此数符合上述性质。

## 输入格式

第一行为数据组数 $K$ （ $K$ 约等于 $1000$ ）

在接下来 $K$ 行，每行一个整数 $n$ （ $1\leq n\leq20000$ ）

## 输出格式

对于每个测试用例，输出数字的最小倍数，该数字仅由数字```1```和```0```（从```1```开始）组成。

## 样例 #1

### 样例输入 #1

```
3
17
11011
17
```

### 样例输出 #1

```
11101
11011
11101
```

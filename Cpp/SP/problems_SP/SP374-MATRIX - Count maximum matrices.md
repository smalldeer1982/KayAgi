# MATRIX - Count maximum matrices

## 题目描述

给定一个 $M$ 行 $N$ 列的矩阵 $A$，矩阵元素为数字 $0$ 或 $1$。对于其中的任意一个子矩形（边长不小于 1），定义 $X1$ 为该子矩形四条边上数字 $1$ 的数量，$X0$ 为该子矩形四条边上数字 $0$ 的数量，定义该子矩形的“价值”为 $X1 - X0$。我们需要找到所有子矩形中具有最大价值的值 $W$，以及具有该最大价值的子矩形的数目 $S$。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的数量（$t \le 15$）。对于每个测试用例，第一行包含两个整数 $M$ 和 $N$，表示矩阵的行数和列数（$1 \le M, N \le 200$）。接下来 $M$ 行中，每行包含 $N$ 个数字 $0$ 或 $1$，表示矩阵的元素。

## 输出格式

对于每个测试用例，输出一行，包含两个整数 $W$ 和 $S$，分别表示最大价值和具有该最大价值的子矩形的数量。

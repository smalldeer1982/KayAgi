# VCIRCLES - Area of circles

## 题目描述

给出$N$个圆，求它们的并的面积

## 输入格式

第一行一个正整数$N(1 \leq N \leq 50)$表示圆的个数

接下来$N$行每行三个整数$x_i,y_i,r_i$表示一个圆，保证圆覆盖的面积的并的所有点的横纵坐标的绝对值不会超过$10^4$

## 输出格式

一行一个浮点数表示圆的并的面积，保留五位小数

## 样例 #1

### 输入

```
2
5 6 3
5 5 5
```

### 输出

```
78.53982
```

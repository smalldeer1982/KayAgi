# MCOINS - Coins Game

## 题目描述

### 题目大意
Asen 和 Boyan 正在玩如下的游戏。它们选出两个不同的正数 K 和 L，并用一个有 n 个硬币的塔开始游戏。Asen 开始，然后是 Boyan，以此类推。当轮到某个玩家时，他可以从塔上取出 1 或 K 或 L 个硬币。取走最后一个硬币的人赢得比赛。玩了很久之后，Asen 发现有一种方法，无论 Boyan 怎么玩，他都可以赢。反之亦然。

所以，在游戏开始之前，Asen 很想知道自己会赢还是会输。写一个程序，帮助 Asen 用给定的 K，L，N 来预测游戏结果。

## 输入格式

输入包含了 m 局游戏。

第一行包含三个整数 K，L 和 m，$1 < K < L < 10, 3 < m < 50$。第二行包含 m 个数字：$N_1,N_2,...,N_m$

## 输出格式

输出包含一个长度为 m 且由 A, B 组成的字符串。如果 Asen 能在不论 Boyan 怎么玩的情况下赢得游戏，就输出 A，反之就输出 B。

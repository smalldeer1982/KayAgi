# MTRIAREA - Maximum Triangle Area

## 题目描述

在一个平面上给定 $n$ 个不同的点，你需要找到一个面积最大的三角形，其顶点来自这些给定的点。

## 输入格式

输入由多组测试数据组成。每组数据的第一行是一个整数 $n$，代表平面上的点的数量。接下来的 $n$ 行中，每行有两个整数 $x_i$ 和 $y_i$，表示第 $i$ 个点的坐标。输入的最后一行是一个整数 $-1$，表示输入结束，这一行不需要处理。可以假设 $1 \le n \le 100$，且所有坐标的绝对值不超过 $10^4$。

```
样例输入
3
3 4
2 6
2 7
5
2 6
3 9
2 0
8 0
6 5
-1
```

## 输出格式

对于每组测试数据，输出一行，包含最大面积，结果保留两位小数。你可以放心，总是可以找到一个面积大于零的解。

```
样例输出
0.50
27.00
```

## 说明/提示

- $1 \le n \le 100$
- 所有坐标的绝对值均不超过 $10^4$。

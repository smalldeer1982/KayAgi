# MCIRGAME - Point Connection Game in a Circle

## 题目描述

这是一款古老而有趣的小游戏。你需要将数字 1, 2, 3, ..., 2n−1, 2n 顺时针依次写在地上，围成一个圆圈。然后，用一些直线段将这些数字两两连接，形成数字对。每个数字必须恰好与另一个数字配对，并且不允许任意两条线段相交。这听起来很简单，对吧？不过，当你完成这个任务后，能否告诉我，有多少种不同的方法可以将这些数字配对呢？生活是不是比想象中更具挑战性呢！

## 输入格式

输入文件中的每一行包含一个正整数 n，表示有 2n 个数字需要配对。输入的最后一行为数字 -1，表示输入结束。你可以假设 1 ≤ n ≤ 150。

### 样例输入
```
2
-1
```

## 输出格式

对于输入的每个 n，输出可以将 2n 个数字配对的方法数，每个结果占据一行。

### 样例输出
```
2
```

## 说明/提示

注意：由于 n 可达到 150，结果可能是一个很大的数字。

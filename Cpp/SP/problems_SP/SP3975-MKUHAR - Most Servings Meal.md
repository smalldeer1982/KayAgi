# MKUHAR - Most Servings Meal

## 题目描述

丽莎在一家餐厅当服务员。今晚是丽莎的生日，因此她请求厨师为她的朋友们准备一道特别的餐点。这道餐点需要用到 $N$ 种食材。制作一份餐点需要每种食材一定的数量。

厨房里已经备有部分食材，丽莎需要去杂货店购买剩余的部分。店里所有的必要食材都有售，每种食材都有小包装和大包装两种规格。丽莎手头有 $M$ 美元，她希望能够合理地使用这笔钱，让厨师制作出尽可能多份餐点。

## 输入格式

第一行包含两个整数 $N$ 和 $M$，分别表示食材的种类数和她的预算金额（$1 \le N \le 100, 1 \le M \le 10^5$）。

接下来的 $N$ 行中，每行描述一种食材，包含六个整数：$X, Y, S_M, P_M, S_V, P_V$，它们分别代表：
- 每份餐点对该食材的需求量 $X$（$1 \le X \le 100$）
- 厨房已有的这种食材数量 $Y$（$0 \le Y \le 10000$）
- 小包装中含有的食材数量 $S_M$（$1 \le S_M < S_V$）
- 小包装的价格 $P_M$（$1 \le P_M < P_V$）
- 大包装中含有的食材数量 $S_V$（$1 \le S_V \le 10000$）
- 大包装的价格 $P_V$（$1 \le P_V \le 10000$）

## 输出格式

输出一个整数，表示丽莎花费预算后，厨师最多可以制作的餐点份数。

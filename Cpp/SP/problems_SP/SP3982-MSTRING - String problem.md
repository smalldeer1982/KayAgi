# MSTRING - String problem

## 题目描述

定义一个字符串 $A$ 的子序列为，从该字符串中选取一个或多个字符，并且不改变其原有顺序而得到的新字符串（字符不需要连续）。

现有两个字符串 $VOKI$ 和 $TOKI$。请编写程序，计算出 $VOKI$ 的所有子序列中，长度最短且不是 $TOKI$ 的子序列的那个子序列的长度。

## 输入格式

输入文件的第一行是字符串 $VOKI$，第二行是字符串 $TOKI$。输入保证只包含小写英文字母（‘a’- ‘z’）。每个字符串的长度不会超过 $1000$。

注意：输入的数据保证一定存在解。

## 输出格式

在输出文件的第一行打印出需要找到的子序列的长度。

# BONFIRE - Equatorial Bonfire

## 题目描述

设计师们提出一个想法：
沿着赤道连续放置火药，然后在不同的时间在火药带的不同点来点火。
火会沿着赤道向两个方向蔓延，随后整个赤道都会燃烧。
设计师们关心的一个问题是：什么地方是最后一个起火的地方，因当在什么时候起火？

## 输入格式

输入内容最多由10个块组成，每个块指定赤道篝火的单独方案。每个区块的第一行以每小时经度指定火力前进的速度。下一行包含篝火的数量N(N<=5000) ，篝火将沿着赤道。接下来的N行中的每一行都包含两个数字T和L，用于指定篝火的时间T(T为正数)小时 和位置L(-180<=L<=180),(L以经度为单位)。
从格林尼治标准时间12:00am开始。除外，所有数字的精度最多为小数点后2位。每个块后面都有一个空行。最后一个块后面是一个空行，然后是一个包含-1的行。

## 输出格式

对于输入文件中的每个块，输出一行，其中包含上次着火的时间和坐标。时间和坐标的单位和范围精度为小数点后3位。
提示： 如果有多个解决方案，输出从零子午线向东（即正向）移动时最接近的解决方案。

## 样例 #1

### 输入

```
2
2
1 90
1 -90

10.0
3
1 40
2 45
6 -80

-1
```

### 输出

```
46.000 0.000
15.500 -175.000
```

# DROOT - Multiplicative digital root

## 题目描述

给定一个整数，找出它的乘法数字根。具体做法是，对此整数的所有非零数字进行相乘，并不断重复这个过程，直到结果是个位数。这个个位数就是该整数的乘法数字根。例如，2009 的乘法数字根是 8，因为在第一次迭代中，数字 2 和 9 相乘得到 18，接着在第二次迭代中，数字 1 和 8 相乘得到 8，因此结果为 8。

## 输入格式

输入的第一行包含一个整数 $T$，表示测试用例的数量。接下来的 $T$ 行中，每行包含一个大正整数 $n$。

## 输出格式

对于每个整数 $n$，输出其对应的乘法数字根。

## 说明/提示

$$1 \le T \le 10^5, \quad 1 \le n < 10^{100}$$

### 输入

```
4
6

555555555

```

### 输出

```
6
8

2
```

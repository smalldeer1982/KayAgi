# FCANDY - Candy (Again)

## 题目描述

你和朋友有一大袋糖果。为了保持身材，你希望两个人分到的糖果热量尽可能一致。也就是说，你需要将糖果分成两组，使每组总热量差尽可能小。

## 输入格式

第一行包含一个整数 $N$，表示糖果的种类数量（$1 \le N \le 100$）。接下来的 $N$ 行中，每行包含两个整数，$k_i$ 和 $c_i$，分别表示这一种类糖果的数量和每颗糖果的热量。假设 $1 \le k_i \le 500$，且 $1 \le c_i \le 200$。

## 输出格式

输出一个整数，表示两组糖果的热量差最小值。

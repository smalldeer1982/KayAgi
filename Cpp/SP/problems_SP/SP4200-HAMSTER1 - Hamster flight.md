# HAMSTER1 - Hamster flight

## 题目描述

在 Hamsterburg 正在进行一场飞鼠比赛。每只参赛的飞鼠都是通过弹弓发射的。评委根据飞行的距离和高度来评分。假设飞行距离为 $X$ 米，飞行中的最大高度为 $Y$ 米，那么该飞行将获得的分数为 $K1 \times X + K2 \times Y$ 分。飞鼠的初始速度为 $V0$ 米/秒。重力加速度为 $g = 10$ 米/秒$^2$。我们忽略空气阻力，飞鼠和弹弓的大小也可忽略不计。当飞鼠从弹弓中发射时，起始高度为 0 米。你需要计算出一个最优的发射角度，使得飞鼠获得的分数最高。

## 输入格式

输入的第一行为一个整数 $t$，表示测试的数量。接下来的 $t$ 行中，每行包含三个用空格分隔的整数，分别是 $V0$、$K1$ 和 $K2$。

## 输出格式

对于每个测试，输出两个数值：发射飞鼠的最佳角度（以弧度表示）和飞鼠获得的最高分数。这两个数之间用空格隔开，并且结果要求保留三位小数。

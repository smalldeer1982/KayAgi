# ASSIGN - Assignments

## 题目描述

无

## 输入格式

输入的第一行包含测试组数量c（1 <= c <= 80）。每个测试组的学生数n（1 <= n <= 20）。接下来的n行中的每行包含n个整数，这些整数描述一个学生的偏好。在第i个位置上为1表示该学生喜欢第i个话题，为0则表示他不喜欢该话题。

## 输出格式

对于每个测试组，输出共有多少种分配方案。

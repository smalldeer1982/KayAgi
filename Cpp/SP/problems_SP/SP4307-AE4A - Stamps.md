# AE4A - Stamps

## 题目描述

小约翰正用他的小魔法印章在一张 $k \times k$ 的方格纸上画兔子。这张纸分成许多边长为 1 的小方格。小约翰的印章是一个 $3 \times 3$ 的正方形，其中有两个凸起的小方格，它们在同一行或者同一列。每当小约翰用这个印章按在纸上，凸起的部分正好会对应纸上的某些小方格，并且这些方格的颜色会翻转——从黑变白或从白变黑。印章可以部分伸出纸张边界，但凸起的小方格必须完全位于纸张内。而且，印章只能平移，不能旋转。

开始时，整张纸都是白色的。兔子由部分黑色方格组成，剩下的都是白色。小约翰试了很久，仍然无法用他的小印章画出兔子（这并不意味着不可能，只是非常困难）。于是，他向哥哥大约翰求助。

大约翰可以给他一个更大的魔法印章。大印章的尺寸是 $s \times s$，上面有若干个凸起的小方格（这些方格不一定在同一行或同一列）。这个大印章的使用方式和小印章一样，但它只能在完全位于纸张内部的情况下使用。

在给小约翰大印章之前，大约翰想确定这两个印章是否能够一起合作完成兔子的图案。他请你帮忙来判断这一点。

## 输入格式

第一行输入一个整数 $t$（$1 \le t \le 10$），表示测试用例的数量。

每个测试用例的第一行包含两个整数 $s$ 和 $k$（$1 \le s \le k \le 1000$，$1 \le s \le 200$），分别表示大印章的尺寸和纸张的尺寸。

接下来的三行描述小约翰的印章，每行由三个数字 0 或 1 组成。这里 0 表示白色，1 表示黑色。在这三行中，恰好有两个字符是 1，并且它们要么在同一行，要么在同一列。请注意，这种描述并不是印章的真实设计，而是印章在纸上的效果。

接下来的 $s$ 行以类似格式描述大约翰的印章；这部分描述可能包含任意数量的 1。之后的 $k$ 行描述了兔子的图案，格式与之前的印章描述相同。1 依然表示黑色，0 表示白色。

## 输出格式

对于每个测试用例，输出一行，内容为 TAK（表示“是”）或 NIE（表示“否”），判断是否可以用这些印章顺利画出兔子的图案。

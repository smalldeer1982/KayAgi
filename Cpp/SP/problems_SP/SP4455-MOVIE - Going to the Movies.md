# MOVIE - Going to the Movies

## 题目描述

### 题目背景
$N（N \le 120）$ 位朋友决定一起去当地的电影院看电影。他们都买了同一排的票。因为还有一段时间电影才开始，所以他们每个人都拿着票去购物了，直到电影开始才回来。

他们回来时都迟到了，电影已经开始了。站在门口的引座员同意让他们一个接一个地进去。每个女孩都应该找到自己的座位并坐下。

然而，打印他们票的机器出故障了。他们并没有收到连续的座位号，而是每个人都收到了一个介于 $1$ 和 $K$ 之间的随机座位号，其中 $K$ 是他们那一排的座位数。他们收到的座位号并不一定都是唯一的。

当女孩们试图坐下时，她们会从标有 $1$ 号座位的那一端进入排座，并一直走到她们票上的座位号所在的位置。如果她想要的座位是空的，她就会直接坐下。如果那个座位已经被占了，她就会继续朝同一个方向走，直到找到第一个空座位并坐下。
当然，也有可能某个不幸的女孩会一直走到排座的尽头也找不到座位。在这种情况下，引座员会过来把她赶出去。

给定数字 $N$ 和 $K$。$（1 <= K <= 200）$

假设每个女孩的票上的号码在 $1$ 到 $K$ 之间（包括 $1$ 和 $K$）。每个号码都是均匀随机抽取的，且抽取是独立的。

还假设当第一个女孩开始找座位时，整排座位都是空的。

计算至少有一个女孩遭遇被引座员赶出去的悲惨命运的概率。

## 输入格式

输入文件的第一行包含一个整数 $T$，指定测试用例的数量。每个测试用例之前有一个空行。

每个测试用例由一行组成，包含两个整数 $N$ 和 $K$。

## 输出格式

对于每个测试用例，输出一行，表示概率的最简分数。（在`/`符号之前或之后不要输出任何空格。）

## 样例 #1

### 输入

```
3

1 10

2 3

3 3
```

### 输出

```
0/1
1/9
11/27
```

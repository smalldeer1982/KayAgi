# ANTTT - The Ant

## 题目描述

### 题目大意
在一个平面直角坐标系上有 $N$ 条线段，其中第 $i$ 条线段的两条端点分别在 $(x1_i,\ y1_i)$ 和 $(x2_i,\ y2_i)$ 上。\
现在有 $q$ 次询问，一次询问表示有一只蚂蚁在第 $x$ 条线段上，问蚂蚁能否爬到第 $y$ 条线段上。

## 输入格式

**本题包含多组数据。**\
第一行一个整数 $T$ ，表示有多少组数据。\
对于每一组数据，先输入两个整数 $N,\ q$ ，含义已在大意中表示。\
紧接着输入 $N$ 行，第 $i$ 行输入四个数 $x1_i ,\ y1_1 ,\ x2_i ,\ y2_i$ 。\
最后 $q$ 组询问，一组询问输入两个数 $x ,\ y$ 。

## 输出格式

对于一组数据中的一组询问，输出答案，能爬到第 $y$ 条线段上则输出 `YES`，反之输出 `NO`。

### 样例
（由于题目给的样例过于奇怪，不好调试，所以提供一下改正的）\
输入：
```
2

3 3
1 3 4 3
3 4 3 1

1 3

```

```

YES
YES
YES
NO
NO 
```

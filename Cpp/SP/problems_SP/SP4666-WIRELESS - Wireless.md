# WIRELESS - Wireless

## 题目描述

Bob 想在家里用电脑之余，增加一点社交互动，于是他决定带着电脑去咖啡馆。

他掌握了很多关于城市中无线网络和咖啡馆的信息。在他的城市，每个街道交叉口都有一家咖啡馆。具体来说，Bob 所住的城市有 $M$ 条东西向街道和 $N$ 条南北向街道。

在 $K$ 家咖啡馆中，有无线网络设备。每台设备的信号覆盖范围是一个圆，其半径是 $R$ 米，圆心位于这家咖啡馆。如果某人恰好在 $R$ 米范围内，便可以连接到无线网络，但如果超出这个范围，就无法连接。

假设每家咖啡馆最多安装一个无线网络设备，由于其他网络设备靠得近，在同一个咖啡馆可能会连接到多个无线网络。

Bob 的电脑具备一个特殊功能，能够充分利用所有可连接的无线网络的比特率。

他想知道，自己可以获得的最高比特率是多少，还有拥有这种最高比特率的咖啡馆有多少家。

## 输入格式

第一行输入一个整数 $M$，代表东西向街道数量。第二行输入一个整数 $N$，代表南北向街道数量。第三行输入一个整数 $K$，代表有无线网络的咖啡馆数量。接下来的 $K$ 行，每行包含 4 个整数。第一个整数 $x$ 指出咖啡馆所在的南北向街道编号，其中 $1 \leq x \leq N$。第二个整数 $y$ 表示咖啡馆所在的东西向街道编号，其中 $1 \leq y \leq M$。第三个整数 $R$ 表示无线网络的覆盖半径。第四个整数 $B$ 表示无线网络的比特率。

## 输出格式

输出结果共两行。第一行输出一个整数，代表在所有咖啡馆中可以得到的最大比特率。第二行输出一个整数，代表能够提供这种最大比特率的咖啡馆数量。

## 说明/提示

- $1 \leq M \leq 100,000$
- $1 \leq N \leq 100,000$
- $1 \leq K \leq 100,000$
- $1 \leq R \leq 20,000$

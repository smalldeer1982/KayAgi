# TFOSS - Fossil in the Ice

## 题目描述

一群考古学家正在南极进行研究。他们的传感器探测到若干个洞穴，洞中有引人注目的化石。然而，每个洞穴的入口都被厚厚的冰层封锁。虽然考古学家拥有可以在冰层中烧出隧道的设备，但所需的燃料非常昂贵。为了精确测量每块化石的大小，团队通过小钻孔发射了一些探测器。每个成功命中化石的探测器会发出一个信号，包含其所在的 x 和 y 坐标。你的任务是计算出隧道所需的最小尺寸，该尺寸应等于任意两个探测器之间的最大距离，以确保在提取化石时不会损坏它。钻探设备需要知道这个最大距离的平方值。

给定一组探测器坐标点，找到任意两个探测器之间的最大距离的平方。

## 输入格式

```
t [测试用例的数量，最多为20]
[空行]
n [活跃探测器的数量，最多为100000]
x1 y1 [第一个探测器的坐标]
...
xn yn [第 n 个探测器的坐标]
[坐标为整数，范围从 -50000000 到 50000000]
[空行]
[下一个测试用例的输入...]
```

## 输出格式

```
o1 [第一组测试用例中的最大距离平方值]
[下一组测试用例的输出...]
```

## 说明/提示

- 测试用例数 $t$ 最多为 20。
- 活跃探测器数 $n$ 最多为 100000。
- 坐标范围是从 -50000000 到 50000000 的整数。

# FACT0 - Integer Factorization (15 digits)

## 题目描述

给出一个数n，分解质因数。

## 输入格式

一个正整数n（会有若干行(大约10行)，并以0结束）（n<10^15）

## 输出格式

每输入一行，会以x^y形式输出(x为质数，y为x的乘方，之间用空格分开)

## 样例 #1

### 输入

```
3111989
13091989
77145199750673
0
```

### 输出

```
317^1 9817^1
17^2 89^1 509^1
328439^1 234884407^1
```

# MONONUM - Monotonous numbers

## 题目描述

有些整数有一个有趣的特性：其每位数字都不大于右边的数字。我们把这样的整数称为递增整数。相反地，如果一个整数的每位数字都不小于右边的数字，我们称之为递减整数。例如，24558 是一个递增整数，而 888410 是一个递减整数。数字 5 则是既递增又递减的数字。现在，给定一个整数 $n$，计算所有 $n$ 位递减整数与 $n$ 位递增整数的比例。注意，我们只考虑正整数，且不允许有前导零。

## 输入格式

第一行输入一个整数 $t$，代表测试用例的数量。在接下来的 $t$ 行中，每行包含一个整数 $n$。

## 输出格式

对于每个测试用例，输出计算得到的比例，结果保留小数点后六位。

## 说明/提示

$$1 \le t \le 100$$
$$1 \le n \le 100$$

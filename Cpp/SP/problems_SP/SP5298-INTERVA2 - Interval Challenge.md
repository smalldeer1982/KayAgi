# INTERVA2 - Interval Challenge

## 题目描述

给定 $N$ 个区间，区间 $m$ 用 $[A_{m}, B_{m}]$ 来表示。

现给出区间覆盖的定义为：对于两个区间 $S$ 和 $T$，如果 $A_{T} \le A_{S}$ 和 $B_{S} \le B_{T} $，我们说 $S$ 被 $T$ 覆盖。

求对于每个区间，有多少个区间可以覆盖它而不被它覆盖？

## 输入格式

多组测试。
每组测试第一行为 $N$ ($ 1 \le N \le 2e5$)，下面 $N$ 行表示 $A_{i}$ 和 $B_{i}$

## 输出格式

对于每组测试，输出一行，其中包含 $N$ 个整数，其中第 $i$ 个整数指定可以覆盖但不被它覆盖的区间数。

## 样例 #1

### 输入

```
30 1-1 2-2 3&nbsp;20 10 1
```

### 输出

```
\n2 1 00 0
```

# LPRIME - Primes of Lambda

## 题目描述

Lambda 用一种很特别的方法来判断一个数是不是素数。他的判断标准包括两个条件：

- 所有数字在十进制形式下的每一位都必须是素数或数字 1，即只能是 1、2、3、5 或 7。
- 这个数不能是 2、3、5、7、11 或 47 的倍数（为什么是 47？我也不知道）。

为评估他的方法准确性，他希望你计算符合这些条件的、长度为指定值的十进制整数的数量。

## 输入格式

输入由一行组成，包含一个整数 $n$，代表整数的长度。

## 输出格式

输出一行，包含在计算结果后对 9973 取模的结果。

## 数据范围

1 ≤ n ≤ 10^5

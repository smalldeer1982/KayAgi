# KSEQ - 123 Sequence

## 题目描述

123序列是长度 $\ge$ 2的非递减序列，其中每个数字是``1``、``2``或``3``。因为123序列只包含``1``，``2``，``3``，所以任何一对之间的差只能是``0``，``1``或``2``。给定 $X$ 、 $Y$ 、 $Z$ ，分别表示123序列差中``0``、``1``、``2``的个数，找出 $X$ 、 $Y$ 、 $Z$ 可能组成的不同的123序列的个数。

如果 $A_i$ 不等于 $B_i$ ，那么两个123序列A和B被认为是不同的。

## 输入格式

第一行一个整数 $T$ （$T \le 10000$），接下来 $T$ 行，每行三个整数 $X$ ， $Y$ ， $Z$（$0 \le X,Y,Z \le10^8$，$X + Y + Z > 0$）。

## 输出格式

输出共 $T$ 行，每行一个整数，表示不同的123个序列的数量。

## 样例 #1

### 输入

```
\n3\n0 2 1\n1 2 3\n1 3 2\n\n
```

### 输出

```
\n1\n0\n2\n
```

# QCJ2 - Another Box Problem

## 题目描述

在桌子上摆放着 $N$ 个编号为 $B_1, B_2, \ldots, B_N$ 的盒子。请写一个程序，计算出将 $N$ 个相同的球放入这些盒子的所有可能方案数，要求在任意前 $k$ 个盒子中（即 $B_1, B_2, \ldots, B_k$）放置的球最多不超过 $k$ 个，其中 $1 \le k \le N$。由于结果可能会非常大，最后需要将答案对 $761238923$ 取模。

## 输入格式

输入包含多组测试用例。每行给出一个整数 $N$，满足 $1 \le N \le 100$。输入的最后一行是数字 0，表示输入结束，不用对这行进行处理。

## 输出格式

对于每组测试用例，输出一个整数，表示所有可能方案数对 $761238923$ 取模后的结果。

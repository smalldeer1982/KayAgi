# QCJ4 - Minimum Diameter Circle

## 题目描述

在平面上有 $n$ 个点，要求找出一个可以包含所有点的最小圆的直径。需要注意的是，圆上的点同样被视为在圆内。

## 输入格式

输入的第一行是一个整数 $n$，表示平面上点的数量（$n < 301$）。接下来的 $n$ 行，每行包含两个整数，分别表示一个点的坐标，这些整数位于 $[0, 1000]$ 的范围内。

## 输出格式

输出一个实数，表示所求圆的直径，结果需要保留到小数点后两位。

### 输入

```
\n4\n1 1\n1 0\n0 1\n0 0\n
\n\n
```

```
\n1.41
```

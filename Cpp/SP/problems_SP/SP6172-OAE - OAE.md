# OAE - OAE

## 题目描述

给定一个 $n$ ，请输出所有满足以下条件的数的数量：

1. 每位均由 $0$ ~ $9$ 构成。（如 $000200$ 是合法的）
2. 数中含有偶数个 $0$。

将答案对 $314159$ 取模。

## 输入格式

有多组数据。

先给定 $t$ ，表示输入数据的组数。

下面 $t$ 行，每行一个数 $n$ ，如题意所示。

## 输出格式

$t$ 行，每行输出一个数，如题意所示。

## 样例 #1

### 输入

```
2
2
1
```

### 输出

```
82
9
```

# YODANESS - Yodaness Level

## 题目描述

尤达是他那个时代最具智慧、或许也是最强大的绝地武士。作为一个神秘人物，他有许多奇特的习惯，其中之一就是喜欢改变句子中单词的顺序。例如，他常说的一句话是「Or I will help you not.」。我们称一个句子的「尤达度」为相对于标准语序，有多少对单词的位置被交换过。编写一个程序来计算尤达所说的不同句子的「尤达度」。

## 输入格式

第一行输入一个整数 $t$，表示测试样例的数量。接下来有 $t$ 个测试样例。每个测试样例包含三行：

- 第一行是一个整数 $n$，表示句子中的单词数目。
- 第二行是尤达的说法，包含 $n$ 个用空格分隔的单词。
- 第三行是标准说法，同样包含 $n$ 个用空格分隔的单词。

所有词都是不同的，并由小写字母组成。

## 输出格式

对于每个测试样例，输出该句子的「尤达度」值。

## 数据范围
- $1 \le t \le 100$
- $1 \le n \le 1000$

这样确保了程序能处理足够复杂的句子组合。

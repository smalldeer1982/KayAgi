# EXPER - Experiment

## 题目描述

在一个实验中，平面上有 $n$ 个点在运动。每个点都以固定的速度移动。我们的任务是找出在每秒结束时，能够包含所有这些点的最小圆的半径，实验总共持续 $T$ 秒。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的数量。接下来是 $t$ 组测试用例的描述。对于每个测试用例的第一行，包含两个整数 $n$ 和 $T$，分别代表点的数量和实验持续的时间（即秒数）。接下来的 $n$ 行，每行有四个整数，分别是 $x, y, vx, vy$，表示每个点的初始坐标和速度。

## 输出格式

对于每个测试用例，输出 $T$ 个实数 $R_i$（$i$ 从 $1$ 到 $T$），表示每一秒钟结束时包含所有点的最小圆的半径。保留两位小数。

## 说明/提示

- $1 \leq t \leq 10$
- $1 \leq n \leq 100$
- $1 \leq T \leq 100$
- $-1000 \leq x, y, vx, vy \leq 1000$ 

这些条件确保输入数据量适中，适合用于实时计算最小圆半径。

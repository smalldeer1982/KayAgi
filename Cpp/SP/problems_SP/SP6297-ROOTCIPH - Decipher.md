# ROOTCIPH - Decipher

## 题目描述

雷达在边境发现了正在接近的敌机。然而，由于某些传输错误，部分数据无法解读。我们只能获取到一个三次多项式的系数，而这个多项式的根就是敌机相对于雷达的坐标。现在，你需要计算到敌机的距离。请编写一个程序来解决这个问题。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量（$1 \le T \le 1000$）。接下来的 $T$ 行中，每行包含三个整数 $a$、$b$ 和 $c$，它们是多项式 $x^3 + ax^2 + bx + c$ 的系数。这个多项式的根即是敌机的坐标，其绝对值不超过 $10^8$。

## 输出格式

对于每个测试用例，输出与敌机距离的平方。

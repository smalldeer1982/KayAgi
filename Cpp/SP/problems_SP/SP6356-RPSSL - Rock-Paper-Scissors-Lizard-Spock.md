# RPSSL - Rock-Paper-Scissors-Lizard-Spock

## 题目描述

Daniel 非常喜欢看电视剧，其中他最喜欢的是《生活大爆炸》。这部剧的主要角色包括：天才理论物理学家 Sheldon、擅长实验物理的 Leonard、他们迷人的金发邻居 Penny，以及他们的朋友——航空航天工程师 Howard 和粒子天体物理学家 Rajesh。

有一天，Rajesh 对 Sheldon 说：“要不我们来玩个石头剪刀布游戏？”  
Sheldon 回应道：“哦，我觉得没什么意思。根据经验，在石头剪刀布游戏中，彼此熟悉的玩家有 75% 到 80% 的概率会打成平手，因为结果有限。我建议我们玩石头剪刀布蜥蜴史波克。”  
“什么？”  
“很简单。剪刀剪纸，纸包石头，石头压蜥蜴，蜥蜴毒翻史波克，史波克砸剪刀。剪刀斩蜥蜴，蜥蜴吃纸，纸驳倒史波克，史波克蒸发石头，而石头依旧压剪刀。”  
“好吧，我觉得我懂了。”  
于是，他们决定玩石头剪刀布蜥蜴史波克。我们想知道在这个游戏中，Rajesh 打败 Sheldon 的概率是多少。我们已知每位玩家选择各种物体的概率，比赛规则是先赢得两局的人获胜。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的数量。接下来是 $t$ 个测试用例的具体描述。每个测试用例有两行。第一行包含五个整数 $R_{R}$, $R_{Sc}$, $R_{P}$, $R_{L}$, $R_{Sp}$ —— 这些数字分别代表 Rajesh 选择石头、剪刀、纸、蜥蜴和史波克的概率。第二行同样包含五个整数 $S_{R}$, $S_{Sc}$, $S_{P}$, $S_{L}$, $S_{Sp}$ —— 分别表示 Sheldon 选择石头、剪刀、纸、蜥蜴和史波克的概率。（注意：输入中物体的顺序是石头、剪刀、纸、蜥蜴、史波克。原题目在俄语版本中剪刀排在纸之前，但游戏规则不变。）

## 输出格式

对于每个测试用例，输出 Rajesh 打败 Sheldon 的概率，以百分比形式显示，并四舍五入到最接近的整数。

- $0 \le S_{R}, S_{Sc}, S_{P}, S_{L}, S_{Sp} \le 100$
- $R_{R} + R_{Sc} + R_{P} + R_{L} + R_{Sp} = 100$

# PERMUT1 - Permutations

## 题目描述

在一个有n个元素（1~n）的序列中，**查找这个序列的全排列中一共有多少个排列含有k个逆序对。**
比如当n=4，k=1时，一共有3个排列含有1个逆序对
（友情提示：1243、1324、2134）

## 输入格式

第一行输入一个d（1<=d<=10),表示一共有d组数据。
对于每组数据，会输入整数n(1<=n<=12) 和整数k(0<=k<=98)

## 输出格式

每行输出每组数据的答案。

## 样例 #1

### 输入

```
1 
4 1
```

### 输出

```
3
```

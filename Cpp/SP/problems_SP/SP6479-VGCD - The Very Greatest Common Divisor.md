# VGCD - The Very Greatest Common Divisor

## 题目描述

给定两个整数 $a$ 和 $b$，求它们的最大公约数。

## 输入格式

第一行包含一个整数 $n$，表示测试用例的数量，其中 $0 < n < 1000$。接下来的部分为 $n$ 组测试用例，每组由两行组成。第一行是整数 $a$（满足 $0 < a < 10^{12540}$），第二行是整数 $b$（满足 $0 < b < 10^{12540}$）。已知输入文件中的所有数都是一个特定形式的方阵行列式的结果。

## 输出格式

对于每个测试用例，输出对应的两个整数 $a$ 和 $b$ 的最大公约数，每个结果占一行。

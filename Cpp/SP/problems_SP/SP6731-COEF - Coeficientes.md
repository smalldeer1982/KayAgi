# COEF - Coeficientes

## 题目描述

你的任务是计算多项式 $(x_1 + x_2 + \cdots + x_k)^n$ 展开式中的系数。

## 输入格式

输入由多组数据组成，每组包含两行。第一行有两个正整数 $n$ 和 $k$，它们用空格隔开，分别表示多项式的幂次和变量的个数。第二行包含 $k$ 个非负整数 $n_1, n_2, \ldots, n_k$，并且满足 $n_1 + n_2 + \cdots + n_k = n$。

## 输出格式

对于每组输入，输出一行，表示多项式 $(x_1 + x_2 + \cdots + x_k)^n$ 展开后，对应单项式 $x_1^{n_1} x_2^{n_2} \cdots x_k^{n_k}$ 的系数。

## 说明/提示

- 输入中的所有整数均满足 $0 < n$ 且 $0 < k$。
- 确保每组数据中的 $n_1 + n_2 + \cdots + n_k = n$。

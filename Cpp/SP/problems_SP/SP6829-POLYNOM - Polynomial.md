# POLYNOM - Polynomial

## 题目描述

世界杯的观众数量年年攀升。为了在未来几年的比赛中销售广告位，你需要计算出一家公司为获得广告位需要支付的费用。而这就需要你根据过去比赛的观众数量来预测未来的观众人数。

你觉得这些观众数量可能可以被一个三阶以下的多项式模型精确描述。你的任务是验证这个想法是否成立。

## 输入格式

输入以一个正整数 $N$ 开始，表示测试用例的数量。接着每个测试用例占一行，首先是一个整数 $n$，后面跟着 $n$ 个整数 $x_1, \ldots, x_n$，这些整数代表过去比赛每场的观众人数，并且满足 $0 \leq i < n$。

## 输出格式

对于每个测试用例，若存在一个实数系数的三阶以内多项式 $p$，满足对所有 $i$，$p(i) = x_i$，则输出 **YES**。否则，输出 **NO**。

## 说明/提示

- $1 \leq N \leq 100$
- $1 \leq n \leq 100$
- $0 \leq x_i \leq 10^9$

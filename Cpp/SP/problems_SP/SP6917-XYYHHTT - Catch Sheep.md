# XYYHHTT - Catch Sheep

## 题目描述

### 题意简述：
给定一张有 $N$ 个节点的图。

现在有 $K$ 个机器人。他们将从同一点开始行走，使每条边至少走过一次。所有的机器人都可以到达任意节点。问机器人行走的总距离最小是多少？

## 输入格式

第一行两个整数 $N,K$。

接下来 $N-1$ 行：$a,b,c$（$a$ 与 $b$ 相连，边权为 $c$。

## 输出格式

$N$ 行：如果他们都从城市 $i$ 出发，机器人行走的总距离。

### 输入样例：
```
5 3
1 2 7
2 3 5
3 4 14
3 5 8
```
### 输出样例：
```
42
39
34
42
42
```

## 说明/提示

$N \leq 15000，k \leq 30$

$1 \leq a, b \leq N, 0 \leq c \leq 100$

## 样例 #1

### 输入

```
5 3
1 2 7
2 3 5
3 4 14
3 5 8
```

### 输出

```
42
```

# NECKDEC - Necklace Decomposition

## 题目描述

将一个字符串在环上顺时针排列，起始位置从第一个字符开始。

## 输入格式

输入的第一行是一个正整数 $n$，表示后续有 $n$ 组测试用例。每组测试用例只包含一行，由一个长度不超过 100 的二进制字符串组成，该字符串仅由 0 和 1 构成。

## 输出格式

对于每组测试用例，输出一行，展示字符串的项链分解形式。要求项链以 `(项链)` 的格式输出。

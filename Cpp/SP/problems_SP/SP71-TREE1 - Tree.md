# TREE1 - Tree

## 题目描述

如果考虑将一个排列依次插入到一颗二叉搜索树中，将会唯一地得到一棵二叉搜索树。我们称这个排列对应这个树。

现在给定一颗二叉搜索树，询问有多少个排列对应这个树。

## 输入格式

第一行一个整数 $d$，表示数据组数。

对于每组数据，第一行一个整数 $n$ 表示节点个数。第二行 $n$ 个整数，表示这颗树的先序遍历。

## 输出格式

对于每组数据，输出一行一个整数，表示答案。

## 说明/提示

$d\leq10, n\leq30$。

## 样例 #1

### 输入

```
5 
3 
2 1 3 
3 
1 2 3 
1 
1 
4 
2 1 3 4 
4 
1 4 2 3
```

### 输出

```
2 
1 
1 
3 
1
```

# DINOSM - Dinosaur Menace

## 题目描述

某个 DNA 项目失败后，实验室里突然出现了许多恐龙，它们吞噬了大部分的工作人员。Jeff 是参与这个项目的科学家，他躲在实验室的西南角才幸免于难。现在恐龙们都睡着了，他准备想办法逃出实验室。实验室的出口在东北角。

Jeff 很清楚，如果有任何一只恐龙醒来，他绝无生还的机会。因此，他需要选择一条路径，使得他在这条路径上与任何恐龙的最小距离尽可能大。路径的长度对于 Jeff 来说无关紧要。

在这个问题中，我们把 Jeff 和恐龙都视为平面上的一个点，而 Jeff 的路径是连接实验室西南角和东北角的一条连续曲线。Jeff 希望这条曲线与任何一个恐龙所在位置之间的最小距离最大化。

## 输入格式

输入由多个测试用例组成，每个测试用例包含多行。第一个测试用例的第一行包含三个整数 $N$、$W$ 和 $H$，使用空格分隔。$N$ 表示实验室中恐龙的数量（$1 \le N \le 10^3$），$W$ 和 $H$ 表示实验室在 $x$ 和 $y$ 坐标上的尺寸（$2 \le W, H \le 10^4$）。Jeff 的起始位置是 $(0, 0)$，实验室的出口是 $(W, H)$。接下来的 $N$ 行，每行有两个整数 $X$ 和 $Y$，表示一只恐龙的位置（$1 \le X \le W-1$，$1 \le Y \le H-1$）。所有恐龙都不在边界上，并且没有两个恐龙处于相同位置。输入的最后一行有三个 $-1$，表示输入结束，这行无需处理。

## 输出格式

对于每个测试用例，输出一行，表示到最近恐龙的最大可能距离。结果应取到小数点后三位，若有并列情况，选择较大者。

# GARBAGE - Garbage Collection

## 题目描述

一个大办公室使用清洁机器人来清空每个工作间的垃圾桶。每个工作间是一个正方形，办公室的布局被组织成一个 $R$ 行 $C$ 列的矩形矩阵。清洁过程从机器人通过入口进入左上角的工作间开始，并通过同一入口离开结束。进和出分别需要 26 秒。当机器人在一个工作间内时，它可以用 13 秒来清空垃圾桶。机器人还可以移动到与其当前工作间相邻的工作间，这种移动需要 38 秒。机器人必须至少进入每个工作间一次以清空垃圾桶。

要完成整个清洁过程所需要的时间取决于机器人实际行走的路径。在所有可能的路径中，我们需要找出耗时最小的那条。

根据给定的办公室布局，你需要计算出完成整个清洁过程所需的最短时间，其中包括进出、清空每个工作间的垃圾桶以及移动的时间。需要注意的是，最优路径可能会多次经过某个工作间，但机器人只需在每个工作间清空垃圾桶一次。

## 输入格式

输入包含多个测试用例，每个测试用例由一行构成。该行包含两个整数 $R$ 和 $C$，由空格分隔，分别表示行数和每行的工作间数量（$1 \le R, C \le 100$）。当出现一行包含两个为 -1 的整数时，表示输入结束，该行不应被处理。

## 输出格式

对于每个测试用例，输出一行，表示机器人完成清洁过程所需的最短秒数。

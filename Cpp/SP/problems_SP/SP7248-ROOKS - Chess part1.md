# ROOKS - Chess part1

## 题目描述

在一个 $N \times N$ 的棋盘上放置两个车。每个格子中都包含一个非负整数。需要注意的是，这两个车必须放置在不同的格子上。

我们认为，当某个车所在行或列上的格子被其控制时，这些格子就被攻击了。然而，车所在的格子本身不算被攻击。

我们的目标是放置这两个车，使得被攻击的格子上的数值之和达到最大。请编写程序来寻找这个最大值。

## 输入格式

第一行为整数 $N$，其中 $2 \le N \le 300$。

接下来的 $N$ 行中，每行包含 $N$ 个整数，表示棋盘上的数值。每个数值都大于等于 $0$ 且小于 $1000$。

## 输出格式

输出单行，包含一个整数，表示题目要求的最大和。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
3
0 1 4
3 0 2
1 4 1
```

### 输出

```
15
```

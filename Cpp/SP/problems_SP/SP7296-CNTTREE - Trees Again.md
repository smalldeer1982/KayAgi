# CNTTREE - Trees Again

## 题目描述

给出一棵树，你需要计算：直径为$D$，且$D<=K$的子树的数量。

## 输入格式

第一行包含一个整数 $t$，表示接下来有 $t$ 组测试数据。

对于每组测试数据：

第一行包含两个变量，$N,K$.

接下来的$N-1$行中，每行包含两个变量$a_i,b_i$，表示 $a_i$ 与 $b_i$ 有一条边相连。

每组数据后有一个空白行。

## 输出格式

共输出$T$。其中第 $i$ 行为第 $i$ 组测试数据的答案。

## 样例 #1

### 输入

```
2
3 1
0 1
1 2

6 3
0 1
1 2
2 3
2 4
3 5
```

### 输出

```
5
23
```

# RANJAN02 - Tower Of Hanoi - Revisited

## 题目描述

有三个柱子：左边柱子 A、中间柱子 B 和右边柱子 C。需要找到一组最短的移动操作，将 n 个盘子从左柱 A 移动到右柱 C，并且要求直接从 A 移动到 C 是不允许的（每次移动都必须经过中间柱 B）。

限制条件：
1. 开始时，左柱 A 上按由大到小的顺序叠放着 n 个盘子。
2. 每次只能移动一个盘子，且遵循不能将大盘子放在小盘子上的规则。
3. 移动次数总是小于 $2^{64}$。
4. $1 \leq n \leq 35$。

## 输入格式

输入的第一行为一个整数 $t$，表示测试用例的数量。接下来的 $t$ 行中，每行包含一个整数 $n$，表示需要移动的盘子数量。

## 输出格式

对于每个测试用例，输出把 n 个盘子从柱子 A 转移到柱子 C 的最小移动次数。

### 输入

```
\n4\n1\n2\n5\n10\n\n
```

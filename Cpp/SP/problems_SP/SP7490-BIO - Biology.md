# BIO - Biology

## 题目描述

早在 1869 年，儒勒·凡尔纳通过他的科幻小说《海底两万里》激起了人们对海洋深处的浓厚兴趣。驾驶鹦鹉螺号的尼莫船长带领船员们拜访了失落的城市亚特兰蒂斯，并见识到了各种奇异的海洋生物。

近一个世纪后，皮卡尔和沃尔什造访了地球上最深的挑战者深渊，并在海面以下 10,924 米处插上了一面瑞士国旗。他们搭乘的特里斯特潜艇在这些深度中看到了令人惊叹的动物生命。

![](https://cdn.luogu.com.cn/upload/vjudge_pic/SP7490/3c76113ce4c4ded39c98cfc461c3db129dd98fcb.png)

最近，一队生物学家决定研究马里亚纳海沟的深处，尤其是那些「非常奇特稀有的居民」（SWERC）。他们进行了一项初步研究，发现这些物种的生境非常局限，投影到海面上则呈现为一系列凸多边形。所有生境位于相同深度，有些甚至可能重叠。生物学家计划在每个生境安装架子和摄像头，以吸引并拍摄这些生物。尽管架子上的食物诱人，但没有物种会跨越其生境的边界。因为这些摄像设备及其配套的通信系统成本极高，所以希望尽可能减少使用数量。你能帮助生物学家规划摄像头的位置，告诉他们最少需要多少个摄像头吗？注意，每个摄像头-架子组合必须位于生境内部，才能有效吸引相关物种。

## 输入格式

输入由多个测试用例组成，每个测试用例之间用空行分隔。每个测试用例由整数 $S (0 \le S \le 20)$ 开始，代表物种数量。接下来的 $S$ 行描述每个生境，其格式为一个整数 $n_i$ 表示凸多边形的顶点数，后面是这些顶点的坐标 $x_1 y_1 x_2 y_2 \ldots x_{n_i} y_{n_i}$（坐标范围为 $|x_i|, |y_i| \le 1000$）。当读入 $S = 0$ 时，输入结束，该测试用例不需处理。

## 输出格式

对于每个测试用例，输出监控所有物种所需的最少摄像头数量。

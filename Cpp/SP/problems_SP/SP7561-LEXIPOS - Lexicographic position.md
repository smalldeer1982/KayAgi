# LEXIPOS - Lexicographic position

## 题目描述

考虑一个从 1 到 $N$ 的整数集合。将这些整数按照字典序排列（就像在词典中的排列方式一样）。例如，当 $N = 11$ 时，这些数字的字典序排列为：1, 10, 11, 2, 3, 4, 5, 6, 7, 8, 9。

定义数字 $K$ 在这种排列中的位置为 $Q_{N,K}$。例如，$Q_{11,2} = 4$。

给定 $N$ 和 $K$，请计算 $Q_{N,K}$。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。接下来的 $T$ 行中，每行包含两个整数 $N$ 和 $K$，它们之间用一个空格分隔。

## 输出格式

对于每个测试用例，输出 $Q_{N,K}$ 的值，每个结果占一行。

## 说明/提示

1. $1 \leq T \leq 100$
2. $1 \leq K \leq N \leq 10^5$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
\n7\n1 1\n11 2\n215 211\n215 215\n215 26\n215 99\n1000000000 999999999\n\n
```

### 输出

```
\n1\n4\n126\n130\n135\n215\n1000000000
```

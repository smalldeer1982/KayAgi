# BOI7FEN - Fence

## 题目描述

Leopold 真是个幸运儿，他在彩票中赢得了一座豪华庄园。除了主宅外，庄园里还有几座宏伟的建筑。Leopold 打算以后都住在主宅里，但令他担忧的是，庄园目前没有围栏来保护他免受入侵者的打扰。经过考虑，他决定为主宅建一个围栏。然而由于经费有限，他只能围住主宅，而不可能围住整个庄园。同时，有一个重要的要求：围栏不能靠近其他建筑。

具体来说，从上往下看，每个建筑都被一个禁止区域包围，呈矩形状。在这些矩形范围内，围栏不得通过。所有的矩形边都与 x 轴或 y 轴平行。同时，围栏的每一段也必须与 x 轴或 y 轴平行。

![图 5.1：主宅（黑色）和其他三座建筑及其周围的禁止矩形，粗黑线表示围住主宅的最短允许围栏。](http://vn.spoj.com/content/boi7fen.jpg "Fence")

## 输入格式

第一行是一个正整数 $m$（表示建筑物的数量，其中 $m$ 满足 $1 \le m \le 1000$）。接下来的 $m$ 行，每行四个整数 $x_1, y_1, x_2, y_2$，表示每个建筑物禁止矩形的左下角和右上角的坐标，且 $-10^9 \le x_1 < x_2 \le 10^9, -10^9 \le y_1 < y_2 \le 10^9$。其中，第一个建筑物是主宅。

## 输出格式

输出一行，表示围住主宅的最短允许围栏的长度，结果为一个正整数。

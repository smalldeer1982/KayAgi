# HS10SQFT - Almost square factorisation

## 题目描述

给定一个整数 $n$，要求找出所有形式为 $n = (a^2 - 1) \times (b^2 - 1)$ 的因式分解，其中 $1 < a \leq b$。

## 输入格式

第一行输入一个整数 $T$，表示测试用例数量，其中 $T \leq 1000$。接下来的 $T$ 行中，每行包含一个整数 $0 < n < 2^{62}$。

## 输出格式

对于每个测试用例，首先输出测试用例的编号，然后换行按 $a$ 的值递增顺序输出所有因式分解。如果找不到符合要求的因式分解，则输出错误信息。请参考样例输入输出了解具体格式。

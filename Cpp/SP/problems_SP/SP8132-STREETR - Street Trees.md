# STREETR - Street Trees

## 题目描述

有一排树种在一条直线上，KOI 计划再种植一些树，使得所有相邻树木之间的距离相等。为了简化问题，每棵树只能种植在整数坐标上。

比如，假如最初 4 棵树分别种在坐标 (1, 3, 7, 13) 上，如果 KOI 在坐标 (5, 9, 11) 处再种 3 棵树，那么所有相邻树木之间的距离就可以变得相等。

你的任务是计算 KOI 最少需要再种几棵树，才能使所有相邻树木的距离相等。

## 输入格式

第一行输入一个整数 $N$，表示已经种植的树木数量，且 $3 \le N \le 100,000$。

接下来的 $N$ 行中，每行输入一个整数 $X$，表示每棵树的坐标，且 $1 \le X \le 1,000,000,000$。

你可以假设这些坐标值都是唯一的。

## 输出格式

输出 KOI 至少需要再种植的树木数量。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
4
1
3
7
13
```

### 输出

```
3
```

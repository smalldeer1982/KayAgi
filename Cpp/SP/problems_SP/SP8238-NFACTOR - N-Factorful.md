# NFACTOR - N-Factorful

## 题目描述

如果一个数正好具有 **n** 种不同的质因数，我们称该数为 **n**-因子数。现在给你三个正整数 **a**、**b** 和 **n**，请你找出在区间 \[**a**, **b**\] 内的所有整数中，有多少个是 **n**-因子数。注意，这里规定 1 是 0-因子数。

## 输入格式

输入第一行为整数 **T**，表示测试用例的数量。接下来有 **T** 行，每行包含三个整数 **a**、**b** 和 **n**（如题意所述）。

$ T > 10000 $  
$ 1 \le a \le b \le 10^6 $  
$ 0 \le n \le 10 $

## 输出格式

对于每个测试用例，输出一行，表示区间 \[**a**, **b**\] 中有多少个 **n**-因子数。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
5
1 3 1
1 10 2
1 10 3
1 100 3
1 1000 0
```

### 输出

```
2
2
0
8
1
```

# PLAYSIGN - color the balls

## 题目描述

你和 Dope 正在拜访 DAKSH。突然，Dope 发现了一些白色的球，这些球都有一定的质量。让人感到有趣的是，这些球通过按下开关可以变成红色或绿色。Dope 心情不错，想要和你玩个游戏。他提出让你们轮流给球换颜色，只能选择还未改变颜色的白色球进行操作。在所有球变色完成后，你需要根据红色球和绿色球质量总和的绝对差额支付一笔费用。Dope 会尽可能增加你要支付的金额，而你自然希望支付最少的钱。Dope 邀请你先开始游戏。假如你和 Dope 都采用最佳策略，你需要支付多少费用给他？

## 输入格式

输入包含 $T$ 组测试用例。每组测试用例由两行组成。第一行为一个整数 $N$，表示白色球的数量。第二行包含三个整数 $a$、$b$ 和 $c$，用于生成 $N$ 个球的质量，生成公式为：  
$$\text{mass} = (a \cdot i + b) \mod c; \quad 1 \leq i \leq N$$

## 输出格式

对每组测试用例，输出一行，即你需要支付的金额。

## 说明/提示

$$1 \leq T < 1000$$  
$$1 \leq N < 10000$$  
$$1 \leq a, c < 1000$$  
$$0 \leq b < 1000$$

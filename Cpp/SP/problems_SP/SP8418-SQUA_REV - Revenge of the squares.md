# SQUA_REV - Revenge of the squares

## 题目描述

给定一个正整数，计算其中所有大于零的数字的乘积 $N$。然后，找出有多少种不同的方式将 $N$ 表示为 $A \times A + B \times B$，其中 $A$ 和 $B$ 是非负整数。最终输出这些不同表示方法的数量 $R$。

## 输入格式

一共二十组测试，每组输入一个不超过 $10^{20}$ 的正整数。

## 输出格式

对于每组测试，输出对应的数量 $R$。

### 输入

```
5\n7\n78185824586267361855
```

### 输出

```
1\n0\n3
```

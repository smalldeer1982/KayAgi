# MAIN75 - BST again

## 题目描述

求有多少棵大小为n的深度为h的二叉树。（树根深度为0，且左右子树有别）

## 输入格式

第一行一个整数T（1<=T<=10），表示数据组数。

以下T行，每行2个整数N和H（1<=N<=500,0<=H<=500）。

## 输出格式

共T行，对于每组数组每行输出一个整数表示答案(答案对1000000007取模)

感谢@白苏小公子喵 提供的翻译

## 样例 #1

### 输入

```
1
2 1
```

### 输出

```
2
```

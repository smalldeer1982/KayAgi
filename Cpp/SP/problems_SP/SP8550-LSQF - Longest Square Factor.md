# LSQF - Longest Square Factor

## 题目描述

给定一个字符串 $x$，如果把 $x$ 自身重复一次连接起来，就称为 $x$ 的平方。现在给你一个字符串 $s$，找出最长的字符串 $x$，使得 $x$ 的平方是 $s$ 的一个子串。如果有多个满足条件的 $x$，请输出字典序最小的那个。

## 输入格式

输入包含一行，由小写字母组成的字符串 $s$，其长度不超过 $10^5$。

## 输出格式

输出两行数据。第一行是字符串 $x$ 的长度，第二行则是字符串 $x$ 本身。

测试数据保证一定存在满足条件的字符串 $x$。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
abcabc\n\n
```

### 输出

```
3\nabc
```

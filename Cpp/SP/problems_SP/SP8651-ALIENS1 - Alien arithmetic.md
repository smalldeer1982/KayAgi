# ALIENS1 - Alien arithmetic

## 题目描述

在字节之地的小约翰尼，被来自仙女座星系的外星人绑架了！目前他被困在一艘停在他家后院的飞碟的小房间里。约翰尼急切地想打开房门，结果发现门需要解答一系列算术问题：两个数的加法、减法、乘法或者幂运算（外星人不懂除法）。他本以为这些问题很简单，能够轻松解答然后逃脱。然而，当他看到题目后，才发现外星人使用的是 42 进制！约翰尼开始慌了，这时他请求你的帮助。

## 输入格式

第一行包含一个整数 $T$（$T \leq 500$），表示测试用例的数量。接下来的 $T$ 行中，每行由 $A\ \text{op}\ B$ 构成，其中 $A$ 和 $B$ 是用 42 进制表示的数字，最多有 10000 位数字且没有不必要的零。$\text{op}$ 是 `+`、`-`、`*` 或 `^`，表示相应的算术操作。42 进制的表示方式如下：

- '0' 到 '9' 表示数字 0 到 9
- 'A' 到 'Z' 表示数字 10 到 35
- 'a' 到 'f' 表示数字 36 到 41

## 输出格式

对于每个测试用例，输出一行结果：计算后的结果需要在 42 进制下对 LIFE$_{42}$ 取模，并且用 42 进制表示，注意不要包含多余的零。

## 说明/提示

- 测试用例数量 $1 \le T \le 500$
- 每个 42 进制数最多含有 10000 位数字
- $\text{op}$ 为 `+`、`-`、`*` 或 `^` 中的一个符号。

# MAXLN - THE MAX LINES

## 题目描述

# 题目简述
在半径为r的半圆上找一个点A，使得这一个点到半圆的一个端点的距离平方+这一个点到半圆的另一个点的距离最短

## 输入格式

第一行T

第2~T+1行 每行一个r

## 输出格式

Case Number: ans

其中ans是两位小数

## 说明/提示

$1<=T<=10^3$

$1<=r<=10^6$

## 样例 #1

### 输入

```
1
1
```

### 输出

```
Case 1: 4.25
```

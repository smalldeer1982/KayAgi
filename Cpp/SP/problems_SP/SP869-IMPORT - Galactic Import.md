# IMPORT - Galactic Import

## 题目描述

借助新型的 ThrustoZoom 超维驱动器，位于新泽西的进出口集团 HyperCommodities 如今能够与宇宙中最遥远的星系进行贸易。HyperCommodities 希望从 Plural Z 区域的一些星系中进口商品。这些星系的行星出口各类有价值的产品和原材料，如真空密封材料、透明铝、二石墨和量子钢。初步报告揭示了以下几点：

- 每个星系至少包含一个，并且最多包含 26 个行星。每个行星以字母 A 到 Z 标识。
- 每个行星专注于一种商品的生产和出口。同一星系内的行星出口的商品各不相同。
- 一些行星之间通过超空间运输线连接。若行星 A 和 B 互连，则可直接进行商品贸易。如果行星 C 与 B 相连但不与 A 连接，则 A 和 C 仍能通过 B 进行贸易，但 B 将从中扣除 5% 的运费。（这样 A 只能收到 C 发来的 95% 的商品，C 也只能收到 A 发出的 95%。）通常情况下，只要两个行星通过某些线路相连，就可以进行贸易，但每经过一个中转行星，都会扣除 5% 的运输量（这并不一定是原始货物的 5%）。
- 每个星系中至少有一个行星愿意开通一条通往地球的 ThrustoZoom 运输线。就贸易而言，ThrustoZoom 运输线与星系内的其他运输线并无不同。例如，如果行星 K 开通了一条通向地球的 ThrustoZoom 线，那么地球可以自由地与 K 或 K 相连的任何行星进行贸易，但仍需承担通常的运费。

HyperCommodities 对每个行星的主要出口商品分配了一个相对价值（小于 10 的正实数）。数值越高，商品就越能在国内市场上以高利润率出售。问题在于确定，考虑到运输费用之后，哪个行星的出口商品最为有价值。输入包含一个或多个星系的描述。每个星系的描述从一行整数 $N$ 开始，表示星系中的行星数量。接下来的 $N$ 行描述每个行星的情况，具体格式为：

1. 行星的标识字母。
2. 一个空格。
3. 行星出口商品的相对价值，格式为 _d.dd_。
4. 一个空格。
5. 一个包含字母和/或字符 `\*` 的字符串；字母表示与该行星有运输线连接，`\*` 表示愿意开通一条通往地球的 ThrustoZoom 运输线。

对于每个星系的描述，输出一行，显示「Import from P」，其中 P 是在考虑运输费用后，出口商品价值最高的行星的标识字母。（若多个行星的最大出口价值相同，则输出字母顺序最靠前的行星）。下面是一个输入示例：

```

5

10

E 7.54 GC

G 7.43 IE

```

对应的输出示例应为：

```
Import from F
Import from A
Import from A
```

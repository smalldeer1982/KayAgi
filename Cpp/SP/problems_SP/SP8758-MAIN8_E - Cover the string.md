# MAIN8_E - Cover the string

## 题目描述

给定两个字符串 A 和 B。你的任务是从字符串 A 中找出一个最短的子串，其中 B 是这个子串的子序列。

## 输入格式

第一行是一个整数 T，表示测试用例的数量。接下来的 T 组数据中，每组包含两行，第一行是字符串 A，第二行是字符串 B。

## 输出格式

对于每一个测试用例，输出一个整数，表示所找到的最短子串的长度。如果不存在这样的子串，则输出 -1。

## 数据范围

字符串 A 的长度不超过 20000，字符串 B 的长度不超过 100。

 **本翻译由 AI 自动生成**

### 输入

```
2
aaddddcckhssdd
acs
ghkkhllhjkhthkjjht
hh
```

### 输出

```
10
3
```

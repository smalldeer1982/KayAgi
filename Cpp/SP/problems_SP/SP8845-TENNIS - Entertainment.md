# TENNIS - Entertainment

## 题目描述

Maryanna 和 Lucyanna 从十年前开始每个周日下午都会打一场网球比赛。

这场比赛由五盘组成，通常情况下，先赢得三盘的选手将获得整场比赛的胜利。每盘由多个局组成，而每个局又是由若干分数构成。

一个局是在同一名选手发球的一连串得分构成的。赢得一个局的标准是某选手至少赢得四分，同时比分还需比对手高出至少两分。在网球比赛中，分数的描述十分独特：从零到三的得分分别用「love」、「fifteen」、「thirty」和「forty」表示。如果双方都至少得了三分且分数相同，就称为「deuce」。如果双方得分均在三分及以上，且有一方仅领先一分，则称为该方的「advantage」。在非正式比赛中，这种情况下也可以称为「ad in」或「ad out」，这取决于领先方是发球方还是接球方。

一个盘子由一系列局构成，发球权在局与局之间交替。先赢得至少六局且比对手至少多赢两局的选手赢得这一盘。

另外，连续两盘比赛的第一发球者是不能相同的。

Maryanna 发现，如果她是发球方，她赢得某一分的概率是 **M**%；而如果是她的对手发球，这一分的赢得概率为 **Y**%。 Maryanna 想知道，如果她在比赛中先发球，她赢得整场比赛的概率是多少。

## 输入格式

第一行包含一个整数 **T**（**T** ≤ 10000），代表测试用例的数量。接下来的 **T** 行，每行包含两个用空格分隔的正整数 **M** 和 **Y**，其中 0 < **M**, **Y** < 100。

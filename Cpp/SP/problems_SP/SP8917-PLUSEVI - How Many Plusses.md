# PLUSEVI - How Many Plusses

## 题目描述

Mirko 是个不按常理出牌的男孩，他在一张布满 0 和 1 的方阵纸上写下了许多数字。如今他想研究，究竟这个矩阵里有多少个「加号」。

所谓「加号」，是指一个边长大于 1 的奇数正方形，其中正方形内部除了中间一行和中间一列，其他单元格全部为 0，而中间的那一行和那一列则全部是 1。例如，在以下矩阵中可以找到两个「加号」，其中一个嵌套在另一个里面：

```
00100  
00100  
11111  
00100  
00100
```

## 输入格式

第一行输入一个整数 $N$，表示矩阵的大小。

接下来的 $N$ 行，每行由 0 和 1 组成，代表矩阵的不同行。

## 输出格式

输出矩阵中存在的「加号」的总数。

## 说明/提示

假设 $1 \le N \le 1000$。

## 样例 #1

### 输入

```
8\n00010000\n00010000\n00010000\n11111111\n00010000\n00010010\n00010111\n00010010\n\n\n
```

### 输出

```
3
```

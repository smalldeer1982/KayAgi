# TOHU - Help Tohu

## 题目描述

Tohu 想要考入莫斯科国立大学，他需要通过一次数学考试。由于他对数学一窍不通，所以向你寻求帮助。你需要解决的问题是计算序列 $\{ a_n \}$ 的累加和 $S_n = a_1 + a_2 + \ldots + a_n$，该序列满足对于所有的正整数 $k$，都有 $a_1 + 2a_2 + \ldots + ka_k = \frac{k+1}{k+2}$。

## 输入格式

输入的第一行有一个整数 $T \leq 20000$，表示测试用例的数量。

接下来的 $T$ 行中，每行有一个整数 $1 \leq n \leq 10^9$。

## 输出格式

对于每个给定的 $n$，输出对应的 $S_n$ 值，并精确到小数点后 11 位。

# BISHOP2 - Chessboard Billiard

## 题目描述

### 题意：
选择四个对角线方向中的第一个，台球朝那个方向运动。当它到达棋盘边缘的正方形时，台球从中反射出来，它的运动方向改变了90度，并继续运动。在弹道的任何一个方块上，台球都可以停止运动，并且在这个方块上，台球的移动被认为是完成的。你找出台球的最大数目，成对不能击败对方，放在 $n$ × $m$ 大小的棋盘上。

## 输入格式

输入的第一行包含两个整数 $n$ 和 $m$

## 输出格式

为每个输入一行，其中包含一个数字，即不成对击球的台球的最大可能数目。

### 样例输入：
```
3 4
2 2
5 5
```
### 样例输出：

```
2
2
5
```

## 样例 #1

### 输入

```
\n3 4\n2 2\n5 5\n\n
```

### 输出

```
\n2\n2\n5
```

# GCPC11I - Suiting Weavers

## 题目描述

织巢鸟威利急切地希望能赢得最美丽动人的雌鸟威尔玛的芳心。不过，想要俘获威尔玛的可不仅仅是威利一个。

为了吸引雌鸟的注意，雄鸟们会用叶纤维编织精美的巢穴。明天就是大日子，威尔玛将来查看所有的巢穴。此刻，正值一场猛烈的暴风雨，日出前没有一只雄鸟能离开自己的巢穴。然而，暴风雨会带来大量的叶纤维堆积，所有雄鸟都有机会提升自己的巢穴。威利想知道，他是否能编织出最引人注目的巢穴，让威尔玛最终选择与他结婚。在这个过程中，巢穴的大小尤其重要，因此威利想要估算他的巢穴以及对手们的巢穴可能有多大。

为此，威利考虑了所有他所知道的，可提供适合构巢的叶纤维的地方。由于雄鸟不喜欢离开它们熟悉的领域，这些地方许多只能被一部分雄鸟接近，有的地方甚至无法被任何雄鸟到达。

为了简化问题，威利不打算制定具体的飞行计划。这意味着他不考虑对手的具体策略，也不假定它们一次能带回多少纤维或飞行的速度与时间。因此，有可能某只雄鸟在自己的领地内收集了所有的纤维。最终，威利假设所有的雄鸟都像他一样诚实：不会从对手的巢穴中偷走纤维。

问题来了：在所有叶纤维被收集之后，威利的巢穴有没有可能不被任何其他雄鸟的巢穴超越（相同大小也算成功）？

## 输入格式

第一行包括测试用例的数量 $T$ ($1 \le T \le 50$)。每个测试用例的第一行包含两个整数。第一个整数 $W$ ($1 \le W \le 50$) 是雄鸟的数量（包括威利自己）；第二个整数 $P$ ($1 \le P \le 100$) 是可以提供叶纤维的地点数量。

接下来有 $W$ 行，每行用四个整数 $x$, $y$, $f$, 和 $r$ ($-10^6 \le x, y \le 10^6$, $0 \le f \le 10^6$, $0 \le r \le 10^6$) 描述一个雄鸟的巢穴：$x$ 和 $y$ 表示巢穴的位置，$f$ 表示巢穴中纤维的数量，$r$ 表示巢穴主人在其领地内寻找更多纤维的半径。这 $W$ 行中的第一行描述了威利的巢穴。

接下来是 $P$ 行，每行定义了一个含有叶纤维的地方，包含三个整数 $x$, $y$, 和 $f$ ($-10^6 \le x, y \le 10^6$, $0 \le f \le 10^6$)：$x$ 和 $y$ 表示该地点的位置，$f$ 是该处可获得的叶纤维数量。

## 输出格式

对于每个测试用例，如果在所有纤维被采集后威利仍有机会赢得威尔玛，则输出 `Suiting Success`；否则输出 `Lonesome Willy`。

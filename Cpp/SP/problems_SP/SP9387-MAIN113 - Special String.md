# MAIN113 - Special String

## 题目描述

如果一个由 X、Y、Z 组成的字符串包含任意三个连续的字母且这三个字母分别是 X、Y 和 Z，那么这个字符串就被认为是特殊的。例如，字符串 XYZXYZ 是特殊的，而 XXYYZ 则不是。你的任务是计算长度为 $n$ 的字符串中不特殊的有多少个。

## 输入格式

第一行输入一个整数 $T$（$1 \le T \le 30$），表示测试用例的数量。接下来的每一行包含一个整数 $N$（$1 \le N \le 30$），表示每个测试用例中字符串的长度。

## 输出格式

对于每个测试用例，输出长度为 $N$ 的不特殊字符串的总数量。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
1
2
```

### 输出

```
9
```

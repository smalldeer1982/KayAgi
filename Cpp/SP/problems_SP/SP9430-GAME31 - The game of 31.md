# GAME31 - The game of 31

## 题目描述

说明：由于洛谷上此题残缺，所以使用SPOJ原题。

### 题目背景

31号游戏是以前乘坐火车的骗子艺术家最喜欢玩的游戏。


游戏是这样的：首先会有24张牌，这里有四张标记为1、2、3、4、5、6。这里面的牌双方全部可以看到，即A和B都知道。A和B轮流从一打牌中抽出一张牌，放在一堆牌上。游戏目标就是成为最后一个放牌的人。

这一堆的总量不超过31。你的任务是决定游戏的最终赢家。

例如，在下面的游戏中，玩家B获胜:
- 玩家A放3
- 玩家B放5
- 玩家A放6
- 玩家B放6
- 玩家A放5
- 玩家B放6

## 输入格式

输入将由几行组成；每一行由零个或多个数字组成

代表部分完成的游戏。第一个数字是玩家A的举动；第二个数字是b的举动；诸如此类。你要为双方玩家使用一个最好的策略来完成游戏，并来决定谁会赢。

## 输出格式

对于每个游戏，打印一行，包括输入，后跟一个空格；然后是A或B，表示游戏的最终赢家。

### 示例输入:
```
356656
35665
3566
111126666
552525
```
### 示例输出:
```
356656 B
35665 B
3566 A
111126666 A
552525 A
```

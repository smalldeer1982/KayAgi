# THREETW1 - Connecting three towns (variation)

## 题目描述

有三个给定的城镇，请找出一个点，使该点到所有城镇的距离之和达到最小。要求输出的点的坐标，精确到小数点后6位（±10$^{-6}$）。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量（$T < 100$）。接下来的 $T$ 行，每行提供三个城镇的坐标，使用空格分隔，坐标值为整数，满足 $0 \le x, y \le 100$。

## 输出格式

对于每一个测试用例，输出找到的满足条件的点的坐标。

### 输入

```
2\n2 2 1 1 2 1\n1 0 0 0 2 1
```

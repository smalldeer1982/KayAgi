# IM - Intergalactic Map【RMJ接口有误不支持提交】

## 题目描述

#### 题意简述

给定一个 $n$ 个点 $m$ 条边的无向图，询问是否存在一条不经过重复点的从 $1$ 号节点出发，途经 $2$ 号节点到达 $3$ 号节点的路径。

## 输入格式

本题包含多组测试数据，输入第一行一个数 $t$ 表示数据组数，每组数据第一行两个数 $n,m$ 表示点数和边数，接下来 $m$ 行每行两个数 $u,v$ 描述一条连接 $u$ 和 $v$ 的无向边。

## 输出格式

对于每组数据输出一行，若存在一条这样的路径，输出 `YES`，否则输出 `NO`。

注意：

1. 数据可能存在 $u>n$ 或者 $v>n$ 的情况，请忽略这种边。

2. $1 \leq m \leq 10^5$。

## 样例 #1

### 输入

```
\n

2\n
3 3\n
1 2\n
2 3\n
1 3\n
3 1\n
1 3\n


\n
```

### 输出

```
\n

YES\n
NO\n
```

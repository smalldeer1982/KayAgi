# CODESPTF - Palindromes

## 题目描述

给定一个字符串，你可以随机交换其中的任意两个字符，直到字符串变为回文。请问完成这个任务期望需要进行多少次交换？

## 输入格式

输入为一行，包含一个由小写字母组成的字符串 $s$。

## 输出格式

输出一行，为期望的交换次数。结果四舍五入，保留到小数点后六位。

## 说明/提示

- $1 \le |s| \le 1000$。

# SCPC11G - Indomie

## 题目描述

经济不景气的时候，阿姆贾德需要排队领取「SembakoPlus」物资。「Sembako」是印度尼西亚的「九种基本食品」，包括大米、糖、食用油、肉类、鸡蛋、牛奶、玉米、煤油和碘盐。「SembakoPlus」则在此基础上增加了一种物品：阿姆贾德最爱的印多美方便面！（所以他愿意排这么长时间的队伍也就不足为奇了。）

每个人只能从中选择一种物品。自然地，阿姆贾德只想要印多美方便面。不幸的是，「SembakoPlus」的库存快要用完了，现在只剩下大米、糖和印多美方便面这三种。阿姆贾德从远处观察，比较确定大米和糖的数量是足够的。

现在，给定剩余的印多美方便面的数量，以及阿姆贾德前面排队的人数，请你计算他可能拿到印多美方便面的概率。由于他现在紧张得无法思考，只好请求你的帮助！

## 输入格式

本题包含多组测试数据。每组测试数据包含两个整数 $N$ 和 $M$，表示剩余的印多美方便面数量和阿姆贾德前面排队的人数。

## 输出格式

对于每组测试数据，输出一个浮点数，表示阿姆贾德能拿到印多美方便面的概率。答案的绝对误差或相对误差不超过 $10^{-6}$ 视为正确。

## 数据范围

- $1 \le N, M \le 10^5$

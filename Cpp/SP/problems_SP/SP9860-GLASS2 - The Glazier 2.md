# GLASS2 - The Glazier 2

## 题目描述

玻璃匠 Jozo 再次制作了 $N$ 块正方形玻璃板。这些玻璃的边长分别是 $1, 2, 3, \ldots, N$，因此它们的面积分别为 $1^2, 2^2, 3^2, \ldots, N^2$。

这时有四位顾客前来购买。Jozo 计划将 $N$ 块玻璃全部售出，并且每位顾客得到的玻璃总面积相等（等于所有 $N$ 块玻璃面积之和的四分之一）。

请帮助 Jozo 将这些玻璃合理分配给四位顾客。可以假设所有的测试数据中都存在符合要求的解（解可能不唯一）。

## 输入格式

输入一个整数 $N$，表示玻璃的数量，满足 $10 \leq N \leq 50000$。

## 输出格式

第一行输出四个整数，分别表示分配给第一、第二、第三和第四位顾客的玻璃数量。

接下来的四行中，第 $i$ 行输出分配给第 $i$ 位顾客的所有玻璃的边长。（每块玻璃只能分配给一位顾客。）

# NWERC11A - Binomial coefficients

## 题目描述

小明有一个梦想，他希望获得一块“魔法石”，这种石头可以让他实现任何愿望。为了得到这块石头，他需要完成一个复杂的任务。

## 输入格式

第一行包含一个整数 $n$，表示任务的步骤数。

接下来的 $n$ 行中，每一行描述一个步骤。每个步骤由两个整数 $a$ 和 $b$ 组成，表示需要完成的子任务。

## 输出格式

输出一个整数，表示小明完成所有任务后获得的积分总和。

## 说明/提示

- 对于所有的输入数据，有 $1 \leq n \leq 1000$。
- 对于每一步的子任务，$0 \leq a, b \leq 100$。
- 每个子任务完成后，小明会获得 $a + b$ 的积分。

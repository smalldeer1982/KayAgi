# NWERC11B - Bird tree

## 题目描述

给你一个由 N 个整数构成的数组，你需要找到其中两个元素，使得它们的和为给定的目标值 target，并返回这两个元素的索引。假设每种输入只对应唯一的答案，并且同一个元素不能使用两次。

## 输入格式

输入的第一行包含一个整数 N（表示数组元素的个数）。  
第二行包含 N 个整数，表示数组中的元素。  
第三行包含一个整数 target，表示需要寻找的目标和。

## 输出格式

输出两个整数，表示找到的两个元素的索引（索引从 0 开始计数）。这两个索引需要按照从小到大的顺序输出。

## 说明/提示

- 2 ≤ N ≤ 10^5
- -10^9 ≤ 数组中的元素 ≤ 10^9
- -10^9 ≤ target ≤ 10^9 
- 假设每种输入只会有一个解。

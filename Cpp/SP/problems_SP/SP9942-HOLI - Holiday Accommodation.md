# HOLI - Holiday Accommodation

## 题目描述

给出一棵$N$节点的树，每个节点代表一个城市，每个城市有一个人，每个人离开自己的城市到另一个城市，每个城市只能有一个人，问这$N$个人移动距离和的最大值。

## 输入格式

输入的第一行包含一个整数$T(1 <= T <= 10)$，表示测试用例的数量。每个测试用例包含几行。
第一行包含一个整数$N(2<=N<=10^5)$,代表城市数。
然后接下来的$N - 1$行分别包含三个整数$X,Y,Z(1 <= X,Y <= N,1<=Z<=10^6)$，意味着在城市$X$和城市$Y$之间有一条高速公路，其长度为$Z$。
您可以假设所有城市都已连接并且高速公路是双向的。

## 输出格式

每个输出一行，表示所有人员的最大总行程，在前面加上"Case #测试点编号:"(不含引号)。
  
#### 输入样例1
```
2 
4 
1 2 3 
2 3 2 
4 3 2 
6 
1 2 3
2 3 4
2 4 1
4 5 8
5 6 5
```
#### 输出样例1
```
18
62
```

## 样例 #1

### 输入

```
2
4
1 2 3
2 3 2
4 3 2
6
1 2 3
2 3 4
2 4 1
4 5 8
5 6 5
```

### 输出

```
Case #1: 18
Case #2: 62
```

# GUANGGUN - 111&#8230;1 Squared

## 题目描述

如果一个位数为$n$的十进制正整数每个数字都是$1$,则称之为$S(n)$.求$S(n)^2$的各位数字之和。

## 输入格式

输入包含多组数据，每行一个正整数$n$.

## 输出格式

对于每一个$n$,输出$S(n)^2$的各位数字之和，需要换行。

## 样例 #1

### 输入

```
9
10
```

### 输出

```
81
82
```

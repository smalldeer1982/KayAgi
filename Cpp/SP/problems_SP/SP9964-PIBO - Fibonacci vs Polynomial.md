# PIBO - Fibonacci vs Polynomial

## 题目描述

定义一个数列 _Pib_(n) ，具体规则如下：

- _Pib_(0) = 1
- _Pib_(1) = 1
- 对于 n ≥ 2，_Pib_(n) = _Pib_(n-1) + _Pib_(n-2) + 多项式 **P**(n)

其中，**P** 是一个多项式。

给定整数 **n** 和多项式 **P**，请计算 _Pib_(n) 对 1,111,111,111 取模后的结果。

# PELL2 - Pell (Mid pelling)

## 题目描述

给定 $d$，求方程 $x^2=dy^2+1$ 的最小正整数解。

请注意，方程可能有解，也可能无解。 例如: 当 $d=2$ 时，$3^2= 2 \times 2^2+ 1$，有解 $x = 3$ 和 $y = 2$。 当 $d=3$ 时，$2^2= 3 \times 1^2+ 1$，有解 $x = 2$ 和 $y = 1$。 当 $d=4$ 时，无解。

## 样例 #1

### 输入

```
3
2
3
4
```

### 输出

```
3 2
2 1
-1
```

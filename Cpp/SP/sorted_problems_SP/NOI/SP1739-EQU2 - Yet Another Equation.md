# EQU2 - Yet Another Equation

## 题目描述

**题目大意：**

对于方程

$$x^2 - ny^2 = 1$$

$n$ 是一给定整数，求解其最小正整数解 $(x, y)$。

## 输入格式

输入第一行一个正整数 $t \le 30$，表示数据组数。  
接下来 $t$ 行每行一个整数 $2 \le n \le 1000$，保证对于给定的 $n$ 方程有解。

## 输出格式

请你每行输出两个正整数 $x\ y$，即每个 $n$ 对应的最小解。

## 样例 #1

### 输入

```
3
2
6
61
```

### 输出

```
3 2
5 2
1766319049 226153980
```

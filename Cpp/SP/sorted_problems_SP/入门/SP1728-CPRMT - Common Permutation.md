# CPRMT - Common Permutation

## 题目描述

给定两个由小写字母组成的字符串 $a$ 和 $b$，将 $a$ 和 $b$ 中同时出现的字符统计到字符串 $x$，如果有一个字符出现多次则统计多次，最后按字母的字典序输出。

## 输入格式

输入文件包含多行输入，连续的两行构成一组输入。这意味着，第一行和第二行为一组输入，第三行和第四行为一组输入，以此类推。一组输入的第一行为 $a$，第二行为 $b$。每个字符串在单独的一行上，都为长度不超过 $1000$ 的小写字符串。

## 输出格式

对于每一组数据，输出包含一个 $x$，$x$ 中的所有字母需要按照字典序输出。

## 样例 #1

### 输入

```
pretty
women
walking
down
the
street
```

### 输出

```
e
nw
et
```

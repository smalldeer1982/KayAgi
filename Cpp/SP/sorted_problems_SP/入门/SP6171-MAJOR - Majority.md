# MAJOR - Majority

## 题目描述

有$t$组数据，每组数据中有$n$个数，问有没有一个数它在数据中出现的个数超过$n$的一半，有就输出$“YES”$，再输出这个数，否则输出$“NO”$。

## 输入格式

输入的第一行包含测试用例$t（1<=t<=100）$。它后面是$2*t$行，每个测试用$2$行。第一行输入包含一个数字$n（0<=n<=$10^6$）$，然后在下一行添加n个元素。每个数字从-10^3~10^3。

## 输出格式

如果发送成功，则需要输出“YES”，然后输出发送的号码，否则输出“NO”。

## 样例 #1

### 输入

```
3
4
2 1 2 2
6
1 1 1 2 2 2
5
1 2 4 5 1
```

### 输出

```
YES 2
NO
NO
```

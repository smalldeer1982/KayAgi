# FIBOSUM - Fibonacci Sum

## 题目描述

斐波那契数列 $F(N)$ 的定义如下所述：
 - $F(0) = 0$
 - $F(1) = 1$
 - $F(N) = F(N - 1) + F(N - 2), N \ge 2$

现在给出两个非负整数 $N, M$，你需要计算 $(F(N) + F(N + 1) + ... + F(M)) \bmod 1000000007.$

## 输入格式

第一行为一个非负整数 $T$，代表数据组数。接着是 $T$ 行，每行两个整数 $N, M$。

## 输出格式

$T$ 行，为每组数据的答案。

## 说明/提示

- $T \le 1000$
- $0 \le N \le M \le 10^9$

## 样例 #1

### 输入

```
3
0 3
3 5
10 19
```

### 输出

```
4
10
10857
```

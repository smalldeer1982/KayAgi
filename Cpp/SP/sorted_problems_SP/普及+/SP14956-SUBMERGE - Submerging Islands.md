# SUBMERGE - Submerging Islands

## 题目描述

给你一张有 $N$ 个点、$M$ 条边的无向图。求这张图一共有多少个割点。

## 输入格式

**本题有多组测试数据。**

对于每组测试数据，第一行两个正整数 $N$ 和 $M$，分别表示图中点的数量和边的数量。

接下来 $M$ 行，每行两个正整数 $U_i$ 和 $V_i$，表示点 $U_i$ 和点 $V_i$ 之间连有一条边。

当 $N=M=0$ 时表示输入结束，该数据不需要进行处理。

Translated by [Waldin](https://www.luogu.com/user/965484)。

## 输出格式

对于每组测试数据，输出一个整数表示图中割点的数量。每组测试数据的输出之间需要换行。

## 说明/提示

对于所有测试数据，满足 $1\le N\le10^4$ 且 $1\le M\le10^5$。

## 样例 #1

### 输入

```
3 3
1 2
2 3
1 3
6 8
1 3
6 1
6 3
4 1
6 4
5 2
3 2
3 5
0 0
```

### 输出

```
0
1
```

# AMZRCK - Amz Rock

## 题目描述

# 翻译
- $AmzMohammad$ 是一位摇滚乐粉丝。他有 $n$ 张摇滚唱片（标号从 $1$ 到 $n$ )。现在他想选择一个播放列表。他认为一个好的播放列表是不含两张连续唱片的。有多少种方式？

## 输出格式

- 说明/提示：好的播放列表可能包含 $0$ 张唱片。

## 样例 #1

### 输入

```
2
1
2
```

### 输出

```
2
3
```

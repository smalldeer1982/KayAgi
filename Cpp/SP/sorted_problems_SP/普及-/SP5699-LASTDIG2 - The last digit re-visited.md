# LASTDIG2 - The last digit re-visited

## 题目描述

求a^b的个位数

## 输入格式

第一行包含整数t，代表数据有t组  
接下来t行每行各有一个整数a和b

## 输出格式

每行输出一组数据的运算结果

## 说明/提示

( $ 0 \le a < 10^{1000} $)  
( 0 <= b <= 922*10^15 )  
( t<=30 )

## 样例 #1

### 输入

```
3
3 10
6 2
150 53
```

### 输出

```
9
6
0
```

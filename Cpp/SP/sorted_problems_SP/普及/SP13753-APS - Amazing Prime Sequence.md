# APS - Amazing Prime Sequence

## 题目描述

Bablu 非常喜欢序列,他按照以下方式设计了自己的序列：$a_0=a_1=0$，对于 $n(n>1)$，$a_n=a_{n-1}+f_n$，其中 $f_n$ 是 $n$ 的最小素因子。 他也非常喜欢编程，因此制作了一个小程序来找到 $a_n$，但由于他不擅长编程。所以，他请你帮忙找到 $a_n$。

## 输入格式

**本题有多组数据**。

第一行输入包含 $T(T\le 100)$，即测试用例的数量。

接下来的 $T$ 行包含单个数字 $n(1<n<10^7)$。

## 输出格式

共 $T$ 行，每行包含对应的 $a_n$。

## 样例 #1

### 输入

```
3
2
3
4
```

### 输出

```
2
5
7
```

# AFS - Amazing Factor Sequence

## 题目描述

给定 $n$，求
$$\sum_{i=1}^n\sum_{j|i} j$$

## 输入格式

第一行一个正整数 $T$，表示数据组数。  
接下来 $T$ 行，每行一个正整数 $n$。

## 输出格式

输出 $T$ 行，每行一个整数对应一组数据的答案。

## 说明/提示

【数据范围】  
对于 $100\%$ 的数据，$1\le T \le 100$，$1\le n \le 10^6$。

## 样例 #1

### 输入

```
3
3
4
5
```

### 输出

```
2
5
6
```

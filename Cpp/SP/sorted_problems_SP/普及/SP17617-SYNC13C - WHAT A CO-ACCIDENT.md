# SYNC13C - WHAT A CO-ACCIDENT

## 题目描述

Ramesh 和 Suresh 玩游戏，这个游戏的规则是这样的：首先给定两个盒子中的巧克力数量 $c1$ 和 $c2$，然后一名玩家拿走两个盒子中任意一个盒子中的巧克力，之后再把另一个盒子中的巧克力分成数量可以相同但都非空的两份，若不能把这个盒子中的巧克力分成两份，则这名玩家输掉比赛，另一名玩家赢得比赛。
 
 他们轮流玩且 Suresh 开始这个游戏。
 
 假设两个玩家都发挥最佳状态。

## 输入格式

输入的第一行包含数字 $T$ $(1 \le T \le 1000)$，表示游戏进行的轮数。然后是 $T$ 行，每行包含两个整数 $c1$ 和 $c2$ $(1\le c1 \le c2 \le 10000)$ 用空格分隔。

## 输出格式

输出获胜者的名字（Ramesh 或 Suresh）。

## 样例 #1

### 输入

```
2
3 1
4 5
```

### 输出

```
Ramesh
Suresh
```

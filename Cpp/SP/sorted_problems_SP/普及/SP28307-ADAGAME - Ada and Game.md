# ADAGAME - Ada and Game

## 题目描述

## 题意简述
Ada 和 Vinit 在玩数字游戏，Ada 先开始，先给定一个四位数 $s$ ，每次操作，他们都必须将四位数的某一位加 $1$ （如果此位为 $9$ ，将变成 $0$ ）， 如果在 $n$ 次操作后此数大于原数，Ada 获胜，否则 Vinit 获胜。（他们每次都按照最优策略操作）

## 输入格式

第一行一个整数 $t$ ，表示测试组数。

接下来 $2$ 到 $t+1$ 行，每行两个整数 $s,n$ 表示四位数及操作次数。

## 输出格式

每行一个字符串 "$Ada$" 或 "$Vinit$" ，表示谁获胜。

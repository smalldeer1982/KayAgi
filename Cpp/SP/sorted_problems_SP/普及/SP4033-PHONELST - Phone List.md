# PHONELST - Phone List

## 题目描述

**【题目来源】**

	Poj3630

**【问题描述】**

	给定n个长度不超过10的数字串，问其中是否存在两个数字串S，T，使得S是T的前缀。有多组数据，数据组数不超过40。n<=10000。

## 输入格式

第一行，一个整数T，表示数据组数。
	对于每组数据，第一行一个数n，接下去n行，输入n个数字串。

## 输出格式

对于每组数据，若存在两个数字串S，T，使得S是T的前缀，则输出“NO”，否则输出“YES”。

**【数据规模】**

	n<=10000

## 样例 #1

### 输入

```
2
3
911
97625999
91125426
5
113
12340
123440
12345
98346
```

### 输出

```
NO
YES
```

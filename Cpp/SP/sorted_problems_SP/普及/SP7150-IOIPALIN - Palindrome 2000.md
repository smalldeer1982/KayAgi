# IOIPALIN - Palindrome 2000

## 题目描述

回文串是一种对称的字符串，即从左向右和从右到左读是相同的。请你写一个程序，对于给定的字符串，确定给出能让原字符串成为回文串的插入的最小的字符数。例如，通过插入两个字符，“Ab3bd”可以被转换为一个回文串（“dAb3bAd”或“Adb3bdA”）。而插入少于两个字符就不能形成回文串了。

## 输入格式

第一行包含一个整数：
字符串的长度
第二行包含一个字符串：
要成为回文串的字符串

## 输出格式

第一行包含一个整数：
所需的最少字符个数
## 输入输出样例
### 输入样例
5
Ab3bd
### 输出样例
2

## 样例 #1

### 输入

```
5
Ab3bd
```

### 输出

```
2
```

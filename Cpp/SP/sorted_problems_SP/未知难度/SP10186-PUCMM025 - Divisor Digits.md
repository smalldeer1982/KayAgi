# PUCMM025 - Divisor Digits

## 题目描述

给定一个整数，统计并输出这个整数中有多少个数字可以整除该整数。本题需要统计所有满足条件的数字出现的次数，不仅仅是首次出现。

例如，整数 12345 可以被 1、3 和 5 整除，所以在这个例子中，答案是 3。

## 输入格式

每个测试用例输入一个整数，这个整数的取值范围是 1 到 $10^{240}$ 之间。

## 输出格式

对于每个测试用例，输出整数中可以整除它的数字个数。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
\n12345\n661232\n312\n730000000\n\n
```

### 输出

```
\n3\n3\n3\n0
```

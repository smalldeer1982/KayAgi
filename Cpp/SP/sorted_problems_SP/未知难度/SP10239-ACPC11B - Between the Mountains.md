# ACPC11B - Between the Mountains

## 题目描述

你有没有尝试过乘坐缆车？这是一项非常有趣的活动。我们的公司计划在两座高山之间修建一条新的缆车线路，并邀请你免费体验。要获得这次难得的免费体验，你需要帮公司完成缆车线路的建设工作。

公司计划在每座山上各建一个平台，然后在这两个平台之间架设缆车线。每座山上提供了多个适合作为平台的地点，并且我们已经获得了这些地点的海拔信息。

有一位才华横溢的工程师建议，如果能让两个平台的海拔相同或尽可能接近，公司可以节省大量的能源。你的任务是从每山的可选地点中各选择一个，使它们之间的海拔差最小。

## 输入格式

输入的第一行是一个整数 $T$，表示测试用例的数量（$1 \le T \le 100$）。

接下来每个测试用例包含两行，分别描述两座山上适合建平台的地点海拔高度。每行的第一个整数 $N$ 表示该山上报告的海拔数量（$1 \le N \le 10^5$），随后是 $N$ 个整数，表示各个地点的海拔高度（$0 \le \text{altitude} \le 10^9$）。

## 输出格式

对于每个测试用例，输出一行，一个整数，表示从两座山上各选择一个地点时，其海拔高度的最小差值。

# CADYDIST - Candy Distribution

## 题目描述

有 $N$ 个糖果盒摆成一排。第 i 个糖果盒中有$A_i$颗糖果。你需要选择连续的一段盒子并把其中的所有糖果平均分配给 $M$ 个孩子。也就是说，请你求出满足以下条件的整数对($l$, $r$)。

$1 \le l \le r \le N$，并且$\sum \limits _{i = l} ^{r}A_i$是$M$的倍数。

## 输入格式

输入按照以下格式给出

$N$  $M$

$A_1$  $A_2$  $A_3$  $A_4$ ...... $A_N$

## 输出格式

输出一个整数表示满足条件的整数对的数量。

## 说明/提示

$1 \le N \le 10 ^ {5}$

$2 \le M \le 10 ^ {9}$

$1 \le A_i \le 10^{9}$

## 样例 #1

### 输入

```
4\n1 1 1 1\n2 2 2 2\n5\n10 80 37 22 109\n6 8 8 20 15\n0
```

### 输出

```
8\n2120
```

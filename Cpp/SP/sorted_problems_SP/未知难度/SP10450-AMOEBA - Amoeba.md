# AMOEBA - Amoeba

## 题目描述

### 题意翻译
富拉诺博士发现了一种新的微生物，即快速生长的变形虫。这种变形虫的菌落生长变化很快。为了找出菌落的准确生长速度，他连续几天写下了菌落中变形虫的数量。于是他知道每个变形虫每天产生$6$个新个体。由于变形虫的数量增长非常快，富拉诺博士只记住了最后的$K$位数。当他意识到所记的数字重复时，已经太晚了！这项研究已经取得了很大进展，没有时间重新叙述。解决办法是向你寻求帮助。



富拉诺博士会给你一个$K$位数$（0 $$\le$$ K\le$$10）$，这个数可能有前导零。你应该在他写下这个数字的第一天回答。注意在第一天，菌落由$1$个变形虫组成，并且还要考虑变形虫在产生子代时不会消失。

如果对给定的数字没有答案，请输出$-1$。

## 输入格式

输入若干个$K$位数。

## 输出格式

对于输入中的每一行，输出对应一行。注意输出不包含前导零。

## 样例 #1

### 输入

```
01\n143\n743\n600
```

### 输出

```
1\n20\n12\n-1
```

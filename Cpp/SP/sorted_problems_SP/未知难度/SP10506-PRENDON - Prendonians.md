# PRENDON - Prendonians

## 题目描述

普伦多尼亚的古文明曾生活在如今的日德兰半岛，历史可以追溯到公元前 5000 年至公元前 4500 年。

## 输入格式

输入的第一行是一个整数 $T$，表示测试用例的数量，满足 $1 \le T \le 100$。接下来是 $T$ 组测试用例。每组测试用例的第一行是一个整数 $N$，表示名字的数量，满足 $1 \le N \le 100$。接下来有 $N$ 行，每行包含三个单词（仅由小写字母组成，每个单词的长度不超过 103 个字符），其中第一个单词是唯一的。

## 输出格式

对于每组测试用例，如果所有名字都有效且属于一个有效的普伦多尼亚家族树，并且所有名字之间都有亲属关系，则输出 `YES`；否则，输出 `NO`。

# KOPC12D - K12-Generating Big Numbers II

## 题目描述

给定几个数字的出现次数，你需要使用这些数字构造一个数 $X$。在 $X$ 中，每个数字不能出现在它的倍数位置上。例如，2 不能出现在第 2、4、6、8 等位置；而 7 不能出现在第 7、14 等位置。注意，这里的个位为位置 1，十位为位置 2，百位为位置 3，依此类推。举例来说，假设 $X = abcdef$，那么 $f$ 在位置 1，$e$ 在位置 2，$d$ 在位置 3，$a$ 在位置 6。

## 输入格式

输入的第一行是一个整数 $T$，表示有 $T$ 个测试用例。接下来是 $T$ 行，每行包含 8 个用空格分隔的整数。这 8 个整数分别表示各个数字从 2 到 9 的数量。

注意：不会给出 1 的数量，因为 1 可以放在任何位置；0 也可以放在任何位置。

## 输出格式

输出应包含 $T$ 行，每行对应一个测试用例构造的数。

### 输入

```
2
2 0 2 0 0 0 0 0

```

```
160
```

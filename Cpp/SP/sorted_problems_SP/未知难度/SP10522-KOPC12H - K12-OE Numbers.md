# KOPC12H - K12-OE Numbers

## 题目描述

Rishi 非常钟爱数字及其规律。他想要在区间 \([a, b]\) 中找出 OE 数。OE 数是一种特殊的数字，其偶数位的和（依次指 0, 2, 4, 6, 8）大于奇数位的和（依次指 1, 3, 5, 7, 9）。请帮他计算区间 \([a, b]\)（包括 \(a\) 和 \(b\)）中有多少个这样的 OE 数。

### 示例：
- 对于数字 4563：奇数位的和 \(O = 5 + 3 = 8\)，偶数位的和 \(E = 4 + 6 = 10\)，因为 \(E > O\)，所以 4563 是一个 OE 数。
- 对于数字 1233：奇数位的和 \(O = 1 + 3 + 3 = 7\)，偶数位的和 \(E = 2\)，因为 \(E < O\)，所以 1233 不是 OE 数。
- 对于数字 10：奇数位的和 \(O = 1\)，偶数位的和 \(E = 0\)，因为 \(E < O\)，所以 10 不是 OE 数。

## 输入格式

第一行输入一个整数 \(T\)，表示测试用例的数量。接下来是 \(T\) 行，每行包含两个用空格分隔的整数 \(a\) 和 \(b\)。

## 输出格式

对于每个测试用例，输出一行，表示区间 \([a, b]\) 中 OE 数的个数。

## 数据范围
1 ≤ \(a\) ≤ \(b\) ≤ \(10^8\)  
\(T\) ≤ 35

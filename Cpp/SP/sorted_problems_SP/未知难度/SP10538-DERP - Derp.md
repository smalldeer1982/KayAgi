# DERP - Derp

## 题目描述

生产甲基苯丙胺通常是一项繁琐且乏味的工作。题目中需要计算一些粒子之间的最远距离。

## 输入格式

输入包含若干个测试用例。对于每个测试用例，第一行是一个整数 $N$，表示该样本包含的粒子数量（$2 \le N \le 10^5$）。

接下来的 $N$ 行中，每行包含三个整数 $l_i, x_i, y_i$，分别代表第 $i$ 个粒子的所在层数、X 轴位置和 Y 轴位置（$1 \le l_i, x_i, y_i \le 10^5$）。

## 输出格式

对每个测试用例，输出一个整数 `DISTANCE`，表示该样本中最远的两个粒子之间的距离。

## 说明/提示

- 粒子数量范围：$2 \le N \le 10^5$
- 每个坐标的取值范围：$1 \le l_i, x_i, y_i \le 10^5$

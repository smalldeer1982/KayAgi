# BUNNIES - Bunnies

## 题目描述

Pompom 兔子有 $N$ 个奇怪的蛋。要打破第 $i$ 个蛋，需敲击恰好 $A_i$ 次。

## 输入格式

第一行是一个整数 $T$，代表测试用例的数量。接下来有 $T$ 个测试用例。每个测试用例的第一行包含两个整数 $N$ 和 $K$，然后第二行是 $N$ 个整数 $A_1, A_2, \ldots, A_N$。

## 输出格式

对于每个测试用例，输出在最坏情况下所需的最少敲击次数。

## 数据范围
- $1 \le T \le 10$
- $1 \le K \le N \le 3000$
- $1 \le A_i \le 1000000$ (即 $10^6$)

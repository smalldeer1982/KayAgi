# OFFSTRAT - Problem Offensive Strategy

## 题目描述

Foo 足球俱乐部是一支非常重视进攻的球队。教练正在制定一种全新的进攻策略。  
为了保持进攻队的风格，教练想保证这个新策略能够展现出强烈的进攻性。他脾气古怪，规定球员们只能进行策略表中指定的传球。实际上，他甚至不允许持球球员移动。

一个进攻策略被认为是强烈进攻性的，满足两个条件：第一，球场上每个传球位置都必须传向场内距离对方球门线相同或更近的位置；第二，从策略表中任何传球位置都必须能够最终传到对方球门。

## 输入格式

输入的第一行是一个整数 $T$（$T \le 20$），表示测试用例的数量。接下来每个测试用例描述一个具体的策略。

每个测试用例以两个整数 $W$ 和 $H$ 开头（$1 < W, H \le 100$，且 $W$ 和 $H$ 为偶数），它们表示足球场的宽度和高度。球门线是 $x = W$ 这条直线。如果从 $(x, y)$ 能够到达 $(W, H/2)$，那么视为可以射门得分。

然后是一个整数 $M$（$1 \le M < 1000000$），表示策略中包含的传球数量。接下来的 $M$ 行每行包含四个整数 $x_0, y_0, x_1, y_1$（$0 \le x_0, x_1 \le W$ 且 $0 \le y_0, y_1 \le H$），表示传球的起点 $(x_0, y_0)$ 和终点 $(x_1, y_1)$。

## 输出格式

对于每个测试用例，如果策略是强烈进攻性的，输出 `Yes`（不带引号）；否则，输出 `No`（不带引号）。

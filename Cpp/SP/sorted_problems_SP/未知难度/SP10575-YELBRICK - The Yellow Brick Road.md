# YELBRICK - The Yellow Brick Road

## 题目描述

自从多萝西在奥兹国的精彩冒险之后，黄砖路上的交通愈发繁忙，许多路段因坑洼而难以通行。奥兹国的工程师面临着铺设黄砖路的难题，因为黄色石头的供给紧缺，使得他们不得不从不同的供应商处采购石料。在奥兹国，每条道路必须使用完美且大小一致的立方体砖块来建造。为了保证道路的耐久性，同时减少砖块使用数量变得至关重要。然而，各个供应商提供的石料尺寸各异，需要切割才能制成适用于道路的砖块。

你能帮助工程师们确定使用哪种尺寸的砖块可以使道路更加耐用吗？

## 输入格式

每组测试数据由多行组成。第一行包含一个整数 $N$，表示黄色石头供应商的数量（$2 \le N \le 10^5$）。接下来的 $N$ 行中，每一行提供两个整数 $L_i$ 和 $W_i$，分别表示第 $i$ 个供应商提供的石料的长度和宽度（$1 \le L_i, W_i \le 10^6$）。

## 输出格式

对于每组测试数据，输出一行结果，表示可以从给定石料中切割出的最少数量且相同大小的立方体砖块。

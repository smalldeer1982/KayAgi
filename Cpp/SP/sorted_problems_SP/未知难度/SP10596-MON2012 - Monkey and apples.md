# MON2012 - Monkey and apples

## 题目描述

猴子克里斯非常喜欢苹果，因此他种了很多苹果树。在这些苹果树上，他可以进行两种操作：

- 第一种操作 "1 x y"：用来统计在区间 \[x+c, y+c\] 内成熟的苹果树的数量。
- 第二种操作 "2 x y"：可以让区间 \[x+c, y+c\] 内的所有苹果都变成熟（变得可以食用）。

开始时，\(C\) 的初始值为 0。在每次执行 "1" 操作后，\(C\) 会更新为该操作的查询结果。

## 输入格式

输入的第一行是整数 \(M\) （\(1 \le M \le 10^5\)），表示接下来会有多少操作。在接下来的 \(M\) 行中，每一行描述一个操作，可能是 "1" 或 "2"，并且每个操作都满足 \(1 \le X+C, Y+C \le 10^9\)。

## 输出格式

对于每一个 "1" 操作，输出一行结果，表示指定区间内成熟的苹果树的数量。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
3
2 5 8
2 7 10
1 1 10
```

### 输出

```
6
```

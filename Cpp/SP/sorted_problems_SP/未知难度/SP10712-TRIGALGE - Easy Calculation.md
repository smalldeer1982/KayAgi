# TRIGALGE - Easy Calculation

## 题目描述

求x，使得 Ax + Bsin（x）= C。

## 输入格式

第一行输入一个整数 T ，表示测试样例的个数，

接下来 T 行，每行三个整数 A，B，C $(A \geq B \geq 0)$ 。所有整数都 $\leq 100000$

## 输出格式

T行，每行一个实数 x ，保留6位小数。

## 样例 #1

### 输入

```
1
1 1 20
```

### 输出

```
19.441787
```

# WACHOVIA - Wachovia Bank

## 题目描述

Danilo Gheyi 是一位名声在外的银行劫匪，以其在巴西塞阿拉州福塔雷萨完成最成功的银行抢劫而闻名。他和他的同伙们通过挖隧道进入了金库内部，金库里有多袋装有不同金额的钱或珠宝的袋子。但由于卡车的载重有限，他们不得不留下约一半的总价值。

Danilo 并未被警方抓获，现在他为了证明自己的能力，计划对美国最安全的银行之一——Wachovia 银行再次下手。他需要你的帮助，以尽可能多地盗取财物，同时避免像在福塔雷萨那次损失过重。

你需要编写一个程序，根据卡车最大的载重能力及每个袋子的具体信息，帮助 Danilo 算出他最多能偷到多少价值。

## 输入格式

输入由多组测试实例组成。每组实例的第一行包含一个整数 $N$（表示袋子的数量）。接下来 $N$ 行，每行包含两个整数 $V_i$ 和 $W_i$，分别表示第 $i$ 个袋子的价值和重量。最后一行包含一个整数 $C$，表示卡车的最大载重量。

## 输出格式

对于每个测试实例，输出格式为：“Hey stupid robber, you can get P.”，其中 $P$ 代表 Danilo 所能盗取的最大价值。

## 说明/提示

- $1 \le N \le 1000$
- $1 \le V_i, W_i \le 10^6$
- $1 \le C \le 10^6$

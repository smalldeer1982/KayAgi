# TETRAHRD - Sum of Tetranacci numbers

## 题目描述

四阶斐波那契数列是这样定义的：
$$ a_n = a_{n-1} + a_{n-2} + a_{n-3} + a_{n-4} $$
其中，前几个数为 $a_0 = a_1 = a_2 = 0$，而 $a_3 = 1$。

## 输入格式

输入首先给出一个正整数 $t$，表示测试用例的数量。接下来每个测试用例包含两个整数 $m$ 和 $n$，表示你需要计算数列和的范围。

## 输出格式

对于每个测试用例，计算从$a_m$到$a_n$的数列和，即 $a_m + a_{m+1} + \cdots + a_n$，然后输出这个和对 $1000000007$ 取模的结果。

## 说明/提示

- 测试用例数量 $t$ 满足：$1 \le t \le 10^5$。
- 下标范围 $m$ 和 $n$ 满足：$0 \le m \le n \le 10^5$。

# DCEPC206 - Its a Murder!

## 题目描述

侦探赛卡特在调查一起谋杀案时，走楼梯前往犯罪现场，注意到每级楼梯上都写有一个数字。他觉得这些数字很可疑，于是决定记住所有看到的数字。在回忆这些数字时，他发现其中存在一些规律。因此，他决定对于每个楼梯上的数字，记录下之前所见到的所有比当前数字小的数字之和。请你计算出他在笔记本中记录的所有数字之和。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。

接下来有 $2T$ 行。

每个测试用例的第一行输入一个整数 $N$，表示楼梯的数量。

每个测试用例的第二行输入 $N$ 个整数，表示写在楼梯上的数字。

## 输出格式

对于每个测试用例，输出一行，表示笔记本中记录的最终总和。

## 说明/提示

- $1 \le T \le 10$
- $1 \le N \le 1000$
- $1 \le \text{数字} \le 100000$

## 样例 #1

### 输入

```
1\n5\n1 5 3 6 4\n\n
```

### 输出

```
15
```

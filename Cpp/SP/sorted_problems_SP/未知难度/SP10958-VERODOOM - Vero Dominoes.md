# VERODOOM - Vero Dominoes

## 题目描述

Vero 有一天玩了一种喝橙汁的游戏。游戏规则是翻开一个骨牌，数一数上面的黑点，然后喝下等同于黑点数量的分升（1 分升 = 0.1 升）橙汁。例如，在一副最大为 6 点的骨牌中，黑点的总数是 168。

她想知道在一个简单的游戏中最多能喝多少分升的橙汁。她不太相信你的编程能力，所以准备了超过 15 万个测试用例来验证你的程序。

**输入格式：**

输入的第一行为整数 $T$，表示测试用例的数量。接下来的 $T$ 行每行包含一个整数 $N$，表示骨牌的最大点数。

**输出格式：**

对于每个给定的整数 $N$，输出一副最大为 $N$ 点的骨牌上的总点数。

**数据范围与提示：**

$$1 \leq T \leq 150,000$$

$$0 \leq N \leq 50,000,000$$

**样例输入：**

```
2
6
1
```

**样例输出：**

```
168
3
```

**样例解释：**

对于第二个测试用例，当骨牌的最大点数为 1 时，只存在三张不同的骨牌：0-0、0-1 和 1-1（0-1 和 1-0 表示的是同一张骨牌）。

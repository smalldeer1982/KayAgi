# MAIN12B - PrimeFactorofLCM

## 题目描述

每个人都喜爱史瓦米。史瓦米是一只住在城市下水道里的鳄鱼，他一直渴望过上更接近人类的生活。有一天，他参加了一场数学竞赛，想要向其他鳄鱼朋友展示自己非凡的实力。比赛的任务是要求他计算出多个数字的最小公倍数的质因数。经过一番辛苦努力，他感到疲惫，于是来请你帮忙。他相信你能帮助他解决这个难题。

## 输入格式

输入的第一行是一个整数 $T$，表示有 $T$ 组测试用例。接下来的每组测试用例，第一行包含一个整数 $n$，表示有 $n$ 个数字。第二行是 $n$ 个用空格分隔的整数，分别是 $a_1, a_2, \ldots, a_n$。

## 输出格式

对于每组测试用例，输出 `Case #X: M`，其中 $M$ 是 $a_1, a_2, \ldots, a_n$ 这 $n$ 个数的最小公倍数的质因数的个数。接着在随后的 $M$ 行中按从小到大的顺序依次输出这些质因数。

## 数据范围
- $1 \le T \le 10$
- $1 \le n \le 100$
- $1 \le a_i \le 10^6$

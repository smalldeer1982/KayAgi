# IEEEBGAM - The Ball Game

## 题目描述

Taru 和 Hanaku 正在玩一个球类游戏。在这个游戏中，有 $N$ 个白球、$N$ 个黑球以及 $N$ 个盒子。Hanaku 会随机选择一个盒子，然后从中随机拿出一个球。Taru 的任务是将这些球巧妙地分配到盒子中，以使得 Hanaku 选中白球的概率达到最大。在安排过程中，每个盒子至少要有一个球。你需要找出在这种策略下，Hanaku 选到白球的最大概率。

## 输入格式

第一行输入一个整数 $T$，表示有多少个测试用例。接下来的 $T$ 行中，每行包含一个整数 $N$，表示游戏中球和盒子的数量。

## 输出格式

对于每个测试用例，输出在最佳安排下选到白球的最大概率，并四舍五入到小数点后 8 位。

## 数据范围
$$1 \le T \le 100,000, \quad 1 \le N \le 100,000$$

# SPWORLD - Korra in the Spirit World

## 题目描述

阿凡达科拉被困在灵界，召唤了阿凡达安昂前来帮助。为了考验科拉是否足够聪慧，有能力赢得古老的互质多边形游戏，安昂准备了这个挑战。两个多边形的边数如果是互质的，我们就称它们为互质多边形。比如说，三边形和四边形是互质的，而四边形和六边形则不是。游戏开始时有 $N$ 个多边形，游戏规则如下：

1. 玩家需要选择一个多边形，将其切割成两个互质多边形。切割必须沿着连接多边形两个顶点（不能是相邻的顶点）的线进行。这两个新生成的互质多边形将加入游戏中。如果某位玩家无法对任何多边形进行切割，这位玩家就输掉游戏。
2. 阿凡达安昂先开始游戏，每完成一次切割后，轮到另一位玩家进行下一次切割。每回合中，玩家只能切割一个多边形。
3. 虽然安昂认为科拉不够聪明，但他忽略了科拉拥有所有阿凡达的智慧，其中也包括他的。因此，她完全了解这个游戏。所以两人都会以最优策略进行挑战。

## 输入格式

第一行包含测试用例的数量 $T$。接下来有 $T$ 行，每行包含一个测试用例。每个测试用例首先是一个整数 $N$，表示多边形的数量。接着是 $N$ 个整数，表示这些多边形的边数。每个多边形至少有 3 条边。

## 输出格式

对每个测试用例，输出本场游戏的获胜者。

## 说明/提示

- 测试用例的数量 $T$ 满足 $T \leq 15$
- 多边形的数量 $N$ 满足 $N \leq 7$
- 多边形的边数至少为 3，最多为 1000。

**样例输入**

```
2
3 5 6 7
2 6 7
```

```

Korra
```

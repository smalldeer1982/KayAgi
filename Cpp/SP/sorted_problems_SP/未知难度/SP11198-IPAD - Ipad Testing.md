# IPAD - Ipad Testing

## 题目描述

### 题意：

对于任何 $a,b\ (0 \le a,b)$，有多少正整数不能表示为$ax+by$?

其中 $\gcd(x,y)=1$且 $ x,y \ > 1$。

请同时找出所有这些可能整数的最大值。

## 输入格式

**本题有多组数据**

第一行是一个数 $T$ ，表示数据的组数。

接下来的 $T$ 行中，每行包含两个整数 $x,y$。

## 输出格式

对于每个测试数据，输出两个可能的数，它们在同一行里被空格隔开。

## 说明/提示

$T \le 1000$

$1 \ < x,y <10^8$


答案不一定适于32位整数（原OJ截图）：

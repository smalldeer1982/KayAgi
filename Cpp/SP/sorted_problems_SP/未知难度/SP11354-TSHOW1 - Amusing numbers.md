# TSHOW1 - Amusing numbers

## 题目描述

有趣的数字是指只由数字 $5$ 和 $6$ 组成的数字。给定一个整数 $k$，输出第 $k$ 个有趣的数字。

## 输入格式

第一行输入代表测试用例数的整数 $N$ 。

接下来的 $N$ 行输入 $N$ 个整数组成（$1$<=$k$<=$10$^$15$）

## 输出格式

$N$ 行，每行显示对应的第 $k$ 个有趣的数字

## 样例 #1

### 输入

```
2
1
5
```

### 输出

```
5
65
```

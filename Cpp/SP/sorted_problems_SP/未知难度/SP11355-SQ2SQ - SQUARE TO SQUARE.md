# SQ2SQ - SQUARE TO SQUARE

## 题目描述

有些数字是平方数，而且它们的数字可以重新排列后还是一个平方数。我们称这类数字为 S2S 数。

例如：

144 可以重新排列成 441，这两个都是平方数。

## 输入格式

输入一行，包含两个整数 $X$ 和 $Y$，表示你需要找出在 S2S 数列表中第 $X$ 项到第 $Y$ 项的数字。

## 输出格式

输出 S2S 数列表中从第 $X$ 项到第 $Y$ 项的所有数字（包括第 $X$ 项和第 $Y$ 项）。

## 说明/提示

$1 \le X \le Y \le 10^5$。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
\n1 3
```

### 输出

```
\n144\n169\n196\n\n(Xth,Yth term&lt;=10^12)\n(Source Code Limit= 500B) \nNote: 1-&gt;100-&gt;1 or other similar numbers are NOT S2S numbers(ex 9-&gt;900-&gt;9)\nTry Tutorial at -&gt; http://www.spoj.com/problems/SQ2SQ2/\n\n
```

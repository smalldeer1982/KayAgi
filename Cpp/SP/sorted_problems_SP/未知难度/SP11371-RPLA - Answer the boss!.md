# RPLA - Answer the boss!

## 题目描述

Eloy 是一位勤奋的员工，但总是遭到上司的欺负。他对此感到恼火，于是想知道在什么样的“等级”下，员工可以欺负比自己等级低的人，同时也想找出真正能欺负 Eloy 的人。

现在，你需要为他求出每个员工的“等级”。这里，等级 1 代表“老板”（没有人能够欺负他们）。如果多个员工在同一等级，则按编号的字典序升序输出。

### 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。接下来有 $T$ 个测试用例。每个测试用例的第一行包括两个整数 $N$ 和 $R$，分别表示员工数量和关系数量。在接下来的 $R$ 行中，每行有两个整数 $R1$ 和 $R2$，表示员工 $R1$ 的等级低于员工 $R2$。

### 输出格式

对于每个测试用例，先输出一行「情景 #i:」，其中 $i$ 是当前测试用例的编号。然后输出 $N$ 行，每行两个整数，分别表示员工的等级和员工编号。如果有多个员工处于同一等级，则按编号的字典序排列。每组测试用例之间以空行分隔。

### 数据范围与提示

- $1 \leq N \leq 1000$
- $1 \leq R \leq 10000$

请注意，可能会有多个“老板”（不被任何人欺负）。

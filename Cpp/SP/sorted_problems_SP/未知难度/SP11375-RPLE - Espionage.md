# RPLE - Espionage

## 题目描述

小马科斯决定成立并管理一家间谍公司。他创办了公司并担任负责人，乐呵呵地雇佣了一批间谍，用于监视全国的人！然而，他很快发现自己的间谍中有人互相监视，这对他的事业是不利的。他希望有一个程序能够在间谍相互监视的时候提醒他。

## 输入格式

输入包含 $T$ 组测试数据。每组测试数据首先有两个整数 $N$ 和 $R$，分别表示需要评估的人数和监视关系的数量。接下来的 $R$ 行中，每行有两个整数 $R1$ 和 $R2$，表示 $R1$ 正在监视 $R2$。

## 输出格式

输出应包含 $T$ 行。对于每组测试数据，第 $i$ 行以字符串 `Scenario #i: ` 开头，其中 $i$ 是测试数据的编号。如果每个间谍的目标都是平民，输出 `spying`；如果有间谍在监视其他间谍，输出 `spied`。

**输入示例**

```
3
3 2
0 1
2 1
3 3
0 1
2 1
0 2
4 2
2 3
0 1
```

**输出示例**

```
Scenario #1: spying
Scenario #2: spied
Scenario #3: spying
```

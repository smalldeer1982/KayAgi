# RPLH - Hard Launching

## 题目描述

Felix 是一只爱玩的猫，某天下午他决定用一些西瓜做实验。他造了一个“西瓜发射器”，并开始瞄准各种目标。然而他发现，有时西瓜会飞过目标，或者根本达不到目标。

西瓜发射器的角度可以在 0 到 45 度之间任意调整。

Felix 不擅长数学，所以请你帮他写个程序。给定目标的距离（单位：米）和西瓜的发射速度（单位：米/秒），计算出使西瓜击中目标的最佳发射角度。

请注意，重力加速度为 9.806 米/秒²，而 π 的值是 2 * acos(0.0)。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。接下来的 $T$ 行中，每行包含两个整数 $Ta$ 和 $Sp$，分别表示目标距离和西瓜速度。

## 输出格式

对于每个测试用例，输出一行，格式为 “Scenario #i: ”，其中 $i$ 是当前测试用例的编号，后面紧跟一个数字 $D$，表示达到目标所需的发射角度（单位：度）。如果无法完成发射，则将 $D$ 设为 -1。数字需要保留两位小数。

## 说明/提示

$$
1 \le T \le 100, \quad 1 \le Ta, Sp \le 1000
$$

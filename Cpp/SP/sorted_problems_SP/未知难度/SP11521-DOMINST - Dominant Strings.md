# DOMINST - Dominant Strings

## 题目描述

**支配字符串**

给定两个字符串 $s_1$ 和 $s_2$，如果 $s_1$ 中的字符集合（考虑字符出现次数）是 $s_2$ 中字符集合的真超集，则称 $s_1$ 支配 $s_2$。例如，“acmicpc”支配“camp”，但不支配“chimp”或“macpac”。对于字符串集合 $S$，那些不被集合中其他字符串支配的字符串称为 $S$ 的支配字符串（即使它们没有支配集合中的其他字符串）。

你的任务是找出集合中所有的支配字符串。

## 输入格式

输入由若干行字符串组成，每行一个字符串。每个字符串由至少1个、最多10个小写字母组成。集合中最多会有15000个字符串，并且不会出现重复字符串。输入以文件结束标志（EOF）结束。

## 输出格式

输出这些支配字符串，按照字典序排序，每行输出一个字符串。

## 样例 #1

### 输入

```
acmicpc\ncccp\nmacpac\nchimp\ncamp
```

### 输出

```
acmicpc\nchimp\nmacpac
&nbsp;
```

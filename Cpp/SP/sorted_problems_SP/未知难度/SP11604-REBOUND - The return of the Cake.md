# REBOUND - The return of the Cake

## 题目描述

在比特国，有个关于扔蛋糕的有趣现象：反射角不是入射角，而是其两倍！

假设你站在点 A(0, z)，打算将蛋糕扔到 Leo 的脸上。Leo 站在 B(x, y) 而且相当危险，因此你决定用地面的反弹来帮助你击中目标。

要实现这次完美的投掷，你需要选择一个整数 t，使得它落在 \[0, x\] 之间，并能让蛋糕在 C(t, 0) 点反弹。但并不是每次都能成功！

## 输入格式

第一行是一个整数 T，表示测试用例的数量。接下来的 T 行中，每行包含三个整数 x、y 和 z，它们分别表示你的位置 A(0, z) 和 Leo 的位置 B(x, y)。

## 输出格式

对于每个测试用例，找到并输出唯一一个满足 0 ≤ t ≤ x 的整数 t，使得投掷可以成功实现。如果无法做到，请输出 `Not this time.`。

## 说明/提示

- 1 ≤ T ≤ 100
- 1 ≤ x, y, z ≤ 100000

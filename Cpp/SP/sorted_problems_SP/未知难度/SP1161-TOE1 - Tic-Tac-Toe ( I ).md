# TOE1 - Tic-Tac-Toe ( I )

## 题目描述

给出一个9*9的棋盘，判断这个棋盘能否成为井字棋游戏的一部分，其中X先下，O后下。

## 输入格式

第一行输入一个数n,表示输入的棋盘总数。

接下来多行表示棋盘，每两个棋盘之间有一个换行。

## 输出格式

对于每个输入样例，输出'yes'或'no',表示这能否成为井字棋游戏的一部分。

## 样例 #1

### 输入

```
2
X.O
OO.
XXX

O.X
XX.
OOO
```

### 输出

```
yes
no
```

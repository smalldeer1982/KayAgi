# MAKESUM - MAKESUM

## 题目描述

给定 $n$ 个数的所有两两之和（即 $\binom{n}{2}$ 个结果），这些和已经按升序排好。需要从中推断出这 $n$ 个数，并输出。如果存在多个可能的解，请输出字典序最小的解。

要求输出的数必须是自然数。

## 输入格式

输入为 $\binom{n}{2}$ 个整数，这些数均不超过 $10^5$。

满足 $\binom{n}{2} \le 50$。

## 输出格式

输出按字典序排列的最小的一组自然数。

2

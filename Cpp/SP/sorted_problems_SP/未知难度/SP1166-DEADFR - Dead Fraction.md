# DEADFR - Dead Fraction

## 题目描述

题干大意：给你多个无限循环小数的前至多 $18$ 位，要求你输出可能的分母最小的最简分数。

## 输入格式

有多组数据。

每组数据的小数部分至多有 $18$ 个数（不包括省略号），数据以 $0$ 结尾。

## 输出格式

每组数据一行，即输入数据的最简分数形式。

## 样例 #1

### 输入

```
0.2...
0.20...
0.474612399...
0
```

### 输出

```
2/9
1/5
1186531/2500000
```

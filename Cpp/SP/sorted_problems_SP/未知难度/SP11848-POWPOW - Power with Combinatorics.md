# POWPOW - Power with Combinatorics

## 题目描述

给定一个计算公式，你需要计算 $a^{(\text{exp}^b)}$ 的值，其中：

- $a$ 和 $b$ 是输入提供的整数，满足 $0 \le a, b \le 10^5$。
- $\text{exp}$ 是通过以下公式计算得到的：$\text{exp} = \sum_{r=0}^{n} \binom{n}{r}^2$，这里的 $n$ 也由输入提供，满足 $0 \le n \le 10^5$。

需要注意的是，由于结果可能非常大，因此最后的答案需要对 $10^9 + 7$ 取模。

在此，$\binom{n}{r}$ 代表从 $n$ 个元素中选择 $r$ 个元素的组合数。

## 输入格式

输入第一行包含一个整数 $t$，表示测试用例的数量（$1 \le t \le 1000$）。

接下来有 $t$ 行，每行包含三个用空格分隔的整数 $a, b, n$。

## 输出格式

输出由 $t$ 行组成，每一行对应一个测试用例的结果。

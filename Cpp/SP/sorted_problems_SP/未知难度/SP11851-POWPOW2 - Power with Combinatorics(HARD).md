# POWPOW2 - Power with Combinatorics(HARD)

## 题目描述

你需要计算以下表达式的值： \(a^{(b^{\text{exp}})}\)。

- 参数 \(a\)：从输入中读取，满足 \(0 \le a \le 10^5\)
- 参数 \(b\)：从输入中读取，满足 \(0 \le b \le 10^5\)
- \(\text{exp} = (nC0)^2 + (nC1)^2 + (nC2)^2 + \cdots + (nCn)^2\)，这里 \(nCr\) 表示从 \(n\) 个元素中选取 \(r\) 个元素的组合数。
- 参数 \(n\)：从输入中读取，满足 \(0 \le n \le 10^5\)

特别注意：当计算 \(0^0\) 时，结果应为 1。

由于计算结果可能非常大，最后需要对结果取模 \(10^9 + 7\)。

## 输入格式

第一行输入一个整数 \(t\)，表示测试用例的数量（\(t \le 1000\)）。

接下来有 \(t\) 行，每行依次包含三个整数 \(a\)、\(b\) 和 \(n\)，整数之间用空格分隔。

## 输出格式

输出结果包含 \(t\) 行，每一行对应一个测试用例的答案。

 **本翻译由 AI 自动生成**

### 输入

```
1 1 1
```

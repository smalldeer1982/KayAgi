# BFTRI - Drawing Triangles with Brainf##k

## 题目描述

现在你的任务是画三角形！听起来挺简单的。不过，你只能用Brandfuck语言来编写这个程序。

## 输入格式

第一行是一个整数 $T$ ，表示测试用例的数量

每个测试用例包含一个字符 $c$ 和一个整数 $i$ ，两者用空格分开。

## 输出格式

对于每个测试用例，输出长和高为 $i$ 三角形 （用字符 $c$ 绘制）。

## 样例 #1

### 输入

```
3
&amp; 1
@ 3
? 5
```

### 输出

```
&amp;
```

# PK11I - Paths in a Tree

## 题目描述

给定一棵树（即无环的连通图），树中的边具备方向性。你的任务是添加尽可能少的特殊路径，确保从任意节点都可以到达其他任意节点。特殊路径需满足以下规则：

1. 特殊路径由树中的一些连续边和节点构成。
2. 在特殊路径中，每条边的方向必须与树中原有的方向相反。
3. 在一个特殊路径中，每个节点或边至多只能被访问一次。
4. 不同的特殊路径可以共享节点或边。

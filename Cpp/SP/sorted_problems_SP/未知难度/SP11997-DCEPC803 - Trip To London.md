# DCEPC803 - Trip To London

## 题目描述

Mary 和 Saina 在 2012 年伦敦奥运会上为印度赢得了奖牌。庆祝之后，她们计划在伦敦旅游，因为这是她们第一次来到这里，所以她们买了一张路线图。然而，这张地图并不完整，缺少了一条路线的信息。尽管她们向旅游部门询问，得到的仍只是最短路径的地图，而非原始的直接路线图。Mary 和 Saina 更愿意选择直接路线，而不是最短路线。你能帮助她们计算出原始地图上未知的直接距离可能有多少种取值吗？假设任意两地之间的直接距离不超过 100 公里。

## 输入格式

第一行包含一个整数 $N$，表示在伦敦的地点数量。

接下来的 $N$ 行中，每行包含 $N$ 个整数，表示直接距离矩阵。第 $i$ 行第 $j$ 列的值要么是一个非负整数（表示从第 $i$ 个地方到第 $j$ 个地方的直接距离），要么是 -1（表示该信息缺失）。

再接下来的 $N$ 行中，每行同样包含 $N$ 个整数，表示最短路径矩阵。第 $i$ 行第 $j$ 列的值是一个非负整数（表示从第 $i$ 个地方到第 $j$ 个地方的最短路径距离）。

请注意，从第 $i$ 个地方到第 $j$ 个地方和从第 $j$ 个地方到第 $i$ 个地方的直接距离可能不同，并且一个地方到自身的距离总是 0。

## 输出格式

输出原始地图中被标记为 -1 的缺失直接距离可能的不同取值数量，假设最短路径矩阵数据正确。

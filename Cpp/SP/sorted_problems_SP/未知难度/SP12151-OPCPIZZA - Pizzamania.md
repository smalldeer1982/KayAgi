# OPCPIZZA - Pizzamania

## 题目描述

辛哈姆和他的朋友们都非常喜欢披萨，但这次他们资金紧张，所以决定相互帮助。他们打算两人一组去买披萨。我们的任务是找出在给定披萨价格的情况下，可能的组合数。因为卖披萨的小哥没有零钱找零，如果一对朋友的钱加起来超过所需金额，他们也无法买到披萨。值得注意的是，每个朋友手里的钱数都是不同的。此外，在辛哈姆的世界里，资金的数值可以为负数。

## 输入格式

第一行输入一个整数 $t$（$1 \le t \le 100$），表示测试用例的个数。接下来的 $2 \times t$ 行中，每个测试用例包含两行数据。第一行为两个整数 $n$ 和 $m$，其中 $n$（$1 \le n \le 100000$）表示辛哈姆的朋友数量，$m$ 表示每份披萨的价格。第二行为 $n$ 个用空格分隔的整数，表示每个朋友各自拥有的钱数。

$m$ 和这些钱数值均在 C 和 C++ 的 int 类型范围内。

## 输出格式

输出一个整数，表示有多少对朋友可以凑够钱买披萨。

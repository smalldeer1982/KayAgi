# ROULETTD - Natalia Spins The Roulette

## 题目描述

La Vega 轮盘赌类似于普通的轮盘赌。它是一个圆形，圆周上写有数字。与常见的轮盘不同的是，所有数字的颜色相同，并且在轮盘的正上方有一个指示标记。

Natalia 和 Luis 感到无聊，于是决定使用 La Vega 轮盘赌玩一个叫做“旋转大战”的游戏。在游戏中，玩家首先将轮盘转动，使第一个数字位于指示标记下方。然后，她旋转轮盘并记录指示标记所在的数字，同时记录轮盘的位置。在后续回合中，玩家需要先将轮盘转回到上次回合结束时的位置，再旋转轮盘，将新的指示数字加到当前总和中，并再次记录位置。

两名玩家各轮流进行回合后，必须决定是否继续游戏或宣布比赛结果。设玩家 1 和玩家 2 的总和分别为 $S_1$ 和 $S_2$。如果 $S_1 - S_2 > K$，则玩家 1 获胜，游戏结束；如果 $S_2 - S_1 > K$，则玩家 2 获胜；如果 $S_1 - S_2 = 0$，则游戏为平局并立即结束。如果以上条件都不满足，游戏继续进行。

你需要根据每个回合中轮盘旋转的次数，输出比赛结果。

## 输入格式

第一行输入包含两个用空格分隔的整数 $N$（$2 \le N \le 10^5$）和 $K$（$1 \le K \le 10^9$），分别表示轮盘上的数字数量和获胜所需的差值 $K$。接下来的两行每行包含 $N$ 个用空格分隔的整数，分别表示在每个回合后 Natalia 和 Luis 的轮盘旋转次数。

## 输出格式

输出比赛结果。如果玩家 1 获胜，输出 `P1`。如果玩家 2 获胜，输出 `P2`。如果比赛平局，输出 `BAD LUCK`。

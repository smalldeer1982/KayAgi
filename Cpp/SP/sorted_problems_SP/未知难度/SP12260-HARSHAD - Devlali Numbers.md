# HARSHAD - Devlali Numbers

## 题目描述

Devlali 数是由印度数学爱好者 D. R. Kaprekar 提出的一个有趣的数学概念。

对于每个正整数 $n$，我们定义 $d(n)$ 为 $n$ 加上其每一位数字之和。例如，对 $n = 199$ 而言，有 $d(199) = 199 + 1 + 9 + 9 = 218$。

若一个正整数 $m$ 无法通过任何正整数 $r$ 使得 $d(r) = m$，那么 $m$ 就被定义为 Devlali 数。早期的几个 Devlali 数是 1, 3, 5, 7，等等。

在 Devlali 数中，如果某个数本身是质数，则称其为 Devlali 质数。前几个 Devlali 质数包括 3, 5, 7，等等。

## 输入格式

第一行输入一个整数 $Q$，表示测试次数。

接下来的 $Q$ 行，每行包含两个整数 $A$ 和 $B$。

## 输出格式

对于每组输入，你需要输出 $Q$ 行结果，表示在区间 $[A, B]$（包含 $A$ 和 $B$）内有多少个 Devlali 质数。

## 数据范围

$$1 \le Q \le 10^5$$

$$1 \le A \le B \le 10^5$$

# RANDG - Red And Green

## 题目描述

**红与绿**

有若干个正方形排成一行，当前每个正方形被涂成了红色或绿色。你可以选择任意一个正方形，重新涂成红色或绿色。目标是经过重新涂色后，红色的正方形全部位于绿色正方形的左边。我们希望你以最少的重新涂色次数达到该目标。

这些正方形从左到右排列。我们会给出一个字符串，表示初始排列，其中字符 $i$ 为 'R' 表示第 $i$ 个正方形是红色，'G' 表示是绿色。你需要输出达到目标所需的最小涂色次数。

### 输入格式

输入包含若干组测试数据。每组测试数据是一行长不超过 50 个字符的字符串。输入以 EOF 结束。

### 输出格式

对于每组测试数据，输出一行代表结果。

### 数据范围与提示

- 输入长度在 1 到 50 个字符之间。
- 每个字符都是 'R' 或 'G'。

### 样例输入

```
RGRGR
RRRGGGGG
GGGGRRR
RGRGRGRGRGRGRGRGR
```

### 样例输出

```
2
0
3
8
```

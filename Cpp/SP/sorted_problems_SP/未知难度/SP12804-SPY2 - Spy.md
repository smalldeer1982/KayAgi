# SPY2 - Spy

## 题目描述

Blue Mary 十分热衷于制作 PPT。她已经完成了 **L** 个 PPT，现在唯一剩下的工作就是为每一个 PPT 设置背景图片。Mary 总共有 **N** 张背景图片可供选择，每个 PPT 需要使用一张背景图片，不同的 PPT 可以选择同一张背景图片。这意味着有 **$N^L$** 种可能的组合。

对于每种背景图的组合，Mary 给它加权：权重定义为 **$(k+1)^{-1}$**，其中 **k** 表示在这组组合中没有被使用的图片数量（共有 **N** 张图片）。现在，Mary 希望你帮她计算出所有组合的权重总和。Mary 是一个比较奇特的女孩，时常进行这些看似无意义的计算。

## 输入格式

输入包含多组测试数据，最多有 500 组。每组测试由一行两个整数 **N** 和 **L** 组成，分别用空格分开。所有输入的数字都是正整数且小于 $10^6$。输入直到文件结束（EOF）。

给定的数据是接近对数均匀随机生成的。

## 输出格式

对于每组测试数据，输出计算结果，每个结果单独占一行。保证计算结果对于所有输入数据都是一个整数。由于可能计算出非常大的数，所以结果需对 $10^9 + 2015$ 取模。（为什么不是 $10^9 + 7$？因为 Blue Mary 是个比较特别的女孩！）

 **本翻译由 AI 自动生成**

### 输入

```
2 2
```

### 输出

3

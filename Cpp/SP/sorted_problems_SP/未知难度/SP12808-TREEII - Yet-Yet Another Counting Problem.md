# TREEII - Yet-Yet Another Counting Problem

## 题目描述

计算有 $n$ 个节点的有根树数量，要求满足以下条件：

如果节点 $A$ 和节点 $B$ 到根节点的距离相同，那么 $A$ 和 $B$ 必须拥有相同数量的直接子节点。

当且仅当通过 $n$ 个节点的双射可以将一棵树映射为另一棵树时，这两棵树被认为是相同的。

由于结果可能非常大，请输出答案对 $1000000007$ 取模后的余数。

## 输入格式

每个测试用例由一行组成，包含一个整数 $n$，满足 $1 \le n \le 1000$。你需要处理直到文件结束。

## 输出格式

对于每个测试用例，输出一行结果。具体格式请参见示例。

## 说明/提示

输入中的整数 $n$ 满足 $1 \le n \le 1000$。

# MULTII - Yet Another Multiple Problem

## 题目描述

给定一个正整数 **N**（$1 \le N \le 10000$）和一些禁止使用的数字，要求找出最小的正整数 **N** 的倍数，使其十进制表示中不包含这些禁止的数字。

## 输入格式

每个测试用例包含两行。第一行有两个整数 **N** 和 $m$，用空格隔开。第二行为 $m$ 个空格分隔的数字。输入一直读到文件结束。

## 输出格式

对每个测试用例，输出格式为“第X个案例：结果”，如果没有符合条件的倍数存在，则输出“第X个案例：-1”。案例编号从1开始。

## 说明/提示

- $1 \le N \le 10000$
- $1 \le m \le 9$

### 输入

```
2345 3
7 8 9
100 1
0
```

### 输出

```
Case 1: 2345
Case 2: -1
```

# VPL0_C - Collision on Christmas Eve

## 题目描述

在拿到圣诞礼物之前，Danny 必须完成以下问题：给定两个整数 $N$ 和 $K$，在如下数列中找出因数最多的那个数：

- $A_0 = 1$
- $A_1 = A_0 + K + 1$
- $A_2 = A_1 + K + 2$
- ...

并且要求 $A_n$ 的值不能超过给定的 $N$。

## 输入格式

第一行是一个整数 $T$，表示测试用例的数量。接下来每个测试用例用两行给出，分别是整数 $N$ 和 $K$，表示数值上限和数列中的常数参数。

## 输出格式

对于每个测试用例，输出格式为：`Scenario #i: `，其中 $i$ 是当前测试用例编号（从 1 开始），后面跟着的是那个因数最多的数及其因数数量。如果发现多个数具有相同的因数数量，则选择最小的那个数。

## 说明/提示

- $1 \le T \le 100$
- $1 \le N \le 10^5$
- $0 \le K \le 10^3$

### 输入示例

```
4
4 0
28 1
2 2
78 3
```

### 输出示例

```
Scenario #1: 4 3
Scenario #2: 28 6
Scenario #3: 1 1
Scenario #4: 40 8
```

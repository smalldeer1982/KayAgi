# SGAME - SHAPE GAME

## 题目描述

你可能玩过一个不用抬笔就能画出完整图形的游戏。不过，这听起来似乎对我们太简单了。  
让我们增加一些挑战！  
从某个点开始构建一个图形，并最终回到起始点，期间不能抬笔。

注意：图形是封闭的，不能重复走过某条弧线或直线。

## 输入格式

输入包含若干组测试数据。首行有一个整数 $t$，表示共有 $t$ 组测试用例。每组测试用例：
- 第一行为整数 $n$，表示图形起点和终点的索引。
- 接下来多行，每行包含两个用空格分隔的整数 $i$ 和 $j$，表示从点 $i$ 到点 $j$ 的一条边，直到遇到 `-1 -1` 结束。

## 输出格式

如果能构造出符合条件的图形，输出 `YES`；否则输出 `NO`。

## 说明/提示

- $t \leq 100$
- $1 \leq n \leq 300$
- $1 \leq i, j \leq 300$
- $i \neq j$

**样例**

```
输入：
1
1
1 2
1 4
2 3
2 5
2 6
3 6
4 7
5 6
6 7
-1 -1

输出：
YES
```

# PCPC12J - Amr Samir

## 题目描述

Amr 刚刚开始学习除法。他给朋友提供了一组数字列表，并让朋友帮忙找出每个数字的所有因数。之后，Amr 生成了一个新列表，其中包含了原始列表中所有数字的因数。

## 输入格式

输入由多组数据组成。
每组数据的第一行是一个整数 $n$，代表数字列表的长度。
接下来的行中为 $n$ 个整数 $a_1, a_2, \ldots, a_n$，它们构成了 Amr 提供的数字列表。

## 输出格式

对于每组数据，输出一行，包含找到的所有因数。因数应按从小到大的顺序排列，且以空格分隔。

- $1 \le n \le 100$
- $1 \le a_i \le 10^6$

# ADITYA13 - FizzBuzz Happy 2013

## 题目描述

在成功完成了 ACM ICPC 在线赛后，你得到了一份教比特兰高中生学习 Java 编程的工作机会。你愉快地接受了这份工作。教学数日后，你布置给学生们一个编程任务，让他们编写一个 [FIZZBUZZ](http://en.wikipedia.org/wiki/Fizz_buzz) 程序。学生们要么聪明，要么懒惰，他们抱怨这个任务太难，并要求你在不使用任何英文字母的情况下完成类似的程序。于是，你去请教资深程序员 [Mitch](../../../users/cyclops/)，他建议你使用 BF 语言解决这个问题。[Tjandra](../../../users/tjandra/) 则建议你可以用 Whitespace 语言。

## 输入格式

输入包括三部分：
- 第一行：两个由空格分隔的一位整数 $A$ 和 $B$。
- 第二行：一个最多包含 5 个字符的字符串 $C$。
- 第三行：一个最多包含 5 个字符的字符串 $D$。

题目保证 $\gcd(A, B) < A$ 且 $\gcd(A, B) < B$。此外，所有字符串仅由字母和数字组成。

## 输出格式

对于从 1 到 99 的每个数字，按照以下规则输出：
- 如果数字能被 $A$ 整除，输出字符串 $C$；
- 如果能被 $B$ 整除，输出字符串 $D$；
- 如果同时能被 $A$ 和 $B$ 整除，输出字符串 $C$ 和 $D$ 的连接；
- 如果既不能被 $A$ 也不能被 $B$ 整除，输出数字本身，保证两位数格式（不足两位数时前面补零）。

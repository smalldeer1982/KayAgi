# STICKS - Magic Sticks

## 题目描述

我们有N根棍子排成一排，它们之间的距离不同，每根棍子都靠在左边的棍子上（见图）。  
L：棍子的长度。  
D：杆头与左侧杆头之间的距离。          
我们抓住第一根棍子（长度L0）并保持垂直。            
确定如果第一根棍子移开，预计所有棍子掉落的总时间（假设每个棍子的底部头部不会移位，当棍子到达地面时，它会消失，右侧的棍子开始掉落）。            
在时间=1秒时（如果我们假设棍子不会停止并保持相同的运动类型，则从棍子开始落下），您将获得V（杆头的速度）。   
假设力在方向和值上对棒有稳定的影响

## 输入格式

第一行：N，L0                                                     
接下来N-1行：Lk，Dk，Vk（1）

## 输出格式

总时间（毫秒为单位）

## 样例 #1

### 输入

```
2 8.75\n10 8.7 25000
```

### 输出

```
2
\n\n\n\n2 &lt;= N &lt;= 5000\n1 &lt;= L &lt;= 1000000\n1 &lt;= V &lt;= 1000000\n\n.\n
```

# CHASE - A Chase In WonderLand

## 题目描述

给定平面上的 $k$ 个点 $(0 < k \le 2200)$，求一条直线能通过的最多点数，请输出这个答案 **减一**。

## 输入格式

先输入一个整数 $n$ ，表示有 $n$ 组测试数据。

对于每一组测试数据，第一行有一个整数 $k$。接下来的 $k$ 行，每行两个整数，描述这个点的位置。

## 输出格式

输出 $n$ 行，每行一个整数，表示所求答案。

## 样例 #1

### 输入

```
2
5
0 0
1 1
2 2
4 8
2 75
3
0 0
1 2
3 4
```

### 输出

```
2
1
```

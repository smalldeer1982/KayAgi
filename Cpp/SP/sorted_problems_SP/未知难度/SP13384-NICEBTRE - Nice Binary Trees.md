# NICEBTRE - Nice Binary Trees

## 题目描述

例如，以下的树是一个「好」二叉树。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。接下来有 $T$ 行，每行是一个由字母组成的字符串，表示一棵「好」二叉树的前序遍历。字符串中的字母 `l` 代表叶子节点，`n` 代表内部节点。可以保证输入的每个字符串都是某棵「好」二叉树的前序遍历。

## 输出格式

对于每个测试用例，输出一个整数，表示对应二叉树的深度。

## 说明/提示

- $1 \le T \le 10^5$

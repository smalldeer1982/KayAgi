# PRLOVE - Expected Time to Love

## 题目描述

Alice 遇到了一点麻烦。她深爱着 Bob，但羞于面对自己真实的情感。因此，她决定通过互联网从自己的电脑发送一封信给 Bob，以表达心意。

这个互联网网络由 $N$ 台电脑组成，每台电脑的编号从 $1$ 到 $N$。Alice 的电脑编号为 $1$，而 Bob 的电脑编号为 $N$。

然而，由于一些编程错误，网络中的电脑开始出现异常行为。当电脑 $i$ 接收到文件时，会以概率 $P_{ij}$ 将文件转发给电脑 $j$。从电脑 $i$ 转发文件到电脑 $j$ 所需的时间为 $T_{ij}$。

请你帮助计算 Bob 接收到 Alice 情书的期望时间。

**注意：** 一旦 Bob 的电脑接收到信件，就会直接将其交给 Bob，不再进行转发。

## 输入格式

第一行包含整数 $T$，表示测试用例的数量。

每个测试用例如下：

- 第一行包含整数 $N$，表示网络中电脑的总数。
- 接下来的 $N$ 行，每行有 $N$ 个整数。第 $i$ 行的第 $j$ 个数表示 $P_{ij}$ 的值（百分比）。
- 接着的 $N$ 行，每行有 $N$ 个整数。第 $i$ 行的第 $j$ 个数表示 $T_{ij}$ 的值。

## 输出格式

输出一个实数，表示预期的传输时间。

如果每个数字的绝对误差或相对误差小于 $10^{-6}$，则认为输出是正确的。

## 说明/提示

- $1 \le T \le 10$
- $2 \le N \le 100$
- $0 \le P_{ij} \le 100$
- $0 \le T_{ij} \le 1000$

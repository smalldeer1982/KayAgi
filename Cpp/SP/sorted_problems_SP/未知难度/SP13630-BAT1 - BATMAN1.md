# BAT1 - BATMAN1

## 题目描述

卢修斯·福克斯：这种对话通常以一个不寻常的要求结束。

## 输入格式

第一行有一个整数t，代表测试数据的组数。

第二行有三个整数：n,m,k。

n代表一共有几批武器装备，m代表每批武器装备的数量，k代表布鲁斯韦恩（即蝙蝠侠）可以在武器装备上花费的金钱。

接下来的n*m个整数代表第i批第j个武器装备的价格。

再接下来的n*m个整数代表第i批第j个武器装备的威力值。

## 输出格式

布鲁斯韦恩可以支付的武器装备所能提供的最大威力值。

## 说明/提示

1<=n,m<=20

k<=1000

物品的价格<=20,物品的威力值<=100

## 样例 #1

### 输入

```
1
2 4 20
3 4 
3 2 3 2
3 2 3 5
3 2 3 2
4 5 6 5
```

### 输出

```
40
```

# BAT2 - BATMAN2

## 题目描述

“阿福：我会把这个交给福克斯，但到此为止。我把你带大成人，我帮你疗伤续骨，我不会为你下葬。我已经埋葬过太多韦恩家族的人了。”

## 输入格式

第一行t表示数据的组数。

第二行n表示房间的数量。

第三行的n个整数表示房间之间的标签关联。

## 输出格式

可以搜索到的最大房间数量。

## 说明/提示

1<=n<=100

## 样例 #1

### 输入

```
1
6
5 3 4 6 1 2
```

### 输出

```
5
```

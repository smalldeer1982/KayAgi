# MAX_NUM - finding maximum possible number

## 题目描述

给定一个数字 $n$，通过精确删除 $k$ 个数字，找出最大可能的数字。

如果 $n$ 的值等于 $k$ ，则只需打印一行新行。

## 输入格式

$T$ ，$n$ 和 $k$。

## 输出格式

一行，最大可能数。

## 说明/提示

$T$ 为测试用例数（$1\le t \le 10^3$）。

$1 \le n$ 中的位数 $\le 10^5$。**（n可能包含前导 $0$）**。

$0 \le$ $k \le$ $n$。

## 样例 #1

### 输入

```
2
1223 2
8756 2
```

### 输出

```
23
```

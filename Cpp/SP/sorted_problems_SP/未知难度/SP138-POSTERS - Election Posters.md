# POSTERS - Election Posters

## 题目描述

### 题意翻译
你有 $n$ 张海报。所有海报都从上到下延伸，但悬挂在墙上的不同位置，宽度也可能不同。墙壁水平分为多个部分，海报完全占据两个或多个相邻部分。

随着时间的推移，一些海报（部分或全部）被其他海报覆盖。你知道所有海报的位置和悬挂顺序，需要确定最后有多少海报至少有一个可见部分。

## 输入格式

$t$ 组数据，每组数据的第一行包含一个正整数 $n$（$1\le n\le 40000$），表示海报的数量，然后 $n$ 行，每行两个正整数 $l$，$r$（$1\le l<r\le 10^7$），表示每张海报的左端和右端。

## 输出格式

每组数据一行，表示有多少海报至少有一个可见部分。

## 样例 #1

### 输入

```
1
5
1 4
2 6
8 10
3 4
7 10
```

### 输出

```
4
```

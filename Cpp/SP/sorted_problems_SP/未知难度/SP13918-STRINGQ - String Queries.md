# STRINGQ - String Queries

## 题目描述

给定一个由小写字母组成的字符串 $S$。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。

对于每个测试用例：
- 第一行为字符串 $S$。
- 第二行为一个整数 $N$，表示查询的数量。
- 接下来的 $N$ 行中，每行是一个形如 "X a b" 的字符串，其中 $X$ 是 `Q` 或 `U` 之一，$a$ 和 $b$ 是满足 $1 \leq a \leq b \leq |S|$ 的正整数。

## 输出格式

对于每个测试用例，输出 $X+1$ 行，其中 $X$ 为类型 `Q` 的查询数量。

- 对于每个类型为 `Q` 的查询，输出一个数字作为结果。
- 最后一行输出更新后的字符串 $S$。

## 说明/提示

- $1 \leq T \leq 100$
- $1 \leq |S| \leq 10^5$
- $1 \leq N \leq 10^3$

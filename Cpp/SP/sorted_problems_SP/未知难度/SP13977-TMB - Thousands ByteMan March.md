# TMB - Thousands ByteMan March

## 题目描述

Leo 邀请了他的所有朋友参加比特国的和平大会。所有来宾都乘坐满员的大巴车到达。去年只来了 4 个人，分别是 A、B、C 和 D。Leo 喜欢有条理的事物，所以他想把这些人分成小组。去年的小组划分方式如下：

```
{{A,B,C,D}}：一个 4 人小组（1 种方式），
{{A}, {B}, {C}, {D}}：四个 1 人小组（1 种方式），
{{A,B}, {C,D}}，{{A,C}, {B,D}}，{{A,D}, {B,C}}：两个 2 人小组（各 1 种方式，共 3 种方式）。
```

合计有 5 种分组方式。今年，预计会有更多人参加。

## 输入格式

第一行输入一个整数 $T$，表示测试用例数量。接下来的 $T$ 行中，每行包含两个整数 $N$ 和 $K$。这里，$N$ 表示到达会议的大巴车数量，$K$ 表示每辆大巴车的乘客容量。

## 输出格式

对于每个测试用例，需要计算出人员可以组成均匀小组的总数。由于答案可能非常大，可能无法存储在一个 64 位的数据类型中。

## 说明/提示

- $1 \le T \le 10^5$
- $1 \le N, K \le 10^5$

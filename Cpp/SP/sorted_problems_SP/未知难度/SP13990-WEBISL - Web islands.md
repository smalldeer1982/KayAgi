# WEBISL - Web islands

## 题目描述

给定一组网页，我们希望找到一些尽可能大的子集，使得在这些子集中，任意一个网页都能通过链接到达子集内的其他所有网页。

## 输入格式

第一行包含两个整数 $N$ 和 $M$，分别代表网页的数量和链接的总数。网页的编号从 0 编号到 $N-1$。接下来的 $M$ 行中，每行包含两个整数，第一个整数表示链接的起始网页编号，第二个整数表示链接的目标网页编号。

## 输出格式

输出共 $N$ 行，每行对应一个整数，表示每个网页所属于的连通分量的编号，该编号为该连通分量中最小的网页编号。

## 说明/提示

1 ≤ N, M ≤ 100,000

## 样例 #1

### 输入

```
3 3\n0 1\n1 0\n1 2\n\n
```

### 输出

```
0\n0\n2
```

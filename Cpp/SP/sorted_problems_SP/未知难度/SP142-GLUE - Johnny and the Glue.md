# GLUE - Johnny and the Glue

## 题目描述

给出 $n$ 个点，求能覆盖全部点，且每条边至少接触两个点的可能的矩形数量，矩形可以斜着摆放。

## 输入格式

第一行一个正整数 $t$ 表示测试数据组数。

每组测试数据的第一行为正整数 $n$ $(n<=10000) $。接下来的 $n$ 行每行包含两个整数 $x_i y_i$ $(-15000<=x_i,y_i<=15000)$，表示每个点的坐标。

## 输出格式

对于每组测试数据，输出一行一个数，表示可能的矩形摆放数量。

## 样例 #1

### 输入

```
1 
8 
1 0 
1 4 
0 3 
5 4 
5 0 
6 1 
6 3 
0 1
```

### 输出

```
2
```

# KPPOLY - Projections Of A Polygon

## 题目描述

给定一个笛卡尔坐标系中的凸多边形。这个多边形在 X 轴和 Y 轴上各有一个投影。你可以任意旋转这个多边形。你需要求出这个多边形的投影长度之和的最小值和最大值。

## 输入格式

第一行输入一个整数 $N$（$3 \le N \le 100$），表示多边形的顶点数。接下来的 $N$ 行中，每行包含两个整数，分别是顶点的坐标 $X_i$ 和 $Y_i$ 。所有顶点的坐标都是整数，按顺时针或逆时针顺序给出。没有顶点重合，任意三个连续顶点都不在同一条线上。坐标的绝对值均不超过 $10000$。

## 输出格式

输出两个浮点数，分别为多边形投影和的最小值和最大值。两个数用空格分隔。答案与正确值的误差不应超过 $10^{-6}$。

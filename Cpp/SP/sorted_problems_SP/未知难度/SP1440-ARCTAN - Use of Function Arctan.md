# ARCTAN - Use of Function Arctan

## 题目描述

很容易知道 $\arctan(\frac{1}{2})+\arctan(\frac{1}{3})=\arctan(1)$。给定一个数字 $A$，编写一个程序计算 $B+C$ 的最小值。$A$、$B$、$C$ 都是正整数且满足如下的等式：
$$
\arctan(\frac{1}{a})=\arctan(\frac{1}{b})+\arctan(\frac{1}{c})
$$

## 输入格式

第一行包含一个正整数 $T(1\le T\le 1000)$，后面 $T$ 行中每一行包含一个整数 $a(1\le a\le 60000)$。

## 输出格式

$T$ 行，每行包含一个表示 $B+C$ 最小值的整数。

## 样例 #1

### 样例输入 #1

```
1
1
```

### 样例输出 #1

```
5
```

## 样例 #1

### 输入

```
1
1
```

### 输出

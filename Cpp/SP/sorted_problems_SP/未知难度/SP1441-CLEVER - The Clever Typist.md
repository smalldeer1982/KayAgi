# CLEVER - The Clever Typist

## 题目描述

玛丽是某个秘密部门的打字员。现在她必须在一小时内输入非常多的密码，每个密码的长度都为6个字符。并且她必须保证在按键盘的次数最少时完成。

输入密码的键盘是专门为保密而设计的。键盘有6个特殊的键，而不是10个数字键。从左到右定义屏幕上的6个位置1,2,3,4,5,6。这些键的操作如下所示：

操作1：交换光标位置的数字和位置1的数字。光标不移动。如果光标现在位于位置1，屏幕上的数字将不会更改。

操作2：交换光标位置的数字和位置6的数字。光标不移动。如果光标现在位于位置6，屏幕上的数字将不会更改。

操作3：将光标位置的数字增加1。如果光标位置的数字为9，则不会发生任何更改。

操作4：将光标位置的数字减少1。如果光标位置的数字为0，则不会发生任何更改。

操作5：将光标向左移动一个位置。如果光标位于位置1，则不会发生任何更改。

操作6：将光标向右移动一个位置。如果光标位于位置6，则不会发生任何更改。

## 输入格式

第一行包含一个整数t（0<t<=1000）。接下来的t行包含两个6位数的字符串，每个字符串在屏幕开始时显示数字以及玛丽要输入的密码，用一个空格分隔。

## 输出格式

t行，每行包含一个整数，即最少操作的次数。

## 样例 #1

### 输入

```
1
123456 654321
```

### 输出

```
11
```

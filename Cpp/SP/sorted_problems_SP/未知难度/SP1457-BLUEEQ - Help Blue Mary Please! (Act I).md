# BLUEEQ - Help Blue Mary Please! (Act I)

## 题目描述

## 题意翻译
有一些方程式，仅由加号，乘号及十个阿拉伯数字组成。现在用一些$ a$~$m$的字母替换了部分字符，保证同种字符用同种字母替换。数据保证原方程式合法，即有且仅有一个等号且不在末尾。请由给定的替换后的字母串确定部分能够确定的字母代表的字符。

## 输入格式

* 先输入一个$t$,表示数据组数。
* 对每一组数据，第一行一个字母$n$,下面$n$个字符串，表示替换后的串。

## 输出格式

按顺序输出能确定的字母代表的数字。具体格式见样例.

## 说明/提示

* 对$90$%的数据，保证$n≤5$
* 对$80$%的数据，保证$n≤2$
* 对$70$%的数据，保证至多有$5$个未知数。

## 样例 #1

### 输入

```
1
2
abcdec
cdefe
```

### 输出

```
a6
b*
d=
f+
```

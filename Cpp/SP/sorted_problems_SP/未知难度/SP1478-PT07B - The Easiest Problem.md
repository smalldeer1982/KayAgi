# PT07B - The Easiest Problem

## 题目描述

给定一棵无权无向树 $T$。我们称这棵树是“特殊树”，如果它满足以下条件：

“所有度数大于等于 $3$ 的节点，其周围最多有两个度数大于等于 $2$ 的节点。”

请找到这棵树的一个最大子树，使其成为一个“特殊树”。

## 输入格式

输入的第一行是一个整数 $N$ —— 表示树中的节点数量（$0 < N \leq 10^6$）。接下来的 $N-1$ 行每行给出一条边的信息，包含两个整数 $(u, v)$，表示节点 $u$ 和节点 $v$ 之间存在一条边（$1 \leq u, v \leq N$）。

## 输出格式

第一行输出所找到的最大“特殊树”的节点数量。接下来的若干行，每行输出这棵子树中的一条边，用两个整数 $u$ 和 $v$ 表示节点 $u$ 和 $v$ 之间有一条边。

## 说明/提示

节点数量 $1 \leq N \leq 10^6$。

### 输入

5

```

```
5
1 2
2 3

2 5

# KIMO1 - abdou set

## 题目描述

阿卜杜拥有一组不同的正整数。他想向这个集合中添加若干个（可能为零）新的正整数，目的是使得在对集合排序后，对于任意两个相邻的数 $X$ 和 $Y$，都有 $\text{abs}(X \mod m - Y \mod m) = 1$。你的任务是计算他至少需要添加多少个新数，才能达到这样的要求。

## 输入格式

输入包含多组测试数据。

- 每组数据的第一行有两个整数 $n$ 和 $m$，分别表示初始集合中的元素个数和模数 $m$，范围为 $1 \le n, m \le 10^5$。
- 随后的一行为 $n$ 个不同的正整数 $a_1, a_2, \ldots, a_n$，其中每个数 $a_i$ 满足 $1 \le a_i \le 10^9$，代表初始集合中的元素。

## 输出格式

对于每组测试数据，输出一行，包含需要添加的最小新数的数量。

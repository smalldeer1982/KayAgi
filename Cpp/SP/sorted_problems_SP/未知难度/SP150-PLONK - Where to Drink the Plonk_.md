# PLONK - Where to Drink the Plonk?

## 题目描述

一个以正方形为边界的城市，其$n$ 条水平和$n$ 条垂直街道将其划分为$(n+1)^2$ 个正方形街区。这个城市的所有居民都生活在十字路口。一群朋友想在他们其中一人的住处聚会，共度一个欢乐的夜晚。考虑到到回到各自家里可能步行距离又长又短，所以请你算出每个人去聚会的最短路径，并算出所有人的路径总和。

PS：每个人都沿着街道走，只在十字路口转弯，任何相邻十字路口之间的距离为$1$。

## 输入格式

第一行，为一个整数$t$ ，表示有几组数据。

对于每组数据，第一行包含一个正整数$n$ ，表示有$n$ 个朋友希望见面。

接下来$n$ 行，为每个朋友的所在地。

## 输出格式

$t$ 行，为每组数据的答案。

## 样例 #1

### 输入

```
1
7
1 3
3 2
3 5
6 9
10 1
12 4
5 7
```

### 输出

```
39
```

# IITKWPCD - Partition the sticks

## 题目描述

给定 $n$ 根棍子，要求将棍子每三根一组并组成三角形（必须满足三角形性质），求所有三角形面积之和的最大值。

## 输入格式

第一行一个整数 $T$（$1\le T\le 5$），表示数据组数。

对于每组数据，第一行一个整数 $n$（$1\le n\le 15$），表示木棍数量。

第二行共 $n$ 个数，第 $i$ 个数表示第 $i$ 根木棍的长度 $l_i$（$1\le l_i\le 10^3$）。

## 输出格式

对于每组数据输出一行一个数，表示能组成的三角形面积之和的最大值。精确到小数点后 $6$ 位。

# STC05 - Garden

## 题目描述

Byteen 是 Byteasar 的妻子，她对花朵情有独钟，计划在家附近开辟一个花园。她追求完美，希望花园是一个正方形，且边与南北和东西方向平行。同时，她希望花园的每个角落都能有一棵苹果树。

Byteasar 在世界杯足球赛决赛前夕得知了妻子的计划。他知道妻子对这个花园非常期待，因此必须马上开始建造。为了省时间，他询问妻子她理想中的花园位置，并知道妻子会在作出最终决定前检查所有可能性。请帮 Byteasar 算出妻子需要多少时间检查完所有可能的花园位置。假设每个位置的检查需要正好一秒钟。

## 输入格式

第一行是一个整数 $N$，表示 Byteen 和 Byteasar 家附近有 $N$ 棵苹果树，$1 \le N \le 10^5$。为了方便，每棵树的位置用笛卡尔坐标系表示。接下来的 $N$ 行中，每一行包含两个空格分隔的整数 $X_i$ 和 $Y_i$，范围是 $-10^6 \le X_i, Y_i \le 10^6$，代表第 $i$ 棵苹果树的位置。输入中不会有重复的坐标。

## 输出格式

输出一个整数：Byteen 检查所有可能花园位置所需的时间（单位：秒），即花园四个角有苹果树的所有位置的数量。

## 示例
### 输入
```
6
0 0
0 1
1 0
1 1
3 0
3 1
```

### 输出
```
1
```

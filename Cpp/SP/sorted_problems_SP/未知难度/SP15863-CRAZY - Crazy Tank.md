# CRAZY - Crazy Tank

## 题目描述

《疯狂坦克》是一款远在十年前风靡一时的游戏，几乎每个孩子都玩过。虽然后来时光流逝，孩子们渐渐长大，但快乐的回忆仍然留存于心。

你现在控制着一辆名叫 Laotu 的坦克，它位于一个高为 $H$ 米的平台上。由于这辆坦克过于老旧，你只能在游戏开始前选定一个射击角度（任意角度均可），而无法在发射过程中进行调整。你将发射 $N$ 枚炮弹，每枚炮弹的初速度为 $V_i$。

在 Laotu 的右侧地面上，有一辆敌方坦克，坐标区间为 $[L_1, R_1]$，以及一辆友方坦克，坐标区间为 $[L_2, R_2]$。炮弹若落在 $[L_1, R_1]$ 之间（包括边界），即认为击中了敌方坦克。同理，若落在 $[L_2, R_2]$ 之间，则击中友方坦克。Laotu 的水平位置坐标为 0。

游戏的目标是在不击中友方坦克的前提下，尽可能多地击中敌方坦克。

重力加速度 $g = 9.8$。

## 输入格式

输入包含多组测试数据。

每组测试数据有 3 行：

- 第一行是整数 $N$，表示要发射的炮弹数量（$0 \le N \le 100000$）。
- 第二行包括 5 个浮点数 $H、L_1、R_1、L_2、R_2$：$H$ 表示平台的高度，$L_1、R_1$ 表示敌方坦克的坐标区间，$L_2、R_2$ 表示友方坦克的坐标区间（$1 \le H \le 100000$；$0 < L_1 < R_1 < 100000$；$0 < L_2 < R_2 < 100000$）。敌我坦克可能有重叠。
- 第三行包含 $N$ 个浮点数，每个表示一枚炮弹的初速度。

当输入的 $N = 0$ 时，输入结束。

## 输出格式

对于每组测试数据，输出一个整数，表示在不击中友方坦克的情况下，最多能击中敌方坦克的炮弹数量。

## 说明/提示

- $0 \le N \le 100000$
- $1 \le H \le 100000$
- $0 < L_1 < R_1 < 100000$
- $0 < L_2 < R_2 < 100000$

提示：

- 在第一个例子中，可以选择水平射击，使第一枚炮弹落在 14.3 处，第二枚落在 28.6 处。
- 在第二个例子中，没有任何角度可让炮弹不落在 $[2, 30]$ 的同时击中 $[35, 40]$。

# TETRA - Sphere in a tetrahedron

## 题目描述

给定一个四面体的六条棱长，求其内接球体的半径。

## 输入格式

第一行输入一个数n，测试数据的数量。（n<=30）
之后n行每行六个整数，四面体的六条棱长。（棱长不长于1000）
对于四面体WXYZ，边的顺序WX,WY,WZ,XY,XZ,YZ。

## 输出格式

对于每一个n，输出一个保留四位小数的实数，四面体内接球体的半径。

## 样例 #1

### 输入

```
2
1 1 1 1 1 1
1000 999 998 5 5 6
```

### 输出

```
0.2041
1.4189
```

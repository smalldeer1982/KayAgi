# QUE2 - Queue (Pro)

## 题目描述

有 $N$ 个人排成一队。现在我们知道每个人的身高以及在他前面有多少个比他高的人。你的任务是根据这些信息确定每个人在队列中的准确位置。

## 输入格式

第一行是一个整数 $T$，表示测试用例的数量。接下来是 $T$ 个测试用例，每个测试用例包括三行。每个测试用例的第一行是一个整数 $N$，表示队列中的人数。第二行包含 $N$ 个整数，表示这些人的身高。第三行也包含 $N$ 个整数，这些数字分别表示在每个人前面有多少个比他高的人。

## 输出格式

对于每个测试用例，输出一行，按他们站位顺序排列这些人的身高。

## 说明/提示

- $1 \le T \le 10^5$
- $1 \le N \le 10^3$
- $1 \leq \text{身高} \le 10^9$
- $0 \leq \text{前面比他高的人数} < N$

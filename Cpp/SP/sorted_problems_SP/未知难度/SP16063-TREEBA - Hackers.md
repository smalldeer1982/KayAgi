# TREEBA - Hackers

## 题目描述

黑客正在监听米克和斯拉夫的对话，试图破解他们正在传递的信息。米克和斯拉夫使用的词典中包含 $M$ 个单词，任何他们之间发送的消息都由这些词组成，并以空格分隔。不过，由于黑客的设备存在一些问题，截获到的消息发生了变化。

## 输入格式

第一行包含一个整数 $T$（$1 \le T \le 10^5$），表示有多少组测试数据。

对于每组测试数据：
- 第一行包含两个整数 $M$ 和 $N$（$1 \le M, N \le 10^5$）。
- 第二行是截获的消息，由一串连续的小写字母组成，总长度为 $N$。

## 输出格式

对于每组测试数据，输出解码后消息中最多可以找到的词典单词数量。

## 说明/提示

- 测试数据组数 $T$ 最大为 $10^5$。
- 词典单词数量 $M$ 和截获的消息长度 $N$ 均不超过 $10^5$。

# CARRHIM - Can you play carrom !!!

## 题目描述

卡罗姆是一种流行的桌面游戏，通常有四名玩家参与，以双人对抗模式进行。在这种模式下，队伍由对角线上的玩家组成。

如图所示，玩家 A 和玩家 C 组成队伍 1，玩家 B 和玩家 D 组成队伍 2。

在游戏中，每个队伍都会使用一种颜色的棋子。若队伍 1 先行，他们将使用白色棋子，而队伍 2 则使用黑色棋子。如果队伍 1 在队伍 2 之前将所有自己的棋子和红色棋子全部收入囊中，他们将获胜。否则，如果队伍 2 把他们的所有棋子收完，那么队伍 2 就会胜出。

棋盘上共放置 9 个白色棋子、9 个黑色棋子和 1 个红色棋子。游戏遵循特定规则进行。

在本题中，规则如下：

- 每个普通棋子的分值为 1 分，红色棋子的分值为 5 分。
- 假定最先开始游戏的玩家使用白色棋子。一个队伍必须在游戏中收获红色棋子，并可获得该棋子的分数。
- 如果玩家错误地收取了对方队伍的棋子，则游戏顺序交给下一位玩家。

你的任务是根据游戏情况，判断哪支队伍获胜，以及他们的最终得分。

得分是通过棋盘上剩余棋子数加上获胜队伍所取得的红色棋子分数来计算的。

## 输入格式

输入由多组记录组成，每组最多有 100 行。每行包含一个字符和三个整数。第一个字符是玩家代号（A, B, C 或 D），接下来的三个整数 W, B, R 表示该玩家在此回合中获得的白色、黑色和红色棋子的数量（1 ≤ W, B ≤ 9）。输入结束符为一个以 "#" 开头的行。

## 输出格式

对于每组记录，如果游戏结束，若队伍 1 获胜，则输出“Team-1 win and the point is N.”；若队伍 2 获胜，则输出“Team-2 win and the point is N.”。这里，N 是获胜队伍的总得分。如果游戏未完成，则输出“Incomplete game.”。

## 说明/提示

- 每行输入中 $1 \leq W, B \leq 9$
- 一次输入最多 100 行

**样例**

**输入:**

A 5 1 0

B 1 2 0

C 2 1 1

D 1 0 0

\#

B 1 1 0

C 1 2 0

D 3 1 1

A 1 2 0

B 3 0 0

\#

A 1 1 0

B 1 5 1

C 1 1 0

\#

**输出:**

Team-1 win and the point is 10.

Team-2 win and the point is 8.

Incomplete game.

**解释:**

对于第一组输入来说，玩家 A 率先开始，使用的是白色棋子。输入结束后，队伍 1 以获得所有白色棋子和红色棋子告捷，棋盘上剩余 5 个黑色棋子。因此，队伍 1 获胜，总分是 5（剩余黑棋数）+ 5（红棋分）= 10。

# CODGRF - Deconnecting

## 题目描述

我们称 Beta 图为一个由边相连的节点集合。该图的特征是：任意两个节点之间至多有一条边，并且没有节点有边连接自己。每一个节点都有一个「度数」，表示该节点与其他节点连接的边的数量。

## 输入格式

第一行是一个整数 $t$，表示测试用例的数量（$1 \le t \le 10^5$）。接下来的每个测试用例中，第一行包含两个整数 $n$ 和 $m$，分别表示节点数量和边的数量（$1 \le n, m \le 10^5$）。随后有 $m$ 行，每行包含两个整数 $u_i$ 和 $v_i$，表示节点 $u_i$ 和节点 $v_i$ 之间有一条边（$1 \le u_i, v_i \le n$）。

## 输出格式

对于每个测试用例，在单独的一行中输出执行上述过程之后可以保留的最大节点数。

## 说明/提示

$$1 \le t \le 10^5, \quad 1 \le n, m \le 10^5, \quad 1 \le u_i, v_i \le n.$$

# ANAGR - Anagrams

## 题目描述

如果一个字符串的字母可以重新排列后得到另一个字符串，那么这两个字符串被称为"变位词"。在本题中，你将获得两个字符串，你需要判断它们是否是变位词。如果它们不是变位词，你需要找出一个字典序最小的回文串（限小写字母），将其添加到一个字符串的末尾，使两个字符串成为变位词。

在判断时，忽略大小写字母的区别，空格和标点符号可以忽略不计。

**输入格式：**

第一行输入一个整数 $T$，表示测试用例的数量。接下来有 $T$ 个测试用例，每个测试用例由两行构成，每行是一个字符串。

**输出格式：**

对于每个测试用例，输出一行。如果两个字符串是变位词，输出 `YES`。如果不是，输出可以使它们变为变位词的字典序最小的回文串。如果找不到这样的回文串，则输出 `NO LUCK`。

**数据范围与提示：**

$1 \le T \le 100$

每个字符串的长度范围是 1 到 100。

**样例输入输出：**

```
输入：

4
Computer programmer
mature germ romp crop
Awaaay
away
internet
web
the terminator
I’m rotten hater

输出：

YES
aa
NO LUCK
YES
```

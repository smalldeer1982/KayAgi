# TREEOI14 - Trees

## 题目描述

给定一串序列$h_{i}$，定义序列的无序度为$|h_{1}-h_{2}|+|h_{2}-h_{3}|+...+|h_{n-1}-h_{n}|$，你可以任选一个数与给定数进行交换，当然也可以不交换。求交换后的最小无序度。

从标准输入读取数据，并使用标准输出。

## 输入格式

第一行输入一个整数$n$，第二行输入$n$个整数，每两个整数之间以空格分隔。

## 输出格式

输出包括$n$行，第$i$行输出一个整数，代表将第$i$个数与其他数进行交换所得到序列的最小无序度。

## 样例 #1

### 输入

```
5
7 4 5 2 5
```

### 输出

```
7
7
8
7
7
```

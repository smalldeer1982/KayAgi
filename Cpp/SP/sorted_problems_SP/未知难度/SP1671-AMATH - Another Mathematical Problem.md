# AMATH - Another Mathematical Problem

## 题目描述

给出两整数 $n(1 \le n < 10^{100})$ 与 $k(1 \le k \le 100)$ ，你需要计算出是否有正整数 $T$ 使得对于任意正整数 $a$ ，$n^{a+T}-n^a$ 可以被 $10^k$ 整除。

## 输入格式

**多组测试数据**。每组数据给出两个数 $n,k$ 。数据组数不超过 $20$。

## 输出格式

对于每组数据，如果有符合条件的 $T$ 则输出最小的那一个，否则输出`-1`。

## 样例 #1

### 输入

```
32 2
```

### 输出

```
4
```

# EC_MODE - Modems

## 题目描述

奥鲁罗省希望所有的城镇都能接入互联网，并且各个城镇之间能通过至少一条线路进行通信（不一定是直接相连）。负责的工程师希望你帮助他计算实现这一目标的最低成本。

全省共有 $N$ 个城镇（$1 \le N \le 1000$）。UTP 电缆的最大使用距离为 $R$（$1 \le R \le 10000$），如果距离超过 $R$，则必须使用光纤电缆。UTP 电缆的单位成本为 $U$，而光纤的单位成本为 $V$（$U \le V$；$1 \le U, V \le 10$）。此外，还有 $W$ 台卫星调制解调器可以使用（$1 \le W < N$）。这些调制解调器可以被放置在任意城镇，一旦安装，一个城镇便能够通过调制解调器连接互联网，并与其他任何安装了调制解调器的城镇互联。

## 输入格式

第一行输入测试用例的数量。接下来的每个测试用例开始于一行，由五个整数 $N, R, W, U, V$ 组成。之后的 $N$ 行中，每行包含两个整数 $x_i, y_i$（$-10000 \le x_i, y_i \le 10000$），表示第 $i$ 个城镇的坐标。

## 输出格式

对于每个测试用例，输出格式如下：

`Caso #TC: A B`

其中 `TC` 是测试用例编号，`A` 是使用 UTP 电缆的总成本，`B` 是使用光纤的总成本。这两个值都需要保留三位小数。

# EC_ESTA - Statistics Applied

## 题目描述

在这个问题中，我们将寻找数据集的中值。中位数是有序数据组中的中心元素。

例如：对于集合{2,6,3,3,2}，中值为3。一般来说，如果我们有$n$个元素{$a$$_1$,$a$$_2$,$a$$_3$…$a$$_n$}，如果$n$是奇数，我们将中值定义为元素$a$$_($$_n$$_+$$_1$$_)$$_/$$_2$，否则定义为$(a$$_($$_n$$_/$$_2$$_)$+a$_($$_n$$_/$$_2$$_)$$_+$$_1$)$/2$。

------------

## 输入格式

你会得到$N$个数字，你必须计算$N$个中位数。第i个中值取子集$a$$_1$,$a$$_2$,$a$$_3$…$a$$_i$，其中（$1$<=$i$<=$N$）。

------------

## 输出格式

对于每种情况，用中线打印N行。如果结果为非整数，则使用小数点打印精确值（见示例）。

## 样例 #1

### 输入

```
2
4
3 5 7 3
2
3 4
```

### 输出

```
3
4
5
4
3
3.5
```

# CRYPTO1 - The Bytelandian Cryptographer (Act I)

## 题目描述

某人给你发送了一串加密过的数字，并告诉你这是从1970.01.01 GMT午夜到攻击时刻的秒数，并除以4000000007的数字，“我给你一秒钟告诉我这是什么意思！”

## 输入格式

输入为一个加密过的整数。

## 输出格式

输出解密后的日期，格林尼治标准时间的日期和时间，在1970年至2030年之间，使用标准的26字符格式。

## 样例 #1

### 输入

```
1749870067
```

### 输出

```
Sun Jun 13 16:20:39 2004
```

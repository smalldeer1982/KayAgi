# NFURY - Training Land of Fury

## 题目描述

神盾局（S.H.I.E.L.D.）正在招募士兵来对抗洛基的军队。尼克·弗瑞来到了曼哈顿，希望寻找一块大的训练场地。他遇到了一位性格有些憨厚的地主。这位地主出租边长为整数的正方形地块，而且不论地块大小，仅仅按地块的数量收费。尼克·弗瑞需要购买总面积恰好为 $A$ 平方单位的土地。请帮他计算一下，最少需要购买多少块地才能以最低支出满足需求。

## 输入格式

输入的第一行是一个整数 $T$，代表测试用例的数量。接下来的每个测试用例都是一个整数 $A$，表示尼克·弗瑞所需的土地面积。

- $10 \le T \le 100000$
- $1 \le A \le 1000$

## 输出格式

对于每个测试用例，输出他最少需要购买的地块数。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
4
1
2
3
10
```

### 输出

```
1
2
3
2
```

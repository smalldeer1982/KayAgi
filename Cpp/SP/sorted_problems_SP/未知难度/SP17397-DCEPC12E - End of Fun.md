# DCEPC12E - End of Fun

## 题目描述

考试后是假期时间，**Kappi**和**Pushap**正充分享受着。但是，他们的教师**Sid**却是个虐待狂，看不得他们开心。是不是很可悲？为了破坏他们的乐趣，他给了他们一个编程问题，在接下来的$5$个小时内解决。他给了他们两个相同维度的方阵，矩阵$A$和矩阵$B$，并对这些矩阵进行多个查询。查询的目的是改变矩阵中的特定元素，并输出它们的乘积元素的和。**Sid**提出了很多这样的查询。**Kappi**和**Pushap**不想结束乐趣，所以他们请你代表他们解决这个问题。

## 输入格式

第一行包含一个整数$N$，表示矩阵的维度。

接下来的$N$行，每行包含$N$个以空格分隔的整数，表示矩阵$A$。

接下来的$N$行，每行包含$N$个以空格分隔的整数，表示矩阵$B$。

下一行包含一个整数$Q$，表示**Sid**提出的查询数量。

接下来的Q行，每行包含一个查询，格式为$“A i j K”$或$“B i j K”$**（为了清晰起见加上引号）**，表示将矩阵A或B中第$i$行第$j$列的元素改为$K$。

## 输出格式

输出$Q$行，每行对应一个查询，包含矩阵$A*B$的元素和。

## 样例 #1

### 输入

```
2

1 2

3 4

4 3

2 1

3

A 1 1 2

B 0 1 3

A 0 0 10
```

### 输出

```
40

40

103
```

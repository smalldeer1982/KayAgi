# POLEVAL - Evaluate the polynomial

## 题目描述

你的任务是计算一个多项式在几个特定点上的值。这个多项式的最高次数为 $n$（$0 \le n \le 999$），其形式为：

$$ p_n(x) = c_n x^n + c_{n-1} x^{n-1} + \cdots + c_2 x^2 + c_1 x + c_0 $$

需要计算这个多项式在 $k$ 个点 $x_1, x_2, \ldots, x_k$ 处的结果。多项式的系数以及这些点的值都在区间 $[-100, 100]$ 内，并且保证计算结果不会超过 $2^{63} - 1$。

## 输入格式

每个测试用例由以下 4 行组成：
1. 第一行：一个整数 $n$，表示多项式的最高次数。
2. 第二行：$n+1$ 个整数，依次为多项式的系数 $c_n, c_{n-1}, \ldots, c_1, c_0$，用空格分隔。
3. 第三行：一个整数 $k$，表示需要计算多项式的点数目。
4. 第四行：$k$ 个整数，表示要计算的点 $x_1, x_2, \ldots, x_k$，用空格分隔。

输入的最后一个测试用例以 $n = -1$ 开头，这个用例无需处理。

## 输出格式

对于每个测试用例，输出 $k + 1$ 行：
- 第一行：输出“Case #t:”，其中 $t$ 为测试用例的编号（从 1 开始）。
- 接下来的 $k$ 行：分别输出多项式在每个给定点的计算结果。

请参考样例以了解输出格式。

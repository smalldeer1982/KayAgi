# CHOTU - Faridi and Yadav

## 题目描述

两位科学家——**Mr Yadav** 和 **Mr Faridi**，偶然间发现了一种名叫 mahobanium 的神奇放射性化合物，这种材料会以恒定速率衰减，同时不断减轻重量。他们设计了一个游戏，游戏规则是这样的：两个人分别坐在距离一个点 C 等距离的位置上，一人在西边，一人在东边。由于他们的名声，很多学生被吸引来看这个化合物。

这两位科学家有一个名叫 CHOTU 的小机器人。在一开始，Mr Faridi 手上有所有重量相同的 mahobanium 石头。Faridi 会一块一块地把这些石头交给 CHOTU，指挥他把石头给尚未见过它的学生看，然后立刻去找 Mr Yadav。（假设给学生看的过程中不耽误时间，CHOTU 总是走最短路径，且速度稳定在 5 米/秒）。学生们已经将自己安排得很好，以确保到达 Yadav 那里的所有石头重量相同（假设石头一开始放进 CHOTU 手里就开始衰减）。一个学生最多可以在距离中心 C 的东西方向 X 单位，以及南北方向 Y 单位的地方站定。

**现在，给出测试用例 t（< 1000），以及 X 和 Y 的值（假设 X > Y 且 X < 1000），请你找出 Mr Yadav 和 Mr Faridi 之间可能的最大距离，并确保上述条件总是满足。输出结果保留三位小数。**

# QTGIFT1 - New year love story

## 题目描述

在越南的春节期间，有一个传统习俗叫做“利是”，即成年人给孩子们红包以示祝福。DB 的家庭有一种独特的发放红包的方式：

假设有 $n$ 个红包，每个红包中有 $a[i]$ 越南盾（VND）。DB 将从中选定一个正整数 $k$（满足 $1 \le k \le n$），他可以从这些红包中选择连续的 $k$ 个，拿走其中所有的钱。

请帮助 DB 找到能获得最大金额的方法。

**输入格式：**

- 第一行包含两个整数 $n$ 和 $k$，分别表示红包的数量和选取红包的数量。
- 第二行包含 $n$ 个整数，第 $i$ 个数代表第 $i$ 个红包中的金额 $a[i]$。

**输出格式：**

- 输出一个整数 $s$，表示 DB 可以获得的最大金额。

**示例：**

**输入：**

```
5 3
6 19 8 7 13
```

**输出：**

```
45
```

**数据范围与提示：**

- $0 < n \le 10^5$
- $1 \le k \le n$
- $0 \le a[i] \le 10^9$

# ATOMS - Atoms in the Lab

## 题目描述

Yagami先生是Bhabha原子研究中心的科学家。他们正在进行核裂变的实验室实验。在核裂变中，一个原子分裂成不止一个相同类型的原子。

最初，实验室中有$N$个原子。从现在开始（$t = 0$），每秒钟之后，每个原子都将分裂为$K$个相同类型的原子。

科学家们不希望原子数超过$M$，因此必须在$t$ = `T`的某个时间停止反应。请您为Yagami先生找到值`T`。

## 输入格式

第一行一个整数$P$,表示测试数据的组数

接下来$P$行,每行3个整数,即题目所说的$N,K,M$

## 输出格式

对于每组测试数据,输出一行,即`T`的值

## 输入输出样例

输入 #1

```
2
2 2 7
2 2 8
```

输出 #1

```
1
2
```

## 说明/提示

$1 \le P \le 10^4$

$2 \le N,K,M \le 10^{18}$

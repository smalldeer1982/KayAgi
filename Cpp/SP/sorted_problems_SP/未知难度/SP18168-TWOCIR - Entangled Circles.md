# TWOCIR - Entangled Circles

## 题目描述

在这个题目中，我们需要判断两个圆在三维空间中是否缠绕在一起。每个圆都由3个不共线的点定义，并且这两个圆不相交。如果两个圆像链条一样缠绕在一起，即不能在不破坏它们自身的情况下分开，就认为它们是缠绕的。

## 输入格式

输入由两行组成，每行包含6个整数，分别是 $x_1, y_1, z_1, x_2, y_2, z_2, x_3, y_3, z_3$，代表定义圆的3个点的坐标。

## 输出格式

输出一行，如果两个圆缠绕在一起则输出 `YES`，否则输出 `NO`。

# IITWPC4A - Maggu and Strings

## 题目描述

Maggu 和 Coder 正在进行一个字符串游戏。在每一轮中，Maggu 会交给 Coder 一个字符串。Coder 可以选择将其中连续的 $m$ 个 'a' 替换成连续的 $n$ 个 'b'，这个操作可以无限次进行。通过这样的操作，他需要得到两个结果：一个是字符串可能的最大长度，另一个是可能的最小长度。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量（$1 \le T \le 100$）。  
接下来对于每个测试用例，有两行输入。  
第一行是一个只包含小写字母 'a' 到 'z' 的字符串 $s$（$1 \le \text{length}(s) \le 10^5$）。  
第二行包含两个由空格分隔的整数 $m$ 和 $n$，分别表示可替换的 'a' 的个数和替换后的 'b' 的个数（$1 \le m, n \le 10^5$）。  
所有测试用例的字符串长度和不超过 $10^6$。

## 输出格式

对于每个测试用例，输出一行，包含两个由空格分隔的整数，分别代表 Coder 能获得的字符串 $s$ 的最小可能长度和最大可能长度。

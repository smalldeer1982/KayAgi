# IITWPC4B - Maggu and Triangles

## 题目描述

Maggu 拥有一根长为 $n$ 的铁丝。他需要用这根铁丝制作三角形，要求每个三角形的边长都是整数。他想知道，可以用这根铁丝制作出多少个不同的（形状不全等）三角形。注意，制作每个三角形时必须完全使用这根铁丝。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。（$1 \leq T \leq 10^5$）  
接下来的每一行输入一个整数 $n$，表示铁丝的长度。（$1 \leq n \leq 10^9$）

## 输出格式

对于每个测试用例，输出一个整数，表示 Maggu 可以制作的三角形数量。

### 输入

```
\n3\n5\n7\n9
```

### 输出

```
\n1\n2\n3
```

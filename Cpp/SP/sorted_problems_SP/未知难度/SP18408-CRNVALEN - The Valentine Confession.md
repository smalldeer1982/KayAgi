# CRNVALEN - The Valentine Confession

## 题目描述

我们这里谈论的是 DTU。和印度大多数工程学院一样，DTU 的男女比例极不平衡。不过，DTU 的女生很慷慨高尚，很多人出于社交需求，同时和两个男生约会。 

情人节周的晚上11点，宿舍的电视房里聚集了 $n$ 位女生，她们正在聊天。每个女生要么只和一个男生交往，要么同时和两个男生交往。她们决定互相坦白，于是告诉彼此自己是只约会一个男生，还是更高尚地同时约会两个男生。请注意，这些是 DTU 的女生，她们在彼此之间从不撒谎。然而，宿舍管理员 Chameli 也想知道这其中的秘密。 

于是，这些女生玩了一个游戏：每个女生告诉管理员，剩下的 $n-1$ 个女生中有多少人在同时约会两个男生。你需要帮助管理员 Chameli 确定总共有多少女生在同时约会两个男生。

## 输入格式

第一行包含一个整数 $T$，表示测试用例的数量。

每个测试用例由两行组成。第一行包含一个整数 $n$，表示宿舍中女生的人数。

第二行包含 $n$ 个用空格分隔的整数。第 $i$ 个数字表示第 $i$ 个女生告诉管理员的数字。

## 输出格式

对于每个测试用例，输出一行，表示同时约会两个男生的女生总数；如果给出的数据无效，则输出 -1。

## 说明/提示

- $T \leq 100$
- $2 \leq n \leq 10000$
- 所有的 $n$ 个数字都满足 $\leq 10^9$。

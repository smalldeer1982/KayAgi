# UOFTCC - A Subtle Surf

## 题目描述

Alice 收到了 Bob 的邀请，在接下来的 $D$ 天（$1 \leq D \leq 100$）一起看电视！虽然和 Bob 一起共度时光很愉快，但 Alice 更在意他们究竟要看哪些频道。毕竟，Bob 是个男生，可能会对那些不那么雅致的节目感兴趣。

每天都会有 $N$（$1 \leq N \leq 100,000$）个不同的电视频道可供选择，频道编号从 $1$ 到 $N$。每个频道 $i$ 都有一个特定的女孩度 $G_i$（$0 \leq G_i \leq 10^9$），反映出 Alice 对该频道的喜爱程度。当 Alice 到达 Bob 家时，电视总是已经设定在频道 1，但她希望尽快切换到一个女孩度最高的频道。

Alice 希望她的换台行为不太引人注目。她认为如果在一个频道上停留的时间不到 $T$ 秒（$1 \leq T \leq 1000$）就切换，或者新频道的女孩度比当前频道高出超过 $C$（$1 \leq C \leq 10^9$），Bob 可能会察觉。Alice 需要一个计划，在最短的时间内切换到女孩度最高的频道。

## 输入格式

第 1 行：一个整数 $D$，表示天数。

**对于每一天：**

第 1 行：三个整数 $N$、$C$ 和 $T$，分别表示电视频道数量、可容忍的女孩度增量和最短停留时间。

第 2 行：$N$ 个整数 $G_{1..N}$，分别表示每个频道的女孩度。

## 输出格式

**对于每一天：**

输出两个整数，分别为 Alice 可以达到的最大女孩度以及达到该女孩度所需的最短时间。

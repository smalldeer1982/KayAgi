# FONTSIZE - Font Size

## 题目描述

你需要在一个广告牌上显示一条消息，并尽量使用最大的字体。现在需要根据广告牌的宽度和高度，以及要显示的消息，计算出能够完整显示该消息的最大字体大小。

## 输入格式

输入包含三行：
- 第一行是两个整数 $W$ 和 $H$，分别代表广告牌的宽度和高度，单位为英寸，其中 $0 \le W \le 7500$，$0 \le H \le 7500$。
- 第二行为空行。
- 第三行是由字母和数字组成的非空消息，单词之间用一个空格分开。消息的最大长度为 10,000 个字符。

## 输出格式

输出一个浮点数，表示可以完整显示消息的最大字体大小，单位为英寸。结果需精确到小数点后三位。

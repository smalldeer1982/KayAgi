# TREEORD - Tree _order

## 题目描述

给定三种树的遍历序列，判断这些序列是否可能来自同一棵树。

## 输入格式

第一行包含一个整数 $N$，表示每种遍历中的节点数量，满足 $0 < N \le 8000$。  
第二行是前序遍历的 $N$ 个节点，节点之间用空格分隔。  
第三行是后序遍历的 $N$ 个节点，节点之间用空格分隔。  
第四行是中序遍历的 $N$ 个节点，节点之间用空格分隔。

## 输出格式

如果这三种遍历可能来自同一棵树，输出 `yes`；否则，输出 `no`。

# INS14B - TRISQRS

## 题目描述

从 `2013` 年的 `ACM` 世界总决赛回来后，迪戈决定通过观看大量动作片来庆祝他的表现。迪戈坐在他的笔记本电脑看着他最喜欢的明星之间的较量，突然受到了加入中央情报局的启发。迪戈是一个严肃的人，他发誓要在年底前加入中央情报局部队。请注意，他可是非常认真地对待自己的誓言！

在野外辛劳了几个月之后，终于到了采访他的时候了。面试官认为脑力与体能一样重要，并决定用数学难题来测试我们的英雄。

**面试官递给迪戈一个边长为** $N$ **的正方形，并要求他将该正方形分成多个直角三角形，每个三角形的面积相等。三角形的两条边应与正方形的两条边平行。** 作为 `ACM` 世界决赛入围者，迪戈在一瞬间解决了这个问题。面试官印象深刻，决定用一个更难的问题来测试他。**他让迪戈写出可能的这种分区的数量。具有相同数量直角三角形的所有分区被认为是相同的。**

译者提示：只有黑体字是解题需要的信息

## 输入格式

第一行为一个整数$T$，表示测试用例的数量。接下来的 $T$ 行每一行都为一个整数$N$，每个 $N$ 表示正方形的边长。

## 输出格式

每个测试用例输出一行。每行包含一个整数，它是对于边长为$N$的正方形的解决方案。

#### 数据范围 / 提示
保证$T$在$1$和$1000$的范围之内；\
并且$1 \le N \le 1000$。

#### 输入和输出样例
##### 输入样例1
```
2
1
2
```
##### 输出样例1
```
0
2
```

# INS14D - Digo Needs Guns

## 题目描述

Digo 负责保护一个非常重要的房间，里面住着许多名人。政府为他提供了一些自动枪，可以安装在房间的任意两面墙的交线上。这些枪能绕着安装点旋转，但无法穿透墙壁射击。由于国家正面临经济危机，而自动枪的成本很高，Digo 希望使用尽可能少的枪，并将剩余的退回。然而，房间的具体布局对他来说是机密的，他只知道房间的墙数。房间的形状可以是任意简单多边形。要确保每种可能的多边形房间都能被完全保护，他至少需要携带多少把枪呢？

## 输入格式

第一行为一个整数 $T$，表示测试用例的数量。  
接下来的 $T$ 行中，每行包含一个整数 $N$，表示房间的边数。

## 输出格式

对于每一个测试用例，输出所需的最少枪数。

## 说明/提示

- $1 \le T \le 1000000$
- $3 \le N \le 1000000000$

## 样例输入

```
1
4
```

## 样例输出

```
1
```

# RECTANGL - Rectangles

## 题目描述

在平面中给定一组 $N$ 个点的 $S$，并且必须计算其四个顶点都位于 $S$ 中的不同轴平行矩形的数量（即，计算那些两条边平行于 $x$ 轴，另外两条边平行于 $y$ 轴的矩形）。

## 输入格式

输入的第一行是 $N$ （$1 ≤ N ≤ 250000$），即 $S$ 中的点数。然后是 $N$ 行，其中第 $i$ 行的格式为 “$x_{i}   y_{i}$”，给出 $S$ 中点 （$x_{i}$， $y_{i}$） 的坐标。所有给定的点都是不同的，并且所有坐标都适合 $32$ 位有符号整数。

## 输出格式

您的输出应由一个数字组成，即四个顶点都位于 $S$ 中的不同轴平行矩形的数量，后跟一个换行符。

## 样例 #1

### 输入

```
6
-1 0
-1 1
0 0
0 1
1 0
1 1
```

### 输出

```
3
```

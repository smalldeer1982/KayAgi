# APPROB - GETTING AN AP

## 题目描述

小约翰对数列产生了浓厚兴趣，特别是等差数列。

他准备了三个盒子，每个盒子中放一些纸片。为确定纸片数量，他随机选一个数 $n$：在第一个盒子放 $n$ 张纸片，第二个盒子放 $2 \times n$ 张，第三个盒子放 $3 \times n$ 张。每个盒子中，纸片编号从 $1$ 到纸片数。

接着，他从每个盒子中随机抽取一张纸片，按次序排列（先第一个盒子的纸片，然后是第二个盒子，再是第三个）。他想知道，这样得到的顺序能否形成一个等差数列。

给定小约翰选择的数 $n$，计算他抽取的纸片构成等差数列的概率。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。

接下来 $T$ 行，每行一个整数 $n$，表示小约翰选择的数。

$1 \le T \le 100000$

$1 \le n \le 100000$

## 输出格式

对于每个测试用例，输出一个最简分数 `p/q`，表示抽取纸片构成等差数列的概率。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
\n2\n1\n2\n\n
```

### 输出

```
\n1/3\n1/8\n\n\n\n\n\n\n\n\n\n
```

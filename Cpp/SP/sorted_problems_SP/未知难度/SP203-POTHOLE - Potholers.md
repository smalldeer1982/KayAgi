# POTHOLE - Potholers

## 题目描述

一支洞穴探险队正在比特山脉的大洞穴中训练。训练过程中，每个探险队员须从顶层房间出发，到达底层房间。探险队员只能顺着路径向下移动，这意味着路径上后续房间的层数必须低于前一个房间。此外，每位探险队员必须通过与他人不同的走廊从顶层出发，并且每人也需要通过不同的走廊进入底层房间。其余的走廊则可以被多个探险队员共享使用。请问，最多能有多少名探险队员同时参与训练？

### 任务

编写一个程序来：

- 读取洞穴的结构描述，
- 计算出可同时训练的最大探险队员人数，
- 输出结果。

## 输入格式

第一行输入一个整数 $t$，代表测试用例的数量。随后是 $t$ 个测试用例，每个用例之间用一个空行隔开。每个测试用例的第一行是一个整数 $n$（$2 \leq n \leq 200$），表示洞穴中的房间总数。房间按顺序编号，从高到低排列，高层房间的编号比低层房间大（顶层房间为编号 $1$，底层为编号 $n$）。接下来的 $n-1$ 行描述了各个房间之间的走廊。第 $(i+1)$ 行列出了第 $i$ 个房间连接到编号较高房间的走廊信息，第一个数字 $m$（$0 \leq m \leq n-i+1$）表示走廊的数量，接下来的 $m$ 个整数是这些走廊通往的房间编号。

## 输出格式

对于每个测试用例，输出一个整数，表示能够同时进行训练的最大探险队员人数。

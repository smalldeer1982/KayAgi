# REMGAME - Stone Removing Game

## 题目描述

有一个 5×5 的棋盘游戏。在游戏开始时，每个格子里都有一个棋子。两位玩家轮流从棋盘上移除棋子。每次移除时，玩家可以选择从任意一行或一列中去掉任意数量的连续棋子。例如，在下面示意的棋盘配置中（1 表示位置上有棋子），玩家可以移除一个棋子（如 **A1**、**A2** 或 **B1**），也可以同时移除相邻的两个棋子（如 **A1** 和 **A2**，或者 **A1** 和 **B1**）。游戏的目标是避免成为那个被迫拿走最后一个棋子的玩家，若你成功做到这一点，那么你便获胜。

```
    1  2  3  4  5
A   1  1  0  0  0
B   1  0  0  0  0
C   0  0  0  0  0
D   0  0  0  0  0
E   0  0  0  0  0
```

## 输入格式

第一行包含一个整数 $n$，表示测试用例的数量。接下来的每一个测试用例都给出一个 5×5 的棋盘的初始配置。

## 输出格式

对于每个测试用例，输出「winning」表示该配置下先手玩家能获胜，输出「losing」表示先手玩家无法获胜。

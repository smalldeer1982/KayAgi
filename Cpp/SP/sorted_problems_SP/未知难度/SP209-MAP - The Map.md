# MAP - The Map

## 题目描述

在Byteland绘图办公室的新行政区划后，该国开始使用新的人口统计地图。由于技术原因，只能使用几种颜色。地图应带有颜色，以使人口数相同或相似的区域具有相同的颜色。对于给定的颜色$k$，定义一个$A(k)$，使得：

至少一半颜色为$k$的区域的人口不大于$A(k)$;

至少一半颜色为$k$的区域的人口不少于$A(k)$。

区域的**着色误差**是$A(k)$与区域人口之间的差的绝对值。**累积误差**是所有区域的**着色误差**的总和。我们正在寻找一种最佳的地图颜色（一种具有最小**累积误差**的颜色）。

### 你的任务：

编写一个程序，该程序：

从输入中读取Byteland的区域总数，
计算最小的**累积误差**，输出最小的**累计误差**。

## 输入格式

输入的第一行是一个整数$t$，然后是以空行分隔的$t$组数据。

在每组数据的第一行中，输入一个整数$n$，表示Byteland中的区域总数$(10<n<3000)$。

在第二行中，写有m表示用于为地图着色的颜色数$(2<=m<=10)$。

在随后的$n$行中的每行中，都有一个非负整数$a_i$表示第$i$个区域人口，保证$a_i$不超过$2^{30}$。

## 输出格式

输出一行，包括一个数，表示给出数据的**最小累积误差**，这可以在对地图进行着色时算出。

## 样例 #1

### 输入

```
1
11
3
21
14
6
18
10
2
15
12
3
2
2
```

### 输出

```
15
```

# UCF - Under Construction Forever

## 题目描述

UCF（中佛罗里达大学）如今是美国第二大学生人数最多的大学。为了满足发展的需求，学校计划多次对校园进行重组。

## 输入格式

第一行为一个正整数 $n$，表示需要进行的校园重组次数。接下来的每次重组都由多行信息构成。第一行包含两个整数：$b$（建筑物的数量）和 $c$（建筑物之间的连接数量），其中 $1 \le b, c \le 1000$。接下来的一行提供 $b$ 个整数 $w_i$，表示重建第 $i$ 栋建筑物的费用 ($1 \le w_i \le 1000$)。接下来的 $c$ 行，每行包含两个整数 $x_i$ 和 $y_i$，表示建筑物 $x_i$ 和 $y_i$ 之间存在直接连接关系 ($1 \le x_i, y_i \le b$)。需要注意的是，所有建筑物都是通过某种方式直接或间接连通的。

## 输出格式

对于每次重组，首先输出 “Case #d: ”，其中 $d$ 代表测试用例编号，从 1 开始。然后输出三个用空格隔开的整数：在达到目标配置时，剩余的最少建筑数量、所需的最低成本，以及在满足最小规模和最低成本条件下实现这种配置的方法数。输出格式如示例所示。

## 说明/提示

$$1 \le n \le 100$$
$$1 \le b, c \le 1000$$
$$1 \le w_i \le 1000$$
$$1 \le x_i, y_i \le b$$

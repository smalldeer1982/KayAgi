# CALCULAT - Digital Calculator

## 题目描述

### 题目大意
给定三个数 $n,\ x,\ y$ ， 问 $ n! $ 的前 $x$ 位和后 $y$ 位分别是多少（其中 $n!$ 表示 $n$ 的阶乘）。

## 输入格式

**本题包含多组数据。**\
第一行一个整数 $T$ ，表示有 $T$ 组数据。\
对于一组数据，输入三个数 $n,\ x,\ y$ 。

## 输出格式

对于一组数据，输出两个数，表示 $n!$ 的前 $x$ 位和后 $y$ 位，答案用空格隔开。

## 样例 #1

### 输入

```
3

6 2 1

10 3 2

8 5 5
```

### 输出

```
72 0
362 00
40320 40320
```

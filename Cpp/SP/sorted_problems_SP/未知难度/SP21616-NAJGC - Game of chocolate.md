# NAJGC - Game of chocolate

## 题目描述

本和格温从爷爷那里各得到了一袋巧克力。本的袋子里有 $X1$ 个奇巧（Kit-Kat）巧克力和 $Y1$ 个帕克（Park）巧克力，而格温的袋子里则有 $X2$ 个奇巧巧克力和 $Y2$ 个帕克巧克力。然而，本不喜欢吃巧克力，所以他打算把所有的巧克力都送给格温。不过，他想和格温玩一个小游戏，如果格温赢了，她就能拿到本所有的巧克力。游戏规则如下：

1. 首先，本从自己的袋子里随机拿出一块巧克力，这块巧克力是未知的。
2. 接着，他将这块巧克力放入格温的袋子。
3. 然后，格温从她自己的袋子里随机拿出一块巧克力。
4. 如果格温拿到的巧克力与本最初拿出的巧克力类型相同，那么她就赢得了游戏，并可以得到本的全部巧克力。

现在，请计算格温赢得游戏的概率。

## 输入格式

输入以一个整数 $T$ 开头，表示测试用例的数量。接下来的每一行包含四个整数 $x1, y1, x2, y2$，分别表示本袋子里的奇巧和帕克巧克力数量，以及格温袋子里的奇巧和帕克巧克力数量（$0 \leq x1, y1, x2, y2 \leq 1000$）。

## 输出格式

对于每个测试用例，输出一行，格式为 `Case x: a/b`，其中 `a/b` 表示格温赢得游戏的概率，且是不可约分的分数形式。如果概率为零，则直接输出 `0`（如：`Case x: 0`）。

## 样例输入输出

### 输入

```
2
3 5 4 5
1 3 4 8
```

### 输出

```
Case 1: 9/16
Case 2: 8/13
```

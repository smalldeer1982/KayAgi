# SOPARADE - Soldiers on Parade

## 题目描述

### 题面简述
Byteland 中的协议真的很奇怪。例如，要求士兵在军官面前展示武器时，应站成一排（从 $1$ 到 $n$ 的位置）。士兵有 $4$ 种可能的军衔，以肩章上的波浪线数量（$1$ 到 $4$ 之间）来区分。并排站立的士兵必须至少有两个波浪线的等级差异。此外，还有其他规则集（每个省都不同）。每条规则都规定，站在行中某些给定位置的士兵的等级必须至少相差一个波浪线。

从新的一年开始，一些省份正在改变他们的一套协议规则。作为礼宾部高级军事秘书，你的任务是批准新规则。令你惊讶的是，一些省份提出了完全不可能实现的礼仪规则，即使士兵是专门挑选来展示武器的。侦查所有此类违规省份，绝不批准其法律。

## 输入格式

输入的第一行包含一个正整数 $t(t<=10)$，表示提出新法律的省份数量。遵循 $t$ 组规则，用空行分隔。

每组规则都以包含两个非负整数 $n,p(n<=100000,p<=100000)$ 的行开头，分别是该省安排的士兵人数和提议的规则数量。接下来的每一行 $p$ 都包含一个规则：一个整数 $b_i(2 \le b_i \le n)$，后面跟着 $a_1,a_2,...,a_{b_i}(1 \le a_k \le n)$。这样的规则意味着士兵站在阵地 $a_1,a_2,...,a_{b_i}$ 必须具有不同的等级。

## 输出格式

对于输入端呈现的每组规则，如果无法根据协议安排士兵单位，则输出 `rejected`，反之输出 `approved`。

## 样例 #1

### 输入

```
2

2 1
2 1 2

5 2
3 1 3 2
4 2 3 4 5
```

### 输出

```
approved
rejected
```

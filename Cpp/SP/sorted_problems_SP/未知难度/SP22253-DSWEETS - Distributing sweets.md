# DSWEETS - Distributing sweets

## 题目描述

有一天，AAST的学生在“科学思维”方面取得了成绩，大多数学生的分数都很低。所以讲师想给他们一些让他们开心的东西，所以他检查了一下包，发现了 $n$ 个糖果，分糖果的要求如下：
1. 每个人都会得到至少一个糖果（数量为整数）。
1. 他想让一个学生的分数和这个学生得到的糖果数量之比等于其他任何学生的分数与糖果数量之比来计算。（对于所有学生：一个学生的分数 $ \div $ 他得到的糖果数量 $=$ 一个固定的数，每个学生得到的糖果数量至少为一，这个固定的数可以是任何实数）。
1. 分发糖果结束后不能有糖果剩余。

示例：如果两个学生 $x$ 和 $y$ ，$x$ 得5分， $y$ 得10分，那么 $y$ 必须得到 $x$ 得到的糖果数量的两倍，也就是 $x$ 得到的甜食数量至少为1。

Mohamed Ramzy是他讲座中最调皮的学生，所以他决定惩罚他，让他分发糖果，但Ramzy不知道是否有可能按照讲师所说的条件分发糖果，所以他请你帮助他。

注:每个数据点中还有多组测试数据.

## 输入格式

第一行包含一个整数 $t$ ，表示测试数据的组数。

对于每组测试数据，第一行包含两个整数 $n$ 和 $m$ ，分别代表糖果的数量和学生的数量。下一行包含m个整数，即学生的分数 $a_i$ ，第 $i$ 学生的分数为 $a_i$
 。

注：输入文件中没有空行。

## 输出格式

对于每组测试数据输出一行，如果有满足要求的分法，输出”YES“，否则输出”NO“。

## 样例 #1

### 输入

```
3
5 4
1 1 1 1
4 4
1 1 1 1
3 2
1 2
```

### 输出

```
NO
YES
YES
```

# RELIGION - pK

## 题目描述

pK 降临地球后，面临两大困扰：

1. 他的「遥控器」被 Tapaswi Maharaj 偷走了。
2. 地球上的宗教冲突日益严重（没有无神论者，而且至少有两种宗教）。

上帝怜悯 pK，给予他一个解决人类之间冲突的机会。首先，上帝建造了 $C$ 个城市，每个城市对应一个宗教。他根据宗教信仰把人们分配到相应的城市，且每个城市的居民需遵从城市的首领。另外，上帝在城市间建了各自的「通道」。

为了促进城市间的团结，上帝召开了一个会议，邀请所有城市首领与会，希望他们彼此和平相处。尽管许多城市的首领接受了上帝的提议，但仍有一些不愿意和其他城市友好相处。上帝将这些城市间的通道称为「敌对」通道，而其他的则称为「友好」通道。

pK 担心地球上会有敌对通道存在，因此上帝计划重新排列城市，把部分城市转移到 pK 的星球上，条件如下：

1. 地球上不应有敌对通道。
2. pK 的星球上不应有友好通道（这样 pK 可专注于解决敌对通道的冲突）。
3. 地球和 pK 的星球之间可以存在两种通道（以便上帝和 pK 共同行动解决敌对通道问题）。
4. 地球和 pK 的星球必须各自至少有一个城市。
5. 每个在 pK 星球的城市不需要和地球的城市都有通道连接。

如果无法满足这些条件，冲突和生命损失将不可避免。

邪恶的 Tapaswi Maharaj 做法祈祷并催眠上帝，从而获得了摧毁城市的能力。他决定摧毁一个或多个城市（和相关的通道），以此来破坏上帝的计划，导致冲突发生。即使他只破坏一个城市，也会归还「遥控器」给 pK（被上帝视为 Tapaswi 的妥协）。然而，Tapaswi 的力量只在城市数量超过两个时有效。

如果 pK 通过避免冲突拯救了生命，则 pK 成功；否则，邪恶的 Tapaswi 成功。

# CHICKEGG - Chicks

## 题目描述

在一个农场里，有 $n$ 只母鸡。每只母鸡每产下一个鸡蛋后，孵化时间都会增加 1 天（也就是说，孵化下一颗蛋比上一颗蛋需要更多的时间）。

设定第 $i$ 只母鸡的初始孵化能力为 $D[i]$：

- 第 $i$ 只母鸡在第 $D[i]$ 天产下第一颗蛋。
- 第 $i$ 只母鸡在第 $2 \times D[i] + 1$ 天产下第二颗蛋。
- 第 $i$ 只母鸡在第 $3 \times D[i] + 3$ 天产下第三颗蛋。
- 第 $i$ 只母鸡在第 $4 \times D[i] + 6$ 天产下第四颗蛋。
- 第 $i$ 只母鸡在第 $5 \times D[i] + 10$ 天产下第五颗蛋。

如此类推……

现在需要计算的是：在农场中，至少要产下 $K$ 颗鸡蛋所需的最短时间。已知鸡蛋不会损坏，也不会孵化出小鸡。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的数量。对于每个测试用例，第一行包含两个整数 $n$ 和 $K$。接下来的一行是 $n$ 个整数，表示每只母鸡的初始孵化能力。

## 输出格式

对于每个测试用例，输出至少需要多少天才能生产出 $K$ 颗鸡蛋。

## 数据范围

$$1 \le t \le 10^5, \quad 1 \le n \le 10^5, \quad 1 \le K \le 10^9, \quad 1 \le D[i] \le 10^5$$

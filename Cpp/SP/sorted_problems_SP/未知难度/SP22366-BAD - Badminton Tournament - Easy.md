# BAD - Badminton Tournament - Easy

## 题目描述

在一场羽毛球比赛中，所有 $n$ 个选手都要与其他 $n-1$ 个选手各进行一场比赛，比赛的胜负结果会记录下来。每场比赛结束后，每个选手需要列出他们战胜的对手名单（称为“名单1”），并且还要记录下这些对手所战胜的其他选手。例如，如果选手 A 战胜了选手 B，而选手 B 又战胜了选手 C，那么选手 A 的名单中就会包括 B 和 C。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的数量。每个测试用例首先输入一个整数 $n$，表示选手的数量。接下来，有 $\frac{n(n - 1)}{2}$ 行数据，每行有两个整数 $a$ 和 $b$，表示选手 $a$ 击败了选手 $b$。

## 输出格式

对于每个测试用例，输出一行，包含两个由空格分隔的整数 $p$ 和 $q$。其中，$p$ 是拥有最多选手名单的选手编号，$q$ 是该选手名单中的最大的可能选手数量。如果有多个选手名单的长度相同且最大，则选出编号最小的那个选手。

## 说明/提示

- $1 \le t \le 10$
- $2 \le n \le 100$

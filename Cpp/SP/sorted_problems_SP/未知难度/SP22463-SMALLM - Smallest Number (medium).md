# SMALLM - Smallest Number (medium)

## 题目描述

60可以被2、3、4和5整除。没有比 $60$ 更小的数字可以满足条件。

## 输入格式

在第一行，输入两个整数 $T$（测试用例的数量）和 $M$（一个整数）。接下来的 $T$ 行中的每一行都包含一个整数 $N$。

$M$ 为模数。

## 输出格式

对于每个测试用例，在一行上输出，这是可被 $1 \sim N$（包括$1$和$N$）的所有整数整除的最小数字。由于答案可能很大，因此将其模 $M$ 输出。

## 样例 #1

### 输入

```
1 1000000007
5
```

### 输出

```
60
```

# JEWELS - Jewelry and Fashion

## 题目描述

## 珠宝 - 珠宝和时尚


您在一家小型珠宝商公司工作，该公司以其生产的精美项链和多色琥珀绳而闻名。在过去的三个世纪里，仅琴弦的销售就足以让业务顺利进行。然而现在，时尚的影响力比以往任何时候都大，除非你适应构成你主要客户群的社会相当不寻常的部分的需求和幻想，否则你就会面临即将破产的前景。这些老太太最近决定时尚已经发生了变化：绳子被淘汰，耳环被引入。对此我们无能为力 - 您必须遵守并转而生产耳环。

仍然存在一个问题：如何处理您商店中堆积的令人印象深刻的琥珀线？你的一位助手有一个好主意：他建议将绳子切成两部分，去除一些宝石，使两部分具有相同的颜色图案（立即或旋转 180 度后），然后将剩下的东西作为耳环出售。经过片刻的思考，您决定继续执行该计划。但您细心的管理眼光告诉您，尽量减少浪费（移除）的宝石数量可能并不像听起来那么容易......

 ![Example of string2earring conversion ;)](https://cdn.luogu.com.cn/upload/vjudge_pic/SP226/bd90ef4791e6ca17c49823a0b19aa8159ca35295.png)

## 输入格式

输入的第一行包含一个整数 t<=500，即测试用例的数量。接下来的 t 行各包含一个测试用例，其形式为最多 8000 个字符的字符串 'a'-'z' （以新行结尾，可以选择在前面加上应忽略的空格）。该线的第 i 个字符对应于它所代表的琥珀色绳中第 i 颗宝石上的设计。输入文件的总长度不超过 100kB。

## 输出格式

对于每个测试用例，输出两个数字：可以从绳子中产生的一对耳环的最大可能总长度，以及一个正整数，表示应该切割绳子以达到此目的的石头编号。如果可能有多个切割位置，则输出最左侧（最小）的切割位置。

## 样例 #1

### 输入

```
3
abcacdd
acbddabedff
abcbca
```

### 输出

```
4 3
6 4
4 2
```

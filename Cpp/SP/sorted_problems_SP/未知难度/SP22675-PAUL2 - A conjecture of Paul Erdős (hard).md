# PAUL2 - A conjecture of Paul Erdős (hard)

## 题目描述

给出 $T$ 个整数n ($T \le 500000$，$n \le10^{12}$) ，请找出形为 $x^2+y^4$ 且小于n的整数的个数 ($x,y$都为整数)。

## 输入格式

输入的第一行包含一个整数 $T$ ，表示数据组数
接下来 $T$ 行，每行一个整数 $n$ ，如题所述。

## 输出格式

输出共 $T$ 行，每行一个整数表示答案。

## 说明/提示

$T \le 500000$，$n \le10^{12}$，x、y 都为整数。

## 样例 #1

### 输入

```
6
1
2
10
9999999
500000000000
1000000000000
```

### 输出

```
0
1
2
13175
25874902
42377120
```

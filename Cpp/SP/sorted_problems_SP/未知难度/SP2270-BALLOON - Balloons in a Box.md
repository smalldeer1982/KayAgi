# BALLOON - Balloons in a Box

## 题目描述

在一个长方体盒子里，有n(n<=6)个点。在其中任何一个点上放一个很小的气球，那么这个气球会一直膨胀，直到触碰到其他气球或盒子。而且必须等一个气球膨胀完毕才能让下一个气球膨胀。问按照什么样的顺序在这n个点上放气球，才可以使放置完毕后所有气球占据的总体积最大？

## 输入格式

输入包含几个测试用例。每个测试用例的第一行包含一个整数n，n指点数（1 <= n <= 6）。第二行包含三个整数，它们表示框角的（x，y，z）整数坐标，第三行包含框的对角角的（x，y，z）整数坐标。测试用例的后n行每行包含三个整数，分别代表集合中各点的（x，y，z）坐标。盒子在每个维度上的长度都不为零，并且其侧面平行于坐标轴。

## 输出格式

对于每个测试用例，请打印一行输出，其中包括测试用例编号，然后是未被气球占据的盒子的体积。将音量四舍五入到最接近的整数。在每个测试用例的输出之后放置一个空白行。

## 样例 #1

### 输入

```
2
0 0 0
10 10 10
3 3 3
7 7 7
0
```

### 输出

```
Box 1: 774
```

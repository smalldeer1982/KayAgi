# SHAMAN - Shamans

## 题目描述

在遥远的荒芜之地，居住着一个神秘的部落。他们年年遭遇干旱之苦，但始终坚守对神灵的信仰，绝不离开自己的家园。为了解决干旱问题，部落中的四位大祭司必须在艰难时期进行祈祷，希望得到神恩雨水的帮助，促进粮食生产。

四位大祭司将各自选择领地中的一个山峰上进行祈祷。他们所形成的四边形区域（祭司们分别位于四个顶点）就是他们法术的影响范围（上帝从天上俯瞰时会看到这个区域）。这片土地崎岖不平，部落为祭司们选择了多个山顶进行祈祷。当然，为了达到最好的效果，四边形的面积越大越好。因此，在祭司们出发前，他们需要仔细挑选出最适合的四个山顶。

## 输入格式

第一行是一个正整数，表示测试用例的数量，之后是一行空白。测试用例数量最多不超过 80 个。

对于每个测试用例，第一行是一个整数 $n$（$4 \le n \le 2000$），表示山顶的数量。接着的 $n$ 行中，每行包含两个整数 $x, y$（$-20000 \le x, y \le 20000$），表示山顶的坐标。

每组测试用例之间用一个空行分隔。

## 输出格式

每组测试用例输出一个浮点数，保留一位小数：祭司们所能覆盖的最大四边形面积。

# CWC2015 - CWC-2015

## 题目描述

在2015年板球世界杯上，孟加拉队的教练 Chandika Hathurusingha 想要选出最佳的球队组合。然而由于出色和经验丰富的球员太多，选择变得相当复杂。于是他计划采取几个步骤来挑选最佳组合。第一步是将 $N$ 名球员分成两个组，使得每组的人数相同，并且每组玩家的经验值总和也相同。经过尝试后他发现这项任务极其困难。于是，他向你求助，期待你拿出可以实现这一步的方法。你的任务是判断是否能够按照他的计划，将球员分组。

输入：

输入的第一行是一个整数 $T$，表示共有多少个测试用例。接下来的每个测试用例以一个整数 $N$ 开头，表示球员的数量。接下来的行中包含 $N$ 个整数，每一个表示相应球员的经验值。

输出：

对于每个测试用例，请输出 "Case X: Yes" 或 "Case X: No"。其中 $X$ 是测试用例的编号（从 1 开始）。如果可以成功完成分组，输出 "Yes"；否则输出 "No"。

样例输入：
```
2
6
1 2 3 4 5 6
4
2 3 4 5
```

样例输出：
```
Case 1: No
Case 2: Yes
```

输入以一个整数 $T$ 开始，表示测试用例的数量。接下来每个测试用例的第一行是一个整数 $N$，表示球员数量。第二行紧接着给出 $N$ 个整数，表示每个球员的经验值。

对于每个测试用例，输出一行，格式为 “Case X: Yes” 或 “Case X: No”。如果可以完成分组，输出 “Yes”；否则输出 “No”。

# DCEPC14C - The Long Pile Game

## 题目描述

Alice 和 Bob 在进行一场特殊的游戏。

他们选择了两个不同的正整数 $K$ 和 $L$，并从拥有 $N$ 个硬币的塔开始游戏。游戏规则是 Alice 先走，Bob 后走，然后再轮到 Alice，如此交替进行。在每一轮中，玩家可以从塔中取走 1、$K$ 或 $L$ 个硬币。

最终能够取走最后一个（或一组）硬币的人获胜。

经过长时间的游戏，Alice 发现某些情况下，她能够确保自己获胜，无论 Bob 如何应对。在其他情况下，只要 Bob 战略得当，他就能赢得比赛，不论 Alice 怎么出招。

现在给定整数 $N$、$K$ 和 $L$，请预测在双方都采取最佳策略的情况下，Alice 是否能取得胜利。

**数据范围**：

- $1 \le K \le 20$
- $1 \le L \le 20$
- $1 \le N \le 10^9$
- $T \le 10^5$

## 输入格式

第一行为一个整数 $T$，表示测试用例的数量。

接下来的 $T$ 行中，每行包含三个整数 $K$、$L$ 和 $N$。

## 输出格式

对于每个测试用例，如果 Alice 能赢，则输出 `Alice`（不包含引号），否则输出 `Bob`（不包含引号）。

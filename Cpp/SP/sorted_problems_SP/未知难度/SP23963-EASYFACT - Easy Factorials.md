# EASYFACT - Easy Factorials

## 题目描述

给定一个正整数 $n$,现在求 $n!$ 拆分为若干个连续正整数和的方法数。

## 输入格式

**本题采用多组数据。**

第一行一个正整数 $T$,表示数据组数。

接下来 $T$ 行，每一行 $2$ 个正整数 $n,m$，其中 $m$ 表示模数。

## 输出格式

$T$ 行，一行一个正整数，表示答案。

## 说明/提示

$n \le 10^8,m \le 10^9,T \le 5001$。

## 样例 #1

### 输入

```
1
3 7
```

### 输出

```
1
```

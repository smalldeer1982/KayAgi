# MOHIB - Mohib and series

## 题目描述

LBW 有一个不同的正整数序列，其平均值为$(a+1)$。

如果 LBW 在序列中加一个整数 $x$，那么序列中整数的平均值将是 $a$。

LBW 想知道序列中可能出现的最大整数。

## 输入格式

**本题有多组数据**。

第一行包含 $T$，表示测试用例的数量。

接下来的 $T$ 行包含一个整数 $x$和 $A$。

## 输出格式

对于每个测试点，输出序列中可能出现的最大整数。

## 说明/提示

对于全部数据：

$1 \le T \le 5000$。

$1 \le x \le 49999$。

$x+1 \le A \le 50000$。

## 样例 #1

### 输入

```
1
1 2
```

### 输出

```
3
```

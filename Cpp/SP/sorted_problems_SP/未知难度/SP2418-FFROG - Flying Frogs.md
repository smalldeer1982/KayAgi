# FFROG - Flying Frogs

## 题目描述

青蛙王国的国王 WiseFrog 临终前召集了他的两位聪明绝顶的儿子：SensibleFrog 和 SmartFrog。为了选出下任国王，WiseFrog 为他们设计了一个策略游戏。

比赛场地是一个 $m \times n$ 的矩形，每个小方格标记为 $(i, j)$（从左上角开始编号，$i=0,1,2,\ldots,m-1; j=0,1,2,\ldots,n-1$）。每个方格中随机分布着若干飞行青蛙，数量可以任意。

比赛开始后，由于 SensibleFrog 是国王的偏爱，他首先出手。两位王子轮流下达命令。在每个回合中，他们可以选择下达 1 到 $K$ 次命令，每个命令允许一只青蛙在指定方向上跳跃一定的距离（方向仅能为向上或向左，距离必须为正数），跳跃不能使青蛙离开场地。飞行青蛙可以轻松越过任意远的距离。

如果某一轮开始时，该王子发现没有任何可行的命令（即所有青蛙均已在左上角的方格中），则另一位王子获胜。根据提供的初始条件，你需要判断谁将成为青蛙王国的新国王。

## 输入格式

输入的第一行为一个整数 $T$，表示测试用例的数量。之后每个测试用例开始于一行，包含三个整数 $m, n, k$，依次表示矩形的行数、列数和每轮可执行的最大命令数。接下来的 $m$ 行中，每行包含 $n$ 个整数，代表相应位置的飞行青蛙数量。

## 输出格式

对于每个测试用例，输出一行。如果 SensibleFrog 获胜，打印 `SensibleFrog Wins!.`；否则，打印 `SmartFrog Wins!.`。

## 说明/提示

- $1 \le T \le 1000$
- $1 \le m, n \le 100$
- $1 \le k \le 100$
- 每个方格中的青蛙数量在 $0$ 到 $1000$ 之间。

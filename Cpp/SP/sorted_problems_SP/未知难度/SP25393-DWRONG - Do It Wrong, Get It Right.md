# DWRONG - Do It Wrong, Get It Right

## 题目描述

在小学里，学生们学习减分数时，通常会先找到一个共同的分母，然后再进行分子的运算。然而，有时候学生们即使用了错误的方法，也可能得到正确的答案。例如，对于以下计算：

5/4 - 9/12

一种错误的方法是直接相减分子，再相减分母，最终化简竟然也能得到正确答案，如下所示：

5/4 - 9/12 = -4/-8 = 4/8 = 1/2

现在，给定一个分数 $b/n$，你的任务是找出所有满足条件的分数 $a/m$，其中 $a \geq 0$ 且 $m > 0$，使得以下等式成立：

$$\frac{a}{m} - \frac{b}{n} = \frac{a-b}{m-n}$$

## 输入格式

输入包括多组测试数据。每组测试数据是一行，由两个整数 $b$ 和 $n$ 组成（满足 $1 < b < n < 1000$）。

## 输出格式

对于每组测试数据，在一行中输出所有符合条件的分数，按从小到大的顺序排列。如果有分数等价，输出其中分子较小的一个。每个分数用“a/m”形式表示，分数中的“/”两侧不应有空格，相邻分数之间用单个空格隔开。请勿多输出空格，也不要用空行隔开不同组的答案。

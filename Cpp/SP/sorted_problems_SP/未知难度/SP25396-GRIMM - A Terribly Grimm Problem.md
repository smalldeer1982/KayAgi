# GRIMM - A Terribly Grimm Problem

## 题目描述

Grimm 猜想指出，对于一组连续的合数，可以为每个合数分配一个不同的质因数。

例如，对于区间 242 到 250，可以为每个数分配不同的质因数，具体如下：

![](../../../content/joshkirstein:666)

现给定一个连续合数序列的下界和上界，请为其中的每个合数分配一个不同的质因数。如果存在多种可能的分配方案，请优先输出第一个质因数最小的方案。如果依然有多种方案，接着比较第二个质因数，以此类推。

## 输入格式

输入包含多组测试数据。每组测试数据占一行，包含两个整数 $l$ 和 $r$，分别表示合数序列的下界和上界（$1 \le l \le r \le 10^5$）。

## 输出格式

对于每个测试数据，请输出一组不重复的质因数，以空格分隔，按从小到大的顺序排列在同一行上。注意不需要输出多余的空格，也不必在不同答案之间添加空行。

- 输入满足 $1 \le l \le r \le 10^5$。

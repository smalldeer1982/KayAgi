# BITDIFF - Bit Difference

## 题目描述

给定一个 N 个整数的整数数组，求所有可以由数组元素形成的对中的位差之和。一对 （x， y） 的位差是 x 和 y 的二进制表示形式中相同位置的不同位的计数。例如，2 和 7 的位差为 2。2 的二进制表示形式是 010,7 是 111（第一位和最后一位在两个数字中不同）。

## 输入格式

输入以包含整数 T（1<=T<=100） 的行开头，表示测试用例的数量。然后 T 测试用例紧随其后。每个测试用例都以一行开头，该行包含整数 N（1<=N<=10000），表示数组中的整数数，后跟一行，其中包含 N 个空格分隔的 32 位整数。

## 输出格式

对于每个测试用例，以 Case X： Y 格式输出一行，其中 X 表示测试用例编号，Y 表示可以从模数数组元素形成的所有对中的位差之和10000007。

## 样例 #1

### 输入

```
1
4
3 2 1 4
```

### 输出

```
Case 1: 22
```

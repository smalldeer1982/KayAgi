# HIGHBABA - Highly Spiritual Baba

## 题目描述

#### 题目大意

有 $T$ 组数据，每组数据给定两个字符串 $a$、$b$，要求 $a$ 的一个**最短子串**，使它包含 $b$ 这个**子序列**。输出这个子串的长度，如果不存在输出 `-1`。

## 输入格式

第一行一个整数 $T$，表示数据组数。

对于每组数据：第一行输入 $a$，第二行输入 $b$。

## 输出格式

先输出字符串 `Case x: ` 其中 `x` 表示是第几组数据，然后输出答案。

## 说明/提示

$1 \le T \le 100$，$1 \le |a| \le 2500$，$1 \le |b| \le 80$。保证字符串中只出现小写英文字母。

## 样例 #1

### 输入

```
2
aabbaaccanbbbncc
abc

objectorientedprogramming
ii
```

### 输出

```
Case 1: 6
Case 2: 15
```

# NECSTASY - Ecstasy

## 题目描述

![ecstasy](http://3.bp.blogspot.com/-HxggbGGL_GM/VlVloZmYfpI/AAAAAAAABCU/vznikJ2Tzro/s1600/ecstasy.jpg "Ecstasy")

Swapnil 非常喜欢在公园荡秋千，这让他感到无比欢愉。然而，最近公园的地被用来修建公共建筑，Swapnil 因此感到失落。直到他的妈妈决定在他们家旁边安装一个新的秋千，这才让他重新振作。

现在，Swapnil 的妈妈 Swagata 面临一个新的问题：虽然秋千的位置已经确定，但是秋千的高度却需要仔细计算。

Swagata 在院子里设定了一个点 A，该点直接位于悬挂点的正下方，且与他们家房子的底部之间的垂直距离为 $d$ 米。为了安全，Swapnil 的脚需要在距离地面 $x$ 米，并且离建筑物 $y$ 米的高度处。只要荡秋千的角度不超过 $\theta$ 度，Swapnil 就能保持平衡。你的任务是帮助计算出秋千的最大高度 $h$。

## 输入格式

输入包含多个测试用例。每个测试用例的第一行由三个正浮点数 $d, x$ 和 $y$ 组成，单位为米，并满足 $0 < x, y < d < 100$。接下来的第二行是一个浮点数，表示最大的荡秋千角度 $\theta$，单位为度，满足 $0 < \theta < 180$。可以确保每个测试用例都有有效解。

## 输出格式

对于每个测试用例，输出秋千的最大高度 $h$，单位为米，结果保留两位小数。

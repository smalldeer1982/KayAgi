# HELPBTW - Help BTW

## 题目描述

BTW打算给她的男朋友买礼物，她计划购买一串整数数组。由于整数数组大多价格昂贵，BTW从市场上购买了最便宜的数组。但是她的男朋友非常抠门，往往按照数组中**最小的元素**来评估数组的质量。因此，她决定提高这串数组的质量。对数组元素进行增加操作的成本非常高，需要**一整天的时间**才能完成。此外，可以**同时增加最多M个连续元素**。她还剩下剩下K天的时间。请帮助BTW计算她可以获得的阵列的最大可能**质量**。

P.S. BTW是人名

## 输入格式

第一行为样例数T
每个测试用例的第一行为数组大小N、M和剩余天数K，第二行为N个整数，表示数组的值。

## 输出格式

输出BTW可达到的最大数组质量的值

## 样例 #1

### 输入

```
3 2 1
2 2 3
```

### 输出

```
3
Sample test case 2
Input
3 2 1
2 3 2
Output
2
```

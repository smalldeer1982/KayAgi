# CRICKDP - Cricket Selection 

## 题目描述

他已经参加了一些比赛，并且对自己在每场比赛中的表现进行了评分。为了尽可能提高他的总评分，他考虑通过贿赂裁判来提升部分比赛的评分。

## 输入格式

- 第一行输入一个整数 $T$，表示测试用例的数量。
- 对于每个测试用例，第一行包含三个整数 $N, K, M$，分别表示参加比赛的场次、他拥有的金钱总量以及他可以贿赂的裁判数目。
- 接下来的行中包含 $N$ 个整数，表示他在每场比赛中的初始评分。
- 接下来的 $M$ 行中，每行包含三个整数 $L_i, R_i, C_i$，表示他可以贿赂裁判的比赛场次范围是第 $L_i$ 场到第 $R_i$ 场，并且需要支付的费用是 $C_i$。

## 输出格式

对于每个测试用例，输出可能获得的最大总评分，结果显示在单独的一行。

## 说明/提示

- $1 \leq T \leq 10^5$
- $1 \leq N \leq 10^5$
- $0 \leq K \leq 10^9$
- $0 \leq M \leq 10^5$
- $1 \leq L_i \leq R_i \leq N$
- $1 \leq C_i \leq 10^9$
- 每场比赛的评分都在 $1$ 到 $10^9$ 之间

通过这些信息，你需要决定如何妥善使用他的资金来获取最大可能的总评分。

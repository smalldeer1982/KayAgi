# BALLSUM - Ball sum

## 题目描述

你有一个装满 $N$ 个球的袋子。每个球上有一个从 $1$ 到 $N$ 的不同数字。你要从袋子里取出两个球。你需要计算两个球上数字之和不大于给定数 $K$ 的概率。

## 输入格式

输入由任意组测试用例组成。每组测试用例包含两个整数 $N$ 和 $K$（$0 \le K \le N \le 10^9$）。

当 $N,K=-1$ 时，结束输入。

## 输出格式

每行两个整数（两个球上数字之和不大于 $K$ 的概率），以 `p/q` 的形式输出（除非答案为 $0$ 或 $1$）。

## 样例 #1

### 输入

```
3 2
100 5
10 6
-1 -1
```

### 输出

```
0
2/2475
2/15
```

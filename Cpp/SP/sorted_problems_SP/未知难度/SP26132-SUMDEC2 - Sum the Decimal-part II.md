# SUMDEC2 - Sum the Decimal-part II

## 题目描述

在这道题目中，你需要计算给定数的平方根后，小数部分前 1000 位的数字之和（忽略整数部分）。

举个例子，假设给出的数是 2。那么 2 的平方根为 1.4142135623......  
在这个例子中，你需要忽略小数点之前的数字（即 1），然后将小数部分的前 1000 位数字相加，最后输出结果——在这个例子中是 4482。

**特别注意：如果给出的数是一个完全平方数，那么输出结果应该是 0。**

## 输入格式

输入的第一行是整数 $t$，表示测试用例的数量。  
接下来的 $t$ 行中，每行包含一个非负整数 $n$。

## 输出格式

请为每个整数 $n$ 输出单独一行结果。

#### 数据范围与提示

1 ≤ t ≤ 100  
1 ≤ n ≤ 100000

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
2
4
2
```

### 输出

```
0
4482
```

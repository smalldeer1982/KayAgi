# PFACTORS - Pisano Factors

## 题目描述

给定一个整数 $n$。

计算满足条件的整数 $c$ 的个数，使得它们的皮萨诺周期是 $n$ 的因数。

其中 $1 \le c \le 10^5$。

注意：题目包含多个测试用例。

## 输入格式

第一行输入一个整数 $t$，代表测试用例的数量，$1 \le t \le 100$。

接下来有 $t$ 行，每行输入一个正整数 $n$，$1 \le n \le 10^9$。

## 输出格式

对于每个测试用例，输出相应的答案，每个答案占一行。

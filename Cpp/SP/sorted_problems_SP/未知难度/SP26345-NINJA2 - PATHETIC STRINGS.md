# NINJA2 - PATHETIC STRINGS

## 题目描述

如果一个字符串中所有字符出现的次数都相同，那么这个字符串就被称为“PATHETIC”。现在给你一个长度为 $n$ 的字符串，字符串仅由小写字母组成。你的任务是计算出，将这个字符串转换成“PATHETIC”所需进行的最少字母替换次数。替换操作允许将一个字母变为任意其他字母。

## 输入格式

第一行输入一个整数 $T$，表示测试用例的数量。接下来的 $T$ 行中，每一行都是一个由小写字母构成的字符串，代表一个测试用例。

## 输出格式

对于每个测试用例，输出一行表示将其变成“PATHETIC”所需的最少替换次数。

## 说明/提示

- $1 \le T \le 10^5$
- $1 \le n \le 10^5$

**样例输入：**

```
2
bbaccaaa
ccaacb
```

**样例输出：**

```
2
1
```

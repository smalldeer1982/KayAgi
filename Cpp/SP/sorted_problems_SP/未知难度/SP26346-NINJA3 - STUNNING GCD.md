# NINJA3 - STUNNING GCD

## 题目描述

### 题目背景：

给定三个正整数 $N,a,b$，通过将 $N$ 重复 $a$ 次得到 $X$，将 $N$ 重复 $b$ 次得到 $Y$，请求出 $gcd(X,Y)$。

## 输入格式

第一行有一个正整数 $T$，代表有 $T$ 组测试数据，每组测试数据仅一行，依次给定正整数 $N,a,b$。

## 输出格式

每组数据一行，给出 $gcd(X,Y)$ 的值。

### 输入输出样例：

见原题目

### 样例解释：

$N$ 为 $123$，重复两次为 $X$，即 $X=123123$，重复三次为 $Y$，即 $Y=123123123$，$gcd(X,Y)=123$

## 说明/提示

$1\le T\le 100$

$1\le N,a,b \le 10^{18}$

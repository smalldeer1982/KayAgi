# PUTIN - Put a Point in a Hyperspace

## 输入格式

首先，输入一个整数，表示测试用例的数量。

对于每个测试用例：

输入一行，包含三个空格分隔的整数 K、S 和 M。其中 K 表示超空间的维度数

（2<=K<=30），S 表示点 B 的坐标和的上限（2<=S<=10000），M 表示给出的点的数量（0<=M<=20）。

接下来输入 M 行，每行包含 K 个非负整数，表示在 K-D 超空间中的一个点的坐标。每个点的坐标用空格分隔，且这些点各不相同，并且都不在任何坐标轴上。

## 输出格式

对于每个测试用例，输出一个整数，表示满足以下条件的点 B（b1, b2, ..., bk）的数量：

点 B 不在任何坐标轴上。
对于每个给出的点（a1, a2, ..., ak）（1<=i<=M），存在至少一个维度 j（1<=j<=k），使得 bj < aij。
对于每个维度 j（1<=j<=k），bj 是一个非负整数。
点 B 的坐标和（即 b1 + b2 + ... + bk）不超过 S。

## 样例 #1

### 输入

```
1
2 4 2
1 3
2 1
```

### 输出

```
2

Hint
The two points are (1,1) and (1,2).
```

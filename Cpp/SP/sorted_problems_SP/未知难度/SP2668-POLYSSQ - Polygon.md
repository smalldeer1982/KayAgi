# POLYSSQ - Polygon

## 题目描述

你在飞机上有N个不同的点。没有任何三个是共线的。编写一个程序，找出从给定点取K个顶点的凸多边形的最小面积。

## 输入格式

两个整数N和K被写入标准输入的第一行。它遵循N条线，每条线包含对应给定点的一对坐标。输入中每行上的两个数字用空格隔开。约束条件：0<N<50，0<K<11。给定点的坐标是非负整数，小于9999。

## 输出格式

你的程序必须输出一个整数，它等于最小面积的整数部分。如果不存在如上所述的任何凸多边形，则程序必须输出0。

## 样例 #1

### 输入

```
4 3
0 0
1 1
0 10
10 0
```

### 输出

```
5
```

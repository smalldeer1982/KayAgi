# ICCSECRET - Secret Key

## 题目描述

给定一个仅由 `A`,`B` 组成的字符串，你需要找出一个最小的与之对应的正整数。

我们称一个仅由 `A`,`B` 组成的字符串 $b$，与一个正整数 $a$ 对应（$a_i$ 表示 $a$ 在十进制下从高位往低位数的第 $i$ 位的数），当且仅当：

- $a$ 中 $1$ 到 $9$ 不重复出现，且每一个数均属于 $[1,9]$。
- 若 $b_i=$ `A`，则 $a_{i+1}>a_i$。
- 若 $b_i=$ `B`，则 $a_{i+1}<a_i$。
- $|a|=|b|+1$。

## 输入格式

第一行一个正整数 $T$，表示测试数据组数。

接下来 $T$ 行，每行一个字符串。

## 输出格式

输出共 $T$ 行，每行一个正整数，表示最小的与该字符串对应的正整数。

## 说明/提示

字符串长度不超过 $8$。

这里补充一下输出：

```C++
125436
43215
```

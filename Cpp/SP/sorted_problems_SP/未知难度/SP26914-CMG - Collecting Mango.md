# CMG - Collecting Mango

## 题目描述

风暴过后，米娜带着篮子去花园采摘芒果。她开始一个接一个地将芒果放进篮子，如果愿意，还可以把最后一个放入的芒果扔掉。在摘芒果的过程中，米娜需要你帮助记录篮子里芒果的最大大小。任何时候她都可能问你哪个芒果最大，你要时刻准备作答。

作为一名优秀的程序员，你需要编写一个程序以帮助米娜回答以下问题与指令：

1. 类型 1：将一个大小为 $x$ 的芒果放入篮子。
2. 类型 2：将最后放入篮子的芒果扔掉。（这种操作可能会连续出现。）
3. 类型 3：询问当前篮子里的芒果哪个最大。

## 输入格式

输入的第一行为正整数 $T$，表示测试用例的数量。接下来每个测试用例包含一个正整数 $N$，表示米娜在摘芒果过程中要执行的操作总数。接下来的 $N$ 行，每行代表一种操作（A, R, Q）。`A x` 表示放入一个大小为 $x$ 的芒果；`R` 表示扔掉最后放入的芒果；`Q` 表示询问当前篮子里最大的芒果的大小。

## 输出格式

对于每个测试用例，先输出测试用例编号，然后输出米娜每次询问的答案。如果篮子此时为空，则回答 `Empty`。

## 说明/提示

- $1 \le T \le 25$
- $1 \le N \le 100000$
- $1 \le x \le 100000$

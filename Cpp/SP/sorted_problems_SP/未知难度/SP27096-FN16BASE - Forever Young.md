# FN16BASE - Forever Young

## 题目描述

## 翻译
#### 我要过生日了！为了让我看起来更年轻，我会把我的年龄用 $b$ 进制来表示。年龄的每一位只能由 $0～9$ 组成，$b$ 进制下我的年龄的表示方式转换到十进制后也不能太小。否则我的伎俩就会被识破！
#### 给出我的年龄 $y$ 与 $b$ 进制下我的年龄的表示方式转换到十进制后的最小值 $l$。求出最大的 $b$。

## 输入格式

#### 每一个测试点里含有多组数据。每组数据占一行。每行含有两个十进制整数 $y$$(10≤y≤10^{18})$ 和 $l$$(10≤l≤y)$。

## 输出格式

#### 对于每组数据，输出最大的进制 $b$。
Translated by @[Mysterious_Mini](https://www.luogu.com.cn/user/476150)

## 样例 #1

### 输入

```
32 20
2016 100
```

### 输出

```
16
42
```

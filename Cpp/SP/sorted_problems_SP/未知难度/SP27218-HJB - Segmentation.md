# HJB - Segmentation

## 题目描述

Bobs刚从他母亲Ksenxi那里得到了一个有n个整数的数组A。她让他算一算A数组的好处。

## 输入格式

输入的第一行是一个整数n。
第二行数组A1,A2,……An

## 输出格式

打印出一个整数——a的优点，因为答案可能是一个很大的数，你应该打印它的模10 ^ 9 + 7 (1000000007))

## 样例 #1

### 输入

```
5
5 3 0 9 8
```

### 输出

```
21
```

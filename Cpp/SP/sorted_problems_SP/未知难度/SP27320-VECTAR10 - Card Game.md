# VECTAR10 - Card Game

## 题目描述

Changu和Mangu的有序牌组有n张牌，编号从1到n，顶牌1，底牌n。他们想玩游戏。只要甲板上至少有两张纸牌，他们就会执行奇怪的操作：Changu扔掉顶部的纸牌，然后Mangu将现在在纸牌顶部的纸牌移到纸牌的底部。您的任务是找到最后剩余的卡。

## 输入格式

输入的每一行（除了最后）包含一个正数 N ，最后一行包含 0 和这个行不应被处理。

## 输出格式

对于输入中的每个数字，产生一行输出，给出最后剩余的卡片。

## 说明/提示

$N \leq 10^6$，最多有 $5\times10^5$ 行输入。

## 样例 #1

### 输入

```
2
3
4
0
```

### 输出

```
2
2
4
```

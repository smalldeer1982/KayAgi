# NDS - Increasing numbers

## 题目描述

Subham 和 Dewang 在玩一个数字游戏。Subham 给 Dewang 一组数字，要求他找出一个长度为 **L** 的递增序列中可能的最小最后一个数字。

**提示：** 请查看样例输入输出以获得更具体的理解。

## 输入格式

输入的第一行是测试用例的数量 **T**。接下来的每个测试用例以三行数据表示。第一行是数组的大小 **N**。第二行是 **N** 个用空格分开的数组元素。第三行是长度为 **L** 的递增序列。

## 输出格式

对于每个测试用例，输出一个递增序列长度为 **L** 的最小可能最后一个数字。如果这样的序列不存在，则输出 `-1`。

## 说明/提示

- $1 \le T \le 100$
- $1 \le N \le 10^5$
- $1 \le L \le N$
- $1 \le \text{数组元素} \le 10^9$

### 输入

```
1
7
9 7 2 5 4 11 12 
3
```

### 输出

```
11
```

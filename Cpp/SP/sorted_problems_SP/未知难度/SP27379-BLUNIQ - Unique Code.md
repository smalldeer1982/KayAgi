# BLUNIQ - Unique Code

## 题目描述

一个城市，来了几位游客。每位游客都有一个最喜欢的数字，要给每位游客都安排一个密码，其中每位游客的密码都不能重复。并且密码要尽量接近该游客的喜爱数字，如果该密码已经有人有了，就选择比这个数大并且最接近的数字作为密码，如果这个数也被选了则再进行这个操作。

PS：如果游客离开了那他的密码就又可以用了。

## 输入格式

第一行，为操作次数N。

接下来N行，每行两个数`x` `a`，若x为1则表示来了一位喜爱数字为a的人，若x为2则表示幸运数字为a的游客离开了。

## 输出格式

对于每行x为1的输入，输出一个数，表示该游客的密码。

## 样例 #1

### 输入

```
5
1 42
1 42
1 43
2 43
1 42
```

### 输出

```
42
43
44
43
```

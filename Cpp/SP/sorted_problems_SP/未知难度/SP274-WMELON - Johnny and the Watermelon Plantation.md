# WMELON - Johnny and the Watermelon Plantation

## 题目描述

Johnny 要种西瓜，于是，他卖掉了所有的啤酒瓶，用这些钱买了一块 1km × 1km 的田地，他种下了种子，令所有人惊讶的是，很多种子都发芽了，很快，许多植物都出现了果实的迹象（有些甚至不止一个）。很出乎意料的是，当西瓜还未成熟到不能吃的时候，冬天开始了。约翰尼知道他必须建造一座温室来保护田地，但由于预算有限，他买不起覆盖整个地区的玻璃。他决定让 $k$ 个水果在玻璃屋顶下经受住考验就足够了。出于建筑规划的考虑，温室必须是一个矩形，两侧与地块边缘平行。

## 输入格式

输入的第一行包含整数 $t \le 100$，即测试用例数。每个测试用例从第一行开始，包含两个整数 $n, k$，分别表示要保护的植物总数和西瓜果实数（$1 \le n \le 1000$，$1 \le k \le {10}^6$，$k$ 不超过西瓜植株总数）。接下来的 $n$ 行中的每一行描述一个植株，第 $i$ 行包含三个整数（植株的 $x$ 和 $y$ 坐标，以及植株上西瓜果实的数量（$1 \le X_i$））

## 输出格式

对于每个测试用例，输出一个整数，表示足以覆盖种植园的至少 $k$ 个水果的最小矩形玻璃屋的面积。

## 样例 #1

### 输入

```
1
6 11
1 1 2
1 2 2
3 1 2
3 2 3
4 2 5
3 3 2
```

### 输出

```
2
```

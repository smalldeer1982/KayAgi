# VECTAR13 - Changu Mangu in a Football Team

## 题目描述

Changu 和 Mangu 是一支准备参加足球锦标赛的球队成员。在这个锦标赛中，总共有 $n$ 支球队参加。每支球队将与其他每支球队进行两场比赛（主客场各一场）。获胜的球队将获得 3 分，打平的双方各得 1 分，而失败的球队不得分。

比赛结束后，球队将根据总积分进行排名，排名从 1 到 $n$。某个积分为 $p$ 的球队 $t$ 的排名等于比它积分高的球队数量加 1。可能会有多支球队获得相同的排名。

除了获得第一名的球队，还有一支可能被冠以“幸运”称号的球队。这支“幸运”球队必须是赢得比赛最多、进球数最多、失球数最少的球队（这些条件必须是绝对唯一的，即没有其他球队能与之相同）。

Changu 一直梦想成为“幸运”球队的一员。你的任务是确定“幸运”球队可能的最差排名。

## 输入格式

第一行为整数 $T$，表示测试用例的数量。接下来的 $T$ 行，每行包含一个整数 $n$，表示参赛球队的数目。

## 输出格式

对于每个测试用例，输出“幸运”球队可能的最差排名。

## 说明/提示

- $1 \le T \le 10^5$
- $1 \le n \le 10^{18}$

**示例**

输入：

```
2
1
3
```

输出：

```
1
1
```

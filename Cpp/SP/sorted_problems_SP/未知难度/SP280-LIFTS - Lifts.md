# LIFTS - Lifts

## 题目描述

## 题目背景

塞尔吉非常喜欢游戏。最近他在电脑里发现了一款街机游戏。控制英雄在地图上移动并收集各种物品。在游戏的某个阶段，塞尔吉遇到了一个的问题。为了继续他的冒险，英雄应该越过一道沟。为此，可以使用位置一致的电梯，看起来像平台。每部电梯在某些楼层之间上下移动。英雄可以在下一个相邻的平台之间通过，但是只有在他们处于同一水平时才能通过。同样，只有当电梯出现在边缘水平时，才能从裂缝边缘进入电梯。


沟的边缘在同一水平面上。对于每一次提升，给出了移动的高度范围、初始位置、初始时间和移动方向。所有电梯都以每秒一米的速度移动。判断英雄是否能到达沟的另一边，如果可以，那么达到这个目的所需的最短时间是多少。

## 输入格式

一个正整数n表示开始提升的次数，（n<=100），然后是n行。第i行（0<i<=n）包含四个整数li、ui、si、di，

li表示电梯的最低位置

ui表示电梯的最高位置

si表示电梯的初始位置

di表示移动的初始方向（1表示向上，−1表示向下）（−100<=li<=si<=ui<=100，l1<ui）。

## 输出格式

输出到达沟的另一端所需的最短时间。如果不可能则输出-1。

## 样例 #1

### 输入

```
1

4
-1 2 1 -1
0 3 0 1
-4 0 0 -1
-2 1 0 -1
```

### 输出

```
29
```

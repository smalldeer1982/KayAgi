# CATINV - Cats Invitation

## 题目描述

库图斯和普图斯的结婚一周年纪念日，他们决定邀请所有的猫朋友来庆祝。由于各自的缘故，猫们无法同时出席，有些提前到达，有些中途离开。库图斯希望让普图斯尽可能的快乐。在聚会的某一时刻，如果有 $L$ 只猫在场，普图斯的快乐指数就是 $L$。库图斯记录了每只猫到达和离开的时间，并希望你帮忙解答几个问题：普图斯的快乐指数为 $L$ 的最长时间段。

## 输入格式

输入首先包含一个整数 $T$，表示测试用例的数量。每个测试用例首先会有两个整数 $N$ 和 $Q$，分别表示到访的猫的数量和要进行的查询数量。接下来的一行包含 $N$ 对整数 $X_i$ 和 $Y_i$，表示第 $i$ 只猫的到达和离开的时间。之后的 $Q$ 行，每行包含一个整数 $L$，即查询普图斯幸福指数为 $L$ 的最长时间段。

## 输出格式

对于每个测试用例，输出“Case #x: ”，其中 x 是测试用例编号，接着给出普图斯幸福指数为 $L$ 的时间段 $X$ 和 $Y$。如果有多个解，输出最早的时间段。如果没有满足条件的时间段，输出 -1。

**数据范围与提示:**

- $T \le 10$
- $1 \le N \le 100000$
- $1 \le Q \le 100000$
- $1 \le X_i \le Y_i \le 100000$
- $1 \le L \le 100000$

# BLMIRANA - Mayonnaise Arrow

## 题目描述

Mirana 是一位拥有超能力的神射手。她的箭在飞行路径越长时威力越大。目前，Mirana 正在一个战场上。

很巧，一个商人正在卖蛋黄酱。借助蛋黄酱的神奇力量，Mirana 的箭能够穿透所有的障碍物。但是，这份蛋黄酱只能支持一支箭的发射。

Mirana 站在坐标系的原点 (0, 0)。从这个位置出发，她需要用一支箭尽可能多地击中敌人。每个敌人都有一个圆形的命中区域。如果箭穿过这个命中区域，敌人就被击败了。请找出用一支箭最多能击中多少个敌人。

## 输入格式

第一行包含一个整数 $T$，表示战斗轮数。对于每一轮战斗，第一行包含一个整数 $N_i$，表示敌人的数量。接下来的 $N_i$ 行中，每行都有三个整数 $x$、$y$ 和 $r$，表示敌人的命中区域的圆心坐标和半径。

- $N_i < 125000$
- $N_1 + N_2 + \ldots + N_T < 500000$
- $1 < r < x < 1000$
- $-1000 < y < 1000$

## 输出格式

对于每轮战斗，输出一行，表示用一支箭最多能击中的敌人数目。

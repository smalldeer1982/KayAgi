# ADAQUEUE - Ada and Queue

## 题目描述

瓢虫艾达有很多事情要做。她把它们放到她的队列中。无论如何，她非常优柔寡断，所以有时她使用顶部，有时使用背面，有时她决定反转它。

## 输入格式

第一行由 1 、 查询数组成。每个命令都包含以下命令之一

back - 从背面打印数字，然后擦除它

front - 从前面打印数字，然后擦除它

reverse - 反转队列中的所有元素

push_back **N** - 将元素 **N** 添加到底层

toFront **N** - 将元素 **N** 置于最前面

所有数字均为 0

## 输出格式

对于每个后/前查询，请打印适当的数字。

如果您将收到这种类型的查询并且队列为空，请改为打印 **“No job for Ada？”**。

输入输出样例
无

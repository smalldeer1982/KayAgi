# TAP2016L - Leonardo de Pisa

## 题目描述

莱昂纳多·德·皮萨是个非常谨慎的人。尽管距离圣诞节还有几个月，他已经提前买好了圣诞树。这颗树非常高，甚至比比萨斜塔还要高。莱昂纳多打算用彩色球和灯串来装饰这棵树。他购买了很多球，这些球的直径是从 1 到 $N$ 的所有整数值。实际上，他买的球太多了，甚至不知道该如何搭配使用。

输入文件包含多个测试用例。每个测试用例的第一行给出两个整数 $N$ 和 $K$，分别表示球的最大直径和灯串中灯的数量（$2 \le K \le 10^5$）。第二行包含 $K$ 个整数 $L_1, L_2, \ldots, L_K$，表示灯串的灯的类型。其中，第 $i$ 个整数 $L_i$ 表示灯串中第 $i$ 个灯泡的种类（$1 \le L_i \le N$，$i$ 从 $1$ 到 $K$）。

## 输出格式

对于每个测试用例，输出一行，一个字符，表示莱昂纳多是否可以将灯串成功放置。若可以，输出字符 'S'；否则，输出字符 'N'。

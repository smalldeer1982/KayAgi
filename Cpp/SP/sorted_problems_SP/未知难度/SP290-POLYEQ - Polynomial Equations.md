# POLYEQ - Polynomial Equations

## 题目描述

给你一个多项式 $F(x)$，它由多个单项式组成。每个单项式的形式是：
\[
\text{系数} \times x^{\text{指数}}
\]
或仅为一个常数（即系数）。其中，$\text{系数}$ 和 $\text{指数}$ 为整数，并满足 $-30000 \leq \text{系数} \leq 30000$，$0 \leq \text{指数} \leq 6$。方括号中出现的参数可以省略。

你的任务是找到这个方程 $F(x) = 0$ 的所有解。

## 输入格式

第一行是一个整数 $t$，代表测试用例的数量。接下来有 $t$ 行输入，每一行是一个字符串形式的多项式 $F(x)$。该字符串的格式如上所述，且长度不超过 300 个字符。

## 输出格式

对于每个测试用例，输出所有满足方程的解（包括重根），按从小到大的顺序排列。这些解在区间 $[-100.0, 100.0]$ 内，每个解的精度误差不超过 $0.01$。保证所有解均为实数。

## 说明/提示

- $1 \leq t \leq 100$
- $-30000 \leq \text{系数} \leq 30000$
- $0 \leq \text{指数} \leq 6$
- 字符串的长度不超过 300 个字符。

# WOWSUBSTR - Counting WOW-Substrings

## 题目描述

## 题意

定义一个 WOW 字符串为不含重复字符的字符串，求出给定字符串中子串是 WOW 字符串的长度 （$\bmod 100007$）。

## 输入格式

一个整数 $T$，表示测试组数。

对于每组数据，一行一个字符串表示给定字符串。

## 输出格式

对于数据 $i$，输出 $\text{Case }i: ans$，$ans$ 表示答案。

## 样例输入
```
2
aaa
ab
```
## 样例输出

```
Case 1: 3
Case 2: 4
```

# BLLUCK - Coincidence

## 题目描述

Toriel 正在筹划一场带有奖品的游戏。在这场游戏中，有 $N$ 名参与者和 $N$ 个盒子，每个盒子里都藏有一张写有某位参与者名字的纸条。每位参与者的名字都是独一无二的，并且盒子中的名字也不重复。游戏最多可以有 $1000$ 名参与者。

接下来，每位参与者随机选择一个盒子，确保每位参与者获得的盒子都不相同。如果某位参与者获得了标有自己名字的盒子，那么获奖几率将会增大。为了设计奖品，Toriel 需要计算出期望中能够拿到自己名字的参与者的人数。

## 输出格式

针对每局游戏，输出获得自己名字的参与者的期望数量。请保留小数点后 $6$ 位精度。

## 说明/提示

$$1 \le N \le 1000$$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
1\n1&nbsp;
```

### 输出

```
1.000000
```

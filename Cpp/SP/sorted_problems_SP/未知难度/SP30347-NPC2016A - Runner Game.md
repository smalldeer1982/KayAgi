# NPC2016A - Runner Game

## 题目描述

今天，瓦卡将玩跑步游戏。在这个游戏中，玩家将被放置在一个方形竞技场中，他们将跑到竞技场的检查站。\
游戏的竞技场是一个边长为N的正方形。玩家将被放置在竞技场内的坐标$(X,Y)$处。以下是游戏规则：
- 玩家最初将位于坐标$(X,Y)$处，其中坐标$(0,0)$是竞技场的左下角。
- 然后，玩家应该跑到竞技场的一边去检查点位于竞技场的每一侧（顶部、左侧、底部、右侧），玩家应接受所有检查点（即他们必须访问所有侧面），然后回到玩家的初始位置，即$(X$、$Y)$。
- 如果玩家到达竞技场的某个角落，他可以通过$2$个检查站。例如，如果玩家到达坐标$(0,0)$，则认为他获得了左侧和底部的检查点。\
现在，瓦卡很好奇，到达所有检查站并回到最初位置的最小距离是多少？下图是跑步者游戏的可能动作之一。（红色圆圈是瓦卡的初始位置$(X,Y)$）
![](https://cdn.luogu.com.cn/upload/vjudge_pic/SP30347/c46a9e6ebeeab7b507ce368e63cf409e5b56c412.png)

## 输入格式

- 输入包含$3$个整数，$X$、$Y$和$N$。

## 输出格式

- 输出瓦卡完成游戏所需的最小距离。如果差值不超过$10^{-6}$，则你的答案就是正确的。

## 样例 #1

### 输入

```
6 4 10
```

### 输出

```
28.284271
```

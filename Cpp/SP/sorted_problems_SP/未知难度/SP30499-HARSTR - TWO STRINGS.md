# HARSTR - TWO STRINGS

## 题目描述

给你两个字符串 $a$ 和 $b$。你的任务是从字符串 $b$ 中删除最少数量的连续字符，使得剩下的字符串成为 $a$ 的「子多重集」。可能不需要删除任何字符，也可能需要将 $b$ 清空。

## 输入格式

第一行是字符串 $a$，第二行是字符串 $b$。这两个字符串不为空，只包含小写英文字母。每个字符串的长度最长为 $10^5$。

## 输出格式

在第一行输出按照字典序排列的字符串，表示从 $b$ 中获得的 $a$ 的「子多重集」。如果存在多个可能的答案，输出字典序最小的那一个。

如果最终结果为空，请输出 `«-»`（减号）。

## 说明/提示

- $1 \le |a|, |b| \le 10^5$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
abacaba\nabcdcba
```

### 输出

```
aabbc
```

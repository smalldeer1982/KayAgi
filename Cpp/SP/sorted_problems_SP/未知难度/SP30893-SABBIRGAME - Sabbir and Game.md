# SABBIRGAME - Sabbir and Game

## 题目描述

Sabbir 是个爱玩游戏的小男孩。一天，朋友 Taskin 给他提议了一个有趣的新游戏。游戏里有 $n$ 个关卡，玩家需要在每个关卡保持正的生命值才能顺利通过。在某些关卡中，打败反派会提升生命值，而在另一些关卡中，若无法打败反派则会损失生命值。Sabbir 已经知道在每个关卡会增加或减少多少生命值。请你计算出，在游戏开始时，Sabbir 至少需要多少生命值才能保证在任何时候生命值都是正的，从而通过所有关卡。

## 输入格式

输入最多包含 100 组测试案例。

第一行是一个整数 $T$，表示测试案例的数量（$1 \le T \le 100$）。

每个测试案例由两行组成。第一行是一个整数 $n$，表示关卡的数量（$1 \le n \le 1000$）。

第二行包含 $n$ 个用空格分隔的整数 $a_1, a_2, \ldots, a_n$，表示每个关卡中生命值的变化（$-10^7 \le a_i \le 10^7$）。

## 输出格式

对于每组测试数据，输出一个整数，表示 Sabbir 最少需要的初始生命值。

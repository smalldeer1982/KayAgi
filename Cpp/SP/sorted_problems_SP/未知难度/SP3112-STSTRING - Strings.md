# STSTRING - Strings

## 题目描述

现有两个字符串 $A$ 和 $B$，我们为字符串 $C$ 定义一个运算符 $\hat{c}$，其表示为：$C \hat{c} \{A, B\}$。

```
如果 $C$ 的长度大于 $A$ 且小于 $B$，则 $C$ 满足该运算符。
否则：
    如果 $C$ 的长度等于 $A$，那么 $C$ 必须在字典序上大于 $A$。
    如果 $C$ 的长度等于 $B$，那么 $C$ 必须在字典序上小于 $B$。
```

## 输入格式

输入为两个字符串 $A$ 和 $B$，其中 $\text{length}(A) \leq \text{length}(B) \leq 6$。字符串中的字符均为大写字母 A 到 J。

## 输出格式

输出满足上述所有条件的字符串 $C$ 的总数量。此外，字符串 $C$ 必须满足以下条件：字符串 $C$ 中任意两个相邻字符不能相同，也不能是连续的（换句话说，相邻字符的 ASCII 值之差绝对值必须大于 1）。

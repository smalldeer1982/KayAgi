# PALIM - Yet Another Longest Palindrome Problem

## 题目描述

一个字符串称为回文串，如果它正着读和反着读是一样的。例如，「abdba」是回文串，而「abbaa」则不是。

现在给你一个字符串，请找出在该字符串中出现的最长连续回文子串，并输出它的长度。

## 输入格式

- 第一行：一个最多包括 $100000$ 个字符的字符串。所有字符的 ASCII 编码范围在 $32$ 到 $127$ 之间。
- 第二行：一个用于安全验证的魔法密钥。

## 输出格式

- 第一行：输出找到的最长回文子串的长度。
- 第二行：输出用于安全验证的魔法密钥。

## 说明/提示

- 字符串长度满足 $1 \le |s| \le 100000$。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
abaabbabaaba
MAGICAL KEY
```

### 输出

```
6
MAGICAL KEY
```

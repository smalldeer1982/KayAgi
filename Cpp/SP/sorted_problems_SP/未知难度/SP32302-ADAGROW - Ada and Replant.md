# ADAGROW - Ada and Replant

## 题目描述

你可能已经知道，瓢虫艾达是一个农民。此刻，她所有的蔬菜都在一条沟里。她打算把它们重新种植到一些新的犁沟里（同时保持蔬菜的顺序）。

种植蔬菜的总成本将等于相邻蔬菜之间差的绝对值之和。艾达想把成本降到最低，你能帮她吗？

## 输入格式

第一行输入包含一个整数 $T$ ，**表示有 $T$ 个测试用例。**

每个测试用例的第一行包含两个整数 $N$ 、 $K$

下一行包含 $N$ 个整数 $a_i$ ，**即蔬菜的成本。**

**注意**：测试用例的数量因数组的大小而异（最长的数组不会是多个文件）。

## 输出格式

对于每个测试用例，输出最低成本。

### 输入输出样例
无

# WINDMILL - The Tall Windmills

## 题目描述

在职业生涯的后期，Johnny 买下了一块狭长的土地，计划在上面建造一排风车，希望能依靠这些风车产生的电量来生活。然而，他很快发现自己被坑了——大多数时间，风都是顺着土地的长向吹，而不是横着刮。这样一来，风在遇到第一个风车时就会耗尽大部分动力，导致后面的风车无法运行。为了解决这个问题，Johnny 想出了一个办法：让相邻的风车高度有所不同。

具体来说，Johnny 准备用相同的间距（一个比特兰德弗隆）沿直线排列建造 $n$ 个风车。经专家确认，如果两个风车相距 $k$ 个比特兰德弗隆，那么它们的高度差至少要有 $n-k$ 个比特兰德码。每个风车的高度至少为 1 比特兰德码，而有些风车可能需要更高。然而，高昂的建造成本使得要尽量保证最高的风车高度最小。因此，你需要帮助选择各个风车的高度配置，从而使得最高的风车也尽量矮一些。

## 输入格式

首先输入一个整数 $t$，表示测试用例的数量（$t \leq 100$）。接下来是 $t$ 组测试用例。

每组测试用例包含一个整数 $n$（$1 \leq n \leq 100$），表示 Johnny 打算建造的风车数量。

## 输出格式

对于每组测试用例，输出一行，包含 $n$ 个整数，表示顺序排列的风车的高度配置。

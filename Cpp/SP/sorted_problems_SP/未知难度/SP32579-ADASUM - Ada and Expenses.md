# ADASUM - Ada and Expenses

## 题目描述

瓢虫 Ada 给你一个十进制数字串 $s$，它希望你求出 $s$ 的所有**子串**代表的数字之和。

由于结果可能很大，你只需要求出结果对 $10^9 + 7$ 取模的值。

## 输入格式

一行，一个十进制数字串 $s$。

## 输出格式

一行，一个整数，表示所求的值。

## 说明/提示

对于 $100\%$ 的数据，$1 \leq |s| \leq 2 \times 10^6$，**$s$ 可能包含前导零**。

## 样例 #1

### 输入

```
123

```

### 输出

```
164

```

## 样例 #2

### 输入

```
001

```

### 输出

```
3

```


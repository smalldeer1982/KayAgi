# ADADIG - Ada and Digits

## 题目描述

艾达女士是一位酷爱数学的农场主。有一天，在重复且单调的农活中，她开始思考关于数字的问题。她想知道，有多少个数字的数字和与数字积是相等的。

尽管她对一些较小的数字和（同时也是数字积）已经找到了答案，但当这些数字变得很大时，问题就复杂了。你能帮助她解决这个难题吗？

### 输入格式

第一行包含一个整数 $1 \le T \le 10^5$，表示测试用例的数量。

接下来的每一行包含一个整数 $1 \le N \le 10^5$，代表需要计算出数字和与数字积相同的数字个数。

### 输出格式

对每个测试用例，输出符合条件的数字个数。由于这个数量可能非常大，请将结果对 $10^9 + 7$（即 $1000000007$）取模后输出。

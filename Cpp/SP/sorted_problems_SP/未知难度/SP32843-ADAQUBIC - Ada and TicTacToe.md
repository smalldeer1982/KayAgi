# ADAQUBIC - Ada and TicTacToe

## 题目描述

瓢虫艾达正在和她的好朋友Vinit一起在3x3x3的棋盘上玩井字棋。

遗憾的是，Vinit必须要提前回家了，但是比赛还没有结束。

艾达想要知道，谁能赢得这场比赛（假设她们都发挥得很好）。

你还知道艾达优先开始比赛。

## 输入格式

第一行包含一个整数，表示测试数据的数量。

每个测试数据包含9行，每行3个字符，表示每一个方格（“.”表示空位，“x”表示Ada的棋子，“o”表示Vinit的棋子）。

很确定的是，游戏未完成且有解。

前三行表示最顶上的方格，接下来的三行表示中间的方格，最后三行表示最底下的方格。

每一个测试样例前都有一个空行。

## 输出格式

对于每一个测试样例，输出获胜者的姓名（Ada或Vinit）。

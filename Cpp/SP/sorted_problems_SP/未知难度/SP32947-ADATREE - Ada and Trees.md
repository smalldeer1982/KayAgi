# ADATREE - Ada and Trees

## 题目描述

瓢虫艾达是一位农民。她有一条长长的沟渠，她在那里种植树木。每棵树都有一定重量。任务很简单，她想知道沟渠某处最大的树，且高度不超过某个**H**，由于艾达经常查询这个问题，她请你为此编写一个程序。

## 输入格式

第一行包含两个整数 **1表示树木数量和问题数量。**

下一行包含 **N** 个整数**0，表示树木高度。

接下来的和 **Q** 包含三个整数**0，表示她感兴趣的沟渠部分和0**。

## 输出格式

对于每个查询，输出最大树的大小小于或等于 **H**，或者如果该树在给定段上没有生长，则输出 0。

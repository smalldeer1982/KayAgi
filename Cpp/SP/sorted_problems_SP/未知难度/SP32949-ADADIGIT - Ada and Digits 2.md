# ADADIGIT - Ada and Digits 2

## 题目描述

瓢虫艾达喜欢数字。她的朋友知道，所以他们想给她一个"号码"作为礼物。不幸的是，他们在宜家买了这个"号码"，所以艾达不得不自己组装这个"号码"。现在她遇到了以下情况——她前面有几个数字，她想知道她将创建什么数字。由于她不懂这本手册（它是瑞典语的），她有可能组装任何东西。

她还不确定，不管怎样，她最喜欢的选择是把除数最多的数或除数和最大的数组合起来。你能帮她找到吗？

**注意：要从数字中组合数字，只需选择数字的任意排列。前导零是完全可以的，但是你必须使用所有的数字。**



------------

## 输入格式

每个样例的第一行将包含一个整数$1$，即要组合的位数。

下一行将包含$N$个数字$0$。



------------

## 输出格式

输出两个整数-除数最多的数和除数和最大的数。如果有多个这样的数字，请选择较小的一个。只要使用前导零，就输出不带前导零的数字。

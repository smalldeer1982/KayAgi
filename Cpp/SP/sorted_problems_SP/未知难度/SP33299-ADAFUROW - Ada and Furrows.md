# ADAFUROW - Ada and Furrows

## 题目描述

您可能已经知道，Ada是一个农民。她有许多种蔬菜的犁沟。她永远不会在同一犁沟中种植多个相同种类的蔬菜。现包含多个操作（包括种植与查询等等）您能快速回答她的所有询问吗？

## 输入格式

输入的第一行将包含一个整数Q（Q≤3*10^5），表示操作数。

接下来的Q行将包含 **? x y** (0 ≤ x, y ≤ 2*10^4)，其中?可能为 + - v ^ ! \，其含义如下：

**+：** 将种类为y的蔬菜种到犁沟数x处（注意：在同一犁沟中永远不会有多个相同种类的蔬菜）

**-：** 收获犁沟x中类型为y的蔬菜（请注意，总会有这种类型的蔬菜）

**v：** 找出x和y沟中一共有多少种蔬菜。

**^：** 找出有多少种蔬菜同时在犁沟x，y中

**！：** 找出x和y中有多少种蔬菜，且满足只在x,y中的一个里面出现。

**\：** 找出x中有多少种y里面没有的蔬菜

## 输出格式

对于最后四种查询，每行输出一个整数表示答案

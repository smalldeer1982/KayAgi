# HIGHWAY - Cestarine

## 题目描述

在一天内，Luka 的 $N$ 辆卡车行驶了一条特定的高速公路。高速公路有许多出口和入口。具有特定编号的出口与具有该编号的入口位于同一位置。

进入高速公路后，卡车司机会收到一张标明他使用的入口的单子。下车时，司机支付的通行费等于入口和出口编号的绝对差。例如，如果一张单子说他使用了 30 号入口，那么从 12 号出口离开将花费他 18 美元。

Luka 已经找到了一种节省公司日常花费的通行费的方法。任何两名司机都可以在高速公路上相遇并换单子，即使他们的路线不相遇。车票可以交换任意次。

但是，如果司机的换完的单子上写着他使用的同一个入口，那么他就不能出去，因为这会很可疑。

编写一个程序来计算司机通过换票可以达到的最小通行费总额。

## 输入格式

输入的第一行是整数 $T$（$1 \le T \le 5$），即测试样例的数量。$T$ 个案例后跟每个案例都以单个整数 $N$（$2 \le N \le 100,000$） 开头。接下来的 $N$ 行中的每一行都包含两个介于 $1 \sim 10 ^ 9$ 之间的整数，表示卡车的入口和出口编号。请注意，没有两辆卡车的入口或出口编号相同。

## 输出格式

对于每个测试样例，输出 Luka 的公司必须支付的最小通行费总额。

## 样例 #1

### 输入

```
2
3
3 65
45 10
60 25
3
5 5
6 7
8 8
```

### 输出

```
32
5
```

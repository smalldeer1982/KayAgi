# HIST2 - Histogram

## 题目描述

- 在统计学中，柱状图用来体现事件频数。一张柱状图可以看做是多个矩形底部对齐排列构成。
- 本题中给定多个矩形，每个矩形底边长度相等，高度不同。这些矩形不同的排列方案所构成的柱状图周长不同，你的任务是求解柱状图的最大周长以及矩形排列方案数。

- (a)图中矩形高度为 $ \{1,2,3,4\} $ 其周长为 $16$

- (b)图中矩形高度为 $ \{3,1,2,4\} $ 其周长为 $20$

## 输入格式

输入包含多组测试样例。每组数据第一行为一个整数 $n$ ，表示矩形个数。

第二行为 $n$ 个整数，表示每个矩形的高。

当 $n=0$ 时，表示测试结束。

$2 \leq n\leq 15$ ，测试样例不超过50组。

## 输出格式

对于每组数据，输出一行两个整数表示最大周长和排列方案数

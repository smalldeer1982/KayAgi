# TREX - Taming a T-REX

## 题目描述

进化论中指出，哺乳动物是在恐龙大规模灭绝之后才开始占据主导地位的；然而，有些人认为人类与恐龙可能共存过一段时间。甚至有人猜测，人类可能像《摩登原始人》中的情节一样，驯服并利用了恐龙。Shankar 是这样一种说法的信徒，而 Sunil 则对此持怀疑态度。

有一天，Sunil 问 Shankar：“如果你的观点正确，你打算如何驯服一只霸王龙，并用它来做什么？”Shankar 回答说：“我们可以用它来运输牛，从一个村庄搬到另一个村庄，并通过在固定时间间隔内给它喂食来令它平静下来。”Sunil 强调，霸王龙的载重量有一个最大值，即 **C**。假设需要运输的距离是 **d** 公里。若距离过长，霸王龙可能在到达另一个村庄之前就吃掉所有的牛。Shankar 声称他知道一种方法，可以在到达目的地时保证最多的活牛数 **M**。Sunil 回应道：“真的吗？那我来设定一些条件。霸王龙每行驶一公里就会吃掉一头牛，而且它在每公里内的任意时刻都可能进食。因此，霸王龙任何时候都不能没有牛可以吃。如果你在途中放下牛，只能在整数公里处进行。而且，到达目的地时所有牛都必须是完整的，不能分割（即只能是整数头牛）。”

Shankar 感到困惑，他需要你的帮助。给定 **I**（起始村庄的牛的数量）、**d**（距离）和 **C**（霸王龙的最大载重），请你计算出 **M**，即最终能运送到目的地的最大活牛数量。

## 输入格式

输入包含多个测试用例。第一行是一个整数 **n**，表示测试用例的数量，$n \leq 300$。接下来的 **n** 行中，每行包含三个整数 **I**、**d** 和 **C**，分别表示起始村庄的牛的数量、距离和霸王龙的最大载重能力，这三个整数用空格隔开。**I** 和 **C** 的范围是 $1 \leq I, C \leq 10^6$，**d** 的范围是 $1 \leq d \leq 10^5$，距离单位为公里。

## 输出格式

对于每个测试用例，输出一个整数，表示在满足条件下能运输到目的地的最大牛的数量，结果各占一行。

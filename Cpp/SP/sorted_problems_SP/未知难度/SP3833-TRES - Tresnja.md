# TRES - Tresnja

## 题目描述

拉娜住在一个小而快乐的村庄里。大街旁边有一排樱桃树。拉娜用从1开始的连续整数给树编号。



经过大量的研究，拉娜注意到，这棵树的数量决定了它所产樱桃的数量。



对于一棵树，考虑树编号中的连续数字组。对于每组数字，将数字乘以该组长度的平方。把这些数字加在一起就得到了这棵树的樱桃总数。



例如，在编号为77744007的树中，组是777、44、00和7。樱桃的数量为7.3^2+4.2^2+0.2^2+7.1^2=86个单位。



采摘樱桃树的时候到了，村民们同意采摘所有编号为A到B（含B）的树。编写一个程序来计算采摘的樱桃总量。



###### 简述：给你一个区间[l,r]，求出每一个数的贡献和，每一个数的贡献如下计算，先把数分成几个连续的数字组，比如77744400077分成777,444,000,77，对于每一个数字组，它的贡献是它的数值$\times$数字组长度的平方，例如777的贡献为7*3^2，每一个数字的贡献为所有数字组贡献的和。

## 输入格式

输入由两个整数A和B（1≤A≤B≤10^15）组成，这是要拾取的第一个和最后一个树。

## 输出格式

输出一个整数，将采摘多少个樱桃单位。

## 样例 #1

### 输入

```
100 111
```

### 输出

```
68
```

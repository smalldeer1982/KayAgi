# FOOL - Any fool can do it

- 他自信地说：「当然知道！集合就是用一对大括号把一个可能为空的元素列表括起来。列表中的每个元素要么是另一个集合，要么是字母表中的一个字母。元素之间用逗号分隔。」
- 你继续问：「如果我给你一个字符串，你能够判断它是否是集合的正确表示吗？」

```
List         ::= Element | Element "," List
Element      ::= Atom | Set

```

# MRECTCNT - Rectangles  Counting

## 题目描述

对于一个 _a_ 行 _b_ 列的矩阵，将 _f(a,b)_ 记作该矩阵被一条对角线穿过的格子数目，求有多少对 _a_ , _b_ 满足 _f(a,b)=N_ 。( _a_ , _b_ 与 _b_ , _a_ 算同一种)

## 输入格式

输入共一行一个整数 _N(0 < N <=10^6)_ 。

## 输出格式

输出共一行一个整数，即所求答案。

## 输入样例
```
4
```

## 输出样例
```
4
```

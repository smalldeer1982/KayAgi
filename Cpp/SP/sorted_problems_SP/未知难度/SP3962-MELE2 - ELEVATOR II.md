# MELE2 - ELEVATOR II

## 题目描述

爱德华在非平凡电梯工程、研究与建造公司（NEERC）担任工程师。他的新任务是为一栋拥有 $h$ 层的摩天大楼设计一款新型电梯。

爱德华坚信，使用四个按钮就足够控制电梯的上下移动。他最近的设计方案提出了以下四个功能按钮：
- 上升 $a$ 层。
- 上升 $b$ 层。
- 上升 $c$ 层。
- 返回到第一层。

在一开始，电梯位于第一层。乘客可以通过使用前三个按钮到达所需的目标楼层。然而，如果乘客试图移动 $a$、$b$ 或 $c$ 层，而超过了大楼的最高层（超过第 $h$ 层），电梯将不作任何移动。

为了验证他的设计是否有效，爱德华想知道，从第一层出发，通过这些按钮究竟可以到达多少个楼层。请帮助他计算这个可以到达的楼层总数。

## 输入格式

输入的第一行包含一个整数 $h$，表示摩天大楼的总层数（$1 \le h \le 10^{50}$）。第二行包含三个整数 $a$、$b$ 和 $c$，分别代表电梯上升的层数（$1 \le a, b, c \le 1000$）。

## 输出格式

输出一个整数，表示从第一层出发，可以通过电梯按钮实际到达的楼层数。

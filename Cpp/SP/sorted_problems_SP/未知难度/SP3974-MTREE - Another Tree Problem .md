# MTREE - Another Tree Problem 

## 题目描述

给定一棵树（边权非负），定义路径(u,v)的权值为从u到v经过的所有边的权值乘积，定义树的权值为该树上所有路径权值之和。请编程求出给定的树的权值，答案对1e9+7取余。特别地，在计算树的权值时，(u,v)和(v,u)只计算一次。

## 输入格式

第一行一个整数n，表示共n个节点。 

第二行到n+1行，每行(x,y,z)描述一条边权为z树边。

## 输出格式

输出该树的权值，对1e9+7取余。

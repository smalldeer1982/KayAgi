# TWOPROF - Two Professors

## 题目描述

在 X 学院中，有两位教授彼此间关系极差。为了保护隐私，我们称他们为教授1和教授2。学院里一共有 $n$ 位教授，每位教授的课程都有明确的、固定的开始时间和结束时间。不过，这些课程的上课地点还无法确定。显而易见，如果两节课的时间有重叠，它们不能在同一间教室进行；但如果一节课恰好在另一节课结束时开始，那么可以共用同一间教室。你的任务是计算出安排所有课程所需的最少教室数量。需要特别注意的是，教授1和教授2绝对不能在同一间教室里授课，因为他们互相厌恶。

## 输入格式

输入包含多组测试数据。第一行输入一个整数 $t$ （表示测试数据组数，$t \leq 250$）。对于每组测试数据，第一行包含一个整数 $n$（表示教授人数，$2 \leq n \leq 100000$）。接下来的 $n$ 行中，第 $i$ 行包含两个整数 $start_i$ 和 $end_i$，表示第 $i$ 位教授的课程开始和结束时间（$0 \leq start_i < end_i \leq 1000000000$）。

## 输出格式

对于每组测试数据，输出一个整数，表示安排所有课程所需的最少教室数量。

# DELCOMM - DEL Command

## 题目描述

给定一组文件，判断是否能通过在 MS-DOS 操作系统中执行一次 DEL 命令就完成所有需要的文件删除操作。目录中没有嵌套的子目录。

## 输入格式

输入的第一行包含一个整数 $M$，表示有 $M$ 组数据集。之后有一行空行，用于分隔。

每组数据集由文件全名列表构成，文件名之间没有空行和空格。每个文件名前都有一个控制符：'-' 代表需要删除的文件，'+' 代表需要保留的文件。全文件名不重复。每组数据集中至少有一个文件需要删除，且总文件数不超过 1000 个。

## 输出格式

对于每组数据集，在输出的第一行写出正确的 DEL 命令（只需提供一种方案）。如果没有解决方案，则输出 "IMPOSSIBLE"。注意："DEL" 和通配符之间需要一个空格。 **在每组数据集的输出后插入一个空行用于分隔。**

## 说明/提示

- 总共有 $1 \le M \le 1000$ 组数据集
- 每组数据集的文件数量不超过 1000 个。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
1

-BP.EXE
-BPC.EXE
+TURBO.EXE
```

### 输出

```
DEL ?P*.*
```

# NUMQDW - Number of quite different words

## 题目描述

## 题意
给出$c$个英文大写字母组成的字母表。当$c$为$6$时,字母表即$A，B，C，D，E，F$，

如果**两个单词之间不存在长度超过一的公共子序列**，我们将称两个单词完全不同。

例如，`ABC`和`CBA`完全不同，但是`ABBA`和`CADDCAD`却没有，因为`AA`是两个单词的子序列。

给定单词$w$，您将发现长度为$n$的单词数量与$w$完全不同。

## 输入格式

第一行将包含测试用例的数量$T$（$T\leq20$个）。

每一组测试用例会有两行，第一行包含数字$n$（$n$为int范围内的非负整数）和$c$（$1\leq c\leq 6$），第二行包含字符串$w$。$w$将仅由英文大写字母的前$c$个字母组成，最多包含$10000$个字符。

## 输出格式

为每个测试用例仅有一行输出，代表与w完全不同的单词数。

由于这个数字可能很大，因此您只需要输出除以$4242$所得的余数。

## 样例 #1

### 输入

```
3
3 3
ABC
4 4
CADDCAD
100 3
A
```

### 输出

```
10
13
2223
```

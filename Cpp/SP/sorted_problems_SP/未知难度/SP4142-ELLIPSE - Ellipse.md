# ELLIPSE - Ellipse

## 题目描述

给出一个椭圆上的五个点，求这个椭圆的面积。

你需要保证你的输出的相对误差在 $10^{-6}$ 以内或绝对误差在 $10^{-9}$ 以内。

## 输入格式

每个测试点有多组测试数据，每个测试数据中给出 $10$ 个实数，为椭圆上 $5$ 个点的坐标。

## 输出格式

对于每个测试点，输出一行一个正实数表示椭圆的面积。

特殊地，如果无法确定或是不存在满足输入的椭圆，则输出`IMPOSSIBLE`。

## 样例 #1

### 输入

```
6 1 3 2 -2 -3 -3 -2 1 6
7 -3 2 7 6 3 5 5 -2 -9
```

### 输出

```
IMPOSSIBLE
157.079633
```

# BFROTATE - rotate it

## 题目描述

请将给定的字符串从末尾向前旋转指定位置，旋转时允许循环。值得注意的是，这次你只能使用 BrainFck 编程语言来实现！

## 输入格式

输入一行，其中包括一个两位数字的整数和一个长度不超过 99 的字符串，而且两者之间没有空格。可以假设该字符串仅由可打印字符构成，并且字符串的长度大于给定的整数。

## 输出格式

输出一个与输入字符串长度相同的新字符串，字符顺序根据题目的要求进行了重新排列。

## 附加说明

- 整数范围：10 到 99。
- 字符串长度：大于整数且不超过 99。

# FACTOR1 - Factorization, Factorization, Factorization

## 题目描述

将表达式 $x^n - 1$ 分解成若干个不可约的整数多项式。

## 输入格式

程序需要处理多组输入数据。每组数据仅包含一个整数 $n$（满足 $2 \le n \le 1200$）。输入以一个零结束，表示没有更多数据。

## 输出格式

对于每组输入，输出式子 $x^n - 1$ 的多项式分解。

为了确保分解结果的唯一性，我们对不可约多项式进行排序，规则如下：

1. 低次的多项式在字典序上优先于高次的。
2. 对于相同次的多项式，根据系数进行排序。从高次到低次进行比较，包括省略的项（视为系数为0）。系数比较时，首先考虑绝对值大小，然后再考虑正负符号。绝对值较小的系数排在前面；如果绝对值相同，负数的系数排在正数前面。

具体排序和输出格式可以参照示例。

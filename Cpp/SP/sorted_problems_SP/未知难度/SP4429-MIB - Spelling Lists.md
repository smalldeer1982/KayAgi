# MIB - Spelling Lists

## 题目描述

黑衣人中的 J 正在学习一种外星语言，明天要进行拼写测试。不过，J 已经厌倦了学习那些毫无意义的（通常无法发音的）单词。

相反，他正在研究有多少种方法可以重新排列他的拼写单词表。在对列表中的单词进行了所有可能的排列组合后，他对重新排列的列表按字典顺序进行了排序（从第一个字母按字母表顺序比较）。然后问题是，原始的拼写单词列表在这个排序中处于哪个位置，其中第一个按字典顺序排列的列表位于位置1。

## 输入格式

第一行是拼写列表的数量 n（不超过 10 个）。

每一个拼写表都是一行是单词的数量（不超过 1000），下一行是原拼写表。

拼写表中的所有单词都是唯一的。每个单词由字母 a-z 组成，少于 100 个字符，后跟一个空格。

## 输出格式

n 行，每行是原始的拼写单词列表在这个排序中处于的位置

## 样例 #1

### 输入

```
\n4\n4\na b c d \n4\nd c b a \n1\nmrsmith \n6\na aaaaaa aaaaa aaaa b bb \n\n
```

### 输出

```
\n1\n24\n1\n55\n
```

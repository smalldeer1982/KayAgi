# UCI2009D - Digger Octaves

## 题目描述

经过多年玩《矿工》游戏，小伊万终于意识到他以前没有利用好「八度」。哦，抱歉！很多人可能在《矿工》问世时还没有出生！

《矿工》是一款加拿大开发的电脑游戏，最初设计于1983年，适用于IBM个人电脑。游戏的目的是在一个废弃矿井的地下探索，收集珍贵的金子和绿宝石。

对于我们这些《矿工》的高手来说，连续八个绿宝石被称为一个「八度」。需要注意的是，这里的连续性意味着玩家可以依次采集它们。在游戏中，你的移动设备可以向四个方向移动：北、南、西和东。

在一个简化的游戏版本中，地图仅由绿宝石和空格构成，任务就是数一数地图中有多少个「八度」。

## 输入格式

输入以一个整数 $T$ 开头，表示测试用例的数量（$1 \le T \le 20$）。每个测试用例由一个地图组成，具体如下：

首先是一个整数 $N$（$1 \le N \le 8$），表示地图的边长。接下来有 $N$ 行，每行包含 $N$ 个字符。字符 'X' 表示一个绿宝石，字符 '.' 表示空格。

## 输出格式

对于每一个测试用例，在一行中输出地图中所包含的「八度」的数量。

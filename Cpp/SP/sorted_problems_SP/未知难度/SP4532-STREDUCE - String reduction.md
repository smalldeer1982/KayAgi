# STREDUCE - String reduction

## 题目描述

给你一个字符串，仅由字符 'a' 和 'b' 组成。你可以通过将形如 a?a 或 b?b 的子串（其中 '?' 可以是 'a' 或 'b'）替换为一个字符 '?' 来缩短这个字符串。

## 输入格式

输入为一行字符串。

## 输出格式

输出最小可能的字符串长度。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
\nbaaba\n\n
```

### 输出

```
\n1\n
```

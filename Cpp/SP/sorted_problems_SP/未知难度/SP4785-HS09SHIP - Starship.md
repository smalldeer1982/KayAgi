# HS09SHIP - Starship

## 题目描述

**题目大意：**

你乘坐星际飞船旅行的任何时候，都可以朝着6个方向之一移动：向前、向后、向上、向下、向左或向右。换句话说，在每一秒钟内，你的位置的三个坐标中的一个坐标都会准确地改变一个单位。假设你在（x 1，y1，z1），你想达到（x 2，y 2，z 2）。不幸的是，你的飞船是第一代星际飞船，这意味着所有的运动都是完全随机的，所以每秒钟你都会以1/6的概率向前/向后/向上/向下/向左/向右移动。你能计算一下在**n-**秒内到达目的地的概率吗？

## 输入格式

第一行包含整数T，表示测试用例的数量。每个测试用例都以正整数n开头，下一个用例给出星际飞船的起始位置，而最后一个用例给出目的地。已知：T<30000，0<n<=1000。x、y、z坐标的绝对值小于10的六次方。10个点有5个输入集。

## 输出格式

一行
T个答案，第i个答案给出第i个测试用例到达目的地的概率。保留小数点后10位，每个答案用" \ "隔开

## 样例 #1

### 输入

```
\n5\n2\n0 0 0\n0 0 0\n4\n0 0 0\n0 0 0\n100\n2 -3 4\n-4 5 6\n100\n2 -3 4\n-4 5 7\n1000\n0 0 0\n0 0 0\n\n\n
```

### 输出

```
\n0.1666666667\n0.0694444444\n0.0001389381\n0.0000000000\n0.0000208505
```

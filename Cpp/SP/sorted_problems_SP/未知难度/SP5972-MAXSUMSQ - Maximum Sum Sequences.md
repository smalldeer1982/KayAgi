# MAXSUMSQ - Maximum Sum Sequences

## 题目描述

给定一个 $n$ 个数的数组 $a$，求 $a$ 的最大子段和 $x$，并求出子段和等于 $x$ 的子段数量。

## 输入格式

输入第一行一个正整数 $T$，表示数据组数。

接下来 $2\times T$ 行，每 $2$ 行表示一组数据。每组数据中第一行一个正整数 $n$，第二行 $n$ 个正整数 $a_i$。

## 输出格式

输出 $T$ 行，每行输出两个整数，分别表示第 $i$ 组数据的最大子段和，以及子段和最大的子段个数。

## 样例 #1

### 输入

```
2\n3\n-1 -1 -1\n4\n2 0 -2 2
```

### 输出

```
-1 3\n2 4
```

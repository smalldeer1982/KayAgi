# HOMEC - Homecoming

## 题目描述

人类经过艰苦奋斗，终于取得了一场全面胜利。这场胜利归功于两个部落的共同努力，但第二个部落却想独揽功劳，并欲通过消灭第一个部落来统治世界。战争结束后，两个部落的成员将各自返回城市。战士们会分组通过已建好的桥梁回去，此时第二部落计划实施他们的阴谋。第一个部落的国王意识到这一点，并希望确保这种谋害不会发生，因此设定了一条约束：任何时候，在桥上或两岸，第二个部落的人数都不能超过第一个部落。此外，由于天色已晚，过桥时必须携带唯一的一支火把，因此每次至少有人需带火把返回。更糟糕的是，每次过桥的人数有限制。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的数量（$1 \le t \le 50$）。接下来有 $t$ 行，每行包含三个整数 $n$、$m$ 和 $c$，分别表示第一个部落的战士人数、第二个部落的战士人数，以及每次能同时过桥的最大人数（$1 \le n, m \le 100$，$1 \le c \le 50$）。

## 输出格式

对于每个测试用例，输出一行，包含一个整数，表示桥被穿越的总次数。一次往返算作两次。如果在给定的约束条件下无法完成旅行，则输出 -1。

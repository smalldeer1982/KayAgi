# PYRA - Treeramids

## 题目描述

Daniel 正在建造塔楼。他拥有许多尺寸为 $1 \times 1 \times K$（$K$ 是某个正整数）的积木。按照以下规则，Daniel 用这些积木建造塔楼：最长的积木块被放在塔楼的底部，多座塔楼可以建在同一个大底座上。塔楼之间的距离为 1，与底座边缘的距离也是 1。每座塔楼的顶端必须有一个长度为 1 的积木块。Daniel 使用根树结构来表示每个塔楼。例如：

![](https://cdn.luogu.com.cn/upload/vjudge_pic/SP6288/071828f856716bf33e2c0afdb9f3ae4bfe32fc8c.png)

## 输入格式

第一行为整数 $t$，表示测试用例的数量。接下来是 $t$ 个测试用例的描述。每个测试开始时有一个整数 $n$，表示树的节点数。接下来的 $n-1$ 行中，每行有两个整数 $a$ 和 $b$，表示由一条边连接的两个节点。这些节点编号为从 0 到 $n-1$。树的根节点始终是编号为 0 的节点。除此之外，节点和边的顺序没有特定要求。

## 输出格式

对于每个测试用例，输出根据题目规则建造的结构的总体积。

## 说明/提示

- $1 \leq t \leq 10^5$
- $2 \leq n \leq 10^5$

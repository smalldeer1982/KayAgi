# QUEST4 - Dungeon of Death

## 题目描述

题目背景：

为了找到宝藏，Jones 必须穿过“死亡之屋”。


这个房间的地板是正方形，边长为 
$N$。它是由尺寸为  $1 \times 1$ 的方形瓷砖铺设成网格。但是，在房间中的某些地方，瓷砖丢失了 $N$ 块。当这间屋子的门一打开，有毒气体就会从这些丢失的瓷砖中冒出来。
从这种气体中逃出的唯一方法是在房间外用木板完全覆盖这些位置。每块木板的尺寸为   $N \times 1$，并且木板只能平行于地板两侧放置。

他发现如果要找到宝藏，就必须尽可能使用少的木板。显而易见，即使木板重叠，仍然可以成功阻止丢失的瓷砖中排出有毒气体。现在请帮助 Jones 完成这项任务。

## 输入格式

**本题有多组数据**。

-  第一行一个整数 $T$，表示数据组数。

对于每组数据：

-  第一行是正整数 $N$（$n \leq 120$），表示缺少瓷砖位置的数量和房间的边长。  


*   接下来是 $N$ 行。每行包含两个整数 $x$ 和 $y$ （$0 \leq x,y<120$），由一个空格分隔，代表缺失瓷砖位置的坐标。

## 输出格式

*   对于每一组数据，请输出 $1$ 个数字，每个数字占一行，代表每一个房间 Jones 需要的最少木板数。

## 样例 #1

### 输入

```
2
3
1 0
2 0
3 0
4
1 1
2 2
3 3
4 4
```

### 输出

```
1
4
```

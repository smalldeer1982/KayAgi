# QUEST5 - Nail Them

## 题目描述

**Jones** 面临寻找宝藏的最后一个挑战。他来到一张桌子前，桌上放着许多木板，这些木板沿着桌子的长度摆放。**Jones** 发现桌子的宽度恰好等于每块木板的宽度。这些木板异常沉重，无法手动挪动，有些木板甚至彼此重叠。在他的手中有一把锤子，并且神灵赐予了他无限数量的钉子。**Jones** 需要将所有的木板固定在桌子上，确保每块木板至少用一个钉子和桌子连接。钉子足够长，只需垂直地钉入桌子。假如多个木板有重叠部分，它们可以共用一个钉子进行固定。请帮助 **Jones** 计算需要用多少个最少数量的钉子来固定所有的木板。

## 输入格式

- 输入的第一行是一个正整数 $t \leq 20$，表示桌子的数量。
- 之后是 $t$ 个桌子的描述，每个桌子的描述如下：
  - 第一行包括一个正整数 $n (n \leq 10010)$，表示桌子上的木板数量。
  - 接下来的 $n$ 行中，每一行有两个整数 $a$ 和 $b (0 \leq a \leq b \leq 10000010)$，表示每个木板左端和右端与桌子左端的距离。

## 输出格式

- 输出共 $t$ 行，每行对应一个桌子。第 $k$ 行的输出为一个整数 $i_{k}$，表示第 $k$ 张桌子上固定所有木板所需的最少钉子数量。

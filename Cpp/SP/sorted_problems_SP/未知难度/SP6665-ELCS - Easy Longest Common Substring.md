# ELCS - Easy Longest Common Substring

## 题目描述

给定 $N$ 个**仅由小写字母组成**的字符串，字符串编号从 $0$ 开始。现在有 $Q$ 组询问，每组询问给定两个数字 $a,b$ ，询问编号为 $a,b$ 的字符串最长公共前缀的长度。

## 输入格式

第一行一个整数 $N$ ，表示一共有 $N$ 个字符串。字符串编号从 $0$ 开始。

接下来的 $N$ 行，每行一个字符串。

下一行包含一个整数 $Q$ ,表示询问组数。

下面 $Q$ 行每行两个整数 $a,b$ ，询问编号为 $a,b$ 的字符串最长公共前缀的长度。

## 输出格式

共 $Q$ 行，每行一个整数，表示本次询问的答案。

## 说明/提示

对于 $30\%$ 的数据， $N\leq 100 ,Q\leq 10000 ,$ 每个字符串长度 $\leq100$

对于 $100\%$ 的数据，字符串总长度 $\leq 500000, N\leq 100000, Q\leq 100000$

## 样例 #1

### 输入

```
\n5dyljqlqpwsjzt30 11 20 2
```

### 输出

```
\n010
```

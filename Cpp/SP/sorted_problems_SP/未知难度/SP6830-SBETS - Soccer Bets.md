# SBETS - Soccer Bets

## 题目描述

FIFA 世界杯的小组赛阶段已经结束，16强队伍已然确定。我的老板已经对所有剩余的比赛进行了分析，并将结果写在一张纸上，让我拿着 _1000_ 美元去投注。由于紧张且有些笨手笨脚，我不小心摔了一跤，把所有投注单弄乱了。所以，我现在无法判断哪个结果对应的是决赛、半决赛或者其他比赛。

为了不让老板失望，我决定只押一注，直接赌最终的冠军队伍。我所知道的是，在每一轮比赛中，获胜队伍（即进球数多于对手的队伍）将晋级到下一轮，而败者则被淘汰。但半决赛除外，因为半决赛的失利者还将参加三四名的争夺。总共存在 _16_ 场比赛。

你能依据我老板的下注单，告诉我哪支队伍最终将赢得世界杯吗？

## 输入格式

输入的第一行是一个整数 _c_ (_1 ≤ c ≤ 100_)，表示测试用例的数量。每个测试用例有 _16_ 行，描述了所有比赛（顺序随机）。每场比赛的格式如下：$t_{1}$ $t_{2}$ $g_{1}$ $g_{2}$。其中，$t_{1}$ 和 $t_{2}$ 是队伍的名称，使用三个大写字母缩写；$g_{1}$ 和 $g_{2}$（$0 \le g_{1}, g_{2} \le 10$；且 $g_{1} \neq g_{2}$）是两支队伍的进球数。

## 输出格式

对于每个测试用例，输出一行，显示赢得 FIFA 世界杯的队伍名称（基于我老板的准确分析）。

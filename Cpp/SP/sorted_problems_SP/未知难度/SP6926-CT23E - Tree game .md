# CT23E - Tree game 

## 题目描述

最近，Bob 发明了一种利用树结构的全新游戏。注意，树是一个没有环的连通图。在游戏中，他可以任意删除这棵树上的若干条边（包括不删除任何边），然后计算删除后各个连通子图的节点数乘积。你的任务就是计算出，对于给定的树，Bob 通过这种操作能得到的最大乘积是多少。

## 输入格式

输入包含多组测试数据。

对于每组测试数据，第一行是一个整数 $n$，表示树的节点数。

接下来的 $n-1$ 行，每行由两个整数 $u$ 和 $v$ 组成，表示树中的一条边连接了节点 $u$ 和节点 $v$。

## 输出格式

对每组测试数据，输出一个整数，表示 Bob 可以得到的最大乘积。

## 说明/提示

$$1 \le n \le 300,000$$

每组测试数据给出的一棵树由 $n$ 个节点构成。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
\n8\n1 2\n1 3\n2 4\n2 5\n3 6\n3 7\n6 8
\n\n
```

### 输出

```
\n18\n
```

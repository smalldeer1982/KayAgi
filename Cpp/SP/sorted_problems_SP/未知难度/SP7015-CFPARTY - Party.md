# CFPARTY - Party

## 题目描述

$n$ 个人参加了一个聚会，然后陆续离开。首先是没有朋友的人离开，然后是有 $1$ 个朋友的人离开，接着是有 $2$ 个朋友的人离开，有$3、4、5、6……n-1$个朋友的人陆续离开，求最后剩下几人。

## 输入格式

第一行一个数字 $t$（数据组数），接下来 $t$ 行，一行一个数字 $n$。

## 输出格式

一行一个数字，为最大留下的人数。

## 样例 #1

### 输入

```
1
3
```

### 输出

```
1
```

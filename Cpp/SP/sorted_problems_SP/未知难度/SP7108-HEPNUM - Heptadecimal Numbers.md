# HEPNUM - Heptadecimal Numbers

## 题目描述

计算机增强数字工厂（FCEN）需要他们的开发委员会（DC）设计一种方法来处理17进制的数字。众所周知，17进制在许多科学、工程以及其他实际应用中都非常重要。虽然17进制的数字看起来有点棘手，但如果方法得当，它们也极具亲和力。

在17进制中，使用17个字符来表示数字：从0到9的数字沿用其原本的数值，而大写字母A到G则分别表示10到16。由于17是一个质数，17进制数并不要求以非零数字开头，因此每个数可能有多种表示方式。例如，十进制数字117可以表示为6F、06F，甚至00000006F。因为这种前导零的情况，导致17进制数之间的比较变得困难。

作为FCEN-DC的一员，你需要编写一个程序来帮助解决这个复杂而具有挑战性的任务。

## 输入格式

输入由多行组成，每行是一个测试用例，包含两个由空格分隔的17进制数，长度都不超过10万位。输入的最后一行由两个星号（*）隔开，表示输入结束，不作为测试用例处理。

## 输出格式

对于每个测试用例，如果第一个17进制数小于第二个，输出`<`；如果第一个17进制数大于第二个，输出`>`；如果两个数相等，输出`=`。

# BYECAKES - Bye Bye Cakes

## 题目描述

John 即将搬迁到另一个城市，他想在搬家之前用光所有易腐食品，以避免浪费。所幸的是，他的食材只有鸡蛋、面粉、糖和牛奶，因此他决定制作他出名的蛋糕，并把它们作为告别礼物送给朋友们。John 只能制作完整的蛋糕，而不能做半个或三分之一的蛋糕。因此，他需要购买足够的各种原料，使得可以恰好做出整数个蛋糕，并且没有剩余。同时，为了节省花费，他希望购买的数量尽可能少。你的任务是帮助 John 计算出需要购买的每种原料的具体数量。

## 输入格式

输入由多组测试数据构成。每组数据包括一行，包含八个整数 E, F, S, M, E', F', S' 和 M'，由空格分隔。这里，E 和 E' 分别代表鸡蛋的现有量和价格，F 和 F' 为面粉的克数和价格，S 和 S' 为糖的克数和价格，M 和 M' 为牛奶的分升数和价格。

## 输出格式

对于每组测试数据，输出一行，包含四个非负整数，以相同的顺序表示 John 需要购买的每种原料的数量，单位与输入一致。

## 说明/提示

- \(0 \leq E, F, S, M < 10^5\)
- \(1 \leq E', F', S', M' \leq 100\)

 **本翻译由 AI 自动生成**

### 输入

```
\n2 3 4 5 1 1 1 1\n3 6 9 0 1 2 3 4\n-1 -1 -1 -1 -1 -1 -1 -1\n\n
```

```

```

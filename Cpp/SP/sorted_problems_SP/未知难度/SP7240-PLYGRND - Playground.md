# PLYGRND - Playground

## 题目描述

乔治手上有 $K$ 个弧形片段。

## 输入格式

每组数据由一个正整数 $K$ 组成，满足 $0 < K$。

输入以一行单独的 $0$ 结尾。

## 输出格式

对于每个测试用例，如果能用这些弧片构建出一个简单的连通图形，则输出「YES」；否则，输出「NO」。

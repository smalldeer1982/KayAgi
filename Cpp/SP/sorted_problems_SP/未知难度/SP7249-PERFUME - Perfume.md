# PERFUME - Perfume

## 题目描述

一家大型香水店正在通过混合不同的香料来制作香水。

## 输入格式

输入的第一行有一个整数 $T$，表示测试用例的数量，满足 $T \le 100$。每个测试用例的第一行包含一个整数 $N$，表示店铺已经拥有的香料混合物数量，其中 $1 \le N \le 200$。接下来的 $N$ 行分别描述每种混合物的成分比例，每行包含两个浮点数 $A$ 和 $B$，表示混合物中成分 $A$ 和 $B$ 的百分比，注意 $0 \le A, B \le 100$ 且 $A + B \le 100$。然后再有一行是整数 $Q$，表示需要验证的新混合物数量，其中 $1 \le Q \le 5151$。接下来的 $Q$ 行也包含两个浮点数 $A$ 和 $B$，对应新混合物的成分比例，满足同样的限制条件。不同的测试用例之间用一个或多个空行分隔。

## 输出格式

对于每个待验证的新混合物，如果可以通过现有的混合物制成，则输出 `Yes`，否则输出 `No`。在输出不同的测试用例结果之间，需要加入一个空行。

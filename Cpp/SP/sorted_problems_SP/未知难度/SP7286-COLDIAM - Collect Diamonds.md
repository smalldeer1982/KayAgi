# COLDIAM - Collect Diamonds

## 题目描述

米斯蒂在一家珠宝店赢得了一场比赛，现在她被允许收集钻石作为奖品。但店长不想这么慷慨。他在地板上铺了几颗钻石，现在他要求米斯蒂用她的金链子包围它们。之后，她可以把所有被包围的钻石放在链子上。米斯蒂可以设法从她的朋友那里得到任何长度的金链子。但她必须尽量减少长度，还想收集地板上的所有钻石。链条上的钻石也被认为是在里面。帮助她找到所需的金链子的最小长度。

## 输入格式

输入的第一行由单个正整数 $n \leq 1000$  组成。接下来的 $n$ 行由两个整数 $x,y$ 组成，表示钻石在地板上的位置（坐标）。$0 \leq x，y \leq 1000$。

## 输出格式

需要最小长度的金链，以便米斯蒂可以收集所有钻石。答案应四舍五入到小数点后两位。

## 样例 #1

### 输入

```
\n4\n1 0\n0 1\n1 1\n0 0\n\n
```

### 输出

```
\n4.44\n
```

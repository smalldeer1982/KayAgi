## 题目描述

# GRIDCOIN - Placing Coins on a Grid

在 $N \times M$ 的网格上，有多少种方式放置 $R$ 枚硬币，使每行和每列至少有 $1$ 枚硬币？

## 输入格式

**本题有多组数据**。

第一行一个整数 $T$，表示数据组数。

对于每组数据：

一行三个整数 $N,M,R$

## 输出格式

输出 $T$ 行，每行一个整数表示方案数，请把方案数对 $1000000007$ 取模。
## 样例 #1

### 样例输入 #1

```
3  
1 1 1
2 1 1
2 3 3
```

### 样例输出 #1

```
1
0
6
```

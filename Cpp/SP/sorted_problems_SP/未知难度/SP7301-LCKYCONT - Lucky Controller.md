# LCKYCONT - Lucky Controller

## 题目描述

Egor 是一名公交车售票员。他每天都会收到一包车票，并负责销售这些车票。最近，他很好奇每天有多少张车票是幸运的。他认为，幸运车票越多，他的一天就会越幸运。因此，他想知道第二天会有多幸运。每张车票的编号都由 $n$ 位数字组成。如果一张车票的前 $\frac{n}{2}$ 位数字之和等于后 $\frac{n}{2}$ 位数字之和，那么这张票就是幸运的。Egor 得知他将收到的车票的编号可以在区间 $[a, b]$ 内随机开始，并且每张票的编号都是连续的。这包车票共有 $k$ 张。请帮助 Egor 计算这包车票中预计有多少张是幸运车票。

## 输入格式

输入的第一行为一个整数 $n$，表示测试用例的数量。接下来的 $n$ 行中，每行包含三个整数 $a, b$ 和 $k$（满足 $0 \le a \le b < 10^{12}, 1 \le k \le 100000$）。整数 $a, b$ 和 $b + k$ 的位数相同，这个位数正是每张车票编号的位数。它们可以以零开头。需要注意的是，$a$ 和 $b$ 的位数总是偶数。

## 输出格式

输出这包车票中预计的幸运车票数量，答案应以最简分数的形式给出。如果结果为整数，则直接输出数值，不应包含斜杠。

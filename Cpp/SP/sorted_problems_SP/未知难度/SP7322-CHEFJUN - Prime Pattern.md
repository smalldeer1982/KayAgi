# CHEFJUN - Prime Pattern

## 题目描述

### 题意翻译
有一块白色的正方形田地，场上有一个坐标系：$y$轴从南向北，$x$轴从西向东，正方形的侧面与轴线平行。现在在$(0,0)$的广场上有一个机器人，机器人会以如图所示的螺旋状移动。首先，它向东移动一步，向北移动一步。然后再向西两步，向南两步。之后向东三步，向北三步，向西四步，向南四步。依此类推。每走一步，它就会移动到一个新的广场。当它移动时，它会计算经过的正方形数量，如果此时正方形的数量是素数，那么机器人会用黑色填充这个正方形。机器人会从$(0,0)$的正方形开始计数，其中$(0,0)$数字为$0$。现给定一个正方形的坐标，你要计算从这个正方形到离它最近的黑色正方形的距离。对于坐标为$(x1,y1)$和$(x2,y2)$的两个正方形，它们之间的距离为$|x1-x2|+|y1-y2|$。

## 输入格式

输入文件的第一行是一个整数$T(T\le500)$表述测试数据的数量。下面的$T$行包含两个整数$x$和$y$，即正方形的坐标。每个整数用空格分隔。

## 输出格式

输出此正方形与最近的黑色正方形之间的距离。

## 说明/提示

$-2000001\le x\le 2000001$,$-2000001\le y\le 2000001$。

## 样例 #1

### 输入

```
8
0 0
1 0
1 1
0 1
3 3
-3 -3
-1 2
0 -3
```

### 输出

```
1
1
0
0
1
1
2
2
```

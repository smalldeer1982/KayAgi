# FUNFACT - Factorial challenge

## 题目描述

Ling 说：“Stir，我们出去玩你最喜欢的游戏吧。”

Stir 回应道：“我正在玩一个关于阶乘的程序，很有趣呢。”

Ling 继续说道：“那我给你一个挑战吧。如果你搞不定，就得跟我出去玩。”

Stir 同意了。

Ling 给 Stir 一个数字 $x$，要求 Stir 找到最大的整数 $n$，使得 $n!$（即 $n$ 的阶乘）不大于 $x$ 位数字能够组成的最大数。Stir 被这个挑战卡住了，需要你的帮助。现在轮到你让 Stir 继续享受阶乘带来的乐趣了。

## 输入格式

第一行包含一个整数 $t$（约 $10^5$），表示测试用例的数量。接下来的 $t$ 行中，每行包含一个整数 $x$（$1 \le x \le 10^9$）。

## 输出格式

总共输出 $t$ 行，每行对应一个输入用例，输出该用例对应的 $n$ 值。

### 输入

```
\n2\n1\n7\n\n
```

### 输出

```
\n3\n10
```

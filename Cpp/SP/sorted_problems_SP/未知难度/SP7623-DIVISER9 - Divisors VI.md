# DIVISER9 - Divisors VI

## 题目描述

找出正好有N个奇数因数的最小整数，其中每个因数均大于或等于1！

例如，对于N=3，最小9（因数1、3、9）。

## 输入格式

每行输入一个N

## 输出格式

输出每一个N对应的答案

## 样例 #1

### 输入

```
\n3\n1024\n\n
```

### 输出

```
\n9\n97947700\n\n
```

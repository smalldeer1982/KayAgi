# BPORT - Building Ports

## 题目描述

在比特兰河上，需要建造 $N$ 个港口。由于贸易活动仅在这条河流上的 $Y$ 英里范围内进行，因此从起始点到最后一个港口的距离必须为 $Y$ 英里。（可以将起始点视为第 $0$ 个港口。）

为了优化重要港口之间的船只移动，设计要求任意两个连续港口之间的距离必须限制在某个整数英里范围内。

你的任务是计算所有可能的港口排列方案的数量，并对结果取模 $1000000007$。

# SHOPPERS - SHOPPERS

## 题目描述

有 $N$ 位顾客来到一家超市购物，超市里有 $M$ 种商品。每位顾客只买自己感兴趣的商品，他们计划总共购得 $K$ 件商品。每位顾客最多只能购买一件商品。有的顾客可能什么都不买，有的商品可能没有人买。

## 输入
第一行是测试用例的数量。

每个测试用例的第一行包含三个整数 $N$、$M$ 和 $K$，用空格隔开。接下来的 $N$ 行每行有 $M$ 个字符，其中第 $i$ 行的第 $j$ 个字符若为 `1`，则表示第 $i$ 位顾客喜欢第 $j$ 个商品，否则为`0`。

## 输出
对于每个测试用例，输出可以购买 $K$ 件商品的不同方案的总数量。

## 说明/提示

- $1 \leq K \leq N$
- $M \leq 13$

## 示例
### 输入
```
1
4 4 2
1111
0100
0100
1100
```

### 输出
```
14
```
在这个例子中，有 14 种不同的方法让顾客买到总共 2 件商品。

# RANJAN05 - Road Map

## 题目描述

在 Andhere Nagar，有 $x$ 个村庄。每个村庄都有一个从 $1$ 到 $x$ 的编号。首都的编号为 $a$。这个国家的所有道路都是双向的，道路布局呈现为树状结构，这意味着从首都到每个村庄都有且仅有一条路径。在 Andhere Nagar 的历史记录中，道路图是这样保存的：对于每个编号不是首都 $a$ 的村庄 $i$，记录了从首都到 $i$ 的路径中倒数第二个村庄的编号 $p_i$。

有一天，国王 Choupat Raja 决定将首都从村庄 $a$ 迁到村庄 $b$。这样的改变使得先前记录的道路图信息不再准确。请协助国王重新整理出以村庄 $b$ 为新首都的道路图，并按之前的格式进行描述。

## 输入格式

第一行包含三个用空格分隔的整数 $x$、$a$ 和 $b$（$2 \leq x \leq n$），分别代表村庄的数量、旧首都的编号以及新首都的编号。

接下来一行包含 $x-1$ 个用空格分隔的整数，这些整数是旧的道路图表示。对于每个村庄（除去首都 $a$ 之外），给出了一个整数 $p_i$，表示从首都到村庄 $i$ 的路径上倒数第二个村庄的编号。所有村庄按照编号递增的顺序列出。

## 输出格式

输出 $x-1$ 个整数，它们表示新的道路图，格式与输入相同。

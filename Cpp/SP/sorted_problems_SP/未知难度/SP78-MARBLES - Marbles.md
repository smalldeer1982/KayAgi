# MARBLES - Marbles

## 题目描述

Hänschen 梦见自己置身于一个拥有无穷多弹珠的商店。他被允许从中挑选 $n$ 个弹珠。这些弹珠有 $k$ 种不同的颜色，每种颜色的弹珠数量也是无限的。Hänschen 想给每种颜色至少选一个弹珠，但他的选择方式还是多种多样。在他努力做出决定时，他醒了过来。现在，他询问你，在梦中，他到底有多少种不同的选择可能性。请注意，相同颜色的弹珠之间是无法区分的，并且弹珠的选择顺序不影响结果。

## 输入格式

第一行输入一个整数 $T \le 100$，表示有多少个测试用例。接下来的每一行包含一组测试用例，由两个整数 $n$ 和 $k$ 组成。其中，$n$ 表示 Hänschen 选择的弹珠总数，$k$ 表示弹珠的不同颜色数量。保证 $1 \le k \le n \le 1000000$。

## 输出格式

对于每个测试用例，输出 Hänschen 能够选择的不同方案数量。你可以假设这个数可以用有符号的 64 位整数表示。

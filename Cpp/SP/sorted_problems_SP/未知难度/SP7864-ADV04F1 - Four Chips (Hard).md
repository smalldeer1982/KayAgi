# ADV04F1 - Four Chips (Hard)

## 题目描述

有一个 $n$ 列的木板，每一列分别用 1 到 $n$ 的整数编号。

前四列中有一个滑块，我们要移动滑块以让它们到达对应的位置。

移动时，滑块只能在木板上移动，且每列最多只能有一个滑块。

因此，一次可以将一个滑块移动到**相邻单元**，或相对任何其他滑块**对称地移动**它。

比如，如果相对于第 13 列中的滑块对称地移动第 10 列中的滑块，它将最终移动到第 16 列中。

您需要确定达到特定滑块配置所需的最小次数。

## 输入格式

**本题有多组数据**。

第一行一个数字 $T$，表示测试用例的数量。

接下来的 $T$ 行，每行一个测试用例。

对于每组测试用例：

由四个整数 $a_1$、$a_2$、$a_3$、$a_4$ 组成，表示滑块应处于最终配置中的列数。

## 输出格式

对于每个测试用例，打印答案。

## 样例 #1

### 输入

```
2
1 2 3 4
1 3 4 6
```

### 输出

```
0
1
```

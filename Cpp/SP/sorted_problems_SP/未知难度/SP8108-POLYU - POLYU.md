# POLYU - POLYU

## 题目描述

在巨国，国王 AC.AC 和他的数十亿位后宫们都非常喜欢吃一种名叫巨果的水果。巨果的形状相当特别，它可以被看作一个三角形。在笛卡尔坐标系里有 $N$ 个巨果，它们可能互相重叠。国王想知道被这些巨果覆盖的总面积。

## 输入格式

第一行是一个整数 $T$，表示测试用例的数量（$T \leq 100$）。

对每个测试用例，第一行包含一个整数 $N$（$N \leq 100$），表示巨果的数量。接下来 $N$ 行中，每行包含六个浮点数 $x_0, y_0, x_1, y_1, x_2, y_2$，表示一个巨果的三个顶点的坐标（所有 $|x_i| \leq 10^4$ 且 $|y_i| \leq 10^4$）。

## 输出格式

对于每个测试用例，输出一行。首先输出 `Case idx: `，其中 `idx` 是测试用例编号，从 1 开始。然后输出覆盖的总面积，保留三位小数。

# DIST - Distance

## 题目描述

给定 $N$ 个城市的坐标 $(X_i,Y_i)$ ，以及一个对应的人气值 $P_i$ 。 你需要加入一个新城市 $(X,Y)$ ，使得该城市到其他城市的距离之和最小。

两城市之间的距离计算公式： $|X_i-X|^{P_i} + |Y_i-Y|^{P_i}$

## 输入格式

输入包含多组测试数据。

第一行为测试数据的个数。

对于每组测试数据，第一行为整数 $N$，表示城市的个数。

接下来有 $N$ 行，第 $i$ 行用来描述第 $i$ 个城市的 $(X_i,Y_i,Z_i)$

输入仅包含整数

## 输出格式

对于每一个测试数据，输出最小的距离之和，精确到小数点后 $3$ 位

## 输入输出样例 

### 样例输入 #1

```
1
3
1 2 0
4 5 1
2 4 2
```


### 样例输出 #1

```
4.500
```

## 说明/提示

对于每个测试点，测试数据的个数 $\le 10$

对于 $100\%$ 的数据，$N \le 10^5$

对于 $100\%$ 的数据，$|X_i,Y_i| \le 1000$

对于 $100\%$ 的数据，$0 \le P_i \le 3$

## 样例 #1

### 输入

```
\n1 \n3 \n1 2 0 \n4 5 1 \n2 4 2\n
```

### 输出

```
4.500\n\n
```

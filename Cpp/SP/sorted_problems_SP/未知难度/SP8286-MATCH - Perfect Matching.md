# MATCH - Perfect Matching

## 题目描述

给定一个二分图，其中每边各有 $N$ 个节点（$1 \leq N \leq 300$）。请判断这个图的完美匹配数是奇数还是偶数。

## 输入格式

第一行是一个整数 $T$（$1 \leq T \leq 20$），表示测试用例的数量。接下来有 $T$ 组数据，每组数据的首行是一个整数 $N$（$1 \leq N \leq 300$），表示每边的节点数。随后有 $N$ 行，其中每行是一个由 $0$ 和 $1$ 组成的字符串。如果字符串的第 $i$ 行（$1 \leq i \leq N$）中，第 $j$ 个字符（$1 \leq j \leq N$）为 $1$，则表示左边的第 $i$ 个节点与右边的第 $j$ 个节点之间有一条边。具体格式请参照示例。

## 输出格式

请输出 $T$ 行，每行写 `Odd` 或 `Even`，表示对应测试用例中完美匹配数的奇偶性。具体输出格式请参照示例。

# ROADTRIP - Road trip

## 题目描述

菲利亚·福克和路路通现在正开着他们的新车去旅行。它们从加油站$A_0$开始，需要到加油站$A_N$，汽车的储油量上限为 $C$，汽车一开始的油箱是空的，问要成功旅行需要的最少费用。

## 输入格式

输入第一行一个整数表示数据组数 $T(\leq10)$

接下来每组数据第一行两个整数$n(\leq50000)$和$C(\leq108)$

接下来$n$行，每行一个整数表示第0个加油站到该加油站的距离$(\leq108)$，第0个休息站的距离默认是0。数据保证休息站距离递增；

接下来$n$行，每行一个整数表示加油站的加油费用$(\leq108)$，即汽车行驶一公里所需的费用,最后一个休息站的费用不会给出（也不需要）。

## 输出格式

对于每组数据，输出一行一个整数，表示最少油费。

## 样例 #1

### 输入

```
2

10

40
50
1
2
1

10
20

```

### 输出

```

```

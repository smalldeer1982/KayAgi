# CCHESS - COSTLY CHESS

## 题目描述

在罗马国，国际象棋被视为一种皇家游戏。每走一步，玩家都必须向皇帝 Jurg 缴纳一定的费用。小绿人（LGMs）在下象棋方面颇具才华，但由于国际象棋的游戏成本高昂，他们请你帮忙计算从一个位置移动到另一个位置所需支付的最小费用。马可以通过多步移动达到目的地。

**约束条件：**

- 棋盘为 8×8 的大小，左下角的单元格为坐标 (0, 0)。
- 马只能按标准的国际象棋规则移动：可以走“日”字，即两格长、一格宽或一格长、两格宽。
- 如果马从位置 (a, b) 移动到 (c, d)，小绿人需要支付 a*c + b*d 的费用给皇帝 Jurg。

## 输入格式

测试用例数量在 100 到 150 之间。每个测试用例由四个空格分隔的整数构成。前两个数 a 和 b 表示马的起始位置，后两个数 c 和 d 表示马的目的位置。输入直到文件结束。

## 输出格式

对每个测试用例，输出他们需要支付的最小费用，结果为单独的一行。如果无法到达目的地，则输出 -1。

# BTCODE_C - Fun With Inequalities

## 题目描述

现有 $n$ 个不等式，每个不等式由四种类型之一构成：
- 类型 1：$x > v$
- 类型 2：$x < v$
- 类型 3：$x = v$
- 类型 4：$x \neq v$

其中，变量 $x$ 只能是非负整数。

你的任务是找到一个特定的 $x$ 值，使得能够满足最多数量的不等式，并输出这个最大数量，以及实现这一目标的最小 $x$ 值。

## 输入格式

第一行输入一个整数 $n$，表示不等式的总数。  
接下来 $n$ 行，每行有两个空格分隔的整数 $t_i$ 和 $v_i$，分别表示不等式的类型和不等式右边的数值。

## 输出格式

输出两个整数，用空格隔开。第一个整数表示可以满足的不等式的最大数量，第二个整数表示达到该最大数量时的最小 $x$ 值。

## 数据范围

$$1 \le n \le 10^5, \quad 1 \le t_i \le 4, \quad 0 \le v_i \le 10^9$$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
4
1 10
2 9
3 7
4 4
```

### 输出

```
3 7

Constraints:
```

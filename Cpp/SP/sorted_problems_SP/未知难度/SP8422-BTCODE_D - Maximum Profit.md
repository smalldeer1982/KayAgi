# BTCODE_D - Maximum Profit

## 题目描述

Chakra 是一位年轻有为的企业家，他拥有并经营着 Quickbyte 餐厅连锁，目前有 $M$ 家餐厅正在运营。他把每天分为 $N$ 个时间段。在每家餐厅 $i$ 的每个时间段 $j$ 内，都有 $A_{ij}$ 名服务员与 $B_{ij}$ 名顾客。作为一个注重质量的人，Chakra 希望每个服务员在任何一个时间段最多只接待一位顾客。因为他日程繁忙，每家餐厅一天只在一个时间段内营业。由于一天中顾客的饥饿程度和需求不同，他们愿意支付的价格也不同，在餐厅 $i$ 的时间段 $j$ 内价格为 $C_{ij}$。

你的任务是，根据给定的 $A_{ij}$、$B_{ij}$ 和 $C_{ij}$ 的数值，计算出 Chakra 在一天中可以获得的最大利润。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的个数。
对于每个测试用例，第一行包含两个整数 $M$ 和 $N$，代表餐厅的数量和时间段数量。
接下来的 $M$ 行，每行包含 $N$ 个整数。第 $i$ 行的第 $j$ 个数表示 $A_{ij}$。
再接下来的 $M$ 行，每行也包含 $N$ 个整数，第 $i$ 行的第 $j$ 个数表示 $B_{ij}$。
接下来 $M$ 行的每行同样包含 $N$ 个整数，表示 $C_{ij}$ 的值。

## 输出格式

针对每个测试用例，输出一个整数，表示 Chakra 在这一天中可以实现的最大利润。
允许多家餐厅在同一时间段内同时营业。

## 说明/提示

- $1 \le t \le 10$
- $1 \le M \le 100$
- $1 \le N \le 100$
- $0 \le A_{ij} \le 100$
- $0 \le B_{ij} \le 100$
- $1 \le C_{ij} \le 1000$

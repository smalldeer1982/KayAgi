# PSWITCH - Party Switching

## 题目描述

Seraph 是个聪明的男孩，在他的生日那天，他为庆祝活动装饰了许多灯。这些灯总共有 $N$ 个，编号为 $1$ 到 $N$。这些灯接入了一个有四个按钮的控制器，每个按钮的功能如下：

1. 按下此按钮，所有灯的状态将从关闭变为开启，或从开启变为关闭。
2. 按下此按钮，所有奇数编号的灯的状态将改变。
3. 按下此按钮，所有偶数编号的灯的状态将改变。
4. 按下此按钮，编号为 $3K + 1$ 的灯的状态将改变。

控制器上有一个计数器 $C$，记录按下按钮的总次数。初始时，所有灯处于开启状态，并且计数器 $C$ 被设为 $0$。随后，你将获得活动结束时灯的状态信息，需要根据提供的信息计算出可能的配置数量。

## 输出格式

输出所有可能的配置数量，并按字典序排列。如果没有符合条件的配置，输出 `Impossible`。

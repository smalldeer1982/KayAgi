# MAIN73 - Manoj and Pankaj 

## 题目描述

Manoj和Pankaj在N*M网格上玩以下游戏，每个单元格要么是空的，要么包含一块石头。

## 输入格式

每个测试用例的第一行包含两个整数N和M(1<=N，M<=200)，下一行N行中每一行都包含一个字符串，如果存在“*”，则下一行N行上的JTH字符是“。”(空)输入结束时N，M=0，0。这是不能处理的。

## 输出格式

对于每个测试用例，如果Manoj获胜，则打印‘Manoj’，否则打印‘Pankaj’。

## 样例 #1

### 输入

```
2 2
.*
.*

2 2
.*
*.
0 0
```

### 输出

```
Pankaj
Manoj
```

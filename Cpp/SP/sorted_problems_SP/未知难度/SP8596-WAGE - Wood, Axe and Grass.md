# WAGE - Wood, Axe and Grass

## 题目描述

Danny 在一个二维网格上开创了一项新文明。最初，每个网格位置可能被三种生命形式之一占据：树林、斧头或草。每天，这些生命形式会在水平或垂直相邻的格子上发生战斗。在这种战斗中，树林总是能战胜斧头，斧头则总是能打败草，而草又总能胜过树林。一天结束后，胜利者将占领失败者的领地，而失败者则会从该位置撤退。请计算经过 $n$ 天后，每种生命形式在网格上的分布情况。

## 输入格式

第一行输入一个整数 $t$，表示测试用例的数量。每个测试用例的开始由三个不超过 100 的整数组成：$r$ 表示网格的行数，$c$ 表示网格的列数，$n$ 表示经过的天数。接下来的 $r$ 行，每行包含 $c$ 个字符，字符可以是 W、A 或 G，分别代表网格上被树林、斧头或草占据。

## 输出格式

对于每一个测试用例，输出第 $n$ 天结束时的网格状态。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
\n2 \n3 3 1 \nWWW \nWAW \nWWW \n3 4 2 \nWAGW \nAGWA \nGWAG
\n\n
```

### 输出

```
WWW \nWWW \nWWW \n&nbsp;\nWWWA \nWWAG \nWAGW
```

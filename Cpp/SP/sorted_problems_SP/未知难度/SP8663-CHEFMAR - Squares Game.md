# CHEFMAR - Squares Game

## 题目描述

### 题意翻译
Fat Tony和Fit Tony正在玩方块绘画游戏。

平面上有 $n$ 个正方形。正方形的边与坐标轴平行。

正方形不相交，但其中一些可以在其他正方形的内部。

每回合中，轮到的玩家可以选择任何正方形，并将其内部区域涂成黑色。涂漆的方块内的所有方块也都涂成了黑色。

玩家不能画已经画过的方块。不能再操作的玩家失败。

Fat Tony先手，假设两名玩家都足够聪明，求出游戏的获胜者。此外，如果Fat Tony能赢得比赛，求出他必须在第一回合画哪个方块才能获胜。如果有许多方块保证Fat Tony获胜，请选择数字最小的一个。

## 输入格式

第一行输入正整数 $t$ ，表示测试用例的数量。

对于每个测试用例：

第一行一个平方数 $n$ 。

接下来的 $n$ 行，每行三个整数组成 $x$ , $y$ , $a$ 依次表示正方形左下角的坐标和边长。

输入中的方块按其列出的顺序从 $1$ 到 $n$ 编号。

## 输出格式

对于每个测试用例：

如果Fat Tony获胜，输出“Fat x”。

其中 $x$ 是为使Fat Tony获胜，需要在第一回合上绘制的方块编号，（如果有很多这样的方块，请选择数字最小的方块）

如果Fit Tony获胜，输出“Fit”。

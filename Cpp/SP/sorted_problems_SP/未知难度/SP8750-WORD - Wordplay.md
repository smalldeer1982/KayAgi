# WORD - Wordplay

## 题目描述

Ivana 发明了一个有 $N$ 个字母组成的长单词，然后她记录下了这个单词中所有长度为 $K$ 的连续子串。例如，如果原始单词是 BANANA，且 $K=3$，那么 Ivana 会写下这些子串：BAN、ANA、NAN 和 ANA。显然，这些子串的数量是 $N-K+1$。

Ivana 将这些子串按字典序进行了排序（在这个例子中，排序结果为 ANA、ANA、BAN、NAN）。

然而，Ivana 一不小心忘记了原来的单词！现在，你的任务是帮助她重构这个原单词。在所有测试数据中，保证每个问题都有唯一的解。

## 输入格式

- 输入第一行为两个整数 $N$ 和 $K$。
- 接下来的 $N-K+1$ 行，每行包含一个按字典序排列的，由大写英文字母组成的子串。

## 输出格式

- 输出一行，即重构得到的原单词。

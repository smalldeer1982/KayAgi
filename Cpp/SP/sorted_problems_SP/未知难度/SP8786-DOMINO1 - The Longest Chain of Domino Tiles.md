# DOMINO1 - The Longest Chain of Domino Tiles

## 题目描述

有 $N$ 个多米诺骨牌。每个骨牌由若干个小方块组成（方块个数不一定是两个），每个方块的颜色是白色或黑色（我们用字母 B 表示白色，字母 C 表示黑色）。

你的任务是找到这些骨牌可以排列成的最长链。每个骨牌只能使用一次，也不能旋转（例如，BC 不能变成 CB）。链的构成规则是：相邻骨牌的接触方块颜色必须一致。

## 输入格式

首先输入一个整数 $N$，表示骨牌的数量。

接下来的 $N$ 行中，每行包含一个长度在 $1$ 到 $100$ 之间的字符串，表示一个多米诺骨牌。

## 输出格式

输出能够构成的最长链的长度。

## 说明/提示

$$1 \le N \le 100$$

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
4
CB
BCC
BBCC
BCBBC
```

### 输出

```
11
```

# LCDS - Longest Common Difference Subsequence

## 题目描述

给定两个整数序列，找出最长的公共子序列，使得子序列中相邻的元素之间的差值相同。  
例如，如果序列 $A$ = {$1, 5, 8, 3$} 和序列 $B$ = {$3, 10, 5$}，则具有相同相邻差值的最长公共子序列为 $A_L$ = {$1, 8, 3$} 和 $B_L$ = {$3, 10, 5$}，因为 $A_L$ 中的差值为 $7$ 和 $-5$，$B_L$ 中的差值也是 $7$ 和 $-5$。

## 输入格式

输入的第一行包含两个整数 $n$ 和 $m$，分别表示序列 $A$ 和序列 $B$ 的长度。  
接下来两行是序列 $A$ 和序列 $B$ 的元素。

## 输出格式

输出一行，表示符合要求的最长公共子序列的长度。

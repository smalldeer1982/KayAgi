# COMPLETE - Complete the Set

## 题目描述

考古学家在拓扑神庙中进行了一次令人振奋的发现。这座千年前用作仪式之地的神庙，出土了许多文物。其中，一张羊皮纸特别引起了科学家们的关注。这张羊皮纸上用古代符号记录着许多数字。

通过解密石刻文字，科学家们了解到这些数字构成一个特殊的整数集合，并且满足两个条件：

1. 集合中的任意多个整数进行按位与运算后，结果仍属于该集合。
2. 集合中的任意多个整数进行按位或运算后，结果仍属于该集合。

由于羊皮纸年代久远，部分破损，致使部分数字丢失。你的任务是根据剩余的整数，尽可能少地补全原始集合，使其成为一个完整的集合。

## 输入格式

输入由多组测试数据组成，测试数据总数小于 1100。每组测试数据的第一行为一个整数 $n$（$n > 1$），接下来的第二行包含 $n$ 个不重复的整数 $a_i$（$0 \le a_i < 2^{16}$），即羊皮纸上遗留的整数。

## 输出格式

针对每组测试数据，输出一个整数，表示为了使集合完整，最少需要补充的整数数量。如果当前数字集合已经是完整的，输出 0。

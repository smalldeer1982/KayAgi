# PATT - Pattern Matching

## 题目描述

正则表达式是由普通字符和元字符组成的字符串。

## 输入格式

输入由多组模式匹配问题组成，每组包含两行。第一行是正则表达式，第二行是要进行匹配的字符串。任何一行的长度均不超过 80 个字符。输入中若出现仅含一个 `e` 的行，则表示输入结束。

## 输出格式

对于每个匹配问题，应输出一行结果。结果行包含要匹配的字符串，但其中能够匹配正则表达式的最左边的子串需要用括号括起来。如果没有任何子串能够匹配，则直接输出原始输入字符串。

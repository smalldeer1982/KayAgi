# PROBOR - Probablistic OR

## 题目描述

众所周知，或运算（OR）是一种基本的位运算。现在，我们定义一种全新的运算，称为“概率或”（Probabilistic OR），记作 #。对于给定的实数 $p$（满足 $0 \le p \le 1$）和两个二进制位 $a$ 和 $b$，该运算有以下规则：

- 当 $a = 1$ 且 $b = 1$ 时，$\#(a, b) = 1$；
- 当 $a = 0$ 且 $b = 0$ 时，$\#(a, b) = 0$；
- 在其他情况下，$\#(a, b) = 0$ 的概率为 $p$，$\#(a, b) = 1$ 的概率为 $1 - p$。

接下来，我们引入对两个非负整数 $x$ 和 $y$ 进行“位级概率或”运算的概念。这个运算的结果是通过对 $x$ 和 $y$ 的每一个相对应位置的位进行 # 运算得到的。例如，当 $p = 0.5$，$x = 2$，$y = 4$ 时，结果可能是 $0$、$2$、$4$ 或 $6$，每个结果出现的概率为 $0.25$。现给你一组非负整数，要求你编写程序计算在给定 $p$ 的情况下，将这些数字依次进行“位级概率或”运算后所得结果的期望值。

## 输入格式

第一行输入一个实数 $p$（保证精确到小数点后两位，$0 \le p \le 1$）。接下来一行为一个整数 $n$（$1 \le n \le 100$）。再下一行包含 $n$ 个整数 $a_i$，它们表示以顺序参与运算的数（$0 \le a_i \le 10^9$）。

## 输出格式

输出在给定的 $p$ 下，对这些数进行“概率或”运算后结果的期望值，并保留两位小数。

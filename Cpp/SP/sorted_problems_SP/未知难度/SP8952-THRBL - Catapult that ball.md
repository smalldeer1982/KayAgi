# THRBL - Catapult that ball

## 题目描述

Bob 面临一个奇特的难题。在比特国，遍布着许多山丘和城市。比特国的国王要求 Bob 将魔法球从一个城市传送到另一个城市。由于需要传送的魔法球数量众多，步行将无法满足时间要求。因此，Bob 想到了一个巧妙的办法——使用投石机来快速传送。

比特国的地形被划分为若干区间，每个区间中包含一个城市和一个山丘。

当从城市 A 到城市 B 的路径上没有任何山丘的高度超过 A 所在位置的山丘时，Bob 可以成功地将魔法球从城市 A 精准地投射到城市 B。

## 输入格式

每个测试用例以两个整数 $N$ 和 $M$ 开始（$N \leq 50000$, $M \leq 50000$），分别代表区间的数量以及所需投射的魔法球数量。

接下来的一行中有 $N$ 个整数 $H_i$（每个 $H_i \leq 10^9$），以空格分隔，代表各个区间中的山丘高度。

接下来的 $M$ 行中，每行包含两个整数 $A$ 和 $B$（$1 \leq A, B \leq N$），表示需要将魔法球从城市 A 投射到城市 B。

## 输出格式

输出一个整数，表示 Bob 成功投射的魔法球总数量。

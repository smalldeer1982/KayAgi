# GUESSLNK - Number Guessing Game 2

## 题目描述

我对「GUESSING」题目特别感兴趣，经过三年，我决定再出一道基于它的“半交互式”题目。

现给定一组「GUESSING」题目的链接，要求找出服务器的响应结果。

## 输入格式

每行都包含一个链接，格式为 `http://www.spoj.com/problems/GUESSING/XXXXXXYYYYYY/`，这里的 XXXXXXYYYYYY 则是一个长度正好为 12 位的数字串。

## 输出格式

输出服务器的响应，格式为 `XAYB`。

 **本翻译由 AI 自动生成**

## 样例 #1

### 输入

```
http://www.spoj.com/problems/GUESSING/123456123456/
http://www.spoj.com/problems/GUESSING/000000000000/
```

### 输出

```
1A3B
1A5B
```

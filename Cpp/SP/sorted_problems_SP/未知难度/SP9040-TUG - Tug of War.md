# TUG - Tug of War

## 题目描述

$N$个人之间将举行一场拔河比赛，给出每个人的力量大小，问能否从中选出若干个人，将他们分成两组，使得双方力量之和相等。

## 输入格式

第一行一个整数$T$，表示测试数据的组数。（$1 \leq T \leq 200$）

每一组测试数据的第一行一个整数$N$，接下来一行$N$个整数，表示每个人的力量。（$1 \leq N \leq 100000$）

每个人的力量是一个大于0，小于100的整数。

## 输出格式

对每组测试数据输出一行`YES`或者`NO`，表示是否能将所有人分成两组，使得双方力量之和相等。

## 样例 #1

### 输入

```
2
4
10 20 30 40
31
0 18 15

```

### 输出

```
YES
NO
```

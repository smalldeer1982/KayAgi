# HANDS - Clock Hands

## 题目描述

分针和时针之间如胶似漆，而秒针则是一名间谍，时刻关注着它们在一天中的某个时间区间里重叠了多少次。你的任务是帮助秒针计算出这个次数。

## 输入格式

第一行是测试用例的数量 $t$。接下来的每个测试用例包含两行，分别是起始时间和结束时间，格式为 (hh:mm)。其中，hh 的取值范围是 00 到 23，mm 的取值范围是 00 到 59。如果分针和时针在起始时间正好重叠（例如 12:00），那么这一重叠不计算在内。时间条件为 00:00 <= '开始时间' < '结束时间' <= 23:59。

## 输出格式

对于每个测试用例，输出一行，表示在给定时间区间内分针和时针重叠的次数。

**示例**

输入：
```
5
00:00
03:00
00:00
00:10
06:00
12:00
11:59
12:01
00:00
12:00
```

输出：
```
2
0
6
1
11
```

# TETRAHED - Point in tetrahedron

## 题目描述

在三维空间中，给定 5 个点，每个点的坐标都是三个整数，并且满足 $0 \le x, y, z < 100$。这 5 个点的顺序是随机的，其中有 4 个点构成了一个四面体的顶点，而其中 1 个点位于这个四面体的内部。请你找出这个内部点在点数组中的位置（索引为 1 到 5）。

## 输入格式

输入的第一行是测试用例的数量 $T$（$T < 1000$）。  
对于每一个测试用例，接下来 5 行，每行提供 1 个点的坐标，坐标由 $x$、$y$ 和 $z$ 三个整数组成，数值之间用空格分隔。

## 输出格式

对于每个测试用例，输出一行，表示位于四面体内部的点的索引。

## 说明/提示

- 坐标值满足 $0 \le x, y, z < 100$
- 测试用例数量 $T < 1000$

 **本翻译由 AI 自动生成**

### 输入

```
1\n5 72 66\n92 23 68\n60 60 49\n74 78 33\n67 76 27
```

### 输出

```
3
```

# GHOSTS - Ghosts having fun

## 题目描述

鬼魂们住在一个拥有 $K$ 个房间的大城堡中。历史悠久的他们感到疲惫，于是决定购买一些传送门。每个传送门只能单向使用，以避免冲突。鬼魂们已经确定了想要建设的传送门以及建造的先后顺序。鬼王 Bob 希望你帮助检查传送门的清单，并选择哪些传送门不应该建造。他不希望鬼魂们通过传送门进入无穷无尽的循环。

## 输入格式

第一行包含一个整数 $K$，表示城堡中的房间数量，$K \leq 1000$。

第二行包含一个整数 $T$，表示传送门的数量，$T \leq 300000$。

接下来的 $T$ 行中，每行包含两个整数 $a, b$，表示一个从房间 $a$ 到房间 $b$ 的传送门，其中 $1 \leq a, b \leq K$。

## 输出格式

输出不应建造的传送门，每行一个。以 `0 0` 结束结果输出。

## 说明/提示

$K \leq 1000$

$T \leq 300000$

# DSUBTREE - Distinct Subtrees

## 题目描述

在一棵无根树中，子树被定义为这棵树的一个连通部分。如果存在一种节点之间的映射，使得两个子树在映射后有相同的边，便认为这两个子树是相同的。

你的任务是计算这棵无根树中有多少种不同的子树。

## 输入格式

输入包含多组测试用例。第一行输入一个整数，表示测试用例的数量。

对于每个测试用例，第一行包含一个整数 $n$，表示树的节点数（$1 \le n \le 15$）。

接下来有 $n-1$ 行，每行包含两个整数，表示树的一条边。节点编号从 1 开始。

## 输出格式

对于每个测试用例，输出这棵树中不同子树的数量。

## 说明/提示

1 ≤ n ≤ 15

## 样例 #1

### 输入

```
2
3
1 2
1 3
9
9 4
4 3
1 3

```

Case #1: 3
Case #2: 21

# MARIO2 - Mario and Mushrooms

## 题目描述

马里奥通常沿着蘑菇王国附近的小道散步来放松自己。邪恶的库巴国王注意到了这一点，在路上放了很多蘑菇。有两种类型的蘑菇，好蘑菇和坏蘑菇。坏蘑菇会使马里奥的生命值降低m点，另一方面，好蘑菇会使Mario的生命值增加1点。蘑菇被随机放置在轨道上，马里奥将一个接一个地接收它们。一旦马里奥收到蘑菇后生命值变为零或更低，他就会死亡。
请注意，马里奥的HP以零开始，因此如果第一个蘑菇是坏的，马里奥将立即去世。幸运的是，如果马里奥收到所有蘑菇，他的HP为1。换句话说，如果路上有k个坏蘑菇，也将有m*k+1个最大蘑菇。
桃子公主想知道马里奥活着的可能性。请帮她算出答案。

## 输入格式

有几个测试用例。第一行只包含一个整数，表示测试用例的数量。
对于每个测试用例只有一行，包含两个整数：m和k，表示如果马里奥收到坏蘑菇，他将减少的HP点数，以及轨道上坏蘑菇的数量。（1<=m<=1000，1<=k<=1000）

## 输出格式

对于每个测试用例，只输出实数，表示如果马里奥一个接一个地收到所有随机放置的蘑菇，他将存活的可能性。答案应四舍五入至小数点后的八位数字。

**忽略额外的空白。**

## 样例 #1

### 输入

```
2
1 1
60 80
```

### 输出

```
Case #1: 0.33333333
Case #2: 0.00020488
```

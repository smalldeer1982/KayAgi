# AND - Magic Bitwise AND Operation

## 题目描述

## 题目简意
给定$n$个整数，你的任务是从中取$k$个数使得取得的数进行按位与（$AND$）操作后是一个最小值。

## 输入格式

数据共有多组，每一个测试样例会有两个整数$n,k$，随后一行共有$n$个整数，每个整数$a_i< 2^{60}$。  
**注意：有大约$1000$个测试用例，不过其中的$90\%$数据强度都是比较小的。**

## 输出格式

每一个测试用例输出一个整数，代表最小值

## 说明/提示

$1\le n\le 40,1\le k\le n$

## 样例 #1

### 输入

```
2
3 2
5 6 7
8 2
238 153 223 247 111 252 253 247
```

### 输出

```
Case #1: 4
Case #2: 9
```

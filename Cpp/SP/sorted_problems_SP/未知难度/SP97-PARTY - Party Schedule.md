# PARTY - Party Schedule

## 题目描述

## 题意简述
你有一定的金钱来参加 $n$ 个派对。你事先得到了这些派对各自的入场费，并估算好了每场派对可以得到的乐趣值。你需要写一个程序来获取用你这些金钱来参加这些派对能够得到的最大乐趣值。

## 输入格式

**本题包含多组数据，当你输入的金钱量和 $n$ 均为 $0$ 时结束输入。**

每组数据输入的第 $1$ 行为金钱量和 $n$ （表示可以选择的派对数） 

金钱量 $<= 500, n <= 100$。

随后的 $n$ 行每行表示一场派对，包括两个数。分别是这场派对的入场费和参加这场派对能获得的乐趣值

$5 <=$ 入场费 $<= 25, 0 <=$ 乐趣值 $<= 10$。 

随后是一个换行代表这组数据的结束。

## 输出格式

对于每组数据而言，输出你能得到的最大乐趣值所要付出的入场费的总和以及最大乐趣值，请用空格分开。

### 输入

```
50 10

```

```

```

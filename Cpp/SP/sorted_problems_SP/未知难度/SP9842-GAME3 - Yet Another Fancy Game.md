# GAME3 - Yet Another Fancy Game

## 题目描述

伊维卡和玛丽卡正在玩一个有意思的游戏。

在游戏中，她们先选择了一个自然数 $n$，并令 $m=1$。

接下来，伊维卡和玛丽卡轮流行动，伊维卡先行动，在每次行动中，行动者可以选择将 $m$ 加 $1$ 或将 $m$ 乘 $2$，需要确保行动后 $m$ 不超过 $n$。

当一名行动者的行动结束后，如果 $m=n$，那么这名行动者失败，另一名获胜。

你需要编写一个程序来确定获胜者，你可以假设她们都绝顶聪明。

## 输入格式

先输入一个正整数 $T$，表示游戏的次数。

接下来 $T$ 行，每行一个自然数 $n$，表示在这一轮游戏中，她们选择的数 $n$。

## 输出格式

输出共 $T$ 行，如果伊维卡获胜，输出`Ivica`，否则输出`Marica`。

## 说明/提示

$1\le T\le 5,1\le n\le 10^{15}$。

## 样例 #1

### 输入

```
4\n2\n3\n4\n5\n
```

### 输出

```
\n\nMarica\nIvica\nMarica\nMarica
```

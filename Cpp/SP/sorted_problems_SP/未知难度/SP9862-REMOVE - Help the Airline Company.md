# REMOVE - Help the Airline Company

## 题目描述

在遥远的某个国家，有 $N$ 座城市，每两座城市间都有一条双向直飞航线。由于经济危机的影响，航空公司决定尽量多地减少航线数量。

这些航线将通过逐条取消的方式来减少。为了确保城市间的连接不受到严重影响，每删除一条航线时，它必须属于一个长度为 4 的环。换句话说，如果存在四座城市 $A$、$B$、$C$ 和 $D$，当前有航线连接 $AB$、$BC$、$CD$ 和 $DA$，那么我们可以从中删除任意一条航线。

可以证明，在这个过程中，最终至少能保留 $N$ 条航线（也就是说，无法在给定条件下再多删一条）。你不需要去证明这一点，只需编写一个程序来协助航空公司逐步删除航线，直到仅剩下 $N$ 条航线为止。

## 输入格式

输入仅包含一个整数 $N$，表示城市的数量。

## 输出格式

对于每条被删除的航线，输出一行包含四个整数 $A$、$B$、$C$ 和 $D$，代表形成环的城市。这些数字表明你正在移除航线 $AB$。

## 说明/提示

- $4 \le N \le 10^5$

# NWERC11G - Smoking gun

## 题目描述

给定一个包含若干整数的数组，你需要找出其中的一个子数组，使得该子数组的所有元素之和最大。输出这个最大和。

## 输入格式

第一行输入一个整数 n，表示数组的长度。

接下来一行输入 n 个整数，表示数组中的元素。

## 输出格式

输出一个整数，表示最大子数组和。

## 说明/提示

- 1 ≤ n ≤ 100,000
- 每个数组中的元素的绝对值不超过 1000

要求：算法的时间复杂度为 O(n)。

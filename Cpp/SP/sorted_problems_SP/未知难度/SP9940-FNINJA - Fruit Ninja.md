# FNINJA - Fruit Ninja

## 题目描述

在一个无限大的屏幕上，有许多水果，每个水果都可以看作是一个圆。现在，我们需要确定在一次触摸中最多能切到多少个水果。

## 输入格式

输入的第一行是一个整数 **T**，表示测试用例的数量（$1 \le T \le 50$）。

对于每一个测试用例，首先输入一个整数 **N**，表示水果的数量（$1 \le N \le 1000$）。接下来的 **N** 行描述每个水果的具体参数，每行包含三个整数 **X** $ _{i} $，**Y** $ _{i} $，**R** $ _{i} $，这三个数分别表示水果的圆心坐标 (**X**, **Y**) 和半径 **R** $ _{i} $ （$-1000 \le X, Y \le 1000$, $1 \le R_{i} \le 1000$）。

## 输出格式

对于每个测试用例，输出一个格式为 "Case #X: Y" 的字符串，其中 **X** 是测试用例的编号（从 1 开始），**Y** 是在一次触摸中最多可以切到的水果数。

题目还说明，甚至将水果的半径稍微扩大 $10^{-5}$ 后，结果依然是相同的。

# HIGH - Highways

## 题目描述

给定一张无向图，求出其生成树的个数
(请使用Matrix定理求解)

## 输入格式

第一行一个整数T，表示测试数据的个数
每个测试数据第一行给出$n,m$ 分别表示点数与边数
接下来$m$ 行，每行给出两个数$a,b$ ，表示$a,b$ 之间有一条无向边

## 输出格式

每个测试数据，输出一个整数，表示给出的无向图的生成树的个数

## 样例 #1

### 输入

```
4
4 5
3 4
4 2
2 3
1 2
1 3

2 1
2 1

1 0

3 3
1 2
2 3
3 1
```

### 输出

```
8
1
1
3
```

# PON - Prime or Not

## 题目描述

给你一个正整数，你需要回答一个问题：“这是一个素数吗？”

## 输入格式

第一行一个整数 $t$ 代表数据的组数，接下来 $t$ 行，每行一个整数 $n$（$1 \leq t\le 500,2\le n\le 2^{63}-1$）。

## 输出格式

对于每一组数据，若它是素数，则输出 `YES`，若不是素数，则输出 `NO`。

## 样例 #1

### 输入

```
5
2
3
4
5
6
```

### 输出

```
YES
YES
NO
YES
NO
```

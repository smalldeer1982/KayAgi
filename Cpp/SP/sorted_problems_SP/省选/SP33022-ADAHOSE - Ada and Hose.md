# ADAHOSE - Ada and Hose

## 题目描述

Ada是个农民。她拥有一块正方形的田地。每块子田上都缠绕着一根软管。所有的软管在它们相交的地方都被另组成一个更大的软管，所以水可以在它们两个，甚至是四个之间任意流动。每根软管都有自己的流量。

田地上面有一口大井（拥有无限的水源），田地下面有一辆洒水车。此外，更大的软管（无限大）要从井里将水引到田地的顶部，从底部引入洒水车。见下图：

![morass_flow.png](https://i.loli.net/2020/10/05/CMsHZDJRiWwjom6.png)

你的任务很简单：计算水从水井到洒水车的最大总流量。

## 输入格式

输入的第一行包含一个整数 $n \le 1000$ ，表示田地的大小。

接下来的 $n$ 行每行包含 $n$ 个整数 $a_{i,j} \le 1000$ ，表示每个软管的流量。

## 输出格式

输出可实现的最大流量。

## 输入输出样例

输入：$\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $输出：
```
4                           8
2 0 0 2
0 2 2 0
2 0 0 2
0 2 2 0
```

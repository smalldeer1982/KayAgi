# TWICE - Twice

## 题目描述

给定一个字符串，找到在字符串中至少出现两次的最长子字符串（出现次数可能重叠）。

## 输入格式

第一行输入字符串的长度。
第二行输入该字符串。

## 输出格式

输出在字符串中至少出现两次的最长子字符串的长度。如果没有这样的子字符串，则输出0。

## 样例 #1

### 输入

```
\n11\nsabcabcfabc\n\n
```

### 输出

```
\n3\n\nInput:\n18\ntrutrutiktiktappop\n\nOutput:\n4\n
```

# SUBLEX - Lexicographical Substring Search

## 题目描述

给定一个字符串，求本质不同排名第k小的子串

## 输入格式

第一行给定主串(len<=90000)

第二行给定询问个数T<=500

随后给出T行T个询问，每次询问排名第k小的串，范围在int内

## 输出格式

对于每一个询问，输出T行，每行为排名第k小的串

感谢@Creeper_LKF 提供的翻译

## 样例 #1

### 输入

```
aaa
2
2
3

```

### 输出

```
aa
aaa
```

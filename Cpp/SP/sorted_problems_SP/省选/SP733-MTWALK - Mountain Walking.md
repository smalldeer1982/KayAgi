# MTWALK - Mountain Walking

## 题目描述

题意简述：现在给一个N*N的矩阵，找一条路径从左上角走到右下角，每次可以向上下左右四个方向中某个方向走。要求走过的点中，数字最大的减去最小的。要求值越小越好。现在就是要求这个值。

## 输入格式

第一行给出一个数字N(2 <= N <= 100),代表矩阵的大小。接下来一个N行N列的矩阵，里面每个数字的值在[0,110]之间。

## 输出格式

一个数字，如翻译中所述

## 样例 #1

### 输入

```
5
1 1 3 6 8
1 2 2 5 5
4 4 0 3 3
8 0 2 3 4
4 3 0 2 1
```

### 输出

```
2
```

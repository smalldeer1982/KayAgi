# JZPEXT - Beautiful numbers EXTREME

## 题目描述

我们定义“美丽的数”：当且仅当它可以被它的每个非零数位上的数整除。给定一个区间[l，r]，请计算有多少个美丽的数字n满足l<=n<=r。

## 输入格式

先输入t

t行，每行两个数字l，r，代表查询间隔[l，r]

## 输出格式

t行，第t行是第t个查询的答案。

## 样例 #1

### 输入

```
1
1 100
```

### 输出

```
33
```

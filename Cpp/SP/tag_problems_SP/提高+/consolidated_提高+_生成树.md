# MDST - Minimum Diameter Spanning Tree

## 题目描述

求最小直径生成树的直径。

## 输入格式

T:T组测试数据



n:接下来有n行数据，每行第一个数是结点，第二个数m表示有m个结点与第一个数表示的结点相连。

## 输出格式

每行输出一个数据，表示这个图的最小直径生成树的直径。

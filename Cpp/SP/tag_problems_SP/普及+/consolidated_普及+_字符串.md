# NAJPF - Pattern Find

## 题目描述

### 题意：

给出两个字符串 $A,B$ ，求 $B$ 在 $A$ 中所有出现的次数和每一次出现的位置。假如没有出现输出`Not Found`。

## 输入格式

本题有多组数据。

第一行一个整数 $T（ 1≤T≤50）$，表示数据组数。

对于每组数据：

每行两个字符串 $A,B$。所有字符均为小写英文字符， $1≤| A |,| B |≤{10^6}$
 。

## 输出格式

对于每次询问，第一行一个整数 $n$，表示 $B$ 在 $A$ 中出现的次数。

第二行 $n$ 个整数,表示每一次出现的位置。

如果没有出现，输出`Not Found`。

每次询问之间有一个空行。

### 输入输出样例
#### 输入 #1
```
3
ababab ab
aaaaa bbb
aafafaasf aaf
```

#### 输出 #1
```
3
1 3 5

Not Found

1
1
```

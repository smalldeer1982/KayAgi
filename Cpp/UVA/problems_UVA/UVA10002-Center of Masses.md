# Center of Masses

## 题目描述

找出凸多边形的质心。

## 输入格式

输入会包含一系列的凸多边形。首先输入一个数字 $n$ $(n \le 100)$ 表示多边形的顶点数量，后跟 $n$ 个不同的整数对（无特定顺序），表示每个顶点的 $x$ 和 $y$ 坐标。最后输入一个有 $m$ $(m < 3)$ 个点的不应被处理的虚假多边形。在任何多边形中，不存在三点共线。

## 输出格式

对于每个多边形，单独输出一行， 包含该多边形质心的 $x$ 和 $y$ 坐标，结果保留三位小数。

## 样例 #1

### 输入

```
4 0 1 1 1 0 0 1 0
3 1 2 1 0 0 0
7
-4 -4
-6 -3
-4 -10
-7 -12
-9 -8
-3 -6
-8 -3
1
```

### 输出

```
0.500 0.500
0.667 0.667
-6.102 -7.089
```

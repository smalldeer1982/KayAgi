# Joseph's Cousin

## 题目描述

约瑟夫问题的变形，有 $n$ 个人，即 $1,2,3,\ldots,n$，他们排成了一个圈（环）。

约瑟夫不再是走 $m$ 步杀死一个人，现在他在第 $i$ 次杀人时将走上第 $i$ 个质数步，比如第 $1$ 次杀，他会走 $2$ 步，因为第 $1$ 个质数是 $2$。

## 输入格式

给出 $n$，$n \le 3501$，即有 $n$ 个人。

## 输出格式

输出最后活下来的那个人的位置编号。（$1\sim n$）

## 说明/提示

有多组输入输出，输入 $0$ 代表输入结束。

## 样例 #1

### 输入

```
6
0
```

### 输出

```
4
```

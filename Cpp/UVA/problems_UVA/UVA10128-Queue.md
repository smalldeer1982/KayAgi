## 说明/提示

**翻译**

有一行队列有 $n$ ( $1 \leq n \leq 13$ )个人，每个人都有不同的身高 $a_i$ 。

**输入格式**

第一行 $t$ 。

对于每组数据：第一行 $n$ 。

接下来后面两个整数，表示从头可以看到的人数 $p$ 和从尾可以看到的人数 $r$ 。

**输出格式**

对于 $t$ 组数据，每组数据输出一个整数，表示可以看到的总人数。

感谢 @rpmcpp @ylinxin2010 提供的翻译。

## 样例 #1

### 输入

```
3
10 4 4
11 3 1
3 1 2
```

### 输出

```
90720
1026576
1
```

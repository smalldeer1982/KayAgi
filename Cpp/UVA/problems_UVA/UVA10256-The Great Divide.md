# The Great Divide

## 题目描述

输入多组数据，每组数据给定 $n$ 个红点坐标和 $m$ 个蓝点坐标，判断是否可以用一条直线将两种颜色的点完全隔离开（直线不能经过某个点）。

## 输入格式

每组数据第一行为两个整数 $n,m$ $(1\leqslant n,m \leqslant 500)$，接下来 $n+m$ 行每行两个整数 $x,y$ $(-1000 \leqslant x,y \leqslant 1000)$ 表示一个坐标，前 $n$ 行为红点，后 $m$ 行表示蓝点。

## 输出格式

如果存在这样一条直线输出 $Yes$，否则输出 $No$ 。

## 样例 #1

### 输入

```
4 3
100 600
200 400
600 500
300 700
400 100
600 200
500 300
4 3
100 600
400 100
600 200
500 300
200 400
600 500
300 700
0 0
```

### 输出

```
Yes
No
```

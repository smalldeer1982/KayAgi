# Again Palindrome

## 题目描述

给出一个字符串 $S$，你可以删除一些字符，使得剩下的字符串是一个回文串。 

求有多少种不同的删字符的方案。

## 输入格式

**本题有多组数据**

第一行一个整数 $T$ $(1\le T < 15)$，表示数据的组数  。
接下来 $T$ 行，每行一个长度不超过 $60$ 的字符串 $S$，表示一组数据。

## 输出格式

对于每组数据，输出一行一个整数，表示删除字符的方案数。

感谢 @[_Wolverine](https://www.luogu.com.cn/user/120362) 提供的翻译。

## 样例 #1

### 输入

```
3
BAOBAB
AAAA
ABA
```

### 输出

```
22
15
5
```

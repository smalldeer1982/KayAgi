# LCM Cardinality

## 题目描述

一对数字只有一个最小公倍数，但是一个最小公倍数不只对应一对数字。

-----

给定一个数 $n (0\le n \le2*10^{9})$ ，求出最小公倍数为 $n$ 的数对个数。

----

最简题意：给定一个数 $n (0\le n \le2*10^{9})$ , 求出：
$$\sum_{i=1}^n\sum_{j=i+1}^n[lcm(i,j)=n]$$

----

## 输入格式

循环输入一个数 $n$ ，当 $n=0$ 的时候停止输入。

## 输出格式

对于每一个输入的 $n$ ，换行输出两个数 $n,c$，用空格隔开，$c$ 表示题目所求的答案。

## 样例 #1

### 输入

```
2
12
24
101101291
0
```

### 输出

```
2 2
12 8
24 11
101101291 5
```

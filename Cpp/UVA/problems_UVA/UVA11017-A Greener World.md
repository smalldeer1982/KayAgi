# A Greener World

## 题目描述

有一个网格，每个格点都有一棵洛谷树。整个网格错切了 $\theta$ 度，于是每个格子都变成了一个菱形。现在在每个菱形的中间种一棵洛谷树，如下图所示。

![390770](https://cdn.luogu.com.cn/upload/image_hosting/0kzdbche.png)

给定一个格点多边形，求它的面积和内部的洛谷树棵数。（边界上的树不统计在内）

## 输入格式

输入包含最多 $15$ 组数据，每组数据的第一行为 $3$ 个整数 $d,\theta,N$，其中 $N$ 是多边形的顶点数。以下 $N$ 行每行包含两个整数 $x,y$，即多边形的顶点。顶点按照顺淑珍或逆时针顺序排列。

输入结束标志为 `0 0 0`。

## 输出格式

对于每组数据，输出洛谷树的棵数和多边形面积（四舍五入到最接近的整数）。

## 说明/提示

$1\leq d<10000,44<\theta<136,0\leq x,y\leq100000$。

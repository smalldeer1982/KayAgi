# Necklace

## 题目描述

现在有 $a$ 个白色珍珠，$b$ 个灰色珍珠，$c$ 个黑色珍珠，求能用这些珍珠能组成的本质不同的项链有多少种，其中两个项链视为相同，当且仅当它们能通过翻转，旋转变为完全一样的。

## 输入格式

输入的第一行为一个正整数 $n$，表示数据组数。接下来的 $n$ 行，每行三个非负整数 $a,b,c$，分别表示白色，灰色，黑色珍珠的数量。

## 输出格式

输出共 $n$ 行，第 $i$ 行对应第 $i$ 组数据的答案。

## 说明/提示

- $1\le n\le 2500$
- $3\le a+b+c\le 40$

## 样例 #1

### 输入

```
2
3 2 1
2 2 2
```

### 输出

```
6
11
```

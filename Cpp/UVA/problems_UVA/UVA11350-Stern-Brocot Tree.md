# Stern-Brocot Tree

## 题目描述

给定一个字符串，这个字符串由 `L` 和 `R` 组成。将 Stern-Brocot 树模拟成下面一个结构：

![](https://oi-wiki.org/misc/images/stern-brocot1.png)

从 $\dfrac{1}{1}$ 开始，`L` 表示左边儿子节点，`R` 表示右边儿子节点，往下进行遍历，直到字符串结束。

比如说字符串 `RLR` ，按照步骤从 $\dfrac{1}{1}$ 开始往下遍历，可以得到 $\dfrac{1}{1}\to\dfrac{2}{1}\to\dfrac{3}{2}\to\dfrac{5}{3}$ 。

## 输入格式

**本题有多组数据**

第一行一个整数 $N$，代表数据组数。

对于每组数据：

一行一个字符串，由 `L` 和 `R` 组成，用于遍历的字符串。

## 输出格式

对于每组数据：

一行一个分数，格式为 $a/b$ ，$a$ 为分子，$b$ 为分母，按照字符串遍历后的结果。

## 说明/提示

对于所有的数据，$0<N\le 10000$ ，字符串长度不长于 $90$ 。

## 样例 #1

### 输入

```
3
RL
RLR
RRL
```

### 输出

```
3/2
5/3
5/2
```

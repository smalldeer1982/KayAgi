# 拿行李（极限版） GCD - Extreme (II)

## 题目描述

得定 $n$，求
$$\sum\limits_{i=1}^n\sum\limits_{j=i+1}^n\gcd(i,j)$$
其中 $\gcd(i,j)$ 指的是 $i$ 和 $j$ 的最大公约数。

## 输入格式

**本题有多组数据。**

对于每组数据，输出一个整数 $n$，如果 $n=0$ 就终止程序。

## 输出格式

对于每组数据，输出计算结果。

## 说明/提示

对于 $100\%$ 的数据，$1 \le n \le 4\times 10^6+1$。

## 样例 #1

### 输入

```
10
100
200000
0
```

### 输出

```
67
13015
143295493160
```

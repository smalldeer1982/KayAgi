# Automatic Answer

## 题目描述

有 $T$ 次询问，对于每次询问，每次给出一个数 $n$，求出：$(n \times 567 \div 9 + 7492 ) \times 235 \div 47 - 498$ 的十位数字是多少？

## 输入格式

第一行一个数 $t$。

接下来的 $t$ 行，每行一个整数 $n$。

## 输出格式

输出共 $t$ 行，每行一个整数，表示 $(n \times 567 \div 9 + 7492 ) \times 235 \div 47 - 498$ 的十位数字。

## 说明/提示

对于 $100\%$ 的数据，$1 \leqslant t \leqslant 100$，$-1000 \leqslant n \leqslant 1000$

## 样例 #1

### 输入

```
2
637
-120
```

### 输出

```
1
3
```

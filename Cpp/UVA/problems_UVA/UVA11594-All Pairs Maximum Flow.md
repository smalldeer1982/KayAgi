# All Pairs Maximum Flow

## 题目描述

给出一个 $n$ 个节点的完全图，和一个 $n\times n$ 的描述边的矩阵 $a$。  
其中，$a_{i,j}$ 表示点 $i,j$ 之间有一条流量为 $a_{i,j}$ 的边。

请你求出每一对 $(s,t)$ 的最大流。

**本题有多组数据**

## 输入格式

第一行一个整数 $T$，表示数据的组数。

对于每组数据：  
第一行一个整数 $n$，表示节点的个数。  
接下来一个 $n\times n$ 的矩阵，表示连边的情况。

## 输出格式

对于第 $i$ 组数据，第一行输出一个字符串`Case #i:`，注意该字符串中的 $i$ 是变量（实在不懂可以去观察样例）。

接下来，一个 $n\times n$ 的矩阵 $g$。其中，$g_{i,j}$ 表示 $i$ 到 $j$ 的最大流。

**每组数据之间不需要输出空行**

## 说明/提示

$0 \le n \le 200$  
$0 \le a_{i,j},g_{i,j} \le 10000$  
感谢 @[_Wolverine](https://www.luogu.com.cn/user/120362) 提供的翻译

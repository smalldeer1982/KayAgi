# Abstract Names

## 输入格式

第一行一个数 $\texttt{a}$ ，表示有  $\texttt{a}$ 组测试点。
接下来有 $\texttt{a}$ 组，每组2行，一行一个 $ \texttt{string}$ 。

## 输出格式

如果每组的两行 $ \texttt{string}$ 删除元音字母后完全相同输出 $ \texttt{Yes}$ ，否则输出 $ \texttt{No}$ 。

## 样例 #1

### 输入

```
5
pele
polo
pele
pola
ronaldo
ronaldino
pele
pelet
pele
bele
```

### 输出

```
Yes
Yes
No
No
No
```

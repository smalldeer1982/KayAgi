# Expert Enough?

## 题目描述

一家名叫 ACM 的公司搜集了很多汽车制造商的汽车价格数据，包含这家汽车制造商所制造的汽车的最低价格和最高价格。接下来，ACM 公司打算开发一个名叫 AES 的程序，用于查询某个价格在哪家汽车制造商的最低价格和最高价格之间。然后，程序输出这家汽车制造商的名称。如果数据库里不包含或包含多个符合要求的数据，则程序输出 `UNDETERMINED`。

这不是很困难，对吧？所以你接到了 ACM 公司的任务，开发 AES。

## 输入格式

第一行一个整数 $T$（$T\le10$），代表测试样例组数。

对于每个测试样例：

- 首先输入一个整数 $D$（$D<10^4$），代表数据库的数量。

- 然后 $D$ 行，输入一个字符串和两个数字，代表汽车制造商名称、最低价格、最高价格。最低价格一定低于最高价格，且最高价格低于 $10^6$。

- 然后输入一个整数 $Q$（$Q<10^3$），代表查询次数。

- 然后 $Q$ 行，输入一个数据，代表用于查询的价格。

## 输出格式

对于每个查询，输出符合要求的汽车制造商的名称。如果数据库里不包含或包含多个符合要求的数据，请输出 `UNDETERMINED`。

注意，UVA 毒瘤输出来了！每个测试样例间输出一个空行，但每个查询之间不能输出空行。

#### 4、输入样例

```
1
4
HONDA 10000 45000
PEUGEOT 12000 44000
BMW 30000 75900
CHEVROLET 7000 37000
4
60000
7500
5000
10000
```

#### 5、输出样例

```
BMW
CHEVROLET
UNDETERMINED
UNDETERMINED
```

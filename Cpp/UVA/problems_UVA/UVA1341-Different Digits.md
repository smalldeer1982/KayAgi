# Different Digits

## 题目描述

给定一个正整数 $n$ 。

求一个正整数 $m=kn(1 \le k)$ 使得在 $m$ 上用的数字**种类**最少。

## 输入格式

不断输入 $n$ 直到 $n=0$。

## 输出格式

对于每一个数 $n$ ，输出一个 $m$ 。

若有多个可能的方案，输出最小的那一个

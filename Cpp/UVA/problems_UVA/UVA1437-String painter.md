# String painter

## 题目描述

有两个仅含有小写字母的等长字符串 $A$ 和 $B$，每次操作可以将 $A$ 的其中一个子串的所有位置修改为同一个任意字符。求将 $A$ 覆盖成  $B$ 的最小操作次数。

$1\le |a |=|b|\le100。$

## 输入格式

输入包含多组数据，每组数据由两行组成，第一行为字符串 $A$，第二行为字符串 $B$。

## 输出格式

对于每组数据，输出最小操作次数。

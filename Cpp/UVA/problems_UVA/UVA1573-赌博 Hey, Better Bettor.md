# 赌博 Hey, Better Bettor

## 题目描述

####  _“在赌场，最重要的规则是让他们继续玩，让他们回来。他们玩得越久，输得就越多，最后，我们得到了一切。”_ 
####  _（摘自1995年的电影《赌场》）_ 
# 题目大意
你在赌场上玩一个游戏，每次赌注是1美元。赢了会得到2美元，输了什么也得不到。赌场有一个优惠:在任何时候，赌场可以补偿 X% 的损失，使用优惠之后你可以退出赌场，也可以继续玩下去。退出赌场之前你只能使用一次这样的优惠。

例如，x = 20,你玩了 10 次，赢了3次，总共损失 10 - 3 * 2 = 4元，使用优惠之后损失 3.2 元。但如果你赢了6次，总共获利 6 * 2 - 10 = 2元。

假定每局比赛胜率为 p% ，输入 x，p（0≤x<100,0≤p<50），输出最优策略下最大的期望获利。

## 输入格式

输入由多个测试用例组成。测试用例由退款百分比x（0≤ x<100）和获胜概率百分比p（0≤ p<50）组成。x和p最多有两位小数点。

## 输出格式

对于每个测试用例，显示最优策略下最大的期望获利，误差不超过10的−3次方。
# 输入输出样例
输入#1
```
0 49.9
50 49.85
```
输出#1
```
0.0
7.10178453
```

# Sum of MSLCM

## 题目描述

$lcm$ 表示最小公倍数。对于一个给定的数 $n$ , $MSLCM(n)$ 表示最大的 满足 $lcm$ 为 $n$ 的数集 的数字和。

## 输入格式

多组输入数据。每组最多有200个。每行给出一个 $n~(1<n\le20000000)$,请你求出
$\sum\limits_{i=2}^n MSLCM(i)$

## 输出格式

对于每一个 $n$ ,输出 $\sum\limits_{i=2}^n MSLCM(i)$

## 样例输入
```
10
1000
0
```
## 样例输出
```
86
823080
```

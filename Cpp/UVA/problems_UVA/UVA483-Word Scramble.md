# Word Scramble

## 题目描述

给你多个单词序列，你需要将每个单词倒序输出（包括标点符号），同时使每个单词在序列中的位置保持不变。

## 输入格式

输入有多行，每行都包含一个单词序列。

## 输出格式

对于每一行，你需要将其中的每个单词倒序输出，且不改变单词在序列中的位置。

## 样例 #1

### 输入

```
I love you.
You love me.
We're a happy family.
```

### 输出

```
I evol .uoy
uoY evol .em
er'eW a yppah .ylimaf
```

# What Day Is It?

## 题目描述

## 题目背景

### 儒略历(Julian Calendar)：

1月、3月、5月、7月、8月、10月和12月是31天；

4月、6月、9月和11月是30天；

2月非闰年28天，闰年是29天，闰年四年一次，即4的倍数都是闰年（这是因为当时的古罗马天文学家计算出一年仅有365.25天长，与现在不符）。

### 格里高利历(Gregorian Calendar)：

1582年，天文学家注意到一年不是365.25天，而是接近365.2425天。因此，闰年修改为：

**如果是4的倍数且不是100的倍数，或者是400的倍数，那么是闰年。**

这样新旧日历就对不上了。为了适应新日历，1582年10月5日至1582年10月14日的十天凭空消失了，1582年10月4日的第二天为10月15日。

### 美国日历：

因为两国关系不好，当时英国和它的帝国（包括美国）没有改用公历。直到1752年，美国才改用公历，9月3日至9月13日十一天凭空消失，9月2日的第二天为9月14日。


输入一个日期，根据美国日历，算出这个日期是星期几。

## 输入格式

输入是一堆正整数（大于0），每行三个整数，代表一个日期，格式是“月-日-年”，其中月是介于1（表示1月）和12（表示12月）之间的数字，天是介于1和31之间的数字，年是正数。输入以三个零结束。

## 输出格式

输出是该日期的星期，格式如示例所示。如果是美国日历的无效日期或不存在的日期，应输出一条错误消息。

## 输入输出样例

### 输入
```
11 15 1997
1 1 2000
7 4 1998
2 11 1732
9 2 1752
9 14 1752
4 33 1997
0 0 0
```
### 输出
```
November 15, 1997 is a Saturday
January 1, 2000 is a Saturday
July 4, 1998 is a Saturday
February 11, 1732 is a Friday
September 2, 1752 is a Wednesday
September 14, 1752 is a Thursday
4/33/1997 is an invalid date.
```

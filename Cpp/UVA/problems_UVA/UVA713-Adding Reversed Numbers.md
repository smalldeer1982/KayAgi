# Adding Reversed Numbers

## 题目描述

Malidinesia 的古代喜剧演员们更喜欢喜剧而不是悲剧。不幸的是，大多数古代剧作都是悲剧。因此，ACM 的戏剧顾问决定将一些悲剧改编成喜剧。显然，这项工作非常艰难，因为剧作的基本意义必须保持不变，尽管所有事物都被转换为它们的对立面。例如，数字：如果悲剧中出现任何数字，它必须在被接受为喜剧之前转换为其反转形式。

反转数是用阿拉伯数字表示的数字，但数字的顺序是反转的。第一个数字变为最后一个，反之亦然。例如，如果悲剧中的主角有 $1245$ 个草莓，现在他有 $5421$ 个。注意，所有的前导零都会被省略。这意味着如果数字以零结尾，反转时零会丢失（例如 $1200$ 变为 $21$）。还要注意，反转数从不带有尾随的零。

ACM 需要用反转数进行计算。你的任务是将两个反转数相加并输出它们的反转和。当然，结果不是唯一的，因为任何特定的数字都是几个数字的反转形式（例如 $21$ 可能是 $12$、$120$ 或 $1200$ 的反转）。因此我们必须假设没有任何零在反转时丢失（例如，假设原始数字是 $12$）。

## 输入格式

输入包含 $N$ 个用例。输入的第一行包含一个正整数 $N$。接下来是这些案例。每个用例由一行组成，包含两个由空格分隔的正整数。这些是你需要相加的反转数。数字的长度最多为 $200$ 个字符。

## 输出格式

对于每个用例，输出一行，只包含一个整数——两个反转数的反转和。输出中应省略所有前导零。

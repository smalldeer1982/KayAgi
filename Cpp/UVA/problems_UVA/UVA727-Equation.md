# Equation

## 题目描述

## 题意

编写程序将中缀表达式转换为对应的后缀表达式。

## 输入格式

1. 需要转换的中缀表达式在输入文件中给出，每行一个字符，每个表达式最多 $50$ 行。

1. 输入文件的第一行为一个整数，表示测试数据的组数，后面接着一个空行，之后是测试数据。每组测试数据表示一个中缀表达式，后面接着一个空行。

1. 程序只需处理四则运算符：``+``，``-``，``*``，``/``。

1. 所有运算数均为个位数。

1. 运算符 ``*`` 和 ``/`` 拥有最高优先级，运算符 ``+`` 和 ``-`` 拥有最低优先级。具有相同优先级的运算符按照从左至右的顺序进行运算。括号可以改变运算符的优先级顺序。

1. 每组测试数据给出的中缀表达式均为合法的中缀表达式。

## 输出格式

对于每组测试数据，输出其对应的后缀表达式，要求将后缀表达式在一行上输出。在相邻两组测试数据的输出之间打印一个空行。

## 样例 #1

### 输入

```
1
(
3
+
2
)
*
5
```

### 输出

```
32+5*
```

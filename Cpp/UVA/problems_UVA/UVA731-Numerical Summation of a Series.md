# Numerical Summation of a Series

## 题目描述

给级数

$$\phi(x)=\sum_{k=1}^{\infty}\frac{1}{k(k+x)}$$

在 $x=0,0.1,0.2,\cdots,300.0$ 时 $\phi(x)$ 的值，误差小于等于 $10^{-10}$。

## 输入格式

没有输入。

## 输出格式

你可以使用下面的 C 语言输出：

```c
printf("%6.2f%17.12f\n",x,phi(x));
```

输入和谐一部分。

## 说明/提示

在规定的时间（3 秒）内完成如此多项的加法似乎有点慢，并且精度似乎也有点差。

为了加速和式的收敛，请注意下面的式子：

$$\frac{1}{k(k+1)}=\frac{1}{k}-\frac{1}{k+1}$$

上式指出了 $\phi(1)=1$。我们可以采用 $\phi(x)-\phi(1)$ 快速收敛，并且精度也提高很多。

这样的快速找收敛是一种很好的方式，当然可能会有更快的。

下面的式子也许对判定收敛的项数更有帮助（其中 $k>1$ 并且 $r\ge 1$）：

$$\sum_{k=n}^{\infty}\frac{1}{k^r}<\int_{n-1}^{\infty}\frac{\mathrm dx}{x^r}$$

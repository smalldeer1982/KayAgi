# Largest Submatrix

## 题目描述

#### 设A是由0和1组成的N（1≤N≤25）阶矩阵，定义A的子矩阵为全部由1组成的小矩阵，请确定A的最大子矩阵的元素个数。

## 输入格式

#### 第一行为一个正整数T，表示有T组数据。下面给出T个矩阵，并且每个矩阵之间有一个空行。

## 输出格式

#### 分别输出每组数据找到的最大子矩阵的元素个数,两个输出之间间隔一个空行。

## 样例 #1

### 输入

```
1
10111000
00010100
00111000
00111010
00111111
01011110
01011110
00011110
```

### 输出

```
16
```

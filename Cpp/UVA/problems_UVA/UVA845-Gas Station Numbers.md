# Gas Station Numbers

## 题目描述

### 题目翻译
许多加油站使用塑料的数字牌来标识油价。这种数字牌中，$6$ 和 $9$、$2$ 和 $5$ 可以看作互相颠倒。

一个加油站已经用完了所有的数字牌，但是员工还想继续提高油价。她可以改变数字的顺序，也可以按照上文颠倒一些数字。在提高时，必须保证整数部分的位数和小数部分的位数**都不变**。给出原来的油价，请你给出油价经过提高之后的**最小值**或者报告无解。

## 输入格式

输入数据包括若干行，$1$ 行 $1$ 个小数表示原来的油价。**特别地，当 $1$ 行只包括小数点时，输入数据结束。**

## 输出格式

输出数据包括若干行，$1$ 行输入数据对应 $1$ 行输出数据。如果有解，$1$ 行 $1$ 个小数表示油价经过提高之后的**最小值**。如果无解，$1$ 行 $1$ 个字符串 `The price cannot be raised.`。

## 说明/提示

对于 $100 \%$ 的数据，表示原来的油价的小数中，有 $2$ 至 $30$ 个数字，小数点后有且只有 $1$ 位数字，当且仅当原来的油价小于 $1$ 时会有前导零。

## 样例 #1

### 输入

```
65.2
76.7
77.7
.
```

### 输出

```
65.5
77.6
The price cannot be raised.
```

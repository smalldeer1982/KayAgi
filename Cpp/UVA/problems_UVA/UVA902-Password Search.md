# Password Search

## 题目描述

给出一个字符串，请问：出现次数最多的长度为 $n$ 的子串是什么？

## 输入格式

有多个测试样例。

对于每一个测试样例，输入一个正整数 $n$ 和一个字符串。

## 输出格式

对于每个测试样例，输出一行，一个字符串，代表所求的答案。

By [dengzijun](https://www.luogu.com.cn/user/387836)

## 样例 #1

### 输入

```
3 baababacb
```

### 输出

```
aba
```

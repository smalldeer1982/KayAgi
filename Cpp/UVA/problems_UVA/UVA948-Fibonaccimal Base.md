# Fibonaccimal Base

## 题目描述

众所周知， 斐波那契数列的前两位是 `0` 和 `1`。 然后的每一个数都等于前两个数的和。 

然而任意一个非负数都能由多个（也可以是一个）斐波那契数的和表示。

现在给你 `N` 组数据 `x`， 让你求斐波那契数的和所组成的 `x` 的二进制 (其中结果的二进制第 `k` 位表示第 `k` 个斐波那契数)。

## 输入格式

第一行为一个整数 `N`， 表示共有 `N` 组数据。

接下来的 `N` 行， 每行一个整数 `x`。

## 输出格式

共 `N` 行， 每行一个二进制数， 表示由斐波那契数的和组成的对应的 `x`。

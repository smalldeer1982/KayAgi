# Safe Salutations

## 题目描述

一个圆上的边上有 $n$ 对点（即 $2n$ 个点），你需要连 $n$ 条直线将这些点连起来（每次连两个点）。

问一共有多少种连线方式使得这些直线没有在圆内相交。

## 输入格式

包含多组数据，每行一个数 $n(1\le n\le10)$ 。

## 输出格式

每行一个数，方案总数。

## 样例 #1

### 输入

```
4
```

### 输出

```
14
```

## 翻译
**题目**

请你求出切 $N$ 刀最多能个有几个面。

**输入格式**

对于每组数据，一行 $N(0 \leq N \leq 210000000)$ 遇到负数时表示结束。

**输出格式**

对于每组数据，输出一个整数，代表能切出的最大面的数量，不含前导或尾随空格。

感谢 @ylinxin2010 提供的翻译。

## 样例 #1

### 输入

```
5
10
-100
```

### 输出

```
16
56
```

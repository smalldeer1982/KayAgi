# The Knights Of The Round Table

## 题目描述

# 题目背景

亚瑟王计划在他的新房间里造一张圆桌，但是他希望用玻璃建造房顶，这样就能在房间里享受到阳光了。他还希望他的桌子能看起来尽管闪闪的，尤其是在正午。所以，他需要他的桌子能完全被阳光照射。但是，兰斯洛特需要屋顶的玻璃部分是三角形的（没人知道他为什么要这么做，可能是他订立了什么乱七八糟的誓约吧）。于是，房间里会有一个三角形区域在正午十分有阳光直射，而桌子必须恰好在这个区域中。

现在，亚瑟王想在桌子在这个三角形区域中的前提下做出尽可能大的桌子。可惜亚瑟王的几何很差，于是他请加拉哈德来帮忙（兰斯洛特也很擅长几何，但是亚瑟王怕他又想出什么鬼点子，所以他请了加拉哈德）。

你能帮助加拉哈德（显然他并不擅长计算机）写一个程序来计算出能放在这个三角形区域中的最大的桌子的半径吗？你可以假设这个桌子是个完美的圆形。

## 输入格式

房间的数量是任意多个。每个房间的三角形区域由三个实数 ($a$, $b$, $c$) 表示，三角形的大小小于 $1\times10^6$ 且保证 $\max(a,b,c) \le (a + b + c) \div 2$。

你需要一直读取直到末尾。

## 输出格式

对于任意一间房间，你需要输出如下一行文字：

```text
The radius of the round table is: r
```

$r$ 表示能放在这个三角形区域中的最大的桌子的半径，保留 $3$ 位小数输出。

## 样例 #1

### 输入

```
12.0 12.0 8.0
```

### 输出

```
The radius of the round table is: 2.828
```

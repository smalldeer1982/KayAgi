# Flip Sort

## 题目描述

**本题有多组数据**。

给定一个长度为 $N$ 的数组（$N$ $\le$ $1000$），你需要对这个数组进行冒泡排序，求共需要交换几次。

## 输入格式

**多组数据**，每组数据有两行。第一行是一个整数 $N$ ，第二行有 $N$ 个整数，表示需要处理的数组。数据以 `EOF` 结尾。

## 输出格式

`Minimum exchange operations : ans`。其中 `ans` 表示答案。

## 样例 #1

### 输入

```
3
1 2 3
3
2 3 1
```

### 输出

```
Minimum exchange operations : 0
Minimum exchange operations : 2
```

# Wavio Sequence

## 题目描述

Wavio 是一个整数序列，它有以下几个有趣的特性：

+ Wavio 的长度是奇数，这里设其长度 $L = 2 \times n + 1$。

+ Wavio 的前 $n + 1$ 个数严格单调上升。

+ Wavio 的后 $n + 1$ 个数严格单调下降。

+ Wavio 中不存在两个相邻且相等的数。

举个例子，数列 $1,2,3,4,5,4,3,2,0$ 是一个长为 $9$ 的 Wavio，而 $1,2,3,4,5,4,3,2,2$ 不是一个合法的 Wavio。在这个问题中，给定一个整数序列，请你输出其最长的 Wavio 子序列的长度。

考虑数列 $1,2,3,2,1,2,3,4,3,2,1,5,4,1,2,3,2,2,1$，此时最长的 Wavio 子序列是 $1,2,3,4,5,4,3,2,1$，所以输出 $9$。

## 输入格式

**本题多测**。输入文件的子问题数少于 $75$ 个，输入以文件结束为结束标志。

每个子问题第一行输入一个整数 $N$（$1 \le N \le 10000$），下一行 $N$ 个整数。

## 输出格式

对每个子问题，输出其最长 Wavio 子序列的长度。

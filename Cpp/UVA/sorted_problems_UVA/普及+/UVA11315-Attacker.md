# Attacker

## 题目描述

## 题目大意
给定一个 $n \times m$ 的棋盘，上面有 $k$ 个攻击者。  
第 $i$ 个攻击者位于 $(x_i, y_i)$ 处，攻击范围为 $r_i$。
当且仅当满足 $|x - x_i| + |y - y_i| ≤ r_i$ 时，$(x, y)$ 处的方格能被第 $i$ 个攻击者攻击。  
计算至少被一个攻击者攻击的方格的数量。

## 输入格式

有多个输入样例，用 $0$ 表示输入的结束。第一行包含三个整数 $n$，$m$，$k$。接下来的 $k$ 行中，每行包含三个整数 $x_i$，$y_i$，$r_i$，表示每个攻击者的位置和攻击范围。

## 输出格式

一个数，表示至少被一个攻击者攻击的方格的数量。

## 样例 #1

### 输入

```
4 4 3
1 1 1
3 1 1
3 3 1
1 10 1
1 1 1
0
```

### 输出

```
Case 1: 10
Case 2: 2
```

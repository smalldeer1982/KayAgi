# Sigma Function

## 题目描述

$\sigma$ 函数是一种数论中的有趣函数。这个函数记某个数的所有因数之和，例如 $\sigma(24)=1+2+3+4+6+8+12+24=60$。对于较为小的数字来说直接计算很容易但对于大数就不是那么回事了。然而，数学家找到了一种好方式计算它：对 $n$ 作唯一分解
$$n=\prod_{k=1}^np_n^{e_n}$$
而后我们就能拿到 $\sigma(n)$ 的表达式
$$\sigma(n)=\frac{p_1^{e_1+1}-1}{p_1-1}\cdot\frac{p_2^{e_2+1}-1}{p_2-1}\cdot\cdots\cdot\frac{p_n^{e_n+1}-1}{p_n-1}=\prod_{k=1}^{n}\frac{p_k^{e_k+1}-1}{p_k-1}$$
自然，我们可以对 $\sigma(n)$ 分析它是奇数还是偶数，求 $\sigma(1)\sim\sigma(n)$ 有多少个是偶数。

## 输入格式

多组数据，每组数据一个整数 $n(0<n\le 10^{12}+1)$，以 $n=0$ 结束。

## 输出格式

输出对于 $\sigma(1)\sim\sigma(n)$ 中有多少个是偶数。

## 样例 #1

### 输入

```
3
10
1000
0
```

### 输出

```
1
5
947
```

# What is the Median?

## 题目描述

中位数在统计学中起着重要的作用。根据定义，中位数是一个将数组分成两部分的值。在这个问题中，你需要确定一些长整型数的当前中位数。假设我们有五个数 $\{1,3,6,2,7\}$，在这种情况下，$3$ 是中位数，因为它的左右各有两个数。分别是 $\{1,2\}$ 和 $\{6,7\}$。如果有偶数个值，比如 $\{1,3,6,2,7,8\}$，单个值无法将数组分成相等的两部分，因此我们取中间值的平均数 $\{3,6\}$。因此，中位数将是 $(3 + 6) / 2 = 4.5$。在这个问题中，你只需要输出整数部分，而不是小数部分。所以，按照这个问题的要求，中位数将是 $4$。

## 输入格式

输入文件由一系列整数 $X$（$0 \le X < 2 ^ {31}$）组成，整数的总数 $N$ 小于 $10000$。数字可能有前导或后导空格。

## 输出格式

对于每个输入，打印当前的中位数值。

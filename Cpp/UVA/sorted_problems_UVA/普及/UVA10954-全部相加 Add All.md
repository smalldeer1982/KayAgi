# 全部相加 Add All

## 题目描述

## 题意翻译
给你 $n$ 个数，求将这 $n$ 个数合并成一个数的成本（成本即为此次相加之和）的最小值。

## 输入格式

本题多测，每组数据的第一行为 $n$，然后一行 $n$ 个整数，表示需要合并的数据。当 $n=0$ 时，表示输入结束。

## 输出格式

一行一个整数，表示成本的最小值。

## 说明/提示

对于 $100\%$ 的数据，$1 \leq n \leq 5 \times 10^3$，$1 \leq a_i \leq 10^{5}$。

## 样例 #1

### 输入

```
3
1 2 3
4
1 2 3 4
0
```

### 输出

```
9
19
```

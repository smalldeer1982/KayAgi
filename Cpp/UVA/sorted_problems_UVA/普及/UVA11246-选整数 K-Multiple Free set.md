# 选整数 K-Multiple Free set

## 题目描述

有 $T$ 组询问，每组询问给定两个参数 $n$ 和 $k$，要求在 $\boldsymbol{1 \sim n}$ 中选择尽量多的数，组成一个集合，集合中的任意元素不能为另一元素的 $k$ 倍。求集合最多能有多少个元素。

## 输入格式

第一行一个整数 $T$ 表示测试数据的组数。

接下来 $T$ 行，每行两个整数 $n, k$。

## 输出格式

输出一行包含一个整数，表示这个询问的答案。

## 样例 #1

### 输入

```
3
10 2
100 2
1000 2
```

### 输出

```
6
67
666
```

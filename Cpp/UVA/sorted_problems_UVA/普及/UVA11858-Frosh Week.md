# Frosh Week

## 题目描述

给定一个长度为 $n$ 的数列，求其中逆序对的个数。

多测。

## 输入格式

对于每组测试数据：

第 $1$ 行输入 $n$。  
第 $2$ 行至第 $n+1$ 行，每行输入一个数，代表这个数列的一项。

## 输出格式

对每组测试数据，输出一行表示数列中逆序对的个数。

------------
翻译由 @[a1ioua](/user/296967) 提供。

## 样例 #1

### 输入

```
3
3
1
2
```

### 输出

```
2
```

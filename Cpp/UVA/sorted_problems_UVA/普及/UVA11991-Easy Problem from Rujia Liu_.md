# Easy Problem from Rujia Liu?

## 题目描述

给出一个包含  $n$ 个元素的数组，你需要回答若干询问，每次询问两个正整数  $k$ 和  $v$，输出从左到右第  $k$ 个  $v$ 的下标。

## 输入格式

输入包含多组数据。每组数据第一行为两个整数  $n$ 和  $m$ （ $1 \leq n,m \leq 10^5$），第二行包含  $n$ 个不超过  $10^6$ 的正整数，即待查询的数组。以下  $m$ 行每行包含两个整数  $k$ 和  $v$（ $1\leq k \leq n$， $1 \leq v \leq 10^6$）。输入结束标志为 EOF。

## 输出格式

对于每个查询，输出查询结果。如果不存在，输出  $0$。

## 样例 #1

### 输入

```
8 4
1 3 2 2 4 3 2 1
1 3
2 4
3 2
4 2
```

### 输出

```
2
0
7
0
```

# I Love Big Numbers !

## 题目描述

给定一个整数 $n$，求 $n!$ 的各个数位数字之和。

## 输入格式

**输入有多组数据。**

对于每组数据，输入一个整数 $n$。

## 输出格式

对于每组数据，输出一个整数 $ans$ 表示 $n!$ 的各位数字之和。

## 说明/提示

为描述方便，设 $|x|$ 表示 $x$ 的位数。

对于 $100\%$ 的数据，$1\le n\le1000$，$1\le |n!|\le2^{31}-1$

## 样例 #1

### 输入

```
5
60
100
```

### 输出

```
3
288
648
```

# The Roots

## 题目描述

一个多项式方程可以表述成这样：

$$a_nx^n+a_{n-1}x^{n-1}+\cdots+a_1x^1+a_0=0$$

这里未知数是 $x$，而 $a_n,a_{n-1},\cdots,a_1,a_0$ 是系数，因此为限定一个多项式方程我们只需要限定它们的系数。多项式的根是当这个多项式为 $0$ 时 $x$ 的值 $z$。你可以认为多项式方程 $n$ 就有 $n$ 个根，并且它们严格不同。

## 输入格式

多组数据。每组数据数据第一行一个整数 $n$，而后是 $(n+1)$ 个浮点数 $a_n,a_{n-1},\cdots,a_0$，代表多项式的系数。以 $n=0$ 结束（把程序写的鲁棒一些！！！）。

$n\le 5,|a_i|\le 10^9,|z|\le 25$。

## 输出格式

对于每组数据，输出 `Equation S`（第 $S$ 个方程），而后是 $N$ 个浮点数，都是输入的方程的根，从小到大排列，保留四位小数。

## 样例 #1

### 输入

```
2 1 -9.5750000000 -179.5585140000
4 1 53.3120000000 958.2510390000 6677.1763593480 15733.6254955064
```

### 输出

```
Equation 1: -9.4420 19.0170
Equation 2: -22.8060 -18.1170 -6.7350 -5.6540
```

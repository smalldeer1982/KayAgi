# The longest constant gene

## 题目描述

给$n$个仅仅由A、C、G、T组成的字符串，求这$n$个字符串的最长公共子序列长度。

多组数据。

## 输入格式

一个正整数$T$表示数据组数

对于每一组数据，一个正整数$n$表示字符串个数，接下来$n$个字符串。字符串总长度不超过$10^6$，字符串仅包含A、C、G、T。

## 输出格式

对于每一组数据，输出一行一个正整数，表示最长公共子序列长度。

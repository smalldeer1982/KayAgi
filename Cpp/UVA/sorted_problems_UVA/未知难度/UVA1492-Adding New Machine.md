# Adding New Machine

## 题目描述

一个W*H的矩形里有N个矩形旧机器，现在要添加一个新机器，这个新机器需要一行内M个连续的单元，且不能占用有旧机器的单元，问选择新机器不同位置的数量。

## 输入格式

有多个样例（不超过50个）。

每个样例第一行包含4个整数W,H,N,M，意义如上述。

接下来N行每行四个整数 Xi1,Yi1,Xi2,Yi2，表示第i号旧机器的位置。

## 输出格式

对于每个样例输出一个整数，表示此样例将机器成功放置的方法数。

## 说明/提示

对于100%的数据，1<=W,H<=10000000，0<=N<=50000，1<=M<=1000，1<=Xi1<=Xi2<=W，1<=Yi1<=Yi2<=H。

数据保证不会有一个单元同时放置两个以上的旧机器。

### 输入样例

3 3 1 2

2 2 2 2

3 3 1 3

2 2 2 2

2 3 2 2

1 1 1 1

2 3 2 3

### 输出样例

8

4

3

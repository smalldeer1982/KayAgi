# 乱糟糟的网络 Network Mess

## 题目描述

有一棵$n(n\leq50)$个叶子节点的无权树。输入两两叶子的距离，恢复出这棵树并输出每个非叶子节点的度数。

## 输入格式

输入包含多组数据。

每组数据第一行为一个整数$n$。
接下来的$n$行，每行$n$个整数，$a_{ij}$表示$i$节点与$j$节点之间的距离。

输入当$n=0$时结束。

## 输出格式

对于每组数据，输出一行为答案。

# 【样例输入】
```
4
0 2 2 2
2 0 2 2
2 2 0 2
2 2 2 0
4
0 2 4 4
2 0 4 4
4 4 0 2
4 4 2 0
2
0 12
12 0
0
```
# 【样例输出】
```
4
2 3 3
2 2 2 2 2 2 2 2 2 2 2
```

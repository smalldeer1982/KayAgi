# Scanner

## 题目描述

# 题目大意

有一款图像处理器，它能处理长为 $15$ 宽为 $10$ 的图像。它由 $4$ 款传感器组成：

$1$ 号传感器，扫描每一行，并且报告每行有多少个像素（共报告 $10$ 个数据）。

$2$ 号传感器，扫描每一个左下到右上的斜线，并且报告每个斜线上有多少个像素（共报告 $24$ 个数据）。

$3$ 号传感器，扫描每一列，并且报告每列有多少个像素（共报告 $15$ 个数据）。

$4$ 号传感器，扫描每一个右下到左上的斜线，并且报告每个斜线上有多少个像素（共报告 $24$ 个数据）。

（具体的可以看 PDF 中的图片）。

现给你每个传感器的数据，请求你输出对应的图像。

## 输入格式

第一行一个整数 $n$，表示要处理的图像数量。

对于每组数据，输入 $4$ 行数据，其中，第 $i$ 行表示 $i$ 号传感器的数据,对于每个传感器报告数据的顺序，看 PDF 中的图片。

## 输出格式

对于每组数据，输出一个长 $15$ 宽 $10$ 的图像，其中像素点用 `#` 表示，不是像素点的用 `.` 表示。如果有不止一张图像满足给出的数据，则输出一张长 $15$ 宽 $10$ 的点图。

每组输出之间以一个空行分隔。

# Mirror Maze

## 题目描述

## 题意
你有一个n行m列的镜子迷宫，每格有一个镜子或一个黑洞或什么都没有。镜子可以摆放为45°(/)或135°(\\)；黑洞可以把激光完全吸收。

迷宫的边缘只有两个开口，其他都是黑洞。你要改变镜子的方向，使得激光可以从一个开口垂直入射走到另一个开口。

## 输入格式

多组数据。

每组数据第一行两个正整数m和n，然后n行每行m个字符，\*代表黑洞，/或\\代表镜子，.代表空。

一行-1 -1表示结束。

## 输出格式

对于每组数据，输出n行m列表示镜子排好后的迷宫。每组数据用一个空行分隔。 （样例输出中应加一个空行）

## 样例 #1

### 输入

```
4 5
****
*\/*
*./.
*..*
*.**
4 4
*.**
*.\*
*\\*
**.*
-1 -1
```

### 输出

```
****
*/\*
*.\.
*..*
*.**
*.**
*.\*
*\\*
**.*
```

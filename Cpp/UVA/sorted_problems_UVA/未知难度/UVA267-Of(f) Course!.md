# Of(f) Course!

## 题目描述

在近几周里，Fat Chance 航空公司有数架飞机偏离航线。当最后一个错误几乎把一架飞机飞到了敌人的领空，一项调查显示它们的导航技术没有正确地考虑风速和风向。
   
为了使新手能更容易上路，航空公司让你的团队制定一个程序能够为飞行员提供合适的航向和风速。

对于每一组数据，你将得到飞机的航向，飞机在风中的真实速度（相对于空气的速度），风速和风向。飞行员需要每次飞机的航向和有效地面速度（飞机相对于地面在所需航向上移动的速度）。

所有速度以段为单位，所有方向以度（$0°\leq x<360°$）为单位（$0°$=北，$90°$=东，$180°$=南，$270°$=西）。

## 输入格式

输入将由每一次的飞行组成。每次飞行由风速、风向、飞机的航向和飞机在风中的真实速度组成，均以自由格式给出，各个数据之间用一个空格隔开。

## 输出格式

每一次飞行输出六行。六行中的格式都为下面的示例。前四行输出输入的数字，其余两行表示为每个路段的航向和有效地面速度，并且要用类似于下面示例的格式。每个数字必须精确到$\frac{1}{100}$。

注意：输出不应该包含-0.00或360.00的方向值，或-0.00的速度值

## 样例 #1

### 输入

```
15.0 290.0 260.0 100.0
15.0 270.0 135.0 200.0
28.0 290.0 5.0 195.0
```

### 输出

```
WIND SPEED 15.00
WIND DIRECTION 290.00
DESIRED COURSE 260.00
TRUE AIRSPEED 100.00
AIRCRAFT HEADING 264.30
GROUND SPEED 86.73
WIND SPEED 15.00
WIND DIRECTION 270.00
DESIRED COURSE 135.00
TRUE AIRSPEED 200.00
AIRCRAFT HEADING 138.04
GROUND SPEED 210.33
WIND SPEED 28.00
WIND DIRECTION 290.00
DESIRED COURSE 5.00
TRUE AIRSPEED 195.00
AIRCRAFT HEADING 357.03
GROUND SPEED 185.87
```

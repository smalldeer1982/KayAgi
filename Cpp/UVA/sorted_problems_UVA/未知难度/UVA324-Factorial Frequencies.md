# Factorial Frequencies

## 题目描述

给定一个数 $n$，求 $n!$ 中各个数字的出现次数。

## 输入格式

有多组数据。

每一组数据有一个数 $n$。

最后一行是 $0$ 表示结束。

## 输出格式

见样例（要看 PDF，落谷 RMJ 有问题）。

## 样例 #1

### 输入

```
3
8
100
0
```

(0) 0
(5) 0

(0) 2
(5) 0

# Long Multiplication

## 题目描述

对于每一组 $(x,y)$ ，输出 $x \times y$ 的竖式乘法.

对于每一行，输出 $x$ 乘上每一个数位 $y$ 的结果，在每一个竖式的结尾输出 $x \times y$.

## 输入格式

每一组数据输入 $(x,y)$ 。最后以 $0$ 结尾

## 输出格式

1.不写乘号

2.第一行的"-"的个数为 $max(x,y)$ 的数位.

3.第二行的"-"的个数为 $x \times y$ 的数位

题目保证 $1 \le x,y \le 10^9$

**如果乘数中有 $0$ ，直接输出 $0$ 即可**

### 输入

```

135 46
12345 862
0 123456789
0
```

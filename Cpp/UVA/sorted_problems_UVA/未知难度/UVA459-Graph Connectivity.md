# Graph Connectivity

## 题目描述

有一个由几个节点组成的图，每个节点用**连续且唯一**大写字母编号，每个节点在初始时互不相连（即：图在初始时没有边）。接下来多次连接两个节点，最后求图中有多少个连通图。

## 输入格式

本题有多个测试样例。

第一行输入一个数，代表本题测试样例数量。

接下来，对于每个测试样例：

- 第一行有一个大写字母，代表图中节点字母的最大值。

- 接下来的每行，一行有两个大写字母，代表连接这两个大写字母对应的节点。

- 一个测试样例的输入以空行结束。

## 输出格式

对于每个测试样例，输出一个数，代表最后连通块的数量。

需要注意，每个测试样例的输出以空行分隔（UVA 的毒瘤输出）。

# Numeric Puzzles Again!

## 题目描述

## 翻译
### 题目大意
一种拼图游戏，其中每块拼图的形状都不一样。就像下面的拼图：
~~~~
11223
11223
12233
11233
11133
~~~~
由以下拼图组成：
~~~~
11
11
1
11
111

  22
  22
22
  2

  3
  3
33
33
33
~~~~
你要编写一个程序，使其能够使用这些拼图组装成一个完整的**正方形**。

---------------------------

## 输入格式

输入由多组数据组成。

第一行输入**完整**拼图的长度 $l$ 。（$l \le 20$）

第二行输入拼图的个数$m$。($m \le 9$)

往下多行描述每块拼图。每块拼图由数字组成。

每组数据的结尾为“#”，若输入结束则输入“0”。

## 输出格式

输出完整的拼图。

### 输入

```
7
6

33
3333
33
33
7 7
7 7
7777
88888
6
666
66
22
222
2
5
55
55

0
```


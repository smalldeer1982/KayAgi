# Art Gallery

## 题目描述

新建的的巴尔干合作中心非常具有未来主义。其艺术画廊采用了多边形的结构（不一定是凸多边形）。当组织一场大型展览时，监视所有展品是一个重要的安全问题。你的任务是编写一个程序，对于给定的画廊，找到一个地板区域，使得画廊墙上的每个点都可见。如图（见原题面），第一个图示给出了画廊在某个坐标系中的地图。所需区域在第二个图示中用阴影标出。

## 输入格式

输入文件第一行读入一个整数 $T$，表示测试数据组数。接下来若干行是对每组数据的描述。每组数据第一行读入一个整数 $N$ $(5 \le N \le 1500)$。接下来的 $N$ 行中，每行包含一个多边形顶点的坐标 - 两个以空格隔开的 $16$ 位整型的整数。相邻两组测试数据间无空行。

## 输出格式

对于每组数据，你需要在单独一行输出所需的区域面积（该数字应四舍五入到小数点后第二位）。

# 信息解码 Message Decoding

## 题目描述

考虑下面的01串序列：
```
0，00，01，10，000，001，010，011，100，101，110，0000，0001，···，1101，1110，00000，···
```
首先是长度为1的串，然后是长度为2的串，依此类推。如果看成二进制，相同长度的后一个串等于前一个串加1。注意上述序列中不存在全为1的串。你的任务是是编写一个解码程序。

## 输出格式

感谢@hedwig 提供的翻译

## 样例 #1

### 输入

```
TNM AEIOU 
0010101100011 
1010001001110110011 
11000 
$#**\ 
0100000101101100011100101000
```

### 输出

```
TAN ME 
##*\$
```

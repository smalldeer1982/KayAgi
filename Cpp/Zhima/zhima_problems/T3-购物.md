# 题目描述

## 题目描述


你来到了一家商店，商店里有 nnn 个物品，购买第 iii 个物品可以获得 wiw_iwi​ 的快乐度。


商店为了推广产品，购买第 iii 件商品时：



- 若这是你第一次购买第 iii 件商品，需要支付 111 元。

- 否则，需要支付 kik_iki​ 元。



你现在只带了 CCC 元钱，求你能获得的最大快乐度是多少。


**注意：你并不需要花光这 CCC 元钱。**


## 输入格式


第一行，两个正整数 n,Cn, Cn,C。


接下来 nnn 行，每行两个正整数 ki,wik_i, w_iki​,wi​。


## 输出格式


共一行，一个正整数表示你能获得的最大快乐度。


## 输入输出样例


### 输入样例 #1


```
3 9
2 9
1 4
2 6

```

### 输出样例 #1


```
46

```

### 样例解释 #1


购买 444 件商品 111，111 件商品 222，111 件商品 333，价格为 1+3×2+1+1=91 + 3 \times 2 + 1 + 1 = 91+3×2+1+1=9，能获得的快乐度为 9×4+4+6=469 \times 4 + 4 + 6 = 469×4+4+6=46，可以证明不存在更优方案。

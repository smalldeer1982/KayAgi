# 题目描述

## 题目描述


给你一个网格图，有些格子是黑色的。你要用若干零片不重不漏地拼满白色格子。你有以下两种零片：




一个 1×31 \times 31×3 的矩形。有 aaa 块。




一个 2×22 \times 22×2 的矩形去掉左上角的一格。有 bbb 块。





注意：




**这些零片允许任意角度地旋转**。




你不需要用完 (a+b)(a + b)(a+b) 个零片，可以有剩余。





你需要输出有多少种拼图方案。


## 输入格式


本题有多测。


第一行两个正整数 c,Tc, Tc,T 分别表示测试点所在子任务编号和测试数据数量（1≤T≤101 \le T \le 101≤T≤10）。


在每一组测试数据中：




第一行，四个非负整数 n,m,a,bn, m, a, bn,m,a,b。




接下来 nnn 行，每行 mmm 个字符。表示这个 n×mn \times mn×m 的网格。其中 `.` 代表白色，`#` 代表黑色。





## 输出格式


对于每组测试数据，输出一行一个非负整数，表示有多少种拼图方案。
